Palo Alto Networks 将以 1.73 亿美元收购 RedLock，以增强云安全技术

原文：https://web.archive.org/web/https://techcrunch.com/2018/10/03/palo-alto-networks-to-acquire-redlock-to-beef-up-cloud-security/

Palo Alto Networks 将以 1 . 73 亿美元收购 RedLock 以增强云安全

帕洛阿尔托网络公司于 2005 年在防火墙时代成立。众所周知，企业很久以前就已经超越了防火墙的舒适界限，像 Palo Alto 这样的供应商现在也已经开始在云中保护数据。为此，公司宣布打算支付 1.73 亿美元收购 RedLock ，这是一家帮助公司确保其云实例被锁定和安全的早期创业公司。

云供应商负责保护他们自己的基础设施，在大多数情况下，主要供应商都做得很好。他们做不到的是从他们自己手中拯救他们的客户，这就是像雷德洛克这样的公司出现的原因。

正如我们一次又一次看到的那样，数据已经暴露在亚马逊 S3 这样的云存储服务中，这不是因为亚马逊本身的任何过错，而是因为错误的配置使数据暴露在开放的互联网中。RedLock 会观察这样的配置，并在出现问题时向公司发出警告。

当该公司一年前从隐身状态中崛起时，公司创始人兼首席执行官 Varun Badhwar告诉 TechCrunch 这是亚马逊共同责任模式的一部分。Badhwar 去年告诉 TechCrunch，“他们有责任保护物理基础设施，但最终是客户负责保护内容、应用程序和防火墙设置。”。

Badhwar 在关于此次收购的视频采访中表示，他们一直专注于帮助开发者安全可靠地构建云应用，无论是亚马逊网络服务、微软 Azure 还是谷歌云平台。“我们认为[红锁]是保龄球馆的护栏或缓冲道，只是不让某人得到那个球，从安全的角度来看，只是确保我们不偏离最佳实践，”他解释说。

他补充说:“我们构建了一个完全基于云的技术平台，可以快速实现价值，因为客户只需通过 API 打开它，我们喜欢向客户展示如何安全地迁移到公共云。”

此次收购也将与云基础设施安全初创公司的 Evident.io 完美契合，该公司于 3 月份以 3 亿美元收购了。Badhwar 认为，客户将受益于 Evident 的合规能力与 Red Lock 的分析能力相结合，以提供更完整的云安全解决方案。

RedLock 于 2015 年推出，已融资 1200 万美元。对于信任这家初创公司的投资者来说，1.73 亿美元的收购似乎是一个巨大的回报。
