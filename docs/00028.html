收购计划失败后，Rethink Robotics 关闭

原文：https://web.archive.org/web/https://techcrunch.com/2018/10/04/rethink-robotics-closes-after-acquisition-plans-fall-through/

我就在这几页上说过:如果硬件很难，机器人几乎是不可能的。本周 Rethink Robotics 的关闭更好地证明了这一真理。这家总部位于波士顿的公司在自动化领域享有盛誉，它生产的一对机器人已经成为研究机构和仓库的中流砥柱。

在提供给 TechCrunch 的一份声明中，Rethink 用直白的语言阐述了自己的灭亡。“Rethink Robotics 关门了，”上面写道。“我们很早就向市场推出了领先于时代的创新产品，不幸的是，我们没有取得预期的商业成功。对该公司的收购计划在最后一刻告吹了。Rethink Robotics 的所有员工都被其他机器人公司积极招聘。”

关于罗德尼·布鲁克斯 】领导的初创公司的传言已经在机器人行业流传了一段时间，但这并没有减少这次关闭的影响。在该公司长达十年的存在中，它生产了 Baxter 和 Sawyer，这是协作类别中最具标志性的两个机器人。

“罗德尼·布鲁克斯和 Rethink Robotics 是协作机器人领域的重要先驱，”自动化推进协会/机器人行业协会主席杰夫·伯恩斯坦在提交给 TechCrunch 的一份声明中说。“就像所有领域的初创企业一样，并非一切都一帆风顺。有时早期产品并不完全像你希望的那样好，有时新的想法不会立即被市场接受，有时其他市场参与者会更快地获得关注。这些因素中的部分或全部可能已经影响了 Rethink。”

鉴于维持这一类别的难度，或许这一切都不应该令人感到意外。据 Crunchbase 的统计，这个 100 人的团队在成立期间筹集了近 1.5 亿美元，但这最终不足以让它继续走下去。如果这是 2012 年，亚马逊可能会在 Kiva systems 上建立自己的机器人业务时关注该公司。

两年后，该公司可能非常适合安迪·鲁宾(Andy Rubin)刚刚起步的谷歌机器人公司(Google Robotics ),该公司一直专注于工厂自动化。即使是现在，软银的雄厚资金也可以在很大程度上维持它的运营——毕竟，这个金融巨头非常乐意从谷歌手中接过波士顿动力公司，将其加入已经包括胡椒制造商、 阿鲁迪巴机器人公司的投资组合。

然而，最终，听起来该公司的灭亡是多种因素的综合结果。当然，这似乎与 2014 年总部位于湾区的 Willow Garage 的关闭有一些相似之处，其 PR2 此前是世界各地大学机器人实验室的熟悉景象。

从 Redwood Robotics 这样的衍生公司到 Fetch 这样由员工领导的公司，再到机器人操作系统 ROS 的继续存在，Willow Garage 的遗产中肯定有希望被发现。
