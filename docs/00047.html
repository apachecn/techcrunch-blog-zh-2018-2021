从谷歌的Pixel 3事件中可以期待什么

原文：https://web.archive.org/web/https://techcrunch.com/2018/10/06/what-to-expect-from-googles-pixel-3-event/

苹果、亚马逊和微软已经举办了他们的秋季大型活动，现在轮到谷歌了。在过去几年中，随着该公司继续进军各种硬件类别，10月份的活动已经成为一个越来越重要的平台。说真的，这是谷歌在假期前大出风头的最后机会。

毫无疑问，Pixel 3将是这场展览的焦点。谷歌对此毫不讳言——从官方批准的预览和尼亚加拉瀑布大小的泄露来看，很明显我们已经看到了手机所能提供的东西。 当然，这年头，大事讲的远不止这些像素。去年这个时候，该公司推出了一系列智能扬声器家用产品，包括家用Mini和Max。我预计在这方面会看到大量的新闻，这次也是如此。

该活动将于10月9日在纽约市拉开帷幕。当然，我们会去的。同时，这里是我们认为我们将会看到的，从最明显的开始。

希望在手机方面会有一些惊喜，但我不会指望它。我们已经从每个可以想象的角度看到了 Pixel 3 和 Pixel 3 XL ，无论是在静态图像还是视频中。事实上，谷歌在泄密问题上给了三星一次机会。

Pixel 3xl将采用迄今为止我们见过的最大的剪纸之一来拥抱Android的notch love。另一方面，像素3可以完全跳过该凹口。一种新的颜色显然也在酝酿之中——浅绿色，以配合最近发布的谷歌Home薄荷色。

据说这些手机将坚持采用单一的后置摄像头配置，这在过去一直很好，但一些新的增强现实技巧显然正在工作中，以帮助展示ARCore的最新产品。HTC推出的挤压界面也通过谷歌日本的一个真正可爱的T2官方视频得到了证实。预计盒子里还会有一副有线Pixel Bud式耳机。 新手机也应该有自己的充电支架——类似于三星(或者就此而言， AirPower )最近推出的。有趣的是，这个支架的设计本质上是将Pixel变成一个临时的智能显示器——类似于亚马逊通过Show Mode对其Fire平板电脑所做的事情。

在今年早些时候与许多第三方合作后，谷歌似乎准备在智能显示游戏中投入更多内容。总部已经在一系列泄密事件和FCC批准中露了脸，谷歌最终与Echo展开正面较量。我们知道，鉴于最近增加的颜色，家庭迷你可能不会得到全面更新，但第一代家庭似乎早就应该进行翻新，希望它看起来不像一个空气清新剂。

与Hub一样，新的Chromecast也通过了FCC的审查，尽管关于新功能的信息似乎很少。鉴于最初的像素芽不冷不热的接受，希望我们能看到这方面的更新。一款新的Pixelbook似乎也完全可行，还有传言中的可转换Pixel Slate平板电脑，为其Chrome OS产品增添了又一款高端设备。
