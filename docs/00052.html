最近，我多次听到各种关注苹果的播客的主持人说，适应新的 iPhone XS Max 就像“回家”对于这些所谓的“Plus Club ”(这个古怪的名字指的是过去选择 Plus 型号的用户群)的成员来说，回到一台拥有如此大尺寸显示屏的设备会立即感到熟悉，甚至舒适。

在使用更小的 5.8 英寸 iPhone X 一年后，我也体验到了这些舒适和熟悉的感觉。我已经测试了近两周的 iPhone XS Max，这是苹果提供给我的一款评测设备，每次我使用它时都会想起为什么我会爱上 Plus 型号。俗话说，越大越好。

虽然苹果最新 iPhone 的头条新闻是更好的摄像头系统，但对我来说，作为一个视障人士，关键的故事是我回到了最大屏幕的 iPhone。XS Max 和 Plus 一样令人愉快(也一样容易使用),通过包含 Face ID 和窄边框显示屏而变得更好。

根据尺寸和重量进行调整

在上个月的发布会上，苹果营销主管菲尔·席勒强调了一个事实，那就是 iPhone XS Max 有一个更大的显示屏——该公司称这是 iPhone 有史以来最大的显示屏——采用了更小的工业设计。席勒在台上说，这使得 XS Max 感觉起来更像 iPhone 8 Plus。在我的用法中，他的比较似乎是正确的；拿着 XS Max 的感觉和我以前的 Plus 手机一样。

为什么从可访问性的角度来看这是值得注意的，这是一个技巧问题。如果你像我一样，患有脑瘫或其他身体运动疾病，那么当你拿着和携带一个物体(任何物体，不仅限于智能手机)时，它在你手中的感觉值得认真考虑。在这种情况下，如果你有困难操纵 XS 最大由于这种电机延迟，这很可能是决定性因素，你是否选择它或选择较小的 XS 大小。

在我三年前对 iPhone 6s 的评论中，我说 6s Plus 不适合我，部分原因是“它提供的浮士德式的优惠”是提供一个大屏幕手机，但代价是使用一个物理上笨重的设备。当时，我认为普通的 6s 已经“足够好了”，因为我不想要这么大的手机。

此后不久，我确实换成了 6s Plus，而且我再也没有回头。事实证明，大显示器是最好的，我已经习惯了拿着大显示器也很好。

显示屏够大吗？

我坦率地承认，在测试 XS Max 的过程中，我一岁的 X 就在旁边，我想知道后者的 5.8 英寸屏幕是否足以满足我的需求。(苹果也给了我一个普通的 XS 来测试，但由于 X 的大小几乎相同，我没有像 Max 那样彻底地使用它。)无论如何，它都不小，我很高兴在过去的一年里，在一个较小的机身中拥有一个相对较大的显示器。X(显然还有 XS)肯定比它们更大的兄弟更容易携带和装入口袋。老实说，在我使用 iPhone X 的整个过程中，我从未希望我的手机屏幕更大。

然而，重申一下我在开头写的内容，当我打开 XS Max 并从我的 iCloud 备份中恢复时，真的感觉就像回到了家。忘记有机发光二极管，忘记像素密度——拥有 6.5 英寸的显示屏对我的眼睛来说超级好，也更舒服。更多的屏幕意味着更多的内容，意味着更少的眼睛疲劳和疲劳。考虑到这些因素，我更喜欢什么是无可争议的。虽然我有多种残疾，但我的视觉障碍可以说是最重要的，也是我应该优先考虑的。我做到了，我为此感到高兴。

没错，iPhone XS Max 是苹果迄今为止最容易上手的 iPhone。

这里的教训不是无关紧要的，它说明了残疾人每天面临的实际生活选择。我对 iPhone X 非常满意；如果这是苹果今年发布的唯一一款新 iPhone，我会跳到 XS。但事实并非如此——XS Max确实存在，而且它的大显示屏对我(和我的视力)来说太有诱惑力了。

我确实怀念 iPhone X/XS 外形的金发女郎式“恰到好处”的特性。但是如果机动性的损失带来视觉上的收益，我每次都会做出这样的权衡。

关于“先进”Face ID 的思考

当 Face ID 去年首次亮相时，我很快发现了一个问题，尽管我的脸已经在 iPhone X 上注册，但它在识别我的脸方面存在很大问题。经过一些故障诊断，我发现这个问题是由于我左眼的斜视。它的通俗说法是“懒眼”，但这是一种一只或两只眼睛不直的状况，它对原深感摄像头系统造成了严重破坏。即使系统“识别”了我的脸，我的手机也不会解锁，因为 Face ID 认为我没有看手机，尽管我知道我实际上肯定在看手机。

解决方法是禁用 Face ID 设置中的“需要注意”选项。当你这样做时，iOS 会警告你，这会使面部识别系统变得不安全，但这是我像其他人一样从 Face ID 中受益的唯一方式。我已经有一年多没有遇到任何问题了，随着时间的推移，我的 iPhone X 似乎越来越好地看到了我；在极端的角度下尤其如此，比如当我把手机放在厨房的桌子上，俯身看着它的时候。

XS Max 上的 Face ID 一直是可靠的，当然有需要注意的地方。我唯一的吹毛求疵仍然是因为我通常把我的手机靠近我的脸来看，我仍然没有始终保持足够远的距离来正确解锁。我得到了有趣的“摇头”动画，并输入了我不喜欢的密码，但我想本能的习惯很难打破。至少每次我这样做，Face ID 都能更好地了解我，这对苹果来说是一个很好的机器学习。

底线

我在总结最近几篇 iPhone 评论时说，每一款都是“迄今为止最容易上手的 iPhone”。无论多么陈腐，我不得不再做一次，因为这是一个完全准确的描述。

我是一个快乐的回归者。更大的显示屏，加上 Face ID 和窄边框设计，使用起来很有乐趣。没错，iPhone XS Max 是苹果迄今为止最容易上手的 iPhone。然而，说实话，尽管 XS 系列不错，但我对蓝色 iPhone 的存在更感兴趣，蓝色是我最喜欢的颜色。它实际上是一个最大值，我也得到了我的蓝色。
