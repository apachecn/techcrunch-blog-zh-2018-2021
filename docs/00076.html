谷歌的 Home Hub 有一个屏幕，但没有摄像头，“因此在私人空间也很舒适”

原文：https://web.archive.org/web/https://techcrunch.com/2018/10/09/googles-home-hub-has-a-screen-but-no-camera-so-that-it-is-comfortable-in-private-spaces/

谷歌终于把消费者的隐私担忧放在心上了吗？今天，这家搜索和安卓巨头揭开了它的、149 美元的家庭中枢，这是一款新的基于屏幕的智能家居设备，可以让你与谷歌照片等谷歌服务和联网的智能家居设备进行互动。这是谷歌对亚马逊回声秀和脸书新门户的回应。但在这个隐私时代，该公司做出了一个有趣的功能选择:它将推出第一款没有内置摄像头的设备。

在今天的演示中，产品管理副总裁 Diya Jolly 在介绍该设备的其他功能时指出:“我们也有意识地没有设置摄像头，以便在您家中的私人空间(如卧室)对我们来说很舒适。”。

这一特点——或者说缺乏这一特点——是值得注意的。就在昨天，脸书发布了自己的联网主屏幕设备，门户，亚马逊一直在努力推动和更新其 Echo Show ，这是一个基于 Alexa 的带屏幕的家庭中枢。亚马逊和脸书目前都专注于如何展示图像，但如何捕捉图像，并使用视觉线索来建立更智能的服务。

谷歌似乎正在采取一种不同的方式:这些天来，科技公司一直因其处理隐私的方式而受到关注，谷歌决定在这款设备中去掉摄像头，这体现了科技公司正试图对用户想要的——也许是需要的——更加敏感，因为我们已经有这么多其他带有摄像头的设备。

特别是对谷歌来说，时机尤其重要:就在本周，该公司宣布将关闭其命运多舛的社交网络 Google+ ，该网络存在一个暴露用户隐私信息的漏洞。在许多人想知道这些智能家庭扬声器能接收多少信息的时候，推出一款带摄像头的新设备的光学效果(双关语)看起来确实非常糟糕。

另一方面，省去摄像头可能会为谷歌带来其他好处。

这有助于降低设备成本，149 美元是一个非常有竞争力的价格点。

它还可以帮助谷歌防止这种设备与它向用户推销的其他设备竞争，特别是其手机，其新的 Pixel Slate 平板电脑，它有一个用于视频聊天的前置摄像头；还有 Pixel 支架，当它与新的 Pixel 3 智能手机结合时，基本上可以将该设备变成一个基于屏幕的家庭中枢，而和都有摄像头。

最后，它有助于谷歌开始构建一个路线图，如果它发现用户有需求，它可以在以后的迭代中添加到主家庭中枢产品中。
