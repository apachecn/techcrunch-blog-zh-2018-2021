微软展示政府云服务，绝地大限将至

就在谷歌决定退出五角大楼100 亿美元的 10 年绝地云合同竞标的一天后，微软宣布增加对政府客户的支持服务。在的一篇长文中，该公司展示了其专注于政府的云服务。

虽然今天的公告与 JEDI 本身没有直接关系，但在 10 月 12 日提交 RFP 截止日期的前三天，这个时机很有意思。今天的发布是为了展示该公司的政府专用云服务有多全面。

在一篇博客文章中，微软 Azure 公司副总裁 Julia White 明确表示，公司正在努力专注于政府业务。“在过去的六个月里，我们为 Azure Government 添加了 40 多项服务和功能，并为 Azure Government 地区发布了新的路线图，为我们即将发布的版本提供了持续的透明度，”她写道。

“展望未来，我们正在简化联邦机构的法规遵从性方法，以便我们的政府客户能够更快地获得创新。此外，我们正在增加购买和加入云服务的新选项，以便更轻松地迁移到云。最后，我们正在为政府带来一系列新的混合和边缘功能，以确保政府客户能够充分利用智能边缘和智能云时代的技术，”White 补充道。

虽然这篇帖子的大部分内容都围绕 Azure 的总体价值主张，如安全、身份、人工智能和边缘数据处理服务，但也有许多内容专门针对政府客户。

首先，该公司正在提高其 FedRAMP 合规性，这是一系列旨在确保供应商向联邦政府客户安全提供云服务的法规。具体来说，微软正在 50 项服务上从 FedRAMP 中等到高评级。

“通过采取业内最广泛的监管合规方法，我们正在使商业创新更容易为政府所采用，”怀特写道。

此外，微软宣布它正在扩展 Azure Secret Regions，这是一个专门为在云端处理高度机密信息而设计的解决方案。这一次似乎直接瞄准了绝地。“我们在交付这种旨在满足国防部和情报界的监管和合规要求的云方面取得了重大进展。今天，我们宣布这些最新地区将于 2019 年第一季度末上市。此外，为了满足美国政府不断增长的需求和要求，我们正在确认我们提供 Azure 政府服务的意图，以满足最高分类要求，并具备处理绝密美国机密数据的能力，”怀特写道。

该公司的声明包括许多之前已经宣布过的其他内容，显然是为了在一项重大政府合同即将出炉之际炫耀其政府印章。该公司在八月宣布了 Azure Stack for Government ，这是这篇博文中提到的另一件事。
