Pixel 3的最佳新功能TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/10/09/the-pixel-3s-best-new-features/

谷歌周二在其大型硬件活动上发布了数量的新产品，包括谷歌Home Hub和谷歌Pixel Slate。但是该公司的新智能手机 Pixel 3 才是真正的明星。

Pixel 3有两种尺寸，起价为799美元，有三种颜色，后置1220万像素摄像头和双前置摄像头。手机里有许多新的应用和功能。该设备今天可以预订，并将于10月18日开始发货。

以下是Pixel 3中一些最好的新功能和更新功能的分类。

呼叫屏幕

呼叫屏幕选项使Google Assistant能够为您接听来电。用户只需点击呼叫屏幕按钮，手机就会接听电话，并询问是谁打来的，为什么。

然后对话会被实时转录在屏幕上，让你决定是回复还是发短信回复。

周二推出了Pixel 3，并将于11月推广到其他Pixel设备。

安全性

Pixel 3配备了一种新的安全芯片，谷歌称之为Titan M。这种定制的芯片有助于保护手机的密码和操作系统。

扬声器

前置扬声器比之前的Pixel智能手机声音大40%，声音更丰富。

YouTube音乐

新的Pixel 3配备了YouTube音乐流媒体应用程序和六个月的免费订阅。

谷歌镜头

谷歌镜头本质上是一款人工智能相机。智能手机摄像头捕捉图像，人工智能算法为你识别它。Lens中的“风格搜索”可以识别图片中的产品，帮助您在线找到该产品。

Google Lens还可以识别地标、植物和动物，并将事件添加到您的日历中。将摄像头对准外卖菜单，它会高亮显示要拨打的电话号码。

集体自拍摄像头

Pixel 3配有双前置摄像头，用于需要更宽视野的自拍。用户只需双击电源按钮来打开相机，并轻拂手腕两次来激活自拍模式。从这里，用户缩小到团体自拍。

最佳镜头

此照片功能在HDR plus中捕捉备选照片，然后推荐最佳照片。例如，用户可能试图拍摄他们的孩子在生日蛋糕上吹灭蜡烛的照片。在过去，射偏了就是射偏了。Top shot捕捉你按下快门拍照之前或之后的瞬间，然后自动推荐最好的一张。

如果你想要另一个镜头，你也可以选那个。

操场

游乐场功能添加了反应人物(如一只狗或一个跳舞的音响)，并为您的照片和视频添加标题和动画贴纸。用户可以通过Playmoji让他们的照片“栩栩如生”，这些角色可以相互反应，也可以对你做出反应。

一些选项是像《漫威电影宇宙》中的钢铁侠这样的角色。谷歌正在与唐纳德·格洛弗合作，在今年晚些时候将孩子气的Gambino带到游乐场。

照相亭模式

是的，另一个相机功能。这个功能会在你做鬼脸或者微笑的时候自动拍照。
