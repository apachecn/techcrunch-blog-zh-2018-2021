Shujinko 为其云安全合规平台带来了 280 万美元的收入

原文：https://web.archive.org/web/https://techcrunch.com/2018/10/10/shujinko-brings-in-2-8m-for-its-cloud-security-compliance-platform/

舒金科——是的，就像真人快打这个角色一样——在从不寻常的风险投资公司、Defy、Vulcan Capital、PSL 风险投资公司和 Vas Ventures 筹集了 280 万美元的种子资金后，今天正悄然出现。

总部位于西雅图的云安全合规平台正在推出创业工作室先锋广场实验室，该实验室本身最近筹集了 1500 万美元来扩大其孵化计划。

Shujinko 由星巴克云工程团队的前董事兼经理 Scott Schwan 和 Matt Wells 创建，为至少拥有 5000 万用户的基于云的 IT 企业自动执行审计流程。

Schwan 是这样向我解释的:“当你通过会计师纳税时，基本上你最终会收集所有这些收据，你拥有所有这些你长期收集的证据，你最终不得不与审计员讨论这些证据，以证明你没有任何问题。这是一个痛苦的过程……在 IT 部门，他们定期进行这些工作，有时一年要进行多次相当于 IT 审计的工作。”

“当公司进行 IT 审计时，他们通常会花几个月的时间手动收集证据，如防火墙配置或[操作系统]配置的截图，”Wells 补充道。“这需要几个月的手动工作，我们试图通过自动化这一过程来减轻这种痛苦。”

施万和威尔斯有着悠久的职业历史。他们于 2008 年在 Tommy Bahama 相遇，当时 Schwan 是安全和 PCI 项目经理，而 Wells 是应用系统管理员。几年后，施万加入了面向商户的软件平台 Cardfree。不久之后，他鼓励威尔斯跟随他。

施万说:“我在威尔斯身上看到的是一个真正有动力继续学习的人。“我觉得我也有这种感觉。”

威尔斯最终同意加入 Cardfree，当施万在 2015 年离开这家初创公司加入星巴克的云工程团队时，威尔斯再次效仿。

对他们两人来说，在这样一家大型企业工作的经历是变革性的。

“我们在那里看到了行业趋势，以及该领域的其他竞争对手如何努力实现云迁移，特别是在云合规性需求方面，”Wells 说。

威尔斯和施万在一月份离开了星巴克，加入先锋广场实验室作为常驻企业家，致力于后来成为 Shujinko 的工作。

至于名字，Shu jinko——除了它在真人快打的从属关系——是日语中英雄或主角的意思。

Schwan 说:“Matt 和我在基础设施和安全领域工作了很长时间，我们从来都不是主角。“你 看不到我们在做什么，但这真的很重要……有了舒金科，我们真的想走出来，成为我们故事中的主角。”
