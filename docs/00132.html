Sidewalk Labs 的分支机构 Coord 筹集了 500 万美元来帮助移动服务更好地融入城市

从 Alphabet 旗下的 Sidewalk Labs 分拆出来的移动数据初创公司 Coord 已经筹集了 500 万美元的 A 轮融资，由 Alliance Ventures 领投，Trucks，Urban 参与其中。美国和 DB 数字风险投资公司。

这笔资金的计划是继续增强 Coord 的 API 和地理覆盖范围，以及“在私营部门和公共部门之间建立一座桥梁”，Coord 的联合创始人兼首席执行官 Stephen Smyth 在 Medium 上写道。

Coord 为其客户提供了一些产品，包括 Zipcar、Mozio 和谷歌地图产品等公司。有收费 API，它跟踪收费公路、桥梁和隧道，以确定出行成本；Curbs API 旨在帮助司机轻松了解该地区的停车和载客规则(比如打车司机)、计价器价格等等；以及使用实时信息来显示最佳多模式路线的路由 API。

随着自行车共享和踏板车共享在全球范围内继续扩张，Coord 还提供了一个共享车辆 API，使其客户能够整合自行车和踏板车的实时可用性、价格和位置。

“我们的目标是帮助公共和私营部门在城市交通方面使用相同的语言，”史密斯写道。“虽然许多私营公司没有很好地集成到今天的现有交通系统中，但我们相信最终用户将最终要求城市中所有系统的互操作性。为此，我们正在推动城市间交通相关数据的标准化。”
