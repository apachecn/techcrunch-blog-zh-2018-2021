Docker 筹集了 9200 万美元的新资金 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/10/15/docker-has-raised-92-million-in-new-funding/

根据提交给美国证券交易委员会的一份文件，Docker 是一家在创造当今现代集装箱化计算环境方面做得比任何其他独立公司都多的公司，它已经在目标 1.92 亿美元的融资轮中筹集了 9200 万美元。

这笔新的资金是一个信号，尽管 Docker 可能在与谷歌的 Kubernetes 关于谁的工具包将被最广泛采用的竞赛中失败了，但这家总部位于旧金山的公司已经成为那些希望转向现代混合应用开发和信息技术运营编程模式的企业的冠军。

为了理解容器在现代编程中的重要性，解释一下它们是什么可能会有所帮助。简而言之，它们是不需要操作系统就能工作的虚拟应用程序环境。在过去，这种类型的功能是使用虚拟机创建的，包括软件和操作系统。

相比之下，容器效率更高。

因为它们只包含应用程序和库、框架等。它们依赖于，你可以在一个主机操作系统上安装很多。服务器上唯一的操作系统是一个主机操作系统，容器直接与它对话。这使得容器很小，开销非常低。

企业正在迅速转向容器，因为他们希望改进他们开发和管理软件的方式——并且做得更快。但他们无法独自做到这一点，需要像 Docker 这样的合作伙伴来帮助他们实现这一转变。

许多人忽略的是 Docker 不仅仅是容器编排层——Kubernetes 赢得了这场战争——而是构建和管理这些容器的完整工具链。

对于每一个开源项目，技术公司都会很快采用(并适应)开源项目，并精通如何使用它。更多不太懂技术的主流大企业将转向 Docker 这样的公司，帮助他们管理用工具包开发的项目。

这是一家为大企业客户服务的科技初创公司的自然演变，当他们变得更有利可图时，他们变得索然无味。企业用。他们赚钱。炒作已经过去了。因为一旦一家公司向一家大型企业客户销售产品，他们就会永远与该供应商合作。

今年早些时候，当 Docker 的创始人兼前首席执行官所罗门·海克斯(Solomon Hykes)离开公司时，他也承认了这一点

…在我们的首席执行官传奇人物史蒂夫·辛格(Steve Singh)的领导下，Docker 已经悄然转型为一家收入增长迅猛、拥有数百万开发人员社区的企业。我们的策略很简单:世界上的每一家大型企业都在准备将其应用程序和基础设施一起迁移到云中。他们需要一种解决方案来可靠、安全地做到这一点，不需要昂贵的代码或流程更改，也不需要将自己局限于单一操作系统或云。目前，满足这些要求的唯一解决方案是 Docker 企业版。这使得 Docker 处于巨大增长机会的中心。为了利用这个机会，我们需要史蒂夫身边的首席技术官有几十年为世界上最大的公司运送和支持软件的经验。因此，我现在有了一个新的角色:帮助寻找理想的 CTO，偶尔提供一些建议，并在团队继续建立庞大的业务时，离开他们。作为股东，我非常乐意接受这个角色。

有了这笔钱，Docker 很可能会增加其销售和营销人员，开始创造 2019 年上市所需的收入数字。该公司已经建立了一个独立董事的名单(另一个明显的迹象是，它正试图打开一扇进入公开市场的窗口)。

Docker 已经是价值超过 10 亿美元的“独角兽”了。据报道，Docker 上一次筹集资金是在 2017 年底，当时华尔街日报发现了证券交易委员会的一份文件，表明该公司已经筹集了 6000 万美元，目标是 7500 万美元。当时的投资者包括 AME 云风险投资公司、Benchmark、Coatue Management、高盛和 Greylock Partners。当时，这笔投资对该公司的估值为 13 亿美元。
