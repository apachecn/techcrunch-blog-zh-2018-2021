Premiere Rush CC 是 Adobe 新推出的用于桌面和移动设备的一体化视频编辑工具

今年早些时候，Adobe 预览了 Project Rush ，一款新的多平台视频编辑工具。今天，在 Max 大会上，公司宣布【Project Rush 现在是 Rush CC 的首播，也是 Creative Cloud suite 的官方部分。

拉什背后的想法非常简单。它旨在为视频创作者提供一个现代化的一体化视频编辑解决方案，允许他们快速编辑视频并将其发布在 YouTube 和其他社交网络等平台上。事实上，它很有可能成为 YouTube 一代的视频编辑工具。

Rush 采用了 Adobe 的视频和音频编辑工具套件的核心部分，并将它们结合在一个单一的移动和桌面体验中。这意味着你会得到一套动画模板，例如，这些模板是专门为让 Rush 用户轻松访问可定制的标题而设计的。色彩校正系统建立在相同的技术之上，该技术支持功能更全面、更复杂的 Premiere Pro 编辑工具。音频编辑功能，包括一次点击闪避，与它们在 Audition 中的对应部分由相同的代码驱动。

所有这些编辑都可以在平台之间轻松同步，让创作者能够在手机上开始编辑，然后在笔记本电脑上完成工作。

Rush 的当前版本与 Adobe 几个月前发布的版本非常相似，但该公司也利用今天的发布来预览下一步的内容。很快，你还可以在 Android 上编辑(Adobe 承诺在 2019 年发布)，并获得速度控制，这样你就可以加速和减速你的视频(YouTube 创作者肯定会过度使用这一功能)，以及更容易地为多个平台创建不同版本的视频。该团队还承诺随着时间的推移提高绩效。

Premiere Rush 现已面向所有 Creative Cloud All Apps、Premiere Pro CC single app 和学生计划订户开放。还有个人 9.99 美元/月(团队 19.99 美元/月)的单个 app 计划。此外，还有一个免费的启动计划，让用户可以访问所有的应用程序和功能，但仅限于三个项目的出口。
