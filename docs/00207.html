6D.ai 开放测试版 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/10/18/6d-ai-opens-up-its-beta/

经过十多年的努力，开发了一种可以创建物理世界三维地图的技术， 6D.ai 的团队终于准备好向开发者开放其工具包，该公司表示已经做到了这一点。

当公司首席执行官马特·米斯尼克(Matt Miesnieks)在 3 月份宣布推出 6D 时，他为其发展规划了一个愿景，该愿景有三个目标:公司将构建 API 来捕捉世界的三维几何图形；它将应用三维数据来构建语义 API，以便应用程序能够理解世界；它将合作并扩展这些 API 来为现实创造一个操作系统。

已经实现了第一个目标，公司现在正在进行第二个目标。

“这家公司的整个目的不是‘嘿，这是新技术！’Miesnieks 说:“它是 AR 在其完全实现的形式下可以做什么，以及在以前的媒体中不起作用的 AR 的原生体验是什么，是什么阻止了这些东西发挥作用，以及如何解决这些问题。”

对于 Miesnieks 来说，增强现实面临的问题归结为创建可信的视觉对象，无缝地融入世界。根据 Miesnieks 的说法，这种创造行为依赖于坚持、封闭和互动。

对于 Miesnieks 来说，互动应该无缝地发生，而不是需要一个多步骤的过程，这位 6D 首席执行官称之为“只是一座太远的桥”

“你需要做的是说，‘嘿，加入我的游戏吧。’它就这样工作了。"

Miesnieks 认为，同步所需要的那种精度需要一种设备上的本地化，这正是 6D 声称它能够实现的。

“一旦有了 3D 模型，虚拟内容就可以从 3D 模型中反弹出来。可以正确的做阴影。将它扩展到大范围，这样它就不仅仅在我客厅的一个角落起作用，而是可以在任何地方起作用，”Miesnieks 说。“我们需要这些模型，唯一的方法是使用深度相机或离线摄影测量。”

6D 已经与 Massive Attack 和 Aphex Twin 等乐队合作，对其技术进行了一些早期测试。维多利亚和阿尔伯特博物馆也使用了这项技术。不久，它将与一家未披露的日本游戏开发商(拥有类似于神奇宝贝的知识产权)合作推出一款游戏，并与日本社交网络 Gree 合作推出一款类似 YouTube 的虚拟应用。

对于 Miesnieks 来说，最有趣的应用可能是一家大型、未披露的运输公司，该公司对地面和其他移动导航感兴趣。

“当我们成立公司时，我们确信我们想告诉开发者这是现实。我们会给你多人共享的坐标，”Miesnieks 说。

所有这一切的背后是对安全性的担忧，即谁可以看到用户地图空间中的内容。但是 Miesnieks 说公司已经解决了这个问题。

Miesnieks 说:“只有当你身处某个空间时，你才能获得该空间的数据。”“我举起我的手机，它会查看你的客厅，根据它看到的内容，它会查询服务器，如果有匹配，它会将数据发送到那个位置。”

根据研究，6D 生成的点云与地理结构没有直接联系。它是稍微随机化的，所以用户不能看着点云看什么是什么。

“它不能被任何已知的科学逆向工程成人类可读的图像，”Miesnieks 说。“所有的图像看起来就像是一大堆点和斑点。这就是我们目前正在做的事情。”

随着该公司构建出其三维世界地图，它鼓励开发者将其视为一种新型的增强现实平台。

“我们的业务是网络服务满足 Waze，”Miesnieks 说。
