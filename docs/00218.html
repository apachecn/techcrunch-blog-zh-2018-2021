流行的联网存储驱动器中的漏洞软件可能会让黑客读取私人数据 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/10/19/flaws-connected-storage-drives-can-let-hackers-read-private-data/

安全研究人员在四个流行的连接存储驱动器中发现了缺陷，他们说这些缺陷可能会让黑客访问用户的私人和敏感数据。

研究人员 Paulos Yibelo 和 Daniel Eshetu 表示，他们测试的三种设备上运行的软件——NetGear Stora、Seagate Home 和 Medion life cloud——可以允许攻击者远程读取、更改和删除数据，而无需密码。

Yibelo 本周与 TechCrunch 分享了这项研究，并于周五发布了调查结果，他说许多其他设备可能也面临风险。

由科技公司 Axentra 开发的软件 Hipserv 是他们发现的四个缺陷中的三个的主要原因。Hipserv 是基于 Linux 的，使用几种 web 技术——包括 PHP——来增强 web 界面。但是研究人员发现，漏洞可以让他们在没有任何认证的情况下读取驱动器上的文件。这也意味着他们可以以“root”身份运行任何他们想要的命令，root 是具有最高访问级别的内置用户帐户，使设备上的数据容易受到窥探或破坏。

我们在周四联系了 Axentra 进行评论，但在撰写本文时尚未收到回复。

Netgear 发言人表示，Stora“不再是受支持的产品…因为它已经停产，是一种生命周期结束的产品。”希捷没有在截止日期前发表评论，但如果情况有变，我们会及时更新。现在拥有 Medion 的联想没有回应置评请求。

研究人员还报告了一个影响 WD My Book Live drives 的独立漏洞，该漏洞允许攻击者远程获得 root 访问权限。

WD 的一位发言人表示，该漏洞报告影响了最初于 2010 年推出并于 2014 年停产的设备，并且“不再包含在我们的设备软件支持生命周期中。”WD 补充说:“我们鼓励希望继续操作这些传统产品的用户配置他们的防火墙，以防止远程访问这些设备，并采取措施确保只有本地网络上的可信设备才能访问设备。”

在所有四个漏洞中，研究人员表示，攻击者只需要知道受影响驱动器的 IP 地址。在这个时代，这并不困难，这要感谢 Shodan 这样的网站，一个为公众提供设备和数据库的搜索引擎，以及类似的搜索和索引服务。

受影响设备的数量会因您的关注点而异。 Shodan 给出的数字是 311，705 台，但是 ZoomEye 给出的数字接近 180 万台。

尽管研究人员描述了这些缺陷的适度细节，但他们表示，他们没有计划发布任何利用代码来防止攻击者利用这些缺陷。

他们的建议是:如果你运行的是云驱动器，“确保将你的设备从互联网上移除。”
