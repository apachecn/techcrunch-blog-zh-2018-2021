Expedia 收购 Pillow 和 ApartmentJet 以征服短租市场

为了跟上美国城市对短期租赁不断增长的需求，并与家庭共享巨头 Airbnb 竞争，旅游预订网站 Expedia 已经收购了两家风险投资支持的酒店初创公司 Pillow 和 ApartmentJet 。

两家公司的员工都将加入 Expedia。该公司拒绝透露交易的财务条款。

该公司在一份声明中解释说:“收购 Pillow 和 ApartmentJet 将有助于释放城市增长机会，随着时间的推移，这将有助于 HomeAway 为其市场和 Expedia 集团品牌的市场增加更广泛的住宿选择，确保旅行者总能找到完美的住宿地点。”

2015 年，Expedia 为 HomeAway 及其旅游品牌组合支付了 39 亿美元。这笔交易是它在替代住宿领域的第一个重大举措，也是一系列努力超越风投宠儿 Airbnb 的开始。它的最新目标是为物业管理公司提供软件工具，方便他们在 Airbnb 的竞争对手 HomeAway 和 VRBO 上管理短期租赁。

Pillow 位于旧金山，在不违反租约的情况下，帮助居民将公寓列为短租。自 2013 年以来，它已经筹集了总计 1650 万美元的风险投资，包括去年由梅菲尔德牵头、斯特林参与的 1350 万美元的融资。风险投资，峰值资本合伙人，扩展风险投资，克里斯安德森，加里·维纳查克，丹尼斯菲尔普斯和维尔软件投资。

ApartmentJet 帮助业主从空房中赚钱。这家总部位于芝加哥的初创公司成立于 2016 年，已从 Network Ventures 和 BlueTree 筹集了据报道120 万美元的资本。

总部位于贝尔维尤的 Expedia 集团拥有多个旅游品牌，包括 HomeAway、VRBO、Travelocity、trivago、Orbitz 和 Hotels.com。该公司既是初创公司的积极投资者，也是收购者。

Expedia 的股价周四上涨了 9.4%，此前其第三季度收益超过了分析师的预期。该公司公布的收入为 32.8 亿美元，与去年的 29.7 亿美元相比有显著增长。
