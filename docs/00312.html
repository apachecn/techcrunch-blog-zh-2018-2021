Reelgood 收购 Guidebox，将流媒体电视数据带到更多地方 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/10/26/reelgood-acquires-guidebox-to-bring-streaming-tv-data-to-more-places/

Reelgood 是一家初创公司，旨在帮助插队者在他们订阅的服务上找到可以观看的节目和电影，该公司进行了收购，希望将 Reelgood 的数据带到更多地方。该公司已经收购了 Guidebox ，这是一家流媒体可用性数据提供商，为 Roku、TVGuide、Metacritic 和其他公司提供支持。

交易条款没有透露，但我们知道价格在“数百万美元”

Guidebox 于 2012 年成立，当时是一个面向消费者的网站,它将完整的节目和剧集数据集中在一个地方，然后告诉你可以在哪里观看——事实上，这非常像今天的 Reelgood。但多年来，它将重点转向了与出版商和设备制造商合作。例如，众所周知，这项服务为 Roku 的通用搜索功能提供了动力。

最近有报道称 Guidebox 被出售给了视频数据和推荐初创公司 ColorTV。然而，Reelgood 表示，这笔交易实际上从未发生过——宣布收购还为时过早。(事实证明，彩色电视现在似乎被关闭了。我们试图联系他们进行确认。)

Reelgood 表示，它决定收购 Guidebox，因为它的目标不仅仅是为流媒体电视提供指南。

“想象一下，问你的 Alexa，‘我的哪个节目有新一集？’或者在网上阅读一个节目，嵌入文章中，看看你可以在哪里观看，”该公司在关于该交易的公告中解释道。为了让电视“正常工作”，我们需要让 Reelgood 的数据也能更容易地应用到其他产品上。"

在当今的割线时代，更好地组织流媒体服务内容的需求比以往任何时候都更加重要，因为消费者越来越多地放弃有线电视和卫星电视订阅，建立自己的视频服务捆绑包。Reelgood 表示，美国普通家庭现在使用四种不同的流媒体应用程序，此次收购将使其覆盖范围扩大到超过 5000 万个家庭。

该公司表示，它将基于现有的 Guidebox 技术，让公司更容易帮助自己的用户找到流媒体内容。这些数据将通过 API 提供。

这也意味着 Reelgood 不会关闭 Guidebox 或结束其现有的业务关系——它的目标是扩大这些关系，并寻求新的商业机会。据我们所知，它目前正在与大型电视和有线媒体公司重新谈判 Guidebox 的一些交易，这些公司为一些大型网络提供服务。

Guidebox 去年曾与狮门影业、WWE 和 Fandor 等内容提供商合作。

就 Guidebox 团队而言，并非所有人都是长期加入的。管理团队正在实施一项盈利计划，将帮助整合 Reelgood 的技术并转变客户关系。Reelgood 聘请了一些从事数据完整性和质量保证工作的员工来帮助扩展产品和服务。

Reelgood 创始人兼首席执行官大卫·桑德森(David Sanderson)在关于这笔交易的一份声明中表示:“没有人愿意花时间在应用程序中寻找合适的节目。”“人们希望他们的设备能帮助他们决定看什么以及在哪里看。无论是搜索引擎、网站、流媒体播放器还是语音助手，这都是公司获得正确体验的机会。”
