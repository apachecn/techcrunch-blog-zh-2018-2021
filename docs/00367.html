WeWork 旗下的 Meetup 让大卫·西格尔担任首席执行官

去年夏末，Meetup 创始人兼首席执行官斯科特·海夫曼(Scott Heiferman)宣布他有意出任董事长一职。今天，Meetup 宣布大卫·西格尔将执掌这家有着 16 年历史的公司。

西格尔来自 Investopedia，他在那里担任了三年的首席执行官，在此期间，公司的收入增加了两倍，流量增加了一倍。在他加入 Investopedia 之前，Siegel 是 Seeking Alpha 的总裁，负责监管美国的职能部门，包括销售、营销、产品和业务开发。

2017 年底，联合办公巨头 WeWork 以 2 亿美元收购了 Meetup 。Meetup 的整个前提是基于社区的理念——使用互联网让人们离开互联网，在现实生活中交谈。这是 WeWork 战略的核心主题。

以下是西格尔对这一转变的看法:

在当今世界，技术往往会加大人与人之间的距离，Meetup 利用技术为全球数百万人带来真实的、面对面的、改变生活的联系。与 WeWork 一起，Meetup 正在重塑人们每天工作、生活、学习、娱乐和创建社区的方式。我很高兴能成为这个令人激动的项目的一部分，让更多的人聚在一起。

Meetup 目前拥有超过 4000 万名会员，320，000 个 Meetup 群组，每天在世界各地促成 12，000 场 Meetup。
