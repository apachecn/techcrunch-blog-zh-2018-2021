这个小工具给苹果的电源适配器增加了两个 USB 3.0 端口

这很聪明。USB-C 集线器由 HyperDrive 公司制造，可以安装在苹果 USB-C 电源适配器上，并增加了两个 USB 3.0 端口。仅此而已。我喜欢它，它解决了苹果目前笔记本产品线的一个主要缺点。

苹果放弃了全尺寸 USB 端口，转而支持多功能 USB-C。这在某些层面上是有道理的。USB-C 几乎支持所有可用的总线格式，但仍有许多设备带有较旧的 USB 插头。比如 iPhone。如果一个人走进苹果商店，购买了最新的 iPhone 和最新的 MacBook Pro，iPhone 将需要一个加密狗来为 MacBook Pro 充电。为什么不做成这个加密狗呢？

类似的设备在市场上已经存在很长时间了，但倾向于使用电源端口来添加 USB 端口。这一款使用 USB-C 的电源，这使得适配器比其他产品更小。

HyperDrive USB-C Hub 有两种风格，可以匹配苹果的两个版本的电源适配器。61W 电源适配器的 USB-C 集线器售价 39.99 美元，而 87W 电源适配器的 USB-C 集线器售价 49.99 美元。两者现在都可以从 Hyper 以 25%的折扣预购。
