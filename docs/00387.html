现在，全球的谷歌员工都走出来抗议性骚扰

谷歌员工厌倦了这家搜索巨头在处理性骚扰和不当行为指控时缺乏透明度。

今天早上，成千上万的谷歌人从旧金山到都柏林，希望能给公司带来真正的变化。此前，《纽约时报》上周报道称，尽管谷歌对安卓联合创始人安迪·鲁宾提出了可信的不当性行为指控，但谷歌仍向他支付了9000万美元。

抗议者有五把钥匙反问道:

  1. 终止骚扰和歧视案件中的强制仲裁。
  2. 结束薪酬和机会不平等的承诺。
  3. 一份公开披露的性骚扰透明度报告。
  4. 一个清晰、统一、具有全球包容性的流程，用于安全、匿名地报告不当性行为。
  5. 提升首席多元化官，直接向首席执行官负责，并直接向董事会提出建议。并任命一名员工代表进入董事会。

罢工计划本周早些时候出台，就在爆炸性的NYT报告发布几天后。根据 BuzzFeed 消息，大约200名谷歌员工开始发起抗议；该集团迅速发展到数千人，包括非美国的谷歌员工。据报道，谷歌首席执行官桑德尔·皮帅周二在给员工的内部电子邮件中宽恕了抗议活动。

皮查伊在今天提供给TechCrunch的一份声明中表示:“本周早些时候，我们让谷歌员工知道，我们已经了解了今天计划开展的活动，如果员工愿意参与，我们将为他们提供所需的支持。”。“员工就我们如何改进政策和流程提出了建设性的意见。我们正在接受他们的所有反馈，以便将这些想法转化为行动。”

皮查伊还在一封由人力运营副总裁艾琳·诺顿联署的信中回应了NYT的报告，信中承认仅在过去两年里，公司就有48人因性骚扰被解雇，其中包括13名高级雇员。
