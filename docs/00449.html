网飞的黑客马拉松通过ARKit和Face ID TechCrunch开发了一种导航其iOS应用的方法

原文：https://web.archive.org/web/https://techcrunch.com/2018/11/07/netflixs-hackathon-produces-a-way-to-navigate-its-ios-app-with-arkit-and-face-id/

从几年前让你在原版《NES》上运行网飞的到最近通过音频描述将《网飞》系列变成老派广播节目的，网飞内部的黑客马拉松一直在产生有趣且通常很傻的黑客。今年的黑客马拉松也没有让人失望，新的黑客既像往年一样愚蠢又有趣，包括一个ar和Face ID驱动的黑客，让你只用眼睛在网飞导航，另一个是为“Sharknado”粉丝设计的，等等。

“跳到鲨鱼”让观众直接跳到非常糟糕但很好的“鲨鱼”的精彩部分，这样他们就可以观看鲨鱼的血腥动作镜头，而不是一直坐到电影的实际情节结束。如视频所示，这非常棒。

增强现实黑客Eye Nav也相当令人印象深刻。

黑客使用苹果的ARKit和使Face ID能够跟踪眼睛位置和面部表情的技术。它跟踪你的眼睛位置，在屏幕上移动指针，然后测量在同一区域花费的时间，以触发“点击”

如果你想关闭一个屏幕，你可以伸出你的舌头。

虽然最终的破解肯定很有趣，但也暗示了未来的可访问性用例。

该黑客是在24小时内制作的，因此它可能不够稳定，无法在现实世界中使用，但这绝对是一个有趣的想法。

第三起黑客攻击与网飞无关，而是网飞员工使用的生产力软件Slack。

“午餐机器人”通过邀请太忙而没有时间去吃午饭的同事一起吃午饭——实际上，是在闲聊的时候。该应用程序还会检查每个人的日历，以确保他们都有空。

今年的其他黑客攻击包括产品改进、内部工具增强和一些只是为了好玩的攻击。其中一些展示在其Hackday 2018视频中，例如用于定位工作室制作资源的地图，“轻松登录”系统和使用网飞人物的Animoji版本。

但是网飞黑客马拉松的更大目标，正如你可能知道的，并不一定是创造以后会产品化的特性(尽管，来吧…跳到Shark！)，但该公司表示，它们有时会成为未来功能的灵感来源。
