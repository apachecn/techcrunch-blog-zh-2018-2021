Datacoral 为其数据基础设施服务融资 1000 万美元

Datacoral 旨在通过抽象出所有复杂的基础设施来组织和处理数据，使企业更容易构建数据产品。该公司今天宣布，它已经筹集了 1000 万美元的 A 轮融资，由 Madrona Venture Group 牵头，社交资本也参与了 2017 年 400 万美元的种子轮融资。

Datacoral 首席执行官 Raghu Murthy 告诉我，该公司计划使用新的资金来发展其业务团队，以便能够接触到更多的潜在客户，并扩大其工程团队。

Datacoral 的承诺是为企业提供端到端的数据基础设施，使企业及其数据科学家能够专注于就必须管理和集成其数据源的问题提出见解。因为没有人想在云之间移动大量数据——并承受随之而来的性能损失——data coral 就位于一家公司的 AWS 系统中。尽管这仍然是一项完全托管的服务，但数据是加密的，永远不会离开客户的虚拟私有云。

Murthy 说:“随着公司希望他们的数据能够带来价值，数据从业者发现配置和管理他们自己的数据基础设施是一项耗时的工作，不仅成本高昂，而且错误百出。“我们构建了一个平台，可以轻松、自动地将来自不同应用程序和数据库的数据汇集在一起，在任何查询引擎中组织这些数据，并根据对其业务运营至关重要的见解采取行动。一个至关重要的组成部分是，它在客户的云中安全、私密地工作，而不是我们从他们的系统中获取数据。”

Murthy 是脸书的早期工程师，是负责扩展该公司数据基础设施的团队成员，并在 Bebop 管理工程团队的一部分，Bebop 是 Diane Greene 的初创公司，后来被谷歌收购。

为了扩展 Datacoral，该团队将赌注押在了无服务器平台本身上。它在亚马逊的云计算平台上大量使用 AWS Lambda 和其他 PaaS 解决方案。不过，这并不意味着 Datacoral 计划只支持 AWS。Murthy 告诉我 Azure support 是下一个目标。“我们计划与所有顶级云提供商合作，利用他们独特的服务，为我们的客户提供一致的‘以数据为中心的界面’，本质上是‘云最佳’而不是‘云不可知’。”"

目前的 Datacoral 用户包括 Greenhouse、Front、Ezetap、Swing Education、mPharma 和 Mason Finance。
