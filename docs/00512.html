Bubble 可以让你创建没有编码经验的 web 应用程序

原文：https://web.archive.org/web/https://techcrunch.com/2018/11/11/bubble-lets-you-create-web-applications-with-no-coding-experience/

认识一下 Bubble 这是一家自举创业公司，它一直在构建一项强大的服务，即使你不知道如何编码，它也能让你创建一个 web 应用程序。许多大小公司的网站都依赖于 Bubble。

我不得不说，当我第一次听到泡沫的时候，我是相当怀疑的。许多创业公司已经试图让编码像玩乐高积木一样简单。但它总是有限的令人沮丧。

Bubble 比你一般的建站服务更强大。它在一个可视化界面中重现了 web 编程的所有主要支柱。

它从设计选项卡开始。您从空白画布开始，可以通过在屏幕上拖放可视元素来创建网页。你可以把元素放在任何你想放的地方，调整地图、文本框、图片等等的大小。你可以随时点击预览按钮查看你那个时代的发展版本。

在第二个选项卡中，您可以创建站点背后的逻辑。它的工作原理有点像 Mac 上的 Automator。您可以添加块来创建按时间顺序排列的动作。您可以在每个块中设置一些条件。

在第三个选项卡中，您可以与数据库进行交互。例如，您可以创建一个注册页面，并将个人资料信息存储在数据库中。您可以随时导入和导出数据。

有数百个插件，可以让你用 Stripe 接受支付，嵌入一个类型表单，通过聊天使用内部通信进行客户支持，使用 Mixpanel 等等。您也可以在 Bubble 之外使用您的 Bubble 数据。例如，你可以建立一个依赖于你的泡泡数据库的 iPhone 应用程序。

许多小公司开始使用 Bubble，对其中一些公司来说效果不错。例如， Plato 的所有后台都使用 Bubble。 Qoins 和 Meetaway 运行在泡泡上。股息融资筹集3.65 亿美元并利用泡沫。

初创公司负责为您托管应用程序。随着应用程序变得越来越大，每次调整实例的大小，您都要付出更多。

尽管该公司从未筹集到任何资金，但它已经每月产生 11.5 万美元的经常性收入。Bubble 仍然是一家小型创业公司，这可能会让更大的客户感到害怕。但该公司希望改进产品，让客户看不到 Bubble 的局限性。现在，挑战在于增长速度要快于客户的需求。
