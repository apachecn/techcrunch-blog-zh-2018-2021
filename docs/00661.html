跨境金融科技初创公司 Instarem 筹集 2000 万美元用于全球扩张

原文：https://web.archive.org/web/https://techcrunch.com/2018/11/21/instarem-raises-20m/amp/

图片来源: 斯维特拉娜·卢金科 / 摄影爱好者

Instarem 是一家总部位于新加坡的初创公司，帮助银行向海外廉价转移资金，已经筹集了超过 2000 万美元的 C 轮融资，用于全球扩张。

这轮融资由 MDI Ventures 和 Beacon 牵头，MDI Ventures 是印度尼西亚电信运营商 Telkom 的风险投资部门，Beacon 是泰国 Kasikorn 银行旗下的基金，现有投资者包括 Vertex Ventures、GSR Ventures Rocket Internet 和 SBI-FMO 基金。

尽管 Instarem 联合创始人兼首席执行官普拉吉特·纳努(Prajit Nanu)告诉 TechCrunch，这家初创公司计划将 C 系列扩大到 4500 万美元，但迄今为止，这笔资金已使四岁的 Instarem 筹集到近 4000 万美元。额外的资本预计将于明年 1 月完成，Nanu 尤其希望引进战略投资者，帮助该公司在拉丁美洲和欧洲的新兴市场实现增长。

“我们处在一个金钱的颜色非常重要的阶段，”他在接受采访时说。“对我们来说，将能够为我们的业务增加价值的人纳入进来非常关键。”

纳努补充说，该公司正在与其他潜在投资者中的大型美国基金洽谈。

Instarem 与银行合作，降低他们的海外转账成本，提供一种“面向企业的转账”服务。不过，与使用全球银行网络向世界各地汇款的 Transferwise 不同，Instarem 使用的是已经进行海外货币交易的中型银行。正如我之前解释的那样，这个过程在财务上相当于把几个箱子放在一艘即将出发的 UPS 货船上，因此支付的费用只是你必须自己找一艘船运送的费用的一小部分。

主要集中在东南亚，它服务于 50 多个市场的转让。该公司确实为消费者提供服务，但金融机构——它们有持续的需求和更高的平均支出——是其主要目标。

普拉吉特·纳努在 2014 年与迈克尔·伯明罕一起创立了 Instarem

该公司在新加坡、孟买和立陶宛设有办事处，并在西雅图开设了办事处，因为它开始寻求扩大业务，目前已经包括东南亚十大银行中的三家。Nanu 表示，该公司将尝试与银行和金融服务合作，如跨境服务，其目标用户最初与拉丁美洲和墨西哥有联系。在亚洲，它正在等待日本和印度尼西亚的支付牌照，这将使它能够在这两个国家提供更多服务。

TechCrunch 了解到，该公司即将与 Visa 达成协议，该协议将允许其客户推出品牌预付卡，为其产品增加另一项金融服务。当我们问及与 Visa 的交易时，纳努拒绝置评。

TechCrunch 还了解到，Instarem 在今年早些时候被东南亚的一家独角兽公司收购。纳努拒绝透露竞标者的名字，但他告诉 TechCrunch，这个报价“对我们来说不是合适的时机。”然而，他越来越多地考虑通过 IPO 退出。

去年，当 Instarem 筹集了 1300 万美元的 B 轮融资时，他表示该公司可能会在 2020 年上市。现在，这个目标日期已经推迟到 2021 年，Instarem 首席执行官告诉 TechCrunch，在时机成熟时，美国仍然是公开上市的首选。
