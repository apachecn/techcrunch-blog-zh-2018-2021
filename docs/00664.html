当台湾准备就 LGBTQ 问题进行投票时，一个仇视同性恋的团体在 YouTube TechCrunch 上的儿童视频前播放广告

原文：https://web.archive.org/web/https://techcrunch.com/2018/11/22/as-taiwan-prepares-to-vote-on-lgbtq-issues-a-homophobic-group-is-running-ads-before-kid-videos-on-youtube/

本周六，包括婚姻在内的几个与 LGBTQ 平等相关的问题将在台湾中期选举中进行公投。在国家最高法院支持同性婚姻一年多之后，由于反同性恋团体的努力，LGBTQ 群体再次为他们的权利而斗争。这场斗争已经蔓延到包括 YouTube 在内的社交媒体平台，一个名为 3 Yes 的团体正在播放一则广告，经常出现在流行的儿童视频之前，声称在学校教授 LGBTQ 问题会让孩子们对自己的性别身份感到困惑。

“3 Yes”的名称指的是反同性恋团体希望选民在周六选举期间批准的三项公投项目(提案 10、11 和 12)。的问题问选民“你是否同意婚姻应该被限制在一个男人和一个女人之间，根据民法典？”；“你是否同意教育部和中小学不应该教授《性别平等教育法》中详述的同性恋知识？”[2004 年通过，旨在促进性别平等和防止歧视]；以及“你是否同意民法定义的婚姻之外的结合，以保护同性伴侣永久生活在一起的权利？”

为了抵制这些提议，LGBTQ 倡导团体引入了另外两项公投 (14 和 15)，询问“你是否同意同性伴侣结婚的权利应该受到民法典婚姻条例的保障？”以及“你是否同意《性别平等教育法》中定义的性别平等教育应该在国家课程的所有阶段进行，并包括关于情感、性和同性恋的教育？”

尽管台湾宪法法院在 2017 年裁定支持同性婚姻，但这个问题和其他问题在今年付诸公投的事实凸显了由宗教保守派领导的反同性恋团体的力量，包括 生命之粮基督教会、幸福下一代联盟 和台湾宗教团体保护家庭联盟。除了 YouTube 视频，此类团体还通过示威游行、传单、横幅和其他在线平台，包括 YouTube、和台湾最受欢迎的信息应用之一 Line，传播仇视同性恋的宣传和错误信息。

一个传单谴责 LGBTQ 问题教育的例子。

3 Yes' YouTube 频道有三个视频。有两部影片似乎向同性伴侣展示了婚姻是一男一女之间的事情，然后他们突然转向传达这一信息。一周前上传到 3 Yes 频道、目前正在播放的第三部更为恶劣。

故事发生在一所学校，年轻的学生正在看一本教科书。一个女生说“他必须八成喜欢一个女生”，指的是一个男生，他不爽了就跑回家。心烦意乱的他问父母“我的同学都在笑我八成像女生。你以为我是男生还是女生？”当他们震惊地看着课本时。他的姐姐然后转向他们的父亲说:“爸爸，你必须是 100%的女孩”，因为双方的父母都在恐惧地盯着她。广告切入口号“支持准确的、适龄的性别平等教育”

(考虑到视频中对女性的厌恶，值得指出的是台湾政府不得不发布反对性别选择性堕胎的指导方针，因为对男孩的偏好扭曲了该国的性别比例)。

我是一个蹒跚学步的孩子的父母，当我们在 YouTube 上观看受欢迎的儿童频道时，我开始注意到这个广告。嵌入的示例是在 Super Simple Songs 的视频之前拍摄的，这是一个拥有超过 1200 万订户的频道，为非常年幼的孩子提供童谣和其他内容。虽然我在其他时候也看到过，但它在下午和傍晚出现的频率表明，3 Yes 计划在这些时段播放广告，因为许多孩子放学回家后，会看到针对他们的内容。

(值得注意的是，虽然 YouTube 内容创作者可以在播放视频之前过滤掉某些广告客户，但许多人可能不知道像 one by 3 Yes 这样的广告，因为他们不在同一个市场。超简单的歌，3 Yes，Google 都联系过评论。当联系到他们的脸书页面时，3 Yes 的人说他们不做电视广告。我澄清说我指的是 YouTube 广告，正在等待回复。)

这则广告尤其令人担忧，因为它有助于台湾反同性恋团体传播关于性和性别的错误信息。全民公决需要辩论，在其中一次全民公决中，全国家长协会主席联盟执行主任杨春子表示，儿童不应该在小学和初中学习同性恋问题，因为这可能会影响他们的性取向。越来越多的证据表明，性取向主要是由遗传因素决定的，这一观点已经被揭穿。

事实上，像杨和 3 Yes 这样的声明支持在台湾的学校进行更多而不是更少的关于 LGBTQ 问题的教育。《3 Yes》的视频似乎也违反了谷歌反对错误信息的新举措，尽管尚不清楚它是否违反了谷歌反对仇恨言论的政策。

但是，即使它没有煽动身体暴力，它也促进了对 LGBTQ 人群的有害刻板印象，并且是反同性恋广告在 YouTube 上出现的其他情况的可悲回声，有时出现在 LGBTQ YouTubers 的视频之前。这也提醒人们，YouTube 的试图使其平台对儿童更安全，包括“保护未成年人的情感和身体健康”，这是很难执行的。

想象一下，作为一个台湾的 LGBTQ 儿童，当你正在观看你最喜欢的频道时，看到这样的广告弹出来，而此时成年人即将就 LGBTQ 人群的基本权利进行投票，而你已经每天都面临着宣称你属于较低人类类别的横幅和传单。这是一个严峻的提醒，社交媒体平台和内容创作者必须在不同的市场更加关注允许针对儿童的广告和内容，特别是在选举和其他政治紧张时期。
