美国邮政服务暴露 6000 万用户数据

原文：https://web.archive.org/web/https://techcrunch.com/2018/11/26/the-us-postal-service-exposed-data-of-60-million-users/amp/

一个被破坏的美国邮政服务 API 通过允许研究人员向服务器发送通配符请求来提取数百万行数据，从而暴露了超过 6000 万用户。在向美国邮政总局反复请求后，由此产生的安全漏洞已被修补。

美国邮政的服务名为 InformedDelivery ，允许你在邮件到达你家之前查看邮件，并提供了一个 API，允许用户将其邮件连接到 CRMs 等专业服务。我们在 2017 年对这项服务进行了简介。

这位匿名研究人员表示，该服务在许多搜索中接受通配符，允许任何用户看到网站上的任何其他用户。Brian Krebs 在他的网站上有一份 API 代码的拷贝。

USPS 告诉 Krebs，它已经调查了数据泄露事件，并且:

计算机网络不断受到试图利用漏洞非法获取信息的犯罪分子的攻击。与其他公司类似，邮政服务的信息安全计划和检查服务使用行业最佳实践来持续监控我们的网络以发现可疑活动。

任何表明犯罪分子试图利用我们网络中潜在漏洞的信息都会被严肃对待。出于大量的谨慎，邮政服务正在进一步调查，以确保任何人可能试图访问我们的系统不适当的是追求最大限度的法律。

Krebs 还报道称，身份窃贼正在滥用这项服务来查看哪些邮件会在哪天到达用户家中，从而让他们随意获取重要文件和支票。API 漏洞已经被修补，但是还不知道在这个强大的工具中会出现什么其他处理不当的特性。
