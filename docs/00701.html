来自九个国家的议员告诉扎克伯格的问责替身 TechCrunch，问题在于脸书

原文：https://web.archive.org/web/https://techcrunch.com/2018/11/27/the-problem-is-facebook-lawmakers-from-nine-countries-tell-zuckerbergs-accountability-stand-in/

在今天早些时候的听证会上，一个由国际议员组成的大委员会(grand committee)没有人主持马克·扎克伯格(Mark Zuckerberg)的听证会，此前这位脸书创始人拒绝接受多次邀请，以面对有关恶意、滥用和不当使用其社交媒体平台的问题——包括所谓“假新闻”的民主削弱影响

英国 DCMS 委员会一直带头指控脸书对数据滥用丑闻和选举干预负责，现在世界各地的国际立法者也加入了这一努力。但仍然不是扎克伯格本人。

总之，来自九个国家的议员在房间里向扎克伯格的替身、政策副总裁理查德·艾伦提出了尴尬的问题——包括询问脸书正在采取什么措施来阻止 WhatsApp 被用作在南美传播政治虚假信息的载体；为什么脸书拒绝删除斯里兰卡一段高度煽动性的反穆斯林仇恨言论(直到斯里兰卡封锁了对其平台的访问)；脸书如何继续跟踪比利时的非用户，以及它如何在欧洲严格的新 GDPR 框架下证明这样做是合理的；更普遍的是，在公司深陷隐私和信任丑闻的情况下，为什么任何人都应该相信公司在这一点上说的任何话。

当选代表代表英国、阿根廷、比利时、巴西、加拿大、法国、爱尔兰、拉脱维亚和新加坡的近 4.5 亿人集体发声。他们最常挂在嘴边的问题是为什么扎克伯格不在那里？

艾伦代表他缺席的老板看起来很不自在，在三个小时的大部分时间里，他都在做各种安抚性的手势，他谈到希望与监管机构合作，找到“正确的监管”，以控制社交媒体的反社会、反民主影响。

加拿大议员鲍勃·齐默(Bob Zimmer)代表房间发言，打断了艾伦的另一段辩护:“在这里，我们再次听到脸书的另一个道歉——‘看，相信我们，你们都在监管我们，但我们在全球事务中真的没有那么大的影响力。’

“在这个房间里，我们监管着 4 亿多人，不让你们的首席执行官坐在那张椅子上是对我们所有人的冒犯，实际上也是对我们公民的冒犯。”

“[黑莓联合创始人]吉姆·巴尔西莱说，当我在我们的委员会中问他时，如果我们不改变加拿大的法律以应对监控资本主义，我们的民主会有危险吗？”齐默继续说道。“他说:‘毫无疑问’。你怎么看？”——艾伦以此为线索，开始了另一系列“我们需要 tos”和喋喋不休的“许多有问题的载体”，他说脸书正试图用许多“工具”来解决这些问题

会议在很大程度上充满了这种令人沮丧的重构的废话，因为艾伦试图转移，方得和化解委员会的问题——导致它不止一次地指责他重复《纽约时报》最近报道的“延迟，否认，转移”策略。

艾伦声称没有——声称在那里“承认”问题。但是他旁边的空椅子看起来很尴尬。

在收盘时，加拿大的查理·安格斯试图扫除脸书的热气，他指责艾伦用症状的讨论分散了辩论的注意力，以转移监管机构对问题根源的注意力，他尖锐地将问题定义为脸书本身。

“我们与脸书的问题是从来没有问责制。因此，当我们谈论监管时，我要告诉你，也许最好的监管是反垄断，”安格斯说。“因为不喜欢脸书的人——哦，他们可以去 WhatsApp。但是，哦，我们在南美有一些问题，我们在非洲有问题，我们必须回到扎克伯格先生身上，他不在这里。

“我的女儿们可以离开脸书。但他们会去 Instagram。但现在由脸书控制。或许最简单的监管形式是拆分脸书——或者将它视为一种公用事业，这样我们就能感觉到，当我们谈论监管时，我们谈论的是允许竞争，计算实际上诚实和真实的指标，而脸书破坏了如此多的信任，允许你简单地吞并所有形式的竞争，这可能不符合公共利益。

“所以当我们谈论监管时，你有兴趣问问你的朋友扎克伯格先生，我们是否可以就反垄断进行讨论？”

艾伦伸手去拿一个“这取决于我们试图解决的问题”的回答，试图重新构思这个问题。

“问题是脸书，”安格斯反驳道。“我们在谈论症状，但问题是对各种形式的社会话语和交流的前所未有的经济控制。那是脸书。这是我们需要解决的问题。”

委员会主席达米安·科林斯(Damian Collins)也对艾伦试图模糊地重新定义这一系列问题的做法不屑一顾——因为监管机构主张“关闭互联网”(而不是安格斯实际上呼吁的:一种从企业获得“可信的民主回应”的方式)——他插话说:“我认为我们也应该区分互联网和脸书，以表明它们不一定是一回事。”

房间确认它符合那个。

会议开始时，科林斯透露，委员会不会——至少目前不会——公布它在本周末从一家初创公司的创始人那里戏剧性地获得的文件缓存，该公司自 2015 年以来一直在起诉脸书，称它“无法这样做”尽管在后来的新闻发布会上，他说目的是公布这些文件。

在会议期间，DCMS 委员会的成员似乎还在几个地方取笑了从文件中摘录的一些新细节，比如问脸书是否曾为开发者做出 API 决定，条件是他们在其平台上做广告。

艾伦说没有——似乎是试图表明，委员会可能一直在阅读的电子邮件是“正常”内部商业讨论的结果，讨论的内容是如何为移动优先的时代发展脸书最初基于桌面的商业模式。

科林斯在听证会上确实详细说明了一条新信息，他明确指出这条信息来自被扣押的文件，特别是来自一名脸书工程师 2014 年 10 月发出的一封内部电子邮件，这表明这条信息似乎与重大公共利益有关。

“脸书的一名工程师在 2014 年 10 月通知该公司，拥有俄罗斯 IP 地址的实体一直在使用 Pinterest API 密钥，通过 ordered friends API 每天提取超过 30 亿个数据点，”他透露道，并问艾伦脸书是否“当时向任何外部机构”报告了这一情况

脸书副总统回应称，被扣押文件中包含的信息是“部分的”，因为它是通过“敌对诉讼当事人”获得的

“我不希望你利用这个机会攻击诉讼人，”柯林斯反驳道。“我希望你回答这个问题……当一名工程师向公司报告此事时,(脸书)的内部流程是怎样的？他们是否向外部机构通报了这一活动？因为如果俄罗斯的 IP 地址从平台上下载了大量数据——这些数据是被报告了，还是只是被保留了下来，就像经常出现的情况那样，只是保留在‘家庭’内部，而不被谈论。”

艾伦回应道:“你所看到的包含在电子邮件缓存中的任何信息，往好里说都是片面的，往坏里说都有误导的可能。”。“根据我们随后的调查，关于我们是否相信俄罗斯人在那个时候有活动的具体问题，我会回来找你。”

我们联系了 Pinterest，询问脸书是否曾告知其滥用 API 密钥的情况。在撰写本文时，该公司尚未回复我们的置评请求。

Pinterest 的一名发言人告诉我们:“我们目前正在调查，并且已经联系了脸书，以获取他们可能掌握的任何信息。”
