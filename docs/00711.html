AWS 推出托管区块链服务

就在一年前，AWS 首席执行官安迪·贾西说他对区块链服务不感兴趣。很明显，在过去的一年里发生了一些变化，因为今天，该公司推出了两项新的区块链服务:量子账本数据库和亚马逊管理的区块链。

顾名思义，AWS 托管区块链是一项托管区块链服务。它支持以太坊和超帐结构。

“这项服务将使你更容易使用两个最流行的区块链框架，”AWS 首席执行官 Andy Jassy 说。他指出，当企业知道其区块链网络中的成员数量，并希望获得强大的私人运营和能力时，它们往往会使用 Hyperledger Fabric。AWS 承诺，该服务将扩展到数千个应用程序，并将允许用户运行数百万个交易(尽管该公司没有说明会有什么样的延迟)。

对 Hyperledger Fabric 的支持现已推出。以太坊支持将在几个月后推出。

开始使用托管区块链是一个使用 AWS 控制台和配置节点、添加成员和部署应用程序的问题。

“当我们听到人们说‘区块链’的时候，我们感觉他们在奇怪地混淆他们真正想要的东西，”杰西说。“当我们花时间与客户一起工作，并找出他们真正想要解决的工作时，这就是我们认为人们试图与区块链一起做的事情。”
