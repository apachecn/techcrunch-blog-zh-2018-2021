AWS 推出新的时间序列数据库

AWS 今天在拉斯维加斯的 AWS re:Invent 上宣布了一个新的时间序列数据库。名为 Amazon Timestream 的新产品是一个完全托管的数据库，旨在随着时间的推移跟踪物品，这对于物联网场景尤其有用。

AWS 首席执行官 Andy Jassy 解释说:“对于时间序列数据，每个数据点都由一个时间戳和一个或多个属性组成，它真正衡量了事物如何随着时间的推移而变化，并有助于推动实时决策。”

不过，他认为现有的开源和商业解决方案存在一个问题，他说这些解决方案不能很好地扩展，并且难以管理。这当然是 AWS 这样的云服务经常帮助解决的问题。

毫不奇怪，因为他们的客户正在寻找一个好的时间序列数据库解决方案，AWS 决定自己创建一个。Jassy 表示，他们从头开始构建了 Amazon Timestream，采用了一种按时间间隔组织数据的架构，并实现了时序特定的数据压缩，从而减少了扫描次数，提高了性能。

“Timestream 自动执行汇总、保留、分层和压缩，因此可以高效地存储和处理时间序列数据。“Timestream 的查询引擎可以适应数据的位置和格式，使查询时间序列数据更容易、更快，”AWS 的杰夫·巴尔在一篇总结包括 Timestream 在内的几个新发布的博客文章中写道。

AWS 声称 Timestream 的速度将会快一千倍，而成本仅为关系数据库的十分之一，当然，它可以根据需要扩展和缩小，并包括您需要了解您正在跟踪的所有数据的所有分析功能。
