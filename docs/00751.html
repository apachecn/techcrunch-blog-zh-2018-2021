穿梭业务 Via 可能会在其产品线中增加滑板车

Via 是一家基于班车的拼车服务和平台，与美国和欧洲的城市合作，可能很快就会将滑板车加入其业务。

首席执行官兼联合创始人 Daniel Ramot 在 TechCrunch Disrupt Berlin 的舞台上表示，该公司正在尝试增加滑板车作为其穿梭业务的补充。

Ramot 说:“我们也在为我们的伙伴城市增加小型摩托车，他们将为居民提供一个整体的交通解决方案，作为公共交通产品。”

威盛面向消费者的班车位于芝加哥、华盛顿特区和纽约。该公司还与城市和交通部门合作，让客户能够访问他们的平台，部署自己的班车。例如，奥斯汀的首都大都会运输管理局使用 Via 平台为该市的接送服务提供动力。德国铁路公司 Arriva Bus UK 也使用 Via 的平台，提供连接通勤者和英国肯特高速火车站的第一英里和最后一英里服务

威盛还没有推出滑板车。但是 Ramot 在后台告诉 TechCrunch，Via 正在考虑在萨克拉门托推出滑板车，并且已经在与市政府官员进行谈判。方法是在已经有代步车的城市增加代步车。Ramot 说，如果没有核心的穿梭业务或平台，就不会推出踏板车。

威盛仍然非常专注于打造自己的航天飞机平台。Ramot 说，到明年年底，威盛希望在大约 300 个城市为公共交通系统供电。
