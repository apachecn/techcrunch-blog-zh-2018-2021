AWS想要统治世界TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/12/02/aws-wants-to-rule-the-world/

AWS曾经是亚马逊电子商务业务的一个不错的小配角，多年来已经成长为一个庞然大物，其运营成本为270亿美元，并且仍在以每年约45%的速度增长。从任何角度来看，这都是一项非常成功的业务，但当我上周在拉斯维加斯的 AWS re:Invent conference 上听AWS高管演讲时，我没有听到一个团体满足于坐以待毙，让增长自己说话。相反，我听说有一家公司想要主宰企业计算的每一个领域。

无论是硬件，如新的推理芯片和AWS前哨，新的本地服务器或区块链基站，如果AWS看到了机会，他们不会向任何人让步。

去年，AWS宣布了令人震惊的1400项新功能，据说今年将会超过这一数字。尽管他们拥有大量的市场份额，但他们并没有固步自封，而是像一个小得多的公司一样继续创新，这一点得到了很多赞誉。

功能膨胀可能不会永远持续下去，但至少现在他们没有显示出放缓的迹象，因为公告再次以疯狂的速度发布。虽然他们会告诉你，他们做出的每一个决定都是为了满足客户需求，但很明显，其中一些公告也是为了应对竞争压力。

更加努力地追赶竞争对手

在过去，AWS将对竞争对手的批评保持在最低限度，可能会给甲骨文一点点刺戳，但今年他们似乎变本加厉。在他们的主题演讲中，AWS首席执行官安迪·贾西和亚马逊首席技术官沃纳·威格尔不断抨击数据库市场的竞争对手甲骨文，但作为一家云公司，甲骨文目前几乎不构成主要威胁。

他们直奔甲骨文的市场，不过推出了一个名为AWS Outposts的新本地系统，该系统允许AWS客户使用单个AWS控制面板或VMware的控制面板在本地和云中操作。这是Larry Ellison可能提出的云愿景，但Jassy并不认为这是在追求甲骨文或其他任何人。“我不认为前哨站是任何人的箭。如果你看看我们正在做的事情，它在很大程度上是由客户告知的，”他在上周的新闻发布会上告诉记者。

AWS首席执行官Andy Jassy上周在AWS re:Invent的新闻发布会上

然而，AWS并不仅仅批评甲骨文。它还瞄准了微软，抨击了微软的SQL Server，还宣布了Amazon FSx for Windows File Server，一款专门设计用于将微软文件转移到AWS云的工具。

谷歌也未能幸免于推出推理和弹性推理，这让谷歌注意到AWS不会将人工智能市场让给谷歌的TPU基础设施。所有这些工具，以及更多的工具，不仅仅是为了满足客户需求，而是为了在企业计算的各个方面引起竞争。

向上增长轨迹

云市场继续以惊人的速度增长，作为市场领导者，AWS已经能够利用其市场主导地位。Jassy回应了Google的Diane Greene和Oracle的Larry Ellison，表示整个行业在采用云技术方面仍然处于起步阶段，这意味着仍然有大量的市场份额有待争夺。

“我认为，在美国以外的地区，我们在企业和公共部门的应用还处于初级阶段。我认为我们落后了12-36个月。所以有很多主流企业现在才开始规划他们的云计算之路，”Jassy说。

Moor Insights & Strategy的创始人兼首席分析师帕特里克·穆尔黑德(Patrick Moorhead)表示，AWS一直在利用其市场地位不断向不同领域扩张。“AWS目前的规模可以做许多其他公司做不到的事情，特别是像谷歌云平台和甲骨文云这样的小公司。他们试图用他们推出的数以千计的新产品和新功能来表明自己的观点。他对TechCrunch表示:“从长远来看，这对其他参与者来说是一种抑制，我相信这将导致一场洗牌。”。

至于创新的狂热步伐，穆尔黑德认为它不可能永远持续下去。“对我来说，问题是，我们什么时候能达到95%的需求得到满足，并且不需要创新率。每一个市场，实际上每一个市场，都会达到这种情况，所以这不是是否会发生的问题，而是什么时候发生的问题，”他说。

当然，像AWS地面站这样的领域表明，AWS愿意超越企业计算的传统界限，进入外层空间，帮助公司处理卫星数据。这种超越云计算资源传统用途的思考能力显示了一定程度的创造力，这表明AWS可能还有我们尚未想象到的其他未开发市场。

随着AWS进入企业计算堆栈的更多领域，无论是在内部还是在云中，他们都表现出了统治企业计算世界各个方面的愿望。上周，他们展示了他们不愿意向任何人放弃任何领域。
