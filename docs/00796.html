据报道，微软正在用基于 Chromium 的浏览器取代 Edge

原文：https://web.archive.org/web/https://techcrunch.com/2018/12/04/microsoft-is-reportedly-replacing-edge-with-a-chromium-based-browser/

据报道，微软正在用基于 Chromium 的浏览器取代 Edge

自 2015 年推出以来，微软 Edge 一直未能吸引公众的注意力，所以你不能责怪该公司改变策略。根据首次出现在 Windows Central 的新报道，浏览器在这个世界上并不长久。微软最早可能在本周宣布其替代品。

至于 Windows 10 默认浏览器的下一步，据报道，该公司正在寻求谷歌在这方面的帮助。据说下一代浏览器会发现微软用 Chromium 替换了 Edge 的 EdgeHTML 渲染引擎。

对于这个内部名称为“Anaheim”的项目来说，所有这些仍处于早期阶段，但 Internet Explorer 的替代品有望解决一些阻碍采用的稳定性和兼容性问题。它还可以帮助浏览器在 ARM 驱动的 Windows 机器上更好地工作。

Chromium 浏览器可能会有更大的灵活性，如果该公司真的选择全力以赴与 Chromebook 竞争，而不是推出 Windows 10 的精简版。区分浏览器和 Chrome 完全是另一个问题。

如果微软希望从谷歌手中夺回可观的浏览器市场份额，它需要在这方面得到一些帮助。根据最近的数据，Chrome 控制了超过一半的全球浏览器市场。
