微软 Edge 采用 Chromium(和 MacOS)

原文：https://web.archive.org/web/https://techcrunch.com/2018/12/06/microsoft-edge-goes-chromium-and-macos/?guccounter=1

谣言是真的:微软 Edge 正在向开源 Chromium 平台转移，这个平台也是谷歌 Chrome 浏览器的驱动平台。一旦做到了这一点，微软也将为 macOS 带来优势。此外，微软正在将 Edge 从 Windows 更新过程中分离出来，以提供更快的更新节奏——这样，它也将为 Windows 7 和 8 用户带来新的优势。

不过，这一切发生还需要一段时间。今天没有要测试的代码，第一个预览版还有几个月的时间。但是在 2019 年的某个时候，微软的 EdgeHTML 和 Chakra 将会消失，而 Blink 和 V8 将会取而代之。该公司预计明年初发布第一个开发者预览版。

显然，这里有很多东西需要解开。不过，有一点是清楚的，微软承认 Chrome 和 Chrome 是当今事实上的标准，对用户和开发者都是如此。

多年来，特别是在微软放弃了 Internet Explorer 品牌之后，Edge 在很大程度上已经成为一个完全可用的浏览器，但微软承认总是存在兼容性问题。虽然它投入巨资来解决这些问题，但我们从微软那里听到的是一个非常务实的信息:它根本不值得再投资于工程资源。毕竟，微软要做的是对某些网站上的问题进行逆向工程。

在某种程度上，这是因为 Edge 从未真正获得过 T2 的市场份额，在那里开发者们足够关心在平台上测试他们的代码。随着网络越来越大，不兼容网站的长尾仍然大量存在。

因为许多 web 开发人员在 MAC 上工作，他们无法访问 Edge，所以对它的测试变得更像是事后的想法。因此，微软在放弃为 Mac 电脑开发的ie 浏览器 15 年后，努力为 Mac 电脑带来优势。该公司预计 Edge on Mac 不会获得任何显著的市场份额，但它相信，在每个平台上都可以使用它将意味着更多的开发者也将使用 Edge 测试他们的网络应用。

微软也承认，Edge 只在 Windows 10 上运行并没有帮助，Edge 更新也绑定到了 Windows 更新。我一直不太确定为什么会这样，但正如微软现在高兴地承认的那样，这意味着数百万使用旧版本 Windows 的用户被抛在了后面，甚至那些使用 Windows 10 的用户也经常无法获得最新、最兼容的 edge 版本，因为他们的公司仍然落后一些更新。

无论好坏，Chrome 已经成为默认，微软也在随波逐流。该公司本可以选择开源 EdgeHTML 及其所有的 JavaScript 引擎(有些部分已经开源)。这一选择摆在桌面上，但最终它选择不这样做。该公司表示，这是因为当前版本的 Edge 与 Windows 10 有太多的联系，如果微软想将新的 Edge 带到 Windows 7 和 Mac 上，这样做根本没有多大意义。公平地说，这可能是一个傻瓜的差事，因为很难想象一个围绕 Edge 的开源社区会在解决实际问题上有很大的不同。

通过这一举措，微软还计划增加其在 Chromium 社区的参与。这意味着它将为 Chromium 带来一些曾让 Edge 在触摸屏上运行良好的工作。而且，正如之前报道的，该公司现在公开指出，它正在与谷歌和高通合作，在 ARM 上为 Windows 10 带来 Chrome 浏览器的原生实现，使其比当前严重依赖仿真的版本更快、更电池友好。

微软希望，如果它能让兼容性问题成为过去，用户仍然会被它的浏览器所吸引，因为它与众不同。也许这是它的 Cortana 集成或与 Windows 和 Office 的新集成。或者，这些可能是新的消费者服务，或者对于企业用户来说，是使 IT 经理的生活变得更轻松的特定功能。

几天前，当这一变化的谣言首次出现时，一些专家认为这对 web 来说并不好，因为它给了 Chromium 项目更多的权力来控制 web 标准。

我同意其中一些担忧，但微软正在为这一举措进行非常务实的论证，并指出 Edge 的市场份额很小，无论如何都不允许它在这一过程中有所作为。通过在 Chromium 社区中变得更加活跃，它将拥有更大的发言权——或者说它希望如此——并且能够倡导网络标准并为 Chromium 带来自己的创新。

你的浏览器可能是目前你电脑上运行的最复杂的软件。这意味着关掉引擎绝不是小事。该公司没有详细说明其开发流程将是什么样子，以及它将如何进行，但我们被告知，该公司正在考虑保留 Edge 体验的哪些部分，然后将与 Chromium 社区合作，将这些内容也引入 Chromium 引擎。

顺便说一下，微软强调它不会放弃 Edge。浏览器不会去任何地方。如果你现在是一个快乐的 Edge 用户，那么从长远来看，这一举措可能会让你成为一个更快乐的 Edge 用户。如果你不是，微软希望当新的基于 Chromium 的版本发布时，你会给它一个新的外观。微软现在要做的是打造一款与众不同的浏览器，足以让人们再试一次。
