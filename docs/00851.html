Krisp 使用机器学习来降低通话噪音，它很快就会出现在 Windows 中

原文：https://web.archive.org/web/https://techcrunch.com/2018/12/10/krisp-reduces-noise-on-calls-using-machine-learning-and-its-coming-to-windows-soon/

如果你的运气和我一样，只要你接了一个重要的电话，就会有人决定是时候吹掉你窗外人行道上的树叶了。2Hz 的 Krisp 是一款新的桌面应用，它使用机器学习来消除背景噪音，或者人群，甚至是哭泣的孩子——同时保持你的声音完好无损。它已经在苹果电脑上发布了，很快也会在 Windows 上发布。

今年早些时候，我在加州大学伯克利分校的 Skydeck 加速器会见了 Krisp 的创始人，包括 2Hz 的联合创始人 Davit Baghdasaryan，他们展示了当时的原型技术。

所涉及的技术很复杂，但想法很简单:如果你创建一个机器学习系统，它理解人类的声音，平均起来，那么它可以听一个音频信号，并只选择那一部分，删除大量的背景噪音。

Baghdasaryan 以前在 Twilio 工作，最初想创造一种可以在移动网络上运行的东西，所以 T-Mobile 或任何人都可以吹捧内置的噪音消除功能。然而，这种平台方法被证明太慢，所以他们决定直接面向消费者。

“吸引客户的速度很慢，这对一个年轻的初创公司来说是个问题，”巴格达萨扬后来在一封电子邮件中说。然而，人们喜欢“静音”的想法，所以我们决定改变我们所有的关注点，打造一款面向用户的产品。"

顺便说一下，那是我和他们面对面交谈的时间，仅仅六个月后他们就发布了 Mac。

很简单:你运行应用程序，它会修改传出和传入的音频信号，正常的嘈杂信号从一端传入，干净、专注于语音的信号从另一端传出。一切都发生在设备上，延迟非常短(大约 15 毫秒)，因此不涉及云，任何东西都不会发送到任何服务器，甚至不会存储在本地。该团队正致力于让软件动态适应和学习，但还没有实现。

这种方法的另一个好处是，它不需要任何特殊的调整就可以使用 Skype 而不是 Webex。因为它在操作系统的声音处理层面上工作，所以无论你使用什么应用程序，都会听到 Krisp 修改的信号，就好像它是从你的麦克风中清除出来的一样。

他们在 Mac 上推出，因为他们觉得早期采用者更有可能在苹果的平台上，这个赌注似乎得到了回报。但是 Windows 版本很快就要推出了——具体日期尚未确定，但预计会在本月底或 1 月初推出。(上线后我们会通知您。)

它应该或多或少与 Mac 版本相同，但将有一个特殊的游戏为重点的版本。Baghdasaryan 指出，游戏玩家更有可能拥有运行 Krisp 的 GPU，并且也真正需要清晰的通信(作为一名 PUBG 玩家，我可以对开放的麦克风和噼啪的按键表示厌烦)。因此，可能会有一些专门针对游戏玩家的超级用户功能，但还不是板上钉钉的。

你可能会像我一样感到奇怪，为什么他们不去追查芯片制造商，也许会把 Krisp 作为一种技术嵌入到手机或电脑的音频处理器中。

他们亲自提出，这最终也太慢，限制太多。与此同时，他们看到在软件领域没有真正的竞争，而进入软件领域要容易得多。

“目前所有的噪音消除解决方案都需要多个麦克风和一个特殊的外形，嘴巴必须靠近其中一个麦克风。我们没有这样的要求，”巴格达萨良解释说。“我们可以使用单话筒，也可以通过来自网络的音频流进行操作。这使得在任何您想要的环境(边缘或网络)和任何方向(入站或出站)运行软件成为可能。”

如果你对技术方面的事情感到好奇——它是如何用一个麦克风或以低延迟完成的，等等——Baghdasaryan不久前为 Nvidia 博客写了一个很好的解释。

此外，Krisp 可以轻松运行的人工智能芯片的激增意味着进入移动和嵌入式领域很容易。“我们已经成功地将我们的 DNN 移植到 NVIDIA GPUs、英特尔 CPU/GNA 和 ARM 上。Baghdasaryan 指出:“高通正在酝酿之中。

为了完成这项工作，该公司迄今已筹集了 200 万美元:50 万美元来自 Skydeck 以及朋友和家人的种子前融资，然后是由 Sierra Ventures 和盛大集团牵头的 150 万美元融资。
