价值19亿美元的InVision获得1.15亿美元的F系列

原文：https://web.archive.org/web/https://techcrunch.com/2018/12/11/invision-valued-at-1-9-billion-picks-up-115-million-series-f/

“屏幕正在成为世界上最重要的地方，”InVision首席执行官兼创始人克拉克·瓦尔伯格说。事实上，你很难在没有听到的情况下与他交谈。而且，考虑到他的公司年经常性收入已经增长到1亿美元，他有理由相信自己的断言。

InVision是一家希望成为设计销售力量的初创公司，随着1.15亿美元的F轮融资结束，该公司正式获得了独角兽地位，使该公司的总资金达到3.5亿美元。这笔交易对InVision的估值为19亿美元，是其继1亿美元E轮融资后，截至2017年年中估值的近两倍。

Spark Capital领投了这轮融资，参与方包括高盛(Goldman Sachs)，以及Battery Ventures、亚由美资本、Tiger Global Management、FirstMark和Geodesic Capital等现有投资者。Atlassian也参加了这一轮。今年早些时候，Atlassian和InVision建立了更深入的集成，允许吉拉、Confluence和Trello用户通过InVision即时协作。

作为交易的一部分，Spark Capital的Megan Quinn将与现有的董事会成员和观察员Amish Jani、Lee Fixel、Matthew Jacobson、Mike Kourey、Neeraj Agrawal、Vas Natarajan和Daniel Wolfson一起加入董事会。

InVision于2011年成立，作为一个简单的原型工具。它允许设计人员构建他们的体验，而无需要求工程/开发团队实际构建，然后发送给工程、产品、营销和执行团队进行协作和/或批准。

多年来，该公司在该过程的上下游都做出了努力，建立了一个名为InVision Cloud的完整协作套件，以便组织的每个成员都可以参与到设计过程中；工作室，一个设计平台，旨在承担Adobe和Sketch之类的工作；和InVision Design System Manager，设计团队可以从一个地方管理他们的资产和最佳实践。

但也许比InVision为设计师打造设计产品的能力更令人印象深刻的是它吸引非设计师用户的能力。

“最初，我认为我们没有意识到免费增值模式在一个组织内部扮演了一个飞轮的角色，”Quinn说。“这些设计师不仅邀请自己团队或其他团队的设计师，还邀请项目经理、市场营销、客户服务和高管合作并批准设计。从外面看，InVision像是一家设计公司。但实际上，他们从作为核心客户的设计师开始，并在一个组织内病毒式传播，为大众服务。”

InVision简单地主导了原型和协作，今天宣布它已经超过500万用户。更何况，InVision的客户种类繁多。这家初创公司拥有一长串令人印象深刻的数字优先客户——包括网飞、优步、Airbnb和Twitter——但也服务于《财富》100强中97%的企业，客户包括阿迪达斯、通用电气、美国宇航局、宜家、星巴克和丰田。

其中一部分可以归因于产品的质量，但向数字化的根本转变(正如瓦尔伯格所预测的)肯定正在进行中。不管品牌喜欢与否，客户越来越多地在屏幕后与他们互动，数字化客户体验对所有T2公司越来越重要。

事实上，麦肯锡的一项研究表明，在麦肯锡设计指数中排名前四分之一的公司在收入和股东总回报方面都超过了同行两倍之多。

但是和任何转变一样，有些人反对改变。Valberg认为行业教育和布道是InVision面临的两大挑战。

“组织不会很快改变像设计这样的事情，这就是为什么我们建立了一个设计转型团队，”Valberg说。“团队走进去，接触品牌，帮助他们采用新的做法，并在组织内实现设计成熟。”

凭借1.15亿美元的新收入和500万用户，InVision几乎拥有了进入新竞争层次所需的一切。即使是像Adobe这样的巨头，仅在第三季度就为带来了22.9亿美元的收入，InVision也提供了既能互补又能竞争的产品。

但是奎恩认为，视觉的未来取决于执行力。

“和大多数公司一样，最大的挑战将是持续卓越的执行，”奎因说。“InVision拥有合适的团队、优秀的产品和优秀的客户，可谓顺风顺水。这一切都是为了在团队前进的道路上提前构建和执行。”
