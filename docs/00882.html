AtScale 获得由摩根士丹利牵头的 5000 万美元投资

帮助公司将大量数据转移到商业智能和分析工具的初创公司 AtScale 今天宣布了 5000 万美元的 D 轮融资。

摩根士丹利领投了这轮融资，之前的投资者 Storm Ventures 和 Atlantic Bridge 也加入了进来。新投资者富国银行也参与其中。这笔资金几乎是在该公司宣布其 2500 万美元的 C 轮后整整一年。今天的资金筹集总额达到 1.2 亿美元。

引入摩根士丹利这样的机构投资者通常是一个信号，表明该公司已经到了至少开始考虑未来某个时候上市的可能性的阶段。AtScale 首席执行官克里斯林奇(Chris Lynch)承认了这种联系，但没有做出任何广泛的承诺(如你所料)。林奇告诉 TechCrunch，“我们离 IPO 还很远，但这是未来选择摩根士丹利的考虑因素。”。

该公司所做的是帮助获取大数据，并将其转移到客户可以更好地利用它的工具中。AtScale 联合创始人戴夫·马里亚尼(Dave Mariani)曾在雅虎工作，他在 2009/2010 年期间帮助推动了大数据的使用。不幸的是，当时的系统无法处理如此大的数据量——这仍然是一个问题，AtScale 表示它旨在解决这个问题。去年在 C 轮融资时，Mariani 告诉 TechCrunch】“我们把一堆数据孤岛放在数据平台上，并以一致的方式公开它们。这使得公司能够获得其数据的整体视图，而不是以较小的块消耗数据。

AtScale 报告了辉煌的一年，在零售、金融服务、广告和数字销售等目标垂直领域带来了 50 家新客户。其中包括乐天、戴尔科技、道明银行和丰田。更重要的是，该公司今年进行了扩张，利用上一轮融资向欧洲和亚洲的国际市场扩张。

该公司成立于 2013 年，总部位于加利福尼亚州圣马特奥。
