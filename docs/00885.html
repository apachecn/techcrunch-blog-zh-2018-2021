已确认的VPN希望给VPN行业带来透明度

VPN行业烂透了。数十家公司向你承诺完美隐私这一不可能实现的梦想。但这简直是弥天大谎。一家名为的公司证实，VPN 希望通过让VPN公司承担责任来改变这种状况。

VPN公司让你在你的设备和某个数据中心的服务器之间建立一个加密隧道。虽然没人能看到隧道里有什么，但VPN公司可以看到他们服务器上的一切。

许多可疑的公司利用这一点来分析你的浏览习惯，卖给广告商，在不安全的页面上注入他们自己的广告，或者窃取你的身份。最糟糕的是，他们可以向当局提供大量关于你网络生活的数据。

他们在隐私政策上撒谎，甚至经常没有一个关于为这些公司工作的人的名字的页面。他们花费大量金钱购买评论和代言。

不要相信他们中的任何人。

换句话说，VPN服务不会让你在互联网上更安全。安装 HTTPS Everywhere ，安装一个广告拦截器，把你的DNS设置改成 Quad9 或者Cloudflare的1.1.1.1。这些是保护你的连接的更好的方法。

现在我已经把这个问题解决了，Confirmed VPN是又一个想要尝试新事物的VPN服务。其背后的团队(Duet Display的拉胡尔·万德和前iCloud工程师林剑衡)已经开源了其客户端和服务器端组件的代码。然后，它会自动在Amazon Web Services上部署新的提交。

该公司使用AWS CloudWatch来监控服务器上的异常活动，以证明他们没有下载日志或类似的东西。安全专家也可以使用只读凭证登录AWS。确认的VPN也已经完成了两次安全审计，并且有一个bug奖励计划。

我不是安全专家，所以我不能认可或推荐确认的VPN——记住，我仍然认为你不应该使用VPN服务。但是这种透明的方法本身就很有趣。现在我们来看看竞争对手是如何反应的。
