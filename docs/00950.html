脸书开源 PyText NLP 框架

原文：https://web.archive.org/web/https://techcrunch.com/2018/12/14/facebook-open-sources-pytext-natural-language-processing-framework/amp/

脸书人工智能研究所正在开源一些对话式人工智能技术，用于支持其门户视频聊天显示和 Facebook Messenger 上的 M 建议。

该公司今天宣布，其基于 PyTorch 的 PyText NLP 框架现已向开发者开放。

自然语言处理涉及系统如何解析人类语言，并能够做出决策和获得洞察力。脸书说，PyText 框架被该公司视为人工智能研究人员更快地在实验和部署之间移动的渠道，对文档分类、序列标记、语义解析和多任务建模等任务特别有用。

该公司构建了一个框架，可以非常无缝地适应研究和生产工作流，强调健壮性和低延迟，以满足公司的实时 NLP 需求。该产品负责为脸书每天超过 10 亿次预测提供动力的模型。

另一大亮点是该框架的模块化，允许它不仅从头开始创建新的管道，还可以修改现有的工作流。PyText 连接到 ONNX 和 Caffe2 框架。除了通过多次运行来训练模型的分布式训练之外，它还支持一次训练多个模型。

该公司显然还没有完成对其 NLP 框架的改进。脸书表示，未来他们将特别关注为在移动设备上运行的模型建立端到端的工作流程。

PyText 可以在 GitHub 上获得。
