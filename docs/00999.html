2019 年，Target 的当天送达服务将包括“所有主要产品类别”TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/12/18/targets-same-day-delivery-service-shipt-will-include-all-major-product-categories-in-2019/

Target 对 Shipt 有着宏伟的计划，Shipt 是一年前以 5 . 5 亿美元收购的当日杂货配送服务。虽然如今人们普遍认为 Shipt 是 Instacart 的竞争对手——实际上是 Target 对沃尔玛和亚马逊杂货配送业务的回应——但在过去一年里，Shipt 已经扩展到包括更多 Target 的产品。该公司表示，到 2019 年，Shipt 将提供“所有主要产品类别”的当天交付

如今，Shipt 为超过 55，000 种杂货和其他必需品以及部分电子产品、玩具和其他产品提供当天送达服务。

希普特告诉 TechCrunch，明年，送货服务将扩展到所有主要产品类别。

这意味着服装、毛巾等商品将可以在当天送达。该零售商拒绝透露具体细节，如将涉及多少额外的 SKU，或 2019 年这一类别扩张将于何时开始。

然而，此举可能会改变塔吉特百货的游戏规则，该公司一直在改造其业务，以更好地适应人们想要的所有购物方式，包括在线和离线。T2 一直在改造其商店，为免下车顾客增加停车位，即那些在网上下单当天从商店取货的顾客。它已经更新了商店的布局，因此当你走进商店时，在线订单提货、自助结账和外卖杂货必需品就在前面。

但 Target 仍然面临的挑战之一是，很难确定哪些商品可以通过哪种送货或提货方式获得。即使你在 Target 的移动应用中搜索和过滤“店内”商品，你也会看到缺货的商品，以及那些你只能在线订购才能送货到家的商品。

与此同时，塔吉特百货的很多商品——比如冷杂货和衣服——都无法通过“驱车到达”或“送货上门”这两种当天送达服务获得。(杂货只通过 Shipt 服装只能通过免下车服务或订单提货。)

最终的结果是，除非你走进商店，否则你无法完成包括所有类型商品的完整的“目标运行”。这也可能引起顾客的困惑，因为免下车和送货上门提供了不同种类的产品。为什么当天送达的商品不能同时用于取件或送货？

但如果 Shipt 在 2019 年发展到包括所有主要产品类别，它可能会成为目标客户的首选当天服务。

2018 年，Shipt 经历了大规模扩张，此前 Target 承诺，它和 Drive Up 将于 2018 年假期在全国范围内推出。

这家零售商在一篇关于 Shipt 一周年的公司博客文章中指出，送货服务现在已经覆盖了美国 46 个州的 200 多个市场，而在收购时只有 70 个市场。零售商 Shipt 表示，自加入 Target 以来，其会员人数增加了两倍，并在伯明翰、阿拉巴马州和旧金山的办事处雇佣了超过 375 名新员工。

Target 表示，Shipt 将在 2019 年继续进行地理扩张，并将伯明翰总部的员工人数增加“数百人”。
