Corel确认已收购虚拟化专家Parallels

原文：https://web.archive.org/web/https://techcrunch.com/2018/12/20/corel-confirms-acquisition-of-parallels/

上个月，我们爆料称，Corel正在收购虚拟化领域的先驱Parallels T1，以扩大其在商业软件领域的影响力。现在这笔交易是正式的:今天总部位于加拿大的 Corel 证实它已经收购了这家公司。现在，像CorelDraw这样的产品和WordPerfect这样的生产力应用程序将与Parallels Desktop for Mac、Parallels Toolbox for Windows and Mac、Parallels Access、Parallels Mac Management for Microsoft SCCM和Parallels Remote Application Server(RAS)并列使用——这些产品主要面向运行计算机网络的企业，这些计算机网络不是统一运行一个操作系统，而是跨多个位置。

Corel首席执行官Patrick Nichols在声明中表示:“我们很高兴欢迎新的Parallels员工加入Corel的全球组织。“从我们高度互补的产品组合到我们共享的商业模式和战略，Corel和Parallels非常契合。得益于我们的技术和团队的合力，我们看到了巨大的机会来推动我们的业务持续增长和成功，最重要的是，我们的客户。”

交易条款尚未披露，但我们从接近该公司的消息人士处了解到，这是一笔全现金交易，而且这不是一笔大的退出。

商业世界中不断变化的实践——基于云的服务的兴起，越来越普遍的自带设备硬件政策，以及传统的单一办公室之外更广泛的工作环境——创造了对更好的软件管理实践的需求，不仅涵盖安全性等重要事项，还包括可用性。这给虚拟化公司带来了机会。Corel表示，它预计将“对Parallels业务进行重大投资”，因为它认为它拥有“非凡的增长机会”。在这种情况下，这可能包括其现有产品和Parallels产品之间的更紧密集成。

“Parallels和Corel拥有共同的愿景、市场领导地位和创新热情。作为Corel的一部分，我们期待着继续为我们的客户和合作伙伴提供一流的软件，”Parallels首席执行官Jack Zubarev在一份声明中说道。

Corel自20世纪80年代在渥太华成立以来，已经多次易手，多次公开上市。它现在归 Vector Capital 所有，后者本质上是购买Parallels并制定投资策略的公司。

Parallels成立于1999年，起源于俄罗斯，目前总部位于华盛顿贝尔维尤。

它从未大肆宣传其融资或估值。根据PitchBook的说法，它的最后一轮融资是在2015年，来自Endeavour Vision、KG Investments、Maxfield Capital、Savano Capital Partners和其他公司的金额未披露。前年，它从Ingram Micro融资3亿美元。

正如我们之前所写的，我们并不完全清楚出售的理由是什么，只是我们听说它的投资者是长期投资者，正在寻求退出，而Corel一直在慢慢整合一些较小的软件业务，这与它在那里的更大战略非常吻合。另一项最近的收购是今年早些时候收购的来自德国的Gravit Designer。
