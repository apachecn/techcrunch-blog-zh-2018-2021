Corel 确认已收购虚拟化专家 Parallels

原文：https://web.archive.org/web/https://techcrunch.com/2018/12/20/corel-confirms-acquisition-of-parallels/

上个月，我们爆料称，Corel 正在收购虚拟化领域的先驱 Parallels T1，以扩大其在商业软件领域的影响力。现在这笔交易是正式的:今天总部位于加拿大的 Corel 证实它已经收购了这家公司。现在，像 CorelDraw 这样的产品和 WordPerfect 这样的生产力应用程序将与 Parallels Desktop for Mac、Parallels Toolbox for Windows and Mac、Parallels Access、Parallels Mac Management for Microsoft SCCM 和 Parallels Remote Application Server(RAS)并列使用——这些产品主要面向运行计算机网络的企业，这些计算机网络不是统一运行一个操作系统，而是跨多个位置。

Corel 首席执行官 Patrick Nichols 在声明中表示:“我们很高兴欢迎新的 Parallels 员工加入 Corel 的全球组织。“从我们高度互补的产品组合到我们共享的商业模式和战略，Corel 和 Parallels 非常契合。得益于我们的技术和团队的合力，我们看到了巨大的机会来推动我们的业务持续增长和成功，最重要的是，我们的客户。”

交易条款尚未披露，但我们从接近该公司的消息人士处了解到，这是一笔全现金交易，而且这不是一笔大的退出。

商业世界中不断变化的实践——基于云的服务的兴起，越来越普遍的自带设备硬件政策，以及传统的单一办公室之外更广泛的工作环境——创造了对更好的软件管理实践的需求，不仅涵盖安全性等重要事项，还包括可用性。这给虚拟化公司带来了机会。Corel 表示，它预计将“对 Parallels 业务进行重大投资”，因为它认为它拥有“非凡的增长机会”。在这种情况下，这可能包括其现有产品和 Parallels 产品之间的更紧密集成。

“Parallels 和 Corel 拥有共同的愿景、市场领导地位和创新热情。作为 Corel 的一部分，我们期待着继续为我们的客户和合作伙伴提供一流的软件，”Parallels 首席执行官 Jack Zubarev 在一份声明中说道。

Corel 自 20 世纪 80 年代在渥太华成立以来，已经多次易手，多次公开上市。它现在归 Vector Capital 所有，后者本质上是购买 Parallels 并制定投资策略的公司。

Parallels 成立于 1999 年，起源于俄罗斯，目前总部位于华盛顿贝尔维尤。

它从未大肆宣传其融资或估值。根据 PitchBook 的说法，它的最后一轮融资是在 2015 年，来自 Endeavour Vision、KG Investments、Maxfield Capital、Savano Capital Partners 和其他公司的金额未披露。前年，它从 Ingram Micro 融资 3 亿美元。

正如我们之前所写的，我们并不完全清楚出售的理由是什么，只是我们听说它的投资者是长期投资者，正在寻求退出，而 Corel 一直在慢慢整合一些较小的软件业务，这与它在那里的更大战略非常吻合。另一项最近的收购是今年早些时候收购的来自德国的 Gravit Designer。
