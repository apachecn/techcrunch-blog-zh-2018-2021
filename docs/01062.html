Spot 是一个加密货币应用程序，可以控制你所有的钱包和交换账户

原文：https://web.archive.org/web/https://techcrunch.com/2018/12/21/spot-is-a-cryptocurrency-app-to-control-all-your-wallets-and-exchange-accounts/

认识一下 Spot ，这是一款设计精美的移动应用程序，用于控制您的加密货币。Spot 看起来像一个投资组合跟踪应用程序。但该公司已经建立了一个强大的基础，将在未来几个月增加更多功能。Spot 希望成为您通往加密货币世界的唯一门户。

“Spot 的愿景不是建立一个投资组合追踪器——我们在这个功能上做得有点过头了，”联合创始人兼首席执行官爱德华·斯蒂格曼(Edouard Steegmann)告诉我。“最终，我们希望成为管理你所有密码的应用程序，一种带有加密 DNA 的 Revolut。”

首次安装该应用时，您可以通过添加公共地址将其连接到现有的钱包。即使你将代币存储在硬件钱包中，Spot 也可以读取你钱包的公共细节，以在应用程序中显示它们。

“我们在以太坊、比特币、莱特币、恒星和其他货币上有自己的节点，可以追回你钱包里的钱，”Steegmann 说。数据还会与第三方服务进行交叉检查，以确保一切正常。

Spot 还允许您使用 API 密钥连接到 exchange 帐户。目前，该应用程序支持币安、北海巨妖、Bitfinex 和 Poloniex，但该公司已经计划增加更多的交易所。

然后，该应用程序会向您详细介绍您在所有服务和钱包中的持有情况。您可以查看详细的图表，并发现哪个令牌的性能优于其他令牌。这也是我今年见过的设计最好的移动应用之一——动画和交互非常华丽。

但是 Spot 并不依赖 API 来获取每个令牌的定价信息。“我们从头开始重建了 CoinMarketCap，我们是少数几家做到这一点的公司之一，”Steegmann 说。该公司在 150 家交易所存储了数十种代币的定价信息。那是许多配对。

如果你点击应用程序顶部的 Spot 标志，你就可以看到你的投资组合的最大价值，如果你在最高价格的交易所兑现你的代币。该公司确保有足够的交易量向你展示一致的价格。

Spot 认为控制自己的数据太重要了，不能依赖 API 调用。当您拥有自己的数据时，您就没有任何 API 速率限制，没有主要的依赖性，并且可以更从容地进行扩展。

接下来，你可以直接在应用程序中交易。该公司不打算建立自己的交易所，但你可以在第三方交易所买卖代币，而不必访问网站。

“我们认为许多东西将被令牌化，没有用户友好的界面来传输、接收、购买和销售，”Steegmann 说。

该公司从基马风险投资公司和商业天使那里筹集了 120 万美元(准确地说是 105.6 万€)，其中包括 Ledger 的 Eric Larchevêque 和 Thomas France，Jean-Daniel 海底平顶山，Thibaud Elzière，Eduardo Ronzano，Nicolas Steegmann，Sébastien Lucas 和 Nicolas Debock。
