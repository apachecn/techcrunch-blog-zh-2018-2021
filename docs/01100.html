所以你想安全地、私密地浏览网页？事实是这几乎是不可能的。

你的网络提供商收集关于你的海量数据，比如你访问的网站，将其出售给广告商，促使一些州完全阻止这种做法。更不用说美国政府可以从你的网络浏览活动中收集数据。但也许最惊人的数据收集者是你访问的网站和你使用的应用程序。社交媒体巨头吸收你的数据，广告商在网络上跟踪你，你的搜索历史可以揭示你的健康问题、政治信仰和浏览习惯。

任何时候你使用互联网，你都会留下一串数据。你不能阻止这一切；这就是互联网和网络浏览器的工作方式。但是你可以做很多事情来减少你的数字足迹。

以下是一些安全浏览网页的小贴士:

VPN 不会让你匿名

你可能听说过很多关于虚拟专用网络(VPN)的事情，以及它们如何隐藏你的网络流量，不让窥探者发现。不完全是。

VPN 允许你通过一个专用的、通常是加密的隧道来传输所有的互联网流量，允许你对你的互联网提供商隐藏你的网页浏览历史。这可能有助于你绕过限制你的河流进入你所在地区的审查系统或地理区块。但除此之外，你只是将所有的互联网流量发送给了 VPN 提供商。

本质上，你必须决定你更信任谁:你的 VPN 提供商还是你的互联网提供商。

问题是，大多数免费的 VPN 提供商通过出售你的数据或为你提供广告来赚钱——有些只是彻头彻尾的黑幕或有安全问题会把你置于危险之中。即使你为了隐私而使用高级 VPN 提供商，他们也可以将你的支付信息与你的互联网流量联系起来，许多 VPN 提供商甚至懒得加密你的数据。

正如 TechCrunch 的 Romain Dillet 解释的那样，最好的 VPN 是你自己建立和控制的。你可以在几分钟内创建自己的 Algo VPN (如果你想使用云)或 WireGuard 服务器。Algo 是由纽约一家非常受信任和尊重的安全公司 Trial of Bits 创建的。GitHub 上有源代码，这使得偷偷在代码中插入后门变得更加困难。

除了将你的流量从一个地方转移到另一个地方，还有更好的方法来最小化你的数字足迹。

你需要一个安全的 DNS

“你的网络提供商知道你访问什么网站”到底是什么意思？

互联网依靠 DNS(域名系统)将网址转换成计算机可读的 IP 地址。大多数设备自动使用 DNS 解析器，该解析器由您所连接的网络(通常是您的互联网提供商)设定，这意味着您的互联网提供商知道您正在访问哪些网站。几年前，国会通过了一项法律，允许网络提供商向广告商出售你的网络浏览记录。

你需要一个安全和私人的 DNS 提供商。

最受欢迎的隐私友好产品之一是 Cloudflare 的安全 DNS ，它称之为1.1.1.1。Cloudflare 会对您的流量进行加密，不会使用您的数据提供广告服务，并且不会存储您的 IP 地址超过 24 小时。上手很简单，可以从苹果应用商店和 Google Play 下载 Cloudflare 的应用。

对于 Firefox 用户来说，HTTPS 域名系统(或 DoH)是内置的，默认情况下是启用的。DoH 对您的 DNS 查询进行加密，这可以向第三方隐藏您的网页浏览历史记录，同时也使您的 DNS 查询更难被拦截和劫持，从而将您重定向到恶意网站。

卸载旧插件，检查您的扩展

传统 web 插件的时代已经一去不复返了。Flash 和 Java 已经在网上被弃用多年了。因此，删除它们是安全的，因为这将减少恶意软件针对你的方式。

对于 Java hold-outs，有指令删除它的 Windows 和 macOS 。

如今，web 浏览器扩展为您的浏览器增加了功能。但是它们也会带来隐私和安全风险。像你手机上的应用程序一样，浏览器扩展通常需要访问你的浏览器、你的数据，甚至你的电脑，尽管受到网络浏览器制造商的审查，但扩展可能会偷偷摸摸，窃取或分享你的数据，或者秘密挖掘加密货币。

对于什么是好的扩展，什么不是，没有简单的规则。运用你的判断力。确保您安装的每个扩展不会要求比您认为它需要的更多的访问权限。并确保卸载或删除任何不再使用的扩展。

这些网络浏览器扩展可以保护您

  • 广告拦截器:广告拦截器可以很好地拦截广告——顾名思义——但也可以拦截隐私侵犯代码，这些代码可以跨网站跟踪你。 uBlock 是一个流行的、开源的高效拦截器，它不会像其他的一样消耗太多的内存。uBlock Origin 在 Chrome 、 Firefox 和 Edge 中工作，而扩展是开源的，所以任何人都可以看看它是如何工作的。
  • 跨网站追踪器拦截器: 隐私獾是一个伟大的工具，它阻止隐藏在网页上的微小“像素”大小的追踪器，但从一个网站到另一个网站追踪你，了解更多关于你的信息，为你提供广告。对广告商和追踪者来说，就好像你消失了一样。 Ghostery 是另一个高级反追踪器的例子，旨在保护用户免受隐藏追踪器的默认攻击。Safari 还带有内置的跨站追踪保护和一种隐藏你的 IP 地址的方法
  • Firefox 容器: Firefox 用户还可以利用多账户容器，这是一种浏览器功能，可以让你隔离浏览活动。这样，一个容器包含您的工作选项卡，另一个容器包含您的个人选项卡。通过将标签分开，网站很难跟踪你在网上的浏览活动。

改变长期存在的习惯对保护你的数字隐私也大有帮助。

你也可以考虑换一个更注重隐私的搜索引擎，比如 DuckDuckGo ，这是一个流行的搜索引擎，它承诺永远不会存储你的个人信息，并且不会跟踪你来提供广告。iPhone 和 iPad 用户也可以将 iOS 中的默认搜索引擎改为 DuckDuckGo。Brave 还提供关注隐私的网络浏览器和测试版的搜索引擎 T21。

使用 Tor 匿名

如果你想匿名，使用 Tor。

Tor 是一个匿名网络，它通过散布在世界各地的一系列中继服务器反弹你的互联网流量，这使得跟踪你的浏览活动变得非常困难。你可以使用 Tor 浏览器访问 Tor 网络，这是一个预配置的锁定版 Firefox。

您可以像访问任何其他 web 浏览器一样访问常规网站，也可以访问特殊网站。洋葱域名只能通过 Tor 访问。这些天来，许多大公司已经推出。洋葱域名为他们的服务，像脸书、推特和英国广播公司，在很大程度上是为了应对日益增长的全球审查。

Twitter 在 Tor 浏览器中运行的截图。图片:TechCrunch

Tor 让任何人都几乎不可能窥探你的网络流量，知道你在访问哪个网站，或者知道你是访问这个网站的人。活动家和记者经常利用 Tor 来规避审查和监视。

使用 Tor 并不是万能的。就像其他网络浏览器一样，你必须确保 Tor 浏览器是最新的。(它应该会自动下载更新，但请确保您重新启动以安装它们。Tor 项目有一个有用的指南，告诉你使用 Tor 需要知道什么。

只是不要期望 Tor 很快。虽然随着 Tor 的不断扩展，它的速度越来越快，但对于流媒体视频或访问带宽密集型网站来说，它并不实用。
