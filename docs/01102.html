为什么你需要使用密码管理器

原文：https://web.archive.org/web/https://techcrunch.com/2018/12/25/cybersecurity-101-guide-password-manager/

如果你认为密码时代将很快结束，请再想一想。他们会留在这里——暂时。

密码很麻烦，也很难记住——当你想起来的时候，你会被告知再次更改密码。有时密码可以被猜到，也很容易被破解。虽然没有人喜欢密码，但它们是生活中的事实。虽然的大科技和的初创公司的小公司试图用指纹和面部扫描技术取代密码，但这两种技术都不完美，许多人仍然求助于可靠(但令人沮丧)的密码。

如何让密码生活变得更简单？你需要一个密码管理器。

什么是密码管理器？

把密码管理器想象成一本你的密码书，由一把只有你自己知道的万能钥匙和其他安全保护装置锁住，保护你最敏感的秘密。

密码管理器不仅可以存储您的密码，还可以在您注册新网站时帮助您生成和保存强而唯一的密码。这意味着无论何时你登陆一个网站或应用程序，你都可以打开你的密码管理器，复制你的密码，粘贴到登录框中，你就进入了。通常，密码管理器带有浏览器扩展，可以自动为您填写密码。

因为许多密码管理器都有加密的跨设备同步功能，你可以把你的密码带到任何地方，甚至在你的手机上。

您可能认为密码管理器的“主密钥”听起来像是单点故障。有人拿到我的主密码怎么办？那是一种合情合理的恐惧。但是，假设您选择了一个强有力的、唯一的、但容易记住的、您在其他任何地方都没有使用过的主密码，这是保护您的其他密码不被非法访问的一种近乎完美的方法。此外，大多数密码管理器还带有额外的保护措施，如双因素认证，这意味着一个坚定的黑客无法仅凭您的主密码访问您的密码管理器。

为什么需要用一个？

密码管理器消除了创建和记住强密码的麻烦，并存储其他秘密，如令牌、信用卡甚至密钥。就这么简单。但是有三个很好的理由让你应该关心。

密码经常被盗。网站和服务不断面临被入侵的风险，就像你面临试图诱骗你交出密码的网络钓鱼攻击一样。尽管公司在存储密码时会对密码进行加密——这一过程被称为散列法——但并非所有公司都使用强大或现代的算法，这使得黑客很容易逆转散列法，以纯文本形式读取密码。有的公司根本懒得哈希！这将使你的账户面临欺诈的风险，或者你的数据面临被用来针对你的身份盗窃的风险(T2)。

但是你的密码越长越复杂，黑客破解的时间就越长。这可以是一个由大写和小写字符、数字、符号和标点符号混合而成的密码，或者，正如许多人正在走向的那样，使用故意冗长的密码短语，这些短语组成几个独特的单词，容易记住，但比较短的密码强得多。

另一个问题是我们必须记住的密码的数量。银行、社交媒体账户、我们的电子邮件和公用设施——只使用一个通用密码很容易。但这让“的证件填充”变得更加容易。这就是黑客从一个被入侵的网站获取你的密码，并试图在其他网站登录你的账户。使用密码管理器可以更容易地生成和存储每个站点独有的更强密码，从而防止凭据填充攻击。

而且，当你在一个拥挤或繁忙的地方——比如咖啡店或飞机上——想想谁在你周围。输入密码可以被附近的窃听者看到、复制和使用。在许多情况下，使用密码管理器可以消除键入任何密码的需要。更受欢迎的密码管理器功能还可以让你安全地与你的朋友或同事共享密码。

您应该使用哪个密码管理器？

简单的回答是，这取决于你。所有的密码管理器执行大致相同的任务，但不同的应用程序会比其他应用程序具有更多或更相关的功能。

最新的 iPhone 和 iPad 用户默认内置了密码管理器，所以没有理由不使用。您可以使用 iCloud 钥匙串在设备之间同步密码。苹果公司也为那些想在另一个浏览器或 Windows 电脑上访问他们的 iOS 密码的人提供了一个 Chrome 扩展。

对于其他人来说，大多数密码管理器都是免费的——可以选择升级以获得更好的功能。

例如，如果您希望您的密码能够跨设备同步，那么1 密码被广泛使用，而与 Troy Hunt 的 Pwned 密码数据库相集成，因此您可以判断(并避免！)先前已经在数据泄露中泄露或暴露的密码。

DashLane 为单个设备提供免费服务。

一些密码管理器是开源的，比如 KeePass 和 BitWarden 允许任何人阅读源代码。KeePass 不使用云，所以它永远不会离开你的电脑，除非你移动它。这对超级偏执狂来说更好，但对那些可能面临更广泛威胁的人来说也是如此——比如那些在政府工作的人。因为它是开源的，其他人已经接受了这个项目并创建了他们自己的开源版本，像 KeePassXC ，它包含了一系列更现代的功能。

尽管大多数浏览器，如 Chrome、Firefox 和 Safari T21 都允许你保存密码，但一些安全专家警告说，窃取密码的恶意软件以浏览器为目标，希望窃取密码和其他敏感机密。

像所有软件一样，任何密码管理器中的漏洞和弱点都会将您的数据置于风险之中。但是只要你保持你的密码管理器最新——大多数浏览器扩展会自动更新——你的风险就会大大降低。

简而言之:使用密码管理器比不使用密码管理器对你的整体安全来说是更好的 T2。
