开源货币化初创公司 Tidelift 融资 2500 万美元 B 轮

开源软件的诅咒在于，它几乎被用于这个星球上的每一个应用程序和设备，然而，它几乎没有商业模式。维持开源对于软件的未来来说是一个关键问题，因为没有持久的收入来源，这些关键项目背后的开发者就不能投入全部精力来改进、维护和保护它们。

Tidelift ，一家由一群长期开源工程师和高管创立的初创公司，已经以令人信服的方式解决了这个问题。正如我去年在深入研究开源可持续性的文章中所写的那样:

Tidelift 旨在“围绕软件的安全性、许可和维护等领域”提供保证，[Tidelift 首席执行官唐纳德]费舍尔解释道。这个想法起源于将 Linux 商业化的 Red Hat。这个想法是，当公司可以获得关键漏洞和长期支持等问题的保证时，他们愿意为开源付费。此外，Tidelift 还处理为商业化建立开源的日常任务，比如处理许可问题。

这仍然是该公司的使命，现在它有更多的资源来发展。该公司今天宣布，它已经在 B 轮融资中筹集了 2500 万美元，由 return backers General Catalyst，Foundry Group 和 Matthew Szulik 领导，后者是开源领导者 Red Hat 的前董事长兼首席执行官，去年被 IBM 以 340 亿美元的重磅交易收购。这是去年 1500 万美元首轮融资的后续。

自从我去年 6 月报道 Tidelift 以来，该公司已经从最初在 Javascript 生态系统中推出扩展到为 Java、Python、PHP、.NET 和 Ruby 生态系统。今天 Tidelift 订阅的著名开源项目包括 Apache Struts、Vue.js、Gulp、Carbon、Jekyll、Beautiful Soup 和 Mongoose。Tidelift 称其订阅现在覆盖了数百个开源软件包。

除了覆盖更多的包，Tidelift 去年 9 月宣布他们在开源维护者方面的承诺已经达到了 100 万美元。在一份新闻稿中，该公司强调社区和讨论平台话语是客户。

CEO Fischer 告诉我“我们的底线是开源不仅仅需要‘资金’。它需要一个对创作者和用户都适用的大规模商业模式。“该公司打算使用新的资金来进一步扩大其流行的开源软件包的覆盖范围，并与更多的开源创作者合作。
