阿里巴巴以 1.03 亿美元收购德国大数据初创公司 Data Artisans TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/01/08/alibaba-data-artisans/

阿里巴巴向€支付了 9000 万英镑(1.03 亿美元)收购 Data Artisans，这是一家总部位于柏林的初创公司，为企业提供分布式系统和大规模数据流服务。

这项交易首先由欧洲媒体宣布，包括 EU-的初创公司，然后由 T2 的阿里巴巴和 T4 的数据专家通过博客证实。

Data Artisans 于 2014 年由领导开发 Apache Flink(一种开源的大规模数据处理技术)的团队成立。这家初创公司提供自己的 dA 平台，包括开源的 Apache Flink 和应用程序管理器，面向企业客户，包括网飞、荷兰国际集团、优步和阿里巴巴本身。

两家公司在声明中表示，这家中国电子商务巨头自 2016 年以来一直与数据工匠合作，通过支持和开源工作来帮助软件的架构和性能。据记录，Data Artisans 在两轮融资中筹集了 650 万美元，最近一轮是 2016 年由英特尔投资公司牵头的 A 轮融资，但根据 Data Artisans 联合创始人 Kostas Tzoumas 和 Stephan Ewen 的博客文章显示，去年有一轮似乎未宣布的 B 轮融资已经结束，看起来阿里巴巴也参与其中。

现在，阿里巴巴的所有权——你也可以假设，资源——可以帮助该企业通过其开源技术达到“新的地平线”，包括“扩展到我们过去没有探索的新领域，并确保 Flink 成为现代数据驱动的实时企业的更有价值的数据处理框架，”两人写道。

“共同前进，数据工匠和阿里巴巴不仅会继续，而且会加速对 Apache Flink 和开源大数据的贡献，”Tzoumas 和 Ewen 补充道，并解释说阿里巴巴是 Flink 的最大用户和社区贡献者之一。

为了标志这个新时代，阿里巴巴承诺向社区提供其自己开发的 Flink 它称之为 Blink 的内部开发。

阿里巴巴集团副总裁周在一份声明中表示:“通过利用双方团队的技术专长和共同的热情来开发开源社区，我们相信这一战略合作将进一步加强 Flink 社区的发展，加速数据处理技术，并帮助为热衷于流处理和为现代企业提供实时应用的全球开发者营造一个开放、协作和建设性的环境。”。

这笔交易让人想起阿里巴巴 2017 年对 MariaDB 的投资，MariaDB 是一家开源初创公司，以提供数据库管理系统 MySQL 最受欢迎的替代产品而闻名。虽然不是完全收购，但这种合作关系见证了两家公司在社区的新产品上的合作，这也是我们的目标。

“特别是在许多开源技术和公司决定采用更少合作、更“封闭”的方法的时候，我们很高兴看到阿里巴巴致力于开源和我们的使命，渴望将 Flink 的技术进步提升到一个新的水平，”Tzoumas 和 Ewen 在公告博客帖子中写道。

进军开源和基础设施技术对阿里巴巴来说是有意义的，阿里巴巴最知名的是电子商务，但也经营云业务、流媒体服务等。鉴于上个季度的营业收入为 124 亿美元，净利润为 26.6 亿美元，这家中国公司当然有足够的资金来实施这一战略。

阿里巴巴副总裁周回答了 TechCrunch 的两个问题。

当被问及阿里巴巴为何寻求收购，而不是继续合作或投资时，他表示:

我们相信我们的联合能力能够为 Flink 做出更大的贡献，无论是在技术方面还是在社区发展方面。正是共同的愿景和有效的工作关系将这两个团队带到了一起。我们感谢相互学习，并努力使数据处理开源社区更加开放、协作和多样化。

我们还问阿里巴巴将如何回应人们对世界上最大的科技公司之一将如何使用和尊重开源社区的担忧，更不用说总部设在中国的企业了——对此，周回答道:

自 2016 年以来，我们一直是数据工匠的强大合作者，我们一直分享着同样的热情，使用开源流处理框架来统一实时事件驱动的应用程序和实时分析。数年来，数据工匠和阿里巴巴一直以绝对尊重 Apache 软件基金会的精神参与 Apache Flink 社区，我们打算继续这样做。

从技术的角度来看，这对于全球的开发人员来说是真正令人兴奋的事情，他们正在努力为每个人开发一个简单的流处理。对于我们的客户和合作伙伴来说，我们与数据工匠的合作为许多企业在数字化转型时代面临的挑战提供了一个有效的选择，在这个时代，实时应用和分析经常在董事会议上讨论，但没有很多供应商支持如何有效地以可扩展的方式进行。
