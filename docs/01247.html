1。 IBM 推出首款商用量子计算机

20 量子位系统将量子和经典计算部分结合在一起，将这样的机器用于研究和商业应用。虽然值得强调的是，20 量子位的机器远远不够强大，无法用于大多数商业应用，但 IBM 认为这是解决经典系统过于复杂的问题的第一步。

②。 苹果的万亿市值始终是一个虚假的偶像

没有什么东西会永远生长，即使是苹果。早在 8 月份，我们在全球头条大肆宣扬苹果作为世界上第一家价值 1 万亿美元的公司的短暂任期，但最终这并不重要。快进四个月，苹果公司的股票价值已经损失了超过三分之一，上周该公司一天之内市值损失了 750 亿美元。

3。 GitHub 免费用户现在可以获得无限的私有库

从今天开始，免费的 GitHub 用户将获得最多三个合作者的无限私人项目。之前，GitHub 对其免费用户有一个警告，如果他们不为服务付费，代码必须公开。

4。 【优步】的 IPO 可能没有我们预期的那么让人大跌眼镜

优步今年晚些时候的首次公开募股无疑是 2019 年最受期待的 IPO，但该公司的高估值(有人估计高达 1200 亿美元)让一些投资者感到不安。

5。 亚马逊对车载 Alexa 与 Telenav 的交易越来越认真

亚马逊宣布与总部位于圣克拉拉的联网汽车服务提供商 Telenav 建立新的合作关系。此次合作将在扩大亚马逊向司机提供相关信息的能力方面发挥巨大作用，并进一步推动该公司将 Alexa 融入生活方方面面的使命。

⑥。 我在一辆时速 90 英里的车上用了 VR，没有生病

车载娱乐的未来可能是虚拟现实。奥迪在 CES 上宣布，它将推出一家名为 Holoride 的新公司，为汽车带来自适应 VR 娱乐。这里的秘密酱是将 VR 内容与车辆的轻微运动相匹配，以帮助那些经常晕车的人。

7。 【威瑞森】和 T-Mobile 在& T 因假 5G 标签

没有什么比消费电子展上的戏剧更能开始你的一天了。美国电话电报公司最近分享了一个阴暗的营销活动，将其 4G 网络标为 5G，而竞争对手威瑞森和 T-Mobile 却不这么认为。
