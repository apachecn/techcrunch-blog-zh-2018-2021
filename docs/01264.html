IBM发布其首款商用量子计算机——第675页——TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/01/08/ibm-unveils-its-first-commercial-quantum-computer/675/

在CES上， IBM 今天宣布其第一台用于实验室外的商用量子计算机。20量子位系统将量子和经典计算部分结合成一个包，用于研究和商业应用的机器。当然，这个名为IBM Q system的包仍然很大，但它包含了一家公司开始量子计算实验所需的一切，包括冷却量子计算硬件所需的所有机器。

虽然IBM将其描述为第一个为科学和商业用途设计的完全集成的通用量子计算系统，但值得强调的是，20量子位的机器远远不够强大，无法满足人们对具有更多量子位的量子计算机的大多数商业应用的想象——量子位的使用时间超过100微秒。难怪IBM强调这是第一次尝试，这些系统“旨在有朝一日解决目前被视为经典系统无法处理的过于复杂和指数化的问题。”现在，我们还没有完全实现，但是公司也注意到这些系统是可升级的(并且易于维护)。

“IBM Q System One是量子计算商业化的一大进步，”混合云高级副总裁兼IBM研究总监Arvind Krishna表示。“这个新系统对于将量子计算扩展到研究实验室之外至关重要，因为我们正在努力为商业和科学开发实际的量子应用。”

然而，最重要的是，IBM似乎对Q系统的设计感到自豪。IBM与设计工作室地图项目办公室和环球设计工作室以及Goppion合作，该公司建造了包括英国皇冠珠宝和蒙娜丽莎在内的陈列柜。IBM显然认为Q系统是一件艺术品，事实上，最终的结果是相当惊人的。这是一个9英尺高、9英尺宽的密封盒子，量子计算吊灯挂在中间，所有部件都整齐地隐藏起来。

不过，如果你想给自己买一台量子计算机，你必须与IBM合作。亚马逊上的两天免费送货服务不会很快推出。

在相关新闻中，IBM还宣布了 IBM Q Network ，这是一项与埃克森美孚公司和欧洲核子研究中心及费米实验室等研究实验室的合作，旨在建立一个社区，将商业和研究兴趣聚集在一起，探索量子计算的用例。与IBM合作的组织将获得其量子软件和基于云的量子计算系统。
