一个简单的错误很容易欺骗谷歌搜索结果传播错误信息

原文：https://web.archive.org/web/https://techcrunch.com/2019/01/09/a-simple-bug-makes-it-easy-to-spoof-google-search-results-into-spreading-misinformation/amp/

图片来源: TechCrunch

任何人都可以很容易地利用谷歌的一个漏洞，很容易地踢出看起来完全真实的被操纵的搜索结果。

伦敦的安全专家 Wietze Beukema 记录了搜索操纵漏洞，他警告说，恶意用户可能会利用这个漏洞产生错误信息。

这是通过拼接来自谷歌搜索结果的“知识图”的值来完成的，知识图是在搜索结果中弹出的卡片，以视觉效果和快速事实来补充搜索查询。任何来自国家、星球、科技新闻网站等的信息都有卡片出现在谷歌搜索结果的右侧，显示其他有价值的信息。

在博客文章中，Beukema 解释说，输入到谷歌搜索结果中的简短、可共享的 URL 可以被剪切并添加到任何其他搜索查询的网址中。

因此，当你搜索“英国的首都是哪里”时，你会期望伦敦返回。实际上，你可以让它变成任何值——比如火星。

如果你搜索“谁是美国总统？”也是有效的你可以通过操作结果来读取“Snoop Dogg”T9】

一个 bug 使得将知识卡的内容放入搜索结果变得很容易。(图片:TechCrunch)被操纵的搜索查询不会破坏 HTTPS，所以任何人都可以制作一个链接，用电子邮件发送，在推特上发布或在脸书上分享——而收件人，人们认为，不会知道。但是，在一个民族国家行为者发起错误信息运动后对互联网公司不信任的时代，这可能是一个真正的问题。

Beukema 警告说，这个搜索操纵漏洞可能被用来传播事实上不正确的信息，甚至是宣传。

“谁该为 911 负责？”可以指向乔治·布什，一个被广泛持有的阴谋论。“巴拉克·欧巴马出生在哪里？”可以指向肯尼亚，另一个阴谋论主要由他的继任者唐纳德·特朗普传播，他后来收回了这一说法。

甚至“我该投哪个党的票？”可以指向共和党或者民主党。

难怪这么多人认为选举被操纵，如果他们认为他们可以点击一个按钮，让搜索引擎告诉他们投票给谁。

Beukema 告诉 TechCrunch，任何人都可以“生成看起来正常的谷歌网址，做出有争议的断言，”这可能“在谷歌上看起来很糟糕，或者更糟的是，人们会认为它们是真的。”

他说，他在 2017 年 12 月首次向谷歌报告了该漏洞，但该报告在该公司没有采取任何行动的情况下被关闭。

“我描述的‘攻击’依赖于人们对谷歌的信任和它所呈现的事实，”他说。

在撰写本文时，该 bug 仍处于活动状态。其实知道快三年了。在一年多前首次发现这个问题后，Beukema 只是将这个问题公之于众。但是它已经引起了黑客社区的兴趣。一名开发人员 Lucas Miller 只花了几个小时就编写了一个 Python 脚本来自动生成基于搜索查询的虚假结果。

尽管有人声称存在政治偏见(尽管没有证据表明这是真的)，但谷歌为何花了这么长时间来修复其搜索结果中的一个基本弱点，从而使服务更值得信赖，这是一个谜。

谷歌发言人告诉 TechCrunch，它正在“努力解决”这个问题。
