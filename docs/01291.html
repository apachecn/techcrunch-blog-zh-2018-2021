动手操作 Ledger 的蓝牙加密硬件钱包 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/01/09/hands-on-with-ledgers-bluetooth-crypto-hardware-wallet/

法国初创公司 Ledger 本周在 CES 上发布了一款新的硬件钱包。虽然这款设备要到 3 月份才会上市，但该公司让我试用了这款设备的原型版本。莱杰纳米 X 感觉就像使用纳米 S，但在移动。

当该公司之前的硬件钱包首次问世时，那是在加密货币繁荣之前，在莱杰筹集到7500 万美元之前。用户体验也不好。

你必须安装多个 Chrome 应用程序来管理多种加密货币，当你想访问你的余额和管理你的加密资产时，在每个应用程序之间切换。但是当该公司在 macOS、Windows 和 Linux 上发布 Ledger Live 时，事情变得好了很多。

有了这个新的应用程序，您最终可以从同一个桌面应用程序查看您的投资组合余额并管理多个加密资产。合乎逻辑的下一步是移动。你必须为此买一个新的硬件钱包。

莱杰纳米 X 看起来或多或少像莱杰纳米 S，但略大。它的形状像一个 USB 钥匙，有一个小屏幕来确认设备上的交易。它有一个 100 毫安的小电池和一个稍大的屏幕。当你不使用钱包时，电池应该可以持续几个月，并且在积极使用时大约可以持续 8 小时。microUSB 端口已被 USB-C 端口取代。按钮现在位于屏幕的每一侧，而不是设备的侧面。

将设备与手机配对后，您可以从 iOS 或 Android 手机控制一切。你可以在 Ledger Nano X 上安装应用程序，访问你的钱包并发送加密货币。在 iOS 上，你可以使用密码锁定应用程序，也可以选择 Face ID 或触控 ID。

当您需要验证您的 Ledger Nano X 上的交易时，您的手机将通过蓝牙与您的 Ledger 设备配对。然后，您可以在分类帐设备上查看交易信息，并在设备本身上批准交易。

Ledger 之所以如此安全，是因为您的私钥永远不会离开您的 Ledger 设备。交易直接在设备上签名。您的私钥永远不会通过蓝牙发送，即使您的智能手机受到威胁，您的加密货币也会保持安全。

Ledger 现在使用 ST33 安全元素，比以前版本的 ST31 稍微安全一些。现在，只有一个芯片，直接连接到屏幕和按钮上，这降低了有人破坏你屏幕上信息的风险。

屏幕现在增加了一倍，这样您就可以查看完整的公共地址，而无需滚动视图。你现在可以安装多达 100 种不同的加密货币应用。您仍然可以将设备插入电脑，并使用桌面应用程序。该设备售价 120 美元/€120。
