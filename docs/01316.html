不可思议和史诗游戏建立了 2500 万美元的基金，帮助开发者在 Unity 崩溃后转向“更开放的引擎”

原文：https://web.archive.org/web/https://techcrunch.com/2019/01/10/improbable-and-epic-games-establish-25m-fund-to-help-devs-move-to-more-open-engines-after-unity-debacle/amp/

图片来源:安东尼·哈

不可思议在今天早些时候宣布 Unity 已经撤销了其在流行游戏开发引擎上的运营许可后，采取了一个大胆的举措。

这家总部位于英国的云游戏初创公司与 Unity 的竞争对手 Epic Games 签署了一份深夜新闻稿，该公司运营虚幻引擎，是《堡垒之夜》的创作者，建立了一个 2500 万美元的基金，旨在帮助游戏开发商转向“更开放的引擎”

Epic Games 首席执行官蒂姆·斯维尼和“不可思议”首席执行官赫曼·纳鲁拉撰写了一篇博文，内容如下:

为了帮助那些被今天推出的新引擎和服务不兼容问题搁置一旁的开发者，Epic Games 和 implementable 共同建立了一个 25，000，000 美元的联合基金，帮助开发者过渡到更开放的引擎、服务和生态系统。这笔资金将来自各种来源，包括虚幻的开发赠款，不大可能的开发商援助基金，和史诗游戏商店的资金。

这对于 implevable 来说是相当大胆的，似乎表明在 implevable 发表了一篇署名“你们[Unity]是一家非常重要的公司，一个糟糕的日子并不能带走你们给我们的一切”的博文后，Unity 没有给他们打电话。让我们为我们的社区解决这个问题，你知道我们的号码。”

Unity 方面声称，他们给了不太可能的充分通知，违反了他们的服务条款，并且两人已经深深陷入了一个“合作伙伴”协议中，这显然是不够的。不可能的 Unity 许可的终止基本上切断了他们与一大部分基于 Unity 开发的独立开发者的联系。

Epic Games 首席执行官蒂姆·斯维尼很快在今天早些时候报道了这一消息，对 Unity 的行为进行了指责。

“Epic Games 与 implevable 的合作伙伴关系，以及 implevable 基于云的开发平台 SpatialOS 的整合，是基于共同的价值观，以及对公司应该如何合作以直接、毫不意外的方式支持共同客户的共同信念，”博客帖子写道。

从某种程度上来说，这对《不可能》来说是一个积极的发展，这表明 Epic Games 致力于坚持创业，但与此同时，人们不禁要问，根据今天的公开言论，Unity 和《不可能》的关系是如何如此迅速地恶化的。
