原文：https://web.archive.org/web/https://techcrunch.com/2019/01/11/202-million-job-seekers-personal-data-exposed/

一个不安全的数据库暴露了中国 2.02 亿求职者的个人信息

由于数据库不安全，中国超过 2.02 亿求职者的个人信息，包括电话号码、电子邮件地址、驾照和期望工资等信息，被任何知道去哪里寻找的人免费获取长达三年。

这是根据安全研究员 Bob Diachenko 发表的发现，他在 12 月下旬找到了一个开放且未受保护的 MongoDB 实例，其中包含 202，730，434 条“非常详细”的记录。该数据库在数据搜索引擎 Binary Edge 和 Shodan 中被编入索引，无需密码或登录即可自由查看。直到迪亚琴科在推特上发布了关于它存在的信息后，它才被保密。

迪亚琴科是 Hacken 的网络风险研究主管，他无法将数据库与特定的服务相匹配，但他找到了一个三年前的 GitHub 应用程序库，其中包括“与曝光简历中使用的相同的结构模式。”同样，所有权在这一点上并不明确，尽管这些记录似乎确实包含了从中国分类广告中搜集的数据，包括类似 Craigslist 的 58.com。

58.com 发言人否认这些记录是其创造的。相反，他们声称自己的服务受到了第三方的损害。

“我们已经搜遍了我们的数据库，并调查了所有其他存储，结果证明样本数据不是从我们这里泄露的。一名发言人告诉迪亚琴科:“这些数据似乎是从许多简历网站上搜集数据的第三方泄露的。”。

TechCrunch 联系了 58.com，但我们尚未收到回复。

虽然数据库现在已经得到保护，但在长达三年的时间里，它仍有潜在的脆弱性，而且已经有证据表明它被定期访问。不过，还是不清楚是谁干的。

“值得注意的是，MongoDB 日志显示，至少有十几个 IP 可能在数据离线前访问过这些数据，”迪亚琴科写道。

这里有很多谜团——不清楚 58.com 是不是这个漏洞的幕后黑手，或者它是一个竞争对手的服务还是一个刮刀——但更确定的是，这个漏洞是中国发现的同类漏洞中最大的一个。
