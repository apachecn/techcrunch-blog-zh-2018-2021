特斯拉将 Model 3 带入 Pwn2Own，这是世界上最艰难的黑客竞赛之一

原文：https://web.archive.org/web/https://techcrunch.com/2019/01/14/tesla-is-entering-the-model-3-into-pwn2own-one-of-the-worlds-toughest-hacking-contests/

特斯拉今年将向 Pwn2Own 移交其新的 Model 3 轿车，这是汽车首次被列入一年一度的高调黑客大赛。

获奖安全研究员的奖品:一台 Model 3。

Pwn2Own 是由趋势科技的 Zero Day Initiative 举办的第 12 届比赛，是业内最激烈的黑客竞赛之一。在该项目的整个生命周期中，ZDI 已经拨款 400 多万美元。

Pwn2Own 的春季漏洞研究大赛 Pwn2Own Vancouver 将于 3 月 20 日至 22 日举行，将分为五个类别，包括网络浏览器、虚拟化软件、企业应用程序、服务器端软件和新的汽车类别。ZDI 选择的目标包括苹果、谷歌、微软、Mozilla、甲骨文和 VMware 的软件产品。当然还有特斯拉。Pwn2Own 与 CanSec West conference 同时运行。

自 2014 年该公司推出第一个 bug bounty 计划以来，特斯拉一直与黑客社区保持公开关系。从那以后，它一直在成长和进化。

去年，该公司将最高奖金从 10，000 美元提高到 15，000 美元，并增加了其能源产品。今天，特斯拉的车辆和所有直接托管的服务器、服务和应用程序现在都在它的赏金计划范围内。

该公司去年还对其 bug bounty 计划进行了重大改革，通过允许车主在遵守规则的情况下黑掉自己的车，来支持【安全港】。特斯拉的产品安全政策现在表示，如果通过“善意的安全研究”，你用砖头砸了你的车，该公司将通过无线方式或在服务中心刷新软件。该公司表示，如果他们破解了软件，他们也不会取消汽车的保修。

特斯拉(现在还有其他汽车制造商)推出 bug 赏金计划是有原因的。特斯拉汽车以软件为中心，通过启用无线软件更新，可以修复故障和安全问题，提高性能并添加其他新功能，在许多方面改变了行业。这使得特斯拉能够赢得消费者的支持，让他们相信他们的汽车会随着时间的推移变得越来越好。

但随之而来的可能是安全问题。自 2014 年以来，该计划已导致特斯拉发布了多项 安全改进，包括其软件的加密验证，其密钥卡的更强大的加密技术以及旨在防止密钥卡克隆中继攻击的 PIN-to-Drive 的推出。

当然，不能保证 Pwn2Own 温哥华的黑客会找到任何漏洞。趋势科技发言人告诉 TechCrunch，成功尝试的百分比各不相同，但通常是 50%左右的可用目标。

这位发言人说，还不清楚研究人员是否会进入汽车类别，因为这是今年的新类别，她补充说，她希望人们进入“因为我们很想看看汽车研究的艺术水平到底是什么。”
