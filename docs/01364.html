Box 聘用前 SAP 高管担任首席信息安全官

Box 今天宣布，已经聘请拉克什米·汉斯帕尔(Lakshmi Hanspal)担任公司新任首席信息安全官(CISO)。她拥有 20 年的安全经验，包括在 SAP Ariba 和美国银行担任执行安全职务。她还在 PayPal 担任过高级职位。

在宣布招聘的博客文章中，该公司这样定义她的角色:“在 CISO 的角色中，拉克什米将负责 Box 的网络安全实践、安全运营以及数据和平台保护。”

Hanspal 看到了 Box 与她在 SAP Ariba 时的相似之处，但她认识到她将面临一系列不同的挑战。“我在 Box 的角色与我在 SAP Ariba 的角色相似，最大的不同是 Box 的地理足迹。她告诉 TechCrunch:“box 是一家诞生于云公司，并在全球范围内迅速扩张，因此我的重点也将包括为我们的客户保护公共云运营(未来堆栈)和风险透明度。

她说，这将涉及通过自动化提高服务成熟度和可持续性，同时继续确保 Box 公司和产品平台的最高安全水平。

Box 首席执行官 Aaron Levie 表示，安全性是 Box 所做的一切的核心，因此找到合适的首席信息安全官绝对至关重要。“Lakshmi 不仅带来了她在 SAP、PayPal 和美国银行的令人印象深刻的多元化领导经验，而且她是一个令人难以置信的团队建设者和文化补充，将我们的安全团队提升到一个新的水平，”Levie 说。

汉斯帕尔是 Box 执行团队中的第三位女性，加入斯蒂芬妮·卡鲁洛，她在 2017 年被聘为首席运营官，并在克里斯蒂·莱克担任首席人力官。
