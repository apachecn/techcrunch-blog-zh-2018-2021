Fiserv 以 220 亿美元的金融科技大交易 TechCrunch 购买首批数据

原文：https://web.archive.org/web/https://techcrunch.com/2019/01/16/fiserv-is-buying-first-data-in-a-22b-fintech-megadeal/

虽然我们将继续看到金融科技(fintech)领域的小型创业公司之间的大量整合，但也有一些更大的组合在发挥作用，以帮助利用规模经济来应对当前和未来的竞争。今天， Fiserv 宣布将以价值 220 亿美元的交易收购First Data——分别是金融服务和电子商务支付领域的巨头。

这是一次合并，但 Fiserv 将在交易中占据上风:其首席执行官 Jeffery Yabuki 将成为合并后实体的首席执行官，而 First Data 的首席执行官 Frank Bisignano 将成为总裁兼首席运营官。

这将是一次全股票交易。具体来说，第一数据股东将获得每股第一数据普通股 0.303 股 Fiserv 股票的固定交换比例，“股权价值为 220 亿美元。”Fiserv 表示，其支付的价格比昨日收盘时的五天成交量加权平均价格高出 29%。交易完成后，Fiserv 股东将持有合并后公司 57.5%的股份，而 First Data 股东将持有 42.5%的股份。

这一合并突显了整合金融服务生态系统不同部分的大趋势，为客户提供一站式服务，并从整体上构建更加一体化的服务。

传统银行及其向用户提供的服务——从储蓄账户到信贷和贷款服务，再到汇款和转账服务和支付——在过去 10-15 年中被打乱了，因为出现了一大批初创公司，它们采用基于云架构、应用程序、引人注目的营销、人工智能和机器学习的更快、更敏捷的解决方案来提高响应能力和整体用户体验，并降低银行提供的一些利率。

后起之秀包括 PayPal(你可能会说从这个意义上来说它不再具有破坏性)、Stripe、Square、TransferWise、Ant Financial(某种形式的友敌关系)等等。亚马逊(Amazon)和苹果(Apple)等其他科技公司也在“拥有”客户的财务支出方面发挥作用。

Fiserv 现在已经 35 岁了，它实际上在帮助银行应对这一更大趋势的努力中发挥了作用，例如，它曾经开发了自己的基于智能手机的读卡器来与 Square 的竞争，它打算将 Square 的出售给银行，以应对较小的公司。

与此同时，First Data 主要在自己的车道上游泳，充当有趣的金融科技初创公司的整合者，如 Clover 、 Perka 、 Gyft 、 Blue Pay 、 Spree 等。它于 2015 年上市，并表示其技术活跃于 100 多个国家的 600 万家实体企业和 4000 家金融机构，每秒处理约 3000 笔交易，每年处理 2.4 万亿美元。

现在，他们将把各自的工作整合成更加完整的产品。作为公司给出的例子，它将合并 First Data 的数字商户账户注册功能可以集成到 Fiserv 的数字银行解决方案中，这些解决方案为成千上万的金融机构提供服务。

他们表示，未来五年还将投资 5 亿美元用于商业解决方案、数字服务、风险管理和支付等领域的新技术。

Fiserv 总裁兼首席执行官 Jeffery Yabuki 在一份声明中说:“通过这种变革性的结合，我们希望重新定义人们和机构移动资金和信息的方式。”“我们钦佩 First Data 在商户收单和全球发行服务方面的卓越表现，以及他们在 Frank 的领导下取得的巨大进步。我们希望这种结合能够促进和支持我们的集体客户及其顾客的价值主张。”

First Data 董事长兼首席执行官 Frank Bisignano 在一份声明中表示:“我一直很钦佩 Fiserv 多年来取得的成就，我期待着与两家公司的才华横溢的同事合作，为行业设定更高的创新和服务标准。”。“我们在 First Data 的目标一直是为我们的客户提供最全面的创新、高度差异化的解决方案和服务，与 Fiserv 的结合为所有利益相关者创造了巨大的价值，我对此感到非常兴奋。”

虽然这是为了创造产品，更好地与金融服务领域的其他公司竞争，但也是为了节省两家公司的资金。Fiserv 和 First Data 表示，他们预计运营成本将节省 9 亿美元，收入协同效应将达到 5 亿美元或更多。

这些公司表示，他们预计这笔交易将在 2019 年下半年完成。
