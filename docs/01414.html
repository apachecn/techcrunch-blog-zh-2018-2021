谷歌的 Pixel 3 Lite 可能会带回耳机插孔

圣诞节后，关于 Pixel 家族下一个成员的消息开始泄露。现在传闻中的 Pixel 3 Lite 正获得更多的时间来展示自己，这要归功于一个三分钟的 YouTube 视频，该视频强调了谷歌旗舰硬件系列的预算增加。

也许这里最有趣的是耳机插孔的明显回归(除了第三个 Pixel 3 型号的存在)。在对第一个像素上的端口大吵大闹后，该公司很快改变了方向，转向其继任者。

https://www.youtube.com/watch?v=jqnSkALsdkk

然而，增加一部中档手机是恢复该端口的理想理由(可能会持续一段时间)。毕竟，虽然蓝牙耳机近年来变得越来越普及，但对于希望节省几(或几百)美元的人来说，专业耳机仍然是一个很大的要求。

有一些削减成本的措施，包括骁龙 670，塑料机身和没有第二个自拍相机。总的来说，这款设备有点像谷歌的 iPhone XR，但值得注意的是，它似乎与更昂贵的同类产品具有大致相同的后置摄像头配置。这很可能是因为人工智能——而不是硬件——正在为新手机做大部分繁重的图像提升工作。

值得注意的是，Pixel 设备的价格通常已经低于苹果和三星的旗舰产品，但新的产品可能是谷歌展示 Android 如何在低成本设备上发光发热的好机会。
