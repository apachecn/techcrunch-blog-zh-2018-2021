Wynd 为其商店管理服务筹集了 8200 万美元

法国初创公司 Wynd 从 Natixis、Sofina 和 BNF Capital 又筹集了 8200 万美元(€7200 万美元)。该公司开始为餐馆和其他实体商店提供销售点解决方案。现在，在管理您的线下和线上销售时，它为您的所有数字需求提供一站式服务。

这家初创公司已经从今天的新投资者以及索迪斯、奥兰治和阿尔文那里筹集了总计 1.27 亿美元(€1.12 亿美元)。

Wynd 为一切可以由计算机驱动的东西提供了一个软件即服务平台。该服务管理你的库存，处理订单和付款，并告诉你的员工他们应该做些什么来为你的客户准备订单。

一切都是全渠道的，这意味着在线销售和线下销售在系统中以相同的方式处理，只是在交付方面有不同的参数。您的库存在您的电子商务网站和商店中是统一的。而且 Wynd 还可以代替你的产品信息管理服务。

如果你已经在为你的业务的某些部分使用其他服务，Wynd 有一个 API 并与第三方服务集成。例如，你可以把 Wynd 和你的 ERP 连接起来。

Wynd 还能让你获得关于你的产品和员工的详细报告。在这方面，Wynd 与 Excel 和优秀的老静态出口竞争。例如，拥有动态仪表板可以帮助你更积极地反应，并理解某个特定产品为什么会畅销。

而现在，大品牌都在用 Wynd 来管理他们的销售，比如家乐福、道达尔、MK2 和 Monceau Fleurs 该公司 30%的收入来自其他国家。
