微软移动版 Edge 现在包括一个内置的假新闻检测器 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/01/23/microsoft-edge-newsguard/

2019 年，我们还真不知道该拿假新闻怎么办。由于没有什么可以阻止病毒式的超级标题和其他证实偏见的做法，网上的错误信息似乎和以前一样猖獗。这是一个棘手的问题，特别是因为它需要读者去质疑他们的信仰，因为这太离谱了。换句话说，如果没有某种技术解决方案或大规模的文化转变，假新闻的困境不会很快自行解决。

也就是说，微软的移动边缘浏览器正在对其进行适度的调整。在 Android 和 iOS 上，微软 Edge 应用程序现在安装了一个内置的假新闻检测器，名为 NewsGuard 。这次合作是微软保卫民主计划的延伸，本月早些时候首次宣布了 Edge 的 NewsGuard。

虽然 NewsGuard 在默认情况下是不打开的，但任何使用 Edge 的人都可以通过设置菜单中的简单切换来启用它。当我下载该应用程序进行测试时，Edge 实际上把我推到了设置菜单，然后是一个带小蓝点的名为新闻评级(这可以启用 NewsGuard)的选项。这个点不是一个红色警报通知，但可能足以引起我的兴趣，并为我指明背景，即使我没有写这个故事。

目前，NewsGuard 的评级集中在美国网站上，但国外的主要网站也包括在内。TechCrunch 在 NewsGuard 上获得了一张健康的绿色支票，表明我们坚持“准确性和问责制的基本标准”。点击地址栏旁边的绿色徽章，会出现一个选项来查看 TechCrunch 的完整“营养标签”——相关信息的概要，如我们的所有权和融资、内容和可信度。这些信息非常微妙，一直到“观点并不总是被清楚地标注出来”，这很公平。它甚至包括一个纠正故事的例子，以及我们如何处理它。正如《卫报》指出的那样,《每日邮报》的日子并不好过。

影响 NewsGuard 评级的社论深度挖掘令人印象深刻，尽管它们确实例证了另一个让打击假新闻变得特别棘手的问题。即使新闻来源经过一系列因素的评估，做出这些决定仍然需要一定程度的主观评估。虽然有很多实体可以做出这些决定，但是我们如何就谁应该做这件事达成共识呢？

NewsGuard 由《华尔街日报》的前出版商戈登·克罗维兹和斯蒂芬·比尔共同领导。像其他具有编辑意识的新闻实验一样，NewsGuard 依赖于人类团队而不是算法。该公司的顾问包括前中情局局长迈克尔·海登和信息创始人杰西卡·莱辛。

Edge 并不是一个非常受欢迎的浏览器，但它仍然是针对低质量在线信息的棘手战争中一个有趣的案例研究。这也说明了假新闻时代的第 22 个主要问题:最需要假新闻检测器的用户最不可能使用它。微软对 NewsGuard 的 Edge 实验并不能解决这个问题，但是在浏览器中加入某种新闻验证工具确实让人觉得是朝着一个令人信服的方向迈出了一步。
