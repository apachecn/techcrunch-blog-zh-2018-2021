假设电视现在可以提醒你电视套餐的变化，这样你就能买到最划算的东西

原文：https://web.archive.org/web/https://techcrunch.com/2019/01/23/suppose-tv-can-now-alert-you-to-changes-in-tv-packages-so-you-get-the-best-deal/

去年，一家名为的初创公司假设电视进入市场，以帮助消费者找到电视服务的最佳交易。该网站提供了一个在线工具，可以让你比较不同的服务，包括频道选择和定价。但是这样做仍然需要在输入标准、设置渠道优先级和指定其他要求方面做很多工作。今天，该公司推出了一个新的自动化功能，使整个过程更加简单。通过“电视服务提醒”，这家初创公司现在可以在电视服务改变价格、频道阵容或功能时向你发送电子邮件。这样，你可以做出一个节省成本的转变，或者跳到一个更适合你的转变。

随着流媒体领域变得越来越分散，这种东西可能会成为那些被各种选择淹没、没有时间跟上所有变化的人的必备工具。

今天的流媒体服务正在不断调整他们的产品——编辑他们的捆绑包，以及提高和降低价格，因为他们试图找出最有效的方式。

例如，去年 Sling TV 、 YouTube TV 、 DirecTV Now 和 PlayStation Vue 都提高了直播电视的价格，网飞也提高了其订阅视频服务的价格。网飞也在本月再次提价，而 Hulu 降低了付费视频的价格，但提高了直播电视的价格。

与此同时，这些方案也在不断变化。例如，Sling TV 增加了一个新的免费层和零点频道订阅；Hulu 放弃了一些频道，而则将它们与新增频道一起放入可选的附加捆绑包。而且大多数公司会随着时间的推移，不断将渠道添加到其产品包和附加选项中。

谁能跟上这一切？

假设电视能帮上忙。这个网站可能不漂亮，但很方便。在这里，您可以通过设置频道选择和功能首选项来自定义您的首选阵容。随后，随着各种流媒体服务包的变化，它现在可以通过电子邮件向您发送个性化提醒，告诉您是否能获得更好的交易。

假设 TV 说，这项服务已经处于测试阶段几个月了，在今天推出之前大约有 1000 名测试人员。

这家公司的网站还可以帮助你找到当地广播和地区性体育网络可以为你提供的服务——具体情况因市场而异。(目前在美国最大的市场和部分较小的市场都可以买到。)现在，它还能为你提供流媒体服务的促销折扣和优惠。

这种工具今年将变得更加有用，因为新的服务开始到来，如迪士尼+、康卡斯特的 NBCU 流媒体服务以及即将到来的 AT & T/WarnerMedia 提供的等等。

对于消费者来说，假设电视的工具可以帮助他们做出更好的决定，但初创公司的业务却在别处。

该公司今天还推出了一个 API，提供对其直播电视数据库和推荐引擎的访问，这将允许合作伙伴帮助他们的客户找到流媒体服务的正确组合。

“通过我们的 API 解决方案，假设支持宽带提供商、电视操作系统平台或零售商等合作伙伴公司提供电视服务选择工具，”假设电视联合创始人 Andrew Shapiro 在关于 API 发布的声明中表示。"这些公司有能力帮助他们的客户注册和管理他们的电视服务。"
