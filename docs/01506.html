谷歌与索尼影业合作推出开源的VFX渲染管理器

谷歌今天宣布，它已经与索尼视觉效果和动画工作室索尼电影图像工作室合作，推出 OpenCue ，一个开源渲染管理器。OpenCue不处理任何实际的渲染过程，但它提供了所有的工具来分解这些不同的步骤，然后调度和管理大型渲染场(本地和云中)的不同渲染作业。

当然，谷歌有兴趣将这些工作负载带到它的云平台上，它已经做出了协调一致的努力将好莱坞电影公司带到它的云平台上。这包括去年推出的洛杉矶云区域的T8，以及2014年收购的云渲染器的T10。谷歌也是学院软件基金会(Academy Software Foundation)的创始成员之一，这是一个开源基金会，专门研究电影行业的工具。索尼电影娱乐公司/图像公司不是创始成员，但在基金会成立几个月后加入了。

Cue 3实际上是Imageworks的内部排队系统，它至少有15年的历史了。谷歌与该公司合作开源该系统，谷歌和Imageworks已经将该系统扩展到15万个内核。

“随着内容制作在全球范围内继续加速，视觉特效工作室越来越多地转向云，以满足对高质量内容的需求，”谷歌产品经理托德·普里维斯在今天的公告中写道。“虽然内部渲染场仍在大量使用，但云提供的可扩展性和安全性为工作室提供了适应当今快节奏的全球制作日程所需的工具。”

值得注意的是，这不是索尼第一次涉足开源领域。该公司之前也开源并贡献了像 OpenColorIO 和 Alembic 这样的工具。
