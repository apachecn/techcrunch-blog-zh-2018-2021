Dropbox 以 2.3 亿美元收购 HelloSign，获得工作流程和电子签名 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/01/28/dropbox-snares-hellosign-for-230m-gets-workflow-and-esignature/

Dropbox 今天宣布打算收购 HelloSign ，这是一家提供轻量级文档工作流和电子签名服务的公司。该公司为此特权支付了 2.3 亿美元。

Dropbox 的工程 SVP Quentin Clark 认为这不仅仅是在 Dropbox 解决方案中增加电子签名功能。对他来说，【HelloSign 在 2017 年增加的工作流功能是这次购买的关键。

“W 帽子 是 独有关于 HelloSign 是 那个 投资 他们已经 制造了API 和工作流产品 是 真的 所以 对齐 “这不仅仅是用 Dropbox 多做一项活动，它真的会帮助我们追求更广阔的愿景，”他补充道。这一愿景包括扩展存储功能，这是 Dropbox 解决方案的核心。

这也可以从去年 Dropbox 增加的扩展功能的背景中看出。HelloSign 实际上是参与发布的公司之一。虽然克拉克表示，该公司将继续鼓励公司扩展 Dropbox 解决方案，但今天的收购让它拥有了自己的能力，不需要合作伙伴关系，并且已经通过扩展与 Dropbox 相连。

快速集成

Deep Analysis 的创始人兼首席分析师艾伦·佩尔茨-夏普(Alan Pelz-Sharpe)多年来一直关注这个市场，他表示，事实上，它是一个扩展合作伙伴，这应该能够比像这样的收购中通常发生的情况更快地进行整合。“与中小型企业相关的简单文档流程在很大程度上仍然是手动的。Pelz-Sharpe 解释说，HelloSign 拥有针对房地产、保险和登机客户/员工等问题的解决方案，加上对 Dropbox 的现有扩展，这意味着它可以迅速用于 Dropbox 的收入增长。

他补充说，这笔交易的规模表明，对这类能力的需求很高。“这是一个非常高的倍数，但在这样一个快速增长的领域，对一家表现出如此增长潜力的初创公司提出这样的要求并非不合理。这一价格表明，几乎可以肯定，还有其他热情高涨的竞标者在竞购这笔交易，”他表示。

HelloSign 首席执行官约瑟夫·瓦拉(Joseph Walla)表示，成为 Dropbox 的一部分，使 HelloSign 能够获得一家更大的上市公司的资源，这应该使它能够进入比单独一家公司更广阔的市场。“通过与 Dropbox 合作，我们可以为更多的客户带来更无缝的文档工作流，并大大加快我们的影响，”瓦拉在宣布这笔交易的博客帖子中说。

HelloSign 保持独立

HelloSign 的首席运营官 Whitney Bouck 曾在 Box 和 EMC Documentum 任职，他说该公司将保持独立的实体。这意味着它将作为 Dropbox 家族的一部分，继续以目前的管理结构运营。事实上，克拉克表示，作为交易的一部分，HelloSign 的所有员工都将在 Dropbox 获得就业机会。

“我们将继续有效地保持 Dropbox 家族中的独立业务，这样我们就可以继续专注于开发我们拥有的优秀产品并创造价值。因此，好消息是我们的客户不会真正体验到任何巨大的变化。他们只是得到了更多机会，”布克说。

Constellation Research 专门研究企业工作流的分析师 Alan Lepofsky 认为 HelloSign 为 Dropbox 提供了一个企业级工作流工具，但他补充说，Bouck 的加入和她在企业内容管理方面的背景也是 Dropbox 在这笔交易中的一个不错的奖励。“虽然这不是一次收购，但 Dropbox 最终还是得到了 Whitney Bouck，一位在将产品扩展到企业规模客户方面久经考验的领导者。我相信她会在 Dropbox 与前雇主 box 的斗争中发挥重要作用，”莱波夫斯基告诉 TechCrunch。

Clark 说，现在说它将如何捆绑和整合扩展之外的 HelloSign 功能还为时过早。但他预计，即使 HelloSign 作为一家拥有自己客户的独立公司运营，该公司也会找到一种方法，在有意义的地方整合这两种产品。

当你考虑到 2011 年成立的湾区初创公司 HelloSign 仅筹集了 1600 万美元时，这对投资者来说似乎是一个令人印象深刻的回报，也是该公司的一个坚实的退出。

该交易预计将在 Q1 完成，按照惯例，将取决于监管机构的批准。
