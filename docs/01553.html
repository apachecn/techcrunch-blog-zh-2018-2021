洛杉矶正在使用打车初创公司 Via 运送人们乘坐公共交通工具

洛杉矶正在与按需班车服务 Via 合作，作为试点项目的一部分，该项目将让人们乘坐公交车前往三个繁忙的公共交通站。

该试点项目于周一启动，旨在解决第一英里和最后一英里的问题，这使得人们很难往返于公共交通站。根据试点计划，Via 将提供往返于阿泰西亚、埃尔蒙特和北好莱坞站的按需乘车服务。Via 将多名乘客送往同一方向，将他们预定到一辆共享汽车。

与 Via 合作的试点项目旨在应对公共交通乘客量下降的问题，这一问题受到了优步和 Lyft 打车服务以及个人汽车拥有量上升的影响。像洛杉矶这样的城市正在寻找方法，将动态的按需功能引入公共交通，从而使打车变得流行。

“这个创新的试点项目将让乘客们对洛杉矶未来的综合交通系统有一个新的了解。许多地铁用户面临着从家到车站的挑战，反之亦然，”洛杉矶县主管和地铁董事会主席希拉·库尔在一份声明中说。“他们需要一种快速、简单、廉价的门到门解决方案，而这个新的试点项目是可以考虑的。”

Via 和 Metro 之间为期一年的试点项目价值 250 万美元，部分资金来自联邦运输管理局 135 万美元的拨款。

运载 3 至 6 名乘客的车辆。地铁低收入补贴项目的注册用户可以免费乘坐。对于 TAP 卡持有者，乘车费用为 1.75 美元一次。其他人的费用是 3.75 美元。

Via 在芝加哥、华盛顿特区和纽约运营面向消费者的班车。该公司还与城市和交通部门合作，让客户能够访问他们的平台，部署自己的班车。例如，奥斯汀的首都大都会运输管理局使用 Via 平台为该市的接送服务提供动力。德国铁路公司 Arriva Bus UK 也使用 Via 的平台，提供连接通勤者和英国肯特高速火车站的第一英里和最后一英里服务
