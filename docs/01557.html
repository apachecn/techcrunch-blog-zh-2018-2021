Nvidia 大幅降低收入指导 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/01/28/nvidia-significantly-lowers-revenue-guidance/

Nvidia 是跌不下去的。该公司已经下调了对其即将发布的收益报告的预期。该公司预计将产生 22 亿美元，而不是之前宣布的 27 亿美元。

这是 18.5%的巨大降幅。目前股价为 139.60 美元，比昨天 160.15 美元的收盘价下跌了 12.83%。

Nvidia 再次引用了导致其可怕、可怕、不好，非常糟糕的一年的相同原因。就像许多科技公司一样，英伟达也受到中国需求和关税的影响。

“然而，不断恶化的宏观经济条件，特别是在中国，影响了消费者对 NVIDIA 游戏 GPU 的需求，”该公司在新闻稿中写道。

这只是故事的一部分。Nvidia 在 2017 年取得了巨大的成功，因为该公司出售了大量的 GPU。该公司计划于 2 月 14 日发布 2019 财年第四季度的财报，预计将于 1 月底结束。

将 2019 年第四季度与 2018 年第四季度进行比较将会很困难，因为这正好对应于 2017 年底的加密货币繁荣。在挖掘加密货币时，GPU 非常有用。这导致了短缺和价格飙升。相比之下，英伟达的表现会显得很弱。

但是 Nvidia 已经知道了。该公司表示，高端 GPU 的需求低于预期。听起来新的 Nvidia GeForce RTX 产品系列的销售不如预期。

“使用英伟达新图灵架构的某些高端 GPU 的销量低于预期。这些产品通过实时光线跟踪和人工智能在性能和创新方面实现了革命性的飞跃，但一些客户可能会推迟购买，同时等待更低的价格点和 RTX 技术在实际游戏中的进一步展示，”该公司表示。

换句话说，一系列因素导致了今天的修订。让我们看看该公司在接下来的几个季度中进行预测时，是否能够回到正轨。
