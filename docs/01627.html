特斯拉报告连续盈利季度 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/01/30/tesla-reports-back-to-back-profitable-quarters/

特斯拉周三公布利润为 1.39 亿美元，或每股 0.78 美元，销售额好于预期，但未能达到分析师对第四季度收益的预期。股市收盘后股价下跌，截至太平洋时间下午 2:30，股价下跌 1.7%。

由于 Model 3 的销售，尽管第四季度出现了几次逆风，特斯拉已成功连续两个盈利期，包括非控股权益产生的 5400 万美元的非现金费用，来自中国的零部件进口关税提高，Model S 和 Model X 在中国的降价，以及 Model 3 的低价中档版本的推出。

10 月，特斯拉在连续七个季度亏损后首次盈利。这是该公司历史上第三次达到这一里程碑。

特斯拉第四季度的盈利与其去年同期的财务状况形成鲜明对比，去年同期该公司亏损 6.75 亿美元，合每股 1.75 美元。

或许与汽车制造商的收入同样重要的是它的现金状况。特斯拉报告称，尽管计划在第四季度偿还 2.3 亿美元的可转换债券，但其现金状况仍改善了 14.5 亿美元。

“我们手头有足够的现金，可以轻松兑现将于 2019 年 3 月到期的可转换债券，”致投资者的股东信中表示。

以下是一些亮点:

  • 特斯拉第四季度收入为 72 亿美元，高于第三季度的 68 亿美元
  • 特斯拉第四季度的运营现金流减去资本支出后达到 9.1 亿美元
  • 第四季度末，现金和现金等价物增加了 7.18 亿美元，达到 37 亿美元
  • 到 2019 年底，弗里蒙特的 Model 3 产量应达到每周 7，000 辆的持续水平

分析师对特斯拉第四季度收益报告的反应是谨慎的，指出了该公司的成就以及未来的挑战。

埃德蒙兹行业分析执行董事杰西卡·考德威尔(Jessica Caldwell)在一份电子邮件声明中表示，“特斯拉在美国的情况真的不会比 2018 年底好到哪里去。”“扭亏为盈、创造性地解决生产挑战以及将 Model 3 推向大众是巨大的里程碑，但保持这种势头几乎是不可能的。特斯拉的产品阵容开始变得陈旧，现在由于联邦税收抵免的取消，购买一辆汽车从未如此昂贵。虽然拥有一辆特斯拉仍然有一个很酷的因素，但这可能不足以说服买家选择一辆特斯拉，而不是奥迪、保时捷和捷豹的全新车型。特斯拉正处于一个尴尬的炼狱之中，既要成为一家初创公司，又要成为一家主流汽车制造商，2019 年面临的最大问题是该公司真正的出路在哪里。特斯拉习惯于成为焦点，但明年我们可能会看到一个更低调的特斯拉，因为该公司在规划未来的同时，正在采取小步骤保持事情向前发展。”

即使联邦税收抵免继续减少，特斯拉面临的挑战将是提高或至少保持利润率。

展望未来

特斯拉预测，由于其加利福尼亚州弗里蒙特工厂全年的高生产率，Model 3 的产量将在 2019 年“大幅增长”。

特斯拉表示，计划在年底前在其上海的“超级工厂”开始生产 Model 3 汽车。汽车制造商表示，他们打算削减在中国工厂生产 Model 3s 的成本，这是一个大胆的预测，如果成功，将有助于提高利润率。

“我们预计这家工厂每单位产能的资本支出不到我们在弗里蒙特的 3 型生产线的一半，”该公司表示。

特斯拉还计划在美国降低成本，预计 2019 年运营费用的增幅将低于 10%。该汽车制造商表示，第一季度采取的重组行动——包括本月发生的裁员——将每年减少约 4 亿美元的成本。

该公司预测，特斯拉第一季度的财务状况将反映一次性重组成本。

因此，特斯拉表示，如果能够成功降低成本，应对欧洲和中国的物流和交付挑战，它仍然乐观地认为，第一季度将实现小额利润。

特斯拉 1 月 2 日报告称，第四季度交付了 90，700 辆汽车，略低于分析师预期。该公司当时表示，它交付了 13，500 辆 Model S 轿车，14，050 辆 Model X SUVs 和 63，150 辆 Model 3。
