特斯拉称 Y 型车将于 2020 年问世

该公司周三在给股东的一封信中表示，特斯拉 Model Y-首席执行官埃隆·马斯克自 2015 年以来一直在取笑和谈论的 SUV 电动汽车-将于 2020 年底开始批量生产。

该汽车制造商尚未展示该车的原型，但表示将于今年开始为 Y 型车制造工具。马斯克在财报电话会议上表示，与特斯拉的其他电动汽车不同，Model Y 最有可能在该公司位于内华达州的大型“gigafactory”生产。

马斯克对 Model Y 持乐观态度，预测该车的销量将高于其新款 Model 3。

特斯拉还预测，Model Y 生产线的成本应该大大低于加利福尼亚州弗里蒙特的 Model 3 生产线，因为它将建立在相同的平台上，并与 Model 3 共享约 75%的零部件。

该公司预测，Y 型车的生产也应该更快。

马斯克表示，在 gigafactory 生产 Model Y 应该会降低该公司的执行风险，以及从内华达州向组装 Model 3 的加利福尼亚州转移零件的成本。

“对特斯拉来说，今年应该是真正令人兴奋的一年，”该公司在股东信中表示。“Model 3 将成为全球产品，我们业务的盈利能力应该会持续增长，我们在上海的新超级工厂应该会开始生产汽车，我们将开始为 Model Y 的生产提供工具。”
