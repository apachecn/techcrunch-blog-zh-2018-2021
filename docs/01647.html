任天堂正在为 iOS 和 Android 制作《马里奥医生》

任天堂推迟了几年开发智能手机游戏，但现在他们就是停不下来。他们以短命的 Miitomo 开始了一个小小的失误，但随后以超级马里奥 Run 找到了一个观众。然后是火徽英雄。然后动物穿越:口袋营地，和德拉加利亚迷失。

接下来呢。马里奥医生。

任天堂今天下午宣布，它正在与 Line 合作开发一款名为《马里奥世界》(Dr. Mario World)的游戏(比如开发 Line 聊天应用的公司；他们还制作迪斯尼的手机游戏《NHN》。

对于那些可能太年轻而不记得超级马里奥作为一名医学博士的人来说，马里奥医生是一款落瓦式游戏，让玩家快速尝试安排……药丸。来杀死病毒。

这就是盒子艺术。任天堂就像，“马里奥现在是医生了，”每个人都像，“哦，好吧，酷。”那是 90 年代，好吗？

任天堂除了称之为“动作益智游戏”之外，并未透露游戏的具体内容。他们表示，该产品将于 2019 年“初夏”上市，并将在 iOS 和 Android 上免费下载(通过应用内购买)。
