微软从 BrightBytes 收购 DataSense management，以加强其在 Azure TechCrunch 的教育业务

原文：https://web.archive.org/web/https://techcrunch.com/2019/02/04/microsoft-acquires-datasense-management-from-brightbytes-to-step-up-its-education-play-in-azure/

微软 Azure 云业务中历史最悠久、规模最大的垂直领域之一是教育，今天它宣布了一项收购，希望能够帮助它深化其影响力:它已经从一家名为 BrightBytes 的教育技术公司收购了 DataSense ，以将功能集成到 Azure 中，data sense 是一个数据管理平台，可用于收集、整合和报告来自一系列在线教育应用程序和服务的信息。

DataSense 是一个主平台，学校和教育机构使用它来获取信息并将其报告给州和其他机构，涵盖不同的应用程序和其他数据源。BrightBytes 说，即使在被微软收购之前，它已经有了很大的影响力，目前被用于管理美国数百万学生的数据。微软的收购应该会加速其增长。

根据 PitchBook 的数据，看起来这笔交易实际上是在 12 月完成的，尽管今天才宣布。

收购条款尚未披露，但作为参考，BrightBytes 在 2015 年最后一次融资时的估值为 1.2 亿美元，由 Insight Venture Partners 牵头，Bessemer、Learn Capital 和 Rethink Education 参与了 3300 万美元的融资。

这家初创公司总共筹集了 5000 多万美元，其他投资者包括新学校风险基金和安迪·佩查克。看起来 BrightBytes 将继续运营，专注于其 Clarity 数据分析平台，该平台已被美国 25，000 所学校使用。

BrightBytes 首席执行官 Traci Burgess 在一份声明中表示:“我们对此次销售为我们的使命提供的全球加速感到兴奋。”“世界各地的学校将从数据集成、互操作性和供应能力中受益匪浅。”

这笔交易将包括总部位于旧金山的 BrightBytes 公司曾从事 DataSense 产品的人员，他们将继续支持现有客户，并开发未来的产品。

一位微软人士没有具体说明具体有多少人会过来，他将其描述为“主要位于佐治亚州亚特兰大地区的远程团队”。这个小团队将加入微软的教育团队。”

微软将把 DataSense 集成到其 Azure 平台中，以扩大微软在其平台上为学校和教育工作者服务的方式，同时继续保持 it 和 Clarity 之间的联系。

“在未来几个月，我们很高兴开始将 DataSense 技术集成到我们的学校产品中，提供一个更安全的基于微软的服务，为学校释放数据分析的力量，”微软教育战略和平台总经理 Steve Liffick 在一份声明中说。它还将继续为该产品的现有客户提供服务。

BrightBytes 和微软有一个共同的目标。两人都认识到，教育已经成为一个非常大的数据业务，但其中有大量的遗留基础架构和碎片化。

随着教育当局开始更新他们的系统，以更好地与政府和其他组织合作，并试图更好地组织他们系统中已经存在的数据，微软和 BrightBytes 等公司已经介入提供帮助。

就微软而言，该公司正在以的数量级在这里工作——提供的硬件，面向不同的利益相关者，从教师和学生到 IT 管理员，以及通过软件、云服务、培训等等。这项交易专门针对帮助车轮转动的 IT 管理员，尽管它最终将触及影响许多其他人的数据。
