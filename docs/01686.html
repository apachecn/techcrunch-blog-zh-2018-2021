Coda 的可编程文档编辑器推出测试版，推出 iOS 应用程序

原文：https://web.archive.org/web/https://techcrunch.com/2019/02/05/codas-programmable-document-editor-comes-out-of-beta-launches-ios-app/

Coda 的可编程文档编辑器推出测试版，推出 iOS 应用程序

今天刚刚结束有限测试的 Coda ，想要彻底改变你对文档和电子表格的看法。鉴于 Word、Excel 等工具以及谷歌、Zoho 等公司的同类产品根深蒂固，这是你能给自己设定的最大挑战。Coda 的秘密武器是，它将文本和电子表格功能结合到一个文档中，能够在其中构建一些基本的编程，并添加第三方服务的功能。

除了向任何人开放这项服务，Coda 今天还推出了新的 iOS 移动应用程序(Android 将在未来某个时候推出)。

“这是最好的文档、电子表格、演示文稿和应用程序——都放在一个新的界面上，”Coda 创始人兼首席执行官(前 YouTube 产品负责人 Shishir Mehrotra 告诉我。“但我们喜欢用的短语是，Coda 允许任何人让文档像应用程序一样强大。”

你不会使用 Coda，它成立于 2017 年，得到了 Greylock，Khosla Ventures 和 NEA 等风投巨头的资助，是一个成熟的低代码/无代码服务。对那个来说还是有点太局限了。但你可以用它来构建你自己的定制库存系统，或者构建一个基本的 CRM 或待办事项应用程序来满足你的特定需求。或者你可以把它当作一个在线文本编辑器，然后根据需要慢慢添加像 Slack 或 Figma 这样的第三方集成功能。对于曾经使用过 Excel 或 Google Sheets 中的函数的人来说，所有这些都很简单。

到目前为止，已经有数万人在私人测试阶段使用了这项服务。Mehrotra 告诉我，他们中大约有 15%的人来自湾区，其中很多人只是将这项服务作为基本的文档编辑器。

不出所料，新的 iOS 应用程序主要专注于消费内容和使用你在 web 应用程序中内置的功能。毕竟，你不太可能想在手机上建立全新的体验。在我看过的演示中，Coda 很好地将单元格及其功能转换成 iPhone 上可用的表格和卡片。
