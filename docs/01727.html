Skype 现在可以在视频通话中模糊背景

由于 WhatsApp 和其他免费通话和消息应用的兴起，Skype 不再是许多人曾经的首选服务。但该公司刚刚增加了一项新功能，可能会提高其对用户的吸引力。

微软拥有的这项服务最新增加了一个屏幕模糊功能，旨在掩盖你凌乱的房间或任何其他背景细节，你不希望在电话中向对方显示。

Skype 表示，这项功能——，类似于去年微软团队加入的模糊功能——使用人工智能来保持对呼叫者的关注。这也意味着它将检测头发、手和手臂等特征。

这项功能正在向 Skype for desktop 推出，尽管它还不支持网络和移动应用程序。可以在 Skype 的设置中启用，也可以通过服务内的视频通话按钮启用。

注意:这篇文章的原始版本已经被更正，以反映该功能在网络或手机上不可用。
