树莓派商店比苹果商店酷多了

树莓基金会刚刚公布了一个全新的项目——一个真正的商店。如果你住在英国的剑桥，你现在可以买一堆甜覆盆子，用来修补和开发一些很酷的东西。

Raspberry Pi 一直致力于让编码更容易理解。实体零售空间正符合这一要求。该基金会开发了一系列极其便宜的电脑，配有基于 ARM 的处理器、一系列端口、Wi-Fi 和蓝牙。

最新的旗舰机型树莓 Pi 3 Model B+售价仅为 35 美元。但是如果你想要更小更便宜的，还有其他型号可以满足各种需求。

也许你只是需要一台微型电脑来完成某个物联网项目。在这种情况下，您可以选择树莓 Pi 3 型号，价格为 25 美元。它的内存和端口少了一点，但它的工作方式非常像任何树莓派。还有价格不到 10 美元的节能机型——树莓派 Zero 机型。

我从来没有真正想过树莓派商店。但是介绍视频为这种商店提供了强有力的支持。产品阵容变得有点复杂，能够和别人谈论你的项目总是好的。

而且，基金会可以把这家店作为展示一些很酷的例子的橱窗。你也可以买一些好东西，比如杯子和毛绒玩具。红白相间的键盘和鼠标看起来也很酷。
