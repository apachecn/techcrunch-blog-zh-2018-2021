“亚马逊直播”是这家零售商通过直播视频挑战QVC的最新举措

原文：https://web.archive.org/web/https://techcrunch.com/2019/02/08/amazon-live-is-the-retailers-latest-effort-to-take-on-qvc-with-live-streamed-video/

亚马逊正在与QVC展开竞争，推出了 Amazon Live，Amazon Live，其中包括来自亚马逊达人的直播视频节目，以及那些通过新应用 Amazon Live Creator广播自己的直播视频的品牌。在直播节目中，主持人谈论并展示亚马逊上出售的产品，就像他们在QVC上做的一样。在那下面有一个旋转木马，购物者可以浏览产品细节并进行购买。

亚马逊网站上同时有多个视频流，因此购物者可以调到他们最感兴趣的那个。

例如，亚马逊Live目前正在播放情人节礼品店秀、以烹饪为主的节目(在厨房里与@EdenEats一起)和Back to Business Live，后者正在展示针对日托中心和学校的产品。

您可以点击不同的视频来更改视频流，向下滚动来观看最近直播的视频记录，或者查看接下来的直播节目。

在网上，直播网站可以在Amazon.com/Live的找到，但它还没有在亚马逊的主导航菜单中列出，所以仍然很难找到。在手机上，从最近的应用更新开始，iOS和Android应用的主导航菜单上都出现了一个名为“Amazon Live”的部分。

我们已经确认Amazon.com/Live页面是新添加的，尽管这不是亚马逊第一次提供直播。

这家零售商过去曾涉足直播，结果喜忧参半。

两年前，它终止了短暂的努力，Style Code Live，它也提供了类似QVC的家庭购物体验。这场直播秀以拥有电视和广播背景的主持人为特色，并请来了专家来谈论美丽和时尚技巧。

但是时尚代码直播只关注时尚和美丽。

另一方面，Amazon Live 涵盖各种产品，从智能家居到游戏、玩具、厨房用品、家居用品、电子产品到厨房用品等等。定位也不一样。直播不是一个只展示亚马逊人才和嘉宾的单一直播视频节目，而是亚马逊向希望接触更广泛观众并发现他们产品的品牌开放的东西。

上图:亚马逊直播主持人——根据LinkedIn，他们不是亚马逊员工

您过去可能已经看过一些品牌的直播视频。

在2017年和2018年的Prime Day，亚马逊播放了直播视频流，宣传Prime Day的一些交易。这些视频是由品牌制作的，非常像你现在在亚马逊Live上看到的一些视频。

该公司还在其今日交易页面上播放了直播内容，并允许品牌在今日之前在他们的产品页面、商店和Amazon.com/Live上播放。

亚马逊现在的目标是让品牌更容易参与亚马逊直播。

在一个详细介绍亚马逊直播的网站上，亚马逊宣传直播视频如何推动销售，让品牌与客户实时互动——包括通过直播期间的聊天——并接触更多的购物者。一个早期的测试者，卡牌游戏制造商“看你的嘴”，被引用说，直播帮助增加了5倍的产品详情页面的日访问量，并“显著增加了我们的销售额”。"

该信息网站还将品牌指向亚马逊的新应用程序直播，亚马逊直播创建者。

该应用仅在iOS上可用，允许品牌在桌面、移动和亚马逊移动应用内将其视频内容直接传输到Amazon.com。该应用程序支持直接从智能手机本身或通过使用专业相机的编码器进行流式传输。

它还包括内置分析，因此品牌可以确定他们的信息流表现如何，包括他们在“提升”(一种接触更多购物者的支付方式)上花费了多少预算，总浏览量，未静音浏览量和其他指标。

根据Sensor Tower的数据，Amazon Live Creator于昨天发布，时间是2019年2月7日，目前在App Store上解除排名。它没有评论，但有五星评级。

亚马逊网站称，目前，直播功能对在亚马逊品牌注册处注册的美国专业卖家开放，不支持来自中国大陆和香港的直播。

 

亚马逊对直播感兴趣已经有一段时间了。公司去年为其围绕视频直播购物的想法申请了专利，而在此之前被发现雇佣为其亚马逊直播服务。

然而，亚马逊当时声称，其直播购物体验“并不新鲜”

这是真的，因为直播有时会出现在大减价的时候，比如Prime Day。但自Style Code Live以来，亚马逊一直直接向在线购物者推广其直播视频。

本周推出的亚马逊品牌直播应用以及亚马逊在其移动应用上创建亚马逊直播流专用链接的举措表明，直播视频正在成为这家零售商的一项更大努力，尽管它试图将此作为“旧闻”扔掉。

这种对直播视频的日益关注也正值 Instagram被传言正在开发一个独立的购物应用程序，并大力推动其专注于创作者的 IGTV产品进入用户的家庭订阅。 QVC本身刚刚宣布其新身份，计划更深入地涉足电子商务，以及可购物视频应用。当然，YouTube已经利用了品牌和影响者的现场和预录视频演示来帮助销售化妆品、电子产品、玩具等产品。

亚马逊正式 拒绝置评。
