电子商务初创公司 Zilingo 融资 2.26 亿美元，将亚洲时尚供应链数字化 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/02/11/zilingo-raises-226m/

如果你正在东南亚寻找下一个独角兽，Zilingo 可能就是它。这家成立 3.5 年的电子商务公司今天宣布，它已经筹集了价值 2.26 亿美元的 D 轮融资，以寻求数字化亚洲时尚供应链的机会。

自 2015 年推出以来，Zilingo 从投资者那里获得了 3.08 亿美元的新一轮投资。D 轮融资由现有投资者红杉印度、新加坡主权基金淡马锡、德国的布尔达和索菲娜提供，索菲娜是 Flipkart 旗下时尚网站 Myntra 的欧洲支持者。新加坡经济发展局(Economic Development Board)的企业投资机构“新投资者”( new investor)EDBI 首次加入了这场派对。

Zilingo 没有对这一轮的估值发表评论，但一位了解交易的消息人士告诉 TechCrunch，这距离 10 亿美元有“舍入误差”。最近几个月，我们听说这家初创公司已经接近独角兽地位，所以这可能是迟早的事——特别是考虑到 Zilingo 已经如此迅速地进入了 D 轮。

融资超过 3 亿美元使 Zilingo 成为东南亚资本最高的创业公司之一，但其去年的迅猛增长来自于从消费者电子商务到企业对企业服务的扩张。

前红杉印度和麦肯锡首席执行官安基蒂·博斯(Ankiti Bose)和首席技术官 Dhruv Kapoor 首先建立了一项服务，利用东南亚日益增长的互联网连接，将曼谷和雅加达等城市街头市场的小型时装摊贩纳入电子商务范围。

Zilingo 仍然运营着面向消费者的在线零售店，但其关键举措是寻求供应链中的 b2b 机会。也就是说，它正在建立一个供应链网络——制造、物流、支付等——可以带给零售商或品牌。因此，理论上，任何想要进入自有品牌或时尚销售的人都可以使用 Zilingo 作为端到端的解决方案来制造和采购他们的产品。

Bose 告诉 TechCrunch，过去一年的收入增长了 4 倍，其中 b2b 占总收入的 75%。她拒绝提供原始数据，但表示净收入在“数亿”美元。该公司拥有 400 多名员工，尚未盈利，但首席执行官 Bose 表示，b2b 部门通过帮助抵消昂贵的电子商务战争，为其提供了“一条实现收支平衡的清晰道路”。

Ankiti Bose 和 Dhruv Kapoor 在 2015 年创立了 Zilingo。

供应链的“过时技术”

在建立分销后进入供应链是有道理的，但 Zilingo 长期以来一直关注服务。

这种以业务为中心的推动始于一套帮助 Zilingo 卖家管理电子商务业务的基本产品。这些服务最初包括库存管理和销售跟踪，但后来逐渐发展到更深层次的服务，如融资、采购和采购，以及识别未来时尚趋势的“时尚猎人”。Zilingo 还扩大了目标，从在东南亚经营的小供应商的长尾，到更大的商家和品牌，甚至到欧洲、北美和其他地区的时装业，寻求进入亚洲生产商的渠道，据估计，亚洲生产商在 30 亿美元的全球时装制造市场中占 1.4 万亿美元。

Zilingo 今天的目标是为任何卖家提供 Zara 等品牌通过多年工作为自己建立的功能、洞察力和网络。

在东南亚，这意味着帮助小商户、中小企业和大型零售商通过 Zilingo 商店在线采购销售商品。但在欧洲和美国，Zilingo 没有经销店，而是直接找卖家。这可能意味着从亚洲寻求批发机会的零售商或 Instagram 名人等在线影响者热衷于利用他们的存在进行电子商务。除了挑选商品出售，Zilingo 还希望利用其供应链网络帮助他们建立自己的自有品牌。

自去年 Zilingo 完成 5400 万美元的 C 轮融资以来，这个世界其他地方的计划就一直在酝酿之中，但现在旅程的下一阶段是与工厂的更深入整合。

Bose 在一次采访中说，“如果你想想这些制造产品的工厂，那里的工艺并没有优化。”“管理工厂的人可能没有技术，他们甚至不会使用 Excel。所以我们要去中小型工厂，提高产能利用率，帮助管理工资，获得贷款和其他金融科技服务。”

她的联合创始人卡普尔补充道，时尚供应链“被过时的技术破坏了”

“对于我们来说，当务之急是开发能够向中小企业有效引入机器学习和数据科学的产品，同时还要易于使用、被采用和快速扩展。我们正在用这种镜片重新连接整个供应链，这样我们就能增加最大的价值，”他在一份声明中补充道。

Zilingo 鼓励零售商和品牌通过利用其建立的供应链网络开发自己的自有品牌

时装供应链的 AWS

Bose 表示，Zilingo 的早期努力已将工厂效率提高了约 60%，并使发展与零售商的联系成为可能，同时也使工厂能够发展自己的自有品牌系列，而不是简单地生产无品牌或无描述的产品。

Bose 说，与工厂合作的很大一部分是基于咨询的，Zilingo 聘请了供应链专家来帮助提供质量指导和观点以及它提供的软件工具。

她在很多方面将其与亚马逊如何构思 AWS 进行了比较。在建立技术来解决内部问题后，它将服务商业化，提供给第三方。因此，Zilingo 开始提供面向消费者的电子商务平台，但它正在以一定的成本向任何人开放其采购网络——几乎就像 API 上的供应链。

这使其业务有了两个(如果不是三个)方面的重点，即通过Zilingo.com向东南亚消费者销售产品——目前在泰国、新加坡、马来西亚和印度尼西亚有销售，菲律宾和澳大利亚也将很快推出——通过 Zilingo Asia Mall 接触海外零售商，以及发展 b2b 业务。

在它的本土市场东南亚，Zilingo 没有强迫它的商家只在它的平台上销售——“如果他们去 Instagram、Lazada、Tokopedia 和 Shopee，我们不介意，”Bose 说——但在美国，它没有消费者渠道。随着该公司考虑潜在的合作伙伴关系，这种情况可能会发生变化，尽管它似乎不太可能推出自己的消费者游戏。

Zilingo 曾经注定要与 Lazada、和 Tokopedia、等大公司竞争，前者由阿里巴巴所有，后者由纽约证券交易所上市的 Sea 运营，而 Tokopedia 是软银愿景基金的一部分，价值 70 亿美元，但其供应链重点已经转移到使能者的位置。

这帮助它避免了专业电子商务服务的艰难时期，这些服务面临激烈的竞争、价格战和充满挑战的动态，而是成为东南亚资本最高的创业公司之一。该公司的美国计划雄心勃勃，启动时间比预期的长，但这使它成为 2019 年值得关注的创业公司。这也是创业之旅没有定义的一个例子，因为在某些情况下，最大的机会不会立即出现。
