Datadog 收购应用测试公司 Madumbo

热门监测和分析平台 Datadog 今天宣布收购了基于人工智能的应用测试平台 Madumbo 。

Datadog 的首席执行官 Olivier Pomel 说:“我们很高兴有 Madumbo 团队加入 Datadog 。“他们已经建立了一个复杂的人工智能平台，可以快速确定 web 应用程序是否正常运行。我们看到他们的核心技术增强了我们的平台，并为我们的客户扩展了许多新的数字体验监控功能。”

总部位于巴黎的 Madumbo 在 F 站孵化，于 2017 年推出，为用户提供了一种测试他们的网络应用程序的方法，而无需编写任何额外的代码。它承诺让开发人员通过简单地与网站交互，使用 Madumbo 测试记录器来建立测试，并帮助他们即时建立测试电子邮件、密码和测试数据。然后，Madumbo 系统会监视您的站点，并根据您所做的任何更改调整其检查。这个 bot 还监视 JavaScript 错误和其他警告，并且可以集成到部署脚本中。

该团队将加入 Datadog 现有的巴黎办事处，并将开发新产品，Datadog 表示将于今年晚些时候公布。Datadog 将在未来几个月内逐步淘汰 Madumbo 平台。

“加入 Datadog 并将 Madumbo 的人工智能测试技术引入其平台是一个惊人的机会，”Madumbo 首席执行官 Gabriel-James Safar 说。“我们长期以来一直钦佩 Datadog 及其领导地位，并很高兴通过与 Datadog 的其他产品紧密集成来扩展我们现有技术的范围。"
