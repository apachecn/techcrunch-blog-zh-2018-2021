Mode 是一个专注于为数据科学家赋权的合作分析平台，刚刚获得了 2300 万美元的新资金

总部位于旧金山的合作分析平台 Mode 已经在 Valor Equity Partners 领投的 C 轮融资中筹集了 2300 万美元。

分别领导 Mode A 轮和 B 轮融资的 Foundation Capital 和 REV Venture Partners 也加入了这轮融资，使该公司的总融资额达到 5000 万美元。

在某些方面，这项投资是对数据科学家持续需求的押注，尽管许多公司专注于让数据分析对更广泛的员工可用并可理解，如雪花和 BigQuery。

Mode 联合创始人兼首席执行官 Derek Steer 认为，由于今天的工具，组织可能需要更少的数据科学家。但他们也需要更好地授权这些个人有效地回答关键问题，如客户如何使用他们的产品。Mode 通过一个集成的 SQL 编辑器、Python、R notebooks 和 visualization builder 来实现这一点，它说这让用户可以灵活地为给定的数据集选择他们想要的抽象级别。

加入董事会的 Valor 合伙人大卫·奥布兰德(David Obrand)表示，新一轮融资在很大程度上也是对 Steer 的押注，他之前曾在 Yammer 与 Steer 合作，Yammer 是一家企业级社交网站，于 2012 年被微软收购。

奥布兰德是 Yammer 的首席客户官，他认为 Steer 是“加速(Yammer)成为数据驱动型企业的关键。”他进一步认为 Steer“体现了他所服务的客户的个性”

同样重要的是，在 Yammer，Steer 学会了如何建立一个“免费增值”软件业务，在一小部分员工开始积极使用其免费版本后，该业务将被一个组织采用。事实上，Mode 的剧本大同小异，让数据科学家获得一个名为 Mode Studio 的免费产品，希望对许多人来说，它将成为他们工作流程的核心，然后他们会要求整个组织的决策者使用它。

这个计划似乎正在起作用。Steer 告诉我们，现在有 600 家机构为 Mode 付费，包括 Twitch、Lyft、Shopify、Meredith 和 Conde Nast。(另外 2000 家公司正在使用它的免费产品。)

目前，该公司的目标是拥有多达 5000 个席位的企业。它也主要迎合美国观众。但随着新资本的增加，该公司计划在地理上进行扩张，包括首次在旧金山以外的地方雇佣员工。Steer 建议期待新的功能。该公司计划扩大每个部门，尤其注重社区工作。
