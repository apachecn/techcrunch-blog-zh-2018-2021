任天堂为 Switch TechCrunch 推出了马里奥、塞尔达、俄罗斯方块等游戏

原文：https://web.archive.org/web/https://techcrunch.com/2019/02/13/nintendo-makes-the-old-new-again-with-mario-zelda-tetris-titles-for-switch/

下午，任天堂在其一个直接视频宣传片中发布了一系列多事之秋，2019 年将成为这一转变的标志性一年。这是该公司宣布的一切，从超级马里奥制造 2 到意外翻拍的 Game Boy classic Link 的觉醒。

stream cold 以新马里奥制作人的开场，老实说，这对于一天来说已经足够了。但是男孩他们有更多的袖子。

第一部《其实新玩意儿:

最后展示但很可能获得互联网大部分回应的是超过四分之一世纪前在 Game Boy 上推出的林克觉醒的翻拍版。我承认从来没有完成这个，但我喜欢它的感觉，所以我渴望玩这个新的倾斜移动，视角切换的 3D 版本。

白金有一个耐人寻味的新游戏叫做星体链，在游戏中你出现在一些看起来疯狂的机器人(？)-机器人上的动作。来自精彩 101、魔兵惊天录和尼尔的人才:自动机确保这将是值得关注的。

最近流行的《皇家战役》和或许是有史以来最好的游戏《俄罗斯方块》结合在一起，在《俄罗斯方块 99》中，100 个人同时竞争丢方块。看起来很疯狂，而且现在就开始免费上 Switch。

和 JRPG 上的提示:

火徽:三座房子有一个很长的地方介绍了主要人物，毫无疑问你会与他们结盟或者背叛他们。空气中弥漫着浪漫气息！和箭头。

从推出《我是 Setsuna》和《迷失的 Sphear》的回归基础工作室而来的是 Oninaki ，一个动作 RPG，看起来像是一个制作精良的有趣游戏，如果不是特别原创的话。

《勇者斗恶龙 11s》——原版热门游戏的增强版——和《DQ 建造者 2》将于今年晚些时候分别在秋季和 7 月进行切换。

符文工厂 4 特辑是我喜欢的一个系列中的另一个增强版，重制的经典(尽管我希望他们重制边疆)。据宣布, RF5 正在开发中，感谢上帝。

最终幻想 VII 三月底就要来了，最终幻想 IX 现在也有了。我很惭愧地说，我从来没有玩过后者，但这是一个很好的机会。

新老侧滚机:

BOXBOY！+ BOXGIRL！是一个类似益智游戏平台系列的新作品，引入了一些新角色和多人游戏。四月到来。

血迹斑斑:夜晚的仪式有个预告片，但是我们已经听说了很多关于这部恶魔城精神续集的事情。出来吧！

Yoshi 的《精雕细琢的世界》将于 3 月 29 日上映，但今天会有试玩版。

T2】

蛤蟆队长:寻宝者更新了复杂的关卡，增加了多人游戏，很快，新的付费套装也将推出。我可能会等待一个组合版本，但这应该很有趣。

杂而有趣:

新的漫威终极联盟将在今年夏天到来，我已经等不及了。第二个是一个爆炸，但它出来太久了。一个好的合作打架者是一个自然的选择，而且成为超级英雄很有趣。

守护天使 X 玛奇纳，这款看起来很震撼的机甲战斗游戏，在夏季发行前获得了试玩版。他们将整合玩家的改变和建议，所以如果你想帮助塑造游戏，就去做吧。

迪士尼 Tsum Tsum 节 …不知道这是什么。但是看起来很狂野。

Deltarune ！这是心爱的 Undertale 的续集，你现在可以得到 Switch 的第一章。先玩低音，否则你不会明白狗的笑话。
