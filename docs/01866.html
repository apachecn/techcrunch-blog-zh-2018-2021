Qloo 收购文化推荐服务 TasteDive

Qloo 今天上午宣布收购了 TasteDive 。

这两家公司听起来非常相似——根据的公告，Qloo 是“文化和品味的领先人工智能平台”，而 TasteDive 是“文化推荐引擎和社交社区”

有什么区别？嗯，TasteDive 是一个网站，你可以创建一个个人资料，与其他用户联系，根据你喜欢和不喜欢的东西，它会推荐音乐、电影、电视节目、书籍等。与此同时，Qloo 正试图了解消费者口味的模式，然后将这些数据出售给营销人员。

或者，正如 Qloo 首席执行官亚历克斯·埃利亚斯(上图)在一份声明中所说，“TasteDive 为数百万人做了 Qloo 多年来为品牌所做的事情——利用人工智能做出关于文化和品味的更好决定。”

显然，TasteDive 拥有 450 万活跃用户，它将继续作为一个独立的团队和产品运营，创始人安德烈·奥吉纳仍将担任首席执行官。(埃利亚斯将成为主席。)

与此同时，两家公司表示，Qloo 技术的加入将使 TasteDive 变得更加智能，并扩展到不同的类别，而 Qloo 则受益于 TasteDive 的全球客户群及其 API 生态系统。

收购的财务条款没有披露。
