数字海洋推出托管数据库服务

原文：https://web.archive.org/web/https://techcrunch.com/2019/02/14/digitalocean-launches-a-managed-postgresql-database-service/amp/

图片来源:氧气/盖蒂图片

DigitalOcean 最初是一款价格实惠但功能基本的虚拟专用服务器，拥有令人愉悦的用户界面。在过去的几年里，该公司开始增加一些功能，如对象和块存储、负载平衡器和容器服务。今天，它又一次扩展了它的产品组合，推出了它的产品线中非常缺少的一个功能:托管数据库服务。

这些数字海洋管理数据库的第一版只支持 PostgreSQL，这是一种流行的开源关系数据库。今年晚些时候，它将增加 MySQL 和 Redis 支持(可能在 Q2 或第三季度)。至于其他数据库，该公司表示，它将听取客户的反馈，并利用这一点来优先考虑其他产品。

与其他供应商的类似服务一样，托管数据库有望使开发人员的生活更加轻松。数字海洋用户将能够在几秒钟内启动一个数据库，然后该服务处理所有的维护任务，包括更新。与该公司的其他服务一样，除了第三方 Terraform 提供商之外，开发人员可以使用图形用户界面或该公司的 API。

每日备份是免费的，DigitalOcean 承诺您的数据在静态和传输中的端到端安全性。

这项新服务的定价如下:

“我们的产品开发受到一个重要问题的驱动:我们如何让开发人员在更短的时间内做更多有价值的工作？” DigitalOcean 的产品副总裁 Shiven Ramji 说。“有了托管数据库，开发人员及其团队可以专注于创建有意义的应用程序，并与他们的社区共享这些应用程序，而不必管理支持该过程的数据库基础架构。”
