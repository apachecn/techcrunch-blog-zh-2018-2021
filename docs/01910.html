OpenAI 构建了一个非常好的文本生成器，它被认为太危险了，不能发布

原文：https://web.archive.org/web/https://techcrunch.com/2019/02/17/openai-text-generator-dangerous/amp/

图片来源: OpenAI

一场关于一种新语言模型的风暴正在酝酿之中，这种语言模型是由非营利人工智能研究公司 OpenAI 建立的，该公司表示，这种语言模型非常擅长生成令人信服的、写得很好的文本，以至于它担心潜在的滥用。

这激怒了社区中的一些人，他们指责该公司违背了不关闭其研究的承诺。

OpenAI 表示，其新的自然语言模型 GPT-2 被训练成在 40gb 的互联网文本样本中预测下一个单词。最终结果是系统生成“适应条件文本的风格和内容”的文本，允许用户“生成关于他们选择的主题的现实和连贯的延续。”该模型是对第一个版本的巨大改进，产生了更长的文本，具有更强的连贯性。

但是，随着该系统的每一个良好应用，如能够更好地对话和更好地语音识别的机器人，非营利组织发现了更多的应用，如生成假新闻，模仿他人，或自动在社交媒体上发布辱骂或垃圾评论。

也就是说:当 GPT 2 号被要求对“回收对世界有益，不，你错得不能再错了”的提示做出回应时，机器反驳道:

“回收对世界没有好处。这对环境有害，对我们的健康有害，对我们的经济也有害。我没开玩笑。回收对环境没有好处。它对地球具有破坏性，是全球变暖的主要原因。回收对我们的健康没有好处。它会导致肥胖和心脏病、癌症等疾病。回收对我们的经济有害。它增加了产品的成本，进而增加了用该产品制造的所有东西的价格。回收对我们国家没有好处。我们为拥有世界上最先进、最高效的回收系统付出了巨大的代价。回收是对时间、能量、金钱和资源的巨大浪费。”

难怪 OpenAI 担心发布它。

出于这个原因，OpenAI 说，它只是发布了一个较小版本的语言模型，引用了的章程，其中指出，该组织预计“安全和安保问题将在未来减少我们的传统出版。”诚然，该组织表示，它不确定这一决定，“我们相信人工智能社区最终需要在某些研究领域以一种深思熟虑的方式解决出版规范的问题。”

不是每个人都能接受。OpenAI 发布的宣布 GPT-2 的推文遭到了愤怒和沮丧，指责该公司“关闭”其研究，并做了“与 open 相反的事情”，抓住了该公司的名字。

其他人更加宽容，称此举为“道德的新标杆”，为可能的滥用提前思考。

OpenAI 的政策主管杰克·克拉克(Jack Clark)表示，该组织的首要任务是“不允许恶意或滥用该技术”，称这是“对我们来说非常艰难的平衡行动”。

OpenAI 的最初投资者之一埃隆·马斯克被卷入了这场争议，他在一条推特上证实他已经“一年多”没有参与该公司了，他和该公司“友好地”分手了

OpenAI 表示，它尚未就 GPT 2 号的发布做出最终决定，并将在六个月内再次访问。与此同时，该公司表示，政府“应该考虑扩大或开始采取措施，更系统地监控人工智能技术的社会影响和传播，并衡量此类系统能力的进步。”

就在本周，特朗普总统签署了一项关于人工智能的行政命令。几个月前，美国情报界警告称，人工智能是美国国家安全面临的众多“新兴威胁”之一，其他威胁还有量子计算和无人驾驶汽车。
