Redis Labs筹集了6000万美元的E轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2019/02/19/redis-labs-raises-a-60m-series-e-round/

Redis Labs ，一家围绕Redis内存数据存储提供商业服务的初创公司(Redis creator和首席开发人员Salvatore Sanfilippo是其员工之一)，今天宣布它已经筹集了6000万美元的E轮融资，由私募股权公司 Francisco Partners 牵头。

该公司没有参加Redis Labs之前的任何一轮融资，但现有投资者高盛私人资本投资公司、贝恩资本风险投资公司、Viola Ventures和戴尔技术资本公司都参加了这一轮融资。

Redis Labs现在总共筹集了1.46亿美元，该公司计划使用新的资金来加速其上市战略，并继续投资于Redis社区和产品开发。

目前Redis实验室的用户包括美国运通、史泰博、微软、万事达和Atlassian等。该公司现在总共拥有8，500多名客户。因为它是非常灵活的，这些客户根据他们的需求将该服务用作数据库、缓存和消息代理。该公司的旗舰产品是Redis Enterprise，它扩展了开源的Redis平台，为企业提供了额外的工具和服务。该公司提供托管云服务，除了传统的软件下载和自我管理安装的许可证之外，企业可以选择在AWS、GCP和Azure等公共云以及他们的私有云上托管。

Redis Labs首席执行官Ofer Bengal告诉我，该公司的现金还不乐观。他还指出，该公司不需要进行这一轮融资，但他决定这样做是为了加速增长。“在这种竞争激烈的环境下，你必须在产品开发上投入大量资金，大力推进，”他说。

值得注意的是，他强调，Francisco Partners在推动公司发展方面享有盛誉，Redis Labs的下一步合乎逻辑的发展将是IPO。“我们认为我们有一个非常独特的机会来建立一个非常大的公司，值得IPO，”他说。

这种新的竞争环境的一部分也涉及到竞争对手，他们使用其他公司的开源项目来构建自己的产品，而没有回报。Redis Labs是第一批决定在新许可证下提供最新版本的开源公司之一，新许可证仍然允许开发者修改代码，但迫使想要转售代码的竞争对手购买商业许可证。在这种情况下，Ofer特别提到了AWS。值得注意的是，这不是关于Redis数据库本身，而是关于Redis实验室构建的额外的模块。Redis企业本身就是闭源的。

“当我们推出这种新的许可证时，有许多不同的观点，”他承认。“有些人对此表示谴责。但是在最初的喧嚣平静下来之后，特别是在其他一些公司提出了类似的概念之后，社区现在明白开源的最初概念必须得到修正，因为它不再适合现代时代，在现代时代，云公司利用他们的垄断权力采用任何成功的开源项目，而不对其做出任何贡献。”
