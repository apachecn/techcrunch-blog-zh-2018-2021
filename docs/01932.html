SendBird 获得 5200 万美元 B 轮融资以扩展消息传递 API 工具

旧金山地区的一家初创公司 SendBird ，用几行代码帮助开发者将消息添加到他们的应用程序中。这种想法类似于支付领域的 Stripe 或通信领域的 Twilio。今天，该公司宣布了 5200 万美元的 B 轮投资。

本轮由 Iconiq Capital 领投。现有投资者 Shasta Ventures、August Capital、Y Combinator 和 Funders Club 也参与其中。根据 Crunchbase 的数据，今天的投资使融资总额超过 7000 万美元。Y Combinator 参与了这一轮，SendBird 实际上是 2016 年冬季班的成员。

该公司让开发者能够毫不费力地将消息添加到他们的应用程序中。“我们是一个非常灵活、完全可定制的白标消息传递功能。我们提供完全托管的基础架构。因此，基本上，你可以登录任何移动应用程序或网站，并使用我们的信息功能，”公司创始人兼首席执行官约翰·金解释说。

Kim 说它适用于任何需要交流的应用。这可能是在线游戏、约会应用或按需交付服务——任何两个人需要快速沟通的情况。它提供了任何 API 驱动的服务的价值支柱，因为它使开发人员能够用几行代码为应用程序添加消息传递功能，使他们不必自己从头开始创建这样的服务。

他们根据使用具有 SendBird 消息功能的应用程序的用户数量提供分层定价，并提供额外费用的高级功能，如聊天审核工具或自动翻译。

除了 API，该公司还提供 iOS 和 Android 的 SDK 以及 JavaScript，React Native。NET 和 Unity。

这个过程非常简单，该公司已经吸引了 50，000 名开发人员，运行在近 12，000 个应用程序上，每月处理超过 10 亿条消息。客户包括 NBA，雅虎！体育和 Glu 移动。

该公司发展迅速。2017 年，该公司只有 25 名员工。今天，Kim 报告说他们有 74 人，并且正在迅速增加人手。他说，他将利用这笔钱继续扩张，以满足不断增长的市场需求，宣传公司，培养一支营销和销售团队，直接向开发者销售该工具。

到目前为止，它的成功主要是靠口口相传，但他们想用新基金做的一部分是把话传出去，增加他们的市场份额。
