谷歌云的语音 API 变得更便宜，并学习新的语言

谷歌今天宣布了对其云语音转文本和文本转语音API 的更新，引入了一些企业用户应该特别感兴趣的新功能，以及改进的语言支持和降价。

这些更新大多集中在语音到文本的产品上，但云文本到语音正在获得 31 个新 WaveNet 和 24 个新标准语音的重大更新。该服务现在还支持七种新语言:丹麦语、葡萄牙语/葡萄牙语、俄语、波兰语、斯洛伐克语、乌克兰语和挪威语博克马尔语。这些语言目前都处于测试阶段，支持的语言列表已增加到 21 种。

该服务现在还具有为特定设备优化音频播放的功能。这听起来像是一件小事，但它允许您优化一个用于交互式语音应答的呼叫中心应用程序和另一个用于耳机的应用程序。

至于云语音转文本，这次更新的重点是让这项服务在开发者必须支持多种渠道的用户的情况下更可用——想象一下电话会议。为此，该公司去年推出了多渠道识别作为测试版；现在，这个功能已经普遍提供了。

同样，谷歌用于视频和增强手机的优质人工智能模型去年进入测试阶段，承诺比谷歌的另一个模型更少的转录错误，后者主要专注于简短的查询和语音命令。这种模式现在也普遍可用。

除了这些新功能，谷歌还决定降低语音转文本服务的使用价格。该公司决定为那些选择加入谷歌数据记录程序的人将录制视频的标准和高级视频模式的价格下调 33%。通过选择加入，您允许谷歌使用您的数据来帮助训练谷歌的模型。该公司承诺，只有有限数量的员工可以访问这些数据，并且它将只使用这些数据来培训和改进其产品，但可能不是每个人都愿意选择这一点，即使这意味着有折扣。

令人欣慰的是，常规的付费视频模式现在也便宜了 25 %,而不必登录谷歌的数据记录。和以前一样，前 60 分钟还是免费的。
