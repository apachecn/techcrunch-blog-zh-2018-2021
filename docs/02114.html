埃隆·马斯克表示，特斯拉 Model Y 将于 3 月 14 日在其洛杉矶设计工作室发布

根据首席执行官埃隆·马斯克的一条推特，特斯拉将于3 月 14 日在洛杉矶的一场活动中拉开帷幕，推出其 Model Y 跨界车。

自 2003 年公司成立以来，这是第五款来自特斯拉工厂的新车设计。自 2015 年以来，马斯克一直在取笑这款车的发布，在一月份给股东的信中表示，大规模生产将于 2020 年底开始。

特斯拉表示，它将在今年晚些时候开始为 Model Y 制造工具，该公司将在内华达州的“gigafactory”生产这款车。在同一封信中，马斯克预测，Model Y 系列的成本将大大低于加州弗里蒙特的 Model 3 系列，因为它将与新的低成本汽车共享大约 75%的相同部件。

特斯拉的 Model Y reveal 发布之际，该汽车制造商上周宣布了彻底的变革，同时推出了 35，000 美元的低价 Model 3。

特斯拉表示，为了实现这一较低的价格，它将把全球所有销售转移到网上，这意味着该公司将在未来几个月关闭许多商店。马斯克在与记者的电话会议上说，留在高流量位置的商店将被变成信息中心。结果会有一些裁员。马斯克后来表示，他们将雇佣更多的服务技术人员。

该公司在一篇文章中写道:“将所有销售转移到网上，结合其他持续的成本效率，将使我们能够将所有车辆的价格平均降低约 6%，使我们能够比预期更早地实现 35，000 美元的 Model 3 价格点。”

特斯拉的管理层预计，所有这些变化应该会给公司带来更好的结果。“Model 3 将成为一种全球产品，我们业务的盈利能力应该变得持续为正，我们新的上海 Gigafactory 应该开始生产汽车，我们将开始为 Model Y 的生产提供工具，”1 月份的股东写道。

如果 Model Y 最终投入生产，这可能意味着旧的特斯拉车型将被淘汰，尽管这并不确定。

特斯拉还有另外两款正在研发中的车型——跑车和特斯拉 Semi。
