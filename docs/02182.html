特斯拉新增压器大幅缩短充电时间 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/03/06/teslas-new-supercharger-slashes-charging-times/?yptr=yahoo

特斯拉正在推出第三代增压器，旨在大幅缩短其电动汽车的充电时间，以保持对新竞争对手的优势。

周三在该公司位于加州弗里蒙特的工厂发布的 V3 增压器支持远程版 Model 3 高达 250 千瓦的峰值功率。按照这个速度，V3 可以在 5 分钟内增加 75 英里的里程，特斯拉说。

充电时间的改善对该公司至关重要，因为它销售了更多的 Model 3 汽车，这是其销量最高的汽车。在一些受欢迎的超级充电站等待时间可能会很长。早期采用者可能会满足于等待，但随着新的特斯拉客户上线，耐心可能会减少。

特斯拉表示，与今天相比，它的改进将使超级充电站网络在 2019 年底每天为超过两倍的车辆提供服务。

V3 不是该公司前几代产品的改型。这是一个架构上的转变，包括一个新的 1 MW 功率柜，类似于该公司的公用事业规模产品，以及液冷电缆设计，这使得充电速度高达每小时 1000 英里。特斯拉在 V2 增压器上使用风冷电缆。

新的电源柜将为四台增压器提供专用的 250 千瓦功率。这意味着车辆充电时将不再功率共享。

特斯拉在一篇博客文章中宣布了其他改进，包括旨在提高其 Model S 和 Model X 车辆充电率的改进。特斯拉表示，当与 V3 增压器结合使用时，充电时间平均减少了 50%。

一个名为“途中电池预热”的新软件功能也将为所有车辆发布。该公司表示，这一更新使电池组能够在尽可能长的时间内接受车辆的峰值功率，从而将平均充电时间减少 25%。

特斯拉计划在 2019 年开设数千个 V2 和 V3 超级充电器。特斯拉将于 4 月开始安装的 V3 站将被放置在使用率最高的地方。据该公司称，特斯拉在北美、欧洲和亚洲拥有超过 12，000 台超级充电器。

特斯拉还计划更新其 V2 增压器，为单车充电提供 145 千瓦的新峰值充电速率。

特斯拉将在第二季度向所有车主推出 V3 增压系统，以覆盖更广泛的车队——这意味着超出了远程 Model 3s。该公司计划在第二季度和第三季度在北美增加 V3 的安装。V3 安装将于第四季度在欧洲和亚太地区开始。
