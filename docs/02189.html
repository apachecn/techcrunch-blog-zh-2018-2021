播客的美妙之处在于任何人都可以做。这是一种罕见的媒介，制作起来几乎和消费起来一样容易。因此，没有两个人会以完全相同的方式做这件事。有大量的硬件和软件解决方案对潜在的播客开放，所以设置运行从 NPR 工作室到 USB Skype 钻机的范围。

我们请了一些我们最喜欢的播客主持人和制作人来强调他们的工作流程——他们用来完成工作的设备和软件。到目前为止的列表包括:

林德伯格的《灵动狂野》
我自己的播客，RiYL

第三周，我邀请我的老朋友兼专业播客杰西·索恩分享他的想法。“美国电台甜心”是几个长期节目的主持人，包括 NPR 的《T21》和喜剧播客《乔丹·杰西·戈》。他还是总部位于洛杉矶的播客网 Maximum Fun 的所有者和经营者，该网主持了数量惊人的节目，包括《约翰·霍吉曼法官》、《我的兄弟》、《我的兄弟和我》以及《哦，不，罗斯和凯莉》。

他用 Hyphy 的必备配菜突出了他独特的设置。

我在 Craigslist 上买了这个录音棚。多年来，我在自己的房子里录音，墙壁上排列的书籍和许多柔软的家具足以让我的录音听起来过得去，但后来我妻子告诉我去找一间办公室。我们客厅的实习生一直在吵醒婴儿。

办公室在一座混凝土阁楼建筑里，对声音来说很难忍受，所以我最终来到了洛杉矶中部哈佛高地一个家伙的后院。他是一个巨大的萨摩亚花花公子谁运行一个太平洋岛民 R&B 标签，直到他的商业伙伴去巡回演出，作为贾斯汀比伯的声乐教练。我花了 3000 美元买下了它，雇了一个有货车的人把它带到办公室，安装好，我们就开始做生意了。

客人们经常对展位发表评论——从“这是一个小廉租房”到“我是什么，在电锯惊魂电影里吗？”老实说，我认为这有助于保持我们的对话人性化。不太隔音；我们的一个制作人是一个特别爱笑的人，他经常出现在录像带上。但老实说，我有点喜欢。
