F5 以 6.7 亿美元收购 NGINX，进军开源的多云服务

原文：https://web.archive.org/web/https://techcrunch.com/2019/03/11/f5-acquires-nginx-for-670m-to-move-into-open-source-multi-cloud-services/

多云架构是企业中的一个巨大趋势，今天 F5 采取了一项重大举措，使自己的业务更接近这一趋势。这家提供云和安全应用服务的公司，宣布以 6 . 7 亿美元收购了 NGINX ，这是一家支持流行的开源网络服务器的商业公司。

实际上，我们听到这次收购的传言已经有一段时间了，价格标签约为 7 亿美元。最重要的是，我们的消息来源称 NGINX 正在四处采购，其他公司也在关注它，包括 Citrix。那笔交易在价格上失败了。

NGINX 上一次融资是在 9 个月前，由高盛牵头的 4300 万美元融资，以推动扩张，近年来 NGINX 将自己定位为 F5 的有力替代者。(该公司没有披露那一轮的估值。)巧合的是，据说 F5 自己在 2016 年聘请了高盛对其进行收购，尽管这从未导致任何事情。

F5 总裁兼首席执行官 Fran ois Locoh-Donou 在一份声明中表示:“F5 收购 NGINX 通过加速我们的软件和云计算转型，加强了我们的增长轨迹通过将 F5 世界一流的应用安全性和丰富的应用服务组合(用于提高性能、可用性和管理)与 NGINX 领先的软件应用交付和 API 管理解决方案、DevOps 社区中无与伦比的可信度和品牌认知度以及庞大的开源用户群相结合，我们可以在企业的云计算环境中提供一致的应用服务，从而弥合 NetOps 和 DevOps 之间的鸿沟。"

事实上，我们的消息来源指出，该公司的增长有些停滞，这是它对 NGINX 感兴趣的一个原因。今天收盘时，F5 的市值为 96 亿美元。在最近一个季度的收益中，该公司表示，与去年相比，其收入仅增长了 4%。与此同时，自 2004 年作为 Apache 的替代品首次出现以来，NGINX 一直是提供维护和运行网站的开源工具的巨头。该公司目前运营着 3.75 亿个网站，约有 1，500 名付费客户接受额外的服务，如支持、负载平衡、API 网关和分析。

F5 表示，它将把自己的业务与 NGINX 的业务合并，现任 NGINX 首席执行官 Gus Robertson 以及创始人 Igor Syosev 和 Maxim Konovalov 都将加入该公司。

“NGINX 和 F5 有着共同的使命和愿景。我们都相信应用是推动数字化转型的核心。罗伯逊在一份声明中说:“我们都相信，需要端到端的应用基础设施，即从代码到客户的应用基础设施，才能在云计算环境中交付应用。”。“我很高兴能够继续这一旅程，将 NGINX 的开源创新力量融入 F5 的 ADC 领导地位和企业影响力。F5 通过专为 DevOps 设计的解决方案获得深度，而 NGINX 通过接触成千上万的客户和合作伙伴获得广度。"
