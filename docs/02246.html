F5 以 6.7 亿美元收购 NGINX，进军开源、多云服务领域

原文：https://web.archive.org/web/https://techcrunch.com/2019/03/11/f5-acquires-nginx-for-670m-to-move-into-open-source-multi-cloud-services/amp/

影像学分: tr3gi / Getty 影像

多云架构是企业的一个巨大趋势，今天 F5 迈出了一大步，将自己的业务推向更近一步。提供云和安全应用服务的公司,宣布以 6.7 亿美元收购了流行的开源网络服务器背后的商业公司 NGINX 。

实际上，我们已经听到了一段时间关于这次收购的杂音，价格在 7 亿美元左右。除此之外，我们的消息来源称，NGINX 正在四处购物，其他一直在关注它的公司包括 Citrix。那笔交易在价格上失败了。

NGINX 最近一次融资是在 9 个月前，由高盛牵头的 4300 万美元的 T8 轮融资推动了扩张，并在最近几年将自己定位为 F5 的有力替代品。(它没有披露这一轮的估值。巧合的是，据说 F5 本身在 2016 年就聘请了高盛来开发自己的收购兴趣，尽管这从未带来任何结果。

F5 总裁兼首席执行官 Fran ois Locoh-Donou 在一份声明中表示:“F5 对 NGINX 的收购通过加速我们的软件和多云转型，巩固了我们的增长轨迹。”通过将 F5 用于提高性能、可用性和管理的世界级应用程序安全性和丰富的应用程序服务产品组合与 NGINX 领先的软件应用程序交付和 API 管理解决方案、在 DevOps 社区无与伦比的信誉和品牌认知度以及庞大的开源用户群结合在一起，我们在企业的多云环境中以一致的应用程序服务弥合了 NetOps 和 DevOps 之间的鸿沟。"

事实上，我们的消息来源指出，该公司的增长有些停滞，这也是其对 NGINX 感兴趣的原因之一。今天收盘时，F5 的市值为 96 亿美元。在最近一次季度盈利中，该公司表示其收入与去年相比仅增长了 4%。与此同时，自 2004 年作为 Apache 的替代品首次出现以来，NGINX 一直是提供维护和运行网站的开源工具的巨头。该公司目前运营着 3.75 亿个网站，约 1500 名付费客户接受额外服务，如支持、负载平衡、API 网关和分析。

F5 表示，它将把自己的业务与 NGINX 的业务合并，现任 NGINX 首席执行官 Gus Robertson 以及创始人 Igor Syosev 和 Maxim Konovalov 都将加入该公司。

“NGINX 和 F5 有着共同的使命和愿景。我们都相信应用是推动数字化转型的核心。罗伯逊在一份声明中说:“我们都相信，需要端到端的应用基础设施，即从代码到客户的应用基础设施，才能在云计算环境中交付应用。”。“我很高兴能够继续这一旅程，将 NGINX 的开源创新力量融入 F5 的 ADC 领导地位和企业影响力。F5 通过专为 DevOps 设计的解决方案获得深度，而 NGINX 通过接触成千上万的客户和合作伙伴获得广度。"
