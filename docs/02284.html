AgroStar 获得 2700 万美元的 C 系列资金，通过数据分析帮助更多印度农民提高作物产量

印度农业科技初创公司 AgroStar 今天宣布，它已经为其平台筹集了 2700 万美元的 C 轮融资，该平台帮助农民提高作物产量，管理订单和在线购买供应品。这轮融资由贝塔斯曼印度投资公司牵头，回归投资者 Accel、Chiratae Ventures 和 Aavishkaar Bharat Fund 参与。

AgroStar 将其应用程序称为“一站式解决方案”，包含印度 1.35 亿农民所需的一切，包括农学工具，如结合图像识别技术、教育内容的作物疾病诊断工具和销售农产品的电子商务商店。它的新资金使 AgroStar 迄今为止筹集的资金总额达到 4200 万美元，将用于扩大该平台的规模，以利用印度农村地区智能手机普及率不断提高的优势，并雇佣更多的农业专家。

首席执行官兼联合创始人 Shardul Sheth 告诉 TechCrunch，该应用程序已被下载超过 100 万次，并有超过 100 万农民参与。它的目标是在未来两年内达到 1000 万次下载，并将其电子商务业务扩展到印度更多的邦。

在一份声明中，贝塔斯曼印度投资公司董事总经理 Pankaj Makkar 表示，该投资公司将与 AgroStar 合作，“为其供应链制定正确的战略，并将从我们在中国类似领域的投资中获得最佳实践。”
