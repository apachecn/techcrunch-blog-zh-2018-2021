TwitSnap？Twitter 推出新的摄像头功能来降级文本

推特推出一个月前 TechCrunch 报道的对其相机功能的重新设计，该功能旨在让你捕捉照片、视频和现场镜头，并将它们与全球对话联系起来。从今天开始，当你从 Twitter 时间轴上向左滑动时，你将获得不再隐藏在 tweet composer 中的新相机。在你捕获一些媒体后(暂时没有上传)，你可以在一个彩色标签上覆盖一个位置，标签或者一些文字。但真正特别的是，Twitter 将在提要中以更大、更身临其境的格式显示这种媒体，图像出现在你的推文中的文本之前。

Twitter 并没有向对手 Instagram 发布故事或一些专门的图片。但它希望成为一个更实时的世界镜头，而不仅仅是通过人们的话语来解读世界。增强的摄像头可以让更多人拍摄媒体，这可以让 Twitter 更容易被文本墙吓倒的新用户访问。更多的视觉内容也使更多的视觉广告更容易进入提要。

Twitter 告诉我，它不会在主时间轴上给用相机创建的推文算法提升。但一位发言人告诉我，它的人力和技术联合策展团队可能会在探索标签的“正在发生什么”部分关注 Twitter 摄像头的推文。我们将看到媒体机构和品牌是否试图利用新相机在提要中脱颖而出。

当你在时间轴上向左滑动时，你会看到一个 Snapchat 风格的相机快门按钮，轻按一下即可记录照片，如果按住不放，可循环播放长达两分钟的视频。轻轻一扫，你就可以录制视频或纯音频直播，无需任何 Periscope 品牌(该应用程序会成为 Twitter Live 吗？).然后，Twitter 会根据附近的大型活动和其他信号推荐标签，或者你可以添加自己的标签以及位置和文本。你可以为覆盖在这些标签上的电视新闻风格的 chyron 选择六种颜色，这些标签帮助 Twitter 将内容路由到不同的正在发生的部分的图像传送带。

目前，Twitter 摄像头中没有像 tweet composer 中的图像上传器那样的贴纸、滤镜、灯光增强或其他创意工具(tweet composer 也方便地提供了新摄像头的快捷方式)。Twitter 告诉我，它希望专注于将内容引入正确对话的标签，而不是美化媒体。这与刚刚开始推出增强现实人脸过滤器的 YouTube 不同。

从历史上看，推特一直非常缓慢地推出产品变更，因为害怕打扰它的长期忠实者。但推特告诉我，它正在翻开新的一页，即使有粗糙的边缘，也要把新相机推出去，这样它才能开始获得反馈和迭代。这比我们过去使用的推特更接近脸书的建设方式。此举与本周发布的推特测试版原型应用 twttr 相一致。推特似乎终于明白，等待每一个功能的完善和害怕实验已经让推特落后于它的竞争对手。

热爱推特的人在其他任何地方都找不到同样的持续对话的地狱，新相机可能也不会改变这一点。但是，在不贬低自身的情况下，转向视觉交流来追赶故事潮流，可能会让推特在一个越来越多地通过图像交流的世界里变得更加舒适。
