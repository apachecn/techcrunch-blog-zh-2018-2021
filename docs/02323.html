特斯拉 Model Y 是一款 300 英里里程的 Model 3 doppelgnger，将于 2020 年秋季上市

原文：https://web.archive.org/web/https://techcrunch.com/2019/03/14/tesla-model-y-unveil-suv/amp/

图片来源:柯尔斯顿·科罗塞克

乍一看，似乎特斯拉 Model 3 有一个二重身。

经过多年的挑逗和暗示，特斯拉首席执行官埃隆·马斯克终于推出了 Model Y，这是一款中型全电动汽车，计划于 2020 年秋季上市。

Y 型与 3 型有惊人的相似之处。这款车将采用标准的远程双电机全轮驱动和高性能车型，比 Model 3 稍大，可以容纳 7 人(对于选择支付 3000 美元充电费的人来说)。它的位置也比 Model 3 略高。最显著的区别之一是 Y 型车有一个全景车顶。

这就是差异开始消失的地方。

Model Y 的内饰与 Model 3 相同，包括相同的 15 英寸触摸屏界面，以及门把手等其他功能。

现在，y 型车的站姿稍微高了一点，前端更短了。

其他重要规格

标准射程版本起价为 39，000 美元，射程为 230 英里。然而，特斯拉将首先生产性能，双电机和远程版本。想要标准版 Model Y 的客户至少要等到 2021 年春季。性能和双电机版本将能够在一次充电后行驶 280 英里，而远程版本，正如它听起来的那样，将具有最长的里程，为 300 英里。

所有的变体都被设计成具有与其较小的兄弟同类相同的性能。Model Y 的性能版本将能够在 3.5 秒内从 0 到 60 英里/小时，最高时速可达 150 英里/小时。

但是这种性能需要付出更高的代价。性能版起价 6 万美元。双电机版本的起价为 51，000 美元，远程版本的基础价格为 47，000 美元。

马斯克没有说 Model Y 将在哪里生产，也没有透露车辆规格和模糊时间表以外的其他细节。

他确实对 Model Y 做出了乐观的预测，在活动结束时表示，他预计 Y 的销量将超过 Model S 和 Model X 的总和。特斯拉迄今已售出超过 50 万辆汽车，包括 Roadster、S、X 和 3。
