Suse 再次成为一家独立的公司

原文：https://web.archive.org/web/https://techcrunch.com/2019/03/15/suse-is-once-again-an-independent-company/

开源基础设施和应用交付供应商 Suse T1——最古老的 Linux 发行版背后的公司——今天宣布再次成为一家独立公司。该公司今天完成了成长型投资者 EQT 从微焦点(Micro Focus)手中以 25 亿美元收购的交易，微焦点在 2014 年收购了该公司。

很少有公司像 Suse 那样频繁易手，但仍然是其业务中的强大参与者。Suse 于 2004 年首次被 Novell 收购。Novell 随后于 2010 年被 Attachmate 收购，后者于 2014 年被微焦点收购。该公司随后将 Suse 变成了一个独立的部门，然后在 2018 年年中宣布将其出售给 EQT。

尽管微焦点和 EQT 花了一些时间来完成收购，但现在，自 2004 年以来，Suse 第一次独立了。

Micro Focus 表示当 Suse 以 23.5 亿美元收购 Attachmate Group 时，它只创造了该集团总收入的 20%。从那时起，Suse 已经创造了相当多的业务，因为它将产品组合扩展到了核心 Linux 产品之外，进入了利润更高的开源基础设施和应用交付业务，除其他外，还围绕 Cloud Foundry、OpenStack 和 Kubernetes 等大型开源项目提供产品和支持。

Suse 首席执行官 Nils Brauckmann 将继续执掌公司，但公司正在对其高管队伍进行一些调整。例如，Enrica Angelone 被任命为 Suse 的首席财务官，Sander Huyts 现在是该公司的首席运营官。前 Suse 首席技术官托马斯迪·吉亚科莫现在是工程，产品和创新总裁。三个人都直接向 Brauckmann 汇报。

Brauckmann 在今天的声明中表示:“我们真正开放的开源解决方案、灵活的业务实践、没有强制的供应商锁定和卓越的服务对客户和合作伙伴组织来说更加重要，我们的独立性与我们专注于为他们提供最佳服务的专注不谋而合。”。“我们始终如一地满足这些市场需求的能力创造了一个成功、动力和增长的循环，使 SUSE 能够继续提供客户实现数字化转型目标所需的创新，并实现他们推动自身持续创新、竞争力和增长所需的混合和多云工作负载管理。”

由于 IBM 最近以 340 亿美元收购了红帽公司(T2)，Suse 的独立未来能持续多久还有待观察。毕竟，开源市场正在升温。
