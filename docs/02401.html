Iterable 获得 5000 万美元的 C 轮投资，以扩大跨渠道营销平台

帮助公司建立跨渠道的复杂营销活动以减少流失和增加使用的初创公司 Iterable 今天宣布了 5000 万美元的 C 轮融资。

投资者包括 Blue Cloud Ventures、CRV、Harmony Partners、Index Ventures 和 Stereo Capital。今天的投资总额达到了 8000 万美元。

公司联合创始人兼首席执行官 Justin Zhu 表示，Iterable 平台可以从各种来源的消费者那里获取源源不断的数据，使营销人员能够根据消费者的行为建立细分市场或事件触发器。

“客户正在实时更新他们是谁，他们在哪里购买，他们在应用程序中做什么，他们在网站上做什么，我们正在获取所有这些数据并实时提供，”朱解释说。

照片:Iterable

这可以让营销人员根据行为联系人们，例如两周没有打开应用程序的人群。营销人员也可以使用事件触发器来自动联系。在废弃购物车的经典场景中，营销人员可以设置一个触发器，在购物车被废弃两小时后发送电子邮件或 SMS 消息，以提示客户回来。

作为一个平台，Iterable 在一个解决方案中提供了一套工具，营销人员可能需要单独购买。“过去，你通常会做的是拼凑各种单点解决方案。你可能只为手机购买产品，也可能只为电子邮件购买产品。您可能会让工程师拼凑定制代码来处理生命周期管理。有了 Iterable，所有这些都可以在一个地方完成，营销人员也可以完成，这将是他们工作的重点，”朱说。

他说，该公司正在将来自各种数据源的客户数据直接传输给营销人员，因此不涉及与第三方的数据共享。“这是来自我们自己客户的第一方数据，”他说。

该公司报告了三位数的年同比增长率，尽管它不愿透露具体的收入数字。Iterable 有 300 个客户，包括 Box、DoorDash 和 Zillow。它目前有 200 名员工，分布在三个地方，包括位于旧金山的公司总部以及位于丹佛和纽约市的办公室。

朱说，该公司的愿景是成为一家全球性公司，有了这笔资金，它计划在继续建设公司的同时，向欧洲和亚洲扩张。
