微软警告Windows 7用户安全更新即将结束

微软推出了一个补丁，警告Windows 7用户安全更新即将结束。

周三发布的补丁警告用户即将到来的截止日期，2020年1月14日，届时软件巨头将不再推出安全缺陷和漏洞的修复程序。这一最后期限距离Windows 7于2009年首次亮相已经过去了大约10年，比微软最新推出的操作系统Windows 10早了5年多。

微软停止发布安全更新的举动是该公司不断努力推动用户使用其最新软件的一部分，该软件建立在更好的安全基础上，并对减少攻击进行了改进。

从4月18日开始，Windows 7用户将开始收到关于即将截止的警告。

根据Net Applications的数据，Windows 7仍然控制着大约40%的桌面市场。距离截止日期还有整整300天，消费者安全支持的时间越来越紧迫。

企业客户可以选择为2023年前的安全更新付费。

多年来，微软允许Windows 7用户免费升级到Windows 10，以试图鼓励增长和升级。随着这些激励措施的消失，许多人只剩下缺乏安全更新，这将使业务数据和系统面临网络攻击的风险。

微软给生命周期结束的软件打补丁几乎是闻所未闻。2017年，微软为Windows XP发布了罕见的安全补丁——三年前退役——以防止WannaCry的传播，wanna cry是一种勒索软件，它搭载了美国国家安全局开发的泄露黑客工具。

勒索病毒爆发使学校、企业和医院离线。

Windows 7的继任者Windows 8将继续接收更新，直到2023年1月10日。
