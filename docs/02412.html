微软警告 Windows 7 用户安全更新即将结束 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/03/20/windows-7-security-updates/amp/

图片来源: Getty Images

微软推出了一个补丁，警告 Windows 7 用户安全更新即将结束。

周三发布的补丁警告用户即将到来的截止日期，2020 年 1 月 14 日，届时软件巨头将不再推出安全缺陷和漏洞的修复程序。这一最后期限距离 Windows 7 于 2009 年首次亮相已经过去了大约 10 年，比微软最新推出的操作系统 Windows 10 早了 5 年多。

微软停止发布安全更新的举动是该公司不断努力推动用户使用其最新软件的一部分，该软件建立在更好的安全基础上，并对减少攻击进行了改进。

从 4 月 18 日开始，Windows 7 用户将开始收到关于即将截止的警告。

根据 Net Applications 的数据，Windows 7 仍然控制着大约 40%的桌面市场。距离截止日期还有整整 300 天，消费者安全支持的时间越来越紧迫。

企业客户可以选择为 2023 年前的安全更新付费。

多年来，微软允许 Windows 7 用户免费升级到 Windows 10，以试图鼓励增长和升级。随着这些激励措施的消失，许多人只剩下缺乏安全更新可期待，这将使业务数据和系统面临网络攻击的风险。

微软给生命周期结束的软件打补丁几乎是闻所未闻。2017 年，微软为 Windows XP 发布了罕见的安全补丁——三年前退役——以防止 WannaCry 的传播，wanna cry 是一种勒索软件，它搭载了美国国家安全局开发的泄露黑客工具。

勒索病毒爆发使学校、企业和医院离线。

Windows 7 的继任者 Windows 8 将继续接收更新，直到 2023 年 1 月 10 日。
