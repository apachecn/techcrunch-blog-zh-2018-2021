中国女同性恋交友应用Rela曝光了500万用户资料

Rela (热拉), a popular dating app for gay and queer women, has exposed millions of user profiles and private data because a server wasn’t protected with a password.

Rela 于2017年5月从应用商店消失，此前据报道，rela被中国监管机构关闭，尽管政府从未证实其采取了行动。但是据的应用商店列表显示，这款应用一年后在另一家云提供商处回归。LGBTQ+在中国的权利仍然非常有限，尽管它在1997年已经合法化。社区中的许多人仍在与歧视作斗争，人们的态度变化缓慢。

GDI基金会的安全研究员Victor Gevers 告诉TechCrunch，本周发现了被暴露的数据库，其中包含超过530万应用程序用户。

Gevers说，据信该数据库自2018年6月以来一直被暴露，即该应用程序返回后的一个月。

每份记录包括他们的昵称、出生日期、身高和体重、种族、性偏好和兴趣。在用户允许的情况下，记录还包括他们的精确地理位置。该数据库还包含超过2000万个“时刻”，或状态更新，包括私人数据。

“500多万LGBTQ+人群的隐私在中国面临着许多社会挑战，因为没有法律保护他们免受歧视，”Gevers说。“这种多年来一直公开的数据泄露对被曝光的相关人员造成了更大的伤害。”

在一个简短的回应中，公司发言人证实数据库已经得到保护。

同性恋约会应用仍然是一个大生意——即使对中国的公司来说也是如此，尽管法律的复杂性已经导致几个主要应用被关闭。Zank是一款主要由同性恋和双性恋男性使用的流行应用程序，于2017年4月被关闭，理由是政府对传播色情内容的规定。

然而，更多成熟的应用程序，如Blued，在中国仍然很受欢迎。

中国游戏巨头昆仑在2017年购买了美国同性恋约会应用Grindr 60%的股份，后来收购了整个公司，但据报道，由于担心该公司对美国国家安全构成风险，正在出售。
