脸书推出所有活动广告的可搜索透明图书馆——TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/03/28/facebook-ads-library/2048/

现在，你可以在脸书搜索川普过去一年在广告上花了多少钱，哪些页面的广告提到了移民，或者某个页面以前的名字是什么。这是脸书新广告库的一部分，今天推出，兑现了社交网络的广告被用来试图影响2016年美国总统选举后增加透明度的承诺。

2018年5月推出的脸书广告档案库以前只包括与政治或政策问题相关的广告，但现在显示所有关于任何事情的活跃广告，以及不活跃的政治和问题广告。它显示页面创建日期、与其他页面的合并、页面名称更改和页面管理位置，以及报告广告违反策略的选项，所有这些都将在所有页面的新页面透明度选项卡上可见。用户可以通过关键词搜索政治和发布广告，或者通过页面名称搜索其他广告，脸书将提供自动填充建议和以前的搜索。该库还显示Pages在过去一周或自2018年5月以来的总政治广告。五月中旬，脸书将开始提供每日可下载的广告库报告，以及月度和季度报告，而不是像现在这样每周一次。

透明度的提高可以让研究人员、政府调查人员、记者或任何发挥监督作用的人确保广告不会被滥用来传播错误信息、压制投票、加剧两极分化或以其他方式进行粗略的描述。

脸书没有把所有过期的和不活跃的非政治广告存档。它没有说明原因，但一些广告商表示担心，这可能会让竞争对手复制他们的信息和目标战略。另一个例外是新闻出版商，他们联合起来抱怨在他们的广告上贴上“付费”的标签来宣传他们的政治报道，使他们看起来像是在为某一方竞选。脸书在11月豁免了英国出版商，现在脸书将使用合法美国出版商的第三方列表，将他们的广告从广告库的标签和政治标签中豁免。

继10月份在英国推出之后，脸书也将把它的政治广告标签带到整个欧盟。政治和发行广告买家将必须提交文件并通过技术检查，以验证他们的位置和身份，这些文件将由自动系统进行审查，并由用户进行审计，这些用户可以报告没有适当标签的广告。目的是防止国外的人利用广告干涉不同国家的选举。脸书还将在脸书和Instagram上的所有政治和问题广告上添加“付费”标签，并将活跃和不活跃的广告放在广告库中。标签将审查谁买了广告，他们的联系方式，广告的预算，有多少人看到了它，年龄，地点和性别的人口统计资料。

脸书注意到，在巴西、T2、乌克兰和以色列准备选举的时候，它已经有了透明度工具。脸书计划在6月底之前在全球范围内提供政治和问题广告透明度工具。

与此同时，脸书正在扩大研究人员和开发人员对Ad Library API的编程访问，该API此前处于封闭测试阶段。现在，任何拥有脸书开发者账户的人，只要通过身份确认程序并同意平台服务条款，就可以使用软件来筛选和发现数据趋势。为了防止另一个剑桥分析的情况，脸书告诉TechCrunch，它将对API施加费率限制，但不会公开它们，以确保不良行为者不会遵守规则。当被问及鉴于人们不一定遵守服务条款，脸书还会如何保护API时，脸书商业诚信团队的产品经理Satwik Shukla告诉我，“这就是我们实施身份验证流程的原因。我们想知道访问数据的人的身份，如果他们滥用我们的平台服务条款。。。我们有能力撤销访问权限。”

一个有趣的举动是，即使你没有脸书账户，你也可以访问广告库和报告。这可能会避开一些批评者和国会议员，他们对脸书要求用户注册才能获取透明信息或阻止其收集用户数据感到不满。

在访问脸书广告数据的深度和便利性之间，监管机构可以在影响太多选民之前发现有问题的竞选活动。如果有人发现一场运动试图说服人们可以通过短信投票，教皇支持川普，或者故意煽动愤怒以促进参与，他们可以将广告标记给脸书删除。

正如今天美国住房和城市发展部指控脸书住房歧视所证明的，脸书多次未能自行阻止有问题的广告。通过向援助敞开大门，它可能不会减轻审查，但至少所有的目光都可以帮助它对抗选举干预和其他滥用。
