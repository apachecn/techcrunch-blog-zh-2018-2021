DEA称美国电话电报公司仍然提供数十亿电话记录

原文：https://web.archive.org/web/https://techcrunch.com/2019/03/28/hemisphere-phone-records/

根据一份监督报告，一个允许药物代理人从美国电话电报公司获得数十亿通话记录的项目“仍然活跃”。

司法部监察长周四(T1)发布的这份报告证实，该项目——在报告中仅被命名为项目C——继续向缉毒署特工提供“数十亿”国内和国际通话记录。该程序允许代理人从运行该程序的电信提供商那里获取关于呼叫者以及何时何地打电话的信息，而无需法院命令。

项目C——内部称为半球——的细节于2013年首次在披露，几乎是在该项目于2007年开始后的十年。根据监管机构的报告，电信提供商——据说是AT&T——“维护和分析”自己收集的大量电话记录数据，并允许访问几个执法机构。

据说，Hemisphere收集的电话记录比美国国家安全局(National Security Agency)一度备受争议的电话记录收集计划还要多，该计划最初由告密者爱德华·斯诺登(Edward Snowden)披露。美国国家安全局的项目在2017年收集了约5亿条通话记录，但在去年被有效关闭。

电子前沿基金会根据信息自由法诉讼获得的一份文件，详细说明了美国电话电报公司半球项目的范围

美国电话电报公司运行的程序收集通过其系统的每个呼叫的信息，包括其他手机运营商和网络提供商的信息。据报道，该数据库包含了30年来的通话记录，包括呼叫者的位置。它还提供IMSI和IMEI号码来识别呼叫者。

但司法部的监管机构表示，该项目带来了法律问题，因为代理人可能需要比机构发出的传票更高的权限才能访问数据，而机构发出的传票不需要司法监督。

司法部副监察长比尔·布利尔(Bill Blier)表示，DEA“在使用其行政传票权之前，未能进行全面的法律分析。”

监督机构建议美国药品管理局就该项目的许可用途发表法律意见。

执法部门被指控使用数据库进行“平行构建”，这是一种允许警方发现潜在犯罪行为，但反向工程他们的发现以防止数据库公开披露的方式。但监察机构发现，使用平行结构来重新创建用于法庭文件的信息“没有什么固有的不当之处”。

美国电话电报公司表示，像所有公司一样，“法律要求它通过遵守法院命令、传票、合法查询请求和其他法律要求，向政府和执法机构提供信息。”该公司补充说:“当一个人的生命受到威胁或处于其他直接的紧急情况下，我们会向执法部门提供自愿援助，”并且:“我们确保援助请求是有效的，我们的行动符合法律。”

美国药品管理局发言人凯瑟琳·普法夫告诉TechCrunch，该机构“同意”监督机构的建议，“已经开始加强这些程序。”

该机构拒绝透露电信提供商的名称。
