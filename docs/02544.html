微软给创业公司 500 项专利

原文：https://web.archive.org/web/https://techcrunch.com/2019/03/28/microsoft-gives-500-patents-to-startups/

微软今天宣布了其 Azure IP Advantage 计划的重大扩展，该计划为其 Azure 用户提供保护，防止专利流氓。该计划现在还为正在构建连接到 Azure 的物联网解决方案的客户提供 10，000 项专利，以保护自己免受知识产权诉讼。

然而，最有趣的是，微软还向物联网中的初创公司捐赠了 500 项专利。该组织拥有近 400 名成员，包括亚马逊、脸书、谷歌、微软、网飞、SAP、Epic Games、福特、通用汽车、Lyft 和优步等公司，旨在通过向公司提供其成员公司和其他来源的大量专利库来保护公司免受专利流氓的攻击。

然而，这个新项目远远超出了基本的专利保护。加入 LOT 网络的合格初创公司可以获得微软专利，作为其免费会员的一部分，正如安德森强调的那样，初创公司将完全拥有这些专利。LOT 网络将能够向其初创成员提供该集合中最多三项专利。

不过，这里有一个额外的要求:为了有资格获得专利，这些初创公司还必须满足每月 1000 美元的 Azure 支出。不过，正如安德森告诉我的那样，他们不必做出任何形式的预先承诺。该公司只会查看一家初创公司最近三个月的 Azure 账单。

安德森说:“我们希望帮助 LOT 网络扩大其创业网络。“为了提供激励，我们将向他们提供这些专利。”他指出，初创公司显然对获得专利感兴趣，这是他们公司的基础，但也是为了筹集资本和抵御网络钓鱼。

我们在这里谈论的专利涵盖了广泛的技术和地域。安德森指出，例如，我们谈论的是美国专利以及欧洲和中国的专利。

“这个想法是，这些创业公司来自不同的行业，”他说。“我们的希望是，当他们接近 LOT 时，他们会在这 500 项专利中发现基本上任何一家公司都会感兴趣的专利，这些公司可能会为他们的业务需要一套基础专利。”

至于扩展的 Azure IP Advantage 计划，值得注意的是，每个在过去三个月每月花费超过 1000 美元并且在过去两年中没有对另一个 Azure 客户提起专利侵权诉讼的 Azure 客户都可以自动选择该计划组合中的一项专利，以保护自己免受来自 trolls 的无聊专利诉讼(这是一个不同于微软作为启动计划的一部分捐赠给 LOT Network 的专利库)。

正如 Andersen 所指出的，该团队着眼于如何通过关注一些特定领域来加强知识产权计划。微软显然正在向物联网投入大量资金，因此将该计划扩展到这一领域是有意义的。“我们基本上要说的是，如果客户正在使用物联网技术——不管它是否是微软的技术——并且它与 Azure 相连，那么我们将提供这项专利选择权，以帮助客户在专利诉讼中保护自己，”安德森说。

此外，对于那些选择全面使用微软物联网技术的人，微软也将提供补偿。

专利钓鱼者最近开始收购物联网专利，因此他们有可能准备利用这些专利，我们将在未来看到这一领域的相当多的专利诉讼。“我们看到的早期迹象表明，这是客户未来会关心的事情，”安德森说。
