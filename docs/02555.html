索尼的流媒体服务 Crackle 向心灵鸡汤出售多数股权

原文：https://web.archive.org/web/https://techcrunch.com/2019/03/29/sonys-streaming-service-crackle-sells-to-chicken-soup-for-the-soul/

裂纹卖给心灵鸡汤。这不是一个早期的愚人节玩笑，而是一个新闻，索尼的老牌流媒体服务 Crackle 现在已经为 Soul Entertainment(或 CSS Entertainment)找到了一个新家，这家公司现在运营流媒体服务 Popcornflix 和其他流媒体频道。(而且没错，也是知名丛书的背后。)

这个消息是首先由综艺报道的，在今天早上发布的新闻稿中有更详细的报道。

CSS Entertainment 现在将成为新合资公司的大股东，新公司更名为“Crackle Plus”(索尼并没有完全退出。)与此同时，Variety 的报告称，索尼将获得 400 万份五年期认股权证，用于购买 CSS Entertainment 的 A 类普通股。

索尼在 2006 年首次收购了 Crackle，使其成为市场上较早的免费、广告支持的流媒体服务之一。然而，在来自网飞等网站的竞争中，以及它和其他公司——如 Hulu、亚马逊和现在的苹果——在原创内容上的数十亿投资中，它一直难以增长。这导致索尼去年宣布将寻求新的战略合作伙伴来帮助其运营 Crackle。

然而，Crackle 所有权的转移发生在像这样的无广告流媒体服务受到新的关注的时候，亚马逊推出了 IMDb 的 FreeDive，Roku 的 Roku Channel，沃尔玛的 Vudu，维亚康姆的新成员 Pluto，Tubi 和其他服务现在都在盈利。

根据好莱坞报道的说法，作为交易的一部分，索尼将向新公司贡献其美国资产，包括 Crackle 品牌、用户群和广告代理业务。它还将授权索尼电影娱乐公司(Sony Pictures Entertainment library)的电影和电视节目，以及 Crackle 的原创节目，比如它的节目《创业》(Start Up)和《誓言》(The Oath)。

CSS Entertainment 将为 Crackle Plus 带来六个广告支持的网络，包括 Popcornflix、Popcornflix Kids、Popcornflix Comedy、Frightpix、Espanolflix 和 Truli，以及其订阅服务 Pivotshare。

两家公司声称，这一合并将使 Crackle Plus 成为美国最大的广告支持的视频点播平台之一，每月活跃用户近 1000 万，注册用户 2600 万。这项新服务还将获得超过 38，000 小时的节目，90 多个内容合作伙伴和 100 多个网络。

“我们的合资公司将把 Crackle Plus 定位为领先的 AVOD 流媒体平台，在我们自有和运营的网络上拥有近 1000 万活跃用户。CSS 娱乐公司董事长兼首席执行官 William J. Rouhana，Jr .在一份声明中说:“这将使我们的在线网络经常性收入成倍增长。“我们计划通过有机增长和收购，积极打造 Crackle Plus，并从中获利。”

通用汽车和索尼首席数字官埃里克·博格将在交易完成后退出。
