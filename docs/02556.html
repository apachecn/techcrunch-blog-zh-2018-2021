索尼的流媒体服务Crackle向心灵鸡汤出售多数股权

原文：https://web.archive.org/web/https://techcrunch.com/2019/03/29/sonys-streaming-service-crackle-sells-to-chicken-soup-for-the-soul/amp/

【交易时间】爆炸头卖给心灵鸡汤。这可不是四月初的“傻瓜”笑话，而是索尼老化的流媒体服务Crackle最近推出了“心灵娱乐鸡羹”的喜讯——这家公司如今运营着流媒体服务pop矢车菊和其他流媒体频道。(而且没错，背后也是知名的丛书。)

这则消息最初由《综艺》杂志(Variety)在上报道于，并在今天上午发布的新闻发布会上进一步详述。

CSS Entertainment将成为新合资公司的大股东，公司更名为“Crackle Plus”。(索尼并没有完全退出。)与此同时，综艺娱乐的报告称，索尼将获得400万份五年期认股权证，可以购买CSS Entertainment的A类普通股。

索尼于2006年首次收购Crackle，使其成为市场上较早的免费广告支持的流媒体服务之一。然而，在来自网飞等网站的竞争以及it和Hulu、亚马逊(Amazon)以及现在的苹果(Apple)等公司在原创内容上投入的数十亿资金的夹击下，它的增长步履维艰。这促使索尼在去年宣布将寻求新的战略合作伙伴来帮助其运营Crackle。

不过，Crackle的所有权转让适逢此类无广告流媒体服务引起新的兴趣之时，亚马逊推出的IMDb FreeDive、Roku的The Roku Channel、沃尔玛的Vudu、维亚康姆新加入的Pluto、Tubi等服务现在都在盈利。

根据好莱坞报道的说法，作为这笔交易的一部分，索尼将利用其在美国的资产，包括Crackle品牌、用户基础和广告代表业务，为新合资公司做出贡献。它还将获得索尼影视娱乐公司(Sony Pictures Entertainment)资料库中Crackle Plus电影和电视节目的授权，以及Crackle原创节目的授权，比如它的节目《Start Up》和《誓言》。

CSS Entertainment将把它的六个广告支持的网络，包括pop矢车菊、pop矢车菊Kids、pop矢车菊喜剧、superpix、Espanolflix和Truli，以及它的订阅服务Pivotshare，都带到Crackle Plus。

两家公司声称，这一合并将使Crackle Plus成为美国最大的广告支持的视频点播平台之一，每月活跃用户近1000万，注册用户2600万。这项新服务还将获得超过38，000小时的节目，90多个内容合作伙伴和100多个网络。

“我们的合资公司将把Crackle Plus定位为领先的AVOD流媒体平台，在我们自有和运营的网络上拥有近1000万活跃用户。CSS娱乐公司董事长兼首席执行官William J. Rouhana，Jr .在一份声明中说:“这将使我们的在线网络经常性收入成倍增长。“我们计划通过有机增长和收购，积极打造Crackle Plus，并从中获利。”

通用汽车和索尼首席数字官埃里克·博格将在交易完成后退出。
