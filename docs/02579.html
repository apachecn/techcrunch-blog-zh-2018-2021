WordPress.com 母公司推出工作协作平台快乐工具

WordPress.com、WooCommerce 和 Jetpack 背后的 Automattic 公司正在推出一套新的专注于未来工作的产品——快乐工具。Automattic 是一家偏远的公司，有 850 多名员工在 68 个国家工作。多年来，该公司已经开发了一系列产品，用于交流、合作和工作。

通过 Happy Tools，Automattic 计划将这些内部工具转化为实际产品。第一个产品是 Happy Schedule ，Automattic 用它来提供 24/7 的客户支持。

Happy Tools 的 automatic 产品负责人马特·万德拉(Matt Wondra)告诉我说:“关于发布我们内部工具的想法已经在 automatic 周围踢了好几年，但它一直是关于找到合适的时机和合适的产品来主导。”“当我们一年前开始构建 Happy Schedule 时，我们意识到，为我们自己的日程安排需求设计一个工具，也填补了(劳动力管理)领域的一个明显空白。”

“没有任何其他产品能够为我们提供轻松安排全球分布团队所需的灵活性和可见性。因为这是一个全新的内部项目，我们可以从头开始设计，并考虑公开发布。首先在我们非常熟悉的行业——客户支持——推出 Happy Tools 是有意义的。”

快乐日程是一个现代的网络应用程序，它应该感觉更像谷歌日历，而不是一些 SAP 产品。例如，您可以单击并拖动鼠标来创建一个事件，而无需输入开始时间和结束时间。

但这仅仅是一个开始。Automattic 计划随着时间的推移推出更多产品，以便您可以作为远程团队更高效地工作。该公司正在使用软件即服务的方法，每个用户每月花费 5 美元来访问 Happy Tools。

有趣的是，Automattic 从第一天起就承诺推出一套产品。它不会只是一堆不同的产品。当你订阅 Happy Tools 时，你应该能够访问多个协同工作的产品，就像 G Suite 订阅可以让你访问 Gmail、Google Calendar、Google Drive 等。随着时间的推移，这一战略将提高参与度和粘性。
