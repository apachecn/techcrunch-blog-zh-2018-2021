阿卡特兹人工智能正在为企业徽章打造 Face ID

原文：https://web.archive.org/web/https://techcrunch.com/2019/04/02/alcatraz-ai-is-building-face-id-for-corporate-badges/

阿卡特兹人工智能正在为企业徽章打造 Face ID

来认识一下阿卡特兹人工智能公司，它是一家初创公司，想要用一个类似 Face ID 的摄像系统取代你办公室里所有的工卡阅读器。阿卡特兹集成了多种传感器，可以毫不费力地识别人脸和开门。

如果你仔细想想，指纹传感器在移动设备上大受欢迎，但每个人仍然在办公室使用塑料徽章，这很奇怪。当然，高度安全的建筑使用指纹和虹膜扫描仪。但在很多情况下，它增加了太多的摩擦。

首先，当每个人都从午休回来时，如果每个人都需要将手指放在传感器上，就会造成交通堵塞。其次，新员工入职时，您需要将他们的生物特征信息添加到系统中。对大公司来说可能很麻烦。

阿卡特兹人工智能承诺通过面部认证提供更快的认证体验。当你加入一家公司时，你也会得到一个实体徽章。你第一次使用徽章时，阿卡特兹人工智能会扫描你的脸，为未来的使用创建一个模型——过一段时间后，你可以把你的徽章留在办公室。

该公司已经建立了带有三种不同传感器的定制硬件，包括传统的 RGB 传感器和用于 3D 绘图的红外传感器。客户付钱给阿卡特兹人工智能安装这些混合徽章/人脸阅读器。之后，公司需要支付年费才能使用该平台。

阿卡特兹人工智能的客户可以获得分析、实时通知，并可以检测尾随情况。这样，如果有人不应该进入秘密实验室，阿卡特兹人工智能可以通过跟踪授权进入那里的人来检测他们是否试图潜入。

这个想法是，持续的许可成本应该包括贵公司支付的保安费用。这家初创公司已经从 Hardware Club，Ray Stata，JCI 风险投资公司，Ruvento Ventures 和 Hemi Ventures 筹集了近 600 万美元。
