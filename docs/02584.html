亚利桑那州饮料被勒索软件攻击下线

原文：https://web.archive.org/web/https://techcrunch.com/2019/04/02/arizona-beverages-ransomware/

TechCrunch 获悉，美国最大的饮料供应商之一亚利桑那饮料公司(Arizona Beverages)在上个月遭受大规模勒索软件攻击后正在恢复。

据一位知情人士透露，以冰茶饮料闻名的该公司在袭击发生近两周后仍在重建其网络，清除了数百台 Windows 电脑和服务器，并实际上关闭了销售业务数天，直到事件响应中心接到电话。

200 多台服务器和联网计算机显示了同样的信息:“你的网络被黑客攻击并加密了。”勒索信中有该公司的名字，表明这是一次有针对性的袭击。

办公室里张贴的通知告诉员工将他们的笔记本电脑交给 IT 人员。海报上写着:“不要开机，不要复制文件，不要连接任何网络。”。“你的笔记本电脑可能受到了威胁。”

该消息人士称，该公司又花了五天时间才派出事故应对人员来处理疫情。许多后端服务器运行的是不再受支持的旧的和过时的 Windows 操作系统。大多数人已经多年没有收到安全补丁了。

该消息人士称，考虑到他们系统的年龄，他们对攻击没有来得更快感到“惊讶”。

攻击发生一天后，员工发现备份系统配置不正确，几天内无法检索数据，直到该公司签署了一份昂贵的合同，聘请思科事件响应人员。思科发言人没有立即置评。该公司的 IT 人员不得不从头开始有效地重建整个网络。自疫情爆发以来，该公司已经在新的硬件、软件和恢复成本上花费了“数十万”。

“一旦备份不起作用，他们就开始砸钱解决问题，”这位人士说。

根据 TechCrunch 看到的一张截图，勒索病毒感染被理解为 iEncrypt(与 BitPaymer 相关)，于 3 月 21 日夜间触发，几周前，联邦调查局联系亚利桑那州，警告称有明显的 Dridex 恶意软件感染。联邦调查局拒绝置评，但消息人士称，事件响应者认为亚利桑那州的系统至少已经被入侵了几个月。

勒索信要求向攻击者发送电子邮件，“以获取赎金金额。”没有已知的 iEncrypt 解密工具。

Dridex 通过恶意电子邮件附件发送。一旦植入安装，攻击者可以获得几乎不受限制的访问整个网络，并可以窃取密码，监控网络流量和提供额外的恶意软件。在国际合作伙伴的帮助下，美国联邦调查局于 2015 年摧毁了窃取密码的僵尸网络，但该恶意软件继续对构成威胁。最近，Dridex 被用来向受害者发送勒索软件。

卡巴斯基表示，两年后的今天，这个恶意软件“仍然是武装的和危险的”

事件响应者似乎认为亚利桑那州早期的 Dridex 妥协可能导致了随后的勒索病毒感染。

安全公司 CrowdStrike 的情报副总裁亚当·梅耶斯(Adam Meyers)说:“最初，Dridex 被用来窃取凭据，以进行网络欺诈，但自 2017 年以来，人们更经常看到它进行更有针对性和更高价值的操作。”他说，该公司已经“观察到这种恶意软件被用来部署企业勒索软件，我们称之为‘大型游戏狩猎’。”"

该勒索软件还感染了该公司基于 Windows 的 Exchange 服务器，导致整个公司的电子邮件瘫痪。尽管其 Unix 系统没有受到影响，但勒索软件的爆发让该公司几乎一周没有任何计算机能够处理客户订单。停机几天后，员工开始手动处理订单。

“我们每天损失数百万美元的销售额，”消息人士说。"这完全是一场闹剧。"

该公司仍有办法从勒索软件攻击中恢复过来。该消息人士称该数字为“大约 60%的启动和运行”，但该公司的安全意识已经提高。

亚利桑那州饮料公司的发言人没有回复要求置评的电子邮件。该公司的电话线路似乎无法正常工作。在文章发表之前，我们通过 LinkedIn 向高管们发送了几条信息，但没有得到回复。

这是最近几周备受瞩目的勒索事件中的最新一起。

去年，德国制造商 KrausMaffei 据说也在 11 月 21 日遭到了相同的 iEncrypt 勒索软件的攻击，这是基于一张泄露的勒索信截图。类似的最初勒索病毒感染与后来的勒索病毒攻击有关。趋势科技去年 12 月表示，Dridex 和 Emotet 等其他恶意软件家族存在关联。在亚利桑那州疫情爆发的几周前，佐治亚州的一个地方县遭到类似勒索软件攻击。
