福特将在重新设计的 2020 款 Escape SUV 中提供混合动力和电动选项 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/04/02/ford-to-offer-hybrid-and-electric-options-in-redesigned-2020-escape-suv/

福特今天推出了最新一代的 Escape。这是一个从里到外的重大改造，采用了新的钣金和动力系统选项。2020 Escape 更具运动感，搭载了多项技术，并提供混合动力和电动选项，显然是为未来打造的。

自 2000 年首次亮相以来，Escape 一直是福特最实惠的 SUV。比健壮的探险者更小，比 Edge 或 Flex 更传统，福特一直将 Escape 定位为一款简单易用的运动型多功能车。即将推出的版本延续了这一传统，同时为买家提供了新的选择。

2020 Escape 有几个版本。基本型号配备 1.5 升 EcoBoost 发动机，或者买家可以升级到 2.0 升 EcoBoost 发动机。福特还提供传统混合动力配置的 Escape，并首次提供插电式混合动力设置。但买家将不得不等待插入式版本。这种装饰水平将于明年春天上市，而该系列的其余产品将于今年秋天上市。

定价尚未公布。目前的 Escape 零售价在 25000 美元到 30000 美元之间。

练习场很有竞争力。福特表示，配备标准混合动力的前轮驱动 Escape 的 EPA 估计行驶里程为 550 英里。这款插电式混合动力车预计仅依靠电力就能行驶 30 英里，而其 11 加仑的油箱确保它在需要停车之前仍能行驶 400 多英里。

这是混合动力系统中提供的第二代 Escape，但却是第一代插电式车型。福特在 2004 年至 2012 年期间销售了 114，000 辆混合动力逸动，但放弃了 2012 年推出的当前车型系列的选项。

福特对混合动力车型有着远大的抱负。2018 年，该汽车制造商表示，它将全力以赴开发混合动力 SUV，同时退出传统汽车。到 2020 年，福特的目标是在市场上推出高性能 SUV，包括五款混合动力车型和一款全电池电动车型。这包括将于今年夏天上市的 2020 款福特探险者混合动力车，以及两款全新的越野 SUV，包括一款新的 Bronco 和一款尚未命名的小型 SUV。此外，“高性能电池电动工具”将构成其整体 SUV 阵容的一部分，该系列将于 2020 年发布，并将率先计划到 2022 年发布六款电动汽车。为了支持这一努力，该汽车制造商计划在第二家美国工厂增加产能，用于其下一代电池电动汽车项目。

Escape 有前轮驱动和全轮驱动两种配置。不过，目前插电式混合动力汽车将只提供前轮驱动，而配备 250 马力 2.0 升 EcoBoost 的车型将只配备全轮驱动。

福特表示，新款逸动拥有一流的第二排腿部空间，带滑动后座。虽然它偷了一点存储空间，但这个功能应该会让后座对乘客来说更舒适一些。在混合动力车型中，电池组位于后座下方，可以防止座椅滑动——即使如此，通过将电池放在座椅下方，它也不会占用车辆后部宝贵的存储空间。

福特确保 Escape 配备了各种标准和可选的驾驶辅助功能。标准套件包括福特所谓的 Co-Pilot360，它包括自动紧急制动、后视摄像头和盲点监测系统。其他系统是可选的:停车辅助，抬头显示和自适应巡航控制。

八英寸触摸屏是 SE 车型的标准配置，采用福特的 Sync 3 系统。

外出的逃亡者需要彻底检查。与其最直接的竞争对手相比，汽车评论经常指出当前的福特锐界不足。大多数评论都指向像本田 CR-V、丰田 RAV4 和马自达 CX-5 这样的选择，声称它们给司机提供了更宽敞的驾驶室和生动的动力系统。福特在 2012 年发布了第三代 Escape，并在 2017 年因销量开始下滑而更新了外观。

福特对其 SUV 有很大的计划。2018 年初，它宣布取消所有轿车的生产，留下野马作为它在北美销售的唯一传统汽车。该汽车制造商将转而转向跨界车和 SUV，以取代 Fusion、金牛座、嘉年华和福克斯。多年来，传统汽车的销量一直在下降，福特显然认为它需要抓住最新趋势，提供大多数买家想要的东西。

2020 年福特 Escape 是福特最小的 SUV 的一次显著飞跃。有了新的混合动力选项和更大的内部空间，它应该会引起那些寻找环保的人力资源的购物者的共鸣。
