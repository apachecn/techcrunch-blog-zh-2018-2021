日本的新帝国时代如何以微小的方式打破互联网 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/04/02/how-japans-new-imperial-era-broke-the-internet-in-a-very-tiny-way/

日本明仁天皇退位，将王位传给他的儿子皇太子德仁，作为交接的一部分，出现了一个新的 gengo，或时代名称:“Reiwa。”它承载着意义和潜台词，但由于文本在互联网上发送和显示的方式，这个名字实际上无法以某种官方方式显示。Unicode 已经发布了一个更新，使之成为可能，但目前还没有 Reiwa 的字符。这是一个奇怪的小问题，很快就会得到解决，但它说明了我们所依赖的系统并不完美。

Now, to be clear, you can definitely type out the kanji for it, 令和, and that’s totally fine (though there are some technical hiccups there too). But gengo get combined single characters for certain situations and contexts. For comparison, the current (soon to be former) era, Heisei, can be written 平成 but has a combined-character name as well: ㍻. Sure, it’s just the two pieces squished together, but they’re squished together in an important, official way.

在不深入日本帝国的历史或政治的情况下，这个国家有一种内部日历，其时代对应于某个特定皇帝的统治(主要是象征性的)。这是一件大事，用于官方文件等，但也是指一个时代的简写——“哦，那是在平成十年”或类似的话，我们可以说，“克林顿总统任期的最后一年。”我可能做错了，但是你明白了。

关于这种做法有趣的部分——嗯，有很多有趣的部分，但对我们今天的目的来说最有趣的是，这些 gengo 基本上都是从零开始发明的。由于日语的书写方式，这意味着根本没有办法在网上正确显示这个名字！

Unicode 基本上是一个很大的(大约 120，000 个条目长)代码集，对应于我们想要键入或发送的实际字母和图像。所以当我写“嘿！🙋“你在你那边看到的和我在我这边看到的一样。但是，对于一个在正式亮相之前一直保密的事物，怎么会有一个符号呢？这个符号是由其他日语单词的一部分组合成一个全新的字符。这有点像宣布他们在字母表中增加了一个新字母。

这是现代计算时代第一次发生时代变革，所以真的没有任何先例，正如该组织去年在预测这一事件时指出的那样。Unicode 真正能够做的唯一事情是为输入字符保留空间，并给它一个显示其代码的占位符图形:㋿.(假设您在添加字符之前看到这个框，您在框中看到的是“32FF”，是代码的一部分。)

在某种程度上，这就是 Unicode 需要做的一切；现在由设计者来组合实际的字符，因为它将以他们的字体显示。这些更新必须分别发送到服务器和设备。当然，这不可能马上实现——与此同时，任何带有新名字的网上标题都不能显示出来！这是一个有趣的小难题。

至于名字本身，当人们争论语言学和文学潜台词时，它引起了一些骚动；日本文字的复杂性和音节的历史以及它们是如何表现的，使得各种各样的解释成为可能，具有不同的政治和哲学内涵。Nick Kapur 的分析是我所见过的最好的分析，它触及了 Reiwa 难以翻译的核心，同时也清楚地表达了它的含义。

从很多方面来说，这是一个独特的挑战，而互联网是唯一不适合的挑战。很快我们就有了这个角色，由发布和正确显示它的能力之间的延迟造成的任何轻微不便都将被遗忘，但当世界向我们认为理所当然的系统扔出一个弧线球时，这总是很有趣的。

希望在 5 月 1 日 Reiwa 时代正式开始时，我们会把一切都整理好。
