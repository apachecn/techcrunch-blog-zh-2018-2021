Red Points 完成了价值 3800 万美元的 C 系列产品，为品牌提供“防病毒”SaaS

总部位于巴塞罗那的 Red Points 已经完成了由 Summit Partners 牵头的 3800 万美元的 C 轮融资。现有投资者 Northzone、Mangrove、Eight Roads Ventures 和 Banco Sabadell 也参与了这轮融资，这是继去年1200 万美元 B 轮之后的又一轮融资。

该公司的“软件即服务”平台针对那些希望保护产品及其经销商网络免受廉价在线假货和/或盗版侵害的品牌，Red Points 的品牌保护技术涵盖了实体品牌商品和受版权保护的数字内容。

它使用市场扫描和机器学习支持的图像识别来自动检测和发送侵权通知，能够大规模发现潜在的假货，并结合可定制的规则，让品牌优先考虑和自动执行行动。

“从检测到执行，整个过程都是自动化的，”首席执行官 Laura Urquizu 告诉我们。“对于我们帮助的公司，我们通常会说我们就像一个杀毒软件。因为我们在那里，造假者转移到其他品牌、其他公司、其他地方。因此，为了保持清洁，用一种行为——或者像我们这样的技术——来对抗是非常重要的，这样才能让互联网、市场和社交网络尽可能地清洁。”

在一份支持性声明中，Summit Partners 的负责人 Steffan Peyer 补充说:“世界各地的品牌正面临着前所未有的在线知识产权侵权上升。 Red Points 是一个类别领导者，提供一个智能和强大的技术平台，旨在解决这一日益增长的难题 point 。凭借与众不同的技术和强大的领导团队，我们相信RedPointsPoints能够加速发展，进一步巩固其在品牌智能市场的领导地位。”

检测和处理网上销售的假货仍然是 Red Points 业务的最大份额，约占 70%,而处理数字版权侵权占 30%。“我们从数字内容开始，然后我们真的做出了一个非常有趣的决定，在品牌保护和假冒产品方面投入了大量精力。这是我们开发的平台非常强大的地方，”Urquizu 说。

她指出，例如，该软件帮助品牌追踪卖家，以突出与非法销售相关的邻近问题，如特定地区的定价过低。

围绕知识产权威胁的分析是 SaaS 提案的另一个核心要素。“另一个非常重要的功能是我们能够带给客户的所有数据。他们自己的数据和我们产生的数据与垂直市场相关，与其他客户相关等。

“客户将这种知识产权‘威胁情报’用于各种用途——例如查看大多数假冒行为发生在哪里，假货在哪些平台上分销。通过这种方式，他们还可以实施自己与特定平台或特定国家相关的战略。他们也知道谁是最大的侵权卖家，所以他们也可以对他们采取法律行动。”

“假冒无处不在，”Urquizu 在讨论品牌在线假货问题的规模时补充道。“所有平台、所有社交网络都受到了影响。当然，当某样东西流行时，它会受到影响。我们在提供非常新潮的新玩意儿的品牌中看到了这一点。他们很快就发现了造假的问题，随着平台和社交网络越来越流行，越来越适合造假者把它当作游乐场。”

Red Points 现在每月为 550 多个客户检测超过 500，000 起非法产品和/或内容事件。去年我们与 Red Points 进行交流时，客户数量超过了 300 家。

客户包括时装、设计、玩具和配件等垂直行业的品牌商品制造商，以及数字知识产权方面的媒体公司。它的客户群包括 Bang & Olufsen、MVMT 和 DOPE 等。

新增长的垂直业务包括电子和工业机械。“目前，我们平均每月新增 40 名客户，”Urquizu 补充道。

在执法方面，她表示，该平台每月代表其付费客户采取“数十万次”执法行动。(一年前，它报告每月删除 20 万多条，因此它似乎不再打破这一指标。)

总体而言，2018 年的业务增长率超过 100%，美国在一年多前开设纽约办事处后占了很大一部分。Urquizu 说，这个市场现在约占其业务的一半。

C 轮融资的一大重点是扩大在美国的销售和营销。“我们的增长很大一部分来自美国，我们才刚刚开始，所以我们想做的是增加我们的存在和足迹。”

即便如此，Urquizu 表示，该公司没有将总部迁出欧洲的计划。“美国办事处将会发展壮大。它在增长…我们去年才开始，那是三月/四月。现在我们在美国已经有 20 多人了。我们今年将继续在那里发展……但我们认为，对我们来说，技术和产品团队可以在巴塞罗那过得很好，因为这里人才济济。我们发现，从这里开始，我们真的可以很好地为世界其他地方服务。”

“每个月我们都在非常非常好地成长，”她补充道。“现在我们的收入已经远远超过 1000 万美元，所以我希望这些收入每年都能翻倍。”

新的资金也将用于更多的 R&D。或者像 Urquizu 所说的那样，“推进我们所做的一切，发现我们可以解决的新问题，为客户带来更多的数据和见解”。

在竞争方面，她不情愿地提到了美国老牌公司 mark monitor T1，说它提供一些相同的服务，同时认为方法有很大的不同。

“我们向客户、市场和公司提供的技术是非常新的、非常创新的，与现有的任何技术都非常不同，”她声称。“一个非常不同的事情是，我们已经能够使用机器学习和人工智能来自动化从检测到执行的过程。”

当被问及一项有争议的数字版权改革提案时，Urquizu 并不担心，该提案上周由欧洲议会批准，将用户侵犯版权的责任转移到主要平台。

她认为，人们对盗版和假冒问题的意识日益增强，对于一个从公司付钱让假货下架中赚钱的企业来说，这是个好消息。

“我们认为，这种新的立法通常需要很长时间才能产生影响，所以我们将在几年后或多或少地看到真正的影响，”她建议道，然后补充道:“我们总是把这种事情视为我们的一个机会。因为目前在这个问题上有太多可以做的事情——也有助于我们提高品牌和公司的意识，我们总是认为这是非常非常积极的。

“这些事情真的支持了我们的工作，因为它让公司和品牌意识到必须做些什么。因此，我们总是与平台和社交网络有着非常大的合作，所以对我们来说，我们总是把它视为共同发展的机会。”
