新兴市场贷款机构 Branch International 与 Visa 合作，融资 1.7 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2019/04/08/partnering-with-visa-emerging-market-lender-branch-international-raises-170-million/

总部位于旧金山的初创公司 Branch International，在新兴市场提供小额个人贷款，已经筹集了 1.7 亿美元，并宣布与 Visa 合作，为非洲、南亚和拉丁美洲的分支客户网络提供虚拟预付费借记卡。

Branch 在旧金山、拉各斯、内罗毕、墨西哥城和孟买拥有 150 名员工，向新兴市场和前沿市场的个人提供起价为 2 美元的贷款。该公司还使用一种算法模型来确定信用度，建立信用档案，并通过手机提供流动性。

“我们将利用(这笔钱)深化在非洲的现有业务。今年晚些时候，我们将在非洲推出高收益储蓄账户，”Branch 联合创始人兼首席执行官 Matt Flannery 表示。

Flannery 表示，来自 Foundation Capital 及其新借记卡合作伙伴 Visa 的 1.7 亿美元将支持 Branch 的国际扩张，其中可能包括巴西和印度尼西亚。去年在墨西哥和印度开设了分公司。在非洲，它在肯尼亚、尼日利亚和坦桑尼亚提供服务。

潜在的分行客户

Branch-Visa 合作伙伴关系将允许个人获得虚拟 Visa 账户，用于在 Branch 的应用程序上创建账户。这使 Branch 在尼日利亚等国家拥有更大的影响力，尼日利亚是非洲人口最多的国家，有 1.9 亿人，在这些国家，在将无银行账户和银行服务不足的人群与金融联系起来方面，信用卡比移动货币发挥了更大的作用。

Branch 成立于 2015 年，开始在肯尼亚运营，在那里，Safaricom 的 M-Pesa (不需要卡或银行账户)等移动支付产品已经取得了显著的规模。根据肯尼亚通信管理局发布的行业统计，M-Pesa 现在拥有 2500 万用户。据公司统计，该分行拥有 300 多万客户，处理了 1300 万笔贷款，支付了超过 3.5 亿美元。

根据 Flannery 的说法，按照尼日利亚和肯尼亚的 Google Play 应用程序数量总和，Branch 是非洲下载量最多的金融科技应用程序之一。

弗兰纳里说，Branch International 已经盈利，预计今年的收入将达到 1 亿美元，其中大约 70%来自非洲。

除了 Visa 和 Foundation Capital，1.7 亿美元的 C 轮融资还包括 Branch 现有投资者 Andreessen Horowitz、Trinity Ventures、Formation 8、IFC、CreditEase 和 Victory Park 的参与，同时增加了新的投资者 Greenspring、Foxhaven 和 B Capital。

Branch 上一次融资 7000 万美元是在 2018 年。对于一家主要专注于非洲的初创公司来说，该公司的总风险投资和 1 亿美元的收入是相当大的数字。泛非电子商务初创公司 Jumia 也于上个月宣布在纽交所 IPO，在 2018 年创造了 1.4 亿美元的收入(未盈利)。

为非洲 12 亿人口打造金融科技的初创公司获得了投资者的关注。Partech 表示，作为一个行业，金融科技(或金融包容性)在 2018 年吸引了非洲初创公司约 11 亿美元资金的 50%。

Branch 最近一轮在国际上增加国家的计划也跟踪了金融科技相关产品在非洲增长，然后向外扩张的趋势。这包括 M-Pesa，在全球 10 个国家运营之前，它在肯尼亚创造了巨大的销量。尼日利亚支付初创公司 Paga 去年年底宣布了其在亚洲和墨西哥的扩张计划。支付服务，如 T4 的 SimbaPay，也已经连接到中国的微信等全球网络。
