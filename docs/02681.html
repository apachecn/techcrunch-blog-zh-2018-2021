为了减少垃圾邮件，Twitter 减少了你每天可以关注的账户数量 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/04/08/to-cut-down-on-spam-twitter-cuts-the-number-of-accounts-you-can-follow-per-day/

Twitter 刚刚迈出了又一大步，以帮助将垃圾邮件发送者赶出其平台:Twitter 用户可以关注的账户数量从每天 1000 个减少到 400 个。新限制的想法是，它有助于防止垃圾邮件发送者通过“大量、攻击性或不加选择地”关注然后取消关注 Twitter 账户来快速扩大他们的网络——这违反了 Twitter 规则。

许多服务最近因为做同样的事情被禁止在 Twitter 的 API 中使用。

几家公司一直在提供工具，让他们的客户不费吹灰之力就能自动跟踪大量用户。这是一种增长策略，因为有些人出于礼貌会跟随，而没有意识到他们已经跟随了一个机器人。

这些公司还提供工具，让那些没有通过关注机器人来回报的人大量取消关注他们的 Twitter 账户。其他自动化工具也经常被提供——比如创建那些烦人的自动 DMs 的工具。

年初，Twitter暂停了一些应用程序，因为它们违反了“关注和追随流失”的规则但是，启动这些公司只能解决那些通过提供垃圾邮件自动化服务来获利的公司，其他人可以使用这些服务。

为了真正打击垃圾邮件制造者，Twitter 用户可以关注多少人的限制也必须在 API 层面上进行改变。

然而，一些人认为 Twitter 今天的举动还不够。

在 Twitter 关于新限制的推文中，一些人回应询问为什么选择“400”这个数字，因为这仍然远远超过一个普通 Twitter 用户一天需要遵循的数字。一些用户说，花了几年时间才达到关注数百人的地步。与此同时，关注 400 人的业务用例有些值得商榷，因为 DMs 可以保持开放，而公司可以在推特上发布一个特殊的 URL 将客户发送到他们的收件箱以继续对话——双方都不需要关注或不关注。

虽然较小的企业可能仍然采用大量跟踪技术来吸引客户，但这至少给这些努力设置了更多的限制。

Twitter 向 TechCrunch 解释了选择这个数字的原因。

一位发言人说:“我们研究了不同阈值下的关注行为，选择 400 作为合理的限制，可以阻止大多数垃圾邮件，同时不影响合法用户。”他们还指出了网站诚信负责人 Yoel Roth 关于这个话题的一条更长的 Twitter 帖子。

这些新的限制和打击垃圾邮件经销商并不是 Twitter 最近几个月为解决其平台上的垃圾邮件问题而采取的唯一变化。

该公司还更新了其举报工具，允许用户举报垃圾邮件，如虚假账户；此外,在账户验证和注册方面引入了新的安全措施,此外还有其他一些旨在更主动识别垃圾邮件发送者的变化。去年夏天， Twitter 还从人们的追随者指标中清除了之前因垃圾邮件而锁定的账户。

综合起来，这一系列行动旨在降低垃圾 Twitter 的吸引力，并大大增加其规模。这不仅影响了那些利用垃圾邮件获取资本的人，也影响了新一波假新闻兜售者寻求颠覆民主和破坏选举——这使得美国政府现在考虑增加对社交媒体的监管。

这些变化的短期影响可能是 Twitter 每月用户增长的下降(Twitter 最近停止分享的数字)，但这是对该平台长期健康的押注。
