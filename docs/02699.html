Personio 收购西班牙薪资初创公司 Rollbox

就在获得 4000 万美元 B 轮融资的三个月后，德国人力资源和招聘平台 Personio 宣布收购了提供基于 API 的薪资解决方案的西班牙初创公司 Rollbox 。两家公司一直在一起工作，但现在 Personio 正在将 Rollbox 的技术和团队带入公司内部。

交易的全部条款仍未披露。然而，据我所知，此次收购包括现金和股权的组合。许多 Rollbox 的投资者被发行 Personio 股票，而其他人则完全退出。

Rollbox 成立于 2016 年，得到了 SaaS 投资者 Point Nine 和柏林风投 La Famiglia 的支持，还有个人投资者 Paul Forster(Indeed 创始人)和 Brian Pietras(work day 战略副总裁)。

与此同时，我被告知 Rollbox 的创始人哈维·里尔和伊斯梅尔·桑切斯将加入 Personio，并将成为未来管理团队的关键成员。Rollbox 品牌将退役，技术将并入 Personio 平台。Rollbox 客户被邀请迁移到 Personio。

Personio 联合创始人兼首席执行官 Hanno Renner 告诉 TechCrunch:“这笔交易将允许 Personio……通过[添加]一个完全集成的薪资引擎来扩展其人力资源操作系统，该引擎可以帮助客户完全自动化薪资。”其功能包括实时验证、自动化政府、保险和税务沟通以及即时工资计算。

Renner 引用 Rollbox 的国际足迹——该公司在西班牙、英国和德国运营——还表示，此次收购将有助于促进 Personio 在欧洲的国际化，这一国际化始于今年早些时候。

在这方面，Personio 表示将保留办公室和所有员工。总人数将超过 200 名员工:目前 Personio 慕尼黑总部有 190 名员工，马德里有 25 名员工。
