谷歌云为微软的活动目录增加了托管服务

不管你喜不喜欢，微软的活动目录仍然是企业中最常用的身份服务之一。谷歌的云平台长期以来一直允许你手动设置活动目录部署，但今天，谷歌通过宣布托管服务的测试版更进一步。顾名思义，谷歌将为你管理这项服务，并自动化从服务器维护到安全配置的一切。

毫不奇怪，鉴于 Google 最近对混合云部署的关注，您也可以使用该服务将现有的内部 Active Directory 域扩展到云中。

正如谷歌指出的那样，依赖于活动目录并正在向云迁移的应用程序和服务器的数量持续增加。其中许多是传统应用程序，但许多新应用程序也依赖于它，因为它只是特定公司的标准。这也给 IT 团队带来了新的挑战，例如，他们现在必须管理额外的延迟和安全要求。新的托管服务旨在使所有这些变得更容易，正如谷歌所说，允许“IT 和安全团队专注于更高价值的项目。”
