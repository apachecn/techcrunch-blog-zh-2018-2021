随着物联网安全业务的突飞猛进，Armis 获得了 6500 万美元的系列 C 技术危机

原文：https://web.archive.org/web/https://techcrunch.com/2019/04/11/armis-nabs-65m-series-c-as-iot-security-biz-grows-in-leaps-and-bounds/

随着物联网安全业务的突飞猛进，Armis 获得了 6500 万美元的系列 C

Armis 正在帮助公司在不使用代理的情况下保护网络上的物联网设备，这显然是一个引起市场共鸣的问题，因为这家初创公司报告去年增长了 700%。这引起了投资者的注意，他们给了他们 6500 万美元的 C 系列投资，以帮助继续加速增长。

在新投资者 Insight Venture Partners 和 Intermountain Ventures 的帮助下，红杉资本引领了这一轮融资。回归的投资者贝恩资本、红点资本和 Tenaya 资本也参与了进来。根据该公司的数据，今天的投资使融资总额达到 1.12 亿美元。

该公司正在解决网络设备管理的难题。如果您的设备无法通过代理跟踪，您如何管理它们？该公司的联合创始人兼首席技术官 Nadir Izrael 说，您必须非常小心地进行操作，因为即使是扫描端口对旧设备来说也可能太多了，它们可能会关闭。相反，他说，Armis 对安全采取被动的方法，观察、学习和理解正常的设备行为是什么样子的——一种行为指纹。

“我们观察网络上的设备。我们观察他们的行为，从中找出我们需要知道的一切，”Izrael 告诉 TechCrunch。他补充道，“简而言之，Armis 是一个巨大的设备行为众包引擎。基本上，Armis 的每个客户端都在不断学习设备的行为。那些统计模型，那些机器学习模型，它们被合并成主模型。”

无论他们在做什么，他们似乎都遇到了一个安全难题。几乎就在一年前他们宣布向提供 3000 万美元的 B 系列，他们又回来索要更多，因为他们发展迅速，需要资本来雇佣员工来维持。

这种增长对任何初创公司来说都是一个挑战。该公司预计在今年年底前将其 125 人的员工队伍增加一倍，但该公司正在努力建立系统来整合这些新员工，并为所有新客户服务。

当然，该公司计划招聘更多的销售和营销人员，但他们将专注于客户支持和建立伙伴关系计划，以获得系统集成商、独立软件开发商和 MSP 的帮助，他们可以为他们提供一些客户帮助。
