无光盘Xbox One S可能于5月7日登陆

微软即将推出一款更便宜的Xbox One S，为了削减成本，该公司将完全移除蓝光光盘驱动器。根据 WinFuture (通过 Thurrott )发现的泄露的营销图片，这款游戏机可能会在5月7日为德国的€229推出。

鉴于距离发布只有几周时间，而且这些营销图片与之前的传言完全吻合，这很可能是真的。

正如你在WinFuture的图片上看到的，它看起来就像一个没有光盘插槽的Xbox One S。这款游戏机名为Xbox One S All Digital，配有1TB硬盘——目前大多数标准Xbox One S游戏机也配有1TB硬盘。

微软明确表示，这款游戏机只用于数字游戏。如果你已经有了实体的Xbox One游戏，你将无法将它们插入主机。

客户可以通过下载代码免费获得三款游戏——《我的世界》、贼海和极限3。然后，您可以在在线商店购买更多游戏，或者订阅Xbox Game Pass来访问游戏库。

这种型号的€在德国的售价应该是229英镑，但你也许能以更低的价格买到它。例如，Xbox One S在微软网站上的官方售价为€299美元，但你可以在亚马逊和其他零售商那里以€200美元的价格轻松买到。

微软通常使用相同的美元价格，所以如果Xbox One S All Digital在美国的官方售价为229美元，我不会感到惊讶。

很明显，微软正在用这款游戏机测试市场。该公司一直在转向订阅模式。Xbox品牌正在从游戏主机品牌演变为服务品牌。这应该是微软与下一代游戏机的关键区别因素。
