EA 的 Origin 客户端中的安全漏洞使游戏玩家暴露于黑客之下

原文：https://web.archive.org/web/https://techcrunch.com/2019/04/16/ea-origin-bug-exposed-hackers/

EA 的 Origin 客户端的安全缺陷使游戏玩家暴露在黑客面前

在安全研究人员发现他们可以欺骗一个毫无戒心的游戏玩家在他们的电脑上远程运行恶意代码后，电子艺术已经修复了在线游戏平台 Origin 中的一个漏洞。

该错误影响了安装了 Origin 应用程序的 Windows 用户。数千万玩家使用 Origin 应用程序购买、访问和下载游戏。为了更容易从网络上访问个人游戏商店，客户端有自己的网址方案，允许玩家通过点击地址中带有origin://的链接打开应用程序并从网页上加载游戏。

但是两个安全研究人员,戴利·贝和张秀坤·彭纳(来自失败者安全)发现，该应用程序可能被骗运行受害者电脑上的任何应用程序。

“攻击者可以运行他们想要的任何东西，”Bee 告诉 TechCrunch。

“弹出计算器”演示 Origin 中的远程代码执行错误(图像:已提供)

研究人员给了 TechCrunch 概念验证代码，让我们自己来测试这个 bug。该代码允许任何应用程序以与登录用户相同的权限级别运行。在这种情况下，研究人员打开了视窗计算器——黑客可以使用的应用程序，显示他们可以在受影响的计算机上远程运行代码。

但更糟糕的是，黑客可能会发送恶意的 PowerShell 命令，这是一个内置应用程序，经常被攻击者用来下载额外的恶意组件并安装软件。

Bee 表示，恶意链接可能以电子邮件的形式发送，也可能在网页上列出，但如果恶意代码与在浏览器中自动运行的跨站点脚本攻击相结合，也可能被触发。

也有可能使用一行代码窃取用户的帐户访问令牌，从而允许黑客无需用户的密码即可访问用户的帐户。

Origin 的 macOS 客户端没有受到该错误的影响。

艺电发言人约翰·雷斯堡证实，周一已经推出了一个解决方案。TechCrunch 确认该代码在更新后不再有效。
