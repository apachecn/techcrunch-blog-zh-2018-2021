原文：https://web.archive.org/web/https://techcrunch.com/2019/04/18/nvidia-launches-its-isaac-sdk-to-help-democratize-ai-powered-robot-development/?tpcc=ECFB2019

Nvidia 推出其 Isaac SDK，以帮助民主化人工智能驱动的机器人开发

今天，在加州大学伯克利分校举行的 TechCrunch TC Sessions:Robotics+AI 活动上，Nvidia 工程副总裁 Claire Delaunay 宣布，该公司的 Isaac SDK 可供下载。上个月宣布，该软件开发套件是芯片制造商持续推动机器人开发为更多用户所用的一部分。

该系统旨在提高机器人人工智能和人工智能关键功能的可访问性，包括障碍检测、语音识别和立体深度估计，其中每一项都将成为未来甚至是基本机器人系统的关键组件。

据该公司称:

使用计算图形和实体组件系统，Isaac 机器人引擎允许开发人员将复杂的机器人任务分解为更小、更简单的步骤网络。使用 Gems 可以简化复杂系统的开发，Gems 是用于传感、规划和驱动的模块化功能，可以轻松插入机器人应用程序。

当然，该系统将与 NVIDIA 自己的机器人硬件组件如 Jetson Nano 和 Jetson AGX Xavier 配合良好。在今天的活动中，Delaunay 在舞台上演示了该系统的一些功能，使用了 Nvidia 的双内部参考平台，两轮 Carter 和四轮 Kaya。
