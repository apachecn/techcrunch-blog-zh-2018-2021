Y Combinator grad Fuzzbuzz 获得 270 万美元的种子资金，提供 fuzzing 服务

最近一届 Y Combinator 班的毕业生fuzz buzz得到了每个早期创业公司都想听到的消息，它获得了 270 万美元的种子资金，以云服务的形式帮助提供一种特殊的自动化软件测试，即 fuzzing。

燃料资本领涨本轮。Homebrew 和 Susa Ventures 也参加了，还有各种天使投资人，包括 Docker 联合创始人所罗门·海克斯，Mesosphere 联合创始人弗洛里安·莱伯特和 Looker 联合创始人本·波特菲尔德。

Fuzzbuzz 的联合创始人兼首席执行官安德烈·塞尔班(Andrei Serban)说，Fuzzbuzz 具体做的是大规模自动化模糊化。“这是一种自动化软件测试，每秒钟可以进行数千次测试，”他解释道。Fuzzbuzz 还利用人工智能和机器学习基础，使用结果反馈来自动生成新的测试，这样它应该会越来越智能。

目标是覆盖尽可能多的代码，比人类测试人员更快更有效地发现漏洞和错误。这是每一家生成代码的公司显然都想做的测试，但问题是，到目前为止，这个过程一直很昂贵，需要高度专业化的安全工程师来承担。像谷歌和脸书这样的公司能够雇佣这样的人来构建模糊化的解决方案，但在很大程度上，这对于一般的公司来说是遥不可及的。

Serban 表示，他的联合创始人 Everest Munro-Zeisberger 曾在谷歌 Chrome fuzzing 团队工作，该团队使用这种技术发现了超过 15，000 个漏洞。他想让任何人都能进行这种测试。

“今天，任何人都可以在不到 20 分钟的时间内开始在 Fuzzbuzz 上 fuzzing。我们直接连接到 GitHub 和您的 CI/CD 管道，对发现的每个 bug 进行分类和去重，然后通过 Slack 和吉拉等工具通知您。使用 Fuzzbuzz CLI，开发人员可以在本地测试并修复错误，然后将他们的代码推回 GitHub，”该公司在宣布资助的博客文章中写道。

现在还为时尚早，这家初创公司正在与一些初始客户合作。这笔资金应该可以帮助三位创始人 Serban、Munro-Zeisberger 和 Sabera Hussain 雇佣更多的工程师，并将更完整的解决方案推向市场。这是一项雄心勃勃的事业，但如果它成功地创建了一个模糊服务，这可能意味着交付更少错误的代码，这对每个人都有好处。
