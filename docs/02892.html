制造业巨头 Aebi Schmidt 被勒索软件 TechCrunch 攻击

原文：https://web.archive.org/web/http://techcrunch.com/2019/04/23/aebi-schmidt-ransomware/

TechCrunch 获悉，在美国开展业务的欧洲制造业巨头 Aebi Schmidt 受到了勒索软件攻击。

据了解该事件的消息人士称，这家总部位于瑞士的机场维护和道路清洁车辆制造商在恶意软件感染后于周二中断了运营。

该公司的国际网络(包括其美国子公司)出现系统故障，但大部分损失发生在该公司的欧洲总部。世界各地与 Aebi Schmidt 网络相连的许多系统都处于瘫痪状态。该消息人士称，攻击发生后，生产运营所需的系统无法使用。该公司的电子邮件据说也受到影响。

目前还不清楚是哪种勒索软件导致该公司系统离线。

这家跨国制造巨头最近通过收购除雪和清洁机制造商 M-B 公司扩大了其在美国的业务，此前该公司收购了冬季维护设备制造商 Meyer Products 和 Swenson Products。

在多次通过电子邮件、电话或主动发来的 LinkedIn 消息联系该公司后，发言人托马斯·希伊斯(Thomas Schiess)在脸书的一条消息中确认了系统中断，具体是“电子邮件系统故障”。“我可以证实，其他系统的可用性曾经或可能仍然受到限制，我们的专家仍在努力解决这个问题，原因尚不清楚，”他说，但不愿进一步置评。

周三，Schiess 表示，其运营系统(包括 SAP 业务和销售系统)正在工作，生产“正常运行”但是，他说，该公司的 Windows 网络“受到了病毒的影响”,为了以防万一，其他系统已经关闭。

“技术人员正在努力恢复 Windows 环境，但这需要一些时间，”他说。

但消息来源称，员工被遣送回家——在某些情况下被迫休无薪假。

席斯不愿具体评论勒索软件的说法，但消息人士称，员工在周三的全体会议上被告知，该事件是一次“勒索软件攻击”

Aebi Schmidt 是最近几周被勒索软件击倒的最新一家公司。

铝制造巨头 Norsk Hydro 在 3 月份的一次交易攻击后被迫暂时下线。该公司在实施备份恢复流程后很快恢复。这是比饮料公司亚利桑那饮料更好的回应。一个月后，该公司遭到勒索，导致其系统关闭一周——尽管联邦调查局几周前警告称，该公司感染了潜伏的恶意软件。
