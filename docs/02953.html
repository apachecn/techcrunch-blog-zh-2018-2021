沃尔玛推出人工智能驱动的未来商店，现已向公众开放 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/04/25/walmart-unveils-an-a-i-powered-store-of-the-future-now-open-to-the-public/

沃尔玛今天早上公布了一个新的“未来商店”和新兴技术的测试场，包括人工智能摄像头和互动显示器。这家商店是一个名为智能零售实验室(Intelligent Retail Lab)的工作概念，简称为“IRL ”,在纽约莱维敦的沃尔玛社区市场经营

这家零售商说，这家商店对顾客开放，是沃尔玛最繁忙的社区市场商店之一，包含 3 万多种商品，这使它能够在现实环境中测试技术。

与 Amazon Go 的便利店类似，该商店在天花板上安装了一套摄像头。但与亚马逊 go 不同，亚马逊 Go 是一家占地面积较小的商店，沃尔玛的 IRL 占地 50，000 平方英尺，拥有 100 多名员工。

此外，在沃尔玛的案例中，这些人工智能摄像头并没有被用来确定客户正在购买哪些商品，以便自动对他们收费。它仍然有传统的收银台。相反，摄像头将监控库存水平，以确定员工是否需要从后台冰箱中拿出更多的肉来补充货架，或者一些新鲜商品是否在货架上放置了太长时间，需要撤出。

这个想法是，人工智能将帮助商店员工更准确地知道在哪里和什么时候重新进货。反过来，这意味着顾客会知道，当他们到达时，农产品和肉类总是新鲜的，有库存。

沃尔玛表示，利用技术做到这一点并不简单。这意味着自动化系统将需要能够检测货架上的产品，识别它看到的确切产品(1 磅碎牛肉与 2 磅)。等。)，然后将货架上的数量与即将到来的销售需求进行比较。

对于商店员工来说，该系统允许他们停止不断地在商店走动以更换库存，相反，他们会在当天向顾客开门之前知道从里屋拿出什么。

商店中的摄像头和其他传感器每秒钟产生 1.6 TB 的数据，相当于三年的音乐，这需要一个大型数据中心。在 IRL 商店，它被玻璃包裹，沐浴在蓝光中，向公众展示。

这看起来有点吓人——人工智能摄像机和巨型服务器。但沃尔玛表示，这些数据仅存储不到一周。

商店里还有信息站，顾客可以在那里了解更多关于正在使用的技术。店内还有一个欢迎中心，供想要了解更多技术规格并获得常见问题答案的顾客使用。

互动墙让客户享受人工智能的乐趣——它展示了人工智能系统如何估计身体位置。但实际上，这是为了让所有这些新技术看起来不那么可怕。

IRL 首席执行官 Mike Hanrahan 表示:“技术让我们能够实时了解我们的业务。“当你将我们在 IRL 收集的所有信息与沃尔玛 50 多年的商店运营专业知识相结合时，你可以创造真正强大的体验，改善我们顾客和员工的生活。”

沃尔玛有兴趣将人工智能用于零售中更实际的目的——这位首席执行官在一份声明中可能含蓄地提到了亚马逊 Go。

“你不能过分迷恋人工智能闪亮的物体元素，”汉拉汉说。“有很多闪亮的物体正在做我们认为不切实际的事情，并且可能从长期来看，对消费者没有好处。”

沃尔玛将在 IRL 测试未来的概念，而不是专注于自动结账解决方案，在肉类库存水平使用人工智能系统确保购物车随时可用，收银机开放并配备人员后。

该公司坚称，这项技术不会取代工作，而是让员工有时间与客户互动。这也是它在向其商店推出更多机器人时所做的声明。但很难想象，随着时间的推移，更高效地经营商店会需要像现在这样多的员工。

IRL 是由沃尔玛的科技孵化器 Store N o 8 设计的一个概念，它经营着几家合资企业来测试零售业的新想法。今年早些时候，推出了一家提供虚拟现实旅游的初创公司，以增强购物体验，2017 年它开始在纽约测试一项名为 Code Eight 的个人购物服务。
