脸书被指控阻止对其广告平台 TechCrunch 进行更广泛的研究

原文：https://web.archive.org/web/https://techcrunch.com/2019/04/29/facebook-accused-of-blocking-wider-efforts-to-study-its-ad-platform/

脸书被指控阻止独立研究人员有效研究政治虚假信息如何通过其广告平台流动的能力。

社交网络业务旨在赚钱的广告——至少——有影响人们和推动选民投票的潜力，正如剑桥分析公司脸书数据滥用丑闻去年强调的那样。

自从那件事爆发成脸书的重大全球丑闻后，该公司就面临着大西洋两岸政策制定者的齐声呼吁，要求增加透明度和问责制。

它以大量的模糊、误导和更糟的作为回应。

在脸书应对虚假信息对其业务构成的威胁的争议较少的努力中，有一项是它所谓的“选举安全”举措，如对政治广告客户进行身份检查。面对有关方面试图利用其工具在全球市场上放大虚假信息的担忧，即便是这些努力也显得毫无进展、支离破碎。

也许更重要的是——在强大的政治压力下——脸书推出了一个可搜索的广告档案。访问脸书广告数据肯定有可能让外部研究人员对该公司的声明负责。

但前提是访问权限不是同样的杂乱、零碎，有选择性地访问广告数据的风险最终会像脸书平台上的其他东西一样受到控制和操纵。

到目前为止，脸书在这方面的努力仍因做得不够而招致批评。

“与他们声称正在做的事情相反……”

该公司上个月开放了对广告档案 API 的访问，通过它可以提供对关键词搜索工具的限速访问，让研究人员查询历史广告数据。(研究人员首先需要通过身份检查流程，并同意脸书开发者平台的服务条款，然后才能访问 API。)

然而，非营利组织 Mozilla 对该工具的评估认为，该 API 是一种“透明清洗”，而不是一种支持公共利益研究的善意尝试，这种研究可以真正帮助量化脸书广告业务的社会成本。

“事实是，API 没有提供必要的数据。它的设计方式阻碍了研究人员的重要工作，这些研究人员向公众和政策制定者通报错误信息的性质和后果，”它在一篇博客文章中写道，其中它认为脸书的广告 API 只达到了它以前设定的五个最低标准中的两个——得到了来自牛津大学、阿姆斯特丹大学、布鲁塞尔自由大学、新 Verantwortung 等研究机构的 60 名学者的支持。

Mozilla 没有像专家认为的那样提供全面的政治广告内容，而是写道“不可能确定脸书的 API 是否全面，因为它要求你使用关键字来搜索数据库。”

“它并不提供所有的广告数据，也不允许你像几乎所有其他在线数据库那样，使用特定的标准或过滤器进行过滤。由于你不能批量下载数据，API 中的广告也没有唯一的标识符，脸书不可能获得他们平台上运行的所有广告的完整图片(这与他们声称的完全相反)。

脸书的工具也因未能提供广告的目标标准和参与信息而受到批评——从而使研究人员无法了解其平台上的广告商向该公司支付的费用是为了达到什么目的；以及这些脸书广告的效果如何。

去年，在对网上虚假信息进行为期数月的调查期间，英国议员向多名脸书高管提出了这个问题。有一次，脸书的首席技术官被直截了当地问及该公司是否会提供广告定位数据，作为计划中的政治广告透明度措施的一部分——结果只是给出了一个模糊的答案。

当然，脸书有很多理由不愿意让真正独立的外部人士量化其平台上政治广告的效果，进而量化其广告业务。

当然，包括剑桥分析公司数据盗窃本身的具体丑闻，这是由当时隶属于剑桥大学的学者亚历山大·科岗博士实施的，他利用他对脸书开发平台的访问权部署了一个问答应用程序，旨在在(大多数)人不知情或不同意的情况下收集用户数据，以便将这些信息出售给名誉扫地的数字竞选公司(该公司曾参与美国的各种竞选活动，包括特德·克鲁兹和唐纳德·川普的总统竞选)。

但这只是凸显了如此多的市场力量集中在一家广告科技巨头手中的问题的严重性，该公司没有动力自愿报告关于其真正影响力和影响全球 20 多亿脸书用户的能力的完全透明的指标。

除此之外，在去年对多个负面头条的典型危机公关回应中，脸书一再试图将科岗描绘成一个流氓演员——暗示他根本不是其平台上广告活动的代表性样本。

因此，出于同样的原因，脸书将真正的研究贴上类似风险标签的任何努力都理应得到有力的反驳。一个人的历史行为，尽管是一个学者，不应该被用作关闭一个受尊重的研究社区大门的借口。

“当前的 API 设计给研究人员带来了巨大的限制，而不是让他们发现平台上真正发生的事情，”Mozilla 认为，脸书施加的各种限制——包括搜索速度限制——意味着研究人员可能需要“几个月”来评估特定地区或特定主题的广告。

再一次，从脸书的角度来看，推迟发布更多来自其膨胀的历史数据柜的平台使用框架会有很多好处。(它去年大张旗鼓地宣布的“历史应用审计”继续以自己选择的速度进行。)

脸书的 API 在两个方面得到了 Mozilla 的初步认可，即提供对最新和历史数据的访问(数据的七年可用性被评为“相当好”)；并且 API 可以被普通公众访问和共享(至少在他们通过脸书的身份确认流程后)。

尽管在这两种情况下，Mozilla 也警告说，仍然有可能出现进一步的封锁策略——这取决于脸书如何支持/限制访问。

对脸书 API“不充分”的批评发生在同一天，这看起来并不完全是巧合，因为脸书已经推出了关于开放访问其用户自 2017 年以来链接的网址数据库的宣传,该数据库正在向一组选定的学者开放。

在这种情况下，美国社会科学研究委员会挑选了来自 30 个机构的 60 名研究人员。

值得注意的是，脸书选择的研究数据集完全跳过了 2016 年美国总统大选，当时俄罗斯的选举宣传臭名昭著地瞄准了数亿美国脸书选民。

从 2017 年 1 月开始，英国 2016 年的英国退出欧盟投票也不在数据集范围内。

脸书确实表示，它“致力于推进这一重要举措”，暗示它可能会扩大数据集的范围和/或谁可以在未来某个未指明的时间访问它。

它还声称“隐私和安全”的考虑阻碍了更快发布研究数据的努力。

“我们理解许多利益相关者渴望数据尽快可用，”它写道。“虽然我们仍然致力于推进这一重要举措，但脸书也致力于花必要的时间来纳入最高的隐私保护，并建立一个以安全方式提供数据的数据基础设施。”

在欧洲，脸书去年在 T2 签署了欧盟委员会关于虚假信息的行为准则，承诺支持诚信和公共利益研究。

EU-的广泛准则包括一项具体承诺，即平台签署者“授权研究社区通过对平台数据的隐私合规访问来监控在线虚假信息”，此外还有其他行动，如处理虚假账户，以及使政治广告和基于问题的广告更加透明。

然而，在这一点上，脸书似乎也在以“隐私合规”为借口，淡化其向外部研究人员提供的透明度。

TechCrunch 知道，私下里，脸书对其广告 API 限制的担忧做出了回应，称无法向研究人员提供更多关于广告的数据——包括广告的目标标准——因为这样做会违反其在欧盟通用数据保护条例(GDPR)框架下的承诺。

这个论点当然是纯粹的“蛋糕主义”又名脸书是试图有它的蛋糕，吃它的隐私和数据保护有关。

说白了，脸书正试图利用欧洲隐私法规来保护其业务免受更深入、更有意义的审查。然而，正是这家公司——这里出现了丰富的 fudgy cakeism——其他地方的声称其平台上普遍收集的用户兴趣的个人数据不是个人数据。(在这种情况下，脸书也被发现允许敏感的推断数据用于定向广告——专家认为这违反了 GDPR。)

所以，TL；可以发现，脸书博士在符合其商业利益时会抓住隐私监管不放——即试图避免外部研究人员评估其广告平台和业务对更广泛的社会和民主的影响所需的透明度水平……但当隐私监管通过向用户灌输针对每个人利益的无处不在的监控所针对的侵入性广告，阻碍了用户眼球的货币化时，他反对 GDPR。

这种矛盾丝毫没有逃过隐私专家的眼睛。

艾伦图灵研究所(Alan Turing Institute)研究员迈克尔·维尔(Michael Veale)说:“实际上，GDPR——不仅仅是脸书通常对它的微弱解释——并不能阻止组织发布总体信息，例如哪些人口统计数据或地理区域看到了或成为了某些广告的目标，这些数据不够精细，无法挑出个人，”迈克尔·维尔是 Mozilla 支持的有效广告应用编程接口指南的 10 名研究人员之一。

他还告诉我们:“脸书需要一个合法的基础来进行以出版为目的的汇总，这并不困难，因为提供数据以使公众能够审查数据处理的合法性和道德性是我见过的合法利益。”“脸书不断地重复使用数据用于不同的和不明确相关的目的，因此声称他们不能合法地重复使用数据来将自己的活动置于聚光灯下，坦率地说，这是可悲的。

“统计机构早就熟悉差分隐私等技术，这种技术可以阻止聚合信息泄露特定个人的信息。许多差分隐私研究人员已经在脸书工作，所以专业知识是显而易见的

“看起来更有可能的是，脸书不想公布目标锁定的信息，因为这可能会让[它]和他们的客户感到尴尬，”维尔补充道。“还有一种可能是，脸书与特定的广告商签订了保密协议，这些广告商可能会因为超出公众预期的做法而被当场抓获。数据保护法没有挡住透明的消毒光，脸书挡住了。”

当被问及脸书今天向选定的研究人员发布的 URL 数据库时，维尔说这是一个受欢迎的步骤——同时指出了进一步的限制。

“脸书开始在研究问题上更加开放，这是一件好事，尤其是那些可能指向该平台使用问题的问题。他说:“最初的一批学生似乎来自不同的地理区域，这令人耳目一新——尽管似乎没有来自印度大学的学者，而印度大学无疑是脸书最大的用户群。”。“时间会证明这个有限的数据集是否会在以后扩展到其他问题，以及如果研究人员希望继续友好接触，他们会在多大程度上缓和他们的发现。”

“脸书很有可能有效地挑选数据集，试图避免他们知道存在的问题，但你也不能在所有方面和问题上开始建立一个协作流程。时间会告诉我们跨国公司希望开放到什么程度，”维尔补充道。

我们联系了脸书，就其广告存档 API 的批评发表评论。
