我们为IPO秘密提交文件TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/04/29/wework-files-confidentially-for-ipo/

WeWork，现在被称为We公司的合作巨头，已经向美国证券交易委员会提交了首次公开募股的机密文件，该公司在周一的新闻稿中证实。

据《T2纽约时报》报道，该公司最初于去年12月提交了IPO文件。

WeWork今年1月的估值为470亿美元，自2010年由亚当·纽曼(Adam Neumann)和米格尔·麦凯尔维(Miguel McKelvey)创立以来，已经通过债务和股权融资筹集了84亿美元。WeWork是几家获得软银愿景基金(SoftBank Vision Fund)数亿美元(实际上是数十亿美元)支持的科技独角兽之一。最近，日本电信巨头盯上了该公司价值160亿美元的多数股权，但在最后一刻冷却了他们的飞机。

WeWork的收入从2017年的8.86亿美元翻了一番，达到2018年的约18亿美元，净亏损达到惊人的19亿美元。对于上市前的企业来说，这些指标并不吸引人；此外，尽管增长萎缩，优步目前正在完成备受关注的IPO路演。以下是来自 Crunchbase新闻的更多关于WeWork顶线财务的报道:

  • we work 2017年收入:8.86亿美元
  • we work 2017年净亏损:9.33亿美元
  • WeWorks 2018年收入:18.2亿美元(增长105.4%)
  • we work 2018年净亏损:19亿美元(增长103.6%)

从好的方面来看，根据Axios的数据，WeWork在2018年的入住率达到了90%，会员总数增长了116%，达到40.1万人。

WeWork经常被认为是硅谷倾向于抬高估值的完美例子。房地产公司WeWork现金消耗很快，无疑将不得不努力让公开市场投资者相信它的长寿，以及它作为一家科技公司的地位。

WeWork得到了软银、Benchmark、T. Rowe Price、富达、高盛和其他几家公司的支持。
