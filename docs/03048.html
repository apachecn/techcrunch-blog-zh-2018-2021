微软推出拖放式机器学习工具

原文：https://web.archive.org/web/https://techcrunch.com/2019/05/02/microsoft-launches-a-drag-and-drop-machine-learning-tool-and-hosted-jupyter-notebooks/

微软今天宣布了三项新服务，旨在简化机器学习的过程。这些产品包括用于完全自动化模型创建过程的工具的新界面，用于构建、培训和部署模型的新的无代码可视化界面，以及面向高级用户的托管Jupyter风格的笔记本。

机器学习入门很难。即使是进行最基本的实验也需要大量的专业知识。所有这些新工具都通过隐藏代码或给那些想要编写自己代码的人提供一个预先配置好的平台，极大地简化了这个过程。

Azure的自动化机器学习工具的新界面使创建模型变得像导入数据集一样简单，然后告诉服务要预测哪个值。用户不需要编写一行代码，而在后端，这个更新的版本现在支持许多新的算法和优化，应该会产生更准确的模型。虽然大部分都是自动化的，但微软强调，这项服务提供了“算法的完全透明性，因此开发人员和数据科学家可以手动覆盖和控制这个过程。”

对于那些希望从一开始就获得更多控制权的人来说，微软今天还为其Azure机器学习服务推出了一个可视化界面preview，该界面将允许开发人员在不接触任何代码的情况下构建、训练和部署机器学习模型。

这个工具，Azure机器学习视觉界面，看起来很像现有的 Azure ML Studio ，这是微软首次尝试构建视觉机器学习工具。事实上，这两种服务看起来完全一样。然而，该公司从未真正推动这项服务，而且几乎似乎已经忘记了它，尽管事实上它似乎一直是一个非常有用的机器学习入门工具。

微软表示，这个新版本结合了Azure ML Studio和Azure机器学习服务的优点。在实践中，这意味着虽然界面几乎是相同的，但Azure Machine Learning可视化界面通过在Azure Machine Learning服务上运行并添加该服务的安全、部署和生命周期管理功能，扩展了ML Studio的可能。

该服务提供了一个简单的界面来清理您的数据，在不同算法的帮助下训练模型，评估它们，并最终将它们投入生产。

虽然前两个服务明显针对新手，但Azure机器学习中的新托管笔记本显然是面向更有经验的机器学习从业者的。这些笔记本电脑预先打包了对Azure机器学习Python SDK的支持，并在该公司描述为“安全的企业就绪环境”中运行。虽然使用这些笔记本电脑也不简单，但这一新功能允许开发人员快速入门，而无需使用所有必要的云资源设置新的开发环境。
