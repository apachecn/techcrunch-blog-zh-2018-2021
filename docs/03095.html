微软推出React Native for Windows

微软今天宣布了一个新的开源项目，面向那些希望瞄准Windows的React原生开发者。“React Native for Windows”，正如该项目的名字一样，是在MIT许可下React for Windows的一个新的“面向性能”的实现。

能够使用来自脸书的跨平台开发框架 React Native 来定位Windows并不新鲜。该框架允许开发者用JavaScript编写代码，然后在Android和iOS上运行，已经有了针对Windows和 macOS 的插件和扩展。

在React Native for Windows中，微软重新实现了React Native，并且用C++ 重写了许多组件，以获得最佳性能。它允许开发者针对任何Windows 10设备，包括PC、平板电脑、Xbox、混合现实设备等。有了微软对该项目的支持，这些开发者现在将能够为他们的用户提供更快、更流畅的应用。

微软该项目现已在GitHub上发布，并准备供开发人员测试，在不久的将来会有更成熟的版本推出。
