微软推出在线代码编辑器Visual Studio Online

微软今天宣布推出 Visual Studio Online 的私人预览版，该公司将这款在线代码编辑器定位为Visual Studio和Visual Studio代码的伴侣。

该服务基于微软广受欢迎的免费开源桌面代码编辑器Visual Studio Code。这意味着Visual Studio Online还将支持当前可用于Visual Studio代码的所有扩展，以及像Visual Studio代码工作区这样的流行功能。对 IntelliCode 的支持也是内置的，这是微软的人工智能辅助开发工具，后来成为今天普遍可用的。

这里强调的是Visual Studio Online是一个“伴侣”它并不意味着成为开发人员的默认环境，而是作为一种快速编辑、查看拉取请求或加入实时共享会话的方式。

如果你认为Visual Studio Online这个名字听起来很熟悉，那是因为微软实际上在重复使用这个名字。不久前，Visual Studio Online是微软所有DevOps的中心，在DevOps成为流行语之前。去年，该公司将其更名为Azure DevOps，并保留了其他用途。坦率地说，从名称上看，许多人可能总是认为Visual Studio Online是集成开发环境的基于web的版本，但后来失望地发现它不是。

值得注意的是，如果你不想等待微软向更多用户开放私有预览，也有像 Coder 这样的创业公司可以为你提供远程Visual Studio代码环境。
