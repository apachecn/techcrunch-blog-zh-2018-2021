微软用于人工智能辅助编码的 IntelliCode 发布预览版

微软的人工智能辅助编码工具 IntelliCode 现在已经上市。它支持 Visual Studio 中的 C#和 XAML，以及 Visual Studio 代码中的 Java、JavaScript、TypeScript 和 Python。默认情况下，它现在也包含在 Visual Studio 2019 中，从 16.1 版本的第二个预览版开始，该公司还宣布，

IntelliCode 本质上是微软非常流行的代码完成工具 IntelliSense 的下一代。IntelliCode 的不同之处在于，该公司通过向它提供 GitHub 上千个至少有 100 颗星的开源项目的代码来训练它。使用这些数据，该工具可以给出更智能的代码完成建议。在提出建议时，它还会考虑当前的代码和上下文。

默认情况下，IntelliSense 会为开发人员提供一个按字母顺序排列的列表，这很有用，但通常情况下，您需要的代码是列表中的几个项目。

值得注意的是，像 Kite 这样的初创公司提供了类似的智能代码完成工具，可以跨开发环境工作，尽管 Kite 目前只支持 Python 代码。

像 Kite 和 IntelliCode 这样的工具的承诺是让开发人员的生活变得更容易，提高生产力，减少出错的可能性。随着这些工具变得越来越智能，它们很可能能够向前看得更远，甚至可能会根据您试图实现的内容以及其他人如何解决类似问题的知识，建议自动完成程序代码的更大部分。不过，在那之前，它们已经是避免去几次 StackOverflow 的一个很好的方法了。
