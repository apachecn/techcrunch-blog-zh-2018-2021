点击此处观看微软 2019 年 Build 主题演讲

现在是五月初，这意味着所有大型开发者大会的时间到了。继上周脸书·F8 之后，本周轮到微软和谷歌，微软先走。该公司本周将在西雅图举办年度 Build 开发者大会，约有 6000 名开发者参加。据我们所知，该活动并未售罄，但微软仍然预计约 6000 名与会者将在西雅图。

可能是因为谷歌将其 I/O 主题演讲安排在明天，而传统上，微软将在明天举行更具技术性的第二天主题演讲，微软实际上已经开始了，在上周四太平洋时间下午 2 点的奇怪时间宣布了几乎所有的云新闻(包括一个小的 HoloLens 更新)。这意味着今天的庆祝活动可能会集中在该公司的其他平台上，包括 Windows、Office 365、微软 365 等。

该活动将于太平洋时间上午 8:00/欧洲中部时间 17:00 以微软创新杯学生竞赛的决赛开始，这是第一次在 Build 举行。塞特亚·纳德拉将在太平洋时间上午 8:30/欧洲中部时间 17:30 发表他的“愿景主题演讲”活动的这一部分可能不太技术性，更多地关注微软对未来工作的整体愿景。这应该持续大约一个半小时。

上午晚些时候，在太平洋时间上午 11 点/欧洲中部时间 20:00，微软将主题演讲分为两部分，由斯科特·格思里和拉杰什·贾同时进行技术主题演讲。

该公司还承诺为主题演讲提供物理环境，这很容易被证明是一个脱离活动内容的怪异噱头。
