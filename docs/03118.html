原文：https://web.archive.org/web/https://techcrunch.com/2019/05/07/google-launches-jetpack-compose-an-open-source-kotlin-based-ui-development-toolkit/

Google 发布了 Jetpack Compose，这是一个基于 Kotlin 的开源 UI 开发工具包

谷歌今天宣布了 Jetpack Compose 的第一个预览版，这是一个新的开源 UI 工具包，面向希望使用类似 React Native 和 Vue.js 的反应式编程模型的 Kotlin 开发者。

Jetpack Compose 是一个未捆绑的工具包，是谷歌为 Android 开发人员提供的整体 Android Jetpack 软件组件集的一部分，但不要求使用任何其他 Jetpack 组件。通过 Jetpack Compose，Google 基本上把 UI 作为代码的理念带到了 Android 开发中。Compose 的 UI 组件是完全声明性的，允许开发人员通过简单地描述 UI 在代码中的样子来创建布局。Compose 框架将为开发人员处理 UI 优化的所有细节。

开发人员可以混合搭配 Jetpack Compose APIs，并查看基于 Android 原生 API 的 API。开箱即用，Jetpack Compose 还原生支持谷歌的材质设计。

作为今天整个 Jetpack 更新的一部分，谷歌还推出了许多新的 Jetpack 组件和功能。这些功能包括支持为 Android for Cars 和 Android Auto 构建应用程序，以及支持将应用程序与企业移动管理解决方案和内置基准工具更轻松地集成的企业库

不过，最突出的功能可能是 CameraX，这是一个新的库，允许开发人员构建以相机为中心的功能和应用程序，使开发人员能够访问与原生 Android 相机应用程序基本相同的功能。
