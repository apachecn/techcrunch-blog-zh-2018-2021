Kotlin 现在是谷歌开发 Android 应用程序的首选语言

原文：https://web.archive.org/web/https://techcrunch.com/2019/05/07/kotlin-is-now-googles-preferred-language-for-android-app-development/amp/

谷歌今天宣布， Kotlin 编程语言现已成为安卓应用开发者的首选语言。

谷歌在今天的公告中写道:“首先，Android 开发将变得越来越科特林”。“许多新的 Jetpack APIs 和功能将首先在科特林提供。如果你正在开始一个新项目，你应该把它写在 Kotlin 中；用 Kotlin 编写的代码对你来说通常意味着更少的代码——需要键入、测试和维护的代码更少。”

直到两年前的，在 2017 年 I/O 大会上，谷歌才宣布在其 Android Studio IDE 中支持 Kotlin。考虑到 Java 长期以来一直是 Android 应用程序开发的首选语言，这有点令人惊讶，但在那一年的 I/O 上很少有宣布获得更多掌声。在过去的两年里，科特林的知名度只增不减。谷歌称，超过 50%的专业 Android 开发者现在使用这种语言开发他们的应用，在最新的 Stack Overflow 开发者调查中，它被列为第四大最受欢迎的编程语言。

因此，谷歌增加对 Kotlin 的支持是有道理的。“我们宣布，我们正在采取的下一个重大步骤是，我们将 Kotlin-first，”Android 的首席倡导者切特·哈斯(Chet Haase)说。

“我们知道现在不是每个人都在科特林，但我们相信你应该去那里，”哈斯说。“可能有充分的理由让你继续使用 C++和 Java 编程语言，这完全没问题。这些不会消失。”
