在安全领域，没有什么是“不可攻击的”当它被宣称时，安全研究人员只看到一个挑战。

输入英国网络安全公司Pen Test Partners的最新发现。他们的最新项目是拆开“无法破解”的eyeDisk，这是一个据称安全的USB闪存驱动器，使用虹膜识别来解锁和解密设备。

在去年的Kickstarter活动中，eyeDisk筹集了超过21000美元；它在三月份开始发货。

只有一个问题:它绝不是“不可破解的”

Pen Test Partners的研究员戴维·洛奇发现，使用一种能够嗅探USB设备流量的软件工具，可以很容易地获得设备的备份密码——在设备故障或突发挖眼事故时访问数据。

秘密密码——“secret pass”——可以在明文中看到(图片:Pen Test Partners)

“那个红色的字符串，是我在设备上设置的密码。畅通无阻。他在博客文章中详细描述了他的发现，他说。

他说，更糟糕的是，即使输入了错误的密码，设备的真实密码也能被窃取。洛奇解释说，这是因为设备首先透露其密码，然后在发送解锁密码之前，根据用户提交的任何密码进行验证。

洛奇说，任何使用这些设备的人都应该在设备上使用额外的加密技术。

研究人员向eyeDisk透露了这个缺陷，eye disk承诺修复，但尚未发布；eyeDisk没有回复置评请求。
