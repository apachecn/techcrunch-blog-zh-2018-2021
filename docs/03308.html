HBO NOW 订户增长的冬天即将来临 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/05/17/winter-is-coming-for-hbo-now-subscriber-growth/

粉丝对《权力的游戏》最后一季的反应可能不一，但这部剧对 HBO 的网络和其顶级流媒体服务 HBO NOW 来说无疑是好的。第八季首播当天吸引了 1180 万现场观众和 1740 万各平台观众，以及HBO NOW 创纪录的注册人数，今年 3 月报道称其拥有 800 万订户。但是节目的大结局将于本周日播出，因此 HBO 将会看到大量流媒体用户的流失。

根据 Mintel 本周发布的新研究，HBO NOW 用户在特定节目结束时取消订阅的可能性是其他流媒体服务用户的两倍。

唯一在这方面表现较差的服务是 YouTube Premium。这并不完全是苹果与苹果之间的比较，因为它的用户群还包括想要无广告的 YouTube 观众——而不仅仅是那些观看其原创内容的人。

这些新发现说明了 HBO 多年来对《权力的游戏》的依赖程度。此外，这些指标表明 HBO 母公司 WarnerMedia 即将推出的流媒体服务将面临潜在的挑战。由于将在今年晚些时候推出测试版，这项服务将由 HBO content 主导。但是如果没有新一集的《权力的游戏》,它将不得不依靠其他受欢迎的节目来吸引观众，比如《西部世界》。

然而，尽管《西部世界》是 HBO 收视率第二高的节目，但《权力的游戏》的观众数量是它的三倍。

该网络显然意识到了《权力的终结》将给其流媒体平台带来的负面影响。它已经批准了《权力的游戏》前传的计划，目前正在拍摄的。此外，它还在酝酿其他的衍生产品。

前传可能不会像第一部那样吸引人们的热情，但它可以帮助观众回归。然而，与此同时，HBO NOW 将在周日晚上之后看到大量订户取消订阅。

Mintel 还发现，除了已经订阅了四个或更多顶级流媒体服务的消费者之外，HBO 现在没有任何显著的吸引力。这些用户为网飞、亚马逊 Prime Video 和 Hulu 付费，然后为了获得《权力的游戏》而将 HBO 加入其中。他们不一定忠于网络本身，也不一定对它的其他节目感兴趣。每月 14.99 美元，HBO NOW 是一个相当昂贵的附加服务。

随着苹果和迪士尼准备在未来几个月推出新的流媒体服务，许多消费者可能会将他们的 HBO NOW 美元转移给新人，或者干脆将他们的存款装进口袋。

研究人员还认为，规模较小、知名度较低的流媒体服务可以通过将其产品定位为比 HBO NOW 更实惠的替代产品而受益。

这一点尤其正确，因为该研究发现，消费者对“完美”流媒体包的理想价位——拥有他们想看的一切——将是每月 20 美元左右。今天，这个数字可以让他们购买两项或最多三项服务。第四种服务，如 HBO NOW，更像是一种奢侈的支出——也许是《权力的游戏》播出时的必备服务，但在节目结束时，没有一种服务会让消费者感到舒服。

不过，新的报告没有对 HBO 即将看到的取消数量做出明确预测。

Mintel analyst 分析师巴迪·罗(Buddy Lo)说:“我不太确定是订阅还是取消。”他告诉 TechCrunch:“我们从研究中得知，近 20%的 HBO NOW 消费者表示，他们会取消某个特定节目的服务，但我们没有明确询问他们是否会取消《权力的游戏》。

当然，很难想象 HBO NOW 的订户在回应时会想到什么其他节目。

Mintel 并不是唯一一家陷入旗舰系列结束对 HBO NOW 订户增长的潜在影响的公司。上个月， Second Measure 指出了历史趋势,这有助于预测未来的大用户下降。

例如，在第七季播出期间，HBO NOW 在美国的用户数量跃升了 91%，但在第七季结束后的六个月里，用户数量稳步下降。报告称，在《权力的游戏》第七季期间首次付费的 HBO NOW 用户中，只有 26%在六个月后仍然是用户。

它还发现，HBO NOW 的用户远不如其他流媒体服务的用户忠诚，依次包括网飞、Hulu 甚至 CBS All Access——后者要感谢《星际迷航:发现》的粉丝群。

HBO NOW 和 CBS All Access 的留存率都远不及网飞和 Hulu，后者的六个月留存率分别为 74%和 60%。

第二项调查还发现，网飞和 Hulu 比竞争对手更具排他性——也就是说，更多的用户只为他们的服务付费。

在网飞，这个数字是 78%。相比之下，HBO NOW 只有 27%的独家用户。

该公司预测，随着消费者对流媒体内容需求的增长，未来几年对单一服务的忠诚度将继续下降。

竞争的加剧将使 HBO 更难独自发展。这就是为什么华纳媒体正在利用它的其他资产来创造一个 HBO 主导的“捆绑包”,感觉比单独的 HBO 更有吸引力。
