资本主义是 16 世纪左右在欧洲出现的社会和经济制度，我们仍然生活在其中，它与性别歧视和种族主义的历史有着深刻的关系。认识到性别歧视和种族歧视本身是历史现象非常重要。

它们并不是在所有地方都以同样的方式存在。它们在不同的时间呈现不同的形式。我觉得很有希望认识到这一点，因为这意味着他们可以改变。

非常重要的一点是，不要认为男人总是讨厌女人，这种性别战争总会存在，最好的情况是，在传统异性恋的压抑休战中得到暂时解决。我们生活的环境并不是唯一可能的环境，也不是不可避免的。

马克思主义的一个基本观点是，资本主义必然包含剥削。为了发展，一家公司需要支付给员工的报酬低于他们工作的价值。种族和性别使得这个剥削过程看起来很自然。

图片来自 Getty Images / gremlin

某些人自然倾向于做某些地位较低、工资较低的工作，为什么有人要为做自然发生的事情而获得高额报酬呢？碰巧的是，我们不太看重的工作被认为是更自然的“女性工作”这不仅仅是指那些被认为是女性的护理职业——护理和教学等，尽管也包括这些职业。

事实上，计算机编程的历史提供了一个最好的例子。在最初的几十年里，当编写软件被视为死记硬背的工作和较低的地位时，它主要是由女性完成的。正如 Mar Hicks 和其他历史学家所展示的那样，随着这个职业变得越来越有声望，越来越有利可图，女性被非常积极地排挤出去。

你甚至可以在特定的编码语言中看到这一点。比如说，随着越来越多的女性学习 Javascript，它被视为女性化的——被视为不如 Python 那样令人印象深刻或有价值，是一种“更软”的技能。这种认为女性拥有某些应该免费或廉价的自然能力的观点，有着与资本主义历史重叠的悠久历史。在某种程度上，这是雇佣劳动兴起的副产品。

对于一个中世纪的农民来说，说他的妻子带着孩子在农场工作、分娩、杀鸡做饭或做家务不算“工作”是没有意义的，(但当他)带着鸡去市场出售时，那才是。对吗？

一长串女权主义思想家以不同的方式引起了人们对这一问题的关注。70 年代的一个口号是，“谁的工作产生工人？”从这一过程中获利的女性，无论是公司还是国家，都不希望为此买单。

我为什么要说这些？我的观点是:在历史上，种族和性别对于免费获得资本主义的东西非常有用，对于证明这个过程是正确的也非常有用。当然，它们也非常有助于分化被剥削的人。所以一个白人男性员工讨厌他的黑人同事，或者他的偷腥的妻子，而不是他的老板。
