Nvidia 宣布推出 Studio 系列笔记本电脑，与 MacBook Pro 展开竞争

在台北 Computex 开幕前一天的新闻发布会上， Nvidia 宣布一个新系列的笔记本电脑将运行其 RTX 图形处理器，以及一个名为 Studio 的新软件平台，带有 SDK 和驱动程序，使图形渲染和其他任务更快。这些设备面向创意专业人士，如视频编辑、摄影师和平面设计师，旨在与 15 英寸的 MacBook Pro 竞争。

Nvidia 的新工作室笔记本电脑之一，旨在与 MacBook Pro 竞争

该系列将包括由英伟达的制造合作伙伴(包括宏碁、华硕、戴尔、技嘉、惠普、微星和雷蛇)制造的十七款笔记本电脑。这款笔记本电脑将于 6 月开始零售，起价为 1599 美元。

这 17 款笔记本电脑将配备 Quadro RTX 5000、4000 或 3000 GPUs 或 GeForce RTX 2080、2070 和 2060 GPUs。英伟达声称，它们的性能比 MacBook Pro 快 7 倍。配备 Quadro RTX 5000 GPU 的工作室笔记本电脑将拥有 16GB 的图形内存，一些设备还将配备 4K 显示器和英伟达的 Max-Q 技术，用于构建轻薄的笔记本电脑。

Nvidia Studio 套件包括 CUDA-X AI 平台，用于自动执行颜色匹配视频或标记照片等任务。
