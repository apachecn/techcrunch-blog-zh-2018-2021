参观柏林工厂，欧洲“最大的创业俱乐部”

据柏林工厂的初创公司称，这不仅仅是另一个合作空间。毕竟，这家公司的名字来自安迪·沃霍尔在纽约市的著名工厂，它自称为“欧洲最大的创业俱乐部”

去年年底，我们参观了柏林工厂位于格利泽公园的五层楼、14，000 平方米的厂房。是的，这是一个创业公司可以租用工作空间的建筑，但作为参观的一部分，我们有机会与几位企业家交谈，每个人都形容它是一个真正的社区。

“成为社区的一部分，对我们来说，意味着不要把自己与外部世界隔离开来，” Code 大学创始人汤姆·巴切姆说。“或者特别是在柏林，从我们拥有的伟大的创业生态系统中——而是真正深入地融入其中。”

同样，甜甜圈的尼尔·波帕特指出工厂的区块链活动和展示是一个主要的好处，而新学校的基普·卡特表示，他的团队已经使用工厂的信息应用程序来寻找可以与新学校的孩子一起工作的专家。

创始人兼首席执行官 Simon Hossell 说，对于不是柏林人的企业家来说，这是一个很好的基地:“事实是，你知道，尽管你可能是一个陌生人或外国人，但在一个新的城市，总有一群人——你周围志同道合、聪明、聪明的人，总是在那里帮助和鼓励你。”
