苹果发布了 5000 美元的专业显示器 XDR，这是一款 32 英寸、6K 的显示器，将于今年秋天上市

苹果终于回到了显示器的游戏中。今天，在 WWDC 2019 上，该公司揭开了 Pro 显示器 XDR 的面纱，并搭配了新的 Mac Pro。这是自 2016 年该公司停产 Thunderbolt 显示器以来的第一款苹果显示器。

屏幕上覆盖着一层新型的哑光涂层。苹果公司表示，玻璃被蚀刻以取代哑光效果，但没有缺点。重新设计的蓝色阵列塑造并控制光线，而显示器的背面充当散热器，允许显示器无限期地全屏亮度 1000 尼特。屏幕也可以旋转到纵向模式。

背面采用了与新款 Mac Pro 相同的奶酪刨丝器设计。背面也有四个雷电 3 端口。

苹果公司拥有生产一流显示器的悠久历史，当该公司停止生产其最后一款显示器时，在市场上留下了一个漏洞。取而代之的是，苹果转向了 LG 的超细线。显示器是伟大的高保真和雷电 3 连接。但它不是苹果显示器，这让许多人向往电影院显示的日子。

这款显示器自 2017 年以来一直在工作，当时苹果向 TechCrunch 透露，它正在为 2019 年发布的新显示器工作。

Pro Display XDR 将于今年秋季上市，价格为 5000 美元或 6000 美元，可选哑光涂层。支架和 VESA 支架需要额外的费用，因为是苹果的产品。
