库伯内特是如何统治世界的

开源已经成为构建软件的事实标准，这些软件支撑着复杂的基础设施，运行着从你最喜爱的移动应用到你公司几乎不可用的费用工具的一切。在过去的几年中，许多新软件被部署在 Kubernetes 之上，这是一个用于管理运行容器的大型服务器集群的工具，谷歌在五年前将其开源。

今天，Kubernetes 是发展最快的开源项目，本月早些时候，两年一次的 KubeCon+CloudNativeCon 大会吸引了近 8000 名开发人员来到阳光明媚的西班牙巴塞罗那，使该活动成为欧洲迄今最大的开源会议。

为了谈论 Kubernetes 是如何形成的，我采访了谷歌 Kubernetes 的联合创始人之一克雷格·麦克卢奇(Craig McLuckie)(他后来创办了自己的初创公司 Heptio，并将其出售给了 VMware)；Tim Hockin ，另一位谷歌人，他是这个项目的早期成员，也是谷歌 Borg 团队的成员；还有 Gabe Monroy ，他联合创立了 Deis，这是最早成功的 Kubernetes 创业公司之一，然后把它卖给了微软，他现在是 Azure Container Compute 的首席项目经理(也是微软在这一领域努力的公开代表)。
