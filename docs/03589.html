在一份罕见的公告中，美国国家安全局敦促用户修补 BlueKeep 漏洞

原文：https://web.archive.org/web/https://techcrunch.com/2019/06/05/nsa-advisory-bluekeep-patch/

美国国家安全局(National Security Agency)发布了一项罕见的建议，警告用户更新他们的系统，以防范 BlueKeep，这是一种新的安全漏洞，有能力在计算机之间迅速传播。

影响运行 Windows XP 和更高版本的计算机的“严重”级漏洞可被利用来在系统级别远程运行恶意软件，该漏洞对计算机具有完全访问权限。由于该漏洞可被远程利用，任何连接到互联网的未打补丁的计算机都可能面临风险。

只有 Windows 8 和 Windows 10 不容易受到该漏洞的影响。

微软在 5 月份发布了补丁，然而大约一百万台面向互联网的计算机和服务器仍然没有受到保护。

该情报机构敦促电脑所有者“面对日益增长的威胁”修补漏洞，因为他们担心恶意行为者可能会发动攻击，类似于 2017 年wanna cry 勒索软件攻击的规模。

截至本文撰写之时，安全研究人员只能开发出能够远程关闭受影响计算机的概念验证漏洞，即所谓的拒绝服务攻击。但他们表示，这些漏洞被用来发送勒索软件或窃取数据的恶意软件只是时间问题。

“国家安全局敦促每个人投入时间和资源来了解你的网络，并使用最新的补丁运行支持的操作系统，”该机构的建议说。

很少看到 NSA 干预消费者网络安全事务。国家安全局发言人指出，其 BlueKeep 咨询是该机构今年的第三份网络安全通知。在 NSA 经常利用漏洞进行监视和间谍活动的地方，通常是国土安全部发布关于可能被黑客利用的严重安全缺陷的建议。

两年前，该机构因高度机密的黑客工具被盗而颜面扫地，黑客后来在网上公布了这些工具。泄露的永恒之蓝漏洞像万能钥匙一样工作，打开了互联网上几乎 10 亿多台 Windows 电脑中的任何一台。据信与朝鲜有关的黑客利用泄露的永恒之蓝漏洞大规模传播勒索软件。在安全研究人员发现了一个中和恶意软件的“杀死开关”后，恶意软件才停止传播。

国家安全局从未公开承认盗窃。

愤世嫉俗者可能会看到，美国国家安全局正在积极行动，以避免另一场公关灾难，此前它的一项最高机密漏洞被泄露，并被用于全球勒索软件攻击。然而，乐观主义者可能会说，政府正试图警告用户，如果某个漏洞被利用或公布，要防止大规模破坏。

就其本身而言，NSA 表示，修补 BlueKeep“不仅对 NSA 保护国家安全系统至关重要，对所有网络也是如此。”
