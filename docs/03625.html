财富 500 强巨头技术数据暴露客户和计费数据 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/06/06/tech-data-server-leak/

安全研究人员表示，IT 巨头 Tech Data 的安全漏洞允许他们访问客户和账单数据。

在研究人员 Noam Rotem 和 Ran Locar 发现并报告泄漏数据后不久，这家财富 500 强信息技术巨头就保护了一台暴露的服务器。

该服务器运行一个数据库，用于记录其 StreamOne 云服务的内部公司事件，该服务允许客户从各种提供商和供应商购买云服务。记录数据包含错误数据，技术数据人员可以使用这些数据来解决客户尝试在线购买服务时出现的问题。

但这家科技巨头没有在服务器上设置密码，允许任何有网络浏览器的人查看过去几个月的每日日志。

Rotem 和 Locar 与 TechCrunch 独家分享了他们的发现，发布了他们的发现。

TechCrunch 还获得了一部分记录，我们对其真实性进行了检验。

该数据库包含一系列数据，但我们发现了大量客户数据，包括姓名、邮政地址和电子邮件地址、职位以及发票数据和收据。这些记录还包含部分支付信息，如卡的类型、持卡人姓名和到期日期。

除了模糊的卡号，所有数据都没有加密。

不知道数据库中到底有多少客户记录。我们获得的那部分数据包含数万名客户的数据，但数据库的规模要大得多。

Rotem 和 Locar 说他们还发现了私人钥匙，在某些情况下还发现了密码。

在一次泄露后，数据库被关闭。我们向 Tech Data 发送了几个问题——特别是它是否计划向客户或监管机构通报安全漏洞——但该公司没有回复我们在发表前发送的电子邮件和后续邮件。

在我们发表后，发言人 Bobby Eagle 证实了这一消息。“在得知这一情况的几个小时内，安全漏洞就被纠正了，服务器也被禁用了，”他说。但是公司没有回答我们的具体问题。

这是研究人员最近几个月发现的一系列暴露数据库中的最新一个。

本周早些时候，研究人员披露了一个公开的数据库，暴露了犹太交友应用 JCrush 的用户记录和私人信息。他们之前的发现包括加拿大手机网络自由移动和在线零售商 Gearbest 。
