AWS 现在让所有客户都可以使用亚马逊个性化服务

亚马逊个性化服务，在去年 11 月 AWS re:Invent期间首次发布，现在所有亚马逊网络服务客户都可以使用。该 API 使开发人员能够在他们的应用程序中添加定制的机器学习模型，包括个性化产品推荐、搜索结果和直接营销的模型，即使他们没有机器学习经验。

该 API 使用最初为亚马逊自己的零售业务 创建的算法处理数据，但该公司表示，所有数据都将“完全保密，完全由客户所有”该服务现已面向美国三个地区(东部(俄亥俄州)、东部(北弗吉尼亚)和西部(俄勒冈州)、两个亚太地区(东京和新加坡)以及欧盟的爱尔兰的 AWS 用户，更多地区将很快推出。

已经将 Amazon Personalize 添加到其应用程序中的 AWS 客户包括美国雅马哈公司、赛百味、Zola 和 Segment。在亚马逊的新闻稿中，美国雅马哈公司信息技术总监 Ishwar Bharbhari 表示，亚马逊个性化“与我们自己构建和配置环境相比，为我们的机器学习模型节省了高达 60%的设置和调整基础设施和算法所需的时间。”

亚马逊个性化的 定价模式对上传到亚马逊个性化的每 GB 数据收取 5 美分，对使用他们的数据训练定制模型的每训练小时收取 24 美分。实时推荐请求根据上传的数量来定价，订单越多越有折扣。
