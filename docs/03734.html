研究人员称，一个广泛使用的输液泵可以被远程劫持

原文：https://web.archive.org/web/https://techcrunch.com/2019/06/13/alaris-infusion-pump-security-flaws/amp/

图片来源:

据安全研究人员称，一个用于对接广泛用于医院和医疗设施的输液泵的工作站存在严重的安全缺陷，允许它被远程劫持和控制。

医疗安全公司 CyberMDX 的研究人员在医疗设备制造商 Becton Dickinson 开发的 Alaris Gateway 工作站中发现了两个漏洞。

输液泵是医院里最常见的设备之一。这些设备控制静脉输液和药物的分配，如止痛药或胰岛素。它们通常连接到一个中央监控站，这样医务人员就可以同时检查多个病人。

但是研究人员发现，攻击者可以在泵的机载计算机上安装恶意固件，该计算机为输液泵供电、监控和控制。该网关运行在 Windows CE 上，在智能手机出现之前，Windows CE 通常用于掌上电脑。

在最坏的情况下，研究人员表示，通过安装修改后的固件，可以在某些版本的设备上调整泵的特定命令，包括输注速度。

研究人员表示，也有可能远程砖板载计算机。

根据国土安全部的公告，该漏洞在行业标准通用漏洞评分系统中获得罕见的最高分 10.0。第二个漏洞得分较低，为 7.3 分(满分为 10.0 分)，可能允许攻击者通过 web 浏览器访问工作站的监控和配置界面。

CyberMDX 的研究负责人埃拉德·鲁兹(Elad Luz)在给 TechCrunch 的一封电子邮件中说，研究人员表示，创建一个攻击套件“非常容易”，并且“工作稳定”。但是攻击链很复杂，需要多个步骤、访问医院网络、了解工作站的 IP 地址以及编写定制恶意代码的能力。

换句话说，有比利用这些细菌更容易杀死病人的方法。

CyberMDX 于 11 月向 Becton Dickinson 和联邦监管机构披露了这些漏洞。

Becton Dickinson 表示，设备所有者应该更新到最新的固件，其中包含对漏洞的修复。发言人特洛伊·柯克帕特里克(Troy Kirkpatrick)表示，该泵不在美国销售，但不愿透露有多少设备“出于竞争原因”容易受到攻击

“大约有 50 个国家拥有这些设备，”柯克帕特里克说。他证实，8 个国家拥有超过 1000 台设备，3 个国家拥有超过 2000 台设备，但没有一个国家拥有超过 3000 台设备。

这些缺陷再次提醒人们，任何设备都可能存在安全问题，尤其是医疗领域的救生设备。

今年早些时候，国土安全部就美敦力除颤器的一系列严重漏洞发出警告。政府发布的警报称，该设备的专有无线电通信协议不需要认证，允许附近的攻击者在特定情况下通过无线方式拦截和修改命令。
