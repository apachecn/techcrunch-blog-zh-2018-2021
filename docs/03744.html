Target 推出了一个由 Shipt 提供支持的当天送达的专用购物网站

原文：https://web.archive.org/web/https://techcrunch.com/2019/06/13/target-launches-a-dedicated-shopping-site-for-same-day-delivery-powered-by-shipt/

在亚马逊的计划提高 Prime 速度以及沃尔玛最近在选定市场推出免费次日送达的消息传出后，塔吉特正在加大其当天送达的努力。今天早上，塔吉特宣布直接从网站上购买当天精选商品变得更加容易，送货由 Shipt 提供。

零售商在 2017 年以 5.5 亿美元收购了杂货配送服务 Shipt，以加速其数字化履行努力。交易完成后，Target 继续通过 Shipt 为杂货和其他日常需求(如家居用品、宠物和婴儿用品)提供当天送达服务。虽然 Shipt 也为其他当地杂货商和商店送货，但 Target 在 Shipt 的价格并没有提高。

然而，到目前为止，塔吉特购物者必须登录 Shipt 网站或下载应用程序，才能浏览塔吉特百货当天的商品。这不是一次完整的经历。随着今天在 target.com/sameday开设的 Target.com 购物专区的推出，这种情况有所改变。

在这里，网上购物者可以浏览 Target.com 提供的 65，000 种当天送达的商品，包括杂货、家居用品、宠物用品、婴儿用品甚至一些服装。他们还可以轻松地重新订购他们以前购买的商品，或者搜索特定类别的新商品进行订购。

据我们所知，Target.com 当天的商品种类与 Shipt 应用程序提供的商品大致相似，但成人饮料除外，成人饮料仅在特定市场通过 Shipt 提供。

Target.com 的当天购物体验提供了与 Shipt 网站不同的外观和感觉——它不仅仅是 Shipt 网站周围的目标品牌包装。

塔吉特的网站在页面顶部提供了可点击的类别，比如“杂货”、“家庭必需品”、“宠物”、“婴儿”等等，每个类别都有一个彩色的图标。单个产品页面看起来也像 Target.com 上的任何其他产品页面，有各种取件或送货选项，以及订阅折扣定期购买，或将商品添加到列表或注册表中的功能。

关键的区别是，现在的交付选项是“当天”，这是以前没有的。(如果订单不紧急，还可以通过次日送达服务 Target Restock 发货。)

在结账时，购物者可以选择支付每份 9.99 美元的当天送达费，也可以选择为期四周的 Shipt 年度会员资格试用。如果你选择后者，在免费期结束后，除非你取消，否则试用将转换为每年 99 美元的会员资格。

Shipt 会员资格还包括从其他地方和国家零售商处购物的选项。该应用支持 60 多家零售商，但会员只能看到附近的零售商。

在 Target.com 推出专门的当天选择服务，这是没有 Shipt 会员资格的目标购物者首次可以付费利用当天送货服务。该选项加入了 Target 允许人们在其网站上购物的其他几种方式，包括当天商店提货、免下车服务(路边提货服务)、通过 Target Restock 的第二天送货以及传统送货。

对于 Target REDcard 持有者来说，使用新的当天网站还有另一个好处——他们现在可以用 REDcard 支付，节省 5%的购物费用。

此外，塔吉特的在线购物者可以利用每周的广告促销活动，比如“用价值 5 美元的塔吉特礼品卡购买 3 件商品”，这在 Shipt 中是不可能的。

与 Shipt 一样，Target.com 的当天购物者可以在送货前一个小时编辑或添加商品，并保存多个与其账户相关的地址。

Target Digital 高级副总裁 Dawn Block 在一份声明中表示:“通过在 Target.com 体验中直接提供当天送货服务，我们让我们的客人在 Target 购物变得更加容易，同时还能获得他们所期望的超值、精选的产品组合和有益的客户服务。”。
