Plaid 将 Quovo 收购权用于新投资产品

今年早些时候，当 Plaid 收购 Quovo 时，很明显，该公司希望超越支票和储蓄数据，转向更广泛的投资视角。今天，该公司宣布投资，这是一个基于 Quovo 的 API，使客户能够将投资数据纳入他们的应用程序。

T2 格子公司合作伙伴负责人、Quovo 公司前首席执行官和联合创始人洛厄尔·普特南说，客户和开发者一直渴望这种数据，收购 Quovo 有助于加速格子公司内部的这种需求。

“我认为产品的起源来自于我共同创立的一家公司，也来自于过去几年对格子花呢的广泛需求。普特南告诉 TechCrunch:“我们的许多客户都希望获得客户的 360 度全景，这并不奇怪。

投资账户与支票账户或储蓄账户相比具有独特的挑战，支票账户或储蓄账户在任何给定时间都有一定数量的现金。Putnam 说,“投资账户不是在一个账户中只持有现金，而是在同一个账户中持有多种证券，这些证券的价值会发生变化，不管(你是否从事贸易活动)。

Plaid 认为这种能力极大地扩展了开发人员使用 API 的方式。该公司在宣布新产品的博客文章中写道:“通过连接到投资账户，Plaid 可以服务于一系列全新的用例，使开发者、技术平台和金融机构能够提供更有洞察力和更具吸引力的产品。”

他说，两家公司认识到将这种信息引入任何金融应用程序的力量，甚至在收购之前就认识到了这一点，并且从那以后他们一直在努力实现这一点。“我们意识到，我们有机会将 Quovo 的投资数据专业知识引入 Plaid API 堆栈，并通过我们过去几年一直在思考的所有调整、变化和增强来实现这一点，”他说。

新产品从今天开始在美国上市。今年还将在其他地区推广。
