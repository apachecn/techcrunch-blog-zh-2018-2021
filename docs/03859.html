树莓基金会推出树莓Pi 4

原文：https://web.archive.org/web/http://techcrunch.com/2019/06/23/the-raspberry-pi-foundation-unveils-the-raspberry-pi-4/

树莓Pi 4已经上市，这是一次令人惊叹的升级。之前有传言称，树莓Pi的重大升级还需要一段时间，但今天就可以开始了。

说到物理设计，Raspberry Pi 4 Model B看起来很像之前的旗舰机型Raspberry Pi 3 Model B+这是一台单板计算机，有许多连接器，大小相当于一副纸牌。

但是一切都更新了。它从更快的片上系统开始。处理器现在使用Cortex-A72架构(1.5GHz的四核64位ARMv8)。例如，它支持H.265硬件视频解码。

树莓派多年来一直停留在512MB或1GB的内存上。第一次，想要更大内存可以买内存更大的型号。基本型号仍然从1GB内存开始。但你可以选择购买2GB内存甚至4GB内存的型号。

除了原始内存容量，内存传输速度应该更快，因为基金会正在从LPDDR2切换到LPDDR4。

Raspberry Pi基金会已经给了我一个Raspberry Pi 4，我计划运行一些基准测试并分享结果。我只是在等待Raspbian的更新，因为现有的版本不能在新的架构上运行——我意识到，在格式化microSD卡后，用Raspbian Lite (oopsie)取代了预装的NOOBS操作系统。

谈到连接，两个大的变化是，你现在得到了真正的千兆以太网(而不是USB 2.0上的以太网)。它应该为服务器和无头Raspberry Pi设备开辟了大量的潜在用例。

现在有两个USB 3.0端口和两个USB 2.0端口。现在，您可以为电源模块获得一个USB-C端口。蓝牙也正在从蓝牙4.2升级到蓝牙5.0。

最后一个大的硬件变化是，全尺寸的HDMI端口不见了。你现在有两个微型HDMI端口，可以使用一个Raspberry Pi以每秒60帧的速度插入两个4K显示器。我还没有测试那个设置。例如，我确信在你的办公室运行两个statics仪表盘是不错的，但是我不期望疯狂的双屏性能。

其余的规范对于过去用过树莓派的人来说应该很熟悉。有一个microSD卡插槽，这样你就可以把操作系统和用户数据放在存储卡上。有一个40针GPIO接头，应该与现有的附加板兼容。

该产品今天通过授权的覆盆子Pi零售商推出。基本型号仍为35美元，而2GB RAM型号为45美元，4GB RAM型号为55美元。

覆盆子Pi最初只是一台简单的电脑，用来教孩子如何编程，但现在它已经成为一款拥有多种使用场合的多功能设备。在过去的几年中，我一直在使用一些，并在编程、系统管理、Docker容器和网络方面学到了很多。看来，今天的更新将是一个热门的孩子，家长和制造商。
