谷歌新的媒体素养项目教孩子们如何辨别虚假信息和假新闻——第767页——TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/06/24/googles-new-media-literacy-program-teaches-kids-how-to-spot-disinformation-and-fake-news/767/

谷歌 今天早上宣布，它正在扩大其为期两年的儿童数字安全和公民课程“成为互联网牛逼”，现在包括媒体素养——特别是识别所谓的“假新闻”和其他虚假内容的能力。该公司正在为课程推出六项新的媒体素养活动,这将有助于教孩子们如何避免网络钓鱼攻击，什么是机器人，如何验证信息是否可信，如何评估来源，如何识别网上虚假信息，识别虚假网址等等。

新的媒体素养课程——坦率地说，一些成年人也应该通读——是与网络安全合作组织的执行董事安妮·科利尔(Anne Collier)和《媒体素养教师指南》(The Teacher ' s Guide to Media Literacy)的合著者、国家媒体素养教育协会(National Association for Media Literacy Education)的共同创始人费丝·罗戈(Faith Rogow)博士合作开发的。

“我们需要正确的工具和资源来帮助孩子们充分利用技术，虽然家庭有良好的数字安全和公民资源，但在媒体素养方面还可以做得更多，”教育家和teachmama.com创始人艾米·马斯科特(Amy Mascott)今天在谷歌博客上的声明中写道。“我曾与数十名教育工作者一起工作，他们认为媒体素养对于数字时代的安全和公民意识至关重要，但也同意这是一个很难涵盖的话题。”

这些课程不仅为孩子们提供指导，还结合了各种活动和讨论，旨在帮助他们在寻求在线资源时发展批判性思维技能。

课程材料解释说，它的总主题是帮助孩子们理解他们在网上找到的内容不一定真实或可靠，甚至可能涉及恶意窃取他们的信息或身份。

孩子们学习网络钓鱼是如何工作的，为什么它是一种威胁，以及如何避免它。然后，他们通过表演和讨论对可疑在线文本、帖子、好友请求、图片和电子邮件的反应来练习他们的反网络钓鱼技能。

在机器人部分，他们学习人工智能如何工作，并比较和对比与机器人交谈和与人类交谈。

在接下来的媒体素养部分，孩子们学习什么是可信的消息来源，如何判断消息来源的动机，并学习“仅仅因为一个人是某一方面的专家并不意味着他们是所有方面的专家。”

在一个相关的课堂活动中，孩子们选择一个与他们在网上看到或在课堂上学到的东西相关的问题，并试图在网上获得答案，同时判断来源是否可信。

他们还学会用其他可信来源来核实可信来源，以此来寻找各种来源。

该课程解释道:“如果你找不到各种可信的来源与你正在检查的来源一致，你就不应该相信那个来源。”

此外，孩子们还被教导如何利用欺骗性网址等线索识别虚假信息，以及检查信息来源的可信度。他们被告知，有些人不知道如何做到这一点，并在网上分享虚假信息——这就是它的传播方式。

“有很多人和团体对他们的信仰如此狂热，以至于他们扭曲事实来让我们同意他们的观点。当扭曲的信息被伪装成新闻故事时，那就是虚假信息，”该课程说。

孩子们还被告知，一些假新闻组织很难被发现，因为他们使用的名字听起来像真的一样。

该课程还探究了一些网站使用的各种技巧——比如使用与故事无关的照片，使用“令人震惊”或“令人发指”等他们知道会让人好奇的点击诱饵词，“使用粗体、下划线、感叹号或全大写字母”，以说服你同意他们的观点。

这一部分以一个在线游戏“现实之河”结束，这个游戏要求孩子们运用他们最好的判断来穿越急流。这发生在因兰德，这款游戏是作为谷歌数字安全和公民课程的配套开发的。

媒体素养课程的总体目标是鼓励孩子们养成查看所有新闻和信息的习惯，而不仅仅是那些他们认为可疑的。

谷歌表示，新课程可以在网上供教师和家庭使用，并以英语、西班牙语和其他八种语言提供。

谷歌还与多个城市的基督教青年会(T2)和国家家庭教师协会(T4)合作举办在线安全研讨会。
