WordPress 管理网站 WP Engine 收购 Flywheel，其估值和首次公开募股(IPO)将达到 100 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2019/06/24/wordpress-management-site-wp-engine-acquires-flywheel-as-it-moves-to-a-1b-valuation-and-ipo/

WordPress 现在占全球所有网站的 34 %,今天，帮助处理 WP 托管网站的创建和管理的主要公司之一正在通过更广泛的生态系统中的一些整合而变得更大。WP Engine ，与企业合作建立和管理他们的 WordPress 托管网站，已经收购了一个较小的竞争对手飞轮。

WP Engine 的首席执行官兼董事长希瑟·布鲁纳(Heather Brunner)在接受采访时表示，交易的财务条款尚未披露，但她向 TechCrunch 证实，该交易涉及她的公司从现有投资者那里筹集一小轮资金(金额也未披露)，以帮助为交易融资。WP Engine 的投资者包括银湖(去年向该公司投入了2.5 亿美元的巨额资金)和 GuidePost Growth Equity(原名北桥)。WordPress 开发者 Automattic、Silverton 和 Eric Ries(因“精益创业”而出名)是前投资者。

布鲁纳也拒绝谈论 WP Engine 的估值，尽管她指出，目前的年经常性收入为 1.32 亿美元，飞轮为 1，800 万美元，目前的增长率为 50%，两家公司有望在 2020 年前在 ARR 中赚到 2 亿美元，并有可能在上市途中突破 10 亿美元。

“我们的目标是建立一家上市公司，此次收购是实现这一目标的一部分，”她说。(具有讽刺意味的是，这可能意味着 WordPress 的合作伙伴，有时是竞争对手，可能会先于它上市。)

这笔交易标志着围绕 WordPress 建立的生态系统的整合。WP Engine 是这个生态系统中名副其实的发电站，是这个领域的早期推动者——WordPress 早在 2011 年就支持它——现在正在为 150 个国家的大约 12 万个品牌和代理机构建立和管理网站(实际网站数量可能是这个数字的数倍)。

正如布鲁纳所描述的，WP Engine 主要专注于中端市场和大型企业，而飞轮公司——成立于奥马哈，目前总部位于奥马哈——则专注于小型企业。这使得这两个自然互补，但布鲁纳指出，将有更多的联盟获得。

“那里的团队非常注重产品，”她指出。“他们已经建立了一个套件，我们认为它主要面向小型机构，但它们也是大型机构将从中受益的工具类型。”她指的是 Flywheel 的产品 Local，这是一个由超过 150，000 名开发者使用的本地开发应用程序。

飞轮公司成立于 2012 年，只筹集了大约 600 万美元的资金，包括几年前的一轮 400 万美元的融资。WP 引擎带来的规模经济将使它获得更广泛的曝光率和接触新客户的机会。

飞轮首席执行官兼联合创始人达斯丁·戴维森在一份声明中说:“我们创立飞轮的信念是，为了帮助创意人员做最好的工作，我们需要创造一种鼓励我们的员工也这样做的内部文化。”“这一理念引导我们建立了一个不可思议的公司和一些在 WordPress 中最受欢迎的产品，由一群令人印象深刻的人才和世界上最卓越的开源社区提供支持。”

在此之前，WP Engine 已经收购了 WordPress 生态系统中的其他合作伙伴，使其成为该领域的整合者。Brunner 指出，虽然该公司的一些增长努力可能会导致进一步的收购，但它也在寻求与第三方合作伙伴合作的第二条道路，并作为公司引入其他服务以帮助运营其网站的中介平台。她指出，WP 引擎生态系统中的合作伙伴——当然还有 WordPress 本身——包括亚马逊网络服务、Cloudflare、谷歌、HubSpot 和 New Relic。
