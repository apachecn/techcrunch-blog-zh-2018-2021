由软银视觉基金牵头，可互换融资 2 亿美元，帮助公司处理越来越多的海量数据

原文：https://web.archive.org/web/https://techcrunch.com/2019/06/27/fungible-raises-200-million-led-by-softbank-vision-fund-to-help-companies-handle-increasingly-massive-amounts-of-data/

由软银视觉基金(SoftBank Vision Fund)牵头，可互换基金筹集了 2 亿美元，用于帮助公司处理日益庞大的数据量

可替代，一家希望帮助数据中心应对新技术产生的越来越多的海量数据的初创公司，已经筹集了 2 亿美元由软银视觉基金(SoftBank Vision Fund)牵头的 C 系列，参与方包括诺威斯特风险投资公司及其现有投资者。作为这一轮谈判的一部分，软银投资顾问公司的高级管理合伙人 Deep Nishar 将加入可互换公司的董事会。

成立于 2015 年的可替换公司现在有大约 200 名员工，已经筹集了超过 3 亿美元的资金。它的其他投资者包括电池企业，梅菲尔德基金，红线资本和瓦尔登河伍德企业。它的新资本将用于加快产品开发。该公司的创始人兼首席执行官帕拉德普·辛杜和伯特兰·塞莱特表示，可替换公司将在今年晚些时候发布更多信息，说明其数据处理部门何时可以投入使用以及他们的安装过程。他们表示，安装过程不会要求客户改变他们现有的应用程序、网络或服务器设计。

Sindu 之前创建了 Juniper Networks，在那里他担任首席科学家和首席执行官。在 2011 年离开苹果并创建 up tole 之前，Serlet 是苹果软件工程部门的高级副总裁。up tole 是一家存储初创公司，于 2017 年被西方数字收购。Sindu 和 Serlet 将可互换的目标描述为将数据中心从“以计算为中心”的模式转变为以数据为中心的模式。虽然该公司经常被问及是否考虑英特尔和英伟达的竞争对手，但他们表示，可互换数据处理单元(DPU)是对其他芯片制造商技术的补充，包括中央和图形处理单元。

Sindhu 将可替换的分布式处理单元描述为数据中心基础设施中的一个新的构建模块，允许它们更有效地处理更大数量的数据，也有可能支持新类型的应用。它的 dpu 是完全可编程的，并通过以太网局域网和本地总线(如 PCI Express)与标准 IPs 连接，后者又连接到 CPU、GPU 和存储。介于两者之间的分布式处理单元就像一个“超级充电的数据流量控制器”，执行由中央处理器和分布式处理单元卸载的计算，以及将 IP 连接转换成高速数据中心结构。

Sindu 表示，这可以更好地为数据中心准备新技术产生的大量数据，包括自动驾驶汽车，以及个性化医疗保健、金融服务、云游戏、农业、呼叫中心和制造业等行业。

在一份新闻声明中，Nishar 表示，“随着全球数据爆炸和人工智能革命的展开，全球计算、存储和网络基础设施正在经历根本性的转变。可替代的产品使数据中心能够利用其现有的硬件基础设施，并从这些新技术范例中受益。我们期待与该公司富有远见且卓识的管理团队合作，为下一代数据中心提供支持。”
