来源:跨境支付 API Currencycloud 在 E 轮 TechCrunch 的第一部分融资 3200 万英镑

原文：https://web.archive.org/web/https://techcrunch.com/2019/07/02/hey-you-get-off-of-my-series-e-cloud/

来源:跨境支付 API Currencycloud 已经在 E 轮融资的第一部分筹集了 3200 万英镑

Currencycloud 是一家为跨境支付提供 API 和服务的公司，被许多金融科技公司和大型公司使用，包括最近的 Visa，它已经完成了一轮 E 轮融资的第一部分。

据消息人士称，这家总部位于伦敦、已有 10 多年历史的公司几周前在内部宣布，它正在接受新的融资，而最近的监管文件显示，迄今为止，E 系列发行的新股总数接近 3200 万股。然而，我知道这只是第一批，额外的 E 轮融资将在接下来的 2-3 个月内正式宣布。第一档还包括两个略有不同的股价，因为它看到早期的债务融资转换为股权。

关于谁在支持 Currencycloud 的 E 轮融资，一位消息人士告诉我，高盛也在竞选中，可能会在这一轮中领先。据说，现有投资者 GV (之前的谷歌风投)正在跟进。我还听说，另一个新的投资者可能是西班牙桑坦德银行，通过其风险部门桑坦德创新风险投资公司，这将标志着一个重大的战略投资和/或合作伙伴关系。Currencycloud 拒绝置评。

Currencycloud 在 2011 年筹集了 A 轮融资后，于 2012 年更广泛地推出，长期以来被视为伦敦金融科技生态系统的支柱(该公司在早期甚至被 TransferWise 大量使用)，为跨境支付和转账建立了支付基础设施。具体来说，该公司为需要向客户提供国际转账的企业提供了一个 API。

现在，该公司在欧洲以及美国和加拿大开展业务，迄今已处理超过 500 亿美元的转账，向 180 多个国家汇款。Currencycloud 在全球范围内合作的银行和金融科技公司包括 Starling Bank、Standard Bank South Africa、Visa、Travelex 和 Klarna。该公司的团队现在拥有 200 多名员工(见上图)，分布在多个国际办事处，包括伦敦、阿姆斯特丹和纽约。
