大众的 ID R 电动赛车不断打破记录，这次是在古德伍德两次

打破古德伍德速度爬山节记录花了 20 年时间。而且在同一个周末发生了两次。

新的纪录保持者是大众的 ID R 电动赛车。由车手 Romain Dumas 驾驶的这辆车以 41.18 秒的成绩在英格兰南部的古德伍德完成了 1.86 公里(1.15 英里)的赛道，打破了纪录。第二天，杜马斯以 39.90 秒的成绩完成了比赛，打破了自己的记录。

著名的爬山赛之前的纪录是由尼克·海菲尔德在 1999 年创造的，他驾驶的是一辆 780 马力的迈凯轮-梅赛德斯 MP4/13 内燃机车。海菲尔德以 41.6 秒跑完全程。

大众 ID R(和 Dumas)最近一直在狂奔，在纽博格林-诺德施莱夫赛道和派克峰赛道创下纪录。ID R 以 6 分 05 秒 336 的成绩在纽博格林-诺德施莱夫赛道完成了比赛，创造了新的电动汽车单圈纪录。之前的纪录是由彼得·邓布利多(Peter Dumbreck)在 2017 年创造的，他当时驾驶的是 Nio 电动汽车。

大众 ID R 被改装成最适合古德伍德爬山。一辆车配备了一个更小的电池，这有助于减轻重量，功率输出为短距离冲刺进行了优化。减阻系统是为纽博格林赛道的高速路段设计的。

为什么所有这些都很重要？这款赛车是大众 ID 电动汽车平台的测试平台和旗舰产品。

大众汽车几年来一直在炫耀其概念电动汽车的 ID 系列。该公司正准备生产其中一些，从 ID.3 开始。大众的目标是每年销售 10 万辆 ID.3 汽车。

ID.3 掀背车是第一款基于该汽车制造商新的模块化电动驱动工具包(即 MEB 电动汽车架构)的车型。MEB 于 2016 年推出，是一个灵活的模块化系统——实际上是一个普通零件的矩阵——用于生产电动汽车，大众表示，这使其更高效、更具成本效益。

其他人将很快跟上。大众计划推出 20 多款全电动车型。该汽车制造商的目标是到 2025 年每年销售 100 万辆电动汽车。
