Raspberry Pi 4并不适用于所有USB-C电缆

树莓派4是一个巨大的小怪兽，但是泰勒·伍德T2发现了USB Type-C连接器的一个缺陷。Raspberry Pi基金会向TechRepublic 证实，设计缺陷是真实的，你的Raspberry Pi 4可能无法与所有USB-C电缆兼容。

这不是一个真正的交易破坏者，但你可以期待未来的电路板修订与USB-C协议的正确实施。但是如果你发现自己在挠头，不明白为什么你的树莓派不上电，现在你知道为什么了。

Raspberry Pi基金会发布了电路板的原理图。还有一个缺失的CC电阻，让复杂的充电器与设备协商电流。

鉴于USB-C是一种复杂的连接器，一些电缆有电子标记，这意味着它们有一个集成的芯片来支持多种设备。

例如，您可以使用带有大量USB-C设备的MacBook Pro充电器。充电器只是计算出它需要提供多少电力。

但Raspberry Pi 4不支持电子标记的电缆，如苹果的USB-C电缆或谷歌的Pixel 3电缆。该设备被错误地识别为音频适配器附件。

幸运的是，它不会损坏树莓派4，也不会产生任何火灾隐患。设备就是不通电。

“我希望这个问题会在未来的主板版本中得到解决，但现在用户需要应用建议的解决方法之一。令人惊讶的是，这并没有出现在我们(相当广泛的)实地测试项目中，”Raspberry Pi基金会创始人Eben Upton告诉TechRepublic。

一个简单的解决方法是购买非e标记的电缆或充电器。例如，Raspberry Pi基金会正在销售一款8美元的USB-C充电器。在我的测试中，它在过去的几周里运行良好。
