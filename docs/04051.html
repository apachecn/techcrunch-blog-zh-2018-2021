苹果推出了一个静默的 Mac 更新来移除隐藏的 Zoom web 服务器

原文：https://web.archive.org/web/https://techcrunch.com/2019/07/10/apple-silent-update-zoom-app/

苹果推出了一个静默的 Mac 更新来移除隐藏的 Zoom web 服务器

苹果为 Mac 用户发布了一个静默更新，删除了流行的视频会议应用 Zoom 中的一个易受攻击的组件，该组件允许网站在未经用户许可的情况下自动将用户添加到视频通话中。

总部位于加州库比蒂诺的科技巨头告诉 TechCrunch，现在发布的更新删除了隐藏的网络服务器，Zoom 在用户安装应用程序时悄悄安装在他们的 MAC 上。

苹果表示，更新不需要任何用户交互，是自动部署的。

在乔纳森·莱茨舒(Jonathan Leitschuh)周一公开漏洞披露后，视频会议巨头受到了用户的抨击，他在其中描述了“任何网站都可以在未经用户许可的情况下，在激活摄像机的情况下，强行将用户加入变焦通话。”即使用户卸载了 Zoom，未记录的 web 服务器仍保持安装状态。Leitschuh 表示，这使得 Zoom 可以在不需要任何用户交互的情况下重新安装应用程序。

他还发布了一个演示该漏洞的概念验证页面。

虽然 Zoom 在周二发布了一个修复的应用程序版本，但苹果表示，它的行动将保护过去和现在的用户免受未记录的网络服务器漏洞的影响，而不会影响或阻碍 Zoom 应用程序本身的功能。

更新后会提示用户是否要打开应用程序，而之前它会自动打开。

苹果经常向 MAC 电脑推送静默签名更新，以阻止已知的恶意软件——类似于反恶意软件服务——但苹果很少公开对已知或流行的应用采取行动。该公司表示，它推出更新是为了保护用户免受暴露的网络服务器带来的风险。

Zoom 发言人普里西拉·麦卡锡(Priscilla McCarthy)告诉 TechCrunch:“我们很高兴能与苹果合作测试这一更新。我们希望 web 服务器问题今天能够得到解决。我们感谢用户的耐心，因为我们将继续努力解决他们的问题。"

全球 750，000 家公司的 400 多万用户使用 Zoom 进行视频会议。
