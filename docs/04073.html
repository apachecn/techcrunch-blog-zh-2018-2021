微软售价 399 美元的 Azure Kinect 人工智能相机现已在美国和中国上市

今年早些时候，在 MWC，微软宣布以人工智能开发者套件的形式回归其 Kinect 传感器。399 美元的 Azure Kinect DK 相机系统包括一个 1MP 深度相机、360 度麦克风、12MP RGB 相机和一个方位传感器，所有这些都在一个相对较小的封装中。该套件已经可以预订几个月了，但正如该公司今天宣布的那样，它现在已经正式上市，并向美国和中国的预订客户发货。

与最初的 Kinect 不同，Azure Kinect 是作为 Xbox 游戏附件推出的，从未在流行过，它完全是商务版。它旨在为开发人员提供一个平台，以试验人工智能工具，并插入 Azure 的机器学习服务生态系统(尽管使用 Azure 不是强制性的)。

为了帮助开发者入门，该公司已经推出了许多 SDK，包括一个身体跟踪 SDK 的预览版，这与你可能记得的 Kinect 的 Xbox 时代很相似。

T2】

相机的核心与微软的 HoloLens 的关系比最初的 Kinect 更大。正如该公司在新闻材料中指出的那样，Azure Kinect DK 使用了该公司为其第二代 HoloLens AR visor 开发的相同的飞行时间传感器。虽然这里的重点显然是使用摄像头，但微软也指出，麦克风阵列也允许开发人员构建复杂的语音解决方案。

该公司将该设备定位为健康和生命科学、零售、物流和机器人领域的用户开始尝试使用深度传感和机器学习的简单门户。我们已经从其他公司看到了一些类似的开发套件，包括微软合作伙伴高通，尽管这些设备通常没有使 Kinect DK 成为 Kinect 的深度摄像头。
