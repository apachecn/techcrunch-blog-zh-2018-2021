SEC 批准首个消费者 RegA+令牌道具用于跨应用程序奖励

原文：https://web.archive.org/web/https://techcrunch.com/2019/07/11/props-reg-a-token/

在打击 ico 后，SEC 刚刚批准了前两种 RegA+代币，这种代币为任何人提供了一种获得公司财务股份的替代方式，即使是未经认证的投资者。Blockstack 获得了批准进行 2800 万美元的数字代币销售以筹集资金，而影响者直播应用 YouNow 的衍生道具获得了消费者实用工具“Howey”的正式绿灯，用户可以在多个应用中获得忠诚度优惠。

Props 已经通过向 Union Square Ventures、Comcast、Venrock、Andreessen Horowitz 的 Chris Dixon 和 YouTuber Casey Neistat 预售代币筹集了 2100 万美元，因此它没有通过出售 Blockstack 等代币从 RegA+筹集任何资金。相反，用户通过与 YouNow 等应用程序合作来赚取或“挖掘”道具，这些应用程序将奖励用于创建广播、观看视频和给创作者小费的代币。拥有更多道具可以让 YouNow 的 4700 万注册用户获得额外功能、VIP 身份和更强的购买力，以及该应用程序的专有积分“酒吧”,迄今为止用户已经购买了价值 7000 万美元的酒吧。

但与大多数只能在单个应用程序中使用的虚拟货币不同，技术上不属于消费者，开源道具区块链系统可以通过 API 集成到其他应用程序中，人们可以将他们的道具导出到加密货币钱包中。这让他们可以在 YouNow 之外的其他应用程序中应用他们的道具。四款合作应用已经准备就绪，包括 xSplit，一款拥有 1700 万注册用户的视频游戏流媒体应用。

虽然道具目前不能兑换成法定货币，但根据 SEC 批准的文件，它们的价值为 0.1369 美元。该公司正在努力让道具在类似于加密货币交易所的替代交易系统中上市。这让 Props 给日常应用程序用户提供了财务激励，让他们看到应用程序网络的增长。因为 10 亿个道具的供应是有限的(目前已经开采了 6 亿个)，如果对道具的需求上升，那么用户可以以更高的价格出售它们。这通过提供一种奖励早期和铁杆用户的方式，为创业公司创造了一种新的增长黑客方法。

“我们提供的道具是第一个通过 SEC 认证的面向消费者的‘豪威代币’。Props 首席执行官 Adi Sideman 告诉我:“这是比特币和以太网之外，SEC 认为符合规定的第一款面向消费者的公用事业令牌。尽管 SEC 官员已经表示，由于比特币和以太网的充分分散性，它们并不是证券，但它们尚未获得正式批准。“我们使用法规 A+ (Reg A)进行资格认证，以便 Props 可以由未经认证的投资者、用户、应用程序和验证者获得，并为其提供功能，同时遵守美国法规。”

然而，这也可能给不太了解的用户带来风险，他们可能会误解代币系统，并过度相信他们会通过观看大量音乐家或喜剧演员在 YouNow 上的流媒体而致富。Props 需要确保整合其令牌的合作伙伴不会夸大其潜力。它花了两年时间争取 SEC 的批准，但如果道具被歪曲，仍可能面临后果。

“Props 使我们能够将创作者转变为网络中的利益相关者，这意味着他们成为网络成功的合作伙伴。PeerStream 首席执行官亚历克斯哈灵顿写道:“这对我们来说是一个重要的工具，可以更好地激励和吸引我们应用程序中最重要的用户。“对于我们这些开发者来说，Props 通过一组简单的 API 抽象了与基于区块链的令牌相关的技术和监管复杂性，我们可以使用这些 API 将令牌集成到我们的应用程序体验中。"

Blockstack 和 Props 开创了 RegA+方法，我们可以看到更多的公司申请使用这种方法筹集资金或与用户分享股份。
