另一个州正在考虑以每小时670英里的速度推进人们通过管道

另一个州——这次是北卡罗来纳州——被hyperloop的想法所吸引，这是一个未来主义的、仍在理论上的运输平台，可以在城市之间以高达670英里/小时的速度运送人员和包裹。

维珍Hyperloop One和北卡罗莱纳州区域运输联盟(T1)周五宣布开始“探索”使用Hyperloop连接该州罗利、达勒姆和教堂山的研究三角。

在hyperloop或Virgin Hyperloop One正在探索的这条特定路线成为现实之前，还有很长的路要走。根据AECOM进行的预可行性研究，从理论上讲，如果这条铁路建成，从罗利到达勒姆或教堂山只需不到10分钟。这将是该地区的关键，该地区是该国一些顶级公司、大学和医疗保健中心的所在地。

这将如何进行现在掌握在北卡罗莱纳州交通部手中。但根据周五一次活动的评论，国家机构不仅对三角研究感兴趣；它还计划在最初的想法上进行扩展，并研究一条连接夏洛特和华盛顿特区的线路

从现在开始，这个过程将是缓慢的。当国家机构调查建造hyperloop的可行性时，维珍Hyperloop One (VHO)正在致力于验证载人技术。这个目前并不存在的认证过程可能需要数年时间。VHO的目标是到2023年获得认证，到2029年拥有一个hyperloop平台。

该声明是在VHO的几个里程碑事件之后发布的，包括最近在华盛顿特区的示威游行和对NETT(非传统和新兴交通技术委员会)的资助，该委员会将在全国范围内研究和资助hyperloop。

VHO在5月份筹集了1.72亿美元。该公司还有一位新的首席执行官——杰伊·瓦尔德，他经营自行车共享运营商Motivate。瓦尔德在11月接管了VHO 。

理查德·布兰森爵士于10月卸任董事长，其职位已由阿联酋航运物流公司DP World的董事长兼首席执行官Sultan Ahmed bin Sulayem接任。

DP World是该公司最大的投资者。DP World于2016年首次投资该公司。这两家公司在2018年成立了一家物流合资企业，以开发货物的hyperloop运输。
