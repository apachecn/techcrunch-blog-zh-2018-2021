特斯拉放弃“标准系列”Model S 和 Model X，降低“远程系列”和 Model 3 的价格

特斯拉对其车型阵容进行了调整，取消了其 Model S 和 Model X 车型的入门级标准系列。这一变化意味着，现在无论是全电动的 Model S 轿车还是 Model X SUV，总体上都更加昂贵，但汽车制造商也降低了每款车的新入门级远程车型的价格，并将 Model 3 的起价降至 38，990 美元。

“为了使购买我们的汽车更加简单，我们正在标准化我们的全球汽车阵容，并简化为 Model S、Model X 和 Model 3 提供的装饰套件的数量，”特斯拉在给路透社的一份声明中就定价和阵容变化背后的原因表示。“我们也在调整定价，以继续提高客户的承受能力。”

减少特斯拉高端车型的数量应该有助于它最大限度地降低成本，并将高端买家的兴趣集中在装饰水平上，从而为汽车制造商带来更大的利润潜力。它在那里获得的好处可以有益地应用于 Model 3 的成本，这日益成为汽车制造商的增长来源。

特斯拉第二季度的汽车交付量创下最高纪录，总计 95，200 辆，其中最实惠的车型 Model 3 约占整体组合的 80%。
