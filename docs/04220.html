谷歌现在将为发现 Chrome 安全漏洞支付更高的奖励

捕虫可能是一项有利可图的工作。根据公司的不同，通过适当渠道报告的严重缺陷可以让最先发现它的人获得数万美元。

谷歌在 2010 年为 Chrome 推出了一项 bug 赏金计划。如今，他们将该计划的最高奖励提高了 2-3 倍。

Chrome 的 bug bounty 计划中的奖励根据 bug 的严重程度和报告的详细程度而有很大不同——细节较少的“基线”报告通常比解释 bug 可能如何被利用、为什么会发生以及如何修复的“高质量”报告赚得少。你可以在这里了解谷歌如何给报告评分。

但在这两种情况下，潜在的回报规模都在增加。基线报告的最高支出从 5，000 美元增加到 15，000 美元，而高质量报告的最高支出从 15，000 美元增加到 30，000 美元。

有一种漏洞是谷歌特别感兴趣的:那些损害运行在访客模式下的 Chromebook 或 Chromebox 设备的漏洞，并且无法通过快速重启来修复。谷歌最初为这种类型的错误提供了 5 万美元的奖励，在没有人成功认领后，在 2016 年将奖励增加到 10 万美元。今天他们涨到了 15 万美元。

他们还为 Chrome OS 奖励引入了一个新的漏洞类别:锁屏旁路。如果你能绕过锁屏(例如，从锁定的用户会话中提取信息)，谷歌将支付高达 15，000 美元。

谷歌为使用其“Chrome Fuzzer 程序”发现的任何漏洞支付额外的奖励，该程序让研究人员编写自动化测试并在大量机器上运行，希望找到只会在更大规模上出现的漏洞。通过 Fuzzer 程序发现的 bug 的奖金将从 500 美元增加到 1000 美元(在你通常从该类别的 bug 中获得的奖励之外)。

谷歌表示，自 2010 年推出 Chrome 漏洞奖励计划以来，它已经通过该计划支付了超过 500 万美元的漏洞奖金。截至今年 2 月，该公司已经支付了超过 1500 万美元用于他们所有的 bug 奖励计划。
