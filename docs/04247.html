VMware 收购 ML 加速启动 Bitfusion

VMware 今天宣布已经收购了 Bitfusion ，该公司曾是我们创业战场竞赛的参与者。Bitfusion 是最早帮助企业在 GPU、FPGAs 和 ASICs 上加速复杂计算工作负载的公司之一。在最早的迭代中，超过四年前，该公司的重点不太放在人工智能和机器学习上，而是更多地放在高性能计算的其他领域上，但是，不出所料，随着近年来对人工智能和 ML 的兴趣增加，这种情况发生了变化。

VMware 将使用与供应商和硬件无关的 Bitfusion 技术，为其客户带来类似的功能。具体来说，它计划将 Bitfusion 集成到其 vSphere 平台中。

VMware 云平台业务部门高级副总裁兼总经理 Krish Prasad 写道:“一旦收购完成，Bitfusion 将通过虚拟化硬件加速器来支持 VMware 的基于人工智能和 ML 的工作负载战略。”。“多供应商硬件加速器及其周围的生态系统是交付现代应用程序的关键组件。这些加速器可以在环境中的任何位置(内部和/或云中)使用。”

普拉萨德还指出，为了充分利用 GPU 等硬件加速器，大多数企业将它们部署在裸机上。然而，VMware 认为这将导致利用率和效率低下(考虑到它从事的是虚拟化业务，这是理所当然的)。普拉萨德写道:“这为虚拟化他们提供了一个绝佳的机会，增加了资源共享，降低了成本。”。

两家公司没有披露收购价格。Bitfusion 在 2017 年筹集了 500 万美元，并在 2018 年从三星风险投资公司获得了一笔较小的战略性投资。
