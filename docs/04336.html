AWS 的收入增长略有下滑，但仍是亚马逊的摇钱树

像上周的 Azure 一样，AWS 今天报告了其云收入增长数据的一些下滑，从去年同期的 49%下降到今年的 37%。来自 Q2 的总收入从去年的 61.05 亿美元增加到今年的 83.81 亿美元。今天的报告显示，仅 AWS 一项，该公司的运营成本就超过了 330 亿美元。那是一家成功的公司，更不用说是一个更大组织的一个部门了。

亚马逊肯定会为此感到兴奋，因为 AWS 占该公司本季度总收入的 13%，继续帮助这家在线零售商引领潮流。从大多数方面来看，所有这些都是好消息，但在一个持续加速的市场中，增长放缓对微软和亚马逊来说都很奇怪，因为两家公司都继续主导云基础设施市场。

亚马逊控制着大约 33%的市场，而微软大约占 16%。不管你看谁的数字，AWS 的市场份额大约是微软的两倍，但微软仍然是迄今为止唯一一家市场份额达到两位数的公司。

两家公司的下跌可以简单地归因于大数定律，即尽管这两家公司规模很大，但随着时间的推移，维持它们的增长率变得越来越困难，即使是在一个加速增长的市场中。

“我们看到的是大数定律，收入如此之高，增长百分比就变得越来越难。Moor Insights & Strategy 首席分析师帕特里克·穆尔黑德(Patrick Moorhead)对 TechCrunch 表示:“AWS 的美元收益仍高于 2-5 家(竞争对手)的总和。

随着市场的成熟，增长自然会放缓，无论市场有多热，亚马逊都不可能永远保持 40%的增长。无论如何，收入仍然令人印象深刻，AWS 继续帮助推动亚马逊的成功。
