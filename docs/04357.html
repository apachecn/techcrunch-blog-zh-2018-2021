被失败的网络钓鱼攻击的 Bellingcat 记者–TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/07/27/bellingcat-targeted-failed-phishing-attempt/amp/

图片来源: Bellingcat / ThreatConnect

调查新闻网站 Bellingcat 证实，其几名员工的 ProtonMail 账户遭到了网络钓鱼攻击，记者和电子邮件提供商表示攻击失败。

调查新闻网站的创始人 Eliot Higgins 在推特中写道:“Bellingcat 再次发现自己成为网络攻击的目标，几乎可以肯定这与我们对俄罗斯的工作有关。”。“我想衡量我们影响的一个方法是俄罗斯联邦的代理人试图攻击它的频率，无论是他们的黑客，巨魔还是媒体。”

有消息称，本周有少量 ProtonMail 电子邮件帐户成为攻击目标，其中几个属于 Bellingcat 的研究人员，他们从事与俄罗斯政府活动有关的项目。一封据称来自 ProtonMail 本身的网络钓鱼邮件要求用户通过攻击者设置的类似名称的域来更改他们的电子邮件帐户密码或生成新的加密密钥。根据安全研究人员的分析，记录显示该假冒网站是匿名注册的。

在声明中，ProtonMail 称网络钓鱼攻击“没有成功”,并否认其系统或用户账户遭到黑客攻击或泄露。

ProtonMail 的首席执行官 Andy Yen 说:“从 ProtonMail 用户的收件箱中获取电子邮件数据的最实际的方法是危害用户，而不是试图危害服务本身。”"因此，攻击者选择了直接针对记者的网络钓鱼活动."

Yen 表示，攻击者试图利用 ProtonMail 使用的第三方软件中的一个未打补丁的缺陷，该软件制造商尚未修复或披露该缺陷。

“然而，这个漏洞并不广为人知，这表明攻击者的技术水平更高，”Yen 说。

谁是这次袭击的幕后黑手还不得而知。然而，Bellingcat 和 ProtonMail 都表示，他们认为攻击的某些策略和指标可能指向与俄罗斯政府有关的黑客。例如，攻击的目标是 Bellingcat 的研究人员，他们正在调查俄罗斯军队击落 MH17 航班以及在英国定点清除中使用神经毒剂的事件

希金斯在一条推特中说本周的未遂攻击可能针对“数十”人，而不像早些时候归因于俄罗斯政府支持的黑客组织 APT 28 或 Fancy Bear 的攻击。

Bellingcat 在过去的一年中因其对俄罗斯政府的调查而获得了关键的赞誉，它揭露了涉嫌参与 2014 年炸毁马来西亚 MH17 航班的疑似导弹袭击的俄罗斯特工的姓名。研究小组还发现了俄罗斯特工的名字，他们被指控在 2018 年英国索尔兹伯里的神经毒剂袭击中毒害前俄罗斯情报人员谢尔盖·斯克里帕尔和他的女儿尤利娅。

研究人员使用开源情报和信息收集，而警察、执法部门和情报机构经常失败。

黑客针对 Bellingcat 已经不是第一次了。2016 年和 2017 年，在民主党全国委员会遭到入侵后，其研究人员多次成为目标，数千封内部电子邮件被盗并在网上发布。

记者致电俄罗斯驻纽约领事馆请求置评，但未获回复。
