Google 与 VMware 合作，将更多企业引入其云技术

原文：https://web.archive.org/web/https://techcrunch.com/2019/07/29/google-teams-up-with-vmware-to-bring-more-enterprise-customers-to-its-cloud/

谷歌与 VMware 合作，将更多企业引入其云

谷歌今天宣布与 VMware 建立新的合作伙伴关系，这将使企业更容易在谷歌云上运行其 VMware 工作负载。具体来说，谷歌云现在将支持 VMware Cloud Foundation ，该公司用于部署和运行混合云的系统。该解决方案由 CloudSimple 开发，而不是 VMware 或谷歌，谷歌将与 CloudSimple 合作提供一线支持。

虽然谷歌肯定希望所有企业都迁移到容器并利用其 Anthos 混合云服务，但大多数大公司目前都使用 VMware。他们可能希望将这些工作负载迁移到公共云，但他们不准备放弃一个长期为他们工作的工具。有了这个新功能，谷歌没有提供任何特别新的或创新的东西，但这不是我们要做的。相反，谷歌只是给企业更少的理由去选择竞争对手，甚至不考虑它的产品。

“客户要求我们为 VMware 提供广泛的支持，现在有了 CloudSimple 的 Google Cloud VMware 解决方案，我们的客户将能够在 GCP 运行基于 VMware vSphere 的工作负载，”该公司在公告中指出，我们获得了该公告的早期副本，但由于我们不知道的原因，该公告将于明天在该公司的博客上发布。“这为客户如何在混合部署中运行其 VMware 工作负载提供了广泛的选择，从采用 Anthos 的现代容器化应用程序到采用 VMware 的 GCP 基于虚拟机的应用程序。”

新的解决方案将支持完整的 VMware 体系，包括 vCenter、vSAN 和 NSX-T 等

VMware 客户运营首席运营官 Sanjay Poonen 表示:“我们与谷歌云的合作伙伴关系一直致力于解决客户的需求，我们很高兴能够扩展这一合作伙伴关系，使我们共同的客户能够在谷歌云平台的 VMware Cloud Foundation 上运行 VMware 工作负载。借助 Google 云平台上的 VMware，客户在执行云战略时将能够利用 VMware 工具和培训的所有熟悉度和投资保护，并快速将新服务推向市场，在混合云环境中无缝、更安全地运营这些服务。"

虽然谷歌的声明强调了该公司与 VMware 的长期合作历史，但有趣的是，至少这种合作关系的技术方面更多地是关于 CloudSimple 而不是 VMware。同样值得注意的是，VMware 长期以来与谷歌的云竞争对手 AWS 有着密切的关系，T2 的微软 Azure 也在其云上提供运行基于 VMware 的工作负载的工具。
