移动消息金融咨询服务stack in’增加银行功能并筹集现金TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/07/29/mobile-messaging-financial-advisory-service-stackin-adds-banking-features-and-raises-cash/

两年前，当stack in ' T1最初将自己定位为Techstars洛杉矶加速器计划的一部分时，该公司是一个针对千禧一代观众的金融建议视频平台，这些观众对传统的咨询服务过于熟悉。

如今，近两年过去了，该公司已将面向千禧一代的金融建议从视频转向了基于文本的建议，并开始为数字银行提供新的线索挖掘服务。

该公司与Radius银行合作推出了一项新的免费支票和储蓄账户功能，为用户提供1%的存款年收益率。

Stackin已经从Experian Ventures、Dig Ventures和Cherry Tree Investments筹集了400万美元的新资金，新老投资者也做出了补充承诺，包括Social Leverage、Wavemaker Partners和Mucker Capital。

益百利全球消费者服务集团总裁泰·泰勒(Ty Taylor)在一份声明中表示:“‘stack in’有一种独特而高效的方法，可以围绕金融与整整一代年轻消费者建立联系和沟通。”。

Stackin两年前由Uproxx Media的前创始人斯科特·格瑞恩斯和担任Uproxx高级副总裁的凯尔·阿博(Kyle Arbaugh)创立，最初它标榜自己是个人理财的Uproxx。

事实证明，消费者并不想要另一个视频平台。

“Stackin '正在从根本上改变金融关系的形式和背景，它创造了一个有趣、包容和无判断的环境，让我们的用户能够通过信息学习和采取行动，”Stackin '的首席执行官兼联合创始人斯科特·格瑞恩斯在一份声明中说。“这笔资金使我们能够围绕银行和投资开发新功能，从而增强与客户的关系。”

今年秋天晚些时候，该公司表示将推出一项新的投资功能，鼓励Stackin用户参与股票市场。很可能这个功能看起来有点像Acorns模型，它鼓励用户投资多样化的金融工具，以便在允许个人股票交易之前熟悉股票市场。

根据Grimes的说法，该公司在2018年3月从视频转向文本，并在Twilio上建立了一个定制的消息平台，为该公司的50万用户服务。

“在很短的时间内，我们已经建立了一个庞大的客户群，这是一个很难接触到的群体。像益百利这样的金融机构作为投资者加入进来，证明了这种模式正在发挥作用，”格里姆斯在一封电子邮件中写道。
