保时捷 Taycan 全球首发前预订量突破 30，000 辆

在这家德国汽车制造商将推出全电动跑车的一个多月前，保时捷已经为 Taycan 获得了 3 万英镑的定金，这些数字表明，有足够的需求支持该公司第一年生产 4 万辆的计划。

彭博和保时捷人力资源总监安德里亚斯·哈夫纳在接受德国商业刊物商报采访时引用了最新的预订数字。

保时捷最初的目标是第一年生产 20，000 辆 Taycan 电动汽车。但是对这款车的兴趣促使汽车制造商在第一年将计划年产量翻了一番，达到 40，000 辆。预订需要 2500€的押金(2785 美元)。

如果保时捷能够在第一年生产并交付 4 万辆 Taycans，这款电动跑车将超越其一些标志性的内燃车型，包括 718 Boxster 和 911。2018 年，保时捷在全球销售了 35，573 辆 911 和 24，750 辆 718。

Taycan 仍将落后于保时捷的其他流行的交叉和 SUV 车型，如卡宴和 Macan。

Taycan 还可能给特斯拉 Model S 带来压力，这款广受欢迎的豪华电动轿车长期以来一直主导着这个行业的利基市场。特斯拉结合了 Model S 和 X 的交付编号。2018 年，该公司交付了 99，394 辆 Model S 和 X 汽车。

自 2012 年开始生产以来，Model S 已经进行了多次更新，但自 2016 年 4 月前面板被改为看起来更像 Model X 以来，它还没有进行过重大翻新。

特斯拉首席执行官 Elon Musk 本月早些时候表示，该公司不计划“更新”其 Model X 或 Model S 车辆。用汽车行业的话来说，更新通常意味着对车型的小修改，超出了制造商通常每年进行的更新。刷新不是一个重大的重新设计，虽然通常有一个明显的变化，车辆模型。

马斯克当时表示，该公司将对豪华电动轿车和运动型多功能车进行微小的持续变化。即使有了这些持续的更新，潜在客户可能会选择更新的 Taycan。

保时捷没有依靠其第一款电动汽车的新奇来推动销售。该公司正在推出其他激励措施，特别是计划在全美数百个电气化美国公共站为 Taycan 车主提供三年免费充电。Electrify America 是大众汽车公司设立的实体，作为其与美国监管机构就柴油排放作弊丑闻达成的和解的一部分。

该汽车制造商还额外投资 7000 万美元，为保时捷经销商增加 DC 快速充电器。
