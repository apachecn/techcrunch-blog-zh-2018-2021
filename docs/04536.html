俄罗斯人来了！俄罗斯人是...复杂！

原文：https://web.archive.org/web/https://techcrunch.com/2019/08/08/the-russians-are-coming-the-russians-are-complicated/

你知道吗，俄罗斯的安全部门，尤其是那些与黑客/信息安全相关的部门，多年来一直处于恶性的高风险内斗的痛苦之中。你知道吗？西方对俄罗斯战略的分析中所认为的俄罗斯教义实际上从未存在过。你知道吗，克里姆林宫的秘密已经建立了一个很大程度上毫无根据的谣言和阴谋论的家庭手工业，这些谣言和阴谋论是基于一些确实泄露的诱人的细节。

好吧，你可能知道最后一部分。如今，所有人，或者至少是所有称社交媒体陌生人为“俄罗斯机器人”的人，都是俄罗斯阴谋论者。当然，从暗杀、黑客攻击到社交媒体爆炸，俄罗斯广泛恶意活动的证据比比皆是。

但是，确切地说，哪些俄罗斯组织在做什么，为什么做——这就很难确定了。我想起了旧的冷战间谍小说，其中克里姆林宫学家分析了政治局成员的几次公开露面，错误地解读了谁在何时何地站在哪里的重大意义，因为他们没有什么其他线索。就像那些糟糕的旧时光一样，我们现在的本能是把“俄罗斯”视为一个单一的、运转良好的、紧密配合的恶性机器。

当然不是那么回事。相反，它是一个复杂的、沸腾的、多层次的泥沼，在一个深刻而迅速变化的时代，许多人物和机构经常相互激励。今天，我参加了金伯利·曾兹的黑帽演讲，她在开场时呼吁对俄罗斯和俄罗斯的活动进行细致入微的思考。当然，她是对的，但可悲的是，互联网往往是细微差别消失的地方。

然而，这种细微差别尤其引人入胜，是间谍惊悚片的素材。2017年，许多俄罗斯情报官员和黑客——莫名其妙地，还有卡巴斯基实验室的调查负责人——突然被捕。据Stratfor报道，一个人“显然被强行带离了与联邦安全局官员的会议——头上套着一个袋子被护送出去”。最终他们被指控犯有“支持美国的叛国罪”

今年有四个人被判处长达22年的监禁。(他们很有吸引力。)俄罗斯信息安全中心(Information Security Center)长期负责人安德烈·格拉西莫夫(Andrei lavrov)，“是一个影子单位……被认为是俄罗斯在国内外网络能力方面最大的检查机构，包括黑客攻击，”在这起案件曝光一周后辞职。

斯特拉特福再次表示:“因为指控是叛国罪，该案件被国家视为‘机密’，这意味着不会公布任何官方解释或证据。”从这个秘密的迷雾中，六个不同的谣言和理论已经散发出来。这些指控完全是为了消灭对手而捏造的吗？是不是有人泄密到美国攻击对手，结果却看到这种事与愿违的壮观景象？金融稳定委员会是不是变成了一个黑客组织，然后发现了一些他们真的不应该知道的关于一个强大寡头的事情？谁能说呢？

当然，另一种阴谋论是没有细微差别的“润滑良好的恶性机器”理论，在这种情况下，这只是所述机器从内部排出一点砂砾的一个实例。值得注意的是，这种“铁板一块的俄罗斯单一声音群体思维”分析变得如此普遍。下面是政治的例子，在上述丑闻爆出后:“最近，俄国似乎正从各种矛盾的角度攻击美国……困惑？除非你不懂格拉西莫夫主义。”

请注意，这一原则是以瓦列里·格拉西莫夫将军的名字命名的，而不是重复上面提到的现已名誉扫地的前联邦安全局局长安德烈·格拉西莫夫的名字。这一原则被用来解释俄罗斯的所有活动，即使是看起来自相矛盾的活动，也是为了“在敌国内部实现永久动荡和冲突的环境”这是昨天在另一个黑帽演讲中引用的，我对此很不以为然，我会委婉地避免进一步讨论。时至今日，俄罗斯政策分析人士一直引用这一观点。

但是，作为现代克里姆林宫学基石的格拉西莫夫主义的问题在于——根据创造这个术语的人所说！— 它实际上从未存在过。(讽刺的是，这源于格拉西莫夫将军的阴谋论:中央情报局煽动了阿拉伯之春。)相反，与其说这是一场基于统一原则的运动，不如说俄罗斯的行动是:

很大程度上是机会主义的，支离破碎的，甚至有时是矛盾的。一些主要的行动是协调的，主要是通过总统行政部门，但大多数不是。相反，行动是由一群令人困惑的“政治企业家”构思和执行的，他们希望自己的成功能赢得克里姆林宫的青睐

这听起来是一个非常重要的区别，这引出了关于曾兹女士的演讲最有趣的事情(对我来说);她提到“俄罗斯政府认为俄罗斯网络罪犯是一种战略资产”，这起叛国案的一个副作用是，它极大地削弱了俄罗斯和西方之间关于网络威胁的信息共享和合作。

这种战略地位是否反过来意味着俄罗斯黑客很可能是政府特工，和/或与政府勾结的俄罗斯信息安全公司？我不是克里姆林宫学专家，但在我看来，这个问题是错误的，不应该问。相反，在法治发达的国家，“私营部门”、“政府”和“罪犯”之间相对明显的区别，在现代俄罗斯这样的国家并不存在，在那里，这些区别可能，而且经常会模糊在一起。
