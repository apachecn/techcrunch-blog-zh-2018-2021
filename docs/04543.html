Preclusio 使用机器学习来遵守 GDPR 和其他隐私法规

随着像 GDPR 和 T2 这样的隐私法规激增，越来越多的创业公司希望帮助公司遵守这些法规。进入 Y Combinator 年夏季班的成员precusio，该班开发了一种基于机器学习的解决方案，以帮助公司遵守这些隐私规定。

“我们有一个部署在客户环境内部的平台，可以帮助他们识别他们正在收集的数据、使用数据的方式、存储数据的位置以及应该如何保护数据。公司联合创始人兼首席执行官希瑟·韦德(Heather Wade)告诉 TechCrunch，我们帮助公司整合其数据的广泛视图，然后我们持续监控他们的数据基础设施，以确保这些数据继续受到保护。

她说，该公司经过深思熟虑后决定将该解决方案保留在本地。“我们真的相信让我们的客户控制他们的数据。Wade 解释说:“我们不想成为另一个第三方 SaaS 供应商，你必须向它提供数据。

也就是说，客户可以在他们希望的任何地方运行它，无论是在本地还是在 Azure 或 AWS 的云中。不管它存储在哪里，这个想法是让客户直接控制他们自己的数据。“我们真的试图提醒我们的客户实时关注他们环境中发生的威胁或潜在的隐私异常，身处他们的环境中是促进这一点的最佳方式，”她说。

该产品通过对数据进行只读访问来工作，然后开始使用机器学习以自动化的方式识别敏感数据。“我们的产品会自动查看数据的模式和样本，并使用机器学习来识别常见的受保护数据，”她说。该过程完成后，隐私合规团队可以审查调查结果，并根据需要调整这些分类。

Wade 于今年 3 月创办了这家公司，她表示，这个想法是在她之前负责实施隐私政策的职位上形成的，她发现市场上没有足够的解决方案来提供帮助。“我不得不直接面对处理隐私和合规性的挑战，并看到资源是如何真正从我们的工程团队手中夺走的，不得不在内部分配这些资源来解决这些问题，特别是在 GDPR 第一次通过的早期，市场上真的没有太多可用的工具，”她说。

有趣的是，韦德的联合创始人是她的丈夫约翰。她说，他们通过坚持自己的专业领域来应对已婚和创业的压力。他是营销人员，她是技术人员。

她说，他们申请 Y Combinator 是因为他们想快速发展，随着更多隐私法即将出台，这个时机很重要。社区在帮助他们实现目标方面的慷慨给她留下了深刻的印象。“几乎难以形容的是，其他经历过 YC 项目的人对即将到来的批次是多么慷慨和乐于助人，他们真的有那种向前支付的精神，”她说。
