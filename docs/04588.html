ClockworkPi GameShell 是一款超好玩的便携式游戏 TechCrunch 上的 DIY 旋转游戏

原文：https://web.archive.org/web/https://techcrunch.com/2019/08/13/the-clockworkpi-gameshell-is-a-super-fun-diy-spin-on-portable-gaming/

便携式游戏机并不新鲜，多亏了 Switch，它们基本上是世界上最受欢迎的游戏设备。但是，ClockworkPi 的 GameShell 是完全独特的东西，当谈到移动游戏时，它完全令人耳目一新。这个聪明的 DIY 控制台套件提供了您在家中组装自己的口袋游戏机所需的一切，运行基于 Linux 的开源软件，并使用欢迎未来定制的开源硬件设计。

GameShell 是 Kickstarter 成功营销的结果，去年开始向其支持者发货，现在可以直接从该公司或亚马逊购买。159.99 美元(在撰写本文时售价为 139.99 美元)包括了构建主机所需的一切，如集成 Wi-Fi、蓝牙和 1GB DDR 3 RAM 的 ClockworkPi 四核 Cortex A7 主板——但它是未组装的。

你不必拿出烙铁——电路板附带所有元件。但你将组装屏幕、键盘、CPU、电池和扬声器模块，用附带的电缆将它们连接起来，并安装在光滑的 GameBoy 风格的塑料外壳中。这可能看起来是一个令人生畏的任务，取决于你的技术专业水平:我知道当我打开各种盒子并把所有部件摆在我面前时，我发现自己有点担心。

但是附带的说明只是插图，就像乐高或宜家提供的说明一样，非常容易理解，并且将任务分解为所有技能水平的人都可以管理的任务。总的来说，我在不到一个小时的时间里就把我的组装好了，尽管我确实在某一点上用牙齿咬了进去(在组装可选的 Lightkey 组件时去掉了一点塑料小块，这给控制台增加了额外的功能键)，但我从未感到不知所措或被打败。下面的延时记录了我的整个组装过程，从开始到结束。

当你完成后，你得到的是一个功能齐全的便携式游戏设备，它运行的是该公司开发的基于 Linux 的开源操作系统“发条操作系统”。它包括过去几十年来最著名的独立游戏之一《洞穴故事》(Cave Story)，以及许多内置的模拟器(使用模拟器在道德和法律上是有问题的，但它确实提供了一种简单的方法来玩一些你已经拥有的具有更高便携性的 NES 和 SNES 游戏)。

GameShell 周围有一个非常活跃的社区，其中包括许多可以在主机上玩的独立游戏，以及修改和优化使用的技巧和诀窍。它还被设计为一种 STEM 教育资源，为孩子们提供了一种很好的方式来了解他们日常使用的电子产品面板背后实际发生的事情，甚至开始编写自己的代码来构建在控制台上运行的软件。由于随附的 microSD 存储卡和通过 Wi-Fi 轻松连接以从 Windows 和 Mac 电脑上移动软件的能力，加载软件很容易。

GameShell 的一切都是可编程的，它具有 micro HDMI out，内置音乐播放器和蓝牙耳机连接支持。对于技术水平非常有限的人来说，它可以立即访问，对于那些确实想更深入地挖掘它还能提供什么的人来说，它可以无限扩展和破解。

可交换的面板和背板，加上每个硬件组件的开放 3D 模型，意味着社区开发的硬件附件和修改也是完全可能的。该设备的模块化特性意味着，随着更高容量的电池模块和改进的开发板，它在未来可能会变得更强大。

我以前肯定见过和使用过像 GameShell 这样的设备，但很少有人能同时做到易用、强大和可定制。GameShell 的速度也很快，声音很棒，显示屏也很棒，而且似乎非常耐用，电池续航时间约为三小时，连续使用时间略长，具体取决于你是否在使用 Wi-Fi 和屏幕亮度等因素。
