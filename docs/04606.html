VMware 表示正在寻求收购 Pivotal TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/08/14/vmware-says-its-looking-to-acquire-pivotal/

VMware 今天证实正在就收购软件开发平台 Pivotal Software 进行谈判，该服务因将开源云铸造平台商业化而闻名。在拟议的交易中，VMware 将以每股 15 美元的价格收购所有已发行的 Pivotal A 类股票，这比 Pivotal 当前的股价高出了一大截(不出所料，Pivotal 的股价在声明发布后立即飙升)。

自 2018 年 4 月该公司 IPO 以来，Pivotal 的股价一直在挣扎。该公司最初于 2012 年从 EMC Corporation(现为 DellEMC)和 VMware 分离出来，专注于 Cloud Foundry，这是一个开源软件开发平台，目前被大多数财富 500 强公司使用。这些企业中有很多都在与 Pivotal 合作，以支持他们的云铸造工作。戴尔本身继续拥有 VMware 和 Pivotal 的大部分股份，VMware 也已经拥有 Pivotal 的股份，并向其客户销售 Pivotal 的服务。这有点像交易的大毒蛇。

Pivotal Cloud Foundry 一直是该公司的主要产品，但除此之外，它还提供额外的咨询服务。尽管自上市以来，Pivotal 的执行情况有所改善，但由于其股价仅为 IPO 价格的一半多一点，Pivotal 在上一个财政季度仍然亏损了 3170 万美元。事实上，VMware 提供的每股 15 美元与 Pivotal 的 IPO 价格相同。

VMware 的收购将使 Pivotal 的旅程回到起点，尽管这肯定不是 Pivotal 团队预期的旅程。VMware 是 Cloud Foundry Foundation 白金会员，另外还有 Pivotal、DellEMC、IBM、SAP 和 Suse，因此我预计 VMware 对 Pivotal 核心平台背后的整个开源生态系统的支持不会有任何重大变化。

不过，收购是否真的会发生还有待观察。在一份新闻稿中，VMware 承认了两家公司之间的讨论，但指出“不能保证会就潜在交易达成任何此类协议，除非达成最终协议，否则 VMware 不打算就此事进行进一步沟通。”律师喜欢写这种句子。不过，如果这笔交易没有达成，我会很惊讶。

收购 Pivotal 在 VMware 最近的收购大计划中也是有意义的。今年早些时候，该公司收购了 Bitnami ，去年又收购了由 Kubernetes 项目三位联合创始人中的两位创建的初创公司 Heptio，该项目现已成为许多新的企业云部署的基础，最近还收购了 Pivotal Cloud Foundry。
