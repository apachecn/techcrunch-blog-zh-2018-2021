VMware表示正在寻求收购Pivotal TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/08/14/vmware-says-its-looking-to-acquire-pivotal/amp/

图片来源: 枢纽

VMware今天证实正在就收购软件开发平台 Pivotal Software 进行谈判，Pivotal Software是一家以将开源云铸造平台商业化而闻名的服务公司。在拟议的交易中，VMware将以每股15美元的价格收购所有已发行的Pivotal A类股票，这比Pivotal当前的股价高出了一大截(不出所料，Pivotal的股价在声明发布后立即飙升)。

自2018年4月该公司IPO以来，Pivotal的股价一直在挣扎。该公司最初于2012年从EMC Corporation(现为DellEMC)和VMware 中分离出来，专注于Cloud Foundry，这是一个开源软件开发平台，目前被大多数财富500强公司使用。这些企业中有很多都在与Pivotal合作，以支持他们的云铸造工作。戴尔本身继续拥有VMware和Pivotal的大部分股份，VMware也已经拥有Pivotal的股份，并向其客户销售Pivotal的服务。这有点像交易的大毒蛇。

Pivotal Cloud Foundry一直是该公司的主要产品，但除此之外，它还提供额外的咨询服务。尽管自上市以来，Pivotal的执行情况有所改善，但由于其股价仅为IPO价格的一半多一点，Pivotal在上一个财政季度仍亏损了3170万美元。事实上，VMware提供的每股15美元与Pivotal的IPO价格相同。

VMware的收购将使Pivotal的旅程回到起点，尽管这肯定不是Pivotal团队预期的旅程。VMware是Cloud Foundry Foundation白金会员，另外还有Pivotal、DellEMC、IBM、SAP和Suse，因此我预计VMware对Pivotal核心平台背后的整个开源生态系统的支持不会有任何重大变化。

不过，收购是否真的会发生还有待观察。在一份新闻稿中，VMware承认了两家公司之间的讨论，但指出“不能保证会就潜在交易达成任何此类协议，除非达成最终协议，否则VMware不打算就此事进行进一步沟通。”律师喜欢写这种句子。不过，如果这笔交易没有达成，我会很惊讶。

收购Pivotal在VMware最近的收购大计划中也是有意义的。今年早些时候，该公司收购了Bitnami ，去年又收购了由Kubernetes项目三位联合创始人中的两位创建的初创公司Heptio，该项目现已成为许多新的企业云部署的基础，最近还收购了Pivotal Cloud Foundry。
