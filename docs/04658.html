原文：https://web.archive.org/web/https://techcrunch.com/2019/08/19/microsoft-acquires-jclarity-an-open-source-java-performance-tuning-tool/

微软收购 Java 性能调优工具 jClarity

微软今天早上宣布收购 jClarity ，这是一项旨在调整 Java 应用程序性能的服务。从现在开始，它将在 Azure 上这样做。此外，该公司一直在提供一种叫做 AdoptOpenJDK 的 Java，他们称之为 Oracle Java 的免费替代品。两家公司没有讨论交易条款。

正如微软在宣布收购的博客文章中指出的,他们看到 Azure 上越来越多地使用大规模的 Java 安装，包括内部的《我的世界》等平台和外部的大客户，包括戴姆勒和 Adobe。

该公司相信，通过增加 jClarity 团队及其工具集，它可以帮助更好地服务这些 Java 客户。该公司在博客中写道:“该团队由 Java 冠军和数据科学家组成，在数据驱动的 Java 虚拟机(JVM)优化方面拥有成熟的专业知识，将帮助微软的团队利用 Java 平台的进步。”

微软实际上是 AdoptOpenJDK 项目的一部分，还有其他企业公司的名人录，包括亚马逊、IBM、Pivotal、Red Hat 和 SAP。

联合创始人兼首席执行官马丁·韦尔伯格(Martijn Verburg)在宣布交易的公司博客文章中写道，不出所料，他热情洋溢地谈论了他即将成为其中一员的公司。“微软在支持开发者及其社区方面走在世界前列，在与他们的工程和项目领导交谈后，进入正式讨论是显而易见的。凭借微软员工的热情和深厚的专业知识，我们将能够比以往任何时候都更好地支持 Java 生态系统，”他写道。

Verburg 还花时间感谢支持开源项目的员工、客户和社区，他的公司就是建立在开源项目之上的。Verburg 在微软的新头衔是微软首席工程组经理(Java)。

目前还不清楚社区对另一种风格的 Java 被另一家大型供应商吸收会有什么反应，或者参与该项目的其他大型供应商对此会有什么感觉，但无论如何，jClarity 的 Java 风格及其性能工具现在都是微软的一部分。

注意:本文最初声明 jClarity 的所有产品都是开源的。它的表现工具是付费服务。
