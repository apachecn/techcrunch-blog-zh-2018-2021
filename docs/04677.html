GitHub 的免费学生开发者包增加了 21 个新伙伴

几年来，GitHub 一直在开展一项计划，作为其 GitHub 教育计划的一部分，该计划为全球学生提供免费使用 GitHub Pro 和其他合作伙伴提供的各种免费和打折服务。总共有超过 150 万名学生注册了该计划，其中约有 75 万名学生目前处于活跃状态。随着新学年即将开始，该公司今天宣布，通过增加 21 家公司，将学生开发者包项目的合作伙伴数量增加一倍。

新的合作伙伴涵盖了广泛的开发工具和服务。这些工具从 web 设计工具如 Bootstrap Studio 到域名注册商不等，Bootstrap Studio 实际上在今年夏天悄悄加入了这个项目。TECH ，SSH client Terminus ，用于特性标志和配置管理的 ConfigCat ，以及用于使应用程序看起来更好的 Icons8 。

GitHub Student Developer Pack 的高级项目经理斯科特·萨尼奇(Scott Sanicki)说:“我们的理念是，让学生为他们在工作场所将遇到的所有重要工具做好准备。这意味着也可能有竞争产品的合作伙伴，但正如 Sanicki 指出的那样，GitHub 目前还没有看到包括微软在内的现有合作伙伴的任何抵制。

事实上，今年夏天，GitHub 的新主人微软也将其 Azure 云计算服务添加到学生包中。这并不奇怪，但值得注意的是，AWS、DigitalOcean 和 Heroku 已经是其中的一部分，为学生提供免费和打折的云计算资源。正如 Sanicki 告诉我的那样，他们仍然留在项目中，这是公司承诺保持开放的一部分，即使在被微软收购之后。

Sanicki 说:“(GitHub 首席执行官 Nat Friedman)在收购时被直接问到我们是否会终止学生开发者包，他承诺我们会继续这个包，事实上，会让它变得更强大，这也是我们一直试图继续做的事情。”。“我们已经看到了来自微软和领导层的支持，他们给了我们所需的资源，不仅让我们对 pack 和 GitHub 教育有了更大的梦想，还让我们现在尝试去实现它。”

Sanicki 预计，在接下来的几个月中，该团队将继续增加新的合作伙伴。要注册并查看您是否有资格参加该计划，请点击这里。
