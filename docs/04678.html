谷歌的轻量级搜索应用Google Go向全球安卓用户推出

原文：https://web.archive.org/web/https://techcrunch.com/2019/08/20/googles-lightweight-search-app-google-go-launches-to-android-users-worldwide/

Google Go是Google搜索应用的一个轻量级版本，今天全球所有Android用户都可以使用。第一款于2017年推出，此前进行了几个月的测试，这款应用主要是为新兴市场设计的，在这些市场，人们第一次使用低端安卓设备通过不稳定的连接访问互联网。

像许多为新兴市场开发的“精简”版本的应用程序一样，Google Go在手机上占用的空间更少——现在只有7MB多一点——并且它包括离线功能，以帮助那些互联网连接缓慢和断断续续的人。谷歌还声称，该应用的搜索结果经过优化，可以保存高达40%的数据。

除了网络搜索，Google Go还包括其他发现功能，如挖掘热门话题的能力，语音搜索，图像和GIF搜索，一种在语言之间切换的简单方法，以及由人工智能驱动的大声朗读网页的能力。

在今年春天的谷歌I/O开发者大会上，该公司宣布也将把Lens引入Google Go 。

Lens允许用户将智能手机摄像头对准现实世界的物体，以便调出相关信息。在Google Go中，镜头功能将帮助阅读困难的用户。当摄像头对准文本时——比如公交时刻表、标志或银行表格——Lens可以大声读出文本，并在朗读时高亮显示。用户也可以点击某个特定的单词来了解它的定义或翻译文本。

虽然Lens只增加了100KB，根据Google 的说法，自发布以来对Go应用程序的更新增加了它的大小。最初是一个5MB的app现在是7MB多一点。

此前，Google Go仅在少数国家的Android Go版设备上可用。根据Sensor Tower的数据，它在全球已安装了大约1750万次，其中印度的用户比例最大(48%)。Sensor Tower表示，其下一个最大的市场是印度尼西亚(16%)、巴西(14%)、尼日利亚(6%)和南非(4%)。

该软件已在29个国家的Android Go edition设备上提供，包括:安哥拉、贝宁、博茨瓦纳、布基纳法索、喀麦隆、佛得角、科特迪瓦、加蓬、几内亚比绍、肯尼亚、马里、毛里求斯、莫桑比克、纳米比亚、尼日尔、尼日利亚、菲律宾、卢旺达、塞内加尔、坦桑尼亚、多哥、乌干达、赞比亚和津巴布韦。

谷歌表示，该应用程序现在拥有“数百万”用户。

今天，谷歌表示，Play Store将向全球所有用户开放。

谷歌表示，它决定在全球范围内推出这款应用，包括在带宽不是问题的市场，因为它知道每个人都会遇到手机存储空间有限或连接不稳定等问题。

另外，这是一个用于阅读和翻译文本的轻量级应用程序。在谷歌I/O大会上，该公司指出，全球有超过8亿成年人阅读困难——当然，并不是所有人都在新兴市场。

谷歌Go是谷歌为新兴市场开发的众多轻量级应用之一，例如，还有 YouTube Go 、文件Go 、 Gmail Go 、谷歌地图Go、 Gallery Go 和谷歌助手Go。

运行Android Lollipop或更高版本的全球用户将可以在Play Store 上使用Google Go应用程序。
