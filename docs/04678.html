谷歌的轻量级搜索应用 Google Go 向全球安卓用户推出

原文：https://web.archive.org/web/https://techcrunch.com/2019/08/20/googles-lightweight-search-app-google-go-launches-to-android-users-worldwide/

Google Go 是 Google 搜索应用的一个轻量级版本，今天全球所有 Android 用户都可以使用。第一款于 2017 年推出，此前进行了几个月的测试，这款应用主要是为新兴市场设计的，在这些市场，人们第一次使用低端安卓设备通过不稳定的连接访问互联网。

像许多为新兴市场开发的“精简”版本的应用程序一样，Google Go 在手机上占用的空间更少——现在只有 7MB 多一点——并且它包括离线功能，以帮助那些互联网连接缓慢和断断续续的人。谷歌还声称，该应用的搜索结果经过优化，可以保存高达 40%的数据。

除了网络搜索，Google Go 还包括其他发现功能，如挖掘热门话题的能力，语音搜索，图像和 GIF 搜索，一种在语言之间切换的简单方法，以及由人工智能驱动的大声朗读网页的能力。

在今年春天的谷歌 I/O 开发者大会上，该公司宣布也将把 Lens 引入 Google Go 。

Lens 允许用户将智能手机摄像头对准现实世界的物体，以便调出相关信息。在 Google Go 中，镜头功能将帮助阅读困难的用户。当摄像头对准文本时——比如公交时刻表、标志或银行表格——Lens 可以大声读出文本，并在朗读时高亮显示。用户也可以点击某个特定的单词来了解它的定义或翻译文本。

虽然 Lens 只增加了 100KB，根据 Google 的说法，自发布以来对 Go 应用程序的更新增加了它的大小。最初是一个 5MB 的 app 现在是 7MB 多一点。

此前，Google Go 仅在少数国家的 Android Go 版设备上可用。根据 Sensor Tower 的数据，它在全球已安装了大约 1750 万次，其中印度的用户比例最大(48%)。Sensor Tower 表示，其下一个最大的市场是印度尼西亚(16%)、巴西(14%)、尼日利亚(6%)和南非(4%)。

该软件已在 29 个国家的 Android Go edition 设备上提供，包括:安哥拉、贝宁、博茨瓦纳、布基纳法索、喀麦隆、佛得角、科特迪瓦、加蓬、几内亚比绍、肯尼亚、马里、毛里求斯、莫桑比克、纳米比亚、尼日尔、尼日利亚、菲律宾、卢旺达、塞内加尔、坦桑尼亚、多哥、乌干达、赞比亚和津巴布韦。

谷歌表示，该应用程序现在拥有“数百万”用户。

今天，谷歌表示，Play Store 将向全球所有用户开放。

谷歌表示，它决定在全球范围内推出这款应用，包括在带宽不是问题的市场，因为它知道每个人都会遇到手机存储空间有限或连接不稳定等问题。

另外，这是一个用于阅读和翻译文本的轻量级应用程序。在谷歌 I/O 大会上，该公司指出，全球有超过 8 亿成年人阅读困难——当然，并不是所有人都在新兴市场。

谷歌 Go 是谷歌为新兴市场开发的众多轻量级应用之一，例如，还有 YouTube Go 、文件 Go 、 Gmail Go 、谷歌地图 Go、 Gallery Go 和谷歌助手 Go。

运行 Android Lollipop 或更高版本的全球用户将可以在 Play Store 上使用 Google Go 应用程序。
