终于袋周又到了！TechCrunch 年度最精彩的一周。在返校的时候，我们将为您带来各种包的评论:从背包到滚轴到信使再到腰包。

今年，和去年一样，我决定专注于包袋社区的一个特定领域:打蜡帆布。去年，我看了一些来自 Ona、Filson 和其他上蜡商品供应商的包。但是还有很多可以选择的，所以我收集了第二把，并且用了足够长的时间来了解它们的优缺点。

上蜡的帆布是一种极好的材料。浸有蜡的天然纤维具有防水性、结构性、保护性和美观性，随着使用时间的推移只会越来越好。这是我最喜欢的材料，也应该是你的。唯一的问题是，它可能很贵。但请记住，这些包包是那种你会随身携带一二十年的包包。

在这篇文章中，我关注的是笔记本电脑包，但在稍后的庆祝活动中，我会有几个更像“信使”风格的打蜡包，所以请保持警惕。

在我为这个综述看过的所有包中，这个包可能是最简单的，因为它不是可转换的，超重的，超轻的，蓝色的或类似的。它只是一个由上蜡的帆布和皮革制成的结实的多功能笔记本电脑包，因此可以作为比较其他所有东西的基准。

[gallery ids="1839091，1839097，1839088，1839093，1839096，1839095，1839094，1839092，1839090，1839087"]

Bolt 的帆布不像其他 Waterfield 包那么厚，因为它的内部有衬里和衬垫。它仍然有一个很好的完成，虽然，皮革基地和装饰也同样高品质。像该公司的其他包一样，表带是尼龙的，我更喜欢帆布，但其中包含的 grippy 皮革肩垫是我用过的最实用和舒适的。

Bolt 的优势不在于纯粹的空间，因为它是一个相当紧凑的包(如果你喜欢，可以选择更大的尺寸)，而在于感觉空间得到了很好的利用。前面有按扣口袋，还有一个更大的拉链口袋，方便快速取用，都有一个小翻盖保护，但仍然很容易拿到。背面有一个翻盖口袋和行李带，因此它可以安全地放在您的滚筒包上。

通过其防风雨拉链打开主隔层，折叠袋愉快地打开，露出笔记本电脑、平板电脑、笔记本电脑和其他插槽，方便取用。我甚至喜欢里面的颜色！

我只希望它能激发更多的爱。无论如何，这都不是一个难看的包，但它感觉非常单调——似乎没有做出什么风格选择。很实用但不个性。对于一些无关紧要的人来说，这是一个结实的袋子。但是它缺少某种该公司不得不在它的其他袋子里省下的东西。

店铺链接

有时候，你只是带着平板电脑或笔记本电脑和书出门，而不想带着一整个邮差风格的包或公文包。这个小家伙有点介于笔记本电脑套和包之间，如果你不介意它像钱包一样的性质，它是那些更简单的旅行的完美伴侣。

笔记本电脑隔间舒适且填充良好。外面有一个滑动口袋，里面有一些放笔之类的角落，足够大，可以放一个 8.5×11”的笔记本或不太厚的书。只是不要试图把食品杂货或任何东西放在那里。

[gallery ids="1839103，1839102，1839101，1839100，1839099，1839098"]

结束是一种足够安全的磁性咬合，但我更希望有一些更实际的东西。我想提一下，在上面的照片中，封闭带看起来有点邋遢，但在一般使用中它真的不像那样，会穿得很好。虽然用肩带携带这个轻便的小家伙感觉很好(它可以很好地收藏)，但像袖子或手包一样携带它并不那么热——一个小把手或肩带会更好。

我会把这个推荐给那些有一个大旅行包，但不想每次去咖啡店都要打包和打开的人。如果你在大袋子里有足够的空间，这可以作为一个小袋子或者笔记本电脑套。

店铺链接

来自一家以手提行李和轻便时尚的日常都市生活用品而闻名的商店，这一件对他们来说是沉重的负担，但与本综述中的其他一些相比却很轻。它的风格微妙而直白，但质量很高。

这种材料是一种重量较轻的彩色帆布，具有酥脆感，可以很快显示出使用模式，例如，一个前袋通常用于存放书籍或钥匙。这可能是我用过的最轻的打蜡包了，这当然对任何想保持简约的人都有好处。简单的皮革表带坚固而舒适，尽管其弹性和直立的特性意味着它们偶尔会妨碍使用。

[gallery ids="1839045，1839054，1839046，1839053，1839052，1839051，1839050，1839044，1839049，1839048，1839047"]

我喜欢巨大的黄铜拉链和拉手，虽然我可以没有皮革位(你可以删除它们)。起初，我并不喜欢这种朴素的天然帆布表带，但它就像包的其他方面一样，让我越来越喜欢。

设计的简洁是好的，但也导致了一些问题。除非你仔细看，否则很难分辨哪一面是正面——只有拉链盖和小标签能暗示出来。一些固定或区分前袋或后袋的东西，甚至简单到移除后面的分隔物，都是受欢迎的。

里面有三个部分，但是起伏的，非结构化的帆布加上有限的拉链入口会使存放和取回有点尴尬，至少比翻盖包更尴尬。笔记本电脑或平板电脑在这里有一个更紧凑的隔间会更好，而不是让它在一个无差别的大区域里游泳。也没有衬垫，所以我建议把你的设备放在它自己的盒子里(这也有助于它填满空间)。

店铺链接

上蜡的帆布通常是棕褐色或棕色的，但这只是传统。我喜欢去年的 Croots 包的森林绿或下面的鞍背包，但这个曼哈顿 Portage Token 包的浓郁海军蓝也很棒。这种材料非常轻，编织精细，几乎没有任何蜡。这意味着它可能不会显示赋予这种包个性的特有磨损和图案。(随时可以自己打蜡。)

这款包带有半翻盖和顶部提手，横跨在笔记本电脑包和公文包之间。它不是特别厚，但有足够的空间放大型文件、笔记本电脑和其他长物品。它的结构意味着即使吃饱了也能保持相对苗条——不会结块。

[gallery ids="1839062，1839056，1839058，1839061，1839059，1839063，1839060，1839055，1839057"]

皮革表带和饰边是漂亮的巧克力色，与蓝色相得益彰。它不重也不硬，特别是肩带非常柔韧——尽管太长了，我不得不打结以避免多余的东西(幸运的是，这样看起来很酷)。按扣的感觉可能有点棘手，但连接牢固。把手可以折叠起来，但在你需要的时候会弹出来，这是一个天才——可能是这一系列包中最好的把手，尽管没有马鞍包那么结实(但什么是呢？).

内部不是我喜欢的。与精致的外部相比，印有品牌图案的红色尼龙似乎有点笨拙，同时感觉这种材料的选择应该允许更多的小口袋。不过，它应该有助于保持干燥，考虑到蜡帆布层的厚度，这是很好的。

当我在 CES 或其他大型展会上需要走很多路，但需要随身携带基本物品时，我经常希望我能把我的笔记本电脑包变成背包，反之亦然。萨拉托加实现了这一点，虽然它最终折衷了两种形式，但它也从根本上解决了一个重要的问题。

这种材料是一种手感柔软的帆布，感觉不太粗糙，但已经显示出一种很好的磨损模式。风雨密封拉链对于任何想在雨中外出的人来说都是好消息，但这种拉链实在太多了。六个在外面，五个在前面！这东西像喜庆的小精灵一样叮当作响。

[gallery ids="1839068，1839067，1839066，1839065，1839069"]

包的背面是一个大口袋，包带放在里面，当它们在里面时，提供额外的衬垫。你把它们拿出来，夹在一些不显眼的小 D 形环上，嘣，它就成了一个背包。反过来做就有点难了，因为你需要确保带子不会在他们的袋子里打成一团。

我会更喜欢一个更优雅的口袋解决方案，尤其是因为一些口袋在这种或那种配置下没有多大意义。皮革底部，虽然在公文包模式下很棒，但在背包模式下似乎有点不平衡。显然，这些是可切换设计固有的缺点，这带来了它自己的好处，但它们值得考虑。我可能喜欢前面有一个大口袋，可以从侧面或顶部打开，里面有子口袋。

内部，虽然它是与上面相同的带水印的红色尼龙，但有大量的小口袋和有用的储物空间，它们都可以独立关闭，这意味着当你从一种模式转换到另一种模式时，无需重新打包。

(我似乎再也找不到这种东西出售了——但如果你喜欢它，就睁大你的眼睛。)

店铺链接

这个袋子上贴着一个标签，上面写着该公司的座右铭:“当你死了，他们会为它而战。”我倾向于相信它。这绝对是迄今为止我有幸评论过的最重的上蜡帆布袋，可能会也可能不会符合你的口味。

橄榄色的帆布很厚很硬，而且从头到尾都打了蜡，而不只是在外面打了一层。缝合是工业级的，可能用了半英里的线。四分之一英寸厚的皮革板像木板一样坚硬，保护着包的背面和底部，另一个用来连接提手。表带是一种折叠的帆布，摸起来比皮革还要坚韧。顶部是一个独特而实用的厚皮革手柄，必要时可以折叠，但感觉非常结实。

[gallery ids="1839078，1839081，1839083，1839082，1839084，1839086，1839085，1839080，1839079，1839077"]

然而，肌肉材质和结构排除了小口袋和笔套等细节。相反，有两个主要的外部口袋，简单地折叠起来，通过翻盖保持关闭；它们和主隔间之间也有空间。在厚重的金属带下面的小侧袋是放手电筒的好地方，但是钢笔可能会消失在底部。

这东西也重得要命。空的，它和另一个轻负载的包一样重。对一些人来说，这种重量会让人放心，但对另一些人来说，这就太多了。

在主隔间里有足够的空间，但几乎没有组织；只有一个翻盖可以固定笔记本电脑(我的 13 英寸 MacBook Pro 非常适合)，除此之外，它就是一个很大的空白空间。这是唯一一款可以和菲尔森的(在我的上一篇综述中)包相媲美的公文包。不过，在最初的几次旅行中，这个肯定会让你的东西变得蜡质。
