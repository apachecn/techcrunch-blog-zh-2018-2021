谷歌提议对网络进行新的隐私和反指纹控制

原文：https://web.archive.org/web/https://techcrunch.com/2019/08/22/google-proposes-new-privacy-and-anti-fingerprinting-controls-for-the-web/

谷歌今天宣布一项新的长期计划，如果完全实现，将使在线营销商和广告商更难在网络上跟踪你。这项新提议遵循了该公司的计划，即改变 Chrome 中 cookie 的工作方式，并使用户更容易阻止跟踪 cookie。

今天的新开放标准提案通过研究 Chrome 如何填补漏洞来扩展这一点，数字广告生态系统可以利用这些漏洞来规避这一点。很快，这可能意味着你的浏览器将拥有新的选项，让你在不失去匿名性的情况下更好地控制你分享的内容。

在过去的几个月里，谷歌开始谈论“隐私沙箱”，这将允许一定程度的个性化，同时仍然保护用户的隐私。

“我们在安全方面享有盛誉。[……]我觉得我们赢得声誉的方式是真正推动网络向前发展，”谷歌负责 Chrome 安全和隐私的工程总监贾斯汀·舒赫(Justin Schuh)告诉我。“我们提供了大量的福利，在许多不同的领域开展工作。我们今天试图做的基本上是为隐私做同样的事情:对我们认为隐私应该如何在网络上发挥作用，我们应该如何默认浏览器和网络更加隐私，有同样大而大胆的愿景。”

这是谷歌今天提议的技术方面:为了防止那种使你的机器可以被唯一地识别为你的机器的指纹识别，谷歌提出了隐私预算的想法。有了这个，浏览器可以允许网站进行足够多的 API 调用，以获得足够多的关于你的信息，从而将你归入一个更大的群体，但不会达到你放弃匿名的程度。一旦一个网站用完了这个预算，浏览器就会停止响应任何进一步的调用。

一些浏览器也已经实现了非常严格的 cookie 拦截。谷歌辩称，这有意想不到的后果，需要有一套商定的标准。“在很大程度上，我们认为其他浏览器供应商真的致力于开放网络，”Schuh 说，他还强调谷歌希望这是一个开放标准，并与网络生态系统中的其他参与者合作开发。

“肯定有很多非故意的错误信息，但只是关于网站如何赚钱和出版商如何实际获得资金的不正确数据，”Schuh 强调说。事实上，谷歌今天指出，它的研究表明，当读者屏蔽 cookies 时，出版商平均损失 52%的广告收入。对于新闻网站来说，这个数字甚至更高。

此外，根据谷歌的说法，屏蔽所有第三方 cookies 不是一个可行的解决方案，因为开发者将通过依赖用户机器的指纹来找到绕过这一限制的方法。然而，虽然你可以选择退出 cookies，并从浏览器中删除它们，但你不能选择不被采集指纹，因为你的机器上没有存储任何数据(除非你定期更改笔记本电脑的配置、你安装的字体以及其他使你的笔记本电脑独一无二的可识别特征)。

谷歌基本上想做的是改变广告生态系统的激励结构。隐私预算不是试图绕过浏览器的 cookie 和指纹限制，而是结合行业在联合学习和差异隐私方面的工作，这意味着在不伤害出版商的情况下，给广告商他们需要的工具，同时仍然尊重用户的隐私。这不是一个容易的转变，谷歌坦率地承认，这将需要数年时间。

“这将是一个多年的旅程，”舒赫说。“我能说的是，我非常有信心，我们将能够改变激励结构。因此，我们致力于采取非常有力的措施来保护用户隐私，我们致力于打击滥用用户隐私的行为。[……]但在我们这么做的时候，我们必须推动平台向前发展，让平台本身提供更强大的隐私保护。”

大多数大型科技公司现在都明白，他们有责任帮助用户保留他们的在线隐私。然而与此同时，个性化广告依赖于对特定用户尽可能多的了解，而谷歌本身的绝大部分收入来自其各种广告服务。听起来这应该会在公司内部制造一些紧张气氛。然而，Schuh 认为谷歌的广告部门和 Chrome 团队是独立的。“归根结底，我们是一个网络浏览器，我们关心我们的用户群。我们将做出最符合他们利益的决定，因此我们必须权衡所有这些因素的适合程度，”舒赫说。他还指出，广告方对用户透明度和用户控制有着非常坚定的承诺——如果用户不信任广告生态系统，那也是一个问题。

不过，就目前而言，Chrome 浏览器中还没有任何可供你尝试的内容或新版本。目前，这只是一个提议，也是 Chrome 团队开始对话的努力。不过，我们应该期待该公司在不久的将来开始尝试这些想法。

就像它提议的改变广告商和网站如何使用 cookies 一样，这对该公司来说是一个非常长期的项目。一些用户会认为，谷歌可以采取更激烈的措施，简单地利用其技术实力来阻止广告生态系统通过 cookies、指纹识别和广告技术研究人员接下来会想到的任何东西来跟踪你。然而，如果谷歌的数字是正确的，那肯定会伤害出版商，很少有出版物能够承受收入下降 50%。我可以理解为什么谷歌不想独自做这件事，但它确实有市场地位，可以更积极地推动这些变化。

苹果在广告业务中没有任何既得利益，它已经通过最新发布的 Safari 做出了更激烈的举动。它的浏览器现在屏蔽了许多追踪技术，包括指纹识别，而没有对广告商做出任何让步。对于出版商来说，T2 的结果与谷歌的 cookie 研究是一致的。

至于 Chrome 的其他竞争对手，Firefox 也开始增加反指纹技术。新贵 Brave 也为所有第三方内容添加了指纹保护，而微软的新 Edge 目前专注于防止跟踪的 cookies。

由于试图找到一条中间道路，Chrome 冒着落后的风险，因为用户今天正在寻找保护他们隐私的浏览器——特别是现在又有令人信服的替代选择了。
