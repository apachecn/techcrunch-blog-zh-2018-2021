原文：https://web.archive.org/web/https://techcrunch.com/2019/08/24/driving-volkswagens-all-electric-id-buggy-concept/

大众电动 ID越野车的概念是令人愉快的和明亮的，结实的和微笑的。这是一辆适合阳光和沙丘的车，或者也许是一辆不太适合在海滩上缓慢行驶的车。

因此，我第一次驾驶全电动越野车的原型车——沿着加州蒙特雷西班牙湾附近的海岸——就带着些许悲伤。毕竟，ID Buggy只是一个概念。它不属于这个世界。至少现在不是。

ID小车仍有机会投入生产。TechCrunch证实，大众已经在与“至少一家公司”进行谈判，以将这款车投入生产。

在三月份的第89届日内瓦国际汽车展上，ID小车概念在全球首次亮相，旨在展示大众的电动未来，并展示其模块化电动驱动工具包底盘(即MEB)的多功能性。2016年推出的MEB是一个灵活的模块化系统——实际上是一个普通零件的矩阵——用于生产电动汽车，大众表示，这使其更高效、更具成本效益。

使用这一MEB平台的第一批车辆将属于ID品牌，尽管这一平台可以并将用于大众集团其他品牌的电动汽车，如斯柯达和西亚特。(大众品牌奥迪和保时捷不会使用MEB，它们正在开发自己的电动汽车平台。)

大众展示了几个ID概念。其中一些，如ID Crozz和ID Buzz即将投入生产。2020年底，的量产版卷发将登陆美国。其他的，比如这辆车，目前还没有进入生产轨道。

驾驶ID小车驱动器

ID Buggy很简单，这正是它应该有的。没有杂乱或惊人的物质享受。相反，这款受20世纪60年代Meyers Manx启发的休闲车没有车顶或车门，尽管防水布可以在挡风玻璃框架和Targa bar之间展开，作为太阳帆或阳光保护。没有门，司机爬进去，相对容易，取决于一个人的一般健康和灵活性。

ID童车超越了它的灵感——标志性的迈耶斯曼克斯童车，在20世纪60年代的加州海滩和冲浪文化中变得流行。

ID越野车也是一个更安静，更平稳的乘坐比迈耶斯曼斯。我还花了一些时间在一辆经典的鲜红色越野车上，它配有四速手动变速器和汽油发动机，这可能是一个触摸化油器。当我换入第一档并驶离时，马恩岛咆哮着，而电动ID越野车安静而平稳地驶出了沙地停车场。

ID Buggy内部的主要细节是缺乏特性和do-dads。上图所示的六边形方向盘上没有乱七八糟的开关；横杆上只有几个控制器。方向盘右侧的一个小集成杆允许驾驶员将车辆移动到前进档、倒档和驻车档。数字仪表组提供基本信息，如速度。

就连刹车和油门踏板也延续了这种简约的设计风格。

仪表板和乘客区同样缺乏特色。这种“东西”的缺乏更多的是关于功能而不是形式，尽管底部的哑光绿色和纹理化的灰蓝色确实做出了视觉声明。ID小车注定要在风雨无阻的环境中行驶。所以设计师们把室内设计成防水的。

在ID小车的车身下是好东西住的地方。

后轮驱动的越野车配备了一个电动马达，可产生201马力和228磅-英尺的最大扭矩。它有一个62千瓦时的电池，一次充电可以行驶155英里(根据WLTP标准)。没有环保局对该范围的估计。它可以在7.2秒内从静止加速到每小时62英里。

不幸的是，这辆原型车上有一个令人窒息的限速器，破坏了我沿着海岸飞驰的计划。

尽管如此，ID越野车提供了一个有趣和容易，轻松的乘坐。它可以轻松处理道路的弯道，其宽大的车身和较高的车尾提供了一种安全感，即使在其他更大的客车中行驶时也是如此。

构建ID小车

目前还不清楚哪家或哪些公司正在洽谈生产这款小车。大众不愿透露姓名；在蒙特雷汽车周期间，即使是海风和万里无云的天空或无尽的超级跑车也不足以让大众员工放松警惕。

有可能这家不知名的公司就是e.Go Mobile。大众今年3月宣布，e.Go Mobile将成为其第一个外部合作伙伴，利用其MEB电动平台推出除大众车型系列之外的其他电动汽车。大众当时表示，一个专用汽车项目已经在计划之中。

一位大众发言人告诉TechCrunch，还没有决定在与e.Go Mobile的合作下生产哪种汽车。可能是马车；也可能是其他交通工具。

然后是福特。今年早些时候，两家汽车制造商宣布了一项合作计划，其中包括福特基于大众开发的MEB 生产电动汽车。

蒙特里的大众汽车公司的人确实表示希望有第三方来制造这款车，或者是它的改良版。正如一位发言人后来告诉TechCrunch的那样，“正如蒙特雷的驾驶所显示的那样，这款车是大众汽车和电动汽车的伟大大使。我相信它会找到很多顾客。”

最后，ID越野车是一个圆滑的巡洋舰，而不是像20世纪60年代的原始海滩轰炸机。它成功展示了大众电动平台的多功能性。毕竟，大众预见到在不久的将来，用于驱动多种消费电动汽车的ID越野车中的关键部件。这是一个公平的假设，ID小车的生产表兄弟将有更多的小工具，包括像门这样的愚蠢的东西。
