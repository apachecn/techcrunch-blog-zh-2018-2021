一些 Windows 7 客户将获得 Windows 7 安全缓刑

原文：https://web.archive.org/web/https://techcrunch.com/2019/08/26/microsoft-enterprise-windows-7-security-updates/amp/

图片来源: Getty Images

再过四个月，Windows 7 将达到生命周期终点，不再接收安全更新。

对于一些仍在运行已有十年历史的操作系统的企业来说，这将是一个问题。从 2020 年 1 月 14 日开始，Windows 7 电脑将停止接收安全补丁，使企业容易受到恶意软件的攻击。

根据最新数据，约 37%的台式机消费者和企业电脑仍然运行 Windows 7，Windows 10 以 41%的比例略微领先。

然而，对于拥有有效 Windows 10 订阅的企业客户来说，将会有一些缓刑。

微软发布的一份鲜为人知的文件称，拥有 Windows E5、微软 365 E5 和政府 E5 订阅的顶级客户将免费获得一年的扩展安全更新。一年期满后，微软将向每台企业设备收取第二年 50 美元的更新费用，第三年每台设备 100 美元。

该文件称，合格订阅必须保持有效，直到年底，并在整个延长的安全更新期内继续接收安全更新。

但对于其他 Windows 订阅计划的其他人，微软将从 1 月份 Windows 7 停止支持的那一刻开始收费，并于 2023 年 1 月最终停止扩展安全更新。

这个软件和服务巨头在三月份开始警告用户，他们将很快停止接收关键和必要的安全更新。微软建议用户升级到 Windows 10，或者作为“最后手段”获得扩展的安全更新。

《计算机世界》最先报道了安全更新扩展的消息。
