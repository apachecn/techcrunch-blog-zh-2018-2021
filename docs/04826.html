有消息称，中国利用 iPhone 黑客攻击维吾尔族穆斯林

原文：https://web.archive.org/web/https://techcrunch.com/2019/08/31/china-google-iphone-uyghur/?guccounter=1

TechCrunch 了解到，在两年多的时间里，许多曾经侵入 iPhones 手机的恶意网站都是针对维吾尔族穆斯林的。

知情人士表示，这些网站是国家支持的攻击的一部分——很可能是中国——旨在针对中国新疆的维吾尔族社区。

这是中国政府在近代史上镇压穆斯林少数群体的最新努力的一部分。根据 T2 联合国人权委员会的数据，在过去的一年里，中国政府在拘留营拘留了超过一百万的维吾尔人。

谷歌安全研究人员发现并且最近在本周披露了恶意网站，但是直到现在还不知道他们的目标是谁。

这些网站是针对宗教团体的运动的一部分，通过访问一个陷阱网页，用恶意代码感染 iPhone。在不受约束地访问 iPhone 软件的过程中，攻击者可以读取受害者的信息、密码，并近乎实时地跟踪他们的位置。

苹果在二月份修复了 iOS 12.1.4 中的漏洞，几天前谷歌私下披露了这些漏洞。黑客行动的消息在本周首次披露。

谷歌称，这些网站在至少两年的时间里每周都有“成千上万的访问者”。

在我们发表文章后，福布斯证实了我们的报道，并表示针对 iPhones 的相同网站也被用来针对 Android 和 Windows 用户。这表明，针对维吾尔人的这场运动的范围比谷歌最初披露的要广得多。

受害者被诱骗打开一个链接，当打开该链接时，将加载一个用于感染受害者的恶意网站。用间谍软件攻击手机用户是一种常见的战术。

其中一名消息人士告诉 TechCrunch，这些网站还感染了无意中访问这些域名的非维吾尔族人，因为它们在谷歌搜索中被编入索引，这促使联邦调查局提醒谷歌要求将该网站从其索引中删除以防止感染。

谷歌发言人拒绝对已发表的研究发表评论。联邦调查局发言人表示，他们既不能确认也不能否认任何调查，也没有进一步置评。

谷歌因没有公布攻击中使用的网站而面临一些批评。研究人员表示，这些攻击是“不分青红皂白的水坑攻击”，没有“目标歧视”，并指出任何访问该网站的人都会被黑客攻击。

但该公司不愿透露谁是这些攻击的幕后黑手。

苹果公司未予置评。一封要求中国驻纽约领事馆置评的电子邮件没有得到回复。
