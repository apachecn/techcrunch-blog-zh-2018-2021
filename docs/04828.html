特斯拉的 Model 3 内饰(甚至方向盘)现在是 100%无皮革的

特斯拉周六表示，其 Model 3 内饰现在完全没有皮革，实现了首席执行官埃隆·马斯克在今年的年度股东大会上做出的承诺。

几年来，特斯拉一直在接近无皮革内饰。但症结在于方向盘，马斯克在 6 月份的公司股东大会上提到了方向盘，以回应 PETA 活动人士的要求。

”我 相信 我们 被 合 合 合 一 【非 受热 转向 车轮， 那是 不是 皮革，”马斯克当时说。有 有 有 挑战 当 热度 热度 非皮 材质也有 如何做好呢

马斯克表示，到 2020 年，Model Y 和 Model 3 将成为纯素食主义者。他不确定该公司是否能在 S 型和 x 型上达到同样的目标。

 

 

维权股东在 2015 年提出一项提案，要求特斯拉在 2019 年前不再在其电动汽车的内部使用动物来源的皮革。虽然股东拒绝了这一提议，但特斯拉确实开始在其汽车中推出更多的“纯素食”内饰组件。

该公司开始提供无皮座椅作为选择。两年前，特斯拉在其 Model 3、Model X 和 Model S 汽车中制定了合成材料标准。
