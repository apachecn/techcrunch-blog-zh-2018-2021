Starboard Value 持有 Box TechCrunch 7.5%的股份

原文：https://web.archive.org/web/https://techcrunch.com/2019/09/03/starboard-value-takes-7-5-stake-in-box/

Starboard Value，LP 上周在提交的 SEC Form 13D 文件中透露，它拥有云内容管理公司Box7.5%的股份。

Starboard Value 投资于那些股价已经急转直下的公司，这可能不是巧合。自 2015 年以来，Box 的股价一直处于过山车状态，当时其股票最初的定价为 14.00 美元/股，但随后在首日飙升至 23.23 美元。近年来，其股价曾高达 28.12 美元，但下跌幅度很大:其 52 周低点为每股 12.46 美元。

“虽然我们不评论与投资者的互动，但 Box 致力于与股东保持积极参与的对话。董事会和管理团队专注于实现增长和盈利，以推动长期股东价值，因为我们继续开拓云内容管理市场，”Box 发言人告诉 TechCrunch。

事实上，现在判断 Starboard 的投资长期来看意味着什么还为时过早。但更普遍的是，众所周知，该公司在其投资组合的公司中扮演着非常积极的角色，有时会增持股份以确保董事会席位，并利用这一地位倡导管理层变革、重组、出售等。

从某种意义上说，Box 现在受到了关注。来自右舷的文件:

“视各种因素而定，包括但不限于发行人的财务状况和投资策略、股票价格水平、证券市场状况以及总体经济和行业状况，报告人可能在未来就其在发行人的投资采取其认为适当的行动，包括但不限于与发行人的管理层和董事会进行沟通， 与发行人的股东或其他第三方就发行人和[Starboard 的]投资进行讨论，包括涉及发行人或其某些业务的潜在业务合并或处置，就涉及发行人或其某些业务的资本化、所有权结构、董事会结构(包括董事会组成)、潜在业务合并或处置的变更向发行人提出建议或提议，或改善发行人财务和/或经营业绩的建议，购买额外股份， 出售其部分或全部股份，从事与股份相关的卖空或任何对冲或类似交易……”

Box 最初是一家消费存储公司，但在 2005 年推出几年后，它转型为企业软件。在此过程中，它筹集了超过 5 亿美元，在 2014 年申请 S-1 之前，它一直是硅谷 SaaS 的宠儿。

S-1 披露了大量的销售和营销支出，批评者对该公司进行了严厉的批评。这导致了记忆中最长的 IPO 延迟之一，从该公司提交申请到 2015 年 1 月在最终完成 IPO ，耗时 9 个月。

虽然最近的损失似乎越来越小，但它们仍然是该公司财务状况中非常突出的一个方面。在上周的 Q2 收益报告中，Box 宣布该季度的收入为 1.725 亿美元，接近 7 亿美元，但它也表示其 GAAP 运营亏损为 3630 万美元，占收入的 21%(去年 GAAP 运营亏损为 3720 万美元，占收入的 25%)。

同时，非 GAAP 运营收入为 50 万美元，占收入的 0%(去年非 GAAP 运营亏损为 650 万美元，占收入的 4%)。负自由现金流也高达-1900 万美元，而一年前为-1000 万美元。换句话说，这些正是吸引激进投资者投资高调上市公司的指标。
