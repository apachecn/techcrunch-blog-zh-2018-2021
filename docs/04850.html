网上发现了一个庞大的脸书用户电话号码数据库

原文：https://web.archive.org/web/https://techcrunch.com/2019/09/04/facebook-phone-numbers-exposed/amp/

图片来源:亚当·贝瑞/盖蒂图片公司

在网上发现了上亿个与脸书账户相关的电话号码。

被暴露的服务器包含超过 4.19 亿条跨地域用户的记录，其中包括 1.33 亿条美国脸书用户的记录，1800 万条英国用户的记录，以及超过 5000 万条越南用户的记录。

但是因为服务器没有密码保护，任何人都可以找到并访问数据库。

每条记录都包含用户唯一的脸书 ID 和账户上列出的电话号码。用户的脸书 ID 通常是与他们的账户相关联的长的、唯一的和公开的号码，可以容易地用于辨别账户的用户名。

但是自从脸书限制访问用户的电话号码以来，电话号码在已经一年多没有公开过了。

TechCrunch 通过将一个已知的脸书用户的电话号码与他们列出的脸书 ID 进行匹配，验证了数据库中的许多记录。我们还通过将电话号码与脸书自己的密码重置功能进行匹配来检查其他记录，该功能可用于部分泄露与用户账户相关的电话号码。

一些记录还包括用户的姓名、性别和所在国家。

来自英国数据库的一组经过编辑的记录。“44”表示+44，即英国的国家代码，“7”表示手机号码。

这是自剑桥分析公司丑闻以来发生的一系列事件后，涉及脸书数据的最新安全漏洞，剑桥分析公司丑闻导致超过 8000 万份个人资料被刮擦，以帮助识别 2016 年美国总统选举中的摇摆选民。

自那以后，该公司发生了几起高调的抓取事件，包括 Instagram，它最近承认批量抓取了个人资料。

最近的这起事件暴露了数百万脸书用户的电话号码，使他们面临垃圾电话和 SIM 卡交换攻击的风险。利用他人的电话号码，攻击者可以强制重置与该号码相关的任何互联网帐户的密码。

安全研究员兼 GDI 基金会成员 Sanyam Jain 发现了这个数据库，并在无法找到所有者后联系了 TechCrunch。经过对数据的审查，我们也不能。但在我们联系了网络主机后，数据库就离线了。

贾恩说，他发现一些个人资料，上面有几个名人的电话号码。

脸书发言人杰伊·南卡罗表示，在脸书切断用户电话号码之前，这些数据已经被搜集。

这位发言人说:“这个数据集是旧的，似乎在我们去年做出改变以消除人们使用他们的电话号码找到其他人的能力之前就已经获得了信息。”“数据集已经被删除，我们没有看到脸书账户受到损害的证据。”

脸书后来声称该服务器包含“大约 2.2 亿条”记录。

但是问题仍然存在，到底是谁搜集了这些数据，什么时候从脸书搜集的，为什么。

脸书长期以来限制开发者获取用户电话号码。该公司还让 T2 搜索朋友的电话号码变得更加困难。但这些数据似乎是在上个月底加载到暴露的数据库中的——尽管这并不一定意味着这些数据是新的。

这一最新的数据曝光是最新的无密码在线公开存储数据的例子。尽管通常与人为错误而非恶意破坏有关，但数据泄露仍然是一个新出现的安全问题。

最近几个月，金融巨头第一美国左派数据曝光，同样曝光的还有 MoviePass 和参议院民主党。
