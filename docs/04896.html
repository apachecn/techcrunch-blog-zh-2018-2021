Spendesk 为其公司卡和消费服务筹集了 3840 万美元

法国初创公司 Spendesk 在 B 轮融资中又筹集了 3840 万美元，现有投资者 Index Ventures 领投该轮融资。该公司多年来已经筹集了 4940 万美元(€4500 万英镑)。

Spendesk 是一种一体化的企业费用和支出管理服务。它可以让您跟踪整个公司的开支，为您的员工提供清晰的审批流程，并简化您的簿记工作。

这项服务基本上像 Revolut 或 N26 一样工作，但是是为了公司的需要。注册后，你就可以在 IBAN 上获得自己的 Spendesk 帐户。您可以充值该帐户并定义不同的策略集。

例如，你可以根据每个人的工作设置支付限额，并定义谁负责批准昂贵的支付。之后，每个人都可以生成虚拟卡用于在线支付，并获得一张实体卡用于商务旅行。

在旅途中，您可以像使用任何公司卡一样，直接使用 Spendesk 付款。如果您必须用现金或其他卡支付，您可以从 Spendesk 移动应用程序中拍摄收据的照片，然后取回您的钱。

许多 Spendesk 用户也将该服务用于其他用例。例如，你可以定义一个营销预算，让营销团队用虚拟卡把它花在脸书或谷歌的广告上。

您还可以从 Spendesk 界面跟踪您的所有在线订阅，以确保您不会为类似的工具付费。如果你雇佣自由职业者，你可以将你所有的发票上传到平台，导出一个 XML 格式的未结发票，然后导入到你的银行门户。

Spendesk 试图超越传统的费用解决方案。例如，该公司试图利用光学字符识别(OCR)来匹配收据和付款，自动填充增值税率等。

随着今天的融资，该公司计划在柏林和伦敦开设办事处，增加更多货币并开发新功能。在过去的一年里，公司从 20 名员工发展到 120 名员工。欧洲现在有 1500 家公司使用 Spendesk。
