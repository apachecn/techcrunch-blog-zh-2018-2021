Mozilla 推出了 VPN，带回了 Firefox 测试试点计划

原文：https://web.archive.org/web/https://techcrunch.com/2019/09/10/mozilla-launches-a-vpn-brings-back-the-firefox-test-pilot-program/

Mozilla 今天宣布将恢复 Firefox 测试试点计划，允许用户在准备好主流使用之前尝试新功能。虽然这个名字很熟悉，但新计划的总体目标与上一次有所不同，重点不在疯狂的实验上，而是更多地放在几乎可以供公众消费的测试产品上。

测试试点计划中的第一个新项目是 Firefox 专用网络 VPN 服务的测试版，现在美国的 Firefox 桌面用户可以使用该服务

Firefox 测试试点项目已经经历了多次迭代。三年前首次推出，它迅速成为许多新功能的孵化地。然而，今年一月，该组织决定将其关闭。

为什么现在带回来？显然，Mozilla 从试飞员用户那里获得了有价值的反馈，他们无疑是最忠实的火狐粉丝。

该组织表示，它希望花时间来发展该计划，这个新版本确实有些不同。“与新推出的测试试点计划的不同之处在于，这些产品和服务可能在 Firefox 浏览器之外，我们将更加完善，距离公开发布仅一步之遥，”该团队解释道。

新的测试试点项目并不是给用户机会来测试 Firefox 团队的一些更古怪的想法，而更像是一个传统的公共测试项目。

该小组写道，新的 VPN 项目是这种方法的一个很好的例子。这是一个试验项目，因为团队希望在公开发布之前对其进行更多的微调。

Firefox 的私有网络并不是试图绕过地理限制，而是主要专注于让用户在使用公共 Wi-Fi 时访问私有网络，并帮助他们向网站和广告追踪者隐藏自己的位置(事实上，许多新的测试试点项目将专注于隐私)。这可能也是 Mozilla 不把它称为 VPN 的原因，尽管很明显它就是 VPN。

Mozilla 的玛丽莎·伍德(Marissa Wood)今天写道:“从最近的事件中得到的一个重要教训是，人们对隐私功能的需求越来越大。Firefox 专用网络是一个扩展，它提供了一个安全、加密的网络路径，以保护您在使用 Firefox 浏览器的任何地方的连接和个人信息

Mozilla 与 Cloudflare 合作进行此次发布，Cloudflare 为其提供代理服务器。它是 Firefox 的扩展，但只在美国提供给 Firefox 桌面用户。目前，它是免费的，尽管有一些迹象表明 Mozilla 将在某个时候开始对这项服务收费。因为这不是一项完整的 VPN 服务，所以该组织能够收取多少费用还有待观察。去年，Mozilla 与 ProtonVPN 合作，提供每月 10 美元的服务。

值得注意的是，Opera 也包括一个免费的内置 VPN 服务，它可以将你的位置设置为美洲、欧洲或亚洲。
