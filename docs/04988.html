脸书推出新的视频工具，加上 Instagram 和 IGTV 日程安排功能

原文：https://web.archive.org/web/http://techcrunch.com/2019/09/16/facebook-rolls-out-new-video-tools-plus-instagram-and-igtv-scheduling-feature/

脸书周一在阿姆斯特丹举行的国际广播大会(IBC)上宣布了一系列针对视频创作者和出版商的更新。这些更新包括对视频直播、脸书的观看派对和创作者工作室的改变，它们包括工具的增强、扩展的功能集和改进的分析等。

亮点包括更好地准备和联播直播，更好地利用观看派对活动的方式，跟踪视频表现的新指标，以及备受期待的提前六个月安排 Instagram/IGTV 内容的选项。

现场视频

在视频直播方面，脸书表示，它听取了那些在其平台上直播的人的反馈，现在正在向脸书页面推出几个广受欢迎的功能(不是个人资料)。)这些变化是为了更好地适应专业广播公司，这些公司希望使用脸书的直播功能，而不是 YouTube 等其他平台。

通过 Live API，出版商现在可以使用“排练”功能仅向页面管理员和编辑进行直播，以便在向所有观众直播之前测试新的制作设置、交互功能和展示格式。QVC 已经测试了这一功能，因为他们每月在脸书直播数百小时，并希望尝试新的工作流程和格式。

出版商还将能够修剪直播视频的开头和结尾，并可以直播长达 8 小时——是之前 4 小时限制的两倍。

后一种能力已经被美国宇航局使用，例如，它广播了八小时的太空行走，它还为直播体育比赛、新闻事件和类似 Twitch 的游戏广播留下了空间。

或许最值得注意的是，该公司意识到直播公司需要为脸书以外的观众服务。现在，出版商将能够使用应用程序，让他们通过直播 API 同时播放多个流媒体服务。

该公司还指出，脸书 Lite 最近也推出了直播视频。

观看派对

此外，脸书还宣布了其共同观看功能 Watch Party 的一些新更新，包括页面提前安排派对以建立预期的能力，支持“重播”，让其他人在播放后欣赏视频，在品牌内容和新分析中标记业务合作伙伴的能力。

至于后者，Creator Studio 中增加了两个新指标:观看分钟数和唯一 60 秒观众数(在观看聚会中观看至少 60 秒的唯一用户总数)。这些是对现有指标的补充，如覆盖面和参与度。

现场评论功能，允许主持人在观看派对中现场分享他们自己的评论，现在也在全球范围内可用。

创作者工作室

总结所有这些是对 Creator Studio 的更新，这是出版商用来在脸书和 Instagram 上发布、管理、货币化和衡量他们的内容的工具。

仪表板将很快在忠诚度洞察中添加一个新的可视化层，以帮助创作者通过测量哪些视频驱动返回观看者来查看忠实粉丝想要查看哪些视频。

一个新的分发指标将根据页面在一系列指标上的历史平均值对每个视频的表现进行评分，这些指标包括:1 分钟浏览量、平均观看分钟数和保留时间。这项功能将在未来几个月推出，将提供一个易于阅读的视频性能快照。

Creator Studio 现在还支持 13 种语言的自动字幕:阿拉伯语、中文、德语、印地语、意大利语、马来语、俄语、他加禄语、泰米尔语、泰语、土耳其语、乌尔都语和越南语。除此之外，还有其他语言，包括英语、法语、葡萄牙语和西班牙语。

Instagram & IGTV 日程安排

最后，出版商和创作者将能够在长达六个月的时间内发布和安排他们的 Instagram Feed 和 IGTV 内容。该公司表示，再过几个月，Instagram Feed 和 IGTV 起草和编辑也将可用。

在今天的声明之前，这个功能已经在 中被发现了，它让社交媒体管理和影响者社区议论纷纷。它还遵循了去年 Instagram API 的更新，允许第三方应用程序进行调度。然而，本地特性并不像其他一些选项那样受到限制。

该功能现在对所有拥有脸书页面的创作者和出版商开放，而以前一些人只看到它的标签是“即将推出”或无法让它工作。故事安排还没有包括在这里，但是如果在后面看到它的加入也不会令人惊讶。
