华特·迪士尼工作室与微软 Azure 合作建立云创新实验室

似乎现在所有的东西都要上云了，为什么电影制作要被排除在外呢？今天，华特·迪士尼工作室宣布与微软围绕一个创新实验室展开为期五年的合作，寻找将内容制作转移到 Azure 云的方法。

该项目涉及华特·迪士尼工作室，这是一个创新的工作空间，迪士尼员工可以在这里尝试将不同的工作流程迁移到云。电影制作软件公司 Avid 也参与其中。

希望通过合作，三方可以提出创造性的、基于云的工作流程，从而加快这家著名电影制作商的创新周期。每个大公司都在寻找创新的方法，不管他们的核心业务是什么，迪士尼也不例外。

由于电影制作涉及越来越多的计算资源，云是一个完美的模型，允许他们根据需要扩大和缩小资源，无论是渲染场景还是添加特殊效果。正如迪士尼的首席技术官杰米·沃利斯(Jamie Voris)所认为的那样，这可能会使这些流程更加高效，从而有助于降低成本和缩短生产时间。

“通过与微软的创新合作，我们能够简化我们的许多流程，让我们有才华的电影制作人可以专注于他们最擅长的事情，”Voris 在一份声明中说。这与许多大型组织寻求的云价值主张相同。他们希望加快上市时间，同时让技术处理一些更普通的任务。

该合作建立在微软与 Avid 现有的合作基础上，两家公司一直在合作，使用 Azure 上的 Avid 软件解决方案为电影行业构建基于云的工作流程。迪士尼将把自己独特的要求加入到这一组合中，并希望在五年的合作中，在更现代的云环境中简化一些工作流程。
