FairMoney 为其新兴市场挑战者银行筹集了 1100 万美元

金融科技初创公司 FairMoney 正在尼日利亚建立一家挑战者银行。该公司最初提供小额贷款，现在计划扩展到活期账户和储蓄。FairMoney 刚刚筹集了 1100 万美元的 A 轮融资(€1000 万)，由 Flourish，DST Global partners 和现有合作伙伴 Newfund，Speedinvest 和 Le Studio VC 领投。

FairMoney 让你从它的移动应用获得贷款。在回答一些问题并分享财务信息后，这家初创公司会分析这些数据集以及你的地理位置、你手机上安装的其他应用程序和其他因素，在几分钟内给你答案。

平均而言，人们借贷相当于 33 美元。最终，如果你总是按时还款，你可以借到 415 美元。利息因还款期和其他因素而异，但最高年利率为 13% ( 更新:利率可以远高于此。月利率范围从 10%到 28% EIR(每月有效利率)，年利率范围从 120%到 260%。)

当你申请贷款时，FairMoney 会使用传统的银行转账方式来支付贷款——在尼日利亚，银行转账只需几分钟。然后，您可以通过合作银行柜员、银行转账或短信使用现金还款。

FairMoney 在尼日利亚有贷款执照。该公司将与小额信贷机构合作，推出活期账户、储蓄和便利支付。最终，FairMoney 希望它能从中央银行获得自己的小额信贷许可证。

像许多挑战者银行一样，FairMoney 希望成为满足您所有银行需求的金融中心——一个应用程序统治所有银行。这就是为什么在你的钱包里装钱的能力是关键。对于没有智能手机的用户，这家初创公司也在开发一个短信界面来转账。
