好利来为其假日租赁搜索引擎 TechCrunch 募集€4000 万英镑的 C 系列资金

原文：https://web.archive.org/web/https://techcrunch.com/2019/09/17/holidu-raises-e40m-series-c/

好利来为其假日租赁搜索引擎募集€4000 万英镑的 C 系列资金

总部位于慕尼黑的假日租赁搜索引擎 Holidu 已经在 21 个国家市场开展业务，已经为€筹集了 4000 万美元的 C 轮融资。

这轮融资由 Prime Ventures 牵头，coparion 和 MairDuMont Ventures 参与其中。包括 EQT Ventures、Venture Stars、Senovo 和商业天使克里斯希琛在内的现有投资者也紧随其后。

Johannes 和 Michael Siebers 兄弟俩在 2014 年成立了 Holidu 搜索引擎，让你可以轻松搜索和预订假日住宿。他们说，他们在葡萄牙冲浪旅行时试图预订度假租赁的经历令人沮丧。

Holidu 声称使用专有的图像识别技术，在 600 个不同的网站上比较超过 1500 万套出租房屋的价格，包括 Airbnb、Booking.com 和 HomeAway。这使得用户可以通过自动发现不同房源中同一物业的价格差异，节省“高达 55%”的预订费用。

“许多网站提供相同的租金，但价格不同，”Holidu 联合创始人兼首席执行官 Johannes Siebers 早在 2016 年就告诉 TechCrunch 。“此外，由于不同网站之间的日历不同步，酒店被重复预订，因此预订被拒率很高。对于消费者来说，不可能获得一个透明的概览。”

为了帮助解决这一问题，好利来还为度假酒店业主开发了一项服务。它被称为“bookply”，提供了一个单一的界面来列出最大的旅游网站上的属性，包括同步日历，创建多语言描述和采购专业摄影。此外，Bookiply 的团队处理旅行者沟通。

Holidu 表示，Bookiply 已经管理着 5000 家酒店，并声称自己是几个欧洲休闲目的地的市场领导者。该公司表示:“重点是那些尚未上网或其数字形象可以优化的业主。”。

同时，为了促进增长，去年好利来收购了它的西班牙竞争对手。这家初创公司现在声称每月有 1000 万访客，并表示将使用 C 轮资金进行产品开发(包括 Holidu 网站和 Bookiply 软件)。它还将发展其 Holidu 合作伙伴，并与更多的业主签约。为了实现这一目标，该公司表示，它计划开设多个地区办事处。
