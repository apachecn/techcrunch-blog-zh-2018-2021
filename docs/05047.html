Twitch 收购游戏数据库网站 IGDB，以改善其搜索和发现功能

原文：https://web.archive.org/web/https://techcrunch.com/2019/09/17/twitch-acquires-gaming-database-site-igdb-to-improve-its-search-and-discovery-features/amp/

亚马逊(Amazon)旗下的 Twitch 进行了一项小规模但具有战略意义的收购，旨在提高其搜索能力，并更好地将观众导向正确的内容。该公司正在收购互联网游戏数据库 IGDB (与亚马逊的 IMDb 没有关系)，这是一个致力于将所有游戏相关信息整合成面向各地游戏玩家的综合资源的网站。作为收购的结果，IGDB 的数据库现在将进入 Twitch 的搜索和发现功能集。然而，IGDB 网站本身不会被关闭。

IGDB 于 2015 年由克里斯蒂安·弗里瑟夫(Christian Frithiof)和总部位于瑞典哥德堡的一个小型团队成立，其游戏内容通过社区贡献和自动化两种方式获得。

该网站包括每款游戏的有用信息，如流派、支持的平台、描述、会员和评论家的评级和评论、故事情节、游戏模式、发行商、发布日期、角色等。你还可以找到一些不太常见的细节，比如玩这个游戏需要多长时间，或者玩家对这个游戏的看法等等。

类似于 IMDb 组织与娱乐产业相关的一切的使命，IGDB 允许声音天才在其网站上要求他们的简介，除了列出与给定标题相关的全部学分。

为了创收，IGDB 提供了一个开发者 API供较小的商店免费使用，或者每月支付 99 美元，最多可请求 5 万次。感兴趣的合作伙伴(例如华硕)可以寻求特殊定价。据我们所知，迄今为止，IGDB 一直在与数千名 API 用户合作。

Twitch 在一份声明中向 TechCrunch 证实了此次收购。

一位发言人说:“每天都有数百万人来到 Twitch 寻找并联系他们最喜欢的横幅和社区，我们希望让人们更容易找到他们正在寻找的东西。”。“IGDB 开发了一个全面的游戏数据库，我们很高兴能让他们帮助我们更快地改进和扩大 Twitch 上的搜索和发现。”

交易条款没有披露，但从财务角度来看，这可能是一笔小交易。根据来自 Crunchbase 的数据，IGDB 只有 10 个人的团队，迄今为止只筹集了 150 万美元。

然而，从战略角度来看，此次收购更有影响力。

Twitch 首席执行官 Emmett Shear 公开谈论了围绕 Twitch 搜索功能的问题，以及如何在这方面进行改进。

“我们希望 Twitch 上的每个地方都能帮助你被发现。如今，接近三分之一来 Twitch 的人使用搜索来找到他们想要的东西。现在，我要第一个承认，我们的搜索功能并不总是最好的体验，”Shear 今年早些时候在 TwitchCon Berlin 上说。“一个错误的字母和您的搜索结果可能会空手而归，或引导您到一个非常不同的飘带，而不是您正在寻找的飘带。因此，我们将修复搜索，使其实际工作，”他承诺。

最近几周，有迹象表明 IGDB 发生了一些事情。

在2019 年 8 月 19 日的一篇博客文章中，IGDB 宣布它将开始“大规模迁移我们的后端、数据库和主机”，并表示该服务“将经历一些变化，一些是暂时的，另一些是更永久的。”作为改变的一部分，它关闭了用户注册或更新个人资料的功能，并关闭了 pulse 新闻、订阅和推荐功能。

现在作为 Twitch 的一部分，IGDB 将把它的免费和高级 API 合并成一个免费层，将清理其他功能并迁移基础设施。其 IGDB 网站将继续保持在线。

“我们的使命一直是建立世界上最全面的游戏数据库。IGDB 的一篇博客帖子写道:“当你是一个小型创业团队时，这样一项巨大的事业可能会非常具有挑战性。”。“通过加入 Twitch，我们将能够利用他们的经验、资源和技能，这将使我们能够加快我们的进步，并提供我们一直梦想的 IGDB 版本。不仅如此，我们两家公司有着相同的文化、核心价值观和对游戏的热情——这使得这是一个完美的结合，”该帖子说。

这是业界常识 Twitch 以前 用过竞争数据提供商巨弹。通常情况下，该公司可能一直在与 IGDB 讨论进行转换，这导致了收购。(该公司拒绝透露这是如何发生的。)IGDB 与 MobyGames 等其他 API 提供商的不同之处在于，它允许其 API 被商业使用，包括被竞争项目使用，并且允许在本地数据库中缓存和存储数据。

来自 IGDB 的整个 10 人团队将继续留在瑞典，但将通过其观众体验组织向 Twitch 汇报。
