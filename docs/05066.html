私人搜索引擎 Qwant 的新 CEO 是 Mozilla 欧洲资深人士特里斯坦·尼托

原文：https://web.archive.org/web/https://techcrunch.com/2019/09/18/private-search-engine-qwants-new-ceo-is-mozilla-europe-veteran-tristan-nitot/amp/

法国初创公司 Qwant 已经对其高层领导团队进行了一次调整，为下一阶段的增长做准备。q want 的非跟踪搜索引擎作为谷歌尊重隐私的替代产品，在国内市场获得了越来越多的关注。

前 Mozilla 欧洲总裁特里斯坦·尼托(Tristan Nitot)去年加入 Qwant，担任宣传副总裁，现已晋升为首席执行官，接替弗朗索瓦·梅萨格尔(Fran ois Messager)——后者也于 2018 年加入，但现在离开了该业务。与此同时，Qwant 的联合创始人埃里克·莱安德里(Eric Leandri)继续担任总裁。

Nitot 是一名互联网资深人士，曾在 Netscape 工作，并于 1998 年帮助创建了 Mozilla Europe，后来他担任了该公司的总裁，并一直呆到 2015 年，后来他离开公司，写了一本关于监控的书，他在产品和通信角色以及开源方面拥有丰富的经验。

最近，他在个人云创业公司舒适云工作了几年。

“我基本上是来帮助(莱安德里)发展公司和构建公司的，”尼托特告诉 TechCrunch，称 Qwant 的创始人是“令人惊叹的企业家，大胆而有远见”。

近年来，随着欧洲对外国数据挖掘科技巨头的担忧加剧，专注于隐私的谷歌竞争对手的市场逆风一直在改善。

去年法国政府宣布将把默认搜索从谷歌切换到 Qwant。如今，购买本土数字技术显然被视为明智的产品选择和良好的政治策略。

与此同时，对搜索巨头谷歌的反垄断关注，无论是国内的还是国外的，都导致了直接有利于搜索竞争对手的政策转变——比如今年早些时候悄悄发布的chromium 引擎默认列表的更新。

这一幕后变化使得 Qwant 首次成为法国市场用户的一个选择。(听到这个消息后，冷嘲热讽的莱安德里感谢了谷歌——但建议 Qwant 用户选择 Firefox 或 Brave 浏览器，以获得不那么令人毛骨悚然的网络浏览体验。)

“许多公司和机构已经决定并基本上意识到，他们一直在使用一个非欧洲的搜索引擎。它收集数据。大量地。这让他们感到不舒服，”尼托特说。“他们还没有做出有意识的决定。因为他们带来了一台电脑，它有一个默认设置了搜索引擎的浏览器——最终你无法选择你的员工使用哪个搜索引擎，对吧。

“因此，他们正在有意识地决定转向 Qwant。我们在这方面投入了大量的时间和精力，现在回报丰厚。”

根据 Nitot 的说法，除了法国政府约 300 万台台式机被默认切换到 Qwant(预计将在本季度完成)，这个支持隐私的搜索引擎还得到了其他政府部门和地方政府以及大型银行和学校的支持。

他认为，专注于学生搜索产品的势头越来越大，比如为 6-12 岁儿童设计的 Qwant Junior，它将性和暴力从搜索结果中排除，并且没有广告。(它将在未来几周内更新。)它还刚刚得到了 Qwant School 的补充:这是一款针对 13-17 岁学生的学校搜索产品。

“所有这些创造了更多的用户——孩子们和他们的父母谈论 Qwant Junior，父母为他们安装 Qwant。因此，这种增长有很大的推动力，”尼托建议道。

Qwant 表示，它在 2018 年处理了超过 180 亿次搜索请求。

成长中的企业当然需要资金来推动。因此，尼托说，涉及可转换债券的融资努力是他在新职位上将重点关注的一个领域。“我们正在筹集资金，”他证实道。

提高效率——尤其是在工程方面——是新任首席执行官的另一个重点。

“剩下的将是对组织本身的关注，即我们如何构建组织。我们如何发展公司文化。例如，帮助或改善工程团队的交付，”他说。“这并不是不好，只是我们需要确保我们投资的每一美元或每一欧元都能带来尽可能多的回报。”

就产品而言，Nitot 近期的注意力将集中在推出 Qwant 搜索引擎的新版本上，这将涉及重新设计核心技术以提高搜索结果的质量。

“我们(用 v2)想做的是提高搜索结果的质量，”他在谈到核心搜索产品时表示。“就质量而言，你不会注意到与你可能使用的其他真正好的搜索引擎有任何区别——除非你知道你的隐私受到 Qwant 的尊重。

“(随着我们筹集到更多资金)，我们将能够拥有更多基础设施，来运行更好、更强大的算法。因此，我们计划在全球范围内改善这一点……每种语言都将受益于新的搜索引擎。这也是一个资金和基础设施的问题。因为网络是巨大的，而且还在增长。

“新版本包括 NLP(自然语言处理)技术……用于理解语言，理解意图——例如，你是想买东西还是在找参照物……或者一个地方或一件东西。这就是我们正在做的事情，但它将对涉及的每种语言都有很大的改进。”

西欧将是搜索引擎 v2 的重点，从法语、德语、意大利语、西班牙语和英语开始，并计划“以后超越它们”。

Nitot 还表示，还将交错推出(从法国开始)，Qwant 计划并行运行新旧版本，在最终切换用户之前对新版本进行质量检查。

“正如我们过去在 Mozilla 所说的那样，运输很困难，”他评论道，拒绝确定 v2 的发布日期(除了说它将在“不到一年的时间内”到来)。“这是一个普遍规律；发布新产品是困难的，这就是我们想在版本 2 中做的……我从 1980 年就开始写软件，所以我知道软件发布日期是如何预测的。所以我很小心不做承诺。”

Qwant 的另一个重点是开发更多自己的广告技术。在这方面，我们的目标是通过减少对微软等合作伙伴的依赖来提高利润率。

Nitot 说:“到目前为止，我们一直在与合作伙伴合作，尤其是在搜索引擎结果页面方面。“我们在上面放了微软的广告。我们的目标是提升广告技术，这样我们就可以依靠我们自己的技术——一些我们可以控制的技术。这有望带来更好的回报。”

像谷歌一样，Qwant 通过在搜索结果旁边投放广告来赚钱。但与谷歌不同的是，这些是上下文广告，这意味着它们是基于一般位置加上搜索本身的内容；而不是定向广告，定向广告需要持续跟踪和分析互联网用户，以便为广告选择提供信息(因此感觉像是广告在互联网上跟踪你)。

提供情境广告是一个选择，让 Qwant 提供一个可信的隐私承诺，山景城根本无法比拟。

Nitot 指出，直到 2006 年，谷歌还提供情境广告，然后才滑向不利于隐私的微定位。 “这是一个不错的老主意，”他在谈到情境广告时说道。“我们正在使用它。我们认为这确实是一个有价值的想法。”

Qwant 也在研究对隐私敏感的广告技术。当前工作的一个领域是个性化。它正在开发一种基于浏览器的客户端加密数据存储，称为 Masq，旨在通过 WebSocket 连接存储和检索应用程序数据。(这里是项目 Masq Github 页面。)

“因为我们不了解产品的使用者，所以很难进行个性化定制。所以我们计划在客户端实现产品的个性化，”他解释道。“这意味着服务器端不会比我们现在有更多的细节，但在客户端，我们正在开发一种开源的东西，它将数据本地存储在您的设备上，无论是笔记本电脑还是智能手机，在浏览器中，数据都是加密的，因此没有人可以重复使用它，除非您决定要这样做。

“它是开源的，所以它是透明的，可以被审计，所以人们可以信任这项技术，因为它运行在他们自己的设备上，它存储在他们的设备上。”

“现在它处于 alpha 阶段，”Nitot 在谈到 Masq 时补充道，拒绝具体说明它何时可以进行更大范围的发布。

Qwant 的新任首席执行官的最终目标是成为欧洲的搜索引擎——这是一个雄心勃勃的目标，但目前仍遥不可及，因为谷歌仍占据着超过 90%的地区市场份额。(这种支配地位使其业务在欧洲陷入了反垄断的困境。)

然而，今天的互联网与昨天的互联网不同，当时网景是浏览的主要产品——直到微软捆绑其竞争对手新贵作为 Windows 的默认浏览器后，Internet Explorer 取代了它的位置。其余的，正如他们所说，是互联网的历史。

许多事情已经改变，许多事情正在改变。但是滥用市场力量已经是老生常谈了。随着监管者对今天的自利违约采取行动，有像 Qwant 这样的精明的替代品准备好了，并等待着为消费者提供一种不同的价值。

“Qwant 是在欧洲为具有欧洲价值观的欧洲公民创建的，”Nitot 说。“隐私是我们使命的核心价值观之一。法国数据保护机构 CNIL 于 1978 年在法国成立，这并非偶然。这是第一次创造出这样的东西。然后 GDPR[通用数据保护条例]在欧洲诞生了。这不是偶然发生的。这是一个价值观的问题，也是人们看待自己生活和周围事物的方式、政治等等。我们非常关注法国的隐私。它写在欧洲人权宣言中。

“我们生产的产品反映了这些价值观，因此对欧洲用户很有吸引力。”
