Cloudflare 有一个对抗机器人和气候变化的新计划

原文：https://web.archive.org/web/https://techcrunch.com/2019/09/23/cloudflare-bot-fight-mode-climate-change/amp/

图片来源: Getty Images

Cloudflare 正在通过一种新的“战斗模式”加大对机器人的打击力度，它说这种模式将挫败和阻止机器人运营商的恶意活动。

机器人因抓取网站和滥用开发者权限下载大量用户数据而臭名昭著。太多时候，机器人试图利用这个系统，通过压低音乐会或机票价格，以最低价格批量购买，然后以更高价格卖出。更糟糕的是，一些模仿真实用户，用暴力进入网站，里面有被盗密码的列表。

Cloudflare 每天收到 30 亿个 bot 请求。现在该公司表示，它已经“决定反击。”

其新的“机器人战斗模式”，Cloudflare 今天启用了所有帐户的免费选择加入功能，将检测和服务具有故意计算密集型挑战的机器人。当机器人试图解决这个不可能的难题时——实际上只有一小部分代码对机器人可见——机器人的服务器将最大限度地发挥其处理能力，搅动云资源，提高机器人运营商的成本。

虽然该公司表示，从长远来看，它的努力将阻止 bot 活动，但它承认，短期内，它的努力将导致云服务器超时工作，从而消耗更多的电力，需要更多的冷却——所有这些都会导致更大的能源消耗。

“我们喜欢挫败机器人的想法，”Cloudflare 的首席技术官 John Graham-Cumming 告诉 TechCrunch。但他表示，该公司“注意到”电力和冷却等 bot 资源的激增如何与碳排放直接相关。他说，如果该计划会导致对自然资源的利用，一些内部人士最初反对该计划。

该公司找到了一个简单的解决方案:种植树木来抵消机器人活动及其拆除所产生的碳排放。

“通过让机器人做额外的工作，我们可能会增加[处理器]使用的碳排放，并决定通过植树来抵消，这将产生长期影响，”格雷厄姆-卡明说。"从长远来看，我们的目标很简单:结束恶意机器人作为一种可行的做法."

种植的每棵树可以吸收大约一年的双核计算能力。但考虑到树木需要时间生长，Cloudflare 表示，其捐款将导致该公司遇到并关闭的每一个失败的机器人种植 25 棵树。

Graham-Cumming 说，植树可以抵消碳排放，但是减少互联网上的不良机器人数量将带来最大的净效益。

“如果我们成功做到这一点，鉴于如今有多少互联网资源被恶意机器人浪费，环境影响将是巨大和积极的，”他说。

挫败机器人并不是 Cloudflare 的唯一武器。当可以的时候，它会要求它的一个行业合作伙伴将这个机器人下线。如果该机器人由一家作为带宽联盟成员的公司托管，该联盟由一些最大的云和 web 主机组成，Cloudflare 将交出互联网地址以关闭该机器人。

Cloudflare 不是反机器人领域的唯一参与者。今年早些时候,我们对 Kasada 进行了简介，这是一家旨在引诱机器人的初创公司，旨在阻止和抑制机器人运营商瞄准其客户的网站。Cloudflare 表示，其规模和覆盖范围——覆盖超过 2000 万互联网资产——将有助于加速所谓的 bot 经济的消亡。

Graham-Cumming 表示，虽然该功能目前是可选的，但该公司计划在年底前默认向用户推出该功能。
