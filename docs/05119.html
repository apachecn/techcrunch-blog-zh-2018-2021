Unity 用新的模拟产品押注云测试

在过去的几年里，Unity 的估值随着他们成为游戏开发商之外的重要客户的公开野心而膨胀。

实时渲染游戏引擎的一个更有趣的用例是帮助公司在虚拟环境中训练他们的系统。这已经成为机器人初创公司和自动驾驶汽车公司工作流程的一个关键部分，这些公司构建的技术需要在不断变化的环境中反复训练。

今天在该公司的 Unite 会议上，Unity 宣布他们正在为这些用例开发一款专用产品，名为 Unity Simulation 。该产品目前处于封闭测试阶段，但由于 Unity 和谷歌云之间的合作，它允许客户在云连接的硬件上运行模拟。

“通过将 Unity 的实时 3D 开发平台与云的规模和灵活性相结合，Unity Simulation 将帮助企业加速创造更好、更安全、更可靠的产品，”AI 的 Unity 副总裁 Danny Lange 表示，他曾在优步领导机器学习。

对于自动驾驶汽车公司来说，这些模拟可以帮助汽车在边缘情况下建立虚拟的道路体验，可以测试技术的极限。类似地，机器人公司可以在尚未建造的真实模型上测试产品。

较大的客户已经在使用 Unity 解决这类问题，但他们一直在使用自己的硬件。Unity Simulation 依靠 Google Cloud 来运行这些并行测试，可以让想要在虚拟环境中利用测试的小客户更加容易。

该产品可能会吸引游戏世界以外的客户，但用例并不排除游戏工作室。Unity 网站上的一个例子详细介绍了该服务如何帮助工作室快速模拟游戏的数千种结果，以帮助开发者适当调整用户的难度水平。对于多人游戏，像这样的工具可以探索新的物品或能量是否可以组合起来使单个用户过于强大。这项服务还可以帮助电影公司确定他们的片名有多稳定。

Unity 的声明是在这家游戏公司的敏感时期发布的，该公司因 Unity 的一名前女性高管对其现任首席执行官 John Riccitiello 的性骚扰指控而震惊了。里奇泰洛今天早上在哥本哈根 Unity 的“团结”会议上发表了讲话，但没有谈到最近的诉讼。

这家初创公司最近披露，它已经以 60 亿美元的估值完成了一轮 5.25 亿美元的融资，据报道，它的目标是明年进行 IPO。
