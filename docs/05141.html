微软敦促 Windows 用户安装紧急安全补丁

微软警告 Windows 用户安装“紧急”带外安全补丁。

这家软件巨头在一份公告中称，某些版本的 Internet Explorer 存在安全漏洞，使得攻击者能够在受影响的设备上远程运行恶意代码。用户可能通过访问恶意网页或被诱骗点击电子邮件中的链接而被悄悄感染。

“成功利用该漏洞的攻击者可以控制受影响的系统，”微软表示。

微软表示，该漏洞正在被积极利用，尽管该漏洞的细节尚未公开。

根据最近的数据，超过 7%的浏览器用户正在运行受影响版本的 Internet Explorer 9、10 和 11。所有受支持的 Windows 版本都会受到影响，包括 Windows 7、Windows 8.1 和 Windows 10，以及多个 Windows Server 版本。

大多数用户可以使用 Windows Update 安装修补程序。

微软还发布了内置恶意软件扫描器 Windows Defender 的修复程序，如果被利用，可能会触发拒绝服务条件，导致应用程序无法工作。

该公司表示，用户不需要采取任何行动来修复 Windows Defender 中的漏洞。

微软在其典型的每月补丁周期之外发布紧急安全补丁是罕见的，但并非闻所未闻。该公司通常会在每月第二周的所谓“补丁星期二”发布安全补丁，但也会在有可用补丁时，尽快发布正在被积极利用的重大漏洞的补丁。

国土安全部在 T4 发出警告，敦促受影响的用户安装补丁。
