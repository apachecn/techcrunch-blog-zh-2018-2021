亚马逊在 Alexa 活动上宣布的一切 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/09/25/everything-amazon-announced-at-its-alexa-event-today/

就像现在假期前的典型情况一样，亚马逊刚刚在今天的 Alexa 活动中向我们展示了一系列全新的硬件设备，包括新的 Echo 智能扬声器和屏幕，以及其品牌 Ring 和 Eero 的其他智能家居和联网设备。还有一系列惊喜，比如 Alexa 耳塞、眼镜和智能戒指。哦，对了，还有一个新的 Alexa 驱动的烤箱。

新回声

此次活动的头条是亚马逊对其 Echo 系列设备的最新更新。

带时钟的回声圆点

亚马逊并不喜欢过于有创意的名字，它对入门级的 Echo——Echo Dot 进行了漂亮的升级。新款 Echo Dot with Clock 现在的特点是前面有一个数字闹钟，如果你喜欢，可以不时调暗。这种 LED 显示屏还允许点显示天气，当询问时，或倒计时定时器。

然而，新的点并没有取代原来的点。如果你想要额外的功能，只需花 10 美元升级到 59 美元。

回声工作室

音响发烧友一直在等待的回音。新的 Echo 工作室是亚马逊对谷歌的 Nest Max 和苹果的 HomePod 的回应。这款售价 199.99 美元的设备支持 3D 音频和杜比全景声。它有五个驱动器，包括一个向下的 5.25 英寸低音扬声器，一个 1 英寸的前置高音扬声器和三个 2 英寸的中音扬声器，指向不同的方向。

它有一个 24 位 DAC 和一个 100 kHz 带宽的功率放大器，用于高分辨率、无损音乐播放——你知道，就像亚马逊音乐高清服务提供的那样。

回音芽

令人惊讶的是，Alexa 通过亚马逊自己的 129 美元无线耳塞来到你的耳边。 Echo Buds 包括 Bose 的主动降噪技术，并提供对 Alexa 的免提访问，因此你可以做一些事情，如播放音乐，获取方向，订购优步等等。稍后，你也可以在购物时查看全食超市的库存。(对于那些仍然在实体店购物的人来说)。

Echo Buds 提供 5 小时的音乐或 4 小时的通话时间，并带有一个最多可容纳 3 次充电的外壳。只需轻轻一点，Echo Buds 就可以访问你手机的另一个助手，比如 Siri 或谷歌助手。

回声秀 8

129 美元的 Echo Show 8 是 Echo 设备的 10 英寸旗舰型号的一个较小的 8 英寸版本，带有一个屏幕。它比 Echo 5 大一点，Echo 5 更像是一个闹钟的替代品，所以对于观看视频和进行语音通话来说是有意义的……或者顺便看看孩子。该设备具有高清显示屏和内置相机快门。

回声

该公司用更好的声音和新的彩色织物覆盖物(暮光蓝、木炭、希瑟灰和砂岩)更新了现有的回声扬声器，但价格仍然和以前一样，为 99 美元。换句话说，没有必要跑出去买这个回声来取代你的旧的，但如果你已经在市场上，你会得到的回声稍微好一点。

回音辉光

这个有趣的 Alexa 配件基本上是一个小夜灯，可以随着音乐改变颜色和闪烁。但是 30 美元，对于一点点乐趣来说是一大笔钱。让我们等待不可避免的儿童版点+辉光束，好吗？

Echo Flex

你现在可以带着新的 25 美元的 Echo Flex 设备带着 Echo 去浴室了。(不，实际上，这是亚马逊在演示区的设置。)Flex 可以插入任何可用的插座，充当小型智能扬声器。没有摄像头，但它可以响应语音命令，比如在你准备工作时给你提供新闻和天气。它还可以用 14 美元的配件升级，包括智能夜灯或运动传感器。Flex 是 25 美元。


回声帧&循环

亚马逊推出了其测试版，仅限受邀者的硬件设备。一款是一副售价 179.99 美元的支持 Alexa 的眼镜，它在框架中贴了 4 个微型扬声器，用于免提 Alexa。

另一款是售价 129.99 美元的 Alexa 智能戒指，配有 2 个麦克风，可以将 Alexa 戴在手指上，进行快速提问或简短通话。

其他硬件

新 Eero mesh Wi-Fi 路由器

亚马逊推出了下一代 Eero 路由器的，它现在可以更好地与 Alexa 配合使用。该路由器可以让你用语音命令关闭或暂停 Wi-Fi，这些功能将通过 API 的方式从 TP-Link、华硕、Linsky 和 Arris 的其他路由器中推出。这款路由器售价 99 美元，三件装售价 249 美元，今天在美国上市，今年晚些时候在欧洲上市。

亚马逊智能烤箱

没有一个新的 Alexa 驱动的烤箱，亚马逊的活动是什么，对不对？这一次，亚马逊用 250 美元的组合对流烤箱、微波炉、空气油炸锅和食物加热器来应对六月烤箱。一旦与 Echo 配对，你就可以要求 Alexa 做一些事情，比如预热烤箱，开始或停止烹饪，等等。

新型环形凸轮

随着售价 199 美元的 Ring Stick Up Cam Elite 上市，Ring Stick Up Cam 的价格降至 99 美元。与粘贴式摄像机不同，这款新型摄像机将通过以太网供电。环形室内摄像机是一种有线的廉价版本，售价 60 美元。

更多东西

Alexa 变得越来越聪明和古怪

亚马逊今天还花时间提到了 Alexa 功能的一些升级。

  • 这包括一个多语言模式，Alexa 现在可以在美国说英语和西班牙语；加拿大的法语和英语；还有印度的印地语和英语。
  • 一个新的神经文本到语音模型，它将使用机器学习来帮助 Alexa 更具“表现力”
  • 名人声音！因为谁不想让 Alexa 听起来像塞缪尔·L·杰克逊呢？不真的。杰克逊之声将以 99 美分的价格添加到 Echo 设备上，更多名人的添加正在进行中。
  • Alexa 即将来到通用，包括 2018 及更新的凯迪拉克、雪佛兰、Buic 和 GMC 车辆
  • 扫描烹饪是 Echo Show 的升级版，它可以让你扫描条形码，让新的亚马逊智能烤箱正确准备食物。该功能与数百个全食品牌合作，如 365 daily Value，加上 Gardein，玛丽卡伦德等。
  • Certified for Humans 是一项新计划，将向搜索亚马逊的消费者突出易于使用的智能家居产品。不过，简单只是一个要求。另一个是“与 Alexa 合作”
  • Alexa Guard 得到更新，不仅可以监听碎玻璃、烟雾或一氧化碳警报。现在，它还可以监听人类活动，比如当警卫处于离开模式时，是否检测到脚步声、说话声、咳嗽声或关门声。
  • Alexa Guard 也有例行程序(例如，“Alexa，我要出去了”来武装你的系统和锁门，等等。)
  • 将于今年晚些时候推出的 Alexa 补给服务将在智能家居设备电量不足时提醒你，比如智能锁中需要更换的电池，或者准备更换的空气过滤器。
  • 新话语:你可以问 Alexa，“告诉我你听到了什么”和“你为什么这么做？”这将有助于清理意外的 Alexa 活动的问题。(例如黑客…或幽灵)
  • 当你不在家时，Alexa 可以通过响铃来回答你的门铃，比如说“你好，我是 Alexa。”。这是送货吗？或者“嗨，我是 Alexa。”。我怎么帮你？"
  • 在 Alexa 上分享照片在 Echo 设备和 Alexa 应用程序之间分享照片，并通过表情符号做出反应。
  • Alexa connections :提醒顺便访问或通过顺便访问的联系人找到共同的可用性
  • 访问 Spotify 的播客，记住您离开 Spotify、SiriusXM 和 iHeartRadio 的播客的位置

Alexa 驱动的食品网络厨房服务

顾客喜欢在厨房里观看食谱视频的 Alexa。因此亚马逊与其合作伙伴 Food Network 加倍努力，为 Echo Show 用户提供直播和点播的烹饪课程。这项服务将于 10 月份推出，虽然 Alexa 是独家语音合作伙伴，但它也将在手机和平板电脑上提供。哦，巴比·福雷也参与了。

亚马逊人行道

今天更有趣的介绍之一是 Amazon Sidewalk ，亚马逊的低带宽、长距离无线协议，旨在连接你家周围和周围的所有物联网设备——包括蓝牙或 Wi-Fi 无法到达的地方。该网络使用免费的无线 900 兆赫频谱，可以允许像 Alexa 连接的邮箱或院子里的灯这样的事情。
