战地老兵 StrongSalt(前 OverNest)宣布获得 300 万美元种子资金 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/09/26/battlefield-vets-strongsalt-formerly-overnest-announces-3m-seed-round/

2016 年，TechCrunch Disrupt NYC 战地上，StrongSalt 现身，并宣布了一款用于搜索加密代码的产品，这款产品至今仍不常见。今天，该公司宣布了由 Valley Capital Partners 领投的 300 万美元种子融资。

StrongSalt 创始人兼首席执行官 Ed Yu 表示，加密仍然是一个困难的命题，当你看到大多数违规行为时，加密并没有被使用。他说，他的公司希望简化向应用程序添加加密，并推出了一项新服务，让开发人员以 API 的形式添加加密。“我们决定推出我们称之为 API 平台的产品。它就像基础设施，允许您将我们的解决方案集成到任何现有或任何新的应用程序中，”他说。

该公司最初的想法是创建一个搜索加密代码的产品，但俞说，这项技术作为一个跨应用程序适用的 API 有更多的效用，这就是为什么他们决定将其打包为服务。这与用于通信的 Twilio 或用于支付的 Stripe 没有什么不同，只是在这种情况下，您可以内置可搜索的加密。

可搜索部分实际上是一个相当大的问题，因为正如于指出的，当你加密数据时，它就不再是可搜索的了。“如果你对所有数据进行加密，你就无法在其中进行搜索，如果你无法在其中进行搜索，你就无法找到你要找的数据，显然你无法真正使用这些数据。所以我们实际上解决了那个问题，”他说。

开发人员可以将可搜索加密作为他们应用程序的一部分。对于已经在使用商业产品的客户来说，该公司的 API 实际上与流行的服务相集成，使客户能够加密存储在那里的数据，同时保持其可搜索性。

“我们将在 Box、AWS S3、谷歌云、Azure 的基础上提供存储 API，这取决于客户拥有什么或想要什么。例如，如果客户已经有了 AWS S3 存储，那么当他们使用我们的 API 时，在加密数据之后，数据将被存储在他们的 AWS 存储库中，”Yu 解释说。

对于那些没有存储服务的公司，该公司正在提供一项服务。此外，他们还使用区块链来提供共享、审计和管理加密数据的机制。“我们还使用区块链通过记录发送方的授权来共享数据，因此接收方可以检索重建密钥所需的信息，以便检索数据。这简化了共享情况下的密钥管理，并确保发送者共享的可审计性和可撤销性，”于说。

如果你想知道该公司自 2016 年以来是如何生存的，虽然今天才获得种子期，但在此之前它已经有过几次小规模的种子期，并与美国国防部签订了合同，这取代了对大量早期资金的需要。

“国防部正在寻找一种在计算机之间实现安全通信的解决方案，他们需要一种安全存储数据的方法，因此我们为他们提供了一种解决方案，”他说。事实上，正是这项工作引导他们构建了今天提供的商业 API 平台。

该公司成立于 2015 年，目前拥有 12 名员工，遍布全球。
