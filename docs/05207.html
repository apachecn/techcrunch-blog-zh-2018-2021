盖茨比为其现代网络开发平台筹集了 1500 万美元的首轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2019/09/26/gatsby-raises-15m-series-a-for-its-modern-web-development-platform/

盖茨比为其现代网络开发平台筹集了 1500 万美元的首轮融资

Gatsby ，一个使用 React 和 GraphQL 等现代网络技术帮助开发者更快建立更好网站的平台，今天宣布它已经筹集了 1500 万美元的 A 轮融资，由 CRV 领投。之前的投资者 Trinity Ventures、Mango Capital、Fathom Capital 和 Dig Ventures 也参与了投资，Kong 首席执行官 Augusto Marietti 和 Adobe CPO Scott Belsky 也参与了投资。该公司此前筹集了 380 万美元的种子资金。

虽然盖茨比可能还不是一个家喻户晓的名字，但自 2015 年推出以来，它已经迅速增长。如今，它的用户包括 IBM、PayPal、Braun、Airbnb 和 Impossible Burger 等。该公司认为，前 10，000 家网站中约有 1%现已建立在该平台之上，这有望使这些公司摆脱旧的 LAMP 堆栈，转向更现代的堆栈，基于现代开源工具和工程实践。Gatsby 还抛弃了一个单一的 CMS 系统，取而代之的是将各种工具整合在一起，这些工具仍然允许内容创作者使用 WordPress 或 Drupal 这样的平台来创建一个本质上的无头 CMS 系统。在这种情况下，Gatsby 只是成为 CMS 的表示层。

“我们花了四年时间将盖茨比打造成为建设现代网站的最全面的平台，”盖茨比创始人兼首席执行官凯尔·马修斯写道。“在 Gatsby 上开始、构建和部署尖端 web 堆栈需要公司花费数月甚至数年的时间。”

盖茨比本身就是基于 GatsbyJS 开源项目。该公司表示，自该项目启动以来，已有 2500 多人捐款。Matthews 说 Gatsby(该公司)现在每年向开源项目贡献大约 300 万美元，其中包括核心的 Gatsby 工具和围绕它的插件生态系统。

像类似的开源项目一样，Gatsby 通过提供托管服务来帮助开发团队快速建立一个新网站，每个网站的价格从 50 美元/月起，从而将其工具货币化。
