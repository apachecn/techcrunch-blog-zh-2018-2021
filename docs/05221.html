Package Free获得450万美元用于推广可持续CPG产品TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/09/26/package-free-picks-up-4-5-million-to-scale-sustainable-cpg-products/

气候危机仍然只是一场危机。这促使全国(和全球)的人们采取行动，尤其是在涉及到他们自己的生活方式时。

劳伦·辛格就是这样一个人。在NYU大学学习环境科学和政治学后，她开设了一个名为垃圾是给扔垃圾者的的博客，让零废物的生活方式更容易被每个人接受和理解。但是还有一个问题。即使可持续的CPG产品急剧增长，这些品牌也很少具有与传统CPG产品在价格上竞争的规模，并且缺乏面向所有人的分销渠道。

这就是免费套餐发挥作用的地方。今天，Package Free宣布，自2017年推出以来，它已经筹集了第一笔资金，由Primary Ventures牵头提供了450万美元的种子资金。Scooter Braun的TQ Ventures、Day One Ventures、Peleton的Ryan Engel、Wall Group的Brooke Wall和Casper创始人Neil Parikh等人也参与了这一轮投资。

Package Free最初是一个可持续消费品牌的小型弹出式商店，在实体环境中展示他们的商品。这些品牌自己支付1000到3000美元来参与，并从弹出式广告中获得100%的利润。

辛格说，到第一个月结束时，每个品牌都收回了投资。到第三个月末，免费套餐已经成为这些品牌的主要收入来源。那时，他们转而采用传统的零售模式来创收，推出了一个电子商务网站。

今天，Package Free是一个成熟的经销商。这家弹出式商店现在在纽约布鲁克林的时尚街区威廉斯堡拥有永久的地位，在绿点有自己的仓库。该公司批发购买他们的库存，并对与之合作的供应商实施极其严格的指导方针，尤其是无一例外的无塑料政策。

通过无包装销售的品牌不仅必须使用所有天然成分和无塑料，而且还必须运送到无包装仓库而不使用任何塑料。如果货物中含有塑料，该公司实际上会向供应商收取一定比例的费用，如果第二次违规，则会提高这一比例。三振出局，那个卖主就永远出局了。

辛格说:“我们知道不用塑料完全有可能做这些事情，只是现在还不规范。”“因此，我们试图改变可持续包装的基本基准。我真的认为浪费的负担不应该落在消费者身上。它应该首先落在制造商身上，然后是经销商。”

一旦产品到达仓库，Package Free就会重复使用原始货物附带的垫料(包装材料)，这意味着该公司从不使用“原始垫料”。包装免费运送给消费者的盒子是100%可回收的，运输标签也是100%可回收的。事实上，每个免费包装盒上都印有“我不是垃圾”的字样，以及关于垃圾的更多事实。

有了这笔资金，Package Free希望扩展到创造自己的可持续CPG产品，首先解决目前无法通过供应商合作伙伴获得的产品的“空白区域”。辛格拒绝透露更多关于Package Free首批产品的细节。

Package Free也在寻求招聘，特别关注垂直营销，因为该公司迄今尚未进行任何正式营销或付费营销。

最终目标是通过规模经济，将可持续的CPG与传统的CPG产品放在同一个竞技场上。价格是日常消费者和可持续产品之间的主要障碍，Singer的目标是将可持续CPG类别作为一个整体扩大到可以与联合利华和宝洁在全球范围内进行合理竞争的程度。
