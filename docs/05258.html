微软的 Windows 虚拟桌面服务现已正式推出 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/09/30/microsofts-windows-virtual-desktop-service-is-now-generally-available/

微软今天宣布 Windows 虚拟桌面(WVD) ，这是去年九月宣布的基于 Azure 的系统，用于虚拟化 Windows 和 Office 用户体验，现在已经全面上市。使用 WVD，企业可以让员工访问虚拟化应用和远程桌面，包括提供多会话 Windows 10 体验的能力，这使微软自己与提供虚拟化 Windows 桌面和应用的其他供应商有所不同。

除了让这项服务普遍可用之外，微软还在全球范围内推出这项服务，而预览版只在美国推出，最初的计划是在全球范围内慢慢推出。WVD 的首席工程主管 Scott Manchester 也告诉我，有 2 万多家公司报名参加了预览。他还指出，随着视频会议体验的显著改善，微软团队在 WVD 获得了更多支持。

在宣布 WVD 预览版后不久，微软收购了一家名为 FSLogix 的公司，该公司专门提供与微软通过 WVD 提供的相同类型的虚拟化 Windows 环境。正如微软公司负责微软 365 的副总裁在今天的发布会之前告诉我的那样，该公司从 FSLogix 那里获得了很多专业知识，以确保用户在 WVD 上的体验尽可能流畅。

微软 365 的 CVP 布拉特·安德森指出，正如企业越来越习惯于将他们的一些基础设施迁移到云(并让其他人担心管理它)，现在也有越来越多的组织需要同样的桌面体验。“他们将云视为一种方式，表示‘听着，让专家来管理基础设施。他们可以优化它；他们可以微调它；他们可以确保一切顺利，”然后我就有了第一方服务—在这个例子中是微软—我可以利用它来简化我的生活，并使我能够按需增加和减少容量，”Anderson 说。不过，他也指出，确保这些服务始终可用可能比其他已经迁移到云的工作负载更重要。毕竟，如果你的桌面停止工作，你就无法完成更多的工作。

Anderson 还强调，如果客户想要在云中使用多会话 Windows 10 环境，WVD 是唯一的选择，因为这是获得许可的唯一方式。“我们已经建立了操作系统，我们建立了公共云，所以这种组合将是独一无二的，这使我们有能力确保 Windows 10 体验在公共云的基础上是绝对最好的，”他指出。

他还强调，收购 FSLogix 使他的团队能够与 Office 团队合作，优化那里的用户体验。正因为如此，当你启动一个新的虚拟化版本的 Outlook 时，只需要一两秒钟就可以加载，而不是一分钟。

许多公司仍在寻求升级其旧的 Windows 7 部署。微软将很快停止为他们提供免费的安全补丁，但在 WVD，这些用户仍然可以访问虚拟化的 Windows 7 桌面，免费的安全更新将持续到 2023 年 1 月。Anderson 不认为这将是采用 WVD 的主要驱动力，但他确实看到了“正在进行过渡的客户群体”

如果企业拥有符合条件的 Windows 10 Enterprise 或微软 365 许可证，他们可以在 WVD 上访问 Windows 10 Enterprise 和 Windows 7 Enterprise，而无需额外的许可费用(当然，他们消耗的 Azure 资源会让他们付出代价)。
