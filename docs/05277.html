Streamlit 推出开源机器学习应用开发框架

Streamlit 是一家新的机器学习创业公司，由曾在 GoogleX 和 Zoox 工作的行业资深人士组成，今天推出了 600 万美元的种子投资和一个灵活的新开源工具，使机器学习工程师更容易创建定制应用程序，以与他们的模型中的数据进行交互。

种子轮由 Gradient Ventures 牵头，彭博贝塔公司参与其中。solo 投资者的名人录也参加了，包括 Color Genomics 的联合创始人 Elad Gil、#Angels 的创始人 Jana Messerschmidt、Y Combinator 的合伙人 Daniel Gross、Docker 的联合创始人 Solomon Hykes 和 Insight Data Science 的首席执行官 Jake Klamka。

至于产品，Streamlit 的联合创始人 Adrien Treuille 表示，作为机器学习工程师，他和他的联合创始人处于一个独特的位置，可以了解工程师的需求，并建立一个满足他们要求的工具。关键不是构建一个一刀切的工具，而是开发一个足够灵活的解决方案来满足多种需求，这取决于用户正在处理的数据的性质。

“我认为 Streamlit 实际上在这个市场上拥有独特的地位。虽然大多数公司基本上都在试图将机器学习工作流程的某些部分系统化，但我们给工程师提供了这种乐高积木来构建他们想要的任何东西，”Treuille 解释道。

使用 Streamlit 构建的定制自动驾驶汽车数据应用程序，使机器学习工程师能够与数据进行交互

Treuille 表示，训练有素、拥有一套独特技能的机器学习工程师实际上最终会花费大量时间来构建工具，以理解他们拥有的大量数据。Streamlit 正试图帮助他们更快地使用他们习惯使用的编程工具来构建这些工具。

他说，通过几行代码，机器学习工程师可以非常快速地开始构建工具来理解数据，并帮助他们根据数据类型以任何有意义的方式与之交互。这可能意味着构建一组具有不同变量的滑块来与数据交互，或者简单地创建具有对工程师有意义的数据子集的表格。

Treuille 表示，这个工具集有可能极大地改变机器学习工程师处理模型中数据的方式。“作为机器学习工程师，我们已经看到了这一点，并且知道经历这些挑战是什么样的，我们真的很兴奋地说，有一种更好的方法可以做到这一点，而且不是一点点，而是可以将一个需要四周时间和 15，000 行代码的项目变成一个下午就可以完成的事情。”

从今天开始，该工具包可以在 GitHub 上下载。
