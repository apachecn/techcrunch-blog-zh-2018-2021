微软为双屏设备推出 Windows 10X

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/02/microsoft-introduces-windows-10-x-for-dual-screen-devices/

在纽约举行的年度 Surface 硬件活动上，微软今天宣布了其现有硬件系列的预期更新。然而，最大的惊喜无疑是该公司宣布的双屏 Surface Neo，该产品将在 2020 年假期之前上市销售。为了让这种双屏设备成为可能，微软还打造了新版 Windows 10: Windows 10X。

微软表示，它今天将发布硬件和软件，以便在发布前让开发者掌握这些硬件和软件。

就像 HoloLens、Surface Hub 和 Xbox 使用 Windows 10 的核心技术一样，双屏 Surface 也将运行这一新版本，戴尔、惠普、联想和其他合作伙伴的双屏设备也将如此。不出所料，这些设备——以及 Windows 10X——将提供改进的笔支持(和虚拟键盘)。

Windows 10X 是微软努力使 Windows 10 更加模块化的结果，这样它就可以根据需要使用操作系统的各个部分。正如微软在今天的发布会之前告诉我们的那样，Windows 10X 本质上是它对 Windows 10 所做的架构变化的延续，这些变化使其能够实现 HoloLens、Surface Hub 和 Xbox 版本。

该公司强调，这不是一个新的操作系统，而是采用了你今天所知道的 Windows 10，并使其更适应其他形式的因素。这也意味着你不能给自己买一个 Windows 10X 的独立版本。获得它的唯一方法是在这些新的双屏设备上。

通过模块化 Windows 10 核心技术，包括用户界面，微软可以做一些事情，如开始菜单，并在 HoloLens 中显示。Windows 10X 做类似的事情，并允许你根据需要将任务栏或开始菜单放在任何一个面板上。类似地，你可以在任何一个面板上使用开始菜单，这取决于另一个面板上发生的事情。

整体设计看起来与你可能已经熟悉的 Windows 10 没有什么不同，但显然它拥有在设备之间移动应用程序的所有功能——或者跨屏幕移动应用程序。无论您如何握持设备，设备都会自动重排。Windows 10X 还考虑到了 Neo 的键盘盖，它覆盖了大约一半的屏幕，然后露出了微软出于某种原因称之为“WonderBar”的虚拟触控板。

但这种模块化的努力也让微软在幕后做了一些聪明的事情。因此，举例来说，虽然普通 PC 会启动并立即运行运行 Win32 应用程序所需的所有服务，但 Windows 10X 直到需要时才会加载该子系统。该公司认为，这使它能够非常有效地利用机器上的可用资源，并大大延长其电池寿命。

不像 Windows 10 S 那样，Windows 10 只允许你运行一小部分应用程序，Windows 10X 可以让你运行任何你想运行的应用程序，无论是 web 应用程序、UWP 还是 Win32 应用程序。微软表示，开发人员不必做任何特定的事情来让他们的应用程序为 Windows 10X 做好准备。Windows 10X 将为他们做到这一点。

这些工作中的一部分肯定会回流到主流 PC 版本的 Windows 10 中。在某种程度上，这已经发生了，微软在 Windows 10 for Xbox 的图形功能上做的一些工作回流到了 PC 上。
