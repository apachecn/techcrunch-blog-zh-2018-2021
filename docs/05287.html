微软推出基于ARM的Surface Pro X TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/02/microsoft-launches-the-arm-based-surface-pro-x/

在其年度Surface硬件活动上，微软今天宣布了传闻已久的基于ARM的Surface，这是微软首次推出内置ARM处理器的设备。这款13英寸的设备将使用微软自己定制的SQ1芯片，基于高通的骁龙和人工智能加速器，使其成为第一款集成人工智能引擎的Surface。微软和高通还致力于为Pro X构建定制设计的GPU核心，Pro X将运行微软版本的ARM Windows 10。

Pro X将于11月5日上市，起价999美元，现已接受预购。

几年前，微软开始与基于ARM的设备调情，这项工作在惠普、华硕和其他公司推出大量基于ARM的设备时达到高潮，这些设备承诺全天电池续航时间，仍然能够运行几乎所有Windows应用程序，性能可与低端英特尔芯片媲美。为此，微软正在使用二进制翻译器，根据需要将X86指令转换为ARM64，同时仍然为ARM架构编译几乎所有的原生Windows 10库。

与微软的其他新Surface设备一样，基于ARM的13英寸Pro X将采用USB-C端口和LTE连接。该屏幕的对比度为1400:1，分辨率为2880×1920，可以扩展为4K屏幕。最薄处，Pro X薄5.3毫米，重1.68磅。还有一个可移动硬盘，这是Surface line的第一款。

与高通一起，微软为此设计了自己的定制处理器(SQ1)。微软还强调了该团队在芯片组中构建人工智能引擎的工作。

微软还为Pro X推出了一款新的更薄的Surface pen，它将在设备的新型外壳上拥有自己的存储空间。

众所周知，ARM自己也努力将其芯片设计带到笔记本电脑、台式机和服务器上。对于每一代新的设计，该公司都谈到如何让更多的芯片进入这些机器，尤其是现在它们的性能对于许多用例来说往往绰绰有余。通过与微软的合作，我们肯定会更进一步。
