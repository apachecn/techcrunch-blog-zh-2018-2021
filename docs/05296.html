Tiny 收购 Meteor TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/02/tiny-acquires-meteor/?ref=freshfordesigners.com

加拿大科技控股公司 Tiny ，是 Dribble、Flow 和 Unicorn Hunt 等公司的所在地，今天宣布收购了以 JavaScript 为中心的开源应用平台 Meteor 。

Meteor 于 2011 年推出，虽然它一度是开发者的宠儿，但随着其他技术的崛起，近年来它的势头有所放缓。

Meteor 向开发人员承诺，他们可以将自己的应用程序的前端和后端构建为 JavaScript 应用程序，并提供 Galaxy，这是一种针对这些应用程序的交钥匙托管服务。

Meteor 创始人 Geoff Schmidt 将继续专注于 Apollo GraphQL，这是一个让 GraphQL 规模化运行的平台。

“我们发现自己处于一个有趣的境地，我们有两个伟大的产品，阿波罗和流星，生活在同一个屋檐下，”施密特告诉我。“阿波罗发展得如此之快，似乎总是正确的事情将我们增加的时间或金钱投入到阿波罗身上。因此，随着时间的推移，很明显，他们需要在不同的屋顶下，这样 Meteor 就不会缺乏资源，也不会有它需要的生长空间。”

Tiny 承诺将继续投资 Meteor、Galaxy 及其开发者社区。Tiny 和 Apollo 还表示，他们将在过渡期间密切合作，“将他们对平台的深刻了解与 Tiny 的雄心勃勃的计划结合起来。”

当然，具体会是什么样子还有待观察，但 Tiny 在维持其收购的公司方面有着相对良好的记录。施密特还指出，用户不应该期望在不久的将来对 Meteor 或 Galaxy 做出任何重大改变。

“我们与许多更传统的收购者进行了交谈，但最终与 Tiny 合作，因为他们在设计和社区方面具有强大的价值，”施密特解释道。“本质上，我认为 Meteor 是关于开发者体验、社区和授权人们做他们认为力所不及的事情。Tiny 是我们认为最有能力推进这一使命的团队。”
