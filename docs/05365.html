1。苹果 MacOS 卡特琳娜现已上市T5】

该公司正在为未来的公告保留大部分细节，但它确实透露了关于即将到来的游戏控制台的一些事情——比如控制器将包括新的触觉反馈。

3。第九集团收购 PopSugar

今年早些时候，有报道称，由 Thrillist、NowThis、Dodo 和 Seeker 合并而成的 Group Nine 正在谈判收购另一家女性生活方式出版商 Refinery29，该公司最终被 Vice Media 收购。

4。中国公司腾讯、Vivo 和央视因香港推特事件暂停与 NBA 的合作

智能手机制造商 Vivo、广播公司 CCTV 和互联网巨头腾讯今天表示，他们将暂停与全国篮球协会的所有合作，成为继休斯顿火箭队一名高管发布支持香港民主抗议者的推特后最新一家与联盟断绝关系的中国公司。

5。 Opera 的桌面浏览器获得内置跟踪保护

Opera 最新桌面浏览器的最大特点是增加了一个拦截器，这将使广告商和其他人更难在你浏览网页时跟踪你，并且还具有加速浏览会话的额外好处。事实上，Opera 认为，同时打开跟踪保护和内置广告拦截器可以将页面加载速度提高 23%。

6。威盛正在库比蒂诺市推出按需公交网络

目的是让这些按需班车——从六辆印有库比蒂诺市标志的面包车开始——提供更有效的连接到 CalTrain 的方式，并增加整个城市的公共交通。

7。风险投资公司 83North 的劳雷尔·鲍登谈欧洲深度科技和创业生态系统

总部位于伦敦和特拉维夫的风险投资公司 83North 已经关闭了其第五只基金，价值 3 亿美元。在与普通合伙人 Laurel Bowden 的对话中，这位经验丰富的投资者分享了一些关于欧洲和以色列技术场景的想法，公司在团队中寻求什么，以及如何在全球范围内扩展。(需要额外的 Crunch 会员资格。)
