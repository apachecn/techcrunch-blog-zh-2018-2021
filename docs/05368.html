Tinder 前高管的律师申请驳回“报复性”诽谤诉讼

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/08/lawyers-for-former-file-to-dismiss-defamation-lawsuit/

上周，Tinder 前首席执行官 Greg Blatt 对 Sean Rad 和 Rosette Pambakian 提起诽谤诉讼，他们是 Tinder 创始人和前高管的一部分，他们指控 Blatt(上图)性骚扰和性侵犯，作为更广泛诉讼的一部分。

现在 Rad 和 Pambakian 的律师已经提交了他们自己的动议来驳回诉讼，认为它“试图通过昂贵的诉讼来冷却受保护的言论”——换句话说，这是一种被加州反 SLAPP 法禁止的诉讼。

拉德和帕姆巴基安的律师奥林·斯奈德在一份声明中说:“这起诉讼旨在阻止罗塞特和肖恩说出真相，即(IAC 主席)巴里·迪勒和格雷格·布拉特是如何从他们的员工那里偷窃并掩盖性侵犯指控的。”“不幸的是，像这样旨在让受害者沉默并侵犯他们的第一修正案权利的非法报复性诉讼在#metoo 时代太常见了。”

在提交的文件中，拉德和帕姆巴基安的律师还辩称，布拉特提起诉讼“只是为了对帕姆巴基安和向 Match 报告殴打事件的人肖恩·拉德发起公开诽谤运动。与此同时，现在布拉特的公开法庭文件服务于他的媒体目标，布拉特说，他自己选择在法庭上提交的投诉实际上应该被送往私人仲裁。

对此，布拉特的律师维尼特·巴蒂亚发表了以下声明:

我们完全期待 Rad 和 Pambakian 制造这种普通的程序性烟幕。这些论点在法律上是错误的，我们希望在法庭上获胜。底线是，Rad 和 Pambakian 合谋诋毁布拉特先生，应该为此负责。

布拉特的诉讼和新的文件都试图将此案与更广泛的#metoo 运动联系起来(正如斯奈德暗示的那样，该运动已经出现了许多被指控性侵犯的高调人物，然后通过诽谤 诉讼反击)。

布拉特的律师认为，“拉德和帕姆巴基安试图将一场重要的社会运动武器化，通过虚假指控破坏性虐待真正受害者的困境，以寻求 20 亿美元的横财。”

相比之下，Rad 和 Pambakian 的律师表示，“随之而来的报复高潮——让人想起许多好莱坞#MeToo 案件——包括[Tinder 的母公司] Match 围着 Blatt 打转，公开贬低 Pambakian，将攻击行为归结为‘两厢情愿的拥抱’，并在她拒绝签署 NDA 协议后几个月解雇了她。”

在 2018 年夏天提起的诉讼中，Rad (Tinder 的联合创始人和前首席执行官)、Pambakian(时任该公司营销和传播副总裁)、Rad 的同为联合创始人的 Justin Mateen 和 Jonathan Badeen 等人起诉 Match 及其控股股东 IAC，指控他们操纵财务数据，并解除 Rad 的首席执行官职务，以便创造“虚假的低估值”，并剥夺创始人和高管的股票期权。

该诉讼还指控布拉特在 2016 年的一次公司假日聚会上对帕姆巴基安进行性骚扰。布拉特曾担任 IAC 的高管，并在接替拉德担任 Tinder 首席执行官之前担任 Match 的首席执行官。

IAC 和 Match 称这起诉讼毫无价值。在布拉特的诽谤诉讼中，他的律师表示，布拉特和帕姆巴基安在假日派对上的遭遇是两厢情愿的，拉德和帕姆巴基安随后“合谋对布拉特进行性骚扰和性侵犯的虚假指控，具体意图是损害布拉特的良好声誉、个人和职业声誉以及可信度。”

在脚注中，Rad 和 Pambakian 的律师说，因为他们正在进行言论自由辩论，他们驳回布拉特诉讼的动议不需要法院“深入研究事实”然而，他们补充道:

布拉特的虚假陈述——这是两厢情愿的，Pambakian 和 Rad 捏造了攻击指控，以帮助他们的估价诉讼——显然是错误的，令人反感。证据显示，布拉特承认在假日聚会上喝醉了，对帕姆巴基安说了不恰当的话，并在酒店的床上“依偎和爱抚”她。它进一步表明，布拉特在接下来的一周向帕姆巴基安道歉，后来又因自己的不当行为提出辞职。这些不是一个无辜男人的行为，也不是布拉特第一次被指控在工作场所虐待女性。

为了支持这一点，该动议提到了一篇 Gawker 文章，描述了一家大型交友网站“由一家公司所有”的一位未具名的“首席执行官”在城镇另一端的一栋玻璃建筑中所谓的骚扰和辱骂(随后的报道暗示这篇文章是关于布拉特的)。

由于仲裁协议，Pambakian 退出了最初的诉讼，但现在正在起诉 Blatt 和 Match 非法终止合同和性侵犯。
