玩具反斗城重新推出网站，网上销售由塔吉特公司提供支持

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/09/toys-r-us-relaunches-its-website-where-online-sales-are-powered-by-target/

托塔吉特百货公司的福，玩具反斗城又上线了。 Tru Kids ，其母公司在破产后收购了倒闭的玩具连锁店，已经宣布重新启动 ToysRUs.com 网站，开始在美国各地开设零售店。作为其东山再起战略的一部分，当消费者点击“购买”按钮进行在线购买时，玩具反斗城美国网站的产品页面将重定向到 Target.com。

零售商没有讨论交易的条款，但考虑到双方的利益，收入分享协议显然涉及到了这种情况。玩具反斗城将能够从仍然排名第一、信誉卓著的域名 toysrus.com 迅速建立现金流，而塔吉特可以从到访 ToysRUs.com、不知道这家玩具连锁店破产和重新开业的购物者那里获得大量新销售额。

除了将在线购物者重定向到 Target，新网站还提供有关最新玩具趋势和热门品牌的文章和视频，以及深入的产品评论、热门玩具列表和其他品牌体验。这些将在玩具反斗城网站上提供。只有当顾客准备购买时，他们才会被送到 Target 结账。

该网站的“购买”按钮也有明确的标签，所以在结账时不会混淆。在塔吉特红白相间的品牌颜色中，上面写着“现在就在[塔吉特]购买。其中“目标”一词被替换为目标徽标图标。

被送到玩具反斗城的目标购物者获得了与直接购物相同的优惠——也就是说，他们可以下单送货、路边或商店提货，还可以通过 Target Circle 获得忠诚度积分，或者通过 Target REDcard 支付获得 5%的积分。

然而，零售商之间的新伙伴关系不仅仅专注于重新引导消费者的传统电子商务产品销售。

Target 表示，今年秋天晚些时候，玩具反斗城在得克萨斯州休斯顿和新泽西州帕拉默斯开设第一家体验式零售店时，它也将实现在线销售

Tru Kids 此前宣布与科技初创公司 b8ta 达成协议，打造现代化的玩具店体验，包括蒸汽工作室、供儿童玩耍的树屋、电影和游戏影院，以及品牌在更互动的环境中展示产品的方式。

在这些商店，想要购买商店本身没有的商品的顾客可以向商店助理下订单，通过 Target.com 完成订单。

“Target 在玩具、数字和履行方面的领导地位是一个无与伦比的平台，让玩具反斗城与粉丝重新建立联系，同时我们向他们介绍在 Target 购物的轻松和便利，”Target 商品营销高级副总裁 Nikhil Nayar 在一份声明中说。“通过以新的方式应用我们的能力，我们可以服务更多的玩具消费者，推动新的增长，并建立我们的玩具领导地位，”纳亚尔补充说。

与玩具反斗城的新交易并不是塔吉特公司最近几周唯一与玩具相关的重大合作。8 月底，零售商宣布与迪士尼达成协议，将在其零售店内开设迷你迪士尼商店，顾客可以在这里购买玩具、服装、收藏品、家居用品等。25 家迪士尼“店中店”现在已经开业，2020 年还将有几十家。

Tru Kids 首席执行官理查德·巴里(Richard Barry)在一份关于目标合作伙伴关系的声明中表示，“我们在美国的战略是，通过强大的体验式和内容丰富的全渠道概念，以现代的方式恢复 ToysRUs 品牌。”

“该战略的基础需要零售行业领导者的帮助，塔吉特百货是支持新玩具反斗城购物体验的理想零售商，旨在为家庭提供无尽的发现、玩耍和享受玩具的方式。Target 将通过其玩具分类、数字优势和在几小时内将订单交付给购物者的能力，帮助我们实现这种体验，”他说。
