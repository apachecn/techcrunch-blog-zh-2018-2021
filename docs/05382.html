Adobe 的 Creative Cloud 应用程序有了新的外观

随着 Adobe 近年来扩大其产品阵容和基于云的服务，其核心的 Creative Cloud 应用程序，即用于下载其余 Creative Cloud 应用程序的桌面应用程序，开始感到有点受限。今天，该公司推出了一款完全重新设计的 Creative Cloud 应用程序，它可以更好地集成您的 Creative Cloud 库，并提供更深入的搜索和资产管理功能。这样，这个应用程序就不那么像一个安装程序，而更像是你在 Creative Cloud 的应用程序中工作的中央指挥中心。

这次更新的首次展示有点奇怪。今天将在法国和德国开始，明天将在日本开始。下周，美国和其他地方的消费者将会在他们的桌面上看到它。

像以前一样，该应用的核心功能是帮助您安装其余的创意云工具。这是不会改变的。但与以前的版本不同，由于新的全屏管理工具可以预览您的资源，您现在可以更容易地看到您的库中有什么。Adobe 表示，这也让与他人分享图书馆变得更加容易。

Creative Cloud 应用程序现在还提供了一个教程中心，用于了解更多有关如何使用各种 CC 应用程序的信息。这是 Adobe 最近非常重视的事情。例如，Lightroom 的最新 CC 版本现在也提供了内置教程。

该应用程序还可以让你管理你的电脑上安装的字体。不过，像以前一样，看起来你必须使用 web 应用程序来搜索和选择你想要安装的字体。我很惊讶 Adobe 没有把这个相对基本的功能直接内置到应用程序中。
