比尔·麦克德莫特卸任 SAP 首席执行官

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/10/bill-mcdermott-steps-down-as-saps-ceo/

SAP 今天宣布过去九年的首席执行官比尔·麦克德莫特将立即离职。该公司表示，他决定不再续签合同。SAP 执行董事会成员 Jennifer Morgan 和 Christian Klein 被任命为联席首席执行官。

麦克德莫特的商业生涯始于长岛阿米提维尔的一家熟食店老板，最近在我们的 TechCrunch Sessions:Enterprise event 上发言，他于 2002 年加入 SAP，担任 SAP 北美地区的负责人。他于 2008 年成为联席首席执行官，并于 2014 年成为该公司的唯一首席执行官，这使他成为这家德国企业巨头中担任这一角色的第一位美国人。在他的指导下，SAP 的年收入和股价持续增长。在接下来的两个月里，他将以顾问的身份留在公司。

目前还不清楚麦克德莫特为什么决定在这个时候下台。维权投资者 Elliott Management 最近披露了 13.5 亿美元的 SAP 股份，但当被问及对今天的消息有何评论时，Elliott 发言人告诉我们，它没有任何“立即评论”。

同样值得注意的是，该公司最近几个月出现了大量高管离职，SAP 的成功因素首席运营官·布里吉特·麦金尼斯-戴和其云业务总裁兼董事会成员罗伯特·安斯林都离开了公司，加入了谷歌云。

“没有比尔·麦克德莫特，SAP 就不会有今天，”普拉特纳在今天的声明中说。“Bill 为公司做出了不可估量的贡献，他是 SAP 向云转型的主要推动者，这将推动我们未来多年的发展。我们感谢他为 SAP 所做的一切。我们也祝贺 Jennifer 和 Christian 有机会为 SAP 的未来打下坚实的基础。一年前，比尔和我决定扩大詹妮弗和克里斯蒂安的角色，作为将他们培养成我们下一代领导人的长期过程的一部分。我们对他们的愿景和能力充满信心，因为我们将带领 SAP 进入下一个增长和创新阶段。”

麦克德莫特近年来最大的赌注是以 80 亿美元收购 Qualtrics。在上个月的活动中，麦克德莫特将此次收购与苹果收购 Next 和脸书收购 Instagram 相提并论。“Qualtrics 将削弱 M&A 的举措对那些优秀公司的影响，”他说。在他的领导下，SAP 还以 83 亿美元收购了企业开支和差旅管理公司 Concur，以 34 亿美元收购了 SuccessFactors。

麦克德莫特在今天的声明中说:“现在是每个人开始激动人心的新篇章的时候了，我相信詹妮弗和克里斯蒂安会做得很出色。”“我期待着在他们结束 2019 年并为 2020 年及以后奠定基础时支持他们。对于每一位信任 SAP 的客户、合作伙伴、股东和同事，我只能表达我衷心的感谢和永久的敬意。”
