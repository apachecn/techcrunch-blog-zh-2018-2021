New Vector获得850万美元，将更多用户纳入其开放、分散的消息传递矩阵

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/10/new-vector-scores-8-5m-to-plug-more-users-into-its-open-decentralized-messaging-matrix/

New Vector 是一家欧洲初创公司，由一种名为 Matrix 的开放、去中心化通信标准的创始人于2017年创立，旨在推动即时通讯和VoIP应用的替代通讯协议的采用并发展一个生态系统，该公司已筹集了850万美元的A轮融资。

New Vector首轮融资的投资者包括企业技术专家概念资本和黎明资本，以及欧洲种子基金第一分钟资本。

几年来，该团队一直在展示当你在通常的(信息巨头)嫌疑人的专有筒仓之外思考时可能发生的事情——推出一个名为 Riot的松散竞争对手。我回到了2016年，运行在Matrix上——提供一个开放的、可定制的和 安全的 选择。(安全，因为与Slack Riot 不同，提供端到端加密。虽然还没有普及，但扩展e2e加密是a系列计划的一部分。)

Riot的用户也可以选择在自己的服务器上运行该应用程序，这样他们就可以完全控制数据托管。该应用程序包括一个桥接功能，可以与Slack等主流聊天应用程序竞争对手集成。因此，这是现代信息技术的“蛋糕和吃它”的方法:控制加上互操作性和透明性。

firstminute capital的布伦特·霍伯曼(Brent Hoberman)在一份支持声明中评论这笔资金时说:“Slack和WhatsApp已经证明了即时消息对工作场所的生产力有多么重要，但将这种便利与对数据的完全主权和安全结合起来，比以往任何时候都更有价值。”

“在过去的几年里，我们感觉像是在倒退，像Slack和WhatsApp这样的通信平台是有围墙的花园，用户对他们的数据是否安全以及如何被使用的担忧是非常可以理解的，”Capital的乔斯·怀特在另一份声明中补充道。“市场终于有了新的矢量服务的替代品，这些服务基于矩阵协议，提供开放标准并提供完整的数据所有权和安全性。”

New Vector的A轮快速跟进了去年筹集的500万美元资金，当时该团队从一个名为状态的基于以太坊的安全聊天和加密钱包应用获得了战略投资。

Matrix协议的早期开发工作得到了一家大型跨国电信基础设施公司的资助，其创始团队此前曾为该公司开发过通讯应用。但是资金在2017年8月枯竭，这是他们开始寻找替代方案的时候——最初是向支持者寻求捐款。

几年后，随着他们方法的发展势头越来越大Matrix网络今年已经扩展到超过1100万用户和40，000次部署，自2018年以来日活跃用户增长了400%——他们已经在银行获得了一大笔风险投资。

当你看到一些用户他们能够检查名字时，这并不奇怪。比如美国政府；法国政府(今年早些时候，它授权Riot推出了自己的消息应用程序，名为Tchap，并选择Matrix作为其官方通讯平台)；维基媒体；KDE；和RedHat等等。它还表示正在与英国国家健康服务和Mozilla合作。

因此，A系列的计划是加大油门，扩大他们的托管平台，改善产品体验，增强协议，以便能够支持更多寻求数字主权、信息自主和强加密的政府和企业，以便在日益动荡的地缘政治时代保守秘密。

就在上周美国、英国和澳大利亚政府的官员公开向脸书施压，呼吁该公司不要扩大其端到端加密的使用——除非或直到它能够确保在有保证的需求下获得解密的通信。

WhatsApp的e2e加密技术备受推崇。但它的强大也取决于脸书对它的执行。当该公司面临来自本国政府的强大压力，要求其应用程序后门时，这并不令人放心。因此，想要避免数据集中化，既有安全逻辑，也有隐私逻辑——即使它是经过严格加密的。

当然，对于某种高度安全意识的企业和公共部门用户来说，这是Matrix的目标。

如果数据是集中的，它就有可能成为强大利益集团试图获取的目标，并产生大量的元数据，控制商业实体完全可以对其进行数据挖掘。因此，一个不需要这种权衡的健壮的、分散的消息传递标准对那些有资源定制和部署自己的应用程序的人来说将有明显的吸引力。

(郑重声明，Matrix表示，其e2e加密基于由Signal推广的双棘轮算法，但该算法已被扩展到支持对包含数千台设备的聊天室进行加密。它还表示，它使用Olm和Megolm加密棘轮，这些棘轮被指定为在Apache许可下发布的实现的开放标准，并且已经由NCC Group独立审计。)

New Vector首席执行官兼Matrix联合创始人Matthew Hodgson告诉我们，Matrix的增长主要来自公共部门和相邻行业(这些行业需要能够与政府部门安全地通信)；来自开源项目；加密货币；活动家和非政府组织。

“在这里，推动去中心化的因素是希望能够对你的对话拥有完全的自主权和控制权，而不依赖于脸书、谷歌或Slack这样的大公司……不想创造一个孤岛，而是参与到一个更广泛的全球开放矩阵网络中，就像网络本身一样，”他说。“还有，开发商终于要(了！)是一个开放的平台，可以像网络一样构建通信应用，而不是局限于大公司的专有通信平台。”

霍奇森指出，政府本质上是“高度分散的”(即在不同部门、部委、公民等之间)，并补充说他们“真的喜欢端到端加密，尤其是在更广泛的开放网络中”。

或者，好吧，至少有一部分政府没有要求脸书对其应用程序进行后门…

“我们是一个加密但分散的通信平台的首选，该平台可以跨越多个政府部门——根据需要在不同的服务器上实施不同的安全级别，由于Matrix，零供应商锁定，”他继续说道。“它可以让你获得整个公共部门——学术、医疗、军事、公民及其相邻组织(以及相邻国家！)在同一个网络上，无需向脸书、谷歌、Telegram或其他任何人交出控制权。”

“法国和美国公共安全部已经投入使用，其他几个国家正在筹备中，”他补充说，公共部门的部署。“我们预计Matrix将成为未来政府内部和政府间安全通信的支柱。”

以法国为例，政府已经在所有16个部门推广了Matrix覆盖550万用户。

谈到未来，A系列的计划有四个方面。首先:投资改善Riot的用户体验，让这款应用成为霍奇森所说的“真正的主流”——也就是:“WhatsApp和Slack的真正替代品，供那些需要完全在自己控制之下的安全通信的群体使用，而不是由脸书或Slack运营”。

第二，默认情况下，他们会为所有私人对话开启端到端加密。

“去中心化的e2e加密很难，”他强调道。“但是我们已经非常接近缺失的成分(交叉签名密钥验证；支持E2E的全文搜索；支持E2E的机器人)完成了——这意味着我们可以在整个公共网络上为私人房间默认打开它。这是一笔大交易，尤其是考虑到集中式端到端加密日益明显的风险(比如WhatsApp和Signal)。”

第三，这笔资金将继续用于构建他们的旗舰Matrix主机平台(Modular.im)并将其打造为Riot——“以便用户群可以轻松地跳上他们自己的自主服务器”。

“我们a 今天已经有像维基媒体基金会、KDE和GNOME这样的人在使用模块化(希望将来还有Mozilla和 NHSX )，我们将利用这笔资金让尽可能多的人使用模块化来帮助scale Matrix向前发展，”他补充道。

最后，他们打算致力于打击虐待。与任何通信平台一样，人们想要分享的东西也有阴暗面。再加上e2e加密和去中心化，你如何缓和仇恨交流的问题很容易被忽视。但新矢量至少在思考这个问题。

“矩阵是更广泛的开放互联网的一个迷人的缩影，1100万可寻址用户跨越了人类的所有领域，”霍奇森说。“我们正在进行一些非常有趣的工作，让用户能够过滤掉他们不想看的内容(而不是使用集中式算法来这样做)，这可能适用于更广泛的互联网。”

“我们希望Matrix.org基金会(控制矩阵协议的非盈利组织)将推动这项工作，但这也是New Vector非常关注的事情，”他补充道。

当被问及Matrix的安全性和稳定性时，霍奇森说这是6月份big 1.0发布的焦点——当时该协议退出了测试版。

“我们推出了正式的安全披露政策和名人堂(【https://matrix.org/security-】T2披露政策/ )，该协议有着相当好的安全记录——除了在法国推出Tchap的大戏，”他说，指的是该应用推出后立即发现的安全漏洞。

“发现漏洞的研究人员对此大肆宣扬，但实际上这不是矩阵协议本身的漏洞——它是针对法国部署的配置的，在发布前就被发现了，我们在报告后的几个小时内就解决了它，”他补充道。“很明显，在暴露于互联网之前就应该被发现，但随后法国设立了一个成功的漏洞奖励计划(https://yeswehack.com/程序/tchap )以及一个专门的审计来避免问题的发生。

“与此同时，我们让我们的E2EE 在2016年成功通过了NCC集团的审核(从那以后没有实质性的变化)，加上之前提到的E2EE默认工作，我们将继续专注于安全&稳定性。”
