IAC 概述了其比赛小组分拆计划

数字媒体控股公司 IAC 已经采取了剥离 Match Group 的下一步措施，提出了一份概述这一过程的提案。

Match Group(旗下拥有 Tinder、PlentOfFish、OkCupid、Hinge，当然还有 Match 本身)已经是一家上市公司，但 IAC 仍是大股东。随着分拆，IAC 表示，它应该将其 Match Group 的股份分配给 IAC 的股东，“从而形成两家独立的上市公司。”

IAC 首席执行官乔伊·莱文在一份声明中说:“今天，IAC 提出了将 Match Group 从 IAC 中分离出来的重要的第一步。”“IAC 相信，提交给 Match Group 特别委员会的提案为 Match Group 作为一家蓬勃发展的独立公司开始其旅程奠定了坚实的基础。”

根据该提案(IAC 表示，该提案仍需获得其董事会、上述特别委员会以及股东的批准)，Match Group 的双重股权结构将被取消，形成单一股权类别。

该公司在 8 月份表示，它正在探索 Match Group 和 ANGI 家庭服务公司的分拆。

在他今天的声明中，莱文说，“因为这涉及到评估我们在 ANGI 家庭服务公司的所有权股份，我们目前不希望把我们的注意力转向分拆的问题，直到 Match Group 交易完成。”
