Algolia 从 Accel 和 Salesforce 获得 1.1 亿美元，用于 Slack、Twitch 和 8K 等公司的搜索即服务

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/15/algolia-finds-110m-from-accel-and-salesforce-for-its-search-as-a-service-used-by-slack-twitch-and-8k-others/

Algolia 是一家为网站和应用程序提供搜索服务的初创公司，作为谷歌和其他搜索引擎的替代选择，它今天宣布进行一轮大规模融资，以促进其增长。这家初创公司已经拥有 8000 多名客户，包括 Twitch、Slack、Discovery 和 LVMH 等知名公司。该公司已经完成了 1.1 亿美元的 C 轮融资，计划投资于围绕其搜索技术的研发，包括加倍投资语音业务，以及在欧洲、北美和亚太地区的进一步全球扩张。

这一轮 C 轮由 Accel 领投，其他投资者包括 Salesforce Ventures、Alven、DAG Ventures、Founders Circle、Owl Rock Capital、SaaStr Fund 和世界创新实验室。

这笔资金是在 Algolia 强劲增长的时候获得的，其基本前提——为企业提供易于使用的基于 API 的搜索服务，作为购买搜索技术而不是从头开始使用搜索平台的一种方式——已经获得了很大的吸引力。

它已经在计划增长的各个地区活跃起来:Algolia 最初成立于法国，现在总部位于旧金山，自 2014 年以来一直在亚洲，最近在日本的业务翻倍，当它上次在 2017 年6 月筹集资金时，它只有 3000 名客户。

在此之前，Algolia 已经筹集了 7400 万美元，之前的投资者包括 Accel、Point Nine Capital、Storm Ventures、Y Combinator、500 Startups 和一些个人等。

虽然 Algolia 没有披露其估值，但建立一个大型、专注于企业的搜索业务的前景是存在的。作为比较，考虑一下企业搜索公司 Elastic，该公司于 2018 年上市，现在市值约为 67 亿美元，而在它还是私有公司时，市值仅为 7 亿美元。即使 Algolia 现在有 8000 名客户，这也只是冰山一角:Algolia 引用估计今天市场上有大约 18 亿个网站和数百万个应用程序。

在这一轮中，Salesforce 作为战略支持者是值得注意的，因为 CRM 巨头在搜索方面只迈出了较小的步伐:上个月推出的一个工具，旨在更好地搜索 Salesforce 服务，以及其商业云中的一些发现和搜索功能。但是没有与 Algolia 类似的本地搜索产品，Salesforce 选择与第三方整合，如 Algolia 的竞争对手 Coveo。Algolia 的计划是与 Salesforce 整合，继续扩大其在商业云搜索领域的业务，尽管目前还没有产品。

Salesforce Commerce Cloud 首席执行官 Mike Micucci 表示:“Algolia 一直是一个伟大的搜索创新者，为各行各业的客户提供独特的体验。“Algolia 与商业云平台的整合将继续推动我们与开发商、合作伙伴和客户群体的合作，共同取得成功。”

在搜索仍然是一个组织如何在网上展示自己的关键基石的时候，在搜索领域提供抗衡谷歌力量的努力继续快速进行，这基本上给了 Salesforce 在该领域中增长最快的公司之一的财务立足点。

正如我的同事 Romain 之前在他对 Algolia 的报道中指出的，该公司的独特卖点是它提供了一个超快速有效的搜索工具，你可以通过 API 轻松地将其集成到一个网站或应用程序中。

相比之下，有些解决方案要么是在内部从头开始构建，要么是依靠更耗时、更昂贵的集成来启动和运行。除此之外，Algolia 最近发布了辅助工具，如搜索分析和 A/B 测试，以帮助优化结果，更好地了解网站/应用程序访问者想知道什么。

这笔资金出现在搜索领域一个有趣的时刻。多年来，谷歌通过开放的网络方式订购全球信息，有效地控制了市场:主要的入口是 Google.com，虽然你可以根据你的搜索词定制你的结果，但卖点是你可以搜索任何东西。

但是最近几年，我们看到了一个巨大的转变。对隐私和数据保护等问题的认识已经让一些人放弃了由广告驱动的开放式浏览的想法，随着我们和互联网本身变得越来越复杂，有时开放式搜索对我们的目的来说感觉太宽了，网络出版商自己也不太倾向于将搜索流量交给谷歌。

这带来了更专注的垂直搜索服务，甚至更具体地说，是网站和应用本身更好的搜索。这种背景催生了 Algolia 和其他类似的公司(例如，Lucidworks 在 8 月份筹集了 1 亿美元)。

前景是广阔的，但它仍然是阿尔戈利亚认为保持专注的最佳方式。

“我们没有建立消费者服务的计划，”首席执行官 Nicolas Dessaigne 说。“有很多像亚马逊和谷歌这样的公司做得很好。在某种程度上，我们喜欢将他们视为合作伙伴，让全世界了解搜索。”

“凭借世界级的搜索专家团队和热情的客户群，Algolia 已成为搜索即服务领域的市场领导者，”Accel 合伙人 Nate Niparko 表示。“Algolia 正在加速个性化和智能搜索的创新，使公司能够提供出色的用户体验，从而推动业务成果的改善。我们很高兴能加倍投资 Algolia，支持他们引领搜索和发现市场的使命。”
