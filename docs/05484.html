只需164美元，捷豹就可以让你在纽博格林赛道上驾驶它的I-Pace比赛出租车

著名的纽博格林北赛道(Nurburgring Nordschleife)在20世纪60年代末被称为绿色地狱，它是德国一条73个弯道、近13英里的赛道，甚至可以挑战最优秀的职业赛车手。

现在，只需164美元，寻求刺激的人就可以乘坐一圈。捷豹宣布了一项名为 RACE eTAXI 的促销活动，客户可以购买一辆由专业赛车手驾驶的全电动I-Pace汽车在纽博格林赛道上兜风。

促销活动将持续到11月底。该汽车制造商也有一个常规的ol' race出租车计划，大约274美元，让客户乘坐捷豹XE SV Project 8。

纽博格林已经成为大众、Nio和保时捷等汽车制造商争夺“最快电动汽车”吹嘘权的地方。甚至特斯拉首席执行官埃隆·马斯克也表示，该公司计划在“环”上测试其Model S。

今年8月，保时捷Taycan 以7分42秒的成绩完成了全程，这家德国汽车制造商称这是四门电动车的最快圈速。

但这并不是任何电动汽车的最快圈速。这一荣誉属于大众的ID R电动赛车，它以6分05秒336的成绩完成了全程。之前的纪录是由彼得·邓布利多在2017年创造的，他驾驶的是一辆 Nio电动汽车。
