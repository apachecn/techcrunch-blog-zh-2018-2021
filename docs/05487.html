谷歌推出649美元的Pixelbook Go Chromebook

在其年度硬件活动上，谷歌今天宣布推出Pixelbook Go，这是其第一方Chromebook系列的最新版本。Pixelbook Go的起价为649美元，标志着继去年的Pixelbook采用180度转轴和Pixel Slate二合一设备后，又回到了标准的笔记本电脑格式。

这款Go将配备16:9的13.3英寸触摸屏，高清或4K显示屏，两个USB-C端口，内置Titan-C安全芯片，高达16GB的内存和高达256GB的存储空间。它由英特尔酷睿CPU提供支持，从低端的m3芯片和高端的i7芯片开始。现有颜色为黑色和“非粉色”,现在开始预购，但仅限黑色版本。《非粉》即将上映。

它有一个波纹底部，“易于抓握”和“静音键”，据说比以前的Pixelbooks上的键更安静。说到安静，Go还拥有远场麦克风，满足你所有的“嘿谷歌”需求。

"我们 想要 想要 创造 一个 薄轻 笔记本电脑t 帽子 真的是快，也 有 当然， 我们 想要它看起来 和 觉得 漂亮，”谷歌的艾薇·罗斯在今天的公告中说。罗斯还强调，Pixelbook Go拥有更大的电池，但仍然更轻，这要归功于它的镁机身。

与微软的Surface系列不同，谷歌的笔记本电脑似乎更像是定义了Chromebook系列高端产品的理想设备。649美元的Pixelbook Go显然比谷歌之前在这一领域的许多努力都更实惠，该公司显然希望再卖出几台。
