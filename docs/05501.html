阿里安娜·赫芬顿的 Thrive Global 正在收购一家利用神经科学促进应用程序使用的初创公司 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/16/arianna-huffingtons-thrive-global-is-buying-a-startup-that-uses-neuroscience-to-boost-app-usage/

当阿里安娜·赫芬顿辞去在《赫芬顿邮报》的职位，创办 Thrive Global 时，她说她的新公司的目标是帮助一代人“避免如今成功带来的倦怠”

在实践中，这意味着创建一个企业，向一群相信增强身心健康可以提高工作效率的公司出售正念、一般健康和保健技巧和诀窍。

现在，Thrive Global 收购了位于洛杉矶的初创公司 unlimited Mind，为其认知行为疗法的军火库增加了一个技术工具。

该公司最初名为多巴胺实验室，成立于 2015 年，旨在将脸书等社交媒体公司用于提高参与度的一些相同技术应用到更广泛的领域。

交易的条款没有披露，但股票和现金收购将使目前 unlimited 团队的所有九名成员加入 Thrive Global。之前的无限投资者包括 Revolution 旗下的 Rest Rise 种子基金和埃丝特·戴森将加入 Thrive Global 的资本表。

“我们对他们基于神经科学的人工智能印象深刻，他们用人工智能来改变行为，”赫芬顿说。"我们可以利用技术把人们从不健康的行为中解脱出来."

赫芬顿说，unlimited“是鼓励健康习惯的技术应用的缩影”。

从赫芬顿的角度来看，美国的大多数健康问题实际上源于行为问题，而不是生理问题。“直到 100 年前，人们死于传染病……现在大多数人死于行为，”赫芬顿引用无限思维联合创始人道尔顿·库姆斯的话说。

赫芬顿说，美国大约 70%的医疗支出用于行为改变和生活方式相关的疾病。Thrive Global 通过结合现场活动和分发流行心理学和名人建议的数字平台来解决这个问题，而 unboundary 则使用人工智能和机器学习来推动。

unlimited 技术的工作原理是监控手机的 tap 屏幕上正在发生什么活动(类似于苹果的屏幕时间监控)。unboundary 在后端所做的是分析这些数据，并创建提示以鼓励行为——与其他公司的应用程序有通知以提示再次参与的方式非常相似。

展望未来，unlimited 团队希望利用更多来自手机越来越多的传感器阵列的信息，更好地完善其通知。采用该公司软件的结果各不相同，但 unlimited 指出了来自健康、健身、生产力、金融和电子商务应用程序的数据——包括行走增加了 60%，生产力增加了 30%，饮食和运动参与增加了 21%。

阿里安娜·赫芬顿和无限思维的联合创始人和员工

Thrive Global 的业务有三大支柱:现场研讨会、数字健康计划和行为改变工具(如 unlimited)。

该公司已经签署了企业合作伙伴，如埃森哲、摩根大通、希尔顿、美国银行和宝洁，赫芬顿说，客户已经看到了员工流失率降低和员工满意度提高的调查结果。

这种相关性并不意味着因果关系，该公司仍在努力更好地量化采用其工作场所健康协议的好处。Thrive Global 测试大脑训练机制的地方之一是中美洲的呼叫中心

“你可以想象无限的整合将如何让我们超个性化，”茁壮成长全球首席品牌官丹尼谢伊说。在呼叫中心，这可能意味着提示员工在长时间或紧张的通话后休息一下。

“我们很高兴看到 Thrive Global 的持续增长和市场扩张，”IVP 的普通合伙人兼 Thrive Global 董事会成员 Somesh Dash 说。“Thrive 的使命和 unlimited Mind 的技术的结合确实非常出色，这种整合将有助于 Thrive 为全球企业扩展一个改变游戏规则和差异化行为改变的技术平台。”

迄今为止，Thrive Global 已经从包括 JAZZ Venture Partners、IVP、马克·贝尼奥夫、雷伊·达里奥和凯文·杜兰特在内的投资者那里筹集了超过 6500 万美元的资金。
