边缘计算初创公司 Pensando 走出秘密模式，获得了总计 2.78 亿美元的资金

由前思科工程师创立的边缘计算初创公司 Pensando 今天走出了秘密模式，宣布它已经筹集了 1.45 亿美元的 c 轮融资。该公司的软件和硬件技术旨在为数据中心提供更多云计算服务器的灵活性，被定位为亚马逊网络服务 Nitro 的竞争对手。

这轮融资由惠普企业(Hewlett Packard Enterprise)和光速创投(Lightspeed Venture Partners)牵头，目前 Pensando 的融资总额达到 2.78 亿美元。HPE 首席技术官马克·波特和光速创投合伙人巴里·埃格斯将加入彭桑多的董事会。该公司的董事长是前思科首席执行官约翰·钱伯斯，他也是 JC2 风险投资公司的投资者之一。

Pensando 于 2017 年由 Mario Mazzola、Prem Jain、Luca Cafiero 和 Soni Jiandani 成立，这是一个工程师团队，他们率先开发了思科的几项关键技术，并成立了四家创业公司，这些公司被思科收购，其中包括 Insieme Networks。(在接受路透社的采访时，前思科执行副总裁 Pensando 首席财务官 Randy Pond 表示，尚不清楚思科是否有兴趣收购这家初创公司，并补充道“我们目前的愿望是上市。但是，你知道，货币化活动总有其他的可能性。”)

这家初创公司声称，就生产率和规模而言，其边缘计算平台的性能比 AWS Nitro 好五到九倍。Pensando 为边缘计算准备了数据中心基础设施，使其能够更好地处理来自 5G、人工智能和物联网应用的数据。在秘密模式下，Pensando 收购了包括 HPE、高盛、NetApp 和 Equinix 在内的客户。

在一份新闻声明中，波特说:“当今快速变化、高度互联的世界要求企业以比以往任何时候都更大的灵活性和选择来运营。HPE 与 Pensando Systems 的关系不断扩大，这源于我们对企业和云的共同理解。我们很自豪地宣布我们与 Pensando 的投资和解决方案合作伙伴关系，并将继续共同推动满足客户需求的解决方案。"
