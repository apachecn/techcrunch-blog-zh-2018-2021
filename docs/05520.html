Atlassian收购吉拉自动化制造商Code Barrel TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/17/atlassian-acquires-code-barrel-makers-of-automation-for-jira/

Atlassian今天宣布，它已经收购了 Code Barrel ，它是吉拉自动化的制造商，这是一个低代码工具，用于轻松自动化吉拉的许多方面，也是Atlassian市场上最受欢迎的吉拉软件和吉拉服务台的附加软件之一。两家公司没有透露收购价格。

总部位于悉尼的Code Barrel是由Atlassian建造吉拉的第一批工程师中的两位创建的，他们是Nick Menere和Andreas Knecht。通过这次收购，他们在创业四年后重返Atlassian。

“对我和Andreas来说，这几乎就像回家一样，”Menere说，他于2005年加入吉拉团队，当时只有少数几个开发人员在开发该产品。“这是我们非常了解如何开发软件和如何开发产品的地方。对我们来说，这是我们唯一愿意回去的公司。”

顾名思义，吉拉自动化使Atlassian的问题和项目跟踪服务中的重复性任务的自动化变得非常容易。Atlassian技术团队产品副总裁诺亚·沃斯默(Noah Wasmer)告诉我:“越来越多的(我们的客户)不得不在日常琐事上花费大量时间。“我们看到的是，以吉拉为骨干，他们与许多系统交互，复制工作，手动将工作输入不同的系统。因此，我们发现，他们花费了过多的时间来做一些事情，而这些事情实际上并没有帮助他们构建和创造那些有助于改变我们世界的下一代事物。”

如果你想减少这种重复工作，那么自动化是显而易见的。超过6000家公司在Atlassian的市场上发现了Code Barrel的解决方案，加上创始人与该公司的明显联系，吉拉自动化公司肯定是一个明显的收购候选人。

瓦斯默还强调，事实上，他们建立了一个无代码工具，将允许任何人使用吉拉创建脚本，而不必是一个程序员。吉拉自动化允许用户设置基于时间的规则或基于吉拉境内的触发器运行的规则。它还具有与SMS、Slack和微软团队等的第三方集成。

目前，吉拉的自动化仍将停留在Atlassian market并继续以相同的价格销售，10个用户以下的团队为5美元/用户/月，11到100个用户的团队为2.5美元/用户/月，较大企业的价格将从那里下降。当然，Atlassian将开始将该工具的一些功能集成到吉拉，但目前该公司在这方面没有任何消息宣布。
