亚马逊为其试点员工医疗保健计划 Amazon Care 收购健康导航器

亚马逊已经收购了为在线健康服务开发 API 的初创公司 Health Navigator 。据最先报道该交易的美国消费者新闻与商业频道称，健康导航器将成为亚马逊护理的一部分，这是其为员工提供的试点医疗服务项目。

这是亚马逊收购的第二家健康创业公司。第一个是网上药店 PillPack ，该公司于 2018 年以略低于 10 亿美元的价格收购。PillPack 的服务也已经整合到亚马逊护理中，亚马逊护理提供交付处方和远程沟通的治疗计划。

Health Navigator 的平台旨在集成到在线健康服务中，包括远程医疗和医疗呼叫中心，以标准化与患者合作的流程。它的平台包括基于自然语言处理的工具，用于记录健康投诉和护理建议，并通过 API 集成到应用程序中。

这家由内科医生 David Thompson 在 2014 年创立的初创公司尚未就此次收购发表公开声明，但美国消费者新闻与商业频道报道称，该公司告诉客户他们的合同将不会续签。
