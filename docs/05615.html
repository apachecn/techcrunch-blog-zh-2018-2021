埃隆·马斯克预测特斯拉能源可能会比其电动汽车业务“更大”

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/23/elon-musk-predicts-tesla-energy-could-be-bigger-than-its-ev-business/

埃隆马斯克预测，特斯拉能源公司可能会比其电动汽车业务“更大”

特斯拉首席执行官埃隆·马斯克(Elon Musk)预测，该公司的能源业务最终将与其汽车部门的规模相同，甚至更大，这是该公司计划投入更多时间和资源来扩大其太阳能和储能产品的最新迹象。

马斯克在周三的一次财报电话会议上说:“Itt可能会比 更大，但 肯定会与具有类似的数量级。”该公司公布第三季度恢复盈利，令华尔街大吃一惊。

特斯拉的大部分收入来自其 Model S、Model X 和 Model 3 电动汽车的销售。第三季度，汽车收入为 53.5 亿美元。该公司没有公布来自太阳能、储能或其他产品和服务的收入。然而，第三季度的总收入为 63 亿美元，这在一定程度上表明了汽车行业与其其他业务相比的规模。

随着注意力和资源转向 Model 3，特斯拉的能源和太阳能业务停滞了近两年。这种资源转移包括将转向汽车电池电池 生产线 生产线 意为 用于其家用 Powerwall 和商用 Powerpack 储能产品因为公司没有足够的 电池 。

“W e 曾 到 做 它 是因为如果我们不解决 Model 3，特斯拉就不会 活下来，”他说。“如此， 可惜 那个 做空 其他 部分公司。”

现在，该公司致力于扩大储能和太阳能的规模。Kunal Girotra 最初于 2015 年加入特斯拉，担任 Powerwall 的高级产品经理，后来被提升为该公司能源运营的高级总监。

在第三季度，特斯拉部署了 43 兆瓦的太阳能，比上一季度增加了 48%。太阳能装置仍然比去年同期低 54%。

根据周三公布的收益，储能部署继续增长，第三季度达到历史最高水平 477 兆瓦时。

这项新努力的一部分包括其最初于 2016 年推出的太阳能屋顶瓦产品。马斯克表示，其太阳能屋顶瓦的第三代产品将于周四下午首次亮相。
