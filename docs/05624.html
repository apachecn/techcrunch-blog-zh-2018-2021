微软报告了强劲的第一季度财政，但 Azure 的增长率继续下降

微软今天公布了远远超出分析师预期的季度业绩，但 Azure 的增长率继续下降，因为它与 AWS 竞争。

该公司本财年第一季度的收入同比增长 14%，至 331 亿美元。净收入增长 21%，至 107 亿美元，或每股 1.38 美元。

包括办公产品和 LinkedIn 在内的微软生产力和业务流程部门的收入增长了 13%，达到 111 亿美元。LinkedIn 的收入增长了 25%。

与此同时，其智能云部门的收入增长了 27%，至 108 亿美元，其中来自服务器产品和云服务的收入增长了 30%。该公司表示，Azure 的收入增长了 59%，但这代表了一年前开始的增长率下降，当时 Azure 的季度增长率为 76%。自那以后，这一比率下降得更多，今天的报告显示，与上一季度报告的 64%的增长率相比有所下降。

微软个人电脑部门的收入增长了 4%，达到 111 亿美元。

该公司表示，预计第二季度收入将在 351.5 亿美元至 359.5 亿美元之间。
