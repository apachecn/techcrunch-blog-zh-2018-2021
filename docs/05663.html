Grafana Labs获得2400万美元的A轮开源数据分析堆栈

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/24/grafana-labs-nabs-24m-series-a-for-open-source-based-data-analytics-stack/

Grafana Labs获得2400万美元的A轮开源数据分析堆栈

Grafana Labs 是一家商业公司，旨在支持开源 Grafana 项目，今天宣布了2400万美元的首轮投资。光速创投领投，Lead Edge Capital参与其中。

公司首席执行官兼联合创始人Raj Dutt表示，这家初创公司最初是为了在开源Grafana工具的基础上提供一个商业层，但它已经扩展到现在支持其他项目，包括 Loki ，这是一个开源监控工具，与公司去年开发的 Prometheus 没有什么不同。

所有这些都是为了连接数据源和监控数据。“Grafana一直致力于将数据连接在一起，无论数据位于何处，是在专有数据库、本地数据库还是云数据库中。Grafana将超过42个数据源连接在一起，”Dutt解释道。

但该公司的扩张远不止于此。正如它对产品集的描述，“我们的产品已经开始演变为一个单一的产品:世界上第一个可组合的开源可观察性平台，用于度量、日志和跟踪。以格拉法纳为中心。”这正是今年Elastic、New Relic和Splunk等其他监控和日志工具的发展方向。术语“可观察性”是一个经常被用来描述这些度量、日志和跟踪的组合能力的术语。

Grafana Labs是开源项目的商业分支，提供了一些基于这些工具的产品。首先，它有Grafana Enterprise，这是一个软件包，除了开源的Grafana工具提供的功能之外，它还包括以企业为中心的数据连接器、增强的身份验证和安全性以及企业级支持。

该公司还提供SaaS版本的Grafana工具堆栈，该工具堆栈是完全托管的，可以消除试图下载原始开源代码、安装、管理它以及处理更新和补丁的一系列头痛问题。在SaaS版本中，所有这些都由每月付费的客户负责。

Dutt说，这家初创公司在最初的五年里只获得了400万美元的外部投资，并且已经能够建立一个拥有100名员工和500名客户的企业。他特别自豪的是，该公司在这一点上实现了现金流盈亏平衡。

Grafana Labs认为是时候进行这笔巨额投资并加速这家初创公司的增长了，如果没有大笔现金注入，他们真的无法做到这一点。“我们已经看到了这种真正的良性循环，通过这些开源项目在社区中创造价值，建立思想共享，并可以转化为建立可持续的业务。所以我们真的想加速这一过程，这是加薪背后的主要原因。”
