WeFarm吸金1300万美元，为独立农场主拓展市场和网络TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/29/wefarm-rakes-in-13m-to-grow-its-marketplace-and-network-for-independent-farmers/

像脸书和LinkedIn这样的大型网络在社交媒体世界中有着巨大的吸引力——它们的受众规模使它们成为重要的广告平台，也是那些希望信息(无论好坏)尽可能多的人获得的平台。但随着它们的增长，我们看到了专注于更狭隘的特殊利益的平台和网络的持久作用，今天其中一个——专注于所有社区的农民——正在获得一轮融资以推动其增长。

WeFarm是一家面向小农户(即不受大型农业企业控制的农场)的市场和社交网站，已经在首轮融资中筹集了1300万美元，计划用这笔钱继续增加更多用户——农民——以及更多针对他们需求的服务。

这一轮融资使该公司的融资总额达到2000万美元，由True Ventures牵头，AgFunder是June Fund之前的投资者有LocalGlobe、ADV和Norrsken基金会；其他人也参与其中。

WeFarm目前拥有大约190万注册用户，它在提供市场方面的早期举措——帮助农民与当地的商品和设备供应商(如种子和化肥)联系——在运营的前八个月创造了100万美元的销售额，这表明这里有生意可做。这家初创公司指出，事实上，这种增长“比亚马逊和易贝的早期阶段都要快”

WeFarm总部位于伦敦，但尽管这家初创公司的用户确实来自英国和欧洲其他地区，但该公司创始人兼首席执行官肯尼·伊万(Kenny Ewan)在接受采访时表示，它看到发展中经济体的活动和增长更加强劲，在这些经济体中，小规模农业占主导地位，但在新的数字服务方面，那些在农场工作的人却严重得不到服务。

“我们正在代表农民为全球小规模农业建立一个生态系统，”伊万说，并指出全球约有5亿个小规模农场，约有10亿人在这些农场工作，这些农场通常占地1.5-2公顷，通常专注于大米、咖啡、牛或蔬菜等主要经济作物。“这可能是地球上最大的产业，占全球供应链的75-80%,但是没有人为他们建造任何东西。这在许多层面上都意义重大。”

WeFarm提供的服务是双重的。这个网络是免费加入的，首先是作为一个论坛，农民们——他们可能和其他农民一起生活在一个社区里，但也可能很孤独——可以互相问问题或就农业或小块土地问题获得建议。少想脸书，多点堆栈交换。

这为WeFarm的第二个实用轨道提供了一个自然的进展:市场。最初，Ewan表示，它一直在与当地供应商合作，重要的是审查他们，以帮助他们与农民和更广泛的生态系统联系，获得他们可能需要的商品和服务。

从长远来看，目标将是提供一个地方，让小农户能够互相交换商品，或者出售他们生产的产品。

除了提供对待售商品的访问，WeFarm还帮助管理其背后的电子商务流程。例如，在非洲等地区，移动钱包已经成为事实上的银行账户和支付卡的代理，因此人们支付物品的主要方式之一是通过短信。

“对于我们90%的用户来说，我们是他们唯一使用的数字服务，所以我们必须确保我们能够满足他们的信任，”伊万说。“这是全球最大行业的信任网络，我们必须确保它运转良好。”

对于True和其他投资者来说，这是一场长期游戏，财务回报可能没有道德回报那么明显。

True Ventures的联合创始人乔恩·卡拉汉(Jon Callaghan)在一份声明中说:“肯尼和Wefarm团队如何增强世界农民的力量，让我们深受鼓舞，我们看到了他们未来的巨大潜力。”“该公司不仅以影响力为导向，而且Wefarm市场令人印象深刻的增长展示了令人兴奋的商业机会，这些机会将使这些农民获得更多他们需要的东西，从而使整个食品供应链的所有人受益。这是一个庞大的全球性企业。”

尽管如此，考虑到长尾的规模更大，能够整合和管理这个社区的公司也可能拥有非常有价值的业务。
