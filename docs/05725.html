Deadspin 作家在被命令坚持体育运动后辞职

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/30/deadspin-exodus/

作家劳拉·瓦格纳、凯尔西·麦肯尼、汤姆·莱伊、劳伦·泰森、帕特里克·雷德福、艾伯特·伯内科和克里斯·汤普森今天都在推特上宣布他们已经从 G/O Media 旗下的体育网站 Deadspin 辞职。

快速回顾一下:G/O Media 的前身是 Gizmodo Media Group，在此之前是 Gawker Media。今年早些时候，Univision 将该部门出售给私募股权公司 Great Hill Partners，任命前 Forbes.com 首席执行官吉姆·斯潘菲勒(Jim Spanfeller)为其新任首席执行官，从而获得了目前的名称和领导地位。

从那以后，G/O 媒体领导层和编辑人员之间的关系一直不稳定，你可以通过阅读 Deadspin 本身了解到这一点，特别是 8 月份 Wagner 的一篇深度报道，讲述了员工如何对“缺乏关于公司目标的沟通，似乎无视促进公司高层的多样性，以及反复和严重干扰编辑程序”感到不满

几周后，Deadspin 的主编梅根·格伦威尔辞职，称 G/O Media 的新编辑主任保罗·梅德门(Paul Maidment)正在指导员工坚持体育报道——她认为这一决定不是由流量决定的，因为“Concourse 上的帖子，Deadspin 致力于政治和文化以及其他非体育主题的垂直帖子，比主网站上的帖子高出两倍多。”

显然，Maidment 在本周早些时候的一份备忘录中重复了这一法令，这份备忘录被泄露给了《每日野兽》,他在备忘录中说，“Deadspin 将只写体育和以某种方式与体育相关的内容。”

Deadspin 主页随后被非体育内容填满，编辑 Barry Petchesky 发推文称他已经“因不坚持体育而被 Deadspin 解雇”

与此同时，Deadspin 还在网站上发布了一篇批评自动播放广告的文章，声称“我们，Deadspin 的作者、编辑和视频制作人，和你们一样，对我们网站的用户体验现状感到不安。”《邮报》已经停刊，但据报道，批评促使广告商农民保险公司撤回了该活动。

这一切似乎都促使了今天大批人离开了迪兹平。Gizmodo 媒体集团联盟也发表了这份声明:

今天，我们在 Deadspin 的一些同事辞去了他们的职务。从一开始，首席执行官吉姆·斯潘菲勒(Jim Spanfeller)就试图通过削减阅读量最大的网站的覆盖范围来破坏一个成功的网站，因为这让他个人感到不舒服。这不是新闻业的样子，也不是编辑独立的样子。

“坚持运动”一直是“不要对权力说真话”的委婉说法。除了是糟糕的生意，斯潘菲勒的行为在道德上应该受到谴责。GMG 工会与我们现在和以前的同事站在一起，以最强烈的措辞谴责吉姆·斯潘菲勒。

我们已经联系了 G/O 媒体进行评论，如果有回音，我们会及时更新。

更新: G/O Media 向纽约时报提供了以下声明:

他们辞职了，我们很抱歉，他们不能在这个令人难以置信的广泛的覆盖范围内工作。我们对 Deadspin 的未来感到兴奋，我们将在未来几天进行一些重要的更新。
