由同态加密创始人共同创立的安全初创公司 Duality 筹集了 1600 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/30/duality-cybersecurity-16-million/

API 和云解决方案的无处不在，为企业创造了一个有趣的方式世界，无需自己构建服务的每一部分。但它们也释放出了其他东西:由于数据在多个地方以多种方式移动和使用，导致数据泄露的风险增加。现在，一家初创公司已经建立了一种帮助抵御这种威胁的方法——使用同态加密——并宣布提供资金，这是市场需求的一个迹象，也是它为网络安全带来的机会。

Duality 建立了基于同态加密的解决方案——一种加密组织数据的技术，即使该公司与同样处理数据的第三方合作，也能保持加密状态——今天宣布筹集了 1600 万美元的资金。

首轮融资由英特尔投资牵头，Hearst Ventures 和 Team8 参与其中。

Team8 是以色列重量级的网络安全孵化器，它将英特尔(Intel)视为战略合作伙伴(T3)，并拥有一个令人印象深刻的支持者名单(T5)，包括微软、沃尔玛、Eric Schmidt 和埃森哲。

英特尔在这里是一个财务和战略支持者:去年两人合作了一个项目，以揭示人工智能工作负载的安全挑战，该项目利用英特尔平台上的同态加密来最大限度地减少数据暴露。英特尔的应用广泛，包括云服务和大型硬件公司；你可以看到这两者在未来可能会有更多的合作。

Duality 与之合作的其他公司来自金融市场、医疗保健和保险行业，不过该公司表示，由于 NDAs 的原因，它不能透露是谁。它提到的一个客户是英国的 CDA，即网络防御联盟，这是一个跨银行的安全联盟。

另一个很可能是赫斯特，这一轮投资中的另一个投资者。

赫斯特风险投资公司(Hearst Ventures)高级董事总经理肯尼斯·布朗芬(Kenneth Bronfin)在一份声明中表示:“作为一家领先的全球多元化媒体、信息和服务公司，在各个行业拥有 360 多项业务，我们敏锐地意识到数据和数据协作在许多细分市场的公司中越来越重要。“个人和企业不断产生敏感数据；需要有技术来保护这些数据，同时允许我们提取洞察力。我们对 Duality 的使命及其将复杂技术交付给现实世界用例及应用的能力感到兴奋。”

有一些网络安全初创公司和较大的公司正在基于同态加密的原则构建解决方案。

他们包括恩维尔、 CryptoNext Security 和 IBM 。然而，当涉及到这个领域时，Duality 有一个有趣的血统:它的联合创始人之一 Shafi Goldwasser 因其在构成同态加密基础的密码算法方面的开创性工作而获得了图灵奖。

与许多高级数学一样，这项工作在很大程度上是理论性的，因此，由其他联合创始人 Alon Kaufman、Rina Shainski、Vinod Vaikuntanathan 和 Kurt Rohloff 领导的 Duality 所做的工作涉及到将算法变成商业上可行的东西，可供大多数企业使用。

也就是说，处理加密数据仍然需要大量的时间和计算能量，因此二元性的想法是将其用于公司最敏感的信息。随着一些资金流向 R&D，看看算法是否能够改进到足以以实用的方式将这种加密扩展到更广泛的数据集将是一件有趣的事情。

“这里没有免费的午餐，”首席执行官考夫曼在本周的一次采访中说。“同态加密是安全和隐私的圣杯，因为它消除了巨大的挑战。但是有开销。当我们向客户部署它时，我们不会说，“从现在开始，加密所有东西，并假设没有东西是开放的。”这是因为它是一种存储和计算开销。这就是我们关注敏感数据集的原因。他补充说，二元性的一个独特品质是，与其他基于这一原则构建解决方案的方法相比，它的开销显著提高，但同样，“你只在需要时才应用它”。"

“在分析过程中保护数据的能力是未来计算堆栈的关键组成部分，特别是在人工智能的背景下。英特尔投资一直密切关注这一领域，我们很高兴看到安全计算和同态加密变得实用和广泛适用，”英特尔投资副总裁兼高级董事总经理 Anthony Lin 在一份声明中表示。“我们相信人工智能和人工智能中的隐私保护代表着巨大的市场需求，我们正在投资 Duality，因为它拥有独特的创始团队和世界领先的先进密码学和数据科学专业知识。”
