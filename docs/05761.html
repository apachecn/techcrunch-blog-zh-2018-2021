新加坡Qoo10收购印度在线市场shop clues TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/10/31/qoo10-shopclues/

两家公司周四表示，总部位于新加坡的电子商务公司Qoo10已通过全股票交易收购了印度在线市场ShopClues。据两位知情人士透露，这笔交易对ShopClues的估值在5000万美元至8000万美元之间，结束了这家印度公司多年来寻找新家的努力，这家公司在2016年的估值为11亿美元。作为协议的一部分，ShopClues将与易贝支持的Qoo10合并。

ShopClues成立于2011年，是美国Clues Network Inc .的全资子公司，向印度小城镇的用户销售电子产品和家居、厨房和生活用品。截至今年早些时候，该公司声称每天处理超过60，000件快递。它已经建立了一个超过70万小微商户的网络。

两家公司表示，这些来自ShopClues的商家“将能够通过Qoo10在东南亚的存在进入全球市场。”他们补充道，“同样，Qoo10的商家及其跨境物流业务将凭借其高质量、物有所值的产品进入庞大的印度市场。”

他们说，合并已经得到两家公司董事会和主要股东的批准。该交易包括购买Smartship和Momoe enterprise services以及ShopClues运营的Ezonow社交商务平台。

在新加坡、印度尼西亚、马来西亚、中国和香港开展业务的Qoo10表示，希望新协议能帮助其进一步扩大在南亚的业务。

几个月前，ShopClues曾与印度在线零售商Snapdeal就出售事宜进行谈判。当时，ShopClues的估值约为2亿美元。多年来，ShopClues已经从包括Tiger Global和Nexus Venture Partners在内的众多知名投资者那里筹集了超过2.5亿美元的资金。

2016年，当它从新加坡主权财富基金GIC和老虎全球管理公司以及Nexus Venture Partners筹集到约1.4亿美元时，这家初创公司当时声称的估值为11亿美元。ShopClues的联合创始人兼首席执行官桑杰·塞西(Sanjay Sethi)当时告诉TechCrunch，这家初创公司将在一年内上市。

该公司几年来一直在努力提高收入，在此期间，它与每一家电子商务公司进行了合并谈判，包括Flipkart、亚马逊印度和Paytm。去年，ShopClues筹集了1，600万美元——该公司从未公开宣布或承认过这一点——并试图安抚员工，称该公司有重大抱负，计划扩张到美国、尼泊尔、孟加拉国和斯里兰卡。
