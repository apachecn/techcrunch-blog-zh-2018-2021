Twitter的政治广告禁令分散了人们对平台真正问题的注意力TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/11/02/twitters-political-ads-ban-is-a-distraction-from-the-real-problem-with-platforms/

有时感觉好像互联网平台把一切都颠倒了，从政治到出版，从文化到商业，当然还有用真相换谎言。

本周的 bizarro反转是Twitter首席执行官杰克·多西的远景，这位科技首席执行官因完全落后于理解其产品平台的道德曲线而闻名(即纳粹)，在政治言论伦理方面提供了一场即兴的“推特风暴”。

实际上，他在教育脸书的马克·扎克伯格(Mark Zuckerberg)——另一个以与现实世界特别脱节而闻名的科技兄弟，尽管他经营着一个庞大的自由宣传帝国，拥有影响他人生活的巨大权力——站在民主和社会的立场上。

所以不是完全相反。

简而言之，Twitter已经表示将不再接受政治广告。

而脸书最近宣布它将不再事实核查政治广告。又名:谎言是好的，只要你付钱给脸书去传播它们。

你可能会说，脸书的立场有一定的表面清晰度——也就是说，它总结为“当涉及到政治时，我们就是没有任何道德”。大概是希望的结果是“所以你不能指责我们有偏见”。

尽管这实际上是不合理的。通过在政治活动中不应用任何道德标准，脸书为那些最没有道德和最低标准的人提供了帮助。因此，客气地说，它的立场实际上有利于“真理精简版”。(你可以决定哪一方可能会受益。)

Twitter的立场也很明确:全面禁止！政治和问题广告都放入删除箱。但正如我的同事Devin Coldewey很快指出的那样，随着公司开始准确定义什么是(不是)“政治广告”，以及它的少数“例外”可能是什么，边缘可能会变得更加模糊。

事实上，Twitter的定义已经令人惊讶。例如，它显然已经决定气候变化是一个“政治问题”——因此将禁止关于科学的广告。同时，据推测，保持开放，从大石油公司的钱，以促进他们的气候污染品牌…所以是的，混乱。

显然会有人试图进行压力测试，绕过Twitter设定的界限。这项政策听起来可能很简单，但它涉及各种各样的判断，暴露了该公司的政治算计，并使其容易受到偏见和/或道德失败的指控。

尽管如此，就内容标准而言，制定规则是——或者应该是——容易且成熟的事情；对于这些平台来说，执法是真正的汗水辛劳。

这大概也是为什么脸书决定尝试对政治广告不设任何规则——希望避免被迫扮演政治言论警察的角色。

如果这是战略，它已经看起来惊人的愚蠢和弄巧成拙。该公司刚刚为自己设置了一个持续的公关噩梦，它确实被迫从自己的后脚监管故意引发政策的广告——将自己置于“故意腐败的警察”的位置。缓慢地向四周鼓掌。

尽管如此，它至少可以安慰自己，它正在将自己的道德旁路货币化。

正如我们注意到的，Twitter在政治广告上的反对政策也不能免于批评。

事实上，它已经面临的指控,即全面禁止是对公众形象较低的新候选人的歧视。即使这一争论的能量更好地用于倡导广泛的竞选融资改革，包括对选举支出的硬性限制。如果你真的想通过在候选人之间建立公平的竞争环境来重新启动政治，这就是应该做的。

同样至关重要的是:能够对暗钱实施控制的监管，以保护民主政体不被内部通过无形的宣传种子所收买和烹煮，这些宣传种子盗用互联网平台的覆盖范围和数据，将谎言作为民粹主义的真理，将其掩盖在微目标超连接的变形模糊中。

粗略的利益从数据丰富的亿万富翁那里购买廉价的影响力，不受问责或民主审查，是我们新的扭曲的“正常”。但不应该。

这里还有另一个问题。当你考虑到它并不是一个运行政治广告的主要平台时，Twitter禁止政治广告确实是一个令人分心的细节。

在2018年美国中期选举期间，该类别为该公司创造的收入不到300万美元。

其次，任何以推特的形式发布的信息都可以作为政治号召。

正是这些令人愤慨的“有机”推文，才是Twitter平台上真正的政治行动。(嗨特朗普)。)

包括不真实的“有机”推文，这些推文不是一个人真正持有的观点，而是一种伪造的(通常是花钱买的)。称之为“本土化”广告；旨在将谎言作为真理传递的虚假推文，被机器人军队(虚假账户)夸大和放大，作为平行的“非官方”广告基础设施，在众目睽睽之下运作(通常利用Twitter的热门话题)，其使命是产生引人注目的公众舆论哑剧，试图动摇真实的事情。

简而言之:宣传。

当你可以让机器人网络为你做宣传的时候，谁还需要花钱在Twitter上做政治广告呢？

我们不要忘记，多尔西也是一位因没有将其平台的行为规则应用于某些高调政客的推特而闻名的科技首席执行官。(呃，又川普，基本上。)

因此，通过说Twitter禁止政治广告，但继续对世界领导人的推文应用双重标准——最明显的是允许美国总统随意欺负、虐待和威胁，以推进他的民粹主义右翼政治议程——该公司试图拥有自己的蛋糕并吃掉它。

最近 Twitter稍微改进了它的政策，称它将对违反规则的世界领导人的推文施加一些限制。但它继续遵循两套规则。

值得称赞的是，多尔西在他的推特风暴中突出了这种紧张局势——他写道[强调我们的]:

互联网政治广告向公民话语提出了全新的挑战:基于机器学习的消息传递和微定位优化、未经检查的误导性信息和深度假货。所有这些都在以越来越快的速度，越来越复杂，规模越来越大。

这些挑战将影响所有的互联网传播，而不仅仅是政治广告。最好把我们的努力集中在根本问题上，而不是金钱带来的额外负担和复杂性。试图同时解决这两个问题意味着两个都解决不好，而且会损害我们的信誉。

这是多尔西的好东西。出人意料的好，考虑到他和Twitter多年的言论自由原教旨主义——当时该公司因故意无视和无视自由言论在网上蓬勃发展需要公民限制的保护罩这一事实而获得声誉。否则,“放大任何可怕事情的自由”就会变成一种过度伤害少数民族的言论冷却器。

正如多尔西现在指出的，言论自由不同于传播自由。

尽管Twitter在如何定义政治问题上做出了一些令人失望的选择，但就这条广告禁令而言，它与脸书和扎克伯格的对比——仍然在相同的热风中扭曲和旋转；试图证明不连贯的政纲政策是合理的，这些政策为了一种他自己的公司甚至不能坚持的二元意识形态出卖民主——看起来很明显。

多尔西在脸书财报电话会议期间发布推特风暴的时机，显然是为了表明这一点。

“扎克伯格希望我们相信，一个人必须支持或反对没有细微差别、复杂性或文化特异性的自由言论，尽管他经营着一家淹没在复杂性中的公司，”文化历史学家Siva Vaidhyanathan 写道，他在最近的卫报的一篇文章中回应扎克伯格关于自由言论的另一份“宣言”时，面对脸书的道德空虚。“他希望我们的讨论尽可能抽象和理想化。他希望我们不要过于关注脸书本身。”

脸书对言论的立场只是抽象的。正如它的广告定位业务只能在不受监管的默默无闻中运行，在那里，偏见中烘焙的——算法和用户生成的——被安全地隐藏起来，所以人们无法加入他们是如何受到损害的点。

我们不应该对这家丑闻频发的公司如此迅速地被指责为意识形态谎言感到惊讶。由于过去几年平台规模的虚假信息和全球数据丑闻，我们有了更精明的政治阶层。目睹并经历了脸书的政策在现实世界中的转变的人们。比如妥协选举和社区暴力。

随着这些立法者将注意力转向平台巨头，真正有可能对反社会媒体业务进行有意义的监管。

尤其是因为脸书的自我监管一直是另一种危机公关，旨在先发制人，避开真正的危机。这是一种愤世嫉俗的企图，旨在保持对我们注意力的有利控制。该公司从未承诺进行必要的系统性变革，以解决其有毒言论问题。

问题是，最终，毒性和分裂推动了参与，吸引了注意力，并让脸书赚了很多钱。

Twitter可以声称与这种商业模式有一点距离，不仅因为它在通过垄断注意力赚钱方面远不如脸书成功，还因为它为用户提供了更大的余地来建立和关注自己的兴趣网络，不受算法的干扰(尽管它也做算法)。

一段时间以来，中国也一直在走一条自称的改革之路。最近，该公司表示希望负责在其平台上推广“T0”对话健康。没有人会说它还在那里，但也许我们最终会看到一些行动。即使禁止政治广告对Twitter来说是一个快速的公关胜利。

然而，真正艰难的工作仍在继续。即在机器人军队的恶意宣传污染公共领域之前根除它们。Twitter还没有说它很快就能解决这个问题。

脸书也仍然未能阻止其平台上“有机”政治化虚假内容的潮流。通过散布仇恨和谎言来损害我们民主利益的骗子。

对于这类内容，脸书没有提供可搜索的档案(正如它现在对付费广告所做的那样，付费广告被定义为政治性的)，从而为暗钱提供了持续的掩护，通过在群组和页面上免费发帖来进行其操纵民主的黑客工作。

此外，即使脸书声称正在透明地掀起有偿政治影响的帷幕，它也没有做到这一点。它的政治广告API仍然被研究学者抨击为不适合目的。尽管公司政策通过给政客们开绿灯来播放谎言广告，给外部事实审查员增加了压力。

它还被指责在清除“协调的不真实行为”时采用了有偏见的标准，脸书委婉地称之为为扩大和影响而设立的虚假账户网络——当有问题的宣传来自美国内部并倾向于政治右翼时。

 

脸书否认这一点，声称例如其平台上的一个网页网络报道是专门从美国保守新闻网站《每日电讯报》、是由美国真实的人运营的真实网页，他们没有违反我们的政策。(它没有向我们提供如何得出这一结论的任何细节。)

一位公司发言人还表示:“我们正在努力提高透明度，这样将来人们会对脸书上的类似页面有更多的了解。”

所以它仍然承诺“更透明”——而不是实际上是透明。它仍然是解释和应用完全没有法律约束力的政策的唯一法官；那就是虚假监管了。

此外，虽然脸书不时发布禁令，禁止某些国内仇恨言论传播者的有毒内容，如禁止一些信息战争的“亚历克斯·琼斯”页面，但它未能阻止通过新页面再次传播的相同的仇恨。或者实际上是同一伙可恶的人在脸书拥有的不同社交网站上维护着其他账户。政策执行的不一致性是脸书的DNA 。

尽管如此，多尔西决定采取反对政治广告的立场，看起来确实很有政治家风度。

从根本上说，这显然也是正确的做法。购买比你在政治上获得的更多的关注是一种倒退，因为它有利于那些最富有的人。当然，Twitter的立场不会修复这个破碎系统的其余部分，在这个系统中，金钱继续涌入，污染政治。

我们也不知道Twitter的算法如何放大以有机tweet形式包装的政治言论的细节。因此，它的算法杠杆更有可能被触发来推动煽动和煽动的政治推文，还是那些通知和寻求团结的推文。

正如我所说，Twitter的整个平台可以归结为政治广告。该公司确实根据其专有的(和商业的)对“参与质量”的判断，运用算法来掩盖或压制推文。因此，它的整个业务都涉及到如何塑造推特言论的可见性。

这显然包括大量的政治言论。推特不是特朗普的首选平台。

对政治广告的禁令并没有改变这一切。因此，和以往一样，就社交媒体的自我监管而言，我们得到的最多只是在边缘徘徊。

一双愤世嫉俗的眼睛可能会说，Twitter的禁令旨在转移人们对这些吸引注意力的互联网平台中更多结构性问题的注意力。

世界各地的民主国家和社会被迫应对的有毒政治话语问题是互联网平台如何分发内容和塑造公共讨论的结果。所以真正关键的是这些公司如何利用我们的信息来编排我们每个人在第一时间看到的内容。

事实上，我们正在讨论Twitter的政治广告禁令，这可能会分散人们对多尔西顺便提到的“根本问题”的注意力。(尽管他可能会对他们的原因给出不同的定义。在推特风暴中，他只是谈到“努力阻止人们利用我们的系统传播误导信息”。)

脸书对同一个问题的公开诊断总是极其简单，而且是在推卸责任。它只是说有些人是坏的，因此一些坏的东西将被脸书平台化——反映了人性的问题。

这是另一种观点:围绕互联网平台如何传播有毒宣传的所有这些问题的核心问题是，为了操纵我们的注意力而获取人们的数据这一潜在事实。

这种微定位业务——也叫行为广告——把每个人都变成了某种宣传的目标。

不管是唐纳德·特朗普还是迪士尼对你做的，这种做法都很糟糕。因为不对称。太不相称了。这是剥削。它本质上是反民主的。

它还刺激了普遍的、工业规模的个人数据储存，这自然对隐私不利，对安全不利，并消耗大量的能源和计算资源。所以从环境的角度来看也很糟糕。

这一切都是为了最基本的目的。这是平台出卖你，所以其他人可以卖给你东西。不管是肥皂还是政治观点。

扎克伯格为这一过程选择的标签——“相关广告”——只是一个亿万富翁为了润滑管道而说出的巧妙谎言，这些管道吸出了向下游出售我们的注意力所需的数据。

微目标对个人来说是可怕的(意思是令人毛骨悚然的广告；失去隐私；偏见和数据滥用的风险)，出于同样的原因对社会也是可怕的，以及严重的社会层面的风险，如选举干预和敌对势力对来之不易的民主机构的破坏。

个人隐私是一种共同利益，类似于公共健康。预防接种——针对疾病或虚假信息——有助于保护我们所有人免受破坏性传染。

明确地说，微定位也不仅仅是当平台被支付广告定位费用时发生的事情。平台一直都在这样做；应用武器化层来定制他们处理的所有事情。

这就是他们如何分发和编排用户自由上传的大量信息，从日常的人类混乱中创造出最有吸引力的秩序，他们的任务是将自己变成一个引人注目的、个性化的叙事，而不需要支付大量的人工编辑来完成这项工作。

脸书的新闻提要依赖于与行为广告同样的数据驱动原则来吸引和保持注意力。推特的“顶级推特”算法排名视图也是如此。

这是大规模的程序化注意力操纵，被重新包装成一种“社会”服务。一种是利用这些平台通过监视互联网用户学到的东西作为分裂粘合剂来束缚我们个人的注意力，即使这意味着让我们中的一些人相互对立。

这就是为什么你可以发表一篇脸书帖子，提到一个特定的政治问题，并且——实际上在几秒钟之内——从一个你已经多年没联系过的脸书“朋友”那里吸引一个强烈表达反对意见的人。该平台能够传递“直觉穿孔”内容，因为它通过每个人数据的棱镜，对每个人都有上帝般的看法。数据为其算法提供动力，将内容插入“相关”眼球，按订婚火花爆发的最高可能性排序。

不言而喻，如果一个真正的友谊团体包含这样一个玩游戏的跟踪者——他窃听每个人的手机来窥探和监视他们，并利用他们学到的东西来挑拨朋友之间的关系——没有人会想象这会让这个团体更亲密。然而，这就是脸书对待其被俘获眼球的方式。

在扎克伯格最近一次入主白宫时，当他遇到某些尖锐的问题时，你会听到尴尬的沉默，这可能只是小钱的下落。

最终，立法者似乎越来越接近于理解嵌入这些数据内容社会技术平台的真正“根本问题”。

邀请我们凝视他们的平台，以便他们可以永远与我们亲密接触——利用他们从间谍活动中学到的东西，更深入地探查和更快地利用。

所以，虽然禁止政治广告听起来不错，但这只是分散注意力。我们真正需要打破的是反对社会的黑镜平台，在这种平台中，他们可以从各个角度观察我们，同时阻止我们看到他们在做什么，这就是关闭一个全面的隐私屏幕。不针对个人数据。

让他们给我们看内容和广告，当然。他们可以根据一些通用的信息，根据上下文来定位这些内容。他们甚至可以要求我们具体说明，我们是否希望看到有关当今住房或包装消费品的广告？我们可以协商规则。其他一切——我们在站台上或站台下做什么，我们和谁说话，我们看什么，我们去哪里，我们说什么——都必须严格保密。
