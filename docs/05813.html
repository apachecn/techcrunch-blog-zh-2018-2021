你现在可以试用微软基于网络的 Visual Studio 版本

今年早些时候，在其 Build ddevelopers 会议上，微软宣布其正在开发基于网络的 Visual Studio IDE 版本。当时，Visual Studio Online 进入了一个私人预览版，只对部分开发人员开放。现在，在 Ignite 大会上，该公司向所有想尝试一下 的开发者开放了服务 。

有了 Visual Studio Online，开发人员将能够为他们的存储库快速构建一个完全配置好的开发环境。从那里，他们可以使用基于网络的编辑器来处理他们的代码。

“Visual Studio Online 汇集了 Visual Studio、云托管的开发人员环境和基于 web 的编辑器，可从任何地方访问，以帮助开发人员比以往任何时候都更有效率，”微软在其新闻材料中指出。“随着开发变得更加协作，开源工作流(如拉式请求)变得更加普遍，开发人员需要能够在代码库和项目之间快速切换，而不会降低工作效率。”

在目前的形式下，该服务与微软的 GitHub 深度集成(这并不奇怪)，但该公司指出，开发人员也可以将自己的物理和虚拟机连接到他们基于 Visual Studio 的环境中。开发人员也可以直接从 Visual Studio Code 创建这些在线环境，Visual Studio Code 是该公司越来越受欢迎的用于 Windows、Mac 和 Linux 的免费代码编辑器。

基于云的环境，以及对 Visual Studio 代码的扩展支持，现在都在预览版中。
