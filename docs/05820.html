Cloudflare 打败了专利巨魔。现在怎么办？

原文：https://web.archive.org/web/https://techcrunch.com/2019/11/05/cloudflare-beat-a-patent-troll-what-now/amp/

图片来源:史密斯收藏/加多/盖蒂图片社

2017 年夏天，我们报道了总部位于旧金山的互联网安全和内容交付网络 Cloudflare 与两名律师之间的一场战斗，这两名律师此前曾代表众多科技巨头提起知识产权诉讼。律师们聚集在一起，成立了总部位于波士顿和芝加哥的公司 Blackbird Technologies ，该公司迅速积累了数十项专利，然后开始利用这些专利对包括 Cloudflare 在内的公司提起数十起专利侵权诉讼。

这套衣服在各方面都很典型，除了 Cloudflare 对它的反应。Cloudflare 没有像 Blackbird 和其他所谓的专利流氓的一些目标那样悄悄地和解，而是决定以非常公开的方式进行反击，广泛地写博客，与像我们这样的新闻媒体交谈，最重要的是，求助于任何可以帮助它找到现有技术的人。这个想法不仅仅是为了使 Blackbird 用来起诉 Cloudflare 的专利无效，而是为了使 Blackbird 的所有专利无效。Cloudlfare 宣战了。

值得称赞的是，Cloudflare 也赢了。至少，针对 Cloudflare 本身的案件最终被驳回，在昨天发表的一份事后分析中，该公司详细描述了其游戏计划以及围绕其努力众包可能使黑鸟专利无效的现有技术的更多细节。

例如，它显示已收到 155 名个人就 49 项独立专利提交的 275 份独特申请，以及就 26 项专利提交的多份申请。其中大约 40%与针对 Cloudflare 的专利有关，但个人也提出了有助于保护 Niantic(也是试图回击 Blackbird )以及 Lululemon 和 New Balance 的现有技术提交物，这两家公司此前都被 Blackbird 起诉过，原因是 Blackbird 拥有一项与“具有整体存储袋的运动胸罩”有关的专利

Cloudflare 还对 Blackbird 的创始人提出了道德投诉，这些投诉源于职业行为规则，这些规则禁止律师获得代表自己主张的诉讼理由。这些投诉可能会导致什么结果现在还是一个问号，尽管值得注意的是黑鸟的创始人之一，克里斯·弗里曼，以前在柯克兰&埃利斯，现在在一家芝加哥公司工作，为诉讼提供资金。

无论如何，Cloudflare 的胜利是一个在大量坏消息中感觉良好的故事，但它也回避了一个问题:下一步是什么？

有些人可能希望 Cloudflare 会以某种方式继续其反对不公正的运动。如果是这样的话，他们会感到失望——cloud flare 从一开始就表示，一旦与 Blackbird 的法律纠纷结束，它就会退出专利战业务，该公司的总法律顾问道格·克莱默(Doug Kramer)在上周晚些时候的谈话中向我们重申了这一决定。正如他所说，考虑到该公司其他更紧迫的问题(包括 9 月份上市), cloud flare 的改革从未打算成为“终身宣传”。

尽管如此，传递火炬并不容易。克雷默承认，他已经接到“许多来自其他法律总顾问或知识产权律师和首席执行官(他们也是专利诉讼的目标)的电话，他们说，‘除了延期付款和开支票，我们这里还有什么可以做的吗？’"

这让我们想到了昨天的帖子。这不是简单地吹嘘它赢了黑鸟，没有双关语。Kramer 说，更多的是要感谢 Cloudflare 非常依赖的社区。克雷默说，这是为了“明确这里有其他前进的道路”，他希望其他公司将使用 Cloudflare 蓝图的一部分，并建立自己的蓝图。

表明立场并不适合所有人。正如克莱默所观察到的，一旦对一家公司提起专利诉讼，“所有的选择都是糟糕的选择，很多公司选择了最不糟糕的选择”，也就是开一张支票了事。这就是为什么像黑鸟这样的公司获得动力。"他们遇到的阻力很小。"

克莱默不一定会责怪目标容易弃牌。即使事情对公司有利，如 Cloudflare，诉讼也可能需要数年时间，并可能使公司损失数十万美元，甚至数百万美元。“作为一个诉讼问题，我们在第一次推销时就把它否决了，”他说，“但它比我们只写一张支票花费更多。”

但是更多的公司需要采取更多的行动，如果他们希望看到专利按照他们最初的意图被使用，克莱默说“引入更多的阻力”是达到目的的一种方式。他表示，当然，他们可以使用一些工具，包括“非常活跃、聪明、有思想的人，他们跨越内部职位和专门致力于再次抵制这些做法的第三方组织起来。”

另一种是同情的政客，如马萨诸塞州的州参议员 Eric Lesser，他认为专利流氓是对他所在州经济的威胁，并尽他所能驱逐他们和他们的侵权索赔。

此外，还有工程师和其他建造东西的人，他们不喜欢通过许可或诉讼手段而不是通过生产自己的商品或服务来获利的公司的崛起。事实上，并不是每家公司都有 Cloudflare 的财务实力，它在上市前从投资者那里筹集了超过3 亿美元。(它还收到了一笔的匿名捐款5 万美元，以支持其对抗黑鸟的努力。)然而，克莱默说，即使没有这样的资源，公司外部社区的支持也可以积累很多。

“我们开始了解到，有很多人——商业领域的同事、朋友和志同道合的人，以及科技公司的普通人——真的对获得专利的公司滥用专利、要求版税和索赔感到困扰。

这并不能完全解决这些年来专利案件稳步上升的问题，但是每一点——每一个热心的倡导者，他们能提供的每一条有用的信息——都有所帮助。Cloudflare”并没有解决[这个更广泛的问题]。它仍然存在，”克莱默说。“但我们能够利用这种情绪。希望这对其他人来说是一个证据，表明那里有支持。”
