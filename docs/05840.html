Neural Magic 获得 1500 万美元的种子，用于在商用 CPU 上运行机器学习模型

由几名麻省理工学院教授创建的初创公司 Neural Magic ，他们找到了一种在商品 CPU 上运行机器学习模型的方法，今天宣布了 1500 万美元的种子投资。

康卡斯特风投领投，NEA、安德森·霍洛维茨、Pillar VC 和 Amdocs 参与其中。该公司此前获得了 500 万美元的前期种子资金，使迄今为止的融资总额达到 2000 万美元。

该公司还宣布提前获得其第一款产品，即数据科学家可以在运行 CPU 的计算机上运行的推理引擎，而不是像 GPU 或 TPUs 这样的专用芯片。这意味着，通过允许数据科学家使用商用硬件，它可以大大降低与机器学习项目相关的成本。

这个解决方案的想法来自麻省理工学院教授 Nir Shavit 和他的研究伙伴兼联合创始人 Alex Mateev 的工作。正如他所说，他们正在实验室研究神经生物学数据，并找到了一种使用他现有的商用硬件的方法。“我发现，通过正确的算法，我们可以在商用硬件上运行这些机器学习算法，这就是公司开始的地方，”Shavit 告诉 TechCrunch。

他说，有一种错误的观念，认为你需要这些专门的芯片或硬件加速器来拥有运行这些工作的必要资源，但他说并不一定是这样。他说，他的公司不仅允许你使用这种商用硬件，它还可以与更现代的开发方法合作，如容器和微服务。

“我们的愿景是让数据科学团队能够利用他们已经拥有的无处不在的计算平台，以 GPU 的速度运行深度学习模型——以一种只有商用 CPU 才能提供的灵活和容器化的方式，”Shavit 解释道。

他说，这也消除了这些其他方法的内存限制，因为 CPU 可以访问更大量的内存，这是他的公司的方法在节省成本以外的一个关键优势。

“是的，在商用处理器上运行可以节省在 CPU 上运行的成本，但更重要的是，它消除了所有这些巨大的商业化问题，以及基本上整个机器学习领域的巨大限制，即必须处理小模型和小数据集，因为加速器有点有限。这是神经魔法的重大突破，”他说。

主要投资者康卡斯特风险投资公司(Comcast Ventures)的董事总经理吉尔贝达(Gil Beyda)认为，让人们使用商用硬件的方法是一个巨大的市场机会。“神经魔法正在使用软件取代高成本、专门的人工智能硬件。他在一份声明中说:“软件之所以胜出，是因为它释放了深度学习的真正潜力，来构建新颖的应用程序，并解决行业中一些最大的挑战。”。
