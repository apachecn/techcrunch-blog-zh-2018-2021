微软的 HoloLens 2 开始发货

原文：https://web.archive.org/web/https://techcrunch.com/2019/11/07/microsofts-hololens-2-starts-shipping/

今年早些时候，在巴塞罗那举行的世界移动通信大会上，微软宣布了其第二代 HoloLens 增强现实护目镜。如今，售价 3500 美元的 HoloLens 2 已经在美国、日本、中国、德国、加拿大、英国、爱尔兰、法国、澳大利亚和新西兰开始销售，这些国家也是之前可以预订的地方。

在发布之前，在今年早些时候巴塞罗那的一次简短演示之后，我花了一些时间来了解最新的模型。用户会立即注意到更大的视野，这仍然没有覆盖你的整个视野，但与第一个版本相比，提供了更好的体验(在第一个版本中，你经常感觉像是通过邮票大小的窗口观看虚拟物体)。

该团队还大大增强了佩戴该设备的整体感觉。它并不轻，只有 1.3 磅，但有了可翻转的前遮阳板和新的安装系统，它要舒适得多。

在常规使用中，现有用户也会立即注意到打开开始菜单的新手势(毕竟这是 Windows 10)。现在，你只需在手掌上轻轻一点，当你看着它时，就会出现一个微软的标志，而不是经常导致误报的“绽放”手势。

眼球追踪也有了很大的改进，即使在很远的距离上也能很好地工作，新的机器学习模型在追踪你的所有手指方面也做得更好。所有这些都是由许多定制硬件驱动的，包括微软的第二代“全息处理单元”。

微软还增强了它为 HoloLens 构建的一些云工具，包括 Azure Spatial Anchors，它允许在给定的空间中保留全息图，任何使用全息应用程序的人都可以在同一地点看到。

综上所述，所有这些变化将带来更舒适、更智能的设备，减少您在观看周围各种物体并与之互动时的延迟。
