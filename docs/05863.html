Ubiquiti 将路由器和 Wi-Fi 接入点与 UniFi 梦想机相结合

如果你关心网络，Ubiquiti 是一个知名品牌。许多公司、学校和公共场所使用 Ubiquiti 接入点在整栋建筑内广播 Wi-Fi 网络。本周早些时候，该公司宣布了一款名为 UniFi 梦想机器的新设备。

这款新设备非常有趣，因为它直接面向专业消费者。如果你正在寻找即插即用的网状 Wi-Fi 路由器，Ubiquiti 有一系列品牌为 AmpliFi 的路由器。如果你是一个网络专业人士，你可能更感兴趣的模块化系统与机架路由器和交换机。

梦想机器位于这两个部分之间。它结合了一个路由器、一个带有四个以太网端口的交换机和一个 Wi-Fi 接入点。它有一个集成的云密钥，让您控制您的网络和系统流量。

在路由方面，Dream Machine 有一个集成的安全网关，允许您创建防火墙策略。它支持您期望从高端路由器获得的一切，如服务质量、VLAN 支持、站点到站点 VPN 和远程 VPN。

在交换方面，有一个连接互联网的广域网端口和四个千兆以太网端口。在无线网络方面，梦想机器通过 4×4 MU-MIMO 天线支持 802.11 AC Wave 2(“Wi-Fi 5”)。

在幕后，该设备使用 1.7GHz ARM Cortex-A57 处理器。它有 2GB 的内存和 16GB 的存储空间，功耗高达 26W。

当然，你可以使用 Ubiquiti 的网络控制器软件调整每一项设置。这种药丸形状的设备售价 299 美元。这个产品对大多数人来说可能是大材小用了。但是，如果你像我一样是一个网络呆子，你想能够控制你的网络，这可能是一个很好的切入点。
