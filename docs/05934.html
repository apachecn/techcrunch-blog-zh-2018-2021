鉴于 ICE 的合同，GitHub 面临更多的辞职

原文：https://web.archive.org/web/https://techcrunch.com/2019/11/13/github-faces-more-resignations-in-light-of-ice-contract/

微软旗下的 GitHub 最近因其与移民和海关执法局的 20 万美元合同而受到严格审查。现在，另一名员工、工程师爱丽丝·戈德福斯(Alice Goldfuss)已经辞职。

在一条推文中， Goldfuss 说 GitHub 有许多问题需要解决，而“ICE 只是最新的一个。”

同时， Vice 报道今天至少有五名员工辞职。这些辞职与该公司的大型产品会议 GitHub Universe 在同一天举行。在会议之前，技术工人联盟抗议这一事件，设立了一个笼子来代表 ICE 拘留儿童的地方。

上个月，GitHub 的员工工程师 Sophie Haskins 辞职，称她离开是因为公司没有取消与 ICE 的合同，洛杉矶时报报道。

上个月，GitHub 的员工写了一封公开信，敦促该公司停止使用 ICE。此前，GitHub 宣布向非营利组织捐赠 50 万美元，以支持“现任政府针对的移民社区”在该声明中，GitHub 首席执行官 Nat Friedman 表示，ICE 的收购是通过 GitHub 的一个经销商合作伙伴进行的，并表示该交易对该公司来说“在财务上并不重要”。弗里德曼还指出，ICE 负责的不仅仅是移民和拘留设施。

“[……]我们认识到，移民与海关执法局既负责执行我们强烈反对的美国移民政策，也负责对我们的社会至关重要的政策，如打击人口贩运，”弗里德曼写道。“我们不知道内部 GitHub Enterprise Server 许可证将用于哪些具体项目，但我们知道它可以用于支持我们同意或不同意的政策的项目。”

但一些员工没有被弗里德曼的话说服。

“我们对 GitHub 在这个问题上的公开立场不满意，”GitHub 员工在一封公开信中写道。“GitHub 已经占据了‘桌子上的一个席位’超过 2 年，因为这些非法和非人性化的政策已经升级，却没有什么表现。继续持有这份合同并不能提高我们与 ICE 的议价能力。它所做的只是让我们成为他们广泛侵犯人权的同谋。”

据《洛杉矶时报》上个月的报道，GitHub 首席运营官·埃里卡·布雷西亚在回复那封公开信时说，阻止 ICE 使用 GitHub 会“伤害那些我们都想帮助的人”。然而，正如今天上午的行动所示，员工们并没有放松。

当被问及 GitHub 对其与 ICE 合同的态度时，GitHub 让我看了它上个月的博客文章。TechCrunch 已经发送了一份后续说明，看看该公司是否会对辞职发表评论。
