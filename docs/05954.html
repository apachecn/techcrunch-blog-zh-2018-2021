网络安全专家Alex Stamos谈脸书反恐团队和公私分界TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/11/14/cybersecurity-expert-alex-stamos-on-what-scares-him-most-about-the-upcoming-u-s-presidential-election/

亚历克斯·斯塔莫斯作为雅虎和脸书的首席安全官一举成名。但今天，他是斯坦福大学互联网观察站的主任，在那里，他沉浸在安全技术的教学和研究中，比大多数人更了解美国面临的威胁，尤其是在我们迈向下届美国总统选举的时候。

昨晚，在旧金山举行的 StrictlyVC活动上，他与《纽约时报》网络安全记者Sheera Frenkel谈论了这些具有重大影响的问题中的一小部分，首先回顾了2016年总统选举期间发生的事情，然后让观众了解了美国的防御自那以来是否发生了变化。(简而言之:他们没有。如果说有什么好消息的话，那就是联邦政府和州政府至少意识到了这个问题，而上次他们似乎对此一无所知。)

斯塔莫斯最担心的是“对我们选举基础设施的直接攻击”，因为支持它的东西太少了。事实上，采访的一个大主题是公共部门越来越无力保护美国人或美国民主免受危害国家的行为者的伤害。

由于它具体涉及选举基础设施，斯塔莫斯用一个超本地的例子来强调美国目前正在处理的问题。正如他告诉弗伦克尔的，“我住在圣马特奥县。我见过圣马特奥县的首席信息官。非常好的人。我肯定他有一群非常努力工作的员工。圣马特奥县的首席信息官必须挺身而出，保护自己免受GRU、中国国家安全部、伊朗伊斯兰革命卫队或朝鲜拉扎勒斯组织的攻击。。。这太荒谬了。就像，我们不要求圣马特奥县治安部门准备好击退人民解放军的入侵，但我们要求美国的网络对等物。"

斯塔莫斯说，从长远来看，圣马特奥县是美国大约1万个参与选举的地方政府之一。"世界上没有其他人以这种方式进行选举。"

事实上，在后来的讨论中，他告诉弗伦克尔，几乎从所有可以想象的角度来看，“曾经明确属于公共部门的责任，现在变成了私人部门的责任”。他会知道的，因为他亲眼见过。

“当我在脸书担任首席安全官时，”他告诉听众，“我有一个儿童安全小组。在儿童安全领域，除了联邦调查局(FBI)或国土安全部(Homeland Security investigation unit)之外，我们抓到的坏人可能比任何执法机构都多。就像，在美国没有任何一个地方警察部门比脸书儿童安全小组逮捕了更多的儿童掠夺者。这是一个疯狂的T2统计

他表示，脸书也有一个反恐团队——并非所有人都意识到了这一点——该团队在许多方面已经成为该国的第一反应者。的确，斯塔莫斯说“有几起你从未听说过的恐怖袭击，因为它们不是因为我们抓住了它们而发生的。现在，有一些当地执法机构认为这是他们的功劳，但实际上是我们的团队发现了它，并把它交给了他们，上面还有一个蝴蝶结。”

斯塔莫斯表示，美国人可能对这种谁在应对什么的持续变化不屑一顾，但他们这样做是在冒险。斯塔莫斯设法让观众笑起来，即使他描绘了一幅黯淡的画面。正如他指出的，大型科技公司“在没有任何民主监督的情况下行使这种权力。”他说，考虑一下，“(脸书的)授权是人们在加入脸书或Instagram时点击进入、但从未阅读过的服务条款。当你拥有如此不可思议的权力时，你需要遵守一套奇怪的规则。”

Stamos说，另一个巨大的盲点是，公众和日益强大的私营部门显然没有能力——以及缺乏必要的集体决心——来协调他们的工作。在这里，他提供了另一个广泛的例子来使它变得容易理解。“比方说，你在美国有一个有组织的团体，他们经营着一堆脸书广告，但他们的钱来自圣彼得堡的比特币，”斯塔莫斯说。“那是脸书完全看不见的。那个就是也许对FBI来说是可见的。。。但他们无法访问[FB上]实际内容。找到一种方法让这两个群体相互合作，同时又不会大规模侵犯平台上每个人的隐私，这是非常困难的。”

然而，这比听起来还要糟糕，他继续说道。原因是:没有决策树，部分原因是这个问题已经变得如此难以控制，以至于没有人愿意承担出现问题的后果。“现在实际上没有人负责这件事，这是我们作为一个国家面临的最可怕的事情之一。几乎没有人捍卫网络，当然也没有人负责大局，[意思是]我们如何从网络安全角度和虚假信息角度防范选举[干扰]。”

斯塔莫斯甚至开玩笑地提到“美国政府中的一小撮人，他们实际上在躲避白宫，并非常非常努力地试图”逃避它的注意，因为他们面临的任务是找出令人生畏的工作。斯塔莫斯说，除了开玩笑之外，没有人掌舵，“没有真正的跨机构流程，真的没有人负责”。

这意味着“科技公司实际上是这方面的协调机构。这实际上是非常糟糕的。”
