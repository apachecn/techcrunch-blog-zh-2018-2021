谷歌在其 Android Messages 应用程序中为美国带来了 RCS 支持

谷歌今天宣布，它现在向美国所有用户推出对 Android Messages 应用程序中丰富通信服务消息(你可以将其视为下一代短信)的支持，此前最近几个月已经在一小部分用户中进行了测试。对于谷歌来说，推动 RCS 也是该公司更有效地与苹果的 iMessages 竞争的一种方式(尽管它不具备端到端加密功能)，因为谷歌基本上从运营商手中夺走了这一部署的控制权，所以当用户获得这一服务时，它可以发号施令，而不是电信公司。今年早些时候，它已经在英国和法国这样做了，所以该公司在管理这项服务方面有一些经验。

众所周知，谷歌的信息战略，至少对消费者来说，仍然是混乱的，Hangouts 仍然是一个广泛使用的工具。至少在移动领域，谷歌希望 Messages 能够取代这一角色。迄今为止，Messages 基本上是谷歌的短信客户端。与其他消息服务一样，Messages 中的 RCS 支持将允许您通过 Wi-Fi 或移动数据与朋友通话，并发送照片和视频。你还会收到阅读回执、打字通知和所有你期待的常用信息功能。

随着谷歌控制了推出，它现在也负责保持这个网络的运行，有一些合理的担心，该公司拥有这些运营商。然而，另一方面，运营商让他们自己的 RCS 部署尽可能混乱，这并没有给他们带来任何好处，直到谷歌除了自己做这件事之外，真的别无选择。然而，对于 Android 用户来说，这是一个好消息，尽管他们在 iPhone 上仍然会出现一个绿色的泡泡——因此会被他们使用 iPhone 的朋友所评判。
