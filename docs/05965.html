Salesforce 宣布将营销云迁移到 Microsoft Azure

在企业软件的世界里，经常会有奇怪的伙伴。就在昨天，Salesforce 宣布与 AWS 围绕云信息模型展开重要合作。今天早上，它宣布将其营销云转移到微软 Azure。这就是企业伙伴关系有时摇摆不定的方式。

这些公司还宣布他们正在与微软团队合作，将团队与 Salesforce 销售云和服务云集成在一起。

Salesforce 计划在未来几个月内将一直在自己的数据中心运行的营销云迁移到 Microsoft Azure，尽管确切的迁移计划时间表尚不清楚。这对与 AWS 激烈争夺客户的微软来说是一件大事。AWS是该领域明显的市场领导者，但微软一段时间以来一直是强有力的第二名，将 Salesforce 作为客户加入当然是该公司的质量参考。

CRM Essentials 的创始人布伦特·利里(Brent Leary)多年来一直在观察这个市场，他说，这种伙伴关系在很大程度上说明了微软今天的经营方式，它愿意广泛合作以实现其目标。“我认为更大的新闻是 Salesforce 选择与微软深入合作，而不是亚马逊，微软并不担心以 Dynamics 365(其 CRM 工具)为代价来加强 Salesforce，主要是因为他们最大的增长动力是 Azure，”Leary 告诉 TechCrunch。

微软和 Salesforce 一直有着复杂的关系。在史蒂夫·鲍尔默时代，他们在他们的 CRM 产品上进行决斗。后来，塞特亚·纳德拉在 2015 年的《梦想力量》中出现，点燃了一段友谊。自那以后，这种关系时好时坏，但随着这一声明的发布，似乎敌友比敌人更接近朋友了。

但是，我们不要忘记，就在昨天，Salesforce 宣布与 AWS 围绕云信息模型建立合作关系，这将与 Adobe、微软和 SAP 之间的另一种合作关系直接竞争。或者就在去年，Salesforce 宣布了与 AWS 在数据集成方面的重要合作。

这些相互冲突的交易令人困惑，但它们表明，在当今互联的云世界中，在市场的一个领域相互激烈竞争的公司，在对双方和客户都有意义的情况下，可能仍然愿意在其他领域合作。这些公司今天发布的公告似乎就是这种情况。
