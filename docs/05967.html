Target 将 Shipt 的当天送达服务整合到其移动应用程序中

原文：https://web.archive.org/web/https://techcrunch.com/2019/11/14/target-integrates-shipts-same-day-delivery-service-into-its-mobile-app/

Target 的应用程序将实现当天送达。零售商今天早上宣布其当天购物服务 Shipt，目标两年前以 5.5 亿美元收购，现在将直接集成到目标移动应用中。虽然 Shipt 作为一种与 Instacart 和其他公司竞争的在线杂货服务而广为人知，但 Target 正在让这项服务发挥作用，不仅仅是提供食品和各种家居用品。

相反，Shipt 正在变成 Target 自己版本的亚马逊 Prime Now。目前，目标购物者可以从该应用程序订购 65，000 种商品，用于当天送货，不仅包括杂货和必需品，还包括玩具、婴儿护理产品、厨具等。相比之下，亚马逊表示，Prime 现在提供“数万种”一两个小时送货的产品。然而，Shipt 可能没有那么严密的窗口，只是“当天”

虽然亚马逊没有透露通过 Prime Now 可以获得多少产品，因为它因地点而异，2018 年的一项研究表明亚马逊最大的市场在每个城市提供大约 55，000 种 Prime Now 产品，而其他许多城市提供 34，000-41，000 种 SKU。

与 Prime Now 不同，Shipt 不需要会员资格，尽管有一个会员资格。相反，购物者可以选择为每个当天订单支付 9.99 美元的送货费。这类似于沃尔玛超市的运作方式，尽管它现在正在为更多的老顾客推出订阅服务。

Shipt 与目标应用程序的集成并不意味着专用的 Shipt 应用程序将会消失——目前这仍然是一种更方便的购物体验。然而，这是 Target 向更广泛的移动用户群提供 Shipt 服务的一种方式。

今年早些时候，Target.com 推出了一个当天送达的专用购物网站，这也是一个类似的目标。

Target 一直在稳步实现业务现代化，以更好地与亚马逊竞争，并帮助客户以他们想要的方式购物——店内、在线或两者的混合，就像 Drive Up orders 一样。在不到两年的时间里，免下车服务成为了一项顶级服务，例如，通过在今年上半年完成超过 500 万份订单，该服务的订单数量比 2018 年增加了一倍多。与此同时， Target 最近表示在 Q2，五分之一的客户首次向 Target 下当天订单，这表明当天订单有增长的潜力。

同一天 Shipt 集成今天在目标应用程序中推出，它还包括一个新的“我的商店”标签，购物者可以通过“交付”按钮将列表转换为购物车。“发现”按钮还可以帮助他们导航和查找其他功能，如交易和季节性内容。

Target 告诉我们，如今在美国 48 个州的近 250 个市场都可以实现当天送达。
