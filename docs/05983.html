陆军摄影测量技术在几分钟内制作 3D 航空地图

航空影像在军事事务中是一种常见的资产，但是在没有专门设备的情况下，很难在短时间内收集 3D 地图。然而，这种来自陆军工程兵团的新摄影测量技术可以在几分钟内从普通的航空镜头中制作出精确的 3D 地图。

摄影测量是比较同一地点或项目的多张照片以生成其 3D 地图的过程。这是一种众所周知的方法，但在某些情况下，它仍然依赖于人的智能来确定，例如，应该使用视频的哪些帧来产生最佳结果。

弗吉尼亚州陆军地理空间研究实验室的 Ricky Massaro 缓解了这一问题，并开发了一种高效的摄影测量方法，可以在没有任何人工监督的情况下，几乎实时地将航空图像转换为精确的 3D 表面地图。

这张图片显示的深度图是彩色的，红色更高。它是由多个 2D 图像组合而成的。

该系统由第 101 空降师测试，该空降师在肯塔基州的坎贝尔堡上空驾驶无人机，并绘制了一个用于训练演习的模拟城市。它也被部署在伊拉克用于非战斗目的。因此，这不是停留在某个实验室——它已经投入使用，现在正在公开，因为专利申请已经提交，军方正在谈判将该系统商业化。

“无论是对士兵还是农民来说，这项技术都可以快速提供可用的地形和情报产品，”国防部商业技术转让组织 TechLink 的经理昆顿·金说。“我很乐意帮助企业了解如何将 Massaro 博士的研究成果用于自己的产品或应用。”

实时摄影测量不会取代激光雷达或地面测绘系统，但会与它们协同工作。能够从普通的航空图像中产生精确的深度，而不必将大量数据发送到中心位置或涉及人类专家，这使它能够适应各种情况。如果你对细节感兴趣，你可以点击这里查看专利申请。
