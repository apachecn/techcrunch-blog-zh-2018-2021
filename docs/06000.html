Salesforce 与苹果的合作关系开始活跃起来

去年，在 Dreamforce，Salesforce 的大型年度客户大会上，苹果和 Salesforce 宣布开始合作，两家公司将共同努力增强在苹果设备上运行的 Salesforce 产品。今天，随着今年的 Dreamforce 大会的开始，两家公司宣布了这一努力的成果，并在去年的活动中首次发布了两种新工具。

首先，苹果一直在与 Salesforce 合作，重新设计 Salesforce 移动应用程序，以在应用程序中内置苹果 iOS 功能，例如能够使用 Siri 快捷方式更快地完成工作，使用语音而不是打字，这在移动设备上有时很难做到。

照片:Salesforce

例如，您可以说，“嘿 Siri，下一次销售会议”，Siri 可以与 Salesforce CRM 进行交互，告诉您您与谁会面，他或她的公司名称，您上次会面的时间，以及爱因斯坦机会分数，以帮助您预测您今天(或最终)达成销售的可能性。

此外，该移动应用程序利用苹果的 Handoff 功能来立即反映设备之间的变化，并利用苹果的 Face ID 来轻松登录应用程序。

Salesforce 还宣布在 Salesforce Mobile 上试行 Einstein Voice，允许销售代表使用语音输入注释、添加任务和更新 CRM 数据库。爱因斯坦是 Salesforce 的通用人工智能层，语音功能使用自然语言理解来解释代表问的问题。

该公司报告称，超过 1000 家公司参与了更新应用程序的试点，这是该组织历史上最大的试点。

Salesforce 还宣布了其新的移动开发平台 SDK，该平台是专门为使用 Swift 语言的 iOS 和 iPadOS 构建的。这个想法是提供一个工具，让 Salesforce 开发人员能够为 iPad 和 iPhone 构建应用程序，然后用一个名为 Swift UI 和 Package Manager 的新工具将它们打包。

Trailhead Go 是该公司专门为 iPad 和 iPhone 设计的在线学习平台的移动版本。它是使用新的移动 SDK 构建的，允许用户在移动环境中访问他们可以在 web 上访问的相同课程。新的移动工具包括在设备之间切换的能力，以及对画中画和多任务分割视图的支持。

Salesforce Mobile 和 Trailhead Go 从今天开始在 iOS 应用商店免费提供。Salesforce Mobile SDK 将于今年晚些时候推出。

随着这种合作关系的继续发展，两家公司都将受益。Salesforce 可以直接访问苹果的功能，并可以与苹果合作，以优化的方式实现这些功能。在全球最大的企业软件供应商之一 Salesforce 的帮助下，苹果公司更深入地了解了企业。
