野马 Mach-E 是未来电动汽车的激动人心的外形 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/11/18/the-mustang-mach-e-is-the-exciting-shape-of-the-electric-future/

福特刚刚发布了它的第一辆电动汽车 ，它令人惊叹。被称为野马马赫-E，它似乎得到了很多事情的权利。从品牌到小惊喜，野马马赫-E 看起来是一个打击。

福特北美总裁 Kumar Galhotra 在周日发布会前的新闻发布会上表示:“我们知道我们必须做一些不同的、令人兴奋的事情，这些事情只有福特才能做到。”我认为他是对的。野马 Mach-E 是量产电动未来的理想外形。亨利·福特会感到骄傲的。这是一款面向大众的电动车。

这是一辆野马

从品牌开始。我讨厌它。作为一个汽车人，我为野马即将推出四门版感到难过。还有电动的。举起来了。我想说，那不是野马。野马是一种跑车。然而，野马是一种负担得起的，可靠的车辆，这就是为什么福特称其第一辆电动车为野马。

品牌对于电气的未来至关重要。福特正在使用一个能引起买家共鸣的成熟品牌。随着汽车公司推出当前和过去车辆的电动版本，这种情况会一再出现。一辆电动福特 F-150 和一辆电动吉普切诺基。

不是发明一个新的模型线，像雪佛兰试图做的螺栓，公司将期待转换熟悉的模型到电动。这种转变应该有利于更舒适的过渡。对于雪佛兰 Bolt，消费者明白它是电动的，但仍然留下了新的问题。车手如何坐在螺栓里？Bolt 是低端机型吗？一个螺栓三年后的转售价值是多少？有了一款成熟的车型，比如雪佛兰科鲁兹、别克君威或宝马 3 系，很多问题就更容易回答了。消费者熟悉品牌和模型背后的含义。

福特凭借野马 Mach-E 这个名字解决了许多问题。Mach-E 将会很快(确实如此)，它将会很小(这是一款小型跨界车)，而且它的价格将会很有竞争力(确实是 4 万美元)。

例如，其他公司如吉普，本田和斯巴鲁将可能遵循同样的计划。消费者更容易重复使用现有品牌。电动吉普切诺基将是一款功能强大的中档汽车，具有较高的行驶高度、复杂的全轮驱动系统和五人座位。电动本田思域将是一款小型的经济型汽车，而电动斯巴鲁翼豹将在基于汽车的 AWD 平台上提供电动动力系统。

同样，如果 Jeep 或本田或任何其他汽车公司以新的名字将电动汽车推向市场，就会出现不必要的问题。

使用旧名称存在风险。这可能会冒犯铁杆声乐爱好者。福特因为 Mach-E 这个名字而遭到强烈反对。我觉得从汽车人的角度来看这很无礼。那不是野马，我想大喊。退一步讲，很容易看出福特的辩解。

雪佛兰去年重新发布了 Blazer，经历了类似的反抗。运动夹克应该是强壮的越野车，而不是当前版本的小型运动型跨界车。

野马品牌可以说是福特最有价值的资产之一。已经有 50 多年了。汽车公司投入大量资金打造和营销品牌和车型。通常需要几代人的时间才能得到消费者的认可，在这一点上，汽车公司会慎重对待他们。对于野马 Mach-E，福特必须有足够的数据显示预期的成功。

简单的内部

野马 Mach-E 遵循了特斯拉 Model 3 的设计语言:中间是大屏幕，其他没什么。Mach-E 建立在 Model 3 的基础上，通过在驾驶员面前安装一个 LCD 屏幕，使其更容易接受。闪电也做到了这一点，但雪佛兰走得还不够远。随着 Mach-E 和 Model 3 的推出，福特和特斯拉正在利用智能制造技术，这种技术可能会在整个汽车行业复制——无论是好是坏。

每个汽车制造商都制造安装在其所有车型上的开关、刻度盘和屏幕。汽车行话中的零件箱。通常，交换机在不同品牌之间共享。奥迪的开关可能会进入蓝宝，因为这两个品牌都属于大众家族。这样做是为了降低成本。一个车窗开关就是一个车窗开关，为什么要给每个品牌做不同的车窗开关？

Model 3 和 Mach-E 的中央堆栈中没有任何物理按钮。一个巨大的屏幕处理气候控制，媒体播放和更多。福特、特斯拉和其他汽车制造商不再制造和安装大量的小开关，而是用一个屏幕来完成同样的功能。这使得跨品牌和市场的扩展更加容易。突然之间，没有了按钮，汽车公司可以将零件数量、工作时间和故障排除减少到一台设备上。这也使得为不同市场制造汽车变得更加容易。汽车公司只需要重新配置软件，而不是用不同的语言编写或在马路的另一边驾驶。

这种单屏幕设置需要正确的软件，Mach-E 是第一个展示福特 Sync 4 系统的设备。对我来说，它看起来很棒，有持续的气候控制和合理的布局。有功能吗？我还没用过。

谢天谢地，还有一个物理旋钮:音量控制。音量应该总是由旋钮而不是滑杆来控制。一直都是。

这些屏幕允许汽车公司将品牌进一步整合到汽车中。预计林肯车型将使用类似的软件，但与福特车型的设计方案不同。同样，讴歌软件将类似于本田的，但更运动，高档的感觉。

小小的惊喜

Mach-E 有几个惊喜，这要感谢电动平台。

目前市场上的大多数汽车运行的电动系统都是基于一个有几十年历史的系统。这是有限制的，尽管汽车制造商正在尽最大努力推动它。转向电动为设计师和工程师打开了无数的机会。曾经是幻想的功能和细节现在已经成为可能。

Mach-E 没有传统的门把手。相反，它有小按钮来开门。电池没电了会怎么样？在很大程度上，这是非常不可能的，因为 Mach-E 的电气系统不依赖于传统的电池和交流发电机。

Mach-E 有一个基于前端的存储区域——如果你愿意的话，是一个 frunk。这些是大多数电动汽车的标准功能。然而，福特做了一些新奇的事情，他用塑料制造了 Mach-E 的发动机，并增加了一个排水塞。这使得主人可以在冰箱里装满冰块并储存饮料。

有了电动汽车，汽车制造商可以打开他们的剧本，实现品牌特有的功能。吉普应该得到更多的 Jeep-ie。雷克萨斯车型应该能够与丰田车型区别开来，等等。电动平台从根本上比内燃机系统更简单，让工程师和设计师在物质享受方面更具创造性。

这些基于新平台的新功能的缺点将在几年后出现，届时必须进行修复。大多数车主不太可能像现在的汽车一样能够诊断和修理系统。相反，车主将不得不依靠汽车维修服务中心，这些服务中心看起来更像是 IT 商店，而不是车库。

现在，这是我的老雪佛兰伏特的一个问题。我住在密歇根州，Volt 就是在那里设计和制造的。我把它带到全国最大的一家雪佛兰经销店进行维修，那里只有一名技术人员有资格维修这辆车。修理花费的时间比我希望的要长得多，而且随着更复杂的汽车上市，这个问题可能会变得更加复杂。汽车技术人员必须接受再培训，低成本的第三方服务中心可能会落后于昂贵的经销商。

这是一个电动的未来

野马马赫-E 是一个永远做对了。福特从特斯拉、雪佛兰和其他公司的首批量产电动汽车中吸取了教训。乍一看，Mach-E 的命名是非传统的，但仔细观察就会发现它的逻辑。内部令人兴奋，但可扩展，该模型充满了惊喜，将高兴和可能挫败业主。

最终，汽车领域将充满与 Mach-E 相似的车型:回收品牌，更高的行驶高度和围绕简单内饰的独特功能。

有人说，转向电动汽车将会产生陈旧、相似的汽车。这可能是短期结果，因为汽车制造商希望利用熟悉的形状和品牌来吸引消费者。一旦消费者适应了电动汽车，真正的乐趣就开始了。期待下一代野马马赫-E 是更值得野马的名字。
