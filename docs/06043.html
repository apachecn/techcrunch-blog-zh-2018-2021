Twitter 向全球所有用户推出“隐藏回复”功能 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2019/11/21/twitter-rolls-out-its-hide-replies-feature-to-all-users-worldwide/

Twitter 的激进“隐藏回复”功能是自 Retweet 发明以来 Twitter 工作方式的最大变化之一，现在 Twitter 的全球用户群都可以使用。该公司表示，该功能将于今天向所有平台的 Twitter 用户推出，自早期测试以来只有一处轻微调整。

隐藏回复是 Twitter 迄今为止最具争议的功能之一，旨在通过让发帖人重新控制对其推文的回复保持可见来平衡 Twitter 上的对话。当用户选择隐藏回复时，实际上不会从 Twitter 上删除任何回复，它们会被放置在一次额外点击的后面。也就是说，不着边际的、不相关的、侮辱性的或其他不愉快的评论不会主导谈话。

Twitter 的想法是，如果人们知道仇恨言论和不当行为可以隐藏起来，这将鼓励更多的在线文明。

然而，另一方面，人们可以使用“隐藏回复”功能来压制批评者或压制异议，即使是在有正当理由的情况下——比如有人提供事实核查。

这项功能于 7 月首先在加拿大测试，然后于今年 9 月在美国和日本测试，在 T4【通过网络和移动平台】测试。

自推出以来，Twitter 发现大多数人都会隐藏他们认为不相关、跑题或令人讨厌的回复。它还发现，人们使用这种方法，而不是更严厉的降噪控制，如屏蔽或静音。在加拿大，27%的被调查用户隐藏了他们的推文，他们说他们会重新考虑他们将来如何与他人互动，这是一个有希望的指标。

然而，随着它在全球的首次亮相，这个功能有了一点小小的改变。Twitter 表示，一些人在隐藏回复后想采取进一步的行动，所以现在它将检查他们是否也想阻止回复者。它还听到一些用户说，他们害怕报复，因为图标仍然可见。目前还没有在这方面做出改变，但仍在考虑如何解决这个问题。

当新功能首次推出时，Twitter 上经常提到的另一个问题是，当用户遇到带有隐藏回复的推文时，会出现一个大的弹出通知。

一些人发现通知太大了，具有破坏性，这实际上鼓励人们更多地关注隐藏的回复。

然而，Twitter 表示，这个屏幕只在 Twitter 用户第一次看到带有隐藏回复的推文时显示。之后，一个图标会显示人们的回复是隐藏的——这些回复隐藏在另一个页面上，而不是在推文下面。

尽管这是一次性通知，但它要求用户给予的关注超过了它试图传达的信息——本质上，Twitter 已经推出了一个新功能，在这里可以找到它。如果有人正在钓鱼，被告知这个特定的 Twitter 用户正在隐藏回复，可能会让他们更加愤怒。

除了全球推广之外，Twitter 表示，它将很快在其 API 中推出一个新的隐藏回复端点，以便开发人员可以建立更多的对话管理工具。

Twitter 指出，它将测试对话的其他变化，包括关于谁可以回复甚至看到特定对话的更多选项，以及旨在鼓励更健康对话的参与度变化。

“每个人在 Twitter 上交谈时都应该感到安全和舒适，”Twitter 产品管理总监 Suzanne Xie 写道，她最近通过收购加入了。“为了实现这一点，我们需要改变我们服务中的对话方式，”她说。

Twitter 在这一领域的发展很有趣，因为它正在积极尝试各种方式，在一个以热点、讽刺、讽刺和愤怒闻名的平台上鼓励文明。随着时间的推移，它愿意改变和发展它的功能，因为它知道什么是有效的，并放弃那些无效的改变。它甚至与 Twitter 同时运行一个测试产品(twttr ),以尝试新的想法。如果 Twitter 能够通过其功能集扭转局面，那将是产品管理的一个奇迹。

从今天开始，隐藏回复选项将在全球 iOS、Android、Twitter Lite 和 twitter.com 上推出。
