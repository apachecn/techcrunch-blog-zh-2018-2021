反机器人初创公司 Kasada 从中情局 In-Q-Tel 风险基金 A 轮融资 700 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2019/11/26/kasada-series-a/

反机器人初创公司 Kasada 从中情局 In-Q-Tel 风险基金 A 轮融资 700 万美元

Kasada 是一家反机器人初创公司，我们在今年早些时候对其进行了简介，它已经在首轮融资中筹集了 700 万美元，智能社区的非营利风险投资机构 In-Q-Tel 是这家初创公司的最新投资者。

这家总部位于悉尼和芝加哥的公司使用其专有的反机器人平台 Polyform 来帮助打击在线机器人。

机器人不仅用垃圾流量打击网站，试图让它们离线，它们还被用来自动进行欺诈性购买，并从航空公司和艺人等网站抓取内容，试图降低价格。僵尸工具每年都要花费企业不必要的网络服务器和带宽成本。

该公司的反机器人平台在客户网站的外部边缘提出了一个无法解决的加密挑战。该平台使用指纹技术来确定访问者是否是人类。如果平台检测到一个机器人，它会欺骗机器人尝试解决不可能的数学难题，在机器人运行的服务器上搅动服务器和内存资源，并在过多的云资源中消耗机器人运营商的成本。

Kasada 的首席执行官和联合创始人萨姆·克罗泽表示，风投公司的支持是对其技术和团队的“强有力的认可”。

这家于 2015 年成立的公司表示，在过去一年中，其工程和面向客户的团队增加了一倍，并任命前现场运营高管帕斯卡尔·波德温(Pascal Podvin)帮助扩大其收入业务。

In-Q-Tel 的 Peter Tague 说他对 Kasada 的技术“印象深刻”。这也标志着 In-Q-Tel 在去年年底在悉尼开设办事处后在澳大利亚的第一笔投资。迄今为止，In-Q-Tel已经投资企业数据云平台 Cloudera、网络安全巨头 FireEye、开源数据库制造商 MongoDB 和监控软件公司 Palantir。

这一轮 700 万美元的融资将帮助 Kasada 发展和扩大其客户群，因为该公司面临来自竞争对手的日益激烈的竞争。此次融资紧随网络和内容交付巨头 Cloudflare 推出其自己的反机器人“战斗模式”功能之后，该公司表示，这将有助于“挫败”机器人瞄准和攻击其客户。

克罗泽说，Cloudflare 自己的努力是对蚂蚁机器人服务重要性的“证明”，但他说 Kasada 覆盖了其他人没有覆盖的领域。

已更正，说明 In-Q-Tel 是最新投资者，但未领投。
