森林管理员筹集 700 万美元帮助您建立管理面板

法国初创公司 Forest Admin 已经从 Runa Capital 和 Runa Capital 获得了 700 万美元的 A 轮融资。该公司让您建立后端管理面板，使您的团队可以跟踪和操纵数据从一个单一的界面。

Forest Admin 集成了您的数据库和您可能已经使用的第三方服务，如 Stripe、Intercom、Zendesk、Google Analytics、Mailchimp 等。这样，您可以集中所有数据。

之后，Forest Admin 允许您使用最重要的指标构建仪表板。您可以浏览数据并创建段。最后，您可以为用户或细分市场创建工作流，例如生成发票、发放退款、将数据导出为 CSV 文件等。

鉴于林管理员处理敏感的客户数据，您希望确保您的数据是安全的。虽然 Forest Admin 是一家软件即服务初创公司，但它不会从服务器中提取数据。该公司要求你在你的服务器上安装一个插件，以便林管理可以直接与你的服务器通信。

如果你想确保它不会产生任何漏洞，你可以创建一个 IP 白名单，并把森林插件放在 VPN 后面。同样，林管理允许您管理用户权限，以确保员工只能查看他们需要查看的内容。

到目前为止，Forest Admin 已经成功吸引了 2000 名客户。这家初创公司现在有 20 名员工。该公司与不同垂直行业的客户合作，如金融科技、移动和在线零售。

在今天的一轮融资中，Forest Admin 希望与更多的 API(如 GraphQL)合作，与更多的第三方 SaaS 工具集成，并为移动应用优化管理面板。
