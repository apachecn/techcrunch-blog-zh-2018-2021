Nix Pro 和 Nix Mini 彩色传感器评论

原文：https://web.archive.org/web/https://techcrunch.com/2019/11/28/nix-pro-2-and-nix-mini-color-sensors-are-powerful-easy-to-use-additions-to-any-creative-pro-toolkit/

Nix Pro 2 和 Nix Mini 颜色传感器是任何 creative pro 工具包中功能强大、易于使用的附件

将物理世界转化为数字世界一直是一个挑战，特别是当涉及到颜色这样的东西时:颜色实际上并不是静态的东西，我们使用的无数显示器和相机可以用非常不同的方式来表现它们。但初创公司 Nix 的新设备可以帮助消除这种困惑——Nix 系列颜色传感器，包括易于使用的 Nix Mini(通常为 99 美元)和更复杂的 Nix Pro 2(通常为 349 美元)。

这些设备在使用和构造上看似简单，但有一项工作做得非常好——幕后进行了大量工作使之成为可能。他们的主要目的是给你一个模拟颜色的数字解释，这可能是你遇到的任何表面的颜色。为此，他们在一个菱形塑料外壳内放置了一个小镜头和传感器。Mini 和 Pro 都很容易放入口袋，但 Mini 的直径大约是一枚大硬币的直径，而 Pro 的感觉更像是手中的高尔夫球。

这些传感器包括通过微型 USB 充电的内置电池，由于它们在工作时并不真正使用那么多电力，所以一个充满电的电池可以进行大约 3000 次单独的表面扫描。使用它们也非常容易:你下载一个应用程序(Nix 有三个，包括一个用于数字色彩捕捉，一个用于绘画，一个用于专业用户，提供对专业油漆店和其他应用程序有用的附加信息)，配对其中一个设备(一旦充电，它们应该会自动显示)，然后点击应用程序中的一个按钮来扫描表面，将 Nix 举到所述表面。

这个过程非常快，并根据你使用的应用程序提供不同的结果。在 Nix Paints 应用程序中，一旦你选择了你喜欢的品牌，它会给你最接近的现成匹配，如果你正在做修补工作或重新粉刷你房子的一部分，这是很好的。你也可以得到一个免费的调色板或其他匹配的颜色来重新装修。“数字”应用程序可以让你看到你在网页或数字产品设计中使用的所有十六进制和其他值，还可以为项目工作构建调色板。

Nix Pro 提供了一系列颜色读数，供专业人员用于超精确的匹配和测量，您可以再次使用内置的颜料库进行精确匹配，或使用颜色值根据您的规格进行批量定制混合。

Nix 可以扫描几乎任何表面，包括所有类型的油漆和织物，以及瓷砖和其他地板。这是一个附件，它确实使过去相当混乱的过程变得快速而无痛，而且它也是开箱校准的，所以用户在实际使用时不必做任何事情来确保颜色的准确性。
