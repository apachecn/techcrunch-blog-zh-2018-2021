UniFi 梦想机器路由器是网络发烧友的绝佳切入点

原文：https://web.archive.org/web/https://techcrunch.com/2019/12/02/the-unifi-dream-machine-router-is-a-great-entry-point-for-networking-nerds/

几周前，泛在科技发布了 UniFi Dream Machine，这是一款售价 299 美元的多功能网络设备，结合了一台路由器、一台带有四个以太网端口的交换机和一个 Wi-Fi 接入点。它有一个泛在提所说的集成云密钥，可以让你控制你的网络。

过去几周，我一直在我的家庭网络上使用 UniFi Dream Machine，所以这是对该设备的回顾。

泛素是一个著名的网络品牌。大多数人都熟悉公司的接入点——在学校、公司和公共场所随处可见的圆形天线。

但是对于个人用例甚至小公司来说，前期投资总是有点高。UniFi 梦想机器完美地介于专业设备和消费设备之间。如果您使用的路由器具备互联网服务提供商提供的 Wi-Fi 功能，这意味着一次巨大的升级。

UniFi 设备的再处理

Ubiquiti 在 AmpliFi 品牌下为寻求即插即用解决方案的消费者提供了一系列路由器。该公司最近宣布了一款新设备,如果你不想改变网络设置，这款设备将会有很好的规格。

但是如果你正在阅读这篇文章，你很可能知道 UniFi 产品提供了一些你认为消费产品所缺乏的定制。

从一体化网络设备切换到 UniFi 系统总是有点复杂。该公司已将网络堆栈分解为不同的设备，以便为您提供更多控制。

这意味着你必须购买一个安全网关(路由器，网络的“大脑”)、一个交换机(就像一个插线板，但用于以太网端口)和一个接入点(Wi-Fi 天线)。最重要的是，如果你想用该公司的控制器软件管理你的网络，统一的云密钥是必不可少的。

如果你致力于 UniFi 生态系统，你会得到很好的体验。您可以单独管理交换机上的每个以太网端口，您可以从世界任何地方控制 Wi-Fi 设置以及许多许多其他事情。Ars Technica 的 Lee Hutchinson 掉进了 UniFi 兔子洞，写了一篇关于他在家里运行专业网络设备的经历的精彩故事。

UniFi 梦想机器采取了不同的方法。它重新组合了所有独立的部分，使 UniFi 网络变得栩栩如生。你可以购买 300 美元的 UniFi 梦想机器，控制你网络的每一个细节。

规范

简单说一下 UniFi 梦想机的规格。这个药丸形状的设备有一个集成的安全网关，可以让你运行 DHCP 服务器，创建防火墙策略，利用多个 VLANs 等等。

除了将您的设备连接到互联网的 WAN 端口之外，还有四个千兆以太网端口。至于 Wi-Fi，梦想中的机器支持 802.11 AC Wave 2(“Wi-Fi 5”)，带有 4×4 MU-MIMO 天线——遗憾的是没有 Wi-Fi 6。

在幕后，该设备使用 1.7GHz ARM Cortex-A57 处理器。它有 2GB 的内存和 16GB 的存储空间，功耗高达 26W。

使用梦想机器

设置 UniFi 梦想机器是一次很棒的体验。理想情况下，您希望将以太网电缆插入 ISP 提供的路由器，并将其置于桥接模式。这样，它将作为一个哑调制解调器，让 UniFi 梦想机做所有的艰苦工作。

下载手机应用并打开 UniFi Dream Machine 后，你会得到一个模仿 AirPods 配对弹出窗口的弹出窗口。然后，您可以从该移动应用程序或使用电脑上的 web 浏览器来控制您的网络。

这是有趣的时候。

UniFi 的控制器软件通常会列出网络上当前运行的所有 UniFi 设备。有了 UniFi Dream Machine，你就有了一台设备。但是如果您展开该设备，您可以看到一个由三个独立的 UniFi 组件组成的列表——一个网关、一个交换机和一个 AP。

正如所料，你可以控制你的网络的每一个细节。同样，这并不适合所有人，为了优化你的设置，你必须学习很多关于网络的东西。但是如果你是一个数字修补者，这是一股新鲜空气。

UniFi 梦想机在我家充当 DHCP 服务器。我重命名了我的设备，并为我的所有设备分配了固定 IP，以便更容易找到它们。你可以实时看到他们正在使用的网络，以及他们的 Wi-Fi 信号是否良好。

我还在网络级别配置了 Cloudflare 的1.1.1.1公共 DNS。

如果你关心安全的话，有很多可能性。我创建了一个访客 Wi-Fi 网络，只允许我的朋友访问互联网。他们可以毫无问题地浏览 Twitter 和流式播放网飞的节目，但是他们不能通过本地网络访问我的电脑。

我还为物联网设备创建了另一个 Wi-Fi 网络，如连接的扬声器、打印机和机器人吸尘器。联网设备不会获得很多安全补丁，并且比你保持更新的计算机或智能手机有更多的漏洞。我为此 Wi-Fi 网络分配了不同的 VLAN。VLANs 允许您创建具有不同规则集的分区网络。

我对这个 VLAN 应用了防火墙规则，这样我就可以从我的个人设备上控制这些设备，但是它们不能自行向我的设备发出请求。这对于大多数人来说是大材小用，但你可以从 UniFi 的控制器上做到这一点，这很有趣。更多详情此处。

说到 Wi-Fi，一切都是可定制的，性能也是一流的。我住在一个小公寓里，但阳台一直是个问题。我经常在阳台上工作，我一直在使用一个便宜的 Wi-Fi 扩展器，这是我在一个装有小工具和电缆的盒子里找到的。

我拔掉 Wi-Fi 扩展器，尝试连接到 UniFi 梦想机。我获得了更好的性能，即使我将 Wi-Fi 发射功率降低到中等。

这些只是你可以用 UniFi 梦想机器做的事情的几个例子。我觉得我仍然没有充分利用设备(你可以通过 SSH 连接，并从终端控制一切)，但我不会考虑回到具有 Wi-Fi 功能的入门级路由器。

针对潜在消费者

UniFi 梦想机器是我不知道自己想要的联网设备。我永远不会有数百个 Wi-Fi 设备连接到我的家庭网络。我不需要一打以太网端口。然而，我想控制我的网络。如果你想念 T2 苹果的 AirPort Extreme 或者你是一个网络迷，你应该考虑 UniFi 梦想机。

小企业和商店在公司成立之初经常会做出一些糟糕的决定。当你的企业规模扩大时，亚马逊上的廉价 Wi-Fi 路由器并不能满足你的需求。梦想机器可以是一个很好的切入点，因为你将能够建立在基础设备。

但是如果你认为你有更大的需求，就不要试图从一个 UniFi 梦想机器上运行一个大网络。Ubiquiti 出售一些很棒的机架式设备，可以给你更多的灵活性。UniFi 梦想机毕竟是一个受约束的机器。这就是为什么它对企业客户来说不够好，对专业消费者来说却很棒。
