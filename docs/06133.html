AWS 宣布推出由机器学习驱动的新企业搜索工具

今天在拉斯维加斯举行的 AWS re:Invent 上，该公司宣布了一款名为 Kendra 的新搜索工具，该工具使用机器学习在各种内容库中提供自然语言搜索。

AWS 人工智能副总裁马特·伍德(Matt Wood)表示，新的搜索工具使用机器学习，但实际上不需要任何类型的机器学习专业知识。亚马逊正在为幕后顾客解决这个问题。

您首先要确定您的内容存储库。这可以是从 S3 存储库到 OneDrive 到 Salesforce 的任何内容，也可以是存储内容的任何地方。您可以使用来自 AWS 的预建连接器，提供您的凭证并连接到所有这些不同的工具。

然后，Kendra 根据它在连接的存储库中找到的内容建立一个索引，用户可以开始使用自然语言查询与搜索工具进行交互。该工具理解时间等概念，因此如果问题类似于“IT 服务台何时开放”，搜索引擎会理解这是关于时间的问题，检查索引并向用户提供正确的信息。

这个搜索工具的美妙之处不仅在于它使用了机器学习，而是基于用户的简单反馈，如笑脸或悲伤表情，它可以学习哪些答案是好的，哪些需要改进，并且它会自动为搜索团队完成这些工作。

一旦你设置好了，你可以把搜索放在你公司的内部网上，或者你可以在一个应用程序内部使用它，它会像你所期望的搜索工具那样工作，具有像提前输入这样的功能。
