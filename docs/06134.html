AWS 推出定制推理芯片

在其 re:Invent 会议上，AWS 今天宣布推出其推理芯片，该芯片最初于去年发布。这些新芯片有望使推理，即使用你之前预训练的机器学习模型，明显更快，更具成本效益。

正如 AWS 首席执行官安迪·杰西(Andy Jassy)指出的那样，许多公司都专注于让你训练模型的定制芯片(尽管谷歌和其他公司肯定不会同意这一点)。推理在常规 CPU 上运行良好，但定制芯片显然会更快。与 EC4 上的常规 G4 实例相比，通过推理，AWS 提供了更低的延迟和三倍的吞吐量，并且每次推理的成本降低了 40%。

新的 Inf1 实例承诺多达 2，000 个 top，并具有与 TensorFlow、PyTorch 和 MXNet 的集成，以及用于在框架之间移动模型的 ONNX 格式。目前，它只在 EC2 计算服务中可用，但它很快也会出现在 AWS 的容器服务和 SageMaker 机器学习服务中。
