M-Pesa给非洲新的风险投资丰富的金融科技创业公司带来的教训

原文：https://web.archive.org/web/https://techcrunch.com/2019/12/04/lessons-from-m-pesa-for-africas-new-vc-rich-fintech-startups/

在非洲金融科技，2019年第四季度为新进入者带来了大笔资金。

中国投资者向 OPay 和 PalmPay 投资了2.2亿美元——这两家初创公司计划在尼日利亚和更广阔的大陆扩大规模。几名消息人士告诉我，大笔资金已经让尼日利亚不少拥有类似策略、资金规模较小的支付企业感到焦虑。然而，他们可能还不需要担心:从非洲最成功的移动货币案例研究M-Pesa中得到的教训表明，仅靠风投是无法在数字金融领域获得规模的。

非洲的创业公司和金融科技

在过去的十年里，非洲一直处于创业热潮中，伴随着风险投资的大幅增长以及互联网和手机普及率的提高。

一些明确的公司组建、技术中心和投资中心已经出现；尼日利亚、南非和肯尼亚在所有这些类别的人数上都领先于非洲大陆。在非洲的54个国家和12亿人口中，加纳、坦桑尼亚、埃塞俄比亚和塞内加尔是创新和创业的新优势。

2018年，非洲大陆对创业公司的风险投资超过了10亿美元，根据 Partech 和 WeeTracker 的研究，金融科技是大部分资本和交易流的焦点。

根据多项估计，非洲拥有世界上最大比例的无银行账户人口和银行账户不足人口。

这与该地区远离电网的中小企业和经济活动并行不悖——从内罗毕到拉各斯，这些活动通过街头商贩、路边摊和露天市场进行展示和商业活动。

国际货币基金组织估计，非洲的非正规经济是世界上最大的经济体之一。成千上万的金融科技初创公司涌入了这个庞大的无银行账户和排名不足的公民和中小企业群体，寻求增加数字金融产品和市场份额。

在这场竞赛中，西非国家尼日利亚——非洲最大的经济体和人口所在地——正在成为风险投资的中心。许多金融科技相关公司正在采取先扩大规模再向外扩张的策略。

输入PalmPay和OPay

这包括新进入者 OPay 和 PalmPay ，它们在2019年第四季度筹集了如此多的资本。值得注意的是，这两家公司都成立于2019年，主要由中国演员孵化。

面向消费者的支付产品PalmPay于11月上线，获得4000万美元的种子期(2019年非洲最大的种子期之一)，由非洲最大的手机销售商中国转运领投。这家初创公司直言不讳地表达了自己的雄心，在一份公司声明中称其目标是成为“非洲最大的金融服务平台”。

为此，PalmPay方便地与其主要投资者建立了战略合作伙伴关系。这家初创公司的支付应用将预装在Transsion的移动设备品牌上，如非洲的Tecno，预计2020年将覆盖2000万部手机。

PalmPay也于11月在加纳推出，其英国和非洲首席执行官格雷格·里夫(Greg Reeve)证实，计划在2020年扩展到其他非洲国家。

如果说PalmPay的4000万美元种子轮引起了创始人的注意，OPay的1.2亿美元B轮融资则产生了冲击波，就在几个月前，这家基于移动设备的金融科技风险投资公司筹集了5000万美元——这使得OPay的1.7亿美元资本约相当于2018年非洲所有风险投资的五分之一。

OPay由中国消费者互联网公司 Opera 创建，由9名中国投资者支持，是Opera在尼日利亚开发的一套基于互联网的商业产品的支付工具，包括乘车应用ORide和OCar以及食品配送服务OFood。

通过最新的A系列，OPay宣布将在肯尼亚、南非和加纳扩张。

在尼日利亚，OPay在几个月内宣布的1.7亿美元的A轮和B轮融资让新老金融科技公司筹集的资金相形见绌，除了Interswitch。

这家本土支付处理公司是尼日利亚大部分数字金融基础设施的先驱，去年11月，Visa收购了该公司2亿美元的少数股权，成为独角兽。

尼日利亚支付风险企业更常见的融资金额包括老牌金融科技公司 Paga的1000万美元B轮。最近的市场进入者Chipper Cash的2019年5月种子轮投资为240万美元。

尼日利亚金融科技初创公司的融资额分别为1美元和数千万美元，而OPay和PalmPay的融资额分别为4000万美元和1.2亿美元。传统智慧可能认为，在尼日利亚和其他市场，大资本、大支出的公司在扩大数字支付规模方面具有明显的优势。

看看肯尼亚的M-Pesa也许会证明事实并非如此。
