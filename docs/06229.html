据报道，中国已下令在未来三年内更换所有外国个人电脑硬件和操作系统，加剧了一场正在进行的技术战争。该国以前曾半心半意地尝试过这种事情，但这是迄今为止最认真的努力，以使自己免受西方技术部门的影响。

根据金融时报援引中国科技分析师的报道，今年早些时候，这一命令来自中国政府高层。我们的目标不仅仅是用中国的同类产品取代美国和欧洲的软件和操作系统，还包括它们运行的硬件。

中国此前曾下令清除西方软件，但这些行动较为有限，或者与某些安全问题有关；五年前曾有过让这个国家脱离安卓和 Windows 的努力，但最终都以失败告终。

这一次很可能会有所不同。至少可以说，美国和中国之间的关系已经变得紧张，特别是在科技领域，两国已经从真诚的对手变成了真正的对手。美国最近采取行动，禁止中兴和华为等一些大型中国硬件供应商在美国基础设施中使用(华为称该禁令“违宪”)，其他各种政策决定也加大了裂痕。

因此，这一命令的明显决定性就不足为奇了。据报道，目标是到 2020 年底替换 30%的计算机和软件，2021 年替换另外 50%，2022 年替换剩下的 20%。

至少可以说，为期三年的“3-5-2”计划是雄心勃勃的。数以千万计的设备需要更换，但这并不像用惠普机器换中国制造的那么简单。组件和软件也必须是中国的，所以英特尔和 AMD 处理器出局了，Nvidia GPUs，ARM 架构，索尼图像处理器等等也是。

然而，这看起来并不令人震惊，因为许多中国公司多年来一直在为这种可能性做准备。中国已经非常明确地表达了独立于美国公司的愿望，许多国有企业已经有一段时间无法使用美国供应商。

即便如此，中国的类似 Windows 和 Android 的产品还远没有达到成熟的程度，也没有足够的开发者支持来替换它们而不承担任何后果。这项禁令可能会阻碍其他重大努力，如该国主导人工智能生态系统的努力。如果中国政府支持的研究人员不能使用与世界其他地方的学术和私人同行相同的工具，他们的成果几乎肯定会受到影响。

该计划的细节仍然保密，但随着计划开始实施，可能会慢慢透露出来。但随着供应商、开发商和制造商都在学习如何驾驭不同的市场，这可能会成为未来几年行业动态的主要驱动力。
