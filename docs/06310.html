专注于儿童的 STEM 设备初创公司 Kano 因搁置迪士尼电子设备而面临裁员

原文：https://web.archive.org/web/https://techcrunch.com/2019/12/16/kid-focused-stem-device-startup-kano-sees-layoffs-as-it-puts-disney-e-device-on-ice/

总部位于伦敦的 STEM 设备制造商卡诺(Kano)证实，该公司正在裁员，并称这是将重心转移到“教育计算”的重组努力的一部分。

裁员——从 65 人减少到 50 人——早些时候由电讯报报道。Kano 创始人 Alex Klein 在与 TechCrunch 的电话中证实，Kano 明年将有 50 名员工。虽然他说，以儿童为重点的学习编码设备业务也增加了工程和设计方面的工作，并随着重心的转移而减少了其他职位。

他还表示，一些削减是季节性和周期性的——与度过假期有关。

根据克莱恩的说法，随着该公司从树莓 Pi 平台(2013 年的起点和众筹的 DIY 电脑)转向基于 Windows 的学习平台，工作机会正在减少。

他指出的与裁员有关的其他因素包括在中国建立一个新的制造机构，有一个“更简单、更大的合同制造商”；更少的实体零售店需要支持，卡诺更依赖亚马逊(他说亚马逊“支持起来更便宜”)；减少对大型合作伙伴和代理机构的依赖，Klein 声称 18%的美国 6-12 岁孩子的父母现在熟悉该品牌，减少了其营销费用；以及减少公司经理与制造商数量的愿望，因为“我们看到我们的第一方 Kano 产品——计算机套件、像素套件、运动传感器套件——今年的反应比预期的更强烈”。

“我们带来了一些更专注于这个新平台(Kano PC)的角色，一些专注于树莓 Pi 的角色不再与我们在一起，”他还告诉 TechCrunch。

Kano 在今年秋天发布了第一台基于 Windows 的个人电脑。这款 11.6 英寸的触摸屏电脑售价 300 美元，与谷歌的 Chromebook 价格相当。

这家科技巨头一直保持着对教育计算市场的稳定关注，这给像 Kano 这样试图开拓销售自有品牌 STEM 硬件业务的较小参与者带来了竞争压力。针对谷歌巨人，克莱恩吹捧 Kano PC 的相对可修复性和对计算性能的关注等因素(他声称 Kano PC“与 Surface Go 不相上下”)，此外现在还与竞争对手微软(Microsoft)站在一起。

“我们越来越多地进入学校环境，越来越多地与北美主要的学校分销商交谈，我们越来越多地看到人们想要“DIY”产品设计，他们想要工具包的可攻击性和可扩展性，他们想要工具是开源和可操作的，但他们也希望能够运行 Photoshop、运行 Class Dashboard 和运行 Microsoft Office。因此，我们与微软达成了合作关系，”克莱恩说。

“Windows 电脑为教师提供了丰富的内容和课程，并与微软团队进行了整合，这需要不同类型的开发能力，”他补充道。

“我们正在增加的角色围绕着订阅，他们围绕着计算机，为计算机构建新的应用程序和工具，并继续丰富我们的成员现在可以使用的项目数量——所以我们正在做的事情包括允许人们将他们魔杖中的传感器连接到家用物联网设备。我们将在圣诞节期间推出一款新的协作绘图应用。”

根据 Klein 的说法，Kano“已经看到对其基于 Windows 的 PC 的需求在下一个日历年达到 6 万台”，他说这“远远超出了我们的预期……考虑到价格点”。

尽管他没有给出迄今为止 Kano 个人电脑的确切销售数字。

他还证实，卡诺将在明年缩减产品种类。

最近有消息称，卡诺在 2016 年首次推出的自有品牌摄像设备将不会发货。克莱恩还告诉我们，他们计划在 2020 年推出的另一款联合品牌迪士尼产品被“推迟”——目前还没有新的发布日期。

克莱恩否认销售疲软——声称目前的 《星球大战》和《冰雪奇缘》电子产品已经“为我们做得够多了”。(据克莱恩说，虽然联合品牌的哈利波特电子魔杖卖得比预期的快，但他们原本预计 3 月份会有货，但现在“卖完了”。)

“我们所做的重组与增长和用户无关，”他告诉我们。“我们有望在本财年以更高的平均售价销售更多的设备和产品。我们预计一直到三月份都会有货，但我们的魔杖已经卖光了。我们对 Kano PC 的预发布需求超过了以往任何时候。”

关于被推迟的额外的联合品牌迪士尼电子产品——现在可能根本不会在明年推出——克莱恩告诉我们:“事实是我们正在与迪士尼就此进行谈判——以及时间安排。鉴于我们不确定我们是否会在 2020 年这样做，我们带来的一些承包商角色，特别是做许可签署件，围绕这些品牌开发一些内容，一些为管理这些合作关系而设置的设备，我们不再需要了。”

“我们今年推出了三种新的硬件 SKU。我不认为我们明年会推出三种新的硬件产品，”他补充道，并证实了我们的意图是在 2020 年削减设备发布数量，以专注于 Kano PC。

我们采访的一位消息人士暗示，卡诺正考虑彻底放弃其合作伙伴战略。然而，克莱因在给我们的评论中并没有走那么远。

“几年来，我们一直骑在一只熊身上。我们跳到一只新的熊。这总会让人有点兴奋。但我认为这是一个真正有希望的地方，”他是这样表述这个支点的。

“我认为 Kano 比世界上任何人都做得更好的是围绕技术打造一种体验，向更广泛的受众开放其属性，”当被问及硬件或软件是否将是其未来的主要关注点时，Klein 也表示。“硬件元素至关重要且美观，我们制造了一些世界上最有趣的动态物理产品。这是一个经常被讲述的故事，硬件是非常艰难和残酷的——是的，因为如果你做对了，你就改变了社会的结构。

“我很难在企业的硬件和软件之间划清界限，因为我们总是被问到这个问题，进入这个行业七年后，我们发现人们用产品做的最伟大的事情…总是在两者结合的时候。因此，我们很自豪我们善于将两者结合起来，我们将继续这样做。”

STEM 设备领域近年来经历了坎坷，因为早期的宣传和投资未能转化为持续的收入。

这个类别肯定充满了挑战-从低门槛进入导致大量(如果质量不同)竞争，到为(善变的)孩子构建安全、强大和有吸引力的产品的需求，这些产品紧密可靠地集成了硬件和软件，到检查所有相关的盒子和流程以赢得教师并支持学校的课程要求，这对于直接向教育市场销售是必不可少的。

鉴于对 STEM 设备制造商的需求如此之多，今年出现许多这样的初创公司退出其他公司和/或更大的电子制造商也就不足为奇了——比如 Sphero 收购 littleBits 。几年前sp hero 通过自己的支点远离销售联合品牌迪士尼“学习编码”小发明，以放大教育空间。

而另一家总部位于英国的 STEM 设备制造商——pi-top——最近也经历了到 几轮裁员，这显然是其转向美国教育技术市场的一部分。

这一类别很有可能出现更多的整合。鉴于 Kano 和微软之间的新关系，通过收购加入 Redmond 可能是这家初创公司的明显终点。

根据《每日电讯报》的报道，卡诺正在寻求筹集更多资金。然而，当被要求证实该公司的资金状况时，克莱恩没有发表评论。

这家初创公司上一次报告融资是在两年前，当时它完成了由 Thames Trust 和 Breyer Capital 牵头的 2800 万美元的 B 轮融资。Index Ventures、斯坦福工程风险基金、LocalGlobe、Marc Benioff、John Makinson、Collaborative Fund、Triple Point Capital 和 Barclays 也参加了会议。
