播客的美妙之处在于任何人都可以做。这是一种罕见的媒介，制作起来几乎和消费起来一样容易。因此，没有两个人会以完全相同的方式做这件事。有大量的硬件和软件解决方案对潜在的播客开放，所以设置涵盖了从NPR T2工作室到USB Skype设备的所有范围。

三季以来，代言传媒的《家族幽灵》探索了困扰家族几代人的深邃、黑暗、真实的奥秘。节目创作者山姆·丁曼是一位蛾大满贯得主，他也是流行播客“金钱不好”和“你必须记住这个”的制作人

我在2009年爱上了播客，当时我在一家命运多舛的软件公司(RIP LimeWire)担任客户支持代表，经历了一段离奇的职业生涯。我的工作是接电话，告诉那些因非法下载音乐(即色情音乐)而感染病毒的愤怒用户，我们不退款。播客是一种受欢迎的缓解愤怒的方式，不久，我鼓起勇气开办了自己的播客。我开始在录音论坛上消磨掉整个工作日(大喊Gearslutz！)在谷歌上搜索广播演播室的照片，对大型振膜电容话筒垂涎三尺，对前置放大器怦然心动。

不幸的是，我只能买得起一个AT2020 USB麦克风——这导致了一系列令人兴奋却难以理解的初始录音:

幸运的是，在接下来的10年里，我学到了很多东西，也通过音乐之神的恩典，不知何故获得了足够的制作预算，在布鲁克林绿点一个舒适的地下室工作室里建造了我自己的工作室。今年早些时候，我和我的朋友艾伦在地下室度过了两个真正无尽的日子，试图破译建造耳语室的指令，在那里我现在记录了“家庭幽灵”的所有叙述，用洋红色熔岩灯的平静存在缓解了我一直担心整个事情会在我头上崩溃的恐惧。

我在LimeWire上的异想天开的谷歌搜索使我相信，通过将Neumann U87麦克风与Great River ME-1NV前置放大器的丰富模拟电路配对，可以实现圣杯式的声音链，因此，当我们打开耳语室的最后一个螺丝时，我就支持了这两者。每次我把诺依曼从木制珠宝盒里拿出来录音时，我都会轻声说“你好，魔法麦克。”大河坐在我的书桌上，有着庄严的黑色旋钮和简朴的增益表，我喜欢它赋予流经它的声音的温暖和细微差别。

当然，在耳语屋录制旁白只是“家庭幽灵”故事的一半——当我不在演播室时，我通常会背着一个装满野外录音设备的背包:一台Zoom H5数字录音机，两个Rode NTG2猎枪麦克风，两个桌面麦克风支架和XLR电缆，一个墙上适配器和延长线，这样我就不必担心在漫长的采访中耗尽Zoom的电池， 一副索尼MDR-7506耳机和一根立体声“-to-⅛”电缆，这让我可以在Zoom上录制高质量的电话采访。

然后，我将所有作品带回绿点的地下室，将音频加载到Pro Tools中，打开lava灯，买些咖啡，痴迷地重新编排波形，直到深夜凌晨，我永远感激自己找到了一种方法，将尖叫的客户服务电话留在身后。
