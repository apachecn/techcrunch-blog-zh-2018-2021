2019年，印度科技创业公司筹集了创纪录的145亿美元

印度的科技创业公司从来没有这么好过。

根据研究公司Tracxn的数据，2019年，美国本土科技创业公司筹集了145亿美元，超过了去年106亿美元的最高纪录。

今年，印度的科技初创公司参与了817名投资者的1185轮融资，其中459轮是首轮或更高轮融资。

根据风险债务公司InnoVen Capital的一份报告，早期创业公司——即参与天使或首轮融资前的初创公司——今年融资69亿美元，轻松超过去年的33亿美元。

根据InnoVen的报告，通常难以吸引投资者的早期创业公司今年参与的融资交易数量同比增长了22%。他们的估值累计达到260万美元，比去年增加了15%。

Tracxn告诉TechCrunch，总体而言，规模在2500万美元至1亿美元之间的融资交易有81起，高于去年的56起和前年的36起，超过1亿美元的融资交易有27起，高于2018年的17起和2017年的9起。

同样在2019年，印度有128家创业公司被收购，4家上市，9家成为独角兽。根据Tracxn的数据，今年，印度的科技初创公司也吸引了创纪录数量的国际投资者。

今年的融资进一步推动了美国蓬勃发展的创业空间走上稳步增长的道路。

自2016年以来，科技创业公司仅积累了43亿美元，低于上年的79亿美元，生态系统中的资金流大幅增加。2017年，印度初创公司融资104亿美元，按Tracxn计算。

“就创业公司筹集的资金总额而言，这十年来，从2010年的5.5亿美元到2019年的145亿美元，增长了25倍，”Tracxn说。

风险投资基金Lightspeed Venture Partners的合伙人戴夫·哈雷(Dev Khare)在最近接受TechCrunch采访时表示，印度初创公司同样有希望的是他们今天开始应对的挑战。

2014年和2015年，初创公司主要专注于构建电子商务解决方案和复制在西方市场奏效的想法。但今天，他们正在处理广泛的类别和机会，并建立一些在任何其他市场都没有尝试过的解决方案，他说。

Tracxn的分析发现，住宿初创公司今年筹集了约17亿美元——仅Oyo 一家就获得了15亿美元，其次是物流初创公司，如 Elastic Run ，Delhivery和Ecom Express获得了6.41亿美元。

此外，今年还有176个横向市场、150多个教育学习应用程序、160多家金融科技初创公司、120多家卡车运输市场、82家叫车服务公司、42个保险平台、33家二手车列表提供商和13家帮助企业和个人获得营运资本担保资金的初创公司。Tracxn表示，仅金融科技初创公司今年就筹集了32亿美元，超过了其他任何类别的初创公司。

投资者

红杉资本(Sequoia Capital)有超过50笔投资——或共同投资——是今年印度科技初创公司最活跃的风险投资基金。(Rajan Anandan，负责谷歌印度和东南亚业务的前高管，于4月加入红杉资本印度公司，担任董事总经理。) Accel 、Tiger Global Management、Blume Ventures和Chiratae Ventures是另外四大风投。

Steadview Capital对初创公司进行了九次投资，包括打车服务Ola 、教育应用Unacademy 和金融科技初创公司BharatPe ，在私募股权基金中处于领先地位。投资了NoBroker 和的general Atlantic，以及最近由ju的转为盈利的edtech初创公司，投资了四家初创公司。FMO、Sabre Partners India和CDC Group分别投资了三家初创公司。

风险投资公司(Venture Catalysts)今年是印度最大的加速器或孵化器，有超过40笔投资，包括对HomeCapital和Blowhorn的投资。Y Combinator、超过25笔投资、红杉资本的飙升、Axilor Ventures和Techstars今年也非常活跃。

今年，印度的科技创业公司也吸引了来自顶级企业和银行的大量直接投资。高盛本月早些时候投资了金融科技初创公司ZestMoney，今年总共进行了八笔投资。其中，脸书对一家印度初创公司进行了第一笔投资——社交商务公司Meesho——Twitter领投了当地社交网络应用ShareChat 的一轮1亿美元融资。
