在 2019 年的现实检查之后，2020 年无人驾驶汽车的前景如何？

原文：https://web.archive.org/web/https://techcrunch.com/2020/01/02/after-2019s-reality-check-whats-ahead-for-driverless-cars-in-2020/

如果 2018 年是行业震惊清醒的一年，那么 2019 年将是实用主义和试图开发和扩展技术的挑战变得更加真实的一年。结果是:这个行业正处于青春期，变化可能会突然发生，导致混乱和尴尬的遭遇。那是一个别人发展远远快于同龄人的时代。

并且在即将到来的一年里，T4 承诺会有同样的事情发生。

自动驾驶汽车行业的裂缝——被不切实际的热情和似乎无底洞的风险和企业资本所掩盖——在 2018 年的头几个月变得太过明显，不容忽视。

Waymo 与优步之间备受瞩目的商业秘密案件揭示了这个新兴行业残酷和鲁莽的一面。仅仅几周后，一辆自动驾驶的优步汽车在亚利桑那州坦佩的公共道路上测试时撞死了一名行人，导致优步立即停止了所有测试。虽然其他公司只是暂时停止了自己的测试，但这一事件给一个渴望让道路更加安全的行业蒙上了阴影。

华而不实的自动驾驶汽车演示慢了下来，一些人向 TechCrunch 抱怨筹集资金的问题。robotaxis 商业部署的时间表变得更加模糊。研究和咨询公司 Gartner 发布了年度炒作周期图，自动驾驶汽车进入了幻灭的低谷。

因此，每年 1 月在拉斯维加斯举行的大型科技展 CES 2019 上的聊天和公告不应让任何人感到惊讶。不管怎样，它做到了。突然之间，或者看起来是这样，流行词汇不再是“无人驾驶”和“机器人轴”，而是“安全”和“高级驾驶辅助系统”，这是一种在新型轿车、SUV 和皮卡中发现的自动化水平较低的系统。

对 ADAS 的重新关注，以及合作伙伴交换、战略交易和行业整合的开始将是 2019 年的首要主题。今年，距离 CES 2020 只有几天了，ADAS 的爱情故事将随着整合和向有限的 robotaxi 部署的缓慢前进而继续。

ADAS 重置行业预期

英伟达首席执行官黄仁勋在 CES 和 2019 年剩余时间里为其中一个主题搭建了舞台，当时他推出了一个名为英伟达 Drive AutoPilot 的 2 级+系统。新产品是作为一个参考平台推出的，汽车制造商和供应商，如大陆和 ZF，将能够使用它来将更多复杂的自动驾驶功能——而不是完全自动驾驶——引入他们的生产车辆。

对一些行业观察人士来说，这一披露就像是当头一棒。

就在两年前，詹森在 CES 舞台上吹捧英伟达的技术将如何在 2020 年实现 4 级自主。第 4 级是汽车工程师学会(SAE)的一个名称，这意味着车辆可以在没有人为干预的情况下处理特定条件下驾驶的所有方面。第二级系统，其中两个主要功能是自动化的，仍然有一个人在循环中的所有时间。

“完全自主的想法在 2019 年有点落后，ADAS 更多地占据了中心舞台，”Edmunds insights 高级经理 Jeremy Acevedo 在最近的一次采访中告诉 TechCrunch。

Nvidia 并不孤单。这种转变部分是由自动驾驶汽车开发商面临的技术和监管挑战推动的。这也是对汽车制造商寻求短期解决方案的需求的回应，这些解决方案将提高消费类乘用车的技术能力。

对英特尔来说，这是一种证明，英特尔在 2017 年花费了超过 150 亿美元收购了 Mobileye，这是一家领先的汽车开发商和传感器系统供应商，可帮助防止碰撞。作为英特尔内部子公司运营的 Mobileye 认为，如果不首先通过更低水平的自动化，就无法达到第 4 级或第 5 级(在任何和所有条件下完全无人驾驶)。

英特尔高级首席工程师兼 Mobileye 自动驾驶汽车标准副总裁杰克·维斯特(Jack Weast)表示:“你开始看到许多玩家谈论 2+级平台，他们正在考虑将原本应该用于完全自动驾驶的平台重新用于更高端的 ADAS。“我认为这既是意识到‘嘿，这比我们想象的要难一点，要制造出足够好的东西来去掉人’的反映，也是公众认知的反映。”

因此，Weast 认为 2020 年将是配备 ADAS 的车辆成为新常态的一年。Weast 补充说，客户将越来越熟悉自动驾驶技术，这有望减少对全自动驾驶汽车的恐惧。

已经有证据表明这已经开始了。埃德蒙兹的 Acevedo 表示，消费者选择为驾驶辅助和汽车的其他“内容”功能支付更多费用。

2019 年，消费者平均比汽车基价多花了 10，042 美元，比上一年增长 10.4%。再往后看几年差距就更大了。与 2014 年相比，2019 年消费者在 ADAS 等车辆的“内容”附加产品上的支出增加了 36.2%。

这对供应商来说是一个福音。博世负责驾驶辅助和自动驾驶区域业务部门的副总裁 Kay Stepper 表示，博世 2019 年的驾驶辅助系统销售额为 20 亿欧元，比上一年增长 12%。

“这甚至还不包括我们已经看到的新的兴趣，这是将在未来几年推出的 2 级和 3 级系统，”Stepper 说。
