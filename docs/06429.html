被禁一年后，劳拉·迪卡洛带着新的性玩具重返 CES

短短一年内发生的事情令人惊讶。在被禁止参加去年的消费电子展之后，性科技初创公司 Lora DiCarlo 在展会上推出了两款新的性玩具。此外，Baci 和 Onda 这两款新产品还获得了 CES 荣誉创新奖。

Lora DiCarlo 的三种设备都使用微型机器人技术来模仿人类的触摸。Osé旨在帮助产生混合高潮，Onda 针对 g 点，Baci 针对阴蒂。

加利福尼亚州旧金山-10 月 03 日:Lora DiCarlo 创始人兼首席执行官 Lora Haddock 于 2019 年 10 月 03 日在加利福尼亚州旧金山 Moscone 会展中心举行的 TechCrunch Disrupt 旧金山 2019 大会上发表演讲。(照片由金伯利·怀特/盖蒂图片社为 TechCrunch 拍摄)

这些奖项都要感谢 CES 的主办方消费者技术协会(Consumer Technology Association)整整 180。今年 7 月，CTA 宣布了一项计划，允许性科技初创公司参与和竞争健康和健康类别的奖项，为期一年。这发生在去年 CTA 与性技术公司 Lora DiCarlo 发生重大纠纷之后。CTA 撤销了该公司的一项创新奖，该公司正在开发一种免提设备，该设备使用仿生和机器人技术，通过同时刺激 g 点和阴蒂来帮助女性实现混合高潮。5 月， CTA 重新给公司颁奖并道歉。

正如 Lora DiCarlo 创始人 Lora Haddock 之前在 TC Disrupt 2019 上告诉我的那样，这种冷落在公司知名度方面帮了公司一个大忙。去年年底，Lora DiCarlo 开始预售其首款产品 Osé。该公司已经创造了 300 万美元的收入，仅在发布的前 36 个小时就创造了 150 万美元的收入。

Lora DiCarlo 的创始人 Lora Haddock 在一份声明中说:“在去年的事件中，创新奖被取消并恢复了 Osé，我们成为了变革的推动者，发起了一场关于性别平等的重要公共对话，并为所有 CES 参与者创造了一个更安全、更包容的环境。”

“在对我们的产品和使命有了更多了解后，人们开始意识到性健康是整体幸福的重要组成部分。今年，我们在 CES 上继续重塑人们对性技术的看法。这与技术无关。这与高潮无关。这是关于技术增强的体验如何带来更好的健康感，包括改善睡眠、减轻压力和改善心情。”
