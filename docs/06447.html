尽管绝地大溃败，AWS仍保持市场主导地位TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/01/06/despite-jedi-loss-aws-retains-dominant-market-position/

AWS 去年遭受了沉重的打击失去了价值100亿美元的、长达十年的绝地云合同给竞争对手微软。然而，即使没有构建国家联合企业防御基础设施的大型交易，该公司仍然完全控制着云基础设施市场——并且它打算反对这个决定。

事实上，AWS仍然拥有几乎两倍于其最接近的竞争对手微软的云基础设施市场份额。虽然两家公司将在未来十年争夺像JEDI这样的大合同，但目前AWS没有太多担心。

AWS的这一年不仅仅是失去了绝地武士。像往常一样，新闻发布时会发布一系列公告，并对其庞大的产品系列进行增强。更有趣的举动之一是向边缘转移，该公司对芯片业务越来越认真，并发布了大量机器学习产品。

事实是，AWS现在有这样的市场势头，这是一个合理的问题，如果有人，即使是微软，能赶上。不过，市场仍在继续扩大，下一场战斗是争夺剩余的市场份额。AWS首席执行官安迪·杰西(Andy Jassy)在2019年12月的re:Invent客户大会上花了比过去更多的时间来抨击微软，恳求客户更快地迁移到云，并表明他的公司正在准备在未来几年与竞争对手进行一场战斗。

AWS以360亿美元的运营成本结束了2019年，从1月份第一份报告中的74.3亿美元增长到10月份最新财报中的90亿美元。信不信由你，根据美国消费者新闻与商业频道的消息，这个数字没有达到分析师预期的91亿美元，但仍然占亚马逊本季度收入的13%。

无论如何，AWS是一个庞然大物，当你考虑到它在2006年作为Amazon.com的一个附带项目开始时，这是相当惊人的。事实上，如果AWS是一家独立的公司，它将是一项重要的业务。Synergy Research公司副总裁兼首席分析师约翰·丁斯代尔说，虽然去年的增长有所放缓，但当你达到AWS的规模时，这是不可避免的。Synergy Research公司跟踪云市场的各个方面。

“这只是数学和大数定律。平均而言，在过去的四个季度中，它每个季度的收入都增加了超过5亿美元。因此，在12个月的时间里，它的季度收入增长了20多亿美元。

Dinsdale补充说:“从背景来看，季度收入的增长比谷歌在云基础设施服务方面的总收入还要大。在一个年增长率超过35%的巨大市场中，AWS的市场份额保持稳定。”

Dinsdale表示，去年云基础设施市场没有突破1000亿美元，但即使没有第四季度的完整业绩，他的公司的模型预测总额约为950亿美元，比2018年增长37%。AWS拥有超过三分之一的份额。微软回到了17%左右，谷歌以8%或9%的份额排在第三。

绝地失望

如果不讨论JEDI，很难对AWS做任何年终回顾。从国防部宣布其长达十年、价值100亿美元的云RFP的那一刻起，它就一直是一个又一个的大争议。
