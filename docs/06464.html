Arduino 推出新的物联网开发模块化平台

开源硬件平台 Arduino 今天宣布推出新的低代码平台和模块化硬件系统，用于物联网开发。这里的想法是为中小型企业提供开发物联网解决方案的工具，而不必投资专门的工程资源。

这款名为 Arduino Portenta H7 的新硬件具有构建物联网硬件平台所需的一切，包括加密认证芯片和用于 Wi-Fi、蓝牙低能耗和 LTE 以及窄带物联网的通信模块。这些低功耗模块由 32 位 Arm 微控制器(Cortex-M7 或 M4)供电，旨在设计工业应用、边缘处理解决方案和机器人应用。它将运行 Arm 的 Mbed OS，支持 Arduino 代码，以及 Python 和 JavaScript 应用程序。

Arm 物联网服务集团战略副总裁 Charlene Marini 表示:“具有工业需求的中小型企业需要通过安全的开发工具、软件和硬件简化开发，以经济地实现其物联网用例。“Mbed OS 与 Cortex-M IP 在新的 Arduino Portenta 系列中的结合将使 Arduino 的数百万开发人员能够安全、轻松地开发和部署物联网设备，从原型到生产。”

新的 H7 模块现已面向 beta 测试人员开放，预计将于 2020 年 2 月全面上市。
