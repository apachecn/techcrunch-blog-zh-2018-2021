2020年数字媒体的发展前景TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/01/08/2020-media-entertainment-predictions/

随着2020年的开始，媒体和娱乐行业正在发生变化。新技术正在催生新类型的内容，多个内容类别的流媒体平台正在花费数十亿美元争夺市场份额，社交平台和媒体之间的相互作用是全球政治辩论的中心话题(轻描淡写)。

作为TechCrunch的媒体专栏作家，我去年与北美和欧洲的数百名企业家和高管谈论了从垂直方向的视频系列到游戏中的物理引擎再到音乐版税支付等各个领域正在发生的转变。展望未来的一年，我预计我们将在2020年看到媒体领域的一些高层次变化，分为七类:电影和电视、游戏、视觉和音频效果、社交媒体、音乐、播客和出版。

影视

在电影和电视领域，与网飞的竞争仍在继续，竞争比去年更加激烈。在美国，迪士尼在11月推出时拥有1000万迪士尼+订户，这是一个重要的开始，一些人预测到3月将达到2500万(包括免费试用或通过迪士尼与威瑞森的合作免费获得的用户)。与另外两家流媒体公司Hulu和ESPN+捆绑在一起，Disney+将迪士尼与亚马逊和网飞并列为三大巨头。

消费者只会为这么多订阅付费，通常是三大巨头中的一家、两家或全部(因为亚马逊Prime Video包含在更广泛的Prime会员中)，然后是最符合他们个人品味和最喜欢的节目的较小服务。

美国电话电报公司的HBOMax将于5月推出，价格为每月14.99美元，不太可能进入三大巨头的梯队，但可能是一个强大的二线竞争对手。与之并列的将是Apple TV+。通过每月4.99美元的订阅，苹果的服务只包括少量原创作品，这是HBO的一项战略，因为HBO被捆绑到一个更大的图书馆。CBS All Access、Showtime和NBC环球即将到来的(4月份)孔雀开屏也属于这一阵营。

在整个欧洲，地区媒体集团将成功地将SVOD和AVOD的本地竞争对手扩展到网飞，这些竞争对手已经于去年推出或计划在未来几周推出，如英国的BritBox、德国的Joyn和法国的Salto。网飞的增长来自美国以外，所以它的首要任务是购买更多的国际节目，这将迫使新的人口统计数据订阅。

不过，2020年最有趣的新进展将是4月份推出的Quibi，这是一项每月4.99美元的服务，为移动优先观看提供优质节目，已经获得了10亿美元的资金承诺和1.5亿美元的广告收入。Quibi节目将是一口大小的长度(不到15分钟)，垂直方向。该公司已投入数亿美元委托知名品牌打造数十个T2品牌。史蒂文·斯皮尔伯格和吉尔莫·德尔·托罗都有Quibi节目，NBC和CBS正在制作新闻节目。它提供的条款很诱人。

Quibi计划每周发布125部内容(即剧集)，今年将花费4.7亿美元进行营销，这是一个孤注一掷的赌注，如果第一次没有做好，几乎没有迭代的空间；它需要打入主流流行文化的热门节目才能生存。亿万富翁创始人杰弗瑞·卡森伯格和梅格·惠特曼为此次发布设定了极高的期望；预计媒体将在4月份猛烈抨击该平台未能满足这些期望，并预计该平台将在接下来的几个月里随着一些节目获得关注而自我救赎。

与此同时，直播体育仍然是广播电视网的最后希望，因为所有其他节目都将采用流媒体形式。消费者仍然重视实时观看体育比赛。然而，流媒体服务也将用于直播体育，并将在2020年朝着这一目标前进。三周前，DAZN击败了过去20年一直播放比赛的广播公司Sky，获得了2021/22赛季德国冠军联赛的转播权。亚马逊和YouTube继续探索体育转播权的竞标，而脸书和Twitter则从他们的努力中后退。YouTube的“YouTube TV”和迪士尼的“Hulu with Live TV”将导致更多消费者在2020年取消有线电视订阅，转而只看流媒体。

现在电影和电视领域的赢家是顶级制作公司。争夺流媒体视频主导地位的战争正促使几家世界上最富有的公司(和个人)向内容领域投入数百亿美元。大公司拥有这里的分销平台；唯一有实力进入的“创业公司”——DAZN和Quibi——已经由亿万富翁发起，并以10亿美元的支出承诺开始。创业机会在内容创作方面——制作人创作节目，而不是软件开发商创建平台。

游戏

根据研究公司Newzoo 的数据，游戏市场预计将同比增长近9% 从2019年的1520亿美元增长到2020年的1650亿美元，每年约有25亿人玩游戏。游戏现在在所有人群中都很普遍。休闲手机游戏占其中的最大部分(占行业收入的45%)，但PC游戏继续增长(去年增长了4%)，主机游戏是去年增长最快的类别(+13%)。

今年游戏界值得关注的大事:跨平台游戏、更加关注虚拟世界中的社交互动以及云游戏订阅量的扩大。

堡垒之夜用跨平台游戏的好处吸引消费者，这款游戏允许玩家在PC、手机和游戏机之间移动，并且它设定了其他游戏也这样做的预期。去年10月，我们看到《使命召唤》系列游戏登陆手机，并在第一周就达到了创纪录的1亿次下载。这一趋势将会继续，它将会把手机游戏中的免费商业模式传播到许多PC和主机游戏中。

游戏正走向社会前沿。许多人转向像堡垒之夜和PUBG这样的大型多人在线游戏(MMO)来进行社交，游戏性是次要兴趣。游戏是虚拟世界，玩家在其中社交、建造物品、拥有资产，就像在现实世界中一样。这导致游戏社交和现实生活之间的互动越来越流畅，就像Instagram或Twitter等社交应用的虚拟领域的社交现在被视为“现实世界”生活的一部分一样。

预计风投们会在2020年对“游戏是新的社交网络”这一命题下大赌注。一年前，大型投资公司将游戏类别视为不适合风险投资的“内容赌注”，现在它们正积极寻找交易。

在这一点上，有几个初创公司(如Klang Games、Darewise Entertainment、Singularity 6和Clockwork Labs)筹集了数百万美元的风险投资，以创建将于2020年推出(至少处于测试阶段)的开放世界游戏。在这些虚拟世界中，所有玩家都存在于世界的同一个实例中，而不是每个实例有100个左右的玩家。他们对未来的愿景是:数字领域，人们将为游戏中的经济做出贡献，建立友谊，并最终获得收入，就像他们的“现实世界”生活一样。想想下一代第二人生。预计他们将在2021年获得主流牵引力之前，花时间在2020年用早期采用者播种他们的世界。

似乎很少有人像脸书一样对游戏中的社交互动感到兴奋。渴望在社交媒体的下一次范式转变中拥有关键地盘，脸书今年将加快其游戏业务。2019年底，它收购了总部位于马德里的PlayGiga该公司致力于云游戏和5G技术-以及热门VR游戏Beat Saber背后的工作室。它还获得了热门游戏虚拟现实版的独家版权，如育碧的《刺客信条》和Oculus的《分裂细胞》。Horizon是其在VR中进行社交互动的虚拟世界，预计也将于今年推出。

脸书押注AR/VR将取代移动设备，成为消费计算领域的范式转变；它正投入数十亿美元，努力拥有距离R&D黄金时间还有几年的硬件和基础设施。与此同时，消费者转向虚拟世界中的社交互动正在以既定的形式发生——移动、PC和控制台——因此它将努力为消费者搭建从虚拟世界到未来的桥梁。

最后，云游戏随着谷歌Stadia的推出成为去年最大的头条新闻之一，你应该期待今年会再次成为头条新闻。通过将游戏转移到云主机，消费者可以从较低质量的设备上玩最高质量的游戏，极大地扩大了潜在玩家的市场。通过将许多这样的游戏捆绑到订阅服务中，谷歌和其他公司希望吸引消费者尝试更多的游戏。

然而，正如TechCrunch的 Lucas Matney所说，云游戏很可能是现有订阅游戏平台的一个功能——即Playstation Now和Xbox Game Pass——而不是一个新平台的差异化基础。玩云托管游戏固有的轻微延迟使得它对铁杆游戏玩家没有吸引力(他们宁愿下载游戏)。与索尼和微软的产品相比，Stadia有限的游戏选择并不突出。随着亚马逊的预期进入，今年的竞争只会更加激烈。谷歌需要推出与YouTube的Stadia集成，以及它在Stadia公告中宣传的共享状态功能，以真正激发消费者的兴趣。
