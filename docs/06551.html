谷歌将 IBM Power Systems 引入其云技术

原文：https://web.archive.org/web/https://techcrunch.com/2020/01/13/google-brings-ibm-power-systems-to-its-cloud/

随着 Google Cloud 试图说服更多的企业迁移到它的平台，它需要能够为企业现有的遗留基础设施和工作负载提供支持，这些基础设施和工作负载不容易替换或迁移到云。这些工作负载中有许多运行在配备 Power 处理器的 IBM Power Systems 上，而且，直到现在，IBM 基本上是 T2 唯一提供基于云的 Power Systems 的厂商。然而，现在谷歌也加入了这场游戏，与 IBM 合作推出了基于谷歌云的 IBM Power 系统。

更新:总部位于西雅图的Skype tap也提供对 IBM Power 系统的支持，并在其自己的云中以及 Azure 和 IBM Cloud 中提供这些系统。

Google Cloud 全球生态系统副总裁 Kevin Ichhpurani 在今天的公告中写道:“希望通过云计算实现现有基础设施现代化和简化业务流程的企业有很多选择。“一方面，一些组织正在对整个传统系统进行平台迁移，以采用云。然而，许多其他人希望继续利用他们现有的基础设施，同时仍然受益于云的灵活消费模式、可扩展性以及人工智能、机器学习和分析等领域的新进展。”

T2】

鉴于许多公司使用 Power Systems 来处理基于 SAP 和 Oracle 应用程序和数据库的任务关键型工作负载，power Systems 支持显然非常合适。有了它，他们可以将这些工作负载慢慢转移到云中，而不必重新设计他们的应用程序和基础架构。Google Cloud 上的 Power Systems 显然集成了 Google 的服务和计费工具。

这在很大程度上是一种企业产品，没有公开的价格表。有可能，考虑到基于 Power 的服务器的成本，您在这里看到的不是一个便宜的每分钟价格。

因为 IBM 有自己的云产品，看到它与谷歌合作将其服务器引入竞争对手的云有点奇怪——尽管它肯定希望销售更多的 Power 服务器。不过，此举对谷歌云来说非常有意义，谷歌云的使命是为其平台带来更多企业工作负载。该公司可以消除的任何障碍都对其有利，随着企业对其平台的适应，他们可能会随着时间的推移给它带来其他工作负载。
