谷歌云为零售商推出新解决方案 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/01/13/google-cloud-launches-new-solutions-for-retailers/

众所周知，谷歌云管理团队已决定将其工作重点放在选定的一些企业垂直领域，如医疗保健、制造、金融服务、能源和生命科学。零售也一直是该公司的增长市场，特别是亚马逊的竞争对手正在寻求在非 AWS 的云上运行他们的服务。目前的客户包括美国柯尔百货公司、劳氏和家乐福等。因此，毫不奇怪，谷歌今天利用最大的零售活动之一 NRF 2020 发布了一系列针对零售商的服务更新。

今天的一些公告侧重于现有服务的特定垂直版本，包括由 Apigee 提供支持的 Google Cloud API Management for Retail 或 Google Cloud Anthos for Retail，后者专门针对希望实现商店运营和基础设施现代化的零售商。还有谷歌云零售搜索，由谷歌搜索支持，承诺为零售商的应用程序带来更好的产品搜索结果。

此外，谷歌还在扩展更多的客户计划，如零售加速计划，并向更多的客户提供白手套客户可靠性工程服务，帮助零售商更好地计划和管理他们的购物高峰期。

然而，更有趣的可能是像谷歌云 1:1 零售服务这样的新服务，“关于如何以更低的前期成本有效地构建这种类型的数据驱动型解决方案的蓝图和最佳实践指南。”这里的想法是帮助零售商利用谷歌的大数据平台来建立个性化和推荐模型，以更好地了解和吸引他们的客户。

还有一项新的是购买优化和需求预测服务，旨在帮助零售商更好地规划他们的物流业务。

随着时间的推移，我们可能会看到谷歌在更多的垂直领域使用类似的剧本。我们知道谷歌云有在几年内成为第二大云提供商的雄心，为了实现这一目标，它需要让大型企业——尤其是那些仍在试图制定云战略的企业——选择它的服务。
