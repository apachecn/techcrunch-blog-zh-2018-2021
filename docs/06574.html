Cloudflare 向美国政治活动赠送其安全工具 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/01/15/cloudflare-campaigns-security-protections/amp/

图片来源: 康斯托克图片 /盖蒂图片

网络安全巨头 Cloudflare 表示，将免费为美国政治竞选活动提供安全工具和服务，作为其保护即将到来的选举免受网络攻击和选举干扰的努力的一部分。

该公司表示，其新的 Cloudflare for Campaigns 产品将包括分布式拒绝服务攻击缓解、活动网站的负载平衡、网站防火墙和反机器人保护。

这是该公司在其伽利略项目下为记者、民权活动家和人道主义团体提供的安全产品的扩展，旨在保护免受破坏性网络攻击。项目后来在 2018 年将扩展到较小的州和地方政府网站，目的是保护包含选民登记数据和其他选举基础设施的服务器免受攻击。

Cloudflare 的联合创始人兼首席执行官马修·普林斯(Matthew Prince)表示，“显然有必要”帮助竞选活动不仅保护其面向公众的网站，还保护其内部数据的安全。

该公司表示，它正在与无党派、非营利组织捍卫数字运动合作，为运动提供服务。去年，联邦选举委员会改变了规则，允许政治竞选活动接受打折的网络安全援助，这在以前是违反竞选财务的。
