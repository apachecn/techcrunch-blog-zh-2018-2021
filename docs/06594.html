Mozilla 裁员 70 人，等待新产品创收

原文：https://web.archive.org/web/https://techcrunch.com/2020/01/15/mozilla-lays-off-70-as-it-waits-for-subscription-products-to-generate-revenue/

TechCrunch 获悉，Mozilla 今天解雇了大约 70 名员工。

在一份内部备忘录中，Mozilla 董事长兼临时首席执行官米切尔·贝克(Mitchell Baker)特别提到了该公司新创收产品推出缓慢的原因。不过，总人数可能还会更高，因为 Mozilla 仍在研究这一决定将如何影响英国和法国的员工。2018 年，Mozilla 公司(相对于规模小得多的 Mozilla 基金会)表示其在全球拥有约 1000 名员工。

“你可能还记得，我们预计在 2019 年和 2020 年从新的订阅产品中获得收入，并从搜索以外的来源获得更高的收入。这并没有发生，”贝克在她的备忘录中写道。“我们的 2019 年计划低估了制造和交付新的创收产品所需的时间。鉴于此，以及我们在 2019 年学到的关于创新步伐的所有知识，我们决定采取更保守的方法来预测 2020 年的收入。我们还同意量入为出的原则，在可预见的未来，我们不会入不敷出。”

贝克说，被解雇的员工将获得“慷慨的离职补偿”和再就业支持。她还指出，领导团队考虑关闭 Mozilla 创新基金，但决定需要它来继续开发新产品。Mozilla 总共投入了 4300 万美元用于开发新产品。

贝克写道:“展望未来，我们知道我们必须采取大胆的措施来发展并确保我们使命的强度和寿命。”“Mozilla 对未来的收入增长有很强的洞察力，但我们对财务采取了更保守的方法。这将使我们能够根据需要转向应对互联网健康的市场威胁，并捍卫用户隐私和代理。”

该组织上一次报道2017 年大裁员。

在过去的几个月里，Mozilla 开始测试一些新产品，其中大部分产品一经推出就将实行订阅制。这里最大的特色是包括其 Firefox 专用网络和一项设备级 VPN 服务，该服务尚未推出，但每月费用约为 4.99 美元。

所有这些都是该组织计划的一部分，旨在减少对搜索合作收入的依赖，创造更多的收入渠道。2018 年，Mozilla 公布财务记录的最近一年，约 91%的版税收入来自搜索合同。

我们已经联系了 Mozilla 进行评论，一旦我们听到更多的消息，我们会更新这个帖子。

更新(太平洋时间下午 1 点):在发布到 Mozilla 博客的一份声明中，Mitchell Baker 重申 Mozilla 为了资助创新不得不做出这些削减。她写道:“Mozilla 对我们核心业务的未来收入有很强的洞察力。”。“在某些方面，这使得这一行动更加困难，我们对我们的同事受到的影响深感不安。然而，为了负责任地对创新进行额外投资以改善互联网，我们可以而且必须在我们核心资金的限度内开展工作”

大家好，

我有一些不好的消息要告诉你。在整个指导委员会和我们董事会的支持下，我们做出了一个极其艰难的决定:在今天的过程中，我们计划在整个 MoCo 取消大约 70 个职位。这个数字可能会稍大一些，因为根据法律要求，我们仍在英国和法国就可能取消的确切职位进行磋商。我们以最大的尊重对待每一个受影响的人，并将尽最大努力通过提供慷慨的离职补偿和再就业支持来照顾他们。大多数人不会和我们一起去柏林。当我们能够尽可能与受影响的人交谈时，我将发送另一封邮件，以便您知道通知/外展何时完成。

这个消息可能会令人震惊，我很抱歉我们不能一直对你们更加透明。这从来不是我的愿望。只有在探索了所有其他途径之后，指导委员会才考虑将裁员作为我们 2020 年规划和预算工作的一部分。最终决定是在假期前做出的，最终确定受影响角色的工作将持续到 1 月初(英国和法国有例外，我们正在就决定进行咨询。)我决定在我们有一份接近最终的受影响的角色和个人名单之前，不就此事进行沟通。

尽管我认为现在很难理解，但我还是想分享更多导致这个决定的原因。如果方便的话，也许你可以以后再来。

你可能还记得，我们预计在 2019 年和 2020 年从新的订阅产品中获得收入，并从搜索以外的来源获得更高的收入。这种情况没有发生。我们的 2019 年计划低估了制造和交付新的创收产品所需的时间。鉴于此，以及我们在 2019 年学到的关于创新步伐的所有知识，我们决定采取更保守的方法来预测 2020 年的收入。我们还同意量入为出的原则，在可预见的未来，我们不会入不敷出。

这种方法当然是谨慎的，但实际上具有挑战性。在我们的情况下，它需要艰难的决定和痛苦的结果。每年定期加薪、奖金和其他成本逐年增加，以及持续需要维持一个单独的、大量的创新基金，这意味着我们必须在整个 Mozilla 中寻求可观的节约，作为我们 2020 年规划和预算流程的一部分。这一过程最终导致我们决定裁员。

在这一点上，你可能会问我们是否考虑放弃单独的创新基金，继续像我们在 2019 年所做的那样。答案是肯定的，但我们最终决定，本着诚意，我们不能采纳这一点。Mozilla 的未来取决于我们在当前工作中的出色表现，以及开发新产品来扩大我们的影响。创造改变未来所需的新产品需要我们以不同的方式做事，包括为此分配资金，具体来说是 4300 万美元。我们将讨论我们的计划，让创新变得更加强大和成功，因为我们将进入，然后再次在 All Hands，而不是试图在这里这样做。

展望未来，我们知道我们必须采取大胆的步骤来发展和确保我们使命的力量和寿命。Mozilla 对未来的收入增长有很强的洞察力，但我们对财务采取了更保守的方法。这将使我们能够根据需要转向应对互联网健康的市场威胁，并捍卫用户隐私和代理。

我请求大家尽我们所能相互支持，度过这一困难时期。

米切尔
