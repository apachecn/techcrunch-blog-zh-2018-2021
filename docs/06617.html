2020 VC 的悖论是最大的基金在做最小的回合

原文：https://web.archive.org/web/https://techcrunch.com/2020/01/17/the-paradox-of-2020-vc-is-that-the-largest-funds-are-doing-the-smallest-rounds/

我昨天谈到风投们这些天是多么的累。太多的交易，每笔交易的时间太少，以及与其他风投公司对相同股权的持续过度竞争。

我的一位创始人朋友昨晚告诉我，在过去的一年里，他已经收到了 90 多位投资者关于他下一轮融资的入站请求——而且他甚至(大概)还没有开始融资。“我可能错过了一些，”他面无表情——真的，怎么可能没有呢？

然而，所有这些狂热的活动让我们想到了 2020 年风险投资的核心悖论:最大的基金开出了最早、最小的支票。

这是一个悖论，因为大型基金需要进行大轮投资。一个 10 亿美元的基金不可能开出 800 张 100 万美元的种子支票，还剩下美元作为管理费(他们可以，但那会令人讨厌，而且无法追踪)。相反，通常的模式是，随着公司基金规模的增长，其管理合伙人越来越多地转向后期融资，以便能够有效地配置资金。因此，曾经撰写 800 万美元 a 系列的 2 亿美元基金转变为撰写 4000 万美元 b 和 c 系列的 10 亿美元基金。

这是合乎逻辑的。然而，真正的逻辑要复杂一些。也就是说，每个人都在筹集巨额资金。

正如本周来自美国国家风险投资协会的大型风险投资报告明确指出的，2019 年在许多方面都是大型基金年(软银甚至没有融资！).去年推出了 21 只“大型基金”(定义为筹集超过 5 亿美元)，这实际上低于 2018 年的数字。

所有的后期资本都在寻找后期交易，但没有那么多交易可做。当然，有伟大的公司和潜在的巨大回报，但也有几十只基金密谋进入这个资本表，估值是这些投资者从竞争中脱颖而出的唯一杠杆之一。

这是格子在很多方面的故事。Visa 宣布打算以 53 亿美元收购的 fintech 数据 API 层，在 2018 年底从 Index 和 Kleiner筹集了 2.5 亿美元的 C 轮融资，所有这些都是根据 Crunchbase 提供的。多个风投人士告诉我，“每个人”都在关注这笔交易(如果你愿意，每个人都是疲惫的风投)。

但正如一位在 C 轮融资中说“不”的风投本周向我辩护的那样，去年的估值高得令人难以置信。据我所知，该公司 2018 年的收入超过了数千万美元，加上公开报道的 26.5 亿美元的 C 轮估值，意味着收入倍数在 30-50 倍之间——鉴于该公司正在与银行进行斗争，以确保它能够保持对用户账户的数据访问，这是非常昂贵的。

福布斯(Forbes)的杰夫·考夫林(Jeff Kauflin)报道该公司 2019 年的收入现在是数百万的低三位数，这意味着 Visa 很可能支付了类似的昂贵倍数来收购该公司。凯鹏华盈和 Index 的资金在一年左右的时间里翻了一番，没有人应该抱怨这种内部收益率(尤其是在成长型投资领域)，但如果不是因为 Visa 和退出时机的有利炼金术，一切可能都会变得非常不同。

比昂贵的估值更糟糕的是，这些后期融资可能变得非常专有和排他。从声音来看，Plaid 为其 C 轮融资运行了一个相当开放的流程，允许许多公司查看交易，有助于推动估值上升，同时限制早期投资者和创始人的稀释。但这不是唯一的解决方法。

越来越多的早期投资的公司也开始尝试后期投资。投了 500 万美元的 A 轮投资者也想投 5000 万美元的 B 轮和2.5 亿美元的 c 轮。毕竟，他们有资本，已经了解公司，与首席执行官有关系，可以避免在这个过程中耗时的融资。

因此，对于今天的许多交易来说，这些后期的上限表实际上是将新投资者拒之门外，因为已经有太多的资本坐在上限表周围，垂涎欲滴地要加倍下注。

这让我们直奔悖论。为了进入后期阶段的比赛，你必须已经在工资表上，这意味着你必须参加较小的早期阶段的比赛。突然之间，成长型投资者开始回到早期阶段(包括种子期),只是为了获得进入这些初创公司及其融资的选择权。

正如一位风险投资人上周向我解释的那样，“今天奇怪的是，像红杉这样的公司出现在种子期，但他们并不真正关心……任何事情。估价、条款等。这都是为了那些后期回合。”我认为这有点夸张，但说到底，对于最大的基金来说，那些一百万美元的支票本质上是一个舍入误差。真正的回报是在未来的大循环中。

这是否意味着种子基金将不复存在？当然不是，但当你的竞争对手毫不在乎，并认为投资是营销和访问费用时，你很难赚钱并建立一个平衡的、经风险调整的投资组合。至于创始人——如果你能找到合适的风险投资，现在仍然是非常非常好的时机。
