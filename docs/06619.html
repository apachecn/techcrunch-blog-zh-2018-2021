大家好，欢迎回到 TechCrunch 关注风险投资的播客 Equity ，在这里我们揭开头条背后的数字。

本周丹尼和阿历克斯再次聚在一起，反复讨论最新的早期阶段、最新的后期阶段等等。这又是一个堆积如山的星期，迫使我们挑挑拣拣。

然而，首先，这里是过去一周吸引我们眼球的几个回合:

离开早期阶段，走向光谱的另一端，我们谈到了 Cloudinary 突破 6000 万美元的 ARR 大关，ExtraHop 目标是在短期内突破 1 亿美元的 ARR 大关，以及 SiteMinder 的新一轮 7000 万美元的融资，该融资在去年突破 7000 万美元 ARR 后使其估值达到 7.5 亿美元。

明白了吗？就像我们说的，一直很忙。

本周节目中的两个主要故事是大格子交易和美国自己的风险市场正在发生的事情。

通过 Plaid，Visa 花费了 50 多亿美元收购了金融数据 API 服务，这是今年第一批重磅退出的交易之一，这让 Spark Capital 和其他公司的一些风投非常高兴。

与此同时，随着硅谷以外越来越多的地区壮大自己的创业公司，美国风险投资的格局正在迅速变化。如今，硅谷几乎不是风投资金的主要来源，而大西洋中部和东南部等地区正在迅速提升自己的形象。我们谈到了这一点，加上去年推出的十几只大型基金。

总而言之，风险资本家阶层似乎已经厌倦了。我们并不是为他们感到难过，但这表明这些天发生了太多的事情，没有人休息。不管他们有多少钱。
