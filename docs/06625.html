黑客正在利用 Internet Explorer 中的一个漏洞，但没有立即修复

原文：https://web.archive.org/web/https://techcrunch.com/2020/01/18/internet-explorer-security-flaw/

微软称将修复一个主动攻击下的 ie 安全漏洞

微软证实，黑客目前正在利用一个影响 Internet Explorer 的安全漏洞，但目前没有立即修复的计划。

在晚间的一条推文中，负责报告重大安全漏洞的国土安全部 US-CERT 在推特上发布了一个链接，链接到一个详细描述该漏洞的安全咨询，称其“正在被利用”

微软表示，所有受支持的 Windows 版本都受到该漏洞的影响，包括 Windows 7，本周之后将不再收到安全更新。

该漏洞是在 Internet Explorer 处理内存的方式中发现的。攻击者可以利用该漏洞在受影响的计算机上远程运行恶意代码，例如诱骗用户通过搜索查询或电子邮件发送的链接打开恶意网站。

据信这是一个类似于本周早些时候火狐浏览器制造商 Mozilla 披露的漏洞。微软和 Mozilla 都称赞中国安全研究团队奇虎 360 在主动攻击下发现了漏洞。据报道，本周早些时候，奇虎 360 删除了一条提及 ie 浏览器类似缺陷的推文。

奇虎、微软和 Mozilla 都没有说攻击者是如何利用这个漏洞的，攻击者是谁，或者谁是目标。美国政府的网络安全顾问小组也对当前的网络攻击发出了警告。

微软告诉 TechCrunch，它“意识到有限的有针对性的攻击”，并“正在努力修复”，但在下一轮月度安全修复之前，它不太可能发布补丁——定于 2 月 11 日。

微软为该漏洞分配了一个通用漏洞标识符， CVE-2020-0674 ，但该漏洞的具体细节尚未公布。

当联系到微软发言人时，他没有发表评论。
