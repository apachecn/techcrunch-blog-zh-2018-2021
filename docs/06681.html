随着自主性的停滞，激光雷达公司学会了适应

激光雷达传感器很可能是自动驾驶汽车必不可少的，但如果没有后者，你如何利用前者赚钱？在我采访的行业高管中，前景是乐观的，因为他们正在将自己的马车从无人驾驶汽车这颗耀眼的明星上卸下来。事实证明，对于那些有智慧正确运用投资的人来说，几年的疯狂投资会创造奇迹。

CES 2020 的展厅挤满了在更大空间展出的激光雷达公司，数量似乎比以前更多。这似乎与 2019 年是该行业调整年的报告不一致，所以我会见了几家公司的高管和知识渊博的人，听取他们对该行业过去几年转型的看法。

作为背景，2017 年可能是激光雷达的峰值，接近对各种公司近几年狂热投资的尾声。与其说这是一场淘金热，不如说是一场投机性的土地热:据称自动驾驶汽车就在眼前，每辆车都需要一个激光雷达单元……或者五个。投资赢家的竞赛正在进行，导致大量公司声称自己优于竞争对手。

不幸的是，正如许多人会记得的那样，今天的自动驾驶汽车似乎并不比那时更近，因为这项任务的真正困难在于承担这项任务的人。
