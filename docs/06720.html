《星际迷航:皮卡德》打破 CBS 全频道流媒体播放记录

原文：https://web.archive.org/web/https://techcrunch.com/2020/01/27/star-trek-picard-breaks-streaming-records-on-cbs-all-access/

哥伦比亚广播公司(CBS)的流媒体服务 CBS All Access 归功于三个备受瞩目的事件——包括其新的《星际迷航》(Star Trek)系列电影《星际迷航:皮卡德》(Star Trek: Picard)的首映，以及第 62 届格莱美奖，更不用说繁忙的足球月了——帮助它在某个月创造了订户注册的新纪录。该公司表示，2020 年 1 月超过了该服务在 2019 年 2 月的用户注册记录。此外，上周是有史以来第二好的报名周，紧随 2019 年超级碗周之后。

创纪录的大部分与备受期待的剧集《星际迷航:皮卡德》(Star Trek: Picard)的推出有关，该剧让粉丝们最喜欢的帕特里克·斯图尔特回归，饰演让·卢克·皮卡德，现在是一名退休的星际舰队上将，他在家族葡萄园的平静生活即将被打乱。这部剧发生在《星际迷航:下一代》(Star Trek: The Next Generation)最后一部电影《星际迷航:复仇女神》(Star Trek: Nemesis)的 18 年后，不仅利用了斯图尔特的吸引力，还让之前的《星际迷航》演员回归，包括布伦特·斯皮内(Data)、洁蕊·瑞恩(sebastian，九分之七)、玛丽娜·赛提斯(Troi)和乔纳森·弗雷克斯(Riker)。

但与其他希望怀旧能带给观众的重启不同，《皮卡德》的创作者实际上已经考虑到了该剧试图讲述的故事，这使得烂番茄获得了 95%的评论家评分。

哥伦比亚广播公司表示,《皮卡德》的首播也标志着总流量的新纪录，并推动了迄今为止哥伦比亚广播公司 All Access 原创系列的最高订阅量。

与 CBS All Access 的第一个记录相比，《皮卡德》的总流量增长了 115%以上，第一个记录是由它的其他《星际迷航》节目，不太受欢迎的《星际迷航:探索》创造的它还比“探索”之前的订户流媒体 CBS All Access 原创记录增长了 180%以上。

与此同时，哥伦比亚广播公司表示，昨晚的格莱美是迄今为止播放量最大的，也是格莱美周日注册人数的新纪录，新注册人数超过 2019 年的 80%，该服务的独立观众超过 30%。

然而，CBS 不会谈论 CBS All Access 的订户总数，也不会公布有多少人升级到了无广告层。

相反，该公司只分享了 CBS All Access 和 Showtime 的 over-the-top 服务加起来总共有超过 1000 万用户。

无论如何，这一数字远远落后于网飞和 Hulu 等流媒体竞争对手，它们分别拥有 6100 万美国用户和 2900 万 T2 用户。甚至像迪士尼+和苹果电视+这样的新来者也蓬勃发展。据估计，迪士尼+的用户数量在 T4 的 2320 万到 T6 的 2500 万之间。一种估计认为苹果电视+可能会更大，但是分析公司的方法值得怀疑。(毕竟，Apple TV+可能会免费提供给拥有新苹果设备的用户一年，但这并不意味着用户正在观看这项服务，也不会在以后为此付费。)

简而言之，这些数字意味着 CBS 需要的不仅仅是足球、季节性活动和一部新的《星际迷航》系列来实现增长。即使《星际迷航:皮卡德》大受欢迎，单独来看《星际迷航》的粉丝很可能只在节目播出时注册，然后在淡季退订。有些人甚至会等到他们可以尽情享受时再看——甚至可能是在免费试用期。

但作为新合并的 ViacomCBS，该公司现在有了选择。维亚康姆哥伦比亚广播公司的高层已经表示，由于维亚康姆和哥伦比亚广播公司的合并，他们可以将 Nickelodeon、BET、MTV 和 Comedy Central 节目全部带入哥伦比亚广播公司。该公司相信，到 2022 年，CBS 全接入用户将达到 2500 万。

ViacomCBS 首席数字官兼 CBS Interactive 总裁兼首席执行官 Marc DeBevoise 表示:“我们看到了这项服务的持续增长，我们因《星际迷航:皮卡德》、《格莱美》和一个精彩的足球赛季而创下的新纪录，为 CBS All Access 开启了精彩的一年。“CBS All Access 继续建立在它伟大的节目组合基础上——从原创系列到体育和特别活动——我们已经战略性地规划了 2020 年，为订户带来一个必看系列和活动的‘永远在线’日历，”他补充道。
