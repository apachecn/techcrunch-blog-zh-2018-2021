Persona 为身份验证平台筹集了 1750 万美元，该平台超越了用户 id 和密码技术

原文：https://web.archive.org/web/https://techcrunch.com/2020/01/28/persona-raises-17-5m-for-an-identify-verification-platform-the-goes-beyond-user-ids-and-passwords/

基于泄露密码的数据泄露的激增，以及对公司可以收集、存储和使用多少用户信息进行严格限制的监管浪潮，暴露了简单密码和基于记忆信息的验证系统的漏洞。

今天，一家名为 Persona 的初创公司宣布进行一轮融资，以应对市场的转变以及随之而来的对旧行事方式的新替代方案的需求，该公司建立了一个平台，使组织更容易实施基于第三方文档、实时评估和人工智能验证用户的更严密的方法。

这家初创公司已经从一系列令人印象深刻的投资者那里筹集了 1750 万美元的 A 轮投资，其中包括 Coatue 和 First Round Capital，它计划用这笔钱来加倍投资其核心产品:一个企业和组织可以通过 API 访问的平台，该平台允许他们使用各种文件，从政府颁发的 id 到生物识别，来验证客户是否是他们所说的人。

目前的客户包括 Rippling、Petal、UrbanSitter、Branch、Brex、Postmates、Outdoorsy、Rently、SimpleHealth 和 Hipcamp 等。Persona 目前的目标用户是参与任何类型的在线金融交易的任何公司，以验证法规遵从性、欺诈防范以及信任和安全。

这家初创公司很年轻，没有披露估值。根据来自 PitchBook 的数据，此前，Persona 已经从 Kleiner Perkins 和 FirstRound 筹集了一笔未披露金额的资金。该公司的天使投资人包括扎克·佩雷特(Zach Perret)和威廉·霍基(William Hockey)，迪伦·菲尔德(Dylan Field)，斯科特·贝尔斯基(Scott Belsky)和许宏涛(DoorDash)。

Persona 由 Rick Song 和 Charles Yeh 创建，他们分别是 Square 和 Dropbox 的前工程师(这两家公司对身份验证和安全漏洞有自己的担忧), Persona 的主要前提是大多数公司都不是安全公司，因此缺乏人员、技能、时间和资金来建立强大的身份验证和验证服务，更不用说跟上最佳实践的最新发展了。

最重要的是，已经有太多的违规事件凸显了公司持有太多用户信息的问题，这些信息是出于识别目的收集的，但却坐在那里等待被黑客攻击。虽然出现了许多服务来帮助保护产品重复用户的身份，例如企业前端的 Duo 和 Okta，或者作为使用短信的双因素身份验证的更安全替代方案的在线应用程序的身份验证器，但这些服务并没有真正满足作为典型角色客户的各种公司的验证用例。

Persona 的游戏名称是提供易于使用且适用性尽可能广泛的服务。对于那些不能或不访问其应用程序或网站代码进行注册的人，他们甚至可以通过基于电子邮件的链接来验证用户。

“数字身份是最重要的事情之一，但没有灵丹妙药，”首席执行官宋在接受采访时说。“我相信，从长远来看，我们会看到这不是一个放之四海而皆准的方法。”尤其是因为恶意黑客拥有越来越多的工具来绕过每一个安装到位的系统。(最新的例子是模仿人类的深度假货的兴起，这就提出了如何在视频验证系统中绕过这一点的问题。)

在 Persona，该公司目前允许客户选择要求社会安全号码、指纹或照片等生物特征验证，或者政府 ID 上传和电话查找，其中一些(如生物特征)是由 Persona 本身构建的，一些是通过第三方合作伙伴访问的。

除此之外，还有其他工具，如测验和基于视频的互动。宋表示，这个清单正在扩大，该公司正在研究如何使用它正在构建的人工智能引擎——实际上是执行匹配——来为每一笔交易提供最佳工具。

值得注意的是，这个平台是由一家支付公司的工程师构想和构建的。

基于 API 的平台通过在“引擎盖下”完成所有艰苦的工作，消除了支付系统的一些极端复杂性，已经成为许多金融服务如何集成到工作流中的一个组成部分，如果有问题的企业可能依赖它们，但实际上它们本身并不是金融科技公司(或支付技术提供商)。在某种程度上，这是 Stripe、Adyen、CurrencyCloud 甚至 Square 等公司的前提，因为其客户正在集成 Square 为他们构建的工具。

Persona 的另一个关键点是，它通过与其他数据库链接，为客户提供了一种访问和使用信息进行验证的方式，这意味着数据不会由客户自己保存。

这是一个不断变化的目标，也是一个越来越难以关注的目标，不仅是因为恶意黑客攻击的增加，也因为监管限制了在线企业访问和使用数据的方式和时间。

Persona notes 麦肯锡预测，到 2022 年，个人身份识别和验证市场的价值将达到 200 亿美元左右，考虑到谷歌迄今因违反 GDPR 而被罚款近 90 亿美元，或 Equifax 支付的 7 亿美元，或雅虎(现在的姐妹公司)因其用户数据违规而支付的 5000 万美元，这并不是一个令人惊讶的数字。
