微软股价上涨，超过收入和利润预期，Azure 增长 62%

今天，微软报告了其 2020 财年第二季度(日历 2019 年第四季度)的业绩，包括营收 369 亿美元(增长 14%)，净收入 116 亿美元(增长 38%)，每股摊薄收益 1.51 美元。

投资者曾 预计该公司 将公布每股 1.32 美元的利润，收入为 356.7 亿美元。华尔街也曾预计净利润为 101.2 亿美元。在该公司发布财报后，该公司股票在盘后交易中上涨了约 2%。

一切都很好。但是，云呢——微软在 Azure、Office 365 和其他有望将 Redmond 带入未来的产品上做得如何？给你:

  • 与去年同期相比，Office 365 商业收入增长了 27%
  • Azure 较去年同期增长了 62%
  • “办公消费产品和云服务”同比增长 19%
  • Dynamics 365 同比增长 42%

所有上述数字都是 GAAP 结果，因此没有根据货币波动进行调整。在所谓的“不变汇率”基础上，Azure 的增长率略高，为 64%。

除了这些结果之外，LinkedIn(一家相对较新的微软公司)与去年同期相比增长了 24%，而 Surface 以个位数的百分比增长，Xbox 的数字产品同比下滑了 11%。(微软早些时候强调 LinkedIn 的增长率是一个关键的优先事项。)

随着我们继续了解该公司的新业绩，我们很快就注意到了这一切，微软的云转型似乎在继续快速进行，投资者在盘后交易中根据这些结果适度抬高了其股票价格。请记住，微软的股票最近一直在上涨，在我们完成这篇文章时，该公司的估值已经突破了 1.28 万亿美元。

另一家科技巨头苹果公司在昨天公布收益后，股价也有小幅上涨。这是科技成果密集的一周，到目前为止，科技成果进展顺利。

Windows 呢？

微软的收益是稳定的，因为它的业务范围从消费硬件、软件、游戏和操作系统，到企业技术、销售工具、风险投资等等。但是有一种产品总是值得一分钟，那就是微软著名的操作系统 Windows。

值得注意的是，在本季度，Windows 业务表现良好，Windows OEM 收入增长了 18%。更好的是，“Windows 商业产品和云服务”上涨了 25%。诚然，微软的 OEM 业务(向制造个人电脑的硬件商店出售操作系统)可能会受到 Windows 7 寿终正寝的影响，但尽管如此，数字还是不错的。

或者至少比我猜测的几年前生活在后 iPad 时代的我们更好。

总之，一个坚实的季度从雷德蒙。该公司股价仅上涨了几个点，这更能说明其股价已经消化了多少业绩，而不是市场对该公司业务的热情。如果股票价格再次变动的话会更多。
