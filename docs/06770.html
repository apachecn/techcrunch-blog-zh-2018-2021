特斯拉将在第一季度末开始交付 Model Y

特斯拉周三表示，随着全电动紧凑型跨界车的首批交付，Model Y 的生产本月开始，将于第一季度末开始，该公司正在努力扩大其盈利能力。

首席执行官埃隆·马斯克在收益公布后的电话会议上表示，Model Y 的产量目前有限。马斯克补充说，特斯拉的工程师已经能够将全轮驱动版本的 Model Y 的最大 EPA 范围提高到 315 英里。之前估计有 280 英里。

特斯拉周三公布净利润为 1.05 亿美元，摊薄后每股 56 美分，而去年同期为 1.4 亿美元，每股 78 美分。特斯拉报告营收为 73.8 亿美元，比第三季度增长 17%，但仅比 2018 年第四季度的 72 亿美元增长 1%。

特斯拉在第四季度报告中表示，随着利润率更高的 Model X 和 Model S 的交付量减少，它将转向中国和 Model Y 来实现收入增长。特斯拉在第四季度交付了 19，475 辆 Model S 和 Model X，比去年同期下降了 29%。与此同时，特斯拉更便宜的 Model 3 汽车的交付量继续攀升。特斯拉在第四季度交付了 92，620 辆 Model 3 汽车，比 2018 年同期增长 46%。

Model Y 正在特斯拉位于加州弗里蒙特的工厂生产，解决了该公司内部关于在哪里生产这款即将上市的汽车的长期讨论。特斯拉还考虑在其位于内华达州斯帕克斯的大型工厂生产这款汽车，目前该公司在那里生产电池组和电动机。

特斯拉表示，随着到 2020 年年中增加机械设备，它将逐步增加 Model Y 的产量。该公司表示，一旦这些扩张完成，Model 3 和 Model Y 的总装机容量将达到每年 50 万辆。

特斯拉打算最终也在中国生产 Model Y。该公司的上海工厂有能力生产 15 万辆 Model 3 汽车。特斯拉上海工厂的下一阶段已经破土动工，将生产 Model Y。

该公司表示:“鉴于 SUV 汽车市场的受欢迎程度，我们计划让 Y 型车的产能至少相当于 3 型车的产能。”。
