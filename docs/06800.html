电视节目追踪应用TV Time的母公司Whip Media Group融资5000万美元

原文：https://web.archive.org/web/https://techcrunch.com/2020/01/29/whip-media-group-parent-to-tv-show-tracking-app-tv-time-raises-50m/

在流媒体时代，一个节目的收视率和受欢迎程度的数据更难获得。这不再是设置一个尼尔森盒子来获取通过电视、手机、平板电脑和网络观看的节目的数据那么简单。一家为内容所有者、广播公司和流媒体公司解决这一问题的公司是Whip Media Group，它是电视和电影跟踪应用程序TV Time的母公司。该公司今天宣布，它已经筹集了5000万美元的D轮融资，以继续发展其业务。

这轮融资由资产管理公司Eminence Capital领投，Raine Ventures也参与其中。迄今为止，Whip Media Group已经从Raine Ventures、Eminence、IVP和其他公司筹集了1.15亿美元。

威普媒体集团有着丰富的历史。电视时间始于WhipClip ，一个合法收集喜爱节目gif的来源。但在2016年12月该公司收购法国初创公司 TVShow Time 后，它转向成为一个社交电视社区。电视时间应用程序允许用户通过将剧集标记为已观看来跟踪他们最喜欢的节目，以及加入应用程序上的节目社区，用户可以在那里讨论剧集；共享照片、屏幕截图和迷因。参加民意测验；还有更多。它的推荐功能还能帮助用户找到更多值得观看的内容。

该公司更名为Whip Media Group，以反映出它现在拥有一些业务，包括电视时间应用程序，以及电视和电影娱乐数据库TheTVDB，以及最近的内容价值管理平台和 Mediamorph 。

虽然消费者只与电视时间应用程序互动，但这些参与有助于推动Whip Media Group的更大业务。

今天，电视时间应用程序每天有80万到100万活跃用户。50%的用户贡献了某种数据——例如，关注一个节目、创建内容、喜欢另一个用户的帖子、回顾一集、评论等等。迄今为止，电视时间已经跟踪了超过150亿集。

最初，《电视时代》利用这些数据开发了一种新型的评级系统，用于“断链时代”。但是电视时间了解到，一个节目的收视率对于不卖广告的视频点播服务来说并不重要。相反，电视时间可以提供的是用户对节目反应的情感数据。

“通过收集[这些数据]，我们可以建立这些模型，不仅可以说明人们在看什么，还可以开始预测他们接下来会看什么，”鞭媒体集团首席执行官理查德·罗森布拉特说。

此外，参与度数据可以帮助流媒体播放者发现他们以前从未发现过的事情——比如一集的哪个时刻有巨大的用户兴趣峰值，即“最受迷的时刻”这些数据可以帮助他们更好地营销该剧，并帮助他们思考该剧未来几季的发展方向。

现在，通过收购Mediamorph，Whip Media Group也可以帮助对内容进行估值。这使得买家和卖家可以决定在哪里卖节目以及卖多少钱。

这些数据对Whip Media Group的客户非常有价值，其中包括50多家娱乐界的知名公司，如迪士尼、华纳兄弟、Hulu、NBCU、派拉蒙、索尼、狮门影业、BBC、HBO、美国电话电报公司、T-Mobile、Liberty Global、Discovery和United Talent Agency。(流媒体领域还有其他一些家喻户晓的大公司也在使用该公司的数据，但由于保密协议的原因，这些公司不能透露。)

当内容所有者向现代流媒体服务出售一个节目时，他们往往无法知道它的表现如何。

罗森布拉特说，从Q1年底开始，Whip Media Group将能够开始预测可供出售的特定内容的去向。

罗森布拉特解释说:“从第一季度开始，我们将能够推出一个‘参与度评分’，让(内容所有者)能够真正了解一条内容与另一条内容相比，如何吸引特定人群或特定地域的注意力。”。“如果你想想20年前广告网络是如何开始的——你试图将正确的消费者与正确的广告相匹配，而最重要的是他们是否点击。其他都不重要。谷歌赢了，因为他们有最好的数据，最好的模型……这就是我们想要做的，”他继续说道。

“我们希望将正确的内容放在正确的平台、正确的国家和正确的人群中。他说:“我们认为，没有任何其他人像我们一样处于这种地位——在媒体形态和电视时间之间拥有所有这些东西。”。

这个数据比以往任何时候都更重要，在这个时代，核心经典作品的售价高达5亿(比如《宋飞正传》卖给了网飞，或者《老友记》卖给了HBO Max ，甚至更多(比如《生活大爆炸》(Big Bang Theory)十多亿美元的交易也卖给了HBO Max)。)

更广泛地说，全球在线电视剧集和电影收入将在2024年达到1590亿美元；根据 Research and Markets的数据，是2018年680亿美元的两倍多。

Whip Media Group的新一轮融资部分用于支付对Mediamorph的收购，这是现金和股票的结合。但大部分资金都被用于业务增长，包括扩大公司的销售和数据团队，以及加快产品开发。

到目前为止，该公司已经雇佣了20名员工，预计到年底将雇佣50名员工，主要从事数据和工程方面的工作。

Eminence Capital首席执行官里奇·桑德勒(Ricky Sandler)在一份声明中表示:“Whip Media Group正在构建软件和数据解决方案，这些解决方案将改变全球娱乐生态系统中内容的买卖方式。”。“我们相信他们的愿景、卓越的领导力和技术团队，很高兴能与他们合作，共同快速拓展业务。”
