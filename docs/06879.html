Twitter 操纵的媒体政策将删除有害的推文&压制选民，给其他人贴标签

原文：https://web.archive.org/web/https://techcrunch.com/2020/02/04/twitters-policy-on-deepfakes-and-manipulated-media-will-only-remove-harmful-tweets-including-voter-suppression/

Twitter 今天宣布了其“深度造假”和操纵媒体政策的官方版本，该政策主要涉及给推文贴标签，并警告用户操纵、欺骗性修改或捏造的媒体——在大多数情况下，不会删除它们。该公司表示，包含被操纵或合成媒体的推文只有在可能造成伤害的情况下才会被删除。

然而，Twitter 对“伤害”的定义超出了身体伤害，比如对个人或团体身体安全的威胁，或者大规模暴力或内乱的风险。“伤害”的定义还包括对个人或群体的隐私或自由表达意见或参与公民活动的能力的任何威胁。

这意味着该政策涵盖了跟踪、不必要的或强迫性的关注以及包含比喻、绰号或旨在让某人闭嘴的材料的目标内容。值得注意的是，鉴于美国总统大选在即，它还包括压制或恐吓选民。

去年 11 月，T2 首次公布了推特政策的初稿。当时，Twitter 表示将在分享合成和操纵媒体的推文旁边放置一个通知，在用户分享这些推文之前警告他们，并包括解释为什么媒体被认为被操纵的信息链接。这基本上已被确认为官方政策，但有更详细的说明。

Twitter 表示，在制定新政策之前，它使用# TwitterPolicyFeedback 标签收集了用户反馈，结果收集了超过 6500 个回复。该公司引以为豪的是在制定政策决策时让其社区参与进来，但鉴于 Twitter 多年来用户增长缓慢甚至持平，它可能希望尝试咨询那些迄今拒绝加入 Twitter 的人。这将让 Twitter 更广泛地理解为什么这么多人选择退出，以及这与它的政策决定有何关联。

该公司还表示，它咨询了全球民间团体和学术专家，如 Witness ，总部位于英国的路透社研究所和纽约大学的研究人员。

根据反馈，Twitter 发现大多数用户(70%)希望 Twitter 对误导和篡改的媒体采取行动，但只有 55%的人希望删除所有这类媒体。不出所料，反对者提到了对言论自由的担忧。大多数用户(90%)只想删除被认为有害的被操纵的媒体。大多数人(75%以上)也希望 Twitter 对分享这类媒体的账户采取进一步行动。

不像脸书的 deepfake 政策，忽略了视频和断章取义剪辑的剪切和拼接等不真诚的篡改，Twitter 的政策不限于特定的技术，如人工智能支持的 deepfakes。要宽泛得多。

Twitter 网站完整性负责人 Yoel Roth 证实，“像选择性编辑或裁剪或减速或过度配音，或操纵字幕，都是我们在这一政策下会考虑的操纵媒体的形式。”

“我们进行这些评估的目的是了解 Twitter 上刚刚滚动时间线的人是否有足够的信息来了解推文中分享的媒体是否是它声称的那样，”他解释道。

该政策利用三项测试来决定 Twitter 将如何对被操纵的媒体采取行动。它首先确认媒体本身是合成的或被操纵的。然后，它评估媒体是否以欺骗性的方式被共享。最后，它评估潜在的危害。

如果媒体可能导致他人困惑或导致误解，或者如果它试图欺骗人们关于其来源的信息，就被认为是欺骗性的——就像媒体声称它在描述现实，但事实并非如此。

这就是政策变得有点混乱的地方，因为 Twitter 将不得不检查该媒体的进一步背景，不仅包括推文的文本，还包括媒体的元数据，Twitter 用户的个人资料信息，包括在个人资料中链接的共享媒体的网站，或推文本身链接的网站。这种分析需要时间，而且不容易自动化。

然而，Twitter 在制定政策时给自己留了很大的回旋余地，使用“可能”和“可能”这样的词来表示在每种情况下的行动方针。(参见下面的标题)。

例如，被操纵的媒体“可能”被标记，而被操纵的和欺骗性的内容“可能”被标记。被操纵的、欺骗性的和有害的内容“很可能”会被删除。这种措辞给了 Twitter 制定政策例外的余地，而不会像使用“将被删除”或“将被贴上标签”这样更强硬的措辞那样，实际上违反了的政策

也就是说，Twitter 被操纵的媒体政策并不存在于真空中。一些最糟糕的被操纵的媒体，比如非自愿的裸体，已经被推特规则禁止了。那么，新政策并不是 Twitter 做决定时唯一要考虑的事情。

今天，Twitter 还详细说明了被操纵的媒体将如何被贴上标签。如果媒体因为没有“造成伤害”而没有被删除，Twitter 将通过一个提供更多背景信息的登录页面，在推文中添加一个警告标签，以及其他解释和澄清的链接。

事实核查组件也将是这个系统的一部分，由 Twitter 的策展团队领导。在误导性推文的情况下，Twitter 旨在呈现来自新闻机构、专家和其他人的事实，他们正在谈论与误导性推文直接一致的事情。

Twitter 还会在人们转发或喜欢该推文之前向他们显示警告标签，可能会降低推文的可见性，并可能会阻止其被推荐。

Twitter 的公开发布平台的一个缺点是，推文可以像病毒一样迅速传播，而 Twitter 执行其政策的能力可能会落后。在大多数情况下，Twitter 不会主动搜索其网络中的错误信息，而是依赖用户报告的推文进行审核。

这需要时间。尽管有相反的政策，推特多年来一直因其未能 回应骚扰和虐待而受到批评，其也在努力清除不良行为者。换句话说，Twitter 在操纵媒体方面的意图可能会在这项新政策中得到阐明，但 Twitter 在现实世界中的行动可能仍然会有所欠缺。时间会证明一切。

“Twitter 的使命是为公众对话服务。作为其中的一部分，我们希望鼓励健康地参与对话。扭曲或转移人们注意力的事情会威胁到推特上信息的完整性，”推特信任与安全副总裁德尔·哈维说。“我们的目标是为人们提供更多他们在 Twitter 上遇到的特定类型媒体的背景，并确保他们能够围绕他们看到的内容做出明智的决定，”她补充道。
