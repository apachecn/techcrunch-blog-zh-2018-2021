顶级风投在哪里投资开源和开发工具(第1部分，共2部分)

开源软件这个曾经两极分化的世界最近已经成为风投的热门目标之一。

随着开源软件在组织和开发者中的流行，这个领域的初创公司已经达到了新的高度和巨大的估值。

在过去的几年里，我们已经看到像Databricks这样的开源公司达到了独角兽的地位，以及在涉及开源和开发工具公司的大量退出背后套现的风投，如IBM的红帽收购或Elastic的2018年末IPO 。去年，退出狂潮继续进行，如F5 Networks的 收购NGINX的 ，以及从 微软 和 GitHub 等主要公司高调收购。

同样，对该领域新创企业的风险投资也在持续增长。根据Crunchbase的数据，越来越多的投资者开始尝试寻找下一笔巨额支出，在过去五年中，开源和开发工具初创公司的年投资资本以大约10%的复合年增长率(CAGR)增长。此外，该领域诱人的回报似乎为火上浇油，据Crunchbase 数据显示，仅在2019年，开源和开发工具初创公司在该领域的投资就超过20亿美元。
