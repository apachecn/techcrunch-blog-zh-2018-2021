脸书对Onavo间谍软件的使用面临欧盟反垄断调查的质疑-报告TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/02/06/facebooks-use-of-onavo-spyware-faces-questions-in-eu-antitrust-probe-report/

根据《T2华尔街日报》的一篇报道，脸书对其于2013年收购的Onavo间谍软件VPN应用程序的使用——并用于其2014年收购当时的竞争对手WhatsApp消息平台——受到了欧洲反垄断监管机构的关注。

报纸报道称，作为对脸书数据行为初步调查的一部分，委员会已经要求大量的内部文件，该调查于12月日宣布。

《华尔街日报》援引知情人士的话说，监管机构的调查重点是指控脸书试图通过利用用户数据来识别和打击潜在竞争对手，从而扼杀竞争。

脸书在一年前宣布关闭on AVO——面对越来越多的关于其使用VPN工具作为数据收集商业情报搜索网的争议，这不仅对用户隐私不利，还引发了关于反竞争做法的重大问题。

就在最近的 2018 脸书还在积极向其主要社交网络应用程序的用户推广Onavo在“保护”的旗帜下进行营销，旨在让用户相信该工具将帮助他们保护自己的信息。

事实上，VPN允许脸书通过第三方应用程序监控他们的活动，使这家科技巨头能够发现更大的移动生态系统中的新兴趋势。(因此，正如我们之前所说的，“保护脸书的商业”应该是这个工具更准确的标签。)

到2018年年底关于脸书如何在重大收购中使用Onavo作为关键情报杠杆的更多细节浮出水面，当时一个英国议会委员会获得了一份与美国法院案件有关的内部文件，该案件由第三方开发商提起诉讼，指控其应用平台受到不公平待遇。

英国议员得出结论，脸书利用Onavo对客户的移动应用使用情况进行了全球调查，显然是在他们不知情的情况下进行的——利用英特尔不仅评估了多少人下载了应用，还评估了他们使用应用的频率，这反过来帮助这家科技巨头决定收购哪些公司，以及将哪些公司视为威胁。

议会委员会继续呼吁竞争和数据保护机构调查脸书的商业行为。

因此，毫不奇怪，欧洲竞争委员会也应该调查脸书是如何利用Onavo的。据《华尔街日报》消息人士透露，该委员会还在审查脸书对其开发者应用编程接口所做的修改，这些修改影响了它提供的信息。

英国议会公布的内部文件也强调了开发者访问问题——例如脸书将某些受青睐的开发者的用户数据访问列入白名单的做法，引发了用户同意共享其数据的问题——以及相对于未列入白名单的开发者的公平性。

据该报报道，作为初步调查的一部分，监管机构要求脸书提供大量内部文件，包括电子邮件、聊天记录和演示文稿。它表示，脸书的律师进行了反击，试图缩小发现过程，辩称对信息的要求非常广泛，会产生数百万份文件，并可能泄露脸书员工的个人数据。

《华尔街日报》的一些消息人士还告诉它，欧盟委员会已经撤回了最初的命令，并打算发布一份范围更窄的请求。

我们联系了脸书和竞争监管机构，请其置评。更新:在一份由副总法律顾问Timothy Lamb发布的声明中，脸书表示:“我们致力于配合监管机构对我们所处竞争环境的调查。”

该委员会拒绝对《华尔街日报》的报道置评。

回到 2017 欧盟委员会对脸书罚款1.22亿美元，原因是其在收购WhatsApp时提供了不正确或误导性的信息。脸书曾向监管机构保证，用户账户不会在这两个服务之间关联——这为其获准收购WhatsApp扫清了道路——但该公司在2016年宣布将关联用户数据时态度大转弯。

除了就潜在的反垄断问题调查脸书的数据行为，欧盟竞争监管机构也在调查谷歌的数据行为——12月在宣布初步调查。
