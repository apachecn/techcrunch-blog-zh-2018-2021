Brilliant 让您的智能家居更易于管理

原文：https://web.archive.org/web/https://techcrunch.com/2020/02/07/brilliant-makes-your-smart-home-more-manageable/

从你的手机或语音控制你的智能家居设备并不是一件苦差事，但在设置了一堆智能灯、一个 Wi-Fi 锁、恒温器和一些其他智能设备后，我开始怀念用物理开关控制其中至少一些设备的能力。再加上一个简单的事实，你的访客突然不知道如何关灯，你可能只想回到基本的灯开关。令人欣慰的是，业界也已经意识到了这一点，我们现在也看到了更多的智能硬件控制器。

在今年的 CES 上， Brilliant 发布了一款新的智能插头和开关，以补充其现有的触摸屏智能家居控制器。新硬件还有几周的时间，但在发布之前，我有机会试用了现有的 Brilliant 控制器，它已经上市一段时间了，但从那以后已经收到了许多更新和对新集成的支持。最新的集成之一是与西勒奇的编码 Wi-Fi 锁，我也测试了。

智能控制的承诺是，你将能够通过物理和触摸屏控制来控制所有支持的智能家居设备——当然，它还能把你用它取代的电灯开关变成智能开关。它还配有一个内置摄像头(带隐私快门)，你可以用它进行房间到房间的视频聊天，或者在你不在的时候查看你的家。视频质量不是很好，但足以达到预期目的。

支持的设备包括 Wemo 智能插头、响铃报警器、Sonos 扬声器、飞利浦 Hue 和 Lifx 灯，以及西勒奇、耶鲁和奥古斯特锁、等。集成的数量不断增长，涵盖了大多数主要品牌，但如果你已经在其他系统上下了赌注，这不是你的控制器。它还内置了 Alexa 支持，也可以与谷歌助手配合使用。

根据你对家中用电的感觉，智能控制器的物理安装(我测试了 299 美元的单开关和 349 美元的双开关)要么轻而易举，要么会让你做噩梦。不过，如果你曾经换过电灯开关，安装就再简单不过了，Brilliant 提供了深入的印刷安装指南和视频教程。

我自己的经历非常简单，假设你家的电力系统相对现代，符合今天的标准。安装单个开关花了我大约半个小时，而更复杂的双开关在大约 45 分钟左右就准备好了——这是我几年来第一次更换电灯开关。如果你以前从来没有这样做过，你的开关后面的老鼠窝可能需要一点时间来弄清楚，但谢天谢地，现代家庭中的所有电缆都应该有颜色编码。

这里的一个很好的功能是，你首先安装背板，背板上有物理按钮，让你在安装实际的触摸屏单元之前测试你的安装。这样的话，你就不用把所有的东西都拧下来以防你犯了一个错误。

至于软件方面，一旦你放到屏幕上，基于 Android 的界面应该会在几分钟内弹出。从那里，你通过通常的 Wi-Fi 设置程序和最有可能的软件更新。之后，你应该准备走了。

通过触摸屏或侧面的电容带(用于双开关控制及以上)管理直接连接到控制装置的灯非常简单。将第三方设备添加到系统中需要一点时间，但也不会太麻烦，毕竟您只需要做一次。

我发现整个菜单系统有点混乱，需要一段时间来浏览。当你想对场景编程时(也许是打开你客厅或卧室里所有不同的智能灯)，这尤其成为一个问题。为此，你必须编写一个打开所有灯光的场景，每一个灯光都要轻敲几下，然后编写第二个场景，把所有的灯光都关掉。因为你可以复制场景，所以第二步要快一点，但是我忍不住想应该有一个更好的解决方案。同时，虽然，这允许你创造相当复杂的场景。你也可以通过手机上出色的应用程序来完成大部分工作，这可能是一种更简单、更快捷的方式。

然而，一旦一切都设置好了，这个系统实际上非常容易使用，甚至你家从未见过智能插头的客人最终也能打开和关闭你的灯(是的，我知道这在 2020 年应该不是问题，但我们在这里)。我知道这有点老生常谈，但它确实很有效。

我在 Brilliant 上遇到的一个问题是控制器价格昂贵，单开关的起价为 299 美元，双开关的起价为 349 美元。在这样的价格下，你不会把它们放在你的很多房间里(除非你认为这不是很贵，在这种情况下，恭喜)。随着即将到来的无屏幕调光开关，只需要你在家里有一个单一的控制，零售价格将低于 70 美元，这一等式发生了变化。一旦这些新交换机在今年晚些时候上市，我们将会试一试。
