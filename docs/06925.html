印度教育创业公司Byju's从通用大西洋公司融资2亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2020/02/07/indian-education-startup-byjus-raises-200m-from-general-atlantic/

印度教育创业公司Byju's从通用大西洋公司融资2亿美元

印度教育公司Byju's上周五表示，该初创公司的现有投资者General Atlantic已注入额外资金，加入其新的正在进行的融资轮。

尽管这家总部位于班加罗尔的初创公司(使用短视频帮助学生学习数学和科学概念)没有披露通用大西洋公司的支票规模，但一位知情人士告诉TechCrunch，这家总部位于纽约的股权公司在新一轮投资了2亿美元，估值为80亿美元。

今天的声明是在Tiger Global向这家成立九年的初创公司投资2亿美元几周之后发布的，该公司在新一轮融资前宣布盈利。

新一轮融资帮助Byju的估值从一年前的约57亿美元攀升至80亿美元，使这家edtech初创公司成为印度第三大最有价值的私营公司，仅次于金融服务公司Paytm ( 估值160亿美元)和经济型酒店运营商Oyo ( 估值100亿美元)。

这家同名初创公司的创始人兼首席执行官Byju Raveendran表示，新的投资表明“我们的学习项目在全国各地的小城镇越来越被接受。”

Raveendran曾是一名教师，他在2011年创立了Byju's，当时印度只有几百万人拥有智能手机，移动数据价格对大多数人来说过于昂贵。

由于印度低价Android手机的激增和自T9以来数据价格的下降，在过去的十年里，印度有数亿人上网。

Byju还采用了创新的方法，比如用比萨饼来帮助解释复杂的科学概念。它的应用程序已经积累了超过4200万注册用户，其中300万是付费用户。这家初创公司提供针对六年级及以上学生的课程，包括那些准备竞争激烈的本科生课程的学生。

超过65%的Byju学生来自全国前10大城市之外，这让许多学生第一次有机会接受优质教育。(然而，它的一些注册用户的策略已经受到审查。)

“这是对技术如何让学生更容易获得高质量学习内容的验证，无论他们身在何处。事实上，技术作为一种使能手段，有很大的潜力来进一步创新学习方式，使学习变得个性化和有影响力，”Raveendran补充道。

Byju's迄今已融资约14.5亿美元。它的投资者还包括腾讯和Naspers。马克·扎克伯格的Chan-Zuckerberg Initiative也是早期投资者。

Byju的发言人拒绝就通用大西洋新投资的规模和初创公司的估值置评。
