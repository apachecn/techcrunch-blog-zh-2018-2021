一旦你有了产品，销售就成了任何企业家的下一个挑战。在 TRASH (一键视频编辑)，我们将抖音视为一个潜在的营销渠道。随着早期的学习开始滚动，我们决定分享这个爆炸和神秘的野兽里面发生了什么。

第1部分:利用抖音实现增长

拥有一家使用人工智能来帮助加速视频编辑过程的深度技术公司的优势在于，我们可以“免费”这样做。当你考虑到编辑一个半专业的视频至少要花费1500美元和6个小时的后期制作时，这是非常酷的。当我们开始研究分销以及如何降低客户获取成本时，抖音是第一个。

在抖音工作时，有三个关键区域需要注意:

审核流程并进入“为你而做”页面

派往抖音最重要的部分是你的权威排名，即:“你有多大的影响力？”你的权威排名与你的垂直领域(你制作视频的风格)直接相关。

什么有助于你的权威排名

  1. 新账户。就像你的优步五星乘客评分一样，你的每一个帖子都会影响你的分数。
  2. 多个账户。抖音允许多个账户，但专业提示:一部手机上的多个账户会将你标记为商业账户，像许多平台一样，它们会降低你的优先级，除非你是付费广告客户。如果你正在尝试这些事情，限制你的帐户登录到一个设备。
  3. 您发布的前五个视频。抖音希望你能创造出保持在同一垂直领域的视频类型。所以如果你在前五年制作迷因视频，抖音基本上会说，“这是一个迷因账户。”所以，前五条是至关重要的:你需要有一个计划和重点。
  4. 垂直度。抖音不希望你成为实验者。选择一个垂直的内容并坚持下去。变化不定或没有特定主题的内容权重不高。如果你开始制作属于不同类别的视频，这就像是重新开始，因为你在这个垂直领域还没有权威。
  5. 观点。如果你的视频只有100次或更少的浏览量，你将会有一个僵尸账户，所以删除并重新开始。获得1000–3000次观看的视频意味着您拥有一个中端客户。获得10，000+浏览量的视频意味着你有一个“头部”账户。
  6. 查看完成。这是最重要的因素之一。您的视频需要从头到尾观看，才能计入此指标。对此有帮助的关键因素是:
  • 短视频。视频最长可达60秒，但抖音向他们的广告商推荐视频长度为9-15秒(互联网认为抖音的平均长度为30秒)。
  • 循环播放视频。如果视频被重复观看，那么它的完成率将超过100%,并且将增加视频的整体性能评级。一种常见的做法是在视频中创建无缝循环，这样观众就会被诱骗多次观看。
  • 格式。通常会有一个结尾有笑点的挑战形式。人们理解这种形式，所以他们会留下来看笑点。
  • 配合动作配合音乐。观看总是更令人满意。

审查过程，并使之成为FYP

所以，现在你知道了所有可以提高你的权威分数的方法，有了你想要创建的账户类型的计划，并且创建了五个杀手级视频，你就可以开始发帖了。接下来会发生什么，包括如何得到梦寐以求的FYP。

  1. 基于权限的自动分发。根据您的得分，您的视频会被发送到一个拥有大约300-500名观众的本地网络。在这一点上，没有真正的检查你的内容。
  2. 基于完整性的人工智能审查和数据收集。在最初分发给几百人后不久，它就被人工智能逐帧检查是否有不当内容、版权问题等。然后，它会被赋予一个新的权重(完整性等级),要么被除名，要么再次分发。
  3. 延迟爆炸。这是抖音与其他平台的最大区别之一，也是你有第二次机会登上FYP的地方。延迟爆炸是你应该仔细考虑删除旧内容的原因，不管它以前做得好还是差。周期性地(不清楚这发生在什么时间尺度上，可能是几周或几个月)，抖音在FYP上隐藏内容的发布日期。抖音将测试您的旧内容，并重新开始一个循环，看起来像这样:小批量的内容大约两个小时；然后是一个中等的批次，人工智能会查看你的权威评级的关键指标；最后一大堆，包括你的诚信评级(没有“坏”的内容或他们认为“坏”的内容)。此时，它显示类似于“嘿，我们是前5%的视频。”
  4. 人类评论。人类评论者将看到带有这些分数的视频，并决定它是否有可能成为超级病毒视频。他们还会仔细检查版权和可能在第二步中漏掉的“坏”内容。要想被推广到FYP，内容必须符合抖音(作为一家总部位于北京的公司，不可避免地也是中国的公司)对当地好的和受欢迎的东西的想法。人们注意到的常见事物是代表传统美丽标准的人(尽管这也可能是基于人类偏见的算法偏见)，没有强烈的政治观点(除非他们只是在某些国家开玩笑或模因……尽管可能不是小熊维尼),没有违反最具反应性的当地社会规范。这里肯定有某种程度的…同质性。这可能会让你明白为什么抖音希望你创作基于抄袭的内容？这使得更容易审查和坚持的格式不只是什么“工程”(即。正在像病毒一样传播)但是什么符合他们对什么是好的看法。

制作更好的内容(以及你可能做错了什么！)

选择一种格式。因为垂直化是你权威得分的关键，你需要选择一种格式并在其中工作。如果你想有不同的个性，使用不同的帐户。这将提高你的权威得分，也有助于获得追随者，因为他们的期望将根据你制作的内容类型而定。做得好的顶点的例子有喜剧、迷因、舞蹈、视频博客、创作/DIY 和黑客。

复制格式。抖音鼓励多种形式的共同创造，如反应、合作/混合和模仿。这在整个平台上创造了格式、趋势和迷因。观众不会被视为抄袭其他创作者，而是喜欢潮流，并受到启发创作自己的版本。像Charli D'amelio这样的吉他手为流行歌曲创作非官方的舞蹈动作，仅仅复制这些舞蹈动作就能让一首歌登上排行榜的首位。下一个标志性的舞蹈，如“颤栗”、“单身女士”或“Gangam Style”将由某个人创作，他可能与原曲没有真正的联系或所有权。

总的来说，众所周知，与千禧一代相比，Z世代在追求上不那么“孤独”。我们认为这种协作创作方式不仅是社交娱乐的时代标志，也是下一波创作工具和平台的时代标志。了解你的音乐:歌曲是让人们理解你的模因内容的最好方式之一。许多观众根据歌曲就已经知道你的内容是什么了，所以为你想要复制的格式选择正确的歌曲是关键——这不能是事后的想法，这可能是你出错的地方！

准备出售。如果Instagram对于千禧一代来说是 QVC ，那么抖音就是z世代在Supreme store外面排队的地方。购物将会变得更加“真实”和有故事性，而不是那些光鲜亮丽的健身和美容广告。购物视频已经是亚洲的一件大事，据报道，它正在抖音进行测试，很快将推广到世界其他地方。

我们怀疑Z世代会简单地把亚马逊当成谷歌的东西搜索，新的社交平台会变成虚拟商城。我们还怀疑抖音会在FYP算法中更重视可购物内容，因为钱。
