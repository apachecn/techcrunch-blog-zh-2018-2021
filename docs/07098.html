谷歌云收购大型机迁移服务基石

谷歌今天宣布收购了荷兰公司 Cornerstone ，该公司专门帮助企业将其遗留工作负载从大型机迁移到公共云。Cornerstone 提供非常实用的迁移帮助，它将成为谷歌云的主机到 GCP 解决方案的基础。

这一举措非常符合谷歌云的整体企业战略，该战略专注于帮助现有企业将其遗留工作负载迁移到云中(并从一开始就以云原生解决方案的形式启动新项目)。

Google Cloud 客户体验副总裁 John Jester 表示:“这是 Google Cloud 帮助企业客户实现基础设施和应用程序现代化的又一个例子。“我们在更好地为企业客户服务方面取得了长足的进步，包括引入高级支持，更好地调整我们的客户成功组织，简化我们的商业合同流程以使与谷歌云做生意更容易，以及扩展我们的合作伙伴关系。”

许多企业仍然依赖他们的大型机来支持任务关键型工作负载。将它们迁移到云中通常是一项非常复杂的任务，这正是 Cornerstone 和类似供应商的用武之地。许多大型机应用程序是用 Cobol、PL/1 或汇编语言编写的，这并没有什么帮助。Cornerstone 的技术可以自动将这些流程分解为云原生服务，然后在容器化的环境中进行管理。它还可以根据需要迁移数据库。

值得注意的是，谷歌云最近也在它的云中引入了对 IBM Power 系统的支持。这也是帮助企业将其遗留系统迁移到云中的一个举措。有了 Cornerstone，Google Cloud 在此基础上又增加了一层，为那些希望慢慢实现整体堆栈现代化而不必重新设计所有遗留应用程序的用户提供了更多的实际迁移帮助。
