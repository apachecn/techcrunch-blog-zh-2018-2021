Microsoft Dynamics 365 更新专注于利用数据 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/02/19/microsoft-dynamics-365-update-is-focused-on-harnessing-data/

微软今天宣布对其 Dynamics 365 产品线进行重大更新，该产品与企业中不断增长的数据量以及如何收集和理解这些数据以提供更好的客户体验相关。

事实上，这是该领域所有供应商的目标，包括 Salesforce 和 Adobe，它们也希望帮助改善客户体验。本周刚刚晋升为微软商业应用总裁的詹姆斯·菲利普斯(James Philips)表示，微软也一直非常关注利用不断增长的数据量，并帮助在他负责的应用程序中利用这些数据。

“坦率地说，我们在微软所做的每一件事，不仅仅是在商业应用程序中，而是在整个微软云上，都基于这样一种愿景，即数据来自一切，那些能够收集数据、协调数据并进行推理的组织将处于主动而非被动的地位，”飞利浦告诉 TechCrunch。

新的客户参与工具

首先，该公司正在向其客户数据平台(CDP)添加功能，这是所有主要供应商(以及越来越多的初创公司)都接受的概念。它将来自不同系统的所有客户数据汇集到一个地方，从而更容易理解客户如何与您交互，目的是基于这一知识提供更好的体验。微软的 CDP 叫做客户洞察。

该公司正在添加一些新的连接器，以帮助完善客户的形象。飞利浦说:“我们正在为客户洞察添加新的第一和第三方数据连接，使我们的客户能够了解，例如观众成员资格、品牌亲和力、人口统计、心理特征和客户的其他特征，这些都是存储的，然后从 Dynamics 365 客户洞察中利用。”

所有这一切可能会让你想知道他们如何收集这种级别的数据，并保持 GDPR/CCPA 的合规性。飞利浦表示，该公司已经在这方面努力了一段时间。“我们确实在公司层面上建立了一个系统，允许我们和我们的客户搜索并删除微软内部每个产品组的客户信息，包括我的组织，”他解释道。

该公司还增加了新的销售预测工具和 Dynamics 365 销售参与中心。第一种方法允许公司利用所有这些数据来更好地预测销售人员接触的最有可能转化为销售的客户。第二种方法为内部销售团队提供工具，如下一个最佳行动。这些在 CRM 领域并不具有革命性，但确实为微软客户提供了新的功能。

新的操作级工具

运营方面与销售后发生的事情有关，此时公司开始收款并报告收入。为此，该公司正在推出一款名为 Dynamic 365 Finance Insights 的新产品，你可以将它视为客户洞察，除了金钱。

“该产品旨在帮助我们的客户预测并加速他们的现金流。飞利浦说:“它是专门设计来确定你应该集中精力的机会，你可能有最好的机会结清应付账款或应收账款，或者有机会了解你可能存在现金短缺的地方。”

最后，该公司推出了 Dynamics 365 项目运营，为建筑、咨询和法律等基于项目的业务提供了一种跟踪业务需求的方式。

“那些试图以基于项目的方式运营的组织现在拥有 Dynamics 365 项目运营，我们认为这是 Microsoft Project 中使用最广泛的项目管理功能，现在与 Dynamics 365 提供的所有销售、会计和规划后端功能相结合，所有这些功能都建立在同一个公共数据平台上，因此您可以将前端运营和运营规划与后端资源规划、劳动力规划和运营流程结合起来，”他解释道。

所有这些工具都旨在利用进入组织的不断增长的数据量，并提供以更加自动化和智能化的方式运营业务的方法，从而消除运营公司所涉及的一些手动步骤。

需要说明的是，微软并不是唯一提供这种智能功能的公司。这是一个不断发展的趋势的一部分，将智能带到企业软件的所有方面，而不考虑供应商。
