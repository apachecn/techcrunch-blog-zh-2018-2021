脸书的 Creator Studio 获得了一个移动伙伴 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/02/24/facebooks-creator-studio-gains-a-mobile-companion/

脸书的创作者工作室增加了一个移动伴侣。面向创作者和出版商的 insights dashboard 于 2018 年 8 月在全球首次亮相，现已成为 iOS 和 Android 的移动应用。与桌面中心类似，Creator Studio 应用程序允许用户跟踪他们的内容在脸书页面上的表现，以及发布、安排和调整帖子，回复粉丝消息等。

脸书北欧娱乐总监安娜·希格斯(Anna Higgs)在上周的伦敦视频会议上登台，与拥有 400 多万脸书粉丝的创作者 Ladbaby 一起分享了新应用发布的消息。

该应用在几个关键领域对创作者和发布者有用，首先是其指标和见解部分。在这里，用户可以分析页面和帖子级别的见解、保留和分发指标，以便相应地调整他们的策略。例如，他们会发现内容性能指标，如“1 分钟浏览量”、“3 秒钟浏览量”和“平均。观看分钟数”，以及参与度指标，如评论和分享、关注人数、收入等。

该应用程序还可以作为查看已发布和预定帖子的移动伴侣，允许创作者进行快速调整，如编辑视频标题或描述。他们可以使用该应用程序删除或终止帖子，重新安排帖子或发布草稿。

从收件箱部分，用户可以在旅途中回复收到的消息和评论。

创建者可以在同一个会话期间在不同的帐户之间切换，而不必以不同的用户身份注销并重新登录。这可能对那些拥有大量社交媒体的人以及那些业务涉及支持多个创建者页面的人有所帮助。

Creator Studio 应用程序还会发送关键里程碑和其他重要事件的即时通知。

这不是脸书第一次为其创作者社区提供专用应用程序。该公司在 2017 年推出了一款 Creator 应用，该应用还提供了统一的收件箱和分析等功能。但是这个应用程序在去年早些时候被关闭了，创作者被指向页面管理器应用程序或者桌面版的 Creator Studio。在此之前，脸书已经提供了一个提及应用程序，仅适用于经过验证的公众人物和页面。

新的 Creator Studio 应用程序不是 shuttered Creator 应用程序的直接替代品，因为它拥有类似但不完全相同的功能集和新的用户界面。它还明显缺乏 Instagram 集成以及上传和发布新内容的能力——后者导致了该应用发布后用户评论的下降。许多人抱怨页面监视器应用程序重叠太多。但缺失的功能是脸书未来可能会解决的问题，因为它会向应用程序推出更多功能。

值得注意的是，脸书的桌面中心和应用程序的名字类似于 YouTube 的创作者服务——YouTube Studio，于 2017 年从 YouTube Creator Studio 更名为。通过在新应用的名称中包含“studio”和“creator”，它将在应用商店搜索结果中表现更好——包括当有人搜索 YouTube Studio creators 应用时出现的结果。这反映了两家公司之间的竞争性质，两家公司都渴望吸引视频创作人才。

脸书的新应用可以在 iOS 和 Android 上免费下载。
