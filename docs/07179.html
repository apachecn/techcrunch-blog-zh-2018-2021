Minute Media为其用户生成的、基于辛迪加的体育出版平台又筹集了4000万美元

原文：https://web.archive.org/web/https://techcrunch.com/2020/02/25/minute-media-40-million-publishing/amp/

谈到互联网，内容可能是王，但在许多情况下，皇帝没有衣服。也就是说，大众可能会点击有趣的故事、视频、音乐和其他媒体，但围绕这些内容建立一个有利可图的业务可能是一场斗争，除了最大的房产之外，基于广告的模式通常不会提供多少利润(即便如此，这也是一个平衡管理成本的艰难举措)。

一家名为 Minute Media 的初创公司认为，它已经找到了一种方法来应对这一挑战，即通过一个平台，将用户生成的内容引入到它自己的一些主要基于体育的媒体资产中，并通过的收购方式进行有机构建，然后将这些资产联合到第三方出版合作伙伴中。

如今，这家初创公司宣布斥资4000万美元继续增长——特别是继续投资其出版平台；投资已经拥有的财产；并进行更多的收购。

由伦敦道恩资本(Minute Media自称总部位于纽约，但其创建团队不在以色列，最初在首席执行官目前居住的伦敦发展)牵头，其他未透露姓名的前投资者(包括Battery Ventures、高盛、ProSieben、Qumra Capital、Vintage Investments、Gemini Ventures、North Base Media、La Maison、Remagine Ventures、Hamilton Lane和Maor Investments)参与，此次融资使Minute Media筹集的资金总额达到1.6亿美元。

这家初创公司没有披露其估值，但去年我们从一位与该公司关系密切的人士那里了解到，估值在2亿至3亿美元之间。鉴于其去年增长了100%左右，目前预计今年的收入为2亿美元，这可能使目前的估值接近4亿至5亿美元。

在巨人的肩膀上

分钟媒体可能不是一个您非常熟悉的名字，但是如果您是在线体育内容的消费者，您可能会遇到它的一些属性或文章。它目前持有的股份有七种，包括像90min.com(专注于足球，因此得名:这家初创公司的创始人兼首席执行官阿萨夫·佩里德是一个足球迷)这样的名字，以及粉丝、球员论坛报和精神牙线。(其他包括12up、DBLTAP和大线索。)

这些微型媒体中的一些是从零开始建立的，但许多是由于当今媒体行业的更大图景而发展起来的:标题被创造出来，获得观众和品牌认可，然后在前一个所有者无法为网站提供商业案例时，在在线出版世界中传播。

例如，FanSided通过收购Minute Media，就在上个月 : Meredith以1500万美元的价格出售了它，这是它在2018年收购欧文时代公司(Owen Time，Inc .)后进行的更大规模的“非核心”资产剥离的一部分(FanSided曾隶属于《体育画报》)。

与此同时，运动员论坛报报道运动员自己写的故事——有些非常及时，如这篇来自塞布丽娜·约内斯库，女篮的一颗新星，刚刚在周一发表，就在同一周，她的名字因她在科比追悼会上的演讲以及她在球场上的英勇表现而引起轰动。这个网站是由棒球传奇人物德瑞克·基特创建的，T10从名人那里筹集了一些资金，但最终无法作为一个独立的创业公司持续下去。它于去年11月被卖给了Minute Media。

在脸书算法改变后，Mental Floss，一个有点像点击诱饵标题的东西(甚至Monica在中央公园翻阅了印刷版本)，迷失了方向。现在它的家也在分众传媒。

从表面上看，这可能主要像是一场聚合媒体游戏，或者在M&A层面上，是基于私募股权模式的东西，即收购许多疲惫或增长缓慢的品牌，目的是优化它们并继续前进。

但实际上两者都不准确。正如Peled所描述的(风投显然也相信)，有一个技术故事，以及相应的有趣的商业模式，支撑着Minute Media建立的横跨B2C、B2B和C2C出版和发行的东西。

首先，有一个运行网站的集中内容管理系统，“一个开放的CMS系统，允许任何偶然的粉丝创建丰富的内容，”Peled告诉TechCrunch。

虽然一度有多达2万名投稿人，以英语以外的各种语言投稿，但其所有平台上每天发布的文章数量(由人工编辑选择)不到1000篇。只有最多产和长期贡献者才能得到报酬；其他人免费投稿。这构成了公司内容引擎的基础。

这些内容为Minute Media自有资产带来了流量和广告，但这只是该公司赚钱的一个方面。这个平台也是一个基于许可的B2B和B2C平台:它链接了大约十几个或更多的其他出版商和媒体合作伙伴，他们使用它来整合内容和从其他地方引入内容。这里的逻辑是，从其他地方引入联合内容可以帮助其他出版商降低运营成本，同时继续扩大自己网站的内容(从而增加流量);因此，他们与Minute Media合作(并为其付费)。

去年夏天，Peled告诉我，广告和授权收入之间的平衡是“大约50/50，但毫无疑问，B2B开放平台更容易销售，增长也更快。”

体育获胜

尽管Minute Media的平台上有体育以外的其他内容，但体育是一个重点，这是有道理的。

体育内容已经成为在线媒体世界中极其重要的一部分。如果做得好，它可以培养出一大批忠诚的访问者——读者、观众、听众——他们愿意做的不仅仅是点击一次，然后继续前进。如果他们喜欢他们所看到的，他们会再回来，一次又一次。这帮助一些更有趣的体育产业建立了付费内容模式——见体育——和其他基于播客等仍在发展的媒体衍生出的媒体帝国——见林格的巨大成功及其最近出售给Spotify 。

Minute Media对如何建立和扩大体育出版帝国有着自己的看法，符合这一大背景。没有一些拖累其他在线出版业务的开销，这家初创公司建立了一个似乎具有可持续性的出版概念。

黎明资本(Dawn Capital)普通合伙人哈康·奥弗利(Haakon Overli)在一份声明中表示:“Minute Media的一流平台使出版商能够创作、发行高质量的内容，并将其货币化。”。“该公司正在迅速成为新一代在线出版的主要参与者，为创作者和观众提供支持。随着2019年收入的爆炸性增长，我们很高兴再次支持该团队，让他们进一步加快R&D和商业努力。”
