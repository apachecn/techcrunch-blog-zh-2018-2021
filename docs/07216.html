安德森·霍洛维茨(Andreessen Horowitz)支持了Run The World，这是一家提供实时在线活动TechCrunch的初创公司

原文：https://web.archive.org/web/https://techcrunch.com/2020/02/27/andreessen-horowitz-has-backed-run-the-world-a-startup-with-a-timely-offering-live-online-events/

由于对冠状病毒的担忧，每天都有另一个与活动相关的取消。就在今天，微软宣布“出于谨慎”，它将不会出席3月中旬的游戏开发者大会。脸书今天还表示，出于对冠状病毒爆发的担忧，将取消原定于5月举行的年度F8会议。

最后一个是特别大的事情。F8是脸书每年举办的最大的活动，所以难怪它计划在网上举办部分活动。

脸书很可能会为此使用自己的技术。但是对于那些现在还在质疑他们的活动计划的其他公司来说，还有一个新的选择，那就是 Run The World ，一家成立一年的公司，总部位于加州山景城，有18个人，在中国大陆和台湾都有小团队。

它在做什么:在疫情时代，把一个会议组织者可能需要的所有功能融合在一起。想想视频会议、票务、互动和网络。

谁在支持它:安德森·霍洛维茨(Andreessen Horowitz)，尽管该公司已经筹集了430万美元的种子资金，但也有投资者GSR Ventures、Pear Ventures、122 West Ventures、unionly Capital，以及脸书子公司Calibra的产品副总裁凯文·韦尔(Kevin Weil)等天使投资者；Patreon联合创始人Sam Yam和捷蓝航空公司董事长乔尔·彼得森。

谁开创了《经营世界》(Run The World):该公司首席执行官小银·屈(音)，她曾负责脸书和Instagram的产品(“基本上是与娱乐影响者和创作者有关的任何事情，”她谈到自己在脸书的部分时间)。

一年后，她放弃了斯坦福大学的MBA课程，去年与前同事宣江一起创办了这家公司，他曾是脸书活动、广告和故事的技术主管。(蒋拥有佐治亚理工学院的计算机科学硕士学位。)

在从牵头a16z交易的普通合伙人康妮陈(Connie Chan)那里了解到这家公司后，我们昨天与曲进行了交谈。

瞿说，创业的动力与她的母亲有关，她母亲是一名专注于脑膜炎的中国医生，她在2018年底前往芝加哥参加了一个会议，在那里她与一名迪拜的医生建立了联系，这位医生能够与她分享一些关于他自己在脑膜炎方面的工作的罕见而有价值的见解。

对于那些经常旅行的人来说，这似乎不是什么特别的事情，但对于曲的母亲来说，这已经是一场磨难了——她必须获得签证；活动前后休假两周，包括旅行日；在机票和住宿上花了一大笔钱——这是她35年来的第一次重要旅行。

正如瞿半开玩笑地说:“这不像在斯坦福，那里定期举行活动，当地医生甚至可以走过去。”

事实上，就像许多为自己或自己所爱的人解决难题的创始人一样，瞿希望创建一个平台，让她的母亲可以定期与人们见面并建立有意义的工作联系，这意味着通过数字化活动远程进行。

事实证明，她的时机非常好。尽管过去有许多初创公司推出了在线直播活动业务(其中许多已经关闭)，但你可以打赌，更多的组织者正在思考，现在运行世界的平台类型正在微调。

瞿说，虽然四个月前才公开推出，但它已经举办了几十场活动，还有数百场活动正在筹备中。它的客户之一是武汉2020，这是一个大型开源社区，有3000多名开发人员，他们将使用该平台作为长途黑客马拉松的一部分，希望为武汉受冠状病毒影响的人提供技术解决方案。

曲还指出，老挝的一个大象保护区最近通过在该平台上组织的一次会议，在两周内从15个国家的捐助者那里筹集了3万美元。该储备的预算有限，但能够以几乎为零的开销(没有场地，没有餐饮)将分散的受众(不仅仅是富裕的捐助者)聚集在一起，使其成为该组织的一大成功。

较小的活动也找到了平台。举个例子，一位专门与工程师合作的约会教练最近举办了一个研讨会。瞿说，只有40人到场，但这个人却从活动中赚了1300美元。

Run The World保持了简单的成本结构，收取门票销售的25%作为交换，以提供给组织者，从他们创建的模板，到门票销售，到支付处理(通过Stripe)，流媒体，在整个活动中实现社交互动，并帮助组织者在事后跟进参与者。

瞿认为，事实上，除了让组织者以更容易接受的价格接触到更广泛的受众，在线活动的一大优势是更有效的网络潜力。例如，Run The World让每一位活动参与者创建一个类似于Instagram故事的视频档案，帮助其他参与者了解他们是谁，而不是走进一个有时很难知道谁在谈论什么的物理空间。

它还组织相关的“鸡尾酒会”,每次为与会者配对几分钟。

当然，流媒体直播也有不好的一面，去年，一名枪手在脸书直播上拍摄了他在新西兰克赖斯特彻奇大规模屠杀数十人的镜头，这让全世界想起了这件事。

人们还可以想象，这些视频资料可能会吸引一些与会者不必要的注意力，他们可能只想观看一场活动。

这些当然是屈和江所熟知的事情。虽然计划是不断增加新功能(可能包括使用LinkedIn验证参与者的身份)，但瞿指出，另一种确保平台上活动质量保持高水平——以及参与者感到安全——的方法是避开大多数免费活动。

她表示，“当组织者招募自己的人，并管理一个由他们认识或表面上可以了解更多的付费与会者组成的社区”时，事情就保持在这个水平之上，并指出付费与会者的人数也要多得多。

她承认，随着“运行世界”规模的扩大，“我们需要找出新的方法。”

当然，在脸书和Instagram吸取的教训应该会有所帮助，因为该公司正在加速发展，围绕其产品创造更多的结构。除此之外，曲补充道，“对我来说，最理想的活动不是200万人参加的活动。我宁愿我们用50个人举办200万场活动。”
