DocuSign 以 1.88 亿美元收购 Seal Software，增强其人工智能能力

合同管理服务 DocuSign 今天宣布将以 1.88 亿美元现金收购 Seal Software 。此次收购预计将于今年晚些时候完成。DocuSign，值得注意的是，此前在 2019 年向 Seal 软件投资了1500 万美元。

Seal Software 成立于 2010 年，尽管它可能不是一个主流品牌，但它的客户包括 PayPal、戴尔、诺基亚和 DocuSign 本身。这些公司将 Seal 用于其合同管理工具，也用于其分析、发现和数据提取服务。正是这些公司随着时间的推移开发的人工智能，帮助企业分析他们的合同，使 DocuSign 收购了该公司。例如，这可以帮助他们显著减少法律审查的时间。

Seal 软件公司首席执行官 John O'Melia 在今天的声明中表示:“Seal 旨在更简单、更快速地从合同中查找、分析和提取数据。“我们与 DocuSign 有着天然的协同作用，我们的团队很高兴能够利用我们的人工智能专业知识来帮助使协议云更加智能。此外，鉴于公司的规模和广阔的视野，成为 DocuSign 的一部分将为我们的客户和合作伙伴提供巨大的机会。”

DocuSign 表示将继续销售 Seal 的分析工具。然而，对 DocuSign 来说更重要的是，它还将利用该公司的人工智能工具来支持其 DocuSign CLM 产品。CLM 是 DocuSign 的服务，用于自动化整个合同生命周期，具有用于创建工作流的图形界面和用于审查和跟踪变更的协作工具等。DocuSign 认为，与 Seal 工具的集成将使其能够为其客户提供“更快、更高效的协议流程”，而 Seal 的客户将从与 DocuSign 协议云的更深入集成中受益。
