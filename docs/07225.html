在2019年向拉丁美洲的创新基金投资了近20亿美元后，软银本月宣布将向该基金追加10亿美元，继续支持该地区的科技创业公司。尽管这家日本投资者面临着继其愿景基金(Vision Fund)之后筹集第二只全球基金的挑战，但软银仍在大举投资拉丁美洲。

该公司在拉丁美洲的早期投资之一——也是在哥伦比亚的第一笔投资——Ayenda Rooms表现尤为出色，本月从Kaszek Ventures筹集了870万美元。Ayenda是Oyo Rooms的当地版本，Oyo Rooms是软银在印度最大的投资之一，尽管上月出现了金融危机，但该公司仍寻求向墨西哥扩张。事实上，该基金最近受到了 《华尔街日报》 的审查，原因是为类似的交付竞争对手优步、Rappi和滴滴提供资金，这表明存在利益冲突。

最近，软银向墨西哥的贷款机构Alphacredit投资了1.25亿美元，据报道，他们计划继续投资这个利基市场。该公司目前在拉丁美洲监管着650多家公司，主要集中在巴西、阿根廷、智利、哥伦比亚和墨西哥，并计划在年底前向17家公司和两家风投公司投资1亿至1.5亿美元。迄今为止，软银超过50%的投资都进入了巴西，其中大部分都在金融科技领域。

墨西哥新银行首次融资1000万美元

随着新玩家Stori的加入，本月墨西哥新银行市场的竞争变得更加激烈，这是一个自我实现的预言。在过去的几个月里，TechCrunch和 商业内幕 都指出，随着Albo、Klar和Nubank等初创公司争夺市场份额，墨西哥的neobank市场是拉丁美洲值得关注的市场。今年2月，数字银行Stori与贝塔斯曼投资公司(Bertelsmann Investments)和源代码资本(Source Code Capital)的1000万美元A轮投资以及现有投资者Vision Plus Capital一起加入了对话。

由中国投资者牵头的这轮融资，是外国基金开始关注拉丁美洲初创企业生态系统(尤其是亚洲风投)这一日益增长趋势的一部分。腾讯投资了巴西的Nubank，该银行已扩张至墨西哥，还投资了阿根廷的Uala，后者正考虑采取类似举措。软银还投资了巴西和墨西哥最大的贷款和信贷创业公司。

Stori将使用 投资 来改进其人工智能技术，因为它试图通过其包容性数字银行服务覆盖超过100，000名墨西哥人。自2018年成立以来，neobank已经从投资者那里筹集了超过1700万美元的资金。

Grow Mobility退出14个城市

今年1月，Rappi和Lime撤回了在拉丁美洲的业务，以便专注于技术而不是快速增长。巴西顶级的移动创业公司Grow Mobility(由墨西哥的Grin和巴西的Yellow的电动滑板车公司合并而成)也退出了。这家在巴西各地提供电动滑板车和自行车共享的初创公司停止了自行车的运营，并从 14 城市撤出了其滑板车。

成长也 重组 其运营通过裁员影响了全巴西的员工，尽管他们没有评论有多少人受到影响。Grow Mobility的滑板车现在将只在里约热内卢、圣保罗和库里蒂巴运营。

这种爆炸式增长后的撤退模式在拉丁美洲最大的初创公司中变得更加普遍，促使这些初创公司专注于提高收入的技术解决方案，而不是只购买市场份额的闪电式措施。

亚马逊网络服务在巴西翻倍

亚马逊网络服务(AWS)宣布将在未来两年内向圣保罗投资2.36亿美元(10亿雷亚尔)，以加强其在拉丁美洲的基础设施。这一努力可能是亚马逊巩固在拉丁美洲竞争日益激烈的电子商务市场的市场份额的工作的一部分，在这个市场上，像MercadoLibre这样的传统参与者仍然占据主导地位。这项投资将使亚马逊能够扩大其巴西数据中心，并改善向私人和公共合作伙伴提供的本地服务。

亚马逊还宣布将在巴西北部的伯南布哥州建立一个新的配送中心，以支持全国的销售。巴西几乎占据了拉丁美洲电子商务市场的40%，这使得该国对亚马逊在该地区的定位至关重要。

新闻和笔记:Weel、Global 66、Yuca和Memed

巴西应收账款管理平台Weel宣布，巴西第七大银行Banco Votorantim将于2020年2月投资1840万美元。这笔投资是Banco Votorantim继2019年【600万美元 出资后，在该初创公司的第二笔投资。Weel将利用这笔投资探索在巴西的扩张，以及探索智利和墨西哥市场。

智利国际转让初创公司Global 66于2月从英国投资者Venrex获得了325万美元，以继续其在该地区的扩张。这家初创公司目前提供的费率是现有转账服务的八倍，尤其是在拉丁美洲地区。Global 66最近在秘鲁开设了新的办事处，并计划在未来两年内扩展到哥伦比亚、阿根廷和墨西哥。在短短两年的运营时间内，Global 66已经为全球60个城市的25，000多名用户处理了交易。

巴西proptech公司Yuca从Monashees、ONEVC和Creditas筹集了470万美元，帮助对抗巴西最大城市的住房危机。随着巴西城市的扩张——圣保罗是世界上最大的城市之一——Yuca为想要缩短通勤时间的年轻人创造了中央共居空间。受中国初创公司Ziroom的启发，Yuca将年轻的千禧一代专业人士作为客户目标，并计划到今年年底扩大到500套公寓。

巴西的数字处方初创公司Memed最近从DNA Capital和Redpoint eVentures筹集了450万美元，以改善当地医生和患者的处方系统。今天，Memed拥有超过80，000名注册医生，他们开出了超过1，000万张处方，价值超过2.37亿美元。据说，Memed的100%数字处方可以提高巴西复杂的官僚医疗系统的安全性和效率。

虽然巴西仍处于拉丁美洲科技生态系统的前沿，但墨西哥的金融科技公司正在慢慢崛起，尤其是在国际投资者的额外支持下。2020年是一个强劲的开端，暗示着拉丁美洲科技投资又一个潜在的破纪录的一年。
