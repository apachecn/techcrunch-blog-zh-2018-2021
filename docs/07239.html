Zombs Royale 背后的创业公司 End Game 融资 300 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/02/28/end-game-seed-funding/

端游互动(End Game Interactive)首席执行官杨·c·刘(Yang C. Liu)对他和他的联合创始人卢克·兹比里杰(Luke Zbihlyj)的计划有一个令人耳目一新的直截了当的描述:“我们只是在开发游戏。老实说，我们不知道自己在做什么。”

尽管自称无知，但 End Game 刚刚从一群令人印象深刻的投资者那里筹集了 300 万美元的种子资金:这轮投资由专注于游戏的公司 Makers Fund 牵头，参与投资的有《部落冲突》开发商 Supercell、Unity 首席执行官大卫·赫尔加松、Twitch 首席运营官·林义杰、Twitch 副总裁休伯特·蒂埃博、Main Street Advisors 的丹尼·爱普斯丁和亚历山大·科恩以及音乐高管斯科特·布劳恩。

刘告诉我，他和 Zbihlyj 是通过建立与现有游戏相关的网站起家的，比如在《神奇宝贝 GO》中寻找神奇宝贝的网站。然而，他们受到了简单的基于浏览器的多人游戏的启发，如 Slither.io 创造了他们自己的游戏——先是 Zombs.io ，然后 Spinz.io ，然后 Zombs Royale 。

End Game 表示，它的游戏总共吸引了超过 1.6 亿名玩家，单日游戏人数达到 100 万。尤其是《僵尸王》( Zombs Royale)似乎很受欢迎——这款游戏是 2018 年美国谷歌搜索次数最多的游戏之一。

刘表示，该团队的成功让他们将精力集中在游戏开发上:“我们是想做人们简单使用的产品，还是想做人们上学、工作或做梦时想出来的游戏？”

Zombs Royale 应该在不到四周的时间内完成，但刘说，在 2018 年初推出后，团队花了一年的大部分时间来维护和扩展游戏。然后，2019 年是建立一支团队和创造下一款游戏的日子，命运竞技场，这是新汽车象棋流派中的一款游戏，预计将很快在 PC、移动和其他平台上推出。

刘指出，与《末代游戏》之前的作品不同，它的特色是简单的 2D 艺术(“在《僵尸王》和《斯平兹》上，我做了艺术，很糟糕)。《命运竞技场》将以“3D、高保真艺术风格”为特色

但即使该公司的游戏开始看起来不那么原始，目标仍然是快速开发和迭代。刘说，他希望通过这一轮种子期，资助“许多尝试”来开发其他跨平台、多人游戏。

“我们为快速实验感到自豪，”他说，并补充说关键是“不要贪多嚼不烂”。我们从一开始就设计(我们的游戏)规模。我们不一定要成为魔兽世界，在那里你需要做 100 个任务作为基线。我们专注于起点很小，但可以发展成更大规模的游戏。”

Supercell 开发人员关系主管 Jaakko Harlas 在融资公告中的一份声明中提出了类似的观点:

许多公司很快指出他们的行动有多快。然后你会遇到这样的团队，并意识到精益和快速移动的真正含义。杨、卢克和他们的团队已经证明了他们可以推出那些展示真正娱乐天赋的无障碍游戏，我们期待着支持他们开发下一款热门游戏。
