Ampere 推出全新芯片，专为云工作负载打造

由前英特尔总裁蕾妮·詹姆斯运营的芯片初创公司 Ampere 今天宣布了一款新芯片，她说这款芯片是专为云工作负载优化而设计的。

Ampere 产品副总裁 Jeff Wittich 表示，这款新芯片名为 Ampere Altra，其设计的一些功能应该会对云提供商产生吸引力。这涉及三个主要焦点，包括高性能、可扩展性和能效，所有这些要素对于大规模运营的云供应商来说都很重要。

Altra 是一款 ARM 芯片，具有一些强大的功能。“它是双插槽平台中的 64 位 ARM 内核或 160 内核，我们支持单插槽和双插槽[配置]。我们以 3 GHz 睿频运行，所有内核都是 3 GHz，因为云提供计算的方式，您尽可能多地利用所有内核。因此，我们的睿频性能针对所有内核进行了优化，能够始终保持这种性能，”Wittich 解释道。

该公司将这种芯片视为云计算的一种主力。“我们在设计一款专为云环境打造的通用 CPU 时，确实考虑到了这一点，因此您可以像云利用这种类型的计算一样利用这种计算。因此，它支持在云中运行的大量工作负载，”他说。

创始人兼首席执行官詹姆斯表示，该公司一直在与他们的云客户合作，为他们提供所需的信息，以便在粒度配置级别上为他们的个人工作负载优化芯片，这是超大规模计算者特别需要的。

“让我们尽我们所能来构建一个平台，提供原始功能和性能，提供您正在寻找的那种环境，然后有一种设计方法，使他们能够与我们合作，处理重要的事情和控制，这种独特的功能集，因为他们每个人都有自己的软件环境，”James 解释道。

詹姆斯说，交付这种芯片的一个不可预见的挑战是，由于 Corona-19 病毒及其对亚洲的影响，供应链可能会中断，许多部件来自亚洲，芯片是在亚洲组装的。

她表示，该公司已经考虑到这一点，并已经能够建立一个全球供应链，她希望这将有助于解决因供应链放缓而可能出现的问题。
