Sendbird 的消息平台获得了新的审核工具、全文搜索等等

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/03/sendbirds-messaging-platform-gets-new-moderation-tools-full-text-search-and-more/

Sendbird 今天宣布对其网络和移动聊天 API 进行重大更新。凭借这一点，Sendbird 正在为其服务添加许多新功能，使其客户更容易审核图像，为他们的特定需求构建定制审核工具，并在他们的聊天界面中添加反应和交付收据。

这家成立于首尔的公司表示，它目前为全球约 9000 万活跃用户提供服务，每月发送约 15 亿条信息。Sendbird 首席执行官约翰·金告诉我，该公司在按需服务领域尤其强大，客户包括 Gojek、iFood 和 Delivery Hero，以及 Carousell、Paytm、Traveloka 和 SSG 等市场服务，以及 Reddit、Dream 11 和雅虎幻想体育等社区网站(披露:雅虎为 VMG 所有，因此与 TechCrunch 拥有相同的母公司)。

“我们已经看到了医疗保健领域最近的大量增长，无论是护理提供者到患者的聊天，还是福利管理，”他补充道。“这些往往是数字优先的颠覆者，如 Accolade、Livongo 和 Grand Rounds。我们在游戏、约会和直播领域拥有大量客户。”

至于今天的更新，相当一部分集中在适度上。例如，有一个新的图像审核功能，以及一个新的正则表达式脏话过滤器，为版主提供了许多如何阻止特定信息通过的选项。Sendbird 现在还提供了一个符合 GDPR 标准的 API 和一个 moderation API，客户可以使用它们来扩展公司的内置选项。

此外，当服务中断时，离线同步可以在本地缓存消息，按需翻译可以增强 Sendbird 的自动翻译功能和翻译推送通知的能力。

这种对翻译的关注并不令人惊讶。Sendbird 拥有全球客户群，该公司长期以来一直是 APAC 地区的强大参与者。然而，金告诉我，现在它的客户群相当均匀地分布在美国、EMEA 和 APAC。

然而，用户最有可能注意到的是，Sendbird 的客户现在可以在他们的聊天界面上添加反应，这让他们感觉更像是智能手机上的一个现代通讯应用程序。除此之外，Sendbird 还增加了送达回执和全文搜索功能。

“如果你仔细想想，聊天/信息是技术(电话、传真)和互联网(聊天室、IRC 和 ICQ)的最初用途之一，”当我问他为什么认为应用内聊天现在正在兴起时，金说。“在台式机时代，当你在家时，聊天受到使用调制解调器的限制。但是，移动时代和廉价的 3G/4G 网络和 Wi-Fi 接入已经打开了全球和民主化的聊天渠道。随着智能手机的普及，特别是在发展中国家和人口稠密的国家，人们绕过了台式机和电子邮件等旧技术，聊天成为了常态。”

他认为，在即时通讯领域，网络效应将市场推向少数玩家(想想 WhatsApp、微信和 Facebook Messenger)——随着时间的推移，客户只是希望从他们日常打交道的公司那里获得同样的便利。

“现在其他人都必须赶上来，确保他们的经历符合新常态，”他指出。“但大多数人没有才能或专业知识去实现目标或参与竞争。因此，聊天应用编程接口现在越来越流行，越来越普及。”
