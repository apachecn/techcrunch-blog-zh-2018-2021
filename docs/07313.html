随着微服务方法在 web 开发中的发展，Netlify 获得了 5300 万美元的 C 轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/04/netfily-nabs-53m-series-c-as-micro-services-approach-to-web-development-grows/amp/

图片来源:托马斯·巴维克/盖蒂图片社

Netlify ，一家想要杀死网络服务器并改变开发者建立网站方式的初创公司，今天宣布了一笔 5300 万美元的 C 轮投资。

EQT Ventures Fund 领投，现有投资者 Andreessen Horowitz 和 Kleiner Perkins 以及新来者 Preston-Werner Ventures 提供了资金。根据交易条款，EQT Ventures 的交易合伙人兼投资顾问劳拉姚(Laura Yao)将加入 Netlify 董事会。据该公司称，这家初创公司现在已经筹集了 9700 万美元。

像最近的许多创业公司一样，Netlify 的联合创始人克里斯·巴赫说，他们并没有寻找新的资金，但随着公司的快速增长，他们觉得拿这笔钱来帮助继续增长是明智的。

虽然巴赫和首席执行官马特·比尔曼不想讨论估值，但他们说这是“非常慷慨的”，符合他们对自己业务的看法。他们也不想透露具体的收入数字，但确实表示该公司连续三年的收入增长了两倍。

推动这一增长的一个因素是加入该平台的开发者数量。当我们在 2018 年与公司就其 B 系列进行对话时，它有 30 万注册用户。今天，这个数字已经膨胀到 80 万。

正如我们在 2018 年的一篇文章中对该公司的描述，它希望改变人们开发网站的方式:

“Netlify 已经抽象出了网络服务器的概念，它说网络服务器部署缓慢，难以保护和扩展。通过从单一网站转向具有后端微服务的静态前端，它相信它可以解决安全和扩展问题，并更快地交付网站。”

虽然开发人员的受欢迎程度是一个很好的起点，但获得更大的客户是推动更多收入的最终目标，该公司希望利用新注入的资本来建立企业业务。目前的企业客户包括谷歌、脸书、思杰和联合利华。

自去年年初以来，Netlify 的员工已从 38 人增加到 97 人，并希望到今年年底达到 180 人。
