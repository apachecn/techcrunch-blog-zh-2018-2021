Lyft 是最新一家因冠状病毒而让员工回家的科技公司

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/05/lyft-is-the-latest-tech-company-to-send-employees-home-over-coronavirus/

Lyft 是最新一家因冠状病毒而让员工回家的科技公司

拼车公司 Lyft 在得知一名员工与冠状病毒或新冠肺炎病毒接触后，建议其旧金山员工回家。Lyft 发言人 Alexandra LaManna 告诉 TechCrunch，该团队成员没有表现出任何症状，并且正在与医疗专业人员联系。

“我们响应过程的每一步都基于 CDC 的指导，出于谨慎，我们鼓励旧金山总部的员工在本周剩余时间里在家工作，”Lyft 在一份声明中分享道。

拉曼纳还表示，Lyft 总部将有一个“加强清洁过程过夜。”

这是众多科技公司中的又一个回应，这些公司将员工送回家，以限制冠状病毒在员工中传播的机会。本周早些时候，Twitter 鼓励所有员工在家工作。亚马逊、LinkedIn、微软和谷歌也建议一些员工远程工作，因为害怕暴露。

新冠肺炎对科技的连锁反应不止于员工。一些备受瞩目的会议已经取消，包括脸书的 F8 会议和谷歌的物理部分 Cloud Next。SXSW 和 Y Combinator 演示日尚未透露他们的独立会议是否会继续，这些会议聚集了数千人。

冠状病毒也开始影响市场，微软称此次爆发是供应链问题和影响收益的原因。
