脸书承诺向受企业冠状病毒影响的“临时”员工支付薪酬

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/06/facebook-commits-to-paying-contingent-workers-affected-by-corporate-coronavirus-response/amp/

图片来源: Getty Images

昨晚，所有公司都应该效仿的一个举措是，微软宣布将继续支付因公司应对美国冠状病毒爆发而受到办公室关闭影响的小时工

根据该公司的一份声明，现在脸书似乎也在效仿:

“我们正与供应商密切合作，以确保我们优先考虑团队的健康和安全。脸书将支付临时工工资，这些临时工在在家志愿工作期间，当我们关闭办公室，当我们选择让员工回家，或者当他们生病时，由于人员需求减少而无法工作，脸书公司发言人克洛伊·梅耶尔在一封电子邮件中写道。

在宣布微软决定的博客文章中，公司总裁布拉德·史密斯写道:

“虽然保护公众健康的工作需要加快，但经济不能放缓。作为一家公司，我们致力于将公共健康作为我们的第一要务和尽我们所能应对新冠肺炎的经济和社会影响。我们明白，大企业负担得起的东西，小企业可能负担不起，但我们认为，有能力采取这种措施的大企业应该考虑这样做。”

几家科技公司要求员工在已经发现新冠肺炎病例的地方在家工作，比如华盛顿州和加利福尼亚州，包括谷歌、Lyft 和 Square。对新冠肺炎的担忧也导致了重大活动的取消，比如世界移动通信大会和谷歌 I/O 开发者大会。

近年来，科技公司建立了一个双重工人体系，让更多技术和产品导向型员工成为主要公司的全职工人，同时向第三方公司输出劳动力要素。这种基于高等级的系统既令人惊讶又令人愤怒，尤其是当 tech 著名的巴士受到公众攻击的时候。提高薪酬或所有员工不仅是良好的公共关系，也是良好的职业道德。

或许目前的形势将有助于更多的科技公司将更多的员工引入公司内部。无论是 Alphabet 对临时工的依赖，还是脸书对调节工作的外包，或者只是雇佣其他公司来管理他们的业务，内部外包可能太多了。但至少小时工不会在疫情期间减薪。这是一个较低的门槛，也不是所有的科技公司都已经通过了，但总比什么都没有好。
