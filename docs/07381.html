Box 现在允许所有员工在家工作，以降低冠状病毒风险 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/09/box-is-now-letting-all-staff-work-from-home-to-reduce-coronavirus-risk/

Box 加入了许多支持员工在家远程工作的科技公司，以应对新型冠状病毒的爆发。

它将该政策应用于所有员工，不管他们在哪里。

昨天晚些时候，Box 联合创始人亚伦·列维(Aaron Levie)在推特上发布了一份声明，详细说明了这家云计算公司对冠状病毒引起的疾病名称新冠肺炎的回应，用他的话说，就是“确保我们服务的可用性和员工的安全”。

最近几天，Twitter 也同样鼓励所有员工在家工作。而包括亚马逊、谷歌、LinkedIn 和微软在内的公司也建议一些员工远程工作，以降低接触病毒的风险。

Box 在其回应声明中写道，它制定了业务连续性计划，“以确保核心业务功能和技术在任何潜在中断的情况下都可以运行”。

“我们早就认识到由于不利事件(如地震、停电或像新冠肺炎这样的公共卫生危机)导致服务中断的潜在风险，这会影响我们的战略、运营、利益相关方和客户义务。这就是为什么我们有一个业务连续性计划，提供必要的政策和计划来保护 Box 的运营和 T2 的关键业务功能，”该公司写道。

在“劳动力弹性和业务连续性”一节中，该公司指出，在家工作是其业务运营的一个正常部分，但表示，它现在将这一选项扩展到所有员工，无论他们通常在哪里办公——称这是“出于新冠肺炎期间的大量谨慎”。

该公司表示，为进一步降低风险而采取的其他措施包括暂停所有国际旅行，限制非必要的国内旅行；减少大型客户活动和聚会；强调所有办公场所的健康和卫生——“保持卫生用品，鼓励‘如果你生病了，就呆在家里’的心态”。

该公司还表示，它实际上正在进行所有新员工入职培训和候选人面试。

Box 列举了一些它经常用来支持移动和远程工作的工具，包括它自己的安全内容协作服务；Zoom 的视频通讯工具；Slack 消息应用程序；Okta 用于安全 ID；加上额外的未命名的“关键云工具”，以确保“所有员工不间断的远程工作”。

显然，随着越来越多的公司由于新冠肺炎的担忧而转向远程工作，Box 的帖子还链接到了其自己的云计算工具的免费培训资源，这是在窥探新用户的机会。

本报告更新了一项更正，澄清新冠肺炎是由新型冠状病毒引起的疾病；而不是病毒的另一个名字
