欧洲初创公司称赞欧盟委员会重新考虑股票期权的计划TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/10/european-startups-applaud-commission-plan-to-rethink-stock-options/

创业公司欢迎欧盟委员会旨在减少繁文缛节和缩小小企业跨境壁垒的建议，作为今天公布的新欧盟产业战略计划的一部分。

在欧盟执行机构提出的一揽子措施中，成员国签署“创业国家标准”——旨在推广支持创业和扩大规模的最佳实践，如一站式商店、有利的员工股票期权安排和签证处理，以减少企业家在欧盟创办和发展企业的跨境摩擦。

近年来，欧洲初创公司组织了一场围绕股票期权规则的改革运动——来自本土初创公司的30名首席执行官，包括TransferWise、GetYourGuide、Revolut、Delivery Hero、TypeForm和Supercell(仅举几例)两年前签署了一封致政策制定者的公开信，呼吁立法者修正他们所谓的“管理员工所有权的不完整、不一致且往往是惩罚性的规则”

这一努力似乎对欧盟的政策制定领域产生了影响。监管和实际障碍现在都在欧盟委员会的视线之内，它提议成立一个联合工作组来解决商业障碍。

它今天还重申了一项长期警告，反对成员国通过在泛欧盟规则上增加自己的条件来“镀金”。

“单一市场是我们最自豪的成就——然而，70%的企业报告说，他们发现它没有充分整合，”EVP Margrethe Vestager说，他提出了一个工业战略包，主要关注较小的公司，包括那些雄心勃勃要扩大规模的公司。“在整个欧洲，障碍仍在阻止初创公司成长为欧洲企业，我们的报告正在识别这些障碍，然后我们也在单一市场执法行动计划中解决它们。”

在一封回应欧盟委员会欧盟创业国家标准计划的信中，14位欧洲创业公司创始人(如下所列)和一些欧洲创业公司协会对该提议表示欢迎，敦促欧盟成员国支持该提议。

“通过让创业、跨境扩张和吸引顶级人才变得更容易，这一新标准将有助于与美国和中国强大的全球科技中心建立公平的竞争环境，”科技首席执行官和创业倡导组织写道。“我们赞赏欧盟寻求泛欧洲解决方案来满足初创企业需求的雄心。我们还感到鼓舞的是，委员会特别指出，股票期权的处理是关键问题之一。

“正如参加Not Optional运动的500多名欧洲领先企业家所强调的那样，初创公司无法有效利用股票期权来吸引和留住人才，这是欧洲初创公司增长的一个主要瓶颈。”

他们补充称:“欧盟委员会的提议将是向释放欧洲全部创业火力迈出的重要一步——但前提是所有成员国都采纳并实施这些提议。”。“这就是为什么我们今天呼吁所有成员国签署欧盟创业国家标准，包括承诺增加员工持股计划的吸引力。”

Index Ventures的合伙人马丁·米尼奥也对股票期权提议表示欢迎，他是“不可选择运动”的另一位支持者，他说:“如今初创公司面临的最大挑战是招募和留住顶尖人才。这就是为什么我们很高兴欧盟创业国家标准涉及股票期权，使创业公司更容易让员工分享他们的成功。”

他在一份声明中补充说:“我们很高兴看到欧盟委员会认可初创公司对欧洲及其公民的贡献，并寻求一项泛欧洲政策倡议来支持这个不断增长的行业。”“长期以来，欧洲的焦点一直是驯服美国科技巨头。今天的声明证实了欧洲创造自己冠军的雄心。”

欧盟创业倡导成员协会(Allied for Startups)是这封信的另一个签署者。在另一个回应中，它广泛欢迎欧盟委员会的中小企业战略，同时强烈要求将初创企业作为实施该战略的独立行为者，而不是中小企业的子类别。

该游说团体的欧盟政策主任贝内迪克特·布洛梅尔(Benedikt Blomeyer)在一份声明中表示:“拟议的创业国家标准对人才的关注对创业生态系统具有重大潜力，因为获得人才仍然是欧洲创业公司的瓶颈。”。

“通过中小企业战略，我们很高兴看到更好的创业签证和改进的员工股票期权等具体措施摆在桌面上。Allied for Startups在过去几年里多次呼吁采取这两项措施。

“与中小企业不同，创业公司只能在规模上取得成功，”他补充道。“它们从第一天起就走向全球，目标是快速做大。适用于中小企业的具体措施，例如监管豁免，可能不适用于初创企业。相反，这可能会激励初创企业保持小规模。为了解释这些差异，欧盟委员会应该考虑一种侧重于可扩展性的创业战略，作为中小企业战略的补充。”

Allied for Startups也对委员会减轻单一市场内初创公司的行政和监管负担的总体目标表示欢迎——称将监管沙盒作为支持工具包的一部分的考虑“对初创公司具有潜在价值，他们建立了创新的产品和服务。”

欧盟委员会还希望支持中小企业在欧洲上市——在InvestEU SME window下宣布成立中小企业首次公开募股(IPOs)基金，旨在使当地小企业更容易进行IPO。

另一项举措旨在减少中小企业的逾期付款，欧盟委员会今天指出，四分之一的地区小企业因未能按时付款而破产。

它还表示，希望刺激对女性领导的公司和基金的投资，以“赋予女性企业家精神”(值得注意的是，上述信件的所有签名者都是男性。)

工业向数字化转型

更广泛地说，欧盟委员会的新工业战略旨在巩固未来五年的核心欧盟政策优先事项，包括重点推动传统工业的数字化，同时根据泛欧盟绿色协议重新调整以过渡到碳中性经济。

“欧洲拥有世界上最强大的工业。我们的公司——无论大小——为我们提供工作、繁荣和战略自主权。内部市场专员蒂埃里·布雷顿(Thierry Breton)在今天的一份声明中表示:“在新的地缘政治背景下，管理绿色和数字化转型并避免外部依赖需要彻底的变革，而且需要从现在开始。”。

在新闻发布会上，Vestager强调了委员会的观点，即需要新的和更具包容性的工作方法来实现计划的转型。

“数字和绿色的双重转型为整个行业，尤其是中小型企业带来了机遇和挑战。商业模式正在改变。她说:“整个欧洲的公司都面临着消费者信任度下降、透明度需求增加的问题。我们周围的世界也在变化……今天，全球竞争、贸易争端、保护主义的回归——我认为这造成了一种共同的不确定感。

“这是对欧洲工业的挑战，因为它正着手迎接双重转型。幸运的是，欧洲工业正从强势走向这一现实。我们的新战略建立在欧洲的实力和我们的价值观之上。”

关于转向“包容性”工作方法的建议，韦斯塔格尔说，目标是“与大小公司、成员国、研究人员、学术界、社会合作伙伴和其他欧盟机构更密切地合作。”

为此，欧盟委员会提议设立一个新的工业论坛，以便与这些利益攸关方进行更密切的合作，欧盟委员会的目标是在9月份之前设立该论坛。

它还希望致力于确定一些工业生态系统Vestager说，在政策支持方面，“可能需要定制的方法”。

在新闻发布会上，Breton表示可能会有15到20个这样的工业生态系统。

“我们不想漏掉任何人，”他说。“这是一项产业战略，但我们都知道，支撑这一战略的有大公司，但也有很多很多小公司，我们必须让这些公司参与进来。如果没有大企业和小企业，我们就不会有一个充满活力、创新和生机的行业。

“事实上，许多公司已经在自己内部这样做了，但我们确实希望在整个欧盟和内部市场中以更加横向的方式进行，”他补充道。

技能是中小企业战略的另一个重点——欧盟委员会表示，它将把数字创新中心扩展到欧洲的每个地区，以帮助小企业加入尖端技术，并为志愿者和数字技术培训提供更多选择。

帮助中小企业找到他们需要的技能，以转向可持续的工作方式是另一个明确的目标。

委员会已经发布了一份关于工业战略的T4报告。

上个月该执行机构还提出了旨在鼓励工业数据共享和再利用的提案，以及监管人工智能高风险使用的提案。

今年晚些时候，欧盟数字政策的另一项重要内容是即将出台的数字服务法案，该政策旨在解决平台责任和责任，包括对依赖平台作为市场的较小企业的责任和责任。
