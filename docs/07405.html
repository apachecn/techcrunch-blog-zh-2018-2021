Stitch Fix 的大幅下滑标志着科技创业公司的高增长障碍

大家好，欢迎光临回到我们平常的早间话题，看看私营公司、公开市场以及它们之间的灰色地带。

数字化“造型服务”公司 Stitch Fix 的股票今天上午大幅下跌，此前其收益未能激发公开市场投资者。该公司在 2 月份时每股价值还超过 29 美元，今天开盘价仅为 14.75 美元。Stitch Fix 股票在昨天的大范围市场抛售中下跌了近 7%。今天又跌了 30.48%。

该公司盈利且不断增长，这使得其股价下跌似乎超出了近期风险投资支持的 IPO 的正常水平。根据 Crunchbase 的数据，Stitch Fix 作为一家私营公司，在其生命周期内从投资者那里筹集了超过 7900 万美元，这些投资者包括 Baseline Ventures 、light speed Venture Partners和 Bessemer

Stitch Fix 于 2017 年上市，是科技企业的一个很好的例子。它的利润率比我们从软件商店等技术领先的公司看到的要低。它使用现代技术方法如数据科学来帮助其消费者推荐，提高其为用户提供的价值，并增加其支出份额。但该公司最终销售由人类选择中介的第三方实物商品。因此，这是一个可能有吸引力的经济业务，但它不会接近软件即服务(SaaS)的倍数，正如我们将看到的。

我们最近报道了公众市场对科技公司价值的看法，与纯科技公司的估值形成对比；这是一个重要的话题，因为今天市场上的初创公司可能更像后者，其经济状况更像前者。我们知道 Casper 在今年早些时候上市时就遇到了这种情况。

让我们来看看 Stitch Fix 的季度业绩、预测以及这些数字为何未能达到市场预期。我们将梳理出哪里出了问题，然后根据今年其他科技企业的情况来考虑该公司的业绩，看看对毛利率低于 50%的初创公司是否有借鉴意义。
