Twitter改写了开发者政策，以更好地支持学术研究和“好”机器人的使用

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/10/twitter-rewrites-developer-policy-to-better-support-academic-research-and-use-of-good-bots/

Twitter今天更新了其开发者政策，澄清了包括学术研究在内的数据使用规则，以及其对机器人的立场等。Twitter表示，为了简化所使用的语言，使其更具对话性，该政策也被彻底改写。新政策已经从八个部分缩短为四个部分，与之配套的Twitter开发者协议也已经更新，以适应政策的变化。

新政策最显著的更新之一是规则的改变，以更好地支持非商业研究。

该公司指出，Twitter数据被用于研究垃圾邮件、滥用和其他与对话健康相关的领域，并希望这些努力继续下去。修订后的政策现在允许使用Twitter API进行学术研究。此外，Twitter正在简化其关于Twitter数据的再分发的规则，以帮助研究人员。Twitter表示，现在，研究人员将能够共享无限数量的Tweet IDs和/或用户id，如果他们是代表学术机构，并且只是出于非商业研究的目的，如同行评议。

该公司还在修改规则，以澄清当Twitter数据的用例发生变化时，开发者应该如何处理。在新政策中，开发人员被告知，他们必须通知公司对其用例的任何“实质性”修改，并在为此目的使用Twitter内容之前获得批准。Twitter警告说，不这样做将导致他们的API和数据访问暂停和终止。

该政策还概述了何时何地允许“Twitter外匹配”的情况，这意味着Twitter账户何时与使用其他数据建立的个人资料相关联。要么开发人员需要获得相关用户的同意，要么他们只能在信息由用户提供或基于公开数据的情况下进行开发。

上述变化的重点是确保Twitter数据在用于学术研究等有价值的事情时是可访问的，并保护它免受更多有问题的用例的影响。

最后，修改后的政策澄清了并非所有的机器人都是坏的。该公司表示，有些甚至增强了Twitter的体验，或者提供了有用的信息。作为好的机器人的例子，Twitter指出了有趣的帐户@ everycolorbot和信息丰富的【earthquakesSF】。T11】

Twitter将机器人定义为“通过我们的API以某种方式自动创建、发布和交互推文或直接消息”的任何账户。

展望未来，开发人员必须说明他们是否在运营一个机器人账户，这个账户是什么，以及它的幕后操纵者是谁。Twitter解释说，这样一来，“Twitter上的每个人都更容易知道什么是机器人，什么不是。”

当然，那些出于更邪恶目的操作机器人的人——比如传播宣传或虚假信息——很可能会无视这一政策，并希望不被发现。这一特殊的变化是在最近发现四分之一关于气候变化的推文来自发布气候变化否定论信息的机器人之后。此外，最近发现特朗普的支持者和QAnon阴谋者正在使用名为Power10 的应用程序将他们的推特账户变成机器人。

Twitter表示，自2018年7月推出新的开发者审查流程以来，它已经审查了超过100万份开发者申请，批准了75%。它还在过去六个月暂停了超过144，000个来自不良行为者的应用程序，并修改了其开发者应用程序，使其更易于使用。它现在正在开发下一代Twitter API 的，并继续探索新产品，包括通过其测试项目 Twitter开发者实验室。
