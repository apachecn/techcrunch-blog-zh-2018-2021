GitHub 为教师推出新工具，包括 autograding TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/12/github-launches-new-tools-for-teachers-including-autograding/amp/

图片来源: GitHub

GitHub 今天宣布了 GitHub 教室的新功能，这是一套帮助计算机科学教师分配和评估编码练习的工具，以及一套新的免费教师工具。

第一个是亲笔签名，这正是名字所承诺的。教师现在可以将测试添加到他们的工作流中，自动测试作业并相应地给作业评分。然后，这些测试会在每个学生存储库上自动运行。更重要的是，教师还可以通过在线反馈和自动拉取请求来提供具体的反馈。

亲笔签名当然会节省老师相当多的时间。与此同时，GitHub 还推出了教师工具箱，这是一套供教师使用的免费工具，包括许多开发工具、教程、域名等。这些免费服务包括。技术域名，访问 BrowserStack 进行设备上测试和 Termius SSH 客户端。他们还可以免费使用 Arduino 的工具。

目前共有 35 种工具和服务可通过工具箱获得。值得注意的是，这些工具与 GitHub 学生开发包中的工具大部分不同，后者还提供免费和打折的服务，如 Bitnami、Figure Eight、Namecheap 和其他服务(当然包括 GitHub 本身)。
