微软发布Windows漏洞紧急补丁

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/12/microsoft-releases-emergency-patch-for-leaked-windows-10-security-bug/amp/

图片来源: 彭博 / Getty Images

微软发布了针对本周早些时候过早披露的一个安全漏洞的Windows补丁。

这个“严重”级漏洞的细节于周二发布，作为该软件巨头典型的每月安全补丁发布的一部分，它被称为“补丁星期二”。该漏洞存在于最新版本的Windows服务器消息块中，称为SMB，它允许Windows在网络上和互联网上与打印机和文件服务器等设备进行通信。

成功利用SMB漏洞可能允许攻击者在任何易受攻击的计算机上远程运行恶意代码。

Kryptos Logic的安全和威胁情报研究主管杰米·汉金斯(Jamie Hankins)说大约48，000台联网的未打补丁的服务器容易被利用。但是这个数字可能要大得多，因为它没有考虑到连接到这些未打补丁的服务器的所有其他易受攻击的计算机。

关于该漏洞的消息引发了人们对攻击者可能发起“蠕虫式”攻击的担忧，这种攻击会在网络上迅速传播，就像2017年的WannaCry攻击。

现在两天后，微软有补丁修复Windows 10和Windows Server 2019版本1903和1909的漏洞。

早期版本的Windows——包括微软最近停止提供补丁的Windows 7——不受该漏洞影响。

目前还不清楚是什么导致了这次无意中的泄露。安全公司Fortinet和思科的研究人员发布了描述该漏洞的博客帖子，但后来删除了对该漏洞的引用。

这些修补程序可通过典型的更新机制获得，包括Windows Update。
