骑行报告筹集1000万美元帮助城市和运营商管理共享自行车和滑板车

总部位于波特兰的微移动创业公司Ride Report刚刚完成了由不寻常的风险投资公司牵头的1000万美元的一轮融资，以更好地促进城市和运营商之间的关系。

Ride Report 通过监测各个城市的微型汽车(目前全球有60辆)来帮助立法者更好地了解微型汽车对各自城市的影响。Ride Report与运营商合作，提供城市数据、报告和监测工具，以支持政策指导。

“微动力正进入其历史上的一个困难点，”Ride Report首席执行官威廉·亨德森告诉TechCrunch。“我们非常看好市民和居民对更高效的交通方式的需求，这种方式可以让他们从一个地方到达另一个地方，但也有一些成长的烦恼。”

这就是为什么Ride Report致力于在私人运营商和公共实体之间建立更加合作和信任的关系。有了额外的资金，我们的计划是壮大团队，并开始关注新兴市场。

亨德森说:“好消息是，当谈到如何管理微移动性时，每个人都在同一条船上。”“这是一个新事物，没有真正的剧本。这是个好消息，因为在与政府官员和监管机构合作时，你很少有这么大的试验空间。欧洲在隐私和GDPR方面会有独特的限制，但我对此感到兴奋，因为欧洲已经为美国指明了前进的方向。”

Ride Report目前与70个城市合作，包括波特兰、俄勒冈州、奥斯汀、德克萨斯州和新西兰。在运营商方面，Ride Report与包括Bird、Lime和Lynx在内的19家公司合作。

“我们正在进入一个合作的时代，这将打开整个微移动领域，”异常风险投资合伙人安迪·约翰斯在一份声明中说。“这触及了Ride报告的核心内容，以及他们对移动未来的信念——运营商和城市将越来越多地合作，帮助行业发展。”

Ride Report并不是唯一一家试图帮助城市理解微观交通的公司。拥有385万美元资金的popul us，帮助城市从共享交通运营商那里获取车辆和出行数据。但是，考虑到在数百个城市运营的微型汽车公司的绝对数量，似乎还有空间。
