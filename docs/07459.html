随着SaaS股市跌至一年来的最低点，股市再次下跌，进一步陷入熊市

在美国上市的股票今天早上急剧下跌，在昨天的急剧抛售后没有被总统讲话所阻止。T2股市的下跌呼应了今天早些时候亚洲上市股票的下跌。

美国所有主要股指目前都处于熊市区域，从近期高点下跌了必要的20%。今天的大屠杀简直惨不忍睹。当我们写信给你的时候，这里是股票的位置:

  • 道琼斯工业平均指数(DJIA):-1688.5点，或-7.2%
  • 标准普尔500: -191.2，或-7.0%
  • 纳斯达克综合指数:-557.1，或-7.0%

对科技宠儿、、SaaS和云公司来说，坏消息不断。根据Bessemer-Nasdaq cloud index的数据，这些上市公司今天下跌了6.7%。SaaS和云公司现在的交易价格处于一年来的低点，可能会在2018年底至2019年初的几天交易中接近低点。( SaaS公司很早就发现了熊市趋势。)

优步和Lyft的股票今天大幅抛售，因为这两家美国打车公司放弃了最近几个月实现的所有利润承诺驱动的收益。截至本文撰写之时，优步下跌超过10%，而Lyft下跌超过13%。

抛售被证明是如此糟糕，以至于市场断路器再次被触发:

有时被认为是针对其他资产类别的对冲工具的加密债券也大幅下跌，打击了这种想法。例如，比特币不到6000美元。

从市场角度来看，这已经是最坏的情况了。对许多上市公司重新定价的股票的持续大规模抛售也无助于私营企业。随着上市公司的价值下降，初创公司将更难以有利的价格融资，并以有吸引力的估值找到出路。

IPO世界似乎奄奄一息。然而，venture的情况更像是一幅不断发展的画面。鉴于风险融资报告的滞后，很难判断对私人投资者和初创公司来说，情况有多糟糕。当我们有更多的时候。然而，故事仍然是公开市场和恐惧的回归。
