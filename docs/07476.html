白宫和众议院民主党人同意为带薪病假提供资金，为测试提供资金

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/13/white-house-and-house-democrats-agree-to-funding-package-for-paid-sick-leave-funding-for-tests/

虽然股市在总统发表讲话后大幅上涨，但推迟该法案可能会导致进一步的经济不确定性，并使金融市场持续几个月的疯狂，这些市场受到一连串坏消息和旨在提振经济的权宜措施的困扰，但未能解决影响全球市场的实际压力(推动人们投资股市并不能解决经济因应对全国性疫情而陷入停滞所带来的金融冲击)。

随着各城市和州政府鼓励(或在某些情况下强制要求)社会距离和自我隔离，以限制由新型冠状病毒新型冠状病毒引起的新冠肺炎病毒的传播，美国大量的服务行业将受到影响。

这适用于初创企业，如零售连锁店 b8ta 和邻里商品，美容品牌 Glossier ，洛杉矶新千年拱廊连锁店Two Bit Circus；以及最著名的直接面向消费者的创业公司 Warby Parker 。

对于Postmates、Instacart、Lyft、优步、Airbnb等gig和共享经济公司来说，这也是一个因素——这些公司因其对过剩资源(无论是汽车、业余时间还是空间)的新颖方法而成为风险投资的宠儿。

这些公司已经面临国会山立法者的批评，因为他们对可能受冠状病毒爆发影响的工人的补偿做法。

暂停美国在商场和餐馆的购物和餐饮，在酒吧、剧院、音乐会和剧院的娱乐，关闭公共场所，以及在家工作的政策减少了商业区当地企业或零售连锁店的客流量，这将打击低收入工人和酒店员工，他们没有带薪休假，或面临随着业务放缓而失业的风险。

大多数专家认为，这些社交距离措施也是城市减缓病毒传播的最佳机会之一。根据《纽约时报》对该法案通过的报道，带薪休假已被证明可以减少疾病的传播。

“今天，在与政府达成协议后，我们将通过家庭第一冠状病毒应对法案，”议长佩洛西在推特上写道。“这项立法建立在众议院民主党人上周采取的行动基础上，将#家庭优先 与我们强大的两党83亿美元紧急资金一揽子计划相结合。”
