GitHub抓住了JavaScript打包供应商NPM TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/16/github-nabs-javascript-packaging-vendor-npm/

微软旗下的开发者仓库GitHub今天早上做了一笔小交易，它以未披露的金额收购了JavaScript打包供应商 npm 。

正如GitHub首席执行官Nat Friedman在宣布交易的博客文章中所写的那样，npm在JavaScript社区中是一件大事。该公司是节点包管理器、npm注册中心和npm CLI背后的商业实体。

“npm是JavaScript世界的重要组成部分。弗里德曼写道:“npm团队在过去10年的工作，以及数十万开源开发者和维护者的贡献，已经使npm拥有超过130万个包，每月下载量达到750亿次。

仿佛预见到开发者对所有权变化的焦虑，弗里德曼承诺用户不会注意到差别。“对于每天使用公共npm注册表的数百万开发者来说，npm将永远可用，永远免费，”弗里德曼写道。

他还承诺更新工具背后的基础设施，改善体验，并与国家预防机制社区保持沟通。更重要的是，他说公司将把npm技术整合到GitHub平台中。

“展望未来，我们将整合GitHub和npm，以提高开源软件供应链的安全性，并使您能够跟踪从GitHub pull请求到修复它的npm包版本的变化，”他写道。

npm创始人兼首席执行官艾萨克·施鲁特(Isaac Schlueter)在公司博客文章中写道，他认为此次收购是向好的方向转变。“在npm体验中有一些很棒的改进机会，以无数大大小小的方式有意义地改善JS开发人员的生活。他写道:“在我们庞大的相互依存的JavaScript生态系统中，我们将为每个人提供更可靠、更便捷、更互联的服务。

当然，不仅仅是免费版。付费用户也是一个核心群体，弗里德曼表示GitHub将继续支持他们。

他还表示，今年晚些时候，当注册中心更全面地集成到GitHub平台时，付费用户将能够将其私人npm包转换为GitHub包。

根据PitchBook数据，成立于2014年的npm已经以4800万美元的后估值筹集了近1900万美元。“六年来，在创业的磨砺中，我们有着太大而不敢奢望的梦想。下一章是实现这些梦想的机会，”施鲁特写道。
