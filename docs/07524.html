Y Combinator W20演示日的所有公司，第一部分:B2B公司

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/17/all-the-companies-from-y-combinators-w20-demo-day-part-i-b2b-companies/

Y Combinator的演示日这次有点不同。

随着对新冠肺炎蔓延的担忧加剧，Y Combinator 改变了我们已经习惯的在三藩市举行的为期两天的聚会的形式,而是选择通过YC的演示日网站同时向受邀的投资者和媒体公开整个课程。

出人意料的是，YC还将演示日提前了一周，理由是投资者加快了节奏。唉，这意味着每家公司都要改变计划，在演示日网站上播放一段录音；相反，每家公司都通过幻灯片、几段概述他们正在做的事情和他们看到的吸引力以及团队简历来进行推介。目前还不清楚这种新形式——结合快速发展的投资环境——将如何影响这一阶层。

正如我们在每堂课上所做的那样，我们根据从每个公司的推介、网站以及(在某些情况下)我们之前对它们的报道中收集的信息，收集了我们对每个公司的笔记。

为了使内容更容易阅读，我们将内容按类别分开，而不是像一堵巨大的文字墙。这些是B2B公司——主要专注于向其他企业销售的公司。 你可以在这里找到其他类别(如硬件、AI、消费类)。

在 Extra Crunch 上查看TechCrunch员工最喜欢的公司。

B2B公司:

Alude : 巴西的房地产技术是一个炙手可热的市场，Loft和unicorn QuintoAndar等初创公司筹集了大量增长阶段的资金。Alude希望通过其简化的房屋租赁/购买流程来控制分销渠道。它的系统自动化了背景调查、文件收集、保险购买和在线签约的过程。这种现代化的技术对经纪人是免费的，该公司计划通过向客户出售抵押贷款和保险来赚钱。

Vori : Vori希望成为美国杂货供应链的操作系统。即使在2020年，超市仍然有一个旧的学校纸和笔的批发分销商订购流程。Vori充当超市和分销商的B2B市场，帮助商店以更有效的方式从供应商那里生产库存。它说它有24家商店在使用它的技术，并且还在北加州支持超过150家经销商。Vori还充当超市的产品发现引擎，帮助他们保持与全食超市和亚马逊的竞争力。

Linkana : Linkana是拉丁美洲合规驱动的采购软件。拉丁美洲有40，000家公司每年花费60，000美元购买解决方案，创造了25亿美元的市场机会。四人联合创始团队的成员在一起工作了近十年。

Weav : 在工作中面试新的候选人可能会耗费时间，而且也没有一个很好的方法来组织和综合团队的反馈。Weav为招聘团队录制和转录面试，这样公司可以更快地做出决定，减少面试次数。Weav表示，在一个前苹果/谷歌/微软团队的带领下，它正在构建最先进的自然语言处理工具，这些工具利用了信息检索、主题建模和实体识别。

electroniek RPA:每个人都同意机器人是自动化的未来，但实际部署这项技术往往需要大量的专业知识。ElectroNeek正在构建一个桌面和基于云的界面，旨在帮助没有机器人背景的IT员工和商业专业人士简化自动化流程。

reactive:面向动画师、视频编辑和工程师等创意专业人士，reactive旨在用基于云的解决方案取代桌面硬件。该公司声称其产品比传统解决方案快100倍，将八小时的渲染工作缩短到45秒。该公司已经完成了工作室220万美元的采购订单。

Handl : 一个API，用于将纸质文档(包括手写文档)转换为结构化数据，以便随时放入数据库或CRM。虽然该公司表示，大约85%的处理由他们的人工智能处理，但当人工智能的信心较低时，它由人类来验证数据。推出9个月后，该公司的ARR为90万美元。

Eze : Eze正在打造一个二手智能手机市场，其功能就像一个商品交易所。该系统将随着各种不同移动设备的价格波动而实时更新。这些设备将批量出售给批发商，批发商再转手卖给零售商。

Oda : 将来自列表服务和政府记录的房地产数据聚合到一个统一的API中。该公司表示，目前正在与4家试点公司合作。

好的 : 与JIRA、GitHub、GCal和Pagerduty等工具挂钩，为工程经理提供一个仪表板，以便更好地了解他们的团队是如何工作的，并提高效率(例如，通过减少可能破坏生产力的会议)。每位经理收费350美元。

Tajir : 一个帮助巴基斯坦小商店获得库存的市场。该公司表示，目前几乎所有向巴基斯坦“夫妻店”的分销都是在线下完成的；Tajir将这一过程整合到一个移动应用程序中，并提供免费的次日送达服务。

GuruHotel : 一个酒店的网站和物业管理/预订系统。推出8个月后，他们与26家酒店合作，每月收入2万美元。基本计划提供了一个基本的托管/预订引擎，以换取5%的预订佣金，而高级计划则推出了其他功能，如物业管理工具，除佣金外，每月收费350美元至499美元。

Riot Security :一款反网络钓鱼工具，它会根据最近发现的网络钓鱼技术，用假冒的网络钓鱼电子邮件自动测试您的员工。对于员工不足50人的公司，起价为每月200美元，之后价格将转向定制。该工具目前正在六家公司进行试点测试，MRR为1000美元。点击此处查看我们之前对防暴安全的报道。

LabGrid :一个项目跟踪和协作工具，旨在帮助生物技术公司和实验室比通过电子邮件更有效地沟通。

Upflow:Upflow的目标是成为“B2B的Venmo”，专注于支付未付发票。它会自动向未付款的帐户发送定制的电子邮件和挂号信，在帐户状态发生变化时更新指定的团队成员，并处理付款。对于每月少于30张发票的公司，他们每月收取50美元，对于年营业额少于300万美元的公司，他们每月收取225美元。在这里找到我们以前对上流的报道。

Explo : Explo意在让非技术员工分析大量数据，而不必知道如何编写/运行SQL查询，而是为他们提供一个生成报告的点击/点击界面。该团队表示，其候选名单上有超过400家公司。

work bench:work bench正在开发一个专门为硬件公司打造的平台，用于寻找供应商并存储有关他们使用的组件的信息/规格。

Jet Admin :一个无需代码就能构建内部工具的拖放工具，希望能释放开发团队的资源。连接到Stripe/Google Analytics/sales force等数据库和服务，并允许团队通过预构建的小部件来拼凑工具。独立开发者免费，10人以下的团队每位用户19美元。

实战卡 : 通过模拟训练你的销售团队对不满意的客户说什么(包括合成的不满意的客户声音)。团队合作编写不同情况下的应对“剧本”,分享他们发现最有效的答案/措辞。大约推出一个月后，该公司表示，它已经预定了超过3.5万美元的年度经常性收入。

SnackThis : 是一款基于浏览器的协作式运动设计工具。想象一下，一个远程团队在视频中微调移动排版，或者一个应用程序在屏幕之间移动的方式。其中一位联合创始人之前以8000万美元的价格将他以动作设计为主的公司卖给了GoPro。

杰欧汽车 : 为在印度拥有自动车队的公司提供车队管理和跟踪，允许他们查看当前车辆位置、回放过去的行程、计算燃料成本等。兼容50多种不同的GPS设备。该公司表示，它已经搭载了2000名车队所有者，将3万辆汽车带到了该平台上。

Savvy:Savvy是为那些无法提供团体保险计划的公司而设立的，它让这些公司给员工一笔免税津贴，让他们将这笔钱投入到自己选择的个人健康计划中。该公司表示，在两个月前推出后，它正在与30多家公司合作，约占ARR的10万美元。

Flowdash帮助人在回路团队用最少的代码构建新工具，允许他们将它们集成到Slack或Gmail等服务中。他们的基本计划从每个用户每月25美元开始，如果您需要像分析或内部部署这样的东西，可以增加费用。

Dropee:Dropee帮助东南亚的独立零售商从大品牌那里批量购买商品，向这些品牌收取每家店8美元的费用，以了解哪些商品畅销，哪些商品滞销。该公司表示，每月收入超过4万美元。

NUMI : 帮助非洲的零售商和市场进口美国商品，处理货运和海关方面的挑战。目前正在与家乐福开展试点项目，预计每年销售额将超过50万美元。

Pilot : Pilot处理雇佣远程承包商的工资、福利和合规事宜。公司每月向每个承包商支付60美元的试点费用，通过提供福利、股票期权和费用报销来帮助承包商像对待全职员工一样对待他们。随着新冠肺炎quarantines让更多公司熟悉远程工作，通过让他们感觉自己是团队的一部分来确保他们的保留和生产力，可能会有很大的市场。

SEND : SEND是一家面向非洲的数字货运代理和报关行，管理空运、卡车和海运货物。SEND通过在线提供文档，让客户只需与一家公司而不是十几家运输供应商打交道，从而优化了路线，实现了更快、更可靠的交付。SEND的创始人是兄弟，他们认为有机会在这家估值32亿美元的初创公司进入非洲大陆之前，通过征服市场成为非洲的Flexport。

Brokrete:Brokrete是一款交付市场应用，旨在将承包商与价格最具竞争力的混凝土供应商联系起来。这家初创公司的创始人旨在抓住他们所说的1200亿美元的市场机会。他们已经取得了一些进展，首先与承包商演示了该应用程序，然后在12月推出了产品。该公司始于加拿大市场，并于今年春天扩展到休斯顿。

Paneau:Paneau的创始人旨在为企业创造一种新的方式，通过在优步和Lyft车辆内放置互动平板电脑来向乘车客户做广告。乘客可以使用平板电脑进行购物，甚至改变汽车路线。Paneau已经通过每次旅行收取0.96美元的费用创造了收入——每月约11，000美元。

Bego : Bego开发了一款专注于拉美市场的应用，该应用使用机器学习来预测货物交付的未来位置，并帮助卡车司机与客户进行匹配，以减少“空英里”的数量。目前，这家初创公司只有一条路线——从墨西哥城到新拉雷多——墨西哥42%的货物都在这条路线上运输。

99分钟 : 这家Latam初创公司专注于电子商务购物的最后一公里配送。这家初创公司的地理足迹遍布两个国家的19个城市，每天交付15，000次，并计划在今年晚些时候扩展到哥伦比亚和秘鲁。99Minutos现在开始用电动汽车送货，在墨西哥是亚马逊、自由市场和沃尔玛的最后一英里送货合作伙伴。

农场理论 : 农场理论从印度农民手中购买“丑陋”但新鲜且适合食用的蔬菜，然后直接出售并运送到印度的餐馆。蔬菜配送服务表示，它可以为餐馆节省高达30%的费用。

HYPHY : 随着广告和用户生成的社交媒体内容一起成熟，HYPHY的目标是创造一个市场，让消费者直接向品牌出售他们的照片和视频。市场是品牌为广告或营销活动更快获取媒体的一种方式。

Zaam : Zaam正在构建一个平台来简化B2B onboarding，降低复杂性，并通过尽可能自动化来推动客户完成数据和文档请求的繁琐过程。这家初创公司表示，在过去的两个月里，他们已经达到了12万美元。

HireSweet正在建立一个招聘平台，将招聘人员推向理想的候选人，这些候选人可能不会明确地寻找新工作。该平台分析行为，如谁在更新他们的LinkedIn，在Github上添加开源项目，或接近归属悬崖。这个团队上个月在MRR赚了15万美元。点击这里查看我们之前对HireSweet的报道。

Stryve : Stryve希望招聘过程转向视频，用视频聊天问卷取代电话筛选。该团队要求该平台减少无休止的来回调度，提高周转速度。

Paragon : Paragon是一个低代码API构建器，帮助加速构建API、基于API的交互和集成

Syndetic:“Shopify for data”，synde TIC是一个平台，可让组织将其静态数据集变得更加动态和有用

Cadence:Cadence是一个本来应该是邮件的会议平台。早期访问平台专注于消除与共享项目更新相关的会议。它通过与任务管理工具集成来实现这一点，并让员工轻松地在Slack上分享他们一直在做的事情，他们一直在与谁合作，以及正在进行的事情。

Zynq : Zynq正在打造一款企业日历工具，帮助公司全面、高效地安排会议。这项服务有助于在合适的时间将会议安排在大小合适的会议室，这样公司就不会觉得自己的办公室太快就容纳不下了。

卡斯特迪亚 : 卡斯特迪亚将你的数据库与谷歌表单挂钩，确保那里的信息总是最新的，用户不必一次又一次地手动导入CSV文件。

one tool:one tool正在建立一个一体化的平台，允许创业公司订阅和保存，支付单个订阅，同时使用多种供应商来满足他们的需求。该公司希望其平台可以提高新SaaS工具的可发现性，并简化必须管理如此多订阅服务的创始人的生活。

dash works:dash works的目标是建立一个搜索工具，汇集来自各种协作工具和数据库的所有信息。该平台依赖于各种工具的深度集成，包括JIRA、Slack和Google Drive。

laser focus:laser focus正在开发一款应用程序，它可以在你的客户关系管理基础上，让销售人员快速完成与潜在客户的电话、电子邮件和会议。该应用程序希望将目前分散在少数应用程序中的任务重新打包，并减少那些渴望收集潜在客户信息的销售人员的分心。

TrueNorth : 为了帮助解决分散的货运行业的低效问题，TrueNorth为独立的卡车司机提供了一个软件解决方案。可以把它想象成一个操作系统，但是对于卡车来说，它可以帮助完成从燃料和维护到路线优化和负载跟踪的所有事情。

Taiv : Taiv希望帮助您当地的社区体育酒吧更好地利用您在酒吧电视上看到的广告赚钱。每个位置每年收费4200美元，Taiv允许企业用特价或交易的商业广告取代直播商业广告。点击这里查看我们之前对Taiv的报道。

Humanly:Humanly希望为通常收到大量申请的公司自动筛选求职者。该公司表示，其技术有助于保持筛选的一致性，同时消除偏见。客户包括农民保险、羽毛和格林。

build plane:build plane为商业建筑管理设计了下一代工具包。该公司的软件跟踪文档请求，管理变更单，处理分包商的账单和付款，以及报价和定价请求。这个行业类别已经产生了像Procore和PlanGrid这样价值数十亿美元的业务。

SINAI:SINAI Technologies是一个下一代软件平台，它让组织规划他们的碳排放战略:它跟踪公司内的不同部门和流程，然后就如何减少碳排放提出建议，以满足内部和外部目标。点击这里查看我们之前对西奈半岛的报道。

对数实验室 : 对数实验室是一项面向芯片设计人员的项目管理服务，涵盖数据管道、脚本接口、门户和仪表板，用于解析、结构化和分析芯片设计工作中产生的数据

Snapboard:Snapboard提供软件工具，无需代码即可创建仪表盘、可视化和应用。点击这里查看我们之前对Snapboard的报道。

Slingshow:Slingshow是一款用于客户服务和投诉解决的免费视频录制和交付应用。客户所要做的就是拍摄一段带有解释的视频，并通过Slingshow发送出去，以解释问题并获得帮助。

滑轮 : 滑轮是下一代帽表管理工具。随着公司规模的扩大，私营公司可以用它来发行员工和投资者股票，并保持所有权记录。和Carta一样，它也为小型创业公司提供免费服务。与Carta不同，它首先(目前也是唯一)关注的是为创始人服务，而不是投资者。创始人是一位重复创业者，他将之前的公司卖给了微软。

Rosebud AI : 欢迎来到公司代言人塑造活动的反乌托邦未来。Rosebud.ai为任何场合创建数字头像和模型。公司可以根据人口统计、年龄和风格进行筛选。

Termii:Termii是一项多渠道营销和通信服务，为非洲企业提供短信和用户验证的API。

Able Jobs:Able Jobs对印度的候选人进行公司最需要的技能培训，这样公司就能更快地雇佣到更好的候选人。他们在二月份做了130次安置

Skypher:Skypher自动化了安全问卷开发和回复流程。

Terusama:Terusama将物流管理带到码头，为货物提货提供调度软件，并为搬运工提供数字签到。

Mistro : Mistro允许雇主向200多个国家的远程团队提供福利和津贴。他们可以提供医疗保险、共同工作空间、发展课程、食物、IT设备等，员工可以通过Mistro信用卡支付费用。由于冠状病毒，世界各地的团队正在接受远程工作。这种趋势可能会持续下去，为任何能够帮助公司吸引最佳在家工作者的人创造一个巨大的市场。
