Y Combinator W20演示日的所有公司，第二部分:消费公司

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/17/all-the-companies-from-y-combinators-w20-demo-day-part-ii-consumer-companies/

Y Combinator的演示日这次有点不同。

随着对新冠肺炎蔓延的担忧加剧，Y Combinator 改变了我们已经习惯的在三藩市举行的为期两天的聚会的形式,而是选择通过YC的演示日网站同时向受邀的投资者和媒体公开整个课程。

出人意料的是，YC还将演示日提前了一周，理由是投资者加快了节奏。唉，这意味着每家公司都要改变计划，在演示日网站上播放一段录音；相反，每家公司都通过幻灯片、几段概述他们正在做的事情和他们看到的吸引力以及团队简历来进行推介。目前还不清楚这种新形式——结合快速发展的投资环境——将如何影响这一阶层。

正如我们在每堂课上所做的那样，我们根据从每个公司的推介、网站以及(在某些情况下)我们之前对它们的报道中收集的信息，收集了我们对每个公司的笔记。

为了使内容更容易阅读，我们将内容按类别分开，而不是像一堵巨大的文字墙。这些公司主要致力于向消费者销售商品或服务。 你可以在这里找到其他类别(如硬件、AI、fintech)。

在 Extra Crunch 上查看TechCrunch员工最喜欢的公司。

消费品公司

:Apartio为巴西的商务旅客提供长期和短期租赁服务。该公司计划瞄准员工来自巴西的企业，直接在Airbnb和Booking.com上获得游客。

Valienta:Valienta是一款想简化拉美国家直销网络流程的软件。直销占据了价值270亿美元的商品——在拉丁美洲这是一个巨大的机会。这1300万直销员大多是向个人网络转售产品的女性，比如美国的雅芳女性。Valienta希望通过一款应用程序实现这一过程的现代化。

Viya : Viya是面向拉丁美洲市场的移动优先公寓租赁系统。该公司提供市中心房间和公寓租赁以及清洁、维护和洗衣服务，旨在简化租户的生活。

杰米菲 : 杰米菲的定位很简单:音乐家的抖音。随着世界各地直播活动的关闭，这家初创公司当然有一个潜在的市场，为音乐人提供一个现场主持短音乐视频的地方。该公司本月早些时候在印度尼西亚推出。

现代村 : 想成为忙碌家庭的“参谋长”，每月收费30美元，帮助处理诸如儿童保育、清洁工、食品杂货和膳食规划等事务。该公司估计，它每周可以为父母节省大约10个小时。目前处于私人测试阶段。

军团农场 : 电竞教练。军团农场按小时收费，让客户与顶级/职业玩家组队，这些玩家可以在Apex Legends、Destiny 2和堡垒之夜等游戏中指导他们。该公司表示，仅去年一年，它就提供了超过35万小时的指导(球员平均每小时支付约16美元)，ARR为720万美元。我们之前在这里写过关于军团农场的文章。

StayQrious : 面向印度8-15岁儿童的在线小组课程(从编程课开始)。该公司表示，自三个月前推出以来，学生保留率达到了90%。

该公司表示，预计3月份总收入将达到17万美元。

退税巨头 : 帮助来英国的游客退税，他们在离开时能够——但往往不能——获得退税。用户上传他们收据的照片，RefundGiant处理所有的文书工作，从任何退款中抽取25%的提成。

Sayana : 一款基于订阅的精神健康应用，它使用类似聊天机器人的界面来帮助用户跟踪他们的感受，同时提供正念技巧。目前每月收入约为15，000美元，月环比增长率为25%。

HelpNow : HelpNow希望通过动员优步司机，为他们提供培训(AHA认证)和基本的生命支持设备，来缩短在印度获得救护车所需的时间。他们在孟买的道路上有347辆车，为超过9100个求助电话提供服务。该公司是在一位创始人的父亲心脏病发作并被告知要等45分钟以上的救护车后成立的；当他们选择用自己的车送他时，他的父亲得救了。

Global Belly : 为有影响力的人开发和运送定制品牌产品，首先关注食品(如烘焙套件或食谱盒。)该公司表示，目前该平台上有17名有影响力的人，还有200名正在筹备中，目前每月收入约为2.5万美元。

Whatnot : 一个买卖经过专业鉴定的收藏品的市场，从FunkoPop人物开始。想想山羊的收藏价值。该公司表示，大约在推出三个月后，它在GMV每月看到3万美元。在这里找到我们以前的报道。

Pantheon:Pantheon是一款学习应用程序，模仿面向学术学生的基于知识的游戏节目式活动，如科学碗或英国大学挑战赛，在这里，中、高中学生不仅可以相互竞争，还可以与志同道合的书呆子一起聊天。它也是一种“招聘”平台，大学和公司可以在这里发现有前途的候选人。

惊鸿一瞥 : 正如我们被告知要隔离自己以减缓冠状病毒的传播，这里有一款应用程序可以帮助你以不那么孤独的方式打发时间。Glimpse让您可以设置并与朋友或朋友的朋友进行简短的视频聊天，时间不超过2分钟。比起在Instagram上无休止的滚动，有时快速的互动比一整天的会议更有价值。

多元宇宙 : 受龙与地下城&等开放式游戏和Roblox等网站上的DIY游戏构建的启发，多元宇宙是一个让你构建自己的角色扮演游戏的平台。

Nugget:“insta gram for audio”，Nugget允许用户录制简短的音频剪辑，应用过滤器来增强声音，并在社交feed上分享它们。应用于播客媒体的快照的自发性:绝对是我们时代的一个应用。

一起 : 脸书在隐私方面还有很多不足之处。Together的目标是通过一款付费社交媒体应用来填补这一空白，该公司称这款应用可以确保用户的隐私。

泽洛斯 : 泽洛斯让玩家免费获得多个视频游戏的奖励。它的32K周玩家完成挑战(如在一分钟内三杀)，赚取积分，可以收集和兑换虚拟商品、折扣或莱佛士。寻求更多参与的游戏开发商受到激励，与泽洛斯整合，后者从每月5美元的高级订阅中赚钱，以获得更快的积分，并与开发商分成。

The Mercer Club:The Mercer Club是一家豪华男士街头服饰和鞋子租赁服务公司。客户每月支付75美元，每周租赁最多两件商品，这样他们就不必为了在Instagram上穿一件1000美元的古驰帽衫。它的“为男人租跑道”模式已经增长到10万美元。

阿德拉 : 阿德拉给大学女生送去一箱箱的衣服，让她们试穿和购买，然后拿走剩下的。Adla通过高地理密度和校园社区间的病毒传播来提高利润。它对每件商品收取7美元的加价，并希望最终通过推荐需要启动其客户群的品牌来赚取佣金。

维吉尔保险 : 维吉尔保险帮助年满65岁的老年人购买医疗补助的健康保险。在线经纪人使比较计划变得容易，同时提供了不需要在呼叫中心等待的体验。每天有1万名美国人年满65岁，保险经纪人每年从中赚取60亿美元的佣金。创始人此前曾创建金融科技初创公司，如用于取消订阅的Trim和用于为购买运动型汽车融资的Octane Lending。

艺术在Res : 艺术在Res是一个美术市场，顾客可以在这里分期付款购买。这家初创公司与产量超过销量的艺术家签约，然后为他们提供折扣、分期付款和社交媒体粉丝货币化的工具。Art In Res的联合创始人是一名画家，他学会了编码来支付账单，现在他与125名艺术家合作，寻找新的电子商务渠道。

隐匿处 : 隐匿处称，美国餐饮业是一个8600亿美元的市场，然而千禧一代的所有订单中有59%是外卖或外卖。这家初创公司正在建立一个只提供外卖的餐厅品牌组合，首先是日本的katsu sando概念，一个有机碗品牌，很快是意大利的subs和盒饭概念。

Duffl : Duffl通过提供10分钟内交付常用必需品，瞄准大学生的实时且往往是最后一分钟的需求。该公司将经常购买的商品存放在校园附近，并雇佣学生通过电动滑板车运送这些商品。收入来自运输费和通过批量购买产品获得的利润。

 

Thunderpod : 有点像健身燃料的番茄。每个用户都有一个“Thunderpod”头像，它会随着你的每次活动而增长:应用程序会记录你的运动，你可以与朋友进行健身挑战，或者你可以浏览用户创建的健身挑战的大型目录，如舞蹈。越来越多的消费和社交应用来自印度蓬勃发展的科技市场。

Carupi : Carupi在巴西开发了一个P2P二手车市场，并计划在2021年第一季度进入美国。

Motion:Motion的团队正在开发一个扩展，帮助Chrome用户更有效地管理他们在网络上的时间。该工具超越了简单的黑名单和白名单，具有更灵活的结构，允许用户在他们指定为“非生产性”的网站上改变他们的习惯该工具目前是免费的，但该团队希望在成为消费者最喜爱的产品后，最终开发出更广泛的B2B战略。

Cron : Cron正在为谷歌日历打造一个超人。Cron的联合创始人Raphael Schaad之前帮助创建了最初的iA Writer应用程序。他的新创公司每月收费19美元，为G Cal power用户提供一些额外的功能，这些功能可以清理工作流程，并与其他生产力应用程序集成。

月亮 : 月亮自称是拉丁美洲的微笑直接俱乐部。Moons于2019年3月推出，提供正畸治疗和透明矫正器，并声称其价格是SmileDirectClub的一半。该公司月销售额为60万美元，在15个零售点雇佣了200多名员工。在这里找到我们以前对月球的报道。

酸辣酱 : 针对印度新兴的网络用户群体，酸辣酱希望成为“印度大众市场的亚马逊”使用WhatsApp，顾客可以从附近的夫妻店购买新鲜水果和蔬菜，并在第二天提货。

雅西尔 : 雅西尔是一款为非洲法语区设计的应用。它希望为29个国家约4.3亿人口提供一套金融服务。

edu rev:edu rev是一家教育科技公司，面向2亿准备考试的印度学生。备考平台的订阅费用为每年50美元，每月活跃用户超过45万。EduRev不是课后辅导，而是一款以数字化方式向学生提供预备课程的应用。

Riya Collective:印度的婚纱礼服价格昂贵，因此Riya Collective专门围绕它推出了一款租跑道式的产品。这家服装租赁初创公司由两位印第安美国人企业家创建，使用数据驱动的造型和尺码算法。Riya Collective在三个月内从月收入10，000美元增长到月收入50，000美元。

TagMango : 一个“客串印度”的TagMango让用户预订来自顶级影响者和名人的个性化推荐。简单来说，这家初创公司是在通过让宝莱坞变得更具互动性和更易接近来赚钱。TagMango每个视频平均收费20美元。

FitnessAI:FitnessAI拥有18000名付费用户，可以生成个性化的举重计划。创始人杰克·莫尔(Jake Mor)10年来一直在开发应用程序，4年来一直在开发健身应用程序。这家初创公司每年收费90美元。点击这里查看我们之前对FitnessAI的报道。

赠品 : 赠品成立于12个月前，创建了一个点对点的市场来赠送旧物品。它已经在5个国家的用户群中成功完成了超过170，000笔交易。与传统市场不同，赠品让用户通过竞价用虚拟货币购买商品。

深度冥想 : 深度冥想每年收费26美元，提供个性化的冥想练习。该应用目前拥有158，000名月活跃用户。

Pahamify:Pahamify由印度尼西亚一位Youtube科学影响力人士创立，是一款帮助印度尼西亚学生通过高考的应用。这家初创公司反对传统的准备方法，比如面对面辅导，每年向学生收取大约24美元的费用。据该公司称，每个月有65，000名用户在使用Pahamify。

Edlyft:Edlyft由两位大学毕业后在湾区重聚的高中同学和计算机科学专业的学生成立，为可能缺乏计算机科学背景，但正在将其作为专业或职业的大学生提供导师网络、导师和同伴支持。
