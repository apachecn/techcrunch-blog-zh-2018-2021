Y Combinator W20 演示日的所有公司，第三部分:硬件、机器人、人工智能和开发工具

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/17/all-the-companies-from-y-combinators-w20-demo-day-part-iii-hardware-robots-ai-and-developer-tools/

Y Combinator 的演示日这次有点不同。

随着对新冠肺炎蔓延的担忧加剧，Y Combinator 改变了我们已经习惯的在三藩市举行的为期两天的聚会的形式,而是选择通过 YC 的演示日网站向受邀的投资者和媒体公开整个课程。

出人意料的是，YC 还将演示日提前了一周，称投资者加快了节奏。唉，这意味着每家公司都要改变计划，在演示日网站上播放一段录音；相反，每家公司都通过幻灯片、几段概述他们正在做的事情和他们看到的吸引力以及团队简历来进行推介。目前还不清楚这种新形式——结合快速发展的投资环境——将如何影响这一阶层。

正如我们在每堂课上所做的那样，我们根据从每个公司的推介、网站以及我们之前对它们的报道中收集的信息，收集了我们对每个公司的笔记。

为了使内容更容易阅读，我们将内容按类别分开，而不是像一堵巨大的文字墙。这些公司致力于硬件、机器人、人工智能、机器学习或开发者工具。你可以在这里找到其他类别(如生物技术、消费和金融科技)。

在 Extra Crunch 上查看 TechCrunch 员工最喜欢的公司。

人工智能和机器学习

Datasaur : 一种工具，旨在通过自动纠正、自动建议和键盘热键等方式，帮助人类更准确、更高效地标记机器数据数据集。个人贴标机免费，20 人以下的团队每月 100 美元，大型团队可定制价格。

1 构建 : 建筑公司自动化、数据驱动的工作成本估算。你上传你的计划，1build 说它可以“在几分钟内”准备好准确的出价该公司预计收入增长率超过 60 万美元，并表示已经完成了对亚马逊、星巴克和 7-Eleven 等大型公司的估计。

Zumo Labs : 使用游戏引擎为计算机视觉系统生成预先标记的训练数据。该公司表示，通过合成数据，而不是从现实世界的照片/视频中收集数据，它可以更快、更便宜地创建大规模数据集，而且没有隐私问题。

Teleo : 对现有施工设备进行改造，使操作人员可以远程控制。该公司表示，自三个月前成立以来，它已经制造了一种“全功能遥控装载机”，并计划向建筑公司收取每辆车的固定月费。该公司的联合创始人之前是 Lyft 的硬件工程主管和产品经理总监，两人都曾在谷歌的街景团队工作。

图灵实验室公司:对肥皂和除臭剂等消费品的不同配方进行自动化模拟测试。家用产品和化妆品可能是研发实验室数月的工作。图灵已经建立了一个人工智能引擎来帮助这个过程——很像药物发现中使用的人工智能引擎——将时间缩短到几天。它已经与世界上一些最大的 CPG 公司合作。你可以在这里找到我们之前关于图灵的报道。

seg med:seg med 正在为人工智能驱动的医学研究建立数据集。Segmed 与这些组织(目前超过 50 家)合作，并标准化、标记和匿名化数据，而不是要求每个研究人员单独与医院和成像机构合作。

Ardis AI:Ardis AI 想要建立人工通用智能的基础——像人类一样阅读和理解文本的技术。通过结合神经网络、符号推理和新的自然语言处理技术，Ardis AI 可以为不想雇佣团队来做数据提取和标记的公司服务。

Agnoris : Agnoris 分析一家餐厅的销售点数据，以建议对定价、送货菜单和人员配备进行更改。Agnoris 声称，每个餐厅位置每年 3600 美元，就能提高 20%的利润。该公司是在创始人开了一家爆满但亏损的餐馆后成立的，因此它建立了机器学习工具来提高利润率，现在它向所有餐馆出售这种软件。

frog labs:frog labs 为企业提供天气预报 AI，用于预测太阳能和风能产量、交付延迟、人员短缺、销售需求和食品可用性。通过接收数十亿字节的天气数据，它可以确保公司的物流不被中断，从而节省公司的资金。该公司由一位长期在谷歌工作的人创建，他启动了“Loon 项目”,即通过互联网发送气象气球。现在，该公司正在签约电子商务、零售、拼车、餐馆和活动业务。

pillar plus:pillar plus 是一个自动化建筑项目蓝图设计阶段的平台。它从建筑师或承包商那里获得设计，绘制机械、消防、电气和管道细节，并估计材料清单和项目成本，否则这些步骤需要几个月的工作。

Glisten : Glisten 利用计算机视觉和机器学习技术为电子商务公司开发更好、更一致的数据集。它的第一个产品是一个基于人工智能的工具，用来填充和丰富稀疏的产品数据。在这里找到我们以前对闪亮的报道。

nextmv : Nextmv 让客户能够自动创建自己的物流算法——允许企业优化车队和内部管理路线。

视觉一 : 运动检测安全摄像头会带来很多误报:的确有运动，但不一定是有害的。Visual One 建立了一个人工智能平台，与家庭安全摄像头集成，以“读取”它们检测到的特定动作。所有者可以创建定制的提醒，这样他们就可以只收到他们关心的通知。该公司的软件可以检查破坏家具的宠物、偷包裹的小偷、蹒跚学步的孩子不顾死亡的滑稽动作等等。点击这里查看我们之前对 Visual One 的报道。

PostEra : “药物化学即服务”是这里的想法:PostEra 的平台可以比典型的研发实验室更快、更低的成本设计和合成分子，加快药物发现过程中测试新组合所需的研究时间。

硬件和机器人

电脑牙科 : 机器人技术已经使外科手术发生了革命性的变化，这要感谢像达芬奇这样的公司——直观公司。Cyberdontics 的目标是在口腔手术中做同样的事情，从牙冠开始——这是一种更昂贵和更耗时的手术。该公司表示，其机器人能够在 15 分钟内完成通常需要两个小时的手术，收费仅为 140 美元。

Avion:Avion 专注于非洲难以到达地区的居民，正在建立一个基于无人机的送货系统。这些计划包括连接到一个中央枢纽的中程和远程医疗无人机。这些无人机是混合动力和自主的，具有垂直起飞能力，能够携带 5 公斤的有效载荷最远达 150 公里。

躯体 : 工业浴室清洁是一个主要的“沉闷”/“肮脏”的候选，将被自动化取代。体细胞制造大型机器人，通过虚拟现实训练这些机器人打扫厕所。该系统可以喷洒和擦拭物体表面，并能开门和在电梯里上下移动。在这里找到我们以前对体细胞的报道。

RoboTire : 任何坐过服务店等候室的人都知道这个过程有多耗时。RoboTire 承诺将一套四个轮胎的等待时间从 60 分钟减少到 10 分钟。该公司已经开始在美国各地试点这项技术点击这里查看我们之前对 RoboTire 的报道。

Morphle:Morphle 的系统旨在取代过时的模拟显微镜，使用机器人自动化来改善成像。这家初创公司处理的图像分辨率比昂贵得多的系统高，故障率也低得多。Morphle 已经开始向印度的实验室出售其系统。

代达罗斯 : 由 OpenAI 的一名早期工程师创建，代达罗斯正在开发自主软件，允许工业机器人在没有人类编程的情况下运行，从数控机器开始。该公司预计它可以将金属加工市场的生产率提高 5 倍。

Exosonic，Inc. : Exosonic 制造超音速商用飞机，不必产生很大的音爆，因此可以在陆地上空飞行。它的目标是一架能在三小时内从旧金山飞到纽约的飞机。这位首席执行官在洛克希德·马丁公司工作时，曾在美国宇航局的低吊杆 X-59 飞机上工作。Exosonic 现在有一家主要航空公司和两家国防部集团的意向书，加上一份 30 万美元的美国空军合同。

Nimbus:Nimbus 总部位于密歇根州安阿伯，由一位连续创业者创建，正在开发下一代城市交通工具平台。创始人李航·农之前推出了燃油喷射系统开发商 PicoSpray，现在他正在寻找这个问题的答案，“一辆汽车能比现在的汽车节省几倍的空间和能源，同时乘坐起来更舒适吗？”

UrbanKisaan:UrbanKisaan 是一家位于印度的垂直农业公司，为家庭提供新鲜农产品订阅。它的堆叠式水培桌农场可以位于城市附近，只占用传统农业 1%的土地，而且不需要杀虫剂。在一个寻求健康食品的中产阶级日益壮大的市场，送货上门可以让 UrbanKisaan 控制质量和利润。

Talyn Air : 两名前 SpaceX 工程师开发了一种用于乘客和货物的远程电动垂直起降(电动垂直起降)飞机。这家初创公司创造了一种电动固定翼飞机，在起飞和着陆期间用一架定制的有翼无人机抓住半空中，其创始人表示，这种方法使这种飞机的航程达到了竞争对手的三倍，达到 350 英里。

开发者工具

BuildBuddy : 两位前谷歌员工希望通过在谷歌的 Bazel 软件上构建开源 UI/功能集，为所有人提供一个“谷歌风格的开发环境”。该公司表示，他们的解决方案将构建时间加快了 10 倍。它对独立开发者是免费的，根据团队的规模和所需的功能，价格从每个用户 4 美元到 49 美元不等。

Dataline : 意在让网站从使用广告拦截工具的用户那里收集分析数据。该公司声称，大多数广告拦截器用户最关心的是显示广告或跨网站跟踪，并表示第一方分析会受到“附带损害”通过充当运行在子域上的“智能代理”, Dataline 避开了大多数广告拦截系统(目前看来是这样)。)

Cortex : 许多现代在线软件应用程序都由无数独立的、专注于特定目的的工具——或“微服务”提供支持 Cortex 监控你的应用程序的微服务，自动标记正确的人(挂钩到 Datadog/Slack/page duty/等。)一破。

【API tracker】:即使你的网站看起来加载正常，你用来使它工作的 API 可能会有问题，以不那么明显的方式破坏东西。Apitracker…跟踪您的 API。它监视您使用的 API，当其中一个开始失败时向您发出警报，并提供对其整体性能的洞察。

fresh paint:fresh paint 的“自动跟踪”系统收集所有浏览量/点击量/等等。在你的网站上，允许你把它推到像谷歌分析/脸书像素等工具。追溯性地，不需要你的开发团队为每个事件制作手动跟踪器。用户少于 3，000 人的网站可以免费使用基本计划，月用户数达到 50，000 人的网站则需要 300 美元，超过这一限额后，定价将转向定制套餐。

Datree:Datree 允许公司为其代码库设置规则和安全策略，并确保在合并任何代码之前遵循这些规则。向每个开发者收取 28 美元(注意，对独立/开源项目是免费的)，他们迄今为止已经获得了大约 23 万美元的收入。点击此处查看我们之前对 Datree 的报道。

fly.io : 将您的应用部署在距离您的用户更近的服务器上，减少延迟并改善用户体验。如果你的应用在某个城市越来越受欢迎，Fly 会检测到这一点，并相应地调整资源。

Sweeps:Sweeps 声称，通过更加智能地加载网站正在使用的所有第三方工具，他们可以用一行代码让你的网站速度提高 40%。该团队表示，他们的技术不仅提高了速度，还提高了搜索引擎优化。

Orbiter:Orbiter 是一个自动实时监控和警报系统，与 Slack 集成在一起，以确保更好的客户服务和收入管理。

发布 : 产品发布可能会比较棘手。Release 提供了一个登台管理工具包——它在每次有拉取请求时构建一个登台环境，允许更快/更协作的开发周期。

signa dot:signa dot 是一款监控和管理微服务的软件，现代初创公司依靠它来驱动自己的应用和服务，希望在问题对最终用户变得明显之前将其标记出来。

Raycast : Raycast 是开发人员和他们使用的许多工具的通用命令栏。用户可以集成包括吉拉、GitHub 或 Slack 在内的应用程序，并采取超人般的方式来完成表格和任务。该团队将该工具作为帮助工程师快速完成非工程工作的一种方式。

Cotter 正在构建一个基于电话号码的登录平台，该平台在一个工作流程中对用户的设备进行身份验证，该公司的创始人表示，该平台具有基于短信的动态口令的便利性，而没有安全问题。这家初创公司的目标是发展中国家的客户，在这些国家，电子邮件的使用率较低，登录也不太方便。

同上 : 同上的创始人希望创造出文字的图形，帮助团队更周到地设计出他们用来描述产品和工作流程的文案。斯坦福大学室友 Jolena Ma 和 Jessica Ouyang 创建的协作工具目前拥有 80 多家不同公司的用户。

Scout:GitHub 工作流内机器学习实验的持续集成和部署工具包。

ToDesktop:ToDesktop 设计了一种服务，可以自动满足您所有的桌面应用程序发布需求。它可以与 Windows、Mac 和 Linux 一起工作，并为开发人员提供本机安装程序、自动更新、代码签名和崩溃报告，而不需要任何基础设施或配置。

deep source:deep source 是一个代码审查工具，允许开发者检查 Python 和 Go 中的 bug 风险、反模式、性能问题和安全缺陷。

flow bot:flow bot 是一种自然语言，用于 Python 中编码的自动完成搜索工具。它让 Python 开发人员在不记得他们想的确切函数时用简单的英语键入，Flowbot 通过挖掘文档并考虑上下文来找到它认为你正在寻找的代码。

post hog:post hog 是一种软件服务，它让开发者了解他们的用户实际上是如何使用他们的产品的。这是一个面向开源程序员的产品分析工具包。
