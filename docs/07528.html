Y Combinator W20演示日的所有公司，第四部分:医疗保健、生物技术、金融科技和非营利组织

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/17/all-the-companies-from-y-combinators-w20-demo-day-part-iv-healthcare-biotech-fintech-and-nonprofits/

Y Combinator的演示日这次有点不同。

随着人们对新冠肺炎蔓延的担忧日益加剧，Y Combinator 改变了我们已经习惯的在三藩市举行的为期两天的聚会的形式，而是选择通过YC的演示日网站向受邀的投资者和媒体公开整个课程。

出人意料的是，YC还将演示日提前了一周，理由是投资者加快了节奏。唉，这意味着每家公司都要改变计划，在演示日网站上播放一段录音；相反，每家公司都通过幻灯片、几段概述他们正在做的事情和他们看到的吸引力以及团队简历来进行推介。目前还不清楚这种新形式——结合快速发展的投资环境——将如何影响这一阶层。

正如我们在每堂课上所做的那样，我们根据从每个公司的推介、网站以及我们之前对它们的报道中收集的信息，收集了我们对每个公司的笔记。

为了使内容更容易阅读，我们将内容按类别分开，而不是像一堵巨大的文字墙。这些是医疗保健、生物技术、金融科技和非营利公司。 你可以在这里找到其他类别(如B2B、消费和机器人)。

医疗保健和生物技术

简单试纸 : 旨在通过制作可由任何智能手机摄像头读取的试纸，而不是需要专用的血糖仪，使血糖检测更便宜、更容易。该公司表示，预计将在6月份提交给FDA批准。

nplex biosciences:一种更快、更便宜的方式来完成新药开发所需的蛋白质组。该公司表示，已经有超过400万美元的意向书在准备中，其中包括一家大型制药公司的意向书。

:一款旨在帮助非洲用户与医生交流、预约和跟踪实验室结果的应用。该公司表示，它已经实现了盈利，留存率为98%。

drop print Genomics:“单细胞基因组学”软件旨在减少分析单个细胞活动的时间/财务成本，从而实现更好的药物发现。他们在两个月内签了100多万美元的意向书。

纽曼的 : 印度尼西亚一家男性数字化健康诊所。他们正在关注那些通常被视为尴尬的问题(脱发、勃起功能障碍)或通常被放弃的问题(戒烟)，通过远程咨询的方式让医生看病更容易、更便宜、更私密。点击这里查看我们之前对纽曼的报道。

Menten AI:Menten AI表示，它正在使用“量子计算和机器学习”结合合成生物学来设计新的蛋白质药物。

Loop Health:Loop Health表示，印度的大部分健康保险只涵盖“住院，不包括看医生。”他们希望通过提供对指定环路健康诊所的无限制访问以及基于应用程序的远程医疗来改善这种情况。

Synapsica health care:一个“AI报告助手”目前专注于脊柱磁共振成像，该公司表示，通过自动标注测量值和描述椎间盘退化，它节省了放射科医生80%的报告时间。该公司表示，它目前正在进行一个10万美元的试点项目，有250家脊椎按摩诊所提供放射学服务。

Volumetric:Volumetric制造3D生物打印机，制造血管化的人体组织。由两位博士创建的Volumetric向制药公司和科学家出售其光活性组织。它正在利用这些收益来建造生物打印机和生物探针，以生成功能性组织甚至器官。点击此处查看我们之前对体积测量的报道。

奥菲利娅 : 奥菲利娅为美国300万阿片类药物成瘾者用远程医疗代替康复治疗。它让患者可以通过电话会议看医生，获得丁丙诺啡和纳洛酮等处方药，并获得无耻辱感的治疗。这位创始人在一位长期女友死于阿片成瘾后创办了这家公司，奥菲利娅现在已经治疗了40名患者。

Lilia:Lilia是一家冷冻卵子的礼宾服务公司，声称“未来女性毕业后将冷冻卵子”。这家初创公司收取500美元的门房费，当有人被安排到诊所时，还能获得500美元。Lilia表示，其潜在市场总额为330亿美元。

赤道疗法 : 赤道疗法正在研发一种药物，帮助使用者在不运动的情况下燃烧卡路里。由博士阿朵和一名在开发抗衰老药物的公司工作的数据科学家创建的赤道治疗公司的目标是应对肥胖和二型糖尿病的人。

Altay Therapeutics:Altay Therapeutics位于旧金山的拜耳合作者内部，开发出了阻断致病DNA结合蛋白(或转录调节因子)的小分子疗法。该公司最初的疗法主要针对关节炎、纤维化、溃疡性结肠炎和肝癌。

tam BUA Health:tam BUA Health使用“声学”听诊器和专有软件，在没有x射线辐射的情况下提供高级肺部成像。

鲍鱼生物 : 鲍鱼生物是由一系列生命科学企业家创立的，它使用表达数十亿种抗体变体的酵母细胞库来培养能够激活或抑制药物靶标的特定抗体。利用基因测序、机器学习和合成生物学，该公司制造了其抗体的重组蛋白版本，并证实了它们在人类细胞分析中的功效。该公司最初的目标是治疗疼痛、炎症疾病、罕见癌症和罕见肾病的药物。

菲利克斯生物技术 : 由著名的耶鲁大学研究员保罗·特纳创立的菲利克斯生物技术公司正在开发针对细菌和真菌的抗生素耐药菌株的治疗方法。据该公司称，仅在美国，这些病原体每年就导致超过280万例感染和35，000例死亡。平均每15分钟就有一个美国人死于耐药性感染。研究人员警告说，到2050年，死于抗生素耐药性的人数将超过死于癌症的人数。

Genecis生物产业 : Genecis生物产业正在把食物垃圾变成可堆肥的塑料。点击此处查看我们之前对Genecis的报道。

Candid Health:Candid Health为医疗保健行业开发了自动计费软件，该软件跟踪保险公司的情况，并自动对被拒绝的索赔提出上诉。它从每笔付款中抽取5%的佣金。

赭石生物 : 赭石说，尽管存在短缺，但大多数捐赠的肝脏都被丢弃了，因为它们含有太多脂肪，无法成功移植。他们的目标是通过找到移植前的治疗方法来“在体外恢复肝脏的活力”。

金融科技

Facio : 巴西出现了银行问题。由五家银行组成的寡头集团管理着巴西市场，他们速度慢，客户服务差，年利率高，而且不为中小企业服务。Facio希望让员工免于沦为掠夺性债务的受害者，转而通过向员工发放低价工资贷款来获得财务自由。它与雇主整合，直接从他们的工资中扣除贷款。

delt.ai : Delt.ai是一家数字银行，为墨西哥服务不佳的中小企业和自由职业者处理支付、发票和公司卡。这家初创公司的目标是拉丁美洲500多亿美元的商业存款市场。想想Delt.ai是一个Brex或者是一个Mercury，但是专注于拉丁美洲。

内克苏 : 像拉丁美洲的许多其他个人融资业务一样，汽车融资是一个昂贵、低技术含量、艰巨的过程。内克苏是拉丁美洲汽车经销商的融资平台，它使用动态信用评分，让购车者在几秒钟内获得批准。创始团队以沃顿商学院MBA候选人的身份会面。

fonde ora:fonde ora凭借其另类的新银行借记卡，加入了墨西哥饱和的金融科技领域。该公司在其应用程序中提供了一个完全移动的数字储蓄账户。Fondeadora表示，它拥有65000名用户，每月交易额达650万美元。另一家专注于墨西哥的借记卡Albo目前拥有20万每月活跃客户的市场份额，这些客户在其平台上消费和交易，并筹集了2600万美元的资金。但墨西哥的银行业问题已经大到足以让多家初创公司蓬勃发展。在墨西哥1.3亿人口中，45%的人银行服务不足，这意味着他们缺乏旨在通过贷款和储蓄功能帮助他们复合财富的深度金融产品。

詹菲 : 根据业务收入向亚洲的小企业提供贷款，通常约为1万至10万美元。我们之前在这里写过关于简菲的文章。

yBANQ : 印度大型B2B公司的收款和对账系统。该公司表示，自1月下旬推出以来，已找到18名客户，GMV达到约1.8万美元。

ZeFi : 一个在幕后将美元存款转换为“稳定币”加密货币的储蓄账户，ZeFi将这些资金借给借款人以获得利息。

Grain : Grain将你现有的借记卡与“负责任”的信用额度挂钩(目前上限为500美元，并基于你的收入/现金流量)，希望帮助那些信用最低/不良的人随着时间的推移建立他们的信用报告。该公司表示，自推出以来的三个月里，它已经注册了1000名客户，预计每个客户每年能赚80美元左右。

CrowdForce : 让非洲当地的商人充当银行分支，当银行太远时，充当交易的中介。该公司表示，上个月净收入7万美元，平均每个客户每年20美元。

Stark Bank: 为巴西科技公司处理B2B交易的银行API。推出一年多后，该公司表示，每月总销量达到1200万美元。

Bamboo : 一家为非洲高财富个人购买世界各地证券的在线经纪公司。该公司表示，自大约五个月前推出以来，已经有超过2100名投资者在该平台上交易了超过160万美元，目前每月收入超过1万美元。

Swipe:Swipe将自己标榜为“非洲的Brex”，给非洲的中小企业一张信用卡来帮助支付工资和费用。他们通过向企业提供免费的费用/账单工具来发展业务，然后相应地提供信贷。该公司表示，它目前正在与30家公司合作，部署了20万美元的信贷。

goDutch : 一种支付卡，用于在经常分摊账单的群体之间分摊费用，例如室友。聚焦印度。费用被记在一张卡上，并自动从每个成员的账户中扣除。

Paymobil : 使用稳定币加密货币通过Venmo风格的应用程序在全球范围内转移资金。创始人丹尼尔·诺德(Daniel Nordh)提到，他之前在比特币基地领导消费者设计。

Karat : Karat向有影响力的人提供银行、贷款和信用卡。通过使用其受欢迎程度的数据来管理风险，Karat的贷款实现了40%的年利率，平均还款时间为45天。由于其创始人在Instagram建立影响者工具和在高盛构建债务的经验，它已经拥有超过1000万粉丝。

Homestead:Homestead帮助业主将车库改造成出租物业，无需前期费用。宅地支付所有的建设，租户搜索和管理费用，然后与房主分享租金收入。加州一项新法律允许该州800万个车库作为居住空间出租，这创造了一个巨大的市场机会。Homestead的创始人在麻省理工学院建筑和城市规划研究生院相遇，这家初创公司已经实现了100万美元的销售额。

bene pass:bene pass为创业公司和小企业提供福利卡。使用Benepass借记卡，员工可以支付税收优惠福利和健康津贴，如灵活的支出账户、儿童保育、通勤、健身和教育，同时应用程序会跟踪他们的购买情况。Benepass对雇主免费，税率为6%,但可以节省数千英镑的所得税和工资税。随着初创公司不顾一切地与科技巨头争夺顶级人才，Benepass可以确保员工感受到支持。

加油站 : 美国加油站的老板们正竞相升级采用EMV技术的外部加油泵，这是配备电脑芯片的信用卡的全球标准。GAS POS的成立是为了提供一个现代化的销售点系统，帮助北美的180，000个加油站遵守EMV法案，并使交易更加安全。该公司有几个收入来源，对已处理的支付收取3%的费用，设备的SaaS免费，以及向客户提供第二天融资的优惠。

year end:year end正在为纸富们开发税务软件，帮助创业员工报税，同时优化他们的权益。这家初创公司向个人用户收取每年330美元的费用，并希望出售给可以将年终作为员工福利的企业。

GIGI Benefits:印度的GIGI Benefits希望成为该国零工经济工人的福利提供者。这项业务借鉴了去年最热门的Y Combinator初创公司Catch或风投支持的Trupo等公司的做法，为零工经济工人提供医疗保险和退休投资账户等服务。

easy plan:easy plan是印度的Qapital或Digit，允许用户为某些特定目标无缝省钱。

Haven : Haven是一个服务于住房抵押贷款的下一代平台，为贷款人提供更现代的客户界面、更好的支付模式等等。

work pay:work pay将自己描述为“对非洲的热情”——面向该地区中小型企业的下一代薪资和相关服务。

Spenny:Spenny是一款面向印度消费者的储蓄工具，客户可以通过将购买的商品进行四舍五入来开始存钱。

Kosh : Kosh是一个算法增强的印度储蓄和投资平台，允许那些信用良好的人有效担保信用有限的朋友帮助他们借款。
