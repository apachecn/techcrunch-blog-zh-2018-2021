中国云基础设施市场在 2019 年第 4 季度创造了 33 亿美元的收入

研究公司科纳仕公司报告称，中国云基础设施市场在 2019 年最后一个季度增长 66.9%，达到 33 亿美元，就在新冠肺炎病毒袭击该国之前。中国是全球第二大云基础设施市场，份额为 10.8%。

本季度中国市场的运营成本为 132 亿美元。同期，科纳仕公司估计美国市场价值 140 亿美元，占全球市场份额的 47%。

阿里巴巴在中国遥遥领先，市场份额超过 46%。像美国电子商务巨头亚马逊一样，阿里巴巴也有一个云部门，它在中国的主导地位与 AWS 在美国的主导地位非常相似。

腾讯位居第二，占 18%，大致相当于微软 Azure 在美国的份额，百度人工智能云位居第三，占 8.8%，大致相当于谷歌在美国的市场份额。

幻灯片:科纳仕公司

科纳仕公司分析师马修·鲍尔(Matthew Ball)表示，第四季度的数据早于中国爆发新冠肺炎病毒导致的医疗危机。“就第四季度的增长动力而言，我们已经看到对按需计算和存储的持续需求在 2019 年加速增长，因为私营和公共组织开始实施数字化转型项目，并开始构建平台和应用程序来开发新服务。”

鲍尔表示，游戏是一个大的云客户，医疗保健、金融、交通和工业也是如此。他还指出，面部识别技术的增长是智能城市领域的一部分。

至于明年，Ball 表示，尽管 2020 年第一季度受到病毒的影响，但该公司仍认为市场将大幅增长。“一旦业务恢复正常，除了继续开展数字项目，我们预计许多在危机期间首次使用云服务的企业将继续使用并成为付费客户，”他说。在危机期间，云公司已经向企业提供了许多免费的选择。

“全球当前事件的总体结果将是，企业将评估自己的业务连续性措施，并确保如果事件再次发生，它们能够继续运营，”他表示。
