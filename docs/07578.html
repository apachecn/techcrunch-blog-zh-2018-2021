谷歌推出下一个 Android 11 开发者预览版

谷歌今天公布了 Android 11 的第二个开发者预览版。和第一个预览版一样，这个版本也只面向开发者，不提供无线更新。相反，开发人员必须手动下载和刷新他们支持的设备，这些设备目前仅限于 Pixel 2，3，3a 或 4。

不出所料，这个第二版没有任何突破性的新特性，主要是继续了 Preview 1 的工作。在很大程度上，这些都是新的 API 和其他开发人员特性。在接下来的一些版本中，您将会听到更多面向用户的更新。

“这仍然是一个早期版本，但你可以开始看到该操作系统如何在这个版本中实现新的体验，从无缝的 5G 连接到围绕最新屏幕包装你的用户界面，到更智能的键盘和更快的信息体验，”谷歌工程副总裁戴夫·伯克在今天的公告中写道。

第二个预览版中的新功能包括一个 5G 状态 API，用于查看用户当前是否在 5G 网络上，以便开发者可以激活需要 5G 网络的体验。现在，通过一个新的 API，可以更好地支持可折叠设备，该 API 可以从铰链角度传感器读取数据，因此，顾名思义，应用程序可以适应铰链的角度。也有新的 API 用于筛选 robocalls 和支持可变刷新率等等。对于在应用程序中使用神经网络的开发人员来说，Android 11 将包括一个新的“硬 swish-op”功能，该功能将能够更快更准确地训练设备上的模型。

虽然这些更新大多针对开发人员，但用户可能会注意到的一个特性是“启动时恢复”当 Android 11 设备在经过一夜的空中更新后重新启动时，它现在可以立即访问凭据加密存储，并立即开始接收消息。在目前的版本中，在你用你的凭证登录之前，手机大部分时间都处于休眠状态。
