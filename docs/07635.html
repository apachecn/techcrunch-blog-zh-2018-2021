AWS、IBM 启动项目鼓励开发者解决新冠肺炎问题

随着社会开始应对与新冠肺炎病毒相关的日益严重的全球危机，许多公司正在以不同的方式加快步伐。今天，两大科技公司——亚马逊和 IBM——都宣布了鼓励开发人员找到与疫情相关的各种问题的解决方案的计划。

首先，亚马逊的云部门 AWS 宣布了 AWS 诊断开发计划。它已经拨出 2000 万美元，将以 AWS 信用和技术支持的形式分发。该计划旨在帮助和鼓励致力于新冠肺炎诊断问题的团队开发更好的诊断工具。

“在我们的亚马逊网络服务(AWS)业务中，我们听到迫切需要的一个领域是诊断的研究和开发，包括快速、准确地检测和测试新冠肺炎。更好的诊断将有助于加快治疗和遏制，并及时缩短这场流行病的进程，”特蕾莎·卡尔森今天在该公司的第一天博客中写道。

该计划旨在帮助致力于构建诊断解决方案的客户更快地将产品推向市场，并鼓励致力于相关问题的团队相互合作。

该公司还宣布，它正在组建一个由科学家和卫生政策专家组成的咨询小组，以协助参与该倡议的公司。

与此同时，IBM 正在重新关注其 2020 年代码呼吁全球挑战开发者竞赛，不仅解决与全球气候变化相关的问题，这是今年的原始章程，而且通过构建开源工具来解决围绕日益增长的病毒危机的问题。

“在很短的时间内，新冠肺炎揭示了我们习以为常的制度的局限性。该公司在一篇博客文章中写道:“2020 年代码呼吁全球挑战将为你提供资源[……]来构建解决新冠肺炎三个主要领域的开源技术解决方案:紧急情况下的危机沟通，改善远程学习的方法，以及如何激励当地社区的合作。”。

随着越来越多的人被迫呆在室内，所有这些地区都被征税，因为我们试图控制病毒。该公司希望激励致力于这些问题的开发人员帮助解决其中一些问题。

在一个社会和经济极度动荡的时期，社会的各个方面都受到影响，企业、学术界和政府需要共同努力，解决与病毒有关的无数问题。这只是其中的几个例子。
