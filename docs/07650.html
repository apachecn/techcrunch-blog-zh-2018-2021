PSA:是的，您可以在浏览器中参加缩放会议 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/20/psa-yes-you-can-join-a-zoom-meeting-in-the-browser/

新冠肺炎时代的工作，一种由冠状病毒引起的疾病，正在推动视频会议的巨大增长，因为许多办公室工作人员走到远处，在家里登录会议。

Zoom 提供了一系列灵活的功能，如自动转录和虚拟背景，已经成为这一病毒热潮的主要受益者。

回到2 月，研究分析师 Bernstein 引用 Apptopia 的数据估计，该公司在 2020 年的两个月里吸引了比 2019 年全年更多的活跃用户，Apptopia 建立了由 SDK 提供的模型，该 SDK 跟踪第三方应用的大型网络的下载。It 研究表明，截至 2 月底，Zoom 的月活跃用户增加了 222 万，而 2019 年全年的月活跃用户为 199 万。

他们估计 Zoom 总共有 1292 万 mau，自 2019 年底以来增长了 21%。

问题是，你实际上不必下载 Zoom 的应用程序来使用视频会议工具；它可以在浏览器中工作。

击败缩放的黑色图案—从您的浏览器加入

在浏览器中加入缩放会议的选项隐藏得非常好，除非会议主持人调整了默认设置。

就像字面上的隐藏，直到后的你点击加入一个会议——当通过浏览器加入的选项出现在一个更突出的链接“下载& run Zoom”下面的微小文本中。

要么是真的可怕的 UX，要么是为了推动应用下载的黑暗模式。

至少可以说，在浏览器中隐藏加入 Zoom 通话的选项是次优的，因为许多在家工作的人将使用锁定应用下载的公司笔记本电脑来降低安全风险。(而且新冠肺炎有很多网络风险。)

要求下载一个应用程序来加入呼叫也是不相称的。Zoom 应该宣传使用其应用程序的额外好处——它支持更多功能——而不是试图欺骗用户，让他们认为如果他们想开会，他们别无选择，只能下载它。

一些批评者认为，Zoom 的方法看起来不负责任，因为它可能导致沮丧的与会者选择拨入会议，从而错过观看幻灯片等多媒体内容的能力。当世界正在应对全球健康危机时，这真的不是什么好事。

绕过 Zoom 的黑暗模式意味着你要睁大眼睛寻找一个微小的文本链接“从你的浏览器加入”,然后点击它。

主持 Zoom 会议的人可以通过进入设置并切换它们来改善不诚实的情况，以便自动显示“从您的浏览器加入”链接，而不是要求参与者在 Zoom 出现之前先尝试下载它。

玩一个寻找并点击小挂锁的游戏

要进行此更改，您需要转到您的缩放帐户设置，导航到“会议”选项卡，然后在“会议(高级)”中，您应该会找到一个“显示‘从您的浏览器加入’链接”的切换按钮

"验证该设置已启用。如果设置被禁用，请单击状态切换将其启用。如果显示验证对话框，请选择“打开以验证更改”，这是 Zoom 在其网站上的帮助文本中解释该过程的方式。

但还有更多！这样做实际上并没有为所有用户锁定变更。

开关的右手边是一个小小的浅灰色挂锁——眯着眼睛你会看不到它——如果你想让所有用户强制设置这个设置，你必须点击它，然后点击“锁定”以确认更改。

所以再按两下。

还可以显示由特定群组成员主持的所有会议的浏览器链接，这可以通过“群组管理”完成，然后单击群组名称，接着单击“设置”选项卡(然后重复上述步骤)。

对于您自己的缩放会议:登录您的帐户，单击“我的会议设置”(如果您是管理员)或“我的会议”(如果不是)，然后像以前一样导航到“在会议中(高级)”下的“会议”选项卡上的“显示‘从浏览器链接加入’”选项。

根据 Zoom 的帮助文档，如果显示浏览器链接的选项完全灰显，它在帐户或组级别被锁定。此时，您需要首先在帐户级别更改它，或者联系您的管理员请求更改。

如果你能找到的话，Zoom 的网站上有一页关于其网络客户端的信息，在那里它推荐使用谷歌浏览器来获得最大的功能。

但你也可以从 Firefox、Safari、Edge 或 Internet Explorer 加入。谁需要应用程序？
