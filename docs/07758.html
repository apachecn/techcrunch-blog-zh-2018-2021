苹果延长其专业创意应用的免费试用期

原文：https://web.archive.org/web/https://techcrunch.com/2020/03/26/apple-extends-free-trials-for-its-pro-creative-apps/

苹果延长其专业创意应用的免费试用期

苹果公司今天宣布，他们暂时将 Final Cut Pro X 和 Logic Pro X 的免费试用期从 30 天延长至 90 天，让被困在家里的潜在客户有更长的时间试用该软件。

苹果公司表示，该公司希望这一推广将有利于富有创造力的 Mac 用户，尤其是学生，他们发现自己在新的环境中创造和学习。

在新冠肺炎危机期间，苹果加入了许多其他软件公司的行列，延长了他们产品的免费试用期。值得注意的是，苹果不收取订阅费，所以如果你想继续使用视频或音频编辑软件套件，你必须支付全额购买价格。

Logic Pro X 的零售价为 199.99 美元，Final Cut Pro X 的售价为 299.99 美元。苹果公司表示，Final Cut Pro X 的延长试用版已经上线，并将在最近的下载中获得荣誉，而 Logic Pro X 应该会在未来几天获得延长试用版。
