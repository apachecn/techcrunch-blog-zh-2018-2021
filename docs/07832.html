Color 正在启动一个高容量的新冠肺炎测试实验室，并将开源其设计和协议

基因组健康技术初创公司 Color 正在为解决全球新冠肺炎疫情问题尽自己的一份力量，并在周二的一篇新的博客文章和首席执行官 Othman Laraki 的一封信中详细介绍了支持扩大测试工作的步骤。这些努力包括开发一个高通量实验室，每天可以处理多达 10，000 项测试，并在 24 小时内将结果报告给医生。为了从建立这个实验室的努力中获得尽可能多的好处，Color 还将把这个实验室的设计、协议和细节开源给任何希望建立高容量实验室测试的人。

据 Laraki 说，Color 的实验室也几乎准备好开始处理样品——它将在“未来一周”上线。Color 团队与麻省理工学院的 Broad Institute 以及哈佛大学和威尔康奈尔医学院合作，开发其过程和测试技术，与标准的使用中的方法相比，可以允许更高带宽的结果输出。

Color 努力实现这一目标的重点是尽可能使用自动化，并寻求能够从不同供应链获得零部件(包括试剂)的技术。这实际上是能够在全国和全球范围内加大努力的一个关键因素，因为如果每个人都使用相同的实验室处理方法，你将很快遇到供应方面的瓶颈。每天能够处理成千上万的测试在理论上是很棒的，但如果你需要的一种成分也是这个国家其他所有测试实验室所需要的，那就毫无意义了。

Color 还努力在另外两个关键领域解决新冠肺炎响应问题:一线和重要员工的测试，以及测试后的跟进和处理。为了满足那些不顾风险继续在公共场合工作的员工的测试需求，Color 重新调整了其企业员工基础，与政府和雇主合作，提供现场临床测试管理、实验室运输和向患者医生报告结果。

对于其测试后工作流程，Color 正在努力解决其他临床医生和卫生官员报告的挑战，即在跟踪测试结果以及后续步骤时保持一致和有效有多困难。因此，该公司正在为此开放他们自己的平台，他们根据迄今为止的经验重新设计了该平台，并将其提供给任何其他新冠肺炎测试实验室免费使用。这些资源包括检测结果报告、患者指南和说明、追踪接触者的后续调查问卷，以及如何联系与检测呈阳性的患者有关联的潜在接触者的支持。

Color 表示，迄今为止，它已经能够以成本价运营，这在一定程度上得益于公共和私人慈善捐赠的支持。该公司鼓励通过其 covid-response@color.com 电子邮件直接联系，以防有人认为他们可以为该项目和可用资源做出贡献或从中受益。
