也许我们根本不应该使用变焦

由于冠状病毒疫情，现在我们都被困在家里，视频电话已经从新奇变成了必需品。流行的视频会议服务 Zoom 似乎比大多数人做得更好，如果不是最受欢迎的选择，也很快成为其中之一。

但是应该吗？

Zoom 最近的受欢迎程度也让人们关注该公司的安全保护和隐私承诺。就在今天，The Intercept 报道称【Zoom 视频通话没有进行端到端加密，尽管该公司声称它们是这样的。

Motherboard 报道称，Zoom 正在泄露“至少几千”人的电子邮件地址，因为个人地址被视为属于同一家公司。

这是该公司不得不花去年时间清理接二连三的头条新闻，检查该公司的做法和误导性营销的最新例子。也就是说:

  • 在一名安全研究员发现 Zoom 未能披露其在用户的 MAC 电脑上安装了一个秘密网络服务器后，苹果被迫介入保护数百万台 MAC 电脑的安全。研究人员 Jonathan Leitschuh 表示，网络服务器意味着任何恶意网站都可以在未经用户许可的情况下激活安装了 Zoom 的 Mac 网络摄像头。研究人员拒绝了一笔 bug 赏金，因为 Zoom 希望 Leitschuh 签署一份保密协议，这将阻止他披露 bug 的细节。
  • Zoom 悄悄向脸书发送用户的 Zoom 习惯数据——即使用户没有脸书账户。Motherboard 报告称，iOS 应用程序会通知脸书他们何时打开应用程序、设备型号、他们打开应用程序的手机运营商等等。作为回应，Zoom 删除了代码，但还不足以让阻止集体诉讼或纽约州总检察长发起调查。
  • Zoom 因其“与会者跟踪”功能再次受到抨击，该功能启用后，主持人可以检查与会者在通话过程中是否离开了主 Zoom 窗口。
  • 一名安全研究人员发现，Zoom 使用一种“秘密”技术在没有用户交互的情况下安装其 Mac 应用程序。研究人员说“MAC OS 恶意软件正在使用相同的伎俩”。
  • 好的一面是，让一些用户松了一口气的是，我们报道说事实上可以加入缩放视频通话，而不必下载或使用该应用程序。但 Zoom 的“黑暗模式”并不便于仅使用浏览器启动视频通话。
  • Zoom 因其收到的执法请求缺乏透明度而面临质疑。隐私和权利组织 Access Now 呼吁 Zoom 公布其收到的请求数量，就像亚马逊、谷歌、微软和许多其他科技巨头每半年报告一次一样。
  • 然后是 zoom bombing(zoom bombing )( T1 ),在这里，巨魔利用开放或无保护的会议和糟糕的默认设置来接管屏幕共享，并广播色情或其他露骨的内容。美国联邦调查局本周警告用户调整他们的设置，以避免巨魔劫持视频通话。
  • Zoom 本周收紧了其隐私政策，此前它因允许 Zoom 收集用户会议信息——如视频、文字记录和共享笔记——用于广告而受到批评。

除了 Zoom，还有很多更注重隐私的选择。有几个选项，但是都有陷阱。FaceTime 和 WhatsApp 是端到端加密的，但 FaceTime 只能在苹果设备上运行，WhatsApp 一次只能有四个视频通话者。一个不太为人所知的视频通话平台 Jitsi ，不是端到端加密的，但它是开源的——所以你可以查看代码以确保没有后门——它可以在所有设备和浏览器上工作。你可以在你控制的服务器上运行 Jitsi 以获得更大的隐私。

公平地说，Zoom 本质上并不坏，Zoom 如此受欢迎有很多原因。它易于使用，可靠，而且对大多数人来说非常方便。

但是 Zoom 的误导性声明给用户一种错误的安全感和隐私感。无论是举办虚拟快乐时光还是瑜伽课，或者使用 Zoom 进行治疗还是召开政府内阁会议，每个人都应该拥有隐私。

现在，Zoom 比以往任何时候都更需要对用户负责。目前来看，放大自担风险。
