ImmunityBio 和微软合作精确模拟关键新冠肺炎蛋白如何导致感染

原文：https://web.archive.org/web/https://techcrunch.com/2020/04/01/immunitybio-and-microsoft-team-up-to-precisely-model-how-key-covid-19-protein-leads-to-infection/

ImmunityBio 和微软合作精确模拟关键新冠肺炎蛋白如何导致感染

一项涉及结合大量图形处理能力的任务，可以为研究人员提供关键的杠杆作用，这些研究人员希望为当前全球疫情背后的新型冠状病毒开发潜在的治愈和治疗方法。免疫疗法初创公司 ImmunityBio 正在与微软的 Azure 合作，提供总计 24 petaflops 的 GPU 计算能力，用于以非常高的细节程度模拟所谓的“刺突蛋白”的结构，这种结构允许导致新冠肺炎病毒进入人类细胞的新型冠状病毒病毒。

这种新的合作关系意味着他们能够在几天内生产出刺突蛋白的模型，而不是以前需要几个月的时间。节省的时间意味着该模型可以更快地到达研究潜在疫苗和治疗方法的研究人员和科学家的虚拟手中，他们将能够将他们的工作转向详细复制他们试图阻止附着在人类 ACE-2 蛋白受体上的蛋白质，这是病毒感染过程的开始。

致力于治疗的科学家们寻求防止或最大限度地减少病毒在体内传播的主要方法是阻止病毒附着在这些蛋白质上，而最简单的方法是确保刺突蛋白无法与它的目标受体相连。从新型冠状病毒中康复的患者体内自然产生的抗体正是这样做的，正在开发的疫苗专注于先发制人地做同样的事情，而许多治疗方法都在寻求降低病毒在体内复制时锁定新细胞的能力。

实际上，两家公司之间的合作伙伴关系包括 1250 个 Nvidia V100 Tensor Core GPUs，这些 GPU 旨在用于微软 Azure 集群的机器学习应用程序，与 ImmunityBio 现有的 320 个 GPU 集群配合使用，这些 GPU 集群专门针对分子建模工作进行了调整。合作的结果现在将提供给研究新冠肺炎缓解和预防疗法的研究人员，希望他们能够更快更有效地找到解决方案。
