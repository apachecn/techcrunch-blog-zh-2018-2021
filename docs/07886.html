Facebook Messenger发布Mac和Windows应用程序

Facebook Messenger在首次亮相近9年后终于获得了桌面应用程序。在看到桌面浏览器音频和视频通话增长超过100%后，Messenger today 发布了其 Mac 和 Windows 桌面应用。它们带来了与浏览器版本相同的功能，但比将Messenger隐藏在众多标签中的一个标签中更容易保持您的聊天线索。“我们都在寻找更多的方式在一起，即使我们身在异地，”脸书首席执行官马克·扎克伯格写道。

脸书最初在一年前的F8会议上发布了Messenger桌面应用。但鉴于今年的会议因冠状病毒而取消，用户显然现在就想要桌面应用程序，所以在没有大型新闻发布会的情况下推出它们是有意义的。上个月，T2在测试T3时发现了Mac版本。Messenger在2011年曾短暂拥有一款 Windows应用，直到现在才被废弃。

也许Facebook Messenger目前最大的缺点是它最多只能处理8个群组视频通话参与者，而Zoom则可以处理100个甚至500个参与者。这意味着Messenger无法支持一些新的用例，这些用例是由于新冠肺炎病毒导致人们开始就地躲避而出现的。它也没有提供一种方法来轻松共享一个URL，让其他人加入呼叫，所以它对公共网络研讨会或社交活动没有用处。

屏幕共享是另一个主要缺失的功能。无法显示你在看什么妨碍了一些商业用例，以及像共同浏览相册这样的电话社交用例。脸书的工作场所聊天桌面应用程序提供屏幕共享，因此Messenger未来可能会添加该选项似乎是合理的。

聊天应用程序已经从一种在分开时保持联系或安排会议的方式，发展成为我们为那些处于锁定状态的人建立人际联系的唯一形式。这意味着它们不再能作为我们线下生活的补充。他们必须完整地传达我们。
