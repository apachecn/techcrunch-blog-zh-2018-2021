HBO 让一些顶级节目、电影和纪录片免费在 HBO NOW 和 HBO GO 上播放

为了给人们更多的理由呆在家里，遵循旨在阻止新冠肺炎在美国传播的社交距离措施， HBO 表示，从 4 月 3 日周五开始，将在 HBO NOW 和 HBO GO 上免费播放 500 小时的节目，无需订阅。

观众可以观看的节目包括一些有史以来最好的电视节目，如《黑道家族》(The Sopranos)和《火线》(The Wire)，以及其他非常好的 HBO 节目，如《副总统》(Veep)和《六尺之下》

像《精灵宝可梦·大侦探皮卡丘》(pokémon America)、《疯狂、愚蠢、爱情》(Crazy，Stupid，Love)这样的电影和《太阳帝国》(Empire of the Sun)这样的经典电影加入了包括《百万美元》(McMillion$)和《反对阿德南·塞德的案子》(The Case Against Adnan Syed)在内的纪录片的行列，成为免费流媒体服务。

想要观看有史以来最棒的节目(即《火线》)的观众可以下载 HBO NOW 或 HBO GO 应用程序，或者访问 HBONOW.com 的或 HBOGO.com 的。

该公司表示，该网络的分销合作伙伴也将在未来几天免费提供这些节目。该公司表示，这一优惠标志着 HBO 首次在其应用程序的付费墙之外免费提供如此大量的节目。
