苹果和谷歌正在为 iOS 和 Android 发布一款联合新冠肺炎追踪工具

原文：https://web.archive.org/web/https://techcrunch.com/2020/04/10/apple-and-google-are-launching-a-joint-covid-19-tracing-tool/

苹果和谷歌的工程团队已经联合起来，创建了一个分散的接触追踪工具，这将帮助个人确定他们是否接触过患有新冠肺炎的人。

接触者追踪是一种有用的工具，有助于公共卫生当局追踪疾病的传播，并通知潜在的暴露者，以便他们能够接受检测。它通过识别和“跟踪”与新冠肺炎患者接触过的人来做到这一点。

该项目的第一阶段是一个 API，公共卫生机构可以将其集成到自己的应用程序中。下一阶段是一个系统级的联系人追踪系统，将在选择加入的基础上跨 iOS 和 Android 设备工作。

该系统使用设备上的板载无线电在短距离内传输匿名 ID——使用蓝牙信标。服务器会将您最近 14 天的轮换 id 转发给其他设备，这些设备会搜索匹配的 id。基于两个设备之间花费的时间和保持的距离的阈值来确定匹配。

如果发现与另一个告诉系统他们的检测结果为阳性的用户相匹配，您会收到通知，并可以采取措施进行检测和自我隔离。

接触者追踪是一个众所周知和有争议的工具，但它已被卫生当局和大学采用，它们正在从事多个类似的项目。一个这样的例子是麻省理工学院努力使用蓝牙创造一个隐私意识的联系追踪工具，这是受苹果的 Find My 系统的启发。这些公司表示，这些组织发现了他们无法克服的技术障碍，并寻求帮助。

一周前，我们的 Jon Evans 阐述了对更广泛的追踪设备的需求，以及你需要得到苹果和谷歌的认可才能实现的想法。

这个项目是由两家公司的工程师在两周前启动的。这些公司参与其中的原因之一是不同制造商的设备上的系统之间的互操作性很差。有了联系追踪，每次你在多个应用之间分割这样的系统，你就极大地限制了它的有效性。你需要在一个系统中有大量的应用来使联系追踪很好的工作。

与此同时，你会遇到一些技术问题，比如蓝牙耗电、对集中式数据收集的隐私担忧，以及让足够多的人安装有效的应用程序需要付出的巨大努力。

两阶段计划

为了解决这些问题，谷歌和苹果联手创建了一个可互操作的 API，如果用户愿意，它应该允许最大数量的用户采用它。

第一阶段是一个私人的接近接触检测 API，将由苹果和谷歌在 5 月中旬发布，用于 iOS 和 Android 上的应用程序。在今天的简报中，苹果和谷歌表示，API 是一个简单的 API，对于现有或计划中的应用程序来说，应该相对容易集成。该 API 将允许应用程序要求用户选择加入联系追踪(整个系统都是选择加入的)，允许他们的设备向该人“会面”的设备广播匿名的旋转标识符。这将使追踪得以进行，以提醒那些可能与新冠肺炎有接触的人采取进一步措施。

接触者追踪的价值应超越疫情初期，延伸到自我隔离和检疫限制放松的时期。

该项目的第二阶段是通过将跟踪工具引入操作系统级别，为其带来更高的效率和采用率。不需要下载应用程序，用户只需在他们的设备上选择追踪。公共卫生应用将继续得到支持，但这将解决更大范围的用户。

这一阶段预计在未来几个月内完成，将使合同跟踪工具能够在更深的层次上工作，提高电池寿命、效率和隐私。如果它是由系统处理的，那么这些领域的每一项改进——包括加密技术的进步——都将使该工具直接受益。

工作原理

一个这样的系统如何工作的简单例子:

  1. 两个人恰好在对方附近一段时间，比如说 10 分钟。他们的手机交换匿名标识符(每 15 分钟更换一次)。
  2. 后来，其中一个人被诊断患有新冠肺炎，并通过集成了 API 的公共卫生机构应用程序将其输入系统。
  3. 在获得额外同意的情况下，被诊断的用户允许其最近 14 天的匿名标识符被传输到系统。
  4. 他们接触过的人的手机上有一个公共卫生应用程序，可以下载阳性检测的广播密钥，并提醒他们匹配。
  5. 该应用程序为他们提供了更多关于如何从那里开始的信息。

隐私和透明度

苹果和谷歌都表示，隐私和透明度在像这样的公共卫生工作中至关重要，并表示他们致力于推出一种不会以任何方式损害个人隐私的系统。这是美国公民自由联盟在提出的一个因素，它警告说，任何使用手机追踪新冠肺炎的传播都需要积极的隐私控制。

位置数据的使用为零，包括报告阳性的用户。这个工具不是关于受影响的人在哪里，而是关于他们是否和其他人在一起。

该系统的工作原理是给一个人的手机分配一个随机的旋转标识符，并通过蓝牙将其传输到附近的设备。该标识符每 15 分钟轮换一次，不包含任何个人身份信息，将通过一个简单的中继服务器，该服务器可以由世界各地的卫生组织运行。

即使这样，你联系过的标识符列表也不会离开你的手机，除非你选择分享它。测试呈阳性的用户不会被其他用户识别，苹果或谷歌。当不再需要广播系统时，谷歌和苹果可以完全禁用它。

所有匹配的识别都在你的设备上完成，让你可以在 14 天的时间内看到你的设备是否靠近过一个自称新冠肺炎病毒检测呈阳性的人的设备。

整个系统都是选入的。用户将提前知道他们正在参与，无论是在应用程序中还是在系统级别。公共卫生当局参与通知用户他们与受影响的人有过接触。

美国公民自由联盟似乎持谨慎乐观的态度。

“在实现广泛、免费、快速的测试和公平的医疗保健之前，任何接触者追踪应用程序都不可能完全有效。如果人们不信任这些系统，它们也不会有效，”美国公民自由联盟的监控和网络安全顾问詹妮弗·格拉尼克说。“值得称赞的是，苹果和谷歌宣布了一种方法，似乎可以减轻最严重的隐私和集中化风险，但仍有改进的空间。我们将保持警惕，确保任何合同追踪应用程序都是自愿和分散的，仅用于公共卫生目的，并且仅在疫情期间使用。”

苹果和谷歌表示，他们将公开发布有关他们所做工作的信息，供他人分析，以尽可能提高该项目的隐私和安全方面的透明度。

“苹果和谷歌的所有人都认为，合作解决世界上最紧迫的问题之一，这是前所未有的重要时刻，”两家公司在一份声明中表示。“通过与开发商、政府和公共卫生提供商的密切合作和协作，我们希望利用技术的力量来帮助世界各国减缓新冠肺炎的传播，并加快日常生活的恢复。”
