苹果和谷歌讨论他们的冠状病毒追踪工作

原文：https://web.archive.org/web/https://techcrunch.com/2020/04/13/apple-google-coronavirus-tracing/

上周，苹果和谷歌宣布合作,将很快让用户选择加入一个分散的追踪工具，旨在帮助确定一个人是否与新冠肺炎病毒检测呈阳性的人有过接触。

选择加入系统使用蓝牙向附近的设备传输随机和匿名的标识符。然后，用户可以选择上传他们的匿名数据，然后将这些数据广播到其他设备。如果根据花费的时间和附近设备之间的距离找到匹配，用户将被告知他们可能接触了一个携带病毒的人——其身份未被共享。

这是一个类似于麻省理工学院研究人员构想的系统，它也使用蓝牙匿名通知其他人潜在的感染。与苹果和谷歌的新成果一样，该系统也回避了位置数据的使用。

接触者追踪在世界上的一些地方被证明是有效的，有助于当局了解感染热点。但是隐私组织和安全专家担心，为了控制病毒的传播，隐私会被置于个人权利之后。苹果和谷歌表示，这项服务注重隐私。该系统不使用位置数据，用户的随机标识符每 15 分钟改变一次以防止跟踪，收集的任何数据都在设备上处理，不会离开用户的手机，除非他们选择分享。

但是安全和隐私专家很快指出了系统中可能存在的缺陷。前 FTC 首席技术专家 Ashkan Soltani 警告假阳性和假阴性。Signal 加密信息应用的创始人莫邪·马林斯派克也表达了对该系统可能被滥用的担忧。

TechCrunch 加入了苹果和谷歌代表的媒体电话会议，允许记者就他们的冠状病毒追踪工作提问。

以下是电话中讨论的内容。

iOS 和 Android 的哪些版本会获得功能更新？

苹果表示，它将尽可能在最广泛的 iOS 设备上推出这一更新。超过四分之三的 iPhones 和 iPads 都是最新版本的 iOS 13，将会收到更新。谷歌表示，它将更新 Android 的核心部分 Google Play 服务，以便联系人跟踪系统可以在所有 Android 设备(运行 Android 6.0 或更高版本)上运行，而不仅仅是最近更新的设备。

这个追踪系统什么时候可以用？

苹果和谷歌表示，他们将在 5 月中旬推出软件更新，开始支持联系人追踪。公共卫生当局将把接触者追踪 API 整合到他们的应用程序中，然后可以从苹果和谷歌应用商店下载。这些公司表示，他们将在未来几个月内将联系人追踪功能植入 iOS 和 Android，这样用户甚至不必安装应用程序。这些公司表示，这将有助于让更多的人使用该系统。

即使当接触者追踪功能在系统级别被嵌入到操作系统中时，任何阳性匹配的检测仍然会提示用户下载他们所在地区的相关公共卫生应用程序，以接收有关新冠肺炎接触者追踪流程和后续步骤的更多信息。

其他人可以使用这个 API 吗？

这些公司表示，只有公共卫生部门才被允许使用接触者追踪 API。

这种有限的 API 使用将受到限制，就像您将个人医疗保健限制为像医生这样的有执照的医疗专业人员一样。同样，API 的使用将仅限于指定给特定国家或地区的政府指定的授权公共卫生组织。在某些情况下，关于什么是合法的公共卫生机构，可能会有冲突，甚至可以想象，国家和州政府之间会有分歧，所以这听起来可能是一个可能发生摩擦的地方，苹果和谷歌作为平台运营商的地位很微妙。

这些数据会存储在中央数据库中吗？

苹果表示，数据是在用户的设备上处理的，数据是通过世界各地卫生组织运行的服务器“中继”的，不会被集中。这些科技巨头表示，由于数据是分散的，政府进行监控的难度要大得多。

这是否意味着苹果、谷歌或公共卫生当局可以访问这些数据？

苹果和谷歌承认，没有任何系统是完全安全的——这是网络安全领域一个广为人知的概念，即没有什么是“无法破解的”服务器可能被攻破，数据可能丢失。但他们说，在分散数据的过程中，任何怀有恶意的人都很难访问这些数据。

你是如何防止人们制造虚假报告的？

这些公司表示，他们正在与不同的公共卫生组织合作，以确认诊断，如公共卫生当局，以进行验证。苹果和谷歌表示，他们希望用户信任该系统，这包括用户知道该系统是可靠的。

如何识别新冠肺炎确诊病例？

苹果和谷歌指出，虽然阳性检测结果可能是确定案件的最佳方式，但它不一定是唯一的方式。诚然，医学专业人士的诊断实际上并不需要明确确认病毒存在的阳性检测结果——理论上，公共卫生机构可以设定一个较低的标准，例如，只需要基于症状表现的诊断。

两家科技巨头都承认，为了使接触者追踪有效，需要在人群中有高度的病例识别，但也有可能高度的病例识别不一定会转化为一对一的广泛测试，如果任何特定地区的当地卫生当局认为其他识别病例的方法足够可靠的话。

你应该相信这个系统吗？

没有简单的答案。看起来苹果和谷歌创造了一个聊胜于无的系统，但这是一个需要相当大的用户信任的系统。你必须相信苹果和谷歌已经建立了一个可以抵御滥用的系统——无论是来自他们自己还是政府。但是没有一个系统是万无一失的，或者是不会被滥用的。如果你不信任这个系统，你可以不用它。
