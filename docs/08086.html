勤勤恳恳为移动通讯再筹集 4000 万美元，将投资帮助客户响应新冠肺炎 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/04/15/attentive-raises-another-40m-for-mobile-messaging-will-invest-in-helping-customers-respond-to-covid-19/

移动通讯初创公司attention继续带来新的资金。

初创公司去年夏天筹集了 4000 万美元的 B 轮，随后在今年年初筹集了 7000 万美元的 C 轮。今天，它宣布将 C 系列再增加 4000 万美元，使总融资规模达到 1.1 亿美元。

首席执行官布莱恩·龙(他之前和他细心的联合创始人安德鲁·琼斯一起创立了 TapCommerce，然后把公司卖给了 Twitter )告诉我，新的投资一周前刚刚结束。他说，这笔钱来自机构投资者，他们曾想参与 C 轮融资，但“无论出于何种原因，时机并不成熟。”

后来，当这家初创公司想要投资新领域时——尤其是为了应对新冠肺炎疫情事件——龙再次伸出了手。一旦他们看到勤勤公司 2020 年第一季度的数字，这些公司就愿意投资。

显然，新客户注册的数量只会越来越多，Attentive 现在与 1000 多家企业合作。像蔻驰、Urban Outfitters、CB2、PacSun、Lulus 和 Jack in the Box 这样的公司使用该平台来管理他们的移动消息，这些工具包括添加短信订户、创建吸引人的消息以及跟踪这些活动的结果。

虽然我们正处于广告和营销可能大幅放缓的开端，但龙建议，即使企业减少获取新客户，他们仍需要维持与现有客户的关系。

“CRM 对公司来说是一个非常重要的渠道……电子邮件和短信是你最不想关闭的东西，”他说。

红杉资本全球股票(Sequoia Capital Global Equities)和 Coatue 是 C 轮投资的新投资者。红杉资本的风险基金已经牵头(或共同牵头)了 C 轮和 B 轮投资，但龙表示，他有兴趣与该公司的交叉基金和 Coatue 合作，部分原因是他们也投资上市公司。

他并没有立即让 Attentive 上市的计划，但他说，“这只是创造了可选性”，这样无论公司走哪条路，财务压力都会更小。

C 轮的其他投资者包括 IVP、贝恩资本风险投资公司、NextView Ventures、Eniac Ventures 和 High Alpha。

红杉资本(Sequoia Capital)全球股票管理合伙人王建复在一份声明中表示:“勤勤的快速增长表明，消费者渴望找到一个更直接、更个性化、更高效的渠道与企业互动。”。“我们对勤勤的业务增长速度、强劲的客户势头以及团队的专业知识印象深刻。我们很高兴通过我们的全球股票基金加强红杉与勤勤的合作关系。”

至于对新冠肺炎的回应有多认真，这家初创公司计划创建基金，帮助客户应对经济衰退。未来几周将公布更多细节，但龙表示，这个想法是推出专注于电子商务/零售、食品/饮料和教育领域的基金，提供免费的工具和服务，“以帮助这些公司充电。”

龙补充说，他希望到今年年底，勤勤的员工人数能从 260 人增加到 400 人以上。
