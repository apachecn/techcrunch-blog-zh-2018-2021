金融科技初创公司Previse融资1100万美元，帮助供应商更快获得付款TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/04/15/financial-tech-startup-previse-raises-11-million-to-help-suppliers-get-paid-faster/

专注于帮助供应商获得更快付款的金融科技初创公司Previse 宣布，它已获得由Reefknot Investments和万事达卡牵头的1100万美元新融资。回归投资者Bessemer Venture Partners、Hambro Perks和Augmentum Fintech也参加了会议。

Previse成立于2016年，表示目前每天处理约10万张发票，其目标是在未来五年内处理500万供应商的付款。

这一轮融资使Previse迄今为止的融资总额超过2180万美元，并将用于将其InstantPay产品扩展到全球更多的企业买家。Previse正在参加万事达卡的Start Path accelerator计划。Reefknot是由淡马锡控股公司和Kuehne + Nagel于去年在成立的，用于投资物流和供应链初创公司。

Previse的创始人兼首席执行官保罗·克里斯滕森(Paul Christensen)告诉TechCrunch，InstantPay允许企业买家通过使用基于机器学习的技术来分析历史数据，预测哪些发票可以立即支付，哪些发票具有潜在的更高风险，需要人工检查，从而向供应商发送快速付款。

大型买家使用的传统发票支付方式可能需要数月才能完成，这给中小型企业的现金流带来了压力。克里斯滕森说，这是由于公司政策的综合作用，包括销售的条款和条件，以及需要执行的管理任务的数量，包括输入、检查和批准发票。即时支付可以将时间缩短到一天。

他补充说，在新冠肺炎疫情期间，向供应商快速付款甚至更为重要。

“疫情给世界各地大大小小的公司的营运资本带来了巨大压力，造成了严重的现金短缺。他说:“Previse的平台可以释放营运资金，这意味着向大型企业链供货的成千上万中小企业供应商可以在第一天获得付款，而不必等待数周或数月。”

“现在，当供应链被打乱时，这一点至关重要，但当我们从另一边走出来，出现需求激增、供应商必须履行大订单时，这一点也将至关重要。”
