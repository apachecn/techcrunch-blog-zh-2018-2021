Pinpoint 发布仪表板，为软件工程运营带来可见性 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/04/15/pinpoint-releases-dashboard-to-bring-visibility-to-software-engineering-operations/

随着公司寻找更好的方法来了解不同部门如何在粒度级别上工作，工程传统上一直是孤立数据的黑匣子。总部位于奥斯汀的初创公司 Pinpoint 一直在开发一个平台，将这些信息纳入单一视图，今天它发布了一个仪表板，帮助公司从运营角度了解软件工程领域正在发生的事情。

Pinpoint 的联合创始人兼首席执行官杰夫·海涅(Jeff Haynie)表示，该公司过去两年的使命是让工程部门获得更大的可见性，他表示，在当前员工分散在家中的背景下，这一点甚至更加重要。

“公司给工程一大笔钱，他们建造了一堆令人惊叹的东西，但最终，它只是一个黑匣子，我们真的不知道会发生什么，”海尼说。他说，他的公司一直在努力收集所有的数据，尝试将这些数据联系起来，整合在一起，并将这些信息关联起来。

如今，他们推出了一个仪表板，将他们一直在构建的内容整合到一个视图中，这是 100%的自助服务。在此之前，您需要 Pinpoint 人员的大量帮助来启动和运行它，但今天您可以下载产品并登录到您的各种服务，如您的 git 存储库、您的 CI/CD 软件、您的 IDE 等等。

它还为工程人员提供了一种无需离开工具就可以相互交流的方式。

精确定位软件工程仪表板。图片来源:精确定位

“显然，我们会在人们需要时给予帮助，我们有一个企业版的产品，具有更高的 SLA 水平，我们有一个客户成功团队来做这件事，但我们真的把这个新版本的重点放在纯粹的自助服务上，”Haynie 说。

更重要的是，虽然 10 人以下的团队已经有了永远免费的免费版本，但随着今天产品的发布，该公司将在三个月内免费提供对仪表盘的无限制访问。

Haynie 说，他们现在就像任何一家创业公司一样，但他有过与其他几家创业公司合作的经验，经历过 9/11、网络公司崩溃、2008 年等等，他知道如何保持低调并保留现金。与此同时，他说他们看到了很多对该产品的内在兴趣，他们希望找到一种创造性的方法来帮助客户度过这场危机，同时将产品推出供人们使用。

“坦率地说，在这一点上，我们就像其他任何初创公司或任何其他企业一样:我们紧张又害怕。你如何熬过这一关[以及这一关会持续多久]？另一方面，我们急于利用我们正在获得的这种外来兴趣，并试图抓住机会，尝试在如何帮助他们方面有所创新。”

这家初创公司希望，如果公司发现产品有用，三个月后他们不会介意购买完整版本。目前，它只是把它免费放在那里，看看会发生什么——只是另一个试图找到度过这场危机的方法的创业公司。
