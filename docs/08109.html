随着经济形势恶化，独角兽裁员不断增加

今天早些时候，一系列新数据呈现了一幅美国经济的负面画面。今天下午，有消息称，三家知名的、资金雄厚的独角兽公司正在裁员。

随着股市下跌，我们在同一天收到了来自私人市场、公共市场和整体经济的负面信号。让我们花一分钟来设置宏观舞台，然后回顾一下从 Carta ( 最早由彭博报道)、 Zume ( 商业内幕爆料)和开门 ( 通过信息)的最新剪辑。

经济不景气

今天裁员的背景是美国经济摇摇欲坠。浏览一下最近的新闻就足够了。在过去的几个小时里，房屋建筑商的信心创下了“历史上最大的跌幅”，，三月份零售额下降了8.7%，美国消费者新闻与商业频道指出这是“政府数据中有史以来最大的跌幅”，而 CNN商业报道美国工厂的产出在三月份下降了5.4%，“这是自1946年以来最大的一个月下滑。”

因此，我们一整年都在目睹独角兽裁员，这或许并不奇怪。一月份的新闻是，由愿景基金支持的公司削减开支，向盈利靠拢。然后，第一轮新冠肺炎强制裁员落在大公司；像Bird和TripActions这样的公司大规模裁员，因为疫情及其相关的经济和社会变化导致他们的公司核心业务放缓。

小公司的裁员几乎是长期的，TechCrunch也报道了这一点。

然而，今天，三家长期以来一直受到TechCrunch关注的独角兽公司(价值10亿美元或以上的私营公司)遭到了三次裁员。所以，让我们简单地谈谈它们:

  • 由软银支持的旧金山房屋销售初创公司Opendoor今天宣布大幅裁员。在提供给TechCrunch的一份声明中，该公司首席执行官巫启贤表示，将裁员35%，以“确保我们能够继续履行使命”这位首席执行官还表示，离职员工将获得8周的工资，并“报销16周的健康保险。”吴还将他2020年的工资捐给了一个支持员工的基金。在去年3月宣布的3亿美元融资轮中，Opendoor最近的估值为38亿美元。
  • 总部位于旧金山的私人公司股权服务平台Carta宣布裁员16%，或161个职位，根据该公司公开分享的备忘录。以前的eShares，Carta已经从一个为小型私营公司提供股权管理的提供商发展成为一个更大、更广泛的服务和软件公司，为尚未私有化的公司提供支持。Carta最近在去年5月以17亿美元的估值融资3亿美元。
  • 最后是Zume。截至发稿时，Zume没有回应置评请求，也没有发布我们可以找到的公开声明。尽管如此，Business Insider报道称，在2020年裁员后，该公司将再裁员200人。该公司将剩下大约100名员工，从事可堆肥箱子的工作。Zume上一次以略低于19亿美元的估值融资3.75亿美元是在2018年11月。

越来越难追踪所有的伤口了。见鬼，我最近和 TechCrunch的Natasha Mascarenhas 一起帮助打破了Modsy裁员，而且我们也是BounceX裁员的第一人。这是一个艰难、糟糕的经济，它损害了喜欢创业独角兽的成长型公司。

当我们有更多的时候，可能比我们想报告的还要快。
