Instagram创始人推出新冠肺炎传播跟踪器Rt.live

Instagram创始人凯文·斯特罗姆和迈克·克里格联手推出了他们离开脸书母舰后的第一款产品。Rt.live是一个追踪新冠肺炎在每个州传播速度的最新工具。

“Rt”衡量的是被传染者感染的平均人数。数字1越高，新冠肺炎在人群中传播越快，而数字1以下则表明病毒在消退。例如，Rt.live显示，佐治亚州的Rt值最高，最危险，为1.5分，而纽约由于积极的就地安置令而降至0.54分。

Krieger告诉我，“Kevin一直在撰写和出版关于如何每天计算Rt的开源数据分析笔记本。我们希望将这项工作可视化，这样任何人都可以看到他们的国家在遏制传播方面做得如何。”与此同时，克里格也参与进来，建立了 SaveOurFaves ，这是一份出售礼品卡的当地湾区餐馆名录，这样顾客就可以在隔离期间维持运营。克里格夫妇和他的妻子一起建造了这个网站，并将其开源，这样人们就可以为他们的社区建造类似的网站。

Rt.live显示，截至昨天，德克萨斯州和加利福尼亚州的得分为1或略低于1，佛蒙特州的得分最高，为0.33。随着时间的推移，图表揭示了华盛顿和佐治亚州是如何成功地抗击新冠肺炎病毒的，一直下降到1以下，直到病毒最近反弹。数据来源于 COVID跟踪项目，你可以在 GitHub 上检查Rt.live的建模系统。

“当各州决定是否以及如何开放备份时，他们必须小心地管理他们的感染率，我们希望像rt.live这样的仪表板将有助于这样做，”Krieger说。通过更好地说明就地安置政策和合规性方面的微小差异如何能够成倍地改变病毒影响的严重性，它可以帮助说服人们留在室内。这种工具也有助于确定在哪里重新激活某些业务是安全的，并快速捕捉病毒传播是否激增以及严格的社交距离是否需要恢复。

该网站一个引人入胜的功能是可以按地区过滤，这样你就可以看到西部各州在压制新冠肺炎方面比南部做得更好。你也可以查看没有避难所订单的州，看看他们平均做得更差。这些图表有助于确定不同的政治取向及其后续政策如何转化为感染结果。

照片应用巨头建立一个医学统计网站似乎不符合他们的性格。但Systrom长期以来一直在研究病毒，这是他帮助Instagram快速发展的工作的一部分。3月19日，他开始发布自己的统计模型，用于追踪冠状病毒感染和死亡。“我们一直在讨论合作的方式，并由此产生了这一点——我离开学校后的第一份工作实际上是在Meebo做数据可视化/分析，因此从多个方面来看，这是一次来自过去的冲击。”Krieger告诉我。Systrom进行数据分析，Krieger建立网站，反映了他们以前的前端和后端Instagram角色。

“我们建立了 Rt.live ,因为我们相信Rt——有效感染率——是了解COVID如何传播的最佳方式之一”克里奇解释道。“能够再次合作真是太好了，因为我们有着共同的历史背景，所以我们能够在短短几天内从构思到发布。”
