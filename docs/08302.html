为了打击欺诈，亚马逊现在通过视频电话筛选第三方卖家 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/04/27/to-fight-fraud-amazon-now-screens-third-party-sellers-through-video-calls/amp/

图片来源:Beata Zawrzel/Nur photo/Getty Images

亚马逊(Amazon)周日宣布，该公司正在试行一个新系统，旨在通过视频会议验证第三方卖家的身份。该技术是亚马逊用来打击其平台上欺诈行为的一系列卖家验证流程的一部分，该公司称，该平台在 2019 年阻止了 250 万名涉嫌不良行为者向亚马逊发布他们的产品。

今年早些时候，亚马逊开始测试一个过程，由卖家亲自验证。但由于冠状病毒的爆发和社交距离的要求，该公司表示，它在 2 月份转向了现场视频会议。

该试点项目目前正在多个市场开展，包括美国、英国、中国和日本。亚马逊表示，迄今为止，已有 1000 多名卖家试图通过试点体验注册账户。

为了审查卖家，亚马逊的团队建立了一个视频电话，然后检查个人的 ID 是否与他们在应用程序中共享的人和文档相匹配。亚马逊员工还依靠第三方数据源进行额外的验证。此外，该呼叫可用于向卖方提供关于其注册问题以及如何解决这些问题的信息。

“亚马逊一直在创新，以改善卖家体验，让诚实的企业家可以无缝地开设销售账户和开展业务，同时也主动阻止不良行为者，”亚马逊发言人对新举措表示。“在我们练习社交距离时，我们正在测试一个允许我们通过视频会议验证潜在卖家身份的流程。这个试点项目让我们能够与潜在卖家进行一对一的联系，同时让欺诈者更难隐藏，”他们说。

除了视频会议，亚马逊还使用专有的机器学习系统在卖家被允许上网之前对其进行审查。该系统分析数百个不同的数据点来识别潜在的风险，包括例如验证该账户是否与先前从市场中移除的另一个账户相关。卖家的申请在被批准前还要经过训练有素的调查人员的审查。

然而，卖家验证只是亚马逊打击欺诈的一种方式。

这个问题仍然是网上市场的一个严重问题，卖家在网上兜售假冒商品，欺骗消费者。结果，包括耐克和伯肯斯托克在内的一些零售商发现，不值得冒险与亚马逊打交道。

虽然这家零售商长期以来一直被指责回避欺诈问题，但最近承诺花费数十亿美元解决这个问题，并在最近几年卷入了与欺诈卖家和造假者的法律斗争。

例如，在 2018 年与时装设计师 Vera Bradley 和手机配件制造商 Otterbox 合作提起了三起假冒诉讼。它还起诉了购买假评论的卖家和参与假评论行业的其他人。

去年，亚马逊宣布了一项名为 Project Zero 的计划，该计划引入了一系列工具，供品牌用来帮助亚马逊打击欺诈。这些品牌可以选择向亚马逊提供它们的标志、商标和其他关键数据，使零售商能够扫描其数十亿的产品清单，更主动地找到可疑的假冒产品。

另一个工具，电子监管，允许品牌在生产过程中在其产品上包含一个独特的代码，该代码可以在以后被扫描以验证购买是真实的。这个工具，现在被称为透明，去年夏天扩展到其他市场，包括欧洲、加拿大和印度。

但与这些早期的努力不同，卖家验证的目的是首先减少上市的产品——而不仅仅是在上市后删除或阻止欺诈性产品运送给客户。
