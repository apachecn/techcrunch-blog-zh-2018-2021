Twitter 为获得批准的开发人员和研究人员发布了一组新冠肺炎的推文

原文：https://web.archive.org/web/https://techcrunch.com/2020/04/29/twitter-launches-a-covid-19-dataset-of-tweets-for-approved-developers-and-researchers/

Twitter 通过更新其 API 平台，使开发者和研究人员能够实时研究新冠肺炎周围的公共对话。该公司正在向那些参加 Twitter 开发者实验室的人介绍 一种新的新冠肺炎流端点——该计划在新 API 端点和其他功能公开发布之前提供对它们的访问。Twitter 表示，新的新冠肺炎端点将允许获得批准的开发人员跨语言访问新冠肺炎和冠状病毒相关的推文，从而产生一个每天包括数千万条推文的数据集。

这些数据可用于研究与冠状病毒疫情相关的一系列主题，包括疾病的传播、错误信息的传播、社区内的危机管理等。

Twitter 指出，开发人员还可以使用新的数据集来建立机器学习和数据工具，以帮助科学界回答关于新冠肺炎的关键问题。

该公司本身将根据推文中使用的单词来确定哪些推文有资格被纳入这个数据集中——例如，“新冠肺炎”或“冠状病毒”。它还将提取使用常见冠状病毒标签的推文，这些标签往往是语言无关的。顺便说一下，这些关键字与 Twitter 现有的新冠肺炎主题使用的关键字相同，该主题由一个 Tweet 注释提供支持。

Twitter 还将过滤这些数据流，以排除垃圾和低质量的内容。

虽然对终端的访问将是免费的，Twitter 将手动选择哪些开发人员和研究人员将被授权使用它。开发人员还必须告知 Twitter 他们的项目计划，详细说明他们使用大数据的经验，并详细说明他们处理这样一个数据集的可用资源。

“鉴于处理这些数据所需的专业知识和计算资源，并认识到它的敏感性，我们创建了一个专用应用程序来访问这个端点，并计划仔细审查访问请求，以确保它们支持公共利益，”Twitter 在一份声明中指出。“我们还鼓励申请人详细描述他们打算实施的保护措施，以保护这些数据中所代表的人的隐私和安全，包括适用的机构审查和道德筛选，”它说。

Twitter 表示，它将优先处理来自拥有成熟专业知识和资源的研究人员和开发人员的申请。

应用程序和端点将于今天发布。没有开发人员或研究人员可以提前访问。

除了访问新端点的应用程序要求之外，开发人员还需要已经拥有一个批准的开发人员帐户，并遵守 Twitter 的开发人员协议和政策的条款，该协议和政策提供了关于与分析健康相关主题的项目相关的受限用例的指导。为了确保数据保持合规，获得批准的开发人员还将获得对新的合规流端点的访问权。

新的端点是自冠状病毒爆发以来 Twitter 所做的几项努力之一，专注于将人们与疫情的信息联系起来。通过其平台，它引入了变化，使新冠肺炎的事实和可靠的健康信息更容易获得。它还更新了其广告政策，与救援组织合作并匹配筹款捐赠，等等。

Twitter 首席执行官杰克·多西在 Twitter 今天分享的一份声明中说:“公共对话可以帮助世界更快地学习，更好地解决问题，并意识到我们都在一起。”该声明来自最近的一次采访。“面对毁灭性的全球疫情真的让这一点，以及 Twitter 的作用，暴露无遗，”他补充说。
