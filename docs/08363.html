AWS 本季度达到 100 亿美元，运行成本为 400 亿美元

亚马逊的云部门 AWS 本身将会是一项相当成功的业务。今天，公司宣布该季度的收入已经超过 100 亿美元，使云业务的收入超过了 400 亿美元。

这是该公司财报中的一个亮点，净利润为 25 亿美元，比去年同期下降了 10 亿美元。

尽管如此，大多数公司会认为这是整个业务，但 AWS，这个在 2006 年开始作为亚马逊的一个侧推手，已经独自成长为一个强大的业务。33%的增长率，它仍在快速增长，尽管随着大数定律开始对其不利，它的增长速度有所放缓。

尽管微软增长更快——在昨天的报告中，微软报告 Azure 以 59%的速度增长——AWS 有如此大的领先优势，并控制着很大一部分市场份额。

为了让你了解这项业务的增长有多快，彭博的 Jon Erlichman 在推特上发布了自 2014 年以来 AWS 的 Q1 数据，这是非常惊人的增长:

2014 年，这是一项年销售额 40 亿美元的业务。今天，这个数字是 9.1 倍，而且还在继续增长。对每个参与者来说，好消息是这是一个巨大的市场，虽然没有人能够将疫情及其经济影响描述为对任何人来说都是好消息，但事实是，它正迫使公司以超出他们预期的速度向云迁移。

这对所有云基础设施供应商来说都是一个好兆头，即使经济萎缩，这些供应商提供的服务应该比以往任何时候都更受欢迎，这意味着这些数字可能会继续增长一段时间。
