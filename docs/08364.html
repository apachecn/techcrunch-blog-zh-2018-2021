B2B 挑战者银行 Finom 从 Target Global 和 General Catalyst TechCrunch 筹集 700 万美元种子资金

原文：https://web.archive.org/web/https://techcrunch.com/2020/04/30/b2b-challenger-bank-finom-raises-7m-seed-from-target-global-and-general-catalyst/?guccounter=1

正如 B2C 领域出现了挑战者银行一样，针对小企业的 B2B 初创银行也出现了，其中包括 Qonto、Tide、Penta 和 CountingUp 等初创公司。

今天，另一家这样的公司， Finom 已经完成了一轮€650 万英镑(700 万美元)的种子基金，由 Target Global 牵头，General Catalyst 参与其中。其他投资者包括 FJ 实验室、Raisin 创始人 Tamaz Georgadze、Frank Freund 和 Michael Stephan 以及 MarketFinance 的创始人 Ilya Kondrashov。

该公司将主要使用新资本开发其银行产品，并在 2020 年夏天进一步扩展到意大利、法国和德国。

Finom 将面向初创企业和中小企业的会计、财务管理和银行功能整合到一个“移动优先”的产品中。企业可以建立一个在线账户，在相当短的时间内从应用程序和网站上获得应付账款和应收账款。该公司是由一个团队创建的，该团队还为俄罗斯的中小企业推出了名为“neobank”的 Modulbank。

Finom 的联合创始人康斯坦丁·斯蒂金斯(Konstantin Stiskin)告诉 Techcrunch:“欧盟中小企业银行市场规模超过€的 1000 亿欧元。但根据麦肯锡的研究，欧洲企业家将 74%的时间花在非核心活动上，并为昂贵且不方便的产品买单。我们的目标是让欧洲的小企业变得更有效率，并蓬勃发展。”

他补充道:“我们不仅仅是一张有账户的卡。我们的目标是成为中小企业及其日常业务的基础，在一个产品中涵盖银行、会计和财务管理。

Finom 现在在意大利上线，从电子发票开始，这使它能够获得市场知识，并为会计/支付和贷款收集数据。下一个将推出的国家是法国和德国。

Target Global 的普通合伙人兼首席运营官迈克·洛巴诺夫(Mike Lobanov)表示:“在 Target Global，我们是中小企业细分市场的忠实信徒 Finom 背后的优秀企业家团队与我们观点一致，并已经建立了为中小企业提供金融服务的新标准。”

尽管 Target Global 的总部位于柏林，但它在伦敦、特拉维夫和巴塞罗那设有办事处，管理着逾 8 亿€资产。Poortfolio 包括 Auto1、Delivery Hero、Omio(前 GoEuro)、TravelPerk、Rapyd 和 WeFox 等公司。
