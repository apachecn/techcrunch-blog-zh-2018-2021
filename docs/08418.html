macOS 的 Valve drops VR 支持

Valve 正在叫停 macOS 对其虚拟现实平台的支持。一名 Valve 员工在公司的 SteamVR 论坛上发布了一条更新，指出“SteamVR 已经结束了对 macOS 的支持，因此我们的团队可以专注于 Windows 和 Linux。”

苹果在 2017 年 6 月推出了“虚拟现实金属”，并强调了与 Valve 的合作关系。当时，Valve 正在推动 VR 作为一个平台，并与 HTC 合作其 Vive 系统；快进到 2020 年，Valve 拥有了自己的高端耳机，并刚刚发布了备受期待的标题“半条命:Alyx”。

这对开发者的影响比对真实玩家的影响更大。

几乎没有游戏支持 Mac，甚至苹果最高端的 MacBook Pros 也未能满足 Oculus 或 SteamVR 的最低规格要求。正如 上传 的人所指出的，Valve 最近的 硬件调查 显示，该平台上只有 4%的游戏玩家开始使用 macOS，这表明实际游戏玩家中相当小的一部分甚至符合拥有 Mac 的 VR 用户维恩图。

在 Mac 上开发虚拟现实内容的游戏开发者很可能享受在单台机器上开发和测试的能力。随着苹果凭借 Mac Pro 和 iMac Pro 等高价设备积极追逐专业人士，当一个主要软件平台决定你不值得努力时，这看起来并不太好。
