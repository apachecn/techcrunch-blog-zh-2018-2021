英特尔将收购1B智能城市交通初创公司Moovit，以提升其自动驾驶汽车部门TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/03/intel-to-buy-smart-urban-transit-startup-moovit-for-1b-to-boost-its-autonomous-car-division/?tpcc=ECTW2020

智能交通领域的某个大M&A正在以色列进行。根据多个 报道和联系TechCrunch的消息来源，芯片巨头英特尔收购 Moovit 的交易已进入最后阶段，Moovit是一家应用人工智能和大数据分析来跟踪交通并为全球约8亿人提供交通建议的初创公司。该交易预计将在未来几天内完成，价格据信在10亿美元左右。

我们已经联系了Moovit的创始人兼首席执行官Nir Erez以及英特尔发言人，请他们对这些报道发表评论，我们将在了解更多信息后更新这个故事。到目前为止，穆维特的发言人并没有否认这些报道以及我们被直接告知的内容。

“目前我们无可奉告，但如果有任何变化，我一定会让你知道，”穆维特的发言人说。

消息人士告诉TechCrunch，这家初创公司之前得到了英特尔投资的战略投资——将成为英特尔以色列汽车中心的一部分，该中心由自动驾驶公司Mobileye负责，该公司于2017年被英特尔以153亿美元收购。

目前还不清楚Moovit会在这个中心做什么，但通常来说，摄取和处理可靠的实时交通数据和智能路由——moo vit所做的事情的关键——是自动驾驶汽车服务启动和运行的一些最具挑战性的方面。

事实上，Moovit已经与Mobileye和英特尔合作:后者领导了Moovit的上一轮融资，即2018年5000万美元的D轮，作为其中的一部分，英特尔高级副总裁兼Mobileye首席执行官/首席技术官Amnon Shashua教授作为观察员加入了 Moovit 的董事会。

引进人才并将其整合到英特尔的大战略中似乎是这笔交易的重要组成部分。在这10亿美元中，员工将获得最终金额的10%作为保留方案的一部分，这一细节既由以色列希伯来语报纸 The Marker 报道，也由Wadi Ventures的分析师David Bedussa传递给我们。

在Moovit的上一轮融资时，这家初创公司的估值超过5亿美元，但在过去两年里它增长了很多。

它开发了一个受欢迎的独立应用程序，人们用它来找出在城市中导航的最佳方式，它还与优步等公司整合，努力提供多模式路线，使用不同的交通方式，从优步汽车和自行车到公共交通和步行。

2018年，Moovit表示，其 iOS 、 Android 和 Web 应用程序在全球80个国家的2000个城市中被1.2亿人使用。到了2020年，这个数字已经超过了8亿，遍布102个国家的3100个城市，使用45种语言。

除了英特尔，Moovit的其他投资者包括NGP资本、宝马、Sound Ventures、Gemini Israel、Sequoia Israel和LVMH。

此次收购(如果获得通过，还包括M&A的利益)发生在运输业的一个关键时刻。目前，世界各地的许多人被要求减少他们的活动，以减缓新冠肺炎病例在这个已经成为全球性疫情的地方的传播；部分由于同样的公共健康危机，全球经济一直在大幅下滑。两者都对汽车行业产生了直接影响，汽车行业的生产正在放缓，雄心勃勃的下一代战略也在发生一些变化。

与此同时，技术领域的人一直在努力以尽可能优化的方式利用他们的资产，以帮助保持事情的发展(可以这么说)。

因此，虽然随着人们走动的减少，消费者对Moovit应用的使用将大幅下降，但该公司已经推出了一系列新冠肺炎服务，以帮助那些仍然需要保持运营，仍然需要从A到b的人

其中包括为交通数据管理人员提供的一项特殊服务(与普通的B2B产品不同，它是免费提供的)，以接收更新的交通和交通数据，并随后迅速实施“数以千计的短期变化，使乘客能够只使用更新的有效路线来计划他们的行程

它还为Moovit应用用户启动了一项实时服务，以确保他们收到这些提醒。第三，它推出了“紧急动员”服务，让交通管理人员更快地重新调配路线上的公交车，以更好地服务仍在使用公共交通的关键员工。

尚不清楚Moovit是否一直在努力筹集更多资金，或者在完成交易时，它是否感受到了许多其他创业公司感受到的同样的压力，或者这只是一个好得让人无法拒绝的提议，或者甚至在新冠肺炎之前就已经摆在桌面上了。鉴于Moovit现有的规模和范围，这项业务在未来一段时间内似乎值得经营。
