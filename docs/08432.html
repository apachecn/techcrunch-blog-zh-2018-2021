Backblaze 通过使其云存储与 S3 兼容挑战 AWS TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/04/backblaze-challenges-aws-by-making-its-cloud-storage-s3-compatible/amp/

Backblaze 今天宣布其 B2 云存储服务现在与亚马逊的 S3 存储服务API 兼容。

Backblaze 最初是一项价格实惠的云备份服务，但在过去几年中，该公司还利用其存储专业知识，推出了以开发人员为中心的 B2 云存储服务，该服务有望比大型云供应商的类似产品便宜得多。B2 的定价为每 GB/月 0.005 美元起。AWS S3 的起价为每 GB/月 0.023 美元。

然而，存储价格本身并不会让开发者更换提供商。支持多个异构系统也涉及一些成本。

通过使 B2 与 S3 API 兼容，开发人员现在可以简单地将他们的存储重定向到 Backblaze，而不需要任何大量的重写。


Backblaze 的联合创始人兼首席执行官格莱布·布德曼(Gleb Budman)表示:“多年来，企业一直非常喜欢我们极其易于使用的云存储，因为它支持
他们实现令人难以置信的成果。“今天，我们很高兴能够让更多的企业将我们的存储与他们现有的工具和工作流程结合起来，从而做更多的事情。”

目前 B2 的客户包括美国公共电视台、巴塔哥尼亚和威瑞森的复杂网络(威瑞森是 TechCrunch 母公司威瑞森媒体集团的企业霸主)。Backblaze 表示，它在 B2 的服务总共有大约 10 万名客户。今天发布的发布合作伙伴包括 Cinafilm，IBM 的 Aspera 文件传输和流媒体服务，存储专家 Quantum 和云数据管理服务 Veeam 。

“公共云存储已经成为后期制作流程中不可或缺的一部分。Quantum 产品营销高级总监 Eric Bassier 表示:“这一最新增强功能使 Backblaze B2 云存储更加易于访问，无论是对我们供应商还是对客户来说都是如此。“我们现在可以使用新的 S3 兼容 API 将 BackBlaze B2 添加到 StorNext 兼容的公共云存储目标列表中，朝着实现混合和多云工作流又迈进了一步。”
