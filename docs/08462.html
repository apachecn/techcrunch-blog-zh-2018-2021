Firefox 获得了更好的密码管理器

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/05/firefox-gets-a-better-password-manager-and-picture-in-picture-support/

Mozilla 今天发布了其 Firefox 浏览器的版本 76，同时，它还推出了一些新功能，如果你已经在使用开源浏览器，你可能会注意到。

今天发布的亮点是增强的密码管理器。当你试图从浏览器的“登录和密码”页面复制并粘贴凭证时，Firefox Lockwise 会询问你的设备密码。确认设备密码后，您可以查看和复制您的凭据五分钟。这应该会让其他人更难访问你机器上受密码保护的网站，尤其是如果你的电脑经常与他人共享。

Lockwise 还新增了针对易受攻击的密码的警报，这些密码与在已知漏洞中被盗的密码相同(但您永远不会重用密码，对吗？)，以及当您使用的网站被攻破，您的登录名和密码可能被盗时的警告。

此外，Lockwise 的密码生成器现在可以在更多的网站上使用，它将帮助您找到 12 个随机的字母、数字和符号作为您的密码。

随着 76 版的推出，火狐现在也为 YouTube 等视频网站增加了一个改进的画中画模式。有了这个，你可以在继续其他任务的同时继续观看屏幕一角的视频(例如，当你在弹出式窗口中观看时，你不能离开 YouTube 进行浏览)。我希望我能对画中画窗口的大小有更多的控制，因为它太大了，但这只是目前的情况。版本 76 的新功能是双击弹出的视频使其全屏显示。一个小但受欢迎的新功能。

更新: 我们澄清了，PiP 模式本身并不新鲜。只有双击全屏才是。

如果你是一个狂热的 Zoom 用户，你会很高兴听到 Firefox 现在做了一些改变，允许你在 Firefox 中使用它，而不需要任何额外的下载，并且使用 GPU 更快地渲染网站的 WebRender 现在可以在更多的机器上使用。
