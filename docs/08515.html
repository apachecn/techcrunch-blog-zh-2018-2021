由于脸书 SDK 问题，几个主要的 iOS 应用程序在发布时崩溃 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/06/tiktok-tinder-pinterest-spotify-down/

由于脸书 SDK 问题，几个主要的 iOS 应用程序在发布时崩溃

周三晚上，包括 Spotify、GroupMe、Pinterest 和抖音在内的许多热门应用程序都提交了用户 iOS 崩溃报告，根据 GitHub 关于该主题的冗长帖子，这是脸书 SDK 明显问题的结果。

Downdetector 在美国太平洋时间下午 3:30 至 5:30 之间记录了数万份 Spotify 用户崩溃报告，在这段时间，大量报告似乎是在许多流行应用程序中发布的。截至发稿时，用户崩溃报告似乎正在减少。

用户报告称，他们的应用程序在启动后立即关闭。

对于许多用户来说，这可能是一个新闻，他们最喜欢的应用程序在发布时会立即连接到脸书服务器。除了公司的登录门户之外，科技公司的开发人员还依赖脸书的 SDK 获得广告洞察。早期的报告显示，受该问题影响的用户不必使用脸书登录来影响他们。

我们联系了脸书就此事发表评论，并将更新相关细节。
