想象一下医院、制药公司、大学和其他研究机构之间的合作研发模式，其中没有人共享任何实际数据。

这是纽约初创公司 Owkin 的梦想，它已经从投资者那里筹集了 2500 万美元的新融资，包括【BPI France】大型风险投资国泰创新 和MACSF(法国临床医生养老基金)，以及之前的投资者【GV】

该公司的宣传是，数据科学家、临床医生、学者和制药公司都可以登录 Owkin 称为 Owkin Studio 的虚拟实验室。

在这个虚拟环境中，所有各方都可以访问匿名化的数据集和模型，专门用于完善他们自己的研发和研究，以确保对新的生物标志物、作用机制和预测模型的最前沿见解为所有相关方正在开展的工作提供信息。

该公司表示，最终目标是改善患者的结果。

在寻求让更多公司和机构开放和共享信息的过程中——并承诺不会以数据所有者不允许的方式提取或使用信息——Owkin 正在复制其他公司在医疗保健、金融服务等领域所做的工作。

以色列公司 Qedit 已经为金融服务行业开发了类似的技术，最近一批 Techstars 公司的毕业生 Sympatic 正在为医疗保健行业开发类似的技术。

Owkin 通过为制药公司提供远程访问数据集的能力，并将大学开发的模型授权给这些公司来赚钱。这是该公司吸引研究人员加入该平台的一种方式，并为过去 40 年来资金不断减少的研究机构提供了另一种收入来源。

“我们有一个庞大的学术大学圈，可以访问数据，开发算法，我们共享数据，”该公司首席执行官托马斯·克洛泽尔博士说。“最终，它有助于开发更好的药物。”

Owkin 的新投资者和现有投资者的投资使该公司通过 A 轮融资的扩展筹集到了 5500 万美元的总资本。Clozel 说，这一轮总计 5200 万美元。

“我们正是在我们需要的地方，因为这关乎隐私，隐私比以往任何时候都更重要，”克罗泽尔说。

新冠肺炎疫情强调了不同企业和研究机构之间更紧密合作的必要性，这也增加了对该公司技术的需求。“它触及一切…我们可以访问正确的数据集和中心，为 COVID 建立最佳模型，”Clozel 说。“我们很幸运在 COVID 发生之前获得了正确的牵引力，并且我们已经完成了正确的研究。”

事实上，该公司已经推出了 【新冠肺炎开放人工智能联盟【COAI】，并正在利用其平台推进合作研究，加速对冠状病毒感染患者有效治疗的临床开发，该公司表示。它的所有发现将与全球医学和科学界分享。

该公司表示，该研究最初的重点是与全球 50 多个中心合作的国际注册机构合作研究新冠肺炎患者的心血管并发症 。据 Owkin 称，其他研究领域将包括患者结果和分类，以及免疫反应的预测和表征。

“自从我们在 2017 年首次支持 Owkin 以来，我们一直在分享它的愿景，将人工智能应用于抗击地球上最可怕的疾病之一:癌症，”Cathay Innovation 的合伙人 Jacky Abitbol 说。“Owkin 已经成为数字健康领域的领导者，我们很自豪能够增加对该公司的投资，以推动其在医学研究领域开创人工智能的雄心，同时保护患者隐私和数据安全。”
