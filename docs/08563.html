企业搜索一直是一块难啃的骨头。圣杯一直是像谷歌一样运作，但在内部。你输入几个关键词，你会在结果列表的顶部得到近乎完美的回复。试图在本地进行搜索的讽刺之处在于缺乏内容。

虽然谷歌拥有广阔的万维网世界，但企业的反应范围要窄得多。人们很容易认为这应该更容易找到理想的反应，但事实是，它是相反的。数据越多，找到正确文档的可能性就越大。

亚马逊正试图改变企业搜索游戏，将它放入一个更现代的机器学习驱动的环境中，使用今天的技术来帮助你找到完美的响应，就像你通常在网络上做的那样。

今天，该公司宣布亚马逊 Kendra 全面上市，这是该公司去年在 AWS re:Invent上宣布的云企业搜索产品。它使用自然语言处理，允许用户简单地问一个问题，然后在连接到搜索引擎的存储库中搜索，以找到精确的答案。

该公司在一份声明中描述了这项新服务，“亚马逊 Kendra 通过允许最终用户使用真实问题(不仅仅是关键词)在多个数据仓库中进行搜索，并利用机器学习模型来理解文档的内容和它们之间的关系，以提供他们所寻求的精确答案(而不是随机的链接列表)，从而彻底改造了企业搜索”。

AWS 针对包括 IT、医疗保健和保险在内的特定行业调整了搜索引擎。它承诺能源、工业、金融服务、法律、媒体和娱乐、旅游和酒店、人力资源、新闻、电信、采矿、食品和饮料以及汽车将在今年晚些时候到来。

这意味着这些行业中的任何一家公司在搜索方面都应该处于领先地位，因为系统会理解这些垂直行业特有的语言。你可以把你的 Kendra 搜索框放到一个应用程序或网站上，它有你在这样的工具中所期望的类似提前输入的功能。

企业搜索已经存在很长时间了，但也许通过引入人工智能和机器学习，我们最终可以一劳永逸地解决它。
