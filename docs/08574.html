腾讯音乐押注中国拥挤的音频内容领域

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/11/tencent-music-bets-on-chinas-crowded-podcasting-space/

根据市场营销公司 iiMedia 的第三方调查，今年中国的播客、有声读物和其他音频节目的听众估计有 5.42 亿。从 2019 年记录的 4.89 亿用户来看，这是一个健康的跳跃，毫无疑问，它吸引了新玩家加入游戏。

这包括腾讯音乐娱乐(TME)，腾讯的子公司，它有时被认为是中国的 Spotify，但实际上在许多方面都有所不同。该集团的主要业务超出了音乐流媒体，包括虚拟卡拉 ok，直播和音频内容；该公司最近大力推动了这一领域的发展。

在其最新发布的季度报告中，TME 表示，通过增加数以千计的热门 IP 作品和独立制作人作品的新改编，它已经在“扩大”其音频库方面取得了重大进展。知识产权是 TME 相对于竞争对手的一项优势，这得益于与中国领先的在线出版商中国文学的战略合作。TME 的进军加剧了这个已经拥挤的领域的竞争。

就像 Spotify 姗姗来迟地进入播客领域一样， TME 对基于语音的内容姗姗来迟，这是一个总括术语，可以包括从播客、有声读物和广播电台到音频直播等更具创新性的收听体验。巨人想征服所有这些领域吗？

中国播客公司 JustPod 的首席运营官(音译)认为，答案似乎是否定的。腾讯追求的可能是“改编自网络文学的有声读物和有声剧集，而不是纪录片、综艺节目或脱口秀等所谓的‘节目’。”这使得 TME 的新努力更符合听觉模型，而不是通常所说的“播客”

多年来，中国的在线音频领域一直被领先公司西玛拉雅(Ximalaya)占据，西玛拉雅是总部位于三藩市的播客公司喜玛拉雅(Himalaya)的主要投资者，在纳斯达克上市的励志(Lizhi)。TME 进军音频内容并没有立竿见影的效果，因为仍然没有明显的盈利途径。

众所周知，中国用户不愿意为数字内容付费，当他们为教育和自我提升播客付费时，热情往往会很快消退。财大气粗的平台经常求助于免费提供内容来获得市场份额，无情地排挤较小的竞争者。结果就是大家都需要找到更多间接的变现方式。

举例来说，的主要收入来源是通过现场互动音频会议销售虚拟物品，而来自用户订阅和广告的贡献仍然微不足道。这家成立 7 年的公司还没有盈利，去年净亏损 1.33 亿元人民币(1910 万美元)。

间接货币化在中国互联网行业并不是什么新鲜事。以微信最为出名的腾讯，主要依靠其社交网络产品推动的游戏收入。类似地，TME 的大部分收入来自以音乐为主题的直播流媒体的虚拟产品销售，而其音乐流媒体应用的 6.57 亿月活跃用户中只有 6%是付费用户。随着中国在线音乐市场饱和，MAU 增长也陷入停滞。从 2017 到 2020 ，TME 的音乐流媒体服务仅增加了 5000 万新用户。问题是这个音乐巨头能否给邻近的音频行业注入新的活力。
