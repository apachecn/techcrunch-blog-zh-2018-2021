VanMoof 筹集了 1350 万美元，以利用新冠肺炎 TechCrunch 之后的电动自行车热潮

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/12/vanmoof-raises-13-5m-to-capitalize-on-e-bike-boom-in-wake-of-covid-19/

荷兰电动自行车初创公司 VanMoof 于 2009 年成立，在从伦敦风投 Balderton Capital 和自行车组装合作伙伴台湾电子制造商信邦电子(SINBON Electronics)获得 1250 万€(合 1350 万美元)的投资后，现在正式成为一家“扩大规模”的公司。

在推出新的电动 VanMoof S3 和 X3 自行车之后，这些资金将用于国际扩张。

这一宣布来得正是时候。在最近新冠肺炎疫情病毒爆发期间，欧洲各地的城市正小心翼翼地从封锁中走出来，欧洲各国政府迫切希望让经济运转起来。但是大部分官方建议是尽可能避免公共交通，因为社会距离几乎是不可能的。

因此，随着骑自行车在许多城市成为一种可行的选择，但距离仍然是老对手，许多消费者将电动自行车视为通勤的解决方案，而传统自行车无法与电动自行车相比，电动自行车能够让骑行者比普通自行车走得更远。

为了提高使用率，英国政府推出了 20 亿英镑的一揽子计划来开创自行车和步行的新时代，导致自行车零售商 Halfords 的股票在周一上涨了 17%。

在美国，纽约市刚刚承诺在曼哈顿和布鲁克林增加受保护的自行车道。柏林正在扩建一些已经很宽的自行车道。米兰将引入一条 5 英里长的自行车道，以减少封锁后的汽车使用。纽约市报告称，与去年同期相比，骑自行车的人数增加了 50%。在新冠肺炎疫情爆发期间，费城骑自行车的人数增加了 150%以上。

VanMoof 的联合创始人 Taco Carlier 在一份声明中说:“这是一个建立如此有意义的伙伴关系的独特时刻。我们不仅感谢拥有丰富行业经验的投资者投下的信任票，这也是一个很好的迹象，表明投资正在变得更加环保，从化石燃料转向电动汽车。”

Balderton 的负责人科林·汉纳(Colin Hanna)表示:“VanMoof 在一个快速增长的市场中生产出一种领先的产品，这种产品正在让世界变得更好。我们相信 VanMoof 的质量、社区和经验将使他们成为从东京到柏林家喻户晓的名字。”

VanMoof 最近推出了 S3 和 X3 车型，这是以前车型的升级版，价格便宜得多。这一价格反映了这样一个事实，即由于与 SINBON 的合作伙伴关系以及直接面向消费者的销售模式，VanMoof 完全控制了供应链，这意味着它完全拥有从设计到生产，从销售到售后服务的一切。

此外，在过去两年中，该公司已将其 2018 年€1000 万英镑的收入翻了两番，到 2019 年接近€4000 万英镑，到目前为止，2020 年全年的销售额比目标高出 20%，这意味着它的年收入目标是€1 亿英镑。

S3 和 X3 的高性能新型电动面包车拥有一系列新功能，包括电子四速换挡、集成液压制动和两倍的涡轮增压动力。

但 VanMoof 并不是市场上唯一一家风投支持的电动自行车。总部位于布鲁塞尔的 Cowboy 是一家电动自行车初创公司，仅在 2017 年出现，但此后从 Tiger Global 和伦敦的 Index Ventures 筹集了 1950 万美元。

虽然这是一个悲剧，需要一个致命的疫情来启动这个市场，但我们的城市变得更好的前景，更清洁的空气和更健康的骑自行车的公民，似乎正在出现。
