如何为各种预算打造最佳的家庭视频会议设置 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/13/how-to-create-the-best-at-home-videoconferencing-setup-for-every-budget/

与几周前相比，现在你的生活可能需要更多的视频会议——即使已经需要很多了。这不太可能很快改变，所以为什么不充分利用它呢？一般的 MacBook 网络摄像头在技术上可以完成工作，但它远没有给人留下深刻印象。然而，有很多方法可以提升你的游戏水平——花费很少或很多。无论你只是想改善你的日常虚拟单口相声，为在虚拟会议上发表演讲做准备，还是计划一个新的视频播客，这里有一些建议，告诉你如何最大限度地利用你已经拥有的东西，或者如果你真的想最大限度地提高你的视频和音频质量，你应该得到什么。

0 级

打开一盏灯，把它放在正确的地方

要改善视频的外观，最简单的方法之一就是打开手边的任何一盏灯，把它放在摄像头后面，照在你的脸上。这可能意味着移动一盏灯，或者移动你的电脑，如果你所有可用的灯都在一个固定的位置，但它可以产生巨大的差异。看看下面这些例子，屏幕取自我的微软 Surface Book 2(实际上它有一个很好的内置摄像机，就内置摄像机而言)。

上图是除了房间天花板上的灯之外没有任何光线，下图是打开一盏灯，从 Surface Book 的上面和后面将它对准我的脸。这是一个足够的改变，让我看起来不像是开着视频被突然发现，而更像是在参加一个我应该参加的会议。

注意你身后的东西

为你参加的每一个视频通话设置着装，这绝对是一个过分的要求，但花一秒钟来检查一下框架中的可见内容是值得的。理想情况下，你可以找到一个背景相当简单的地方，可以看到一些有组织的装饰。关上门框内的门，尽量不要在无盖的窗户前拍摄。如果你生活在一个由疫情引起的混乱中，只要铲掉这些混乱，直到它脱离框架。

了解您的系统声音设置

了解您的设备和操作系统的输入音量设置。这通常不是什么大问题，因为大多数应用程序和系统都设置了非常合理的默认设置，但如果你也在做一些不寻常的事情，比如坐在远离笔记本电脑的地方，试图在框架中容纳第二个人，那么你可能需要打开输入音频滑块，以确保任何人都能听到你要说的话。

它可能可以在你正在使用的任何应用程序中直接控制，但在 MAC 上，也可以尝试进入“系统偏好设置”>“声音”>“输入”,检查音量对于你正在使用的设备是否可以直接控制，以及调整是否会产生你想要的结果。

一级

获取外部网络摄像头

大多数笔记本和一体机上的内置网络摄像头都不会很好，你几乎总是可以通过购买专用网络摄像头来改善这一点。现在，可能很难在库存中找到它们，因为许多人同时都有提高视频会议质量的相同需求。但是，如果你能得到哪怕是预算升级选项，比如我在下面的剪辑中使用的罗技 C922 Pro Stream 1080p 网络摄像头，它应该有助于清晰度、弱光性能、色彩等。

获得一个基本的 USB 麦克风

专用外部话筒是另一种以相对较低的成本快速提升系统性能的方式。在上面的剪辑中，我使用了流行的 Samson Meteor USB 麦克风，它有内置的腿和专用的音量/静音控制。这款话筒包括您需要的一切，当您通过 USB 插入时应该可以立即工作，它可以产生非常适合人声的出色声音。

拿些耳机来

任何类型的耳机都会让你的视频通话和会议更好，因为它最大限度地减少了你的麦克风从你自己的扬声器拾取音频的机会。大耳朵模型有利于音质，而耳塞在实际视频图像中不太明显。

第二级

使用专用摄像头和 HDMI 转 USB 接口

如果你已经有了一个独立的相机，包括几乎任何具有 HDMI 输出功能的消费类袖珍相机，那么就值得考虑选择一个 HDMI-USB 视频捕捉接口，以便将其转换为更高质量的网络摄像头。在下面的剪辑中，我使用的是索尼 RX100 VII ，这绝对是消费者口袋相机市场的高端产品，但也有一系列选项应该可以提供几乎相同的质量水平，包括索尼的旧 RX100 型号。

在寻找 HDMI 接口时，确保他们宣传它可以在 Mac 和 Windows 上与 Zoom、Hangouts 和 Skype 等视频会议应用程序一起工作，而不需要任何软件:这意味着他们可能具有 UVC 功能，这意味着这些操作系统会将它们识别为网络摄像头，无需任何驱动程序下载或开箱即用的特殊应用程序。这些也是在较高的需求，由于新冠肺炎，所以我在这里使用的埃尔加托凸轮链接 4K 可能没有现货。取而代之的是，寻找像 IOGear 视频采集适配器或 Magewell USB 3.0 采集设备这样的替代品，或者考虑升级到专用直播平台，比如我将在下面详细讨论的 Blackmagic ATEM Mini。

买一个有线 lav 麦克风

一个简单的有线领夹式(lav)麦克风是升级音频游戏的好方法，甚至不需要花费那么多。你可以在亚马逊上花 20 美元买到性能不错的有线 lav，即使你没有 3.5 毫米的输入端口，也可以使用 USB 版本直接连接到你的电脑。 Rode 的领夹式 GO 是一款出色的中端产品，也能与我在下一节提到的无线 GO 发射器和接收器套件配合使用。这样做的主要限制是，根据绳索的长度，在使用绳索时，你的活动范围会受到很大的限制。

获得多盏灯，并有效地定位它们

照明是一个兔子洞，最终会变得非常深，但获得几盏可以移动到最需要它们的地方的灯是一个好的、便宜的开始方式。亚马逊提供了各种各样符合要求的照明套件，或者你甚至可以在鹅颈灯中安装几盏飞利浦色调灯，正确放置并调节到合适的温度和亮度。

第三级

使用可互换镜头相机和快速镜头

体面的紧凑型相机的下一步是具有可互换镜头的相机。这允许你添加一个漂亮的，快速的最大光圈镜头(也就是一个低光圈数)来获得散焦的背景效果。这提供了自然的分离你，主体，从你身后的任何东西，并提供了一个电影的感觉，将哇同事在你的月度所有手。

买个无线 lav 麦克风

lav 话筒很棒，但无线 lav 话筒更好。这意味着您不必担心碰到电缆末端，或者让它与工作场所的其他电缆纠缠在一起，并且它还可以在使用什么音频接口将声音实际传入计算机方面提供更多灵活性。这里的一个很好的选择是 RODE Wireless GO ，它可以单独工作，也可以与 RODE 领夹式 GO 等话筒配合工作，以获得出色、灵活的声音。

使用入耳式监听系统

在这个阶段，你仍然想使用耳机，但最好的是入耳式监听系统，它能尽最大努力消失在视线之外。你可以买一些舒尔制造的专用广播式监听系统，或者买一副真正的好耳机，低延迟，最新版本的蓝牙。苹果的 AirPods Pro 是一个很好的选择，Bang & Olfusen E8 全无线耳塞也是，我已经广泛使用过，没有任何明显的延迟。

使用三点照明

在这个阶段，是时候认真对待照明了。基本上，就优化专门针对流媒体、视频会议和你在办公桌上做的任何其他事情而言，最好的平衡是选择至少两个埃尔加托的关键灯或关键灯。

这些是内置扩散器的 LED 面板灯，没有陡峭的学习曲线，配有非常坚固的带桌夹的铰接管座，可以通过智能手机或桌面应用程序连接到 Wi-Fi 进行控制。你可以调节它们的温度，这意味着你可以根据自己的需要让它们变得更“蓝”或更“橙”，还可以调整它们的亮度。

使用其中的三个，你可以设置一个标准的三点照明设置，非常适合采访或人们直接对着摄像机讲话——也就是你能想到的几乎所有虚拟会议/会议/活动/网络研讨会的使用。

四级

获得一个 HDMI 广播切换器甲板

HDMI-USB 捕捉设备在将大多数相机转变为网络摄像头方面做得很好，但如果你真的想给自己一系列选择，你可以升级到广播转换接口，如 Blackmagic ATEM 迷你。去年发布的 ATEM Mini 包含了许多以前基本上只有视频专业人士才能使用的功能，并以易于使用的形式提供了这些功能，鉴于这款产品的实际功能，价格实际上令人震惊。

单独搭配一个好的摄像头，ATEM 迷你可以为你的视频功能增加很多，包括允许你播放静态图形，切换到计算机输入来显示视频，在图形应用程序中实时工作，演示代码或进行演示。您可以设置画中画视图，放置较低的三分之一，甚至使用专用的硬件按钮渐变到黑色。

但如果你真的想充分利用 ATEM 迷你，你可以添加第二个甚至第三和第四个摄像头。对于大多数用途来说，这可能是太多的摄像头了——最终，一个人说话只能有这么多角度。但是，如果你在相机位置和主题上有创意，这是一种有趣的打断信息流的方式，特别是如果你在做一些更长的事情，比如演讲或扩展演示。较新的 ATEM 迷你专业版刚刚开始发货，并提供内置录音和流媒体服务。

使用广播质量的猎枪麦克风

ATEM 迷你有两个专用的音频输入，在这方面也给你很大的灵活性。例如，将一个附加到 iPod touch 的输出上，可以让你将该设备用作方便的共鸣板，用于提示介绍和标题音乐，以及声音效果。这也意味着，只要你有合适的接口，你就可以从高品质的话筒中发出声音。

对于顶级的流媒体质量，在视频方面需要最小的牺牲，我建议去一个好的，广播质量的猎枪麦克风。 Rode VideoMic NTG 是一个很好的入门级选择，在安装到摄像机上时具有灵活性，但像 Rode NTG3m 这样安装在吊杆臂上并放置在框架外，麦克风末端向下朝向你的嘴，将提供最好的效果。

添加重点照明

你已经有了 3 点照明——但是正如我所说，照明是一个几乎没有尽头的兔子洞。重点照明真的可以帮助你进一步提升视频的专业性，而且使用现成的设备也很容易设置。飞利浦色调可能是我最喜欢的给任何场景增加一点活力的方式，如果你已经是一个色调用户，你可以用他们的任何彩色灯泡。飞利浦最近发布的产品，如 Hue Play 智能 LED 灯条基本上是为此用途量身定制的，你可以在一个电源适配器上菊花链连接多达三个灯条，以创建令人敬畏的重点墙照明效果。

当然，所有这些对于基本的视频会议、虚拟聚会和会议来说都是不必要的。但是，如果您认为远程视频将成为我们未来生活中更大的一部分，即使我们在新冠肺炎事件后恢复了某种常态，那么也值得考虑根据您的预算和需求升级您系统的哪些组件，希望本文能提供一些指导。
