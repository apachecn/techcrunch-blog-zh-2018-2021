阿梅里奥希望从给囚犯免费信件开始，接下营利性的监狱电话诈骗

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/14/ameelio-wants-to-take-on-for-profit-prison-calling-rackets-after-starting-with-free-letters-to-inmates/

在监狱系统的诸多问题中，包括视频通话的巨额费用，这是少数公司以非常高的价格提供的。Ameelio 希望介入，为狱友提供免费的交流选择；它的第一个产品，发送纸质信件，正受到那些有被监禁亲人的人的热烈欢迎。

联合创始人兼首席执行官 Uzoma Orchingwa 表示，Ameelio 诞生于耶鲁法学院学生的思想，是他们在推动长期改革的同时，试图在短期内做出改变。

“我正在研究大规模监禁，我所写的政策解决方案需要很长时间才能实现，”Orchingwa 说。“在我们取得哪怕一点点进展之前，这将是一场漫长的战斗。所以我在想，在我致力于监狱改革的长期项目时，我能做些什么？”

他看到报告称，与亲人定期沟通的囚犯在获释后会有更好的结果，但在许多监狱，这种沟通越来越昂贵，而且受到限制。一些监狱已经完全禁止面对面的会面——这在疫情期间并不奇怪——这使得视频通话成为与亲人面对面交谈的唯一选择。

有时每分钟花费一美元，这些费用很快增加，自然，这对已经脆弱的人群影响最大。前 FCC 专员 Mignon Clyburn，对她来说，这是她任期内特别感兴趣的问题，称监狱通信系统“我作为监管者见过的最明显、最明显的市场失灵。”

值得注意的是，这些私人的、昂贵的呼叫服务并不总是规范的，而是随着私人监狱行业的扩张和从囚犯身上赚钱的方式的增加而诞生的。一些州禁止这种做法，但其他州已经与提供这些服务的公司建立了关系——当然，还有给州政府和监狱的健康回扣。

这个价值数十亿美元的行业由两家公司主导:Securus 和 Global Tel Link。与我们在外面认为理所当然的服务相比，他们提供的服务相当简陋。视频和音频通话被安排、录音、搜索关键词，并保留几个月，以备当局需要。

在视频通话免费提供给世界各地数十亿人的时候，他们也被暂时限制见面，收费似乎是错误的——每分钟收费 1 美元似乎是荒谬的。

Ameelio 的行善法律学生和开发者团队认为他们无法在一夜之间改变私人监狱系统，所以他们开始使用一种不同的产品，但这种产品也给试图与囚犯沟通的家庭带来了困难:信件。

书面邮件是与监狱中的人保持联系的一种常见方式，但有一些障碍可能会阻止不太了解的人这样做。Ameelio 为此提供了一个最新的正确地址列表和写信给全国数千个刑事司法机构中任何一个的惯例，以及查找和识别您试图联系的囚犯的正确方法——很少像在顶部放上他们的名字这样简单。

“监狱地址的工作方式是，囚犯地址不同于实际地址。所以我们收集了地址，并为此建立了一个数据库，并建立了一种方法来找到不同的特质，比如需要多少行，每行要放什么，等等。联合创始人 Gabe Saruhashi 说道。

一旦完成分类，你就可以写你的信，如果你想要的话附上照片，然后打印并发送出去(通过直邮即服务创业公司 Lob)。显而易见，消除印刷、地址等方面的摩擦和成本会导致更频繁的交流。

自从几个月前开始在脸书团体和其他非正式渠道传播这项服务以来，他们已经发出了 4000 多封信。虽然人们能够发送信件是件好事，但 Ameelio 计划迎合更大规模使用邮件的大型组织。

“家庭面临的沟通挑战与刑事司法组织和律师在与客户沟通时面临的挑战是一样的，”Orchingwa 解释说。他们必须管理地址、信件的撰写和发送，以及检查收件人和其他后续行动的人员网络。“我们正在与他们交谈，许多人对我们提供的服务非常感兴趣，所以我们将推出一个面向组织的版本。我们正在创造一种商业模式，在这种模式下，这些组织，其中一些资金充足，可以回报我们，也可以向前支付，并帮助其他人保持免费。”

一个组织如何使用和跟踪写信活动

不过，寄信只是阿梅里奥的开场表演，但这也是他们建立所需联系和研究市场的一种方式。对私人呼叫系统的抗议一直不断，但在州政策下运行的监狱的异质性意味着“我们没有一个系统，我们有 51 个独立的系统，”正如 Orchingwa 所说。还有它能赚很多钱的事实。

“围绕推出 Securus 和 Global Tel 有很多动作，”他说。“但这将从家庭转移到国家支付，所以他们需要赚回他们从回扣中赚的钱。”

一些州已经禁止或者从来不允许付费电话，但是其他州只是在外部压力下才改变他们的政策。正是凭借这些，阿梅里奥希望先成功。

“我们可以从与这些公司没有密切关系的州开始，”Orchingwa 说。“选民会问州和县官员，‘既然有免费的选择，我们为什么还要用他们？’"

你可能会想，一个刚刚起步的年轻公司是否有可能在如此短的时间内建立一个可供部署的视频通话平台。该团队很快解释说，该产品的实际视频通话部分，就像发送信件一样，可以通过第三方来完成。

“目前的障碍根本不在于视频基础设施——企业和 API 将提供这一点。我们已经有了一个 MVP 的样子，”Saruhashi 说。甚至硬件也相当标准——只是贴在墙上的普通安卓平板电脑。

“困难的部分是[惩教部]的仪表板，”Saruhashi 继续说道。“他们需要一种方法来管理传入的连接、安排对话、获取日志并在完成后查看它们。”

但是他们也很好地开发了那部分，这最终也只是一个中等程度的工程挑战，已经在许多其他情况下解决了。

目前，该团队正在评估一些加速器的参与情况，并且已经是 Mozilla 的 Spring MVP 实验室的一部分，这是今天早些时候宣布的更大规模孵化器项目的前身。“我们喜欢它们，”Mozilla 的 Bart Decrem 说。

目前该公司肯定处于早期阶段，计划多于成就，他们很清楚这只是一个开始——正如建立更好的通信选项只是监狱和司法系统更全面改革的开始一样。
