DigitalOcean从Access Industries和a16z再融资5000万美元

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/14/digitalocean-raises-50m-more-from-access-industries-and-a16z/

今天早些时候， DigitalOcean 宣布从之前的投资者 Access Industries 和 Andreessen Horowitz 那里又筹集了5000万美元。此前，专注于中小企业和开发者的云基础设施公司在二月份筹集了价值九位数的债务。

DigitalOcean是一家大型私营公司，截至2019年底，其营收约为2.5亿美元。该公司今天宣布，它已经达到了3亿美元的年度经常性收入，或ARR。(我们最近将该公司加入我们的ARR俱乐部。)这意味着在不到半年的时间里，增长率约为20%，尽管我们不知道该公司何时达到2.5亿美元大关，因此很难计算其真实的增长速度。

重要的是，DigitalOcean在扩张的同时也在走向盈利。

DigitalOcean的首席执行官Yancey Spruill今年早些时候告诉TechCrunch，他的公司将在未来几年内实现自由现金流正数，这一时间表似乎已经提前(不久将有更多相关内容)。如果这家云公司能够在同一时期保持增长速度，它就有望上市。

新的5000万美元对该公司的估值为11.5亿美元，这意味着它的前期投资价值为11亿美元。从收入倍数的角度来看，DigitalOcean目前的估值不像一家SaaS初创公司，尽管它的新估值仍然是旧B轮估值的近两倍(该公司发言人证实了该页面上的数字)。

新资本

TechCrunch想知道该公司在增加负债后为何如此迅速地筹集股本。考虑到世界的经济状况，首都无疑是受欢迎的，但时机值得探究。

据Spruill称，DigitalOcean并没有“寻求额外的资金”，但“在与Access和a16z的投资者一起评估了我们的业务表现和前景后，他们对我们下一阶段的增长有兴趣。”斯普瑞尔说，公司接受了。

据推测，数字海洋的收入从2.5亿美元快速增长到3亿美元，这也是投资决定的一部分。对数字海洋而言，从知名投资者那里获得新的、更高的估值和资金支持甚至可能会提升品牌(参见本文，尤其是考虑到该公司最近吸引的报道)。

关于新首都的计划，Spruill告诉TechCrunch，DigitalOcean现在可以“更好地支持世界各地企业家和中小企业的需求增长，因为越来越多的企业正在向云过渡，尤其是因为COVID。”标志着数字海洋从疫情崛起成为世界上最大的公司之一；大多数公司没有，但是这些公司似乎正在利用这个机会将更多的资本投入到他们的资产负债表中。

TechCrunch还想知道新资本是否为该公司开辟了新的领域，或者它对新资本的优先考虑是否与其之前的目标相似。首席执行官告诉TechCrunch，他的公司的重点是一样的，即扩大业务。

“我们仍然致力于实现10亿美元的收入，在今年下半年实现自由现金流盈利，并最终将数字海洋定位为一家上市公司，”Spruill在一封电子邮件中表示。

这已经很清楚了。

根据这一指标，如果市场复苏，我们预计2021年上半年将出现DigitalOcean S-1。因此，如果目前的计划持续下去，a16z和Access Industries(T4，该公司的长期投资者)可能会看到最近一次检查的快速回报。

该公司的发布记录了“加速增长”，TechCrunch想了解更多。该公司的增长速度有多快？斯普鲁维尔没有分享数字来证实或否认我们根据他的公司的公共收入里程碑所做的粗略计算，但他告诉TechCrunch，该公司“正在积极制定一些举措来加快我们的收入增长率”，并补充道，这些举措在内部被称为“增长更快”。

最后，TechCrunch很好奇新冠肺炎对数字海洋的影响。该公司告诉我们，“由于新冠肺炎，客户流失略有增加”，但情况并不太糟，称“与疫情之前的近期趋势相比”，这一变化“并不显著”

从积极的一面来看，DigitalOcean表示，其“新客户的签约一直在加速”，并看到“一些现有客户的业务增加了。”给SaaS的孩子们加起来:多一点客户流失，好的新标识添加，和一些向上销售的顺风。总的来说，这意味着增长。

当我们有它的时候，更多，但是现在我们至少开始了解公司下一步做什么。
