DigitalOcean 从 Access Industries 和 a16z 再融资 5000 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/14/digitalocean-raises-50m-more-from-access-industries-and-a16z/amp/

图片来源:马特·安德森摄影/盖蒂图片社

今天早些时候，数字海洋 宣布从之前的投资者 Access Industries 和 Andreessen Horowitz 那里又筹集了 5000 万美元。这笔资本来自于专注于中小企业和开发者的云基础设施公司在二月份筹集了价值九位数的债务。

DigitalOcean 是一家大型私营公司，截至 2019 年底，其营收约为2.5 亿美元。该公司今天宣布，它已经达到了 3 亿美元的年度经常性收入，或 ARR。(我们最近在这里把公司加入了我们的 ARR 俱乐部。)这意味着在不到半年的时间里，增长率约为 20%，尽管我们不知道该公司何时达到 2.5 亿美元大关，因此很难计算其真实的增长速度。

重要的是，DigitalOcean 在扩张的同时也在走向盈利。

DigitalOcean 的首席执行官 Yancey Spruill 今年早些时候告诉 TechCrunch】，他的公司将在未来几年内实现自由现金流正数，这一时间表似乎已经提前(稍后将详细介绍)。如果这家云公司能够在同一时期保持增长速度，它就有望上市。

新的 5000 万美元对该公司的估值为 11.5 亿美元，这意味着它的前期投资价值为 11 亿美元。从收入倍数的角度来看，DigitalOcean 目前的估值不像一家 SaaS 初创公司，尽管它的新估值仍然是旧 B 轮估值的近两倍(公司发言人证实了该页面上的数字)。

新资本

TechCrunch 想知道为什么该公司在增加债务后如此迅速地筹集股本。鉴于世界经济状况，首都肯定是受欢迎的，但时机值得深究。

Spruill 表示，DigitalOcean 并没有“寻求额外的资金”，但在“与 Access 和 a16z 的投资者一起评估了我们的业务表现和前景后，他们有兴趣投资我们的下一阶段增长。”斯普雷尔说，公司接受了。

据推测，数字海洋的收入从 2.5 亿美元快速增长到 3 亿美元，这在投资决策中发挥了一定作用。对于 DigitalOcean 来说，从知名投资者那里获得新的、更高的估值和资金补充甚至可能会提振品牌(尤其是考虑到该公司最近吸引的报道)。

关于新首都的计划，Spruill 告诉 TechCrunch，DigitalOcean 现在可以“更好地支持我们从世界各地的企业家和中小企业那里看到的需求增长，因为越来越多的企业正在向云过渡，特别是 COVID 的结果。”把 DigitalOcean down 标为世界上从疫情看到上升的公司之一；大多数公司没有，但是这些公司似乎在利用这个时机将更多的资本放到他们的资产负债表上。

TechCrunch 还想知道新资本是否为公司开辟了新天地，或者新资本的优先事项是否与之前的目标相似。这位首席执行官告诉 TechCrunch，他的公司的重点是相同的，即扩大业务。

“我们仍然致力于实现 10 亿美元的收入，在今年下半年实现自由现金流盈利，并最终将 DigitalOcean 定位为一家上市公司，”Spruill 在一封电子邮件中说。

这已经够清楚了。

根据这一标准，如果市场复苏，我们可以预计在 2021 年上半年看到 DigitalOcean S-1。因此 a16z 和 Access Industries ( 该公司的长期投资者)可以看到，如果目前的计划继续下去，他们最近的检查将很快得到回报。

该公司的新闻稿提到了“加速增长”，TechCrunch 希望更多地了解这一点。公司发展有多快？斯普雷尔没有分享数字来证实或否认我们根据他的公司的公共收入里程碑进行的粗略计算，但他告诉 TechCrunch，该公司“正在积极制定一些措施来加快我们的收入增长率”，并补充说这些措施在内部被称为“增长更快”的措施。

最后，TechCrunch 很好奇新冠肺炎对数字海洋的影响。该公司告诉我们，“由于新冠肺炎，客户流失略有增加”，但情况并不太糟，称“与疫情之前的近期趋势相比”，这一变化“并不显著”

从积极的一面来看，DigitalOcean 表示，其“新客户的签约一直在加速”，并看到“一些现有客户的业务增加了。”给 SaaS 的孩子们加起来:多一点客户流失，好的新标识添加，和一些向上销售的顺风。总的来说，这意味着增长。

当我们有它的时候，更多，但是现在我们至少开始了解公司下一步做什么。
