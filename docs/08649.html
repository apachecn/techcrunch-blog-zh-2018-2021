Scener 现在可以让你在一个多达 20 人的“虚拟影院”中共同观看 HBO 或网飞

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/14/scener-now-lets-you-co-watch-hbo-or-netflix-in-a-virtual-theater-with-up-to-20-people/amp/

允许用户共同观看网飞的浏览器扩展——类似于最近流行的网飞派对 Chrome 插件——正在扩展以支持 HBO NOW 和 HBO GO。除此之外，Scener 也全面重新推出，该公司正在为冠状病毒时代重新设计其产品，作为一种虚拟电影院体验，最多 20 名观众可以在专用的侧栏中通过视频、音频或文本聊天一起观看。

该公司表示，在不久的将来，它还将推出“一对多”版本的虚拟影院体验，允许人们托管更多同步观众，类似于 Instagram Live，但同步到底层的订阅视频内容服务。

一些共同观看的体验是非法的——例如，的 Twitch 被起诉，因为它允许其创作者向观众直播体育比赛。Skype 和 FaceTime 等视频聊天服务多年来一直被用来解决无法从海外观看美国内容的问题。但 Scener 与内容合作伙伴合作，以确保它有适当的交易来合法提供其产品。

HBO 向 TechCrunch 证实，WarnerMedia 接触 Scener 是为了让 HBO NOW 和 HBO GO 的订户通过其网络扩展以保护其知识产权的方式观看内容。除此之外，HBO 不愿对交易细节发表评论。

Scener 也无法确认其对 HBO 内容流媒体的许可是否会延伸到 WarnerMedia 的新 HBO MAX 服务，该服务将于 2020 年 5 月 27 日推出。

然而，Scener 的技术应该能够支持任何流媒体服务，只要该服务提供了一种通过 Chrome 浏览器播放视频内容的方式。要将 Scener 用作私人虚拟影院，浏览器扩展只需要求用户授予它访问相关站点的权限。

建立虚拟影院体验本身很容易。除了扩展之外，没有复杂的软件需要安装。然后，朋友可以通过剧院代码或点击您发送的邀请链接加入。

Scener 最初是 RealNetworks 的一部分，在 2018 年首次开发了 Chrome 扩展，允许用户录制评论视频轨道，可以与网飞、Hulu 和 YouTube 上的内容同步播放。这导致了类似于“神秘科学剧场 3000”的体验，如果你熟悉那个节目的话。

从 RealNetworks 分拆出来后，Scener 在 2019 年开发了一款产品，将网飞内容的共同观看与视频聊天同步。它还与其他内容提供商合作开发手机原型。但新冠肺炎疫情公司重新将 Scener 的努力集中在 Chrome 扩展上，因为随着美国被隔离，该产品在桌面上的使用量飙升了 15 倍。

Scener 的联合创始人和首席运营官·乔·布拉德伍德说，今天发布的新版本是对该产品的完全重建。他承诺，这也仅仅是 Scener 的开始。

“在不久的将来，支持更多(订阅视频点播)服务是我们企业愿景的核心，我们也在与一些(广告支持的视频点播)提供商进行谈判，”他告诉 TechCrunch。

该公司没有宣布可能正在进行的其他交易，但它的扩展正在请求批准读取和更改 Hulu.com 和 DisneyPlus.com 的数据，以及 HBO 网站的数据。Scener 对此不予置评，只是说它目前专注于为 HBO NOW 和 HBO GO 节目提供最佳的共同观看体验。在没有进一步证实的情况下，很难说迪士尼/Hulu 的交易(迪士尼是 Hulu 的大股东)是希望 Scener 的一部分还是一项正在进行的工作。时间会证明一切。

自从 RealNetworks 分拆出来以来，Scener 已经筹集了 160 万美元的资金。它通过与流媒体服务的合作创造收入，围绕其内容创造引人入胜的社交体验，正如今天与 HBO 的交易一样。WarnerMedia 证实，它在 Scener 没有财务股权。
