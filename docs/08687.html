时髦的以太坊钱包应用程序 Argent 秘密出现

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/18/buzzy-ethereum-wallet-app-argent-comes-out-of-stealth/

Argent 将推出其首个面向 iOS 和 Android 的以太坊钱包公共版本。该公司已经推出了几个月的有限测试版，拥有几千名用户。但它已经筹集了种子资金和首轮融资，投资者包括 Paradigm、Index Ventures、Creandum 和 Firstminute Capital 等知名投资者。该公司总共筹集了 1600 万美元。

几个月前，我设法获得了测试版的邀请，并一直在玩它。这是一个精心设计的以太坊钱包，具有一些创新的安全功能。它还可以很好地与 DeFi 项目集成。

许多人将他们的加密资产放在加密货币交易所，如比特币基地或币安。但这是一种集中的模式——你不拥有密钥，这意味着交易所可能会遭到黑客攻击，你会失去所有的加密资产。同样，如果 exchange API 或登录系统中存在漏洞，有人可能会将您所有的加密资产转移到他们自己的钱包中。

在本质上，Argent 是一个非托管的以太坊钱包，就像比特币基地钱包(一个不同于比特币基地交易所的非托管钱包 app)或信托钱包。你控制着钥匙。例如，没有你的授权，Argent 不能启动交易。

但是这种程度的控制带来了很多复杂性。硬件钱包，如分类账钱包，要求你写下一个种子短语，以便你在丢失设备时可以找回钱包。这需要一些训练，如果你不熟悉种子短语的概念，就很难理解。

甚至比特币基地钱包也告诉你在第一次创建钱包时备份你的种子短语。“我们将它们视为开发人员的高级工具，”Argent 联合创始人兼首席执行官 Itamar Lesuisse 告诉我。

这就是为什么新一代钱包试图向最终用户隐藏复杂性，如 ZenGo 和 Argent。在 Argent 上创建钱包是加密货币领域最好的体验之一。你的钱包被一种叫做“守护者”的东西保护着。

信任你的朋友

监护人可以是你认识并信任的人、硬件钱包(或另一部手机)或 MetaMask 账户。Argent 还提供监护人服务，要求你用短信和邮件确认身份。如果你丢失了你的手机，你想在另一部手机上找回你的钱包，你需要和你的监护人谈谈，并得到大多数确认。如果他们都可以确认，是的，确实，你的手机不再工作，你想恢复你的加密资产，恢复过程开始。

我们举个例子。这是你的监护人名单:

  • Argent 自己的监护人服务
  • 两个也在用 Argent 的朋友
  • 一款莱杰 Nano S 硬件钱包

总的来说，涉及到五个不同的因素，你包括。如果您丢失了手机，您可以通过在另一部手机上下载 Argent 来找回您的钱包(因素#1)，要求 Argent 的监护人服务向您发送文本和电子邮件以确认您的身份(因素#2)，并通过 Ledger Nano S 确认您的身份(因素#3)。

您已达到多数，恢复过程开始。你会在 36 小时内拿到你的资金，这样你就有足够的时间取消它这是一次劫机企图。

但是你也可以在另一部手机上下载 Argent 应用程序(因素 1)并直接 pinged 你的两个朋友(因素 2 和 3)。如果他们能确认相同的字符序列(在这种情况下是表情符号)，恢复过程也将开始。

2018 年 7 月，以太坊创始人 Vitalik Buterin 在 TechCrunch 采访中说“我对社会恢复、多密钥方案感兴趣。这不是一个新概念，因为社交媒体应用已经使用社交恢复系统。在微信上，如果你丢失了密码，微信会要求你在一个大名单中选择联系人列表中的人。

在 Argent 的案例中，社会恢复也增加了病毒传播的因素。随着你周围越来越多的人开始使用 Argent，这种体验会变得更好。

除了钱包恢复之外，Argent 使用 guardians 来设置一些限制。就像你对你的银行账户有一些限制一样，你可以设置一个每日交易限额来防止攻击者攫取你所有的加密资产。你可以要求你的监护人放弃超过每日限额的交易。

同样，你可以要求你的监护人锁定你的账户 5 天，以防你的手机被盗。

赌以太坊

Argent 专注于以太坊区块链，并计划支持以太坊提供的一切。当然可以收发 ETH。这家初创公司希望隐藏这方面的复杂性，同时为你支付交易费用(汽油)，并为你提供用户名。这样，你就不必设置交易费用来确保交易成功。

这家初创公司计划将 DeFi 项目直接集成到应用程序中。DeFi 代表分散金融。顾名思义，DeFi 的目标是在分散的区块链和金融服务之间架起一座桥梁。它看起来像传统的金融服务，但一切都编码在智能合同中。

有几十个 DeFi 项目。其中一些可以让你借出和借入资金——例如，你可以通过锁定贷款池中的一些加密资产来赚取利息。其中一些允许您以分散的方式直接与其他用户交换加密资产。

Argent 允许您一起访问 TokenSets、Compound、Maker DSR、Aave、Uniswap V2 流动性、Kyber 和 Pool。该公司已经计划很快推出更多的 DeFi 功能。

总的来说，Argent 是一款完美的应用，能够在安全性和简单性之间找到恰当的平衡。许多加密货币初创公司希望打造“加密革命”。感觉 Argent 有一个很好的开端，很有希望做到这一点。
