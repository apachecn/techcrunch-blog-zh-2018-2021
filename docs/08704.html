Algolia 创始人卸任，新任首席执行官上任

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/19/algolia-gets-a-new-ceo-as-founder-steps-down/

搜索即服务初创公司 Algolia 宣布了该初创公司的一些人事变动。联合创始人兼首席执行官 Nicolas Dessaigne 将在公司担任非运营职务。他仍将是董事会成员，但 Bernadette Nixon 将加入公司，担任首席执行官一职。

Algolia 正在构建一个搜索引擎 API。该公司不想打造下一个谷歌。相反，它希望为你网站或应用程序上的搜索框提供即时的逐字母搜索结果。

该公司正在管理 Slack、Stripe、安德玛、Twitch 和其他 9000 家公司的搜索功能。按照目前的运行速度，Algolia 每年处理 1.2 万亿次搜索。该公司表示，它每天接触六分之一的网络用户。

“故事就在 C 系列之后开始了，”德塞格告诉我。Algolia 在 2019 年底为筹集了 1.1 亿美元的 C 轮融资。“我非常兴奋，但最让我兴奋的是这家公司的潜力。”

“拥有更多市场经验的人可能会更好地实现这一潜力，”他继续说道。

我更直接地问，接替他担任 CEO 的决定是否来自公司董事会。“这真的是从我这边开始的。董事会支持这一决定，但它不是来自他们，”他说。

尼克森之前是 Alfresco 的首席执行官，该公司开发了一家开源企业内容管理初创公司，于 2018 年被私募股权公司 Thomas H. Lee Partners 收购。过去，她在不同的软件公司担任过各种职务，如首席收入官、销售执行副总裁和企业销售高级副总裁。

如你所见，尼克松在销售和运营方面有着丰富的经验。在扩大创业规模时，她的经验将会很有价值。

“我很高兴现在成为 Algolia 团队的一员，并从今天开始领导公司，”尼克松告诉我。带领阿尔戈利亚 C 轮融资的风投公司 Accel 也是 Alfresco 的投资者之一。

过渡将需要几个月的时间，德塞格内将一直呆到 7 月。他说他对下一步要做什么没有任何具体的计划。

过去一年，Algolia 一直在加大高管招聘力度。Jean-Louis Baffier 担任首席收入官，Ashley Stirrup 担任首席营销官，Kristie Rodenbush 担任首席人事官，Iain Hassall 担任首席财务官。换句话说，Algolia 正在成长，为下一阶段做准备。现在让我们看看它是否会导致 IPO 或被更大的公司收购。
