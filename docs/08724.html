微软的量子计算平台现在处于有限的预览阶段

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/19/microsofts-quantum-computing-platform-is-now-in-limited-preview/amp/

图片来源:新华社/西尔维亚·莱德雷尔 via Getty Images / Getty Images

微软今天宣布 Azure Quantum ，这是一个以合作伙伴为中心的量子计算平台，面向想要开始量子计算的开发者，现已进入有限预览。Azure Quantum 在微软 Ignite 2019 上首次发布，它汇集了来自 IonQ、Honeywell、QCI 和微软的硬件，来自 1QBit 等公司的服务，以及 Azure 云的经典计算能力。随着这一举措进入有限的预览，微软现在向少数选定的合作伙伴和客户开放这项服务。

在目前阶段，量子计算对任何企业来说都不是一项关键任务能力，但考虑到事情发展的速度有多快，以及这项技术在未来几年稍微成熟后会有多强大，许多专家认为现在是开始的时候了——特别是因为量子计算与经典计算有多么不同，开发人员需要一段时间才能开发出来。

在 Ignite，微软还开源了其量子开发工具包、编译器和模拟器。

有鉴于此，该公司正采取一种不同于一些竞争对手的方法。此外，微软目前还不得不与量子硬件公司合作，因为它自己的量子硬件努力还没有达到可行的程度。该公司正采取一种与 IBM 或 Rigetti 等公司截然不同的方法，押注于其机器核心的另一种量子位。虽然最近几个月取得了一些突破，但它还没有一个可以工作的量子位——或者说，即使有，它也没有公开谈论过。
