微软的Visual Studio Live Share内置了语音和文本聊天功能

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/19/microsofts-visual-studio-live-share-gets-built-in-voice-and-text-chat/

微软的Visual Studio Live Share内置了语音和文本聊天功能

Visual Studio Live Share 是微软在Visual Studio IDE和Visual Studio代码编辑器中进行实时协作代码编辑的工具。今年早些时候，该公司还在T2推出了基于浏览器版本的工具预览版。虽然共享的实时代码编辑器很棒，但用户总是不得不依赖额外的工具来讨论他们正在做的事情。然而，这种情况正在改变，因为在Build 2020上，微软今天宣布将在这项服务中添加语音和文本聊天。

鉴于目前的情况，任何使在线协作更容易的工具都可能是开发人员工具箱中受欢迎的新成员。与屏幕共享或其他合作方式不同，Live Share的一个优点是，个人开发者可以保留他们自己的、通常有些特殊的设置。像大多数现代协作文档编辑器一样，每个人也可以保留自己的光标。

截至目前，看起来文本和语音聊天并没有与微软的任何其他通信工具(如团队)集成，因此至少目前来说，Live Share中发生的事情仍停留在Live Share中。

新功能现已公开预览。
