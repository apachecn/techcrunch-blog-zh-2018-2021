无头 CMS 公司 Strapi 再融资 1000 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/20/headless-cms-company-strapi-raises-another-10-million/

广受欢迎的开源无头 CMS 背后的公司 Strapi ，也叫 Strapi，已经筹集了 1000 万美元的首轮融资，由 Index Ventures 牵头。该公司此前在 2019 年 10 月筹集了由 Accel 和 Stride.vc 领投的 400 万美元种子轮。

Strapi 是一个无头内容管理系统，这意味着后端和前端完全独立运行。您可以在自己的服务器上运行 Strapi，并通过连接到 Strapi 的管理界面为您的站点编写内容和页面。

之后，应用程序的前端部分可以使用 api 从 Strapi 实例中获取内容，并将其显示给客户和读者。

把前端和后端分开有很多好处。首先，在显示内容时，它给了你极大的灵活性。你可以使用一个流行的前端框架，比如 React，Vue 和 Angular，或者开发自己的定制前端。

当你想更新你的网站设计时，你可以从一个前端切换到另一个前端，Strapi 像往常一样在后台运行。

同样，在服务器架构方面，它提供了更大的灵活性。例如，您还可以利用 Strapi 构建静态网站，并使用内容交付网络(如 Cloudflare 或 AWS CloudFront)分发它们。你可以想象使用 Gatsby 结合 CDN 在边缘部署你的站点。你的大部分流量将通过你的 CDN，而不是直接到达你的服务器。

此外，Strapi 可用于向不同的前端分发内容。例如，您可以为您的网站和移动应用程序的内容使用 Strapi 实例。

Strapi 证明了最终一切都变成了 api。当然，对于大多数项目来说，一个无头的 CMS 可能是多余的。但是如果你正在运行一个大规模的应用程序，Strapi 可以很好地适应你的架构。使用 Strapi 的公司包括 IBM、NASA 和沃尔玛。

许多知名的开源商业天使也投资了 Strapi，如来自 Kong 的 Augusto Marietti 和 Marco Palladino，来自 Sentry 的 David Cramer，来自 Dataiku 的 Florian Douetteau，来自 Docker 的 Solomon Hykes，来自 Cloudup、Socket.io、Next.js 和 Zeit.co 的 Guillermo Rauch，以及来自 Cloudera 的 Eli Collins。
