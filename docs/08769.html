虚拟活动初创公司 Run The World 刚刚从 a16z 和创始人基金 TechCrunch 获得 1080 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/21/virtual-events-startup-run-the-world-just-nabbed-10-8-million-from-a16z-and-founders-fund/

总部位于加州山景城的初创公司 Run The World 刚刚获得 1080 万美元的首轮融资，由早期投资者安德森·霍洛维茨(Andreessen Horowitz)和新投资者创始人基金(Founders Fund)共同牵头。

很容易理解这些公司对这家公司的兴趣，它的平台提供了一个会议组织者在疫情时代甚至之后可能需要的所有功能，因为许多机构正在重新思考如何更持久地举办活动，使包括广泛的参与者。想想视频会议、票务、互动和网络。

几个月前，我们曾报道过这家初创公司，当时它获得了安德森合伙人康妮·陈(Connie Chan)领导的 430 万美元种子资金，其他一些种子阶段的支持者也加入了进来，包括 Pear Ventures、GSR Ventures 和 unionly Capital。据联合创始人兼首席执行官屈称，鉴于当前的大环境，Run The World 受到了相当大的关注，这或许并不奇怪。他此前曾负责和 Instagram 的产品。

“自从我们在 2 月份推出以来——并免除了受冠状病毒影响的活动的所有筹备费用——我们每天都收到数百个入境活动请求，”曲说。更具体地说，她说，这家初创公司已经将其核心团队的规模扩大了一倍，达到 30 名员工，并使来自各个国家的组织者能够监督目前超过 2000 场活动。

瞿说，许多曾经使用 Zoom 举办网络研讨会的活动策划人现在选择了 Run The World，因为它专注于参与和社交功能。例如，该平台上一场活动的参与者被邀请创建一个类似于 Instagram 故事的视频个人资料，这可以帮助其他参与者了解他们是谁。它还组织相关的“鸡尾酒会”，每次可以为与会者匹配几分钟，与会者可以选择他们想随后跟进的人。

对社交网络的高度关注并非偶然。瞿在遇到了她联合创始人宣江，当时宣江是活动、广告和故事的技术主管。

当然,《运行世界》( Run The World)从所使用的模板、门票销售、支付处理和流媒体等所有内容中抽取 25%的门票收入，在 Zoom 领域仍有非常激烈的竞争。自 2 月份以来，这家成立 9 年的公司的消费者采用率飙升，截至 4 月底，每天有 3 亿人使用这项服务。

不仅很难克服这种网络效应，而且 Run The World 也不是唯一一个试图引导活动组织者前进的组织。例如，本周早些时候，由活动系列初创公司 Grind 的创始人共同创立的活动软件业务 Bevy 宣布，它已经在由 Accel 牵头的 B 轮融资中筹集了 1500 万美元。最近几个月，其他年轻的在线活动平台也同样获得了风险投资，包括总部位于伦敦的 Hopin (有趣的是，其最近一轮也是由 Accel 牵头)和总部位于巴黎的 Eventmaker。

尽管如此，新的资金应该会有所帮助。瞿表示，尽管 Run The World 迄今为止“完全通过口口相传实现了有机增长”，但这家初创公司计划壮大自己的团队，而且可能会开始在营销方面至少投入一些资金。

精通社交媒体的投资者很可能会在这最后一条战线上推波助澜。

除了 a16z 和 Founders Fund 之外，它的 A 轮投资中还有许多其他的支持者，包括威尔·史密斯的 Dreamers VC 和凯文·哈特的 Hartbeat Capital。
