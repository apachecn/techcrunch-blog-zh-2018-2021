DHL 收购 MallforAfrica TechCrunch 开发的 Link Commerce 的股份

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/27/dhl-acquires-stake-in-link-commerce-developed-by-mallforafrica-com/

DHL 已经收购了 Link Commerce 的少数股份，Link Commerce 是一家交钥匙电子商务公司，由尼日利亚数字零售初创公司 MallforAfrica.com 发展而来。

Link Commerce 为新兴市场的在线销售提供了一个白标解决方案。

零售商可以接入该公司的电子商务平台，创建一个基于网络的店面，管理支付和物流。

通过这笔投资，这家全球最大的快递服务公司之一希望利用在非洲建立的业务，在全球范围内建立更广泛的客户基础。

“DHL 正试图更多地涉足全球电子商务……他们认为我们的平台是一个很好的方式，”Link Commerce 首席执行官 Chris Folayan 告诉 TechCrunch。

佛拉良最初创立了 MallforAfrica ，为 Link Commerce 铺平了道路。DHL 对该公司的投资(金额未披露)源于与 Folayan 最初的创业公司的合作。

MallforAfrica 于 2015 年开始与 DHL 合作，并于 2019 年推出了 DHL Africa eShop 。该销售平台由 Link Commerce 提供支持，已经为 34 个国家的非洲消费者带来了 200 多名美国和英国卖家，从 Neiman Marcus 到 Carters。

图片来源:DHL

与 MallforAfrica 的模式类似，Africa eShop 允许用户直接从该应用的任何合作伙伴的网站上购买商品。

对于在非洲电子商店上销售的全球零售商来说，阻碍非洲大陆分销的障碍——支付、货币风险、物流——由底层的 Link Commerce 运营平台处理。

“这就是我们的服务。它照顾了整个生态系统，使全球电子商务能够存在，无论你在哪个国家，”Folayan 在 2019 年告诉 TechCrunch。

Link Commerce 的前身是福拉延在美国学习和工作后于 2011 年创立的初创公司 MallforAfrica.com

非洲人的一个普遍做法是给在国外的家庭成员一份商品清单，让他们购买并带回家，这凸显了非洲大陆消费市场的供需缺口。

MallforAfrica Folayan 旨在通过允许非洲大陆的人们直接在线购买全球零售商的商品来缩小这一差距。

MallforAfrica 和 Link Commerce 创始人 Chris Folayan，图片来源:MallforAfrica

这家电子商务网站后来加入了 250 多家全球零售商，现在在俄勒冈州和英国的订单处理设施雇用了 30 名员工。

MallforAfrica 的非洲 eShop 扩张使其有望与泛非电子商务领导者Jumia——于 2019 年在纽约证券交易所上市——和中国的阿里巴巴竞争，后者预计将在某个时候进入非洲大陆的在线零售。

Link Commerce 与 DHL 的交易不会改变这一点，但 Folayan 已经改变了他的业务的雇佣关系，使 Link Commerce 成为主要的运营机构，并将非洲作为众多市场中的一个。

图片来源:链接商务

“我们改变了结构。所以现在 Link Commerce 在 MallforAfrica 之上，MallforAfrica 现在由 Link Commerce 驱动，”Folayan 在最近的一次电话中解释道。

“现在的焦点是在非洲……但我们正在走向全球,”他补充道。

Folayan 和 DHL 计划将该平台扩展到全球新兴市场，其他公司可能会通过围绕其核心业务包装在线商店、支付和物流解决方案来寻求增长。

据 Folayan 称，这可能包括任何想要推出国际电子商务网站的大型实体。

“Link Commerce 专注于银行、移动公司、航运公司，并与它们合作进行全球扩张，”他说。

这相对于佛拉扬最初的企业 MallforAfrica.com 来说是一个巨大的飞跃

这家最初在非洲在线销售品牌牛仔裤和运动鞋的初创公司，已经转向物流巨头 DHL 部分拥有的全球电子商务履行业务。
