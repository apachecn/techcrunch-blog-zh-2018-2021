HBO Max昨天被87K新用户下载(传感器塔)

昨天HBO Max 的发布会进行的怎么样？我们没有来自WarnerMedia的官方数据，但应用商店情报公司 Sensor Tower 表示，HBO Max在苹果应用商店和Google Play上被近87，000名新用户下载。

与其他流媒体发布相比，这个数字似乎很低——比如迪士尼+的400万首日安装量，甚至是《T4》的Quibi 的30万安装量。

但请记住，HBO Max并不是一项全新的服务，无论是从内容的角度来看(它将HBO的图书馆与其他各种电视节目和电影捆绑在一起)，还是从应用的角度来看，因为它是作为现有HBO Now流媒体应用的更新而发布的。

Sensor Tower承认，这些数字不包括只是更新了旧HBO应用程序的人，但它提供了另一种方式来看待昨天的表现:以前，HBO Now平均每天有16，000次新安装，因此比正常情况下多了71，000次下载。

同样值得注意的是，当我在周四下午写这篇文章时，HBO Max目前在App Store的“免费”应用中排名第二，在Zoom之后，但在YouTube、网飞、抖音和Disney+之前。

Sensor Tower估计，自2015年4月推出以来，HBO Now和Max已有3300万人下载，相比之下，网飞有2.6亿人，Hulu有1.2亿人(网飞和Hulu都是从2014年1月开始测量的)，迪士尼+有5000万人。
