数字银行技术提供商 Meniga 在法国银行集团 BPCE TechCrunch 的带领下向€融资 850 万英镑

原文：https://web.archive.org/web/https://techcrunch.com/2020/05/28/meniga-the-digital-banking-tech-provider-raises-e8-5m-led-by-french-bank-groupe-bpce/

数字银行技术提供商 Meniga 向€筹集 850 万欧元，由法国银行 BPCE 集团牵头

总部位于伦敦的金融科技公司 Meniga 向世界上一些最大的银行提供数字银行技术，该公司已经停止向€追加 850 万美元的资金。

这轮融资主要被描述为一项“战略投资”，由法国第二大银行集团 BPCE 集团牵头，葡萄牙信贷银行集团(Grupo Crédito Agrícola)和长期战略合作伙伴意大利联合信贷银行(UniCredit)也参与其中。这三个人都是 Meniga 的客户。

这笔资金将用于继续投资 Meniga 在 R&D 的活动，以及加强 fintech 的销售和服务团队，以满足其所说的不断增长的需求。本轮的其他参与者包括当前的机构投资者 Velocity Capital、Industrifonden 和 Frumtak Ventures。

“我们非常高兴地欢迎 Groupe BPCE 和 Crédito Agrícola 加入我们不断壮大的战略投资者群体，”Meniga 首席执行官兼联合创始人 Georg Ludviksson 在一份声明中表示。“与客户紧密合作是我们成为客户首选数字创新合作伙伴战略的重要组成部分。股权关系是加强这种伙伴关系的绝佳方式。”

Meniga 的数字银行平台帮助银行和金融科技公司使用个人金融数据来创新他们的在线和移动产品。它的各种产品包括一个软件层，该软件层在银行的传统技术基础设施和现代 API 之间架起了一座桥梁，使构建消费者友好的数字银行体验变得更加容易。

Meniga 的产品套件涵盖数据聚合技术、个人和企业财务管理解决方案、现金返还奖励和基于交易的碳洞察。

该公司的技术还旨在支持开放式银行并从中受益，在此帮助下，其产品和服务已经被 30 个国家的 9000 多万银行客户使用。

这使得它在 2019 年在巴塞罗那和新加坡开设了新的办公地点，增加了其在伦敦(该公司总部所在地)和雷克雅未克(其大部分 R&D 所在地)的现有业务，以及在斯德哥尔摩、赫尔辛基和华沙的办事处。

与此同时，主要投资者 Groupe BPCE 在 2018 年首次与 Meniga 合作。Groupe BPCE 公司首席数字和数据官兼 Groupe BPCE 公司管理委员会成员 Yves Tyrode 的提示声明:“迄今为止，我们与 Meniga 的合作关系非常积极。我们共同为 BPCE Groupe 的持续数字创新奠定了基础，以便在一个充满活力的银行市场中更好地为我们的客户服务。我们期待继续转变我们的数字客户体验，并与 Meniga 一起为构建数字银行的未来做出贡献。”
