Priyanka Sharma接管云计算原生计算基金会TechCrunch的领导权

原文：https://web.archive.org/web/https://techcrunch.com/2020/06/01/priyanka-sharma-takes-over-the-leadership-of-the-cloud-native-computing-foundation/

云计算原生计算基金会(Cloud Native Computing Foundation)今天宣布，长期担任该组织执行董事的丹·科恩(Dan Kohn)将离职，GitLab云计算原生联盟(Cloud Native Alliances)主任T2·普里扬卡·夏尔马(Priyanka Sharma)将接任总经理一职。Kohn将继续作为Linux基金会的一员，在那里他将发起一项新的倡议“帮助公共卫生当局使用开源软件来抗击新冠肺炎和其他流行病”

夏尔马曾经参加过她联合创立的创业公司 TechCrunch Disrupt战地竞赛，她在2016年担任Lightstep营销和战略合作伙伴主管期间，成为了整个云原生社区的一部分。她告诉我，她对OpenTracing项目的参与滚雪球般地加深了与CNCF的关系。“一旦我加入GitLab，我就很幸运地被选为CNCF的董事会成员——直到31日，我都是这个角色，”她告诉我。“这真的很有帮助，但这让我了解了这样一个成功的基金会和社区是如何运行的——这里的治理部分是什么——当我在社区方面时，我并没有参与其中。”

Kohn自2016年以来一直执掌CNCF，并从早期开始指导该项目成为有史以来最成功的开源基金会之一。其两年一次的会议吸引了来自世界各地的数千名开发人员。虽然它的招牌项目显然是Kubernetes，但Kohn和他在基金会的团队非常重视整个生态系统。毕竟，该组织的使命是“让云计算无处不在。”今天，CNCF是10个毕业项目的所在地，如Kubernetes、Prometheus、Envoy、Jaeger和Vitess，以及16个所谓的“孵化项目”，如OpenTracing、Linkerd、Rook和etcd。

科恩在一份声明中说:“在过去的几年里，普里扬卡作为议长、管理委员会成员和社区领袖，对CNCF做出了不可估量的贡献。”。“我认为她是领导该组织进入下一阶段的绝佳人选。”

夏尔马说，她将从听取社区意见开始她的任期。“原生云已经成为事实上的标准，”她说。“我们在技术采用和增长方面做得很好，但随着事情的发展，基金会的人数已经有望达到600人，我认为作为一个社区，那里有如此多的增长机会，我们可以更深入地参与开发人员，围绕教育和对所有项目的理解进行更多的对话。现在我们有10个毕业项目——不仅仅是Kubernetes——所以那里会有很多采用。因此，我对我们将迎来的第二波浪潮感到非常兴奋。”

现在每个人都知道DevOps和containers，她想让更多的人加入进来——同时也看看像无服务器和服务网格这样的新技术。“我们已经有了一个轰动的开始，现在我认为我们必须成熟一点，走得更远，”她说。

值得注意的是，现任CNCF首席技术官Chris Aniszczyk将继续在基金会任职。他说:“在过去的几年里，随着公司寻求更加灵活和创新的解决方案来满足他们不断发展的基础设施和应用需求，云原生社区已经取得了突飞猛进的发展。“随着 CNCF 现在在全球拥有近50个项目和90，000名贡献者，我很高兴有机会与Priyanka合作，在下一次发展中培养和发展我们的云原生社区。”
