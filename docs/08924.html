Zigazoo 推出儿童“抖音”,上传和下载量超过 100，000 次——TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/06/02/zigazoo-launches-to-be-a-tiktok-for-kids-surpasses-100000-uploads-and-downloads/amp/

像许多父母一样，Zigazoo 创始人 Zak Ringelstein 担心他的孩子在屏幕前的时间。当新冠肺炎导致学校关闭，孩子们回到远程学习的世界时，他的担忧加剧了。现在，随着锁定时间的延长，林格尔斯坦正在学习拥抱屏幕时间，以此作为一种将教育和娱乐偷偷带入孩子的数字饮食的方式。

Ringelstein 是 UClass 的前创始人(于 2015 年收购了)，他创办了 Zigazoo，他称之为“孩子们的抖音”

Zigazoo 是一个免费的应用程序，孩子们可以通过视频回答简短的视频练习，并与朋友分享答案。练习的范围从如何创造一个小苏打火山到从食物中提取分数，目标是从学龄前到中学的孩子。

为了确保应用程序的隐私，Ringelstein 说父母应该是应用程序的主要用户。用户必须接受好友请求才能看到他们的内容，Ringelstein 认为这是避免不良行为或潜在欺凌的关键。

此外，Zigazoo 通过 SightEngine 使用 API 来调节内容。

林格尔斯坦的第一批用户是他自己的孩子，他说这是一次非常有益的测试。

林格尔斯坦的儿子参与了 Zigazoo 提示。

测试过程让他意识到，孩子们喜欢创作更长的视频，观看更小的视频，因此 Zigazoo 正在计算观看视频的注意力持续时间。目前，每个用户每天在网站上的平均时间已经上升到 19 分 43 秒。

林格斯坦指出《芝麻街》是他的灵感来源。事实证明，将教育和娱乐结合在一起对许多企业来说都是成功的。他回忆说，孩子们在屏幕前看《芝麻街》的角色时会流口水，花几个小时目不转睛地盯着电视机。

“芝麻街的创造者……用这种媒介来教育孩子，同时娱乐他们，”林格尔斯坦说。Vox 将《芝麻街》描述为“教育电视的基石”，将受人喜爱的角色带到前第一夫人米歇尔·奥巴马的桌前，或者用一首愚蠢的歌曲教导孩子们关于回收利用。

在一个月内，Zigazoo 的网站上传和下载了 10 万个视频。

虽然 Zigazoo 声称是儿童的“抖音”，但它正在与平台本身竞争。一些教师转向抖音，开设太阳能电池系统和实验课程。

其他人正在整理“儿童友好型”抖音创作者的指南。抖音自己最近也允许父母为他们的孩子设置内容、数字媒体和屏幕时间的限制。

根据 Ringelstein 的说法，对于学生来说，基于视频的学习是一种更好的积极参与教育活动的方式，而不是被动地阅读谷歌文档中的一段内容。

将教育和娱乐结合起来会带来一系列与儿童安全相关的风险。去年三月,《纽约时报》写了一篇关于“儿童影响者”如何成为一个概念的报道，父母把他们的孩子放在网上，兜售品牌，并从中赚钱。由此产生的伦理问题是为什么 Ringelstein 相信 Zigazoo 是必要的。

“Zigazoo 不是一个孩子玩耍的约会，而是像 YouTube 和抖音这样的成人聚会，它是为孩子的安全、学习和享受量身定制的世界，”他说。

Ringelstein 认为 Zigazoo 的“朋友”与“关注”功能是孩子安全的关键:与抖音不同，那里有公共订阅源，用户可以关注每个人，Zigazoo 要求用户选择被关注，类似于脸书。

这种合作关系将允许 Zigazoo 使用喜爱和知名的角色发布经过验证的内容，以教授孩子们他们关心的主题。在一个数字排毒不再成为现实的世界里，一种更智能的屏幕时间活动似乎非常必要。

最近，Zigazoo 与美国教师联合会合作开展了一个针对数百万 K-12 学生的顶点项目。学生们被邀请使用 Zigazoo 提交一个视频，概括他们在过去一学年的学习经历，AFT 表示，这是“比高风险考试更好的总结学习的方式。”

今年夏天，Ringelstein 将推出“Zigazoo Channels ”,由主要儿童娱乐公司、播客、博物馆、图书馆、动物园、社交媒体影响者等精选群体组成。
