随着WhatsApp Pay艰难推出，谷歌和沃尔玛在印度移动支付市场确立了主导地位

原文：https://web.archive.org/web/http://techcrunch.com/2020/06/03/google-and-walmarts-phonepe-establish-dominance-in-indias-mobile-payments-market-as-whatsapp-pay-struggles-to-launch/

在印度，谷歌(Google)和沃尔玛(Walmart)旗下的PhonePe正在不分上下地争夺移动支付市场的老大地位，而脸书仍因WhatsApp Pay的推出而陷入监管迷宫。

知情人士告诉TechCrunch，Google Pay上个月有超过7500万交易用户，超过PhonePe的6000万用户。

知情人士说，相比之下，软银支持的Paytm应用上个月有3000万交易用户，平均每天有1000万用户交易。Paytm反驳了这些数字。TechCrunch无法确定Google Pay和PhonePe的日常交易用户。

谷歌在商户方面仍落后于Paytm，但安卓制造商在最近几个月保持了整体领先地位，尽管由于印度实施的全球最严格的封锁之一，每个玩家都失去了动力。

据路透社上个月报道，该公司在印度面临投诉，指控其滥用市场地位，在该国不公平地推广Google Pay应用。

Paytm曾经是印度的主导者，近两年来一直在努力维持其用户群。知情人士说，该公司去年1月拥有约6000万交易用户。

一位发言人告诉TechCrunch，这位印度本土球员表示，5月份其应用程序的月活跃用户超过5000万。这个故事发表后，发言人声称每月有超过5000万用户在它的应用上交易。《T4》1月份报道称，Paytm在12月的交易用户不到4000万。

数据集认为交易用户是那些在一个月内通过该应用至少支付一次的人。这是一个令人垂涎的指标，不同于更受欢迎的月活跃用户(MAU)或日活跃用户(DAU ),后者是各公司用来分享业绩的指标。一部分被标记为每月活跃用户的人没有在应用程序上进行任何交易。

一些人说，Paytm近年来一直难以增长，部分原因是印度央行要求移动钱包公司——用户和银行之间的中间人——对用户进行KYC验证，这在很多人中间造成了混乱。尽管该公司成功筹集了超过30亿美元的资金，但这些问题依然存在。

在一份声明中，Paytm的发言人表示，“当谈到移动钱包时，人们必须记住这样一个事实，即Paytm是建立KYC基础设施的公司，并且已经通过与客户见面完成了1亿多kyc。”

Paytm长期以来一直受益于与优步和食品配送初创公司Swiggy等热门服务的集成，但最近几个月，Paytm的每月交易用户中只有不到1000万人依赖这一功能。

两名高管也表示，Paytm抵制采用统一支付接口的想法。由于担心遭到报复，他们和其他人一样要求匿名。这是近两年来由印度一批银行建立和支持的支付基础设施，它使资金能够在不同银行的账户之间直接转账，并且消除了对单独移动钱包的需求。

Paytm在采用该标准方面的延迟给谷歌和PhonePe(另一家UPI的早期采用者)留下了抓住机会的空间。

比谷歌和PhonePe晚一年采用UPI的Paytm驳斥了拒绝加入UPI生态系统的说法。

“我们公司珍视能够改变数百万人生活的创新和技术。我们理解金融技术的重要性，正因为如此，我们一直是UPI的拥护者和支持者。然而，我们在Paytm上推出它比我们的同行晚，因为我们花了更长的时间获得批准，开始基于UPI的服务，”一位发言人说。

2017年2月4日星期六，由One97 Communications Ltd .运营的Paytm在线支付方式的标志展示在印度班加卢鲁一个出售配件的街头摊位上。摄影师:迪拉杰·辛格/彭博

脸书没有加入这场竞争，它将印度视为其用户数量最多的市场。该公司开始与银行谈判，以通过WhatsApp最早在2017年进入印度的移动支付市场，预计到2023年将达到1万亿美元(根据瑞士瑞信银行的数据)。WhatsApp是印度最受欢迎的智能手机应用程序，在该国拥有超过4亿用户。

第二年，脸书向100万用户推出了WhatsApp Pay，但此后一直陷入监管之战，无法将支付服务扩展到其他用户。脸书首席执行官马克·扎克伯格表示，WhatsApp Pay将于去年年底在全国推广，但该公司尚未获得所有批准——新的挑战不断出现。该公司在四月份投资了57亿美元给国家最大的电信运营商Reliance Jio Platforms ，该公司拒绝对此发表评论。

PhonePe是在WhatsApp将目光投向印度移动支付的一年前构思的，随着它增加了几项第三方服务，它一直在增长。其中包括领先的食品和杂货配送服务Swiggy和Grofers，打车巨头Ola，票务和住宿玩家Ixigo和Oyo Hotels，这是一项所谓的超级应用战略。去年11月，约有6300万用户活跃在PhonePe上，其中4500万人通过该应用进行交易。

PhonePe的业务主管Karthik Raghupathy向TechCrunch证实了该公司的交易用户。

他在接受采访时说，三个因素促成了PhonePe的增长。“近年来智能手机和移动数据应用的兴起；早期采用合众国际社，当时印度大多数移动支付公司都在赌虚拟移动钱包模式；并采取开放生态系统的方法，”他说。

“我们很早就向所有商业合作伙伴开放了我们的消费者基础。我们的理念是，我们不会进入电影和旅游在线票务等领域，而是与这些领域的市场领导者合作，”他解释道。

“我们还带着完全开放、可互操作的二维码进入市场，使商家和企业能够使用一个二维码接受来自任何应用程序的支付，而不仅仅是我们的应用程序。在此之前，你会看到一个社区商店维护几个二维码来支持许多支付应用程序。多年来，我们的方法已经成为行业规范，”他说，并补充说，PhonePe也对其他钱包和支付选择持类似的开放态度。

但是，尽管PhonePe取得了增长并采取了开放的方式，但在最近几个季度，它仍难以赢得投资者的信心。加剧投资者担忧的是印度移动支付公司缺乏清晰的商业模式。

PhonePe高管去年举行了融资谈判，对其估值为80亿美元，但谈判破裂。三位知情人士说，今年早些时候的类似谈判也以失败告终，该谈判对PhonePe的估值为30亿美元，此前没有报道过。Raghupathy和PhonePe发言人拒绝对该公司的筹资计划发表评论。

目前，沃尔玛已同意继续为支付应用提供资金，该应用在2018年收购 Flipkart后成为零售集团的一部分。

随着合众国际社进入市场，银行已经取消了对移动支付玩家的任何促销激励，这是他们唯一的收入来源之一。

去年年底在班加罗尔举行的一次活动上，Google Pay和Next Billion User Initiatives的董事总经理兼业务负责人萨吉斯·西瓦南丹(Sajith Sivanandan)表示，目前的地方法规迫使Google Pay在印度没有清晰的商业模式。

冠状病毒让支付公司付出了代价

促使印度政府在3月下旬下令全国封锁的冠状病毒疫情，在接下来的几周内导致移动支付使用量大幅下降，但这是可以预见的。但是，当Paytm继续努力反弹时，PhonePe和Google Pay已经完全恢复，因为印度放松了一些限制。

根据监管合众国际社的NPCI汇编的数据，5月份Paytm上发生了约1.2亿笔UPI交易，低于4月份的1.27亿笔和3月份的1.86亿笔。(Paytm维持着移动钱包业务，这为其整体交易用户做出了贡献。)

仅支持UPI支付的Google Pay在5月份促成了5.4亿笔交易，高于4月份的4.34亿笔和3月份的5.15亿笔。PhonePe 3月份的4.54亿美元下滑至4月份的3.68亿美元，但情况有所好转，上个月的交易额为4.6亿美元。NPCI发言人没有回应置评请求。

PhonePe和Google Pay合计占上个月印度所有UPI交易的83%左右。合众国际社本身拥有超过1.17亿用户。

在竞争对手公司工作的行业高管表示，解雇Paytm将是一个错误，pay TM曾是印度移动支付市场的领导者。

Paytm已经削减了营销费用，并在最近几个季度积极追逐商家。今年早些时候，它推出了一系列小工具，包括一个显示二维码结账代码的设备，配有一个计算器和USB充电器，一个自动点唱机，提供交易和服务的语音确认，以简化商家的库存管理。

该公司的联合创始人兼首席执行官Vijay Shekhar Sharma在今年早些时候接受TechCrunch采访时表示，使用这些设备的商家需要向Paytm支付经常性费用。Paytm还进入了几个业务领域，如电影和旅游票务、贷款、游戏和电子商务，并在多年内建立了一个数字支付银行。

“大家都知道Paytm。Paytm在印度是数字支付的代名词。在外面，有一种感觉是，它真的是印度的支付宝，”一家竞争对手公司的高管说。

这篇报道在发表后用Paytm提供的额外数据进行了更新。
