个人症状追踪器、数字接触追踪和暴露通知工具正在美国和世界各地开发中——它们的采用可以帮助医疗工作者减轻新冠肺炎病毒进一步传播的影响。这些技术也有重大的隐私和安全问题。COVID技术任务组计划在10天后召开会议，讨论与COVID技术相关的关键问题。

作为此次会议筹备工作的一部分，我们收集并审查了美国领先的应用程序。为了帮助公众、州和地方政府更好地了解领先应用程序的隐私和安全功能，我们分享了从构建这些应用程序的团队收集的信息和演示。

我们将演示分为三大类:(1)使用Google/Apple API的联系追踪/暴露通知应用，(2)不使用Google/Apple API的联系追踪/暴露通知应用，以及(3)个人症状追踪应用。

我们就个人数据的隐私、安全和商业化对团队进行了调查。我们在这里提供了调查的结果。我们鼓励您浏览这些回复，并分享您对不同应用如何处理这些重要问题的想法。

这篇文章中的应用程序将在原定于本周举行的联系追踪和技术会议上演示——鉴于围绕种族不公正和警察对美国黑人施暴的重要讨论，我们重新安排了时间，以确保我们不会占用不必要的空间。会议现已改期至6月17日——如果您回复，我们期待您的光临；如果你没有，请做！

该会议将由COVID Tech Task Force主办，并与TechCrunch、哈佛大学伯克曼克莱恩中心、NYU公共利益技术联盟、Betaworks工作室和Hangar合作。COVID Tech Task Force由一群志愿者组成，他们在3月份聚集在一起，帮助召开一个州和地方政府以及技术社区的论坛，共同努力减轻新冠肺炎的影响。

如果您已经开发了联系追踪或曝光通知应用程序，请随时联系partners@crttf.org的，并在此填写调查。

使用谷歌/苹果API的应用程序

谷歌和苹果合作开发了开发工具，以便为公共卫生机构提供一种跨平台的方式来通知个人潜在的暴露。

SafePaths正在为个人、公共卫生官员和更大的社区开发免费、开源、按设计保护隐私的工具，以拉平新冠肺炎曲线，减少恐惧，防止国家对疫情的监控反应。

如果你想了解更多信息，请联系info@pathcheck.org。

CoEpi是一个开源项目，开发了一个分散的、隐私优先的应用程序，用于基于症状共享的匿名蓝牙暴露通知。密切接触的社区可以开始用CoEpi保护自己，而不需要在一般人群中广泛采用；不需要规模就能让小用户群体受益。CoEpi帮助您匿名提醒与您互动的人有关传染性疾病的症状，或者提醒您是否可能在互动中接触过。

如果你想了解更多信息，请联系Dana+CoEpi@OpenAPS.org。

COVID Shield是一款免费的曝光通知解决方案，以保护隐私为首要任务。它是由一群志愿者建造的，其中许多人来自Shopify，目的是帮助加拿大人和世界其他地方的人安全返回工作岗位。

如果您想了解更多信息，请联系 press@covidshield.app 。

该团队由华盛顿大学和微软的公共卫生官员、医生、研究人员和工程师组成，他们共同努力保持公众安全，并帮助公共卫生系统管理疫情。

如果你想了解更多信息，请联系covidsafe@uw.edu的。

COVID Trace是一家非营利组织，使用苹果/谷歌暴露通知API为iOS和Android提供新冠肺炎暴露通知应用程序。使用COVID Trace的人可以期待隐私和简单。借助COVID Trace，卫生部门获得了一个应用程序和指标，这是他们工作的延伸。COVID Trace今天就可以使用了。

如果你想了解更多信息，请联系hello@covidtrace.com的T4。

Zero是一个由公民领导的非营利组织，利用技术促进疫情反应，专注于促进安全的社会行为和内心的平静。他们的目标是阻止新冠肺炎的传播，并给市民提供他们需要的信息，让他们感到安全，有信心参与当地经济。

如果你想了解更多信息，请联系support@usezero.org。

Covid Watch由斯坦福大学和滑铁卢大学的研究人员于2月份创建，是一个开源的非营利团队，由500多名隐私和公共卫生研究人员、开发人员、志愿者和学术顾问组成。Covid Watch开发了开创性的完全匿名TCN协议，该协议后来影响了它现在使用的GAEN协议。Covid Watch正在积极与大学和公共卫生部门合作，将Covid Watch应用程序和可定制的验证门户与公共卫生系统集成，同时保持应用程序用户的匿名性。

如果你想了解更多信息，请联系contact@covid-watch.org。

当前未使用谷歌/苹果API的应用程序

请注意，这些组织中的一些已经表示他们将来可能会使用Google/Apple API。他们中的一些人打算这么做，并且正在等待谷歌/苹果的确认。

NOVID声称是美国发布的第一个(也是目前唯一一个)完全匿名的不使用个人信息的联系人追踪应用。没有GPS，没有电话号码，没有电子邮件——完全匿名。该应用程序利用超声波来提供极其精确的交互距离测量，克服了蓝牙的已知不准确性。该团队由卡内基梅隆大学教授、国际知名数学家陆宝琛领导。

如果你想了解更多信息，请联系feedback@novid.org。

“共同健康”是一个完全整合到公共卫生和企业中的端到端新冠肺炎响应平台。4月份在犹他州推出的Healthy Together移动应用程序支持自我评估、新冠肺炎测试访问和结果、增强的联系人跟踪，以及企业联系人跟踪、工作流工具、数据集成和可视化。利用已扩展到数百万用户并由公共卫生专家提供信息的现有技术，Healthy Together将很快宣布更多的州和企业客户使用该平台来保护居民和员工的健康。

如果您想了解更多信息，请联系 info@healthytogether.io 。

Sharetrace是一个健康护照和联系人追踪应用程序，它的设计是为了保护隐私。它建立在用户拥有的个人数据帐户上，开创了个人数据隐私技术，可以安全地使用敏感数据，而没有公司或政府的主权监控风险。Sharetrace是英国和美国大学的合作项目，包括俄亥俄州的凯斯西储大学。在sharetrace.org的了解更多在线信息。

如果您想了解更多信息，请联系Jonathan . Holt by @ data swift . io。

Coalition App是一款全球性的隐私设计联系人追踪应用程序，利用安全的蓝牙和加密技术通知用户潜在的新冠肺炎暴露。它是由Nodle.io和FireChat背后的团队以及一些独立人士创建的，使用开源的耳语追踪协议构建——由麻省理工学院、斯坦福大学、南加州大学和牛津大学的密码学家进行同行审查——最近被塞内加尔政府采用。法国政府正在利用联盟的蓝牙技术和专业知识。

如果你想了解更多信息，请联系micha@coalitionnetwork.org。

Safe2是一个新冠肺炎暴露警告系统，用于更智能的社交距离。该移动应用程序使用来自GPS和蓝牙技术的匿名数据，私下分享实时暴露警报，以帮助防止病毒的社区传播。Safe2由Jamison D. Day博士、数据科学家和灾难救援专家创建，拥有一支专注于全球健康、技术和危机管理的国际团队，专注于改善健康、经济福祉和隐私。

如果你想了解更多信息，请联系hello@safe2.org。

VIRI是一个由隐私和匿名精神驱动的联系追踪平台，其使命是允许跨实体联系追踪，而无需共享任何个人识别信息。它可以作为API无缝集成到现有的企业应用程序中，允许企业和机构在全球范围内兼容，同时让实体遵守各种医疗保健数据法规。VIRI部署了一个混合后端架构，利用了宽松的区块链技术。

如果您想了解更多信息，请联系 sumit@viri.io 。

症状跟踪器

COVID Near You是一款众包新冠肺炎症状跟踪器，由波士顿儿童医院创新和数字健康加速器内的流行病学家和软件开发人员创建。波士顿儿童医院团队拥有开发传染病监测平台的背景和专业知识，并在为公共卫生响应工作构建基于可视化的工具方面提供技术能力。COVID Near You团队旨在支持新冠肺炎的公共卫生监测措施，并使用自我报告的数据进行研究，以更好地了解这种疾病在北美的影响。

如果你想了解更多信息，请联系covidnearyou@healthmap.org的T4。

“我们的感受”让您可以自我报告您的年龄、性别、邮政编码和您经历的任何健康症状。这款应用是由一个名为“我们的感受项目”的独立非营利组织开发的。他们的技术团队包括Pinterest的首席执行官本·希伯尔曼，以及一个由Pinterest现任和前任员工组成的志愿者团体。他们正在与来自主要机构的科学家、医生和公共卫生专业人员合作，这些机构包括哈佛大学陈公共卫生学院、麻省理工学院麦戈文脑研究所、麻省理工学院和哈佛大学布罗德研究所、霍华德休斯医学研究所、宾夕法尼亚大学、斯坦福大学、马里兰大学医学院和魏茨曼科学研究所。

如果你想了解更多信息，请联系info@howwefeel.org。
