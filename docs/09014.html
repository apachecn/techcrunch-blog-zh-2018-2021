国土安全部警告 Windows 10 存在“蠕虫”漏洞

原文：https://web.archive.org/web/https://techcrunch.com/2020/06/08/homeland-security-windows-10-wormable-bug/

国土安全部警告 Windows 10 存在“蠕虫”漏洞

在上周网上公布了一个“可被蠕虫感染”的漏洞的利用代码后，国土安全部的网络安全咨询部门警告 Windows 10 用户确保他们的系统完全打了补丁。

这段代码利用了微软在三月份修补的一个安全漏洞。在“关键”级错误的细节最初被公布但很快下线后，这个错误引起了混乱和关注。

被称为 SMBGhost 的漏洞代码利用了服务器消息块(SMB)组件中的一个错误，该组件允许 Windows 与打印机和文件服务器等其他设备进行通信。一旦被利用，该漏洞会让攻击者自由访问 Windows 计算机，从互联网远程运行恶意代码，如恶意软件或勒索软件。

更糟糕的是，因为代码是“可蠕虫化的”，它可以在网络上传播，类似于的 NotPetya 和 WannaCry 勒索软件攻击如何在世界范围内传播，造成数十亿美元的损失。

尽管微软几个月前发布了一个补丁，但数万台面向互联网的计算机仍然容易受到攻击，因此发出了这一警告。

在公告中，国土安全部的网络安全和基础设施安全局表示，黑客正在使用新代码“瞄准未打补丁的系统”，并建议用户立即安装更新。

发布代码的研究人员是 GitHub 用户，他的昵称是 chompi 1337，他自己承认他们的概念验证代码“写得很快，需要一些工作才能更可靠”，但警告说，如果恶意使用这些代码，可能会造成相当大的损失。

“除了自我教育之外，把它用于任何目的都是一个非常糟糕的主意。你的电脑会着火。小狗会死，”研究人员说。

如果你最近没有更新 Windows，现在将是一个好时机。
