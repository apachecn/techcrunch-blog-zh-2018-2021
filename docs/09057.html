Pinwheel 是每个 fintech 和 neobank 都需要的收入验证 API 平台

原文：https://web.archive.org/web/https://techcrunch.com/2020/06/10/pinwheel-is-the-api-platform-for-income-verification-that-every-fintech-and-neobank-needs/

许多创始人开始建立一个想法，并在这个过程中，找到另一个更有吸引力的想法。或许最著名的例子是 Tiny Speck，这是一家位于相对无趣的 MMORPG 的游戏发行商，创始人对他们的内部团队沟通工具感到非常失望，以至于他们从游戏开发转向设计一款名为 Slack 的聊天应用。

这就是风车及其创始人库尔特·林、安尼施·巴苏和柯蒂斯·李的故事。李开复曾创立了代客服务公司 Luxe，于 2017 年将其出售给沃尔沃，他曾聘请林担任公司总经理。在一家大型汽车制造商的范围内进行了两年的“创新”后，两人准备好脱离并回到创业的开放世界。与此同时，李和巴苏之前曾在社交游戏公司 Zynga 合作过黑手党战争，并将整个团队联系在一起进行一个新项目。

最大的问题是建造什么。他们首先开发了一个软件平台，让公司能够轻松地为员工提供税前福利，比如将公交通行证费用化和为医疗储蓄账户提供资金。但是他们遇到了编程障碍:没有简单的方法将他们的产品连接到无数的工资单供应商。

“我们已经建立了一个集成到工资系统的内部平台…我们意识到，在我们建立的过程中，没有一种解决方案可以同时聚合和解锁工资系统的访问，这主要是因为它们非常古老，是一种封闭的系统，”林说。当他们与其他 fintech 创始人谈论如何解决这个障碍时，他们意识到缺乏一个可以用于他们产品的 API 实际上本身就是一个潜在的产品。

于是产生了 Pinwheel 的想法，这是一个用于工资数据的 API 层，处理从收入和员工验证到轻松切换和管理直接存款的一切事务。该公司今天正式走出秘密，并宣布已从首轮资本公司的乔希·科佩尔曼和前期风险投资公司的格雷格·贝蒂内利那里筹集了 700 万美元的种子资金，他们都将加入该公司的董事会。

像任何 API 平台一样，有一系列用户和数据需要连接。对消费者来说，最大的吸引力是自动直接存款控制，这将允许消费者控制他们的工资去向。例如，如果他们想将一笔直接存款分割成多个账户，或者定期将一部分薪水转移到 Digit 或 Acorns 等储蓄应用程序中，Pinwheel 可以帮助他们轻松做到这一点。

但真正有趣的用例开始来自该平台的其他金融科技企业用户。以抵押贷款为例。申请抵押贷款的过程非常艰难，需要大量文件来证明收入和就业状况。其中一些现在是数字化的——您可以使用类似公证这样的工具对文档进行数字签名——但几乎没有选项可以直接将工资单数据转换为适用于所有用例的统一的机器可读格式。“即使在现在的 2020 年，大多数人仍然必须在每次需要证实[他们的]数据时提交纸质工资单或税务文件，”林说。Pinwheel 希望成为驱动所有这些数据流的一层。

First Round 的 Kopelman 表示，正是抵押贷款和借贷等金融应用吸引了他加入该公司。他说:“你把四周内会发生的事情(纸风车)变成了四分钟。”。

他第一次了解这家公司是在他投资的一家公司的董事会上，并对此产生了兴趣。当他深入研究时，他喜欢创始人“极度关注用户”，他喜欢对工资的关注。“这是真理的源泉和资金的来源，”他说。他指出，第一轮投资已经投资了许多未来工作公司，如优步和 TaskRabbit，并亲眼看到了对轻松安全地访问工资数据的更好选择的需求。

根据 Pinwheel 的数据，82%的美国人通过直接存款获得报酬——这意味着绝大多数收入数据都存储在工资系统中。

Pinwheel 不是 ADP 等现有薪资提供商或 Gusto 等新贵公司的替代品。相反，它是在它们之上的一层，很大程度上就像，比方说，Plaid 是在现有银行之上的一层，为其他金融科技公司提供对用户银行数据的安全访问。

通过 Luxe，林和李以前建立了一个按需代客服务，我很好奇他们从那次经历中学到了什么，以及如何影响他们的方法。李开复说，最大的区别在于，当谈到建立有价值的公司时，理解什么可行，什么不可行。“某些业务只是有利于取得比其他业务更好的结果，”他说。他指出，与 Luxe 这样的按需业务相比，Pinwheel 的 API play 没有那么资本密集，边际成本有限，而且它是一个纯技术游戏，更容易为初创公司创造价值。

另一个变化是:团队从 Luxe 的总部旧金山搬到了 Pinwheel 的总部纽约。

在该公司目前的名单中，林担任首席执行官，巴苏担任首席技术官，李开复担任执行主席。Lee 也是总部位于纽约的 Primary Venture Partners 的风险合伙人。

除了第一轮和前期投资，Wonder Ventures 还参与了天使投资，如个人财务管理公司 Wealthfront 的前首席执行官亚当纳什(Adam Nash)和支付应用 Venmo 的前首席运营官迈克沃恩(Mike Vaughan)。
