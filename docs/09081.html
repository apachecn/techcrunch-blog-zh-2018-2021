Sentropy 从秘密行动中脱颖而出，推出了一个人工智能平台来解决在线滥用问题，该平台得到了 Initialized and more TechCrunch 1300 万美元的支持

原文：https://web.archive.org/web/https://techcrunch.com/2020/06/11/sentropy-emerges-from-stealth-with-an-ai-platform-to-tackle-online-abuse-backed-by-13m-from-initialized-and-more/

如今，社交媒体上多种形式的在线虐待受到了前所未有的关注:强烈和仇恨的语言不仅会直接伤害个人，而且会煽动仇恨团体和其他极端分子，在线下也激起更强烈(往往是悲剧性的)的反应。如今，一家名为 Sentropy 的初创公司认为，它已经开发出最复杂、但易于实现的系统，通过其基于人工智能的平台来帮助识别、跟踪并最终清除在线滥用，该公司正在着手解决这一问题。

Sentropy 背后的想法既是技术上的，也是哲学上的:与米歇尔·班克、泰勒·瑞恩和伊桑·布雷德共同创立这家初创公司的首席执行官约翰·雷德格雷夫说，这是关于使用他和他的联合创始人所知道的如何操纵和“读取”非结构化数据，将其应用于在线滥用，最终，真的需要认真对待。

“每周我都在观察网络对话对现实世界的影响，”他在一次采访中说。“他们产生了持久的影响。URL 和 IRL 之间有一个链接。对于年轻一代来说，他们的数字自我和物理自我之间的差异将无法区分。这很强大，但也有问题。”

这家初创公司正在摆脱隐形模式，有来自一系列非常知名的支持者的 1300 万美元资金——他们包括 Reddit 的联合创始人亚历克西斯·奥哈尼安，他的风险投资公司 Initialized Capital，King River Capital，Horizons Ventures 和 Playground Global，Riot Games，Nextdoor，OpenAI，Twitch，Twitter 的创始人和高管以及一位“前政府首脑”(其姓名未透露)。

尽管 Sentropy 今天才公开露面，但它实际上已经活跃了一段时间，自 6 月份以来，它的第一款产品——基于 API 的滥用检测引擎 Defend——一直在与许多科技公司合作，进行私下测试。这是一个普遍可用的产品，还有一个名为 Detect 的现成版本。Redgrave 不会透露迄今为止它一直在与哪些科技公司合作，尽管值得注意的是，直到本周，Ohanian 仍然是 Reddit 的董事会成员，并且可能会成为与其合作的明显战略关系，当然，Reddit 是这种平台面临 Sentropy 正在处理的那种滥用的一个主要例子。

“我亲眼目睹了人工管理网络社区的困难，”奥哈尼安在一份声明中说。“这个问题的广度和深度需要大量的资源和机器学习能力。Sentropy 已经建立了社交媒体社区急需的技术。对于那些愿意反对滥用的人来说，用户安全已经成为一个竞争优势。”

Sentropy 可能对其产品产生早期强烈兴趣的另一个原因是创始人的过往记录。他们之前的公司 Lattice Data 是更大的非结构化、黑暗数据领域的专家，尽管目的不同。它于 2017 年被苹果收购，目前团队中的几个人将在出售后继续在那里工作。Redgrave 表示，当 Lattice 还是一家初创公司时，他们实际上就已经开始确定将在线滥用作为非结构化数据问题来处理的想法，尽管这不是他们在那里开发的东西。

Sentropy 既作为基于 API 的产品出售，供开发人员实施和定制，也作为开箱即用的在线社区管理产品出售，分别名为 Detect 和 Defend。今天，这两种实现都是基于为人类提供工具，最终为他们各自的平台做出审核决定，尽管随着时间的推移，可能会添加一种服务来自动化更多的工作。

雷德格雷夫指出，如今在处理网络虐待以及试图阻止和缓和这种行为的任务方面存在许多问题。热门网站上的评论数量之多，使得人类分类的任务变得非常复杂，几乎是一项无法完成的任务(而且本身就有潜在的危害)。与此同时，当前许多自动标记关键词和短语的系统往往不够复杂，无法捕捉到正确的信息。该公司引用皮尤研究称，这导致问题略有缓解，但事实上约 75%的在线骚扰未被举报。

Rhyne 说，支撑这项服务的“大脑”是通过吸收来自开放网络、灰色网络和黑暗网络的许多许多页面的对话而构建的。正如你对人工智能系统的预期，Sentropy 还“学习”特定客户如何使用它来适应他们。反过来，这些知识也被输入到更大的大脑，继续教授它。(Sentropy 解析所有数据的方式使其自己的读取更加“智能”,这当然是这里的秘方。)

随着时间的推移，有很多问题和未开发的领域可能需要解决，即使它在这里取得了进展。首先，Sentropy 开始关注文本。目前，还没有解析音频或视频内容的产品，这些内容需要通过转录器才能被 Sentropy 的算法“读取”。随着时间的推移，鉴于视频应用的流行，以及 YouTube 等网站和脸书等其他平台上的视频的流行，这是一个它必须考虑的领域。

同样，对于一个网站可能希望围绕某些术语进行多少定制，以及在某个时候如何以及是否会被滥用，仍然存在疑问。举例来说，Rhyne 指出，目前，Sentropy 决定条款，因此它本身不能被滥用和用作政治审查的工具。当然，这提出了一些有趣的问题，关于什么是普遍的“对”和“错”。

“我们确实可以选择谁是我们的客户，”雷德格雷夫说。“这不是开源技术，他们必须与我们互动，我们将学习具体的用例。”
