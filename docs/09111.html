苹果增加了新的 MacBook Pro 显卡选项和 Mac Pro SSD 升级套件

在 WWDC 开幕前一周，苹果公司推出了一系列升级产品。从今天开始，16 英寸的 MacBook Pro 和 Mac Pro desktop 都将进行内部升级。

在笔记本电脑正面，16 英寸型号获得了额外的图形配置。AMD 镭龙 Pro 5600M GPU 的加入标志着笔记本电脑系列迄今为止最好的图形性能，比第二高配置镭龙 Pro 5500M 的速度提高了 75%。

价格也很匹配。8GB 内存的 5500M 比该系统 2799 美元的基础内存高出 100 美元。另一方面，8GB HB m2 内存的 5600 将多花 700 美元。但是，如果你要走这条线的顶端，你还不如真的走这条线的顶端，对不对？

一个完整的型号(不含任何额外的软件)将需要 6699 美元。显然，这是一个超高端的专业配置，对我们大多数人来说，这是一个疯狂的过度配置。另一方面，如果你经营一家从事视频制作、3D 设计或特效的工作室，并且想要真正适应未来的东西，它可能值得一看。

说到面向未来，苹果也刚刚推出了一项新功能，旨在让用户更容易升级 Mac Pro。新的 SSD 套件专为用户安装而设计，具有 1、2、4 和 8TB 存储模块。高端塔的升级说明可以在这里找到。

MacBook 更新目前可以通过苹果网站获得。与此同时，SSD 升级应该会在今天晚些时候到来。
