法国联系人追踪应用 StopCovid 已经被激活了 180 万次，但只发送了 14 条通知

原文：https://web.archive.org/web/https://techcrunch.com/2020/06/23/french-contact-tracing-app-stopcovid-has-been-activated-1-8-million-times-but-only-sent-14-notifications/

法国三周前发布了 StopCovid。政府今天将召开新闻发布会，对该应用程序进行更新。特别是，这次会议是分享一些数字的好机会。

在过去的三周里，有 190 万人在 App Store 或 Play Store 上下载了这款应用。在这 190 万人中，有 1，814，048 名用户打开了该应用程序并激活了 StopCovid。

您可以随时禁用 StopCovid 并删除您的个人数据；在过去的三周里，有 23，953 人停用了该应用程序，46 万人干脆卸载了该应用程序。

鉴于这款应用并不要求你登记你的姓名和地址，政府对谁在哪里使用这款应用知之甚少。政府将进行调查以了解更多信息。

仅在上周，就有 19 万人下载了这款应用，但有“数万”人卸载了它。政府表示，人们可能会觉得冠状病毒在法国的爆发已经结束——但每天仍有新的病例出现，所以这还远远没有结束。

“该应用程序运行良好。简而言之，我们可以检测到大多数我们应该检测的接触，”法国数字部长塞德里克 O 说。“如果我有一个遗憾，那显然是它与其他欧洲国家不兼容。”

下载是一回事。但是 StopCovid 并没有导致很多通知。只有 68 名 StopCovid 用户在应用程序中宣布他们被诊断为新冠肺炎阳性。服务器从这 68 个案例中接收了 205 次社交互动。

最后，StopCovid 在过去的三周里只发送了 14 条通知。

“这不是故事的结尾，”塞德里克·O 说。“我们不断改进应用程序。”

如果你正在使用 StopCovid，并且你被诊断为新冠肺炎阳性，你的医院或检测机构会给你一个二维码或一串字母和数字。你可以选择打开应用程序，输入代码来分享你在过去两周内互动过的人的临时 id 列表。

然后，服务器后端将所有这些短暂的 id 标记为属于可能暴露于冠状病毒的人。在服务器上，每个用户都与一个风险分值相关联。如果超过某个阈值，用户会收到通知。该应用程序然后建议你进行测试，并遵循官方指示。

除此之外，政府表示，该应用程序管理了非常少的通知，因为很少有人在应用程序中将自己标记为新冠肺炎阳性。这就是为什么在接下来的几周里，检测机构除了会给你一个二维码之外，还会给你发一封信。这样，你得到了另一个提醒。

在病例最多的法国地区(“橙色”和“红色”区域)，政府将更容易将自己标记为新冠肺炎阳性。例如，全科医生将能够直接告诉你在应用程序中标记自己。

“在目前的状态下，欧洲的应用程序彼此不兼容。如今，为了实现欧洲的互操作性，还有一些工作要做，但是意大利和德国的应用程序之间没有互操作性。

法国不依赖苹果和谷歌的联系人追踪 API。通过以德国和意大利的应用程序为例，Cédric O 希望打消你必须使用苹果和谷歌的联系人追踪 API 来构建欧洲联系人追踪应用程序的想法。即使你正在使用这个 API，你仍然需要调整你的应用程序，使其与其他应用程序一起工作。

但这并不意味着法国的联系人追踪应用程序将能够与其他欧洲应用程序进行对话。Inria 首席执行官 Bruno Sportisse 表示，法国开发团队正在努力。但是听起来 Inria 想要推动它的协议的一个新的迭代。
