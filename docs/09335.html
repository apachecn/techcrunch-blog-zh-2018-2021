塔吉特百货从中西部开始，在全国范围内推广生鲜食品

原文：https://web.archive.org/web/https://techcrunch.com/2020/06/25/target-is-rolling-out-fresh-grocery-pickup-nationwide-starting-with-the-midwest/

Target today 加入了越来越多的杂货店零售商的行列，为新鲜和冷冻商品提供路边和店内提货服务。这家零售商已经暂停了今年春天推出杂货提货的计划，理由是新冠肺炎疫情是一个因素。该公司表示，没有时间对员工进行新流程的培训。今天，这些计划又开始实施了。该公司现在表示，预计在几个月内，美国 1500 家商店都可以买到新鲜和冷冻食品。

新增的服务不是由 Target 的杂货配送服务 Shipt 提供支持，而是将利用商店员工来挑选和包装商品——就像他们今天为 Target 现有的订单取货和取货服务所做的那样。

然而，消费者将无法在发布会上获得 Target 的所有新鲜和冷冻食品。取件服务将提供 750 种新鲜和冷冻食品，此外还有数千种不易腐烂的食品。这包括农产品、乳制品、烘焙食品、肉类和冷冻产品。

在同一订单中，购物者还可以从超过 250，000 种商品中挑选，这些商品涵盖家居、服装、必需品等类别。

一位公司发言人告诉我们，将新鲜和冷冻产品的选择限制在 750 种的决定与该服务在早期测试中的使用方式有关。该公司发现，目标购物者大多在去商店的较大行程之间使用皮卡购买杂货必需品。

据我们所知，免下车服务和订单提货服务将提供相同的产品选择。

Target 表示，新鲜和冷冻商品将存放在商店前面提货区的温控仓库中，直到顾客前来提货。一些商店已经进行了改造，以适应这一计划中的变化。其他人需要一个小型建设项目来增加温控存储，因为冠状病毒的延迟影响了 T2 全面改造数百家商店的计划。

这项服务可以免费使用，不需要最低订购量或会员资格。此外，Target RedCard 的持有者和 Target Circle 的成员也可以在新的杂货项目上享受折扣。

继在双子城和堪萨斯城市场成功测试杂货提货后，Target 目前正在中西部地区推出新鲜和冷冻杂货。这将使这项服务在 400 家商店都可以使用。它预计在美国假日季前将达到 1，500 家商店。

这家零售商是目前在美国为沃尔玛提供提货服务的众多杂货店之一，在线杂货在其不断增长的电子商务销售中发挥了重要作用。在 2019 财年第四季度，在疫情之前，沃尔玛的电子商务销售额增长了 35% 。随着冠状病毒的爆发，越来越多的购物者开始在网上购物，并在沃尔玛销售点之外提货，这家零售商报告 Q1 的电子商务销售额增长了 74%。

与此同时，塔吉特百货称，由于疫情，其在 Q1 的当天送达服务也越来越受欢迎。甚至有几天，它的订单量是网络星期一的两倍。在复活节前的星期五，它通过 Shipt 的送货服务完成了比平常一周更多的量。

同样在 Q1，Target 的数字销售整体增长了 141%，而其当天服务(Shipt，Drive Up 和 Order Pickup)的组合增长了 278%。在使用免下车服务的数百万购物者中，40%的人是这项服务的新用户——这表明疫情已经改变了消费者的行为。

但是 Q1 对零售商来说并不全是好消息。它说，由于疫情对购物者需求的影响，它在劳动力上的支出增加了，销售的高利润商品减少了，并减记了卖不出去的服装和其他商品。该公司承诺，其在网上购物选择和员工方面的投资——例如，上周，T2 将其最低工资提高到每小时 15 美元——从长远来看是值得的。

“我们的履行选项的速度和便利性在全国范围内都是无与伦比的，对于在疫情期间寻找简单安全购物方式的客人来说，它们变得更加重要。塔吉特百货的首席运营官约翰·穆里根在一份声明中说:“通过在我们的客人已经喜欢的送货服务中增加新鲜食品，我们给了他们更多在塔吉特购物的理由。当更多的人寻找不同的方式来获得他们需要的物品时，我们将继续投资，使塔吉特成为最容易和最安全的购物场所，”他补充道。
