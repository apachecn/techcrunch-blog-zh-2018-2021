原文：https://web.archive.org/web/https://techcrunch.com/2020/07/01/match-group-iac-separaton/

Match Group 完成与 IAC 的分离，新董事会包括邓文迪和瑞安·雷诺兹

IAC 和 Match Group 宣布，他们已经完成了“完全分离”。

此前，Match Group(旗下拥有 Tinder、Hinge、OkCupid、PlentyOfFish 和 Match 本身)是一家上市公司，数字控股公司 IAC 是其大股东。去年，两家公司宣布了一项计划,将把 IAC 对 Match 的所有权分配给 IAC 的股东——这项计划于今天上午完成。

这种分离还涉及到领导层的变动，马克·斯坦和格雷格·威尼尔斯基从比赛小组委员会辞职。该公司有四名新的董事会成员:ExecOnline 首席执行官斯蒂芬·贝利，NBA 数字媒体执行总裁梅丽莎·布伦纳，投资者兼企业家邓文迪·默多克和演员瑞安·雷诺兹(也是美国航空杜松子酒和薄荷移动的所有者)。

雷诺兹在一份声明中说:“大多数千禧一代和 Z 世代都不记得在 Tinder、OkCupid 和 Hinge 出现之前的约会是什么样的。”“这些品牌有巨大的责任和机会来影响社会，同时拥抱新技术并保持在流行文化的前沿。我已经准备好卷起袖子，与团队一起为他们未来的成长和成功而努力。”

沙尔山·杜贝将继续担任 Match Group 的首席执行官，她在今年年初在担任这一职务，而乔伊·莱文仍然是 IAC 的首席执行官和 Match Group 的执行主席。

IAC 主席巴里·迪勒(Barry Diller)在一份声明中表示:“这只是我们 25 年来战略核心中最大的一笔交易。”“要投机取巧，保持资产负债表的保守，建立企业，当它们应该独立时，就让它们独立。成为联合企业和反联合企业，这是我们独有的商业模式。”
