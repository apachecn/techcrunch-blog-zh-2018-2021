特斯拉第二季度交付了 90，650 辆汽车，降幅小于预期

特斯拉周四表示，第二季度交付了 90，650 辆汽车，比去年同期下降了 4.8%，原因是新冠肺炎疫情引发的挑战，包括其美国主要工厂暂停生产数周。

尽管逆风，特斯拉仍设法超出预期。

绝大多数交付量(约 80，050 辆)是 Model 3 和 Model Y 汽车。其余 10，600 辆是其更昂贵的 S 型和 X 型汽车。特斯拉没有单独提供每款车型的明细，也没有提供地区交付信息。

分析师曾预计，由于新冠肺炎疫情，交付数量将会减少，他们的预测各不相同，一些人预测交付数量将少至 39，000 辆，多至 83，000 辆。FactSet 调查的分析师一致预计特斯拉第二季度将交付 72，000 辆汽车。

这一结果推动该公司股价周四早盘上涨近 9%，至 1，218.21 美元。

这里有一个快速分类:

  • 特斯拉在 Q2 交付了 90，650 辆汽车，而在 Q1 和 Q2 分别交付了 88，400 辆和 95，200 辆
  • 2019 年，特斯拉在 Q2 生产了 82，272 辆汽车，而在 Q1 和 Q2 分别生产了 103，000 辆和 87，048 辆

据为该公司工作的几名消息人士透露，在第二季度末之前的几周，新的销售目标被放在了员工身上。特斯拉还降低了其汽车在中国和北美的价格。这两项战略都旨在刺激对其电动汽车的需求。

该计划似乎在第二季度发挥了作用，这可能促使分析师对特斯拉今年剩余时间寄予更高的期望。

周三股市开盘后，特斯拉股票暴涨，推动公司市值接近 2080 亿美元，超过丰田成为全球市值最大的汽车制造商。股价上涨的部分原因是，据报道，首席执行官埃隆·马斯克(Elon Musk)在全公司范围内发了一封电子邮件，祝贺员工实现了销售和生产目标。
