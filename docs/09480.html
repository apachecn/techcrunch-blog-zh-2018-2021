DocuSign 以 3800 万美元收购 Liveoak Technologies 进行在线公证

即使在最好的情况下，找到一个公证人也是一个挑战。在疫情中部，这就更难了。今天，DocuSign 宣布以大约 3800 万美元的价格收购了 Liveoak Technologies ，这给了该公司一个在线公证的选择。

与此同时，DocuSign 宣布了一项名为 DocuSign 公证人的新产品，该产品允许公证人与电子签名一起在线进行，从而降低了公证人的要求。随着我们越来越深入疫情，像 DocuSign 这样允许工作流完全数字化的公司比以往任何时候都更受欢迎。这款新产品将于今年夏季晚些时候上市。

鉴于两家公司已经有了合作关系，这笔交易是有意义的。Liveoak 将现场视频、协作工具和身份验证结合在一起，使各方能够获得公证批准，就像您坐在公证人面前一样。

通常，您可能会收到一份需要您签名的文档。如果没有电子签名，你就需要打印出来，签署文件，扫描后返还。如果需要公证人，您需要在公证人在场的情况下签字，这需要亲自前往。所有这些都可以通过在线工作流程进行简化，DocuSign 在此次收购中提供了在线工作流程。

这就像是完美的疫情收购，将手工流程数字化，让人们不必在危险的时候进行面对面的交易。

Liveoak Technologies 成立于 2014 年，是德克萨斯州奥斯汀创业场景的一部分。根据 Crunchbase 的数据，该公司在其作为私营公司的生命周期中筹集了 1350 万美元。

此次收购是疫情不断增长的收购趋势的一部分，大型公共企业公司正在收购早期创业公司，在某些情况下价格相对低廉。最近的收购包括苹果收购 Fleetsmith 和 ServiceNow 上个月收购 Sweagle 。
