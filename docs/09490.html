Koyeb 是一家无服务器的初创公司，它与多家云提供商一起接收、处理和存储数据

来认识一下由 Yann Léger、douard Bonlieu 和 Bastien Chatelard 创建的法国新创业公司 Koyeb ，他们之前在 Scaleway 工作多年。Koyeb 是一个无服务器的初创公司，它可以帮助你以不同的方式操作数据，而不用担心你的服务器基础设施。

云服务提供商之间的竞争变得异常激烈，Koyeb 希望利用这一点。您可以将 Koyeb 与多个云服务提供商集成在一起，让 Koyeb 来完成繁重的工作。

例如，您可以在由 DigitalOcean 管理的对象存储桶中存储大量视频。假设您想要对这些视频进行重新编码，以便针对新设备进行优化。Koyeb 可以从这个桶中导入数据，重新编码这些视频，并将新文件上传到您的桶中。

但是 Koyeb 走得更远，它允许您混合和匹配服务和 API。随着云平台变得越来越智能，它们提供的服务不仅仅是为你运行服务器和存储数据。

例如，谷歌云的语音转文本 API 可以说比亚马逊转录更好。Koyeb 可以从 AWS S3 桶中获取视频文件，在谷歌云上转录这些视频文件的音频，并将结果保存在 AWS S3 桶中，而不必手动设置多云工作流。

Koyeb 有很多用例。从每天将数据从一个 S3 兼容的对象存储提供者复制到另一个对象存储提供者以获得冗余，到用 API 调用触发数据处理。一切都会自动扩展，一旦工作流完成，您就不再需要为运行时间付费。

已经有几十个集成与数据源(作为输入和输出)和现成的处理 API。使用命令行界面或 Koyeb API，一切都可以通过多个处理步骤在 web 界面中进行配置。

该公司刚刚从隐身状态中走出来，并已经在进行更多的产品更新。例如，将来您将能够使用 Docker 容器和自定义功能，这将支持更多的工作流。但这是一个充满希望的开始。
