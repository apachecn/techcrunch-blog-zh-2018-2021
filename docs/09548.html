脸书代码变更导致 Spotify、Pinterest 和 Waze 应用中断

如果你是一名 iPhone 用户，你很可能在早上花了一段时间试图重新打开应用程序，这很令人沮丧。我知道我的早晨散步因为无法启动 Spotify 而扫兴。许多其他用户报告了许多应用程序的类似问题，包括 Pinterest 和 Waze。

这个问题已经解决，脸书注意到这个问题牢牢地落在它的肩上。一个日志页面记录了几个小时前脸书 iOS SDK 产生的错误数量突然激增。脸书说，这个问题是代码变化造成的。

开发团队写道:“今天早些时候，一次代码更改导致一些使用脸书 SDK 的 iOS 应用程序崩溃。”。“我们很快发现并解决了问题。我们对由此带来的不便深表歉意。”

虽然这个问题相对迅速地得到了解决(尽管几个小时可能感觉像一辈子，这取决于你对某个给定应用的依赖程度)，但愤怒的各方无疑会记得五月的，当时 SDK 更新引发了类似的问题浪潮。毫无疑问，这个问题对于那些应用程序使用 SDK 的开发者来说更加令人苦恼。

在最近记忆中的第二个主要问题之后，很容易想象许多人会重新考虑他们与社交网络的关系——毕竟，糟糕的体验可能会让人们完全放弃一个应用程序，正如社交媒体围绕 Apple Music 和 Spotify 的辩论似乎指向今天早上。许多用户最终会将责任归咎于给定的应用程序，而不是导致崩溃的第三方 SDK。

谈到 SDK 的更新，脸书似乎动作太快，打破了太多东西。我们已经联系了该公司，看看它计划如何解决这个问题。
