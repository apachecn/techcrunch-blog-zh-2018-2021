App Annie 推出广告分析产品 Ascend

App Annie 建立在去年收购分析公司 Libring 的基础上，推出新版 Libring，更名为 App Annie Ascend 。

首席执行官 Ted Krantz 告诉我，虽然 Ascend 将出售给 App Annie 的现有客户，但真正的希望是进入“一个截然不同的市场”，因为 App Annie 不仅提供应用市场数据，还提供广告分析——特别是为游戏发行商和广告行业供应方的其他公司。

Krantz 认为，随着移动平台和浏览器对用户跟踪增加了更多限制(最近苹果宣布将给予用户拒绝应用广告跟踪的能力)，对广告商来说，“这个空间将变得非常黑暗”——这为 App Annie 将更广泛的市场数据与出版商自己的第一方数据相结合的方法创造了机会。

为了实现这一目标，Ascend 提供了 Krantz 所说的“数百个连接器”，用于从 AdColony、Unity 和 Chartboost 等不同平台获取数据，使客户能够“并排”查看这些数据集。Krantz 强调，这些数据在很大程度上是供客户自己使用的，并将与 App Annie 更大的市场数据“隔离”，至少在最初是这样。

“随着时间的推移，我们有能力开放基准数据，”他说，并补充说这种方法是 Ascend 独一无二的一部分:“你必须从你的同行群体中获得基准数据。没有市场数据，你无法确定自己的判断是正确的。”

已经在使用 Ascend 的公司包括 Reddit 和 Jam City。

Reddit 的营销总监 Spiros Christakopoulos 在一份声明中说:“Ascend 消除了集成、维护和不断更新数十个 API 的负担，使我们能够专注于重要的事情:实现我们的 KPI 和改进我们的活动。”“Ascend 通过其精心设计的报告工具提供的关键见解的 Thanks，已经成为我们营销分析基础设施的重要组成部分。”
