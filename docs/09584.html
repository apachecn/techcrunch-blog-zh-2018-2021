浏览器公司 Chrome 的竞争对手悄悄筹集了 500 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2020/07/14/chrome-competitor-the-browser-company-quietly-raises-5m/

一些硅谷知名人士支持一家软件初创公司，希望挑战谷歌 Chrome 的主导地位。

这家名为浏览器公司的初创公司由约书亚·米勒(Joshua Miller)领导，他曾担任奥巴马白宫的产品总监，目前是乔什·库什纳(Josh Kushner)创办的投资公司 Thrive Capital 的投资者。

一位消息人士告诉 TechCrunch，这家纽约初创公司已经筹集了超过 500 万美元的资金。该公司的支持者包括 LinkedIn 的杰夫·韦纳、Medium 的 Ev Williams、Figma 的迪伦·菲尔德、opinion 的阿克谢·科塔里和 GitHub 的杰森·华纳。

这家初创公司在公开场合一直对他们到底在做什么含糊其辞。他们正在开发一种新的浏览器，这种浏览器似乎拒绝了最基本的简单性，而是采用了一些现代网络应用程序更灵活的界面。浏览器的后端部分建立在 Chrome 的基础上，利用开源的 Chrome，这使得这款新贵产品在推出时能够无缝支持更广泛的网络标准。

“我们热爱互联网，但它可能会让人不知所措，”这家初创公司在的网站上写道。“如果浏览器可以帮助我们理解这一切，会怎么样？”

在电话中，米勒并没有更多地说明最终版本到底会是什么样子。

米勒在回答一个关于产品能力的问题时说:“我会谨慎一点，因为我们的竞争对手比我们拥有更多的工程师和资金。”

浏览器公司的六人团队并不是唯一一家旨在挑战 Chrome 浏览器市场一刀切做法的年轻初创公司。为了获得更多信息，我深入研究了投资者正在支持的一些年轻的浏览器初创公司。(需订阅。 )

谷歌的 Chrome 平板电脑主导着浏览器市场。2016 年，谷歌详细说明了该应用程序的活跃安装量约为 20 亿次。从那以后，随着火狐和 Internet Explorer 等竞争对手的用户大幅减少，这款产品只是巩固了它的领先地位。

谷歌努力打造一个适合全球数十亿人使用的 Chrome 版本，这导致了一个安全的产品，米勒表示，该产品对人们应该如何使用它不是非常“固执己见”。他说，这家浏览器公司的目标不是取代 Chrome，而是寻找 Chrome 用户的子集，以便更好地满足他们的需求。

“我认为网络浏览器在功能上停滞不前的一个原因是商业模式是建立在搜索广告收入之上的，”米勒说。“我认为 Chrome 和 Safari 是丰田或本田。它们可靠、经济实惠、易于使用且简单。我们试图打造网络浏览器的特斯拉。”

米勒说，浏览器公司希望今年晚些时候开始让用户测试该软件。
