投资者正在浏览 Chromium 初创公司

原文：https://web.archive.org/web/https://techcrunch.com/2020/07/14/investors-are-browsing-for-chromium-startups/?guccounter=1

几个月前，我们宣布“浏览器再次变得有趣，这要归功于主要玩家之间竞争的加剧。现在，随着越来越多的创业公司加入进来，事情变得越来越令人兴奋。

为数不多但越来越多的项目正在构建针对特定类型用户的 web 浏览器。无论被感知的用户是优先考虑提高速度、组织还是与他们的工作流程一致的工具集，企业家们都是在这样的假设下建立这些项目的，即谷歌对 Chrome 一刀切的做法给大量用户留下了次优的体验。

对于一家小型创业公司来说，从头开始构建一个现代网络浏览器并不是最可行的挑战。幸运的是，开源项目使开发者能够在他们想要与之竞争的应用的基础上构建他们进化的网络浏览器。对于不是 Safari、Firefox、Chrome 或少数其他浏览器的浏览器来说，谷歌的 Chromium 开源项目已被证明是一笔无价的资产。

自谷歌 2008 年末首次发布 Chrome 以来，该公司也一直在更新 Chrome。源代码支持的微软 Edge 和 Opera 网络浏览器，但也允许较小的开发团队在构建自己的应用时利用 Chrome 的力量。

这些新贵浏览器通常寻求在隐私方面与主导力量竞争，但随着 Chrome 和 Safari 开始推出更多功能来帮助用户管理他们在网上被跟踪的方式，企业家们正在扩大他们的产品雄心，以解决可用性升级问题。

投资者对定制浏览器越来越多的关注有助于实现这些雄心。Mozilla 联合创始人 Brendan Eich 的 Brave 继续扩大规模，上个月宣布他们以隐私为中心的浏览器拥有 500 万每日活跃用户。

今天，Thrive Capital 的 Josh Miller 与 TechCrunch 谈论了他的项目浏览器公司，该公司已经从一些著名的硅谷运营商那里筹集了 500 万美元。其他热门新贵的努力包括 Mixpanel 创始人 Suhail Doshi 的基于订阅的远程流媒体 Chrome 初创公司 Mighty，以及最近加入的 Blue Link Labs，该公司正在开发一款名为 Beaker browser 的分散式点对点浏览器。
