OpenCV AI Kit 旨在为计算机视觉做 Raspberry Pi 为业余硬件爱好者做的事情 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/07/14/opencv-ai-kit-aims-to-do-for-computer-vision-what-raspberry-pi-did-for-hobbyist-hardware/

一个名为 OpenCV AI Kit 或 OAK 的新工具，旨在复制 Raspberry Pi 和其他最小计算解决方案的成功，但用于不断增长的计算机视觉和 3D 感知领域。其新的多摄像头印刷电路板将大量功能打包到一个小型的开源单元中，现在正在 Kickstarter 上寻求资金。

OAK 设备使用它们的摄像头和板载人工智能芯片来执行许多计算机视觉任务，如识别物体、计算人数、查找帧中物体之间的距离等。这些信息以完善的、现成的形式发送出去。

拥有这样一个可靠、低成本、低功耗的计算机视觉单元，对于任何希望构建智能设备或机器人的人来说都是一个巨大的福音，否则可能需要几个独立的摄像头和其他芯片(更不用说摆弄软件了)。

图片来源:卢克索尼斯

像 Raspberry Pi 一样，它已经成为涉足硬件的业余程序员的首选，几乎所有关于这些设备的东西都是在许可的 MIT 许可下开源的。它正式隶属于 OpenCV，OpenCV 是计算机视觉领域广泛使用的一组库和标准。

实际设备和车载人工智能是由 Luxonis 创建的，该公司之前创建了通勤卫士，这是一种智能自行车刹车灯，可以实时跟踪物体，以便警告骑车人。该团队找不到任何符合要求的硬件，所以他们自己制作了硬件，然后与 OpenCV 合作制作了 OAK 系列作为后续产品。

实际上有两个版本:超小型 OAK-1 和三摄像头 OAK-D。它们共享许多组件，但 OAK-D 的多个摄像头单元意味着它可以实现真正的立体 3D 视觉，而不是依赖于普通 RGB 图像中的其他线索——这些技术现在比以往任何时候都更好，但真正的立体仍然是一大优势。(如果你想知道的话，人类视觉系统两者都用。)

两个橡木设备，与世界上最丑陋的季度规模。图片来源:卢克索尼斯

这个想法是将计算机视觉系统单元化，因此不需要构建或配置它，这有助于更快地启动许多项目。您可以使用现成的内置对象和深度检测，或者挑选您想要的元数据，并使用它来增强您自己对 4K(外加两张 720p)图像的分析。

极低的功耗也有帮助。计算机视觉任务对处理器的要求相当高，因此会消耗大量的能量，这就是像 XNOR 的超高效芯片这样的设备如此有前途的原因(也是该公司被苹果公司抢购的原因)。OAK 设备不会将事情发展到 XNOR 极限，但由于最大功率只有几瓦，它们可以依靠正常大小的电池连续运行几天或几周，这取决于它们的任务。
