亚马逊影响者计划向直播流媒体开放，以便向亚马逊直播 TechCrunch 进行广播

原文：https://web.archive.org/web/https://techcrunch.com/2020/07/15/amazon-influencer-program-opens-to-livestreamers-for-broadcasting-to-amazon-live/

亚马逊为直播流媒体提供了一种新的方式来赚取购买其流媒体中展示的产品的佣金。该公司今天在其现有的亚马逊影响者计划中增加了直播流，在今天之前，该计划允许社交媒体影响者通过在脸书、推特、Instagram 和 YouTube 上的帖子将粉丝引向他们最喜爱的亚马逊产品来赚钱。

影响者计划于 2017 年悄悄亮相，作为亚马逊利用影响者营销这一日益增长的趋势来推动销售的一种方式。该计划本身是亚马逊联盟计划的一个升级，因为它需要批准才能加入，并为有影响力的人提供他们自己的页面，其中包含亚马逊网址，以展示他们的建议。

尽管亚马逊已经通过该计划迎合视频创作者，新的直播选项专注于其自己的亚马逊直播服务。亚马逊 Live 是亚马逊购物网站上直接播放的现代版 QVC，去年推出了，作为零售商通过视频直播吸引消费者的最新努力。

在亚马逊直播秀上，主持人谈论并展示产品，就像他们在家庭购物网上做的一样。在视频下方，一个旋转木马引导消费者购买特色商品。佣金是根据他们销售的产品支付的。

这项服务并不是亚马逊在直播内容方面的第一次尝试——这家零售商终止了其早期在直播内容方面的努力，这是一个名为 Style Code Live 的短命“节目”,由具有电视和广播背景的主持人主持，他们邀请专家来谈论美容和时尚技巧。

然而，Amazon Live 并不局限于时尚和美容。相反，它的内容可以涵盖一系列类别——如烹饪、健身、婴儿、家居、汽车、电子产品、玩具、宠物、运动和电视、工业等等。与有线电视购物网不同，还有多个直播频道可供浏览。

为了向 Amazon Live 直播，视频创作者和现在的影响者使用 Amazon Live Creator 应用程序进行直播，并在观众展示要购买的产品时与他们聊天。在亚马逊 Live 主页上，粉丝还可以在直播视频旁边的类似 Twitch 的侧面板中与主持人和其他人聊天。

你可以看到一些有影响力的人在行动，早期采用者 Mirror & Thread 、Carla 的 Beauty、 The Deal Guy 和 BrickinNick 已经在 Amazon Live 上可用，因为这项服务已经测试了几个月。

此外，作为新计划的一部分，在亚马逊直播上直播的影响者不仅可以在亚马逊直播主页上播放他们的视频，还可以在他们自己的专用亚马逊店面上播放。随着粉丝群的扩大，他们可以从“明日之星”上升到“圈内人”再到“一线明星”。

图片来源:亚马逊

这些层级有不同的奖励和功能。例如，后起之秀通过亚马逊的 Onsite Associates 计划为符合条件的购买提供支付佣金，而更高级别的人除了自己的店面和亚马逊 Live 之外，还可以在产品详情页面上展示他们的视频。亚马逊表示，一线明星还能获得优先支持，并能特别参与亚马逊直播活动和机会。

“我们专注于为客户带来有趣和互动的购物体验，同时帮助有影响力的人在亚马逊上发展他们的业务，”亚马逊直播总监 Munira Rahemtulla 在一份声明中说。“直播可以实现创造力、联系和灵感，机会是无限的——我们很高兴向有影响力的人介绍亚马逊 Live Creator 应用程序，迫不及待地想看看他们将为亚马逊客户创造什么，”她补充道。

TechCrunch 要求亚马逊澄清影响者如何从他们的流中获得报酬，因为今天的社交媒体人物有许多方式与品牌合作盈利——例如，包括通过 YouTube BrandConnect ，脸书的品牌合作和其他项目。亚马逊表示，它根据 Associates Program Standard 收费标准向直播用户支付费用，例如，不同类别的佣金从 1%到高达 10%不等(如奢侈品美容)。
