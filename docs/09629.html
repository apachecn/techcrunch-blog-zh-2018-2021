生成算法正在重新定义软件和音乐的交集

如果你可以混合搭配你最喜欢的艺术家的不同曲目，或者用他们的声音自己创作新的曲目，会怎么样？

这迟早会成为现实，因为类似于用于创建计算机生成的艺术图像和在视频中嵌入深度赝品的人工智能模型正越来越多地应用于音乐。

使用算法来创作音乐并不新鲜。早在 20 世纪 50 年代，研究人员就使用计算机程序生成钢琴曲，那个时代的音乐家如亚尼斯·克塞纳基斯和戈特弗里德·柯尼希甚至用它们来创作自己的音乐。

发生变化的是生成算法的改进，这种算法在 2014 年首次受到欢迎，加上大量的计算能力，越来越多地改变了计算机今天对音乐的处理。

OpenAI 最近发布了一个名为 JukeBox 的项目，该项目使用复杂的原始音频形式，根据个人对流派、艺术家和歌词的选择，帮助创建全新的音乐曲目。与此同时，亚马逊的 AWS DeepComposer 和谷歌 Magenta 项目发布的工具正在帮助开发人员尝试深度学习算法和音乐。

在商业用途方面，让用户创作定制的、免版税的音乐的初创公司，如 Amper Music ，正看到企业在围绕视频背景音乐的一系列用例中采用计算机生成的音乐，唱片公司也开始播放人工智能创作的音乐。

随着计算机生成音乐的技术和质量日益成熟，它可能会给媒体行业带来很多变化，从个体艺术家到唱片公司再到音乐流媒体公司，并带来一系列关于计算机生成音乐权利的法律问题。
