巴基斯坦向抖音发出“最后警告”，并因“不道德、淫秽和粗俗的内容”屏蔽 Bigo 应用程序

原文：https://web.archive.org/web/https://techcrunch.com/2020/07/21/pakistan-issues-final-warning-to-tiktok-and-blocks-bigo-app-over-immoral-obscene-and-vulgar-content/

巴基斯坦向抖音发出“最后警告”,并以“不道德、淫秽和低俗内容”为由屏蔽 Bigo 应用

抖音在另一个国家受到攻击。

巴基斯坦屏蔽了 Bigo 直播应用程序，并向字节跳动抖音发出“最后警告”,称该应用程序中存在“不道德、淫秽和粗俗的内容”。

国家电信管理局表示，已收到来自社会各界的大量投诉，内容涉及在 Bigo 和字节跳动应用上流传的视频的性质，以及它如何影响“整个社会，尤其是年轻人”

巴基斯坦电信管理局在一份声明中表示，它最近向字节跳动和运营 Bigo 应用程序的 Bigo Technologies 转达了这些担忧，并敦促它积极主动地“在法律和道德范围内调节社交和内容”但这些应用的反应并不令人满意。

“因此，在行使 PECA(国家预防电子犯罪法案)赋予的权力时，PTA 决定立即阻止 Bigo，并向抖音发出最终警告，要求其建立一个全面的机制来控制其社交媒体应用程序中的淫秽、粗俗和不道德内容。”

在给 TechCrunch 的一份声明中，字节跳动发言人表示，该公司部署了一系列技术和调节策略来检测和审查违反我们使用条款的问题内容。2019 年下半年，抖音从巴删除了 3，728，162 个用户视频。

“我们致力于进一步加强我们的保障措施，以确保我们用户的安全，同时增加我们与当局的对话，以解释我们的政策，并展示我们对用户安全的承诺，”该发言人说。

几周前，邻国印度出于网络安全考虑，禁止了中国公司开发的抖音、Bigo 和其他 57 款应用。

在禁令之前，抖音将印度确定为其在中国以外最大的市场，印度已经积累了超过 2 亿的月活跃用户。总部位于卡拉奇的叫车初创公司Bykea 的高管 Danish Khalid 说，和在印度一样，抖音在巴基斯坦也非常受欢迎。

根据移动洞察公司 Sensor Tower 的数据，去年抖音是巴基斯坦下载量最大的应用，巴基斯坦有大约 4000 万互联网用户。

一些积极分子谴责巴基斯坦对抖音的警告和对 Bigo Live 的封锁，称这一举动是巴基斯坦试图“测试他们能在审查上走多远”

本周早些时候，巴基斯坦也暂时禁止了流行的手机游戏 PUBG，因为担心该国的年轻人在这款“上瘾”的应用程序上“浪费时间”。

今年早些时候，巴基斯坦政府公布了一些世界上最全面的互联网审查规则，这些规则将严重影响在该国运营的美国科技公司。但是在脸书、谷歌和推特以及其他公司威胁要离开这个国家后，T2 撤回了规定。

字节跳动的招牌应用抖音在美国也面临着紧张局势。国务卿迈克·蓬佩奥本月早些时候表示，美国【肯定会考虑】禁止抖音，因为担心它可能被中国政府用作监视和宣传工具。
