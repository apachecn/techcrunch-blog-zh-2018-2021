给你的 Mac 增加一个外部 GPU 可能是比获得一个新的更好的升级选择

原文：https://web.archive.org/web/https://techcrunch.com/2020/07/22/adding-an-external-gpu-to-your-mac-is-probably-a-better-upgrade-option-than-getting-a-new-one/amp/

图片来源:雷蛇

苹果最近宣布，他们将把 Mac 系列从英特尔处理器转移到他们自己的基于 ARM 的苹果芯片上。这一过程将从今年晚些时候发布的硬件开始，根据苹果公司的预期，将持续两年。虽然新的英特尔驱动的 MAC 电脑将在那个时候发布和销售，但这确实意味着基于英特尔的苹果硬件的写作是在墙上。然而，在过渡完成后的很长一段时间内，现有的英特尔芯片 MAC 仍然有用，软件移植意味着在可预见的未来，它们甚至可能支持更多你现有的最喜爱的应用程序，这就是为什么添加外部 GPU (eGPU)现在可能比以往任何时候都更有意义。

苹果公司几年前增加了对 eGPUs 的支持，这是由于在 MAC 电脑上增加了雷电 3 端口。这些具有非常高的吞吐量，使内部机箱中的 GPU 能够提供几乎与内部连接的 GPU 一样多的图形处理能力。但是，尽管苹果已经直接销售了一些 eGPUs，并且原生支持 AMD 显卡，而不需要任何特殊的驱动程序，但它仍然主要是一个利基类别。但是，对于任何希望将现有 Mac 的寿命延长几年，以等待和观察苹果芯片过渡如何实现的人来说，来自苹果和关键软件合作伙伴的更新使 eGPU 成为一个很好的选择。

这里有几个雷电 3 eGPU 机箱选项，供考虑升级的用户选择，以及每个选项的相对优点。请记住，对于其中的每一个，价格都是针对机箱的—您必须添加自己的 eGPU 才能使其工作。但好消息是，随着图形技术的进步，你可以不断升级和更换这些显卡，给你的 Mac 带来更大的提升。

Razer Core X Chroma(399 美元)

图片来源:雷蛇

Razer Core X Chroma 是 Razer 的顶级 GPU 外壳，它支持宽达三个插槽的全尺寸 PCIe 显卡，最大功率可达 500 瓦。集成电源提供 700 瓦的功率，支持 100 瓦的输出，可为任何连接的笔记本电脑充电。在 eGPU 的背面，您会发现四个额外的高速 USB 端口，以及一个用于联网的千兆以太网端口。色度版本还配有可调 LED 照明，可提供额外的用户定制选项。Razer 为我提供了一台 Core X Chrome、一台 AMD 镭龙 RX 5700 XT 和一台 Nvidia GeForce RTX 2080 Ti，用于测试 Mac 和 PC 系统。

这不是市面上最小的机箱，但部分原因是它支持三插槽卡，这在许多竞争产品中是首屈一指的。它也相对较短和较长，这使得它成为藏在桌下的一个很好的选择，甚至可以放在桌下的支架上(有足够的间隙让风扇排气正常工作)。它在运行时很安静，只有当内部的 GPU 实际上为兼容软件工作时，才会发出任何可以听到的噪音。

我的大部分测试侧重于在 Mac 上使用 Razer Core X Chroma，为了这种使用，你需要坚持使用 AMD 的 GPU，因为苹果并不支持 macOS 中的 Nvidia 显卡。然而，AMD 镭龙 RX 5700 XT 是一个庞然大物，它为改善照片和视频编辑等活动提供了充足的马力，并为您提供了额外的显示输出选项，并为系统提供了额外的资源。

得益于 Adobe 为其 Lightroom、Photoshop 和 Premiere 产品添加 eGPU 支持的工作，你可以在所有这些应用程序中获得大量的整体渲染和输出改进，特别是如果你在一台只有集成 GPU 的 Mac 上。苹果自己的应用程序也是如此，包括 Final Cut Pro X。

根据我的经验，使用 eGPU 极大地改进了 Adobe 和苹果 pro 视频编辑软件的导出功能，将导出时间至少缩短了一半。总体而言，在 Lightroom 中工作速度更快，响应速度更快，缩略图和预览的渲染时间大幅减少，而在我的 2018 Mac mini 上，这通常需要很长时间。

苹果还使用 eGPUs 来加速任何使用 Metal、OpenGL 和 OpenCL 的应用程序的性能，这就是为什么当你插入一个应用程序时，你可能会注意到系统性能的细微改善。很难量化这种影响，但整体系统性能感觉不那么迟缓，响应更快，尤其是在同时运行大量应用程序时。

Razer Core X Chrome 的额外扩展槽、安静的操作和最大功率传输都使其成为您寻找一个处理所有需要的东西的外壳的首选，并且它可以为 MAC 和 Windows PCs 提供很大的冲击——如果您碰巧使用两个平台，两者可以互换。

阿基米德泰坦(329 美元)

图片来源: OWC

如果你想花更少的钱，得到一个更简单但仍能提供出色性能的机箱，请查看aktio Node Titan。机箱制造商 Akitio 被 OWC 收购，后者是一家受欢迎的 Mac 外设制造商和销售商，几十年来一直提供第三方内存、坞站、驱动器等。Node Titan 是他们的高端 eGPU 机箱。

Node Titan 的外壳比 Razer Core X 的略小，并以类似太空灰的颜色完成，这将与苹果的 Mac 笔记本电脑更接近。较小尺寸的代价是它只支持双槽显卡，但它也具有集成的弹出式手柄，这使它更加方便，再加上它更轻、更紧凑的设计，可以随身携带。

Akitio 的 Node Titan 封装了 650w 的电源，这对于高消耗的显卡来说很好，但这也意味着这种情况下相对于 Core X Chrome 的另一个妥协是，Titan 只能向任何连接的笔记本电脑提供 85w 的输出。这低于最新的 16 英寸 MacBook Pro 全速充电所需的 96W，但仍足以让你的笔记本电脑保持供电，并为苹果 Mac 笔记本系列的其他产品提供全速充电。

Node Titan 还在外壳上只提供了一个端口，即用于连接到计算机的 Thunderbolt 输出。不过，配合它使用的显卡会提供自己的显示器连接，用于连接外部显示器。

在性能方面，Akitio Node Titan 在连接时提供了与 AMD 镭龙 RX 5700 XT 相同的潜在收益(以及 AMD 和 Nvidia 卡用于 PC)，因为 GPU 规格是处理机箱时最重要的。它的运行噪音有点大，特别是在发出安静但仍然可以察觉的持续嗡嗡声方面，即使 GPU 没有被占用。

然而，Node Titan 仍然是一个很好的选择，对于那些以最大笔记本功率输出和大量方便的端口扩展为代价寻求更高便携性和更低价格的人来说，它可能是一个更好的选择。

结果

当越来越多的苹果电脑选择用户可扩展内存时，这是一个从外部机器中挤出一点生命，让一台缓慢的机器感觉更快的好方法。现在，苹果的阵容中只有少数几款 MAC 使你的内存升级变得容易甚至可能。添加一个 eGPU 可以产生类似的效果，特别是如果你花很多时间在创意编辑应用程序上，包括 Adobe 的 suite、苹果的 Pro 应用程序或各种其他第三方应用程序，包括 DaVinci Resolve。

eGPU 设置的总价格，包括卡，可以接近甚至匹配新 Mac 的价格，但即使更便宜的卡也有很大的好处，并且您可以根据需要随时更换。值得注意的是，eGPU 在 Apple Silicon Macs 上的未来支持并不确定，尽管苹果已经表示他们将支持 Thunderbolt。尽管如此，eGPU 可以推迟几年的升级需求，更容易等待和观察过程转换对 Mac 用户的真正意义。
