True Link斥资3500万美元为老年人、残疾人和康复中的消费者量身定制金融服务TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/07/28/true-link-taps-35m-for-financial-services-tailored-for-elderly-disabled-and-recovering-consumers/

许多技术都是建立在服务能够瞄准最广泛或最有利可图的用户群的前提下的(当他们能够做到这两者时，他们就会一鸣惊人)。但这忽略了一部分消费者，他们因为各种各样的原因，无论是身体、年龄、财务还是其他情况，都落入了边缘。如今，一家专门针对三个通常被边缘化的市场——老年人、残疾人和正在戒毒的人——建立金融服务的初创公司宣布，在其业务强劲增长的背景下，将提供一些资金，并计划提供更多资金。

True Link Financial提供针对这三类人群的金融服务——具体来说，目前是以预付Visa卡和投资管理的形式——宣布已完成3500万美元的B轮融资，由Khosla Ventures领投，Centana Growth Partners是一家专注于金融服务的投资者。据我们所知，现在的估值约为1.15亿美元。

与克莱尔·麦克唐奈(Claire McDonnell，首席运营官饰)共同创立该公司的首席执行官凯·斯廷奇科姆(Kai Stinchcombe)表示，计划将把资金投入到增加更多产品的组合中，保险(人寿保险)可能是下一个目标，以及更多基于信贷的服务。

有趣的是，大约14个月前，该公司从科斯拉那里筹集了2100万美元的小型B轮融资——当时就开始向TechCrunch宣传这一消息。

Stinchcombe笑着说:“我们大部分时间都在写新闻稿，但后来就没时间了。”。与此同时，该公司的收入翻了一番，然后Centana出现了。由于专注于金融服务，它是True Link非常想加入的投资者，因此它重启了这轮融资，完成了交易，并最终宣布了这一消息。

事实上，Centana发现了为老年人提供服务的机会。

Centana Growth Partners的负责人汤姆·戴维斯在一份声明中表示:“在人们最脆弱的时候，True Link正在从根本上改变他们的财务生活。”。“未来20年，65岁及以上的美国人可能会翻一番，达到8000多万，幸运的是，这个群体比以往任何时候都活得更长，也更独立。在他们最需要支持的时候，任何人都不应该被利用，客户和家人一直在谈论True Link如何使他们保持财务自由。我们很高兴与True Link合作，并投资于他们的使命，为这个市场带来独特的金融产品。”戴维斯将与科斯拉的大卫·韦登一起加入董事会。

具有讽刺意味的是，技术的扩展和日益成熟带来的个性化提升并没有渗透到为潜在客户的长尾构建的更多服务中，所以当你遇到一家正在抓住这一机会的初创公司时，总是令人耳目一新。在True Link的案例中，这家初创公司(成立于2012年)采用了相同的产品，但考虑到特定的用户群，专门对它们进行了调整。

面向弱势老年人的“真正联系卡”是基于这样一种理念，即给予老年人一定的金钱自主权，但另一个人可以看到这笔钱花在哪里和如何花，或者只是能够监督这笔钱花在哪里。它还使老年用户更容易将他们的卡交给护理人员，以代表他们进行购买，而无需跟踪这些卡是否完全按照要求使用(因为有限制)。事实上，Stinchcombe的祖父母在一起欺诈和财务滥用案件中损失金钱的经历促使他创办了这家公司。

对于处于康复期的人来说，真正的联系卡是基于这样一个想法，即处于这种情况下的人通常会进行长期的斗争，以避免重蹈覆辙。这既可以定制为在特定类型的企业(如赌场或酒吧)限制购买，也可以增加取钱的难度。“这里的想法是，如果你在旅途中旧病复发，有一项交易是可以避免的，”他说。"我们的目标是提高壁垒，为复发提供又一个路障."

为残疾人提供的服务旨在提供第三方管理人和客户之间的联系，以便前者拥有更好的记录工具，这对于福利和地方当局确保一个人得到良好照顾是必要的。类似地，那些有认知或其他行为问题的人仍然可以保持一定的财务独立，并学习更多，但如果出现问题，可以有更好的控制。Stinchcombe说，对许多人来说，这代表着他们第一次获得了那种自主权。该公司目前只在美国开展活动，True Link表示，它与州政府管理的ABLE Act项目中近一半的参与者合作。

虽然有些人可能会说“边缘”意味着“小”，但无论是在美国还是在更远的地方，都仍然有很大的机会(Stinchcombe说，True Link可能会与合作伙伴合作推出)。

“True Link拥有一个充满激情的团队，一个庞大而服务不足的市场，以及一个成熟的产品，随时可以扩大规模，”Weiden在一份声明中说。“我们很高兴与他们合作，改善数百万人的财务灵活性和尊严。”
