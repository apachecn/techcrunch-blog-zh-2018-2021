制造改善囚犯生活的产品

我们这些在技术领域工作的人应该经常问自己，“我们到底在为谁建造？”我们设计产品是为了让自己更舒适，还是为了改变我们希望看到的世界而创新？美国监狱系统中的 230 万人是一个长期被排除在技术对话之外的群体，他们被移出了人们的视野和思维。随着技术成为世界进步的关键驱动力，我们应该制造改善囚犯生活的产品，帮助他们在没有复发风险的情况下重新融入社会。

最近，我偶然发现了我在斯坦福刑事司法中心工作后写的一篇文章，分析了挪威人道的监狱系统，并问“它们能在这里工作吗？”这些监狱被设计用来复制墙外的生活。它们融合了瑜伽课和录音室等特色。它们给囚犯提供了接受高等教育的机会，以便他们在重新进入外面的世界时能够有意义地就业。看过纪录片13 号的人都知道，美国的监狱很不一样。为什么？

(快速免责声明:这是一个令人担忧和情绪化的话题。在一篇 1000 字的专栏文章中，很难理解监禁和累犯的复杂性。我感谢那些持不同观点的人的投入和忍耐。)

显而易见，惩教系统有五个目标:

  1. 惩罚违法者。
  2. 使他们丧失能力(让他们远离街道)。
  3. 阻止犯罪。
  4. 回报社会。
  5. 改造人们，使他们不再犯罪。

但可悲的是，根据犯罪学家鲍勃·卡梅伦的说法，“美国人希望他们的囚犯首先受到惩罚，然后才是改过自新。”

这就是为什么挪威的累犯率为 20%,而美国的累犯率徘徊在 75%左右。这是惊人的。四分之三的前囚犯在出狱后有犯罪的风险。这对社会来说是巨大的无谓损失。如果我们投资于囚犯的潜力，这一比率会降低多少？如果我们给他们无缝重返世界的工具？私营盈利企业在这方面有作用吗？如果有，如何利用技术帮助人们永久退出惩教系统？

今天在做什么

大多数技术报道只是集中在用于预测累犯和关押前科犯的工具上，他们中的许多人正试图改造他们的生活。但是有许多创业公司正在开发产品来帮助他们成功前进。

总部位于纽约的 APDS 最近筹集了500 万美元的 B 轮融资，为囚犯提供平板电脑，供他们用于学习目的。这些平板电脑现已在 17 个州的 88 个惩教机构中使用。囚犯可以使用该软件学习英语，获得 ged 或学习创业。北卡罗莱纳州的创业公司 Pokket 帮助囚犯在释放前的六个月里规划监狱外的生活。

使命:Launch 是一个为囚犯举办演示日和黑客马拉松的组织。他们教授金融知识、企业家精神和社区参与。到目前为止，黑客马拉松的参与者已经开发了一个应用程序，可以将来自朋友和家人的在线消息转换为囚犯(与社交媒体隔绝)的书面明信片，还有一个应用程序可以帮助离开惩教系统的人密封他们的记录，以便他们可以再次被雇用。

在重返社会时，与监狱外的朋友和亲人保持联系会有很大的不同。科技公司 Securus 最近宣布在其 290，000 台平板电脑上免费发送信息，这样囚犯就可以与亲属交流，而不必支付高昂的 T2 费用。英国监狱语音信箱提供一种家庭可以支付的廉价电话服务。在所有情况下，当涉及到实施技术以减少累犯时，经济负担不应落在囚犯身上，囚犯是一种代理和收入潜力有限的被囚禁人群。

监狱学者是由一名前囚犯创建的非营利组织，向囚犯教授企业家精神，并帮助他们制定出狱后的商业计划。他们估计，接受教育的囚犯重返监狱的可能性降低了 43%，这意味着每投入 1 美元，社会的投资回报率为 18.36 美元。Defy Ventures 声称其项目毕业生就业率为 82%,再犯率为 7.2%。其他教授数字素养的项目和编码项目也取得了类似的成功。

类似的理工教育直接降低累犯的例子还有很多。但是为什么要在这里停下来呢？科技还能做些什么来产生影响？

我们还能做什么

美国花费 800 亿美元关押囚犯。这为纳税人创造了一个巨大的财政激励来减少累犯。有两个相关的问题需要解决:科技公司真的能从改善监狱系统囚犯生活的产品上赚钱吗？而他们应该吗？

要回答第一个问题——冒着听起来很愚蠢的风险——一个非常简单的商业模式可能是这样的:对于每个成功使用教育产品为出狱做准备的囚犯，州政府向公司支付 0 美元到将囚犯关押在监狱一年的成本之间的费用(约 81，000 美元)。

这笔钱可以分几年支付，这样某人能够不再犯的时间越长，提供者赚的就越多。如果纳税人只支付给技术提供商一个囚犯一年住房费用的 50%，那么技术公司将使每个用户的 LTV 超过 40，000 美元(！).这种经济激励很容易吸引更多有才华的企业家来实现改善监狱系统中人们生活的目标。(与营利性监狱商业模式相反，营利性监狱商业模式产生了维持监狱人口不变的不正当动机。)

盈利性科技公司销售针对这一人群的产品在道德上是否允许，这是一个更难回答的问题。虽然没有正确的答案，但公司可以遵循以下指导原则:

  1. 不要向服刑人员或其家属收费。纳税人有最大的经济动机来减少累犯——以及监狱的所有相关成本——因此，科技公司应该在州惩教预算中寻找收入机会。
  2. 没有古德哈特定律或不正当的激励。产品的设计和销售必须基于原则，例如，“帮助前囚犯重新融入社会并过上完整的生活”，而不是数字目标，例如，“防止前囚犯在出狱后三年内犯下重罪。”基于数字的目标总是可以游戏的。迫使公司牢记给人们提供改善生活的工具这一最终目标。
  3. 收集用户反馈。仅将合同授予与用户关系密切的公司。与标准消费者不同，囚犯面临委托人/代理人问题:服务的购买者(纳税人)不是使用者(囚犯)。各州应该要求技术提供商从其产品的用户那里收集匿名反馈，并且只将合同授予那些获得最高评级的用户。
  4. 当判决结束时，你产品的工作并没有结束。如果旨在减少累犯的产品真的成功了，这意味着随着监狱人口的减少，这些产品的供应商将会慢慢地淘汰他们自己的市场。制造这些产品不仅仅是为了让人们走出监狱，也是为了帮助他们在离开后的岁月里过上有意义的生活。

有太多太多伟大的产品有待为这个群体打造。LinkedIn 或 Craigslist 上的工作，由雇佣前囚犯的雇主填充。直播宗教服务，以便囚犯可以继续参与他们的社区信仰组织。非职业爱好教育平台。大师班或 Udemy 或 Coursera 的限量版。闭环网游。

最后——不用说——当谈到纠正我们的教育系统的错误时，技术甚至还没有开始触及表面。恢复投票权、就业入口和背景调查限制、取消营利性私人监狱、调整暗示等同于契约奴役的监狱工资……我们可以改善的事情有很多。但是，技术仍然可以在改善惩教系统中同胞的生活方面发挥关键作用。

莫汉达斯·甘地风趣地说，“任何社会的真正尺度都可以在它如何对待最弱势的成员上找到。”几乎三分之一的美国人都有犯罪史。美国的监狱人口占世界监狱人口的 25%。让我们停止忽视这一人口统计数据，为那些最需要的人打造真正让世界变得更美好的工具。
