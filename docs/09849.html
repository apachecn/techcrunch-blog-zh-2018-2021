沃尔玛推出了自己的语音助手“Ask Sam”，最初供员工使用

沃尔玛正在扩大语音技术的使用。该公司今天宣布采用名为“Ask Sam”的员工辅助语音技术，并将其提供给全国 5000 多家沃尔玛商店的员工。该工具允许沃尔玛员工查询价格、访问商店地图、查找产品、查看销售信息、查看电子邮件等。最近几个月，Ask Sam 还被用于访问新冠肺炎信息，包括最新的指导方针、指南和安全视频。

Ask Sam 最初是为沃尔玛旗下的山姆俱乐部商店开发的，它于 2019 年在美国各地推出。由于使用了语音技术，与在小屏幕上输入查询相比，Ask Sam 可以加快获取信息的速度。这使得员工能够更好地与客户互动，而不是花时间在他们的设备上寻找信息。

在新冠肺炎时代，这个工具还提供了另一个额外的好处——当你戴着手套时，使用语音应用程序会更容易。

除了价格查找和产品定位器等常见功能，Ask Sam 还可以帮助员工打印、发送电子邮件或查看员工生日或其他活动。内置的紧急警报功能使经理能够快速有效地向所有员工发出紧急情况警报，无论是要求他们留在商店的封锁令，还是要求所有人离开商店的店内紧急情况。

语音辅助技术是使用机器学习技术构建的，这意味着随着时间的推移，它会变得更加智能和准确。此外，一个团队手动审查所提的问题，以帮助找到技术可能遗漏的其他模式和趋势，如热门搜索项目。

这并不是这家零售商第一次尝试使用语音技术。除了早先在山姆会员商店推出的 Ask Sam 产品，沃尔玛自己也在去年与谷歌合作在谷歌助手支持的平台上进行语音订购，以对抗亚马逊在家用 Alexa 上的进步。三年前，在谷歌快递关闭之前，沃尔玛曾与谷歌合作在谷歌 Home 设备上进行语音购物。

沃尔玛尚未表示是否会开发一种面向零售客户的 Ask Sam 技术。但考虑到该产品现在能够回答顾客也想知道的问题——比如在哪里找到一件商品或它的价格——零售商未来扩大产品范围是有道理的。
