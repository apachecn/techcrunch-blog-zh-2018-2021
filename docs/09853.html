Atlassian 收购资产管理公司 mind ville TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/07/30/atlassian-acquires-asset-management-company-mindville/?tpcc=ECTW2020

Atlassian 今天宣布已经收购了 Mindville ，这是一家总部位于瑞典的以吉拉为中心的企业资产管理公司。Mindville 的 1700 多名客户包括美国宇航局、Spotify 和三星等公司。

图片来源:亚特兰蒂斯人

通过此次收购，Atlassian 也进入了一个新的市场，为其服务系列增加了资产管理工具。该公司的旗舰产品是 Mindville Insights ，它可以帮助 IT、人力资源、销售、法律和设施部门跟踪整个公司的资产。至于你在跟踪哪些资产，这是完全不可知的，不过，鉴于 Atlassian 的用户群，大多数公司可能会用它来跟踪服务器和笔记本电脑等 It 资产。但除了实物资产，你还可以使用该服务自动从 AWS、Azure 和 GCP 导入基于云的服务器，例如，该团队还建立了与 Service Now 和 Snow Software 等服务的连接器。

图片来源:明德维尔

“Mindville Insight 使企业能够全面了解其资产和服务，这对于提供出色的客户和员工服务体验至关重要。这些功能是 IT 服务管理(ITSM)的基石，Atlassian 在这个市场继续保持强劲的势头和增长，”Atlassian 的技术团队负责人 Noah Wasmer 在今天的公告中写道。

据 Crunchbase 报道，由 Tommy Nordahl 和 Mathias Edblom 共同创建的 Mindville 从未筹集过任何机构资金。两家公司也没有披露收购价格。

就像 Atlassian 最近收购的其他公司一样，包括 Code Barrel ，该公司已经是 Atlassian 的合作伙伴，并成功地在 Atlassian 市场销售其服务。

“此次收购建立在 Atlassian 对[IT 服务管理]的投资基础上，包括最近的收购，如针对事件管理的 Opsgenie、针对无代码自动化的吉拉自动化以及针对对话式票务的 Halp，”Atlassian 的 Wasmer 写道。

Mindville 团队表示将继续支持现有客户，Atlassian 将继续基于 Insight 的工具，同时努力将它们与吉拉服务台集成。Atlassian 认为，这种整合将让用户更清楚地了解他们的资产，并让他们提供更好的客户和员工服务体验。

图片来源:明德维尔

“我们已经看到 Insight 产品线在许多行业和各种学科中得到广泛应用，包括一些我们从未想到的产品！最受欢迎的领域之一是 IT 服务管理，Insight 在将所有相关资产数据与事件、变更、问题和请求联系起来方面发挥着重要作用，”Mindville 的创始人在今天的公告中写道。“将我们的解决方案与 Atlassian 的产品相结合，可以实现更紧密的集成，以实现更复杂的服务管理，并由底层资产数据提供支持。”
