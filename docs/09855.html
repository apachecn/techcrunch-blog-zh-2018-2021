CBS All Access 在 2021 年更名前增加了 3500 集新节目

维亚康姆哥伦比亚广播公司宣布对其流媒体库进行重大扩展，增加了超过 3500 集的节目，如《阿凡达:最后的气宗》(Avatar: The Last Airbender)、《夏佩尔的秀》(Chapelle's Show)和《拉古纳海滩》(Laguna Beach)

维亚康姆和哥伦比亚广播公司之前合并，然后分开，然后最近再次合并，额外的内容来自该公司的不同品牌——如 BET、Comedy Central、MTV、Nickelodeon、Smithsonian Channel、TV Land、VH1 和派拉蒙影业——加上额外的体育节目，如欧洲冠军联赛和欧洲联赛的美国独家流媒体。

ViacomCBS 此前宣布计划在今年夏天推出 CBS All Access 的扩展和更名版本，以更好地与迪士尼+和 HBO Max 等流媒体产品竞争。今天的公告表明，更名要到 2021 年初才会到来，但这一消息是这一过程的第一步。

“今天标志着 CBS All Access 向 ViacomCBS 订阅流媒体主页的激动人心的演变的开始，也是未来的预览，”首席数字官 Marc DeBevoise 在一份声明中说。

我们之前曾建议注册 CBS All Access 的主要原因是为了观看新的《星际迷航》节目，如《发现》、《皮卡德》和即将上映的动画系列《下层甲板》但这个声明感觉像是试图吸引非迷航迷也来注册。(明确一点:我喜欢《星际迷航》。)
