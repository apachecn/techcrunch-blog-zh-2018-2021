谷歌让手机版 Chrome 上的自动填充更加安全

原文：https://web.archive.org/web/https://techcrunch.com/2020/07/30/google-is-making-autofill-on-chrome-for-mobile-more-secure/

谷歌正在让手机版 Chrome 上的自动填充更加安全

谷歌今天宣布了一个新的自动填充体验，将使用生物认证进行信用卡交易，以及一个更新的内置密码管理器，这将使登录网站更加简单。

图片来源:谷歌

Chrome 已经在 Windows 和 Mac 上使用 W3C WebAuthn 标准进行生物认证。随着这次更新，这项功能现在也来到了 Android。

如果你曾经在 Android 手机上通过浏览器购物，你就会知道 Chrome 总是要求你输入信用卡的 CVC 代码，以确保真的是你——即使你的手机上存储了信用卡号码。这总是有点麻烦，尤其是当你的信用卡不在身边的时候。

现在，你可以使用手机的生物识别认证，只需你的指纹就可以购买新鞋——不需要 CVC。或者你也可以选择退出，因为你不需要加入这个新系统。

至于密码管理器，这里的更新是新的触摸填充功能，通过标准的 Android 对话框显示给定网站的已保存帐户。你可能已经习惯了基于桌面的密码管理器，但这绝对是 Chrome 的一个重要的内置便利功能——越多人选择使用密码管理器，网络就越安全。这一新功能将在未来几周内登陆 Android 平台的 Chrome 浏览器，但谷歌表示“这仅仅是个开始。”
