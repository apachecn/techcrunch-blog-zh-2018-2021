无代码和低代码创业公司的前景如何？

克拉丽丝女团首席执行官布拉德·弗莱塔格:“这归结于计算机科学家的稀缺。”

自从交易所上一次进入低代码和无代码创业资金的世界以来，这个领域又出现了几轮有趣的融资。

本周， TechCrunch 报道了一家名为 Hevo 的初创公司融资 800 万美元，以及 Paragon 融资250 万美元的种子轮。我们报道称，Hevo 是一家“数据管道初创公司”，帮助“客户的员工整合来自 150 多个不同来源的数据——包括来自 Salesforce 和 Oracle 的企业软件，而不需要技术背景”。

Paragon 是 Y Combinator 2020 Winter batch 的一部分，是一项专注于开发人员生产力的服务，据其联合创始人 Brandon Foo 称，该服务“使非技术人员能够更容易地使用我们的可视化工作流编辑器构建集成”。Foo 补充说，Paragon 希望“为产品和工程团队带来低代码的好处，使构建产品变得更容易，而无需为每个集成编写手动代码”，以帮助“简化产品开发过程”。

自我们上次查看以来，该领域还有更多值得关注的回合，如400 万美元用于 Enduvo (无代码 AR/VR)，一个345 万美元的扩展用于快速增长的 Turbo Systems (无代码“参与平台”)，以及种子回合用于 CloudWorx (无代码物联网)，等等。

我们上周注意到，无代码和低代码初创公司正在筹集大量资本的趋势仍然很热。

但是创业公司并不是唯一在这个领域工作的公司:苹果通过其子公司克拉丽丝女团长期涉足该领域，该子公司在以 FileMaker 的名字运营后于去年更名为该名称。当时，克拉丽丝女团首席执行官 Brad Freitag 告诉 TechCrunch，他的公司的愿景是让“每个人都能使用强大的技术”

这不仅仅是陈词滥调:克拉丽丝女团最知名的产品 FileMaker 帮助用户构建低代码应用，其第二款产品叫做 Connect ，这是一项帮助用户使用低代码工具链接 API 的服务。

鉴于克拉丽丝女团在无代码、低代码领域的时间比大多数人都长，TechCrunch 再次采访了弗莱塔格，谈论了市场类别的近期增长，他对低代码术语的看法，当然，还有他对利基领域初创公司的看法。
