亚马逊与印度 Airtel 签署云协议

原文：https://web.archive.org/web/https://techcrunch.com/2020/08/05/amazon-inks-cloud-deal-with-airtel-in-india/

亚马逊找到了一个新的合作伙伴，以扩大其云服务业务——AWS——在印度这个世界第二大互联网市场的覆盖范围。

周三，这家电子商务巨头宣布与印度第三大电信运营商 Bharti Airtel 合作，该公司拥有超过 3 亿用户，以 Airtel Cloud 品牌向该国的中小企业和大型企业销售各种 AWS 产品。

这笔交易可能有助于领先印度云市场的 AWS 进一步扩大其在该国的主导地位。此前，印度最大的电信运营商 Reliance Jio 与 T2 达成了一项类似的协议，向小企业出售云服务。最近几个月，Reliance Jio 从谷歌、脸书和其他知名投资者那里筹集了超过 200 亿美元。两人宣布了为期 10 年的合作伙伴关系，以“服务数百万客户”

Airtel 服务于 2500 多家大型企业和 100 多万家新兴企业，它自己在今年一月与谷歌签署了类似的云协议。Airtel 说，这种伙伴关系仍然存在。

“AWS 带来了超过 175 种服务。我们非常支持云中的任何工作负载。我们拥有最大和最有活力的客户群体，”印度和南亚 AWS 总裁 Puneet Chandok 在周三中午与记者的电话会议上说。

他们表示，这两家公司在 2015 年签署了一项类似的协议，还将合作建立新的服务，并帮助现有客户迁移到 Airtel 云。

Airtel 的全球首席信息官兼云和安全业务负责人 Harmeen Mehta 在电话会议上表示，今天的交易表明了 Airtel 在电信业务之外建立业务的努力。上个月，Airtel 与 TechCrunch 的母公司威瑞森合作，向印度的商业客户出售蓝战士视频会议服务。

十年前，随着科技巨头争相在印度积累用户，与运营商的交易在印度非常普遍。复制一个类似的策略现在说明了这个国家采用云的阶段。

过去十年，印度有近 5 亿人上网。慢慢地，小企业和商家也开始使用数字工具、存储服务，并接受在线支付。

印度已经成为新兴云服务的主要市场之一。根据研究公司 IDC 的数据，到 2024 年，中国的公共云服务市场预计将达到 71 亿美元。
