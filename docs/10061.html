谷歌、诺基亚、高通是芬兰手机制造商 HMD Global 的 2.3 亿美元 A2 系列投资者

原文：https://web.archive.org/web/https://techcrunch.com/2020/08/11/google-nokia-qualcomm-are-investors-in-230m-series-a2-for-finnish-phone-maker-hmd-global/amp/

移动设备制造商 HMD 环球公司 l 宣布提供 2.3 亿美元的 A2 系列融资，这是该公司自 2018 年获得 1 亿美元融资后首次获得外部融资，当时该公司的估值达到了独角兽水平。自 2016 年年底以来，这家初创公司独家授权诺基亚品牌的移动设备，迄今将发运约 2.4 亿台设备。

最近的现金注入在规模(HMD 称其为今年欧洲第三大融资轮)上均引人注目；以及投入资金的战略投资者——即谷歌(Google)、诺基亚和高通——的情况。

一家操作系统主宰全球智能手机市场的科技巨头谷歌(Google)成为欧洲最后一家举足轻重的移动设备原始设备制造商(HMD)的战略投资者，是否会引起地区竞争执法机构的注意，还有待观察。诶，垂直整合有人吗？(也就是说，两年多前谷歌因操作安卓系统违反反垄断法被欧盟监管机构处以 50 亿美元罚款，欧盟委员会表示将继续监控市场“补救措施”。)

另一个奇怪的现象是，在今天的公告发布之前，我们采访了 HMD Global 的首席执行官弗洛里安·塞彻，他没想到投资者的名字会被披露，但一位新闻发言人已经和我们分享了这些信息，所以他正式确认这三位投资者就是我们要找的投资者。(但我不会被谷歌攫取了多少权益所吸引。)

HMD 的智能手机运行在谷歌的 Android 平台上，这给了这家科技巨头一个坚定的商业理由来支持这家移动制造商在全球主要增长型市场增加谷歌包装的硬件的可用性。

HMD 把一贯(而且不断更新)的 Android 风格与谷歌自有品牌 Pixel 智能手机带来的高端“纯粹”Android 体验相提并论，但区别在于这家芬兰公司提供的设备有多种价位，而且硬件都瞄准了发展中市场的移动用户。

其结果是与谷歌的 Pixel 硬件几乎没有重叠，如果 HMD 扩大谷歌的服务用户渠道(因为它通过定向广告赚钱)，谷歌仍有大量业务上升空间。

鉴于安卓在致命扰乱诺基亚利润丰厚的智能手机业务中的作用，移动历史的鉴赏家可能会认为谷歌投资诺基亚品牌智能手机(通过 HMD)不仅仅是一个小小的讽刺——将芬兰巨头从世界头号移动制造商的宝座上拉下来，并开创了一个由安卓推动的亚洲移动巨头时代。但是在科技领域等待足够长的时间，事情往往会有转机。

“我们非常兴奋，”当我们提到谷歌在诺基亚智能手机历史衰落中的关键作用时，Seiche 说。“我们将如何书写智能手机的下一个篇章是该公司的一个关键战略支柱，我们有机会与谷歌在这方面密切合作，从一开始就是非常非常好的合作伙伴关系。这项投资无疑证实了这一点，也是对未来的肯定。”

“这是一个行业的关键时刻，因此有一个清晰的战略——我们相信，有一个清晰的差异化和不同的观点可以提供，这是我们为自己开发的一项了不起的资产。现在是我们加倍努力的大好时机，”他补充道。

我们还问 Seiche，HMD 是否有兴趣利用欧盟委员会的安卓反垄断执法决定，即放弃安卓系统，删除常用的谷歌服务，或许用一些欧洲的替代产品取代它们，这对于在该地区销售的原始设备制造商来说至少是一种可能性，但 Seiche 告诉我们:“我们已经研究过了，但我们坚信消费者或企业客户实际上喜欢(谷歌)服务，因此他们会为自己选择这些服务。”(据推测，谷歌数百万美元的直接投资也有助于谷歌服务商业案例的积累。)

与此同时，诺基亚一直与 HMD 关系密切——后者是由诺基亚前高管建立的，其唯一目的是授权其标志性的手机品牌。(背景故事诺基亚的移动设备部门向微软的出售条款中有一个条款于 2016 年到期，为诺基亚的品牌 在没有之前 Windows Mobile 包袱的情况下重返智能手机市场铺平了道路。)

它对 HMD 的投资现在看起来像是对该公司迄今为止在竞争激烈的移动领域的表现投了一张信任票(HMD 没有透露设备销售的很多细节，但 Seiche 告诉我们，去年它售出了超过 7000 万部手机；这是智能手机和功能手机的综合数字)，也是对未来增长机会范围的乐观评估。

在后一个方面，以美国为首的西方与中国之间的地缘政治紧张局势，看起来确实有可能给 HMD 的商业带来机遇。

例如，移动芯片制造商高通正面临业务损失，因为美国政府的限制威胁到其继续向华为出售芯片的能力；一家主要的中国设备制造商成为美国总统特朗普的主要目标。因此，它对支持 HMD 增长的兴趣，看起来像是高通对冲美国政府针对其移动设备制造商投资组合中的中国公司的干扰的一种方式。

鉴于特朗普最近对抖音应用发出的威胁，似乎可以有把握地认为，没有一家拥有中国所有者的科技公司是安全的。

作为一家欧洲公司，HMD 能够将自己定位为一个安全港 Seiche 的销售宣传强调了对安全细节和整体体验质量的关注，认为这是与 Android 平台不同的关键因素。

“我们从一开始就非常明确、非常一致地选择这些核心原则，这些原则贴近我们的内心，与诺基亚品牌本身密切相关——安全、质量和信任肯定是关键要素，”他告诉 TechCrunch。“这引起了我们全球运营商和零售客户的共鸣，这无疑也是一个核心的基本优势，那些具有长远眼光的合作伙伴清楚地看到了与我们一样的发展机遇。”

HMD 确实在中国以及世界其他一些地方使用制造设施，包括巴西、印度、印尼和越南。

但当被问及是否认为继续使用中国制造商生产‘安全’的移动硬件存在任何供应链风险时，Seiche 回应称:“最重要的[因素]是我们确实能充分控制软件体验。”他特别提到了今年早些时候 HMD 对 T2 瓦罗纳实验室(Valona Labs)的收购。 这家芬兰安全初创公司执行所有软件审计。“他们基本上控制我们的软件，以确保我们能达到可信赖的标准，”Seiche 补充说。

随着西方和远东的地缘政治局势紧张以及其作为替代能源的价值受到关注，欧洲移动设备制造商 HMD 正着眼于在非洲、巴西和印度等增长型市场进行扩张，现在它已经获得了一大笔新资金。 目前，HMD 表示其设备活跃于 8 个地区的 91 个市场，遍布全球 25 万个零售网点。)

该公司还计划在现有旗舰产品诺基亚 8.3 的基础上，以更高的性价比为设备提供 5G 技术。Seiche 还表示，它希望在移动服务方面做得更多。 HMD 首款 5G 设备、旗舰车型诺基亚 8.3 将在几周内登陆欧美市场。Seiche 还建议在明年年中以中等价位推出 5G 设备。

“就市场接受度而言，5G 之旅又在中国开始了。但现在，欧洲和美国是关键的下一个机会——不仅在高端市场，在中端市场也是如此。尽快实现这一目标是我们的目标之一，”他说，并指出与高通在这方面的联合努力。

他补充说:“我们也认为诺基亚在 5G 转型方面有很大的机会，因为他们也在进行大量的私有 LTE 部署，这也是一个有趣的领域，因为……我们在这个大型企业细分市场也有非常强的影响力。”。

在移动服务方面， Seiche 重点介绍了 HMD 连接的推出:一种针对旅行者的数据 SIM 卡——表明它可能在未来扩展到更多的连接服务，与运营商建立更多的合作伙伴关系。

“我们已经推出了几项与硬件业务密切相关的服务——比如为你的智能手机提供保险——但我们现在也把连接视为一个对我们来说很好的领域，”他说。“第一个试点是我们的全球漫游，但我们相信未来消费者或企业客户可以通过他们的设备直接获得连接。我们还与运营商合作来实现这一目标。”

“你可以更多地将我们视为(对运营商的)补充，”他补充道，他认为运营商的业务“动态”也发生了实质性变化——一段时间以来，客户获取已经不再是一种线性游戏。

“同样，当我们谈论谷歌 Pixel 与我们时，我们有不同的足迹。同样，如果你看看运营商，他们今天从哪里获得用户，今天已经是他们自己的直接渠道和他们的合作伙伴渠道的混合。实际上，为什么智能手机玩家不是他们的天然好伴侣呢？因此，我认为你会看到这是一种趋势，可能会在未来几年内演变。”
