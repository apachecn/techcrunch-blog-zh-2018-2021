三星 Galaxy Note 20 Ultra 评测 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/08/18/samsung-galaxy-note-20-ultra-review/

Galaxy Note Ultra 是一款售价 1399 美元的智能手机。即使按照三星的标准，这也是一款高端奢华设备。这款手机是为那些早早登机的人准备的，他们看着我们其他人慢慢地走上飞机，想知道我们离飞机尾部有多近，并从中获得一种自豪感。

当然，大多数或所有这些功能最终将渗透到更便宜的机型中，但这款手机是为那些愿意支付溢价以领先竞争对手一年的人准备的。

5G 就是这种现象的完美例子。在去年的机型上仍然是奢侈品，现在它是整个 Note 系列的标准配置(几乎可以肯定的是，当新款机型在六个月后上市时，Galaxy S 也会有这种配置)。当时世界的蜂窝网络可能还没有准备好支持它，但它是另一种前沿技术，可供愿意支付溢价的早期采用者使用。

事实是，无论你选择 Note 20 还是 Note 20 Ultra，或者就此而言，S20 系列的任何成员，你都将获得一部不错的手机。尽管有 400 美元的差距，但 Note 20 和 Ultra 之间只有几个关键区别。第一个也是最直接的区别是屏幕。这就是你如何知道你有一个真正的优质设备。它真的，真的很大。

在这里，这意味着 6.7 英寸(2400×1080)和 6.9 英寸(3088×1440)显示器之间的差异。这比 6.2 英寸和 6.9 英寸 S20 选项之间的差距要小得多。最终不利于 Note 系列的是，它的最大屏幕尺寸与 S20 相同，而且两款 Note 之间的尺寸差异相对较小。我要说的是，高端手机真的开始碰到智能手机屏幕的天花板了，但我们已经说过一次又一次了，然而我们还是在这里。

图片来源:曹薇儿

在 Note 问世的九年里，它从看似不可能的 5.3 英寸增长到 6.9 英寸的主要因素是三星承诺降低手机的屏幕与机身的比例。即便如此，Ultra 还是一款非常大的手机。我不能用手完全抓住它。老实说，根据你的手和/或口袋的大小，产品的大小很可能是一个交易破坏者。

拥有如此大的手机的结果是，你有更多的空间来放电池。这意味着 4500 毫安时的电池寿命比 20 年代的 4000 毫安时要长。这很好，但仍明显小于 S20 Ultra 的 5000 毫安时，可能是因为 S Pen 插槽的存在，它占用了很大一部分内部空间。我能够一次充电超过 24 小时(接近 28 小时)，绝对达到了该公司的“全天”生活基准。

你的结果，一如既往，会有所不同。但在美国 5G 覆盖率仍不稳定的时候，这一数字翻了一番。三星(TC 的母公司，记录在案)推出了一款带有威瑞森 SIM 卡的机型。在我居住的皇后区，我无法接入 5G 网络，但当我周末走过大桥进入曼哈顿时，事情确实有了转机。在更理想的情况下，我将能够在 LTE 和 5G 电池额定值之间进行更可控的测试，但现在是 2020 年，理想是太多的希望。

图片来源:曹薇儿

Note 20 和 Note 20 Ultra 的另一个主要区别当然是摄像头。再次，相机模块是巨大的。三星更新了一点工业设计，但坦率地说，它仍然相当庞大。这是可以原谅的，但是，当你考虑到手机包装的因素。两款新笔记本都配备了三摄像头系统，但 Ultra 将 1200 万像素的广角镜头换成了 10800 万像素，加入了 1200 万像素的超广角和 1200 万像素的长焦镜头。

图片来源:布莱恩·加热器

设置类似于 S20 Ultra 的，有几个重要的区别。首先，飞行时间深度相机已经被激光自动对焦取代。TOF 肯定感觉更像是一个面向未来的方面。除了当前的肖像模式需求，随着增强现实成为移动软件越来越重要的一个方面，它可能会发挥重要作用。也就是说，激光自动对焦对于日常拍照的需求来说感觉更实用。即使没有启用肖像模式，相机设置在图像上也有真实有效的散景，并且在拍摄特写微距照片时非常有效。

图片来源:布莱恩·加热器

这里的另一个大的更新是关于空间缩放前沿的一些缓和的期望。该功能是为 S20 Ultra 推出的，承诺实现令人难以置信的 100 倍变焦。事情的真相并不令人兴奋，因为任何接近这个最高数字的东西最终都是不可用的。大多数照片最终看起来像抽象印象主义的作品。Note Ultra 保持了仍然令人印象深刻但更易于管理的 50 倍。

图片来源:布莱恩·加热器

在这个数字上，你仍然冒着相当严重的图像质量下降的风险，但是总的来说，结果会比这个数字翻倍更令人满意。总的来说，这个东西的放大效果非常好。使用默认的照片软件，我建议坚持使用三个树图标在三个主要相机之间切换。这将使它保持最大 10 倍的光学变焦。但是如果你需要更多，那就去做吧。

图片来源:布莱恩·加热器

三星对相机质量的方法主要集中在硬件上，结果在它拍摄的图像中显而易见。这与谷歌的方法形成了鲜明的对比，谷歌试图用计算摄影来使自己与众不同。Pixel 的摄像头本身非常好，但在质量变焦等方面无法与 Note 竞争。当然，三星的做法是要花钱的。朋友们，重要的是要记住我们在这里谈论的是一部 1400 美元的手机。

屏幕真的很优秀。对我的口味来说，颜色可能有点过了——特别是亮红色，但这是一个简单的解决办法，在屏幕设置下从“鲜艳”切换到“自然”。对于一些人来说，在后一种背景下，红军可能会有点泥泞。无论哪种方式，这真的只是一个个人喜好的问题，但我建议用它来玩。120Hz 的刷新率使得一些动画非常流畅，但这也是当你需要节省电池时可以很容易禁用的功能之一。

定向麦克风是 S20 上引入的更被低估的功能之一，允许您根据设备的位置确定音频录制的焦点。更酷的是可以在录音时将 Galaxy Live Buds 用作麦克风。这对于站着面试视频来说很方便，尤其是在嘈杂的环境中。

图片来源:曹薇儿

Note 20 是首批配备骁龙 865+的设备之一，本质上是你在 S20 上看到的旗舰 865 的超频版本。这里的时钟速度提高了一点，图形性能提高了大约 10%。移动处理器在 2020 年不会比这更好了。我计划在微软合作伙伴关系上发布一些更完整的东西，为手机带来一些独家游戏通行证内容(老实说，我目前正在等待一个蓝牙移动 Xbox 控制器)。

但这款手机可以满足你的大部分日常任务，可能会成为更好的云游戏手机之一，因为最新的旗舰产品骁龙配备了 12GB 的内存。这里还有很好的 128GB 存储空间，可以扩展到非常好的 512GB。更好的是，借助 microSD 插槽，它可以扩展到 1TB(这在两款 S20 机型上也有，但在常规 Note 中没有)。

我可能每次有新的 Note 出来都会写这个，但我不是一个真正的手写笔人，即使在玩了近十年的 Note 设备之后。尽管如此，我仍然对这款设备识别我糟糕的笔迹的能力印象深刻。也许我会成为这几代人中的一员。我猜更奇怪的事情发生了。S Pen 在经历了这么多代之后变得非常精致，反应也非常灵敏。Air Actions 可以让你在很远的地方使用手写笔作为控制——这是一个足够简洁的功能，但我又一次没有看到自己使用它。其他很酷的新功能包括音频书签，它可以将录音同步到你正在做的笔记上。肯定是有帮助的，尽管我预计当三星推出像谷歌录音机一样的实时转录时会更有帮助。

图片来源:曹薇安

如果你读够了这些两年一次的三星旗舰评论，你可能知道接下来会发生什么。这份说明与其说是更实质性的东西，不如说是对其前身的一种改进。如果你的笔记本是一年或两年前的，你当然不需要跑去更换那个也非常好的手机。这只是我们在移动行业生命周期中所处的阶段。总的来说，更新只是感觉更递增。

但与这些设备一样，Note 20 Ultra 代表了目前你可以购买的一些最好的移动硬件。尤其是相机，它真正出色的性能值得一提。但是，和以往一样，最好的东西是要花钱的。如果你能忍受 1400 美元的安卓手机，它们不会比 Galaxy Note Ultra 好多少。
