Target 在 Q2 创下销售记录，当天服务增长 273%

原文：https://web.archive.org/web/https://techcrunch.com/2020/08/19/target-sets-sales-record-in-q2-as-same-day-services-grow-273/?guccounter=1

继沃尔玛(Walmart)周二公布受疫情推动的收益超过之后，塔吉特百货(Target today)也轻松超过了华尔街的预期，在一系列关键指标上创造了创纪录的季度业绩。这家零售商周三宣布了迄今为止可比销售额最强劲的季度，在 Q2 增长了 24.3%，推动塔吉特的利润同比增长 80.3%，达到 16.9 亿美元。塔吉特指出，网上订购尤其受欢迎，数字销售增长了 195%。免下车服务、订单提货和 Shipt 等当天服务也增长了 273%。

在本季度，Target 的收入、同店销售额、调整后的每股收益和毛利润都超过了预期。它报告的收入为 230 亿美元，而预期为 198.2 亿美元。其创纪录的可比销售额增长 24.3%，远高于预期的 5.8%。每股收益为 3.38 美元，而预期为 1.58 美元。其 GM 为 30.9%，而非预期的 28.98%。

该公司将销售增长归因于一系列因素，包括它在疫情作为一项重要业务保持开放的能力，它的客户对目标品牌的整体信任，它让客户在其产品类别中购物的能力，它的数字服务，以及最重要的是，客户重返 Q2 的商店。

然而，后一项并不一定意味着目标顾客正在过道里走着。

相反，它表明了疫情之前制定的投资目标，即弥合在线订购与实体店之间的差距。例如，在 Q2，Target 的店内订单量增长了 60%以上，因为购物者会走进 Target 去取网上订单。

Target 的免下车服务，允许顾客在线购物，然后在指定的停车点停车，让订单把他们的车开来，在这个季度增长了 700%以上。

Target 的当天送货上门服务比去年增长了 350%。

这意味着，对于大部分目标顾客认为的“网上购物”，他们的销售实际上是由目标商店完成的。事实上，塔吉特百货表示，其门店实现了超过 90%的第二季度销售额。

为了建立其数字履行服务，Target 采取了一种类似科技公司的方法，利用能够快速迭代新想法的内部工程团队。例如，一个八人团队，包括四名工程师，最初从 2017 年 4 月开始建造 Drive Up 。到 2017 年夏天，Drive Up 正在进行内部测试。然后在那年秋天，它推广到了塔吉特百货的国内市场。截至 2019 年 8 月，塔吉特的免下车服务在全国范围内提供。

这家零售商还进行了一些重要的收购来帮助其电子商务运营，包括在 2017 年以 5.5 亿美元收购 Shipt，以及最近的在 5 月份从 Deliv 收购当天送达技术。它还将 Shipt 的当天服务直接集成到自己的网站和应用中，而不是仅仅依靠 Shipt 的专用品牌来接触目标购物者。

这些努力的结果现在在疫情得到了回报，顾客不一定想亲自浏览商店的过道去购物。这导致塔吉特看到了雅虎财经今天所描述的零售业务的“科技公司般的增长”。

Target 的董事长兼首席执行官布莱恩·康奈尔还指出，该公司在 2020 年的前六个月增加了 50 亿美元的市场份额，在此期间，它增加了 1000 万新的数字客户。

“我们第二季度的可比销售额增长了 24.3%，是我们有史以来最强劲的增长，这是我们团队韧性和业务模式持久性的真实证明。我们的商店是这一前所未有的增长的关键，店内电脑销售额增长了 10.9%，商店实现了超过四分之三的塔吉特数字销售额，增长了近 200%。“尽管我们在团队薪酬和福利方面进行了大量投资，但我们在本季度仍实现了出色的盈利能力。我们仍然坚定不移地致力于为我们的客人提供安全便捷的购物体验，他们的信任使我们在今年上半年的市场份额增加了 50 亿美元，”Cornell 继续说道。

“凭借我们与众不同的商品分类、一整套便捷的履行选项、强大的资产负债表和我们高度敬业的团队，我们已做好充分准备来应对疫情的持续挑战，并在未来几年继续实现盈利增长，”他说。

疫情也对消费者的购买行为产生了影响。塔吉特百货表示，其所有五个核心商品类别的销售额都有所增长。这是由电子产品的最强劲销售带动的，该类别同比增长 70%，原因是人们呆在家里工作、学习和娱乐，导致更多购买电脑或游戏系统等物品。电子产品之后是家居产品，增长了 30%，然后是美容、食品和饮料以及必需品类别，增长了 20%。服装甚至从 Q1 下降 20%转变为 Q2 的两位数增长。顾客购物篮规模也增长了 18.8%，因为人们在目标行程中购买了更多商品。

与沃尔玛一样，塔吉特也受到了政府刺激支票的提振，而政府刺激支票可能会在下个季度逐渐减少。但 Target 拒绝提供 2020 年的进一步指导，称新冠肺炎危机使消费者购物模式和政府政策变得不可预测。
