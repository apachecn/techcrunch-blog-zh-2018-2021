曾经被指控利用客户数据开发面部识别技术的谷歌关闭了消费者应用

原文：https://web.archive.org/web/https://techcrunch.com/2020/08/24/ever-once-accused-of-building-facial-recognition-tech-using-customer-data-shuts-down-consumer-app/

云照片存储应用正在关闭，理由是与苹果和谷歌提供的默认服务的竞争加剧。然而，除了试图与科技巨头竞争的小型创业公司的困境之外，该公司还有其他问题。去年， NBC新闻报道了该公司一直在使用其客户的照片开发面部识别技术，然后通过Ever API 向商业客户出售，包括执法部门和军方。

报告称，该公司的真实商业模式没有向访问Ever网站或应用程序的消费者适当披露。

该公司当时辩称，它没有与其面部识别客户分享人们的私人照片或任何识别信息。相反，它利用客户上传的数十亿张图像建立了一种可以从匹配中学习的算法，现在能够根据其他数据训练自己。

北加州的美国公民自由联盟(ACLU)表示，这项业务是“对人们隐私的严重侵犯”，因为很少有人知道他们的家庭照片被用于建立监控技术。

虽然包括亚马逊和微软在内的其他公司近年来也开发了自己的面部识别技术产品，但它们都是使用公共数据集。曾经在未经知情同意的情况下使用过自己用户的照片。(报道称，在NBC新闻开始调查并联系该公司后，Ever的隐私政策中增加了一行。)

新闻报道出来后，Ever 将其Ever AI更名为Paravision，以远离争议。

截至上个月，Paravision仍在继续推销其产品。在7月的新闻稿中，该公司宣布，在7月27日的国家标准与技术研究所(NIST)人脸识别供应商测试(FRVT)报告中，它已经获得了全球前两名的准确率，该报告重点关注戴面具的人脸识别。除了面部检测解决方案，该公司还销售一套活动识别工具。尽管消费者应用程序关闭了，但这项业务似乎还在继续。

不幸的是，2019年不是第一次因其糟糕的商业行为而成为头条新闻。

在来自谷歌和苹果照片技术进步的越来越大的压力下，早在2016年就开始通过短信向用户发送垃圾联系人，邀请他们查看其应用程序。短信邀请垃圾邮件在当时是一种流行的社交应用增长黑客技术，尽管人们普遍不喜欢。在Ever的例子中，它帮助该应用在Android发布之前爬上了iOS排行榜。

同样值得注意的是，Ever试图利用当前对科技公司垄断的关注，作为转移对Ever应用程序关闭的指责的一种方式。

今天，苹果、谷歌和其他科技巨头正在美国接受反垄断调查。，因为政府正在努力确定这些公司是否利用其平台地位损害甚至消除了他们的竞争。

曾在公告中特别点名苹果和谷歌，称:

这项服务已经存在了七年多，但随着过去几年来自苹果和谷歌照片存储产品(本身就是优秀的产品，值得作为替代选择)的竞争越来越激烈，Ever服务不再是可持续的。

这里的含义是，面对如此激烈的竞争，Ever没有机会，现在它的业务已经结束了。

该声明没有提到Ever自己的行为如何在多年来侵蚀其用户的信任方面发挥了作用，或者它后来如何作为一个B2B技术解决方案提供商获得成功。

然而，该公司的关机常见问题解答提到了其面部识别技术。在这里，该公司解释说，一旦Everalbum关闭Ever服务，用户的照片和视频将“永远不会用于任何目的，包括提高人脸识别等计算机视觉能力。”它还表示将删除用户数据，除非法律要求保留这些数据，并确认用户的实际照片从未出售给第三方。

这对Ever的客户来说太少，太晚了，他们永远不会同意让他们的照片用于建立面部识别技术。既然这项技术已经建立起来，似乎就不再需要这些年来收集的初始训练数据了。

Ever服务将于太平洋时间2020年8月31日晚上11点59分关闭。该公司表示，在此之前，客户将能够导出数据并删除他们的账户。

根据来自 Crunchbase 的数据，这家被称为Ever公司剩余部分的公司已经筹集了2900万美元的风险投资。(这包括作为Everalbum筹集的资金。迄今为止，该公司的投资者包括Icon Ventures、Felicis Ventures、Khosla Ventures、Trinity Capital Investment、UpHonest Capital、Atomic和其他几家公司。通常情况下，Atomic 同时扮演联合创始人和投资者的角色。
