以下是Y Combinator 2020年夏季演示日第一天的98家公司

原文：https://web.archive.org/web/https://techcrunch.com/2020/08/24/here-are-the-98-companies-from-y-combinators-summer-2020-demo-day-1/

今天是Y Combinator的两部分2020年夏季演示日的第一部分，近100家公司首次向世界展示了他们的努力。

2020年夏季的一批公司是第一批完全远程的YC群组，正在进行的疫情领导加速器将其计划完全虚拟化。但那天其实是YC的第第二虚拟演示日；随着疫情的严重性在3月份变得更加明显，YC在最后一分钟将2020年冬季课程的演示日部分转移到了虚拟课堂。这样做意味着放弃演示日的关键方面…包括，最值得注意的…演示。W20课程中的每家公司都通过一张幻灯片和对公司的简短文字描述进行推介，而不是现场推介。

YC显然有更多的时间来准备这一次，演示日的体验更符合现场活动。在一系列令人难以置信的现场推介中，每家公司只有60秒的时间向投资者、媒体和其他创始人推销；这足以呈现他们是谁和他们做什么的表面，但通常会将如何/为什么的更深层次的细节留给后续的对话。

以下是我们对今天到场的每家公司的记录:

Evergreen:为员工申请购买和跟踪审批的数字化解决方案。B2B业务是为非集中采购和需要管理不同工具的公司而构建的。

验房 : 一家帮助房产中介为卖房者协调验房的软件公司。这家初创公司希望成长为一个托管市场，充当从维修到保险的所有家庭服务的经纪人。

Blue Onion Labs:这家金融服务初创公司正在帮助公司理解跨多个系统的金融交易数据。他们的API集成套件旨在解决会计团队的一大痛点，作为了解传入交易全部范围的“单一真实来源”。

看中: 配送便利店商品，承诺30分钟内送达。目前专注于英国。Fancy不是从现有商店提货，而是经营自己的“暗店”，以保持更高的利润率。

BukuWarung:一款针对印尼商家的微记账app。它使夫妻店能够为他们的业务带来支付和信用。该服务目前每月有35万活跃商户。

core care:在数百家保险公司、医疗保健提供商和政府机构之间转移数据会导致每年数十亿美元的损失，并可能延误治疗。CoreCare在它们之间同步患者数据，防止错误并为每个人节省资金。

hot plate:hot plate是一种帮助失业厨师在家烹饪食物、出售、送货上门的服务。在创业的前8周，这家公司已经雇佣了10名厨师，他们在GMV创造了大约1500美元的收入。HotPlate从中抽取15%的份额。创业公司的观点是，COVID改变了世界，以至于现在，我们都想要家常菜，而不是我们认为的餐馆准备的食物。我们走着瞧。考虑到每位厨师产生的GMV很小，随着时间的推移，经济效益将会如何将被证明是有趣的。也许订单规模会增长。

Electry:一款招聘机械师、电工的工具，定位为“技术蓝领的LinkedIn”。该团队表示，目前每月收入为7.5万美元，并且是盈利的。

提示 : 我们生活在这么多不同的基于云的平台上，从Google Docs到Figma到Github和Dropbox。Clew希望成为一家公司，帮助你将所有不同的基于云的应用程序整合到一个单一的流平台，作为一个文件系统。到目前为止，Clew已经招募了85名付费用户，并收取50美元的年费。

Arist :类似安全和反种族主义课程的员工培训资料通常是视频或幻灯片。Arist通过交互式短信进行相同的培训，他们说这种方式更快，对今天的用户来说更自然，并导致更高的完成率。他们已经有了几个大客户，并表示很快就会盈利。

Decentro :延续这一批正在成形的X for India潮流，Decentro想为印度打造格纹。该公司提供了一个用于银行集成的API，如Plaid。今年早些时候卖了数十亿美元的格子花呢仍然留在每个有感觉的风险投资人的脑海里，所以这种比较可能证明是诱人的。Decentro的规模仍然很小，总交易额约为100万美元(GTV)，在MRR约为7000美元。尽管如此，目前只有4家客户，还有45家正在筹备中，这是一条不错的道路。

git duck:专为开发者打造的视频聊天工具。它直接与开发人员的IDE挂钩，允许实时代码共享，允许开发人员一起远程编码。每个开发者席位收费20美元。

hello meter:使用现成的摄像头帮助快餐店老板分析顾客得到服务的速度。目前在两个地点进行测试，并有意向在300多个地点进行测试。

volo pay:创业公司想成为东南亚的Brex。VoloPay寻求将审批、账单支付、费用和会计自动化整合到一个平台中。它通过公司信用卡简化了支付，在3个月内，用户已经花费了9万美元。

human loop:在棘手的数据集上构建软件来注释、训练和部署自然语言处理。这家初创公司正在将专业的人工智能引入律师、医生和会计师，以处理他们的数据，否则这些数据将需要外包给昂贵的领域专家。
