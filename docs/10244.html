万事达卡收购并关闭了受到新冠肺炎冲击的体验市场 IfOnly

原文：https://web.archive.org/web/https://techcrunch.com/2020/08/24/mastercard-acquired-and-shut-down-ifonly-an-experiences-marketplace-hit-by-covid-19/

毫无疑问，旅游业是受冠状病毒疫情影响最严重的行业之一，受到以下因素的制约:人们在国家之间和国家内部流动的限制，许多场所关闭，减少聚会的新规定，经济萎缩，以及消费者普遍不愿意外出。在此之后，该领域的一家初创公司被收购。

信用巨头万事达卡悄悄收购并关闭了 if only——一个“体验”市场，其基础是获得独家的、通常昂贵的活动和人物，客人为体验支付的一部分收入用于公益事业。万事达卡告诉 TechCrunch，在 2018 年首次领导对该公司的战略投资后，它已经将技术和团队纳入了无价的——它自己的体验市场。

一位发言人表示:“去年年底，其技术为 Priceless.com 供电的 IfOnly 成为万事达大家庭的一员，带来了他们的专业知识和内部技能。”。“IfOnly 平台将继续帮助推进我们的无价战略，我们合并后的团队将更好地为全球持卡人提供专属体验。”

IfOnly 是由 Trevor Traina 创建并领导的，他是一位商人，美国 T2 最富有的家族之一的成员，也是一位人脉广泛的 T4 的支持者。当特朗普在 2018 年任命他为驻奥地利大使时，特雷纳最终离开了首席执行官的职位。接替他的是约翰·鲍里斯，他曾是 Shutterfly 的 CMO。他仍然将 IfOnly 的首席执行官一职列为他目前的工作。

万事达卡只是 IfOnly 的大型战略投资者之一；其他的还有凯悦酒店、苏富比和美国运通，而金融投资者包括创始人基金、NEA 和科斯拉。投资者总共向这家初创公司投入了近 5000 万美元。根据 PitchBook 数据，IfOnly 的最新估值约为 1.05 亿美元。

虽然万事达卡表示，它已在今年年初收购了这家公司，但鉴于全球形势的变化以及它对旅游业的影响，这最终证明是这家初创公司的软着陆。

直到今年 7 月，IfOnly 才在其网站上发布公告，宣布关闭和收购。(一位读者上周向我们透露了进展情况。)

但在此之前，IfOnly 的业务因冠状病毒疫情而陷入停滞。在网站的存档页面中(通过互联网档案馆的回溯机)，该公司几个月前宣布将暂停提供其体验，“由于新冠肺炎的情况”，并表示将随着了解的更多而更新。

出售(和关闭)结束了一家初创公司，这家公司开始时的独家体验似乎完全是为了那 1%的人。例如，有一个报价(在存档页面上)提供“意大利佛罗伦萨家庭周末盛宴”，四人起价 62，851€(约 74，000 美元)，以及法国香槟地区之旅。

但这家初创公司似乎想拓宽这个领域。另一个提议包括在拉斯维加斯与“山羊瑜伽”的创始人进行一次私人喂养和小山羊瑜伽课程(是的，这是一件事)，起价约为每人 33 美元，取决于群体数量，可能还有山羊的数量和其他参数。每一次体验都与一个特定的慈善机构相关联，该慈善机构将从购买中受益。

看起来，IfOnly 还扩展到了单一的虚拟体验，以及那些可以直接在其网站上竞价的体验，以及与苏富比拍卖行合作的体验。这些包括定制由苏珊·萨拉登创建的语音邮件，或者投标与玛丽·凯·普莱斯共进午餐。

但这种情况可能已经出现在墙上，在全球健康疫情之后，这家初创公司没有在其网站上制定任何形式的“B 计划”。其他围绕体验建立业务的公司——参观景点、旅游、会见名人以及做其他事情，让人们参与离家近或更远的新事物——不得不彻底反思他们的方法。

几年前，Airbnb 大举进军 experiences，以补充和扩大其住宿预订平台。4 月Airbnb 推出了在线体验，为用户提供虚拟旅行和其他基于视频的活动。

总部位于柏林的创业公司 GetYourGuide 资金雄厚，提供独特的旅游和其他基于旅行的体验，该公司已经降薪并重新评估其商业模式，基本上围绕着取消 2020 年(也就是说，假设今年没有人预订)的想法，希望在更长的时间内实现扭亏为盈。

与此同时， Klook 求助于切割棒。然而，像 Omaze 这样的其他公司——和 IfOnly 一样，也将其经验与慈善筹款联系在一起——仍然在筹款和运营，尽管目前需要推迟他们出售的一些经验。

对于万事达卡来说，无价平台是该公司在基本信用卡服务之外扩大业务的更广泛努力的一部分。(这也是万事达卡、维萨卡和美国运通等公司向商业服务领域扩张的原因，比如万事达卡收购了 B2B 支付公司 Nets，美国运通收购了 SMB 贷款平台 Kabbage。)像“无价”这样的服务也有助于万事达卡提高顾客的品牌忠诚度，并有可能通过更直接的零售获得更高的每用户收入。

与 Airbnb 等其他体验供应商一样，这些无价的产品似乎已经进入了完全虚拟的领域，让人们有机会在网上结识体育明星，去著名剧院的后台，并学习如何与知名调酒师调酒。这些现在可能由 IfOnly 提供支持，但只是部分支持:捐赠给慈善机构的选择权似乎并没有在交易中延续。
