Dialpad 收购视频会议服务 Highfive

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/01/dialpad-acquires-video-conferencing-service-highfive/

VoIP 提供商 Dialpad，即广受欢迎的视频会议服务 UberConference 背后的公司，今天宣布收购了资金雄厚的视频会议初创公司 Highfive，该公司专注于为企业提供会议室解决方案。两家公司没有透露收购价格，但在今天的收购之前，Highfive 从 Lightspeed Venture Partners、Andreessen Horowitz、General Catalyst 和 Dimension Data 等公司筹集了 7740 万美元。

Dialpad 的首席执行官克雷格·沃克(Craig Walker)曾将 GrandCentral 出售给谷歌，然后创建了谷歌语音，在他的领导下，dial pad 显然打算在视频领域加倍努力。虽然 UberConference 已经内置了视频会议功能，但这项服务最出名的还是它的通话功能。除了为商业用户提供电话会议解决方案和 VoiP 平台，Dialpad 还提供联络中心解决方案。

“八年前我们做 UberConference 的时候，我们就想，‘看，80%的会议都是人们在打电话。所以让我们把电话、音频会议做得更好，”沃克说然后，很明显，随着时间的推移，情况开始发生变化，然后 COVID 完全加速了这种变化。因此，随着这种加速，我们意识到我们真的想在视频上加倍努力，而不是抱着“嘿，视频作为一个独立的东西将是一项巨大的投资”的心态，而是视频作为商业通信的一部分，必须是优秀的，必须是统一通信即服务(UCaaS)系统的一部分。"

Highfive 是由一群前谷歌工程师发起的，一直专注于视频。沃克指出，这两家公司都诞生在云中，但直到现在为不同的客户服务。

Highfive 首席执行官 Joe Manuele 告诉我:“这一合并真正令人兴奋的是共同的遗产——两家公司都真正诞生在云环境中，运行在超大规模的全球基础设施上。”。" Dialpad 的会议、UCaaS 和 CCaaS 产品仅构建在公共云基础设施上，Highfive 也是如此。虽然视频是 Dialpad 当前产品组合的重要组成部分，但我们通过我们的会议连接器技术带来了连接房间、与其他视频服务互操作的能力，并通过我们的房间连接器提供了传统设备支持。除了产品适合度之外，我个人最感兴趣的是共享的行业愿景，即您可以在超大规模公共云环境中满足您的所有通信需求。"

Manuele 指出，公司董事会曾考虑过其他选择，包括新一轮的融资，但最终，公司决定视频会议服务现在基本上已经成为更大的 UCaaS 堆栈的一部分。

“虽然我们已经开发了一个可扩展的、诞生于云视频解决方案集的解决方案，但与竞争对手竞争变得越来越困难，这些竞争对手正在提供劣质的‘免费’视频服务，作为 UCaaS 堆栈的一部分，”他说。“即使是行业领导者 Zoom 也不得不转向 IP 电话，我们认为这一趋势不可逆转。”

显然，Dialpad 的沃克同意这一观点。“无论我是在打电话，无论是我的商务电话系统，还是我需要进行视频通话，或者我需要进行电话会议，或者我需要进行屏幕共享——如果我需要做这些事情，它都应该是一种[工具]，”他说。

Highfive 真正超越的一个领域是让其服务无缝运行。它通过紧密集成其硬件和软件堆栈来实现这一点，但也通过重新想象其房间系统的一些整体用户体验来实现。

沃克承认，现在没有人真正使用房间系统，但他认为，随着人们逐渐回到办公室，视频和远程会议可能会变得更加重要，因为大多数公司将为员工采用某种混合模式。

他相信此次收购也将使 Dialpad 在整个市场中占据强势地位，并使 Dialpad 能够为其客户提供完整的解决方案。

Highfive 的品牌可能最终会消失，但已经购买该公司系统的客户不会看到他们的服务有任何中断。
