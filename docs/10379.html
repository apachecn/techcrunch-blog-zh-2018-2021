电影无处不在正式推出其数字电影租借功能“屏幕通行证”

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/01/movies-anywhere-officially-launches-its-digital-movie-lending-feature-screen-pass/amp/

图片来源:随处可见的电影

数字储物柜服务电影无处不在今天正式推出其电影共享功能，名为“屏幕通行证”，让你可以将你购买的电影借给朋友或家人。由于冠状病毒政府封锁期间被困在家中的消费者的需求增加，这项功能在今年 3 月匆忙进行了测试，随后在 4 月进行了更加开放的公开测试。

如今，随处电影(Movies Anywhere)应用程序允许客户从多个服务集中访问他们购买的电影，包括 iTunes、Vudu、Prime Video、YouTube、Xfinity 和许多其他服务。

如今，该应用由迪士尼、环球、WB、索尼影业和 20 世纪福克斯联合运营。该产品本身是由 2014 年的迪士尼电影版演变而来，但后来在 2017 年迁移到一个新的平台。该应用程序也进行了重建，以适应更多的运营合作伙伴，更名，现在作为一个不同于过去几年的业务运营。

今年 4 月，电影无处不在(Movies Anywhere)报告称，其应用程序中有超过 6000 部电影符合屏幕通行证资格。从那以后，它又增加了 500 部电影，包括《谁陷害了兔子罗杰？、“一个明星的诞生·安可”、“提线木偶”和“国家宝藏”随着新功能的增加，超过 80%的图书馆书目可以通过新功能共享。总共有 6，700 部符合屏幕通过条件的电影。

图片来源:电影无处不在

若要共享标题，您可以点按标题上的屏幕通行证图标，然后输入详细信息，例如收件人的信息。共享电影可以通过文本、电子邮件或信息发送给收件人，收件人有一周的时间接受。在大多数情况下，共享电影就像数字电影租赁一样工作，因为在电影开始后，观众将有多达 14 天的时间观看和多达 72 小时的时间观看。但与租赁不同，接收者不必付费观看——分享和观看都是免费的。

该公司表示，Screen Pass beta 测试的早期数据表明，该功能有可能推动新的收购和购买。

45%的发送者使用 Screen Pass 共享了一部电影，因为其他人首先与他们共享了一部电影。收到共享电影的人中有 30%是首次使用电影随处看平台。少数几部电影推动了约 9%的总购买量，包括《头号玩家》(Ready Player One)、《威望》(The Prestige)、《墓碑》(Tombstone)、《骡子》(The Mule)、《皇家剧院的糟糕时光》(Bad Times at El Royale)和《大白鲨》(Jaws)。该数据表明，Screen Pass 份额并不像人们预期的那样仅限于较新的游戏，还包括较老的经典游戏。

此外，大约一半的分享者(53%)选择他们借出的电影，而 47%的人让接收者选择。但这可能是由于筛选通过测试的结构，因为接收者将通过接受另一个成员的份额来选择进入测试。很可能早期团队的某个部分只是通过与他们共享一个标题来邀请他们的朋友。(“选择”功能也是后来在试验中添加的。)

除了屏幕通行证共享，电影无处不在最近还推出了一项名为一起观看的共同观看功能，可提供多达九人的同步观看体验。该产品通过 Screen Pass 工作，并与疫情涌现或越来越受欢迎的各种解决方案竞争，包括来自 Hulu 、亚马逊和第三方如 Scener 和网飞党等。

Screen Pass 今天向美国所有的电影随处用户推出。电影随处应用可以在一系列设备上运行，包括 iOS 和 Android 移动设备、Apple TV、Roku、Kindle Fire、亚马逊 Fire TV、Chromecast 以及 LG 和 Vizio 智能电视。
