美国上诉法院裁定NSA通话记录收集非法TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/03/nsa-bulk-records-appeals-illegal/

美国国家安全局(National Security Agency)运行的一个收集数十亿美国人电话细节的项目周四被美国上诉法院裁定为非法。

第九巡回上诉法院发现国家安全局的“批量收集”通话记录违反了法律，但是法官没有裁定该项目违宪。

国家安全局在911恐怖袭击后使用了新的权力——因其在法律书籍中的地位而被称为第215条——通过迫使美国电话巨头交出日常通话日志，每年收集数十亿条电话记录，该机构利用这些记录在感兴趣的目标之间建立联系。这些通话记录包括谁在什么时候给谁打电话，但不包括内容。

该项目的细节由前美国国家安全局承包商爱德华·斯诺登于2013年曝光。

但是这个电话记录项目，被问题、过度收集和对其合法性的质疑所困扰，于去年被关闭。

美国公民自由联盟国家安全项目的高级律师帕特里克·图米(Patrick Toomey)表示，这一裁决是隐私权的“胜利”。

“这项裁决清楚地表明，美国国家安全局大规模收集美国人的电话记录违反了宪法。该决定还承认，当政府试图起诉一个人时，它必须通知它用来收集证据的秘密监视。考虑到如今政府使用的新型间谍工具的扩散，这种保护是至关重要的。"

第九巡回法院的案件涉及巴萨利·莫阿林和其他三人，他们在2013年因向激进组织青年党送钱而被判有罪。政治报道说，莫阿林被定罪的部分原因是因为美国国家安全局收集的通话记录，但是这些数据所起的作用是如此之小，以至于并没有破坏他们的定罪。

国家安全局一直声称该项目对于保护美国本土阻止恐怖袭击至关重要。往届政府声称该计划阻止了50多起攻击。但是在国会审查之后，这个数字被修改为一个确定的个人——莫阿林。

尽管法院没有推翻对莫阿林的定罪，但由三名法官组成的陪审团批评了政府此前关于该计划的实用性和有效性的声明和评论，法院称这些声明和评论“与机密记录的内容不符”

公民自由专家、卡托研究所(Cato Institute)高级研究员朱利安·桑切斯(Julian Sanchez)在推特上写道:“第九巡回法院的意见是，美国国家安全局大规模收集电话记录是非法的，很可能违反宪法，但这无关紧要，因为该项目也毫无价值。”

当被问及国家安全局是否坚持其先前的声明时，该机构发言人拒绝置评。
