荷兰支付初创公司 Mollie 以$1B+估值融资 1.06 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/07/dutch-payments-startup-mollie-raises-106m-at-1b-valuation/

在过去的八个月里，电子商务经历了巨大的飞跃，这是由于在新冠肺炎疫情期间，消费者在网上购买更多的商品，同时花更多的时间呆在家里。如今，一家位于阿姆斯特丹的支付初创公司宣布将获得一大笔资金，以帮助其继续扩大产品和国际足迹，从而满足这一需求。该公司今年也因此实现了大幅增长。

Mollie 是一家初创公司，提供一种简单的、基于 API 的方式将支付集成到一个网站或应用程序中，该公司在 TCV 领导的一轮融资中为€筹集了 9000 万英镑(1.06 亿美元)。创始人兼首席执行官阿德里安·摩尔在接受 TechCrunch 采访时证实，B 轮融资使莫莉为€筹集的资金总额达到 1.15 亿美元，并使这家初创公司的估值超过 10 亿美元。

Mollie 自 2004 年以来一直存在，这只是它第二次筹集资金——第一次是一年前€的 2500 万美元融资——这可能是它一直不在初创公司雷达上的一个原因。

“当我还和父母住在一起时，它就自己构建了后端和前端，”Mol 说。“这是荷兰人的方式。引导你的想法很长一段时间。我认为这是公司的基础。”

然而，Mollie 在过去几年中达到了一些里程碑，这在一定程度上解释了为什么它现在出现在舞台上，看似凭空出现，估值如此之高。

目前，该公司主要专注于中小企业——一个服务水平极低但规模巨大的市场——并拥有约 10 万名商户客户，主要在荷兰、比利时和德国。这些客户包括一些知名度很高的公司，如 Wickey、Deliveroo、TOMS(制鞋公司)和联合国儿童基金会。

今年，它有望处理超过 100 亿€的交易，同比增长 100%，德国等一些市场增长 1000%。到目前为止，它已经盈利多年。

“我们站在效率的正确一边，”Mol 说——Mollie 是他在朋友中的昵称。“但我们需要投资新产品，以保持竞争力。”

事实上，支付服务提供商的市场相当拥挤，Stripe 和 Mollie 的同胞 Adyen 等公司也在提供 API 和几行简单代码的概念上建立了强大的业务，以将支付流程集成到其他服务中。这些公司不仅受欢迎，而且资本充足，准备继续开发更多的工具，并继续扩大其国际足迹。

Mol 说，Mollie 在两个关键方面与他们和其他竞争者不同。首先，它在一个仍然非常分散的市场中提供非常本地化的支付产品，每个国家的消费者和商家对使用何种支付方式的偏好都有很大差异(Stripe 等公司仍未完全提供服务)。

第二，它提供了一个非常快速和简单的集成，隐藏了集成如此多不同类型的支付方法的许多困难。对商家和顾客来说，这种易用性意味着购物车的废弃率更低，因此网站访问者的转化率更高。Mol 表示，转换凸起通常可高达 7%。

“如果旅程非常顺利，他们不会退出，这是我们客户的直接收入，”他说。(定价透明但不统一:取决于成交量，整合使用哪些支付方式。)

这一点，再加上一家非常盈利和高效的公司的增长率，是吸引 TCV 的部分原因。

“易用性非常重要，”TCV 的合伙人约翰·多兰说。“它非常简单，连小孩都可以使用。我们的想法是打造支付世界的苹果。”

Mol 第一次想到建立 Mollie 的想法是在他将支付服务整合到 MessageBird 时，MessageBird 是他之前的初创公司，提供基于 API 的消息服务(可以把它看作是欧洲的 Twilio)。当时，他说他发现所有可用的支付产品都很差，而且很难按照他想要的方式使用，所以他们——特别是工程师 Mol 自己——在内部构建了解决方案(字面意思是他父母的房子:见上文)。

我问该公司为什么不简单地扩张和/或转向在一个保护伞下提供这两种服务，但 Mol 说这两种业务和想法太大了，没有足够的相似性来作为一个业务发展。因此，他离开了 MessageBird 的执行领导层——尽管他仍在董事会——以便全职关注莫丽。

可以说，专注于支付业务的转变取得了回报。

Mol 表示，该公司的下一步将是继续建设与支付相关的服务——例如可能为客户提供营运资金的领域，以及中小企业通常无法从单一提供商那里获得的其他金融服务。“我们看到一大堆传统银行不向中小企业提供的产品，”Mol 说。“他们没有动力投资于此，因为他们从中赚不了多少钱。”他补充说，这些业务可能包括销售点支付、信用卡发行、银行发行和其他银行产品。
