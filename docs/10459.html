Revolut 在日本推出其金融应用

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/07/revolut-launches-its-financial-app-in-japan/

金融科技初创公司 Revolut 正在向日本扩张。在对 10，000 名用户进行测试后，现在任何人都可以注册并开设账户。该公司最初于 2018 年从日本金融服务厅获得运营授权。

当你开户时，你会得到一个电子钱包和一张 Visa 借记卡。你可以给你的账户充值，用你的卡、虚拟卡等消费。Revolut 向您发送即时通知，并允许您从应用程序中冻结和解冻您的卡。

您也可以向其他 Revolut 用户或银行账户汇款。像在其他国家一样，Revolut 允许你在应用程序中兑换货币，并以其他货币汇款。许多用户已经利用这项服务进行旅行，并减少了外汇费用。

日本的用户也可以创建金库，并通过汇总交易和创建重复交易来储存一些钱。暂时就这样了。

该公司已经在日本推出了高级计划，但除了较低的外汇兑换费用、不同的卡设计、更好的支持以及使用 LoungeKey Pass 购买机场候机室的能力之外，它并没有给你带来很多好处。

与英国和欧洲不同，你将无法购买加密货币、交易股票、购买保险产品、为你的孩子创建 Revolut Junior 账户等。Revolut 真的试图在自己的国家建立一个超级应用程序，并在过去几年里大规模扩展了它的功能集。

该公司承诺，加密货币和股票交易等一些功能将在全球范围内提供。但是还没有发布日期。因此，让我们看看该产品在未来几个月将如何发展。

Revolut 目前已在英国、欧洲、美国、新加坡和澳大利亚上市。它目前拥有 1300 万客户。
