谷歌新推出的“核实来电”功能会告诉你为什么一家企业会给你打电话

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/08/googles-new-verified-calls-feature-will-tell-you-why-a-business-is-calling-you/

谷歌今天为 Android 手机推出了一项新功能，通过正确识别品牌名称和致电原因，帮助合法企业通过电话联系客户。这项功能名为“已核实来电”，将显示来电者的姓名、他们的标志、他们来电的原因以及一个验证符号，表明该来电已被谷歌核实。

这项功能的出现正值垃圾电话日益增多之际。根据 RoboKiller 最近的报告，美国消费者在 2019 年收到了 614 亿个垃圾电话，比上一年增长了 28%。美国联邦通信委员会也表示，不受欢迎的电话是其最大的消费者投诉。

谷歌的新系统为合法企业提供了一种与消费者分享信息的方式，以及他们在来电屏幕上打电话的原因。然而，这只适用于那些选择与谷歌的合作伙伴之一签约的参与企业，以便对他们的呼叫进行验证。

根据谷歌的服务网站，企业可以通过与 Neustar、JustCall、Telecall、Zenvia、Prestus、Aspect、Five9、Vonage、Bandwidth、IMImobile、Kaleyra、Quiubas Mobile 或 Datora 等合作伙伴合作，开始进行验证呼叫。

图片来源:谷歌

一旦设置完成，企业将向谷歌的验证呼叫服务器发送其号码、客户的电话号码和呼叫原因，例如“安排您的互联网安装”或“您的食品配送”。然后，谷歌将这些信息发送到 Android 设备的谷歌手机应用程序。该设备将来电信息与谷歌从该公司收到的信息进行比较，如果匹配，手机应用程序会显示该电话为“已验证”谷歌表示，为了保护消费者隐私，用户的电话号码和来电原因会在核实后几分钟内被删除。

该功能在适用于 Android 设备的谷歌手机应用程序中默认处于启用状态，许多 Android 手机都预装了该应用程序，并将于本周晚些时候供更多 Android 设备下载。在发布时，它将在选定的 Android Pie 和更高的设备上工作，包括许多三星和 LG 的旗舰设备。

谷歌表示，在上线之前，它对这项新功能进行了几个月的试点测试，发现验证确实增加了有人接听电话的机会。它没有分享具体结果。

图片来源:谷歌

然而，谷歌现有的验证短信系统已经被的许多品牌采用，包括 1-800-Flowers、Banco Bradesco、Kayak、Payback 和 SoFi 等。谷歌声称，美国和巴西的一项研究发现，经过验证的短信增加了客户对品牌的信任，并提高了购买可能性、品牌满意度和推荐可能性等指标。

Verified Calls 首先在美国、墨西哥、巴西、西班牙和印度推出，随后会在更多国家推出。

谷歌已经为消费者提供了一种方法，通过谷歌助手功能“来电显示”来对抗垃圾邮件。这项功能允许谷歌助理代表用户接听电话，然后询问他们是谁打来的，为什么。一份抄本被发送给手机的主人，然后他可以选择发送一个建议的回复，接电话或挂断电话。

但是根据谷歌的帮助文档显示，在美国，来电显示可以自动使用英语，在加拿大可以手动使用。与此同时，Verified Calls 在全球更多国家提供，并利用行业合作伙伴关系来取代人工智能，使其成为一种更广泛的解决方案。
