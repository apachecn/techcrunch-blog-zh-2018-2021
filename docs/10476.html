Hasura筹集了2500万美元的B轮融资，并在其GraphQL服务TechCrunch中增加了MySQL支持

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/08/hasura-raises-25-million-series-b-and-adds-mysql-support-to-its-graphql-service/amp/

图片来源:Fernando Trabanco foto grafía/Getty Images

Hasura ，一个为开发者提供开源引擎的服务，为他们提供 GraphQL API来访问他们的数据库，今天宣布它已经筹集了2500万美元的B轮融资，由Lightspeed Venture Partners牵头。之前的投资人Vertex Ventures US、Nexus Venture Partners、奋力VC、SAP.iO Fund也参与了此轮。

该团队在新冠肺炎疫情已经启动后筹集的新一轮融资，距离该公司宣布其990万美元的首轮融资仅六个月。现在，哈苏拉总共筹集了3650万美元。

“我们已经看到了2020年企业的快速发展。我们希望加快对 Hasura 社区和我们最近推出的云产品的投资，并确保我们的企业客户取得成功。鉴于风险投资的兴趣，融资有助于我们加大油门，给我们提供舒适增长的空间，”Hasura联合创始人兼首席执行官Tanmai Gopal告诉我。

除了新的资助，Hasura今天还宣布增加了对MySQL数据库的支持。到目前为止，该公司的服务只适用于PostgreSQL数据库。

联合创始人拉乔希·戈什和首席运营官(左)以及联合创始人兼首席执行官坦迈·戈帕尔(右)。图片来源:哈苏拉

正如该公司的首席执行官和联合创始人坦迈·戈帕尔告诉我的那样，MySQL支持长期以来一直是该服务用户最需要的功能之一。这些用户中的许多人(通常在医疗保健和金融服务行业)也在使用传统系统，他们试图连接到现代应用程序，MySQL在这方面发挥着重要作用，因为它已经存在了很长时间。

除了增加对MySQL的支持，Hasura还增加了对SQL Server的支持，但目前还处于早期阶段。

“对于MySQL和SQL Server，我们已经看到医疗保健和金融服务/金融技术用户的大量需求，”Gopal说。“他们有大量现有的在线数据，尤其是在这两个数据库中，他们希望激活这些数据来构建新的功能，并在实现应用程序现代化的同时使用这些数据。

今天的声明也是在该公司为其服务推出完全托管的云服务几个月后发布的，该服务补充了其现有的企业付费专业服务。

Lightspeed Venture Partners合伙人兼Hasura董事会成员Gaurav Gupta表示:“我们对开发人员采用Hasura并采用GraphQL方法构建应用程序的方式印象深刻。“特别是对于使用React等技术的前端开发人员来说，Hasura可以轻松地将应用程序连接到现有的数据库，所有数据都在数据库中，而不会影响安全性和性能。Hasura为将应用程序移植到云原生方法提供了一个可爱的桥梁，因此我们看到这种方法越来越受到企业开发人员和前端开发人员的欢迎。”

该公司计划使用新的资金来增加对更多数据库的支持，并解决一些围绕跨数据库连接和公司应用程序级数据缓存系统的更困难的技术挑战。“我们还在公司建设方面进行了深入投资，以便我们能够协同发展我们的GTM和工程，并在这些职能部门招聘一些高级员工，”Gopal说。
