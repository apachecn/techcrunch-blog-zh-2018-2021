Twitter 开始为它的一些“趋势”添加标题和描述

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/08/twitter-begins-adding-headlines-and-descriptions-to-some-of-its-trends/?tpcc=ECTW2020

Twitter 正在努力使其实时趋势部分不那么令人困惑。上周，公司宣布将开始在潮流页面上发布一条有代表性的推文，这条推文提供了关于潮流的更多见解，并承诺更多变化将很快发生。今天，该公司表示它也将开始为一些趋势撰写标题和描述，所以你会更好地理解为什么一些东西会出现在探索标签中，或者当你进入一个趋势本身时。

综合来看，这些变化可能会让 Twitter 的趋势部分感觉更像新闻播报员的体验，如果它们在趋势产品中得到适当和全面的推广的话。

例如，在撰写本文时，Twitter 并没有只看到说唱歌手“Travis Scott”排名第一，而是通过标题和简短摘要的方式解释说，他的排名是因为他与麦当劳的菜单合作刚刚启动。这很有帮助——尤其是因为当一个名人去世或说了一些离谱的话时，他的名字往往会成为潮流。

不幸的是，尽管 Twitter 最近进行了更新，但它仍然缺乏正确注释其趋势的能力。

有许多趋势仍然无法解释——特别是那些以标签为首的趋势，比如今天的#IJustDontBelieve，用户在推特上发布他们不相信的东西，在推特的其余部分填写他们自己的观点。KPop 团体 BTS 的粉丝们所推动的长标签#BTS2ndNo1ONHot100 也没有被标注。对于不熟悉 BTS 粉丝如何大量使用 Twitter 或 Twitter 的填空迷因受欢迎程度的新人来说，这些趋势可能会令人困惑。

在其他趋势未被标注的情况下，Twitter 让新闻标题做添加背景的工作。

通常，几个热门趋势会集中在当天的新闻上。但是你会发现一个有代表性的新闻标题链接在趋势旁边，而不是 Twitter 提供的摘要和标题。

图片来源:TechCrunch 的推特截图

举例来说，“骄傲的男孩”是写作时美国的第 17 大潮流。但是 Twitter 只链接到网站 Daily Beast 上的一篇文章，这篇文章提到了极右翼骄傲男孩和抗议者之间的暴力冲突。它没有试图解释为什么这篇详细描述了美国众多暴力事件之一的文章会成为热门话题。用户大概是想推断文章本身受到了很多关注。但事实上，推特用户在“骄傲的男孩”潮流下发布了各种内容，包括他们自己的关于暴力袭击的视频和对峙。

通过这种方式，当人们不一定在谈论文章本身时，Twitter 只指向一篇新闻文章，这是在做坏事——他们在分享他们目睹的直接镜头或他们对暴力事件不断增加的总体看法。

与此同时，Twitter 已经让其管理团队开始总结不太重要的新闻，比如关于哈里·斯泰尔斯新形象的趋势。

目前，Twitter 在美国的趋势版块列出了 29 种趋势，但目前只有大约 6 种趋势有 Twitter 撰写的标题或描述。

Twitter 在上周宣布这些变化时曾表示其策展团队撰写的描述旨在提供直截了当、来源明确的背景，解释为什么某些东西在“一些”推文中成为趋势。

然而，尚不清楚的是，Twitter 是如何挑选和选择要注释的趋势的。

从理论上讲，如果 Twitter 在产品上投入更多，它本可以提供更多关于其趋势的背景信息。有第三方 Twitter API 合作伙伴可以生成数据，如趋势何时打破，它看到的推文的速度和数量，围绕趋势的社会情绪，推文生成的位置等等。但这类数据无法在 Twitter 上直接获得。

当被问及为什么只有一些趋势有 Twitter 提供的解释时，Twitter 发言人解释说，Twitter 只会注释那些它认为需要额外信息的趋势。

“如果一个趋势特别令人困惑，很多人都在谈论它，它可能会得到一条固定的推文或描述，”这位发言人说。这意味着 Twitter 正在做出编辑决定，在最需要的时候提供更少的内容。

例如，如果大多数关于抗议的推文要么支持要么对所述事件感到恐惧，Twitter 将这种趋势与数据联系起来的能力将非常有用。但它似乎没有接受这些困难的挑战。

该发言人补充说，Twitter 希望随着时间的推移，为更多的趋势添加更多的背景。
