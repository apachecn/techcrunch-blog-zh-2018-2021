前国家安全局局长基思·亚历山大将军现在是亚马逊的董事会成员

爱德华·斯诺登揭露美国国家安全局(National Security Agency)非法窃听和数据收集项目的惊人程度时，该机构负责人基思·亚历山大(Keith Alexander)将军已加入亚马逊董事会，担任董事。

亚历山大在审计委员会的职责以及其他任何可能需要他的地方都没有详细说明。他目前是IronNet Security的联合首席执行官，该公司是他六年前创立的。在此之前，他是国家安全局和美国网络司令部的负责人。

他也许最被公众铭记的是，在安全第一的后9/11时代，他帮助建立和运作了一套庞大的秘密国内监控计划。当然，管理这个国家的网络安全基础设施远不止这些，但斯诺登泄密事件最终决定了他在政府情报部门职业生涯的终结。

亚马逊本身也面临着通过Alexa驱动的设备网络(以及互联网基础设施、购买习惯和情绪监控智能手表)监控和描述自己用户的指控，虽然它可能会从更有经验的Alexander将军那里获得一些建议，但更有可能的是他在该公司寻求的情报和军事事务的广泛世界中的专业知识和关系。

当试图与联邦政府达成有利可图的交易时，这种事情是有帮助的，这是亚马逊的一个痛处，因为它在漫长的竞标过程中输给了微软。(该奖项仍在接受挑战。)

亚历山大将军将加入百事可乐、星巴克、比尔和梅林达盖茨基金会、布里奇沃特协会等公司的前任和现任高管的董事会。

我已经联系了亚马逊以获取更多的信息和评论，如果有回音，我会更新这篇文章。

更新:亚马逊指引我查看其SEC官方文件以获取更多信息，但想强调“我们将继续遵守严格的政府合同利益冲突规则。”换句话说，他哪儿也进不了他们的大门——尽管毫无疑问，他的简历在其他方面将证明是非常宝贵的。
