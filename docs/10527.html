微软推出新Xbox游戏机月度融资计划

每月融资在Xbox的世界里并不是一个全新的概念。几年前，微软为Xbox One S提供了类似的计划。这个想法很简单:在两年内每月支付硬件和软件费用，直到你完全拥有这个设备。然而，这里的新情况是，该公司正在介绍将于今年晚些时候推出的全新游戏机的计划。

随着S系列产品的发布，微软详细介绍了两个新计划，旨在让那些不愿意或不能预先支付299或499美元购买新系统的人获得游戏机。这一举措极大地扩展了该系统的可访问性，甚至超过了最近宣布的低成本模型。

这一举措符合最近重新燃起的对硬件即服务模式的兴趣。我们已经看到许多像Zoom这样的公司在不同程度上接受了这一点。虽然真的，租赁自有模式与智能手机合同有很多共同之处——尽管这些合同近年来在美国已经在某种程度上开始失宠。

在这里，每月25美元将获得一台S系列主机，与Game Pass Ultimate捆绑在一起。与此同时，每月35美元，你就可以获得Game Pass Ultimate和Series X，无需预先支付任何费用。鉴于Game Pass流媒体服务对下一代游戏机的重要性，这是一笔相当不错的交易。毕竟，Game Pass Ultimate每月只需15美元，不含硬件接入费用。

据估计，PlayStation 5的定价在450-550美元之间，索尼在积极定价方面有一个艰难的举动。尽管PS5可以说比下一代Xbox更令人兴奋，但每月25美元的入门价格很难与之竞争。
