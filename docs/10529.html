提醒:额外 Crunch 会员可获得 1000 美元 AWS 激活积分

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/09/reminder-1000-in-aws-activate-credits-available-for-extra-crunch-members/

召集所有额外的 Crunch 成员！我们已经与亚马逊网络服务合作获得额外的 Crunch 合作伙伴津贴，现在符合条件的会员可以获得 1000 美元的 AWS 激活积分，作为年度额外 Crunch 会员计划的一部分。

如果你已经是一个额外的 Crunch 成员，请向下滚动到帖子的底部，了解如何申请积分。如果你还不是会员，请继续阅读。

什么是 AWS？

AWS (Amazon Web Services)是满足您的应用程序托管需求的首要服务，我们希望确保我们的社区有充足的资源来构建。我们理解托管和基础设施成本可能是科技创业公司的一个主要障碍，我们希望这一提议将有助于更好地支持您的团队。

福利包括什么？

  • 1，000 美元的 AWS 激活信用，有效期为一年
  • 价值 350 美元的 AWS 开发者支持积分，有效期一年

我如何知道我的创业公司是否合格？

并非所有公司都有资格获得 AWS 激活积分。要申请 AWS Activate Founders，你的初创公司必须符合以下标准:

  • AWS 激活创始人新手
  • 以前没有从 AWS 激活产品组合获得积分
  • 拥有有效的 AWS 帐户
  • 启动必须是自筹资金，无支持或自举-没有机构资助或与激活提供商的从属关系
  • 拥有 LinkedIn 个人资料(申请人)
  • 拥有一个公司网站或网页简介
  • 成为一年或两年的额外会员

申请在 7-10 天内处理完毕。

什么是额外嘎吱声？

Extra Crunch 是 TechCrunch 的会员计划，帮助您发现技术趋势和机会，建立更好的创业公司并保持联系。它有数千篇文章，包括每周投资者调查、每日市场分析和关于筹资、增长、货币化和其他工作主题的专家采访。
