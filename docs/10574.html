被吹嘘的云加速度是平掉的吗？

本周我们来看看云软件市场不好的一面。如果你在回避上周的新闻，科技和软件股票 正在挣扎 。请注意，与 2020 年的涨幅相比，这并不算多，但经过几个月的上涨后，最近的下跌已经非常明显。(就在我给你写信的时候，以科技股为主的纳斯达克指数正走向自 3 月份以来最糟糕的一周。)

回调是有一定道理的。眼看着 SaaS 和云的估值 升至历史高点 ，Slack 的收益还不错，但公有云和 SaaS 公司的一组业绩 却不尽如人意 。

我们已经注意到，大多数上市软件公司 都是 而不是 看着自己的收入增长加速 。一些公共软件公司可能会看到他们的增长减速缓慢，但实际上 2020 年公共软件公司的数量 加速 是微小的。 其实——加速组就是 Zoom ，可能还有一两家其他公司。

鉴于我们所听到的关于加速数字化转型的说法，这是为什么呢？ 农闲收益好交代 。企业通信公司最近的文件解释说，其 COVID-bump 已经有所消散，而许多与 COVID 相关的 问题 仍然存在。

在缺乏实质性加速增长和一些流失问题的情况下，看到最近上涨的估值下滑是合理的。

这对创业公司有影响吗？一些。与历史标准相比，公共软件的估值仍然很高，这有助于软件初创公司捍卫自己的估值，并提高估值。我们已经注意到有大量的创业热点，包括 享受阳光的 API 交付的创业公司 ，以及赶上 COVID 相关顺风的 edtech 创业公司 。

下周，我将在 Disrupt 与来自 a16z、Bessemer 和 Canaan 的投资者谈论 SaaS 的未来，收集关于这个特殊问题的私人市场方面的笔记。所以，更多的来了。但现在，我认为我们已经看到了顶峰，现在更多的是面对现实而不是炒作。或者，正如公众投资者可能会说的那样，COVID 交易已经结束，收益将为未来定下基调。

各种各样的

SaaS 和云计算的收益持续流入，这意味着我一周的大部分时间都在和更多上市公司的高管交谈。接下来是来自 Smartsheet、nCino 和 BigCommerce 的简短笔记，以及您周末的一些最终想法。

  • 在估值方面， Smartsheet 首席执行官马克·马德 告诉 TechCrunch，“投资者正在考虑如何在这个仍然非常年轻的领域平衡历史高倍数和历史高潜在回报。”
  • 他补充说，没有人怀疑云“将成为许多事情的答案”，或者“人们将改变他们的工作方式”，但他指出，云公司并非不受宏观逆风的影响，因为“云公司服务于非云公司”，而不仅仅是那些优秀行业的公司。
  • 这正好符合我们上面关于 Slack 的笔记。更多关于 Smartsheet 的收益 这里 。
  • nCino 这个季度表现不错， 在其首次公开收益报告中，表现超出预期，并很好地指导了 。然而，像许多其他 SaaS 和云公司一样，它在最近几周失去了一些估值高度。然而，这仍然比其首次公开发行的价格高了好几英里。
  • 我很好奇这家公司的首席执行官皮埃尔·瑙德(Pierre naudé)在上市后的这段时间过得怎么样，他的回答很有趣。像所有新上市公司的首席执行官一样，他确保记录了他的团队在首次亮相后恢复工作的速度，但他也告诉交易所，他现在确实把过去投资于客户和“创新”的时间花在了与分析师和投资者交谈上。
  • 因此，作为一家上市公司，有值得考虑的时间和关注成本，因为我们看到如此多的科技公司走向公开市场。
  • 然后就是 BigCommerce，最近 上市 。我再次与首席执行官布伦特·贝尔姆(Brent Bellm)通了电话，想多了解一点电子商务市场的现状。
  • 以下是这位首席执行官所说的话，为清晰起见，略加编辑和浓缩:

“我觉得天气一直很热。在大流行后的几周里，令人惊讶的事情是增长加速的速度，以及消费者和企业的采用率增长的速度。我们一直在说，总有一天商店会重新开张，增长率会回落。但是商店实际销售额的增长率仍然非常强劲。你知道，无论你看我们的客户群，还是美国银行或其他机构的信用卡数据，你都可以清楚地看到，电子商务仍然非常非常热门。这是行为上的永久改变。消费者找到了更多他们喜欢网上购物的地方和理由，公司也找到了新的更有效的销售方式。

这是我们所有的房间。拥抱，拳头碰撞，还有好运。
