黑科技管道证明“管道问题”并不存在

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/15/black-tech-pipeline-proves-the-pipeline-problem-isnt-real/

科技行业已经成为一个白人和男性占主导地位的地方。与被误导的普遍看法相反，这与其说是一个渠道问题，不如说是一个保留问题。

黑人技术管道旨在揭穿这一神话，并帮助支持黑人技术专家各自的旅程。由Pariss Athena创建的Black Tech Pipeline有三个关键产品:一个求职板，任何人都可以付费获得推荐，一项招聘和咨询服务，以及与活动潜在发言人的联系。

“黑科技管道的目标是让现有的黑人技术专家社区曝光，”雅典娜说。“我们之所以关注那些在这里的人，是因为我们想揭露整个‘管道问题’并不真实。我们一直在这个行业。我们存在于其中。我们已经在这里很多年了。我们是前辈，我们是中层，我们是一个范围。但我们也想为业外人士创造这种涟漪效应，向他们展示你有一个像你一样的人群，他们会在你的旅程中支持你。”

目前，数据库中有近700名黑人技术专家。截至7月，8.66%的候选人没有工作经验，37.33%有一到两年的工作经验，27.33%有两到三年的工作经验，22%有五到十年的工作经验，10.5%有十年以上的工作经验。

就业公告板上的公司是黑科技渠道的付费客户。这意味着他们可以获得一个定制的登录页面，在这里他们可以描述他们可以担任的角色，并详细说明他们的价值观，公司如何看待多元化和包容性等。这些公司还将出现在Black Tech Pipeline的时事通讯和社交媒体平台上，这些平台总共拥有超过4万名粉丝。

雅典娜说:“我们希望给出一个非常透明的观点，作为一个最有可能成为职场少数派的人，在那里工作意味着什么。”

雅典娜的第一份工作是软件工程师，她被解雇后，在Twitter上变得更加活跃，于是她萌生了创建黑人技术专家数据库的想法。

“当我加入时，我注意到有一个非常小的黑人技术专家社区，这很有趣，因为，至少在波士顿这里——我敢肯定这是无处不在的——但我不仅一直是开发团队中唯一的软件工程师，而且自从我进入这个行业以来，几乎是整个公司中唯一的黑人，”雅典娜说。

“因为我很少看到像我这样的人，所以我想我们并不是真的在这个行业，所以当我看到我们几个人的时候，我就想，‘哦，酷，我想知道我们还有多少人在那里？’我发布了一条推文，问科技领域的黑色推特长什么样？出乎意料的是，这条推文变成了半病毒式传播，来自世界各地的许多黑人技术专家在推文中发布了自己的照片，并为他们在这个行业中所做的事情添加了超长的标题。一夜之间，它真的形成了这个黑科技Twitter的运动和社区。"

她说，同一周，雇主们开始联系雅典娜，请她帮助他们招聘黑人。尽管没有任何招聘经验，雅典娜还是同意帮忙。她开发了一个人才数据库，以便将求职者与雇主联系起来，还开发了一个附带的应用程序，可以自动过滤到数据库中。

在早期，Athena说她注意到虽然许多候选人被录用了，但他们中的许多人没有被保留。

“所以就有了这个留住人才的问题，”她说。“作为一个在白人占主导地位的空间里一直是唯一的黑人的人，我知道发生了什么。”

帕里斯·雅典娜，黑科技管道的创始人。雅典娜供图。

这就是为什么雅典娜创造了一个顾问包，并开始向公司收费。只要候选人通过自己的努力被录用，她就会在头90天内每两周进行一次检查，看看他们做得怎么样。

“这是我确保不把他们送到有害地方的方法，”雅典娜说。“考虑到他们的同意和安全，我会将他们给我的反馈转达给雇主。”

这样做的目的是帮助雇主改善他们的工作文化，以及带有偏见的系统和流程。最初，Athena免费做这件事，只是通过她的Twitter直接消息进行操作。雅典娜已经开始向公司收取这些服务的费用，作为她公司黑科技管道的一部分。

尽管Athena在黑科技管道上投入了所有的努力，这还不是她的全职工作，但她希望它是。理想情况下，她会更专注于咨询部分。

她说:“我真的很高兴能成为这个几乎是外部的人力资源，但是是为候选人，而不是为公司。”。“所以，我在这里不是为了保护公司，确保它看起来很好，不管他们做了多少伤害。我来这里是为了确保这个人有一个好的体验，如果没有，我会让你负责，我们会努力解决这个问题。所以这才是我真正想做的——做出改变，让人们信守诺言。我不想和任何会表演的人一起工作。”

到目前为止，在她工作过的12家公司中，有9家被录用。从技术上来说，有更多，但雅典娜说，她没有计算那些已经被雇用但没有呆很长时间的人。她说，留住人才的问题部分反映了她在大公司遇到的一些问题。

雅典娜说，其中一个问题是，公司中真正希望实施变革的人和不太愿意这样做的高层之间存在脱节。另一个原因是，一些被雇佣的人经历了微暴力，这让他们想要离开。

“所以当这些事情发生时，我会向经理报告，但他们不知道如何处理，”雅典娜说。“他们会说这不够充分，或者说这是巧合。当我无法像这样与雇主沟通时，我会通过检查和报告反馈来完成与他们的合同，但我不会深入了解他们当前的流程，看看他们可以改进的地方。我不再为那些类型的雇主那样做了。所以我只是完成合同，然后继续前进。”
