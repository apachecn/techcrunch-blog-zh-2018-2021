虚拟活动平台 Airmeet 筹集了 1200 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/15/virtual-events-platform-airmeet-raises-12m/

提供虚拟活动平台的初创公司 Airmeet 周二表示，由于总部位于班加罗尔的公司显示出其用户群的加速增长，它已经在新一轮融资中筹集了 1200 万美元。

红杉资本印度公司在一岁的 Airmeet 中领投了 1200 万美元的 A 轮融资。Redpoint Ventures 和现有投资者 Accel India、Venture Highway、Global Founders Capital (GFC)和 Gokul Rajaram(door dash 的 Caviar Lead)也参加了这一轮投资。

据知情人士透露，新一轮融资对 Airmeet 的估值约为 5，000 万美元，比 3 月份融资 300 万美元时的估值高出一倍多。

Airmeet 允许用户和企业举办互动虚拟活动。它的平台直观地复制了实体活动的各个方面，提供了后台，将人们聚集到一张桌子旁，允许参与者相互交流，甚至允许活动组织者与赞助商合作。目前处于公测阶段的 Airmeet 是通过一种免费增值模式提供的，它根据用户的使用情况向企业收费。

在接受 TechCrunch 采访时，Airmeet 的联合创始人 Lalit Mangal 表示，在没有任何广告投资的情况下，该平台的使用量在上个季度增长了 2000%。

最近几个月，Airmeet 一直致力于扩展该平台的用例。他说，除了举办大型会议，Airmeet 现在还被用于大型电影节的专业聚会。最近，它举办了大学资源交易会和技术产业峰会。

“新冠肺炎加速了许多行业的永久性行为转变。随着大部分传统空间的数字化逐年飞跃，价值 8000 多亿美元的全球线下活动空间正在被争夺。红杉资本印度副总裁 Abhishek Mohan 在一份声明中表示:“推动行业向在线赛事转型的玩家潜力巨大。

Airmeet 构建在 WebRTC 之上，这是大多数现代浏览器遵循的标准。这使得 Airmeet 完全可以通过 Chrome 和 Firefox 访问。Mangal 说，所有的会话都是端到端加密的。它没有移动应用程序。Mangal 说，人们倾向于在专业活动中使用笔记本电脑、台式机或 iPads。(然而，用户可以通过他们的移动浏览器使用会话。)

Mangal 说，这家初创公司与 Hopin 和 Andreessen Horowitz 支持的 Run The World 处于同一空间，将使用新资本为 Airmeet 添加新功能，并在全球范围内进行扩展。

“Airmeet 的使命是创建一个全球平台，让全球数百万社区经理和活动组织者能够接触并扩大他们的受众。凭借 Lalit 和团队的专注、执行力和创新思维，他们完全有能力实现自己的目标。
