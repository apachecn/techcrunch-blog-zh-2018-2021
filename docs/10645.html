大白鲨建筑师格伦·戈登将参加12月2日至3日的虚拟活动Sight Tech Global

对于盲人或视力受损的人来说， JAWS 是自由操作Windows PCs的同义词，具有显著的控制程度和精确度，并输出语音和盲文。键盘驱动的应用程序使得浏览网站和Windows程序的基于GUI的界面成为可能。任何一个听过精通JAWS(“语音作业访问”的首字母缩写)的人操作电脑的人都会不禁惊叹于操作者的速度和JAWS本身的快速反应。

近25年来， JAWS 一直主导着屏幕阅读器领域，全球有成千上万的人在使用它。毫无疑问，这是现代辅助技术最伟大的成就之一。我们很高兴地宣布，超过25年的JAWS设计师Glen Gordon将加入视觉技术全球的议程，这是一个虚拟活动(12月2日至3日)，专注于人工智能相关技术将如何影响未来几年的辅助技术和可访问性。出席免费，报名开放。

戈登天生失明，他对可访问性的兴趣源于他所谓的“在一个完全不清楚图形用户界面是否可以被访问的时代，使用Windows的自私愿望。”他拥有加州大学洛杉矶分校安德森学院的MBA学位，他通过“在尝试使用不可访问的软件时遇到的重重困难和挫折”学会了软件开发。他是音频和广播爱好者，也是自由科学播客 FSCast 的主持人。

JAWS的最新公开测试版包含了对这个传奇软件未来的一瞥:它现在可以与某些用户语音命令——“语音辅助”一起工作，并提供更简化的图像描述，这要归功于JAWS团队在JAWS和FUSION(结合了JAWS和ZoomText，一种屏幕放大器)中使用的人工智能技术。这些更新解决了JAWS的两个挑战——现有键盘命令集的复杂性让一些用户感到恐惧，以及图像上的“alt标签”并不总是充分描述图像。

“即将推出的JAWS、ZoomText和Fusion版本使用自然语言处理，允许许多屏幕阅读器命令以口头方式执行，”Gordon说。“你可能不想说出每个命令，但对于不常用的命令，语音辅助提供了一种方法，可以最大限度地减少需要学习的按键组合。”

“总的来说，我们希望让人们更容易使用更小的命令集来高效工作。这从根本上意味着让我们的产品更智能，并能够根据用户之前的行为预测他们想要什么和需要什么。实现这一目标是一个不精确的过程，我们将继续依靠用户反馈来帮助指导我们找到最佳方案。”

下一代屏幕阅读器将利用人工智能和其他技术，这将是12月2日至3日在 Sight Tech Global 上的一个主要话题。立即获得免费通行证。

视线科技全球欢迎赞助商。目前的赞助商包括威瑞森媒体、谷歌、Waymo、Mojo Vision和富国银行。该活动由志愿者组织，活动的所有收益都将捐给硅谷的Vista盲人和视障人士中心。
