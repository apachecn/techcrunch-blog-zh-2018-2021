Pure Storage 以 3.7 亿美元收购数据服务平台 Portworx

公共企业数据存储公司 Pure Storage 今天宣布，它以 3.7 亿美元的现金收购了 Portworx ，这是一家资金雄厚的创业公司，提供基于 Kubernetes 的云原生存储和数据管理平台。这标志着 Pure Storage 迄今为止最大的一笔收购，并显示了这个市场对多云数据服务的重要性。

目前 Portworx 企业客户包括家乐福、康卡斯特、通用电子、克罗格、汉莎航空和 T-Mobile 等。该服务的核心是帮助用户迁移数据和创建备份的能力。它创建了一个存储层，允许开发人员随后访问这些数据，无论这些数据位于何处。

Pure Storage 将使用 Portworx 的技术来扩展其混合和多云服务，并提供跨云的基于 Kubernetes 的数据服务。

图片来源: Portworx

Portworx 首席执行官 Murli Thirumale 表示:“我为我们在 Portworx 构建的东西感到非常自豪:这是一个无与伦比的数据服务平台，面向在混合和多云环境中运行任务关键型应用的客户。“我们在《商业日报》中看到的牵引力和增长表明，容器和 Kubernetes 是下一代应用架构的基础，因此也是竞争力的基础。作为 Pure 的一部分，我们将能够实现加速增长和客户影响，对此我们感到非常兴奋。”

当公司去年筹集 C 轮资金时，Thirumale 告诉我，Portworx 的客户群扩大了 100%以上，预订量从 2018 年到 2019 年增加了 376。

Pure Storage 董事长兼首席执行官 Charles Giancarlo 表示:“随着具有前瞻性的企业采用云原生战略来推进其业务，我们很高兴 Portworx 团队及其突破性的技术能够加入 Pure，以扩大我们在为 Kubernetes 提供多云数据服务方面的成功。“此次收购标志着我们将现代数据体验扩展到传统和云原生应用的一个重要里程碑。”
