以下是今天出席炼金术士加速器演示日XXV的19家公司TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/17/here-are-the-19-companies-presenting-at-alchemist-accelerator-demo-day-xxv-today/

当炼金术士加速器在今年早些时候将其演示日转移到虚拟的时，炼金术士主管兼创始人Ravi Belani告诉我，他预计团队会坚持一段时间。将近半年后，又到了一个演示日——果然，疫情仍在进行，这是另一个虚拟日。

作为一个企业加速器，炼金术士主要关注那些从其他公司赚钱的种子期公司，而不是那些卖给消费者的公司。最近一批(加速器的第25批)看到近20家公司参加了该计划，重点包括物理治疗设备、销售代表的人工智能“教练”以及软件开发人员的生产力工具。

今天下午，加速器还宣布沃尔沃(通过沃尔沃汽车技术基金)已作为投资者加入炼金术士。虽然两家公司没有具体说明沃尔沃的投资金额，但在之前的类似合作中，像通用电气和瞻博网络这样的公司投资了大约200万到300万美元。

想看看这些公司向世界首次亮相吗？炼金术士将在YouTube 上直播其演示日，节目将于太平洋时间下午2点开始。

没时间看完整部？这里有一份按字母顺序排列的所有公司的列表，以及一些关于每个公司正在做什么的笔记

图片来源:安达科技

Anda Technologies:一款简化的智能手表，内置GPS、通话和基于符号的快速信息系统，旨在帮助父母和监护人在智能手机可能太多的情况下保持联系。他们最初专注于拉丁美洲，现在正在扩大对美国和欧洲的支持。

Botco.ai : “对话营销平台”——换句话说，营销聊天机器人意味着增加销售和转化率。潜在客户可以通过短信或消息应用程序与这些机器人聊天，它们的人工智能将利用它对你的业务的日益了解来做出回应。

breakthr rx:一个旨在帮助您的公司在安全漏洞发生时简化响应的平台。他们提供应对行动手册，帮助将任务分配给正确的团队成员，并帮助记录贵公司采取行动的方式和时间。

ClearQuote : 基于计算机视觉的车辆检测。该公司表示，它可以在大约60秒内使用智能手机摄像头扫描整辆车的损坏情况，并即时计算维修成本。首先关注租赁结束检查、二手车检查和租车返还检查。

Copilot : 销售代表的人工智能“教练”，当销售代表打电话/视频电话时，Copilot会分析对话，并生成包含相关信息的“提示卡”。

Evolution Devices:一种可穿戴式电刺激设备，旨在帮助下肢虚弱的人(包括中风幸存者或多发性硬化症患者)进行康复治疗。该设备适应每个用户自己的行走模式，并通过向患者的治疗师报告数据(如步数)来帮助远程护理。

他们的硬件系统会问病人一系列问题，然后带他们进行体检。

HR Messenger : 一款基于WhatsApp/Facebook Messenger开发的人力资源/入职聊天机器人，有助于自动处理预先筛选问题、面试安排和推荐请求等事情。该公司表示，它正在与包括肯德基和H & M在内的客户合作。

Hopthru : 公交机构数据分析平台。连接到这些机构已经收集的数据，对其进行清理，然后将其传输到仪表板，以帮助这些运输机构找到改善其路线和乘客量的方法。

图片来源: Hubly外科

Hubly外科: 为进行“颅骨穿刺”手术的神经外科医生打造更智能的钻头。该公司表示，许多外科医生仍然使用基本的、标准的(手摇！)钻孔，这会导致高并发症率。Hubly的钻头有助于精确调整钻头的角度，防止外科医生钻得太深。预计将于2021年获得FDA批准，并于2022年在美国医院推出。

海波因特 : 致力于航空用高功率、高密度氢燃料电池系统，意在大幅减少航空运输的二氧化碳排放。

【Mobiz】:一个通过短信向你已建立的客户群发送个性化营销信息的平台，根据品牌已有数据为每个用户建立“个性化微网站”。该公司表示，它已经在与汉堡王(Burger King)和伍尔沃斯(Woolworth)等公司合作，目前已获得600万美元的投资。

纳米钻石电池 : NDB的目标是制造一种可自我充电的可持续电池。这个问题可能有点太复杂，无法用一两句话来概括，所以请点击这里查看我们之前对NDB的报道。

节点App : 连接品牌与影响者的市场。Node帮助验证每个影响者的受众，然后通过预先协商的交易条款将品牌与这些影响者联系起来。

纠正 : 一种工具，用于在组织外部共享文档时自动检测和编辑敏感信息。首先关注保险市场。创始人Melissa Unsell-Smith说，Rectify创始团队曾在T公司的法律部门一起工作了15年。

rubi labs Inc .:专注于按需交付医疗产品(疫苗、药物等)的平台。)通过无人机、摩托车和其他专用车辆运送到非洲的医院和药店。该公司估计，它已经挽救了7000多条生命。

第七人工智能 : 将自己标榜为“知识产权宪章”，第七人工智能帮助创始人确定他们业务的哪些部分可以/应该获得专利，更好地了解竞争对手获得了哪些专利，并完成专利申请流程。该公司表示，目前的收入约为25万美元。创始人Alex Polyansky说他在USPTO做了10年的专利审查员。

Tocca : 一个旨在帮助B2B公司举办品牌虚拟销售活动的平台，提供虚拟大厅、舞台、分会场和个人对个人的网络工具等服务。集成到HubSpot和Salesforce等工具中，使事后跟进更加高效。

图片来源:生动地

Veamly : 一个面向开发者的“统一收件箱”feed，将Slack、GitHub和吉拉的线程和消息集中到一个视图中，以及一个可以在这些工具中挖掘的统一搜索。创始人Emna Ghariani表示，该公司的“专有优先引擎”有助于按照重要性对任务和票据进行分类，并分析他们一周内在每个工具上花费的时间。
