Lumos 矩阵自行车头盔评论

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/21/the-lumos-matrix-is-the-ideal-urban-bike-helmet-for-a-smarter-safer-day-trip/

在可预见的未来，我们中的许多人仍然或多或少地被限制在自己的家中和有限的社交空间内，对你们中的许多人来说，这已经导致了对骑自行车的乐趣的重新发现。骑自行车是花时间在户外探索自己的城镇或城市的一种很好的方式，如果你刚刚开始探索这一爱好，或者如果你是一名长期骑自行车的人，正在寻求升级， Lumos Matrix 智能头盔是一种明智的技术，具有可靠的设计，将许多互联功能结合到一个伟大的包中。

基础知识

Matrix 是 Lumos 智能头盔的一个版本，更新了现代的城市头盔美学，背面有一个新的大型 LED 显示屏，可以通过编程显示各种不同的图案，包括简单的图像。除了尾灯面板之外，它还包括一个内置的前灯，以及集成的转向灯，可以与内置的物理车把遥控器配合使用，或者与 Apple Watch 应用程序配合使用。它有亮白和哑光黑两种颜色可供选择。

图片来源:达雷尔·埃瑟林顿

Lumos 设计的 Matrix 可以与各种头部尺寸配合使用，这在一定程度上要归功于头盔内部的两组 Velcro 垫，但主要是因为内部的可调棘轮式尺寸控制装置。这可以很容易地拨紧或放松头盔，帮助它适应 22 至 24 英寸大小的头部。

Lumos 的外壳由 ABS 塑料制成，可以完全不受天气影响，因此您可以在雨中佩戴它，而不必担心嵌入式电子设备的状况。根据该公司的说法，如果你想要额外的安全和安保水平，你还可以添加一个 MIPS(多方向冲击保护系统)选项，尽管这还没有发货，应该“很快就会上市”。

图片来源:达雷尔·埃瑟林顿

头盔的带子上集成了一个按钮，可以让你打开和关闭它，并在内置模式之间循环。你也可以通过蓝牙将头盔与智能手机配对，并使用专用的应用程序来定制功能，包括亮度，甚至为后置显示器创建自己的定制图案。在盒子里，你还会发现一根充电电缆，一端有一个标准的 USB A 连接器，另一端有一个专有的磁性充电表面，用于为头盔和车把遥控器供电。

设计和性能

Lumos Matrix 的特点是基本上连续的表面，头盔顶部有四个通风孔，用于通风，外壳中有一个集成的帽檐。如前所述，头盔内置了一个前置灯，由透明的塑料外壳保护，还有一个 7×11 LED 灯的后面板，可以创建一个点阵式的显示器，可以显示图像或动画，包括滚动文本。这些 led 都是全 RGB，允许用户充分利用自己的或内置的显示创作。

Lumos 还生产 Kickstart，它采用了更符合空气动力学的彻底通风设计。Matrix 的外观更类似于滑板和城市通勤骑自行车者使用的头盔。尽管它的设计看起来更坚固，但在测试中我发现它实际上非常舒适凉爽，允许大量的气流。头盔位于头部稍高的位置，但有充足的硬泡沫填料，绝对感觉像一件坚固的保护装置。总的来说，Matrix 的建筑质量和表面处理水平帮助它获得了更高的价格。

图片来源:达雷尔·埃瑟林顿

Matrix 也很舒适，可调节的尺寸带确保了舒适的贴合性，这意味着头盔在佩戴时不会四处移动。激活按钮位于靠近耳朵的下巴带上，很容易找到并按下，触觉反应与听觉信号相结合，所以你会知道它已经打开。在头盔本身的后部内部，还有一个内置的磁性支架，用于内置的两按钮车把转向灯遥控器，这在戴着头盔外出办事时非常有用。

在智能功能方面，Lumos 为车载照明创建了一套非常合理的默认设置，可以轻松打开头盔开始骑行。内置模式提供了一系列选择，但都是为了增加你的能见度——明亮的照明意味着它增加了你被机动车驾驶员、其他骑自行车的人和行人看到的能力，即使你在明亮的白天骑自行车。

后置点阵显示屏的可定制性也超级得心应手。即使你对创造丰富多彩的设计来表达你的艺术自我不感兴趣，你也可以出于更实际的原因使用它——比如显示一个简单的滚动信息(例如，“和孩子一起骑自行车”)，以便提醒周围的任何人需要高度关注的原因。

图片来源:达雷尔·埃瑟林顿

随附的 Lumos 车把遥控器是开箱即用的，在激活头盔上的转向灯方面极其可靠。对我来说，Lumos 的智能手表应用程序在可靠地识别我的手臂手势以自动发送信号方面更加漫无目的，但这确实是一个增值功能，完全没有必要获得头盔的全部好处。该应用程序与 Apple Health 的集成也非常棒，可以在骑自行车时跟踪锻炼情况，并真正增加了使用 Matrix 头盔的整体体验。

结果

Lumos Matrix 是一款非常棒的自行车头盔，具有令人惊叹的集成智能照明系统，既明亮又高度可定制。苹果商店出售这款产品是有原因的——它在构造、软件集成和设计方面质量一流。也就是说，它的零售价格从 249.95 美元起——考虑到一个没有智能功能的高质量 MIPS 头盔只会花费你大约 60 美元左右，这是一个很大的价格。

然而，当你考虑矩阵中有多少技术时，价格就变得更容易接受了。诚然，专用灯也不贵，但矩阵上的灯质量非常高，在所有照明条件下都非常明显。该矩阵提供了其他地方没有的独特功能，包括主动转向灯和自动刹车灯，这些功能真正增加了您与其他骑自行车的人和车辆安全共享道路的能力。
