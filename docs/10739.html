微软推出Azure Orbital将卫星连接到其云

在其(虚拟)Ignite会议上，微软今天宣布推出Azure Orbital，这是一项新服务，旨在为卫星运营商提供一个完整的平台，与他们的卫星进行通信并处理来自卫星的数据——包括接收这些信号的地面站。

该公司特别将这些服务定位为处理地理空间数据的解决方案，并且已经与Amergint、Kratos、KSAT、KubOS、Viasat、US Electrodynamics和Viasat合作，将该服务推向市场。

图片来源:微软

Azure Networking首席产品经理Yves Pitsch在今天的博客文章中写道:“微软在支持客户收集、传输和处理地理空间数据的需求方面处于有利地位。“随着我们的智能云和边缘战略目前扩展到60多个已宣布的云区域，高级分析和人工智能功能加上世界上最快和最具弹性的网络之一，安全和创新是我们所做一切的核心。”

图片来源:微软

这里的承诺是，卫星运营商将不仅能够在微软的云上运行数据分析，而且能够运行他们所有的数字地面操作。这包括通过微软拥有和运营的地面站(使用X、S和UHF频率)安排与他们的航天器联系的能力。这些数据可以立即流入Azure的各种存储、分析和机器学习解决方案。

随着 AWS地面站，亚马逊已经提供了类似的地面站即服务产品，其中也包括全球天线网络和直接访问AWS云。然而，AWS更进一步，最近推出了一个专门的航空航天和卫星解决方案业务部门。
