微软的 Edge 浏览器将于 10 月登陆 Linux

微软的 Edge 浏览器要来 Linux 了，从 Dev 频道开始。第一部预告片将于 10 月份上线。

当微软宣布将把 Edge 浏览器切换到 Chromium 引擎时，它发誓要把它带到每一个流行的平台上。当时，Linux 并不在这个列表中，但是到去年年底，很明显微软确实在开发 Linux 版本。后来，在今年的 Build 上，一位微软演示者甚至在演示中使用了它。

图片来源:微软

从 10 月份开始，Linux 用户可以从 Edge 内部网站或者通过他们自己的软件包管理器下载浏览器。Linux 用户将获得与 Windows 和 macOS 用户相同的边缘体验，以及访问其内置的隐私和安全功能。在很大程度上，我希望 Linux 的体验与其他平台上的体验不相上下。

微软今天还宣布，到目前为止，其开发人员已经对 Chromium 项目做出了 3700 多项承诺。这项工作的一部分是对触摸屏的支持，但该团队也在核心浏览器基础之上，对辅助功能和开发工具等领域做出了贡献。

目前，微软 Edge 在 Windows 7、8 和 10 上可用，在 macOS、iOS 和 Android 上也可用。
