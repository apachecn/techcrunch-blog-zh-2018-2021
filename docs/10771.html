Shopify 表示，两名支持人员窃取了卖家的客户数据

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/23/shopify-data-merchant-breach/

Shopify 表示，两名支持人员窃取了卖家的客户数据

Shopify 证实了一次数据泄露，其支持团队的两名“流氓成员”从至少 100 家商家窃取了客户数据。

在一篇博文中，这家在线购物网站表示，迄今为止的调查显示，这两名已被解雇的员工“参与了一项获取某些商家的客户交易记录的计划”

Shopify 表示，已将此事提交给美国联邦调查局。

据称，这些员工从“不到 200 个商家”窃取了客户数据，包括姓名、邮政地址和订单细节，但财务数据未受影响。

Shopify 表示，它没有任何证据表明这些数据被使用，但它已经通知了受影响的商家这一事件。

一名商家向 TechCrunch 分享了一份 Shopify 的电子邮件通知，其中称该公司在 9 月 15 日首次意识到违规行为，两名员工获得了可以使用 Shopify 的 Orders API 访问的数据，该 API 允许商家代表他们的客户处理订单。该邮件还称，客户支付卡的最后四位数字在此次事件中被盗。

Shopify 没有说明有多少最终客户受到了商家数据被盗的影响，但 Shopify 发送的电子邮件包含了违规行为中被窃取的客户记录的具体数量。在这个商家的案例中，超过 130 万条客户记录；超过 4900 次访问。

Shopify 的发言人没有回复置评请求。

就在上个月，Instacart 承认其两名第三方支持人员不正当地获取了向顾客交付杂货订单的购物者的信息。
