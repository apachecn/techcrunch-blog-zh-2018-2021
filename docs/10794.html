TC Sessions: Mobility 又回来了，我们很兴奋地给大家最后看一下什么和谁将登上主舞台。

在我们进入谁会来之前，让我们解决2019年首届活动的一个重要变化:今年， TC Sessions: Mobility 将是虚拟的。不要担心，虚拟版的 TC Sessions: Mobility 将会带来你所期待的我们的现场活动，从信息丰富的小组讨论和挑衅性的一对一采访到网络，今年甚至还有一场推介会议。

虽然虚拟与我们过去的活动不同，但它提供了一个巨大的好处:民主化的访问。如果你是欧洲、亚洲、非洲、澳大利亚、南美或美国其他地区的初创企业或投资者，你可以在硅谷倾听、联络和联系其他参与者。此外，你还可以通过我们的配对平台CrunchMatch结识所有与会者。

今年，我们还为早期移动公司举办了一场竞标赛，但你需要确保你有票参加我们的在线活动。世博会门票起价仅为25美元，体验整个活动的普通门票起价仅为195美元。我们还为学生提供50美元的门票。

TechCrunch的记者和编辑将采访一些交通运输领域的顶级领导者，以解决诸如扩大电动汽车公司规模、自动化汽车技术的未来、微移动性、建立AV初创公司和投资该行业等话题。我们的嘉宾包括Argo AI联合创始人兼首席执行官Bryan Salesky、Waymo首席运营官Tekedra Mawakana、Lucid Motors首席执行官兼首席技术官Peter Rawlinson、Ike Robotics联合创始人兼首席工程师Nancy Sun、Formula E赛车手、Cruise全球政府事务总监Prashanthi Raman、Hemi Ventures管理合伙人Amy Gu、Polestar首席执行官Thomas Ingenlath以及TuSimple联合创始人兼首席技术官Xiaodi Hou和Boris Sofman，后者是前Anki Robotics创始人兼首席技术官，现领导Waymo卡车部门。

别忘了普通入场券(包括50美元的优惠)目前是限时供应的；在价格上涨之前，在这里抢你的票。

10月6日星期二

让AVs更上一层楼teke DRA mawa kana(Waymo)

Waymo首席运营官Tekedra Mawakana是Waymo未来的核心，从扩大自动驾驶汽车公司的商业部署和指挥车队运营到发展公司的商业道路。Tekedra将讲述Waymo在推进其成为成熟企业的计划时会面临什么。

与马修·约翰逊-罗伯森 ( 折射艾)、阿里·卡沙尼 ( 邮戳)、的变脸，演讲人待定。

小型创业公司和物流巨头都在研究如何使用自动车辆技术和机器人来送货。Refraction AI的联合创始人Matthew Johnson-Roberson和Postmates的特别项目副总裁Ali Kashani将谈论使用机器人送货的挑战和机遇。

与赖利·布伦南 ( 卡车风投)、艾米·古 ( 赫米风险投资)、以及奥拉夫·萨克斯 ( 马尼夫·机动性 )

Reilly Brennan、Amy Gu和Olaf Sakkers将齐聚一堂，讨论移动技术的不确定未来以及风险投资是否足以推动行业前进。

澄清事实布莱恩·萨勒斯基 ( 阿尔戈·艾 )

Argo AI已经从一家不知名的初创公司发展成为一家为福特和大众提供自动驾驶汽车技术的公司——更不用说这两家全球汽车制造商的数十亿美元投资了。联合创始人兼首席执行官布莱恩·萨勒斯基(Bryan Salesky)将谈论该公司的旅程，下一步是什么，以及如何真正实现自动驾驶汽车技术的商业化。

下一次在微动力方面的机会有丹尼埃尔·哈里斯 ( 元素加速器)德米特里·舍韦连科 ( 王八)阿夫拉·范德齐 ( 超能分子 )

在全球范围内，许多公司都在运营共享的微移动服务——数量如此之多，以至于该行业已经进入整合阶段。尽管市场已经饱和，但新玩家仍有机会。Elemental Excelerator的移动创新总监丹尼埃尔·哈里斯、Tortoise will的创始人Dmitry Shevelenko以及Superpedestrian的战略和政策副总裁。

与南希·孙 ( 艾克 ) 一起打造一家AV创业公司

Ike联合创始人兼首席工程师南希·孙(Nancy Sun)将分享她在自动化和机器人领域的经验，在她开始创办自动驾驶卡车公司之前，她已经从苹果公司(Apple)到奥托和优步。Sun将讨论卡车运输的未来，以及建立一个成功的移动创业公司背后的挑战和秘密。

优步的城市足迹与蔡新培 ( 优步 )

优步的运营涉及交通生态系统的许多方面。无论是自动驾驶汽车、食品配送、卡车运输还是传统的叫车服务，这些产品和服务都需要优步与城市互动，并确保公司站在城市的好一边。这就是蔡新培的切入点。请听Tsay讲述她如何思考优步在城市中的地位，以及她如何驾驭各种监管框架。

全电动空气之路 彼得·罗林森(Lucid Motors)

就在Lucid Motors推出其备受期待的全电动豪华空中轿车几周后，我们将与Peter Rawlinson坐下来讨论建立一家汽车公司和组装第一辆量产车的挑战以及未来的计划。

10月7日星期三

与卢卡斯·迪格拉斯 ( 奥迪运动 ) 的赛车未来

E级方程式车手卢卡斯·迪·格拉斯(Lucas Di Grassi)是一个新的赛车系列的一部分，在这个系列中，骑着高速电动滑板车的车手在城市的临时赛道上相互竞争。想想方程式E，但是用电动滑板车。Roborace的前首席执行官和EsC的可持续发展大使，电动滑板车锦标赛，将加入我们谈论电气化，微型汽车和一种新的赛车运动。

与小迪侯 ( 图辛普尔)和鲍里斯·索夫曼 ( Waymo ) 的货运未来

TuSimple联合创始人兼首席技术官Xiaodi Hou和前Anki Robotics创始人兼首席执行官、现领导Waymo卡车运输部门的Boris Sofman将讨论自动卡车运输的业务和技术挑战。

采用Detlev von Platen的保时捷电气化(保时捷股份公司 )

保时捷在过去几年经历了重大转型，向电动汽车项目投资数十亿美元，并推出了首款全电动汽车Taycan。现在，保时捷正在加速前进。保时捷股份公司的Detlev von Platen是该公司执行董事会的成员，他将谈论保时捷的道路、竞争以及下一步的发展方向。

与大卫·埃斯特拉达(Nuro)梅丽莎·弗罗利希 ( 极光)和乔迪·凯尔曼(Lyft)普拉珊蒂·拉曼 ( 巡航 )

自动驾驶汽车开发商面临着地方、州和联邦法规的拼凑。来自Nuro、Aurora、Lyft和Cruise的政府政策专家讨论了已经取得的进展、仍然存在的挑战以及初创公司如何应对混乱的监管并大规模部署他们的自动驾驶汽车技术。

城市的未来:交付与玛格丽特·奈格尔 ( 翼 ) 一起飞行

Wing的政策和公共事务负责人玛格丽特·奈格尔将谈论无人机如何用于送货，如何重塑城市和提高可达性。

与Thomas Ingenlath(Polestar)一起交付和制造电动汽车

Polestar上市不到四年，已经有两款车上市，还有更多正在研发中。在与首席执行官Thomas Ingenlath的炉边谈话中，我们将讨论公司的重点、战略和时尚设计。

与托尼·阿德西纳 ( 古拉·赖德)、弗雷德里克·耶勒姆 ( VOI科技)和尤温·潘 ( 斯平 ) 一起浏览世界监管框架

尽管无码头滑板车首先出现在美国的街道上，但国外也有大量的滑板车活动。由于不同的监管环境和参与者，滑板车的状况会因你所处的位置而有所不同。滑板车已经在欧洲起飞，许多玩家在欧洲大陆和南美洲经营。现在，共享滑板车和自行车正在非洲兴起。听听Spin首席执行官Euwyn Poon如何将他以美国为中心的公司带到国外，听听VOI联合创始人Fredrik Hjelm如何介绍欧洲的踏板车现状，听听微型移动初创公司Gura Ride的创始人兼首席执行官Tony Adesina如何介绍非洲的机遇和挑战。

启动变桨距

由TechCrunch编辑精心挑选的早期公司将走上舞台，并有五分钟时间介绍他们的公司。

特斯拉之后的生活 JB斯特劳贝尔 ( 红木材料 )

JB Straubel最出名的可能是特斯拉的联合创始人和前首席技术官，他负责该公司一些最重要的技术，尤其是电池方面的技术。但是斯特劳贝尔还没有完成。他推出了自己的回收创业公司，名为Redwood Materials，专注于创建一个循环供应链，最近将亚马逊和松下列为客户。我们将与斯特劳贝尔坐下来谈论他的最新冒险，在特斯拉的时间，当然，电池技术和电动汽车的状态。

与塞利纳·米科拉伊扎克 ( 松下 ) 和JB·斯特劳贝尔 ( 红木材料 ) 一起打造更好的电池技术

北美松下能源公司电池技术副总裁塞利纳·米科拉伊扎克和Redwood Materials联合创始人兼首席执行官JB Straubel将深入探讨电池技术的现状，如何在满足不断增长的需求的同时最大限度地减少对环境的影响，以及他们各自的公司如何合作。
