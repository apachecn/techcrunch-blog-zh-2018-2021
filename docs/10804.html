诺里正在区块链推销碳交易！

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/24/nori-is-pitching-carbon-trading-on-the-blockchain/

加密货币遇上碳交易。

这是总部位于西雅图的新创公司 Nori T1 的推介，该公司刚刚从专注于加密货币的投资者那里筹集了 400 万美元的资金，这些投资者包括 T2 占位符 T3、北岛风险投资公司、顽强风险投资公司和一家大型私营农业跨国公司，该公司拒绝透露该公司的信息。

由前德勤科技员工保罗·甘比尔(Paul Gambill)创立，他于 2015 年离开公司，开始在一家应对气候变化的公司工作，Nori 利用区块链解决碳抵消市场中存在的“重复计算”。

甘比尔说:“在 T4，跨国界出售的非常单一的碳信用额被计算了两次。” 当一个项目发生时，碳信用额本身就是一种资产，然后有人将其出售给其他人。他们卖给经纪人，经纪人再卖给其他人。如果你在市场上看到碳交易……那就是同一吨碳一次又一次地转手交易。”

Gambill 以巴西的一个雨林保护项目为例说，项目的巴西发起者和信用的国际购买者都将把该项目计入他们自己的碳排放减少核算中。

甘比尔说:“这是一个愚蠢的问题，可以通过复式记账轻松解决，但没有人这么做。”。甘比尔说，虽然许多公司现在购买并立即取消碳补偿，因此他们不能在公开市场上再次交易，但这些交易只代表补偿销售总量的一小部分。

甘比尔认为，碳市场是解决气候变化问题的正确方案，他认为这是一个工程问题。

甘比尔说:“2015 年我离开了，我想做一些更大更重要的事情，气候变化对我来说似乎是一件显而易见的事情。“我一直认为这更像是一个工程项目。二氧化碳的位置不对，我们必须把它移走。这是人类必须承担的下一个伟大的工程项目……到 2017 年，我已经组建了一个团队和一个商业模型，我们开始了比赛。”

甘比尔在 2017 年着手解决的问题是，碳市场未能通过将补偿证书从支付机制中分离出来来运作。购买者使用瑙鲁海洋资源公司的代币付款，代币总是适用于一吨固碳排放，但代币的价格将根据供求情况波动。

甘比尔说:“我们想要一个人们可以投机、买卖的商品市场，但我们不想在实际的碳市场上这么做。”。

该公司运营一个电子商务前端，人们可以在那里购买 Nori 令牌来抵消他们的购买(该公司已经在 Shopify 有一个客户)，然后有交易机制，Nori 将在交易所上市交易其令牌，这些交易所也解决碳排放减少和项目融资问题。“我们将来会在应用程序中加入拍卖功能，”Gambill 说。这是该公司成为集成到任何平台或应用程序中的碳去除 API 的路线图上的又一步。

如果这种观点听起来很熟悉，那是因为 Nori 不是唯一一个试图开发碳补偿 API 的公司。

像 Wren 和 Cloverly 这样的公司也在追求减排，但是 Gambill 说他们从等式的另一边处理这个问题。虽然 Cloverly 和 Wren 正在想办法根据客户需求出售现有的碳信用额，但 Nori 的 Gambill 认为，他的公司正在通过促进从农田开始的新的碳封存项目的发展来解决供应方的问题。

该公司在彗星农场有一个令人信服的合作伙伴，这是美国农业部开发的一个温室气体核算和报告系统，它正在与瑙鲁海洋资源公司合作制定其减排核算框架。
