Snyk 收购 DeepCode 以增强其代码审查智能 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/24/snyk-acquires-deepcode-to-boost-its-code-review-smarts/

总部位于瑞士的机器学习代码审查初创公司deep code——它标榜自己是“从语法上为编码人员服务”——已被 Snyk 收购，Snyk 是一家后独角兽估值的网络安全初创公司，专注于帮助开发人员保护他们的代码。

交易的财务条款尚未披露。但据 Crunchbase 称，这家“大代码”解析初创公司自 2016 年成立以来仅筹集了约 520 万美元——主要是最近完成了一轮 400 万美元的种子轮，投资者包括去年的【early bird】、 3VC 和【btov Partners。

DeepCode 首席执行官兼联合创始人鲍里斯·帕斯卡列夫(Boris Paskalev)证实，整个团队都“热切地”加入 Snyk，继续他所说的“让全球每个开发人员都能进行语义人工智能驱动的代码分析的使命。”

Paskalev 告诉 TechCrunch:“deep code 作为一家公司将继续存在(完全由 Snyk 拥有)，我们将保留并计划发展苏黎世办公室，利用这里令人惊叹的人才库，我们将继续支持和扩大全球开发社区的尖端产品供应。”

当被问及 DeepCode 的产品未来是否会继续作为独立产品存在，或者完全融入 Snyk 的平台是否会包括关闭其目前为开发者提供的代码审查机器人时，他说尚未做出决定。

“我们仍在对细节进行评估，但主要目标是保持/扩大我们为所有开发者提供的好处，特别是增加开源的采用和参与，”他说，并补充说:“最初显然什么都不会改变，DeepCode 产品将仍然是一个独立的产品。”

Paskalev 补充说:“两家公司都有一个非常清晰的愿景和对开发人员第一的热情，并帮助开发人员和安全团队进一步降低风险，提高效率。”

在宣布收购的一份声明中，Snyk 表示，它将把 DeepCode 的技术集成到其云原生应用安全平台中——继续宣传其人工智能引擎的优势，称这将使开发人员“更快地识别漏洞”

“DeepCode 的人工智能引擎将帮助 Snyk 提高速度，并确保在发现和修复漏洞方面达到新的精确度，同时不断从 Snyk 漏洞数据库中学习，变得更加聪明，”首席执行官彼得·麦凯写道。“这将使开发人员能够更快地集成，在开发过程中测试问题，而不是作为一个额外的步骤。它将进一步提高我们结果的准确性，几乎消除了浪费时间追查假阳性的需要。”

在 DeepCode 给 Snyk 留下深刻印象的特性中，McKay 称赞代码扫描“比其他产品快 10-50 倍”；他称之为“杰出的开发人员 UX”——这允许“实时的高精度语义代码分析”，因为扫描是在 IDE 和 git 级别进行的。

在自己关于收购苏黎世联邦理工学院(ETH Zurich)的博客文章中，苏黎世大学写道，这家人工智能初创公司的“决定性优势”是“它开发了第一个可以从数十亿个程序代码中快速学习的人工智能系统，能够基于人工智能检测安全性和可靠性代码问题。”

“DeepCode 是现代人工智能系统的一个优秀例子，它可以从数据中学习，在这种情况下是程序代码，但仍然对人类透明和可解释，”它补充道。

支撑 DeepCode 的大学研究工作可以追溯到 2013 年，当时其联合创始人正在研究如何将数据驱动的机器学习方法与基于符号推理的语义静态代码分析方法相结合。

DeepCode 的技术目前覆盖了 400 多万名贡献开发者，超过 10 万个存储库订阅了其服务。
