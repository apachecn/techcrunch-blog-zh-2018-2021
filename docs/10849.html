Axis Security 筹集了 3200 万美元来帮助公司在家办公时保持安全 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/09/29/axis-security-raises-32m-to-help-companies-stay-secure-while-working-from-home/

Axis Security 于去年推出，其理念是帮助客户使承包商和第三方能够以安全的方式远程访问公司的系统，但当疫情来袭时，他们看到了另一个用例，一个已经在他们的路线图上的用例:当员工在家工作时，帮助保持系统安全。

今天，该公司宣布了由迦南合伙人公司牵头的 3200 万美元的 B 轮投资，现有投资者 Ten Eleven Ventures 和 Cyberstarts 也参与了投资。据安讯士称，今天的融资总额达到了 4900 万美元。

联合创始人兼首席技术官 Gil Azrielant 表示，该公司之所以能够如此迅速地转向在家工作的安全场景，是因为它从头开始构建产品，最终支持这一愿景。疫情只是加快了这一进程。

“起初，我们决定专注于第三方和承包商，但我们看到了冰球的发展方向，并明确[设计]了基础架构，以成为成熟的安全访问产品。所以基础设施已经存在，我们只需要添加一些为以后计划的东西，”Azrielant 告诉 TechCrunch。

他说，该公司的产品使用零信任的概念，顾名思义，假设你不能信任你系统上的任何人，并从那里工作。使用基于规则的引擎，客户可以根据您的角色创建一个安全的环境。

“您能看到什么，或能做什么，或能下载或访问什么，都完全由我们的应用访问云控制。这是基于你使用什么设备，你在哪里，你是谁，你是什么角色，以及你通常做什么和不做什么来决定你将获得的访问级别，”他说。

去年 3 月，就在加州疫情关闭开始后的第三天，这家初创公司从隐身状态中脱颖而出，它有两个主要客户——一家连锁酒店和一家制药公司——首席执行官多尔·克纳弗表示，随着 COVID 的出现，“必要性成为了收养之母。”

他补充说，“这两家公司都来找我们，要求我们开始研究所有这些员工访问使用案例，对我们来说，这是令人难以置信的，因为这给了他们所需的推动力，让他们像我们一样生动地看到[远程访问]愿景，”他说。今天，它加入了最初的两个，虽然它不会分享一个确切的数字，但它报告说，它有几十个客户。

如今，这家初创公司有 38 名员工，几乎平均分布在加州圣马特奥和以色列特拉维夫，并计划明年加速招聘，达到 100 人。随着公司规模的扩大，Knafo 表示，他正试图建立一个更加多样化的团队，以便在未来一年雇佣更多的人。

“如今，我们内部有激励措施来帮助我们以更多样化的方式招聘员工。Knafo 说:“我们在这方面进行了大量投资，我们将继续为公司的每个人(将这一点放在首位)。”

Azrielant 补充说，疫情表明，员工不必位于办公室附近，这些办公室今年大部分时间都关闭了，这为建立更加多样化的劳动力提供了更多的可能性，因为他们可以从任何地方雇佣员工。

随着一种产品现在有了更多的用途，该公司将利用新流入的现金来帮助建立其销售和营销业务，并扩大北美以外的销售。

Knafo 表示:“随着 COVID 的加速发展以及从任何地方工作的转变，我们肯定会专注于将我们的产品带给更多面临在家工作这一紧迫挑战的企业。”。
