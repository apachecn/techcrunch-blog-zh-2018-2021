谷歌助手的一项新功能“为我等待”会暂停，这样你就不必等待了

谷歌一直在推进智能手机的功能，这是智能手机最基本的功能之一，也是目前经常被忽视的功能:通话。前几年，该公司推出了来电筛选来审查你的来电，推出了餐厅预订的复式，而就在本月，推出了一项名为核实来电的功能，它将告诉你谁在打电话以及为什么打电话。今天，谷歌推出了一个更方便的功能，为那些仍然使用他们的设备作为一个真正的手机的人提供“帮我拿着”

在今天下午该公司的硬件活动上，谷歌介绍了其 新款 Pixel 智能手机，并简要展示了 Pixel 的最新技术。有一个叫“为我保持”的功能，当你被置于保持状态时，它会为你保持在线，然后在有人接听时提醒你。

谷歌解释说，这项技术是建立在现有通话屏幕和双工技术的智能基础上的——后者是一项基于人工智能的技术，专注于如何通过电话进行对话。

在“Hold for Me”的简短演示中，谷歌展示了 Pixel 设备所有者如何在被搁置后激活新功能。这是通过点击出现在电话屏幕上的新按钮来实现的，该新按钮位于静音、打开免持话筒和其他通话中的电话控制按钮的上方。

激活后，你会收到一条提示信息“不要挂断”，提示你谷歌助手正在为你接听电话，这样你就可以做其他事情了。

这个屏幕上还有一个按钮，可以让你随时点击返回到呼叫，在这个按钮下面，屏幕上的消息会显示“音乐播放”，以指示谷歌助手是否仍在听到保留的音乐。如果线路上正在通话，将会出现实时字幕。该公司表示，这是由谷歌的自然语言理解提供的动力。

您也可以选择按红色的挂机按钮，从该屏幕结束通话。

当有人接通电话时，该设备会提醒您该回电话了。

当人们在等待新冠肺炎相关政府援助(如失业福利)时，一个“为我保留”选项可能不仅仅是一个有用的新功能——对于那些因失业而陷入金融危机的人来说，它可能是一个真正的救命稻草。

谷歌表示，新功能将出现在新的 Pixel 5 设备上，通过下一次“Pixel feature drop”的推出，新一代 Pixel 手机将很快推出。
