Pixel 5 和 4a 5g 得到相同的改进相机，具有后置超宽镜头、夜视人像等

谷歌今天正式发布了其最新的智能手机，推出了泄露很多的 Pixel 4a 5g 和 Pixel 5。两款智能手机都将获得相同的改进相机，尽管两种型号之间的价格相差 200 美元，这对那些专门来谷歌寻求优秀移动相机技术的人来说是个好消息。以下是谷歌在硬件和软件方面对新的和改进的 Pixel 相机所做的概述。

超宽镜头

新款 Pixel 手机最大的新物理变化是在背面的摄像头阵列中添加了一个新的超宽镜头。这提供了一个新的广角视野，让你捕捉一个明显更大的视角，这是一个很大的群体拍摄和风景。这是苹果在最新的 iPhone 上添加的功能之一，谷歌粉丝正在他们的 Pixel 设备上寻找这些功能。

HDR+带括号

谷歌 Pixel 手机的 HDR+功能也很受用户欢迎，为人们提供了一种在照片中获得更好照明的方式，而不必担心事后合成图像以调整场景不同部分的曝光。谷歌通过结合自己的机器学习技术升级了 HDR+功能，并与传统的、更老派的曝光技术相结合，以获得该公司所说的更好的最终产品。

人像模式下的夜视

人像模式自引入智能手机以来一直很受欢迎，并随着时间的推移不断改进，以允许人们通过人工背景模糊获得更准确的深度效果。谷歌在这一代设备中添加了使用人像模式和夜视功能的能力，这意味着即使你使用谷歌的软件欺骗来增加黑暗场景中的照明，也可以获得这种深度效果，以获得清晰的无静电效果，如下图所示。

肖像灯

另一个肖像模式的特点是增加了肖像灯光，这让你可以应用一个可定制的灯光效果来做一些事情，如抵消图像的阴影或褪色部分。这与苹果在 iOS 中自己的肖像模式下的工作室灯光效果类似，但它看起来更具可定制性，并且由于谷歌在 Pixel 设备上的人工智能技术，可能更强大——尽管我们必须让他们进行测试才能确定。

新的视频稳定功能，包括电影平移

最后，在新像素上拍摄视频有三种新的稳定模式——锁定、活动和电影平移。谷歌在活动期间表示，这些都是通过 YouTube 上的教程以及研究好莱坞电影摄影师建立的。对 YouTubers 来说，电影式的平移看起来可能是最有趣的，因为它提供了丝滑、缓慢的效果(实际速度的一半)，使它看起来像电影游记中的一样。
