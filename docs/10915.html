谷歌瞄准“美颜滤镜”,对 Pixel 手机进行设计变革

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/01/google-takes-aim-at-beauty-filters-with-design-changes-coming-to-pixel-phones/

谷歌正在瞄准照片面部过滤器和其他“美化”技术，心理健康专家认为这些技术会扭曲一个人的自信，尤其是当他们被介绍给年轻用户时。该公司表示，在将 Android 相机应用程序使用的照片滤镜的设计原则应用于 Pixel 智能手机时，它现在将依赖专家的指导。谷歌表示，在 Pixel 4a 中，谷歌已经默认关闭了面部修饰功能，并指出该界面将很快更新，以包括谷歌称之为“无价值”的描述性图标和标签，用于该应用的面部修饰效果。

这意味着它不会使用“美丽过滤器”之类的语言，也不会暗示，甚至以更微妙的方式暗示，面部修饰工具可以让某人看起来更好。这些变化也将通过更新推广到其他 Pixel 智能手机中的 Android 相机应用程序。

虽然最终用户可能没有注意到这些变化，但随着时间的推移，这些变化会产生影响。

谷歌表示，Android 上超过 70%的照片是用前置摄像头拍摄的，超过 240 亿张照片在谷歌照片中被标记为“自拍”。

图片来源:谷歌

但是我们的智能手机向我们展示的图像正在驱使更多的人对自己的外表不满意。根据美国面部整形和重建外科学会的数据，去年 72%的会员表示，他们的病人找他们是为了改善他们的自拍，同比增长了 15%。此外，80%的父母表示他们担心滤镜的影响，三分之二的青少年表示他们在照片中的形象受到过欺负。

谷歌解释说，它寻求儿童和心理健康专家的帮助，以更好地了解过滤器对人们健康的影响。研究发现，当人们没有意识到已经应用了照片过滤器时，由此产生的照片可能会对心理健康产生负面影响，因为他们悄悄地设定了一个美丽标准，人们随后会与自己进行比较。

图片来源:谷歌

此外，使用“美丽”、“美化”、“增强”和“修饰”等术语的过滤器暗示某人的外表有问题，需要纠正。谷歌解释说，这表明它们实际上看起来很糟糕。同样的道理也适用于“瘦身”这样的术语，它意味着一个人的身体需要改善。

谷歌还发现，甚至使用的图标也可能导致这个问题。

通常情况下，面部修饰滤镜会在开启该功能的图标上使用“闪亮”的设计元素。这表明使用滤镜可以让你的照片更好。

为了解决这个问题，谷歌将更新其过滤器，使用价值中立的语言，以及新的图标。

图片来源:谷歌

例如，它不会将一个面部修饰选项标记为“自然”，而是将它重新标记为“微妙”它没有闪烁的图标，而是显示一个带有编辑笔的面部图标，以指示按下哪个按钮来启用该功能。

调整水平也将遵循新的指导方针，使用数字和符号或简单的术语，如“低”和“高”，而不是那些涉及美的术语。

图片来源:谷歌

谷歌表示，相机应用程序也应该在启用滤镜时显而易见——无论是在实时拍摄还是事后。例如，当滤镜打开时，屏幕顶部的指示器可以通知用户，这样用户就知道他们的图像正在被编辑。

在 Pixel 智能手机中，从 Pixel 4a 开始，当你使用面部修饰效果时，你会看到更多关于每个设置如何应用以及它将对图像进行哪些具体更改的信息。例如，如果你选择“微妙”的效果，它会解释说，它可以调整你的皮肤纹理，眼睛下方的色调和眼睛的亮度。对应用的效果保持透明有助于揭开面部修饰滤镜对我们的照片进行的有时微妙的调整。

在周三公布的新像素设备中，包括 Pixel 4a 5G 和 Pixel 5，面部修饰也将被关闭。谷歌表示，标签和描述的变化将通过即将到来的更新进入 Pixel 手机，该更新将支持 Pixel 2 和更高版本的设备。
