微软更新了其基于 Arm 的 Surface Pro X 平板电脑，配备了更快的 CPU

微软今天宣布了其基于 Arm 的第二代 Surface Pro X 平板电脑。第一代 Pro X 于去年 10 月推出，自那以来，微软与高通合作设计了支持它的第二代 SQ2 处理器。不过，SQ2 并没有完全取代第一代 SQ1 芯片。相反，微软只是将其用于 Pro X 的高端 SKU。

除了新的处理器，我们仍然在谈论去年推出的同一款 Pro X，内置 LTE 连接，两个 USB-C 端口和一个 13 英寸的触摸屏。

图片来源:微软

除了现有的哑光黑色表面，微软还推出了一种新的颜色，即新的铂金表面。它还为 Pro X 键盘推出了三种新颜色:冰蓝色、罂粟红色和铂金色。

不过，这里还有一些其他重要的更新。由于一些软件改进，Surface Pro X 现在承诺长达 15 小时的电池寿命。

图片来源:微软

但更重要的是，微软指出，更多的软件合作伙伴现在正在为 ARM 架构优化他们的 Windows 应用程序(微软仍然在其材料中拼写为 Arm，但 Arm 公关肯定会联系他们并纠正他们)。当微软推出第一个版本时，它看到了相当多的负面评论，因为缺乏对第三方驱动程序的支持，因此一些应用程序也失败了，而其他应用程序运行缓慢，因为他们必须依赖仿真——除非他们的开发者发布了 Arm 兼容的版本。

本周早些时候，该公司已经注意到其 Arm 上的 Edge 浏览器正在进行更新，以减少电池消耗。此外，微软还指出，它将在 11 月向 Windows Insiders 推出新的 x64 仿真，以增加对运行 x64 应用程序的支持。Visual Studio 也针对 Arm 上的 Windows 10 进行了更新和优化。
