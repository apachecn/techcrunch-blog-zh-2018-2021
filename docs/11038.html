谷歌助手现在可以控制Android应用程序

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/08/google-assistant-can-now-control-android-apps/

Google today 宣布使用语音命令“嘿Google”不仅可以打开Android应用程序，还可以执行特定的任务。该功能将推广到所有支持谷歌助手的安卓手机，允许用户用语音启动应用程序，并在应用程序中搜索或执行特定任务，如点餐、播放音乐、发布到社交媒体、打车等。

例如，用户可以说这样的话，“嘿，谷歌，在Etsy上搜索舒适的毯子”，“在Snapchat上打开Selena Gomez”，“用Nike Run Club开始我的跑步”，或者“在Twitter上查看新闻。”

谷歌表示，在发布时，这些语音命令将在全球范围内Google Play的30多个顶级英语应用中使用，更多应用将很快推出。今天支持的一些应用程序包括Spotify、Snapchat、Twitter、沃尔玛、Discord、Etsy、MyFitnessPal、Mint、Nike Adapt、Nike Run Club、易贝、Kroger和Postmates、Wayfair等等。

如果您用来执行常见任务的特定语音命令有点笨拙，该功能还允许您创建一个自定义快捷短语。这意味着，你可以创建一个命令，而不是说“嘿，谷歌，用Nike Adapt系紧我的鞋，”你可以只说“嘿，谷歌，系紧鞋带。”

要开始使用快捷方式，Android用户可以说“嘿，谷歌，显示我的快捷方式”来获得正确的设置屏幕。

该功能类似于苹果支持将Siri与iOS应用一起使用，还包括打开应用、执行任务和录制自己的自定义短语的能力。

在谷歌的例子中，在应用程序中执行任务的能力是在开发者端通过将用户的意图映射到应用程序中的特定功能来实现的。这一功能被称为应用程序操作，允许用户通过语音命令打开他们喜欢的应用程序，并且，随着功能的增加，用户可以说“嘿谷歌”来在应用程序内搜索或打开特定的应用程序页面。

谷歌表示它已经扩大了目录，包括10个垂直领域的60多个意向，包括金融、拼车、订餐、健身，现在还包括社交、游戏、本地旅游&、生产力、购物和通信。

为了帮助用户了解他们如何以及何时可以使用这些新的应用程序操作，谷歌表示，它正在Android中建立接触点，这将帮助他们学习何时使用某些语音命令。例如，如果用户说“嘿，谷歌，给我看看泰勒·斯威夫特”，它可能会突出显示一个建议芯片，引导用户在Twitter上打开搜索结果。

与此新闻相关的是，谷歌表示还发布了两种新的英语语音，供开发人员在智能显示器上为助手构建定制体验时使用，此外还有其他开发人员工具和为显示器构建工具的资源。

谷歌助手的应用升级是谷歌今天强调的几项安卓改进之一。该公司还表示，它将为Google Duo添加屏幕共享功能，将其验证呼叫反垃圾邮件功能扩展到更多设备(Android 9及更高版本)，并更新Google Play Movies电视应用程序，成为新的“Google TV”应用程序，上周宣布。

在无障碍方面，它推出了针对听力损失的新工具和声音通知以及其他使用动作块进行交流的工具，旨在帮助患有脑瘫、唐氏综合症、自闭症、失语症和其他语言相关残疾的人。

这些功能现已可用。
