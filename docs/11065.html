公众投资者继续热爱科技，因为它是 IPO 的根源和保证

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/10/public-investors-stay-in-love-with-tech-as-root-and-affirm-file-to-ipo/

编者按: 获取 TechCrunch 新闻的免费每周摘要，任何初创公司都可以在每周六早上(太平洋时间上午 7 点)通过电子邮件使用。订阅这里。

为什么现在有这么多科技公司 IPO？创业公司发现，如今他们可以从公开市场获得比私人市场更高的估值，因为如此多的公共投资者希望在科技领域投入大量资金。此外，未来的诱惑，美联储的仁慈，散户投资者的繁荣，一直在等待任何体面时机的独角兽的数量，公司上市的新方式……这些是亚历克斯·威廉在查阅公开市场上关于科技的最新上市和季度数据后找到的一些理由。

各种政治和经济动荡威胁着这一趋势的结束，但对创业世界的影响已经到来。在时事通讯深入到股票、SPACs、新兴产业和其他有用的创业新闻之前，请考虑一下。

从这次 IPO 热潮中，将会有另一波创业公司员工的财富涌入邻近的现实世界，但比脸书和 Twitter IPOs 时期更广泛地扩散到湾区以外。这些员工中的一些人将成为投资者，也许是创始人，而现在上市的初创公司将会用大公司的人来取代这些职位。围绕技术招聘的动态将以令人惊讶的新方式进一步重塑，所有这些都与远程工作等其他变化相结合。

今天，如果你现在正在创办一家初创公司，你现在可以自信地规划新的方式来长期发展你的公司，这是前几代创始人几乎无法想象的。

在未来十年，我们可能会看到一家初创公司上市，首先从种子前滚动资金中筹集资金，吸引新合法化的众筹资金，从数千家目前正在运营的风投公司中匹配合适的风投——或者初创公司筹集债务，因为它做得很好。只要它想利用过去十年中发现的各种融资和二级市场可能性，它就可以保持私有化。然后，当它准备上市时，它可以在传统选择、完美的特殊目的收购公司和直接上市之间做出选择，并保持股东池有利于那些在整个过程中一直陪伴公司的真正信徒。

目前的一批 IPO 还展示了一些其他的东西。科技不再被定义为旧金山的一些无利可图、高价值的消费科技初创公司。它可以来自任何地方，它可以解决实际问题，它可以赚真金白银，它可以不断建设和发展——前提是你可以接受一些持续的风险。难怪如今公开市场喜欢科技股。

看看 Root Insurance，这是一家保险科技独角兽公司，已经帮助定义了俄亥俄州哥伦布市的创业场景。这是一次“创业测试”，Alex 本周详细介绍了其新的 IPO 申请。“你可以找到喜欢的东西(提高调整后的利润率！收入增长！)，而且可以找到不喜欢的东西(螺旋式亏损！负边距！)很轻松。”

这里有更多来自额外的紧缩文章:

许多保险提供商在新冠肺炎期间看到的顺风似乎给 Root 提供了一个很好的推动(疫情期间驾驶下降，导致一些保险提供商返还保费。)鲁特正在利用这一时机，在它能够显示出经济状况大幅改善的时候提出申请。

真聪明。但是这些改进的经济学如何在传统会计中得到证实呢？让我们来看看:

  • Root 的收入从 2018 年的 4330 万美元飙升至 2019 年的 2.902 亿美元。2020 年上半年，Root 管理的收入为 2.454 亿美元，比 2019 年上半年增长 135.73%。
  • Root 的亏损也大幅增加，从 2018 年的净亏损 6910 万美元增加到 2019 年的 2.824 亿美元。随着时间的推移，这家初创公司一直在亏损。最近也是如此，H1 2020 年净亏损 1.445 亿美元，使 H1 2019 年亏损 9700 万美元相形见绌。

本周的另一项申请是确认(Affirm)申请(T1)，该申请为客户提供销售点信用(T2，没有信用卡的所有功能)。对于那些今天正在研究独角兽 IPO 实验的未来创始人来说，这也是几十年来创新如何运作的象征。

这家公司是一只展翅高飞的独角兽，其实际目标来自连续创业者麦克斯·拉夫琴，他也帮助塑造了现代创业公司的概念——从共同创立 Paypal 并在多年来进行了大量天使投资，到 10 年前旧金山的一家无利润、高估值的消费科技公司 Slide。在科技之外，这一点并没有被广泛理解， Slide 和其他社交媒体公司帮助开创了增长和参与技术，随后的初创公司在 SaaS、电子商务、金融科技和现实世界等领域应用了这些技术。今天，Root 和 Affirm 以及这个 IPO 时代的许多其他公司都站在那些年的教训上。

特殊目的收购公司成长的烦恼

随着越来越多的初创公司利用这些投资工具轻松进入公开市场，特殊目的收购公司肯定会提供宝贵的经验。下面是对这一行动的最新观察，首先是康妮·洛佐斯本周从一位专家那里得到的令人不安的引述。

据 Kristi Marvin，一位前投资银行家，现在经营数据网站 SPACInsider 的人说，与过去几年相比，她正在与更广泛的对推出 SPACs 感兴趣的人进行交谈，并听说他们中的所有人都不一定有能力管理这些工具。

“你会问，‘你曾经以 5 亿美元或更多的价格收购过一家公司吗？你在你瞄准的垂直领域有运营经验吗？你了解相关的报告要求吗？她说，“答案通常是否定的。”

那是在一个有争议的前优步高管创办特殊目的收购公司的背景下；康妮还研究了这一新兴高端金融领域的性别代表性。像世界上其他地方一样，参与的人几乎都是男性(事实上，这也是创业融资的情况，亚历克斯报道)。

与此同时，Catherine Shu 调查了陷入困境的电动汽车初创公司法拉第未来如何接近特殊目的收购公司计划，而 Alex 则仔细研究了open door面临的挑战和机遇。
