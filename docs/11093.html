Twilio 证实将以 32 亿美元的全股票交易收购该部门

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/12/twilio-confirms-it-is-buying-segment-for-3-2b-in-an-all-stock-deal/

Twilio 今天宣布了其迄今为止最大的一次收购，除了其现有的用于构建和运行通信服务的基于 API 的工具之外，该公司还向客户数据管理迈出了更大的步伐。

Twilio 将以 32 亿美元的价格收购 Segment，这是一家技术初创公司，允许组织通过 API 将客户数据从一个应用程序转移到另一个应用程序，并使用其平台来更好地控制客户数据的移动，用于营销和其他受益于客户识别和细分的应用程序。

这是一笔全股票交易，具体来说是“以 Twilio 类普通股的形式，在完全稀释且无现金、无债务的基础上进行”，预计将于 2020 年第四季度完成。两家公司表示，Segment 将成为 Twilio 的一个部门。

周五,首先报道了该交易正在进行中。(FWIW 我们也得到了同样的提示，并在这篇报道发表时进行了调查。我们在的消息来源称交易金额将在 30 亿至 40 亿美元之间，可能最早于今天宣布。)

这笔交易凸显了 Twilio 如何更深入地进入客户参与服务，无论是为了营销数据还是其他目的(使其与 Adobe 等公司展开更激烈的竞争)。

Twilio 的联合创始人兼首席执行官杰夫·劳森(Jeff Lawson)在一份声明中表示:“数据孤岛破坏了出色的客户体验。“细分市场让开发商和公司打破这些藩篱，构建客户的完整图景。结合 Twilio 的客户参与平台，我们可以在客户服务、营销、分析、产品和销售方面创建更加个性化、及时和有效的参与。我们很高兴欢迎 Segment 加入 Twilio 团队。”

“Twilio 和 Segment 共同拥有构建未来客户参与平台的绝佳机会，”Segment 联合创始人兼首席执行官 Peter Reinhardt 补充道。“我们创建了 Segment 来帮助企业在数字时代脱颖而出，并提供基于高质量数据的丰富、互联的客户体验。通过联手并将我们的客户数据平台应用到 Twilio 的参与云，我们将能够实现端到端的无缝客户体验。”

这是 Twilio 自 2018 年以 20 亿美元收购 SendGrid 将电子邮件添加到其通信工具系列以来最大的一次收购。在最近一次融资中，的估值为 15 亿美元。

收购细分市场是 Twilio 过去十年发展的自然进程。

Twilio 在 2008 年作为一家初创公司首次成名，它为公司提供了一种将文本和语音服务集成到应用程序中的简单方法。通过使用 API 和几行代码，企业可以将与用户沟通的工具直接放入应用中，从而进入快速增长的智能手机和应用世界。

随着时间的推移，这些基本工具已经形成了更广泛的通信来源，如电子邮件和聊天机器人，以及针对使用它们的不同垂直行业的一组更集中的应用程序，以及更多支持和更好地管理客户关系的工具，以及更好地识别、细分和与这些客户合作的工具。

利用收购来建立这部分业务是公司战略的一部分。一位消息人士告诉我们，Twilio 实际上在 2014 年和 2015 年就试图收购客户关系信息服务 Intercom 。我们的消息来源称，两家公司当时未能就价格达成一致。(看这个空间？)

如今，Twilio 的解决方案业务专注于营销、客户服务和客户身份方面的应用。所有这些都是细分市场的重要用例，细分市场有大约 20，000 名客户使用其平台来“收集、清理和控制”客户数据。

对于细分市场，就其工具的范围和潜在目标公司而言，这是一大进步:

收购细分市场还可以为 Twilio 平台带来更多客户，为向这些客户追加销售 Twilio 提供的其他服务打开大门。鉴于细分市场声明的重点是提供更好的工具来更负责任地处理客户数据，在这种环境下，人们对这一领域的认识(和违规行为)与日俱增，这为 Twilio 提供了更强大的产品组合来满足这种需求。

也有一些有趣的协同作用。不仅两者都是围绕使用 API 将其他服务/端口数据从一个地方集成到另一个地方的架构构建的，而且两者之间已经存在集成，这可能已经被大量使用。

正如一位消息人士向我们指出的，已经有很多消息人士所说的“Xwilions”(前 Twilio 人)在这家小公司工作。

他们包括首席财务官桑迪·史密斯、顾问桑迪·奥格曼和首席市场官科琳·柯立芝，柯立芝还雇佣了一些前 Twilio 工程师。

你可能会说，客户服务(Twilio 一直在构建的主要理念)和客户体验(Segment's jam)是两码事，但这两者是有联系的:通信的一大副产品是数据，公司将寻找更好的方法来将数据的创造与再利用/使用联系起来。换句话说，两者是相辅相成的。
