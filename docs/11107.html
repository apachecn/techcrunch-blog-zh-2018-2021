Cyberpion 筹集了 825 万美元的种子资金，帮助企业在防火墙之外保护资产 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/13/cyberpion-raises-8-25m-in-seed-funding-to-help-businesses-secure-assets-beyond-their-firewall/?tpcc=ECTW2020

劫持公司的 DNS 或向第三方工具注入代码是许多公司没有能力应对的攻击。Cyberpion 是一家由许多经验丰富的安全专家创立的新安全初创公司，旨在帮助企业识别和消除这些通常源于基础设施资产连接的风险，尤其是现在大多数公司都依赖于位于其安全边界之外的各种资产。

该公司今天宣布，它已经筹集了 825 万美元的种子资金，由 Team8 Capital 和 Hyperwise Ventures 共同牵头。该公司已经在美国和以色列建立了团队，计划利用新的资金扩大销售和营销工作，并加快产品开发。

Cyberpion 的联合创始人是该公司首席执行官内塔内尔·格勒恩特(Nethanel Gelernter)，他曾在微软工作过，担任安全研究员 Cyberpion 的联合创始人是 CEO 朗·纳米亚斯，他是 Check Point 的前云安全主管，也曾在 Aqua Security 工作过，担任过微软的企业战略总监。

纳米亚斯说:“最让我着迷的是，Cyberpion 所做的一切都与你所想的、日复一日用传统安全工具处理的事情截然不同。”“对我来说，这非常令人耳目一新，我喜欢在飞行中接触新事物，学习新事物。Cyberpion 提出了一个惊人的挑战，一个非常复杂的问题，也是我了解更多的机会。”

纳米亚斯指出，今天，大多数公司都注重保护和捍卫由他们直接拥有和管理的环境。但这些公司不是在真空中运营的，建立他们的业务涉及到第三方供应商的生态系统。“随之而来的是一个包含潜在利益和潜在威胁的影响深远的生态系统，它成为整个攻击表面区域，而人们的想法和我们所有的工具都没有注意到这一点。”

这里有趣的是，该服务通过查看公司在互联网公共基础设施上建立的连接来分析公司对这些攻击的安全立场，这样它可以查看公司如何连接到一个供应商，以及它们如何连接到其他供应商。

这样，Cyberpion 就能够发现一家公司在其整个生态系统中拥有的已知和未知资产的清单，并能够持续扫描安全漏洞，当然，还能就如何补救这些问题提供警报和建议。

Hyperwise Ventures 的管理合伙人内森·舒查米(Nathan Shuchami)说:“对企业外部数字资产的攻击正成为老练的网络犯罪分子的主要载体。“面对黑客在新的和更广泛的攻击面中发现漏洞， Cyberpion 提供无与伦比的检测和保护能力，为组织的整个在线生态系统提供全面的安全保护。Hyperwise Ventures 很高兴能在 Cyberpion 共同领导种子资金轮
