埃隆·马斯克在推特上说，Model S 的售价将为 69420 美元，因为他还是个孩子

特斯拉 S 型车将变得稍微便宜一点。

特斯拉首席执行官兼常驻 meme 制造商埃隆马斯克(Elon Musk)刚刚在推特上发布消息称，Model S 轿车很快将定价 69，420 美元。这种定价不是一时兴起和一时兴起的。降价很可能是对几小时前 Lucid 在定价上低于特斯拉的回应。

正如你在上面的推文中所看到的，埃隆·马斯克宣称，“挑战已经来临。”

今天早些时候，Lucid 宣布其入门级轿车的价格为 77，400 美元，减去 7，500 美元的美国税收抵免。因为大多数买家都有资格获得信贷，这使得实际价格下降到 69，900 美元。你知道这是怎么回事，对吧？

新的起价为 69，420 美元，这将是本周 Model S 的第二次降价。CNET 昨日路演报道汽车制造商本周早些时候悄悄将 Model S 降价 3000 美元，可能是为了在今天发布之前超过 Lucid。今年早些时候，特斯拉将 Model 3 的起价下调了 2000 美元。

马斯克还在推特上宣布了一些关于特斯拉最新车型的消息，根据他的推文，小型跨界车 Model Y 将很快推出七座版。马斯克没有详细说明座位安排。

Lucid Air 正在成为特斯拉 Model S 的有力竞争者。Air 拥有 Model 3 的所有范围——甚至今天定价的入门级版本也与 Model S 的 400+范围相匹配。空气也更轻，更快，坦率地说，更新鲜。自 2012 年推出以来，Model S 的整体形状和感觉仍然相同。

在最近接受 TechCrunch 采访时，Lucid 首席执行官彼得·罗林森告诉编辑达雷尔·埃瑟林顿，这款空气将以“令人惊讶的低于 8 万美元”的价格出售。虽然今天的价格并不“令人惊讶”，但它将 Lucid Air 与装备精良的中型欧洲跑车，更重要的是特斯拉 Model S 放在了同一价格范围内。
