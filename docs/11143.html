True Ventures还有8.4亿美元投资于新生和突破性创业公司TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/14/true-ventures-has-840-million-more-to-invest-in-nascent-and-breakout-startups/

True Ventures已经成立15年，在加州的帕洛阿尔托和旧金山都设有办事处，今天上午它推出了两只新基金:它以4.65亿美元的资金结束了第七只早期基金，并以3.75亿美元结束了第四只机会型基金——用于支持自己的突破性投资组合公司。

这对True来说是一笔巨大的投资，它由乔恩·卡拉汉和菲尔·布莱克创立并继续领导。话说回来，这家公司比以前更大了，全公司有35名员工，包括投资部门的10名员工，以及公司财务、运营和平台团队的其他同事。

这特别容易理解为什么True会筹集另一只规模略大的机会基金(它的最后一只基金在2018年以2.85亿美元收盘，同时它的最后一只早期基金以3.5亿美元收盘)。正是通过这样一个工具，True能够对消费者健身公司Peloton进行如此大的投资，包括其F轮系列。

当该公司去年秋天以72亿美元的价格上市时，True是该公司在T2的第二大外部股东。大约一年后，公开市场投资者对Peloton的估值超过380亿美元，True仍然参与其中。

本周早些时候，我们与Callaghan和Black讨论了True如何以及何时在公开交易的投资组合公司中解除这种头寸。我们还谈到了该公司继续强调为其创始人和团队创建支持网络，无论他们是否担心创业投资的中心正在转移出湾区等。下面的大部分聊天内容都经过了长度编辑。

TC:过年了，资金更大？

PB:数字更大，但这是很多人的功能，他们想成为我们正在做的事情的一部分，对此我们很感激。我们还有一个更大的团队。

JC:我们超过90%的LP都重新加入了；我们的需求远远超过了我们的供给，包括因为我们认为每次筹集资金时从新的关系中引入新的资本是很重要的，通常是从我们认识很久的人那里。我们积极参与LP基础，这为我们提供了新的思路。

这些风险投资公司中有很多还是没有？似乎越来越多的人都是其他人基金的投资者。

PB:我们有基金的基金，像Greenspring Associates和Foundry Group Next(作为投资者)。我确实认为，当涉及到规模较小、低于5000万美元的基金时，你会看到(风险基金投资于风险基金)，但这不适用于我们。

随着时间的推移，你已经为一家公司20%的股份开出了相当小的早期支票。这在这个市场上还有可能吗？

JC:我们的核心业务保持不变。我们从第一天开始就给一个创始人或小团队写100万到300万美元。我们在看待事物的方式上有所改进。。。大约10年前，我们已经在我们的平台上投入了1500万美元[以利用我们团队的广度和更广泛的网络]。如果我是你的董事会成员，我是你唯一的联系人，那么我就是薄弱环节。

TC:你们有创始人营，真正的大学和众多的文化倡议。就多样性而言，你如何评价这家公司？

JC:我们正在努力做得更好，但作为一个行业，我们还不够好，我们是这个行业的一部分。我们已经资助了非常强大的女性企业家和一些有色人种，但还不够，我们现在正在寻找长期的解决方案。我们也非常关注奖学金(True表示，多年来，True已经通过奖学金招募了165名大学生在True支持的初创公司工作，其中一半人后来在这些公司找到了全职工作)。

我们一直关注性别平等，在最后两节课中更倾向于女性，但我们也关注多元化的候选人和多元化的背景。我们需要提供更多进入技术和创业公司的途径，通过奖学金，我们可以在学生考虑职业生涯之前接触他们。

TC:你投资的公司之一Peloton今年表现特别好。你卖掉那个职位了吗？如何看待公司上市后退钱给LPs？

JC:我们仍然是公司股票的持有者，我仍然在董事会。

PB:我们到底是卖股份还是分发股份，真的是个案处理。这需要时间，因为我们通常是大股东，我们收回资本的能力受到公开市场和(我们拥有的)数量的限制。至于我们是派发现金还是股份，通常是LP喜欢股份。许多家族理财室和基金中的基金更喜欢这些股票，因为它们会将这些股票推荐给自己的公开(市场)团体。其他一些人，尤其是欧洲投资者，出于税收或其他原因，更喜欢现金。

我们开始看到更多由风险基金发起的特殊目的收购公司。你对这些有什么看法？

PB: Jon和我已经有足够长的时间来记住SPACs是一个四个字母的单词。我认为它们是比20年前更好的工具。我们的一些公司正在考虑它们；人们对募集特殊目的收购公司或通过特殊目的收购公司上市的意义充满好奇。人们处于寻找信息的模式。作为一家公司，我们从未想过我们应该自己培养一个特殊目的收购公司。

JC:我认为围绕资本获取的创新非常有趣。我认为现在判断创始人和公司会如何发展还为时过早。这很复杂。我们正在密切关注。

TC:真实往往走在曲线的前面。你投资了硬件公司，比如Fitbit和Peloton，领先于许多其他多面手公司。数字健康和生物学也是如此。你现在对什么感兴趣？

JC:我们的工作是倾听我们的创始人向我们传达什么，我们正在思考的一个领域是工作的未来——不是一年而是五年后会发生什么。第二个主题是我们称之为咆哮的20年代，我的意思是我们正在研究第一次世界大战和第二次世界大战后，消费者行为是如何变化的，以及在这次疫情之后，如果有了疫苗，会发生什么。

TC:你能在这方面再深入一点吗？

JC:我们在考虑艺术、音乐、餐饮、旅游、娱乐。。。百老汇开放后会发生什么？你可以想象混合体验。关于工作的未来，我不确定我们最初会在Zoom上花很多时间(一旦世界重新开放)，但(我们认为)虚拟访问一切，让远程员工有更多(在他们的指尖)，以及郊区与城市的变化，我们已经通过(过去和现在的投资组合公司)Blue Bottle [Coffee]和Sweetgreen和Madison Reed有了一些见解。

TC:许多公司正在将远程工作永久化。你认为这是一个看起来势如破竹的趋势吗？如果是的话，这将如何改变湾区？

JC:我们最近有一个虚拟的异地，在最后的30秒钟里，每个人都在谈论我们是否可以访问COVID测试，以便我们可以把每个人聚集在一起。

人们在人际交往中茁壮成长；我觉得他们需要在一起。我的观点是，湾区仍然是文艺复兴时期的佛罗伦萨，它会很好。这将需要一段时间，但这仍然是很多人才出于各种原因想要去的地方。
