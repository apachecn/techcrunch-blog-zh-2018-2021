恒星区块链将很快支持 USDC 稳定币

由 Circle 和比特币基地共同创建的 stablecoin)最初是一种基于以太坊的代币。在增加了对阿尔格兰德区块链的支持后，中心，管理 USDC 的财团，宣布恒星将成为第三个支持 USDC 的区块链。该项目将在 Q1 2021 期间的某个时候推出。

USDC 是一种跟随美元价值的加密货币。一 USDC 总是值一美元——因此得名稳定币。发行新 USDC 时，USDC 合伙人会在银行账户上保留一些美元。

这些账户定期接受审计，以证明这些账户中流通的 USDC 和美元一样多。目前有 28 亿 USDC 在流通。

当涉及到储存价值时，稳定的硬币提供了一些灵活性，因为你不需要银行账户。但基于以太坊的 stablecoins 也面临着以太坊用户目前面临的同样问题——交易速度慢，交易费用高。

通过让你选择发送恒星区块链 USDC，交易应该更快，更便宜。基本上，你必须选择发送 USDC 的频道——以太坊、恒星或阿尔格兰德。如果你从一个交易所发送到另一个交易所，或者从一个钱包提供商发送到另一个，你必须确保两者都支持特定区块链上的 USDC。

Circle 还开发了一个多链 USDC 交换 API 。例如，如果出现流动性问题，授权开发人员可以利用这个 API 进行跨链互换。

如果有足够多的开发人员采用这种交换 API，应该会改善最终用户的体验。你将不必太注意你用来发送和接收 USDC 的区块链。

所以这仍然是一项正在进行的工作。一些产品可以选择专注于一个特定的区块链，其他产品可以让你在以太坊顶部的 USDC、阿尔格兰德顶部的 USDC 或恒星顶部的 USDC 之间进行选择。其他人可能会使用抽象层，这样你就可以透明地将 USDC 从以太坊钱包发送到恒星钱包。幕后将会发生一些交换。
