由女性打造的女性加速器 Ready Set Raise 宣布进入第三等级 TechCrunch

2018 年，女性创始人联盟的创始人 Leslie Feinzaig ，推出了一个名为 Ready Set Raise 的免费无股权女性加速器。其目标是为缺乏人脉的女性创业者提供筹集资金所需的指导和人脉。

今年，随着对女性创始人的资助下降到 2017 年的水平，Feinzaig 意识到为什么加速器，包括她的加速器，可能不像对男性那样对女性有效:演示日。在大多数加速器中，演示日是一个常见的高潮事件，创始人在这个事件中向投资者、天使和记者推销，希望筹集一轮资金并获得一些报道。

“事实是，你不能在五分钟的高度照本宣科、精心准备和练习的基础上筹集资金，”Feinzaig 继续说道。“你可以在任何时候、任何背景下向任何人推销你的创业，让他们足够兴奋，愿意参与你的旅程。”

因此，Feinzaig 表示，“啊哈时刻”导致 Ready Set Raise 改变了其将持续八周的节目，更加专注于由数百名女性审查的“现实的筹款过程”。

冠状病毒影响了加速器的工作方式。Y Combinator 和 Techstars 搬到了虚拟编程网站 live，它有机会让父母或一年中有三个月不能搬到帕洛阿尔托的人更容易接触到。也就是说，Y Combinator 的最新一批公司在多样性方面有所下降，只有 16%的公司创始人是女性。在前一批公司中，近 21%的公司创始人是女性。

访问权限的下降使得 Feinzaig 的工作变得更加困难和重要。今年，随着准备加薪的申请蜂拥而至，Feinzaig 注意到比以往更多的成熟公司提出了申请。这一细节让《创立者》调查了女性创始人，发现在疫情之前有创业雄心的女性现在不太可能这么做了。尽管如此，她还是很乐观，称他们看到了比以往任何时候都多的“最高水平的申请者”。

今天，Ready Set Raise 宣布了其第三批成员，包括一家将销售户外设备的零售商数字化的初创公司、一个进行道德和法律数据交换的市场以及一个连接黑人女性和文化意识供应商的数字平台。

以下是 Ready Set Raise 的第三批创业公司:

●brillingly:brillingly 由劳拉·维蒂希(Laura Wittig)和丽莎·莫伊塞娃(Liza Moiseeva)创立，是一家结合商业、内容和社区的初创公司，目标是扩大有意识的消费主义。它位于华盛顿的西雅图。

● Womp.ai :由 Gabriela Trueba 创立，Womp 希望帮助任何人探索、创造和分享 3D。它位于纽约的布鲁克林。

●fix fake:fix fake 由 Kathryn Harrison 和 Jason Law 创立，提供决策支持工具以减少电子商务中的欺诈。它位于蒙大拿州的波兹曼。

●TBD health:TBD health 由 Stephanie Estey、Daphne Chen 和 Sherwin Lu 创立，是一个专为女性打造的家庭性传播疾病筛查平台。它的总部设在纽约。

● Gearo :由贾丝汀·巴隆(Justine Barone)创立，Gearo 将户外零售商的运营数字化，并吸引冒险家作为顾客。它位于科罗拉多州的丹佛市。

● 玛丽·露易丝化妆品:玛丽·露易丝化妆品由阿基拉·雷福德创立，销售天然护肤和个人护理产品。它位于加利福尼亚州的洛杉矶。

● datacy: 由首席执行官 Paroma Indilo 和 Kaleb Wilson 创建，datacy 是一个专注于实现道德和法律数据交换的市场。它位于加利福尼亚州的圣何塞。

● Health In Her HUE :该公司由 Ashlee Wisdom 和 Eddwina Bright 创建，是一个连接黑人女性和文化能力提供者的数字平台。它的总部设在纽约。

这门课将于 10 月 19 日开始。会员将接受来自各种合作伙伴的指导，包括库利 LLP、Carta、Grasshopper Bank、Madrona Venture Group、UPS 和 Zendesk for Startups。
