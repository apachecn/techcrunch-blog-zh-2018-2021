Lux 支持的 Flex Logix 宣布为 edge 提供快速廉价的 X1 AI 芯片

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/20/flex-logix-chip-availability/amp/

图片来源: MirageC / Getty Images

在计算机世界里，可能有比你当地超市小吃区更多的芯片可供选择。多样化的计算环境(数据中心和云、边缘、移动设备、物联网等)、不同的价格点以及不同的功能和性能要求正在扰乱芯片行业，重新确定谁现在处于领先地位，谁可能在新的和新兴的利基市场中处于领先地位。

虽然在过去的两年里，风险投资家资助了大量新的芯片创业公司，如 Cerebras 、 SiFive 和 Nuvia ，但 Flex Logix 的立足点要早一些。该公司由 Rambus 前创始人杰夫·泰特和王乘于 2014 年创立，已从投资者 Lux Capital 和 Eclipse Ventures 以及泰特本人那里筹集了 2700 万美元。

Flex Logix 希望将人工智能处理工作流带到计算边缘，这意味着它希望提供将人工智能添加到医疗成像设备和机器人等产品中的技术。在边缘，处理能力显然很重要，但尺寸和价格也很重要。更高效的芯片更容易包含在产品中，定价可能会限制单个组件的成本。

在公司成立的最初几年，它专注于围绕 FPGAs 开发和授权知识产权，或通过软件制造后可以改变的可重编程芯片。这些灵活的芯片在 AI 或 5G 等标准和模型快速变化的应用中至关重要。这是一个由 Xilinx 和 Altera 主导的市场，英特尔在 2015 年以 167 亿美元收购了 Altera。

Flex Logix 通过帮助其他公司开发自己的芯片，看到了成为“FPGAs 的臂膀”的机会。它与桑迪亚国家实验室、国防部和波音公司等机构合作，为其设计赢得了客户的青睐。最近，它一直在开发自己的芯片系列 InferX X1，创造了一种混合商业模式，这种模式与 Nvidia 在收购 ARM 清除监管障碍后的模式没有什么不同。

随着这一背景的消失，Flex Logix 公布了其 X1 芯片的可用性，该芯片目前计划以从 533Mhz 到 933Mhz 的四种速度提供。首席执行官泰特在电话中强调，该公司的关键差异是价格:根据小订单的芯片速度，这些芯片的价格将在 99 美元至 199 美元之间，大订单的芯片价格为 34 美元至 69 美元。

这是一个芯片，好吧。不是很多伟大的股票艺术。但这里是 X1。通过 Flex Logix 拍摄照片。

这些芯片更便宜的原因是，根据 Tate 的说法，它们比英伟达(Nvidia)Jetson 芯片系列中的竞争芯片小得多，尺寸只有 1/7。较小的芯片通常成本较低，因为芯片工厂中的每个晶片可以容纳更多的芯片，将制造成本分摊到更多的芯片上。据该公司称，其芯片性能优于 Nvidia 的 Xavier 模块，但没有独立的基准测试。

Tate 解释说:“我们接触的每一个客户都希望每美元有更高的处理能力，每单位功率有更高的处理能力……凭借我们的芯片尺寸优势，我们可以为他们提供更高的性价比。”。

这些新芯片的客户样品预计将于明年第一季度到达，第二季度开始大规模生产。

该公司的计划是继续其业务的两个方面，并继续发展和成熟其技术。“我们的嵌入式 FPGA 业务现在作为一个独立的业务是盈利的。我们赚的钱超过了工程和商业。现在我们正在开发推理这一新业务，它最终将成为一个更大的业务，因为推理领域的市场增长非常快，”泰特解释道。

该公司的董事会由 Lux 的 Peter Hébert 和 Shahin Farshchi、Eclipse 的 Pierre Lamond 以及微软 Azure 的杰出工程师 Kushagra Vaid 组成。该公司位于加利福尼亚州的山景城。
