Sym 为其安全工作流平台 TechCrunch 筹集了 900 万美元的 A 轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/20/sym-raises-9m-series-a-for-its-security-workflow-platform/

Sym 是一个新的平台，可以让开发人员更容易地将安全和隐私工作流程集成到他们的流程中，今天宣布它已经筹集了 900 万美元的 A 轮融资，由 Amplify Partners 领投。今年早些时候，该公司宣布获得由的安迪·麦克洛林【Uncork Capital 和 芒果资本 的罗宾·瓦桑【Robin Vasan】领投的 300 万美元种子融资。天使投资人包括前谷歌 CISOGerhard Eschelbeck，Atlassian CTOSri Viswanath和 Jason Warner ，GitHub 的 CTO。

Sym 的联合创始人 Yasyf Mohamedali 在过去几年中担任健康技术公司 Karuna Health 的首席技术官。在这个职位上，他非常熟悉在高合规性行业中工作，处理供应商审查和安全审计。为了使这些过程更加有效，他的团队开发了许多小工具，但他意识到业内其他人也在做同样的事情。

图片来源: Sym

“作为一名工程师，当你看到人们一遍又一遍地建造同样的东西时，会感到沮丧，”穆罕默德阿里告诉我。“多年来，我脑子里一直有这样的想法，‘如果我们只做一次，然后人们就不用一遍又一遍地重复做同样的事情，会怎么样？’因此，当我离开卡鲁纳创办 Sym 时，最初，我想做的正是这个——特别是为了 HIPAA。这是一种天真的方法，我就像是，‘你知道，我只是要构建所有的工具，有人需要做 HIPAA 并像黑盒一样开源它。"

然而，他意识到的是，公司有自己的安全和治理工作流，这些工作流往往共享相同的核心，但也有很多可变性。因此，Sym 现在所做的是提供这些核心工具，并让公司从公司创建的以开发人员为中心的工具箱中混合和匹配他们需要的东西。

Mohamedali 解释说:“我们正在构建的是一组工作流模板和原语，它们映射到共享的 20%核心，然后是一组集成，您可以使用这些集成来简化这些工作流模板，并通过将这些模板连接到所有不同的服务来整理最后一英里的差异。

这种方法的有趣之处在于，Sym 提供了一个 Python SDK，让开发人员只用几行代码就可以创建这些工作流和集成。在某种程度上，这是因为该公司的理念是让工程师重新控制安全性，就像 DevOps 允许他们重新控制基础架构和问答一样。所以现在，DevSecOps 需要成为一个东西。我们需要重新获得安全感。我们希望成为实现这一目标的工具，”他说。

Mohamedali 强调，这在很大程度上是一轮机会主义，在接下来的几个月里，这次融资不会改变公司的路线图。但是因为 Sym 开始与大客户签约——并对他们做出承诺——现在是融资的好时机，尤其是因为合适的合作伙伴出现了。这意味着雇佣更多的工程师，但随着时间的推移，该公司显然也计划建立自己的销售和营销团队。不过，该产品本身仍将处于私下测试阶段，直到明年年中左右。届时，Sym 还将推出其平台的自助版本。
