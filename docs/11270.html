Boston Dynamics 的 Spot 明年将安装一个自动充电底座 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/21/boston-dynamics-spot-is-getting-an-arm-and-self-charging-dock-next-year/

波士顿动力公司的车位明年将安装一个自动充电底座

波士顿动力公司的新任首席执行官 Rob Playter 告诉 TechCrunch，截至他上个月在 Disrupt 露面时，该公司已售出约 260 台复杂的 Spot 机器人。虽然该公司面临着一些关于这款价值 7.5 万美元的机器人的商业吸引力的问题，但很明显，许多垂直行业都有兴趣找到部署这项技术的方法。

Spot 的众多吸引力之一是，它的定位是为开发者和第三方提供一种平台，他们可以为从建筑到远程医疗的一系列不同应用开发自己的配件。但波士顿动力公司也在积极开发自己的配件，以帮助 Spot 的应用多样化。

该公司最近宣布，它将提供一种能够执行多种任务的 arm 插件，包括开门和拿起物体。添加是显而易见的，因为这只手臂在多年前的第一部 Spot/Spot 迷你视频中就出现过。事实上，当该配件被排除在该公司第一个商业产品的初始发布之外时，我真的有点失望。

图片来源:波士顿动力公司

机械臂将在下一个临近的时间点到达。它有六个自由度，被设计成随机器人一起移动。“和基本机器人一样，”该公司写道，“手臂不仅仅是硬件。它将配备一个直观的用户界面，并通过平板电脑进行远程操作和监督自主行为。”

开发人员也可以通过 API 访问机械臂/抓手。像开门、抓取和拖动物体这样的应用程序将被自动化，并在 arm 发布时作为测试版功能提供。

图片来源:波士顿动力公司

波士顿动力公司也宣布了一个以企业为重点的机器人版本，该版本具有自充电坞站。像一个大而复杂的 Roomba 一样，Spot 将能够在没有引导的情况下返回到坞站进行充电。该装置是为石油钻塔和辐射危险区等环境设计的，在这些环境中，机器人可以在没有人在场的情况下理想地工作。

这也将在明年年初到来。以上的定价仍然是 TBD。
