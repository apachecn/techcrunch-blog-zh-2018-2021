宣布 TC 会议议程:太空 2020

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/22/announcing-the-agenda-for-tc-sessions-space-2020/

TC Sessions: Space 将于今年 12 月 16 日和 17 日举行，这是我们有史以来第一次专门的太空活动。这是一个为期两天的现场虚拟会议，与会者都是航天工业中最重要的人物，涉及公共、私人和国防领域。

我们很高兴能够邀请到美国国家航空航天局局长吉姆·布里登斯汀、火箭实验室首席执行官兼创始人彼得·贝克、美国太空部队太空作战部长杰伊·雷蒙德将军、洛克希德·马丁公司副总裁兼民用太空项目负责人丽莎·卡拉汉以及其他许多人。除了虚拟舞台的炉边和小组讨论，该活动还将包括网络，启动演示和与来自世界各地的与会者联系的机会。

下面，你会发现 TC 会议的官方议程:太空。这两天已经排得满满的，但我们还准备了一些额外的惊喜，所以请关注未来几周的日程，我们将增加更多优秀的演讲者和会议。

如果你想成为这个活动的一部分，你可以抢到一张票，独家观看这些会议的现场直播(可以观看视频点播)，与改变太空行业的创新者建立联系，发现最热门的初创公司，了解如何为你的太空公司获得资助，招募人才，甚至只需 125 美元的早鸟票就可以找到工作。我们还为群体、学生、现役军人/政府雇员和早期太空初创公司创始人提供折扣，他们希望让自己的初创公司更加引人注目。

议程

12 月 16 日星期三

小行星岩石和登月与丽莎·卡拉汉 ( 洛克希德·马丁太空公司 )

从机器人从星系旅行的小行星表面铲起火箭，到为人类重返月球表面做准备，我们将涵盖太阳系科学和民用探索的所有方面。

从太空岩石回报到财务回报——一个投资者小组，成员有克里斯·波修森 ( DCVC )、迈克·科利特 ( 普罗莫斯风险投资)和泰丝·哈奇 ( 贝塞麦风险投资合伙人)。

一些投资者花了很多时间去寻找下一个风险投资机会。这是一个与众不同的市场，但这会改变基于股票的投资的数学吗？

与帕亚姆·巴纳扎德 ( 五车二太空)彼得·普拉茨 ( 尖顶环球)和拉斐尔·莫德泽夫斯基 ( 冰眼 ) 一起创办俯视地球的企业

地球观测如何成为航天领域真正的赚钱机器之一，以及该行业的前景。

联网中断

通过我们的虚拟平台，与会者可以通过视频聊天进行交流，让人们有机会建立有意义的联系。我们的算法匹配产品 CrunchMatch 将确保你在展会上遇到合适的人，并为更具冒险精神的与会者提供随机匹配。

向约翰·汤姆逊中将 ( 美国太空部队 ) 采购太空安全技术

汤普森中将负责培育非传统太空初创公司的生态系统和太空部队采购的未来，所有这些都是为了保护全球太空公域的最终目标。他将谈论美国在创业伙伴关系和新兴技术中寻求什么，以及它如何与这些年轻的公司合作。

与蒂姆·埃利斯 ( 相对论太空)和克里斯·坎普 ( 阿斯特拉 ) 一起创办一家发射创业公司

发射业务正在蓬勃发展，但除了 SpaceX 和火箭实验室之外，没有人能够真正利用新的太空创业公司。我们将与希望成为下一个目标的公司的创始人交谈。

与詹姆斯·布里登斯汀 ( 美国宇航局 ) 一起将创业带到月球、火星以及更远的地方

美国国家航空航天局将重返月球——这一次，留下来。在管理员 Bridenstine 的领导下，该机构在实现这一目标方面取得了巨大进展，他将加入我们，谈谈他们这次是如何与私人合作伙伴合作的，包括许多初创公司。

12 月 17 日星期四

与 Jay Raymond 将军(美国航天部队)在空间防御领域的公私伙伴关系

听听美国太空部队负责人如何确保一个全新的作战领域的安全，以及美国军方的最新分支如何寻求私营企业来实现这一目标。

从想法到轨道与彼得·贝克 ( 火箭实验室 )

火箭实验室已经迅速成为世界上最受欢迎的发射供应商之一。创始人兼首席执行官彼得·贝克将讨论该公司使太空更容易进入的方法，从更便宜、更快的发射到新的卫星平台。

与梅根·克劳福德 ( 太空基金)和j·克里斯托弗·莫兰 ( 洛克希德·马丁风险投资 ) 一起沟通今天和明天的技术

企业风险投资基金是太空初创公司的一个重要投资来源，部分原因是它们通常涉及有助于创收的合作伙伴关系，还因为它们了解所涉及的时间表。我们将讨论它们如何适应更标准的风险投资来推动生态系统。

如何用威尔·罗珀 ( 美国空军 ) 让空军买你的东西

我们将讨论了解空军需要什么以及如何向他们销售的最佳方式。

地面控制呼叫少校汤姆与约翰·格马克 ( 阿斯特拉尼斯)本·朗梅尔 ( Swarm 科技)和米娜·米特里 ( 开普勒通讯 )

数据连接和通信是商业空间货币化和进一步空间探索和发展的战略计划的关键。听听关键人物对行业现状的看法。

在太空中，没有人能给你换油——然而有了丹尼尔·费伯 ( 轨道工厂)和罗恩·洛佩兹 ( 天文刻度 )

一旦宇宙飞船进入轨道，它就只能靠自己了——但是如果它可以被补给燃料、修理、翻新，如果必要的话，还可以退役呢？OrbitFab 创始人丹尼尔·费伯和 Astroscale 美国公司总裁罗恩·洛佩兹将讨论太空操作如何颠覆今天的工程和商业模式。
