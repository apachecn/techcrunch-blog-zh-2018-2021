脸书为 WhatsApp 业务增加了托管、购物功能和定价层

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/22/facebook-adds-hosting-shopping-features-and-pricing-tiers-to-whatsapp-business/

脸书一直在努力成为中小企业的合作伙伴，这些企业利用互联网与更广阔的世界进行互动，其消息平台 WhatsApp 一直是这一宣传的核心部分，该平台拥有约 5000 万家企业和 1.75 亿用户(总用户数超过 20 亿)。

现在，该公司正在对 WhatsApp 进行三大补充，以实现这一主张。

它推出了一种在 WhatsApp 聊天平台上购物和支付商品和服务的方式；它将与世界上的主机服务提供商并肩作战，推出一款名为脸书主机服务的新产品，来托管企业的在线资产和活动；此外——与其不断扩大的产品范围相一致——脸书表示，它最终将开始向使用 WhatsApp 开展业务的公司收费。

脸书在一篇简短的博客文章中宣布了这一消息，并透露了细节。我们已经联系了该公司，以获取更多关于定价、服务可用性以及脸书是否将自己提供主机服务或与第三方合作的信息，当我们了解更多信息时，我们将更新这篇文章。

更新:脸书做出了回应，我们将把回复放在下面有意义的地方。

目前我们知道的是:

聊天购物:企业已经在使用 WhatsApp 展示产品信息，并发起交易讨论。该领域最近的发展之一是增加了二维码和在聊天中分享目录链接的能力，于 7 月增加。与此同时，脸书一直在扩大企业在脸书和 Instagram 上展示所售商品的方式，最近和在 8 月推出了脸书商店，此前在 Instagram 上推出了类似的产品。

今天的举措听起来像是一种新的方式，让企业转而使用 WhatsApp 链接到脸书本地的目录以及其他产品，然后购买商品，同时仍然保持聊天。

与此同时，脸书将使商家可以在其他地方添加“购买”按钮，将购物者带到 WhatsApp 聊天平台完成购买。“我们还希望让企业更容易将这些功能集成到他们现有的商业和客户解决方案中，”它指出。“这将帮助许多在这次危机中受影响最大的小企业。”

尽管脸书没有称之为 WhatsApp Pay，但这似乎是该公司将支付引入其消息应用聊天流的雄心的下一步。对该公司来说，这是一条漫长而曲折的道路，今年 6 月，终于在巴西推出了使用脸书支付的 WhatsApp 支付，却因未能满足监管机构的要求而被关闭。(计划下一步将扩展到印度、印度尼西亚和墨西哥。)

脸书主机服务:这些服务将在未来几个月内推出，但目前还没有具体的日期。该公司在给 TechCrunch 的一份声明中表示:“我们现在正在分享我们的计划，同时与我们的合作伙伴合作，使这些服务可用。”

不要！这与脸书挑战 AWS 无关。或者…至少还没有？这里的想法似乎是专门针对那些已经使用脸书和 WhatsApp 消息的中小企业销售托管服务，这些中小企业要么已经使用托管服务来管理他们的在线资产，无论是他们的在线商店还是其他东西，要么发现自己现在第一次需要托管服务，因为业务就是要“在线”

“今天，所有使用我们的 API 的企业要么使用内部解决方案，要么利用解决方案提供商，这两者都需要昂贵的服务器来维护，”脸书说。“随着这一变化，企业将能够选择免费使用脸书自己的安全托管基础设施，这有助于为每个希望使用 WhatsApp 商业 API 的公司(包括我们的商业服务提供商)消除一项昂贵的费用，并将帮助他们节省资金。”它补充说，它将分享更多关于数据将在更接近发射的地方托管的信息。

这是一个非常有趣的举动，因为中小型企业托管市场相当分散，有许多公司，包括像 GoDaddy，DreamHost，HostGator，BlueHost 和其他许多也提供这些服务的公司。这种分裂为脸书这样的大公司带来了机会，它拥有全球形象，通过这些中小企业的其他在线服务和世界各地的数据中心网络建立了大量连接，它已经为自己建立了一个非常广泛的网络，现在可以用来为其他人提供服务——这确实与亚马逊和 AWS 的业务方式非常相似。

脸书已经有了一个各种各样的“T0”应用商店，与合作伙伴一起为使用其平台的企业提供营销和相关服务。看起来它计划扩展这一功能，并将在出售主机的同时出售所有这些功能，更重要的是，在脸书上本地托管将加快一切工作的速度。

“提供这一选项将使中小型企业更容易起步、销售产品、保持库存最新，并快速响应他们收到的消息——无论他们的员工在哪里，”它指出。

收费等级:正如你所料，为了鼓励更多的采用，脸书到目前为止还没有对 WhatsApp 业务收费，但它对一些 WhatsApp 业务信息收费——例如，当企业通过脸书的铁路向客户发送登机牌或电子商务收据时。(这些价格各不相同，在这里公布了它们的清单。)现在，随着更多的服务加入进来，企业将自己的命运与在脸书平台上的表现更加紧密地联系在一起，脸书将这种情况转变为付费游戏也就不足为奇了。

“在过去几年里，我们听到的是商业信息的对话性质如何对人们真正有价值。因此，未来我们可能会想办法更新我们对企业的收费方式，更好地反映它的使用情况，”该公司告诉我们。重要的是要注意，这将关系到企业如何发送信息。“和往常一样，人们给企业发信息是免费的，”脸书补充道。

令人沮丧的是，到目前为止，似乎还没有关于哪些服务将被收费，收费多少，何时收费的细节，所以这更多的是一个警告，而不是一个新的要求。

“我们将向商业客户收取我们提供的一些服务的费用，这将有助于 WhatsApp 继续建立自己的业务，同时我们为超过 20 亿人提供和扩展免费的端到端加密文本、视频和语音通话，”它指出。

对于那些可能觉得这很烦人的人来说，从积极的一面来看，对于那些担心数据怪兽不断入侵的人来说，这至少会帮助 WhatsApp 和脸书继续坚持其长期以来的承诺，即远离广告作为一种商业模式。

中小企业双倍下注

这些新服务推出之际，脸书正加倍努力为企业提供服务，这在很大程度上是受冠状病毒疫情的刺激，该病毒已迫使实体零售商和其他公司关闭实际大门，将重点转向使用互联网和移动服务与客户联系并向客户销售。

该公司的首席运营官·雪莉·桑德伯格上个月发布了脸书商务套件，集合了该公司一直在为企业打造的所有工具，以更好地利用脸书、Instagram 和 WhatsApp 的个人资料来宣传自己，以及与客户沟通和向客户销售。桑德伯格牵头宣布这一消息的事实说明了脸书是如何优先考虑这一点的:它在使用其平台的公司面前表现突出，但它看到/希望商业服务可以成为实现其商业模式多样化的一个关键途径，同时也有助于缓冲它——因为许多建立页面的企业也可能做广告。

脸书也一直在脸书和 Instagram 上构建更多功能，专门旨在帮助超级用户和企业以更有效的方式利用这两者。向 WhatsApp 添加更多工具是这一切的自然进展。

诚然，正如我们今年早些时候在指出的，尽管世界各地的企业都在非正式地使用 WhatsApp，但 WhatsApp 业务仍然是一个相当小的产品，在印度和巴西最受欢迎。脸书推出更多如何使用它的工具，不仅可能在这些市场推动更多业务，还可能帮助该公司在其他地方将更多业务转化为使用它。

小企业进入脸书的视野已经有一段时间了。甚至在疫情袭击之前，在许多情况下，零售商或餐馆没有自己的网站，选择脸书页面或 Instagram 个人资料作为他们的 URL 和与世界的主要在线界面；即使他们有独立的网站，他们也更可能在社交媒体上更新人们的信息，传播他们正在做的事情，而不是通过自己的网址。
