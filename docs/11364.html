当然，Hero在一次罕见的edtech收购中收购了Symbolab

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/27/course-hero-symbolab-makes-a-rare-edtech-acquisition/

在其8000万美元的B轮融资几个月后，课程英雄收购了 Symbolab ，这是一款人工智能计算器，可以帮助学生回答和理解复杂的数学问题。

这笔交易的价格没有透露。位于特拉维夫的9人Symbolab团队将加入课程Hero。这些平台将在不久的将来以独立品牌存在，据当然英雄的首席执行官安德鲁·格劳尔说。

Grauer说，Symbolab成立于2011年，是一个高级计算器和问题求解器，今年有望回答10亿个问题。该服务非常注重大学水平的数学，并通过解释和证明解决复杂的几何问题。

课程英雄核心是一个问答平台。增加一个计算器和一家拥有最常被问到的数学问题数据集的公司，可能会让Course Hero在数学服务上拥有更大的优势，该公司表示，这是目前学生中最受欢迎的科目之一。这项服务将作为优惠条件提供给课程英雄的订户。

使用计算引擎为受困学生提供解决方案的模式相当普遍。在一个远程教育的世界里，对于那些可能迷失在课堂上的学生来说，灵活性是关键。虽然教师和导师可能只在一定时间内可用，但人工智能支持的技术服务可能证明是学生可以依赖并支付费用的全天候解决方案。

Symbolab感觉类似于 Wolfram Alpha ，一个流行的计算引擎。虽然Grauer说Wolfram Alpha是一个“强大的工具”，但他认为Symbolab在深度和解释方面做得更好。大公司也增加了类似的服务，例如谷歌在2019年收购了作业助手应用Socratic ，微软在同年开发了 Microsoft Solver 。

格雷尔不得不在建造还是购买之间做出决定。这位创始人最终决定收购这项技术，因为只有当一个平台能够随着时间的推移复合数据时，人工智能才能取得真正的成功。Symbolab成立近十年，后端信息很有价值。Grauer说，他很高兴能够以不同于谷歌和微软的方式解决规划求解问题。

“你不能在短时间内就(这么做)，”他说。“你在寻找我们如何得到正确、准确的答案。但接下来，我将如何获得不仅准确，而且实际上有帮助的逐步解决方案。”

在这个历来资金不足的领域，整合仍很少见。教育科技的收购一直在稳步增长，但速度缓慢。2018年，edtech的收购不到40起。根据Crunchbase的数据，同年，金融科技进行了193起收购。

尽管如此，在edtech更大的繁荣中，这次收购还是很有意义的。Course Hero最近筹集了迄今为止最大的一笔资金，年运营收入超过1亿美元，并开始盈利。因此，该公司可能有足够的资金来支付这笔交易。2012年，Course Hero从InstaEdu的创始人手中买下了红衣主教学者。

Grauer说，他预计Course Hero未来将在各种学科领域进行更多收购。在更广泛的教育技术领域，他认为未来5到10年将会有更多的收购。

“如果你回顾15年前，还没有那么多的教育技术企业，”Grauer说。“现在，我认为有足够多的公司有可能达到这一规模，两家公司都有分销方面的技术指标，并在扩张阶段建立了产品市场。”

编者按:这篇文章称，课程英雄在2012年收购了InstaEdu。这是不准确的。该公司从InstaEdu的创始人手中收购了红衣主教学者。已进行更新以反映这一更正。
