在筹集了 2500 万美元 TechCrunch 后，洛杉矶复制了增加 R&D 和招聘运营管理软件的计划

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/27/las-replicated-looks-to-increase-rd-and-hiring-for-its-operations-management-software-after-raising-25-million/

在筹集了 2500 万美元后，洛杉矶的复制看起来增加了 R&D 和招聘其运营管理软件

总部位于洛杉矶的运营和安全管理软件服务 Replicated 已经筹集了 2500 万美元来增加员工人数，扩大销售和营销工作。

这笔资金由两家适马风险投资公司牵头，包括 Plexo Capital、Amplify 和 BoldStart 等现有投资者，对该公司的估值超过 1 亿美元。

Replicated 最初是一家为 Docker 容器化开发工具提供软件安全和管理服务的开发商，但随着市场转向 Kubernetes，该公司转向为这些应用提供服务。

去年，该公司推出了管理 Kubernetes 部署的工具，并立即看到了销售增长。

“新的产品改变了公司的方向，提高了速度和客户，市场对此反应良好，”首席执行官格兰特·米勒说。“一旦我们开始在 UIPath、Puppet 和 TripWire 上取得成功，他们创造了软件自动化的东西，然后是 FlatFile 和 Fishtown Analytics 等新兴的酷软件公司。”

Replicated 现在有 85 个客户付费访问该平台，并购买客户通过该平台开发的每个内部部署。“如果你是一家像 Puppet 这样的公司，你有数百个本地客户。”

该公司已经开始招聘关键高管，包括一名首席产品官、一名首席收入官以及营销和销售副总裁。一部分资金将支持 Replicated 继续招聘，而另一部分将用于产品开发。

“我们正在尽可能多地自动化需要 IT 管理员将手放在键盘上进行调整和优化的手动软件流程，”Miller 说。“使用 Replicated，一半的服务用于软件供应商管理发布和许可的工具，另一半用于 IT 管理员。关键是他们如何实现应用程序集成的自动化。”

米勒说，该公司目前有 28 名员工，预计到 2021 年底将增加到 75 名员工。

“本地软件市场是 SaaS 市场的四倍，”米勒在一份声明中说。“Replicated 使软件供应商能够以更低的工程开销和更快的上市时间释放本地软件类别的机会。这是云原生计算和 Kubernetes 的融合带来的市场变化。”
