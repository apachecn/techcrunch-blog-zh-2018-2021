Stripe Climate是一个新工具，让Stripe客户进行碳排放购买

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/28/stripe-climate-is-a-new-tool-to-let-stripe-customers-purchase-credits-towards-carbon-removal-efforts/

去年，支付巨头 Stripe 宣布，它将每年捐赠100万美元自有资金给那些正在开发技术从我们的环境中去除碳的公司，今年5月在宣布了这笔投资的受益人。现在，它正在扩大这一承诺，推出一款新产品，旨在让客户也进行投资。

今天，该公司推出了 Stripe Climate ，这是一种新工具，使用Stripe的公司可以集成这种工具，以设置自动捐款，按每笔交易的百分比计算——公司可以自己设置百分比——收益将注入Stripe对碳减排公司的投资之外的附加基金。

目前投资名单上有四家公司: CarbonCure ( 将捕获的二氧化碳注入混凝土，将其从大气中永久清除)； Climeworks ，正在建设除碳工厂；【Vesta项目，该项目致力于“绿沙”等项目，以清除海滩上的碳；以及 Charm Industrial (将废弃生物质转化为生物油)。可以说，这些都是“地球发射”，并不是完全成熟的技术，如果可行的话，运营成本可能太高，但就像月球发射一样，甚至需要大量资本来看看它们如何以及是否能够起飞。

随着时间的推移，很可能会有更多的努力被添加到列表中，也可能会有一些被减去。

目前，使用Stripe Climate的公司没有机会选择如何投资他们的贡献:他们基本上是模仿和遵循Stripe自己的道路。

Stripe Climate是免费使用的，Stripe表示，在封闭测试中测试该服务的25家公司——包括Flexport、Substack、Flipcause和open snow——已经为此投入了数十万美元。

“我们建立Substack是因为，虽然人们很容易对媒体业务的现状感到沮丧，但我们认为对于那些足够大胆的人来说，有巨大的机会保持乐观。Substack的联合创始人兼首席执行官克里斯·贝斯特(Chris Best)在一份声明中说。“我们已经摆脱了陷入萧条的局面。我们希望帮助展示通向更美好未来的道路——更好的是，我们希望给所有Substack作家加入我们的机会。Stripe的气候倡议是一份礼物，因为它消除了积极行动的所有障碍。这个项目使得做正确的事情变得简单而有价值。我们很自豪能成为其中的一员。”

条纹气候正在公司的一些重要主题上发挥作用。

Stripe现在的估值为360亿美元，主要通过简单的支付服务而闻名，网站和应用程序开发者可以通过API使用几行代码进行集成。这帮助该公司快速发展，并获得了大量用户，从个体经营者到更大的企业。

该公司在Stripe Climate上使用了同样的低摩擦原则:这个想法是，虽然公司和个人在理论上可能会致力于对环境事业进行投资，但许多人不知道从哪里开始，或者如何以有效的方式进行投资。这为他们提供了这种方式，使其成为现有支付流程的一部分。

“现在很多社会问题都是集体行动的问题，”Stripe的气候主管南·兰索霍夫(Nan Ransohoff)在接受采访时说。“气候变化是一个集体行动问题。协调可能是复杂和昂贵的。那么，我们能否轻松地将条纹业务整合在一起，使整体大于部分之和？如果我们能做到哪怕一点点，我们作为一个星球，我们将会在一个更好的地方。”

第二个主题是它如何适应Stripe在更具战略性的层面上构建的内容。基本支付可能是该公司的面包和黄油，但除此之外，它还为企业增加了许多其他服务，从帮助他们整合美国业务的工具，到欺诈预防和分析，以及基于他们在平台上现有活动的预付款和信用。另一周，它还进行了有史以来最大的收购，收购了尼日利亚一家名为Paystack的初创公司，以更全面地进入非洲等新的地理区域。

这个想法不仅仅是通过增值服务从他们的客户那里赚更多的钱，而是增加与客户的粘性，如果数据也被集成到他们业务的许多其他部分以及他们的运营方式中，客户可能会更愿意放弃一个简单的API。

Stripe Climate不会为Stripe或其客户赚到任何钱——事实上，这是客户送钱的一种方式——但这是一种非常强烈的善意姿态，可以在一定程度上建立客户的忠诚度和关注度。

Ransohoff表示，该计划还将把Stripe Climate扩展为一种工具，这些公司也可以反过来在结账时向他们自己的客户提供这种工具——这与这些天你在许多网站上点击“立即购买”时可能已经看到的许多为慈善事业捐款的提议没有什么不同。
