Wise 再融资 1200 万美元，加倍投资嵌入式商业银行业务

原文：https://web.archive.org/web/https://techcrunch.com/2020/10/29/wise-raises-another-12-million-to-double-down-on-embedded-business-banking/

金融科技初创公司 Wise 已经筹集了 1200 万美元的 A 轮融资。该公司提供商业银行账户，并采用了有趣的上市策略。明智的合作伙伴与其他公司，使他们能够提供银行账户给自己的客户。

例如，如果你正在运营一个市场或电子商务平台，将公司与个人客户相匹配，你可以利用 Wise 为你的合作公司提供银行账户。 RemoteTeam 正在使用 Wise 改善远程团队的薪资体验。

领投今天这轮融资的是 e.ventures，Grishin Robotics 也参与了。种子投资者 Base10 Partners 和 Techstars 也再次投资。

Wise 不是一家典型的银行即服务公司，因为它不想为新银行提供动力，也不想帮助它们起步。相反，这家初创公司的目标是其他涉及金融服务但无法提供这些服务的公司，因为这是一项如此大的投资。

将 Wise 集成到您的产品中并不需要大量的开发或监管工作。您不必开发整个银行用户界面，因为您只需将您的客户重定向到 Wise 即可。这家金融科技初创公司还处理了解你的客户和了解你的业务(KYC 和 KYB)流程。

当你的客户有他们自己的智能账户时，它可以让他们做你期望从商业银行账户中得到的所有基本事情。你可以持有现金，用银行转账、借记卡、虚拟卡或支票支付，用卡支付、ACH 和支票支付。

在幕后，BBVA 提供银行服务，这意味着你的存款是联邦存款保险公司投保高达 250，000 美元。该公司还将 Stripe 用于一些功能和其他基础设施公司。

Wise 的联合创始人兼首席执行官 Arjun Thyagarajan 将这些合作伙伴描述为积木。例如，该公司可以交换这些合作伙伴，并与其他 API 集成，以便在新的国家推出。

有趣的是，如果你选择向你的合作伙伴提供明智的银行账户，你将分享存款和交换费的收入。

下一步，该公司计划扩展到其他国家，如加拿大。它还将试图解决特定的垂直市场，如远程医疗市场和一般的医疗保健初创公司。这可能需要为不同类型的客户添加不同的功能。

Wise 还在与一些知名公司谈判一些合作伙伴关系，这应该会给该平台带来新的客户。
