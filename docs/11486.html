沃伦获得 140 万美元，帮助当地云基础设施提供商与亚马逊和其他巨头竞争

原文：https://web.archive.org/web/https://techcrunch.com/2020/11/01/warren-gets-1-4-million-to-help-local-cloud-infrastructure-providers-compete-against-the-giants/

作为创始人的副业项目， Warren 现在正在帮助地区云基础设施服务提供商与亚马逊、微软、IBM、谷歌和其他科技巨头竞争。沃伦的自助式分布式云平台位于爱沙尼亚塔林，在东南亚(全球增长最快的云服务市场之一)和欧洲越来越受欢迎。它最近完成了由激情资本(Passion Capital)牵头的 140 万美元的种子轮投资，计划在南美扩张，最近在巴西启动了业务。

沃伦的种子资金还包括柠檬水站和天使投资者的参与，如前诺基亚副总统保罗·梅林和 Marek Kiisa，Superangel 和 NordicNinja 基金的联合创始人。

领先的全球云提供商正在通过在世界各地发展营销团队和数据中心来积极扩展其国际业务(例如，过去几个月，微软在澳大利亚推出了新的数据中心区域，在巴西进行了扩展，并宣布将在台湾建立一个新的区域，与亚马逊网络服务展开竞争)。

但是，对定制服务和数据控制的需求仍然促使许多公司，特别是小公司，选择当地的云基础设施提供商，沃伦联合创始人兼首席执行官 Tarmo Tael 告诉 TechCrunch。

他说:“当地供应商更加注重用当地语言向所有客户提供个人销售和支持，更重要的是，花时间关注中小企业客户，以提供灵活性并满足他们的定制需求。”。“而全球供应商可能只为企业领域的少数大客户提供个人服务。”许多本地提供商还提供更低的价格，并免费提供大量带宽，吸引中小企业。

他补充说，“数据主权方面在为许多客户选择云平台时发挥着重要作用。”

2015 年，Tael 和联合创始人 Henry Vaaderpass 开始从事最终成为 Warren 的项目，同时经营一家电子商务网站的开发机构。他说，从一开始，两人就想开发自己的产品，并测试了几个想法，但对其中任何一个都不感兴趣。与此同时，随着业务的增长，该机构的电子商务客户遇到了挑战。

Tael 和 Vaaderpass 的客户倾向于选择本地云基础设施提供商，因为他们的成本更低，支持更个性化。但是，建立具有可扩展基础设施的新电子商务项目成本高昂，因为许多本地云基础设施提供商使用不同的平台。

“所以我们开始寻找工具来更好、更有效地管理我们的电子商务项目，”Tael 说。“由于我们没有找到我们想要的东西，我们将此视为构建自己产品的机会。”

在创建了他们的第一个原型之后，Tael 和 Vaaderpass 意识到它可以被其他开发团队使用，并决定从 Kiisa 这样的投资者那里寻求天使投资，他们有与云数据中心或基础设施提供商合作的经验。

东南亚是全球增长最快的云市场之一，也是沃伦业务的重要组成部分。沃伦将继续在东南亚扩张，同时专注于其他拥有巨大国内市场的发展中地区，如南美(从巴西开始)。Tael 说，这家初创公司也在与其他市场的潜在合作伙伴进行讨论，包括俄罗斯、土耳其和中国。

沃伦目前的客户包括爱沙尼亚云提供商 Pilw.io 和印度尼西亚云提供商 IdCloudHost。Tael 表示，与 Warren 合作意味着其客户花在处理基础设施软件相关技术问题上的时间更少，因此他们的团队，包括开发人员，可以专注于支持客户和管理他们销售的其他服务。

该公司的目标是为本地云基础设施提供商提供满足不断增长的需求的能力，并最终通过处理更多安装和最终用户的工具进行国际扩张。其中包括自动化维护和 DevOps 流程等功能，这些功能可以简化功能测试和处理不同的平台。

最终，Warren 希望连接网络中的提供商，最终用户可以通过单一 API 和用户界面进行访问。它还设想该网络是一个社区，沃伦的客户可以在这里共享资源，并最终为他们的应用程序和服务提供一个市场。

在竞争方面，Tael 表示，本地云基础设施提供商经常转向 OpenStack、Virtuozzo、Stratoscale 或 Mirantis。这些公司目前比沃伦拥有的优势是更广泛的网络，但沃伦正忙于建立自己的网络。到 2021 年第一季度，该公司将能够将几个地点连接到一个提供商。Tael 说，在那之后，它将“逐步将提供商彼此连接起来，升级我们的用户管理和计费服务，以处理所有这些复杂性。”
