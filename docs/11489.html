AWS 推出了带有 8 个 Nvidia A100 张量核心 GPU 的下一代 GPU 实例

原文：https://web.archive.org/web/http://techcrunch.com/2020/11/02/aws-launches-its-next-gen-gpu-instances/

AWS 发布其下一代 GPU 实例

AWS 今天宣布推出其最新的配备 GPU 的实例。这些被称为 P4d 的新实例是在 AWS 推出其第一套集群 GPU 实例十年后推出的。这款新一代处理器由英特尔 Cascade Lake 处理器和八个 Nvidia 的 A100 张量核心 GPU 提供支持。AWS 承诺，这些实例提供的深度学习性能是上一代的 2.5 倍，而训练一个可比的模型应该比这些新实例便宜约 60%。

图片来源: AWS

目前，只有一种尺寸可用，在 AWS 俚语中是 p4d.24xlarge 实例，八个 A100 GPUs 通过 Nvidia 的 NVLink 通信接口连接，并为该公司的 GPUDirect 接口提供支持。

320 GB 高带宽 GPU 内存，400 Gbps 联网，这显然是一台非常强大的机器。再加上 96 个 CPU 内核、1.1 TB 系统内存和 8tb SSD 存储，按需价格为每小时 32.77 美元也就不足为奇了(尽管一年保留实例的价格降至每小时不到 20 美元，三年保留实例的价格为 11.57 美元)。

图片来源: AWS

在极端情况下，你可以将 4000 个或更多的 GPU 组合成一个 EC2 UltraCluster(AWS 称之为 EC2 ultra cluster ),在本质上是一台超级计算机规模的机器上实现高性能计算工作负载。鉴于价格，你不太可能在短期内建立这些集群来训练你的玩具应用程序模型，但 AWS 已经与许多企业客户合作测试这些实例和集群，包括丰田研究所、GE 医疗保健和怡安。

“在[丰田研究所]，我们正在努力建设一个人人都可以自由移动的未来，”TRI 基础设施工程技术主管 Mike Garrison 说。“上一代 P3 实例帮助我们将训练机器学习模型的时间从几天减少到几小时，我们期待利用 P4d 实例，因为额外的 GPU 内存和更有效的浮点格式将允许我们的机器学习团队以更快的速度训练更复杂的模型。”
