AWS推出了带有8个Nvidia A100张量核心GPU的下一代GPU实例

原文：https://web.archive.org/web/http://techcrunch.com/2020/11/02/aws-launches-its-next-gen-gpu-instances/

AWS发布其下一代GPU实例

AWS今天宣布推出其最新的配备GPU的实例。这些被称为P4d的新实例是在AWS推出其第一套集群GPU实例十年后推出的。这款新一代处理器由英特尔Cascade Lake处理器和八个Nvidia的A100张量核心GPU提供支持。AWS承诺，这些实例提供的深度学习性能是上一代的2.5倍，而训练一个可比的模型应该比这些新实例便宜约60%。

图片来源: AWS

目前，只有一种尺寸可用，在AWS俚语中是p4d.24xlarge实例，八个A100 GPUs通过Nvidia的 NVLink通信接口连接，并为该公司的 GPUDirect接口提供支持。

320 GB高带宽GPU内存，400 Gbps联网，这显然是一台非常强大的机器。再加上96个CPU内核、1.1 TB系统内存和8tb SSD存储，按需价格为每小时32.77美元也就不足为奇了(尽管一年保留实例的价格降至每小时不到20美元，三年保留实例的价格为11.57美元)。

图片来源: AWS

在极端情况下，你可以将4000个或更多的GPU组合成一个EC2 UltraCluster(AWS称之为EC2 ultra cluster ),在本质上是一台超级计算机规模的机器上实现高性能计算工作负载。鉴于价格，你不太可能在短期内建立这些集群来训练你的玩具应用程序模型，但AWS已经与许多企业客户合作测试这些实例和集群，包括丰田研究所、GE医疗保健和怡安。

“在[丰田研究所]，我们正在努力建设一个人人都可以自由移动的未来，”TRI基础设施工程技术主管Mike Garrison说。“上一代P3实例帮助我们将训练机器学习模型的时间从几天减少到几小时，我们期待利用P4d实例，因为额外的GPU内存和更有效的浮点格式将允许我们的机器学习团队以更快的速度训练更复杂的模型。”
