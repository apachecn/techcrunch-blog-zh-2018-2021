在印度，沃尔玛的PhonePe超越Google Pay，合众国际社的月度交易额超过2B TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/11/02/walmarts-phonepe-zips-past-google-pay-in-india-as-upi-tops-2-billion-monthly-transactions/

UPI是由印度最大的银行建立的四年前的支付基础设施，上个月超过20亿笔交易，正好是在达到10亿笔月交易里程碑的一年后。

推动合众国际社交易的是沃尔玛的PhonePe、Google Pay、Paytm和亚马逊Pay。合众国际社已成为印度最受欢迎的数字支付方式，这得益于其开放的架构，允许所有参与支付应用之间的互操作性。

“与中国不同，我们对国内和国外的大小公司都给予了平等的机会，”NPCI首席执行官陈爱龙·阿斯贝在早些时候的一次采访中表示，该公司是UPI背后的支付机构。10月份的20.7亿UPI交易处理了519亿美元的价值，高于2019年10月的257亿美元。阿斯贝此前曾表示，NPCI正计划扩大合众国际社以外的印度。

一年多来，Google Pay首次未能推动合众国际社的最大交易量。据知情人士透露，PhonePe在10月份记录了8.35亿笔UPI交易，而Google Pay的交易量约为8.2亿笔。

知情人士称，Paytm记录了约2.45亿笔交易，而亚马逊支付结算了约1.25亿笔交易。

在一份声明中，PhonePe确认它占据了“市场领先地位”,在合众国际社拥有超过40%的市场份额。谷歌没有立即回应置评请求。

Paytm的发言人告诉TechCrunch，“Paytm向其客户提供一系列资金来源:Paytm UPI、Paytm Wallet、Paytm Payments银行账户、借记卡、信用卡和网上银行。总体而言，10月份通过这些资金来源进行的交易增长了20%以上。个人对商家(P2M)占总交易额的70%以上。”

TechCrunch无法确定这些支付公司在中国积累了多少独立的月度交易用户。今年5月，Google Pay拥有约7500万交易用户，超过PhonePe的6000万和Paytm的3000万。

与Google Pay不同，Paytm和PhonePe都提供钱包服务。钱包服务不是由合众国际社提供的。PhonePe表示，上个月它总共处理了9.25亿笔交易(UPI + wallet)，每月活跃用户超过1亿。

PhonePe最近的交易量激增，因为更多的线下商店开业，商家和消费者选择数字替代方式来完成交易。该应用程序还增加了一系列融资服务，包括60万份保险单。PhonePe还受益于Flipkart的节日特卖和梦幻体育应用Dream 11的增长。

“我们的使命是让数字支付成为每个印度公民的生活方式，我们的下一个目标是到2022年12月注册用户突破5亿。符合我们的品牌精神。PhonePe的首席执行官兼创始人Sameer Nigam在一份声明中表示:“我们将继续为印度社会各阶层推出新的创新产品，并让印度每个村庄和城镇的每个商户都能接受数字支付。”。

随着市场的增长，美国一些顶级支付公司之间也出现了分歧。谷歌在9月份临时撤下了印度最有价值的应用Paytm 的应用，原因是其多次违反Play Store指南。Paytm声称，谷歌的支付应用参与了一系列类似的做法，并自那以后推出了自己的商店，并与印度其他顶级初创公司结成非正式联盟，以减少对安卓制造商的依赖。

行业高管还声称，与印度其他流行的支付应用程序一样，Google Pay向进行某些交易的用户提供返现，使用UPI支付进行此类支付——他们表示，此举有助于谷歌增加其通过应用程序处理的UPI交易量。据瑞士瑞信银行称，印度的移动支付市场预计到2023年将达到1万亿美元。

但这些并不是这些支付公司今天面临的所有问题。至少那些在合众国际社上的人也在努力从中赚钱。去年年底在班加罗尔举行的一次活动上，Google Pay和Next Billion User Initiatives的董事总经理兼业务负责人萨吉斯·西瓦南丹(Sajith Sivanandan)表示，目前的地方法规迫使Google Pay在没有明确商业模式的情况下在印度运营。

随着越来越多的玩家加入竞争，事情将变得更加艰难。在印度拥有超过4亿用户的WhatsApp于2018年开始在其应用程序上测试UPI支付。然而，它仍然被困在监管的迷宫中，这使得它无法向中国的大多数用户推出WhatsApp Pay。
