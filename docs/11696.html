银行即服务 Railsbank 筹集 3700 万美元增长资金 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/11/12/railsbank-growth/

总部位于伦敦的银行即服务平台 Railsbank 已经筹集了 3700 万美元的新增长资金。

领投这轮融资的是 MiddleGame Ventures 和 Ventura Capital，它们都是 Railsbank 的现有投资者。参与的还有 Anthos Capital、Global Brain、Clocktower Technology Ventures、Moneta VC、Mitsui Fudosan 和 Firestartr。

Railsbank 的联合创始人兼首席执行官 Nigel Verdon 告诉我，注入的资本将用于继续扩大金融科技的全球足迹和进一步的产品开发。这将包括在美国推出“信用卡即服务”，并在 APAC(包括菲律宾、印尼、马来西亚、澳洲和日本)扩展其产品。它还将在英国/欧洲等现有市场上加倍下注。

Verdon 也不排除进一步的 M&A 活动，包括继收购英国 Wirecard 之后的其他战略收购

当被问及收购 Wirecard 的好处是什么时，Railsbank 的创始人说，这有助于维持英国和欧洲的有序市场，并有助于保护金融科技行业的声誉。Verdon 表示，最直接的是，它允许数百万持卡人继续使用他们的卡，“客户可以在业务中断最小的情况下继续营业”。

他还表示，此次收购为 Wirecard 和 Railsbank 带来了“才华横溢、经验丰富的人才”,并最终为公司增加了“巨大的股权价值”。

Railsbank 将自己定位为一家“公用事业公司”，其他公司——从金融科技新贵、挑战者品牌(challenger brands)到希望重新整合其技术的现有银行——可以在此基础上建立和销售各种金融服务，或为自己的产品添加金融科技功能。

当该公司关闭其首轮融资时，Verdon 将其比作亚马逊通过 AWS 为数据中心所做的事情。“Railsbank 是一个完整的金融服务后端的公用事业:平台，连接，运营，计划成员资格(如签证)，监管和合规，”他当时告诉我。最近，考虑到最近的金融科技趋势，Railsbank 也在宣传自己是嵌入式金融合作伙伴，这并不令人意外。

Railsbank 的 API 是客户“构建他们能够想象的几乎任何金融用例”的基础。Verdon 说:“用例也是多样化的，前三名是贷款、银行和储蓄相关的，嵌入到金融科技、零售、电信、保险和其他客户旅程中。

为此，Railsbank 的信用卡即服务产品意味着任何公司都可以利用金融科技的基础设施和技术提供品牌信用卡。Verdon 解释说:“在不到 12 周的时间内，我们就可以提供客户品牌的信用卡以及无缝嵌入到客户现有用户体验中的用户之旅。

“我们的使命是对复杂、不透明和错综复杂的信用卡市场进行彻底改造、拆分和民主化，这个市场已有 70 年的历史，仅在美国就价值 4 万亿美元”。

与此同时，Railsbank 现在有 200 多名员工，他们会说 40 多种语言。它在 11 个地方设有办事处:圣莫尼卡(美国)、新加坡、维尔纽斯(立陶宛)、慕尼黑(德国)、纽卡斯尔(英国)、伦敦、马尼拉(菲律宾)、吉隆坡(马来西亚)、墨尔本(澳大利亚)、越南和斯里兰卡。
