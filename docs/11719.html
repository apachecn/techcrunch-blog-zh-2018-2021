2021年，5项关键创新将电动滑板车带入5亿次骑行

Bird

四年前，共享电动滑板车还不存在。如今，它们有望在2021年前在全球范围内超过5亿次，远远超过优步在2009年创立的碳排放严重的打车行业的早期增长。

无论从哪方面来看，这都是城市交通的一个巨大转变，它引发了一个简单但重要的问题:我们是如何走到这一步的？

了解过去几年推动微移动性发展的关键进展，不仅可以让我们深入了解行业的发展方向，还可以让我们了解如何成功塑造行业，以满足全球数亿当前和未来骑行者的需求。

从车辆设计和数据到安全报告和基础设施，这五个创新时刻帮助推动了共享电动滑板车的全球增长，并帮助引领城市走向更健康、更可持续的未来。

#1:推出共享滑板车(2017年秋季)

第一支Bird电动滑板车车队于2017年9月在加利福尼亚州圣莫尼卡启动。到目前为止，微型汽车行业几乎完全由有坞和无坞自行车共享系统组成，这些系统平均每年在美国各地行驶大约3500万次——其中超过一半仅在纽约市。

在一个令人鼓舞的开始之后，美国的共享电动滑板车骑手在2018年进行了近3900万次旅行，第二年又进行了8600万次旅行。大西洋彼岸也出现了类似的轨迹，像意大利、英国、乌克兰这样的国家加入了迅速扩大的国家行列，包括德国、法国、以色列、西班牙、葡萄牙、比利时、丹麦、波兰和其他选择用现代微型交通工具补充城市交通网络的国家。

现在，几乎在世界各大洲的200多个城市都可以找到共享滑板车。

#2:第一款定制设计的共享滑板车发布(2018年秋季)

第一个电动滑板车项目很快教会了我们两件事:对这种微型汽车的需求很高，定制设计的汽车是成功满足这种需求的必要条件。

事实是，与私人拥有的踏板车相比，共享的踏板车被骑得更频繁，处理更多样的路面，忍受更多样的天气条件。这就是为什么波导的车辆团队在2018年10月推出了业内首款定制设计的电动滑板车 Bird Zero 。这款车配备了更长的电池寿命、更好的照明、更强的耐用性和更先进的GPS技术，是一系列旨在提高安全性、可持续性和寿命的全面车辆进化中的第一款，并且取得了成功。成千上万的这种滑板车今天仍在使用，每个月的持续服务进一步降低了它们已经很低的每英里寿命期碳排放量。

随后的定制车辆设计，包括Bird One和Bird Two，都在此基础上有所增加，引入了行业首创的功能，例如:

  • 车载诊断传感器能够检测200多种故障。
  • 车辆智能系统能够每天运行和报告数百万次自主故障检查。
  • 电池的IP67或IP68防水等级。
  • 14，000英里(22，500公里)的电池续航时间，平均每天使用超过10年。
  • 机械设计经过独立测试，可承受超过 60，000次路边撞击。

#3:发布全面的行业安全报告(2019年春季)

从一开始，安全就是共享微移动性最重要的焦点，也是讨论最多的方面。这就是为什么波导在2018年1月发起了业内最早和最全面的免费头盔给所有骑手活动，以及一系列其他安全举措。

2019年4月，这些计划最终形成了一份全面的电动滑板车安全报告。这是对现代微移动系统的第一次深入研究，使用事故报告和其他数据来证明共享滑板车具有类似于自行车的风险和漏洞。该报告为运营商和城市采取合作安全措施奠定了基础，以确保不仅是乘客和行人，而且所有的道路使用者都得到保护。

在过去的一年半时间里，我们利用报告中包含的发现，以及自以来呼应其发现的其他发现，想象并开发了一系列产品创新，帮助建立整个行业的电动滑板车安全标准。其中包括:

  • Shared micromobility首款头盔自拍功能推广头盔使用。
  • Shared micromobility的第一个预热模式功能帮助新骑手。
  • 第一个也是最精确的地理围栏，用于为电动滑板车创建减速区和禁行区。
  • 负责任的数据共享标准和实践，帮助城市为自行车和踏板车建立新的基础设施。

#4:开放式移动基金会成立(2019年夏季)

上面的最后一点尤为重要。城市在限制道路上的汽车数量和最大限度地增加自行车和踏板车的基础设施数量方面发挥着至关重要的作用。这是一个行之有效的提高所有道路使用者安全性的策略，它在很大程度上依赖于一个关键的输入:可靠的、标准化的数据。

自我们首次推出以来，Bird一直是与城市进行负责任的数据共享的大力支持者。然而，缺少的是一个统一的机构来帮助指导和制定整个微移动行业的移动数据标准。

所有这一切都在2019年6月发生了变化，洛杉矶、纽约和旧金山等城市与伯德和微软等公司以及一个名为OASIS的非营利组织联盟一起成立了开放移动基金会 (OMF)。正如LADOT Seleta Reynolds 的主席兼总经理在福布斯中所写的，OMF平台“帮助我们实现重要的城市目标，如增加安全、公平和健康成果，同时降低排放，减少拥堵。”

这些使用开源代码和共享数据标准来管理微移动系统的合作努力可能看起来不靠谱，但它们已经产生了一些非常切实的现实效果。在亚特兰大，共享的电动滑板车数据已经被用来在2021年前将该市受保护的自行车道翻两番。圣莫尼卡最近使用踏板车数据起草并通过了一项修正案，将增加19英里的独立微型移动基础设施。

#5:英国，纽约电动滑板车项目获得批准(2020年春季)

今年英国和纽约州决定将共享电动滑板车合法化，并启动各自的试点项目，这可能不是一个创新，但这是一个关键的发展，将确保该行业在2021年突破5亿次。

从环境和城市流动性的角度来看，伦敦和纽约是世界上最重要的两个城市。加起来，它们是1700万人的家园，每天有超过1000万次汽车出行。将电动滑板车引入这两个人口密集、流动性高的城市将对日常通勤习惯产生巨大影响，尤其是在公共交通乘客量仍因新冠肺炎而受到影响的时候。这对城市、市民和环境来说都是好消息。

从如此大规模的微型交通工具中获得的数据不仅有助于改善纽约和伦敦的基础设施。它将被添加到越来越多的研究中，这些研究正在迅速影响微移动技术，并加速其在世界各地的采用。

展望未来

那么我们能从这些中学到什么呢？共享电动滑板车行业的前四年和5亿次骑行将告诉我们关于微型汽车的未来的什么？

首先，我们应该期待中国经济继续增长。即使在2020年冠状病毒在全球传播之前，对解决汽车拥堵和城市污染的适应性强、环保的解决方案也有很高的需求。现在他们证明了自己是不可或缺的。寻找城市和运营商之间的关系，以加强和变得更加合作，因为滑板车从一个感知的休闲车转变为城市交通网络的重要组成部分。这将包括对骑自行车和骑摩托车的人的受保护的基础设施进行戏剧性的、基于数据的改进。

其次，我们应该预见电动滑板车技术将继续围绕两个关键支柱发展:安全性和可持续性。这适用于车辆本身的形式和功能，也适用于管理它们的日常操作。更长的使用寿命、更好的电池性能、更高的耐用性和更强的诊断能力将是我们衡量这一进步的基准。

最后，我们应该预见到，随着每年数亿次乘车数据的不断积累，我们对城市交通需求的理解将变得更加清晰和细致入微。城市规划决策将能够根据街道和特定时段的需求做出，识别潜在的危险区域，并采取低成本、高影响的行动进行补救。

如果目前的趋势继续下去，而且有充分的理由相信这种趋势会继续下去，那么全球新增5亿电动滑板车用户所需的时间将很快从四年缩短到不到一年。
