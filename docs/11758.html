谷歌用更多的 COVID 信息更新地图，并最终推出其辅助驾驶模式

谷歌今天宣布了谷歌地图的更新，包括许多与 COVID 相关的新功能，以及查看您的外卖或送货订单的实时状态的能力，以及推出期待已久的新助理驾驶模式。

此外，该公司今天分享了一些关于谷歌地图的新数据。例如，该公司表示，现在每天对地图进行 5000 万次更新，尽管这包括用户评论、照片和评级等用户生成的内容。该公司现在还提供全球 2000 万个地方的“流行时报”信息。

图片来源:谷歌地图

就 COVID 而言，这里有两个公告。首先，谷歌正在用一些新信息更新 Android 和 iOS 上的谷歌地图中的 COVID 层，包括一个地区有史以来检测到的病例数和地方政府的 COVID 资源链接。其次，谷歌地图现在可以实时告诉你某条公交线路有多繁忙，这样你就可以避开拥挤的火车或公共汽车。这是基于谷歌地图用户的实时反馈，如果你知道谷歌地图已经可以告诉你某个商店或餐馆目前有多忙，你会觉得很熟悉。

图片来源:谷歌地图

半相关——毕竟，在疫情期间，送货服务正在蓬勃发展(尽管他们继续努力在盈利)——手机上的谷歌地图现在可以向你显示你在美国、加拿大、德国、澳大利亚、巴西和印度的外卖和送货订单的实时送货状态。要做到这一点，你必须在 Android 或 iOS 上从谷歌地图预订订单。

对于没有 Android 自动兼容汽车的谷歌地图用户来说，地图中新的谷歌助手驾驶模式早就值得期待了。该公司第一次在 2019 年 5 月的 I/O 开发者大会上谈到了这套新功能，但正如经常发生的情况一样，在 I/O 上宣布的功能需要一段时间才能上市。最初，这应该在去年夏天推出。

图片来源:谷歌地图

这里的想法是让司机获得来电提醒，让助手读出短信，并在谷歌地图内控制你的音乐。使用助手可以理想地减少司机分心。不过，就目前而言，这种新模式只面向美国的 Android 用户，它支持的功能数量仍然有限。谷歌承诺随着时间的推移将支持更多功能，但尚不清楚它计划在这一模式中添加哪些功能。
