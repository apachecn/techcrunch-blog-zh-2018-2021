微软发布 Pluton，一款内置于英特尔、AMD 和高通处理器的定制安全芯片

原文：https://web.archive.org/web/https://techcrunch.com/2020/11/17/microsoft-pluton-security-chip-intel-amd-qualcomm/

微软发布 Pluton，一款内置于英特尔、AMD 和高通处理器的定制安全芯片

在过去的两年里，一些世界上最大的芯片制造商一直在与一系列硬件缺陷作斗争，如 Meltdown 和 Spectre ，这些缺陷使得直接从处理器中窃取密码和其他敏感机密成为可能——尽管并不容易。芯片制造商推出了补丁，但是要求这些公司重新思考他们如何处理芯片安全问题。

现在，微软认为它的新安全芯片有了答案，它称之为 Pluton。今天宣布的这款芯片是微软与芯片制造商英特尔、AMD 和高通合作的成果。

Pluton 充当硬件信任根，简单来说，它保护设备的硬件免受篡改，例如硬件植入或黑客利用设备低级固件中的缺陷。两家公司表示，通过将芯片集成到未来的英特尔、AMD 和高通中央处理器(CPU)中，黑客通过物理访问计算机来发动硬件攻击和提取敏感数据的难度将大大增加。

微软企业和操作系统安全主管大卫·韦斯顿(David Weston)表示:“微软 Pluton 的设计将在 CPU 上创建硬件和 Windows 操作系统之间更紧密的集成，这将减少可用的攻击面。

微软表示，Pluton 早在 2013 年就首次出现在 Xbox One 中，以使黑客入侵主机或允许游戏玩家运行盗版游戏变得更加困难。该芯片后来升级到微软的云服务 Azure Sphere，用于保护低成本的物联网设备。

现在的想法是将同样的技术，经过一些改进，带到新的 Windows 10 设备上。

这种芯片带来了立竿见影的效果，比如让针对 Windows 设备的硬件攻击更加难以得逞。但是这种芯片也通过保持设备的固件更新解决了一个主要的安全问题。

至于富豪芯片能否经受住时间的考验，则是另一回事。大多数芯片漏洞研究都是由第三方研究人员通过大量且通常繁琐的工作完成的。微软的韦斯顿表示，Pluton 芯片已经通过了内部 red 团队和外部供应商的安全压力测试。但如果出了问题，这可能会反过来困扰该公司。例证:就在上个月，安全研究人员在苹果的 T2 安全芯片中发现了一个“不可修复”的安全漏洞——这是一种在大多数现代 MAC 电脑中定制的芯片，类似于微软的 Pluton——这可能使 MAC 电脑面临该芯片本应防止的安全威胁。

微软拒绝透露是否计划向其他芯片制造商提供 Pluton 芯片设计，或者是否计划将设计开源供任何人使用，但表示计划在未来分享更多细节，为这种可能性敞开大门。
