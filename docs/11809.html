IBM 正在收购 APM 初创公司 Instana，继续扩展混合云愿景 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/11/18/ibm-is-acquiring-apm-startup-instana-as-it-continues-to-expand-hybrid-cloud-vision/

IBM 正在收购 APM 初创公司 Instana，因为它在继续扩展混合云愿景

随着 IBM 从软件和服务转型为一家完全专注于混合云管理的公司，它宣布有意收购应用性能管理初创公司 Instana ，该公司采用的云原生方法完全符合这一战略。

两家公司没有透露收购价格。

有了 Instana，IBM 可以在其内部管理工具的基础上进行构建，为它提供了一种监控运行 Kubernetes 的容器化环境的方法。它希望通过加入这家初创公司，可以为客户提供一种管理复杂的混合和云计算环境的方法。

云计算和数据平台高级副总裁罗布·托马斯在一份声明中表示:“我们的客户今天面临着管理复杂的技术环境，其中充满了运行在各种混合云环境中的任务关键型应用程序和数据——从公共云、私有云到内部云。”。他相信 Instana 将有助于减轻这一负担，同时使用机器学习来提供更深入的见解。

在该公司 2018 年3000 万美元 C 轮之时，TechCrunch 的 Frederic Lardinois 这样描述该公司。“真正让 Instana 脱颖而出的是它能够自动发现和监控构成现代应用程序的不断变化的基础架构，尤其是在运行容器化微服务时。”这似乎正是 IBM 寻求的解决方案。

至于 Instana，创始人认为两家公司非常合适，特别是鉴于 2018 年对红帽的收购是 IBM 混合方法的核心。“Instana 的下一代 APM 和 Observability 平台与 IBM 的混合云和人工智能技术的结合，从 IBM 与我们联系并结合我们的技术的那天起，就让我兴奋不已，”首席执行官 Mirko Novakovic 在宣布这笔交易的博客文章中写道。

事实上，在最近的一次采访中，IBM 首席执行官 Arvind Krishna 告诉美国消费者新闻与商业频道的 Jon Fortt，他们将赌注压在以 Red Hat 为中心的混合云管理上。当你将这一点与剥离该公司的托管基础设施服务业务的决定相结合时，这一收购表明他们打算追求每一个角度

“收购 Red Hat 为我们提供了技术基础，我们可以在此基础上构建基于开源的混合云技术平台，并在我们的客户踏上这一旅程时为他们提供选择。收购的成功为我们提供了动力，然后我们可以采取下一步，也是更大的一步，推出托管基础架构服务。所以公司的其他人可以完全专注于混合云和人工智能，”克里希纳告诉美国消费者新闻与商业频道。

Instana 总部位于芝加哥，在慕尼黑设有办事处，成立于 2015 年 Kubernetes 的早期，这家初创公司的 APM 解决方案已经发展到更加关注云原生环境中的监控需求。该公司在此过程中筹集了 5700 万美元，最近一轮是 2018 年的 C 轮融资。

这笔交易通常需要获得监管机构的批准，但该公司认为应该会在未来几个月内完成。
