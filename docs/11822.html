Zoom Apps 会成为下一个热门的创业平台吗？

原文：https://web.archive.org/web/https://techcrunch.com/2020/11/18/will-zoom-apps-be-the-next-hot-startup-platform/

当 Zoom 上个月宣布 Zapps 时——这个名字已经被明智地改成了 Zoom Apps——VC Twitter 立即开始猜测 Zoom 可能会从成功的视频会议服务跃升为创业创新的发射台。这当然引起了前 TechCrunch 作家、现 SignalFire 投资人 Josh Constine的注意，他在推特上写道“Zoom 的新‘Zapps’应用平台将会粉碎或创造许多创业公司。”

随着 Zoom 在疫情期间的使用激增，它成为商业和教育的关键工具，使用视频会议平台来构建一组相邻工具的想法非常有意义。虽然疫情即将结束，但我们已经对远程工作有了足够的了解，在我们安全返回学校和办公室后，对 Zoom 等工具的需求仍将长期存在。

我们已经看到了像 Mmhmm 、 Docket 和 ClassEdu 这样有前途的初创公司，这些公司正在吸引投资者的注意力。事实上，一些投资者认为 Zoom 可能会成为下一个伟大的创业生态系统。

超越视频会议

十多年前，Salesforce 为 Zoom 铺平了道路，当时它向开发者开放了平台，后来推出了 AppExchange 作为分发渠道。两者在当时都是革命性的想法。今天我们看到 Zoom 在此基础上发展。

Maven Ventures 的创始管理合伙人、Zoom 的早期投资者吉姆·谢曼(Jim Scheinman)表示，他一直认为这项服务有可能成为一个平台。“在任何人意识到之前，我已经公开说过，Zoom 是下一个伟大的开放平台，可以在其上建立数十亿美元的业务，”Scheinman 告诉我。

他说，几年前，也就是 IPO 之前，他和 Zoom leadership 讨论过向外部开发者开放平台的问题。这在当时并不是一个优先考虑的问题，但是新冠肺炎将这个想法推到了前沿。“上市后和 COVID，随着 Zoom 在企业和消费者两方面的大规模增长，很明显，应用市场现在是 Zoom 的一个关键增长领域，这为新生的创业公司创造了巨大的发展机会，”他说。

emergency Capital(Zoom 和 Salesforce 的另一位早期投资者)的创始人兼董事总经理杰森·格林(Jason Green)表示同意:“Zoom 认为，向核心 Zoom 平台添加功能，使其对特定用例更具功能性，是建立合作伙伴生态系统的一个机会，类似于 Salesforce 过去对 AppExchange 所做的那样。”

搭建平台

在一个平台能够获得开发者的支持之前，它需要足够多的用户，Zoom 显然已经跨过了这个门槛。它还需要一套开发人员工具来连接平台上的各种服务。那么庞大的用户群就成了创业公司的现成市场。最后，它需要一种在市场上分发这些作品的方法。

Zoom 一直致力于开发人员组件，并引入了行业资深人士罗斯·梅菲尔德(Ross Mayfield)来运行开发人员项目，他在职业生涯中曾参与过两家合作创业公司。他说，Zoom Apps 开发工具集的设计非常灵活，允许开发人员以他们想要的方式构建应用程序。

首先，Zoom 创建了网络视图，这是一种将功能嵌入 Zoom 这样的应用程序的方式。为了在 Zoom 中构建网络视图，该公司创建了一个 JS 工具包，它与现有的 Zoom APIs 相结合，使开发人员能够在 Zoom 体验中构建功能。他说:“因此，我们在使用 WebViews 创建体验方面给予开发者很大的灵活性，并且使用我们现有平台中非常丰富的 API 集，并创建一些新的 API 来创建体验。”
