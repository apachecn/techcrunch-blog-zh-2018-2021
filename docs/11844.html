微软为其 Edge 浏览器带来新的购物工具

原文：https://web.archive.org/web/https://techcrunch.com/2020/11/19/microsoft-brings-new-shopping-tools-to-its-edge-browser/

微软今天宣布其 Edge 浏览器的一些更新，都是关于购物的。除了扩展该团队上个月宣布的价格比较功能，Edge 现在还可以自动为你找到优惠券。此外，该公司在其必应搜索引擎中推出了一个新的购物中心。这里的时机无疑是由假日购物季驱动的——尽管今年感觉像黑色星期五式的交易几周前就已经开始了。

图片来源:微软

价格比较工具的潜在用途是显而易见的。我发现这在 Edge Collections 中总是工作得相当好——尽管有时这也可能是一种令人沮丧的体验，因为它不会从一些网站上保存的项目中提取任何数据。现在，这个价格比较一直在后台运行，您会看到一个新的徽章在 URL 栏中弹出，让您打开价格比较。当你已经找到了最好的价格，它也会马上告诉你。

至少在 Edge Canary，这已经有一点点了，这也是偶然的。例如，当你在亚马逊购物时，只要一件商品只有一个 SKU，它似乎就能很好地工作。如果有不同的颜色、尺寸或其他选择，它似乎真的不起作用，这有点令人沮丧。

图片来源:微软

优惠券功能也有点令人失望。它的工作更加一致，似乎可以从大多数标准优惠券网站(比如 RetailMeNot 和 Slickdeals)获取数据，但它所做的只是显示整个网站的优惠券。由于大多数优惠券只适用于有限的商品，点击优惠券徽章很快就感觉像是浪费时间。公平地说，该团队实现了一个漂亮的功能，在结账时，Bing 会尝试应用它找到的所有优惠券。这可能会节省时间和金钱。鉴于与 Bing 团队在其他领域的密切合作，这似乎是一个需要改进的领域。我关机了。

微软也利用今天的公告在 Edge 中发布了一个新的网址缩写器。“现在，当您粘贴从地址栏复制的链接时，它会自动从一个长而无意义的 URL 地址转换为一个带有网站标题的短超链接。如果你喜欢完整的网址，你可以使用上下文菜单转换成纯文本。我想这在某些情况下是有道理的。不过，大多数时候，我只是想要链接(中间没有第三方)，所以我希望这个也可以很容易地关闭。
