腾讯音乐娱乐支持浪潮将虚拟娱乐带到中国

原文：https://web.archive.org/web/https://techcrunch.com/2020/11/19/tencent-music-entertainment-is-backing-wave-to-bring-virtual-entertainment-to-china/

已经为约翰·传奇和威肯制作活动的虚拟音乐会平台 Wave 已经签署了一项协议，通过与腾讯音乐娱乐集团的合作将其虚拟音乐会带到中国。

两家公司在一份声明中表示，这笔交易包括腾讯持有音乐服务的少数股权。

根据合作伙伴关系，TME 将能够在其所有平台上播放 Wave 在中国的体验，包括 QQ 音乐、酷狗音乐、酷我音乐和 WeSing。

对 Wave 来说，这是其长征中的又一步，为艺术家为粉丝创造现场活动开辟了一条新途径，并为希望接触中国观众的美国艺术家打开了一个巨大的市场。

这两家公司正在合作将 Wave 的虚拟活动整合到腾讯的 TME 直播节目中。

“我们要做的第一件事就是让我们合作的西方艺术家在中国巡回演出，”Wave 首席执行官亚当·阿里戈(Adam Arrigo)说。 我们的目标是为艺术家提供超级服务，并为他们提供一个进行全球虚拟巡演的渠道

Arrigo 不仅看到了利用腾讯音乐平台的机会，还看到了将该公司的虚拟娱乐环境扩展到腾讯广受欢迎的游戏生态系统中的机会。

对于艺术家来说，替代收入来源还有额外的好处，比如在美国还没有真正站稳脚跟的即时小费、礼物和商品销售。

“这些模式植根于那些亚洲直播观众。这是一个接触西方受众和这些模式的巨大机会，而这些模式需要更长时间才能站稳脚跟，”阿里戈说。

一段时间以来，腾讯一直是中国游戏和娱乐生态系统的主导者，包括其大型多人在线游戏、即时通讯服务、微信和支付服务。

“与 Wave 的合作标志着我们在整合技术和音乐方面迈出了重要的一步，旨在增强用户的沉浸式音乐体验，提高用户参与度，促进内容消费，”TME 内容合作集团副总裁 TC Pan 在一份声明中表示。“通过这一战略合作伙伴关系，我们将通过虚拟表演进一步扩展音乐服务的边界，并建立更广泛的音乐生态系统。”
