FoodBoss 增加了对餐厅直接送货的支持

原文：https://web.archive.org/web/https://techcrunch.com/2020/11/20/foodboss-restaurant-direct/

餐馆搜索引擎 FoodBoss 增加了对餐馆直接送货的支持

FoodBoss 的目标是成为类似在线订餐的 Kayak 在这里你可以搜索不同的服务和应用程序，找到最低的价格和最快的送货时间。

然而，一个限制是，这项服务仅限于第三方服务，如 Uber Eats 和 Postmates，无法从餐厅本身点餐——直到最近，推出了一个名为 Restaurant Direct 的新功能。

FoodBoss 联合创始人兼首席执行官迈克尔·迪贝内代托(Michael DiBenedetto)表示，餐馆越来越重视直接接受送货和取货订单，这既是为了节省支付给第三方服务的费用，也是为了与客户建立直接关系。

“主要问题是他们花了所有这些钱来建设(订购)基础设施，但他们不一定知道他们必须花营销费用来吸引消费者到他们的网站或应用程序，”迪贝内代托说。“这正是我们真正需要帮助的地方。”

图片来源: FoodBoss

餐厅直销可能会带来一些额外的技术障碍，因为它需要 FoodBoss 与各种订购系统相集成。DiBenedetto 表示，该公司在某些情况下将通过 API 进行连接，也可以直接与餐厅 IT 部门合作。

他强调，FoodBoss 将对您如何点餐保持不可知论——目标只是向您展示所有选项，并强调最符合您优先事项的点餐方法。

“在 FoodBoss，我们专注于确保我们正在帮助第三方和[餐馆]降低整体营销成本，”迪贝内代托继续说道。“每个人都想在交货时获利。”

通过 Restaurant Direct 提供的第一家餐厅是芝加哥的 Lou Malnati's，计划明年在多个市场增加 Sbarro。在一份声明中，Lou Malnati 的总裁 Heather Stege 说:“餐馆面临的挑战是能够通过用户喜欢的渠道为顾客服务，同时仍然为他们提供优质的食物。FoodBoss 通过提供多种选择(包括我们自己的选择)来吸引顾客，从而帮助简化这一过程。”
