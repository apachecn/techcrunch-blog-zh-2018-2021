美国东部时间周二下午2点/太平洋标准时间上午11点，请参加Sapphire's Jai Das的现场问答

当然，我们正在进入美国的假日周末，但这并不意味着TechCrunch这艘好船会减速。下周我们将继续关注《T2》第二季Extra Crunch Live 的另一部分，这是我们对初创公司创始人、风险投资家和其他科技界领袖的定期采访系列。

这个系列是为额外的Crunch成员准备的，所以如果你还没有注册，你可以在这里跳上那列火车。

下周，我几乎要和Jai Das坐在一起，Jai Das是Sapphire Ventures的著名董事总经理。

Das在Sapphire期间投资了MuleSoft ( 以65亿美元出售)、Alteryx ( 现已上市)、Square ( 也已上市)、Sumo Logic ( yep、public )等公司，此前曾在英特尔投资和安捷伦风险投资公司从事企业风险投资工作。(Sapphire本身最初是SAP的企业风险投资部门，但它于2011年从母公司分离出来，更名并继续筹集资金。)

这里是贝塞麦的拜伦·迪特在《额外危机现场》最后一集的笔记。

会很有趣的，因为有太多的话题要谈。我的问题列表还在冒泡，所以为了避免给Sapphire公关团队太多的讨论前弹药，我们只能说，企业风险投资在2020年繁荣中的地位对创始人和投资者来说都是一个有趣的话题。

我还想让Das谈谈当前的软件创业市场，我们正处于SaaS倍数的历史弧上，以API为主导的科技新贵的重要性，创始人可能会寻求建立下一个伟大的企业创业公司，以及是否有任何新的平台涌现出来，可以成为未来创始人后来利用的基础。

由于这是一个额外的Crunch Live，我还将回答观众的一些问题(这意味着你，确保你的额外Crunch订阅是live )，以增加我自己的笔记剪贴板。

这将是一个很好的机会。下周二节目见。
