Rockstar 程序员:瑞弗斯·柯摩在编码中找到意义 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/11/25/rock-star-programmer-rivers-cuomo-finds-meaning-in-coding/

“你好，我是乐队 Weezer 的 Rivers ，”瑞弗斯·柯摩微微一笑，挥了挥手。在开始他最好的电视广告宣传之前，他把视线从摄像机前移开了一会儿。“想象你在巡回演出，你坐在你的更衣室或你的旅游巴士里。你在后台。你怯场，你压力太大了。你在来回踱步。最重要的是，你的旅游经理会不停地给你打电话，问你一些后勤方面的问题。”

就互联网营销视频而言，这还不是最普遍的。如果有什么不同的话，这个三分钟的短片到最后失去了任何民粹主义吸引力的希望。在最后一个镜头中，穿着栗色 SpaceX 连帽衫的歌手是最后一个登上私人飞机的人。飞机的门关上了，露出了威泽飞行的“W”标志。

“现在就在 GitHub 上下载 Drivetimes，”科莫在画外音中补充道。“这是 CS50X。”

这不是最完美的应用程序推介视频，库莫的电梯推介可能需要在接触风险投资家之前进行一些改进。然而，就在线编程课程的最终项目而言，这是值得一看的。这些图像在代码页、谷歌电子表格和 POV 镜头之间交替，他走上舞台与 Pixies 合作巡回演出。

这帮助科莫在班上获得了 95 分。

但是，尽管在目前的配置下，Drivetimes 巡演计划工具的吸引力可能有限，但这位音乐家在哈佛后续课程中的最后一个项目 CS50W，对于关注他超过 25 年职业生涯的大批粉丝来说，是显而易见的。

本周，科莫放弃了 2400 多个演示，总计超过 86 个小时。从 1976 年到 2015 年，这些歌曲的质量从磁带录制的草图到更精致的食物不等。有些最终会出现在威泽的 13 张专辑中，或者各种各样的副业。其他人就没这么幸运了。

通过科莫的“里弗斯先生的邻居”网站，这些歌曲被分成九束，每束售价 9 美元。顺便说一句，“库莫在免责声明的底部写道，”这个市场是我在网络编程课程中的最后一个项目。"

五年来，这位白金销量的摇滚明星一直在兼职学习计算机编程。

“我一直是个电子表格迷，”科莫告诉 TechCrunch。“2000 年左右，我想我从微软 Access 开始，然后是 Excel。只是记录下我所有的歌曲、小样和想法。电子表格变得越来越复杂，就像是，‘好吧，我差不多是在用这些公式写代码，除了它超级难用。所以也许我应该去做编程，”"

对于几乎任何其他成功的音乐家来说，这都是一种奇怪的旁门左道。然而，对科莫来说，这是合乎逻辑的下一步。威泽的同名处女作大获成功后，他进入哈佛大学读大二，在宿舍住了一年。他最终离开学校录制乐队备受喜爱的后续作品平克顿，但 1997 年和 2004 年的两次招生发现这位音乐家最终在 2006 年获得了英语学士学位。

CS50 发现科莫回到了哈佛——至少在精神上。该课程由该大学在线主办，是计算机科学的免费入门课程。

“我浏览了一些在线课程，寻找一些看起来有吸引力的东西，所以我看到哈佛 CS50 非常受欢迎，”科莫说。“所以我想，‘好吧，我来试试。’这并没有马上见效。第一周的课程使用 Scratch。我不知道你是否知道，但这就像是一种点击和拖动类型的编程，你正在制作一个小的视频游戏。"

这位音乐家为期六周的课程持续了六个月。同年，这位音乐家——现在是两个孩子的父亲——参加了几十场演出，并录制了威泽的第十张专辑，格莱美提名的白色专辑。

“当我们在课程中途接触 Python 时，”Cuomo 说，“我对它的强大和直观感到惊讶，我可以完成这么多工作。课程结束时，我编写的程序真正帮助我管理了作为一名旅行音乐家的日常生活，还管理了我的电子表格，管理了我作为一名创意艺术家的工作。”

对库莫来说，生产率从来都不是什么大问题。除了今年的黑色专辑之外，该乐队已经完成了两张专辑，他已经开始着手另外两张专辑的后续工作。然而，看起来更大的问题是如何组织这些想法。这就是电子表格和数据库的用武之地。

成千上万的电子表格变成了一个数据库，将库莫自己的演示和他从其他艺术家那里学习的作品分类。

“多年来，这似乎是一种浪费时间或放纵，”他说。“我应该写一首新歌或录制一首歌，而不是仅仅罗列这些旧想法，但我发现，多年后，我能够非常有效地利用这些古老的想法，因为我可以告诉我的 Python 程序，“嘿，给我看看我在一个降半音的基调中以 126 BPM 开始的所有想法，这些想法以三度音阶和旋律开始，并处于多里安模式，并且我的经理给了三颗星或更多星。”"

他承认，这一过程可能缺乏一些摇滚浪漫主义，而这正是乐队粉丝所希望的。但是，尽管绘制了大量的分析，科莫坚持认为仍然有魔力存在。

对库莫来说，生产率从来都不是什么大问题。然而，鉴于他的生产力水平，组织所有这些想法可能会变得棘手。这就是电子表格和数据库的用武之地。

“在我们传统上认为的人类创造力中，仍然有很多自发性和灵感的空间，”科莫解释道。“我心目中的英雄之一是伊戈尔·斯特拉文斯基。有一本他的演讲集叫做《音乐诗学》。"他收藏了一张纸条。他说，他对只使用他的一种能力的作曲家没有兴趣，就像一个作曲家说，“我只会写我在某种创作领域时脑海中自发出现的东西。”。我不会使用任何其他工具，”

“他说，‘不，我更喜欢听这样一位作曲家的音乐，他在运用他所掌握的所有能力，他的直觉，还有他的智力，以及他分析、分类和利用他所拥有的一切的能力。’我发现那些最终成为最狂野、最不可预测和最有创造力的作品。"

而且从来不缺作文。库莫说，除了今年的黑人专辑，乐队还有两张专辑已经完成，他已经开始着手另外两张专辑的后续工作。几十年来，这位歌手一直对 18 个月的大型唱片发行周期心存感激，他表示，在 Demos 项目之后，他对寻找更多直接向粉丝发布音乐的方式产生了新的兴趣。

“我不觉得自己真的擅长理解宏观市场，以及如何对世界产生最大影响，”他说。“我的经理非常擅长这个，但我只是告诉他们，‘嘿，这感觉就像这里的东西。首先，真的很好玩。粉丝们真的很开心。对所有相关人员来说都非常简单。编码部分并不容易，但对其他人来说，只需点击几下，你就可以得到所有这些音乐，而且价格便宜，没有中间人。PayPal 拿了一点，但跟大厂牌没什么两样。所以，这可能是什么。有一些东西，当它直接从我这里传到观众面前时，感觉真好。"

目前，计算机科学继续占据他的大部分时间。Cuomo 估计他已经在编程项目上花费了大约 70%的工作时间。周三晚上，他会帮忙为一个冥想网站编程(这是另一个长达十年的爱好)，他还计划参加哈佛大学的后续课程 cs5000000，该课程主要是开发移动应用程序。

然而，目前还没有计划辞去他的日常工作。

“我看不出我能在一家初创公司找到工作，或者维护某人的网站，”他说。“但也许摇滚明星和网络开发者之间的界限正在变得模糊，因此音乐家将越来越多地使用技术工具。除了音乐软件之外，我们将越来越多地利用发行、组织和创造力的手段，这种方式体现在我们对与观众的联系进行编码的过程中。”
