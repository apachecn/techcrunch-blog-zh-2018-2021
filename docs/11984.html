AWS宣布推出Panorama，这是一款为任何相机添加机器学习技术的设备

原文：https://web.archive.org/web/https://techcrunch.com/2020/12/01/aws-announces-panorama-a-device-adds-machine-learning-technology-to-any-camera/

AWS宣布推出Panorama，这是一款为任何相机添加机器学习技术的设备

AWS已经推出了一种新的硬件设备，即 AWS Panorama Appliance ，它将与 AWS Panorama SDK 一起，将现有的现场摄像机转变为支持计算机视觉的超级监控设备。

将硬件宣传为客户检查生产线上的零件、确保安全协议得到遵守或分析零售店流量的新方式，新的自动化服务是本次AWS re:Invent活动主题的一部分——自动化一切。

除了公司可以使用亚马逊SageMaker 开发的计算机视觉模型，新的Panorama设备可以在联网或联网摄像头的视频馈送上运行这些模型。

很快，AWS希望拥有Panorama SDK，设备制造商可以使用它来构建支持Panorama的设备。

亚马逊以前向开发者和企业推销过监控技术。早在2017年，该公司推出了DeepLens 、，并于一年后开始销售。这是开发人员建立原型机器学习模型的一种方式，也是亚马逊适应计算机视觉功能商业化的不同方式的一种方式。

正如我们在2018年写的:

DeepLens与AWS的其他服务深度集成。这些服务包括AWS物联网服务Greengrass，例如，您可以使用它将模型部署到DeepLens，还有SageMaker，亚马逊用于构建机器学习模型的最新工具……事实上，如果您只想运行AWS提供的预构建样本之一，设置……deep lens并将这些模型之一部署到相机应该不会花费您超过10分钟的时间。这些项目模板包括一个可以区分20个对象的对象检测模型(尽管它在玩具狗方面有一些问题，正如您在上面的图像中看到的那样)，一个以梵高风格渲染相机图像的风格转换示例，一个面部检测模型和一个可以区分猫和狗的模型，以及一个可以识别大约30种不同动作(例如弹吉他)的模型。DeepLens团队还增加了一个跟踪头部姿势的模型。哦，还有一个热狗检测模型。

 

在为视频开发机器学习技术方面，亚马逊有很多经验(和争议)。该公司的Rekognition软件引发了抗议和抵制，导致该技术的暂停使用。

该公司还试图将更多的机器学习功能融入其面向消费者的环形相机中。

尽管如此，企业仍然继续呼吁新的机器学习支持的视频识别技术，以实现安全、安全和质量控制。事实上，随着新冠肺炎疫情的拖延，围绕建筑使用和占用的新协议正在被采用，不仅是为了适应当前的疫情，也是为了提前规划空间和协议，以帮助减轻下一次疫情的严重性。
