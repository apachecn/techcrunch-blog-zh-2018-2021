AWS 推出了 Glue Elastic 视图，使数据从一个专门构建的数据存储转移到另一个更容易

AWS 已经推出了 Glue Elastic Views，这是一个让开发者将数据从一个商店转移到另一个商店的新工具。

在 AWS re:Invent keynote 上，首席执行官安迪·杰西(Andy Jassy)宣布了 Glue Elastic Views，这是一项让程序员更无缝地在多个数据存储之间移动数据的服务。

这项新服务可以从不同的数据仓库中提取数据，并将其复制到目标数据存储中。AWS ETL 服务允许程序员编写一点 SQL 代码来获得可以从一个源数据存储移动到另一个源数据存储的物化视图。

例如，Jassy 说，程序员可以将数据从 DynamoDB 转移到 Elastic Search，允许开发人员设置物化视图来复制数据——同时管理依赖关系。Jassy 说，这意味着如果源数据湖中的数据发生变化，那么它将在数据被重新定位的其他数据存储中自动更新。

Jassy 说:“当你有能力移动数据……并轻松地将数据从一个数据存储移动到另一个数据存储时……这是非常强大的。
