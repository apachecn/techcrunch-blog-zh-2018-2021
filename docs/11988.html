AWS 推出了新的定制 ML 训练芯片 Trainium

原文：https://web.archive.org/web/https://techcrunch.com/2020/12/01/aws-launches-trainium-its-new-custom-ml-training-chip/amp/

在其年度 re:Invent 开发者大会上，AWS 今天宣布推出 AWS Trainium ，这是该公司的下一代定制芯片，专门用于训练机器学习模型。该公司承诺，凭借对 TensorFlow、PyTorch 和 MXNet 的支持，它可以在云领域提供比任何竞争对手都更高的性能。

它将作为 EC2 实例和亚马逊 SageMaker(该公司的机器学习平台)内提供。

基于这些定制芯片的新实例将于明年推出。

这些定制芯片的主要论点是速度和成本。与标准 AWS GPU 实例相比，AWS 承诺吞吐量提高 30%,每次推理成本降低 45%。

此外，AWS 正在与英特尔合作，推出基于 Habana Gaudi 的 EC2 实例，用于机器学习培训。明年，这些实例有望提供比当前基于 GPU 的机器学习 EC2 实例高出 40%的性价比。这些芯片将支持 TensorFlow 和 PyTorch。

这些新芯片将于 2021 年上半年在 AWS 云中首次亮相。

这两种新产品都是对 AWS 推理系统的补充，该公司在去年的 re:Invent 上推出了该系统。推理是这些机器学习部分的推理对应部分，它也使用一个定制的芯片。

值得注意的是，Trainium 将使用与推理相同的 SDK。

AWS 团队写道:“虽然推理解决了推理的成本，这构成了高达 90%的 ML 基础设施成本，但许多开发团队也受到固定 ML 培训预算的限制。”“这限制了改进模型和应用所需的培训范围和频率。AWS Trainium 通过在云中为 ML 培训提供最高的性能和最低的成本来应对这一挑战。通过 Trainium 和 Inferentia，客户将拥有从扩展训练工作负载到部署加速推理的端到端 ML 计算流程。”
