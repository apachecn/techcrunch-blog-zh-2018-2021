Nordigen 推出免费的欧洲开放银行 API

拉脱维亚金融科技初创公司 Nordigen 正转向免费增值模式，这得益于一个免费的开放银行 API。开放银行本应使银行信息的访问民主化，但该公司认为 Tink 或 Plaid 的银行聚合 API 太贵了。相反，Nordigen 认为它可以提供免费的 API 来访问账户信息，并为分析和见解服务提供付费服务。

开放银行是一个宽泛的术语，有不同的含义，从账户聚合到验证账户所有权和支付启动。开放银行最基本的一层是能够查看来自第三方金融机构的数据。例如，一些银行允许你连接到其他银行账户，这样你就可以从一个界面查看你所有的银行账户。

有两种方法连接到银行。有些银行提供应用程序编程接口(API)，这意味着您可以向银行的服务器发送请求，并接收数据作为回报。

尽管由于欧洲 PSD2 指令，所有金融机构都应该有一个开放的 API，但许多银行仍在拖延。这就是为什么开放银行 API 公司通常依赖屏幕抓取。它们模仿网络浏览器的交互，这意味着它很慢，需要大量的服务器资源，而且可能会崩溃。

“如果你想知道我们如何负担得起，我们的免费银行数据 API 完全是按照 PSD2 设计的，这意味着它与现有产品形成了鲜明对比。因此，扩大免费用户规模不会显著增加我们的成本，”Nordigen 联合创始人兼首席执行官 Rolands Mesters 告诉我。

所以你不能得到诺迪根 API 的全部覆盖。这家初创公司目前支持 300 家欧洲银行，覆盖了每个国家 60%到 90%的人口。不过反正是免费产品就很难抱怨了。

一些 Nordigen 客户可能需要更多信息。Nordigen 提供金融数据分析。如果你是一家试图计算信用评分的贷款公司，如果你是一家有最低收入要求和更多要求的金融公司，这可能特别有用。

对于那些额外的服务，你必须付费。Nordigen 目前有 50 个客户，并希望通过其新的免费增值战略吸引更多客户。
