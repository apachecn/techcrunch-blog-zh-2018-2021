Revolut 在英国和网络应用程序中推出早期工资功能

金融科技初创公司 Revolut 本周有两个新功能。首先，该公司将推出一款面向普通用户的网络应用，而不仅仅是商业用户。第二，在英国，Revolut 与 Modulr 合作，让你提前一天拿到工资。

Revolut 历来专注于其移动应用。如果您有一个 Revolut 的商业帐户，您知道您可以看到您过去的交易，并从常规的网络浏览器访问您的帐户。但该公司的 1300 万用户无法通过电脑访问他们的账户。

每个人现在都可以前往 Revolut 的网络应用并登录查看他们的交易历史和卡。从这个界面，您可以冻结和解冻借记卡和控制卡功能。该网络应用还支持使用银行转账、信用卡支付或 Apple Pay(在 Safari 中)进行账户充值。

默认情况下，Revolut 会发送推送通知，以便您可以授权 web 浏览器访问。但如果你丢失了手机，你也可以选择通过电子邮件接收安全码。

你仍然需要使用移动应用程序来访问一些功能，但这是一个开始。

对于居住在英国的用户来说，Revolut 正在加倍加强与 Modulr 的合作关系，以便提前发送你的工资。通过 Bacs 支付计划支付的工资将比平时提前一天到达——在英国，大多数人都是使用这种方法支付的。这都是关于优化支付基础设施的，在长假周末之前，这可能会特别有帮助。

这也应该直接有利于 Revolut，因为许多用户除了使用普通银行账户外，还一直在使用 Revolut。增加让你更容易放弃银行账户的功能，可能会提高该公司的使用数量。这可能有助于该公司增加其卡交换费、订阅收入和其他收入来源。
