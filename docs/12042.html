PhonePe筹集了7亿美元，成为一个独立的实体

原文：https://web.archive.org/web/https://techcrunch.com/2020/12/02/phonepe-raises-700-million-becomes-a-separate-entity/

总部位于班加罗尔的金融服务公司周四表示，PhonePe是沃尔玛收购Flipkart的皇冠上的明珠，正在“部分”剥离。为了开始新的旅程，该公司表示已在新一轮融资中获得7亿美元。

PhonePe表示，这轮融资由沃尔玛牵头，一些现有投资者参与，融资方的名称没有透露。由三名前Flipkart员工创立的PhonePe在新一轮融资中的估值为55亿美元。

今天的声明大大增强了投资者对PhonePe的信心。TechCrunch早些时候报道称，这家初创公司已经与投资者接触了几个季度，至少有一次，在今年早些时候艰难地以30亿美元的估值筹集资金。然而，据知情人士透露，该公司曾希望去年以高达100亿美元的估值融资约10亿美元。

部分分拆已经进行了一年多，这意味着Flipkart在PhonePe的股份将从100%降至87%。PhonePe的创始人兼首席执行官Sameer Nigam在一份声明中表示:“这部分分拆让PhonePe获得了专用的长期资本，以实现我们为10亿印度人提供金融普惠的愿景。”

PhonePe表示，Flipkart仍将是PhonePe的大股东，“两家公司将保持密切合作”。

PhonePe，在印地语中的意思是“在电话上”，目前在印度的移动支付市场处于领先地位。10月，它超过了Google Pay，成为UPI支付应用中的佼佼者。UPI是由印度最大的银行建立的四年前的支付基础设施。这是印度最流行的电子货币交易方式。PhonePe报告10月份UPI交易量为8.35亿笔，领先于当月处理约8.2亿笔交易的Google Pay。

“随着Flipkart Commerce在满足印度客户需求方面继续强劲增长，我们对集团的未来前景感到兴奋。Flipkart集团首席执行官Kalyan Krishnamurthy在一份声明中表示:“这一举措将有助于PhonePe在进入下一个发展阶段时最大限度地发挥其潜力，同时也将最大限度地为Flipkart和我们的股东创造价值。”。

在最近给客户的一份报告中，伯恩斯坦(Bernstein)的分析师认为PhonePe是印度可能在三年内申请IPO的初创公司之一。PhonePe“对母公司Flipkart的依赖已经缩小，从每月交易的50%下降到今天的不到1%。PhonePe首先在UPI上推出，并扩展到新的用例，包括旅行预订、账单支付、金融服务等。该公司还推出了Switch，允许用户在PhonePe和其他应用程序之间切换，如食品、杂货交付等。(超级App)，”他们写道。

分析师在TechCrunch获得的报告中写道，PhonePe已经积累了超过1亿的月活跃用户，可以考虑在印度以外扩张。

如今，PhonePe和Google Pay也在与由软银和阿里巴巴支持的Paytm竞争。Paytm去年筹集了10亿美元，私下估值为160亿美元，最近几个季度，由于UPI目前无法让公司在其平台上赚钱，pay TM已经将注意力转移到迎合商家。

去年年底在班加罗尔举行的一次活动上，Google Pay和Next Billion User Initiatives的董事总经理兼业务负责人Sajith Sivanandan表示，印度目前的地方法规迫使Google Pay在该国没有清晰的商业模式的情况下运营。

如果这还不是一个足够的挑战，WhatsApp是印度最受欢迎的智能手机应用程序，在全球第二大市场拥有超过4亿用户，上个月开始在印度推出支付服务。瑞士瑞信银行称，印度的移动支付市场岌岌可危，预计到2023年将达到1万亿美元。
