点评:来自罗技、Audio-Technica、SteelSeries、HyperX 等公司的无线耳机 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/12/04/review-wireless-headsets-from-logitech-audio-technica-steelseries-hyperx-and-more/

鉴于你这些天在家的时间，你应该得到一副更好的耳机。一个无线的，可以和你的电脑或者你的主机一起工作，有一个麦克风可以打电话，有很好的声音可以播放游戏和电影。幸运的是，有很多选择，我已经测试出你的最佳选择。

我要求领先的音频和外设公司发送他们的旗舰无线耳机，价格从 100 美元到 250 美元不等。超过这个价格范围，回报会迅速减少，但现在这是舒适，声音和可用性的最佳点。

多年来，我一直避免无线耳机，因为有太多的妥协，但我很高兴地说，延迟已经消除，我评测的电池寿命都非常好。(注意:如果无线版本感觉太贵，你经常可以花 50-100 美元买到有线版本。)

为了测试耳机，我将它们用于各种各样的日常任务，从视频通话到电影和音乐(只进行最低限度的均衡以感受它们的自然声音)，到 AAA 游戏和独立游戏。没有一个需要应用程序才能工作，尽管有些有 led 或游戏简介的配套软件。我有一个相当大的脑袋和中等大小的耳朵，不管怎样。所有的耳机都相当笨重，尽管我单独拍摄它们的角度使它们看起来很大——你可以在上面的图片中看到它们的大小大致相同。

这些耳机都没有主动降噪功能，但许多耳机都提供了不错的物理隔离，甚至提供了“监听”功能，可以接收来自外界的声音——如果你在玩游戏，但在等待烤箱预热或其他事情时，这很有用。只有第一套有一个内置麦克风，其余的都有可拆卸的麦克风，质量一般都很好，肯定足以用于流媒体和聊天，尽管对于广播来说，单独的一个会更好。所有这些耳机都使用 USB-A 型转换器，尽管 7P/7X 也有 USB-C 连接器。

7P 钢铁系列/7X–149 美元

图片来源:Devin Coldewey / TechCrunch

7P 和 7X 耳机分别以 PS5 和 Xbox Series X 为设计理念(兼容 PC、Switch 等)，是我第一个也是最毫无保留的推荐。

对我来说，最突出的特点是真正令人惊讶的声音，几乎令人不安的宽广舞台和清晰度。当我放一些我用作参考的熟悉的曲目时，我几乎不敢相信我所听到的。这不是一个 7.1 模拟或任何类似的东西——但毫无疑问，游戏的焦点导致创建一个大的声场。成功了！

我还发现耳机非常舒适，有一个“滑雪护目镜”带，而不是每个波段的调整，让他们坐得很轻，并“记住”你的设置。宽敞的耳罩可旋转，方便旅行或舒适使用。

内置麦克风并不显眼，可以很好地隐藏起来，但如果你对位置很挑剔，调整起来就有点软。许多其他耳机都有更好的麦克风，可以完全分离——也许这对你来说是一个优势，但我往往会失去它们。

我的主要问题是这些控件感觉很便宜，而且布局不是特别好。耳机的底部是一堆杂乱的端口和按钮，音量旋钮没有多少移动空间——一次完整的滑动可以从 0 到 100。(音量控制独立于系统音量。)

加密狗与其他加密狗不同，它本身是 USB-C，但附有 USB-A 电缆。这对兼容性很好，但电缆有三英尺长，连接到一些笔记本电脑等设备上有点傻。不过，你可以很容易地得到自己的短电线。

在 150 美元，我认为这是一个简单的建议，几乎任何人都在寻找这个价格范围。

GWL 音频技术公司——250 美元

图片来源:Devin Coldewey / TechCrunch

这些耳机的价格很高，部分原因是它们是无线版本的耳机，也是有线的，所以如果你想要稳定的音频性能和舒适的佩戴，你可以通过有线方式节省一些钱。

AT-GWLs 的声音丰富，自然侧重于中上音域，这使得媒体中的声音非常流行。我确实发现声音有点受限，这实际上有助于达到“环绕”设置。我知道这种虚拟化通常不被人喜欢，但是这些设置已经被夸大和扭曲了有一段时间了。我发现环绕声更适合游戏，但不一定适合音乐，但它很容易打开和关闭。

耳机很轻，用传统的、严肃的金属带调节，顶部有一个单独的衬垫。我会说它们是我测试过的感觉最轻的一对，SteelSeries 和 Razer 由于一些额外的重量和体积而紧随其后。尽管很紧凑，AT-GWLs 感觉通风，但并不大。皮革-超细纤维组合杯很好，我认为随着时间的推移，它们会很好地适应，提供更好的隔离。

他们的不足之处在于界面。首先，音频技术人员注意:关掉通知噪音！打开耳机，打开或关闭麦克风，或者按下独立于系统的最大音量，都会发出响亮而令人惊讶的蜂鸣声。太大声了！

第二，按钮和转盘很僵硬，很小，感觉一样。快速抬起一只手来调低音量(可能在一声巨大的哔哔声后)，你很容易把电源开关误认为音量旋钮。表盘还兼作环绕模式的按钮，旁边是一个微型按钮，可以打开和关闭周围的声音。这有点混乱——没有什么你不能习惯的，但考虑到这个列表中的其他耳机的控制有多好，不得不说。

HyperX Cloud II wireless——150 美元

图片来源:Devin Coldewey / TechCrunch

(更新:我的错误，我用了一个销售 100 美元的价格作为日常价格，实际上是 150 美元。在这个水平上，这些很难推荐，但仍然非常好。)

HyperX(由 Kingston 所有)直到最近才真正以音频闻名，但它之前的云耳机得到了至关重要的 Wirecutter 的认可，很容易看出为什么。与本综述中的许多其他耳机相比，这款耳机的价格更低，但它的后续产品(我现在正戴着的)具有出色的声音和隔离性能。

我惊讶地发现一个几乎和 7P/7X 一样宽的声场，但更多地关注于有力的低音域，而不是细节和位置。我的音乐感觉又大又近，游戏的气氛也一样，更直接地呈现出来。

Cloud II 的控制简单而有效。与系统音量直接相关的音量旋钮非常棒:抓握感强，动作流畅，摩擦力恰到好处，几乎没有咔哒声。有两个大小合适的按钮，电源一凹和麦克风静音(静音和活跃时发出不同的声音)凸。

不幸的是，对我来说，他们没有名单上的其他人舒服。杯子(虽然有点偏暖)和带子都很好。这是因为它们几乎没有旋转，这意味着没有间隙来适应你的头部形状。我不知道，也许只是我的大圆顶，但它们在我耳朵前面明显比后面紧，所以我不断调整或试图扭曲它们。

我要说的是:如果他们对杯子做了更多的调整，这将是我对 7P/7X 的默认推荐。尽管 SteelSeries 的声音令我兴奋，但 Cloud IIs 似乎更像人们所期待的，控制也更好。

罗技 G-733–130 美元

G733s 的哑光纹理与我的相机有一种奇怪的互动——它们看起来并不特别 IRL。图片来源:Devin Coldewey / TechCrunch

这些是罗技的流光友好，颜色协调，LED 运动套装，但它们比响亮的设计更好。

声音绝对是游戏化的，明确强调低端，声音非常集中，很像 Cloud II。

老实说，我并不指望 G733s 会非常舒适——它们坚硬的塑料外观表明它们会嘎吱作响，压住我的耳朵，压碎我的头。但事实上，它们真的很轻，很舒服！耳罩的位置有很多变化。这种配合有点奇怪，因为 7P/7X 的“滑雪护目镜”表带有一个明显的劣质版本，实际上只有四个设置，而杯子上下滑动约三分之二英寸。这是刚刚足以容纳我的(再次，显然非常大)头。

麦克风的吊杆很短，可悲的是没有指示麦克风何时开或关，这有时是一个小麻烦，有时是一个大麻烦。不过，你可以从静音按钮发出的声音判断出来。

音量调节盘很好，很平滑，虽然“咔哒”声相隔很远。我喜欢它的质地和麦克风静音按钮，电源按钮不太喜欢。但这很有效。

颜色可能不是每个人都喜欢，但我不得不把它交给罗技一路走来。耳机、麦克风甚至 USB 加密狗都是相同的颜色，这使得在我越来越多的耳机和小部件中跟踪它们变得更加容易。

罗技 Pro-X——200 美元

图片来源:Devin Coldewey / TechCrunch

目前罗技最高端的游戏耳机，Pro-X 放弃了其他耳机明亮的塑料外观，选择低调的黑色。

罗技的声音很大，非常清晰，几乎有一种乐队平衡的参考感觉。我觉得在智能低音演奏的中低部分比其他部分更有存在感。有一种“环绕”的感觉，让你感觉更像是在一个配置良好的扬声器而不是耳机的房间里，我认为这是从中央声道的削弱中出现的。媒体“在外面”，而不是“在这里”这不是坏事也不是好事，只是与众不同而已。

控件与 Cloud II 的差不多:一个控制系统音量的漂亮的摩擦音量轮，一个漂亮的麦克风切换按钮和一个相当大的开关，你不太可能故意绊倒。

同样像 Cloud IIs 一样，耳罩没有旋转，这让我觉得它们不如 ATs 和 SteelSeries 以及罗技更便宜的 G-733 舒适。如果可能的话，一个比我自己更大的脑袋肯定会感觉被夹紧了。我确实认为这些会穿得很好，但尽管如此，玩一玩会有很大帮助。

外部材料是光滑的哑光塑料，看起来非常可爱，但绝对是指纹磁铁。考虑到你会经常接触这些东西(老实说，不一定要用刚洗过的手)，你会比我测试的其他人更需要擦拭它们。

雷蛇黑鲨 V2 专业版——180 美元

图片来源:Devin Coldewey / TechCrunch

当 SteelSeries 套装连接到 PS5 时，低调的 Razer Blackshark V2 Pro 很快成为我的 PC 游戏首选。

他们的声音绝对是以游戏为中心的，在低音和中低音中有额外的魅力，但音乐听起来并没有过度向那个方向转移。摄影棚是满的，但并不令人吃惊，一切听起来都很详细而不刺耳。

雷蛇看起来很重，但事实并非如此——每天都在变化，但我认为他们肯定在与 A-Ts 和 SteelSeries 竞争“最舒适”。杯子感觉很宽敞，密封也很好，营造出一种非常隔绝的听觉体验。调整是通过连接到罩杯的电线完成的，这没什么特别的——我有点希望这个设置能让你调整斜面和高度。这种材料就像 Logitech——很容易留下指纹，不过以我的经验来看，不那么容易。

他们的控制设计和布局都很好，都在一边。突出的(独立于系统的)音量旋钮起初可能看起来很奇怪，但你很快就会爱上它。一个大的凹口或咔哒声正好表示 50%，这对于快速“校准”非常有用，旋转旋钮非常平稳，但阻力足够大，我从来没有一次不小心改变它。同时，还有便于放置和区分的静音和电源按钮，以及可拆卸麦克风、充电线和 3.5 毫米输入端口。

我很难想到黑鲨的任何缺点，除了它不能与游戏机一起工作。
