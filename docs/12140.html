网络保险初创公司 At-Bay C 轮融资 3400 万美元，M12 成为新投资者

原文：https://web.archive.org/web/https://techcrunch.com/2020/12/08/cyber-insurance-startup-at-bay-raises-34m-series-c-adds-m12-as-a-new-investor/

网络安全保险初创公司 At-Bay 周二宣布，该公司已在 C 轮融资中筹集了 3400 万美元。

这轮融资由新投资者库姆拉资本(Qumra Capital)牵头。微软的风险基金 M12 也是新的投资者，与 Acrew Capital、Khosla Ventures、Lightspeed Venture Partners、Munich Re Ventures 和以色列企业家 Shlomo Kramer 一起参与了这轮投资，他们共同创立了安全公司 Check Point 和 Imperva。

这对该公司来说是一个巨大的举措，该公司在 2 月份才关闭了 B 轮融资。

到 2025 年，网络安全保险市场预计将成为一个 230 亿美元的行业，部分原因是联网设备的爆炸式增长以及欧洲 GDPR 和最近加州全州隐私法下的新监管制度。但是，传统保险公司难以获得满足日益增长的网络安全保险需求所需的敏锐度，像 At-Bay 这样的初创公司填补了这一空间。

At-Bay 由 Rotem Iram 和 Roman Itskovich 于 2016 年创立，总部位于山景城。在过去的一年中，该公司的员工人数增加了两倍，现在在纽约、亚特兰大、芝加哥、波特兰、洛杉矶和达拉斯都设有办事处。

该公司通过监控其客户网络的边界并提醒他们安全风险或漏洞，使自己与众不同。通过主动寻找潜在的安全问题，At-Bay 帮助其客户在网络入侵和数据泄露发生之前进行防范，避免了公司的损失，同时减少了保险支出，这对保险提供商和客户来说都是双赢的。

“这种现代的风险管理方法不仅推动了对我们保险的强劲需求，还使我们能够改进产品，最大限度地减少投保人的损失，”Iram 说。

这是一个值得的赌注:该公司表示，其索赔频率不到行业平均水平的一半。M12 的合伙人 Lior Litwak 表示，他认为该公司在融合网络风险和分析与网络保险方面有“巨大的潜力”。

现在，随着 C 系列产品在银行上市，该公司计划扩大团队并推出新产品，同时改善其自动化承保平台，使公司能够获得即时的网络保险报价。
