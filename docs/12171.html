该部门筹集 1860 万美元，以服务形式提供银行功能

原文：https://web.archive.org/web/https://techcrunch.com/2020/12/08/unit-raises-18-6m-to-offer-banking-features-as-a-service/

像 Stripe 和 Twilio 这样的公司已经改变了在线业务的游戏规则，它们使得将支付和通信服务集成到客户界面变得更加容易，而不必从头开始构建这些功能，或者进行昂贵的投资来从其他地方集成这些功能。今天，另一家创业公司正在启动并宣布一些资金，希望它也能为金融功能做同样的事情。

Unit 已经建立了一个平台，让第三方通过 API 将支付卡、支票账户、预付现金和转账等银行服务整合到他们自己的业务中。在悄无声息地建立了服务(和客户群)之后，该公司今天宣布，它将开放业务，获得 1860 万美元的资金，并继续增长——通过增加更多功能，雇佣更多人员，并获得新用户。

资金来自投资者的组合，这些投资者谈到了该公司的以色列根源和目前的旧金山基地。它包括 Better Tomorrow Ventures、Aleph、fluore Ventures、Operator Partners 和 TLV Partners，以及 30 位利用金融科技经验的天使投资人。

与 Doron Somech(首席技术官)共同创立该公司的首席执行官 Itai Damti 表示，该公司的使命是让那些已经拥有客户的公司更容易与客户进行某种交易工作——例如，一家按需运输公司与其合同司机车队进行互动；或者是一个为用户提供服务的在线记账平台——通过更多的金融功能，将这种关系扩展到更广泛、更深入、更忠诚的领域。

“自由职业者经济中的公司处于有利地位，可以将更多的银行服务捆绑到他们的自由职业者平台上，”Damti 说。事实上，其中许多公司过去曾涉足提供支付卡等其他服务，以此作为支付佣金的一种方式。现在，“这意味着他们可以帮助他们的自由职业者跟踪支出，并向他们支付款项。”

他的信念是，通过让整合这些功能变得更容易，我们将看到一个名副其实的业务爆炸式增长。“我们已经看到了很多回升，”他说。

现有的大型银行相对缓慢地让自己的服务跟上科技世界的变化步伐，这为许多挑战者打开了大门，他们希望通过提供更个性化的服务、更大的灵活性和更好的利率来赢得市场份额，所有这些都是通过高效的移动应用程序，而不是在旧建筑里排队。

Unit 的赌注是，如果你能找到人们已经在工作的地方，就有更大的机会提供银行服务。据疫情全球卫生组织估计，这种推动之所以增加，仅仅是因为它让更多的人上网，他们希望通过互联网完成更多的工作。

还有其他公司已经发现了“金融服务即服务”的机会，并正在快速发展，进军该市场。

其中最引人注目的可能是 Rapyd，大约一年前，它的最后一次估值为 12 亿美元，其支持者包括 Stripe 等公司和许多其他顶级投资者。

但随着越来越多的业务上线，至少现在每个人都有机会。
