谷歌、英特尔、Zoom 和其他公司发起了一个新的联盟，让企业使用更多的 Chrome

包括谷歌、Box、思杰、戴尔、Imprivata、英特尔、Okta、RingCentral、Slack、VMware 和 Zoom 在内的一批业界重量级公司今天宣布启动现代计算联盟。

这一新联盟的使命是“推动‘硅到云’创新，造福企业客户——推动差异化的现代计算平台，并为集成业务解决方案提供更多选择。”

不管是谁写了这份使命宣言，很明显是想看看他们可以不用说什么就能用多少词。

这个联盟的真正目的是:尽管 Chrome 这个词从未出现在它的主页上，谷歌的合作伙伴也从未提及过它，但它都是为了帮助企业采用 Chrome 和 Chrome 操作系统。“联盟的重点是推动谷歌 Chrome 生态系统的创新和互操作性，增加企业客户的选择，并帮助解决公司今天面临的一些最大的技术挑战，”谷歌发言人告诉我。

我不知道为什么它不被称为 Chrome 企业联盟，但现代计算联盟可能只是有更多的戒指。这也解释了为什么微软不是它的一部分，尽管这只是最初的成员名单，其他人可能会在未来的某个时候加入。

在谷歌的领导下，该联盟的重点是将现代网络应用程序引入企业领域，重点是性能、安全性、身份管理和生产力。当然，所有这些都是为了在 Chrome 和 Chrome OS 上运行良好，并具有互操作性。

“技术行业正朝着开放、异构的生态系统发展，这种生态系统允许自由选择，同时允许跨堆栈集成。这一现实既是挑战也是机遇，”谷歌 Chrome OS 副总裁约翰·所罗门今天写道。

随着企业向云迁移，构建更好的 web 应用程序，甚至是像本地解决方案一样工作的渐进式 web 应用程序，显然是一个崇高的目标，很高兴看到这些公司一起工作。鉴于疫情，所有这些现在也有了新的紧迫性。该联盟的计划是在 2021 年上半年发布产品，尽管尚不清楚这些产品将采取什么形式。希望这些能很好地在任何浏览器上运行。许多这样的“联盟”很快就失败了，所以我们将继续关注这里发生的事情。

额外收获:这种行业联盟已经有很长的历史了。这里有一个有趣的 1991 年的故事，讲的是英特尔、IBM、MIPS 和其他公司的 CPU 联盟。
