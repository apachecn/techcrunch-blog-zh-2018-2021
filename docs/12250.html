亚马逊为 Echo 设备推出实时翻译功能

原文：https://web.archive.org/web/https://techcrunch.com/2020/12/14/amazon-launches-a-live-translation-feature-for-echo-devices/

亚马逊今天宣布了一项新的 Alexa 功能，即实时翻译，该功能将翻译说两种不同语言的人之间的对话。该功能使用亚马逊的语音识别技术和神经机器翻译技术来工作，并支持英语和法语、西班牙语、印地语、葡萄牙语(巴西)、德语或意大利语之间的翻译。

要使用实时翻译，Echo 设备所有者可以发出类似“Alexa，翻译法语”的语音命令，开始在英语和法语之间进行翻译。亚马逊声称，当你听到哔哔声时，你可以用任何一种语言说话，甚至可以在句子之间自然停顿。然后，Alexa 将自动检测正在使用的语言，并翻译对话的每一方。在 Echo Show 设备上，除了听到翻译，还可以看到翻译。

要结束翻译会话，你可以说“Alexa，停止。”

根据雅虎财经 2018 年的一份报告，该公司被披露正在开发通用语言翻译功能。

新增加的功能现在可能使 Alexa 与谷歌设备相比更具竞争力，谷歌设备可以通过谷歌助手利用谷歌现有的翻译服务。谷歌家庭设备在 2019 年初引入了通过名为解释器模式的功能将的对话翻译成多种语言的能力。今天，解释器模式在许多谷歌设备上运行，包括智能扬声器、智能显示屏、智能时钟，甚至手机和平板电脑上的谷歌助手。然而，当谷歌在其 Pixel Buds 中添加实时翻译时，功能最初失败了。

Alexa 的翻译功能将如何工作，需要在今天的发布会后进一步测试。

实时翻译是针对 Echo 设备的一系列语言更新中的最新一项。

该功能是继去年为美国用户推出的多语言模式之后推出的，例如，该模式允许 Alexa 用户说英语和西班牙语、法语和英语、印地语和英语的组合。Alexa 还可以将一个单词或短语翻译成 50 多种支持的语言。

除了帮助用户更好地交流，亚马逊表示，该功能还可以用于语言学习，以及酒店客人和员工之间通过 Alexa for Hospitality 进行交流，Alexa for Hospitality 是其为酒店业设计的平台。
