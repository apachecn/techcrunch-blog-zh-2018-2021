声称受到歧视的前员工称，Pinterest以2250万美元达成的和解突显了科技行业的不平等

原文：https://web.archive.org/web/https://techcrunch.com/2020/12/15/pinterests-22-5m-settlement-highlights-techs-inequities-say-former-employees-who-alleged-discrimination/

Ozoma告诉TechCrunch，当前Pinterest政策团队的Ifeoma Ozoma和Aerica Shimizu Banks在6月份指控Pinterest存在种族和性别歧视时，Pinterest希望让他们完整并解决其涉嫌歧视的文化。但事实并非如此。

仅仅两个月后，前Pinterest网站首席运营官·弗朗索瓦·布鲁格(Franç oise Brougher)起诉Pinterest网站，指控其性别歧视，昨天达成了2250万美元的和解。作为和解的一部分，Pinterest将向Brougher和她的律师支付2000万美元，该公司在一份文件中写道。

“这是你能想到的最明显的不同待遇和歧视的例子，”奥卓玛说。

在今天与TechCrunch的一次通话中，Ozoma和Banks描述了他们与Brougher相比的双重标准。虽然布鲁格获得了2000万美元的赔偿，但奥卓玛和班克斯获得的遣散费还不到一年。

班克斯告诉TechCrunch说:“这遵循了美国由来已久的传统，即黑人女性站出来，开辟一条道路，揭露不公正，白人女性进来并收获所有好处。”

本月早些时候，一群股东对包括首席执行官本·希伯尔曼在内的Pinterest高管提起诉讼，指控他们助长了歧视文化。诉状继续声称，歧视文化损害了Pinterest的声誉，并导致了经济损失。

然而，对于Ozoma和Banks来说，他们说他们已经用尽了所有的法律手段，不会再提起诉讼。班克斯说，重要的是要记住一个事实，即Brougher，前首席运营官，有更多的资源进行诉讼。

“所以我们，就像在很多很多其他情况下一样，黑人妇女把自己放在线上，分享发生在我们身上的所有事情，然后为其他人的突然出现和收集‘进步’奠定基础，”奥卓玛说。“这里没有取得任何进展，因为受到伤害的人没有获得任何权利。”

作为和解的一部分，Pinterest和Brougher都将承诺250万美元，用于在科技行业“提高女性和代表性不足的群体的地位”。

“Francoise欢迎Pinterest为改善工作环境采取的有意义的步骤，并对Pinterest致力于建立一种让所有员工感到被包容和被支持的文化感到鼓舞，”Pinterest和Brougher在一份详细介绍和解协议的联合声明中表示。

Ozoma反对Pinterest和Brougher向慈善机构捐赠250万美元。她说，“它闻起来很臭，”并指出她自己是一个个体，而不是一个慈善机构。

TechCrunch就Ozoma和Banks最近的声明联系了Pinterest。Pinterest拒绝置评，称公司不对法律问题发表评论。然而，6月，Pinterest在给TechCrunch的一份声明中表示:

我们认真对待这些问题，并在提出时进行了彻底的调查，我们相信两位员工都受到了公平对待。我们希望Pinterest的每一位员工都能感受到欢迎、重视和尊重。正如我们在6月2日的声明中所概述的，我们致力于通过在公司和平台上采取行动来推进我们在包容性和多样性方面的工作。作为一家公司，在我们做得不够的领域，我们必须也将会做得更好。

Brougher提起诉讼后不久，Pinterest的员工在8月举行了罢工。除了罢工，一份请愿书在整个公司传阅，要求进行系统性的变革。他们寻求的变革包括晋升级别和留任的完全透明，整体薪酬方案的透明，以及向首席执行官报告的两级人员中至少25%为女性，8%为代表性不足的员工。

从那以后，Pinterest在董事会层面做了一些改变。罢工几天后， Pinterest宣布Andrea Wishom成为公司有史以来第一位黑人董事会成员。10月，Pinterest增加了第二名黑人董事会成员，萨拉姆·科尔曼·史密斯。

Pinterest表示，它还加强了招聘和面试流程，以努力提高高层的多样性，更新了包容性培训，并推出了一个内部维基，详细介绍Pinterest如何做出薪酬决定。

Pinterest一直被认为是多样性和包容性的领导者。当被问及这是否曾经是真的时——如果Pinterest有效地制定了一个坚实的DEI战略——Ozoma很清楚。

“不。如果这是真的，我不认为我们现在会有一个对话。”

歧视，尤其是对黑人女性的歧视，在科技行业是系统性的。本月早些时候，Timnit Gebru博士表示，谷歌解雇了她，因为她在一封电子邮件中谈到了人工智能的道德问题。班克斯和奥卓玛告诉TechCrunch，他们担心会对其他站出来的黑人女性产生寒蝉效应。

班克斯说，有一个人找到她，询问其他黑人女性有什么希望。

“这就是为什么我们说了些什么，”奥卓玛说。“我们所处的位置不像高管层的人那样。但我们的诚信比什么都重要，如果我们能帮助其他人，我们会这样做。”
