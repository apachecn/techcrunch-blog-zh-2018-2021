Twitter 利用 AWS 进军公共云的最新举措

Twitter 有很多事情要做，单靠自己的力量管理这么大的规模并不容易。今天，亚马逊宣布 Twitter 已经与 AWS 签署了一项多年协议，以运行其实时时间线。这是亚马逊云部门的一次重大胜利。

虽然这两家公司已经在某种程度上合作了十多年，但这标志着 Twitter 首次利用 AWS 来帮助运行其核心时间表。

“向 AWS 的扩展标志着 Twitter 首次利用公共云来扩展其实时服务。Twitter 将依靠 AWS 的广度和深度，包括计算、容器、存储和安全方面的能力，以最低的延迟可靠地提供实时服务，同时继续开发和部署新功能，以改善人们使用 Twitter 的方式，”该公司在公告中解释道。

Twitter 首席技术官帕拉格·阿格拉瓦尔(Parag Agrawal)认为，这是利用 AWS 的数据中心网络，向用户提供更贴近用户的内容，从而扩大和改善该公司实时服务的一种方式。“与 AWS 的合作将提高 Twitter 用户的性能，使我们能够在利用 AWS Graviton2 实例基于 Arm 的架构的同时，从离客户更近的数据中心提供推文。Agrawal 在一份声明中说:“除了帮助我们扩展我们的基础设施，与 AWS 的合作还使我们能够在应用 AWS 多样化和不断增长的服务组合时更快地发布功能。”

值得注意的是，Twitter 也与谷歌云有关系。2018 年，它宣布将其 Hadoop 集群迁移到 GCP 。

这一宣布可以被认为是富人变得更富的一个例子，因为 AWS 目前是云基础设施市场的领导者，拥有大约 33%的市场份额。根据 Synergy Research ，微软以大约 18%的份额位居第二，谷歌以 9%的份额位居第三，。在其最近的财报中，亚马逊报告了 116 亿美元的 AWS 收入，使其运营成本超过 460 亿美元。
