欧洲批准Google-Fitbit十年内不得将健康数据用于广告TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/12/17/europe-clears-google-fitbit-with-a-ten-year-ban-on-using-health-data-for-ads/

欧洲批准了谷歌以21亿美元收购健身可穿戴设备制造商Fitbit的交易，提出了一系列条件，旨在减少竞争担忧，即在监管机构对该交易进行了数月的审查后，让谷歌吞噬了T2的大量健康和保健数据。

虽然谷歌在一年多前宣布了收购Fitbit的计划，但它只是在2020年6月15日向欧盟委员会通报了这笔交易，这意味着它花了半年时间才获得欧洲的批准。它现在也在自己的地盘面临正式的反垄断指控——从多个角度(尽管与Fitbit无关)。

根据欧盟批准“Gitbit”的条款，谷歌承诺在十年内不将欧洲经济区用户的Fitbit数据用于广告定位目的。

它表示，将在健康和健康数据之间保持技术隔离，这些数据是通过Fitbit可穿戴设备收集的，与谷歌的其他数据分开存放。

它还承诺确保地区用户有“有效的选择”，允许或拒绝其他谷歌服务(如谷歌搜索、谷歌地图、谷歌助手和YouTube)使用其谷歌账户或Fitbit账户中存储的健康和健康数据。但是看看有多少深色图案设计被应用在那里会很有趣。

有趣的是，委员会说它可能会决定将十年广告承诺的期限从T10年延长到T11年——如果这种延长是合理的话。

它还指出，批准的条件是完全遵守所有承诺——这些承诺的履行将由一名受托人监督，该受托人必须在交易结束前任命。

这个尚未被任命的人将拥有委员会所称的“深远的权限”——包括访问“谷歌的记录、人员、设施或技术信息”。

因此，欧盟监管机构正采取“信任但核实”的方法，让这一大型科技合并案继续进行。

此外，还有更多以竞争为重点的承诺。

在这里，谷歌已经同意通过网络API保持对Fitbit用户数据的访问，而不向第三方开发者收取访问费用(当然要征得用户同意)。

它还同意了一些与竞争对手可穿戴设备制造商获得Android APIs有关的承诺——表示将继续免费许可竞争设备插入其主导的智能手机操作系统Android所需的所有核心功能。

根据欧盟委员会的说法，设备功能的改进包含在协议中，因此它旨在允许竞争对手的可穿戴设备制造商继续创新，而没有制造更好/更强大的设备导致他们被排除在Android生态系统之外的风险。

谷歌还必须在其移动平台的Android开源项目版本中保持API支持。

在这半年的调查和谈判中，欧盟委员会从谷歌那里获得的另一项让步是，它不会试图通过降低用户体验(如显示警告或错误信息)来规避支持竞争对手通过API访问Android的套件的要求。

坦率地说，这是一个监管机构不得不发出的非常不正常的信号。这也凸显了人们对谷歌业务运营方式的不信任程度。

这反过来给欧盟监管机构提出了一个存在性的问题，即为什么他们允许自己屈服，让谷歌-Fitbit继续前进。因此，毫不奇怪，欧盟委员会的公关听起来有点防御性——欧盟立法者写道，该决定“不影响委员会确保数字领域公平和可竞争市场的努力，特别是通过最近提议的数字市场法案(DMA)。

它还指出，监督受托人将有权与谷歌的主要数据保护监管机构爱尔兰数据保护委员会分享其向委员会提供的报告。(尽管考虑到办公桌上积压的大量与技术相关的案件——包括对谷歌业务其他元素的大量调查——这不太可能导致山景城度过任何额外的不眠之夜。)

欧盟委员会也表示，从谷歌获得的承诺包括“第三方可以援引的快速纠纷解决机制”。因此，它显然试图再进一步，为谷歌已经占据主导地位的消费者数字服务领域的进一步整合开绿灯——而此时美国立法者正朝着相反的方向前进。所以，嗯…

自从谷歌收购Fitbit的消息公布以来，欧洲(以及其他地区)的公民社会就一直在大声疾呼——敦促欧盟监管机构阻止这个科技巨头吞噬Fitbit的健康数据，除非或者直到T2的人权保护得到保证。

如今，欧盟委员会回避了这些更广泛的权利问题。

它最多认为自己已经拖延了10年，最多20年。到2030年(或2040年),它希望刚刚提议的规则能够限制像谷歌这样的数字看门人，以防止未来的滥用。

欧盟经常声明的偏好是监管科技巨头，而不是瓦解他们的帝国——或者，事实证明，阻碍帝国的进一步扩张。

在一份声明中评论谷歌-Fitbit的许可时，维斯特格说:“我们可以批准谷歌对Fitbit的拟议收购，因为这些承诺将确保可穿戴设备市场和新生的数字健康领域保持开放和竞争。这些承诺将决定谷歌如何将收集的数据用于广告目的，如何保护竞争对手的可穿戴设备和安卓系统之间的互操作性，以及用户如何继续分享健康和健身数据，如果他们选择这样做的话。”

上周，在接受欧洲议会委员会的提问时，Vestager暗示了Gitbit不可避免的即将被清除——他说美国和欧洲有不同的方法来应对占据市场主导地位的科技巨头。“在欧洲，我们没有禁止垄断，”她告诉欧洲议会议员。“他们在美国有不同的法律基础。我们会说，我们非常欢迎你取得成功，但随着成功而来的是责任——这就是为什么我们有第102条(反对滥用支配地位)。”

这也是为什么委员会认为有必要提出新的法规来加强数字市场的竞争执法——尽管DMA很可能要过几年才能被采用。

与此同时，欧盟监管机构正允许谷歌通过打包Fitbit的敏感健康数据宝库来扩大其对人们私人信息的主导地位——以便以后充分利用。

无论如何，正如哈佛大学教授Shoshana Zuboff 上周所警告的，监控资本主义的商业野心现在已经远远超出了单纯的定向广告。目标是使用数据“，以便随着预测接近确定性而变得更加有利可图”，正如她所说的那样——警告社会必须干预公共利益，以阻止科技巨头的“认知政变”。

对谷歌来说，从健康数据中产生的准确预测可能确实非常有利可图(谷歌近年来一直在加大对健康领域的关注)。

这对人类是好是坏还有待观察——这不是监管者应该感到舒服的那种重大赌博。因此，很多人会说，欧盟委员会在给大型科技公司一个方便的竞争执法的旁路的同时，只是在兜圈子。
