GitHub告别cookie横幅TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2020/12/17/github-says-goodbye-to-cookie-banners/

微软旗下的GitHub今天宣布，它将取消其平台上所有不必要的cookies。由于这一点，从今天开始，GitHub.com和它的子域名将不再有cookie横幅。这样你就少了一个需要点击才能完成工作的cookie横幅。

“没有人喜欢cookie横幅，”GitHub首席执行官Nat Friedman在今天的公告中写道。"但是饼干横幅到处都是！"

当然，这是因为美国的GDPR等法规和欧盟的指令赋予用户拒绝使用减少其在线隐私的cookies的权利。结果是，尽管这些规定考虑到了用户的最大利益，但你今天还是会不断地看到大量的cookie横幅。

“在 GitHub ，我们希望保护开发者隐私，我们发现cookie横幅令人恼火，所以我们决定寻找解决方案。经过简单的搜索，我们发现了一条:不要使用任何非必要的cookies。非常简单，真的，”弗里德曼写道。
