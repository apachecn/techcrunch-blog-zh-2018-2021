新加坡开放金融初创公司 Finantier 获得 Y Combinator TechCrunch 的支持

原文：https://web.archive.org/web/https://techcrunch.com/2020/12/22/singapore-based-open-finance-startup-finantier-gets-backing-from-y-combinator/amp/

图片来源: 金融家

“银行服务不足”并不意味着某人无法获得金融服务。相反，这通常意味着他们没有传统的银行账户或信用卡。但在印度尼西亚等市场，许多人仍然使用数字钱包或电子商务平台，创造了替代的用户数据来源，可以帮助他们获得营运资金和其他金融工具。总部位于新加坡的开放金融初创公司 Finantier ，希望通过一个单一的 API 来简化这些数据，该 API 允许金融服务在用户同意的情况下访问用户数据。它还包括基于机器学习的分析，以实现信用评分和 KYC 验证。

目前有 20 多个客户处于测试模式，Finantier 正忙于准备正式推出。它今天宣布，它已经被 Y Combinator 的 2021 年冬季创业批接受。这家初创公司最近还筹集了由 East Ventures 牵头的未披露金额的种子前资金，AC Ventures、Genesia Ventures、Two Culture Capital 和其他投资者也参与了进来。

Finantier 是今年早些时候由 Diego Rojas、Keng Low 和 Edwin Kusuma 创建的，他们都有为金融科技公司开发产品的经验，其使命是在新兴市场实现开放金融。

开放金融从开放银行发展而来，格子和叮当都是建立在同样的框架上。用户可以决定授权应用程序或网站安全访问他们的在线账户信息，包括银行账户、信用卡和数字钱包，旨在让人们对他们的财务数据有更多的控制权，而不是将其孤立在银行和其他机构中。开放银行业务主要指支付账户，而 Finantier 的专长开放金融则涵盖更广泛的服务，包括商业贷款、抵押贷款和保险承保。

虽然 Finantier 首先专注于新加坡和印度尼西亚，但它计划扩展到其他国家，成为像 Plaid 一样的全球金融科技公司。它已经盯上了越南和菲律宾。

在推出 Finantier 之前，罗哈斯曾为 P2P 贷款平台 Lending Club 和 Dianrong 开发产品，并担任东南亚多家金融科技初创公司的首席技术官。他意识到，许多公司努力与其他平台集成，从银行获取数据，或者从不同的提供商那里购买数据。

“人们正在讨论开放银行、嵌入式金融等，”Finantier 首席执行官罗哈斯告诉 TechCrunch。“但这些是更大事物的基石，即开放金融。特别是在东南亚这样的地区，大约 60%到 70%的成年人没有银行账户或银行服务不足，我们相信可以帮助消费者和企业利用他们在多个平台上拥有的数据。它肯定不需要是一个银行账户，它可以是一个数字钱包、电子商务平台或其他服务提供商。”

这对消费者来说意味着，即使某人没有信用卡，他们仍然可以建立信誉:例如，通过分享电子商务平台上已完成交易的数据。零工经济工作者可以通过不同的应用程序提供关于他们日常乘车或其他类型工作的数据，从而获得更多的金融服务和交易。

建设东南亚的金融基础设施

其他专注于东南亚的开放式银行初创公司包括 Brankas 和 Brick。罗哈斯说，Finantier 的与众不同之处在于，它专注于开放金融，并为金融机构创造基础设施，为最终用户提供更多服务。

金融机构开放金融的好处是可以为更多的消费者创造产品，找到更多收入分成模式的机会。在东南亚，这也意味着惠及更多银行服务不足或缺乏金融服务的人群。

在参与 Y Combinator 的加速器计划的同时，Finantier 还将参与印度尼西亚金融服务管理局的监管沙箱。一旦完成该计划，它将能够与印度尼西亚更多的金融科技公司合作，包括更大的机构。

East Ventures 联合创始人兼管理合伙人王绍博·库卡说，印尼有 1.39 亿成年人银行服务不足或没有银行账户。

这家专注于印度尼西亚的投资公司进行了一项名为“东方创投数字竞争力指数”的年度调查，发现金融排斥是最大的鸿沟之一。雅加达所在的爪哇岛等人口稠密的岛屿与该群岛的其他岛屿之间，金融服务的数量存在巨大差距。

为了促进金融包容性并减轻新冠肺炎疫情的经济影响，政府已经制定了一个目标在今年年底前让 1000 万家微型、小型和中型企业(MSMEs)实现数字化。目前大约有 800 万印度尼西亚微小中型企业在网上销售，仅占该国微小中型企业的 13%。

“提供平等的金融服务将为印度尼西亚经济创造乘数效应，”Cuaca 告诉 TechCrunch，东方风险投资公司决定支持 Finantier。“目前，数百家公司使用自己独特的解决方案将金融服务带给更多的人。我们相信，Finantier 将帮助他们为这部分得不到充分服务的人群提供更多产品和服务。”
