StepZen 获得 800 万美元种子资金，用于构建数据集成 API

原文：https://web.archive.org/web/https://techcrunch.com/2020/12/22/stepzen-snares-8m-seed-to-build-data-integration-api/

StepZen ，一家来自《给你Apigee(2016 年以 6.25 亿美元的价格被卖给谷歌)剧组的新创业公司，对他们的最新公司有着不同的愿景。他们正在构建一个单一的 API，从不同的来源提取数据，以帮助开发人员提供更复杂的在线客户体验。

今天，这家初创公司悄然崛起，宣布获得 Neotribe Ventures 和 Wing Venture Capital 的 800 万美元种子投资。

首席执行官兼联合创始人 Anant Jhingran 表示，凭借多年的 API 工作经验，创始人希望更进一步。“StepZen 是一种产品，它让前端开发人员可以轻松地为后端所需的所有数据创建和使用一个 API，”他解释道。

这一切都是为了提供更流畅、更一致的客户体验。这意味着，无论您是在电子商务网站上访问订单历史，还是在银行应用程序上获取当前余额，这些场景都需要从各种后端数据资源中提取数据。连接到这些资源是一项耗时的任务，StepZen 希望为开发人员简化这项任务。

“开发人员花费了大量的时间来部署和管理访问后端的代码，StepZen 想做的是把这些时间还给他们，”他说。

StepZen 不需要手动编写代码来提取这些数据，而是让开发人员只需提供配置信息和凭证来连接到这些后端数据源，然后构建一个 API 来处理提取这些数据并在需要时呈现这些数据的所有繁重工作。

Jhingran 使用了为客户显示未结订单列表的示例。这听起来很简单，但一旦你考虑到数据可能存在于几个地方，包括 CRM 系统、订单系统或你的快递，这意味着访问至少三个独立的和高度不同的系统。StepZen 将通过它的 API 把这些整合在一起，并流畅地呈现给用户。

今天，该公司有 11 名员工，包括三名创始人，并计划在 2021 年再增加 8 名左右。在这样做的时候，CBO 和联合创始人海伦·惠兰说，他们正在努力建立一个多元化和包容性的公司。虽然创始团队本身是多样化的，但他们希望雇用具有不同背景和思维方式的员工，以建立最完整的产品和公司。

“对于最初的 10 名左右的员工，我们利用了与我们一起工作的人的关系网，你知道这些人可以做得很好。然后，我认为这是有意识地从那里扩大，有意识地花时间去探索和扩大你的候选人渠道，”她说。

该公司刚刚成立 9 个月，今年大部分时间都在构建解决方案并与前 alpha 用户合作。今天，该产品处于 alpha 阶段，计划在明年年初作为软件服务发布。

随着该公司从秘密行动中走出来，它希望继续开发产品，并寻找尽可能消除复杂性的方法。“我们知道如何在后端做困难的事情。我们已经掌握了数据库技术和 API 技术，现在要做的是找到如何使所有这些在外表上变得简单，并且易于开发人员使用，”Whelan 说。
