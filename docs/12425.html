AWS ，亚马逊蓬勃发展的云臂，十多年来一直在快速增长。作为早期的公共云基础架构供应商，它利用率先进入市场的优势成为该领域最成功的参与者。事实上，有人可能会说，如果没有像 AWS 这样的云公司的形成，让他们能够轻松访问基础设施，而不必自己构建基础设施，今天的许多创业公司就不会起步。

在亚马逊最近的财报中，AWS 创造了 116 亿美元的收入，运营成本超过 460 亿美元。这使得 AWS 的下一个里程碑是 500 亿美元的运营成本，如果它继续保持其收入增长速度，不到两个季度就可以达到这一目标。

对相互竞争的公司来说，好消息是，尽管市场规模和相对成熟，但仍有很大的增长空间。

而云计算部门的增长在百分比上有所放缓，因为它坚决反对大数定律，即 AWS 每个季度都必须增长，而收入基数却越来越大。这种动态的结果是，尽管 AWS 的年同比增长率随着时间的推移而放缓，从 2019 年第三季度的 35%降至 2020 年第三季度的 29%，但其增加 100 亿美元年收入的速度正在加快。

在今年的 AWS re:Invent 客户大会上，AWS 首席执行官 Andy Jassy 谈到了这些年来的变化速度，他说，它用了以下几个月的时间以 100 亿美元的增量增长其运行速度:

图片来源: TechCrunch(数据来自 AWS)

从上述趋势推断，AWS 应该用不到 12 个月的时间从 400 亿美元扩大到 500 亿美元。Jassy 说，显而易见的是，“AWS 的增长速度继续加快。”他还花时间指出，AWS 现在是世界上第五大企业 IT 公司，领先于 SAP 和 Oracle 等企业巨头。

令人惊讶的是，AWS 实现其规模的速度如此之快，甚至直到 2006 年才存在。这一增长率让我们不禁要问:有人能指望阻止 AWS 的势头吗？

简而言之，这不太可能。

云市场格局

一个很好的起点是调查云基础设施的竞争格局，看看是否有任何云公司可以赶上市场领导者。根据 Synergy Research 的, AWS 仍然稳稳地处于领先地位，而且看起来任何竞争对手都不可能很快赶上 AWS，除非某些市场动态引起了剧烈的变化。

图片来源:协同研究

凭借约三分之一的市场份额，AWS 显然是领跑者。其最接近和最激烈的竞争对手微软拥有大约 20%的市场份额。稍微客观一点来看，上个季度 AWS 的收入为 116 亿美元，而微软的 Azure 收入为 52 亿美元。虽然微软的云数量增长更快，为 47%，但像 AWS 一样，这个数字已经开始稳步下降，同时它获得了市场份额和更高的收入，并成为同一大数定律的受害者。
