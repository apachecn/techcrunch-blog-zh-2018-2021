Nuro收购自动货运初创公司Ike

原文：https://web.archive.org/web/https://techcrunch.com/2020/12/23/nuro-acquires-autonomous-trucking-startup-ike/amp/

图片来源:布莱斯·德宾

自动送货公司Nuro已经收购了Ike，这是一家由苹果、谷歌和优步先进技术集团的退伍军人创立的初创公司，旨在将自动驾驶卡车商业化。

这笔交易是自动驾驶汽车行业繁忙的收购和整合季节中的最新一起，将两家有着深厚联系和共享技术的公司结合在一起。虽然这两家公司追求自动驾驶汽车技术的不同应用——Nuro用于本地送货，Ike用于长途货运——但他们的创始人表示，在物流方面有着共同的愿景。

交易的财务条款没有披露。

Nuro是这种关系中的巨头，估值50亿美元，员工600多人。据消息人士称，艾克雇佣了约60名员工，截至去年，其融资后估值约为2.5亿美元。然而，这些公司的创始人并不认为这是典型的硅谷招聘。Ike首席执行官奥尔登伍德罗(Alden Woodrow)表示，超过55名Ike员工及其三名创始人将转到Nuro，奥尔登伍德罗与Jur van den Berg和Nancy Sun共同创立了该公司。

伍德罗说，这是实现我们创立公司的一些原则的一个明显的机会，他还于周三在Medium的一篇博客文章中宣布了这项收购。

伍德罗告诉TechCrunch，Ike仍然拥有runway，这意味着它拥有继续独立运营所需的资本。尽管如此，将一家自动驾驶汽车公司发展成商业企业需要超过5200万美元的资金池，以及合作伙伴关系。今年秋天，Ike确实与DHL、Ryder和NFI达成协议，向他们各自的车队提供技术，但这些仍处于早期阶段。

Nuro的联合创始人兼总裁戴夫·弗格森(Dave Ferguson)对TechCrunch表示:“我认为，很明显，艾克的团队是多么不可思议，他们创造的技术质量也是如此。”。“对Nuro来说，特别引人注目的是，Ike在几年前授权了Nuro的技术堆栈，他们建立的所有技术都在该堆栈之上，有一个共享的DNA。他们构建的技术也是我们可以非常容易地转移的东西，几乎可以即插即用到我们的系统中。”

Ferguson补充说，Nuro能够利用Ike建立的技术，并将其应用于自己的本地交付应用程序，以及用于潜在的未来应用程序。

那些未来的应用是什么是一个没有答案的问题。Nuro的专利申请展示了一系列的想法，从送热腾腾的披萨和拿铁到更小的机器人。随着Ike团队的加入，Nuro有可能超越本地交付，扩展到其他类型的物流应用，如中英里交付，这是初创公司Gatik AI获得关注的一个领域，或其他卡车运输用例。Ferguson很快指出，Nuro的本地送货机器人“R2”是它的第一个和主要产品。

虽然Nuro可能看起来是ike的最佳家园，但消息人士告诉TechCrunch，这家自动驾驶卡车初创公司已经与至少一家其他自动驾驶汽车公司就交易进行了谈判。两位消息人士补充说，这些都不是高级会谈。

创始故事

这两家公司的创始人不喜欢将ike描述为Nuro的衍生产品。这在技术上可能是正确的，但它们的根源是交织在一起的。

Nuro于2016年6月由前谷歌工程师Dave Ferguson和朱家军创立。这家初创公司最初是由弗格森和朱创办的，但到了2017年6月，他们很快从Greylock和Banyan那里筹集了9200万美元的资金，并让网易创始人丁磊(又名威廉丁)在Nuro董事会中获得了一个席位。

与此同时，范登伯格和孙离开苹果公司加入奥托(Otto)时，都在苹果公司的特别项目组工作，奥托是一家自动货运初创公司，于2016年被优步收购。伍德罗曾是Google X Makani项目的产品负责人，他也将于2017年2月结束在优步ATG公司的工作，担任其自动驾驶卡车项目的集团产品经理。

到2018年，最后一位Otto的创始人已经离开优步，自动驾驶卡车项目正处于自由落体状态。孙，伍德罗和范登伯格在2018年春天离开了。几个月后，优步关闭了自动驾驶卡车部门，专注于自动驾驶汽车。

这三个人最初在Nuro的办公室工作，那里太小了，容纳不下送货机器人和不断增长的团队。孙的大众野营车在早期充当了艾克的会议室。加利福尼亚州和特拉华州的业务记录显示，在7月份正式合并Ike之前，孙、范登伯格和伍德罗与Nuro的团队密切合作了几个月。该公司的名字Ike是为了向前总统德怀特·戴维·艾森豪威尔和他在签署联邦援助高速公路法案时帮助创建的美国州际公路系统致敬。

最初合作的目的是找出如何利用Nuro的成果，并将其应用到卡车运输的新用例中。伍德罗在最近的一次采访中表示:“这是我们朝着自己的方向前进的机会，在他们继续推进本地交付的同时，我们要真正关注这一点。”。

Ike将许可Nuro的技术，特别是硬件设计、自主软件以及数据记录、地图和模拟。作为回报，Nuro持有Ike的少数股份。

当Ike在2018年10月退出秘密模式时，Nuro 将其与新公司的关系描述为合作伙伴关系，其中“我们给了Ike一份我们的autonomy和基础设施软件，作为交换，Nuro获得了Ike的股权。”

Ike从小处着手，对未来的任务采取了低调的态度。截至2019年2月，Ike雇佣了约30名员工，并在贝恩资本风险投资公司(Bain Capital Ventures)牵头的5200万美元种子和首轮融资后，最终资本雇佣了更多员工。Redpoint Ventures、Fontinalis Partners、Basis Set Ventures和Neo也参加了这一轮投资。贝恩资本风险投资合伙人阿贾伊·阿加瓦尔加入了艾克的董事会。

与其他追求自动驾驶卡车的人不同，ike的创始人当时在一篇博客文章中说，他们没有推动第一辆自动驾驶卡车上路。根据AV货运界的几个消息来源，Ike确实因其系统工程方法、运动规划及其模拟工具而在业内赢得了良好的声誉。

由于Ike默默工作，Nuro在2019年2月获得了软银愿景基金9.4亿美元的投资，从而获得了更高的知名度。Nuro将其团队扩大到600多名员工，并在2018年与 Kroger 合作，在亚利桑那州试点送货服务。最初使用丰田普锐斯车辆的试点，过渡到其R1送货机器人。Nuro还与CVS、达美乐和沃尔玛等公司合作。该公司开发了第二代车辆，称为R2，旨在为餐馆，杂货店和其他企业提供本地送货服务。今年早些时候，R2获得了联邦政府的豁免，允许其作为无人驾驶汽车运营。

投资者并没有放弃Nuro。新冠肺炎疫情推迟了许多硅谷初创公司的计划。但也有亮点，包括本地交付的潜力。Nuro在11月又筹集了5亿美元，T4的投资后估值为50亿美元。C轮融资由T. Rowe Price Associates，Inc .咨询的基金和账户牵头，包括Fidelity Management&Research Company和Baillie Gifford在内的新投资者参与。该轮还包括软银愿景基金1和Greylock等现有投资者。

接下来会发生什么

两家公司将开始整合一些Ike工程师，让他们在Nuro担任更高级的职务。伍德罗将不再担任首席执行官，目前还不清楚他的新头衔是什么。根据他的经验，他的工作可能会侧重于产品开发。

由于Nuro专注于本地交付，Ike将卡车商业化的任务暂时被搁置了。转向本地交付的决定似乎与Ike创始人在无数场合发表的关于他们对卡车的热情的公开评论相反。

“我们在家里吃的、摸的或有的东西，可能在它生命的某个时刻都在卡车上，”孙在10月的TC Sessions: Mobility活动上说。“你不会真正意识到卡车运输对我们日常生活的巨大影响。”

但伍德罗说，对他和他的联合创始人来说，这实际上是让产品进入世界，并“实现自动化的承诺，我们许多人已经在一些不同的公司工作了多年。”

伍德罗说:“最终，我们的结论是，我们认为Nuro处于一个非常独特、非常有吸引力的位置，能够实现这个承诺，而且很快就能实现，而且能够大规模实现。”“所以这才是决定接受他们邀约并以这种方式继续前行的真正动力。”
