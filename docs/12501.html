通用电气更名为 C by GE，扩展到照明以外的领域

你很有可能错过了 5 月份的新闻——毕竟当时发生了很多事情。通用电气将其拥有 100 多年历史的通用电气照明部门出售给智能家居公司 Savant。今天，该部门的新老板宣布了一项关键的品牌重塑努力，这是在最初交易的六个月后。

显然，即使脱离了原来的母公司，通用电气的名字仍有价值。然而，在它的新家，通用电气公司的 C 系列正在更名。似乎完全合理——通用电气的 C 从来都不是最直截了当的名字。展望未来，这条线将被重新命名为“Cync”——如果不是更好，至少是不同的。

随着所有权的变化，这个名字意味着范围的扩大——这在莎凡特的统治下可能是不可避免的。通用电气照明正在其新家成为一个更大的智能家居品牌。下周开幕的 CES 将发现该公司推出了一种新的恒温器和一系列户外产品，包括一种新的智能插头。该品牌还将推出一款连接摄像头和风扇速度开关。

新产品将测试通用电气照明品牌的价值，不仅是通用电气的价值，也不仅仅是照明的价值。智能家居类别已经是一个拥挤的类别(多年来一直如此)，Savant 正在与亚马逊的 Ring 等大牌产品竞争。

根据 Savant 的说法，新的 Cync 应用程序将于 3 月发布，带来“更加用户友好和可定制的体验，增强舒适度、控制力和信心”。下周的消费电子展上有更多消息。
