OpenAI 的 DALL-E 可以为你要求的任何东西创建可信的图像

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/05/openais-dall-e-creates-plausible-images-of-literally-anything-you-ask-it-to/amp/

图片来源:Hiraman/Getty Images

OpenAI 最新的奇怪而迷人的作品是 DALL-E ，简单总结一下，它可能被称为“图像 GPT 3 号”它创建插图，照片，渲染或任何你喜欢的方法，你可以清楚地描述任何东西，从“一只戴领结的猫”到“一个穿着芭蕾舞裙遛狗的萝卜”但是现在还不要写股票摄影和插图的讣告。

像往常一样，OpenAI 对其发明的描述可读性很强，没有过多的技术性。但是它需要一点语境化。

研究人员用 GPT-3 创造的是一个人工智能，给它一个提示，它会试图生成它所描述的似乎合理的版本。因此，如果你说“一个关于一个孩子在森林里发现一个女巫的故事”，它会尝试写一个——如果你再次按下按钮，它会以不同的方式再次写一个。一次又一次。

这些尝试中的一些会比另一些更好；事实上，有些几乎是不连贯的，而另一些可能与人写的东西几乎无法区分。但它不会输出垃圾或严重的语法错误，这使它适合各种任务，正如初创公司和研究人员目前正在探索的那样。

DALL-E(Dali 和 WALL-E 的组合)将这一概念推进了一步。将文本转换成图像已经被人工智能代理人做了很多年，取得了不同但稳步增长的成功。在这种情况下，代理使用 GPT-3 及其底层结构提供的语言理解和上下文来创建与提示匹配的可信图像。

正如 OpenAI 所说:

GPT-3 表明，语言可以用来指导大型神经网络执行各种文本生成任务。图像 GPT 表明，同样类型的神经网络也可以用来生成高保真的图像。我们扩展了这些发现，以表明通过语言操纵视觉概念现在已经触手可及。

他们的意思是，这种类型的图像生成器可以被自然地操纵，只需告诉它做什么。当然，你可以挖掘它的内脏，找到代表颜色的标记，解码它的路径，这样你就可以激活和改变它们，就像你可以刺激真正大脑的神经元一样。但是当你要求你的插画人员画蓝色而不是绿色的东西时，你不会这样做。你只要说，“一辆蓝色的车”而不是“一辆绿色的车”，他们就明白了。

DALL-E 也是如此，它理解这些提示，很少以任何严重的方式失败，尽管必须说，即使在查看一百或一千次尝试的最佳结果时，它生成的许多图像也不仅仅是一点点…错误。其中后来。

在 OpenAI 的帖子中，研究人员给出了丰富的互动例子，说明系统如何被告知对相同的想法做微小的改变，结果似乎是合理的，而且往往相当好。事实是，这些系统可能非常脆弱，因为他们承认 DALL-E 在某些方面是如此，并且说“一个五边形的绿色皮革钱包”可能会产生预期的结果，但“一个五边形的蓝色麂皮钱包”可能会产生噩梦般的燃料。为什么？鉴于这些系统的黑箱性质，这很难说。

图片来源:OpenAI

但是 DALL-E 对这种变化非常稳健，可以可靠地生产出你想要的任何东西。鳄梨色拉酱的圆环，斑马的球体；一个大的蓝色积木放在一个小的红色积木上；一只快乐的水豚的前视图，一只悲伤的水豚的等距视图；诸如此类。你可以在帖子里玩所有的例子。

它还展示了一些非故意但有用的行为，使用直觉逻辑来理解请求，如要求它为同一只(不存在的)猫绘制多个草图，原始草图在顶部，草图在底部。这里没有特别的编码:“我们没有预料到这种能力会出现，也没有修改神经网络或训练程序来鼓励它。”这很好。

有趣的是，OpenAI 的另一个新系统 CLIP 与 DALL-E 结合使用，以理解和排列有问题的图像，尽管它更具技术性，更难理解。你可以在这里阅读有关剪辑的内容。

这种能力的含义是多种多样的，以至于我在这里不打算深入探讨。甚至 OpenAI 平底船:

在未来，我们计划分析像 DALL E 这样的模型如何与社会问题相关联，如对某些工作流程和职业的经济影响，模型输出中潜在的偏见，以及这项技术隐含的长期伦理挑战。

现在，像 GPT 3 号一样，这项技术是惊人的，但很难做出明确的预测。

值得注意的是，它产生的东西很少是真正“最终”的——也就是说，我不能告诉它为我最近写的任何东西制作一个主要图像，并期望它不加修改就能推出我可以使用的东西。即使是简单的检查也会发现各种各样的人工智能怪异之处(詹妮尔·谢恩的专长)，虽然这些粗糙的边缘肯定会及时被打磨掉，但这远远不安全，GPT-3 文本不能未经编辑就代替人类写作发出去。
