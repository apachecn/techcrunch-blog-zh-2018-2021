这6个浏览器扩展将提高您的在线隐私

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/10/these-6-browser-extensions-will-immediately-improve-your-privacy-online/amp/

图片来源:warm world/其他 / Getty Images

互联网不是一个私人的地方。广告试图尽可能多地了解你，以便将你的信息卖给出价最高的人。电子邮件知道你什么时候打开它们，以及你点击了哪些链接。一些最大的互联网窥探网站，如脸书和亚马逊，在你浏览网页时从一个网站到另一个网站跟踪你。

但不一定要这样。我们已经尝试和测试了六种浏览器扩展，通过阻止大多数隐形广告和追踪器，它们将立即改善您的在线隐私。

这些扩展不会阻止每一种窥探，但它们将极大地减少你暴露于跟踪你的互联网活动的大部分努力。你可能不在乎广告商收集你的数据来了解你的品味和兴趣，为你提供有针对性的广告。但是你可能会关心这些广告巨头能看到你正在查找的医疗状况和你正在进行的私人购买。

通过阻止这些隐藏的追踪器加载，网站无法收集你的更多信息。此外，通过删除不必要的大量内容，一些网站会加载得更快。代价是，如果你不让他们跟踪你，一些网站可能无法正常加载或拒绝让你进入。你可以根据需要打开或关闭扩展，或者你可以问问自己，这个网站一开始是不是很好，你能不能不在别的地方找到你要找的东西？

HTTPS无处不在

我们几乎天生就会在浏览器中寻找那个绿色的小锁，告诉我们一个网站是通过HTTPS加密的连接加载的。这意味着您打开的网站在加载之前没有被攻击者劫持或修改，并且您提交到该网站的任何内容都不会被该网站以外的任何人看到。HTTPS Everywhere 是由非营利互联网组织电子前沿基金会开发的一个浏览器扩展，它可以自动加载HTTPS各地的网站，并允许你屏蔽少数不支持HTTPS的网站。扩展是大多数浏览器支持的，包括Chrome、Firefox、Edge和Opera。

隐私獾

EFF开发的另一个扩展，Privacy Badger是阻止网站上看不见的第三方追踪者的最好的一体化扩展之一。这个扩展查看网页的所有组件，了解哪些组件从一个网站到另一个网站跟踪你，然后阻止它们在浏览器中加载。隐私獾也随着你在网上的旅行而学习，所以随着时间的推移它会变得更好。而且它不需要任何努力或配置就可以工作，只需安装它并留给它。该扩展可在大多数主流浏览器上使用。

子块来源

广告让互联网保持免费，但通常是以你的个人信息为代价的。广告试图尽可能多地了解你——通常是通过观察你的浏览活动并在网上跟踪你——这样他们就可以向你投放你更有可能点击的广告。广告拦截器通过阻止广告加载以及随之而来的跟踪代码来阻止他们。

uBlock Origin 是一款轻量级、简单但有效的广告拦截器，被数百万人广泛信任，但它也为更高级的用户提供了大量的粒度和可定制性。(小心冒名顶替者:有很多广告拦截器使用相似的名字，不太可信。)如果你对那些依靠广告获取收入的网站感到不爽(包括我们！)，考虑订阅该网站。毕竟，一个依靠广告跟踪赚钱的免费网络是我们陷入这场隐私噩梦的原因。

uBlock Origin在 Chrome 、 Firefox 和 Edge 中工作，而扩展是开源的，所以任何人都可以看看它是如何工作的。

像素块和清除URL

如果你认为网站中隐藏的追踪器不好，那就等着了解你的电子邮件中潜伏着什么吧。大多数来自品牌的电子邮件都带有微小的、通常不可见的像素，当你打开它们时会提醒发件人。 PixelBlock 是Chrome浏览器的一个简单扩展，它只是阻止这些隐藏的电子邮件打开跟踪器加载和工作。每次它检测到一个追踪器，它会在你的收件箱里显示一个小红眼，让你知道。

这些电子邮件中的大多数还带有跟踪链接，提醒发件人你点击了哪些链接。适用于 Chrome 、 Firefox 和 Edge 的ClearURLs ，位于你的浏览器中，默默地从你的浏览器和收件箱的每个链接中删除跟踪垃圾。这意味着ClearURLs需要比大多数这些扩展更多地访问你的浏览器数据，但是它的制造者在文档中解释了为什么。

Firefox多帐户容器

对于Firefox用户来说，这是一个荣誉，他们可以利用浏览器制造商自己构建的多账户容器来帮助你隔离你的浏览活动。这意味着您可以在浏览器中用一个容器装满您的工作选项卡，用另一个容器装满您的所有个人选项卡，这样您就不必使用多个浏览器。容器也让你的私人浏览和工作浏览活动分开。这也意味着你可以把脸书或谷歌这样的网站放在一个容器里，让他们更难看到你访问了哪些网站，更难理解你的品味和兴趣。容器易于使用和定制。
