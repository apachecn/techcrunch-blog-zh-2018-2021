原文：https://web.archive.org/web/https://techcrunch.com/2021/01/12/techstars-names-maelle-gavet-ceo-as-the-accelerator-group-looks-to-expand/

随着加速器集团寻求扩张，Techstars任命了malle Gavet首席执行官

今天早上，由创业加速器和风险投资基金组成的网络Techstars宣布，的玛勒·加维特将成为新的首席执行官。前首席执行官兼联合创始人大卫·布朗将留在Techstars董事会，而该集团的另一位联合创始人大卫·科恩将成为董事会主席。

TechCrunch今天上午采访了Gavet，讨论了她的新工作、变化的时机、公司的扩张计划以及她在这个职位上的目标。

Gavet说，她被带来帮助Techstars发展，详细介绍了她在更大规模和多个地区的公司的工作经验，包括Compass和Booking.com。

TechCrunch很好奇初创公司市场有多大，以及在不同的地理位置和利基市场还有多少空间可供Techstars和其他公司扩张。Gavet说，当她被招聘担任新职位时，她曾向Techstars问过同样的问题。她说，有大量被忽视的人才和投资不足的地区可以在资本和帮助下得到授权和释放。Techstars希望找到这些创始人并投资他们。 更新:为清晰起见，本段语言在出版后进行了编辑。

我们推测，这意味着更多地方的更多加速器投资更多创业者。

Gavet告诉TechCrunch，Techstars已经投资了超过2300家公司，并且每年向大约500家公司投资。

新任首席执行官解释说，她认为在为企业家提供大量支持并产生积极社会影响的同时，为投资者创造丰厚回报是可能的。这是一份需要立即执行的雄心勃勃的清单，但如果她成功了，她的努力可能有助于科技企业家的多元化，这是长期以来一直需要的。

看到一家初创公司的领导者为不同的、更大规模的运营经验进行优化并不罕见。对于一个元创业公司，一个加速器和投资公司来说，做同样的事情并不奇怪。

TechCrunch定期覆盖加速器群组，包括Techstars ( 一些最近的 笔记这里)和Y Combinator，以及其他程序。历史上，一些最知名的科技公司都出自这样的加速器集团，包括Airbnb ( 现在已经公开)的迈克尔·赛贝尔领导的 Y Combinator 、TalkDesk ( 价值超过30亿美元)的克里斯汀·蔡)领导的 500家初创公司，以及Techstars自己的SendGrid ( 被Twilio以20亿美元收购。

看看Techstars的加速器模式下一步将走向何方将是一件有趣的事情——该集团有时会与公司或美国空军等团体合作，赞助和支持量身定制的项目——就地点和重点而言。但如果它能成功地帮助创始人群体多元化，同时为自己赚钱，它将突显出市场中的其他人如何能做得更好。
