Vantage 使得管理 AWS 更加容易

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/12/vantage-makes-managing-aws-easier/amp/

图片来源:罗恩·米勒/TechCrunch

Vantage ，一项使管理 AWS 资源及其相关支出变得更容易的新服务，将于今天秘密发布。该服务为其用户提供了复杂的 AWS 控制台的替代方案，具有对大多数标准 AWS 服务的支持，包括 EC2 实例、S3 桶、VPC、ECS 和 Fargate 以及 Route 53 托管区域。

该公司的创始人 Ben Schaechter 之前在 AWS 和 Digital Ocean 工作(在此之前，他也在 Crunchbase 工作)。然而，虽然 DigitalOcean 向他展示了如何为个人和小企业打造开发者体验，但他认为底层服务和硬件根本没有超级云强大。另一方面，AWS 提供了开发者想要的一切(可能更多)，但是用户体验还有很多不足之处。

图片来源: Vantage

“这个想法实际上源于‘如果我们能把数字海洋的用户体验应用到三个公共云提供商，AWS、GCP 和 Azure，会怎么样？’”Schaechter 告诉我我们决定只从 AWS 开始，因为那里的体验是最艰苦的，而且它是市场上最大的玩家。我真的认为，在我们做 GCP 和 Azure 之前，我们可以在那里提供很多价值。"

Vantage 的重点是开发者体验和成本透明。Schaechter 指出，它的一些用户将其描述为类似于“AWS 的造币厂”。首先，你给 Vantage 一组 AWS 服务的读取权限，工具会自动分析你账户中的所有内容。该服务每小时刷新一次列表，但用户也可以手动刷新他们的列表。

https://youtu.be/uLYCGFduuv0

鉴于通常很难知道您实际使用的是哪些 AWS 服务，这本身就是一个有用的特性。“这是头号用例，”他说。“我们花钱买什么，我们有什么？”

Vantage 的核心是团队所称的“视图”，它允许您看到您正在使用哪些资源。有趣的是，这是一个非常灵活的系统，允许您构建自定义视图来查看跨区域的给定应用程序使用了哪些资源。这些可能包括 Lambda、存储桶、您的子网、代码管道等等。

在成本跟踪方面，Vantage 目前只提供时间点成本，但 Schaechter 告诉我，该团队计划添加历史趋势，以便让用户更好地了解他们的云支出。

Schaechter 和他的联合创始人创办了这家公司，他指出，在他想为这项服务筹集任何资金之前，他希望看到人们为此付费。目前，Vantage 提供免费计划，以及带有附加功能的付费“专业”和“商业”计划。
