传统保险公司提供的汽车保险存在结构性偏见。它使用信用评分、收入、婚姻状况和教育等指标来计算保险费率，最终通过高费率和低保护不成比例地伤害低收入个人。

Loop，由约翰·亨利和凯里·安妮·纳多共同创立，希望推出一种对所有社区都公平的替代模式。

“结构性偏见融入了金融服务和机构，它们延续并强化了(结构性偏见)，”曾在布鲁金斯学会(Brookings Institute)工作、在麻省理工学院(MIT)围绕流动性课题进行研究的纳多表示。“我们不能只关注银行业，(而且)在金融服务的世界观中，保险业是一个被忽视的丑陋的继子女。”

如何改写规则

Loop 是一家管理总代理(MGA)公司，因此它可以作为保险领域的经纪人和供应商。它营销、收购和服务客户，而不是简单地作为建立在现有保险提供商之上的供应商。这家初创公司也是一家 B 公司，它将利润与环境和社会动态放在一起优先考虑。

这家初创公司试图通过使用两个关键指标来跟踪、创建和收取保险费率，来改写汽车保险的规则:道路状况和司机行为。Loop 根据使用情况确定费率，而传统提供商可能根据人口统计数据确定费率。

Loop 是与保险公司垂直整合的纯移动产品。

一旦用户下载了应用程序，Loop 就会根据用户的位置为其找到报价。秘方是 Loop 的技术:使用 27 个州超过 1 亿起车祸的数据库，Loop 根据用户的位置为他们创建报价。Harlem Capital 的联合创始人亨利(Henry)描述 Loop 的数据“几乎是对每条道路上发生的车祸的神级理解。”

这家初创公司还利用交通量、道路基础设施和天气数据来设定费率。人工智能功能可以让 Loop 引导司机离开具有高撞车风险的道路。或者，它可以简单地奖励他们清理道路没有保险杠刮擦。

其业务的另一部分是基于远程信息处理技术，这使得 Loop 可以随时了解司机的去向和方式。虽然传统航空公司可能会利用事故少来激励降低费率，但 Loop 正在利用数据来设定和降低费率。

交换数据以获得更大的灵活性可能会引起一些质疑，但联合创始人认为，他们的客户群(主要是千禧一代和 Z 世代)对这种模式感到满意，因为它承诺更公平的价格。Loop 从它卖出的每份保单中赚取总佣金。

Loop 还以俄亥俄州的 Root Insurance 为例，说明消费者对分享位置数据越来越放心。这家汽车保险初创公司上市,在许多人看来，这是一家中西部高增长科技公司的成功 IPO 。Root 同样使用驾驶员表现和远程信息处理技术历史等指标。

“他们使用远程信息处理技术，但他们仍然在很大程度上使用传统的保险模式，”亨利说。“我们正在用我们自己的基于人工智能的方法来取代它。”

Root 可能是最明显的竞争对手，但十多年来，基于使用的定价已经通过各种形式在保险行业成为一种上升趋势。Flexible insurtech 最近一直在蓬勃发展，MetroMile 的特殊目的收购公司，Lemonade 的 IPO，以及处于早期阶段的 Marshmallow，英国汽车保险初创公司最近的估值为 1.3 亿美元。

联合创始人相信，他们的技术足够差异化，能够在激烈的竞争中生存下来。

种族清算和推特

创业的想法始于 2020 年 7 月，当时黑人乔治·弗洛伊德被警察谋杀。世界各地爆发了抗议活动，呼吁变革和解决系统性种族主义的办法。风险投资公司纷纷支持黑人创业者，亨利看到了致力于改变的解决方案中的差距。

亨利在推特上回应道:

亨利说:“我突然意识到，我们所寻求的改变不会自然而然地发生。”。"这需要有意识地解决系统性问题."他知道纳多关注的是交通和移动，两人最终决定“大展拳脚”

Carey Anne Nadeau 和 John Henry，Loop 的联合创始人。图像制作者名单:循环

虽然联合创始人承认这个目标雄心勃勃，但他们已经找到了投资者，他们认为 Loop 有一天会成为一个大企业。这家初创公司告诉 TechCrunch，它已经筹集了 325 万美元的种子轮，由 Freestyle VC 牵头，蓝雾资本、Fontinalis Capital Partners、Concrete Rose、起义风险投资公司和 Backstage Capital 参与其中。参与的天使投资人包括 Kristen Dickey、Steve Schlafman、Songe LaRon、Craig J. Lewis、Gerard Adams 和 Joshua Dorkin。

这笔钱将用于雇佣和开发其数据科学基础设施。它还没有上市，但正在俄亥俄州、伊利诺伊州、宾夕法尼亚州和纽约州推出(当然，还有待监管机构的批准)。

该团队会见了 77 名投资者，其中 25%是女性投资者，以获得启动 Loop 所需的资金。

“这比我们想象的要难，”亨利说。“我们从一开始就知道，我们希望筹集更多的种子资金，向市场发出信号，表明我们希望做大。”

Loop 最终关闭了目标轮次和估值。至于让投资者支持一家用大约 300 万美元种子融资扰乱了 2.56 亿美元产业的公司的转折点是什么？

亨利说是任务。

“我现在真的起鸡皮疙瘩，因为这项任务将打开利润无法打开的大门，”他说。
