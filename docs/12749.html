Rapyd 以 25 亿美元的估值融资 3 亿美元，以提升其金融科技即服务 API

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/13/rapyd-raises-300m-on-a-2-5b-valuation-to-boost-its-fintech-as-a-service-api/

在全球新冠肺炎疫情环境的推动下，一波组织正在将他们的商业和金融互动转移到网上，今天，帮助实现这一转变的一个大公司宣布了一轮重要的增长资金，以扩展它为他们提供的工具和服务。

Rapyd 提供了一个基于 API 的“金融技术即服务”平台，涵盖支付、银行服务、欺诈保护等领域，该公司已经筹集了 3 亿美元，首席执行官兼联合创始人 Arik Shtilman 在接受采访时表示，这笔资金将用于扩大其团队，建立更多技术(接下来是:扩大欺诈识别服务和更广阔的市场)，以及进行选定的收购。

Rapyd 的客户群目前约有 5000 家企业，包括市场(劳动力市场和商品市场)、电子商务企业、其他类型的贷款机构以及任何可能希望将交易或新金融服务纳入其更广泛产品的企业。Shtilman 说，目前，Rapyd 的增长势头最为强劲，每周约有 500 名新客户加入。

Shtilman 证实，这笔融资的估值为 25 亿美元。(关于这一点，Rapyd 最近一次估值为 12 亿美元是在 2019 年 12 月。)

这一轮是 D 轮，由多产的增长轮 VC Coatue 牵头，Spark Capital、Avid Ventures、FJ 实验室和 Latitude(都是新的支持者)，以及 General Catalyst、Oak HC/FT、Tiger Global、Target Global、Durable Capital、Tal Capital 和 entre Capital(都是以前的支持者)也参与了。其他过去的投资者，值得注意的是，包括基于 API 的金融服务领域的另一个主要参与者 Stripe 。

与去年需求激增的其他类别的公司一样，金融服务——尤其是那些提供能够通过互联网或电话进行在线交易的服务的公司——已被证明是最具强制性和使用最多的服务。(难怪，因为账单仍然需要支付，食物和其他物品仍然需要购买，贷款仍然需要进行等等。)

这就是许多人所说的“机会主义”加薪，不是为了维持运营或延长跑道，而是因为这笔钱以优惠的条件提供给了 Rapyd，而且有一些聪明的地方可以用来发展业务。

“当我们筹集这一轮资金时，我们并不打算筹集资金，但当疫情进来时，我们的业务开始繁荣起来，”Shtilman 说。“在我们于 2020 年三个月内完成 2021 年增长计划后，现有投资者与我们接触，要求我们超越最初的业务计划。所以我们认为这可能是统治世界的最佳时机。”

施蒂尔曼在一定程度上(只是一定程度上)是在开玩笑——他那种面无表情的表达方式我在这里不太能捕捉到——但这与初创公司的早期有天壤之别，那时“没有人想投资，因为每个人都认为执行起来太难了。 甚至我们的早期投资者也建议我们关注一个更小的概念。但是我们认为全球化建设是行不通的。从小处着手违背了这个想法。在过去的几年里，解释我们做什么的需求几乎消失了。”

Rapyd 在 2017 年首次开业时(最初在以色列孵化)就发现了挑战(和机遇)，即全球商业和金融市场高度分散。各个市场的消费者和企业都有自己偏好的支付方式和需求，各国的法规各不相同，涉及的主要公司也各不相同。

与此同时，API 长期以来一直是集成和连接的重要工具:使用几行代码——假设您自己的服务也是基于代码构建的——您可以将服务编织在一起，并引入需要花费大量时间从头构建的商品化功能，减少所需的努力和工作，从而专注于使您的核心业务更加独特。

虽然 Stripe、Twilio 和许多其他公司已经发现了利用 API 向更广泛的潜在客户扩展功能世界的机会，但 Rapyd 真正发现并构建的想法是，在该提议中加载的不只是一个、两个或三个服务，而是数百个(甚至数千个)功能。在金融科技领域，这些服务很复杂，有大量的服务可供选择，如果你的目标很广，也有大量的潜在客户可以使用它们。

这个想法很聪明，而且正如施蒂尔曼所指出的，非常符合电子商务和金融科技领域存在的规模经济:个人交易归根结底是非常增量的，因此将许多人聚集在一起的服务最终可以开始设想有趣的回报。

当然，这不仅仅是 Rapyd 所认同和经营的事情。也就是说，该公司现在在市场上有许多竞争对手。

就在上周，总部位于德国的 Mambu，也在“银行即服务”的理念下提供基于 API 的服务套件(最近统计为 7，000 个)，以超过 20 亿美元的估值筹集了 1.35 亿美元。Rapyd 的支持者 Stripe 也在继续扩张，增加了许多支付之外的服务。思想机去年也融资了一大轮；特梅诺斯和意大利埃德拉也是这里的强手。

这个领域有如此多的机会，甚至吸引了许多新的进入者:见证单元，另一个有趣的玩家于 12 月在美国悄然出现，拥有自己有趣的支持者名单。

“要建立金融基础设施，不管你是小型夫妻店还是大型企业，你都需要很多东西，如果你想在多个司法管辖区销售，你需要很多这些服务，”Shtilman 提到了金融科技平台提案对规模和广度的需求。他对竞争也非常乐观。

“它们就像雨后春笋般涌现出来，”他说。“但如果你没有竞争，就意味着你没有生意，所以这很好。意味着需求很大。但目前我们是市场领导者。我们认为我们将成为这个领域的 AWS。”

这也是像 Coatue 这样的投资者目前登陆的地方。

“不同国家的支付情况差异很大。一家在全球开展业务的公司可能需要接受数百种本地支付方式。Rapyd 的 API 抽象出了这种复杂性，目前正在为我们认为世界上许多最令人兴奋的公司提供动力，”Coatue 的管理合伙人 Kris Fredrickson 在一份声明中说。“我们很荣幸能与 Arik 和团队合作，开始 Rapyd 旅程的下一阶段。”
