Capsule 为其针对品牌的合作视频平台筹集了 200 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/14/capsule-raises-2-million-for-its-video-qa-platform-aimed-at-brands/

去年，针对企业在疫情期间接触消费者所面临的挑战，一个针对品牌的视频平台应运而生。现在，这家公司已经获得了 200 万美元的前期种子资金。这轮融资由 Array Ventures 领投，彭博贝塔和其他天使投资人也参与其中。

这家初创公司是由最初创建动画 GIF 捕捉工具和社交网络 Phhhoto 的同一团队创建的，最终输给了 Instagram 的克隆Boomerang。Phhhoto 于 2017 年关闭，团队转向体验营销业务， Hypno。这家新公司一直在与举办现场活动和体验的品牌合作，以此作为联系客户的一种方式。Hypno 会为他们提供照相亭和其他允许互动的摄像平台。

随着现场活动的枯竭，新冠肺炎疫情实际上扼杀了 Hypno 的业务。然而，Hypno 合作过的品牌仍然有同样的需求——他们只是必须以不同的方式接触他们的客户。

图片来源:胶囊

胶囊就是这样产生的。这家初创公司于去年推出，为主持问答会议提供了一个完整的平台，品牌从一个模板开始，然后通过改变徽标、颜色、按钮、背景和网址来定制匹配他们的活动——有点像视频问答格式的 Squarespace。

然后，该品牌将以短视频回复的形式，写出他们的问题和提示，供消费者回答。这个问答网址是以公司选择的方式发布的——比如在社交媒体上。一项新功能还允许在网站上嵌入一个“胶囊”。

消费者对问答的反应是为最终的视频产品策划的。让这项技术更加有趣的是 Capsule 如何组装这些镜头。

Capsule 会即时自动处理视频，在播放前或播放后添加音乐和图形等元素，使视频看起来经过专业编辑。这家初创公司通过使用自己的基于 JavaScript 的编程语言来实现这一点，这种语言可以自动混合颜色、音频、图形和动态类型等内容。客户所要做的就是选择他们想要的视频类型——例如，一个充满活力的视频或一个更加阴郁的视频。

今天，Capsule 已经将其库扩展到大约 20 个基本模板。但是每首歌都可以通过改变颜色、风格甚至音乐来编辑——包括直接上传或者从 Capsule 提供的数千首免版税的歌曲。

 

根据 Capsule 联合创始人 Champ Bennett 的说法，该平台的灵活性导致了广泛的用例。虽然该公司的第一批客户来自 Hypno 迎合的现场活动空间，但新客户很快开始采用该产品。

“我们立即开始从现有客户那里听到，然后突然，一群新客户向我们走来，他们说这个平台在许多不同的环境中都很有用，”他解释道。“例如，UGC(用户生成内容)活动或生成社交内容以促进业务，或提高知名度，或产品评论和推荐，甚至是那些只想更快地制作看起来和感觉起来更专业的内容的创作者，”Bennett 说。

在发布时，Capsule 被像网飞这样的公司和像 OkayAfrica 这样的组织使用。自那以来，它已经与数百名客户签订了合同，包括谷歌、三星、Salesforce、德勤和华尔街日报等大型机构的团队，以及其他中小型企业，如帕洛玛健康。USO 也使用了胶囊。

这些品牌和其他品牌特别渴望有工具来帮助他们创作原创的短片内容，这正成为企业营销其产品和服务的一种重要方式。Capsule 指出，研究表明视频点击率比静态图像高两到三倍，95%的企业报告说他们的视频支出逐年增加。

疫情加速了对视频内容的现有需求，但品牌面临着视频难以扩展的挑战。

“越来越多的品牌开始改变他们制作视频的方式，”Bennett 说。“他们这样做的方式之一是利用他们组织内的不同创作者——无论是员工、创始人、合作伙伴、有影响力的人、品牌粉丝还是其他什么人——然后激励他们代表他们创作内容，”他继续说道。“我们称之为社区生成内容，这有点像用户生成内容的迭代。正是这种想法，内容可以来自任何地方。”

Capsule 表示，它希望与 Array Ventures GP Shruti Gandhi 合作，她有工程背景，因为她非常了解核心技术。贝内特说，她将团队介绍给了纽约的彭博贝塔公司，该公司也立即明白了 Capsule 在制造什么。

有了额外的资金，Capsule 将雇用一名产品设计师，并开发新的协作功能，如上传内容的能力。

从长远来看，该公司相信其平台将使更多的人参与视频创作。

“事实证明，企业中有很多不同的人都有能力创作视频内容。他们只是没有做这件事的技术诀窍。他们不是视频编辑，”班尼特说。“因此，我们真正在做的是发掘存在于各地企业内部的这种创造潜力，无论企业规模大小。”
