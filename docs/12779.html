Harness 以 17 亿美元的估值获得 8500 万美元的 C 系列，收入增长 3 倍

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/14/harness-snags-85m-series-c-on-1-7b-valuation-as-revenue-grows-3x/

初创公司 Harness 希望创造一套工程工具，让每家公司都拥有最大公司拥有的技术影响力，该公司今天宣布了 8500 万美元的 C 轮融资，估值为 17 亿美元。

今天的一轮是在 2019 年 6000 万美元的 B 轮之后，B 轮估值为 5 亿美元，显示公司价值快速增长。对于一家三年前才成立的公司来说，这是一个相当不寻常的轨迹。

Alkeon Capital 在新投资者 Battery Ventures、Citi Ventures、Norwest Venture Partners、Sorenson Capital 和 Thomvest Ventures 的帮助下领投了这轮融资。这家初创公司还透露了此前未宣布的 3000 万美元 B-1 轮融资，此前该轮融资为 6000 万美元，使迄今为止的融资总额达到 1.95 亿美元。

公司创始人兼首席执行官 Jyoti Bansal 之前创立了 AppDynamics，他在 2017 年以 37 亿美元的价格将该公司出售给思科。凭借他的过往记录，投资者在这一轮开始寻找他。去年收入增长了近 3 倍，这并没有什么坏处。

“这项业务做得非常好，所以投资者群体一直在主动接触我们，试图投资我们。我们实际上并不打算在今年晚些时候进行融资。我们有足够的资本度过难关，但有很多人想投资，”班萨尔告诉我。

事实上，他说投资者的兴趣如此之大，以至于他本可以筹集两倍的资金，但他觉得没有必要在此时筹集那么多资金。“总的来说，投资者群体看到了开发者工具和 DevOps 市场的价值。在过去的三到五年里，这个领域有如此多的大型上市公司已经退出，这无疑为这个领域创造了更多的认可，”他说。

班萨尔说，他创办这家公司的目标是，在工程效率方面，让每家公司都像谷歌或脸书一样优秀。由于大多数公司缺乏这些大公司的工程资源，这是一个很高的任务，但他认为他可以通过软件解决。

该公司从建立一个持续交付模块开始。接着是云成本效益模块。去年，该公司收购了开源持续集成公司 Drone.io，现在他们正致力于将其构建到平台中，目前处于测试阶段。据 Bansal 称，今年的产品路线图上还有其他模块。

随着该公司在 2020 年继续增加收入和构建平台，它还增加了一系列新员工，在疫情期间从 200 人增长到 300 人。班萨尔说，他计划在今年年底前再增加 200 家。Harness 以工作的好地方而闻名，最近登上了 Glassdoor 的最佳公司名单。

作为一名经验丰富的企业家，Bansal 非常重视建立一个具有欢迎文化的多元化公司。“是的，你必须提供平等的机会，并确保你愿意雇用来自不同背景的人，但你必须更加积极主动，因为你必须确保你的公司环境和公司文化让每个人都感到非常欢迎，”他说。

在疫情期间，建立一家公司，增加这么多新员工，并找到一种让每个人都感到受欢迎和被包容的方式，这是一段艰难的时期。班萨尔说，在疫情期间，他确实看到了生产率的提高，但现在不得不警惕员工的倦怠。

他说，人们在家工作时不知道如何划清界限。他做的一件事是推出了一个计划，让每个人每个月有一个星期五休息充电。该公司最近还宣布，在 COVID 之后，它将成为一家“在任何地方工作”的公司，但 Bansal 仍计划设立地区办事处，人们可以在需要时在那里会面。
