因警告同事纳粹而被解雇的 GitHub 员工正在寻求法律咨询

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/15/fired-github-employee-who-warned-coworkers-about-nazis-is-seeking-legal-counsel/amp/

图片来源:迈克尔·肖特/彭博/盖蒂图片社

特朗普支持者的暴民袭击美国国会大厦的那天，一名忧心忡忡的 GitHub 员工警告他在华盛顿的同事注意安全。

一位同事告诉我，他在 Slack 上发表评论说，“注意安全，兄弟们，纳粹是要来的，”这位同事很生气，说这种言论对工作没有好处。他告诉我，两天后，他被解雇了，一名人事关系代表以“不利于公司政策的行为模式”为由解雇了他。

在接受 TechCrunch 采访时，这位现已离职的员工表示，除了他的犹太家庭成员之外，他真的很关心他在这个地区的同事。

由于担心被解雇员工及其家人的安全，TechCrunch 同意对其身份保密。

作为 商业内幕最先报道的 ，他的被解雇导致员工流传一封内部信，要求 GitHub 声讨白人至上主义和纳粹。员工们也想知道他被解雇的原因。这导致 GitHub 首席执行官 Nat Friedman 告诉员工，公司将调查该员工的解雇事宜。

现在，这位被解雇的员工表示，他目前正在寻求法律顾问，以确保他的家人受到保护，并弄清楚他是否可以获得损害赔偿或其他形式的和解。这位被解雇的员工说，GitHub 已经向他寻求内部调查的帮助，但正在等待与该公司接触，直到他找到合法的代理人。

尽管如此，他说他对调查并不乐观。

“我 90%确定这不是真的，”这名被解雇的员工谈到弗里德曼的回应时说。“这类事情以前就有人说过。它发生在冰的事情上，公司说让我们讨论一下，但是如果你提到冰，你就会被解雇。以前相信这家公司，现在不相信了。”

与一些员工提出的问题类似，被解雇的员工认为这是 GitHub 表明白人至上立场的一个机会。

他说，“我觉得这可能是 GitHub 真正进行清洗的机会，并说‘我们希望这家公司有白人至上主义者吗？我们如何让黑人领导人进入管理层？’"

他说，后者是他加入 GitHub 以来一直想要的东西。但是，当他不停地谈论领导层缺乏多样性时，他说他发现自己的工作处于危险之中。

“当我一直谈论这件事的时候，我在 10 月份受到了被解雇的威胁，”他说。“当我指出销售团队可能只有两个有色人种时，我的两个经理不得不完全为我辩护，恳求他们不要解雇我。”

更新 1/16:被解雇的 GitHubber 后来澄清说，他指的是销售领导团队，而不是整个销售团队。

在发给 TechCrunch 的一份关于这篇文章内容的总括声明中，GitHub 发言人说:

我们非常认真地对待所有这种性质的投诉。我们正在积极调查情况。

这位前雇员说，在他被解雇时，公司给了他两张薪水支票，让他离开了。他说，他愿意接受某种形式的和解，无论是损害赔偿、医疗保险还是其他形式。虽然他不打算重新找工作，但他说他希望在 GitHub 看到更多的工人力量。

他说:“如果我有一根魔杖，我希望 GitHub 的员工能够拥有一个工会，代表来自边缘化社区的人们。”。
