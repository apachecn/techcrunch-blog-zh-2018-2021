鉴于犹太员工 TechCrunch 被解雇，GitHub 的人力资源主管辞职

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/17/githubs-head-of-hr-resigns-in-light-of-termination-of-jewish-employee/

GitHub 的一项内部调查显示，该公司在解雇犹太雇员一事上犯了“判断和程序上的重大错误”,这名犹太雇员在美国国会大厦起义当天警告他的同事华盛顿地区有纳粹存在。

在今天的一篇博客文章中，GitHub 首席运营官·埃里卡·布雷西亚表示，公司的人力资源主管对所发生的事情承担全部责任，并于昨天从公司辞职。GitHub 没有透露辞职人员的姓名，但众所周知，凯莉·奥勒森是 GitHub 的首席人力资源官。

在昨晚的一条推文中，GitHub 全球人力资源服务高级总监 Gia Colosi 对公司和人力资源做了一些评论。该推文已被删除，但截图如下。

在后来的一条推文中，她继续说道，“女人在人力资源部是为了收拾男人的烂摊子。我做了，累了。”

与此同时，GitHub 表示，它已经“撤销了与该员工离职的决定”，并正在与他的代表谈话。

“对于这位员工，我们想公开说:我们真诚地道歉，”布雷西亚在博客中说。

这位被解雇的员工在 Slack 上发表评论说，“注意安全，兄弟们，纳粹是要来的，”这位前员工告诉我，一位同事很生气，说这种言论对工作没有好处。他告诉我，两天后，他被解雇了，一名人事关系代表以“不利于公司政策的行为模式”为由解雇了他。

在本周早些时候接受 TechCrunch 采访时，这位现已离职的员工表示，除了他的犹太家庭成员之外，他真的很关心他在该地区的同事。在那次采访中，他说他对复职不感兴趣，但对其他形式的和解感兴趣。
