CBS All Access的继任者Paramount+将于3月4日在美国、加拿大和拉丁美洲推出

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/19/paramount-the-successor-to-cbs-all-access-launches-march-4-in-the-u-s-canada-and-latin-america/

去年，ViacomCBS 宣布其CBS All Access流媒体服务将很快更名为Paramount+，以更好地反映2019年Viacom-CBS合并后扩大的内容阵容。今天，该公司表示，它已经确定了派拉蒙+在美国的发布日期:2021年3月4日。它还分享了其他国际市场的上市日期，包括拉丁美洲、加拿大和北欧。

该服务将于3月4日在拉丁美洲市场推出，并将同时从CBS All Access更名为加拿大的Paramount+。然而，加拿大要到2021年晚些时候才能获得扩大的阵容。北欧地区将于2021年3月25日提供服务，随后将于“2021年年中”在澳大利亚推出。

自去年早些时候以来，该公司一直在宣传其更名服务的计划，解释新的流媒体服务，即CBS All Access的扩展，将如何让它展示该公司最大的特许经营和其丰富的图书馆，同时也为其不断增长的原创内容收藏提供一个家，如目前在CBS All Access上播放的多个“星际迷航”系列和“好太太”衍生产品“美好的战斗”等。

该服务还将继续传输体育赛事，如NFL比赛和其他联盟的比赛，如NCAA和PGA，以及来自 CBSN 和地方电视台的直播新闻。

去年，维亚康姆哥伦比亚广播公司还宣布了为新服务计划的其他原创节目，包括“The Offer”，这是一部关于“教父”制作的脚本有限系列片；中情局间谍剧《母狮》，由泰勒·谢里丹创作；VH1《音乐背后》的改编版，将聚焦于过去的40年；一个真实的犯罪纪录片，“真正的犯罪心理”，根据虚构的电视热播剧；还有贝特的《游戏》的复兴。

还会有更多的儿童节目，包括一个新的儿童原创系列“Kamp Koral”，来自Nickelodeon的“海绵宝宝”。它将成为“海绵宝宝电影:海绵在奔跑”的订阅视频点播中心

在全面更名之前，CBS All Access已经在扩大阵容，目标是通过整合ViacomCBS旗下品牌的内容，如BET、CBS、Comedy Central、MTV、Nickelodeon和Paramount Pictures，达到超过30，000集和电影。

尽管ViacomCBS计划将Paramount+打造成一个独立的目的地，但它也一直在将其内容授权给其他流媒体公司。在2020年作为一家新合并公司的第一个全年，ViacomCBS与康卡斯特、迪什、威瑞森(TechCrunch的母公司)、Nextstar、梅雷迪斯、考克斯和辛克莱达成了运输协议，并与YouTube TV和Hulu达成了增加收入的协议。结果， YouTube TV 和 Hulu 都在各自的节目中增加了六个ViacomCBS拥有的频道，并提高了价格。

因为Paramount+建立在CBS All Access的现有技术平台上，所以从第一天起，它将与它的前辈一样在平台(电视、网络和移动)上有相同的分布。

今天，CBS All Access估计约有800万订户，这使得它远远小于其他新的竞争对手，如迪士尼+(7300万以上)和HBO Max(1260万“激活用户”)。
