保镖是一款手机应用，在社交平台上隐藏有毒内容

如果你在各种社交网络上有点名气，你很可能会在回复或评论中接触到仇恨言论。法国初创公司保镖最近推出了英文版的应用和服务，这样它就可以隐藏有毒内容，不让你看到。它的法语版已经推出了几年，目前该公司已经吸引了 5 万名用户。

创始人兼首席执行官查尔斯·科恩告诉我:“我们已经开发出一种技术，可以检测互联网上的仇恨言论，准确率为 90%至 95%，误报率仅为 2%。

该公司已经推出了一款任何人都可以使用的手机应用。下载应用程序并将应用程序与您最喜欢的社交网络连接后，您可以选择适度的级别。有几个类别，如侮辱，身体羞辱，道德骚扰，性骚扰，种族主义和同性恋恐惧症。您可以为每个类别选择是低优先级还是高优先级。

之后就不用再打开 app 了。保镖扫描来自其服务器的回复和评论，并决定某件事是否正常。例如，它可以隐藏评论、屏蔽用户、阻止用户等。当你再次打开 Instagram 或者 Twitter 的时候，那些可恶的评论就好像从来没有存在过一样。

该应用目前支持 Twitter、YouTube、Instagram 和 Twitch。不幸的是，由于 API 的限制，它无法处理 Snapchat 和抖音上的内容。

在幕后，大多数审核服务严重依赖于机器学习或基于关键字的审核。保镖选择了不同的方式。它从算法上清理评论，并试图根据上下文分析评论的内容。它可以决定一个评论是否冒犯了你、第三方、一群人等等。

最近，这家初创公司推出了一款 B2B 产品。其他公司可以使用保镖驱动的 API，在其社交平台或自己的应用程序中实时调节评论。该公司使用传统的软件即服务方式向客户收费。
