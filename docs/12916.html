Cloudflare 为任何分发新冠肺炎疫苗的组织引入了免费的数字等候室

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/22/cloudflare-introduces-free-digital-waiting-rooms-for-any-organizations-distributing-covid-19-vaccines/

Cloudflare 为任何分发新冠肺炎疫苗的组织引入了免费的数字等候室

网络基础设施公司 Cloudflare 今天发布了一种新工具，旨在为全球负责推广新冠肺炎疫苗的卫生机构和组织提供一种方法，以维护公平、公正和透明的数字队列——完全免费。该公司的项目 Fair Shot initiative 将向任何有资格的组织免费提供其新的 Cloudflare 等候室，基本上为未来的疫苗接受者提供了一种注册和获得清晰和不断更新的视图的方式，以了解他们在哪里排队接受预防治疗。

Cloudflare 首席执行官 Matthew Prince 通过电子邮件解释说:“Cloudflare 奥斯汀办事处一名高管的妻子试图为她的父母注册新冠肺炎疫苗项目。“注册网站一直在崩溃。她对丈夫说:“为什么 Cloudflare 不建立一个排队功能来帮助疫苗网站？”碰巧的是，我们正在开发这样一个功能，并计划在二月初推出。"

在意识到迫切需要这种工具来帮助缓解当您试图尽快为全球人口接种疫苗以应对病毒威胁时出现的许多基础设施挑战后，Cloudflare 改变了他们的发布时间表，并为该项目投入了额外的资源。

“我们与团队讨论了提前推出等候室功能的计划，”普林斯补充道。“他们夜以继日地工作，因为他们认识到帮助疫苗接种的重要性。这些是真正推动我们团队的项目:当我们能够利用我们的技术专长和基础设施来解决问题，并产生广泛、积极的影响时。”

在技术方面，据该公司称，Cloudflare Waiting Room 易于实施，可以添加到任何建立在该公司现有内容交付网络上的注册网站，而无需任何工程或编码知识。该网站的访问者可以注册，并会收到确认他们在排队，然后会收到后续指引他们到一个注册页面的组织管理他们的疫苗时，轮到他们。进一步的配置选项允许等候室操作员为登记者提供等待时间估计，以及在轮到他们时提供额外的提醒(尽管该功能将在未来的更新中推出)。

正如 Prince 所提到的，Waiting Room 已经在 Cloudflare 的项目路线图上，实际上是为了其他高需求、有限供应的分配项目:想一想必备的音乐会门票，或者最新的热门运动鞋发布。但是 Fair Shot 项目将会把它完全免费提供给那些需要它的组织，而这本来是一个商业产品。感兴趣的人可以在 Cloudflare 的注册页面上注册，加入可用性的等待名单。

普林斯解释说:“通过公平注射项目，我们随时准备帮助确保每个有资格的人都能公平地获得新冠肺炎疫苗，我们和人类其他成员一起，期待着将这种疾病抛在身后。”
