随着在线结账大战继续吸引巨额投资，Fast 融资 1.02 亿美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/26/fast-raises-102m-as-the-online-checkout-wars-continue-to-attract-huge-investment/amp/

图片致谢:图片联盟/ Getty Images

今天早上 Fast ，一家提供在线结账和身份产品的初创公司，宣布它已经完成了 1.02 亿美元的 b 轮融资。新的融资活动由 Stripe 牵头，Stripe 是 Fast 的前投资者。

在线支付巨头 Stripe 去年也领导了 Fast 的首轮融资，这笔交易价值 2000 万美元。Fast 在一份新闻稿中说，迄今为止已经筹集了 1.24 亿美元。

TechCrunch 联系了 Fast，请其就其增长速度发表评论。该公司表示，其结账服务处理的商品总量(GMV)已经“每月增加两倍以上”，并补充说，它预计“这一趋势将继续增长”增长速度很难评估，因为我们缺乏一个扩大规模的基础，但我们现在对 Fast 未来的 GMV 发展有一个预期，可以用作衡量标准。

在 Fast 的超大 B 轮融资之前，许多竞争对手在线结账提供商也进行了大规模融资。

12 月下旬，提供在线结账、身份和支付服务的 Bolt为其 C 轮融资融资 7500 万美元。该公司还分享了一些增长指标，让 TechCrunch 能够掌握其当前的规模，以及对未来业绩的预期。

然后在 1 月中旬，Checkout.com 以 150 亿美元的估值筹集了 4 . 5 亿美元。TechCrunch 当时写道，“Checkout.com 希望为所有与支付相关的事情建立一站式商店，如接受交易、处理交易和检测欺诈。”因此，与 Bolt 类似，并与 Fast 提供的元素竞争。

最后，一天后，Rapyd 宣布以 25 亿美元的估值筹集了 3 亿美元。TechCrunch 指出，Rapyd 通过 API 提供金融科技服务，但由于它支持全球电子商务支付并出售反欺诈技术，它似乎适合这个群体。

加上 Fast 的新 B 轮融资，在过去一个月左右的时间里，我们已经看到 9.27 亿美元——至少——流入了与电子商务基础设施市场目标重叠的初创公司。自 Bolt 轮融资以来，这一数字每天不到 2600 万美元，这在短时间内是一笔巨大的资金。

这些公司是如何以如此快速的方式筹集资金的？这个问题最显而易见的答案是，电子商务如此庞大，对全球经济如此关键，以至于改善卖家和买家在线销售商品的体验，对许多参与者来说都是一个问题空间。在解决在线商务的竞赛中，如此多的初创公司都筹集了资金，这意味着到目前为止，它们都享受着强劲的增长率；这意味着他们都希望进入一个巨大的市场。

在新冠肺炎推动电子商务并普遍加速全球经济数字化的背景下，很难说这些技术将很快受到市场规模的限制。
