印度保留对抖音和其他58款中国应用的封禁

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/26/india-retains-ban-on-tiktok-uc-browser-and-57-other-chinese-apps/amp/

图片来源: TechCrunch

一位知情人士告诉TechCrunch，印度政府已经决定，抖音、UC浏览器、UC新闻、百度地图、小米的视频和社区以及印度在6月下旬禁止的其他53个中国应用程序不会很快回到印度。

上周，新德里告诉这些应用的母公司，迄今为止收到的回应并没有充分解决对他们提出的网络安全问题，该消息人士表示，由于通信是私人的，因此要求匿名。

消息人士称，由于这个原因，新德里告诉这些公司，它将保留对其应用程序的禁令，但不会完全关闭与这些公司的沟通渠道。印度媒体上周晚些时候报道说，印度是世界第二大互联网市场，拥有超过6亿互联网用户，该国将使禁令永久化。

从6月下旬开始，印度最终屏蔽了包括PUBG Mobile 在内的200个与中国有联系的应用，这是在去年两个邻国之间的地缘政治紧张局势下。印度信息产业部表示，所有这些应用都参与了对“印度国家安全和国防构成威胁的活动，最终会影响印度的主权和完整”。

到目前为止，新德里只就对6月下旬被禁应用的回应发送了反馈。

在这一批应用中，抖音是在印度被禁的最引人注目的应用。在被屏蔽之前，字节跳动的皇冠应用在印度拥有超过2亿用户。然而，尽管有禁令，该公司迄今仍保留了大部分印度员工。

字节跳动的一名消息人士告诉TechCrunch，该公司在印度运营着几处房产，包括一个名为Lark的生产力套件，该套件仍在印度运营，该团队继续开发这些应用程序。这一信息以前从未报道过。(UC浏览器也曾在印度非常受欢迎，尽管谷歌Chrome浏览器的日益流行终结了中国应用在该国的主导地位。)

尽管有禁令，抖音和几个被屏蔽的中国应用程序仍在中国维持着数百万用户，他们使用VPN等专门软件访问这些应用程序。根据移动洞察公司App Annie的数据，抖音上个月在印度拥有超过500万活跃用户，PUBG Mobile超过1500万，一位行业高管向TechCrunch分享了该数据。

抖音在一份声明中表示，正在审查印度的通知。“我们不断努力遵守当地法律法规，并尽最大努力解决政府可能存在的任何问题。确保我们所有用户的隐私和安全仍然是我们的首要任务，”一位发言人说。

对抖音的禁令——以及整个美国关于潜在封锁的戏剧性事件——并没有对字节跳动的财务状况产生太大影响。信息周二报道称，字节跳动去年的收入增长了一倍多，达到370亿美元，营业利润从2019年的40亿美元增加到70亿美元。

在过去的十年里，美国和中国公司纷纷涌向印度，寻找下一个十亿用户。但是这个南亚国家对这些公司的利润贡献很小。印度的连续企业家Kunal Shah在2018年的一次会议上表示，这个国家已经成为许多公司的“毛农场”。

不管怎样，自从被禁后，抖音和PUBG Mobile一直在探索在印度卷土重来的方法。抖音与印度最大的企业集团之一Reliance Industries 进行了早期投资谈判，PUBG Mobile 与游戏发行商腾讯断绝关系承诺在印度投资1亿美元。
