SaaS 创业工作室 eFounders 推出金融科技创业工作室

eFounders 正在通过创建第二个名为 Logic Founders 的创业工作室来扩大其关注点。这一次，Logic Founders 将专注于金融科技初创公司。卡米尔·泰恩(上图)将领导新工作室。

在过去的十年里，eFounders 推出了数十家软件即服务公司，试图改善我们的工作方式。投资组合公司包括前线、空中呼叫和 Spendesk 。

Camille Tyan 曾与人联合创立了 PayPlug，这是一家支付公司，后被 NAT ixis(BPCE 集团)收购。他计划遵循以新垂直市场为中心的 eFounders 模式。Logic 创始人会为新的创业公司出点子。它将招募两名联合创始人，并在公司最初的 12 到 18 个月里开始研发产品。

理想情况下，初创公司找到适合市场的产品，并在这个初始阶段后筹集种子资金。创业工作室持有创业公司的股份，但它会继续前进，以便专注于新项目。

如果你一直在密切关注 eFounders，那么这家初创工作室已经与几家金融科技公司合作过，比如 Spendesk、 Upflow 、 Multis 和 Swan 。新的金融科技项目可能会归入 Logic Founders 旗下。

该工作室表示，将推出 API 优先的金融产品。它正乘着嵌入式金融的潮流——许多人认为，金融产品将由平台分销，这些平台并非主要专注于金融，但可能受益于金融科技的功能。你可以期待从事支付协调、资产证券化、借贷 API、加密和 B2B 身份的公司。
