SAP 推出面向数字化转型的礼宾服务“与 SAP 一同崛起”

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/27/sap-launches-rise-with-sap-a-concierge-service-for-digital-transformation/

SAP 今天宣布了一项名为“与 SAP 一起崛起”的新产品，该解决方案旨在帮助该公司的客户完成各自的数字化转型，成为 SAP 所说的“智能企业”RISE 是一项订阅服务，结合了一系列服务和产品。

本周早些时候，当我与 SAP 产品成功负责人 Sven Denecken(以及 S/4Hana 的首席运营官)交谈时，他将其描述为“你可以为你的数字化转型获得的最佳礼宾服务”。“我们需要帮助我们的客户接受他们目前看到的变化，”他说。“转型是一个旅程。每个客户都想成为更聪明、更快、更灵活的企业，但是他们当然也看到了他们现在和未来面临的挑战。这种持续的转变正在企业中发生。我们从与他们的合作中确实知道，实际上他们同意这些基本原则。他们想成为一家智能企业。他们想要适应和改变。但关键问题是怎么去？他们问我们的关键问题是，请帮助我们到达那里。”

借助 RISE for SAP，企业将在 SAP 获得单一联系人，以帮助指导他们的旅程，同时还可以访问 SAP 合作伙伴生态系统。

Denecken 强调，这一过程的第一步不一定是引入新技术，尽管这也是它的一部分，而是帮助企业重新设计和优化他们的业务流程，并在其垂直领域实施最佳实践，然后衡量结果。“业务流程重新设计意味着您要分析您的业务流程是如何执行的。怎样才能得到量身定制的推荐？你如何能以行业标准为基准？这有助于你设定基调，也有助于激励你的员工——你的 IT 人员、你的业务人员——去适应，”Denecken 描述道。他还指出，为了使数字化转型项目取得成功，IT 和业务领导以及员工必须通力合作。

例如，在某种程度上，这包括提供技术和采用机器人流程自动化(RPA)。正如 Denecken 所强调的，所有这些都建立在 SAP 多年来为客户定义业务流程和 KPI 所做的工作之上。

在技术方面，SAP 显然正在提供自己的服务，包括其业务技术平台和云基础设施，但它也将支持所有大型云提供商的客户。RISE 还支持超过 2，200 个 API 来集成各种内部、云和非 SAP 系统，访问 SAP 的低代码和无代码功能，当然还有其数据库和分析产品。

“地缘政治紧张局势、环境挑战和持续的疫情迫使企业比以往任何时候都更快地应对变化，”SAP 首席执行官 Christian Klein 在今天的公告中表示。“能够快速调整业务流程的公司将会蓬勃发展，而 SAP 可以帮助他们实现这一目标。这就是《SAP 崛起》的全部内容:它帮助客户不断探索在云中运营业务的新方式，以保持行业领先地位。”

借助这一新产品，SAP 现在可以为其客户提供大量解决方案，而这些解决方案以前只能通过其合作伙伴生态系统获得。不过，Denecken 并不认为这是 SAP 在与自己的合作伙伴竞争。相反，他认为这在很大程度上是一种合作，这种新的解决方案可能只会给合作伙伴带来更多的客户。

“不用说，这是与那些合作伙伴的谈判，”他说。“因为是的，有时候我们现在接手的话题是他们(以前)接手的。但我们在这里寻求的是规模。市场对数字化转型的需求才刚刚开始。这就是我们与合作伙伴一起看到这绝对是一个大产品的地方。”
