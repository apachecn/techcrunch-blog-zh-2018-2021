这是特斯拉的 Model S 和 Model X 的新内饰

今年，S 型车的内饰将会有一些重大变化。在几个月的传言之后，特斯拉在定于今天下午晚些时候举行的季度收益电话会议之前发布的几张图片中证实了这些修改。

一些变化——比如转向宽屏显示器——是从 Model 3 发展而来的。其他的都是全新的。

(更新:特斯拉已经更新了其网站，更新了订购页面，表明 Model X SUV 也将获得以下版本！)

这是我们目前发现的:

一个飞机风格的转向“轭”(类似于特斯拉新跑车原型中发现的那种)，而不是标准的圆形车轮。

图片来源:特斯拉

前面的中央屏幕现在是 17 英寸的宽屏显示器，而不是 17 英寸的高/纵向显示器。同时，分辨率从 1900×1200 变为 2200×1300。

图片来源:特斯拉

除了方向盘上方的 12.3 英寸驾驶员显示屏，现在后部也有一个 8 英寸的显示屏(大概是为了让乘客更容易在后面玩汽车的内置游戏，如图所示。)

图片来源:特斯拉

特斯拉的收益电话会议定于太平洋时间下午 3:30 开始。如果他们提到任何关于内部的变化，我们会更新这个帖子。
