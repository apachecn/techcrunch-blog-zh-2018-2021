WhatsApp 在其网络和桌面版本中增加了生物认证功能

原文：https://web.archive.org/web/https://techcrunch.com/2021/01/28/whatsapp-is-adding-opt-in-biometrics-to-its-web-and-desktop-versions/

脸书旗下拥有超过 20 亿用户的流行通讯应用 WhatsApp 在最近几周宣布(随后推迟)改变其使用数据的方式后，受到了大量关注和流失用户。它还没有完成如何调整隐私和安全。现在，它正在为这项服务添加一项新的生物特征，为使用其网络和桌面版本的用户带来一个新的认证层。

该公司表示从今天开始，它将允许人们添加指纹、面部或虹膜扫描，通过将 WhatsApp 链接到你的移动应用程序，在桌面或网络上使用 WhatsApp，与现有的二维码认证一起使用。

这项功能是在 WhatsApp 描述为 Android 和 iOS 应用程序上 WhatsApp 网页的“视觉刷新”之后推出的，用于链接和管理连接到你账户的设备。该公司暗示，不久将会有更多的更新。

有了这项新功能，你现在可以在 Android 或 iPhone 手机上添加生物识别登录，使用指纹、面部 ID 或虹膜 ID(取决于设备)来添加第二层身份认证。只有在你的设备上启用了生物特征认证，它才起作用；如果你没有启用，它就不起作用。

当实现时，它将在桌面或网页版本与移动应用程序帐户关联之前出现在用户面前，目前移动应用程序帐户仅依赖于使用二维码。二维码不走了之；这是用户需要采取的第二步，类似于今天你可以选择在手机上实现两步认证来使用 WhatsApp 移动应用程序。

WhatsApp 表示，在 iPhone 上，它将与触控 ID 或 Face ID 操作 iOS 14 及以上版本的所有设备兼容，而在 Android 上，它将与任何兼容生物认证(面部解锁、指纹解锁或虹膜解锁)的设备兼容。如何打开的完整说明在这里。

这项服务是 WhatsApp 向前迈出的又一步，在旗舰移动应用程序之间创造了更多的功能对等，以及当你在其他地方使用这项服务时如何与它互动。

虽然 WhatsApp 最初是一款移动通讯应用，但多年来它一直在开发其他使用方式，例如在 2015 年为 iOS 版本增加了桌面支持。

WhatsApp 的大部分用户仍然是移动用户，但像全球健康大流行这样的事件将我们更多的人留在了家里，可能会导致其网络和原生桌面应用的用户激增，因此它在那里增加更多功能是有意义的。

WhatsApp 告诉 TechCrunch，它今年将增加更多功能，使桌面和移动体验更加紧密。差距还是很大的。比如你不能在 WhatsApp 网页版上打电话。(这个功能可能很快就会出现:上个月，它开始在 beta 测试中被发现。)

生物识别的下一步是什么？

WhatsApp 最近宣布(并撤回)其与脸书之间的数据共享变化，这让很多人对该公司的意图感到不安。

这并不奇怪。这是一个特别敏感的问题，因为短信一直被认为是一个非常私人的空间，有时是私人空间，被视为与人们在更开放的社交网络平台上所做的事情相分离。

当然，随着时间的推移，这种观点已经被数据泄露、群发信息滥用和隐私条款的改变所侵蚀。

所有这一切意味着，可能会有很多人怀疑脸书在这里的意图。

WhatsApp 非常明确地指出，它无法访问你将存储在设备中的生物信息，它使用的标准生物认证 API 与其他安全应用程序(如银行应用程序)使用的相同。

但银行业的应用程序 parallel 在这里值得注意，也许是一个值得思考的问题。想想该公司是如何在 WhatsApp 中加入更多特性和功能的，包括支付商品和服务的能力，以及在印度等市场提供保险和养老金产品的测试。

是的，这项新的生物识别功能今天推出，旨在为人们创建一种更安全的方式来链接不同设备的应用程序。但是，为了功能对等，在未来，随着其他功能也在移动设备之外推出，生物识别技术将如何以及是否会出现，这将是一件有趣的事情。

更新:文本已更正，以反映新的外观将出现在移动应用程序(而不是网络)上，旨在改善如何管理联网设备。
