DesignCrowd筹集了1000万澳元来发展其DIY平台BrandCrowd

原文：https://web.archive.org/web/https://techcrunch.com/2021/02/01/designcrowd-raises-10-million-aud-to-grow-its-diy-platform-brandcrowd/

DesignCrowd 今天宣布，它已经筹集了1000万澳元(约76亿美元)的上市前资金。这笔资金将用于招聘和产品开发，目标是加速其DIY平台 BrandCrowd 的增长。

新的资金来源于DesignCrowd准备在澳大利亚证券交易所进行首次公开募股。本轮投资者包括Perennial Value Management、Alium Capital、Ellerston Capital、Regal Funds Management和CVC，以及回归的支持者海星风险投资公司和AirTree Ventures。DesignCrowd现在已经筹集了超过2200万澳元的资金。

DesignCrowd成立于2007年，总部位于澳大利亚悉尼，建立了设计众包平台的声誉，允许用户从世界各地的设计师那里获得建议。BrandCrowd的推出是为了补充DesignCrowd的众包/市场模式，扩大其潜在用户群，并将其与人们用来寻找设计师的其他网站(如99designs和Fiverr)区分开来。

联合创始人兼首席执行官亚历克·林奇(Alec Lynch)表示，虽然还有其他针对企业家和小品牌的DIY标志制造商，包括Design Hill、Canva和Tailor Brands的工具，但BrandCrowd从一开始就有优势，因为它已经通过DesignCrowd接触到了超过80万名设计师，使该公司能够为其图书馆找到世界各地最好的标志设计师。BrandCrowd更喜欢在发布之前预先购买设计，因为所有的徽标都是该平台独有的(用户可以支付额外的费用从其库中删除徽标)。

BrandCrowd的客户只需支付一次性费用就可以下载一个徽标，并且可以按月或按年订阅。林奇说，许多人同时使用这两个平台。

“例如，如果一家小企业想从DesignCrowd上的设计师那里获得一个定制的标志设计，我们就允许他们在BrandCrowd上的DIY设计工具中使用该标志，以制作他们可能需要的任何东西，从名片设计到Instagram帖子和电子邮件签名，”林奇说。"他们甚至可以使用我们的徽标编辑器工具在BrandCrowd上修改他们的徽标."

DesignCrowd在2020年的净收入同比增长54%，这主要归功于BrandCrowd。该公司表示，BrandCrowd在过去12个月中有超过500万人注册，其中超过一半的收入来自美国。

林奇说，在新冠肺炎疫情期间，该公司在2020年3月和4月经历了一些逆风，但随后全球对在线设计的需求反弹并开始增加。

“我们的假设是，疫情导致更多的人在2020年下半年创办新企业，更多的人需要为这些企业设计，这对我们很有帮助，”他补充道。“除了小企业的这种小‘繁荣’之外，我们认为疫情很可能加速了企业在线而非线下采购设计的现有趋势。”
