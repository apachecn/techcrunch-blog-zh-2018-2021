Adobe 扩展 Acrobat Web，添加 PDF 文本和图像编辑

原文：https://web.archive.org/web/https://techcrunch.com/2021/02/02/adobe-expands-acrobat-web-adds-pdf-text-and-image-editing/

很长一段时间，Acrobat 是 Adobe 的旗舰桌面应用程序，用于处理——尤其是编辑——pdf。近年来，该公司在网上推出了 Acrobat，但它的功能从未像桌面版那样全面，许多用户正在寻找的一项功能，即编辑 pdf 格式的文本和图像，仍然是桌面版的独有功能。这种情况正在改变。随着 Acrobat 在网上的最新更新，Adobe 将这种能力带到了它的在线服务中。

“[Acrobat Web]对我们来说具有战略重要性，因为我们有越来越多的人在浏览器中工作，”Adobe 文档云副总裁 Todd Gerber 告诉我。“他们的一天从登录 G Suite 或 Microsoft Office 365 开始。所以我们希望出现在人们工作的所有地方。”该团队首先推出了创建和转换 pdf 的功能，但正如 Gerber 指出的那样，它需要一段时间才能以高性能和实时的方式编辑 pdf。“我们本可以更早完成，但它达不到快速、灵活和高质量的标准。”他特别指出，使用字体是团队在将这一功能上线时面临的最困难的问题之一。

他还指出，尽管我们倾向于认为 PDF 是一种 Adobe 格式，但它是一种开放的标准，许多第三方工具都可以创建 PDF。这个庞大的生态系统，以及不同实现之间的潜在差异，也使得为 Adobe 提供编辑功能变得更加困难。

随着今天的发布，Adobe 还引入了几个额外的基于浏览器的功能:保护 pdf，将它们分成两个，以及合并多个 pdf。此外，在去年与谷歌合作，利用。新领域，Adobe 现在推出了一系列新的快捷方式，如 PDF.new 。该公司计划在接下来的一年里推出更多这样的产品。

Adobe 表示，该公司现有的快捷方式总共被点击了约 1000 万次，这正好表明每天有多少人试图转换或签署 pdf 文件。

正如 Gerber 指出的，许多潜在用户不一定首先想到 Acrobat。相反，他们想做的是压缩 PDF 或转换它。Acrobat Web 和。他认为，新的域名有助于公司为平台带来新的受众。“这为我们打开了一个新的受众群体，他们最初并没有想到 Adobe。他们考虑 pdf，他们考虑他们需要用它们做什么，”他说。“因此，通过与客户寻求发现并最终交易的方式相关联，我们可以扩大我们的客户群。我们的 Acrobat web 之旅实际上始于这样一个理念:让我们追求非品牌搜索。”

当然，Adobe 将用户明确寻找 Acrobat 的所有品牌搜索汇集到 Acrobat 桌面应用程序，但对于更普通的用户，它将他们带到 Acrobat Web，在那里他们可以轻松地执行他们想要的任何操作，甚至无需注册该服务。
