今年晚些时候，杰夫·贝索斯将不再担任亚马逊首席执行官

原文：https://web.archive.org/web/https://techcrunch.com/2021/02/02/jeff-bezos-will-no-longer-be-ceo-of-amazon-as-of-later-this-year/?utm_medium=TCnewsletter&tpcc=TCdailynewsletter

亚马逊创始人兼现任首席执行官杰夫·贝索斯将在今年第三季度的某个时候担任公司的执行主席，现任AWS首席执行官安迪·杰西(Andy Jassy)将接管这家商业公司的最高行政职位。亚马逊在周二公布财报的同时宣布了这一消息。

亚马逊最初在盘后上涨，因为市场消化了该公司的收益和首席执行官的消息。该公司的每股收益和收入都超过了预期。这使得很难理清市场对其一系列繁忙公告的反应。更新:由于投资者有更多的时间来分析公司的全部公告，亚马逊的股票现在已经跌入负值区域。

亚马逊在2020年第四季度粉碎了的每股收益和营收预期。因此，任何担心贝佐斯退出首席执行官职位的投资者都得到了某种程度的基于业绩的改善。亚马逊这个季度是第一个突破1000亿美元大关的，带来了1256亿美元的收入，而预期为1197亿美元。此外，该公司每股14.09美元的收益几乎是预期的7.23美元的两倍。

此前，Jassy曾被认为是贝佐斯的可能继任者，此前他曾领导亚马逊网络服务公司(Amazon Web Services)取得了成功，目前该公司在云计算领域享有领先地位。AWS本季度收入增长28%，低于去年同期的34%。AWS的净收入从去年第四季度的99.5美元增长到2020年第四季度的127.4亿美元。AWS的营业收入也有所增长，从2019年第四季度的26亿美元增长到最近一个季度的35.6亿美元。

值得注意的是，微软的Azure业务在最近的收益期增长了50%。

贝佐斯向亚马逊员工发送了一封电子邮件，该公司也于周二在其博客上公开发布了这封邮件。在信中，他说，虽然他继续“觉得[他的]工作有意义、有趣”，但他希望能够把适当的时间和注意力放在他的“第一天基金、贝佐斯地球基金、蓝色起源、华盛顿邮报和[他的]其他爱好上。”

贝佐斯的太空公司蓝色起源(Blue Origin)迄今已经取得了很多成就，并定期利用其新谢泼德运载火箭执行亚轨道任务。对于航天公司来说，接下来的一年将是忙碌的一年，因为它很可能将首次开始在新谢泼德载人飞行。

除了自己的载人航天目标，蓝色起源目前正在与其他航天行业的领导者合作，为美国宇航局开发载人登月系统。它还在研究另一种运载火箭New Glenn，这将是一种能够将有效载荷送入轨道的重型火箭。所以那里也发生了很多事情。贝佐斯此前曾将蓝色起源称为“他正在做的最重要的工作”，因为它对人类未来的潜在影响非常大。
