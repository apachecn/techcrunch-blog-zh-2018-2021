塞娜正在为法国市场建立一个保险即服务的创业公司

法国初创公司 Seyna 迎来了新的首席执行官。 Stephen Leguillon 将作为首席执行官加入公司，而 Philippe Mangematin 由于个人原因将退出日常活动，他将成为名誉主席。

如果你不熟悉这家公司，Seyna 是一家保险科技初创公司，已经获得了专注于财产和意外伤害的保险许可证。这是一个重要的举措，因为法国监管机构(ACPR)自 1983 年以来就没有在这一领域颁发过新的许可证。

该公司的走向市场战略也很有趣，因为它不直接向消费者销售保险产品。相反，该公司正在打造保险即服务产品。它与其他公司合作，以自己的品牌提供 Seyna 的保险产品。

在幕后，Seyna 提供了一个 API 来按需生成保险合同——API 是一个编程接口，让两个服务相互交互。您还可以直接连接到 Seyna 的界面来管理合同。

Seyna 创建了自己的核心保险系统，这是一个重要的区别于传统球员的因素。这家初创公司可以产生许多不同的变体，涵盖大约 20 种不同的保险产品——宠物保险、机票取消、房租担保等。客户包括 Garantme 和迪卡侬。

该公司的新任首席执行官斯蒂芬·勒吉伦(Stephen Leguillon)曾联合创立了 La Belle Assiette，这家公司让你可以聘请一名厨师在家做饭，我在 TechCrunch 上报道过。他增加了第二个产品，名为 GoCater，一个企业餐饮服务公司——go cater 是 La Belle Assiette 的子公司，被 ezCater 收购。

Seyna 已经从 Global Founders Capital、Allianz France、Financière Saint James 和一些商业天使那里筹集了 1400 万€(按今天的汇率计算为 1700 万美元)的种子资金。保险公司需要大量的资金来运营，这就是为什么法国市场的种子期看起来很大。
