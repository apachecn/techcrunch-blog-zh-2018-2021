Twitter 悄然推出时事通讯快捷方式 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/02/02/twitter-quietly-rolls-out-revue-newsletters-integration-on-web/?utm_campaign=Coffee%20Marketing&utm_medium=email&utm_source=Revue%20newsletter

如果你渴望在上个月收购荷兰时事通讯平台 Revue后，通过 Twitter 发布电子邮件时事通讯，你将不得不等待一段更长的时间，等待一个功能全面的整合。但是 Twitter 已经悄悄地在它的网络应用上增加了一个时事通讯的链接——将它的社交网络用户引向发布平台。

该公司告诉我们，这两个平台目前仍然是分开的——这意味着那些想注册 Twitter 拥有的 Revue 时事通讯的人仍然可以创建一个单独的(Revue)账户来这样做(尽管 Revue 也提供通过 Twitter 登录的选项)。因此，这是迈向未来更深层次整合的第一小步。

Revue 出版商级付费时事通讯的新注册也暂时处于暂停状态。

T witter 证实，目前它已将 Revue 的出版商报价恢复为私人测试版，同时它专注于投资平台基础设施，为预期的未来需求铺平道路。

付费层将提供更多高级功能，如管理多个时事通讯和定制主题的能力。

Twitter 无法提供该层简讯何时再次对新注册用户开放的时间表。但是它说仍然有可能通过 Revue 的平台在免费时事通讯层下创建付费时事通讯。

它已经做出的一个改变是降低付费时事通讯的费用——降到 5%。

它还告诉我们，它合并了 Revue 在收购之前提供的原始免费和专业层，这意味着免费 Revue 层现在功能更加全面。

一些眼尖的用户，包括我们的主编马修·潘扎里诺，早些时候发现了在 Twitter 的网络应用程序的“更多”标签中增加了一个“时事通讯”的链接。

如果你点击时事通讯链接，就会弹出一个窗口，鼓励 Twitter 用户创建一个免费的时事通讯来联系他们的受众，并标记一些功能，如嵌入推文、导入电子邮件列表、分析参与度以及从订户那里赚钱。

一个大的“了解更多”按钮直接链接到 Revue 的平台。

图片来源: TechCrunch

除非你已经进行了很长时间的数字戒毒，否则你会注意到时事通讯最近很流行。帮助作者通过电子邮件交付格式将其内容货币化的平台的兴起，给 Twitter 带来了明显的风险/机会，Twitter 是一家依赖大量用户免费公开贡献简短内容的公司。

但随着 Twitter 推出自己的内部简讯，推文的价值可能会膨胀——成为 Twitter 用户建立粉丝社区的漏斗顶端，他们可以将这些粉丝转化为以简讯形式提供的较长内容的付费订户。

Twitter 告诉我们，它已经暂停了付费 Revue 层的新注册，因此它可以专注于改善基础设施，以有效满足新客户的需求，并使 Revue 成为出版商更强大的工具。

它补充说,正在迅速推出新版本的新闻通讯工具的付费出版商层，向新客户开放，并指向一个邮件列表，供感兴趣的人注册以获得最新更新(此处为)。

上个月我们报道了 Twitter 正致力于将时事通讯整合到它的平台上，所以它似乎正在快速地加入时事通讯，并利用围绕付费时事通讯的炒作和势头。不过，目前它优先考虑基础设施投资，而不是任何更深入的整合——并告诉我们，它仍在集思广益，如何在实践中发挥作用。所以听起来还有一段路要走。

阻止 Twitter 用户在其他地方找到他们更长形式的想法可能是 Twitter 首先考虑的问题——随着社交领域关注度的竞争加剧，例如来自(竞争对手)简讯平台 Substack (自 2017 以来一直在积聚势头)的竞争；还有时髦的实时语音聊天应用 Clubhouse (最近吸引了喜欢小团体的 technorati )。

在侧边栏的三点“更多”菜单选项下可以找到 Twitter 网络应用程序的当前集成——它会弹出“了解更多”的提示。

Twitter 指出，它在那里标记的功能——例如拖放推文以嵌入它们的能力——已经由 Revue 提供。

Revue newsletter 平台与 Andreessen Horowitz 支持的 Substack 竞争，该平台为许多第三方服务提供集成支持，包括脸书、Instagram、Pocket、产品搜索、Instapaper 甚至 RSS 订阅。一旦用户集成了这些服务，他们就可以将故事从这些提要拖放到时事通讯中。

在当前的头像中，用户可以手动添加订户的电子邮件地址，或者从 Mailchimp 或 CSV 文件中导入列表。在欢迎屏幕上，注意到逆向工程师简·满春·黄上周第一次发现，Twitter 还宣传其时事通讯可以帮助作家和出版商从付费订户那里赚钱。

Revue 的平台还允许用户安排他们的时事通讯。

Twitter 将会给这个平台带来什么样的额外功能还有待观察。

在我们收到来自 Twitter 的额外信息后，该报告进行了大量更正，以澄清 web 应用程序集成目前仅扩展到在标签中显示快捷方式。这两个平台仍然是分离的——因此任何更深入的整合都还没有到来。
