云基础设施市场在 2020 年达到 1290 亿美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/02/04/the-cloud-infrastructure-market-hit-129b-in-2020/

2020 年的云基础设施市场反映了社会本身，最富有的公司越来越富有，而市场底层的公司越来越穷。根据 Synergy Research Group 的数据，这一年增长到 1290 亿美元。这高于 2019 年的约 970 亿美元。

Synergy 还报告称，第四季度云基础设施市场规模达到 370 亿美元，高于第三季度的 330 亿美元，同比增长 35%。

在过去的九个月里，我从世界上每一位创始人那里听说，疫情正在加速数字化转型，其中很大一部分是加速向云的转移。这些数字似乎证明了这一点。

像往常一样，三大巨头是亚马逊、微软和谷歌，阿里巴巴现在牢牢占据第四位，而 IBM 滑落到第五位。但微软比竞争对手亚马逊增长更快，在 2020 年底首次达到 20%的市场份额。请记住，这家总部位于雷蒙德的软件巨头自 2017 年以来，其份额已经翻了一番。这是非常快速的增长。与此同时，谷歌和阿里巴巴分别获得了 9%和 6%的份额。

亚马逊是一个有趣的例子，因为它在连续四年的 Synergy 数据中稳定在 33%左右，但因为它在一个不断增长的市场中占三分之一的份额，这意味着随着类别本身的扩展，它一直在增加其公共云收入。

亚马逊在第四季度的 AWS 收入为 127.4 亿美元，首次超过 500 亿美元。这高于上一季度的 116 亿美元。虽然微软的数字总是很难从其收益报告中解析出来，按照 370 亿美元的 20%计算，它上季度的收入为 74 亿美元，高于上个季度的 59 亿美元。

谷歌的收入为 33 亿美元，高于 2020 年第三季度的 29.8 亿美元，阿里巴巴的收入为 22.2 亿美元，高于同期的 16.5 亿美元。

Synergy 的首席分析师约翰·丁斯代尔(John Dinsdale)表示，领导者在这一点上非常坚定，拥有巨大的绝对市场数量，云提供商之间也存在巨大的差距。“10 多年来，AWS 一直是一个巨大的成功故事，尽管来自众多强大 it 行业公司的竞争日益激烈，但它仍保持着极其强大的市场地位。这对亚马逊和 AWS 领导团队来说都是一个很好的证明，你不得不怀疑这不会随着新的制度而改变。

他认为微软是一个有价值的竞争对手，但在某个时候肯定会遇到增长瓶颈。“微软继续缩小与亚马逊之间的差距当然是可行的，但微软 Azure 越大，就越难保持真正的高增长率。那只是大数定律。”

与此同时，云基础架构领域底层的市场份额持续下降，尽管相关金额持续大幅增加。Synergy 在一份声明中写道:“市场份额的输家是一大群较小的云提供商，他们在过去 16 个季度中共失去了 13 个百分点的市场份额。”。

然而，丁斯代尔说，这些玩家并没有失去一切。“对于较小的参与者(或只有很小市场份额的大公司)，他们可以专注于特定的利基市场(可以基于地理位置、服务类型或客户垂直行业)，也可以尝试向广泛的客户提供广泛的云服务。从事前者的公司可以做得相当好，而从事后者的公司会发现非常困难，”丁斯代尔告诉我。

在某种程度上，这些数字变得如此之大，以至于几乎不再有意义，但尽管公共云收入数字变得如此之大，它们在全球 IT 总支出中所占的比例仍然相对较小。根据 Gartner 的估计，2020 年全球 IT 支出为 3.6 万亿美元(带 T)。这意味着到 2020 年，云基础设施市场仅占总支出的 3.85%。

请想一想:目前只有不到 4%的 IT 支出用于云基础设施，这为未来几年的增长和数十亿美元的增长留下了巨大的空间。

如果有人能进来扰乱领导者，这肯定会变得更有趣，但至少就目前而言，他们很难被推开，除非我们对计算的看法发生了不可预见的戏剧性变化。
