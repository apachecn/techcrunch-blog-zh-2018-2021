WordPress 母公司 Automattic 收购 Parse.ly

原文：https://web.archive.org/web/https://techcrunch.com/2021/02/08/automattic-acquires-analytics-company-parse-ly/amp/

图片来源: WPVIP

与开源网络出版平台 WordPress 相关的盈利公司 automatic 宣布，它已经收购了分析提供商 Parse.ly 。

具体来说，Parse.ly 现在是 WPVIP 的一部分，WPVIP 是 Automattic 内部的一个组织，为包括 TechCrunch 在内的出版商提供企业托管和支持。(我们也用 Parse.ly。)

WPVIP 首席执行官尼克·格纳特(Nick Gernert)将此描述为该组织的第一次大型企业软件收购，反映了一种超越新闻和媒体组织的战略——像 Salesforce(其风险部门早在 2019 年就向 automatic投资了 3 亿美元)、NBA、康泰纳仕、脸书和微软这样的企业现在都使用 WPVIP 来满足其内容和营销需求。

Gernert 说，两家公司有着相似的背景，都有数字出版的“根基”和“非常注重理解内容的影响”

“我们真的开始更多地转向内容营销，并开始更深入地思考传统页面分析提供的内容以外的内容，”他继续说道。这意味着要做的不仅仅是衡量网站的浏览量和时间，而且“真正开始更深入地审视谈话、归属、领域等从营销人员的角度来看具有影响力的东西。”

WordPress 和 Parse.ly 已经合作得很好了，但是他们的计划是让 WPVIP 的用户可以使用这些功能，同时也让 WPVIP 的发布者可以使用更多的 Parse.ly 数据。Gernert 表示，还有机会向 Parse.ly 添加更多与商务相关的数据，因为 Automattic 还拥有 WooCommerce。

他说，目标是“让 Parse.ly 更适合 WordPress，更适合 WPVIP。”

与此同时，他补充道，“我们没有计划让 Parse.ly 成为我们平台上唯一的分析解决方案。我们希望保留[WordPress]的灵活性和互操作性，我们希望从语法的角度确保它仍然作为一个独立的产品存在。这是其未来的关键，我们将继续投资于此。”

据 Crunchbase 报道，Parse.ly 成立于 2009 年，已经从包括 Grotech Ventures 和 Blum Berg Capital在内的投资者那里筹集了 1290 万美元的资金。Parse.ly 创始人 Sachin Kamdar 和 Andrew Montalenti 将加入 WPVIP，由 Kamdar 领导 Parse.ly 的上市战略，Montalenti 领导产品。

Kamdar 在一份声明中说:“我们一直对 WPVIP 作为企业内容团队黄金标准的市场地位深表钦佩，我们很高兴能够一起加入。”“从文化和人员，到产品、市场和愿景，我们步调一致，为客户创造更多价值。这种内容和智能的强大结合将推动行业加速前进。”

收购的财务条款没有披露。
