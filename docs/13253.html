AWS、微软、Mozilla和其他公司成立了Rust基金会

原文：https://web.archive.org/web/https://techcrunch.com/2021/02/08/the-rust-programming-language-finds-a-new-home-in-a-non-profit-foundation/

Rust编程语言在一个非盈利基金会找到了新家

Rust——编程语言，而不是生存游戏——现在有了新家: Rust基金会。AWS、华为、谷歌、微软和Mozilla今天联手推出了这个新的基金会，并承诺在两年内投入100万美元的预算。该预算将允许该项目“开发服务、程序和事件，以支持Rust项目维护者构建尽可能好的Rust。”

Rust最初是Mozilla内部的一个辅助项目，用来开发C/C++的替代品。Rust由Mozilla Research的Graydon Hore设计，并得到了JavaScript创作者Brendan Eich等人的贡献，它成为了Firefox浏览器及其Gecko引擎以及Mozilla伺服引擎的一些基本功能的核心语言。今天，Rust是开发人员中最受欢迎的语言。但随着Mozilla最近几个月的裁员，Rust团队中的许多人失去了工作，该语言的未来在没有主要赞助商的情况下变得不明朗，尽管该项目本身有数千名贡献者和大量企业用户，因此该语言本身不会有任何发展。

一个大型开源项目通常需要某种指导，新基金会将提供这种指导，并且需要一个法律实体来管理社区的各个方面，例如，包括商标。新的Rust董事会将包括来自五个创始成员的五名董事会董事，以及来自项目领导层的五名董事。

Mozilla和Rust基金会董事会成员Bobby Holley在一份声明中写道:“Mozilla培育Rust是为了打造更好的Firefox，并为更好的互联网做出贡献。”。“在有了 Rust 基金会的新家， Rust 将有空间发展自己的成功，同时继续扩大Mozilla与 Rust 社区共享的一些核心价值观。”

所有的企业赞助商都对Rust感兴趣，并使用它来构建(和重建)他们的一些堆栈的核心部分。例如，谷歌最近表示，它将资助一个基于Rust的项目，该项目旨在使Apache服务器更加安全，而微软最近也成立了一个Rust团队，并正在使用该语言重写一些核心的Windows APIs。AWS最近发布了 Bottlerocket，这是一个新的Linux容器发行版，例如，它的特色是主要用Rust编写的构建系统。
