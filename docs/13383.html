如果我们不小心的话，这个专栏的每个条目都可能包含特殊目的收购公司新闻。

特殊目的收购公司，或空白支票公司，无论你喜欢叫它们什么，今天都是巨大的生意。但是它们不是唯一正在发生的事情，我们很快会谈到其他事情。这是对我两天内写了两次关于 SPACs 的道歉。

昨天，我们讨论了风险投资领导的特殊目的收购公司的崛起，以及提供种子资金的风险投资集团是否会在市场上比专注于任何特定创业阶段的公司更有优势。坚持空白支票的主题，今天上午我们来看两个特殊目的收购公司主导的交易，即涉及罗孚和钱利来的交易。

该交易所探索初创公司、市场和资金。每天早上在 Extra Crunch 上阅读，或者每周六获取交易所简讯。T11】

我们加倍努力防止更多与特殊目的收购公司有关的帖子。我们选择 Rover 是因为另一家宠物主题实体 Chewy 已经上市。由于两家公司都有风投支持，我们或许可以对比它们上市后的交易表现。可悲的是，Chewy 专注于宠物电子商务，而 Rover 更专注于宠物服务，但他们可能会证明足够接近一些松散的比较。

为什么和钱利来聊天？因为这是一家风险投资支持的金融科技初创公司，一家 TechCrunch 广泛报道的公司。如果它在特殊目的收购公司的帮助下顺利进入公开市场，它可能会为无数其他尚未上市的金融科技公司铺平道路。

所以这是一份特殊目的收购公司邮报，但由于我们将主要关注两家公司的财务健康状况，这两家公司我们久闻其名，却从未见过其内部，我仍然希望你们能加入我的行列。

我们从 Rover 的投资者演示开始，然后转到 MoneyLion 的演示。

漫游者正在与合并，后者隶属于真风资本。这笔交易预计罗孚的市值约为 16 亿美元，账面现金约为 3 亿美元。

那么，这只新独角兽到底有多大吸引力呢？如果你想边看边看，你可以在这里找到它的投资者平台。

首先，该公司强调，由于疫情和宠物拥有量不断增长的事实，去年数字服务的使用不断增加。两者都是真的。我们已经看到两家公司和的消费者都在加速数字化转型。如果你最近试图收养一只宠物，你会发现很少有人在等待永久的家。

有了这些东西，你可能会想知道为什么罗孚也在追求特殊目的收购公司领导的首次亮相。如果它的市场炙手可热，而且之前已经筹集了风险资本，为什么不通过 IPO 上市呢？因为 2020 年对公司来说很艰难。

收入从 2019 年的 9500 万美元降至去年的 4800 万美元。同期预订量从 420 万下降到 240 万，导致预订总值从 2019 年的 4.36 亿美元下降到 2020 年的 2.33 亿美元。为什么？因为每个人都被困在家里。带着他们的宠物。这种情况限制了对流动宠物服务的需求。
