Onboard 表示，它可以帮助 SaaS 公司更快更好地提升新客户的速度 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/02/16/onboard-says-it-can-help-saas-companies-bring-new-their-customers-up-to-speed-faster-and-better/

虽然公司越来越积极地接受软件即服务公司的产品，但让这些新产品从一开始就以无缝的方式工作并不容易，一些企业客户在数字墨水变干后就感觉被遗忘了。

加入吧，这是一家成立仅 10 个月的初创公司，旨在帮助 SaaS 企业取悦这些新客户，而不是将他们拒之门外。

该公司由首席执行官杰夫·爱泼斯坦(Jeff Epstein)联合创立，他此前推出了推荐营销和联盟营销软件公司 Ambassador，该公司在成立 8 年后的 2018 年出售。

出售给 West Corporation——现在的 Intrado——的条款从未披露，但 Epstein 说这对股东来说是一个“好结果”。(上个月，大使又被卖给了西雅图的一家小公司。)

至于板载是如何工作的，Epstein 让这个过程听起来很简单。“你决定你的客户群的变量，因为不同的计划类型可能意味着公司需要做不同的事情。”(例如，他们可以使用 API 或一些代码片段。)之后，Onboard 与 SaaS 公司合作，创建一个全球任务列表，其中包含从销售过程中收集到的需求，并帮助它创建一种动态的下拉任务列表，其中包含受托人、到期日期、警报和通知。

这在很大程度上是一种自助产品，最终使责任更加透明，尽管 Epstein 将入职过程描述为他的公司和客户之间的“共同责任”。他还表示，他的初创公司已经在努力构建一个更复杂的通知层，它具有自动推送功能，既有用又不讨厌。

这家五人公司目前没有向其数十名测试版客户收费。爱泼斯坦说，它希望在进入盈利模式之前把产品做好。该计划最终将向它所追逐的客户类型——中型公司——收取每月数百美元的费用，外加每人每月的费用。(“我们不打算以任何方式成为企业 y，”爱泼斯坦在谈到公司避免长期合同的计划时说。)

船上不是没有竞争对手。相反，在这个充满问题的企业领域，涌现出了许多新贵。爱泼斯坦的联合创始人之一威廉·史蒂文森(William Stevenson)让他注意到，对于许多新客户来说，这是一个令人恼火的时期。他担任了四年的客户成功大使副总裁，与其他公司中与他职位相同的许多人一样，他试图用不太理想的产品凑合，有时来自周一或 Asana 或 Basecamp 或 Google Docs。

勒德洛风险投资公司的黄邦贤·特里斯特说他很清楚这个问题，他的公司在夏末悄悄领导了一轮 125 万美元的种子轮投资，参与投资的还有泽尔科瓦风险投资公司和底特律风险投资公司。

“一次又一次，在我们的投资组合中，特别是在 B2B SaaS 销售中，”勒德洛的投资组合公司一直“被迫拼凑解决方案或使用不是为他们制造的工具，”特里斯特说。

问题是，Onboard 能否比其他竞争对手更快地站稳脚跟，不出所料，Epstein 认为他的团队具备起步所需的条件。(第三位创始人马特·马耶斯基(Matt Majewski)最近离开了 Ambassador，以帮助公司获得动力。)

他说，爱泼斯坦的简历也有帮助。作为底特律出售公司的创始人，他为当地投资者所知，甚至更多。(“我们能够成为小池塘里的大鱼，”他说。)

爱泼斯坦还表示，投资者意识到“这一领域普遍存在机会”，并补充道，“(风险公司的)合伙人一直在打电话——而不是合伙人——在某些情况下，他们是通过 LinkedIn 上的第三方关系来的。”

爱泼斯坦说，他过去“显然筹集了一些资金”，但他以前从未见过这样的事情。“这很奇怪，”他补充道，“但很酷。”
