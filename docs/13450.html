在申请首次公开募股几天后，Fintech Marqeta 开始涉足信用卡领域

原文：https://web.archive.org/web/https://techcrunch.com/2021/02/18/fintech-marqeta-expands-into-credit-card-space/

Marqeta 正在向消费信用卡领域扩张，以帮助其他品牌推出信用卡计划。

据报道，该支付卡发行公司秘密申请首次公开募股，这使得该公司成为最新一家进入公开市场的 fintech。

据路透社报道，首次公开募股的价值预计在 100 亿美元左右。据报道，正在与高盛和摩根大通合作发行的马凯塔希望在 4 月前完成首次公开募股。

总部位于加利福尼亚州奥克兰的 T4 马凯塔公司去年 5 月以 42 亿美元的估值筹集了 1.5 亿美元，此前 TechCrunch 曾报道称。然后在 10 月份，万事达卡在马凯塔投入了一笔数额不详的资金。

该公司为各种金融服务平台提供工具，以提供卡、钱包和其他支付机制，其客户包括 Cash App、assert、DoorDash 和 Instacart。Marqeta 表示，截至 2020 年底，该公司已通过其平台发行了 2.7 亿张银行卡，高于 2019 年底的 1.4 亿张。该公司有 550 多名员工，分布在 35 个国家。

现在，Marqeta 正与另一家初创公司 with 合作推出新的信用卡计划。

正如 当之无愧的首席执行官卡尔佩什·卡帕迪亚(Kalpesh Kapadia)所解释的，他的公司的技术和开放的 API 平台将为 Marqeta 的项目管理服务提供动力，包括发起 、核保、银监一体化、客户服务、合规和风险管理。

马凯塔创始人兼首席执行官贾森·加德纳(Jason Gardner)将马凯塔扩展至开发新的信贷产品描述为该公司在构建一个能够支持任何卡类型的“真正全面的发卡平台”方面的一个“重大里程碑”。

他告诉 TechCrunch:“这项技术很复杂，我们看到这一市场障碍为我们创造了一个机会，利用我们学到的知识帮助客户在预付费和借记卡领域进行创新，并使其适应信用。”。

Marqeta 相信这样一个观点，即任何目前发行信用卡的企业都在寻找或使用信用卡。

加德纳补充道:“这些创新者想推出现代卡产品，但不得不依赖传统技术，这使得灵活性和个性化的选择少得多，减缓了创新速度。”。

它还押注于消费者希望从信用卡中获得更多，而不仅仅是支付购买费用。

“他们想要无缝的数字体验，与他们的生活方式相匹配的奖励，以及跟踪财务状况的个性化应用程序，但很少有创新能说明这一点，”他说。

随着新冠肺炎·疫情加速非接触式支付——越来越多的人避免面对面互动和购物——对更多数字金融产品的需求激增。

通过新举措，Marqeta 希望能够帮助其客户“在短时间内”推出新的定制信用卡产品，并提供更灵活的控制和功能

 

例如，他们将拥有 Marqeta 所描述的现代信用记录系统，可以根据定制规则实时调整账户参数，如奖励、APR 和信贷额度。客户将能够在获得批准后立即激活持卡人，并将卡直接供应到数字钱包中。

加德纳在向信用卡市场扩张时，称位于门洛帕克的 worth 为“理想的第一战略伙伴”。

“我们计划通过合作伙伴生态系统，利用我们的信用卡发行平台为客户提供项目管理服务，”他说。“它们是与我们试图实现的目标相吻合的基因——坚信开放 API 能够加快上市速度，同时也瞄准了希望打造真正现代卡产品的创新者。他们在信用卡领域经验丰富，有一套独特的要求，也有独特的核保方法。”

就其自身而言，值得一提的是，其 B2B 业务近年来一直在增长，目前每周都有一名潜在客户，每月都有一名新合作伙伴加入其业务。据卡帕迪亚称，在过去三年里，超过 150 万消费者应用了其平台并与之互动，该公司目前(直接或间接)为成千上万的客户提供服务，每月在其平台上交易数千万美元。

卡帕迪亚告诉 TechCrunch:“信贷市场被传统技术、高运营成本、缺乏定制和速度所主导。”“Marqeta 领先的发卡平台与 renewable 的数字卡专业知识相结合，将推动信贷行业的进一步创新，并为消费者提供卓越的用卡体验。”
