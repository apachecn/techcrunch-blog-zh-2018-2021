Fintech Marqeta 在申请首次公开募股几天后进军信用卡领域

原文：https://web.archive.org/web/https://techcrunch.com/2021/02/18/fintech-marqeta-expands-into-credit-card-space/amp/

图片来源: Getty Images

Marqeta 正在向消费者信用卡领域扩张，帮助其他品牌推出信用卡计划。

此前几天，据报道，这家支付卡发行公司秘密申请首次公开募股，成为最新一家走向公开市场的金融科技公司。

据路透社报道，此次 IPO 的价值预计在 100 亿美元左右。据报道，正在与高盛和摩根大通合作上市的 Marqeta 希望在 4 月前完成 IPO。

据 TechCrunch 此前报道，总部位于加州奥克兰的 Marqeta 去年 5 月以 42 亿美元的估值筹集了 1.5 亿美元。然后在 10 月，万事达卡在 Marqeta 投入了一笔数目不详的资金。

该公司为各种金融服务平台提供工具，以提供卡、钱包和其他支付机制，其客户包括 Cash App、Affirm、DoorDash 和 Instacart。Marqeta 表示，截至 2020 年底，该公司已通过其平台发行了 2.7 亿张信用卡，高于 2019 年底的 1.4 亿张。该公司拥有 550 多名员工，在 35 个国家开展业务。

现在，Marqeta 正与另一家初创公司 rewend 合作，推出新的信用卡计划。

正如 应得首席执行官 Kalpesh Kapadia 解释的那样，他的公司的技术和开放 API 平台将为 Marqeta 的项目管理服务提供动力，包括发起 、承保、银行和机构整合、客户服务、合规和风险管理。

Marqeta 创始人兼首席执行官 Jason Gardner 将 Marqeta 在构建新的信贷产品方面的扩张描述为该公司在构建一个“真正全面的发卡平台，能够支持任何卡类型方面的一个“重大里程碑”

他告诉 TechCrunch:“这项技术很复杂，我们看到这一市场壁垒为我们创造了一个机会，让我们能够利用我们所学到的知识，帮助客户在预付费和借记卡领域进行创新，并将其应用到信贷领域。”

Marqeta 认为，任何目前发行信用卡的企业都在寻找或使用信用卡。

“这些创新者希望推出现代卡片产品，但不得不依赖传统技术，这使得灵活性和个性化的选择更少，从而减缓了创新，”加德纳补充道。

它还押注消费者希望从信用卡中获得更多，而不仅仅是支付购物费用。

图片来源: Marqeta

“他们想要无缝的数字体验，符合他们生活方式的奖励，以及跟踪财务健康状况的个性化应用，但很少有创新能解决这个问题，”他说。

随着新冠肺炎疫情加速非接触式支付——越来越多的人避免亲自互动和购物——对更多数字金融产品的需求出现了爆炸式增长。

marq eta 的新计划旨在帮助其客户推出新的定制信用卡产品，“在很短的时间内，提供更灵活的控制和功能。”

 

例如，他们将拥有 Marqeta 所描述的现代信用记录系统，可以根据自定义规则实时调整账户参数，如奖励、APR 和信用额度。客户将能够在获得批准后立即激活持卡人，并将卡直接放入数字钱包。

加德纳称总部设在门洛帕克的该公司是其向信用卡市场扩张的“理想的第一战略伙伴”。

“我们计划通过合作伙伴生态系统，为使用我们信用卡发行平台的客户提供项目管理服务，”他说。“它们是适合我们努力实现的目标的良好 DNA 对开放 API 加快上市速度的力量有着坚定的信念，同时也瞄准了寻求构建真正现代卡产品的创新者。他们在信用卡领域经验丰富，有一套独特的要求，并有独特的核保方法。”

就其本身而言，worthy 表示，其 B2B 业务近年来一直在增长，目前每周增加一名潜在客户，每月增加一名新合作伙伴。据卡帕迪亚称，在过去三年中，超过 150 万消费者申请了该平台并与之互动，该公司目前正在为数十万客户提供服务(直接和间接)，每月在其平台上交易数千万美元。

rewords 还在云端为 Sallie Mae 等公司管理整个 信用卡基础设施，申请和使用 Sallie Mae 信用卡的消费者在幕后与 reworth 互动。它还为 BankMobile 等公司提供发起服务。Opploans、BlockFi 和 Earnest 等其他金融科技公司使用其整个信用卡基础设施来推出自己的信贷产品。

卡帕迪亚告诉 TechCrunch:“信贷市场被传统技术、高运营成本以及缺乏定制和速度所主导。“Marqeta 领先的发卡平台与 renewable 的数字卡专业知识相结合，将推动信贷行业的进一步创新，并为消费者提供卓越的用卡体验。”
