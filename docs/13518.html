Ag 监测初创公司 FluroSat 与土壤碳专家 Dagan 合并，形成 Regrow

原文：https://web.archive.org/web/https://techcrunch.com/2021/02/23/ag-monitoring-startup-flurostat-merges-with-soil-carbon-expert-dagan-to-form-regrow/

Ag 监测初创公司 FluroSat 与土壤碳专家 Dagan 合并，形成 Regrow

FluroSat 和 Dagan 这两家初创公司都在处理农业投入和产出的监测和管理，以更好地了解可持续农业在减少温室气体排放方面的作用，它们已经合并，并以新品牌 Regrow 推出一揽子服务。

两家公司在一份联合声明中表示，昨天宣布的合并将创建一家将 FluroSat 的数据驱动农业管理服务与 Dagan 的土壤生物地球化学建模技术相结合的公司。

合并后的公司将有能力提供卫星收集的数据，以优化作物管理和保护措施的采用，以及针对不同作物、田地、农场和地区的特定地点分析和定制干预。

Dagan 的联合创始人威廉·萨拉斯博士表示，合并后的公司将能够更好地控制碳排放市场——这在很大程度上要归功于 Dagan 在土壤碳方面的工作。

“土壤固碳最终成为一种全球相关的策略，用于减少大气中过量的二氧化碳。捷径、误解和过度宣传有可能阻碍土壤碳的巨大潜力，”萨拉斯在一份声明中说。“但 FluroSat 和 Dagan 的合并将为 行业带来它所需要的信心和完整性，提供一流的土壤健康数据，可以规定特定的战略，并提供准确性和透明度，这将有助于公司在碳市场上取得成功。”

合并的条款没有披露，但根据 Crunchbase 的数据，FluroSat 之前已经筹集了大约 860 万美元的股权和赠款资金，由微软的 M12 风险基金牵头。

微软 Azure Global 首席科学家 Ranveer Chandra 说:“在接下来的十年里，我们需要种植和生产足够的食物，以保护我们的土地和阻止气候变化的方式来养活世界各地的 100 亿人。“Regrow 的计算农业，使用机器学习和 科学建模，将有助于提高计算土壤碳的准确性，并使农民 更接近受益于碳市场。”
