数以亿计的用户，尤其是发展中国家的用户，没有高端智能手机，也买不起高速数据套餐来享受网络上的大部分内容。

谷歌一直在探索多种方式来更好地服务这部分用户群。T2 已经尝试与 T3 合作，让 T4 的数千万用户用得起互联网。它与智能手机制造商合作，为廉价智能手机带来可靠的安卓体验。事实上，目前正在与印度电信运营商 Jio Platforms 合作一个项目，以进一步降低体面的 Android 体验的价格点。

然而，对于手机游戏，谷歌有一个略微不同的想法来吸引用户。120 区，谷歌内部的实验项目孵化器，去年推出了 GameSnacks 。这是一个 HTML5 游戏平台，其中的游戏很小，加载速度快得多，消耗的资源也少得多，因为它们的设计方式。

这个想法似乎正在起作用。

谷歌周二表示，在过去的一年里，它已经在 GameSnacks 上取得了进展，现在准备扩大平台规模，测试盈利模式，以使游戏开发者觉得值得。

在 TechCrunch 的独家采访中，GameSnacks 的总经理 Ani Mohan 表示，该平台已经积累了超过 100 款游戏和数百万用户。

“HTML5 游戏一直在增长，尤其是在美国以外。HTML5 是一个很好的方式，可以让刚上线、可能以前没有玩过游戏的用户玩游戏。这些游戏是跨设备的，在低带宽连接上工作，并且可以立即玩，因为它们不需要用户安装任何文件，”他说。

这些单人游戏可以在任何低至 1GB 内存和 2G 到 3G 数据连接的设备上运行，用户可以通过 GameSnacks 网站获得。它们可以在桌面上播放，也可以在 iPhone 或 iPad 上播放(如果你想试试的话)。

现在，该公司正在利用其规模扩大游戏的覆盖面和可发现性。莫汉说，最近几周，GameSnacks 游戏已经可以在 Chrome 浏览器的新标签页上为印度、印度尼西亚、尼日利亚和肯尼亚的用户提供。

在谷歌最大的用户市场印度，GameSnacks 游戏也开始使用 Google Pay。该公司还在尝试引入 GameSnacks 游戏来发现 feed。

Mohan 说，该公司在选定的国家开始这些整合，因为那里的许多用户面临着该平台试图解决的挑战。“我们认为这只是实验的初级阶段。如果进展顺利，我们将乐于扩大它，”他说。

此外，莫汉表示，该公司正在尝试将 GameSnacks 游戏引入谷歌助手。

“现在这些集成很少是活的，我们希望做的事情之一是与开发者交谈，告诉他们有一种简单的方法可以上谷歌，”他说。

GameSnacks 上的开发者目前通过非独家授权模式将他们的游戏货币化。Mohan 说，这个由六个人组成的团队(尽管更多的人来自谷歌)正致力于帮助这些开发者利用游戏广告格式的下一代 AdSense 从他们的游戏中赚钱。

“随着时间的推移，我们希望帮助他们建立可行的业务，所以我们将开始在这个平台上尝试广告，”他说。然而，这将是为选定数量的游戏目前。

对于 HTML 游戏世界来说，非洲和亚洲等新兴市场并不陌生。例如，在印度，一个名为 Gamezop 的游戏平台去年筹集了 420 万美元，用于扩展其 HTML5 游戏，以吸引更多的开发者，并将它们嵌入到 1000 多个应用程序中。

2018 年，南非电信公司 MTN Group 为其在南部非洲国家的用户推出了 Bonus Bucks HTML5 游戏门户网站。脸书在 Messenger 上运营 HTML5 即时游戏多年，直到将其从即时通讯服务中移除。在我们自己的档案库上快速搜索会返回过去从事 HTML5 游戏的几十家公司，尽管近年来我们看到的例子越来越少。

莫汉仍然乐观地认为，HTML 游戏有很大的机会，而且这种机会不仅限于非洲和亚洲。“我们不认为这些市场是我们唯一的选择。这些只是我们刚刚开始的市场，因为对 HTML5 游戏的需求…尤其迫切。我们认为这方面的市场规模要大得多，因为 HTML 在全世界都有用户，”他说。
