Stitch 凭借其在非洲的 API fintech 游戏 TechCrunch 获得 400 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/02/24/stitch-emerges-from-stealth-with-4m-for-its-api-fintech-play-in-africa/

多年来，世界各地的金融科技基础设施参与者呈增长趋势 。在非洲，一些初创公司在过去三年里推出了此类服务。南非金融科技初创公司 Stitch 就是其中之一；今天，它从幕后走出来，宣布了 400 万美元的种子资金。这使得它成为目前非洲任何 API 金融科技初创公司筹集的最大一轮融资。

由基安·皮莱、娜塔莉·库斯伯特和普里安·皮莱创立的 Stitch 希望从其第一个市场南非 开始，提供全非洲金融账户的 API 访问。通过它的 API，开发者可以将应用程序连接到金融账户。 这允许用户分享他们的交易历史和余额，确认他们的身份和 发起 支付 。

我们已经看到一波以 API 为主导的金融服务公司在全球激增。在美国引领潮流 总部位于瑞典的 fintech Tink 也在整个欧洲占据主导地位，而 Truelayer 和 Belvo 则在英国和拉丁美洲 独占鳌头。

这些公司提供工程和开发工具，减少应用程序连接到用户财务账户所需的技术和运营工作 。通过 API，其他公司只需添加几行代码，就可以整合原本复杂的服务。

像其他金融基础设施公司一样，Stitch services 允许公司和开发商围绕其他服务进行创新，如个人金融、贷款、保险、支付和财富管理 。

创始人利用他们为当地市场 开发 API 产品的经验。2017 年，Kiaan Pillay 担任南非保险 API 平台 Root 运营负责人。他离开后加入了位于旧金山的身份 API 公司 Smile Identity。 在那里，他与非洲各地的金融科技公司合作，发现他们面临着合规性和身份方面的基础设施问题 。

缝合团队。图片来源:庞萨克

此时，在 Root 担任软件架构师的库斯伯特·基安·皮莱(Kiaan Pillay)和普里安·皮莱(Priyen Pillay)正在为非洲打造一款 Venmo 风格的钱包，作为一个附带项目。八个月来试图与银行连接向他们展示了非洲金融科技基础设施的缺乏如何减缓了进展。

“我们正在寻找一种方法，让用户从他们的钱包里取出现金存入他们的银行账户，”基安皮莱告诉 TechCrunch。“我们一开始是手动完成的，后来作为权宜之计，我们使用屏幕抓取自动完成了这个过程。我们意识到，这个手动问题的自动化解决方案本身可以是一个产品，有更好的方式来实现它。”

这使得团队开始着手 Stitch 的工作——基安·皮莱担任首席执行官，库斯伯特担任首席技术官，普里安·皮莱担任首席技术官。他们在 2019 年 10 月开始全职研究这个想法，并在一个月后获得了一轮种子前融资。Stitch 表示，在秘密行动期间，它已经获得了一些客户，其中包括智能债务管理、动量速度俱乐部和灵活俱乐部。围绕面向消费者的产品，该公司也开始吸引一些公司的注意。

截至目前，Stitch 已有一款数据和身份 API 产品，本月将有一款支付产品 加入其产品 中 。像大多数 API fintech 初创公司一样，Stitch 向开发者和公司收取每次 API 调用的费用。 然而 ，对于一些类似预算或个人理财类 app 的产品，也是收取固定费用 。

在广泛而深入的投资者支持下，Stitch 将利用这笔资金巩固在南非 的增长。 也有计划在西非和东非展开行动 。

非洲的金融基础设施空间正在升温

这些市场已经有参与者， 主要是 尼日利亚的初创公司，在 API fintech 领域。他们已经筹集了相当多的资金，背景也令人羡慕。 Mono，半年前才上线的创业公司， 背靠YC；对于 Okra ，是泛非风投公司 TLcom CapitalOnePipe 有 tech stars；总部位于美国但专注于非洲的 Pngme 已经吸引了泛非风险投资公司 EchoVC 和 Lateral Capital 的投资。

目前，这些初创公司不会在两个以上的国家开展业务。例如，Mono，Okra 和 OnePipe 只生活在尼日利亚。Pngme 表示，它在尼日利亚和肯尼亚开展业务，而 Stitch 只在南非开展业务。当竞争和合作扩展到他们的市场之外时，看看它们会如何发展将会很有趣。我们可能不会等太久，因为 Okra 目前正在肯尼亚和南非进行测试，Mono 正计划在年底前扩展到加纳和肯尼亚。

对于我在过去一年中与之交谈过的这些初创公司的创始人来说，竞争对市场来说是健康的，Kiaan Pillay 补充说，从长远来看，可能会出现的情况是，每家公司都在创造他们最擅长的利基功能。

“不像格子图案占主导地位的美国，我认为 非洲需要很多玩家。欧洲就是一个很好的例子；许多大公司都在提供类似的银行 API 服务。对我们来说， 我认为 我们将开始看到的情况是，一些公司将 称为 来做好一件特定的事情，如支付、数据丰富或商户识别 。”

图片来源:缝合

Stitch 拥有令人印象深刻的种子轮投资者阵容，由总部位于伦敦的风险投资公司 、 firstminute Capital 和美国投资公司The Raba Partnership领投。其他参与的投资者包括基金和天使。

基金包括 CRE 和 Village Global、Norrsken(Klarna 联合创始人 Niklas Adalberth 的基金)、Future Africa(flutter wave 联合创始人 Iyinoluwa Aboyeji 的基金)和 500 Fintech 。天使投资人包括 Venmo 的联合创始人伊克拉姆·马格东-伊斯梅尔(Iqram Magdon-Ismail)，Plaid 的一些创始人以及比特币基地、Revolut、Fast 和 Paystack 的高管。

关于这家仍在潜行的初创公司如何成功吸引这些投资者，皮莱表示，这要归功于公司在美国的网络以及每个投资者对产品的信心。

Kiaan Pillay 说:“在旧金山与 Smile 一起工作的大部分时间帮助我们接触到了这些全球世界级的创始人和投资者。“ 我们有机会在整个非洲市场提供新一代金融服务，我们真的很幸运有他们支持我们 ”

对于 firstminute capital 的联合创始人兼普通合伙人布伦特·霍伯曼(Brent Hoberman)来说，该公司决定采用回缝技术，因为它相信非洲的大多数在线业务都将通过回缝技术将金融科技的功能嵌入到他们的应用程序中——促进在线支付，增加贷款能力，简化 KYC 和身份检查。

“作为一名南非同胞，我很高兴能与一支由才华横溢、拥有泛非抱负的当地工程师组成的团队合作，”他补充道 。

也就是说，在经历了一月份的缓慢增长后，非洲的金融科技行业开始升温，农业科技和清洁科技行业在融资轮中占据主导地位 。 本周，南非数字银行 TymeBank 筹集了高达 1.09 亿美元用于在全国范围内扩张并进入亚洲， 扩展了我们过去看到的吸引了超过 30%风险投资资金的大型融资。

对于 Stitch 来说，它的种子期是过去两年非洲 API 金融科技领域一系列引人注目的交易中的最新一笔，所有主要参与者都筹集了 50 万至 400 万美元。
