AWS重组DeepRacer联盟以鼓励更多新手

AWS 在2018年推出了deep racer League作为一种教授开发者机器学习的有趣方式，从那以后它就一直建立在这个想法上。今天，它宣布了最新的联赛赛季，分为两个部分:公开和专业。

正如玛西娅·维拉巴在宣布新联盟的博客文章中写道，“AWS DeepRacer是一款1/18比例的自主赛车，旨在通过在AWS DeepRacer控制台上虚拟比赛或在AWS和客户活动的赛道上实际比赛来测试[强化学习]模型。AWS DeepRacer面向所有技能水平的开发者，即使你没有任何ML经验。当使用AWS DeepRacer学习RL时，你可以参加AWS DeepRacer联盟，在那里你可以在有趣和竞争的环境中获得机器学习的经验。”

虽然该公司开始时是用实体汽车进行现场比赛，但去年疫情迫使他们将比赛变成虚拟活动，但新的形式似乎将新人挡在了门外。因为目标是教人们机器学习，所以让新人参与进来对公司来说至关重要。

这就是为什么它创建了开放联盟，顾名思义，它对任何人开放。你可以测试你的技能，如果你足够优秀，进入前10%，你就可以参加职业组的比赛。每个人都会争夺奖品，比如车辆定制。

职业联赛的前16名每月都有机会参加2021年AWS re:Invent的决赛，这一赛事可能是也可能不是虚拟的，取决于我们在疫情复苏中的地位。

早期阶段是初创企业家和投资者的首次“操作”活动。你将直接听到一些最成功的创始人和风投是如何建立自己的企业、筹集资金和管理投资组合的。我们将涵盖公司建设的各个方面:筹资、招聘、销售、法律、公关、营销和品牌建设。每个环节都有观众参与，每个环节都有充足的时间供观众提问和讨论。
