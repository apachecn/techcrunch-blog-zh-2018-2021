随着维权投资者的出现，Box的下一步是什么？

一家潜力巨大的公司陷入了增长放缓的困境

Box 可能面临多事之秋，如果上周的一则路透社报道准确的话。激进投资者Starboard Value 在2019年9月获得了存储公司7.9%的股份，而在一年前随着其对云公司的参与加深，获得了三个董事会席位。另一只鞋掉下来似乎只是时间问题。

据报道，维权投资者Starboard Value正在争取另外三个董事会席位。

你刚才听到的声音可以说是鞋子，因为据报道右舷增加了三个席位。这些人包括现任首席执行官亚伦·李维(Aaron Levie's)和两名独立董事，他们都将在6月份举行选举。如果该公司获得三个额外的席位，它将控制九票中的六票，并可以为所欲为。

鉴于这种发展(假设这是真的)，公司的未来会怎样？盒子似乎要发生变化了。

下面，我们将探索Box是如何发展到这一步的。如果Box的未来是一次收购，那么谁会在市场上寻找一家为企业规模而构建的云原生内容管理公司呢？很可能会有多个追求者。

Box变幻无常的财务命运

Starboard可能有理由对Box的表现感到沮丧。这家云公司的股价和市值仍然居高不下。它的股价徘徊在每股18美元左右，比2015年上市时的价格高不了多少。它今天的市值是30亿美元，与Dropbox的90亿美元、Slack的230亿美元和Okta的340亿美元相比，这是不足的。

还记得2014年3月Box宣布上市的时候吗？然后，它做了一件极不寻常的事情，将房契延迟了10个月，直到2015年1月。这样或那样的事情阻止了公司扣动扳机并付诸行动。也许这是一个信号。

相反， Box在收到市场的平淡反应后又筹集了1.5亿美元。回过头来看，你可能会说，SaaS模式在2014年只是没有今天那么出名。当然，公众投资者比那时更同情那些以增长的名义出现赤字的软件公司。

但当Box再次提交申请，最终在2015年定价为每股14美元时，它受到了热烈欢迎。据TechCrunch当时报道，该公司的发行价超过了每股11至13美元的IPO区间，并立即飙升。我们在该公司首次公开募股日写道，这家云公司迅速“飙升至每股20美元以上，当时交易价为23.67美元。”

一年后，我们的持续报道发生了逆转，股价在2016年1月停留在10美元。
