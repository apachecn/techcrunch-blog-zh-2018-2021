在疫情的推动下，Daily 为其实时视频平台 TechCrunch 筹集了 1500 万美元的首轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/09/fueled-by-the-pandemic-daily-raises-15m-series-a-for-its-real-time-video-platform/

最近几个月，实时音频和视频开发平台的制造商 Daily 一直在蓬勃发展——部分原因是疫情对远程工作、虚拟活动、远程医疗等的影响。该公司表示，过去一年，每日付费用户数量增长了约 20 倍，收入增长了 30 倍。尽管该团队最初对这种增长犹豫不决，但他们现在已经完成了 1500 万美元的首轮融资。

新一轮由早期 Stripe 员工转为投资者的 T2·拉奇·格鲁姆领导，他通过他的投资组合公司 Interval.com 成为这家初创公司的日常客户。Tiger Global、其他天使投资人和大多数日报的种子投资人也参与其中，使日报迄今为止的融资总额达到 2200 万美元。

Daily 本身成立于 2016 年，是对视频，特别是 WebRTC 的一个大赌注，但受到疫情和由此产生的市场变化的推动。

“我们当时相信，而且有一段时间还相信，视频会无处不在。这将是我们生活、工作和接触各种产品的重要组成部分，”Daily 的联合创始人 Nina Kuruvilla 解释道。“我们认为这种转变需要新的平台和工具，让视频工作变得尽可能简单，”她说。

事实证明，Daily 的押注是正确的——尽管它花了几年时间和一个全球疫情才看到初创公司渴望的曲棍球杆增长。

如今，Daily 被许多专注于未来工作的公司使用，如虚拟办公室创业公司 Tandem 、虚拟总部平台 Teamflow、、演示启动 Pitch 、结对编程工具 GitDuck 、虚拟招聘创业公司 Flo 招聘等。它也被医疗保健和客户支持等领域的企业所采用。

有了 Daily 的平台，开发人员既可以使用预建的 UI 来集成视频通话，只需两行代码，也可以选择构建自己的定制视频 UI 和 UX 。对于前者，开发人员可以嵌入一个具有视频聊天、屏幕共享和录制功能的视频通话小工具。同时，后者让开发者对布局、工作流程以及视频和音频轨道有更多的控制权。

“视频是一个具有挑战性的大问题——实时视频和实时音频。“你需要一些工具来帮助你做到这一点——特别是如果你打算快速推出它，然后扩大规模，”Daily 的联合创始人 Kwindla Kramer 解释道。“我们给你 API，让你快速开发，足够灵活，让你不断改进产品。此外，我们还拥有全球基础设施，使我们的客户无需自己构建一个完全等同于 AWS 的平台就可以进行扩展，”他说。

通过 Daily，开发人员可以访问分布在世界各地的服务器，以防止延迟问题。Daily 还负责确保产品能够在所有平台和设备上良好运行。

从安全和隐私的角度来看，它符合 HIPAA 标准，不会捕获客户数据。关于何时以及如何应用加密，它也是透明的。例如，Daily 的点对点呼叫是端到端加密的，而通过媒体服务器路由的呼叫是加密的，但会在服务器上解密，以便进行转发和记录等操作。

这项技术本身也很灵活。Daily 允许客户指定，出于监管原因，他们是否不希望音频和视频数据包移出欧盟。客户可以选择集成他们首选的转录引擎或其他主要的录制和直播 API。

通过 Daily 的开发者仪表板，客户可以查看呼叫质量洞察和可视化。

为了吸引顾客，Daily 保持其简单的定价模式。它有针对小客户的介绍性计划，可以发展到更大的“规模”计划，每月 199 美元，10，000 分钟。从那里开始，客户只需随着他们的成长按分钟支付额外的费用。

该公司在 COVID 之前已经看到了对视频的稳定需求，特别是在远程工作和医疗保健等领域。但是 COVID 加速了这些用例，也影响了其他用例。2020 年 3 月后，世界很快意识到需要优先考虑视频。

“生产力和工作的未来正在快速变化。在线活动、医疗保健、客户支持、互动直播。物联网和机器人用例。社交和游戏，”Kuruvilla 说，列举了各种日常使用案例。

在疫情的前六个月，由于客户希望快速启动并运行视频，Daily 的采用率有所上升。后来，他们开始转向围绕如何用视频构建数字原生服务进行创新。

这有助于推动视频使用的新模式——比如虚拟健身班，人们不仅可以看到教练，还可以看到他们注册的朋友。或现场活动的混合视频和音频体验；直播过程中的实时视频对话；直播电商活动；还有更多。

该公司认为，这一飞跃意味着，即使在后疫情时代，事情“回归正常”，日报也有未来

Kramer 指出:“我认为一些传统视频通话的使用可能会停滞或下降，但所有这些新的使用案例感觉都在上升，因为这只是早期阶段，它们真的非常有用。”

该团队表示，他们选择与拉奇·格鲁姆合作是因为——与他们在种子阶段投资的数十名投资者不同，他们认为视频是一个利基市场——拉奇知道 Daily 在做什么。

有了额外的资金，Daily 将雇用更多的工程师，并专注于围绕通话质量和可靠性的各种产品计划，扩大其全球基础设施，支持更大的通话和更多的混合用例。
