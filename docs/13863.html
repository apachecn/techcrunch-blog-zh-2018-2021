Pegasus Tech Ventures和Japanet启动新的5000万美元基金，将全球创业公司带到长崎

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/09/pegasus-tech-ventures-and-japanet-launch-new-50m-fund-to-bring-global-startups-to-nagasaki/amp/

图片来源: 不可能的 / Getty Images

Pegasus Tech Ventures ，一家帮助企业启动创业投资基金的公司，今天宣布与 Japanet 建立新的合作关系。日本最大的电视购物公司之一Japanet和Pegasus建立了一个5000万美元的风险基金，投资于世界各地的初创公司，包括长崎的一个重大新开发项目。

Japanet正在寻找创业公司来支持其向新领域的扩张。其中包括将于2024年开放的长崎体育场城建筑项目。它将以一个体育场为中心，包括多用途设施，如办公室、零售店、酒店和活动场所。Japanet还计划专注于老年人和儿童教育的新服务。

Pegasus Tech Ventures的普通合伙人兼首席执行官Anis Uzzaman告诉TechCrunch，体育场城的创建是为了帮助振兴长崎的经济，并为这座城市带来来自世界各地的新产品和服务，包括技术。Uzzaman说，Japanet的计划是“随着时间的推移，与早期创业公司共同开发解决方案，并帮助后期创业公司在体育场城市进行本地化和部署”。

Pegasus Tech Ventures的团队将帮助Japanet寻找来自世界各地的初创公司，包括北美、以色列、欧洲和亚洲。该公司目前管理着15亿美元的资产。通过其“风险资本即服务”项目与之合作的其他公司包括华硕、世嘉萨米控股、阳光健康、信息通信公司和爱信精机。Pegasus Tech Ventures的投资组合包括SpaceX、23andMe、SoFi、Bird、Color和App Annie等初创公司。

Japanet基金对早期创业公司的典型投资额度从10万美元到100万美元不等。后期创业公司将获得100万至500万美元的投资。初创公司将与Japanet及其企业合作伙伴密切合作，包括三菱商事株式会社、JLL购物中心管理公司和MSC Cruises Japan。
