Via 以 1 亿美元收购地图初创公司 Remix

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/09/via-buys-mapping-startup-remix-for-100-million/amp/

图片来源:混音/Via

Remix 是一家开发城市交通规划和街道设计地图软件的初创公司，它诞生于美国代码奖学金(Code for America fellowship)期间的一次黑客马拉松。近七年后，这家总部位于旧金山的初创公司被 Via 以 1 亿美元的现金和股权收购。

Remix 将成为 Via 的子公司，这一安排将使这家初创公司保持其独立品牌。Remix 的 65 名员工和两位联合创始人——首席执行官 Tiffany Chu 和首席技术官 Dan Getelman——将继续留任。

此次收购为威盛不断扩张的业务以及 22 个国家 350 多个地方政府的客户群增加了又一项服务。

朱棣文在最近的一次采访中说，Remix 的优势在于规划，而威盛则带来了软件和运营方面的专业知识。

“通过将这两种优势结合在一起，我们可以成为一个更强大的端到端解决方案——从围绕运输规划的这一想法的最初起源，并将其贯彻到运营中——以一种我们单独无法实现的方式，”Chu 说。

图片来源:混音

Via 于 2012 年开始运营按需班车。该公司在 E 轮融资 4 亿美元后，去年的估值达到 22.5 亿美元，已经从最初面向消费者的重点发展起来。

如今，威盛的核心业务是其软件和运营平台，城市和交通部门使用该平台来规划、调度和部署他们自己的按需和固定路线的公交、辅助交通和校车。威盛在 24 个国家有 200 个合作伙伴。

Via 由 Exor(阿涅利家族控股公司，拥有 PartnerRe、法拉利和菲亚特克莱斯勒汽车公司的股份)，以及 Macquarie Capital、Mori Building、Shell 83North、Broadscale Group、Ervington Investments、Hearst Ventures、Planven Ventures、Pitango 和 RiverPark Ventures 支持。

偶然的创始人

Remix 的硅谷式起源故事是由一些不太可能的企业家推动的。

Chu 之前是 Zipcar 的用户体验设计师，后来她搬到旧金山，完成了为期一年的 Code for America 项目。在这个项目中，Chu 和她后来的联合创始人 Getelman、Sam Hashemi 和 Danny Whalen 一起致力于一个黑客马拉松项目，帮助旧金山市民向旧金山市交通局建议更好的交通路线。

这个交通规划工具被分享到推特上，并迅速传播开来。两周内，30，000 张地图被制作出来。

“它变成了人们在网上探索的这种有趣的、意想不到的扶手椅型交通规划工具，”朱回忆道。但是引起注意的不仅仅是当地居民。大约 200 名城市规划人员伸出援手，要求该团队构建额外的功能，供各机构用于自己的交通规划项目。

“当我们意识到这个被认为是草根性质的市政项目实际上对解决交通方面的实际需求和问题有影响时，这是一个令人兴奋的时刻，”朱棣文说。

不久后，Remix 成立了，公司的创始人申请并被 Y Combinator 接受。公司继续从 Y Combinator、红杉资本和 Energy Impact Partners 筹集了总计 2700 万美元的投资。
