拥抱脸为其自然语言处理库 TechCrunch 筹集了 4000 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/11/hugging-face-raises-40-million-for-its-natural-language-processing-library/

拥抱脸已经筹集了 4000 万美元的 B 轮融资——Addition 领投该轮融资。该公司一直在为自然语言处理(NLP)技术建立一个开源库。你可以在 GitHub 上找到变形金刚库——它有 42000 颗星星和 10000 个叉子。

现有投资者 Lux Capital、A.Capital 和 Betaworks 也参与了今天的融资轮。其他投资者包括德夫·伊蒂切里亚、奥利维尔·波梅尔、王敬实、阿希·马里蒂、弗洛里安·杜埃托、理查德·索彻、保罗·圣约翰、凯文·杜兰特和里奇·克雷曼。

使用 Transformers，您可以利用流行的 NLP 模型，如 BERT、GPT、XLNet、T5 或 DistilBERT，并使用这些模型以某种方式操作文本。例如，您可以对文本进行分类、提取信息、自动回答问题、汇总文本、生成文本等。

NLP 有许多不同的用例。一个流行的是支持聊天机器人。例如，挑战者银行 Monzo 一直在幕后使用拥抱脸来回答客户的问题。总的来说，大约有 5000 家公司正在以这样或那样的方式使用拥抱脸，包括微软的搜索引擎必应。

谈到商业模式，这家初创公司最近推出了一种获得优先支持、管理私有模型和为你托管推理 API 的方式。客户包括彭博和 Typeform。

随着新一轮融资的到来，该公司计划将纽约和巴黎的员工人数增加两倍——也会有远程职位。有趣的是，该公司还分享了一些关于其银行账户的细节。

2021 年 1 月和 2 月，拥抱脸的现金流为正。一年多前，该公司筹集了 1500 万美元的融资——前一轮融资的 90%仍在银行账户上。该公司的估值增长了五倍。这不应该令人惊讶，因为如果你实际上不需要筹集资金，你可以谈判更好的条件。
