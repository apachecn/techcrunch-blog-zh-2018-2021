Disruptel 筹资 110 万美元，让智能电视变得更智能

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/12/disruptel-seed-funding/

总部位于圣路易斯的语音助手初创公司 Disruptel 宣布，它已经筹集了 110 万美元的种子资金。

这笔钱来自一群令人印象深刻的投资者，他们似乎与这家初创公司的目标非常一致——即建立一个语音助手，可以提供关于你的电视屏幕上正在发生的事情的详细信息。这些投资者包括 PJC 和 Progress Ventures(领投了这轮融资)，以及 DataXu 联合创始人兼前首席执行官迈克·贝克、Siri 联合创始人亚当·切尔、Sky 高管安德鲁·奥尔森和 DataXu 联合创始人比尔·西蒙斯。

Disruptel 首席执行官亚历克斯·奎因(Alex Quinn)告诉我，他在高中时就开始追求这个想法——最初的想法更专注于电视手势控制，但他认为“智能电视不知道自己屏幕上发生了什么”这一事实有更大的机会。

所以他说，Disruptel 已经建立了“对屏幕上发生的一切——每件产品，所有数据——有上下文理解的技术。”例如，你可以用这项技术问你的电视，“穿棕色衬衫的人是谁？”

奎因的描述让我想起了亚马逊的 X 射线技术，它可以告诉你屏幕上的演员，以及你在亚马逊 Prime 上观看的任何电影或电视的其他琐事。但他表示，亚马逊的解决方案(以及谷歌的类似解决方案)涉及“静态数据——视频都经过了预处理。”另一方面，使用 Disruptel，“一切都在实时发生”，这意味着理论上它可以处理任何内容。

Disruptel 的旗舰产品 Context 是一款语音助手，旨在与智能电视及其遥控器配合使用。奎因表示，他希望与智能电视制造商和流媒体服务合作，并在今年下半年将这款电视送到观众手中。

与此同时，该公司已经为谷歌 Chrome 开发了智能屏幕扩展，你现在就可以尝试(使用该扩展，我成功地在《闪电侠》一集的多个场景中识别了屏幕上的演员)。Quinn 表示，该公司正在使用该扩展作为测试产品和收集参与数据的方式。

贝克(他在 2019 年将自己的广告技术公司 dataxu 卖给了 Roku)表示，在看到该产品的演示后，他被说服支持该公司:“看到这种体验的力量和流动性很有趣。”

他还表示，Disruptel 的技术为改善智能电视广告体验创造了新的机会，他称智能电视广告体验主要由“垃圾”组成——尽管他也指出 Hulu 是一个可以通过“非侵入式广告和插播广告”获得成功的服务例子

当被问及一名高中生如何创造出这种技术时，奎因(现在 21 岁)说:“我们必须学习。我们的团队非常专注于机器学习，我们的机器学习工程师阅读了一篇又一篇的研究论文。我们认为，我们已经找到了最佳的研究解决方案。”

他补充说，如果 Disruptel 跟随大公司的脚步，专注于预处理内容，“我们甚至永远不会开始这段旅程。”
