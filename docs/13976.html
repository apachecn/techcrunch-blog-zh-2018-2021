被解雇的 GitHub 员工与公司达成“友好解决”

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/15/fired-github-employee-reaches-amicable-resolution-with-company/

这位前雇员告诉 TechCrunch，GitHub 已经与该公司在 1 月份美国国会大厦遇袭后解雇的人达成了“友好解决”。

特朗普支持者的暴力暴徒袭击美国国会大厦的那天，一名忧心忡忡的 GitHub 员工警告他在华盛顿地区的同事注意安全。一位同事在 Slack 上发表评论说，“注意安全，兄弟们，纳粹是要来的，”他生气了，说这种言论对工作没有好处，这位前员工之前告诉我。他之前告诉我，两天后，他被解雇了，一名人事关系代表以“不利于公司政策的行为模式”为由解雇了他。

当月晚些时候， GitHub 首席运营官·埃里卡·布雷西亚表示，公司的人力资源负责人对所发生的事情承担全部责任，并从公司辞职。GitHub 没有透露辞职人员的姓名，但众所周知，凯莉·奥勒森是 GitHub 的首席人力资源官。当时，GitHub 表示，它也“撤销了与该员工离职的决定”，并正在与他的代表交谈。

然而，被解雇的员工并没有恢复工作。

GitHub 的一位发言人告诉 TechCrunch:“我们在审查了调查结果后，立即向该员工提出了复职，但他拒绝了。”。

相反，他告诉我，“我和公司达成了友好的解决方案。我很欣赏他们谴责白人至上主义及其给每个人带来的危险。”

他没有具体说明决议的条款，但他之前告诉我，他正在寻求损害赔偿或其他形式的和解。

以下是他的声明全文，他要求我们全文发表:

我和公司达成了友好的解决方案。我很欣赏他们谴责白人至上主义和它给每个人带来的危险。
