Flutterwave 和 PayPal 合作，允许非洲商家接受和支付款项 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/15/flutterwave-and-paypal-partner-to-allow-african-merchants-to-accept-and-make-payments/

在一些非洲国家，商家几乎不可能从 PayPal 收款。尽管这家支付巨头没有给出原因，但猜测暗示了上述国家监管不足和银行安全状况不佳等因素。

对于一些企业来说，这可能已经成为过去，因为非洲支付公司 Flutterwave 今天宣布与 PayPal 合作，允许 PayPal 全球客户通过其“Pay with PayPal”功能 向非洲商家支付。

通过这种合作关系，非洲企业可以与全球超过3 . 77 亿 PayPal 账户建立联系，并克服非洲大陆高度分散和复杂的支付和银行基础设施 带来的挑战。

根据首席执行官 Olugbenga "GB" Agboola 的说法，这将通过 Flutterwave 与 PayPal 的集成来实现，因此商家可以在非洲大陆之外收款时将 PayPal 作为一种支付选项 。该公司声称，这项服务已经为在 Flutterwave 上注册了商业账户的商家提供，将在 50 个非洲国家和全球范围内运营。 Flutterwave 希望将这项服务也推广到平台上的个人商户 。

他对 TechCrunch 说:“简而言之，我们正在为非洲企业带来超过 3 亿贝宝用户，这样他们就可以接受整个非洲大陆的支付方式。 “我们在公司的使命一直是为无限可能简化支付，从我们开始的时候，它就一直是关于全球支付 。 因此，尽管我们拥有非洲最大的支付基础设施，但我们希望在我们的平台上拥有。”

图片来源:Flutterwave

PayPal 发言人证实了 Flutterwave 与 TechCrunch 的合作。

自从公司扩张到非洲后，就和非洲大陆的大部分国家保持着一边倒的关系，只允许他们汇款 。 而根据其网站显示，只有 12 个非洲国家可以在该平台上收发钱款，但程度不同 。 他们包括阿尔及利亚、博茨瓦纳、埃及、肯尼亚、莱索托、马拉维、毛里求斯、摩洛哥、莫桑比克、塞内加尔、塞舌尔和南非 。

无法享受这种待遇的国家的用户不得不依靠使用朋友或家人的 PayPal 账户，这些账户位于可以接收付款的国家。接下来 ，他们要求通过银行转账的方式获得资金，导致产生更多的费用或使用其他跨境资金平台，如 WorldRemit 。

这是这些企业的痛点，尤其是在尼日利亚。2014 年，PayPal 终于来到了非洲人口最多的国家，一年后，它成为了该公司在非洲的第二大市场。

尽管其采用率很高，金融科技胃口很大，但商家仍无法在该平台上接收来自其他国家的支付。各种消息来源暗示，由于 PayPal 在互联网欺诈方面的历史，它决定不向该国提供这项服务。 无论欺诈与否，尼日利亚的电子商务和整个欧洲大陆的电子商务都在以惊人的速度增长。2017 年，非洲创造了 165 亿美元的收入，到 2022 年，预计将达到 290 亿美元。有了这样的数字，就不难理解为什么贝宝想要参与其中，尽管不完全是。 遂与 Flutterwave 合伙。T47】

该公司通过其 API 向整个非洲大陆的个人和企业提供支付服务。自 2019 年推出以来，这家非洲支付公司已经与 Visa 合作推出易货交易；支付宝将在非洲和中国之间提供数字支付；以及用于非洲支付的 world pay FIS。

但这次与 PayPal 的合作可以说是迄今为止最大的一次合作。现在，非洲企业有更多的机会向全球客户销售产品，使用贝宝在线接收和发送付款。

在某种程度上，Flutterwave 吸收了 PayPal 认为如果其平台对这些国家的商家更加开放将会招致的大部分风险。但同时，这也巩固了 Flutterwave 在寻求进入非洲市场的跨国公司眼中的地位。

就像它与 with Worldpay 的合作恰逢其 B 轮融资一样，这一声明也是在融资后发布的。 上周，支付公司完成了由 Avenir 成长资本和老虎环球领投的 1.7 亿美元的 C 轮融资，成为一家价值十亿美元的公司 。

事后看来，这次巨额加薪表明公司有几个项目正在筹备中。根据这种合作关系，我们可以预计其中大多数都是全球性的 。

然而，这些问题仍然是人们最关心的问题— 当 PayPal 自动 允许这些被忽视的非洲国家的企业开始接收付款 时会发生什么？如果发生这种情况，两种服务会继续共存吗？我们已经联系了贝宝公司进行评论。

无论结果如何，这对 Flutterwave 来说都是朝着正确方向迈出的一步，flutter wave 一次又一次地表明了它愿意为其 29 万商户付出的努力，以及对成为全球支付公司的持续追求。

“通过与 PayPal 合作，我们可以进一步加强我们对客户和服务用户的承诺，因为我们将使他们能够进行交易和扩大业务运营，以开拓新的市场。贝宝的全球影响力是无与伦比的，与他们合作可以让我们的客户探索贝宝嵌入的新市场，”首席执行官说。
