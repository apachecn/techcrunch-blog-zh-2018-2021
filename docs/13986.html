Match Group向非营利背景调查机构嘉宝TechCrunch投资7位数

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/15/match-group-makes-seven-figure-investment-in-background-check-nonprofit-garbo/amp/

图片来源: Tinder

Tinder、Match、OkCupid、Hinge和其他顶级约会应用的母公司Match Group今天早上宣布，它已经向非营利背景调查平台 Garbo 投资了七位数，目的是帮助Match Group的用户在网上约会时就其安全性做出更明智的决定。在这笔交易中，Match将与嘉宝合作，在今年晚些时候将背景调查技术整合到Tinder中，随后是Match Group的其他美国约会应用程序。

图片来源:凯瑟琳·科斯米德斯，via Match Group

总部位于纽约的嘉宝于2018年由凯瑟琳·科斯米德斯(Kathryn Kosmides)创立，她是基于性别的暴力的幸存者，希望让每个人都能够更容易地有能力查找有关某人背景的关键信息，这些信息可能表明暴力历史。

通常情况下，背景调查服务由盈利性公司运营，会暴露各种各样的个人信息，如毒品犯罪或轻微的交通违规，这些信息并不总是与安全和滥用问题相关。另外，嘉宝指出，这些类型的指控通常是针对更弱势群体的成员，与基于性别的暴力无关。

相反，嘉宝提供低成本的背景调查，只收集公共记录和暴力与虐待的报告，包括逮捕、定罪、限制令、骚扰和其他暴力犯罪。要使用该服务，用户需要输入名和姓，或者名和电话号码——这通常是约会应用用户对其匹配对象的唯一信息。

然后，该服务将执行所谓的“公平背景”检查，意味着它将从结果中排除持有毒品指控，以及除了酒后驾车和车辆过失杀人之外的交通罚单。

去年，这家非营利组织在纽约地区的500人中开展了对其技术的测试，并很快通过口口相传将其等候名单扩大到6000多人。嘉宝后来取消了测试，因为该团队意识到这项技术有可能在全国范围内推广——这是该团队希望在向公众推出之前提供的东西。

作为一个没有多少资金支持的小型非营利组织，嘉宝也意识到在这方面可能需要一个更大的合作伙伴。在Kosmides与Match Group的新任安全主管特雷西·布里登(Tracey Graham)建立联系后，两家公司同意合作，将该技术推广到更广泛的美国受众。

“长期以来，世界各地的妇女和边缘群体在资源和安全方面面临着许多障碍，”Match Group的安全和社会倡导负责人布里登在关于《今日新闻》的一份声明中说。“我们认识到，通过技术和基于行动的真正合作，企业可以在帮助消除这些障碍方面发挥关键作用。她补充说:“通过与Match Group合作，嘉宝深思熟虑和开创性的消费者背景调查将为用户提供信息，帮助他们创建更安全的连接和在线社区的公平途径。”。

这是Match Group对外部安全技术提供商的第二次投资，以增强其约会应用的功能集。2020年初，在ProPublica和哥伦比亚新闻调查于2019年12月发表了一篇谴责性的调查报告之后，该公司投资了Noonlight ，以帮助它在Tinder和其他约会应用程序中增加新的安全功能。该报告披露了Match Group如何允许已知的性侵犯者使用其应用程序。它还指出，Match Group没有对其约会应用用户进行背景调查的统一政策，让用户承担保护自己安全的责任。

与此同时，Tinder的主要竞争对手Bumble一直在宣传自己是Tinder等传统约会应用的更受女性欢迎的替代品，并推出了许多旨在保护用户免受不良行为者影响的功能，包括最近的一种方法，防止他们使用该应用的“取消匹配”选项来隐藏受害者。

鉴于获得“不安全”应用的声誉可能会对Tinder这样的品牌以及整个更大的在线约会行业造成严重损害，很明显为什么Match Group现在直接投资解决这个问题。例如，通过对Noonlight的投资， Match Group承诺提供一些功能，比如在Tinder应用程序中触发紧急服务的谨慎方式，类似于优步和Lyft的功能，以及其他反滥用措施。

Match Group表示，嘉宝将利用新投资招聘跨产品、工程和领导层的员工，包括一名工程主管和一个由五名工程师组成的初始团队。这个团队将利用自然语言处理和人工智能等技术来开发嘉宝的能力。

嘉宝还将受益于Match Group投入的大量时间和资源，因为它的产品将全面投入运营，然后从Tinder开始在Match Group的产品中推广。Match Group将帮助嘉宝将非营利组织的技术应用于其他平台，比如拼车公司。

然而，一旦进入Tinder，背景调查功能可能就不是免费的了。

相反，Match Group表示，它将根据用户采用情况、有多少人想使用它、他们想进行多少次搜索以及其他因素来确定定价。它还没有确定集成的深度——例如，它是否会在应用程序外部链接到嘉宝，或者让它看起来更像是应用程序内的功能。

Match Group没有为Tinder的这一功能提供任何确切的时间表，除了“今年晚些时候”，其他美国约会应用也将紧随其后。未来几个月，该公司可能会考虑对其面向国际用户的服务进行类似投资。
