Cohere 为其网络应用远程控制解决方案筹集了 310 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/17/cohere-raises-3-1-million-for-its-remote-control-solution-for-web-apps/

现有的远程桌面解决方案，如 LogMeIn 和 TeamViewer，设置和使用起来可能会很复杂，并且可能会感觉过时。一家名为 Cohere 的初创公司目前获得了 310 万美元的种子资金支持，旨在改善远程桌面和屏幕共享体验。有了 Cohere 的技术，企业可以在几秒钟内帮助客户即时控制他们的屏幕，而无需在客户端进行任何下载或设置。

这种易用性已经为这家初创公司的产品赢得了 50 多个付费客户，包括 TechCrunch Disrupt 2020 冠军 Canix ，CopyAI，Ramp 等。它还与 Podium 签署了第一个企业客户。

Cohere 的三位联合创始人，林、王凯峰和拉胡尔·森戈图韦卢第一次见面是在杜克大学上学的时候。林后来去了公司卡创业 Ramp，其他人都是毕业的。

Cohere 的想法实际上是在疫情期间出现的，那是一次专注于远程工作的黑客马拉松。该团队赢得了比赛，并决定将他们的项目交给 Y Combinator 进行进一步开发。

从本质上讲，Cohere 旨在使团队更容易帮助解决客户的问题，无论是小型创始人还是大型企业。客户只需点击网站上的一个按钮，就可以远程控制他们的屏幕，而不需要下载软件或复杂的安装过程。这可以节省时间，因为另一端的支持人员不必问无数个关于客户在哪个屏幕上或他们看到了什么的问题，或者指导他们点击哪里，他们可以直接接管。

Sengottuvelu 说:“它让你看到人们看到的东西，就像这样，没有设置。“你可以展示给他们看，就像你坐在他们旁边一样。”

图片来源:连贯

但 Cohere 的不同之处在于，它不是一个完整的远程桌面解决方案，在这个解决方案中，另一端的人会接管某人的电脑——这项服务只是转发这个人当前正在查看的个人网页的内容。也就是说，应用程序开发人员只能查看用户在他们自己的网站上做了什么。他们不能切换标签或最小化浏览器来更广泛地浏览用户的电脑。

“我们不像普通的屏幕共享那样在像素级别上操作，在普通的屏幕共享中，他们每秒钟给你的屏幕拍照 60 次，并试图通过网络发送，”Lin 解释说。

这家初创公司的技术本身是基于捕捉网页的状态——对那些理解术语的人来说，可以说是一张 DOM 的图片。然后，它利用像变异观察者和 WebSockets 这样的东西，使得快速实时地看到网页的变化成为可能。Cohere 还花时间来确保它与一系列 web 技术和框架兼容，包括 React、Iframe、Canvas、Vue、Angular 等。随着时间的推移，它希望扩展对更多平台和技术的支持。

“我们捕捉页面的内容，”林说。“因此，我们能够选择性地过滤掉敏感信息，如信用卡信息、密码或社会安全号码——任何个人身份信息，”他补充道。

远程查看器也不能控制，直到用户接受，然后用户可以在任何时候点击屏幕顶部的“X”按钮来启动它们。

除了这些安全控制，Cohere 还符合 SOC 2 Type 1、GDPR 和 CCPA 标准。

目前，Cohere 可以在桌面和移动浏览器上工作，包括 Chrome、Firefox、Edge 和 Safari。它旨在与企业现有的工具集成，如 Zendesk、Slack、Salesforce 和 Intercom，以及未来的更多工具。

虽然该产品在某种程度上与旧的远程桌面应用程序竞争，但其较新的 Cohere Replay 功能允许企业回到过去，追溯查看他们的客户会议。

因为该产品仅限于 web 应用，而不是一个完整的远程桌面解决方案，它往往主要用于客户支持和用户入职等方面。早期创业公司也使用它为他们的第一批客户提供参观，并了解客户如何实时使用产品。

这项付费服务的起价为每个用户每月 49 美元，大型团队为 39 美元。

Cohere 在大约 7 个月前首次推出，当时是 Y Combinator 批次。

它现在筹集了 310 万美元的种子资金，由 Initialized Capital 牵头。本轮其他投资者包括 Y Combinator、BoxGroup、Soma Capital、耸肩资本、Chapter One 和各种天使投资人，如 Zach Perret、Elad Gil、Naval Ravikant、巫启贤、Prasanna Sankaranarayanan、Eric Glyman、Jack Altman、Todd Goldberg、Rahul Vohra、Karim Atiyeh、Vivek Sodera、Dan Romero、Shrav Mehta 和 Oscar Hong。

总部位于纽约的 Cohere 目前仅由三位联合创始人组成，它将利用额外的资金招聘销售、工程和产品人员。它还将投入一些资金来构建企业销售流程，并扩展其集成和用例。
