尼日利亚的Plentywaka得到Techstars的支持，计划向加拿大扩张

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/17/nigerias-plentywaka-gets-backing-from-techstars-plans-expansion-to-canada/

尼日利亚公共汽车预订平台 Plentywaka 今天宣布 已被 接受加入 Techstars Toronto 加速器计划 。

它将加入2021年的其他九家创业公司，并从加速器获得资金，因为它将目光投向了全球扩张 。

这家总部位于拉各斯的公司由Onyeka Akumah、Johnny Ena、John Shaibu和Afolabi Oluseyi创建，运营一种“优步换公交”模式，通过应用程序 将通勤者与公交车连接起来。

根据首席执行官阿库马的说法，Plentywaka于2019年9月推出，在头两个月，平均每天运送6人。到第六个月，这一数字增加到每天约1500人，该公司在这段时间内完成了超过100000次乘坐。

然后在2020年3月，疫情引发的封锁打击了拉格斯和尼日利亚其他州的企业。由于其业务的性质，Plentywaka不得不稍微调整一下，开始在拉各斯运输基本服务，尤其是食品。它还开设了物流服务。

随着全城封锁的缓解和通勤的恢复，该公司转移了60%的产能，而运营成本保持不变 。尽管增长稳定并有所加快，公司还是开始寻求外部投资。8月，它从其母公司EMFATO和其他早期投资者如Microtraction和Niche Capital获得了30万美元的前期种子资金。

在新资金的支持下，Plentywaka已经将核心产品翻倍——通过公交车运送乘客。自 以来，它推出的物流部门和汽车服务已经关闭 。

阿库马表示，对首次公开募股的关注已经带来了回报。 该公司已将其州内服务扩展到尼日利亚的另外两个城市，包括该国的首都阿布贾，并已转移了约30万人 。 在这一宣布之后，他们立即计划在尼日利亚的不同城市推出州际服务 。

这项服务将会看到Plentywaka与一些主要的巴士旅游公司合作，这些公司 合起来 拥有2000多辆巴士， 在全国100多条路线 。 Plentywaka作为一个聚合器，通勤者可以在其平台上看到各种运输公司的选项，比较票价并预订 。

“plenty Waka正在变得越来越像一个聚合者，因为我们在我们的平台上搭载了运输公司 。Plentywaka的联合创始人兼总裁Ena告诉TechCrunch，尼日利亚的州际旅行数据不足，我们希望成为第一家解决这一问题的公司。

除此之外 以及来自Techstars的新资本，Plentywaka正在寻求将其平台扩展到非洲和北美 。Akumah表示，这一全球扩张计划将于2021年第四季度或之前从加拿大的一个城市开始，最有可能是多伦多。

Techstars Toronto董事总经理Sunil Sharma 向TechCrunch证实了这一点。 根据Sharma的说法，Techstars正在支持尼日利亚的移动创业公司，因为它正在解决尼日利亚的一个大问题，这个问题可以 比作 全球其他人口稠密城市的城市交通挑战 。

我们知道西方城市有传统的交通系统。 然而 ，有很多交通挑战，即使是在多伦多这样的城市，他说 。 “和 我们认为plenty Waka在改善市民生活和日常通勤需求方面的技术和方法可以 带到西方的城市就像在非洲一样。”

Plentywaka计划在引入城际模式 之前，在与该国必要的利益相关方接触后，首先推出市内服务。夏尔马认为加拿大的大多数城市都没有良好的公共汽车服务，这导致了城际交通基础设施的损坏。他说，Plentywaka的存在将为这座城市带来急需的选择。

“这里的城镇应该有公交连接，但是他们相当根本没有，我的观点是Plentywaka的到来将是现状的一个直接选项。 它也将作为一种补充现有交通选择的方式引起人们的共鸣， ”他说。

Techstars与Akumah的关系也被证明对Plentywaka加入加速器至关重要。 第二次获得Techstars支持的创始人，Akumah于2016年联合创立了尼日利亚数字农业平台Farmcrowdy。四年前，阿库马在agritech初创公司完成了加速器的亚特兰大项目，现在他正在Plentywaka做同样的事情。他同时担任两家公司的首席执行官。

这位连续创始人表示，与Techstars的关系是该公司向加拿大而不是非洲邻国扩张的原因之一。

“如果我们现在在多伦多拥有的扩张机会与我们在加纳或南非拥有的机会类似的话，当然我们已经在进行这些对话了。 但是，当我们甚至不需要脚踏实地就能获得Techstars、Sunil和多伦多监管机构的支持时，我的意思是，这让我们很高兴能扩展到加拿大， ”这位首席执行官说。

一般来说，尼日利亚或非洲的初创企业很少进入加拿大。Plentywaka正处于这样做的边缘，它将寻求结束投资者的种子期，以实施这些扩张计划并进一步改进其技术。
