Airtel Africa向TPG的Rise基金出售2亿美元的移动货币业务股份

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/18/airtel-africa-sells-200m-mobile-money-business-stake-to-tpgs-rise-fund/

二月份，在伦敦上市的电信公司 Airtel Africa 表示，它正在寻求出售其移动支付业务的少数股份，以筹集现金并出售部分资产T4。

该公司似乎已经找到了一个投资者，因为它宣布投资公司【TPG的全球影响力投资平台The Rise Fund将向其移动资金部门 投资2亿美元。

这笔投资将见证移动货币业务——Airtel Mobile Commerce BV(AMC BV)——价值26.5亿美元。 AMC BV是Airtel Africa的子公司，也是Airtel Africa在14个非洲国家开展移动支付业务的控股公司，这些国家包括肯尼亚、乌干达和尼日利亚 。

AMC BV表示，控股公司将利用投资 减少债务，并投资于各自运营国家 的网络和销售基础设施。 这笔交易将分两部分完成——第一次完成时投资1.5亿美元，第二次完成时投资5000万美元至。

交易完成后，Airtel Africa仍将持有该业务的多数股权，并正在探索在未来四年内将该业务上市的机会。

Airtel Africa首席执行官 Raghunath Mandava 说 “我们的市场为移动货币服务提供了巨大的市场潜力，以满足非洲数千万很少或根本无法获得银行和金融服务的客户的需求，这种需求正在推动增长。" 随着今天的发布，我们 很高兴 欢迎Rise基金成为我们移动货币业务的投资者和合作伙伴，帮助我们实现从 大量 机会中获得的全部潜力，为非洲各地没有银行账户的人提供银行服务 。"

Airtel mobile money business是推动非洲金融普惠的众多参与者之一，它提供一系列服务。 它们包括移动钱包存款和取款、商户和商业支付、福利转移、贷款和储蓄、虚拟信用卡和国际汇款 。

典型的 ，这些服务遍布各个国家，但在尼日利亚的运营方式略有不同。 在西非国家，Airtel走的是与当地银行合作的路线，但现在已经申请了自己的移动银行牌照 。

在最近公布的2020年第三季度业绩中，Airtel Africa 见证了 同比41.1%的收入增长，达到1.1亿美元， 主要受29%的客户群增长 和9.7%的ARPU增长 的推动。 交易价值增长53%，至128亿美元(年化520亿美元)，基本息税折旧及摊销前利润为5400万美元(年化2.16亿美元)，利润率为48.7% 。

AMC BV受益于强大的线下信息亭、迷你商店和代理商，这些与其核心电信业务密切相关。为了推动今年的增长，该业务已经与万事达卡、三星、渣打银行和WorldRemit等达成合作，以扩大其移动支付产品的范围和深度。

领导非洲投资崛起基金的TPG合伙人叶迷·拉卢德表示，金融普惠是一个全球性问题，在非洲尤为突出，电信正在缩小传统金融机构和数百万没有银行账户的非洲人之间的差距。

“我们期待与Airtel Africa合作，增强他们的移动货币服务，拓宽其使用案例，并成长为新的市场 。 他说 通过对Airtel Africa的移动货币业务进行投资，我们 很高兴能够扩大Rise Fund的全球金融科技投资组合，并继续深化我们对改善非洲和世界各地金融包容性的关注。

去年，管理着超过910亿美元资产的TPG向Reliance Jio投资了6亿美元。这家电信运营商是Airtel Africa母公司Bharti Airtel的竞争对手。这是一个有趣的细节，尽管两项投资的目标市场不同，而且都来自TPG的不同基金。
