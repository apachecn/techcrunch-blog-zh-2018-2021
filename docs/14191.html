疫情使互联网成为购物、谋生和维持个人关系的生命线。现在，随着封锁开始解除，房地产行业必须弄清楚这意味着什么。

为了寻找答案，我调查了押注该行业最大、最令人惊讶的变化的人——prop tech 投资者。虽然房东和他们的贷款人可能希望回到过去，但 proptech 的投资者关注的是地方概念的未来走向。我与就 TechCrunch 的许多摘录和我的想法进行了深入的讨论，包括对社区零售的复兴、城市的重生等等的看法。

不过，我无法涉及所有重要的话题，包括住宅销售和租赁市场最近的快速发展、建筑技术、相关的气候技术话题以及其他问题。

随着疫情有望进入最后阶段，这一时期的变化对你的房地产和房地产技术投资策略有何影响？目前的大趋势，如全国范围内的高住房需求和大量的商业空置，是否会重新调整你的投资策略？

最重要的是，我认为 COVID 将成为住宅和商业领域现有趋势的催化剂。我喜欢把它看作是 24 个月内创新加速的十年。

疫情无疑已将注意力重新聚焦于家居生活，这一点再加上低利率，不可否认地让涉足房地产市场的公司迎来了一个不可思议的时期。鉴于我是通过在一家家庭住宅经纪公司的工作进入 proptech 的，这个市场一直是我感兴趣的一个大领域，我认为在 COVID 之前，我们正处于这个市场的一些重大转变之中。住宅交易中断现在分为三个核心类别:iBuyers(直接从卖方购买房屋，并最终希望拥有卖方市场)、neobrokers(通常雇佣他们的代理人，并使用所有权抵押和保险等二级服务来增加收入)和 elite agent tools(专注于顶级代理人的平台或工具)。此外，消费者对替代融资工具越来越开放，包括基于房屋净值的融资模式(在这种模式下，你出售房屋的股份，或者随着时间的推移，你购买房屋的全部所有权。这些新模式的增长和扩散正在巩固整个住宅市场，因此经纪销售佣金和来自抵押贷款、产权和房屋保险销售的佣金现在在功能上是一个庞大且相互交织的可破坏市场。在这种背景下，虽然我们看到的一些估值可能有点乐观，但我认为毫无疑问，我们正处于一个如何购买和持有房屋的大规模和可持续创新的时期，我认为这些趋势将会持续下去。

在商业领域，我们也正处于公司如何构思办公空间的重大转变之中，这在很多方面要归功于 WeWork 带给市场的创新。但疫情在这方面走得更远，从根本上改变了房东/经理(自 20 世纪 50 年代以来基本上一直处于权力地位)和租客(现在权力大得多，更像是奢侈品消费者)之间的关系。因此，我认为最好的房东会认识到，他们将面临压力，从简单地提供物理空间，转变为帮助租户提供多渠道的工作体验。这可能意味着系统的物理/数字混合，包括访问控制、物理空间管理(在中心位置和可能在分支位置)，以及用于会议和协作的数字空间。这些资产需要在更加人性化的环境中提供，重点是满足租户的需求。随着租期不可避免地缩短，租户将需要比过去更积极的方式来争取和支持。

你如何看待大城市适应疫情之后的生活——或者他们会这样吗？对于城市生活的未来，你特别关注哪些解决方案(合住、ADUs、商业向住宅的转化等)。)?

我认为疫情之后的城市经历将会有很大的变化——例如，奥斯汀和纽约州的轨迹可能会看起来完全不同。我个人不确定如何共同生活，甚至商业住宅转换将发挥作用，但我相信，将会有一些一致的兴趣领域。

首先，规模较小的城市生活将继续成为一个显著趋势。我认为，我们将继续看到自行车道的激增，对居住在可步行社区的关注，以及对增强我们与邻居联系感的工具(比如应用程序 nextender)的依赖。

其次，我认为我们将继续看到对某些类型的外包服务的依赖——外卖或快速供应食品杂货和其他必需品的幽灵厨房——促进这些的技术和空间将是一个增长领域。

最后，我确实相信，在一些更为分散的城市环境中，成年人将是一个增长领域，尤其是对于那些寻求额外收入来源以便在家养老的老年人来说。一个警告是，我认为这将是一个赢家通吃的空间，地理优势赢家的数量有限。

目前，对郊区生活方式的需求又回来了。在接下来的一两年里，您在这一类别中重点关注什么(新的住宅服务和便利设施，建造-租赁住房开发，等等。)?

当前从城市环境向郊区环境的转变，伴随着美国公众对待住房所有权方式的一些更广泛的变化而展开。在过去的十年里，自有住房净下降，租房净增长。其中一些是出于必然，一些是出于选择。我认为，我们现在看到的迁移中与技术相关的机会的一个组成部分将围绕租赁平台和多家庭化。在租赁方面仍然没有像 MLS 这样的数据聚合器，许多门户网站(包括 Zillow 租金、公寓列表、Zumper、RentalBeast 和其他网站)正在竞争向消费者提供清单信息。在多家庭方面，许多多家庭运营商意识到工作/生活/娱乐的结合正在为他们的产品带来新的便利和更多服务需求。我认为，我们将继续看到越来越多的重点放在酒店式的品牌创造在多家庭的空间，其中大部分将促进由不同类型的技术促进的设施。

同样重要的是，要注意机构和私人投资者的资金正在涌入独栋租赁市场。(将来可能会有一定比例的人出租他们的主要房屋，但通过拥有这些类型的房地产资产暴露于房地产市场。)这种增长是由 Mynd 等物业管理服务、Roofstock 等市场以及淄博或 Knox Financial 等创新产品推动的。
