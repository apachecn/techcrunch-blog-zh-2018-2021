巴吞筹集 1050 万美元建立“卸货区”，卡车司机可以在这里卸货 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/25/baton-raises-10-5m-to-create-drop-zones-where-truckers-can-hand-off-freight/

在卡车运输业，“停留和滞留”时间是效率、利润和司机的敌人。每年有超过 20 亿个小时的时间因为停留(在配送场地或设施上花费的时间)和滞留(应该开始卸载或装载的时间与实际开始的时间之间的差距)而损失。

Baton 是一家总部位于旧金山的初创公司，由 8VC 的孵化器项目发展而来，它已经开发了一项业务，认为它将解决卡车司机的这些长期问题。这家公司的名字暗示了它的商业模式。巴吞正在开发一个交货区网络，这是它从合作伙伴那里转租的 24 小时设施，位于繁忙的城市中心之外。长途卡车司机可以把满载的拖车停在这些卸货区。然后，Baton 与当地的 8 级卡车车队合作，到达卸货地点，抓取货物，并将货物运送到最终目的地。

这家初创公司开发了一个软件平台，通过一个 API 来协调车辆、卸货区、仓库和当地司机。随着负载的交付，客户还可以通过 API 接收实时自动更新。

“在长途货运中，浪费了大量的时间，”联合创始人安德鲁·伯贝里克在最近的一次采访中说。接力棒的卖点是，它消除了因滞留和滞留而浪费的时间，以及坐在车流中的时间。该公司表示，这也有助于增加司机的工资，并减少碳排放。司机的工资通常是按英里支付，而不是按小时支付。

巴吞已经获得了长途货运公司的客户，包括 CRST，这家私人货运公司为包括沃尔玛在内的一些美国最大的零售商运送货物。它还吸引了各种战略投资者。该公司在 2019 年 12 月结束的种子轮融资中，从房地产公司 Prologis 和 8VC 筹集了第一笔 330 万美元的资金。现在，它在一轮首轮融资中吸引了更多的资本和投资者，由 8VC 和马士基成长(Maersk Growth)共同牵头，马士基成长是物流巨头 AP Moller-Maersk 的企业风险投资部门。

Baton 在首轮融资中筹集了 1050 万美元，现在的融资后估值为 5000 万美元，联合创始人 Nate Robert 和 Berberick 告诉 TechCrunch。Prologis、Ryder、Lineage Logistics、Project44 首席执行官 Jett McCandless、KeepTruckin '首席执行官 Shoaib Makani、Clarendon Capital 运营合伙人 John Larkin、I.S.G .创始人 Trace Haggard 和 Cooley LLC 都参与了这一轮投资。

Baton 在洛杉矶有几个投放区，并计划在该市开设更多投放区。罗伯特和伯贝里克说，他们的计划是在未来 12 至 18 个月内在亚特兰大、芝加哥和达拉斯开设分店。

Baton 的短期目标是结束人类驱动的卡车运输业务中的浪费。但罗伯特表示，这种商业模式非常适合处理他所说的自动驾驶卡车的首次可行应用。“答案只有在高速公路上，”罗伯特说。"要做到这一点，你必须拥有一个全国性的转运中心网络."

Baton 已经在亚利桑那州和加利福尼亚州边境的一家未命名的自动驾驶卡车公司试点这一想法，罗伯特称之为“自动接力”。

“随着我们看到自动卡车以及最终电动卡车成为某些路线的标准，指挥棒网络及其软件提供的协调将被视为核心基础设施。Baton 使向自动化卡车运输的转型成为可能，”8VC 合伙人兼联合创始人 Jake Medwell 说。
