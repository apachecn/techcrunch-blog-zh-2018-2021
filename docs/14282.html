不可替代令牌(NFT)现在比 pog 更热门，在过去几周内发表的关于这个主题的文章数量已经激增到数千篇。因此，我们必须从一开始就乞求原谅，但代币经济被忽视的潜力实在太重要了，不能让它溜走。

在金融资本的世界里，非金融交易只是一个更大发展中的一小部分。让一些人摸不着头脑、忍俊不禁的事情，可能会在十年内彻底改变自硅谷崛起以来一直存在的投资模式。

不可替代的什么？

NFT 以一种奇怪的方式第一次站到了聚光灯下，给极少数人带来了财富，让大多数人感到困惑。在非功能性金融工具被认为是昙花一现之前，可能值得考虑的是，非功能性金融工具在传统投资框架中从来就不是非常有用的。

很难想象这一切会如何发展，但我们已经看到这种新经济的轮廓开始穿透旧模式干枯的皮肤。

一家拍卖行出售一张价值 6900 万美元的 JPEG 图片就像一个马车司机把一个小型核反应堆绑在驾驶室顶上，然后宣布:“这是一辆原子能车！”当马继续突突前进，做所有的工作。你会得到旁观者的关注，但这里没有任何根本性的改变。

最近出现的每一个吸引眼球的 NFT 销售都是这种落后思维的例子。旁观者批评马车司机，说，“核反应堆是炒作，”并没有真正看到长期的影响，或者他们只是不喜欢马。

鲸鱼、狗和独角兽

从早期将投资视为资助越洋航行的一种方式，到如今我们所知的风险资本的兴起，整个金融资本领域一直是一项精英运动。这是因为目前的模式是基于大投资者获得大收益。

几乎整个金融资本世界都是建立在大鲸鱼和独角兽之上的，这些神话般的生物仅仅是凡人能有幸一瞥就已经很幸运了。“结构化”一词在这里是经过仔细挑选的，因为资本的“大狗”理论实际上是建立在强大的中介机构之上的，这些中介机构促进了这些顶级投资者的意愿。

比特币的发明是金融发展的划时代事件。比特币本身已经变成了另一个权力的游戏场，但它所带来的技术震动正开始成为真正的游戏规则改变者。首先，分布式账本技术(DLTs)——区块链只是其中的一个例子——是一项突破，其重要性不亚于能够向世界另一端的人即时发送信息。

DLT 意味着金融资本不再需要强大的中介——或者任何类型的中介。目前，为了在交易、贸易合同或投资中建立信任，中间人是非常必要的。支付这些中间人的服务费用可以作为大公司和富人做生意的成本一笔勾销，但这些费用对许多人来说仍然是令人望而却步的障碍。

DLT 打破了这些障碍，因为信任是由网络本身的体系结构建立并构建的。有了 DLTs，任何有互联网连接的人都可以在他们能承受的任何水平上进行大狗式的商业交易，而这些交易是通过代币进行的。

代币经济将会带来变革

未来几年，所有主要投资者都将采用 DLT 经济，因为分散投资的优势太多，不容忽视——自动化带来的交易摩擦减少、更快的(实时)结果和市场状况分析、透明度带来的更高安全性以及金融产品和服务的更高定制化水平。主要参与者采用分散融资将对其他所有人产生净积极影响。

令牌是这个新系统的生命线，不可替代的令牌只是令牌的一种。在这种新兴模式中，有行为类似于货币的支付令牌，可与股票媲美的安全令牌，提供空间或带宽等功能的实用令牌，以及将这些令牌混合成新形式的混合令牌。如果这听起来有点令人困惑和兴奋，那是因为它确实如此。

这里要理解的主要观点是，代币不仅将取代股票和其他投资产品，还将取代在你和你的购买之间建立中间人的整个想法，无论中间人是投资经纪人、信用卡公司、平台提供商还是银行。分散经济将会是一个更加开放和直接的市场。

橡胶像这样碰到路面

很难想象这一切会如何发展，但我们已经看到这种新经济的轮廓开始穿透旧模式干枯的皮肤。这些突出在经济现实不合理的地方最为明显。

想想新兴的零工经济，没有人真的有稳定的工作，我们每个人都是某种职业雇佣兵，从一个零工到另一个零工。想一想我们大多数人都像磨盘一样挂在脖子上的大量订阅。想想音乐家与流媒体平台、艺术家与画廊之间矛盾而令人沮丧的关系。想想我们星球上仍然存在的极度贫困。

这些都是生活和工作模式不适合旧容器的例子。我们都能感觉到我们生活的这些方面并没有真正发挥最佳作用，但是我们说不出为什么，我们当然也不知道解决方案会是什么样子。分散化、符号化的经济有可能消除所有这些痛点、悖论和杂牌，并用更直观、更优雅的东西取而代之。

这个新的现实在的一些属性中很容易想象:你可以在你想要的时候直接为你想要的内容付费，而不是九种不同的订阅。艺术家不再将一半收入交给画廊，音乐家不再将所有收入交给流媒体平台，他们现在只是通过这类内容建立的流动网络直接支付他们的工作。现在，你可以直接投资于你感兴趣的企业，包括以前遥不可及的领域，如房地产投资，而不是付钱给经纪人来促进你的投资。我们没有打破贫困和阶级之间的界限，而是打破壁垒，让每个人都能获得价值。

代币经济的许多其他发展尚待想象，这可能是最令人兴奋的方面。当我们在全球范围内分配经济时，以一种允许任何有互联网连接的人以有意义的方式互动和贡献的方式，我们正在释放价值数万亿美元的未开发资产的价值。那么，是什么阻碍了我们，我们如何尽快实现这一目标呢？

前面的工作非常清楚

开启这种新经济的最困难的部分已经实现——我们已经从技术上理解了如何分配和分散一种共识系统，这种共识系统与贸易和投资的数字化资产系统相结合。

将这个系统实际上线的剩余工作是相当明显的——首先，我们需要看看这个新系统在其婴儿期的生态影响。我们应该彻底取缔采矿农场，或者对它们的能源中有多少来自不可再生能源设定最严格的限制。如果这个新经济的支柱正在毁灭这个星球，我们需要在它成长之前关闭它，句号。这个系统需要在生态上是可持续的。

第二个最直接的问题是，目前没有标准，没有共同的网络，众多不同的加密货币和令牌都同意。令人震惊和绝对沮丧的是，各种密码几乎没有谈论这一点。

这就好像我们有一群不同的公司不仅发明了灯泡，还发明了他们自己的灯座和接线协议，每一个公司都坚持认为他们是最好的，他们最终会胜出。灯泡很好，但是我们能不能在一个插座上达成一致？除非我们建立一个中立的、可互操作的网络，否则这个美丽的新经济永远不会起飞，而这个网络需要是无感觉的和可扩展的。

最令人担忧的是监管和法律框架。仍有太多的人处于秘密状态，他们有某种无政府主义者的死亡愿望，只是完全被排除在外，这不是为我们社区的长期目标服务的。

我完全赞成将中介机构排除在价值链之外，但这并不意味着要自动建立一个监管机构都不会被邀请去的“永不王国”。分散经济的法律框架与我们的开源、社区建设和透明运营的精神是紧密相连的。我们都需要成为对新生技术进行彻底和精确监管的倡导者。

以生态、互操作性和监管为口号，我们可以开始构建实际的应用程序和其他基础设施，让用户能够利用新经济的力量。用途是无限的，从向你的地区智能电网出售多余的电力，到投资你最喜欢的艺术家的网络，到接受对你自己的劳动的直接支付，到——是的——购买 NFT，这在新经济中更有意义。
