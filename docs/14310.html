谷歌 Nest Hub 2 回顾:固态智能屏幕增加了睡眠跟踪技术

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/30/google-nest-hub-2-review-the-solid-smart-screen-adds-sleep-tracking/

两年半后，Nest Home Hub 仍然是市场上我最喜欢的智能屏幕之一。也许这是对该类别改进速度的评论，或者也许谷歌第一次就把事情做对了。它仍然是市场上最好看的产品之一，由坚固但低调的材料制成。大小是正确的，谷歌显然在功能上花了很多心思。

当然，这些消费电子产品已经花了几十年的时间来训练我们期待产品类别的大规模年度更新(必须保持这种需求)。从这个角度来看，第二代手机有点令人失望。这里没什么新东西。该产品现在可以跟踪睡眠，扬声器的低音更加饱满。老实说，差不多就是这样。

也许新版本最吸引人的地方是谷歌的工程师如何在他们自己强加的限制下工作。我清楚地记得在一次预发布活动上看到了原始的 Google Home Hub，并询问该公司关于不包含摄像头的决定。当然，像谷歌和亚马逊这样的公司致力于在这样的设备上收集尽可能多的信息。

图片来源:布莱恩·加热器

我当时称赞该公司将摄像头从设备上移除——我很高兴地说，它在第二代产品中继续这样做(当然，随着随后发布的摄像头运动 Nest Hub Max，这变得更加容易)。我不得不说，在测试了新的 Echo Hub 之后，它实际上会跟着你在房间里移动，这只会巩固我对故意忽略的赞赏。

毫无疑问，这一决定是 Nest Hub 成为流行床边设备的重要原因。我们中的大多数人不希望在我们睡觉和做…人们在床上做的所有其他事情(吃饼干，看恐怖电影等)时，摄像机对准我们。).

设计一个已经成为流行的床边产品的第二代版本，使睡眠跟踪成为一个显而易见的事情。但有一个问题:摄像头似乎是一种非常明显的睡眠追踪方式。但增加一个摄像头几乎肯定会让人们不太愿意邀请产品进入他们的卧室。你是做什么的？如果你幸运的话，你会发现某个公司花了一大笔钱的技术，但最终却不知道该如何处理。

那么，这种事情发生的几率有多大呢？如果你是谷歌，出乎意料的高。

Soli 项目是谷歌那些怪异的异常现象之一。这是一项寻找问题的很酷的技术。我想，团队最初指出的问题是我们触摸触摸屏太多了。因此，它将这项技术内置于 Pixel 4 中，允许用户与定制的神奇宝贝游戏和其他一些东西进行互动。等到 Pixel 5 滚来滚去的时候，这项技术基本上被遗忘了。

图片来源:布莱恩·加热器

从表面上看，无摄像头睡眠跟踪是 Soli 技术的一个更符合逻辑的实现方式(假设你可以绕过在床边的电子设备中安装微型雷达的最初陌生感)。以下是来自谷歌产品页面的分类:

睡眠感应使用动作感应来跟踪离显示器最近的人的睡眠情况。通过低能量雷达，运动感知可以检测运动和呼吸。Nest Hub 中的其他传感器检测打鼾和咳嗽等声音，以及房间内的光线和温度等环境因素。这就是为什么睡眠感应不仅决定了你什么时候上床睡觉，睡了多长时间，还决定了你的睡眠质量。

当然，睡眠追踪功能并不能让这款产品在消费电子产品中独树一帜。似乎每个公司都在争先恐后地进入睡眠——这是可以理解的。我们大多数人都离得不够近——这种趋势可以追溯到新冠肺炎让我们很多人成为失眠症患者之前。然而，该产品相对独特的地方在于，它承诺在不接触你或床的情况下做到这一点。

作为一个多年来测试过数十种健身追踪器和智能手表的人，我可以证明，戴着可穿戴设备睡觉有点糟糕。我的意思是，我不戴眼镜就已经很难入睡了(如果我不戴眼镜，我可能一开始就不会对跟踪睡眠感兴趣)。

在我看来，如果你正在寻找一个纯粹的睡眠追踪器，我会看看类似 Withings 的睡眠追踪垫，它就在你的床垫下面。这是微创的，不需要在你的床边放一个屏幕。我不能仅仅基于睡眠跟踪来推荐新的 Nest Hub，但如果你已经在考虑在你的床边贴一个智能显示器，这使得市场上最好的型号之一变得更加引人注目。

图片来源:布莱恩·加热器

Soli 跟踪的一个缺点是在放置设备方面没有太大的灵活性。例如，我目前没有床头柜，所以我不得不临时用一把椅子来测试。该设备需要放在床的一侧(而不是头或脚),并与你在床垫上保持水平。当你睡觉时，屏幕应该离你一两英尺远。

还有一个校准过程，尽管它非常快，假设你不打算移动产品，你只需要做一次。这是我第一次发现自己错过了 Echo Show 的一个新功能:特别是一个你可以手动上下倾斜的屏幕。这将是一个伟大的功能，看到所有的产品，向前发展。

我已经使用睡眠追踪系统好几个晚上了，我发现它非常准确——尤其是对于离我床边几英尺远的一个产品。(剧透:我睡得像屎一样)。板载健康功能也会深入了解相关信息。除了持续时间和整体质量等标准信息，它还会告诉你晚上咳嗽了多少次，打鼾了多少分钟，以及整晚的平均呼吸频率是多少。

其他信息也是可用的，包括室温(利用一个仍然相对未被充分利用的内置温度计)和睡眠质量，按醒着/睡着/不安来分类。这是相当基本的，看看该公司最终能够利用给定的硬件深入了解多少细节将是一件有趣的事情。鉴于对呼吸健康的关注，睡眠呼吸暂停似乎是显而易见的，但这可能需要一些更新，加上监管审查。

图片来源:布莱恩·加热器

看起来，就睡眠而言，使用现有的硬件可能有很大的改进空间。既然谷歌收购 Fitbit 的交易已经正式结束，情况就更是如此了。预计在这方面会有更紧密的整合。目前，仍然有一些周到的睡眠集成，如唤醒警报和卧室智能灯功能。

在 99 美元，谷歌的要价下降了整整 50 美元，这无疑减轻了最终是一个相当小的更新的打击。在推出两年半后，Nest Hub 仍然是你能买到的最好的智能屏幕之一，它得到了谷歌可靠的助手和软件产品的支持。新版本不会是我的睡眠追踪器清单上的第一个，但如果你正在寻找一个智能显示器/闹钟，这是一个不错的奖励。
