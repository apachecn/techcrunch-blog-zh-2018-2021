Living Security 为预测性人类风险管理筹集了 1400 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/30/living-security-raises-14m-for-predicitive-human-risk-management/

网络安全培训是每个人都必须做的事情之一，但并不是每个人都期待的事情。

Living Security 是一家总部位于奥斯汀的初创公司，旨在改变网络安全培训，让你期待而不是恐惧。该公司刚刚完成了一笔 1400 万美元的 B 轮融资，以继续其从网络安全意识培训到人类风险管理的扩张。

总部位于华盛顿特区的 Updata Partners 领投了此次融资，现有投资者 Silverton Partners、Active Capital、Rain Capital 和 SaaS 风险投资公司也参与了融资。这笔投资是在去年 4 月由总部位于奥斯汀的 Silverton 牵头的 500 万美元首轮投资之后进行的。

夫妻 Drew 和 Ashley Rose 于 2017 年 6 月在他们的房子外面成立了 Living Security，其使命是通过游戏化的学习，让网络安全培训变得不那么无聊，更有效，并提供真人表演的沉浸式故事情节、基于角色的微模块和报告。

Living Security 推出旗舰产品——网络逃生室。当疫情来袭时，这家初创公司通过推出【cyber escape Online】将面对面培训带到了网上。

随着越来越多的人远程工作，考虑到有多少人出于职业原因和其他原因使用个人设备，对生活保障产品的需求变得更加迫切。员工比以往任何时候都更容易无意中提供进入他们工作的企业网络的入口点，无论是通过社会工程、网络钓鱼还是其他方法。

如今，Living Security 与 100 多家大型企业合作，培训他们的全球员工更好地保护敏感数据和组织安全。这家初创公司的客户名单令人印象深刻，包括 CVS Health、万事达卡、威瑞森、MassMutual、Biogen、AmerisourceBergen、惠普、摩根大通和 Target 等大型企业。

因此，在 2020 年，Living Security 的收入和员工人数增加了两倍，客户数量增加了一倍多，这并不奇怪。该公司拒绝提供硬收入数据，只说 ARR 去年增长了近 200%。

首席执行官 Ashley Rose 表示:“我们已经看到现有账户的增长和扩张显著增加……很大程度上是由于我们数字解决方案的可扩展性。

随着逃生室和游戏化训练的成功，Living Security 的团队开始思考如何让他们的努力“更可预测”

Rose 说:“我们增加了风险管理和评分功能，这样项目和安全负责人就可以更有针对性地专注于培训的实施。

所以现在 Living Security 的目标是使用行为数据和分析 来衡量和管理人类风险。它计划获取这些数据，并为员工提供“预测性干预”。

“我们专注于‘如何将人们从我们最大的风险转化为我们在网络安全方面最大的资产？’”罗斯说这是我们对公司的大愿景。”

形象学分:生活保障

Living Security 希望通过其“Unify”人员风险管理平台提供一个更具可扩展性的解决方案。该公司还计划利用新资本扩大其地理覆盖范围，并扩大直接和渠道销售的努力。

目前，Living Security 有 55 名员工，目标是年底前达到 90 名。

AmerisourceBergen 的信息安全培训和意识总监 Deb Walter 表示，她在 2017 年第一次接触 Living Security 时，她请求了该公司的 网络安全卡牌游戏。

她回忆道:“我想把我展示培训的方式游戏化。

沃尔特表示，在她的培训项目中引入情景游戏化及其“可绑定”的内容，在员工中大受欢迎。

“他们的新平台使我们能够部署一个‘信息安全学院’，以鼓励员工和承包商使用多种培训模式来赢取积分，并在排行榜上跟踪自己，”她说。

Updata 的普通合伙人 Jon Seeber 在 Living Security 的董事会中获得了一个席位，他表示，他的公司在这家初创公司的平台中看到了“突破的潜力”。

他说:“它尽可能地接近人们和他们所操作的系统之间的闭环。”。

他说，此外，它还避免了以合规为中心的“打勾”思维，这种思维经常主导以员工为中心的网络安全解决方案。
