Discord 正在为音频事件推出新的类似俱乐部会所的频道

原文：https://web.archive.org/web/https://techcrunch.com/2021/03/31/discord-stage-channels-voice-events/

现在每个人都在争先恐后地建造一个会所 克隆，但对于不和来说，这完全合理。

在过去的一年里，每个人都被困在家里寻找安全的方式来重新点燃他们的社交生活，Discord 的吸引力爆炸了。该公司援引其在疫情期间观察到的新行为作为 Stage Channels 的灵感，这一新功能将有助于与指定的发言者和听众进行更结构化的语音聊天。

语音聊天已经是 Discord 的核心功能。多年来一直如此，为游戏玩家提供了清晰、无缝的语音聊天服务，将游戏内聊天服务的功能打得落花流水。但不可否认的是，俱乐部会所激发的声音事件的时代精神，即使最终不和谐在许多方面都是第一次出现。

图像来源:不和谐

Discord 表示，这种新的频道将对基于语音的 ama 和访谈、读书俱乐部甚至卡拉 ok 等内容非常有用。新的频道将捕捉 Discord 上已经发生的活动，使任何运行服务器的人都可以更容易地主持正式的对话，而无需与一堆粒度用户权限的东西打交道。

新的频道类型正好符合 Discord 现有的氛围。舞台频道做标签上的事情，允许任何运行不和谐的人策划演讲者体验，并使用主持人工具来控制谁在何时获得麦克风。很像 Clubhouse(或 Zoom)，参与者可以举手发言。他们也可以悄悄溜走。

形象学分:不和谐

舞台频道将专门针对社区服务器，这是针对更大的群体不和谐。为了启用这种新的通道，服务器所有者需要将通道转换为社区服务器，如果它还不是社区服务器的话。

因为不和谐就是不和谐，基于语音的活动发现不会像 Clubhouse 那样工作，club house 将用户创建的直播活动提供给任何打开应用程序的人。Discord 上的社区服务器可以申请出现在服务器发现菜单中，但该应用程序的重点仍然是私人、亲密团体和你已经加入的更大的基于兴趣的社区。

鉴于其健康的用户群和作为休闲、无缝语音聊天的首选应用程序的现有效用，Discord 完全有能力占领一个完全不同的语音活动市场。这款应用是游戏社区的中流砥柱，通常倾向于年轻人，与早期蜂拥至 Clubhouse 的企业家、风投和品牌形成鲜明对比。

Discord 的游戏基因并没有阻止它。近年来，Discord 已经超越了它的游戏根源，并没有背叛它们，扩展到从大学学习小组到有影响力的粉丝中心的无缝聊天体验。去年， Discord 的估值在六个月内翻了一番。仅仅一个季度后，据报道，微软正在与该公司就一笔价值 100 亿美元的交易进行谈判。
