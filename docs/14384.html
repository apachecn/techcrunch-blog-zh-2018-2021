Soda 监控数据并帮助您及时解决问题

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/01/soda-monitors-data-and-helps-you-fix-issues-before-its-too-late/

认识一下 Soda ，这是一个数据监控平台，它将帮助您发现数据处理设置中的问题。这样，您可以尽可能快地做出反应，并确保掌握完整的数据信息。

如果你正在建立一个数字优先的公司，你和你的客户可能会产生大量的数据。你甚至可以利用这些数据来调整你的产品本身——想想酒店定价、在食品配送网站上找到合适的餐厅、向金融科技公司申请贷款等等。这些都是数据密集型产品。

“公司在三大云之一(亚马逊网络服务、谷歌云、微软 Azure)中建立了一个数据平台——正如他们所说的那样。Soda 联合创始人兼首席执行官马腾·马舍林(Maarten Masschelein)告诉我说:“他们把数据放在那里，供分析等使用。

然后，您可以利用这些数据湖或数据仓库来显示分析、可视化您的数据、监控您的服务等。但是，如果您的数据工作流出现问题，会发生什么呢？

你可能需要一段时间才能意识到有一些丢失的数据，或者你算错了一些东西。例如，脸书错误计算了几年来平均视频观看次数。当您发现这个问题时，您业务的一个重要部分可能会受到影响。

Soda 希望通过自动和大规模监控您的数据来尽快发现数据问题。“我们坐在更上游的地方，更靠近数据的来源，”Masschelein 说。

当你用你的数据平台设置 Soda 时，你会立即得到一些提示。苏打水会告诉你是否有什么不对劲。例如，如果您的应用程序今天只生成了 6，000 条记录，而您通常在 24 小时内生成 24，000 条记录，那么很可能出现了问题。或者，如果您通常每分钟都会获得一个新条目，而 15 分钟内没有条目，则您的数据可能不新鲜。

“但这只是被认为是数据问题的一小部分。你需要测试和验证更多的逻辑，”Masschelein 说。

Soda 允许您创建规则来测试和验证您的数据。基本上，考虑软件开发中的测试套件。当你构建一个新版本的应用程序时，你的代码需要通过几个测试，以确保新版本不会出现任何关键问题。

有了苏打水，你可以立即检查数据并得到结果。如果测试没有通过，您可以通过编程做出反应，例如，您可以停止一个进程并隔离数据。

如今，这家初创公司也在推出苏打云。这是一个协作 web 应用程序，让您可以看到整个组织中的数据流。这样，非技术人员可以很容易地浏览元数据，看看是否一切似乎都在正确地流动。

基本上，Soda 客户使用 Soda SQL，这是一个帮助人们扫描数据的命令行工具，以及 Soda Cloud，一个查看 Soda SQL 结果的 web 应用程序。

除了这些产品，Soda 的愿景是数据正在成为软件产品中的一个完整类别。开发团队现在有大量的开发工具可用于自动化测试、集成、部署、版本控制等。但是专门为数据团队设计的工具有很大的潜力。

Soda 最近筹集了 1350 万美元的首轮融资(€1150 万美元)，由总部位于巴黎的新风险投资基金 Singular 牵头，我在本周早些时候报道过。Soda 的种子投资方 Point Nine Capital、蜂鸟创投、DCF 以及各种商业天使也参与其中。
