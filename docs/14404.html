你可能刚刚错过了出售你的初创公司TechCrunch的最佳时机

欢迎回到TechCrunch Exchange，这是一份每周一期的创业和市场时事通讯。它大体上是基于《T2》杂志上的每日专栏，但是是免费的，为你的周末阅读而制作。每周六都想把它收进你的收件箱？在这里报名。

大家周六快乐。我真心希望你精神愉快，身体健康。我正在学习打盹，在我意识到新闻周期永远不会慢下来之后，这已经成为我生活中的一项要求。而且因为我和我的伙伴领养了喜欢早起的第三只狗，请和我一起让午睡成为成年人的酷事，让我们都能为疫苗夏天好好休息。快到了。

关于工作话题，我今天有几件事要告诉你，都与重要的数据点有关:Q1 2021 M&A数据，3月份来自非洲的风险投资结果，以及一些令人惊讶的(至少对我来说)播客数据。

首先，丹·普里马克通过路孚特分享了一些第一季度早期的数据，我想把它们传递给大家。根据金融数据公司的数据，2021年Q1的全球并购活动达到1.3万亿美元，比2020年的Q1增长93%。美国的并购活动在第一季度也创下了历史新高。我们为什么关心？因为这些数据有助于强调过去三个月有多热。

我预计本季度的风险投资数据本身也同样令人印象深刻。但正如本周每个人都注意到的那样，随着第二季度的开始，IPO市场出现了一些裂缝，这可能会使Q2 2021成为一个非常不同的野兽。这并不是说风险资本世界会放缓，特别是考虑到老虎基金刚刚重新注入了67亿美元。

在风险投资话题上，专注于非洲的数据公司 Briter Bridges 报告称，“仅3月份就有超过2.8亿美元被部署到在非洲运营的科技公司，”部分原因是“Flutterwave以10亿美元的估值获得了1.7亿美元的巨额融资。”

这个数据很重要，因为它标志着至少自2017年以来，非洲大陆在风险投资方面最活跃的三月——我想是有史以来最活跃的三月。非洲初创公司往往在下半年筹集更多资金，因此3月份的结果并不是单月的最高纪录。但这仍然是乐观的，并有助于满足我们的普遍情绪，即第一季度的风险投资业绩可能会很大。

最后， Index Ventures的Rex Woodbury 在tweeted上发布了一些Edison的数据，即“8000万美国人(占美国12岁以上人口的28%)是每周播客听众，同比增长17%。”这位风险投资家接着补充道，“美国12岁以上人口中的62%(约1.76亿人)是每周在线音频听众。”

正如我们本周讨论的股权问题一样，鉴于Clubhouse在最近几个月作为一家突破性消费社交公司取得的成功，许多玩家正在押注非音乐流媒体音频市场。支持Discord和Spotify等公司押注的是这些数据点。人们喜欢听其他人说话。比我想象的要多得多，作为一个音乐第一人。

回到消费者投资活跃的时代真好。B2B是伟大的，但不是一切都可以成为企业SaaS。(然而，值得注意的是，Clubhouse似乎正在努力保持自己的宣传力度。)

听着，我跟不上那些该死的风险投资

TechCrunch的早期阶段是在本周，进展相当顺利。但是有一个活动来帮助我，这确实意味着我这周跑的圈数比我希望的要少。所以，如果我有空闲时间，我会把这两个打出来:

  • 斯特瑞姆的【5000万美元C轮。高盛主导了这笔交易。Striim，我相信发音是“stream ”,是一家帮助其他公司在他们的云和本地设置中实时移动数据的软件初创公司。鉴于当今数据市场的活跃程度，我认为Striim的TAM很深？快速流动？你可以在闲暇时提供一个更好的以流为中心的词。
  • 工藤的【2100万美元A轮。去年7月，当Kudo融资600万美元时，我报道了它。该公司提供支持实时翻译的视频聊天和会议服务。你可以想象，它有一个很好的COVID时代。费利克斯在参加种子赛后领先甲A。下周我会看看能否从公司提取一些新的增长指标。值得一看。

另外两轮你也可能错过了你不应该错过的:Holler 在B轮中筹集了3600万美元。据我们自己的Anthony Ha说，“你可能不知道什么是对话媒体，但是你很有可能使用过Holler的技术。例如，如果你在Venmo支付中添加了贴纸或GIF，Holler实际上会管理该应用程序围绕该媒体的搜索和建议体验。

我觉得自己老了。

如果你对拉丁美洲的技术关注不够，这一轮1.5亿美元的乌拉圭回合应该可以帮你理清思路。

各种各样的

本周终于有了一些好消息。如果你读过《交易所》一段时间，你会被迫读到我喋喋不休地谈论贝塞麦云指数(Bessemer Cloud Index)，这是一篮子我非常尊敬的上市软件公司。现在市场上出现了一种新的指数。

满足 Lux健康+科技指数。根据Lux Capital的说法，这是一个“57家上市公司的指数，它们最能代表迅速崛起的健康+科技投资主题。”当然，这在某种程度上是有烙印的，类似于Bessemer系列，它与支持风险投资公司的特定焦点相关联。但新的Lux指数将做的是，像Bessemer集合一样，跟踪特定风险公司本身是如何跟踪其投资组合的公共投资组合的。

这是一个有用的东西。请再来点这个。
