Meesho周一表示，它已在软银愿景基金(SoftBank Vision Fund 2)牵头的新一轮融资中筹集了3亿美元，这家印度社交商务初创公司正努力成为“让所有小企业都能在网上成功的单一生态系统”。

新一轮——a轮E轮——对这家成立五年的初创公司的估值为21亿美元，高于2019年D轮投资的约6亿美元至7亿美元。这家印度初创公司迄今已筹集了约4.9亿美元，它表示，现有投资者脸书、Prosus Ventures、顺为资本、Venture Highway和Knollwood Investment也参与了新一轮投资。

这笔交易似乎是顺为资本近一年来对印度初创公司的首笔投资。去年，T4出台了一项规定，要求中国投资者在给印度公司开支票之前必须得到印度政府的批准。

总部位于班加罗尔的Meesho运营着一个与它同名的在线市场,在社交媒体平台如WhatsApp、脸书和Instagram上连接卖家和客户。它提供的服务包括订单管理、物流管理、在线支付、实时店铺更新，并允许企业让客户订阅。

这家初创公司声称拥有一个超过1300万名企业家的网络，其中大多数是女性，超过10万名供应商遍布近5000个印度城镇，主要经营杂货、服装、家用电器和电子产品。

Meesho表示，它将部署新的资金来帮助该国1亿个人和小企业进行网上销售。Meesho的联合创始人兼首席执行官维迪特·阿特雷(Vidit Aatrey)在一份声明中表示:“在过去的一年里，我们看到了寻求将业务转移到网上的小企业和企业家的巨大增长。

在COVID促使新德里发布长达一个月的封锁期间，Meesho推出了Farmiso，这是一款允许任何人在网上开设在线杂货店而无需任何投资的产品。Aatrey周一表示，Farmiso已经成为Meesho增长最快的业务。(在疫情之前，Meesho也开始在东南亚扩张，但最近几个月,已经缩减了这些努力。)

“在过去的18个月里，我们一直在密切跟踪Meesho，他们的增长、日常参与度指标、对单位经济的关注以及打造强大团队的能力给我们留下了深刻的印象。我们相信Meesho为中小企业供应商和社交经销商提供了一个有效的平台，让他们参与印度的电子商务革命，并帮助他们为消费者提供个性化的体验，”软银投资顾问公司的合伙人Sumer Juneja在一份声明中说。

在最近的一份报告中，瑞银(UBS)分析师认为，社交商务和B2B市场是亚马逊(Amazon)和Flipkart等印度电子商务公司的潜在竞争来源。

图片来源: Meesho
