新加坡职业平台 Glints 获得 2250 万美元的 C 轮融资

总部位于新加坡的职业平台 glitts 今天宣布，它已经获得了由日本人力资源管理公司 PERSOL Holdings 领投的 2250 万美元 C 轮融资。新资金将用于格灵思在新加坡、印度尼西亚、越南和台湾的扩张，以及为其产品和工程团队招聘员工。

Glints 联合创始人兼首席执行官 Oswald Yeo 表示，这是东南亚人才平台迄今为止最大的一轮融资，使这家初创公司的总融资额达到 3300 万美元。其他与会者包括回归投资者 Monk's Hill Ventures、Fresco Capital、Mindworks Ventures、Wavemaker Partners、Flipkart 联合创始人 Binny Bansal 和前高盛 TMT 中国负责人兼合伙人张。

成立于 2013 年的格林斯表示，它已经被超过 150 万名专业人士和 3 万个组织使用，包括 Gojek，Tokopedia，星巴克和 Mediacorp。Yeo 告诉 TechCrunch，其当前的大部分用户来自科技和金融服务行业，但 Glints“广泛横向关注年轻至中级专业人士”，其长期目标是不区分行业。

与活跃在其市场上的其他求职平台(如 LinkedIn、JobStreet 和 CakeResume)不同的一个方面是，为希望推进职业发展的人建立了一个“全栈”服务。该公司表示，除了每月有超过 7000 个活跃列表和 400 万访客的就业市场，格灵思还提供社区功能和技能教育，如在线课程。

格灵思的价值主张之一是帮助公司，尤其是科技公司，应对地区人才短缺问题，这也是它最近在蒙克希尔风险投资公司的综合报告中提到的一个话题。

该报告强调的解决方案之一是在不同的东南亚国家雇佣团队，以解决特定市场的人才短缺问题，如新加坡。格林斯表示，其跨境远程工作中心 TalentHub 的业务在 2020 年翻了一番，因为疫情也让雇主对远程招聘更加开放。
