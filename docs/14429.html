StockX 如何成为炒作的股市

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/05/stockx-ec1-origin/

俗话说，“找一份你喜欢做的工作，你一生中不会有一天是在工作的”，可以肯定这是在优步、“plandids”和股票市场出现之前。StockX 可能是一个价值数十亿美元的庞然大物，在今天的运动鞋文化中具有巨大的影响力，但它始于将个人激情转化为商业计划的飞跃。

对于创始人乔希·卢伯(Josh Luber)来说，将他对运动鞋的热爱与他的职业生涯分开，一开始是非常有意的。当他继续投资于他的爱好时，他看到了他在公司工作中完全没有的东西——数据。随着他建立并深入研究这些数字，一个完全不同的愿景出现了。一场篮球赛，一张支票，一笔生意后来，StockX 诞生了。

最初作为在线销售的基本价格图表，比初创公司 unicorn 更多地尖叫着 Microsoft Excel，现在已经成为世界上最有趣的市场之一。

时机非常巧合。在过去的十年里，运动鞋从一个上升的小众逐渐成为一种狂热，对认证商品的需求也同样飙升。很少有其他公司将市场有效运作所需的核心机制整合在一起。最初作为在线销售的基本价格图表，比初创公司 unicorn 更多地尖叫着 Microsoft Excel，现在已经成为世界上最有趣的市场之一。

运动鞋值多少钱？

在联合创立 StockX 之前，乔希·卢伯在 IBM 做顾问，故意在运动鞋之外工作，把它作为一种严格的爱好。这种设置一直持续到他意识到有机会围绕他心爱的收藏组织数据。

市场没有价格就无法存在，十年前一双运动鞋在二级市场的价格是很难辨别的。当然，零售价格是存在的，但流行的运动鞋往往会随着时间的推移基于需求而增值，而需求可能会随着时间的推移而大幅波动。通过收集易贝上超过 1300 万笔交易的公开数据，Luber 和一个由 17 名志愿者组成的团队建立了 Campless，这是一个不断更新的运动鞋二级市场定价指南，于 2012 年推出。

“虽然它有很多缺陷，需要大量的手工工作，但它可能是当时最好的参考点，”首席运营官和 StockX 的联合创始人 Greg Schwartz 说。无营团队只是简单地从易贝的封闭式拍卖中获取价格，并从中分析趋势，就像任何个人卖家在发布自己的鞋子之前可能会做的那样。然而，通过扩大数据集的规模，他们获得了比以前买卖双方都能获得的更准确的市场出清价格。

类似于汽车行业的凯利蓝皮书(Kelley Blue Book ),该书按车型和年份提供汽车的估计价值，Campless 提供了二手运动鞋市场的深入数据，这些数据最终成为 StockX 的 tentpole 和专有产品。

商场上空的直升机和运动鞋热潮的混乱兴起

当 Luber 和他的团队推出 Campless 时，购买限量版运动鞋并不容易。发烧友们可以通过排队和在店内扎营的方式直接从零售商那里购买，在易贝寻找看起来最合法、价格最优惠的卖家，或者通过外挂或后门渠道获得他们珍贵的鞋子。这三种选择都令人担忧。

乔希卢伯在 TechCrunch 扰乱纽约 2017 年。

Campless 的名字和“知道得更多，营地更少”的标语指的是消费者为了最新、最令人垂涎的运动鞋发布而露营——有时要排队好几天。飞行俱乐部于 2005 年在纽约市开业，最初是为了寄售，通常出售罕见的旧鞋，而不是新鞋。然而，对于个人经销商来说，易贝和 Craigslist 是他们自行决定建立一对一交易的唯一选择，这两个平台都没有必要的运动鞋认证或价格监管措施。

对于一个相对较小的利基市场来说，这个支离破碎的系统可能已经足够了。但自 20 世纪 80 年代以来，它的受欢迎程度一直在稳步上升，2010 年代的规模甚至更大。

在 2014 年接受易贝采访时，卢伯分享了这一时期的重要性，指出一款鞋引起了这一类别受欢迎程度的巨大变化:2012 年 2 月 NBA 全明星周末发布的耐克“银河”Foamposite，这是勒布朗·詹姆斯、彭妮·哈达威、阿玛尔·斯塔德迈尔和凯文·杜兰特等篮球巨星穿着的天体主题包的一部分。

可信的运动鞋博客独家收藏者称这个特别的版本“过去十年中最混乱的运动鞋版本之一”，因为它引起了“全国范围内的骚乱，因为运动鞋爱好者拼命想得到他们的鞋子。”

2012 年，迈克尔·乔丹和神奇小子 23 一起参加乔丹全明星赛。图片来源:耐克(Alexander Tamargo/WireImage)

24 岁的运动鞋爱好者马克·萨比努说:“那绝对是我第一次记得除了我的一群爱鞋的朋友之外的人谈论‘下降’。

运动鞋和街头服饰生活品牌 Kith 的电子商务总监安迪·奥利弗(Andy Oliver)也认为这是一个转折点。“我认为这是正确模型的结合——泡沫是超级热的——与当时真正独一无二的图形处理。然后，从营销角度来看，它与全明星周末密切相关，这在 2012 年是一件大事。当每个人都开始觉得自己大部分都高不可攀时，他们就在另一个层面爆发了。”

布伦丹·邓恩是 Complex Media 的运动鞋展全尺寸 Run 的联合主持人，他说这次发布为混乱和炒作树立了一个新的标杆。“我认为直升机在他们发布的奥兰多购物中心上空飞行的图像是持久的图像。”

一个自 20 世纪 80 年代就存在的社区被打入了主流社会。然而，随着 Instagram 的日益流行，Luber 称之为“限量版运动鞋收藏”的活动更加火上浇油。运动鞋爱好者第一次可以与整个世界分享他们的最爱，向潜在的数百万人展示他们的稀有发现，而不仅仅是他们的朋友。确保全明星周末的下跌不仅意味着冷静，而且意味着全球冷静，加剧了对即将到来的需求规模完全没有准备的市场的压力。
