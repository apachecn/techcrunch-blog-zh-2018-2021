最高法院在谷歌与甲骨文 TechCrunch 的史诗版权战中站在了谷歌一边

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/05/supreme-court-google-oracle-ruling/

最高法院在谷歌与甲骨文史诗般的版权之战中站在了谷歌一边

本周，美国最高法院有很多关于科技的话要说。周一，最高法院对谷歌与甲骨文的长期法律战进行了权衡，推翻了甲骨文公司先前的胜利，后者本可以获得 80 亿美元的赔偿。

在一项 6 比 2 的裁决中，法院裁定谷歌将甲骨文的 Java 软件语言整合到自己的移动操作系统中并没有违反版权法。谷歌抄袭了甲骨文公司为安卓系统编写的 Java APIs 代码，此案引发了一场长达数年的关于既有 API 重用和版权的争论。

2018 年，一家联邦上诉法院裁定，谷歌使用 API 实际上违反了版权法，其实施不属于合理使用。

“在回顾这一决定时，为了便于讨论，我们假设该材料是受版权保护的。但是我们认为本案中的复制仍然构成了合理使用。因此，谷歌的抄袭并没有违反版权法，”大法官 Stephen Breyer 在判决中写道，该判决推翻了甲骨文之前的胜利。塞缪尔·阿利托和克拉伦斯·托马斯法官不同意。

布雷耶写道:“谷歌复制 Java SE API，只包括那些允许程序员将其积累的才能用于新的变革性程序所需的代码行，从法律角度来看，这是对该材料的合理使用。”

谷歌全球事务部的 SVP 肯特·沃克称这一裁决是“创新、互操作性和计算的重大胜利”
