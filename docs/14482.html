浮潜人工智能获得 3500 万美元 B 轮投资，实现机器学习中的数据自动标注

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/07/snorkel-ai-scores-35m-series-b-to-automate-data-labeling-in-machine-learning-apps/

机器学习的一个更乏味的方面是提供一组标签来教导机器学习模型它需要知道什么。潜泳人工智能希望让主题专家更容易以编程方式应用这些标签，今天这家初创公司宣布了一项 3500 万美元的 b 轮投资。

它还宣布了一个名为 Application Studio 的新工具，该工具提供了一种使用模板和预定义组件构建常见机器学习应用的方法。

光速创投(Lightspeed Venture Partners)领投，之前的投资者 Greylock、GV、In-Q-Tel 和 Nepenthe Capital 也参与了这轮投资。新投资者 Walden 和 BlackRock 也加入了进来。这家初创公司报告称，它现在已经筹集了 5000 万美元。

公司联合创始人兼首席执行官亚历克斯·拉特纳(Alex Ratner)表示，数据标签仍然是推动机器学习和人工智能在许多行业向前发展的巨大挑战和障碍，因为它成本高昂，劳动密集型，学科专家很难抽出时间来做这件事。

“今天关于人工智能的不那么隐藏的秘密是，尽管所有的技术和工具都在进步，但一个普通的人工智能项目大约 80%到 90%的成本和时间只是人工标记和收集并重新标记这些训练数据，”他说。

他说，他的公司已经开发了一种简化这一过程的解决方案，使主题专家更容易以编程方式添加标签，他说，这一过程以一种非常戏剧性的方式将应用标签所需的时间和精力从几个月减少到几个小时或几天，这取决于数据的复杂性。

随着该公司开发出这种方法，客户一直在寻求机器学习过程下一步的帮助，即获取训练数据和模型，并构建应用程序。这就是 Application Studio 的用武之地。它可以是银行的合同分类器，也可以是电信公司的网络异常检测器，帮助公司在数据标记后采取下一步措施。

他说:“这不仅仅是关于如何以编程方式标记数据，还关于模型、预处理程序、后处理程序，所以我们现在已经在一种模板化和可视化的无代码界面中实现了这一点。”。

该公司的产品基于 2015 年在斯坦福人工智能实验室开始的研究。在 2019 年推出浮潜之前，创始人在研究阶段花了四年时间。如今，这家初创公司有 40 名员工。拉特纳从多元化的角度认识到科技行业存在的问题，并表示他已经有意识地努力建立一个多元化和包容性的公司。

“我能说的是，我们从第一天起就试图在公司层面、整个团队层面和董事会层面优先考虑这件事，并采取行动。因此，我们一直在与外部公司合作，围绕 DEI 进行内部培训、审计和战略，我们已经将渠道多样性作为我们与招聘公司签订的任何合同中不可协商的要求，”他说。

拉特纳还认识到，自动化可以将偏见硬编码到机器学习模型中，他希望通过简化标记过程，可以更容易地检测到发生的偏见。

“如果你从十几个或二十几个我们称之为标记函数的函数开始，你仍然需要保持警惕并积极主动地尝试检测偏差，但更容易的是通过回过头来查看数百行代码来审计是什么教会了你的模型改变它。”
