1Password收购SecretHub并推出新的企业机密管理工具

与 LastPass 和 BitWarden 等公司竞争的密码管理服务Password 今天宣布了一项超越密码管理基础的重大举措，进军基础设施机密管理领域。为此，该公司收购了机密管理服务 SecretHub ，并正在推出其新的1密码机密自动化服务。

1Password没有披露收购价格。据CrunchBase报道，总部位于荷兰的SecretHub在今天的声明之前从未筹集过任何机构资金。

对于像1Password这样的公司来说，进入企业领域——并为个人用户及其日益复杂的基础设施服务管理公司凭证、API令牌、密钥和证书——似乎是一个自然的举动。通过将1Password与其新的Secrets Automation service相结合，企业可以使用一个工具涵盖从管理员工密码到处理基础设施机密的所有内容。1Password目前在全球80，000多家企业中使用，其中许多肯定也是其Secrets Automation服务的潜在用户。

1Password 的首席执行官杰夫·希纳(Jeff Shiner)说:“公司需要尽可能多地保护他们的基础设施机密，如果不超过他们员工的密码的话。有了 1Password 和Secrets Automation，就有了保护、管理和编排您所有商业秘密的单一信息来源。我们是第一家以一种重要且易于使用的方式将人类和机器秘密结合在一起的公司。"

除了收购和新服务，1Password今天还宣布了与GitHub的新合作伙伴关系。全球最大、最先进的开发平台GitHub的合作伙伴工程和开发副总裁Dana Lawson表示:“我们正在与 1Password 合作，因为他们的跨平台解决方案将使开发人员和安全团队的工作变得更加轻松。“随着即将到来的GitHub和1 passwordSecrets Automation集成，团队将能够完全自动化他们所有的基础架构机密，完全放心他们是安全的。”
