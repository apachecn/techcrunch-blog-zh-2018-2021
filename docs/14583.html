ConsenSys 从摩根大通、万事达卡和瑞银筹集 6500 万美元，为 DeFi TechCrunch 建设基础设施

加密领域的主要参与者和区块链以太坊的主要支持者 ConsenSys 已经从摩根大通、万事达卡和瑞银集团以及区块链主要公司 Protocol Labs、Maker Foundation、Fenbushi、LAO 和 Alameda Research 筹集了 6500 万美元的资金。其他投资者包括 CMT 数码和大湾区国土发展基金。除了菲亚特，还有几只基金投资了总部位于以太坊的 stablecoins，戴和，作为对价。

消息人士告诉 TechCrunch，由于估值风险，这是一轮未定价的融资，融资工具“已满”，因此这轮融资现在正在关闭。

此次融资看起来是一次高度战略性的融资，其理念是传统机构将需要了解影响力越来越大的“去中心化金融”(DeFi)世界，以及以太坊区块链正在开发的 Web3 应用。

在一份关于融资的声明中，ConsenSys 表示，它已经经历了一个“战略演变和增长期”，但大多数外部观察人士会同意这是一种轻描淡写的说法。

委婉地说，在经历了一段相当多的“创造性破坏”之后(几年前，ConsenSys 似乎拥有一切，从风险投资基金到加速器再到多个创业公司)，该公司已经重组为两个主要部门:核心软件业务 ConsenSys；以及投资机构 ConsenSys Mesh、孵化器和投资组合。它还从摩根大通(J.P. Morgan)收购了 Quorum 产品，这使它能够更深入地了解区块链企业生态系统。这意味着它现在有一个非常关键的以太网平台产品套件，包括 Codefi、Diligence、Infura、MetaMask、Truffle 和 Quorum 等产品。

这个套件允许它服务于公共和私人许可的区块链网络。它还可以支持第二层以太网，以及方便访问相邻的协议，如 IPFS，Filecoin 和其他协议。ConsenSys 也是以太坊 2.0 项目的主要贡献者，原因显而易见。

在评论这次融资时，ConsenSys 的创始人约瑟夫·卢宾(Joseph Lubin)在一份声明中说:“当我们开始筹集资金时，耐心地构建一个多元化的上限表对我们来说很重要，这与我们的信念一致，即类似于网络的发展，整个经济将加入下一代协议的革命者。ConsenSys 的软件堆栈代表了对新的自动化客观信任基础的访问，该基础由以太坊等分散协议实现。对于 ConsenSys 和整个行业来说，在这个特别令人兴奋的增长时期，我们很荣幸能够与领先的加密公司和卓越的金融公司合作，进一步融合集中和分散的金融领域。"

由于公共链，金融机构能够看到以太坊上发生的“公开”定义，他们可以看到多少金融系统正在逐渐开始与区块链世界融合。因此，吸引这些主要机构的因素变得越来越清晰。

瑞银集团技术主管迈克·达根(Mike Dargan)表示:“我们对 ConsenSys 的投资为我们的瑞银下一个投资组合增加了分布式账本技术方面的成熟专业知识。”

对于万事达卡来说，这似乎不仅仅是一项纯粹的投资——ConsenSys 一直在与它合作开发一个私人许可网络。

万事达卡数字资产和区块链产品及合作伙伴执行副总裁 Raj Dhamodharan 表示:“企业以太坊是一个关键的基础设施，我们和我们的合作伙伴正在以此为基础构建支付和非支付应用，以推动未来的商业发展……我们的投资和与 ConsenSys 的合作有助于我们为客户带来安全和高性能的企业以太坊功能。”

CMT 数字公司的联合创始人兼首席执行官 Colleen Sullivan 表示:“ConsenSys 是弥合传统金融、集中式加密和 DeFi 之间差距的先驱，更广泛地说，是 Web 2.0 和 Web 3.0 之间差距的先驱。我们很自豪能够参与这一轮融资，因为 ConsenSys 团队继续为全球用户(零售和机构用户)轻松访问加密生态系统铺平道路。”

TechCrunch 了解到，融资大约是在去年 6 月收购 Quorum 的时候开始的。这一轮 6500 万美元的融资主要是法定货币，而不是加密货币，是去年夏天摩根大通完成的一轮融资的补充。

Maker Protocol Labs 等重要加密公司的出现，表明了融资的重要性，超越了简单的交易。该公告也是在比特币基地 IPO 之前发布的，这是一个有趣的时机。

ConsenSys 的产品在开发者、企业和消费者与区块链和 crypto 相遇的世界中变得非常重要。在声明中，该公司声称 MetaMask 现在拥有超过 300 万移动和桌面月活跃用户，在过去五六个月内增长了 3 倍。这与比特币基地的月活跃客户数量大致相当。

康赛思的公告是在比特币基地 IPO 之前发布的。虽然比特币基地作为一个交易所，将菲亚特变成了密码，反之亦然，但它最近也陷入了困境。与 ConsenSys 也有相似之处的是，拥有 300 万用户的比特币基地被用作钱包，同样拥有 300 万用户的 MetMask 也可以被用作钱包。比较就到此为止，但考虑到比特币基地 1000 亿美元的估值，这当然很有趣。

正如首席开发官杰里米·米勒(Jeremy Millar)告诉我的那样:“比特币基地在世界上监管最严格的金融市场之一美国开创了一个交易所，它帮助推动了人们对这个领域的浓厚兴趣。我们与比特币基地保持着非常积极的关系，试图进一步促进生态系统和技术的采用。”

这次融资的背景是，许多早期区块链和加密公司最近筹集了大量资金，但其中大部分是通过加密投资公司进行的。只有少数硅谷风投支持区块链，比如安德森·霍洛维茨。

这份声明有趣的地方在于，这些现任金融巨头不仅对此感兴趣，还与 ConsenSys 合作，在以太坊上投资和制造产品。

ConsenSys 认为，每个支付服务提供商在未来都需要这种金融基础设施，尤其是对 DeFI 而言。

鉴于 DeFi 中有大约 430 亿美元的抵押贷款，主要投资者参与的情况越来越多，而且回报率也越来越高，高于传统收益率和债券或债券收益率。

各国央行转向数字货币的举措，也在迫使企业和政府实现数字货币，而数字货币运行的“区块链轨道”将会继续存在。这是大湾区家园发展基金(深港合资企业)决定参与的建议。

这个故事的另一个方面是，ConsenSys 正坐拥一些极其强大的产品。ConsenSys 有六种产品，服务于三种不同类型的人群。

基于以太坊的服务开发者正在使用 Truffle 开发智能合约。加入 NFT 炒作的用户正在使用 MetaMask。

元掩码钱包允许用户将一个令牌换成另一个令牌。事实证明，这对 ConsenSys 来说非常有利可图，该公司表示，这导致了 18 亿美元的分散式交易所使用量。ConsenSys 从其提供的每笔掉期交易中抽取 0.875%的佣金。

各机构都在使用 ConsenSys 的产品。该公司表示，超过 15 万名开发人员使用 Infura 的 API，450 万名开发人员使用 Truffle 创建和部署智能合同，而其协议组 Hyperledger Besu 和 ConsenSys Quorum 的开发人员——正在为六家央行建立央行数字货币(cbdc ), ConsenSys 表示。

ConsenSys 也在利用 NFT 的繁荣。开发人员将 ConsenSys 产品用于以太坊上存储 NFT 文件的节点和基础设施。

ConsenSys 也在乘风破浪。一个是开发者浪潮，一个是金融系统浪潮。

正如一位发言人所说:“对金钱和发明的兴趣始于以太坊这样的公共网络。所以我们真的相信这些正在融合，并将继续融合，我们的每一款产品都提供公共主网兼容性，因为我们认为这是未来的趋势。”

米勒补充说:“如果我们想帮助世界采用技术，我们需要在采用点满足它，这对许多大企业来说意味着首先在防火墙内。但同样，我们认为，就像公共互联网一样，真正的价值——颠覆性的价值——改变了在更广泛的无许可基础上这样做的能力，特别是当你有足够的隐私和认证可用时。”
