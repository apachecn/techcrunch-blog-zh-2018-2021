美国联邦调查局发起行动，清除被黑客攻击的微软交换服务器的后门

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/13/fbi-launches-operation-to-remotely-remove-microsoft-exchange-server-backdoors/

美国联邦调查局发起行动，清除被黑客攻击的微软交换服务器的后门

休斯顿的一家法院授权联邦调查局的一项行动“复制并移除”美国数百个微软电子邮件服务器的后门，几个月前，黑客利用 T2 的四个以前未发现的漏洞 T3 攻击了数千个网络。

司法部周二宣布了这次行动，称其“成功”

今年 3 月，微软发现了一个新的中国政府支持的黑客组织——铪——以公司网络上运行的 Exchange 服务器为目标。当这四个漏洞连接在一起时，黑客就可以侵入易受攻击的 Exchange 服务器并窃取其内容。微软修复了漏洞，但补丁并没有关闭已经被攻破的服务器的后门。几天之内，其他黑客组织开始用同样的漏洞攻击易受攻击的服务器，部署勒索软件。

应用修补程序时丢弃的受感染服务器的数量。但司法部在一份声明中表示，数百台 Exchange 服务器仍然容易受到攻击，因为后门很难找到并消除。

“这次行动清除了一个早期黑客组织剩余的网络外壳，这些外壳可能被用来维持和升级对美国网络的持续、未经授权的访问，”声明说。FBI 通过 web shell 向服务器发出命令来执行删除，该命令旨在使服务器仅删除 web shell(由其唯一的文件路径标识)

联邦调查局表示，它正试图通过电子邮件通知被移除后门的服务器的所有者。

助理司法部长约翰·c·德默斯(John C. Demers)表示，此次行动“展示了司法部利用我们所有的法律工具，而不仅仅是起诉，来扰乱黑客活动的承诺。”

司法部还表示，此次行动只移除了后门，但没有修补黑客一开始就利用的漏洞，也没有移除留下的任何恶意软件。

据信，这是已知的第一起 FBI 在网络攻击后有效清理私人网络的案件。2016 年，最高法院采取行动，允许美国法官在其辖区外发布搜查和扣押令。批评者当时反对这一举措，担心联邦调查局会要求友好法庭授权在世界任何地方开展网络行动。

其他国家，如法国，在劫持僵尸网络并远程关闭它之前，已经使用了类似的权力。

截至记者发稿时，联邦调查局和司法部都没有发表评论。
