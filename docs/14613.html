为什么总部位于南非的汽车订购公司Planet42要实现碳中和

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/13/why-south-africa-based-car-subscription-company-planet42-is-going-carbon-neutral/

自从优步和博尔特这样的打车公司颠覆了交通运输行业，交通拥堵和污染就成了他们的一大心病 。 研究显示打车出行比私家车出行排放更多 。

为了减少碳足迹并解决后一个问题，两家公司都提出了拼车和其他交通模式的想法，如自行车和滑板车共享服务 。他们还考虑整合公共交通调度，鼓励司机转用电动汽车。 然而 ，这些模型发现很少或根本没有成功。

因此在2018年，Lyft决定更进一步，承诺 实现 碳中和。 据 The Atlantic 报道，该公司计划通过从位于旧金山 的可持续发展公司【3 degrees】购买碳信用额来执行这项计划。

2019年， Lyft表示它已经减少了240万英亩树木在一年内清除的碳量 。它通过购买2，062，500公吨的碳补偿实现了这一目标，但该公司在2020年又回到了老路。

但是，虽然该计划使Lyft骑行碳中和，但这是一个昂贵的过程。该公司声称，从长远来看，其游乐设施的净排放量仍将增加。因此Lyft承诺到2030年只提供电动汽车。 世界上大多数汽车公司都是如此，每个公司都承诺在未来通过电动汽车 达到 碳中和 。

然而 ，总部位于南非的汽车公司Planet希望现在就实现碳中和，而不是以后 。但Planet42不是一家叫车公司。 提供从经销商处购买二手车，通过订阅模式 出租给客户的汽车订阅产品。

Planet42由Marten Orgna和Eerik Oja创建，面向新兴市场的个人，但只在非洲 有业务。 该公司已经在南非购买了近3000辆汽车，并计划在未来几年内将这一数字增加到10万辆——到2025年 全球将达到100万辆。

因此，尽管Planet42不是一家叫车公司，并且因为它向原本不会拥有汽车的人提供汽车而产生了巨大的积极社会影响，但由于其汽车产生的排放物，planet 42对环境的负面影响有限 。

当大多数汽车公司似乎对实现碳中和无动于衷时，Planet42在研究它如何间接造成排放时，去年决定采取行动 。

Oja告诉TechCrunch “很少有人会认为实现碳中和是一个不值得的目标，但在我们看来，世界向碳中和迈进的速度还不够快。“因此，我们没有推出一个到2040年实现碳中和的空洞宏伟计划，而是决定现在就实现碳中和。”

图片来源: Planet42

因为在非洲几乎没有什么大众消费的电动汽车，而且植树成本很高，所以该公司是如何着手的？

在帮助Lyft开始植树项目之前，3Degrees参与了几个风力农场，并从垃圾填埋项目中捕获了温室气体 。Planet42选择通过前者来中和其碳排放；然而，它与南非的当地公司合作实现这一目标。

它的第一个项目是南非北开普省的一个风力发电场；有了碳补偿信用的钱，Planet42已经能够为农场提供几个月的资金。 风力涡轮机产生的电力抵消了其他更有害的能源生产方式，如烧煤和支持低碳全球经济 。

“我们通过在 运营的市场中投资碳抵消项目来抵消这种负面影响。 换句话说，我们对碳中和的投资代表一种自我强加的税收 。我们以身作则，希望非洲和其他地区的公司能效仿我们。”

这家公司已经筹集了2000万美元的债务和股权，当它第一次启动时，实现碳中和甚至不是一个事后的想法。但是现在，它不仅被自然资本合伙人认证为碳中和公司，它的投资者也对这个项目充满热情。

Oja表示，公司的下一步将是通过电动汽车最终实现碳中和 。 然 ，那可能是一个致远星。电动汽车在非洲的采用面临着与美国、欧洲甚至其他新兴市场不同的问题。 首先想到的是电力供应不可靠的严峻形势。 然后就是普遍缺乏税收优惠、补贴和政策，唯一的事实就是对一般非洲车主来说太贵 。

例如，美国道路上有超过100万辆电动汽车，英国道路上有超过31.7万辆电动汽车。 在南非——planet 42的主要市场和非洲顶级电动汽车市场，这个数字正好是1000辆左右。因此，在电动汽车成为主流之前，风力发电场仍将是该公司碳中和努力的关键。

理想情况下， 我们能做的就是让我们的汽车电动化，这就是我们对未来的规划。当我们这样做的时候，就不需要每天都进行补偿，但是我们现在还没有做到。 大家都明白 最终 电动车会比较理想；然而，未来不是现在，我们需要现在就采取行动。
