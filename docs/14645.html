Triller 所有者通过收购 Amplify 获得了新的首席执行官。AI；还收购了直播服务 FITE TV

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/14/triller-owner-gets-a-new-ceo-with-acquisition-of-amplify-ai-also-acquires-live-streaming-service-fite-tv/amp/

图片来源:谢尔顿·库珀/SOPA 图片社/光火箭/盖蒂图片社

将成为抖音的竞争对手 Triller，由母公司 TrillerNet 运营，将获得一个新的首席执行官，该公司今天宣布。这个简短的视频应用程序表示，将收购一个基于人工智能的客户参与平台 Amplify。艾，他的联合创始人 Mahi de Silva 现在将成为 TrillerNet 的首席执行官。现任首席执行官 Mike Lu 将调任 TrillerNet 总裁，并将专注于投资者关系。该公司另外宣布收购 FITE TV，这是一个直播活动和按次付费的战斗体育流媒体平台。

新任首席执行官 Mahi de Silva 在今天之前一直与 Triller 关系密切。该公司今天的新闻稿称，他自 2016 年以来一直担任非执行董事长，但他的 LinkedIn 记录的年份是 2019 年(这将是继 Triller 2019 年由 Proxima Media 提供的资金之后，当时的新闻稿指出他正在担任“董事长”的角色。)当我们寻求澄清时，该公司发现这两种说法都是错误的。正确年份是 2018 年。

在收购之前，德席尔瓦一直担任 Amplify 的首席执行官和联合创始人。AI 自 2017 年以来一直担任 Opera Mediaworks 的首席执行官，Opera media works 是 Opera Software 的营销和广告部门，也是 Botworx 的联合创始人兼首席执行官。

放大。AI 与 CPG、金融服务、汽车、电信、政治和数字媒体等领域的品牌合作，在交易完成后，将继续作为 TrillerNet 的子公司运营。其他团队成员包括前 RSA 和 VeriSign 高管 Ram Moskovitz，他帮助设计和开发了 SSL 和代码签名的数字证书；Amplify.ai 的联合创始人兼首席技术官 Manoj Malhotra 是 B2C 短信服务的先驱。

TrillerNet 今天还宣布收购另一项战略资产，以帮助其业务进一步转向直播活动:FITE TV。该公司表示，这笔交易让 Triller 在直播活动和按次付费流媒体市场获得了更多立足点。因此，拥有 1000 万用户的 FITE 将成为所有 Triller 搏击俱乐部拳击赛事的独家数字分销商。

“收购 FITE 是 Triller 更大战略的一部分，旨在第一次将内容、创作者和商业聚集在一起，这是他们真正互动的唯一地方，”Triller 的瑞恩·卡瓦劳格说，他是电影工作室相对论传媒(和争议人物)的前负责人，他的比邻星传媒在 2019 年成为 Triller 的最大投资者。“我们已经投资了数亿美元，相信我们已经创造了一个更好更高效的电子商务内容平台，”他补充道。

在这次收购之前，TrillerNet 已经进行了其他几项扩展到现场活动的收购，现在在美国成为抖音的替代品不再是一个可行的选择，因为拜登政府暂停了特朗普禁令。Triller 还在 3 月份收购了由 Swizz Beats 和提姆巴兰创立的直播音乐流媒体平台 Verzuz。它还与 Snoop Dogg 合作运营 Triller Flight Club，以及流媒体平台 Triller TV。

虽然具体的交易条款没有透露，但 Triller 告诉 TechCrunch，它已经在收购上总共花费了 2.5 亿美元，包括 Halogen，Mashtraxx，Verzuz，FITE 和 Amplify today。
