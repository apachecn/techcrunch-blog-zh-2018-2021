特斯拉车主现在可以看到他们的电动汽车需要多少太阳能或煤了 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/16/tesla-owners-can-now-see-how-much-solar-or-coal-is-powering-their-evs/

特斯拉车主现在可以确切地看到是什么样的能源在为他们的电动汽车提供动力。TezLab 是一个免费应用程序，就像特斯拉汽车的 Fitbit，本周推出了一个新功能，显示来自充电地点的能源组合——分解化石燃料和可再生能源的确切类型和比例，包括美国各地的超级充电器和第三方网络。

TezLab 的首席执行官兼联合创始人本·希珀斯(Ben Schippers)在最近的一次采访中解释说:“我们正在跟踪与能源相关的数据来源，所以我们知道你是在图森还是布鲁克林(或任何其他地方)，能源来自哪里，以及这些能源的组合是什么样的。”。“因此，我们可以根据你的充电情况，看看有多少碳被排放到大气中，不管你是在家里充电，还是在超级充电站充电。”

来自明日的项目 ElectricityMap 提供了能源数据，TezLab 随后将其整合到面向消费者的应用程序中。一旦下载，该应用程序就知道特斯拉车主何时何地插电。能源组合功能建立在应用程序的现有程序基础上，该程序为车主提供了更多关于他们的充电有多脏或多干净的一般信息。

图片来源:TezLab

以特斯拉在拉斯韦加斯的 Linq High Roller Supercharger 为例，这是一款 V3 增压器，据称支持高达 250 千瓦的峰值功率，并因其使用特斯拉太阳能电池板和 Powerpack 电池来产生和储存操作充电器所需的电力而受到欢迎。

根据 TezLab 的数据，1.7%的能源来自太阳能。可再生能源的主要来源实际上是水电，占 65.6%，这要归功于胡佛大坝。增压器剩余的能源组合大约 33%是天然气。

位于加州霍桑的特斯拉超级充电站是首批安装太阳能电池板的汽车之一，其能源组合包括 0.2%的太阳能、5.5%的核能、13.3%的天然气、27%的煤炭和 49.9%的风能。

前 10 名“最清洁”的增压器——包括森特勒利亚、莱文沃斯、摩西湖和华盛顿州西雅图——由于水力发电而实现了这一目标。太阳能最多的增压器都位于加利福尼亚的同一个电网中。加州巴斯托、奥克斯纳德、卡巴松、圣地亚哥、莫哈韦、因约肯、圣马特奥、海滨和圣安娜的增压器都有 22.7%的太阳能和 15%的风能。这些地方的剩余组合是 0.2%的电池存储、2.9%的生物质能、5.6%的地热、6.3%的水电、6.6%的核能和 40%的天然气。

tez lab诞生于 HappyFunCorp ，这是一家软件工程商店，为移动、网络、可穿戴设备和物联网设备开发应用，客户包括亚马逊、脸书和 Twitter，以及一系列初创公司。HFC 的工程师，包括联合创始人 Schippers(现为公司董事会主席)和 William Schenk，被特斯拉吸引的主要原因是其软件驱动的方法。该小组对 Tesla API 的开放性所创造的机会特别感兴趣。特斯拉 API 在技术上是私有的。但是外部人员可以访问端点。通过逆向工程，第三方应用程序可以直接与 API 通信。

TezLab 于 2018 年推出了一些初始功能，让车主可以跟踪他们的效率、总行驶里程，并使用它来控制车辆的某些功能，如锁定和解锁车门以及供暖和空调。增加了更多功能，主要集中在建立社区，包括允许特斯拉车主评估超级充电站的功能。

所有这些数据都是匿名汇总的。TezLab 表示不会出售这些数据。它确实在网站上发布了从这些数据中收集到的见解，比如车型所有权的细分、平均行程长度和平均插电间隔时间。

随着其他电动汽车进入市场，TezLab 正在将它们添加到应用程序中，包括福特野马 Mach-E。
