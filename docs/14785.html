Hive 为基于人工智能的 API 筹集了 8500 万美元，以帮助调节内容、识别对象等

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/21/hive-raises-85m-for-ai-based-apis-to-help-moderate-content-identify-objects-and-more/

随着内容审核继续成为社交媒体平台如何工作的一个关键方面——他们可能会迫使纠正，或至少在处理方面做得更好——一家初创公司已经建立了一套数据和图像模型来帮助完成这一任务，以及任何其他需要自动检测对象或文本的任务，该公司宣布了一轮大规模融资。

Hive 已经基于全球约 200 万人的众包贡献建立了一个训练数据宝库，然后为一套可用于自动识别物体图像、单词和短语的 API 提供动力——这一过程不仅用于内容审核平台，还用于构建自主系统、后台数据处理等的算法——已经筹集了 8500 万美元的资金，这家初创公司已经确认其现在的价值为 20 亿美元。

Hive 的联合创始人兼首席执行官凯文·郭(Kevin Guo)表示:“我们所做的核心工作是建立人工智能模型，帮助自动化过去由人工完成的工作。”“我们听说过 RPA 和其他工作流自动化，这也很重要，但也已经确定的是，有些事情是非常结构化的，但这些系统实际上无法处理大量其他非结构化工作。”Hive 的模型有助于为其他工作带来结构，郭声称它们提供了“接近人类水平的准确性”

Glynn Capital 领投 5000 万美元 D 轮，General Catalyst、Tomales Bay Capital、Jericho Capital、Bain & Company 和其他未具名投资者参与。Hive 还确认了 2020 年由 Tomales Bay Capital 牵头的 3500 万美元的 C 轮投资，其中包括来自贝恩公司和 Visa 的新战略投资。该公司现在已经筹集了 1.21 亿美元。

该公司自 2017 年成立以来一直受到一定程度的关注，这似乎是创始人凯文·郭之前创业的一个支点，这是一个名为 Kiwi 的问答平台，它本身是他在斯坦福大学期间的一个项目的产品。但自那以后，它已经悄悄地吸引了一些有趣的客户，包括 Reddit、Yubo、Chatroulette、Omegle 和 Tango，以及 NBC 环球、Interpublic Group、沃尔玛、Visa、百威英博等。它总共有大约 100 个客户，去年增长了 300%以上。

Hive 从图像识别开始，并与建立自主系统的公司合作。事实上，如果你和郭讨论 Zoom，你很可能会看到一些作品的截图，背景是汽车飞驰过金门大桥。

然而，这些天来，Hive 的大多数活动(原谅这个双关语)都围绕着适度，其中一些包括图像，但其他包括文本和流媒体音频——这些内容被转换成文本，然后被适度地进行适度。(自动驾驶汽车模型仍然被用作背景，我相信，因为它比内容适度的图像更少干扰，正如你在下面看到的。)

部分是因为这是一个非常经典的问题，你可以想象它将通过人工智能的使用得到解决或帮助，部分是因为它是当今互联网上的一个大问题，有许多其他初创公司建立了平台来帮助管理在线滥用，包括骚扰，并帮助内容节制。

它们包括像 Sentropy 、 Block Party 、l light和 Spectrum Labs 这样的公司，更不用说大型科技公司自己在内部开发的许多工具了。(例如，Instagram 就在今天发布了最新工具，帮助用户打击域名系统中的滥用行为:该公司告诉我，它在内部构建了整个系统。)

但正如郭凯文(Kevin Guo)所描述的那样，可以说，让 Hive 与众不同的是人群。在过去的几年里，该公司通过众包大约 200 万用户的反馈，慢慢建立了一个数据宝库，这些用户获得了报酬——无论是“正常”的钱还是比特币——来查看各种图像和文本项目，以识别“滥用”或其他事情。(郭说，比特币最初是一种边缘产品，现在贡献者获得报酬的大部分来自比特币。)

该数据库反过来为 Hive 的客户使用的一组 API 提供动力，以帮助他们运行自己的审核工具，或任何需要频繁和快速识别的工作流。

目前，该系统中的大多数语言学习都是基于英语和其他几种流行的全球语言，如西班牙语和法语。部分资金将用于帮助扩大其覆盖范围和全球覆盖面，包括更广泛的语言。这也为 Hive 构建的数据和技术带来了更广泛的用例。

其中之一，郭说，包括一种新的广告方法，这种方法的基础是提供与你刚刚在屏幕上读到或看到的内容相关的广告。非常 GDPR 友好，因为它绝对不涉及基于你或你的在线浏览活动的数据(匿名或不匿名)，这正在吸引一些品牌，这些品牌最初可能来 Hive 帮助保护他们的知识产权或声誉管理，现在正在考虑如何使用该工具以更有效的方式传播关于他们自己的信息。

Hive 的人工智能在未来如何使用的可能性是今天吸引投资的一部分。对它如何在云中构建的关注强调了这种可扩展性。

“云计算近年来获得了巨大的采用，但目前只有一小部分公司利用基于云的机器学习解决方案，”格林资本(Glynn Capital)负责人查理·弗里德兰(Charlie Friedland)在一份声明中表示。“我们相信云托管的机器学习模型将代表未来几年云增长的最重要组成部分之一，Hive 在该领域处于领先地位。”

值得注意的是，至少目前 Hive 没有披露其客户中有任何大型科技公司。这可能在一定程度上是由于新发展企业，但郭指出，他们的内部活动，包括大量的人工参与，使他们到目前为止不太愿意成为客户。然而，这种情况可能会发生变化，不仅仅是因为人工智能工具正在改进，还因为当前一些路线出现的问题，例如关于社交媒体内容版主和他们所面临的创伤的争议故事。

就未来的交易而言，这些可能来自 Hive 的一些战略支持者和战略合作伙伴。该公司目前与 Cognizant、康姆斯克和贝恩(贝恩是投资者)等公司合作，这些公司反过来为选择外包一些人力调节工作的大型科技公司提供咨询和服务。无论这些人类主持人是否改变做法，在试图为制定和坚持滥用政策提供更多结构的更大过程中，技术可能会发挥越来越大的作用。
