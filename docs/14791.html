尼日利亚 fintech Okra 在埃森哲风险投资公司和 Susa 风险投资公司的支持下融资 350 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/21/nigerian-fintech-okra-raises-3-5m-backed-by-accenture-ventures-and-susa-ventures/

过去五年里，尼日利亚(以及整个非洲)的金融科技应用以惊人的速度增长 。 但是这些公司和开发者大多发现很难访问实时的银行数据 。这反过来又在客户入职和验证时造成了瓶颈。

自 2019 年以来，格子风格的公司已经出现，但它们的产品有不同的变化，以解决这些问题 。 今天，尼日利亚的秋葵， 可以说是最先获得主流关注的 ，宣布它已经完成了一轮 350 万美元的种子轮 。

总部位于美国的 Susa Ventures 领投了最新一轮投资。 其他投资者包括 TLcom Capital(2020 年 100 万美元 pre-seed round 的唯一投资者)、新加入埃森哲风投的以及一些天使投资人 。Okra 在两轮融资中共筹集了 450 万美元，该公司将利用这笔投资在尼日利亚 扩展其数据基础设施 。

Okra 喜欢将自己描述为一个 API“超级连接器”，它创建了一个安全的门户和流程，以便在客户、应用程序和银行之间交换实时财务信息 。

Fara Ashiru Jituboh 和 David Peterside 于 2019 年 6 月创立了这家公司。 自 2020 年 1 月推出以来，Okra 通过连接尼日利亚的所有银行积极推进，甚至声称有 99.9%的保证正常运行时间 。

其商业模式为开发商和企业提供整合到现有银行服务中，并从 后续 交易 中抽取佣金。这些集成包括账户授权、余额、身份、收入、支付和交易。 按合作伙伴(开发者和企业)来说，他们远远超过 100 家，其中不乏一些知名企业，如 Access Bank、Aella、Interswitch 和 uLesson 。

Ashiru Jituboh 告诉 TechCrunch，除了制作 API，Okra 还从事销售“数字优先-体验和转变”的业务。

她说 “我们正在建立一个开放的金融基础设施，使开发商和企业能够提供数字优先的体验和金融产品。 “我们正处于这样一个阶段，企业意识到数字化转型是大多数董事会议中最热门的话题之一 。 对我们来说，我们本质上是只是 利用我们的 API制造实现大规模数字化转型所需的工具和服务。”

以这样的方式定位公司可能是它在一年多的时间里取得巨大增长的原因。该公司表示，API 调用量平均月环比增长 281%。 秋葵也分析过 2000 多万的交易；上个月，在超过 550 万笔交易中，它分析了这个数字的 27.5%。 为了一点上下文，格子在它存在的八年里分析了超过 100 亿的交易 。

首席运营官·彼得赛德补充道:“我认为就牵引力而言，这是一个很好的迹象，表明我们正处于正确的轨道上。

如果说过去两年人们能从尼日利亚金融科技生态系统中学到什么的话，那就是随着增长而来的是监管审查。 自去年以来，该国一些金融机构的不同监管举措 针对支付、加密和财富科技初创公司。虽然这些监管机构声称要促进尼日利亚公众的利益并保护消费者，但他们的举动散发着扼杀创新和司法游戏的气味。

到目前为止，这些监管机构似乎对 API fintech 基础设施初创公司的活动并不。但如果这种情况发生变化，这些公司准备好应对了吗？

根据彼得赛德的说法，Okra 正在采取主动，并在其领域内与监管机构合作，为不可预见的情况做准备。 自 2018 年欧盟发布《一般数据保护条例》(GDPR)以处理数据保护和由此产生的违规行为以来，大多数非洲国家都为其所在地区制定了这些法律 。在尼日利亚，有尼日利亚数据保护条例(NDPR)，由于与 GDPR 的相似性，Peterside 认为 Okra 没有什么可担心的——至少现在没有。

就 而言法律是怎么说的， 我认为 的细则很清楚不是 只是 在尼日利亚而是在全球范围内，所以我们作为一家企业如何运营是直截了当的 。但就我们的想法而言，监管者是否做出了必要的决定……我们不能真的谈论这个，但总体而言，法律和全球标准是明确的，”他说。

如果公司成功地控制了有害的法规，它就可以随心所欲地发展。 然而 据首席执行官称，可能威胁这一速度的一个祸根是招聘。“我认为我们面临的一个挑战是招聘，”Ashiru Jituboh 说。

现在，尽管才运营了一年多，但秋葵证明其吸引力的重要原因之一是它如何优先考虑速度 。 该公司声称在 24 小时或更短时间内让新客户上线，同时通过特定于其产品的用例为他们提供支持 。

客户的增加意味着问题的增加，这意味着需要更多的人来处理这些问题。因此，除了利用最近的检查来扩展其在尼日利亚的数据基础设施之外，Okra 还将投入相当大的一部分来寻找人才 。

“我们希望确保尽快解决客户的问题，并给予客户所需的支持 。我们希望确保我们的招聘速度与我们的增长速度一致，我认为能够筹集资金是解决这个问题的方法之一……确保我们带来优秀的人才，建立一支优秀的团队，”她补充道。

Ashiru Jituboh 了解对优秀工程人才的需求，因为她有很强的工程背景。 在与彼得赛德一起创办 Okra 之前，她曾与摩根大通、富达投资和戴姆勒奔驰 合作。在 Okra，她同时担任首席执行官和首席技术官，宣称自己是非洲男性主导的金融科技领域最有前途的创始人之一。

TLcom Capital 的高级合伙人 Omobola Johnson 认为，这些品质和 Okra 的提议使该公司成为其第一笔金融科技投资。这足以说服该公司在这一轮 中继续跟进。

一年过去了，秋葵成功地让它的投资者名单更加令人印象深刻。其主要投资者 Susa Ventures 已经对 Robinhood、Flexport 和 Fast 进行了引人注目的早期投资。然而，Okra 是风投公司在安德拉以外投资的唯一一家非洲初创企业。

Susa 的普通合伙人 Seth Berman 说 “我们很高兴与 Okra 合作，因为他们使非洲大陆的开发者能够转变数字金融服务。 “我们被 Okra 团队的质量、开发速度以及客户对其 API 的兴奋所震撼。”

作为《财富》世界 500 强企业的一部分，埃森哲风险投资公司已经投资了 30 多家初创公司。 然而 ，Okra 是其投资组合中第一个黑人创立的创业公司。该公司的总裁兼董事总经理 Tom Lounibos 表示，投资背后的原因是与 Okra 合作将开放金融带到非洲，创始人的能力和他们的技术。

创始人告诉我，埃森哲和苏萨代表着精明的投资者与 Okra 的愿景和技术基础设施计划 保持一致。

这位首席执行官说:“对我们来说，如果我们正在为非洲大陆建立一个 API 基础设施，我们认为埃森哲将是一个真正好的合作伙伴，因为我们本质上是在建立一个基于技术的 API 基础设施。

除了 之外，投资者对于公司的招聘以及即将在肯尼亚和南非实施的泛非扩张计划也至关重要，Okra 目前正在肯尼亚和南非进行测试 。

埃森哲作为投资者加入 Okra 标志着一系列主要公司加入非洲金融科技浪潮的最新一步——收购 Paystack 和 Visa，以及与 Flutterwave 建立 WorldPay 合作关系 。

在 投资方面，埃森哲风险投资公司延续了美国首次投资非洲金融科技 的名单。想到的名字有 Chipper 中的 贝索斯远征队，Flutterwave 中的老虎环球和 Avenir 成长资本，Kuda 中的梵拉。

除了 Susa 和 Accenture Ventures，Okra 还带来了三位天使投资人。 Rob Solomon，GoFundMe 董事长，Accel 前合伙人；以及罗宾汉公司的两位前创始工程师——阿潘·沙阿和·钟。

Okra 并不是唯一一家希望利用 API 金融基础设施领域 的公司。另一家南非 API fintech 公司stitch 获得了400 万美元的资助，从此销声匿迹。Pngme 在二月份筹集了 300 万美元。 其他的像尼日利亚的 Mono 和 OnePipe 已经筹集了六位数的种子前融资，而 得到了 Y Combinator 和tech stars 的支持。

尽管看起来存在竞争，但基础设施行业不同于商品化行业，它有很多赢家的空间。
