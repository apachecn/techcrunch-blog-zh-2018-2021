在Basis Set Ventures，风险资本和软件开发的合并产生了1.65亿美元的新基金

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/22/at-basis-set-ventures-merging-venture-capital-and-software-development-yields-a-165-million-new-fund/

当兰第一次成立Basis Set Ventures时，其目标是利用技术赋予风险资本投资超能力。

从最早开始，当兰聘请前TechCrunch记者约翰·曼尼斯(John Mannes)，然后与数据科学家雷切尔·黄(Rachel Wong)、前advanced Ventures合伙人常旭(Chang Xu)和前Opendoor增长副总裁希拉·瓦希(Sheila Vashee)一起建立团队时，他们关注的重点是技术产品，以及美元投资和咨询服务。

Basis Set与前顾问、现担任公司首席技术官的王妮妮(Niniane Wang)一起，将其决策建立在兰公司的技术诚信基础之上。公司对产品开发不只是口头上说说而已。据合作伙伴称，Basis Set每周向一套庞大的内部和外部产品发布大约三种不同的更新。

这种产品开发是该公司得以保持相对精简的一个原因，也是该公司在筹集第二只基金时获得两倍超额认购的原因。这只基金价值1.65亿美元，最近刚刚结束。

该基金的规模并没有比Basis Set之前筹集的1.36亿美元大多少，但兰认为这个规模符合她的目标，即向她的有限合伙人返还大量资本。

该公司的Pascal系统等内部技术程序允许Basis Set审查大约9000项不同的软件交易，开发该公司核心论点中的技术，即人工智能软件即服务和企业对企业交易。

图片来源: Getty Images

根据兰的说法，在接下来的一年里，这些技术服务将开始支付红利，其形式是一些即将上市的首次公开募股。

由于其软件提供的覆盖支持，Basis Set不受地理位置的限制。“帕斯卡是在湾区以外寻找公司的主要资产，”兰说。

一旦这些公司被确定并纳入投资组合，这些初创公司就可以使用一个名为 HyperGrowth 的工具，该工具将科技公司与来自湾区的导师联系起来，帮助这些公司扩大规模。这是Basis Set产品驱动方法的另一个例子，Lan说。

“当我创办BSV的时候，我们是一个很棒的技术团队和技术人员。每个人都回来问我们如何发展，如何做销售，如何做技术，如何进行首次招聘，介绍客户，介绍顾问。公司的第一需求是走向市场，”兰说。“随着时间的推移，我们开始扩大这项工作。手动介绍，然后举办活动，然后引进一个成长伙伴，希拉。我们开发了这个工具，数百名顾问可以选择加入投资组合公司。”

目前，Basis Set至少有三个不同的项目，都旨在招募和培养典型的湾区公司传统上不关注的人才。第一个是它的持久性平台，旨在帮助女性开发者和创始人建立关系网，培养人脉，培养创意。该公司还有一项名为创始人超能力的服务，帮助创业者在寻找额外工具增强能力的同时，发现并发展优势领域。

"我们很早就有了一个数据科学团队，并且我们已经开始自动化许多事情。这些是到达终点线的人。因为我们的经营方式，我们在不断迭代，”兰说。

Mannes说，该公司的另一个关键因素是试图从典型的硅谷传记中找到更多不同背景的创始人。

Mannes说:“我们谈论了很多如何在湾区之外找到创始人……那些不在蓝瓶24-7闲逛的企业家。Mannes和Lan都表示，持久平台、创始人Superpowers工具以及Basis Set与旨在支持技术社区中代表性不足的成员的组织 Dev Color 所做的工作都是这一点的体现。

图片来源: Getty Images / sorbetto

事务所当前投资组合中的公司包括: Ergeon ，这有助于简化建筑行业的人员配备流程；自主除草和作物喷粉农业科技创业、农事；还有昆斯，Basis Set将其工厂直销模式戏称为反亚马逊。

同时，像对话式机器学习公司，Rasa； 隐私与合规自动化工具包开发者，DataGrail；《工作流自动化软件开发者》面向无桌工作者，Workstream；以及为团队提供技术基础设施支持的Assembled，都筹集了大量后续资金。

该公司强调，所有这些投资都是由技术团队的工作和与投资者的合作支撑的。

瓦西说:“我们完全不知道我们需要建造什么，我们每天都在发货。”。“这就是一流的样子，也是我们努力实现的目标……这是钢铁侠套装。这就是我们希望用这里的产品构建的。我们的目标是尽可能自动化流程的每一个部分，同时与我们的创始人建立共鸣。”
