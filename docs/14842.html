Window Snyder的新创业公司Thistle Technologies筹集了250万美元的种子资金来保护物联网设备

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/22/thistle-technology-seed-security-iot/amp/

图片来源: 豪尔赫·桑斯/ SOPA图片 /盖蒂图片

物联网有一个安全问题。过去十年见证了一波又一波新的互联网连接设备，从传感器到网络摄像头和智能家居技术，这些设备通常是批量生产的，但很少(如果有的话)考虑安全性。更糟糕的是，许多设备制造商没有努力修复安全漏洞，而其他制造商则干脆忽略了提供补丁所需的软件更新机制。

这导致了一大批不安全、不可修复的设备出现故障，当它们出现故障或不可避免地遭到黑客攻击时，注定会被扔掉。

安全老手Window Snyder认为有更好的方法。她的新创业公司蓟科技得到了True Ventures 250万美元的种子资金支持，目标是帮助物联网制造商可靠、安全地向他们的设备提供软件更新。

斯奈德去年创立了蓟，并以这种有尖锐刺的开花植物命名，刺的设计是为了防止动物吃掉它们。“这是一种防御机制，”斯奈德告诉TechCrunch，这是一个适合防御性技术公司的名字。这家初创公司旨在帮助没有人员或资源的设备制造商将更新机制集成到他们的设备软件中，以便接收安全更新，更好地抵御安全威胁。

“我们正在开发一些方法，这样他们就不用自己动手了。他们无论如何都想花时间构建面向客户的功能，”Snyder说。在创办蓟之前，斯奈德曾在苹果、英特尔和微软担任高级网络安全职位，也曾在Mozilla、Square和Fastly担任首席安全官。

蓟在物联网最需要的时候出现在安全领域。众所周知，僵尸网络运营商会扫描互联网，寻找默认密码较弱的设备，并劫持它们的互联网连接，用大量互联网流量打击受害者，使整个网站和网络离线。2016年，Mirai未来组合僵尸网络对互联网基础设施巨头Dyn发起了创纪录的分布式拒绝服务攻击，导致一些最大的网站——Shopify、SoundCloud、Spotify、Twitter——离线数小时。在攻击发生时，Mirai未来组合已经将数以千计的物联网设备纳入其网络。

其他恶意黑客将物联网设备作为进入受害者网络的一种方式，允许他们从内部发起攻击或植入恶意软件。

由于设备制造商没有采取什么措施来解决他们之间的安全问题，立法者正在考虑立法来遏制默认制造商犯下的一些更严重的安全错误，如使用默认密码(通常不可更改)，以及销售无法提供安全更新的设备。

加州在2018年通过物联网安全法后为铺平了道路，在2019年不久后紧随其后。美国没有管理基本物联网安全标准的联邦法律。

Snyder表示，推动引入物联网网络安全法可能是“人们遵守法规的一种简单方式”，而不必雇用大批安全工程师。拥有更新机制也有助于让物联网设备存在更长时间——可能更长时间——只需能够推送修复和新功能。

“构建基础设施，让您能够继续让这些设备具有弹性，并通过软件提供新功能，这对这些设备制造商来说是一个不可思议的机会。因此，我正在建立一个安全基础设施公司来支持这种安全需求，”她说。

Snyder表示，随着种子期的到来，公司将专注于招聘设备和后端工程师和产品经理，并与设备制造商建立新的合作伙伴关系。

True Ventures的联合创始人菲尔·布莱克(Phil Black)是蓟的种子轮投资者，他将该公司描述为“安全技术领域精明而自然的下一步。”他补充道:“Window拥有我们在创始人身上寻找的许多品质。她拥有深厚的领域专业知识，在安全社区中备受尊敬，并被推动行业发展的深厚热情所驱使。”
