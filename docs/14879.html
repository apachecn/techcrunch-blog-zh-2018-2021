Cloudflare 召集军队击退另一个所谓的专利巨魔

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/26/cloudflare-rallies-the-troops-to-fight-off-another-so-called-patent-troll/

大约四年前，我们报道了总部位于旧金山的互联网安全和性能公司 Cloudflare 与黑鸟科技(Blackbird Technologies)之间的一场战斗，后者迅速积累了数十项专利，然后开始利用这些专利对包括 Cloudflare 在内的公司提起数十起专利侵权诉讼。

这套衣服在各方面都很典型，除了 Cloudflare 对它的反应。与许多选择和解的类似诉讼的目标不同，Cloudflare 进行了反击，非常公开地要求帮助找到现有技术，这不仅会使 Blackbird 用来起诉 Cloudflare 的广泛专利无效，而且会使 Blackbird 的所有专利无效。公众响应了这一号召，两年后，275 个独特的提交，对 Cloudflare 的案件被驳回，黑鸟的业务减少。

有人可能会猜测，鉴于 Cloudflare 引起的争议，其他专利钓鱼者可能会选择一个更容易的目标。然而，上个月，Cloudflare 再次被起诉，这一次是由 Sable Networks 起诉，Cloudflare 的总法律顾问 Doug Kramer 说，“这家公司似乎在近 10 年内没有经营过真正的业务——依赖的专利与我们的业务性质或我们提供的服务不太接近”。

不出所料，Cloudflare 不会轻易采取这一最新行动。今天早上，在公开披露诉讼后，它邀请工程界再次“扭转对专利巨魔的局面，邀请他们参与众包努力，寻找现有技术的证据，以使“古老的、20 年前的专利”无效，Cloudlflare 说，赛布尔“试图延伸……超出他们原本打算覆盖的光年。”

Cloudflare 还提供了 10 万美元的奖金，在提供最有用的现有技术参考资料的参赛者之间进行分配，这些参考资料可用于质疑 Sable 所有专利的有效性，而不仅仅是那些针对 Cloudflare 的专利。这个想法是给 Sable 一个足够大的打击，不仅是它对 Cloudflare 的诉讼受阻，而且未来对其他实体的诉讼也会受阻。

克雷默告诉 TechCrunch:“我们感到幸运的是，在我们的历史中，我们没有遇到这种情况，否则它可能真的会让我们偏离我们的道路。”“当我们有了更多的稳定性时，Blackbird 出现了，”他指出。随着 Cloudflare 今天变得更加强大，该公司甚至更有能力"迫使[Sable]定义他们的主张，并坚持他们的主张，以便为其他人留下一些东西，特别是可能落后于我们的较小公司。我们希望对(赛布尔的)整个专利组合进行审查。”

事实上，Cloudflare 并不是唯一一家吸引 Sable 注意的公司。快速搜索显示，赛布尔还起诉了网络安全业务 Fortinet、数据平台 Splunk、网络巨头瞻博网络(Juniper Networks)和思科系统(Cisco Systems)，以及大约十几家其他公司。

其中八起案件——包括瞻博和思科——已经解决。现实情况是，大多数公司认为像赛布尔这样的非执业实体的侵权案件是一种麻烦，需要迅速解决，因为它们会分散注意力，而且打官司的费用往往等于甚至超过和解的成本。

这些公司也经常亏损。尽管在 2017 年，最高法院一致裁定，起诉企业的专利持有人不能寻求友好的法院——他们选择的法院长期以来一直是德克萨斯州的东区，仅在 2015 年就有【2,500 起案件，其中 95%是由 Sable 这样的机构发起的——德克萨斯州的业务仍然活跃，法律团队带来了大量资金，经常成功地将大公司作为恶棍提交给当地陪审员。

休斯顿纪事报去年的一篇报道指出，2020 年前六个月，德克萨斯的企业和个人提交了 747 份专利投诉——是一年前的两倍，是其他州的两倍。为了强调这一点，它指出，尽管去年上半年全国专利侵权诉讼增加了 16%，但德克萨斯州的新纠纷数量飙升了 96%。

其中一些病例发生在德克萨斯州的东部地区(大部分发生在拥有 23，000 人口的德克萨斯州马歇尔市)。一些落在了覆盖休斯顿的南区，根据同一份《休斯顿纪事报》的报道，去年新的专利侵权案件激增了 43%。

但是德州西部的韦科区已经成为专利侵权案件的新中心。这主要是因为该地区包括奥斯汀，许多科技公司在那里设有办事处，值得注意的是，2017 年最高法院裁决的一个关键部分限制了被告实际运营的地点。

所谓的专利巨魔似乎也在美国地区法官艾伦·奥尔布赖特(Alan Albright)身上找到了朋友，这位前出庭律师在 2018 年被前总统唐纳德·特朗普(Donald Trump)提名为韦科联邦法官。在获得参议院批准后的两年里，奥尔布赖特开始主持全国最受欢迎的法院进行知识产权纠纷诉讼，原告胜诉的比例非常高。

难怪世界上的秘密网络继续沿着他们的道路前进。尽早达成和解协议可以带来大生意。(他们的持续成功也是诉讼融资基金继续涌现的原因。)

Cloudflare 现在也是一个更大的猎物。虽然 Blackbird 起诉了 Cloudflare，但它仍被视为一家崭露头角的创业公司，它于 2019 年上市，目前市值为 260 亿美元。

克莱默敏锐地意识到前方的战斗。这就是为什么尽管有资源，Cloudflare 又一次向公众伸出了手。“我不想听起来自私，但我们在这个领域有一群非常热情的工程师和人们，他们定期阅读[我们的]博客，”克雷默谈到他今天早上发表的与此案有关的详细帖子。我也认为这确实触动了一些人的神经，因为他们被专利侵权诉讼的做法所困扰。

Kramer 表示，在 Cloudflare 与黑鸟的斗争中，不可能夸大这些分散在各地的工程师的影响。“不仅仅是人们认为，‘哦，这是一个赚钱的机会，我要去做这件事。’(他们提交的东西)没有很多垃圾。相反，我们让人们说，‘嘿，听着，我 90 年代在这家公司工作时就在研究这个，他们试图说他们发明了这个，这太疯狂了，’他们会把他们写的文章发给我们。

“我们有人在图书馆和类似的地方做研究，”克莱默继续说，“但我们也有在这个行业工作过的人说，‘在他们获得专利之前，我已经研究了三年；他们不可能因为我的所作所为而制造出这种麻烦。"

至于 Cloudflare 是否能再次让其追随者充满活力，以及“同样合适的人，他们受到这一点的激励，并且在这一领域非常非常了解，”Kramer 说，很快就会变得清楚。但对于公司来说，它非常“希望让这帮人重新走到一起。”

鉴于其雄心的规模，Cloudflare 似乎比以往任何时候都更需要帮助。
