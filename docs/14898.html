Weav 筹集 430 万美元，为商业平台打造通用 API TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/26/weav-raises-4-3m-to-knit-together-a-universal-api-for-commerce-platfofrms/

Weav 是一家为商业平台开发通用 API 的公司，如今正悄然崛起，从一群投资者那里获得了 430 万美元的资金，并与 Brex 建立了合作伙伴关系。

Weav 于去年由工程师 Ambika Acharya、Avikam Agur 和 Nadav Lidor 在参与 W20 YC 批次后成立，加入了旨在推动金融科技和金融机构发展的金融科技基础设施公司浪潮。具体来说，Weav 的嵌入式技术旨在让这些组织能够访问“实时的、用户许可的”商业数据，他们可以使用这些数据为小型企业创建新的金融产品。

其产品允许其客户通过一个 API 连接到多个平台，该 API 是专门为商业平台开发的 ，企业使用该平台销售产品和接受支付。Weav 的运作前提是，允许公司建立和嵌入新的金融产品，为电子商务商人、创造者和其他企业家创造新的机会。

从左至右:联合创始人安比卡·阿查里亚、纳达夫·利多尔和阿维卡姆·阿古尔；图片由 Weav 提供

在很短的时间内，Weav 取得了令人瞩目的发展。最近，Brex 针对使用 Weav API 的 Shopify 卖家推出了 即时支付 。它支持 Stripe、Square、Shopify 和 PayPal 等平台集成。(稍后将详细介绍。Lidor 表示，自从其 API 在 1 月份上线以来，“数千”家企业已经使用了基于 Weav 基础设施的新产品和服务。他说，其 API 调用量逐月增长 300%。

这家初创公司吸引了许多知名投资者的关注，包括机构和知名金融科技公司的创始人。Foundation Capital 领投了 430 万美元的种子轮，参与投资的还有 Y Combinator、Abstract Ventures、Box Group、LocalGlobe、Operator Partners、Commerce Ventures 和 SV Angel。

许多创始人和高管也投了钱，包括 Brex 创始人 Henrique Dubugras 和 Pedro FranceschiRamp 创始人卡里姆·阿提耶；Digits 创始人 Jeff Seibert 和 Wayne ChangHatch 创始人汤森阮；GoCardless 创始人马特·罗宾逊和首席运营官·卡洛斯·贡萨雷斯-卡德纳斯；凭证创始人山姆·霍奇斯；Plaid 的查利·马(Charley Ma)以及 Square、Modern Treasury 和派葛亚等金融科技公司的高管。

Foundation Capital 的 Angus Davis 表示，他的公司已经投资金融科技基础设施超过十年。就个人而言，在成为风投之前，戴维斯是商业软件公司 Upserve 的创始人兼首席执行官。他说，在那里，他亲眼目睹了“交易数据对新型贷款产品的价值。”

根据 Davis 的说法，基金会有一篇论文围绕 Weav 开发的嵌入式金融科技类型。它还看到了一个巨大的市场机遇，一类基于 Weav 平台的新型金融应用将进入市场。

“Weav 对商业平台通用 API 的愿景让我们兴奋不已，”Davis 通过电子邮件写道。“就像 Plaid 和 Envestnet 为消费者的银行业务带来了通用 API 一样，Weav 为企业提供了一种新的 B2B 金融科技应用。”

工作原理

Weav 表示，通过使用其 API，公司可以提示其商业客户将他们的账户与销售平台、在线市场、订阅管理系统和支付网关“安全地”连接起来。一旦通过认证，Weav 就可以聚合和标准化跨平台的销售、库存和其他账户数据，并开发洞察，以推动跨一系列用例的新产品，包括贷款和承销；财务规划和分析；实时金融服务和业务管理工具。

Lidor 指出，在过去的几年里，API 公司不断增加，金融系统也变得更加开放，主要关注消费者。

他告诉 TechCrunch:“例如，Plaid 带来了关于消费者的非常丰富的数据，但当你想到企业时，这些数据通常仍被锁定在各种系统中。”。“我们在这里提供一些构建模块和对与销售和收入有关的一切数据的访问。而且，我们真的很高兴为产品提供动力，这些产品旨在使小企业和电子商务、卖家和创作者的生活更加轻松，并能够让他们获得金融产品。”

在 Brex 的案例中，Weav 的 API 允许初创公司立即获得资金，否则企业需要几天或几周才能获得资金。

“小企业需要尽快获得收入，这样他们才能为运营提供资金，”利多尔说。

Brex 联合首席执行官 Henrique Dubugras 表示，Weav 的 API 使该公司能够向更多平台上销售的更多客户提供实时资金，这为该公司节省了“数千个工程小时”,并将其推出时间缩短了数月。

显然，考虑到公司创始人亲自投资 Weav，该公司对所看到的很满意。Weav 是在打造“商业格子”吗？我想只有时间能证明一切。
