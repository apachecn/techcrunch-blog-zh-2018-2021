Arm 推出面向高性能计算、数据中心和边缘的最新芯片设计 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/27/arm-launches-its-latest-chip-design-for-hpc-data-centers-and-the-edge/

Arm 今天宣布推出两个新平台，Arm Neoverse V1 和 Neoverse N2，以及它们的新网状互联。从名字就可以看出，V1 是一款全新的产品，可能是 Arm 在数据中心、高性能计算和机器学习领域雄心的最佳例子。N2 是 Arm 的下一代通用计算平台，旨在跨越从超大规模云到智能网卡和运行边缘工作负载的使用案例。这也是第一个基于该公司新 Armv9 架构的设计。

不久前，高性能计算还被少数玩家所主导，但 Arm 生态系统最近在这里赢得了公平的份额，韩国、印度和 T2 的超级计算机都在赌它。V1 的承诺是，它将大大超过旧的 N1 平台，例如，浮点性能提高 2 倍，机器学习性能提高 4 倍。

图片来源: Arm

“V1 是关于我们能带来多少性能——这是我们的目标, Arm 基础设施业务线的 SVP 和总经理 Chris Bergey 告诉我。他还指出，V1 是 Arm 迄今为止最宽的架构。他指出，虽然 V1 不是专门为高性能计算市场打造的，但它绝对是一个目标市场。虽然当前的 Neoverse V1 平台还没有基于新的 Armv9 架构，但下一代将会。

Bergey 强调说，另一方面，N2 的目标是获得最高的性能功耗比。“这实际上是关于保持我们在 N1 拥有的相同的性能功耗比，但带来更多的性能，”他说。例如，在 Arm 的测试中，NGINX 的性能比上一代产品提高了 1.3 倍。

图片来源: Arm

在许多方面，今天的发布也是 Arm 强调其最近客户成功的一个机会。AWS Graviton2 显然做得相当好，但甲骨文也押注于 Ampere 基于 Arm 的 Altra CPUs 用于其云基础设施。

“我们相信 Arm 将无处不在——从边缘到云。我们看到基于 N1 的处理器提供了客户希望从云基础设施获得的一致的性能、可扩展性和安全性。Oracle 与 Ampere Computing 和领先的 ISV 合作，使 Arm 服务器端开发成为一流、简单且经济高效的解决方案。”

与此同时，阿里云和腾讯都在为其云服务投资基于 Arm 的硬件，而 Marvell 将在其 OCTEON 网络解决方案中使用 Neoverse V2 架构。
