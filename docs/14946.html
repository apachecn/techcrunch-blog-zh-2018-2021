Alchemy 以 5.05 亿美元的估值融资 8000 万美元，成为“区块链的 AWS”

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/28/alchemy-raises-80m-at-a-505m-valuation-to-be-the-aws-for-blockchain/

区块链开发者平台 Alchemy 今天宣布，它已经在由 Coatue 和 add、 Lee Fixel 的 新基金牵头的 B 轮融资中筹集了 8000 万美元。该公司此前共筹集了 1550 万美元，因此自 2017 年推出以来，最新的融资总额达到了 9550 万美元。

最近一轮引起我们的注意有几个原因。

首先，这家自称为区块链行业后台技术的公司，在短短八个月内从公开上市到估值 5.05 亿美元。在此期间，Alchemy 表示，它为全球数千万用户提供了超过 300 亿美元的交易。 S 其次，这家初创公司表示它也已经在为 大多数的 NFT 产业 提供动力。

最后，这轮投资者包括一些知名的机构和个人，如 DFJ 成长公司、K5 环球公司、烟鬼连锁店、演员杰瑞德·莱托和格雷泽家族(坦帕湾海盗队和曼联的所有者)。 他们加入的现有支持者包括雅虎联合创始人兼前 CEO 杨致远、潘迪拉资本、比特币基地、SignalFire、三星、斯坦福大学、谷歌董事长兼斯坦福大学校长约翰·l·轩尼诗、查尔斯·施瓦布、LinkedIn 联合创始人雷德·霍夫曼等人。

了解 Alchemy 运营内幕的消息人士告诉 TechCrunch，自签署 B 轮投资意向书以来，该公司的业务已经增长了八倍以上。他们还表示，Alchemy 有超过 3 亿美元的投资者希望进入这轮融资，并正在以当前估值的“数倍”进行另一轮融资。

TechCrunch 与 Alchemy 的联合创始人 Nikil Viswanathan(首席执行官)和 Joe Lau(首席技术官)谈论了这次加薪，他们对这家初创公司使命的热情显而易见。就像它的爆炸式增长一样。

Viswanathan 告诉 TechCrunch:“我们意识到，为了让这个领域蓬勃发展并发挥其全部潜力，我们需要为区块链建立一个开发者平台层。

Alchemy 的目标是成为考虑在区块链或主流区块链应用程序上开发产品的开发者的起点。其开发者平台旨在消除构建基础设施的复杂性和成本，同时通过“必要的”开发者工具改进应用程序。

这家初创公司推动了区块链几乎所有垂直领域的一系列交易，包括金融机构、交易所、数十亿美元的分散融资项目和联合国儿童基金会(UNICEF)等跨国组织。It 也迅速成为每个主要 NFT 平台背后的技术，包括 MakersPlace、OpenSea、Nifty Gateway、SuperRare 和 CryptoPunks。

“每次你打开 DoorDash，你都在使用亚马逊的基础设施，”刘说。“每次你与 NFT 互动，你都在使用炼金术。它是由引擎盖下的炼金术提供动力的。”

形象学分:炼金术

虽然两家公司不愿提供硬收入数字，但这家作为 SaaS 业务运营的公司表示，2020 年其收入增长了 600%。

对于内线玩家来说，炼金术的努力是在为整个行业铺路。

比特币基地总裁兼首席运营官艾米莉·蔡(Emilie Choi)表示:“加密经济的创新速度比之前的任何技术发展都要快，而炼金术是其中的关键驱动力。" Alchemy 使开发人员能够构建主流区块链应用所必需的丰富的应用生态系统."

潘迪拉资本的 Paul Veradittakit 将 Alchemy 描述为“区块链行业的亚马逊网络服务(AWS)”，即“实现去中心化网络的愿景”

他说:“在 Web 2.0 领域，微软、苹果和 AWS 是世界上最有价值的三家公司，因为它们是推动计算机和互联网行业的开发者平台，Alchemy 也准备为区块链做同样的事情。”

该公司认为与 AWS 进行比较是公平的，并指出:“正如 AWS 为优步、网飞和许多科技行业提供平台，Alchemy 为区块链行业的许多大型企业提供基础设施。”

Alchemy 计划利用新资本将其开发者平台扩展到新区块链，推动全球扩张，并在美国和全球开设新办事处。这家初创公司位于旧金山，并计划在纽约开设一个办事处。

Viswanathan 说:“我们将使用资金来支持我们的开发者平台的新连锁店。“我们还希望今年团队人数增加 5 倍。”

但要明确的是，炼金术以精益和吝啬而自豪。

“我们的员工从 14 人增加到了 22 人，”刘说。"我们有意想让团队尽可能的小。"

最近，投资者对区块链空间越来越感兴趣。

今年 3 月， BlockFi ，自称是一家为加密市场投资者提供金融服务的公司，宣布完成一笔价值 30 亿美元的巨额 D 轮融资。同样在上个月，区块链分析公司chain analysis披露了 1 亿美元的 D 轮融资，使其估值翻了一番，超过 20 亿美元。
