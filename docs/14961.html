希腊的 Viva Wallet 为其面向小企业商户的 neobank 筹集了 8000 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/28/greeces-viva-wallet-raises-80m-for-its-neo-bank-targeting-small-business-merchants/

挑战者银行继续在金融领域掀起巨大的浪潮，较小的机构通过提供更简单的方式吸引客户离开现有银行，让他们不仅可以参与基本的银行服务，还可以利用一波带来更多个性化和更好交易的技术。在最新的发展中， Viva Wallet 是一家针对中小商户建立银行服务的希腊初创公司，它已经获得了 8000 万美元的融资，这笔钱将用于扩大其业务范围和向用户提供的服务，特别是扩大其商户预付款贷款业务。

该公司在 23 个欧洲市场开展业务，并计划很快扩展到克罗地亚、匈牙利和瑞典。

这笔资金之所以引人注目，部分原因在于投资者是谁。腾讯——微信背后的中国科技巨头，也在大举进军金融服务——与欧洲复兴开发银行(EBRD)和布雷耶资本(Breyer Capital)一起参与了该轮融资。我们已经证实，Hedosophia，一个有点神秘的基金，也活跃在美国的 SPACs，也是一个支持者。

Viva Wallet 目前没有披露其估值，但该公司战略和业务发展副总裁亚尼斯·拉里奥斯向我们证实，它正在完成一项大型 D 轮融资，对其的估值为 15 亿€(18 亿美元)。这是一个巨大的飞跃:他还指出，当 Viva Wallet 在 2019 年下半年关闭其 C 系列时，其估值为€3.05 亿美元。

目前尚不清楚 D 轮是否与 2020 年 8 月路透社报道的 €5 亿美元(6.03 亿美元)轮相同，Viva Wallet 正在筹集(我们已经要求)帮助它围绕其商户预付款和向客户提供的其他贷款建立一种新的贷款账户。

从本质上讲，如果得到监管机构的批准，投资者将获得一个新的法律实体的股份，这是一个特殊目的载体，将持有贷款。这不是新银行处理贷款债务的典型方式，但逻辑似乎是，它可以通过消除资产负债表上的一些风险，让初创公司更灵活、更快地扩张。(不利方面:这些贷款的责任可能会减少？)

这轮谈判值得注意的是，它的到来正值欧洲缓慢地、有希望地从新冠肺炎疫情的重压下探出头来，这场危机已经动摇并击倒了许多在公共卫生危机之前就已经摇摇欲坠的经济体。Viva Wallet 主要专注于商户，是可能有助于这些关键业务复苏的科技业务的一个主要例子。

“我们很高兴腾讯、欧洲复兴开发银行和布雷耶资本加入 Viva Wallet，”Viva Wallet 创始人兼首席执行官哈里斯·卡罗尼斯在一份声明中说。“我们相信，我们投资者广泛的专业知识和合作伙伴网络将加速 Viva Wallet 统一分散的欧洲支付市场的计划。我们带给欧洲商户的技术创新将帮助他们为所有客户提供无摩擦、本地化的支付体验，并将他们从维护传统卡终端的麻烦中解放出来。"

如果你认为新银行的世界非常拥挤，特别是专注于中小企业机会的新银行也越来越拥挤(其他一些竞争者包括美国的 Finom 、 Wise 、 Monzo 、 Penta 、 Starling 和 ANNA 等等)，那么 Viva Wallet 受到一些关注的一个原因是它迄今为止的牵引力和跟踪记录。

拉里奥斯表示，截至今年 Q1，这家初创公司已经盈利，其业务在去年增长了 40%以上，目前有 6 万名商户活跃在它的账本上。他说，这一数字有望在今年年底达到 10 万。

他说，其成功的一个原因是，它采取了非常本地化的增长方式，在其活跃的每个国家都设立了实体分支机构，这在今天的市场上有点复古的想法，在今天的市场上，银行经常关闭他们的实体场所，转向虚拟化。“Viva Wallet 证明了其商业模式的弹性，”他表示。

这笔资金将部分用于建立其贷款计划，但也用于扩大 Viva Wallet 已经很强大的领域。其中之一是其销售点 Tap-On-Phone 解决方案，该解决方案将任何 Android 设备(智能手机、平板电脑或企业设备)转变为卡终端，无需单独的硬件即可接受非接触式支付和 PIN 支付。(大多数 POS 系统使用小型、独立的终端，可以连接到平板电脑或电话。)

他还表示，未来将会有一些 M&A 更快地扩展到更多的市场。

该公司不会扩张的一个领域是消费者领域。Revolut 和 Atom 等其他新银行利用其对年轻消费者的吸引力，开始为他们发现的企业提供服务，但拉里奥斯认为，Viva Wallet 不会采取相反的策略，尤其是因为消费者市场迄今已被证明是一个利润率很高(甚至很低)的游戏。

“万岁钱包只专注于企业，并将继续这样做！”他说(感叹他的！).“消费者细分市场没有提供任何盈利空间，我们看到，所有竞争对手的新银行业务模式大多都在烧钱。”

“我们专注于欧洲的中小企业，提供泛欧支付解决方案，但是该解决方案非常本地化，以满足商户在本地支付接受、本地 IBAN 账户、本地 BIN 业务借记卡等方面的真正本地需求。“但是，尽管 Viva Wallet 可能有许多中小企业客户，而且欧洲复兴开发银行的投资肯定是为了支持这一点，但他指出，它也包括中型企业和一些企业，例如像连锁超市这样的大型商家，这将是它将继续扩张的一个领域。

这使 Viva Wallet 具有足够的专业化和差异化，以及迄今为止在这些领域的盈利能力，从而吸引渴望进入经济复苏的知名投资者，既有助于经济复苏，又能在支付股息时乘风破浪。

“我们非常高兴能够帮助 Viva Wallet 统一遍布 23 个国家的分散的欧洲支付生态系统。布雷耶资本(Breyer Capital)的吉姆·布雷耶在一份声明中表示:“Viva Wallet 处于金融科技范式转变的最前沿，我们希望共同改变欧洲的支付行业。”。

“腾讯认同 Viva Wallet 通过创新为用户和合作伙伴创造价值的愿望。我们期待支持 Viva Wallet 在欧洲的扩张，”腾讯投资部总经理马丹英补充道。
