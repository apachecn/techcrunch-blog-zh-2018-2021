这是三星新的旗舰笔记本电脑系列，Galaxy Book Pro

在通常的几周泄露之后，三星刚刚宣布了其笔记本电脑产品线的几个新成员。Galaxy Book Pro和Galaxy Book Pro 360加入了硬件巨头向高端发展的一系列设备，提供了可以合理归类为该公司对MacBook Pro的看法。

Windows机器继续推动该公司模糊其Galaxy PC和移动产品之间的一些生产力界限，包括许多跨设备的软件产品，自然也包括Pro 360附带的S-Pen。顾名思义，360的盖子可以向任意方向旋转，因此可以兼作书写面。

轻薄的设计可能是三星的头条特色。Pro和Pro 360的尺寸分别为11.2毫米和11.9毫米。两者都有13.3英寸和15.6英寸Super AMOLEDs选项。分辨率为1920 x 1080。这一切都由酷睿i5或i7第11代英特尔处理器、8或16GB内存和高达1TB的存储空间(Pro最大512GB)提供支持。

图片来源:三星

还有LTE和5G版本的选项(取决于市场可用性)。这几乎肯定会对电池产生直接影响，该公司将其评为“全天”两款机型都支持通过USB-C端口进行65W快速充电。与以前的型号相比，键盘机制得到了升级，现在的触控板增大了23%。

该系统今天开始接受预订，并于5月14日开始发货。13英寸和15英寸版本的Pro起价分别为1000美元和1100美元，360的起价分别为1200美元和1300美元。
