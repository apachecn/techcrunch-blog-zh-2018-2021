以色列电动动力总成制造商 IRP 系统公司筹集了 3100 万美元的 C 轮融资

电动汽车创新动力系统制造商 IRP 系统公司已经筹集了 3100 万美元的 C 轮融资，使其总融资额达到 5700 万美元。

这笔融资由以色列机构投资者 Clal Insurance 和 Altshuler Shaham 牵头。参与的还有三星创投、雷诺-日产进口商 Carasso Motors 和 Shlomo Group，以及 entre Capital、复星 RZ 资本和 JAL 创投等现有投资者。

IRP 系统公司为包括雷诺在内的一大批电动汽车制造商供货。其电动动力系统声称具有高性能和高效率，同时降低了多种尺寸电动车辆动力系统的重量、尺寸和总成本。

IRP 系统公司首席执行官兼联合创始人莫兰·普莱斯(Moran Price)在一份声明中表示:“随着电气化和数字化成为核心汽车技术，汽车行业近年来正在经历结构性转变。IRP 系统公司是这场革命的中心。有了新的投资，我们将继续创造颠覆性的解决方案，并渗透到新的电动汽车领域。”

IRP 系统公司将利用这一新案例来扩大其电动汽车系统的开发规模，缩短大规模生产的道路，扩大 R&D、运营和客户支持，并推动全球销售和营销。
