肯尼亚的 Ajua 收购 WayaWaya，巩固非洲中小企业的消费者体验

原文：https://web.archive.org/web/https://techcrunch.com/2021/04/28/kenyas-ajua-acquires-wayawaya-to-consolidate-consumer-experience-play-in-africa-smes/

肯尼亚面向非洲企业的消费者体验平台， Ajua 今天宣布收购 WayaWaya ，这是一家总部位于肯尼亚的人工智能和人工智能信息和支付公司 。

WayaWaya 的客户和合作伙伴包括 I & M Bank、Interswitch 和 MTN 等。该公司提供一系列服务，从数字银行和支付服务到金融服务 API 和支付机器人 。

据阿瓜称，此次收购主要是聚焦于 WayaWaya 的支付机器人系统，即 Janja 。 该平台拥有 Airtel、Ezee Money、肯尼亚住房金融公司(HF Group)等客户，支持跨应用和社交媒体平台的无国界银行和支付 。 创办 WayaWaya 的企业家 Teddy Ogallo 加盟 Ajua，担任产品 API 和集成 副总裁。

、WayaWaya 已经 刚刚 筹集了 7.5 万美元。虽然两家公司没有透露收购的财务细节，但预计阿瓜 支付的价格是 WayaWaya 总融资 的 10 倍。

阿瓜，原名 mSurvey ， 由肯菲尔德格里菲斯于 2012 年创立 。该公司正在为非洲企业解决消费者数据问题，以更好地了解他们的业务并推动增长。

格里菲斯告诉 TechCrunch “非洲大陆上有很多商业活动，阿瓜希望公司能够从交易数字转移到这些交易背后的客户身上。“想象一下，如果我们知道是什么驱动了企业的消费者习惯。我的意思是，这对非洲企业来说是一个巨大的指数曲线。”

泰迪·奥加罗(WayaWaya 创始人)和肯菲尔德·格里菲斯(Ajua 首席执行官)

仅尼日尼亚的中小企业市场每年的价值就高达 2200 亿美元。 、而 的企业，大多是 的大企业，买不起客户沟通工具，很大一部分小企业被 冷落。 阿瓜的玩法是用数据和分析将公司和客户实时连接起来 。 “我们已经获得了让企业客户成功的东西，我们正在以一种简单的格式捕捉它，以便中小企业可以拥有同样的工具，”Griffith 补充道。

由于这些中小企业的大多数消费者行为都发生在网下，Ajua 为企业提供了独特的 USSD 代码，以接收付款、获得反馈并向客户提供折扣。 是阿瓜多年来推出的服务于商家的客户反馈产品之一，累计拥有超过 4500 万客户 。

公司的合作伙伴和客户还包括可口可乐、FBNQuest、GoodLife 药房、Java House、Safaricom、渣打银行和道达尔 。

作为一个智能消息机器人，Janja 被 WhatsApp、【Facebook Messenger】和 Telegram 上的 个人和企业用来自动化客户支持和进行跨境支付 。 因此，Janja 整合到 Ajua 的产品堆栈中，将通过自动响应并在客户需要时给予他们想要的东西，来关闭收购方的大部分客户体验循环 。

此次收购发生在 Ajua 宣布与电信运营商 MTN 尼日利亚合作推出面向尼日利亚企业的客户管理产品 一个月后。 名为 MTN EnGauge 的产品具有与 Ajua 相同的功能，但在这种情况下， 是为使用 MTN 网络 的企业单独定制的。此次推出的 预计 将为 Ajua 的成千上万用户产生更多数据。它也将被升级以整合 Janja 和其他服务。

事后看来，由于在消费者体验领域拥有丰富的经验，阿瓜公司似乎可以在内部创造出像 Janja 一样的产品。然而，该公司选择了收购，Griffith 给出了两个原因——制造一个类似的产品需要很长时间，而 Ogallo 似乎非常了解 Janja 的业务和运营，让他加入是有道理的。

“泰迪和我们走的是同一个方向。 我们 只是 以为要 收购 WayaWaya 而不是搞一个真正好的公司拿出两种产品企图解决同一个问题 。对我来说，这一切都是为了一起解决问题，而不是单打独斗，”这位首席执行官说。

在谈到为什么接受收购时，现在担任新职务的 Ogallo 指出，Ajua 提升客户服务和体验以及帮助企业的能力是原因之一，并赢得了他的钦佩 。 “看到 WayaWaya 的技术如何补充 Ajua 的创新产品和服务，并帮助扩大业务规模和实现盈利，对我们来说是一个令人兴奋的机会，我们很高兴我们的团队将合作建设非洲大陆独特的东西，”他补充道 。

这是来自 Ajua 的一个坚实的基础设施项目，来自一个大力提倡收购和整合的创始人。Griffith 认为，这两者都是更快进入非洲新市场和新渠道的战略。

我认为有很多方法可以建立生态系统。有很多年轻的人才在积累东西，他们没有资本进入下一个阶段。问题是，他们是想冲到终点，还是想拖延时间被收购。我认为 如果你想通过收购来解决复杂的问题，非洲有巨大的机会。”

在过去的三年里，一个国家内部和两个国家之间对非洲本地初创公司的收购有所增加。对于前尼日利亚招聘平台 Jobberman 去年收购 NGCareers 想到的是 。还有一些泛非洲的例子，比如总部位于拉各斯的 hub 收购内罗毕的 iHub 埃塞俄比亚软件提供商 Apposit 出售给尼日利亚 fin tech Paga；总部位于约翰内斯堡的 fintech MFS Africa 收购乌干达的 Beyonic 。

这些收购(以及大多数非洲收购)的共同主题是它们未披露的金额 。 对于阿瓜，Griffith 认为监管问题是该公司对数字保密的原因之一 。

根据 Crunchbase 的数据，自九年前成立以来，Ajua 已经筹集了 350 万美元。 鉴于此次收购的性质以及与 MTN 的合作关系，该公司可能会着眼于另一次融资，以大举进入尼日利亚(该公司于 2019 年进入该市场)和其他非洲国家。
