肯尼亚米拉公司筹集 180 万美元在非洲推广 API 保险平台

原文：https://web.archive.org/web/https://techcrunch.com/2021/05/04/kenyas-lami-raises-1-8m-to-scale-api-insurance-platform-across-africa/

根据 2018 年麦肯锡研究对非洲大陆六个保险地区 的比较，非洲的保险市场渗透率为 3%。如果排除南非市场，这个数字下降到微不足道的 1.12%。

与世界其他地区不同，大多数非洲保险公司忽视了定制和负担得起的保险产品对普通非洲消费者的重要性 。拥有 180 万美元种子资金的肯尼亚初创公司米拉科技公司正在寻求改变这种状况。

本轮 由Accion Venture Lab领投，这是一家种子期投资公司，支持针对服务不足市场的金融服务 。 其他参与的风投还有 AAIC、协和、P1 风投、Acuity Ventures、The Continent Venture Partners 和 Future Africa 。

非洲的保险覆盖率低在某种程度上是由于传统的保险单分销方式 。他们 习惯上 依靠实体渠道销售和处理政策。这需要很长的处理周期，客户满意度低，配送成本高。

按顺序 ，保险费 的支付方式 受到 的影响。从 2018 年麦肯锡报告来看，东非总毛保费(GWP)为 33 亿美元。相比之下，南非同年的 GWP 价值为 483 亿美元。

为此，首席执行官韩吉·阿巴斯于 2018 年创立了该公司，旨在实现肯尼亚保险产品的民主化。

“对我们来说，我们想要解决的主要问题是 97%的非洲人不买保险。她告诉 TechCrunch “我们试图理解这背后的方法论，尤其是在肯尼亚，那里有 50 多家保险公司，但普及率只有 2.4%。

“我们的驱动力是让保险 广泛 可用。 我们认为，建立技术基础设施来促进保险的分销是提高在非洲的渗透水平的最佳方式 。”

但是向消费者直接销售将是一个细致的过程，因为他们很少从可信的机构购买保险，更不用说第三方公司 。 因此，米拉采取了 B2B2C 的方式，利用那些与客户日常交流并围绕其进行创新的平台已经建立起来的信任。

通过 API，它允许银行、创业公司和组织等企业向其用户提供数字保险产品 。该产品还可以由 合作企业使用 来管理自己的保险需求。

一些客户，如肯尼亚的 Stanbic 银行，使用米拉的 API 来运行保险业务；人力资源平台 WorkPay 为使用其平台 的企业提供保险产品。 该公司拥有 20 多名保险作家，还在电子商务平台 Jumia 上推出了保险市场。

用户可以通过其 API 获得汽车、医疗或其他定制保险产品的报价。 他们还可以定制福利并调整保费以适合 他们，获取他们的保单文件并访问理赔 。

典型的 ，对于一个普通的非洲保险公司来说，从 到 处理索赔大约需要 90 天。阿巴斯说米拉已经把这个时间缩短到了一周——这是这家成立三年的公司赢得顾客信任的一种方式。

韩吉·阿巴斯(创始人兼首席执行官)。图片来源:米拉科技公司

米拉已经克服的另一个挑战是让保险公司参与进来。 首席执行官表示，从传统的保险方式向数字分销渠道的转变之所以成功，是因为米拉很早就开始展示 客户体验和旅程的价值，这需要在正确的时间为正确的客户提供正确的保险 。

这就是米拉脱颖而出的原因，阿巴斯继续说道。它与其承销伙伴共同设计产品。 以这种方式进行设计有助于企业向其潜在客户群提供独特的保险产品 。

她举例说明了一个公共汽车预订平台，在这个平台上，乘客的保险点数 是以每次出行为基础计算的。上车时算，下车时停。她相信这样一个创新的过程将会把欧洲大陆的保险业带到一个更令人向往的地方。

我认为 保险业有巨大的潜力。尽管渗透率较低，但每年的市场价值超过 600 亿美元。 我认为 相对于其他金融服务而言，人们开始对保险打开眼界 。”

自 成立 以来，insurtech 初创公司已售出 5000 多份保单。它与超过 25 家活跃的承销商合作，包括 Britam、Pioneer 和 Madison Insurance。 这些保险商帮助分销 30 多种产品，从医疗和员工福利到汽车和设备保险 。

米拉将利用种子投资雇佣更多的人，改进技术，扩大在非洲的影响力。

Accion Venture Lab 正在押注米拉的嵌入式金融游戏。以下是该公司非洲部主任阿什利·刘易斯对投资的评价。“… 通过在客户了解和信任的业务中嵌入定制保险，米拉正在让保险服务不足的非洲人口能够获得保险，并使他们能够建立财务弹性 。”

米拉的投资也代表了肯尼亚科技生态系统中的一个亮点，在这个生态系统中，既是本地人又是女性创始人是一个不协调的组合。2019 年的一项研究 显示，在四大科技生态系统 中，肯尼亚的外籍联合创始人最多。 虽然肯尼亚的女性联合创始人比例高于其他国家(1/4)，但肯尼亚女性联合创始人的比例约为 12% 。

有 就有 少数几个融资过百万美元轮的女创始人。尽管阿巴斯在这个显赫的俱乐部里坐得很舒服，但她需要厚脸皮和对自己产品的信心才能进去。

“肯尼亚的资助格局普遍偏向男性创始人，在东非，尤其偏向外国创始人 。因此，让投资者兴奋起来并加入我们的行列要困难得多。对我们来说，我们已经建立了一些相当令人兴奋的东西，虽然它需要一些时间。她说:“我们之所以想宣传这件事，一个关键的原因是，其他女性创始人也可以看到这样做的机会。
