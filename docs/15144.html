Bitski 从 a16z 融资 1900 万美元，成为“NFTs 的 Shopify”

每一个将 NFTs 视为又一个炒作泡沫的密码怀疑论者，都有一个助手将 NFTs 视为开启创意网络未来的钥匙。

Bitski ，一家总部位于旧金山的初创公司，为品牌和创作者建立定制的 NFT 店面，指望后者，他们也有新的投资者押注于此。这家初创公司告诉 TechCrunch，他们已经在 Andreessen Horowitz 领导的 A 轮融资中筹集了 1900 万美元。该公司加入了 Kindred Ventures、Galaxy Digital 以及包括 Jay Z、MrBeast 和 3LAU 在内的一系列创作者和名人，共同支持这家初创公司。

NFT 空间最近变得非常拥挤，在过去几个月里，投资者大肆宣传，交易价值数十亿美元。虽然 Beeple 等艺术家出售的高价艺术品和 CryptoPunks 等独家加密社区一直是活动的温床，但 Bitski 背后的创始人认为，这些区块链支持的数字产品将激发艺术家、有影响力的人和品牌如何将他们的在线人气货币化的巨大变革。

Bitski 的目标是让主流品牌和名人绕过早期市场的加密复杂性，希望给像早期合作伙伴阿迪达斯这样的客户一个进入 NFT 世界的入口，这对于理解数字产品但可能没有完全购买加密的消费者来说更容易接近。这家初创公司向创作者出售各种订阅计划，以支持定制的 NFT 店面，他们可以通过自己的渠道销售，而不是将用户推向广泛的市场。

NFT 平台的建设者和用户之间存在大量争论，围绕着服务的哪些元素应该基于区块链，哪些应该默认为更久的电子商务流。Bitski 经常在用户熟悉度方面出错，允许在平台上使用信用卡购买，“忘记密码”功能和用户钱包托管在 Bitski 自己的服务器硬件上。它们是有争议的入门选择，不会满足加密纯粹主义者和去中心化倡导者，但可能会帮助新用户快速熟悉 NFTs。

随着该公司首轮融资的结束，Bitski 迄今已筹集了约 2340 万美元。
