Desktop Metal在其产品组合中增加了木印技术TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/05/06/desktop-metal-adds-wood-printing-to-its-portfolio/

Desktop Metal今天宣布推出木质3D打印工具Forust。该公司成立于2019年，专注于室内设计的3D打印。该公司的“非破坏性”印刷方法已经设法在很大程度上避开了雷达，直到现在媒体报道很少——这使它们成为一个非常理想的收购候选人。

事实上，根据一份文件，总资产收购实际上发生在2020年10月，其价格为250万美元，包括200万美元的现金对价。从那以后，在正式发布之前，这两家公司似乎一直在合作。

在今天发布的新闻稿中，Desktop Metal将Forust定位为该公司产品组合中新制造工艺的名称。这项技术利用了纤维素粉尘和木质素，分别是木材和造纸工业的副产品。

图片来源:桌面金属

 

“作为桌面金属的一部分，Forust正在让建筑师、设计师和制造商利用设计前沿技术来重新思考木材废料流的使用方式——无论你是想制造一件、一千件还是一百万件，”Forust联合创始人兼首席执行官Andrew Jeffery告诉TechCrunch。“Forust有能力为企业和消费者带来3D打印的可持续木材设计，以创造美丽、坚固的木制品来支持各种行业，如室内设计、家具和家用消费品。”

从今天开始，这些公司将在网上提供样品和设计。产品清单包括建筑装饰、家具和家居产品，如碗和花盆。

图片来源:桌面金属

桌面金属公司首席执行官里克·富洛普在一次新闻发布会上说:“Forust的木材零件的应用真的是无限的。”“如今，聚合物和塑料的应用非常广泛，现在您可以用可持续制造的木质部件来经济高效地替代它们——室内装饰、消费电子产品、仪器仪表、航空、船舶、家居用品中的豪华高端组件，最终还会用于地板和室外屋顶。”

除了Forust，同一份文件还显示，该公司还以350万美元的价格收购了金属板材印刷公司Figur Machine Tools的“单一收购技术资产”。今年8月，Desktop Metal宣布将通过特殊目的收购公司上市，其估值为25亿美元。
