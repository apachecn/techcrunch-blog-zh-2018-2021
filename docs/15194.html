一群 P2P 黑客如何播下独特的费用管理巨头的种子

Expensify EC-1 第 1 部分:起源故事

在企业软件领域，个性通常没有一席之地。在一个单个合同可以轻松达到数百万份的市场中，同质性是可靠性的先驱，有助于让买家放心他们潜在购买的价值。

因此，人们很自然地会认为，从事费用报告管理业务的公司会保持简单，照章办事。但只要看一眼 Expensify 就足以告诉你，这是一家从未找过这本书的公司。

Expensify 的起源故事是一群斗志昂扬的开发人员之一，他们将旅行变成了创意的催化剂，一起经历了高潮和低谷，最终建立了当今企业软件领域最出人意料的原创公司之一。

从其著名的“工作场所”，到其管理结构和决策政策，Expensify 在其 DNA 中就有避免所谓的最佳实践来实现自己的想法——这一哲学植根于其创始人和早期团队的 P2P 黑客背景和自己动手的态度。因此，Expensify 在许多方面都不是创业公司的典型，无论是内在还是外在。

创始人兼首席执行官大卫·巴雷特在我们的第一次电话中明确表示，他的公司与众不同:“我们的招聘方式非常不同。我们有一个非常不寻常的内部管理结构。我们的商业模式本身就很不寻常。例如，我们没有销售人员。我们是一家非常小的公司。我们关注的是员工而不是老板。我们的技术体系完全不同。我们的产品设计方法非常不同。”

即使以创业公司的标准来看，这种描述也会让一些人认为 Expensify 很奇怪，但这种本质上的差异使它在一个由巨头如 SAP Concur 和 Coupa 主导的领域中脱颖而出。这最终对其有利:2020 年，Expensify 的年度经常性收入达到 1 亿美元，EBITDA 利润率高达 25%。在我们采访 EC-1 期间，也有传言称该公司计划上市，但他们在 3 月份停止了与我们的对话，现在我们知道了原因: Expensify 秘密提交于 5 月 3 日上市。

当大卫遇见特拉维斯时…

为了真正理解 Expensify，你首先需要仔细看看一家独特的、短命的 P2P 文件共享公司，名为 Red Swoosh ，这是特拉维斯·卡兰尼克在创立优步之前的初创公司。被卡兰尼克诬陷为他的【复仇事业】在他之前的 P2P 创业公司被起诉 侵犯版权后，Red Swoosh 将成为 Expensify 未来文化和精神的先驱。事实上，Expensify 最初的许多团队实际上都是在 Red Swoosh 认识的，该公司最终在 2007 年被 Akamai Technologies 以 1870 万美元的价格收购。

Barrett，一个自称为头号极客和终身软件工程师的人，实际上是 Red Swoosh 的最后一个工程经理，在他的第一个项目iGlance.com失败后被雇用，这是一个无法与 Skype 竞争的 P2P 一键通程序。“当我正舔着那次经历的伤口时，特拉维斯·卡兰尼克找到了我，他正在经营一家名为 Red Swoosh 的初创公司，”他在一次采访中回忆道。
