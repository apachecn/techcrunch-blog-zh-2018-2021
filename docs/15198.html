Indy风险投资公司Sixty8 Capital推出2000万美元的基金，旨在帮助代表性不足的创始人TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/05/10/indy-vc-firm-sixty8-capital-launches-20m-fund-aimed-at-underrepresented-founders/

很明显，黑人、女性、Latinx和LGBTQ+初创公司的创始人要想在硅谷的风险投资馅饼中分得一杯羹，面临着一场艰苦的战斗。也许这就是为什么总部位于印第安纳州印第安纳波利斯的公司选择启动一个新的2000万美元的基金，旨在为代表性不足的创始人提供早期资金。

该基金的投资者包括印第安纳下一级基金、50南方资本、美国银行、礼来公司、第一互联网银行和中央印第安纳社区基金会。它与另一家总部位于印度的风险投资公司 Allos Ventures 合作，来自Allos的Paul Ehlinger将成为68的风险投资合伙人。

“有了这笔基金，我们要做的是通过将资金直接放到有色人种、女性和其他不同群体的手中，真正开始赋予他们权力。能够直接投资于那些正在构建惊人解决方案的公司，而这些公司恰好是由不同的人创建的。这就是我们推出68的原因。我认为这是一个我们可以抓住的独特机会，我真的很高兴能够在印第安纳州的社区以及中西部和南部部分地区产生影响，”该公司的管理合伙人Kelli Jones解释道。

琼斯告诉我，她在印第安纳波利斯长大，在搬到纽约和洛杉矶在音乐、科技和娱乐的交汇处工作后，她于2016年回到印第，开始帮助她长大的社区的黑人接受培训，以便在科技行业内外获得体面的工作。这导致了专注于黑人创始人的创业孵化器的发展，以及后来的推介比赛。

她说，在那个时候，很明显，与她一起工作的创始人需要获得资本，以便有机会发展他们作为孵化器和推介竞争的一部分而开始的业务，早期基金的想法开始形成。她说印第安纳州因B2B SaaS而闻名，她想利用这种能量。

“你知道我们被称为B2B SaaS，我们在这里有一些令人惊讶的退出，有 ExactTarget和Salesforce 和 Angie's List 和 Interactive Intelligence和Genesys ，所以我们在当地的技术领域发生了很多非常令人惊讶的事情，但没有多少关于多样性的对话，也没有看到更多的有色人种、女性和LGBTQ创始人，”琼斯告诉我。

该计划旨在为每家公司提供25万至50万美元的投资，包括种子期、预种子期以及偶尔的A轮融资。她说，她的其他企业，包括孵化器和推介比赛，都有现成的渠道，她也加入了有很多创业活力的社区。

她表示，他们希望建立一只基金，不仅是为了解决多元化问题，让多元化的人做出投资决策，而且是基于这样一种战略，即公司能够投资于那些可能不总是被视为典型的风险投资支持的商业目标的公司。

琼斯告诉我，第一笔投资是与一家名为 Qualifi 的B2B SaaS公司合作，该公司利用人工智能帮助大量招聘的公司更快地通过资格审查，从7到10天缩短到3天或更少。

这家公司的名字可以追溯到1968年，那是一个历史时期，当时全国各地爆发了许多抗议活动，呼吁有色人种、妇女和同性恋者享有更多平等权利。琼斯表示，当该公司在2019年首次推出时，他们有一个不同的名字，但这似乎更适合一家专注于为多元化群体提供支持的公司。

“感觉我们仍在行进，试图以1968年民权运动时的方式生存下去，在那场运动中，我们失去了伟大的领袖，每个人心中的火都是如此之大。我们为妇女权利、拉丁裔权利和黑人权利而斗争，发生了太多的事情，看起来就像在2021年，我们真的还在那个空间，”她说。
