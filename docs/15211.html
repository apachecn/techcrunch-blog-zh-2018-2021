Apptopia 融资 2000 万美元，将竞争情报业务扩展到移动技术之外

原文：https://web.archive.org/web/https://techcrunch.com/2021/05/11/apptopia-raises-20m-to-expand-its-competitive-intelligence-business-beyond-mobile/

总部位于波士顿的 Apptopia 是一家提供移动应用生态系统竞争情报的公司，已经完成了 2000 万美元的 C 轮融资，旨在推动其在移动应用领域以外的扩张。该公司表示，新的融资由 ABS Capital Partners 牵头，此前 Apptopia 的业务连续三年同比增长 50%，自去年年初以来一直盈利。

包括 Blossom Street Ventures 在内的现有投资者也参与了这轮投资。ABS Capital 的 Mike Avon 是 Millennial Media 的联合创始人，Paul Mariani 将在这一轮加入 Apptopia 的董事会。

Apptopia 表示，品牌对更好地了解其业务的数字方面的需求不断增加，这笔资金随之而来。

今天，Apptopia 的客户包括数百家企业和金融机构，包括谷歌、Visa、可口可乐、塔吉特、Zoom、NBC、Unity Technologies、微软、Adobe、Glu、安德森霍洛维茨和脸书。

过去，Apptopia 的客户从宏观层面审视数字参与和互动，但现在他们希望深入具体细节，需要更多数据。例如，一个品牌以前可能想知道竞争对手的推广在新用户或应用程序会话方面表现如何。但现在他们想知道具体问题的答案，比如有多少独立用户参与了，这些用户是否是现有客户，他们是否在促销结束后返回，等等。

Apptopia 的大部分业务现在都专注于向订阅 Apptopia 数据的企业客户提供这类答案，可能还会订阅来自 Sensor Tower 和 App Annie 等竞争对手的数据，目标是将数据集融合在一起，以便更准确地了解竞争格局。

Apptopia 首席执行官乔纳森·凯承认，从历史上看，Apptopia 自己的数据并不总是被视为最准确的。但这几年已经有所改善。

凯，以前的 Apptopia 首席运营官，现在接替联合创始人埃利兰·萨丕尔的最高职位，随着公司进入下一个发展阶段，他将过渡到董事会主席。

凯解释说，Apptopia 的竞争对手，如 Sensor Tower 和 App Annie，使用移动面板和其他方法来收集应用数据。这些面板涉及面向消费者的应用，如 VPN 客户端和广告拦截器，用户将下载这些应用，而不一定了解他们同意收集他们的应用使用数据。这导致了一些争议，因为应用数据行业的公开秘密被暴露给了消费者，公司因此调整了他们的披露。

但这种做法仍在继续，并没有影响到公司的发展。例如，随着对应用程序数据的需求持续增长，去年筹集了 4500 万美元。所有相关的企业都在为他们渴求数据的客户群开发新的产品和服务。

图片来源: Apptopia

与此同时，Apptopia 决定不依靠移动面板发展业务。(尽管在早期，它确实测试并废除了这样的计划。)

它可以从其应用程序开发者客户那里获得数据——这些数据已经从开发者的苹果和谷歌分析账户中进行了汇总和匿名处理。

最初，这种方法使 Apptopia 处于不利地位。凯说，竞争对手从 2016 年到 2018 年拥有更准确的数据，因为他们使用了移动面板。但是 Apptopia 做出了一个战略决定，不冒这种风险——也就是说，建立一个苹果或谷歌可以随时关闭的业务。

“相反，我们所做的是花费数年时间投资数据科学和算法，”Kay 指出。“我们想出了如何从(竞争对手)可以访问的相同数据集中提取同等或更强的信号。”

使用 Kay 所描述的“大量的历史数据”，Apptopia 随着时间的推移了解到什么样的信号进入了应用程序的应用商店排名。凯指出，很多人仍然认为一个应用的排名很大程度上是由下载量决定的，但现在有多种信号可以显示排名。

“实际上，排名只是苹果和谷歌给分的分析数据点的累积，”他解释道。这包括会话数量、用户数量、在应用上花费的时间等等。“因为我们没有这些面板，我们不得不花费数年时间来找出如何比竞争对手更好地进行逆向工程。最终，我们想出了如何让他们从面板上得到同样的信号。这使得我们在过去几年里取得了如此快速、成功的业务增长。”

由于 Apptopia 已经盈利，它不需要融资。但该公司希望加快向新领域的扩张，包括其在移动应用程序之外的扩张计划。

如今，除了手机和平板电脑，消费者还会在电脑、智能手表和电视上使用“应用程序”。企业不再仅仅想知道移动设备上发生了什么——他们想了解“应用程序”使用的全貌。

“我们找到了一种不依赖于我们的竞争对手过去所做的任何事情的方法，”Kay 说。“所以，我们不会使用任何应用来监视人们，”他说。*

作为对凯声明的回应，Sensor Tower 表示:“我们从未收集任何个人用户的个人身份信息(PII)，也没有收到我们的小组未经用户同意提供的任何匿名使用和参与指标。”]

Apptopia 目前不准备提供有关其未来产品计划的更多细节。但凯表示，他们不排除合作伙伴关系或收购，以实现其未来的目标。

该公司还看到了让其应用程序数据更容易获取的更广阔前景。例如，去年，它与彭博合作，通过彭博终端上的彭博应用程序门户向投资者提供移动数据。它现在与亚马逊的 AWS 数据交换和雪花合作，也可以在其他渠道访问应用数据。未来类似性质的合作伙伴关系可能会发挥作用，成为 Apptopia 数据区别于竞争对手的另一种手段。

该公司拒绝提供其当前的收入运行率或估值，但指出其估值比 2019 年底的上次融资翻了三倍。

除了产品扩张，该公司计划利用这些资金在 2021 年将 55 人的团队再增加 25 人，包括工程和分析师。它还将壮大管理团队，今年增加一名首席财务官、首席采购官和 CMO。

迄今为止，Apptopia 已经筹集了 3000 万美元的外部资本。
