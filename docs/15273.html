Unmind 筹集了 4700 万美元用于建立一个平台，为你的工作场所提供心理健康支持 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/05/12/unmind-raises-47m-for-a-platform-to-provide-mental-health-support-in-your-workplace/

最近，心理健康已经成为人们关注的焦点。对于我们中的许多人来说，我们的生活和生活方式在过去的一年里发生了巨大的变化，与此同时，我们在全球范围内共同面对疫情引发的死亡率，这是几代人都没有遇到过的，这是一场完美的风暴，不可避免地对我们的精神状态和情绪产生了影响。

今天，一家建立了一个平台来帮助人们思考和应对这种情况的初创公司宣布了一轮大规模的增长资金，专门帮助解决所有这些问题，以及它如何在我们生活中更容易引发压力的一个方面——我们的工作场所——发生。

伦敦初创公司 Unmind 已经筹集了 4700 万美元的 B 轮融资，将用于继续投资研发和扩大业务范围。这笔资金由 EQT Ventures 牵头，该公司目前是英国增长轮中非常活跃的投资者，Sapphire Ventures 和之前的支持者 Project A、Felix Capital 和 True 也参与了投资。

Unmind 服务的核心是围绕一系列问题建立的，以帮助员工探索他们自己的心理健康状态，其中可能包括抑郁、焦虑、失眠和一系列其他表现。作为 EAPs 或远程治疗等反应性服务的补充，它可以提供主动、自我指导的工具来衡量、了解和培养他们的心理健康。它还为雇主提供一项服务，与他们分享应用程序中的匿名数据，以便他们也可以考虑如何更好地满足员工的需求。

该应用程序已经获得了一些显著的关注，尤其是在去年，鉴于我们一直生活的环境，这个时候关于心理健康的讨论变得更加普遍和重要。

Unmind 表示，它的服务面向 110 个国家的 200 多万名员工，虽然它不会透露活跃用户的数量，也不会透露他们是如何增长的，但它告诉我，一个组织的员工中有 15%到 60%的人接受和采用了它的应用程序(这取决于规模和组织本身对使用应用程序的重视程度等因素)。它还表示，在使用 Unmind 的员工中，88%的人表示他们在精神健康、工作或关系方面有所改善，而 92%的人报告说他们对精神健康有了更高的信心、认识和理解。

该公司还表示，在过去的 12 个月里，收入增长了 3 倍以上。与此同时，它的客户包括主要零售商，如约翰·路易斯和 M&S，TSB 高街银行，优步，三星，维珍媒体，英国航空公司和 Asos——这些公司周围有很强的客户服务水平，受到了封锁的严重影响，你可以想象肯定有很多人在工作中非常紧张，因为他们处于与紧张的更广泛的消费者接触的第一线。

该公司的联合创始人是尼克·泰勒博士，他之前是一名临床心理学家，在心理健康护理领域工作了多年(在此之前是一名受过古典音乐训练的歌手)，他说，他是在感觉到自己只是第一次看到太多的人，而他们的问题已经非常严重的时候，产生了这个想法。

“我一次又一次地遇到同样的挫折:我希望我六个月前就遇到这个人，”泰勒在一次采访中说。

与所有类型的预防性医疗保健一样，在问题变得更大、更紧迫之前识别并解决问题总是更好的，因此他开始思考如何实现预防性体检和心理健康检查的概念。

工作场所是一个不错的努力基础。它不仅经常是人们压力的来源，而且是他们每天经常去的地方，因此创造一种评估心理健康的方法，这就隐含地创造了一种日常工作。这还可能意味着与雇主建立更紧密的联系，以便在问题出现时以更集体的方式解决问题，这种方式雇主可能不会通过其他方式来做(或发现)。

工作和心理健康之间的联系由来已久，但在去年，这种联系可能比以往任何时候都更加明显。

“我不知道在 COVID 期间精神健康会发生什么，”泰勒回忆道。“实际上我想知道它是否会被降级，”考虑到所有其他冲突的优先级。“但是精神疾病的流行已经升级。已经失控了。在工作场所，这是旷工和离职的主要原因。”鉴于一切都变得如此繁忙，包括可能花费更多的时间工作，因为现在这些都已经融入了我们的家庭生活，我们都知道(很可能是其中之一)许多人现在感到非常疲惫。

泰勒说，事实上，与他早期的怀疑正好相反:心理健康已经成为人们关注的焦点，“耻辱的枷锁正在消失。”

这是真正吸引投资者眼球的一部分:不仅有效，而且与当下非常相关的技术。“现在人们普遍认识到，我们的心理健康与我们的身体健康一样重要，如果不是更重要的话——但却长期被忽视了。这种情况正在迅速改变，”EQT Ventures 的合伙人阿拉斯泰尔·米切尔(Alastair Mitchell)说。“因此，消费者心理健康应用程序的受欢迎程度大幅上升，与此同时，雇主和员工对工作场所心理健康应用程序的需求也在飙升。Unmind 是领先的企业心理健康应用程序，我们很高兴能与尼克博士和团队合作，支持他们在全球范围内的扩展。”EQT 也是一个战略投资者，而不仅仅是一个金融投资者:它在自己的工作场所和许多投资组合公司中主动出击。

应该指出的是，Unmind 并不是唯一一家发现这一“机会”的公司，如果你可以称之为“机会”的话。他们包括其他初创公司，如总部位于旧金山的 Ginger T1，该公司也建立了一个平台，与雇主、医疗保健提供商和其他利益相关者合作，帮助人们识别和管理他们的精神状态。生姜在这些年一直资本充足。同一领域的其他公司包括纽约的 Welbot、伦敦的 Spill 以及许多其他提供精神健康不同方面的公司，如冥想应用程序 calculation 和 Headspace。

我倾向于认为，鉴于问题的规模，精神健康不应该是一场混战，而是需要一个村庄来解决的事情，关键在于每个公司如何处理自己的职权范围，人们如何应对，以及人们最终使用的方法是否会为员工提供更好的桥梁，让他们获得所需的帮助和平静，无论是来自初创公司、个人还是其他人，或者是各种来源的组合。

“我们有责任像对待医疗保健一样，与我们的心理健康联系起来，”泰勒说，他将这种努力比作有时需要多种技能来解决复杂的健康问题。“大医疗整合了多个系统。”
