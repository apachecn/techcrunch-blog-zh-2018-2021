Google Cloud Run 承诺使用折扣和新的安全功能 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/05/13/google-cloud-run-gets-committed-use-discounts-and-new-security-features/

Google Cloud 的容器化应用无服务器平台 Cloud Run 正在获得承诺的使用折扣。承诺在一年内花费一定金额使用 Cloud Run 的用户将获得 17%的折扣。该公司为基于虚拟机的计算引擎实例提供类似的预先承诺折扣方案，并为每月运行超过 25%的机器提供自动的持续使用折扣。

此外，Google Cloud 正在为 Cloud Run 引入许多新的安全功能，包括从 Google Cloud Secret Manager 装载机密的能力和二进制授权，以帮助定义和执行关于如何在服务上部署容器的策略。Cloud Run 用户现在也可以使用和管理他们自己的加密密钥(默认情况下，Cloud Run 使用谷歌管理的密钥)，Cloud Run 内部的一个新的建议中心现在将为用户提供如何更好地保护他们的 Cloud Run 服务的建议。

最近成为谷歌云无服务器平台产品管理总监的 Aparna Sinha 指出，这些更新是谷歌云推动构建她所谓的“下一代无服务器”的一部分。

“我们非常兴奋地推出我们对无服务器的新愿景，我认为这将有助于重新定义这个领域，”她告诉我。“过去，无服务器意味着某种较窄的计算类型，专注于功能或非常特定的应用程序、web 服务等。—我们正在谈论的重新定义无服务器是专注于无服务器的力量，即开发人员的体验和易用性，但将其扩展到一个更加通用的平台，在这个平台上可以运行许多不同类型的应用程序，并以谷歌的方式进行开发、安全和大量集成，以便您可以访问最好的云环境。”

她指出，在疫情期间，Cloud Run 获得了“巨大的采用”,她认为这是因为企业希望加快其应用程序的价值实现时间。以宜家为例，众所周知，宜家在从店内销售转向网上销售的过程中经历了一段艰难的时间，它押注于谷歌云的无服务器平台，在转向这种模式后，它的在线商店和库存管理系统的更新时间从三个小时减少到不到三分钟。

辛哈说:“我认为，这就是无服务器的力量，尤其是展望未来，建立实时应用程序的能力，这些应用程序拥有关于环境、库存和客户的数据，因此可以更加积极主动地响应。”“这是客户对未来的期望，而无服务器是实现这一期望以及响应需求模式的绝佳方式，尤其是在当今不确定的环境下，客户的变化如此之大。”

由于容器模型为企业在这些容器中运行什么提供了很大的灵活性，以及他们希望如何开发这些应用程序，因为 Cloud Run 是语言无关的，所以谷歌现在看到许多其他企业也迁移到这个平台，这既是为了部署全新的应用程序，也是为了使他们现有的一些服务现代化。

对于具有可预测使用模式的公司来说，承诺使用折扣应该是一个有吸引力的选择，而且很可能是更复杂的组织要求 Google Cloud 今天推出的新安全功能。

Sinha 在今天的公告中写道:“下一代无服务器结合了无服务器和容器的优点，可以运行广泛的应用程序，没有语言、网络或区域限制。”“下一代无服务器将帮助开发人员构建未来的现代应用程序，这些应用程序能够轻松适应变化、按需扩展、更快更高效地响应客户需求，同时为开发人员提供最佳的开发体验。”
