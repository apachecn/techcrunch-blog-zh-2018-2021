PayPal 收购退货物流业务快乐退货 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/05/13/paypal-acquires-returns-logistics-business-happy-returns/

PayPal 今天宣布将收购 Happy Returns ，这是一家退货解决方案提供商，为在线购物者提供更简单的方式将不需要的商品退回给零售商，而不必自己装箱发货。该公司目前在美国提供超过 2，600 个送还点的网络，包括超过 1，200 个地铁站和美国各州的送还点。

它还与数百个使用其退货软件和逆向物流服务的品牌建立了关系。该公司表示，作为 PayPal 的一部分，它将继续向在线零售商和购物者提供退货体验。

总部位于圣莫尼卡的 Happy Returns 成立于 2015 年，其价值主张是为在线零售商的退货流程减少一些管理费用和成本。由于网上购物者不能直接检查他们购买的商品，网上零售往往会有更高的退货率，尤其是服装。例如，快乐退货发现，网上商品被退回的可能性是在商店购买的商品的三到四倍。

与此同时，今天的零售商必须与亚马逊和沃尔玛这样的巨头竞争，这两家公司都通过其庞大的实体足迹更容易地为客户退货——亚马逊拥有全食超市的其他地点，沃尔玛拥有自己的商店。事实上，在店内提供亚马逊退货台或储物柜系统的客流量已经导致像美国柯尔百货公司和斯坦马特这样的零售商通过在自己的商店里提供亚马逊退货来迎合购物者，从而拥抱敌人。

如今，Happy Returns 解决方案提供了软件、服务和物流的组合，允许零售商通过自己的零售店、承运商以及 Happy Returns 的“退货栏”位置管理退货。这些都可以在实体零售店找到，如 Paper Source、Sur La Table、Cost Plus World Market 等。这项服务已经被几个数字本土品牌使用，包括 Everlane，Rothy's 和 Parachute Home 等。

报告指出，Happy Returns 在历史上也一直与 PayPal 密切合作。值得注意的是， PayPal 在 2019 年对该业务进行了战略投资，作为 1100 万美元融资轮的一部分。

交易完成后，Happy Returns 将继续与 PayPal 平台内外的零售商和购物者合作。该公司的联合创始人 David Sobie 和 Mark Geller 及其 120 多人的团队将加入 PayPal，并向 PayPal 的 SVP 店内消费者和数字商务负责人 Frank Keller 汇报工作。

PayPal 没有透露交易条款。迄今为止，Happy Returns 已经筹集了 2500 万美元的资金。

“这对我们公司来说是一个令人难以置信的令人兴奋的里程碑，如果没有我们整个团队的辛勤工作和奉献精神，这是不可能的，”在 Happy Returns 网站上的公告写道。“我们为我们团队所取得的成就感到骄傲，并感谢员工每天为工作带来的坚韧、创造力和同理心。我们相信最好的还在后面，并期待着作为 PayPal 组织的一部分，我们的下一个篇章。”
