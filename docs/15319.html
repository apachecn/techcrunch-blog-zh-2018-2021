这是今天出席炼金术士加速器第27届演示日的25家公司

专注于企业的炼金术士加速器带着另一个回来了。今天是它的第27个展示日，预计将有20多家公司走上(虚拟)舞台。

这是迄今为止炼金术士最大的团队，有25家公司参加。与此同时，炼金术士主任拉维·贝兰尼告诉我，加速器的申请增长了100%以上，他们的投资组合迄今已筹集了14亿美元。

除了首次展示其演示日团队，炼金术士还分享了一些消息:它已经启动了一项名为“炼金术士X 的新努力，该努力将利用加速器的工具/网络来帮助企业(从日本的NEC开始)从其内部研发开发衍生公司。炼金术士x将由Rachel Chalmers领导，她最近是Autodesk的董事和Merian Ventures的风险合伙人。

和过去几天的炼金术士演示一样，今天的活动完全是虚拟的；你可以在这里找到太平洋时间上午10:30开始的直播。

以下是今天将首次亮相的公司，按预期顺序排列:

  • Metabob : 一个人工智能工具，旨在帮助你找出Python代码中可能存在的bug。作为NEC的一个分支，这是第一家作为前面提到的炼金术士项目的一部分的公司。
  • Laundris : 一个旨在帮助酒店更好地管理亚麻制品、预测需求和处理采购的平台。
  • Utrust : 帮助企业更轻松地接受和管理加密货币支付
  • EVE : 一个仪表板，帮助公司将他们的车队电动化，概述充电状态和费用。
  • Measurecare : 构建一个旨在与AI/机器学习一起使用的放射学成像数据集。他们建立了一个ReCaptcha风格的系统，其中他们的合作放射科医生通过“同行评审”帮助认证和标记放射学图像，以进一步改善数据。
  • Eunimart cross border:自动化工具，帮助中小型企业扩展到新的地区，确定正确的定价，并确定该地区受欢迎的竞争产品。
  • Growfitter : 印度的“激励健康”平台。Growfitter与当地健身中心合作，鼓励用户成为会员，然后为他们的体育活动提供奖励/返现激励。
  • Bitreel : 建造可定制的3D展厅来展示产品，旨在将宜家之旅变成更像电子游戏的居家体验。该公司表示，其3D浏览器可以在任何现代浏览器中运行。
  • 阿胡拉AI : 阿胡拉说它“用机器学习重现了一对一的辅导体验”；更具体地说，它旨在根据每个员工的学习风格实时调整员工培训计划。
  • 蚂蚁媒体 : 一个低延迟(~0.5秒)流媒体平台，旨在轻松部署在AWS、Azure等平台上。
  • 管道机器人 : 顾名思义，它是一种在管道里行走的机器人。专注于工业/公用事业/化学公司，他们的机器人被建造来导航复杂的管道结构，以更有效地处理检查、3d绘图等。
  • Refactr : 一个自动化平台，旨在帮助公司的网络安全和开发团队更轻松地合作。
  • Stargazr : 旨在帮助财务团队进行预测的人工智能工具
  • :将机器学习算法应用于患者的实验室/健康数据，目标是提高体外受精的成功率。
  • Chatalytic : 构建人工智能，以自动衡量客户支持对话(语音/聊天)的成功，并自动识别“趋势”支持主题，以更快地标记潜在问题。
  • 绽放行为 : 这个平台旨在帮助团队更好地合作，并帮助团队领导识别团队成员认为彼此最强的特质
  • Sif Homes : 结合电力线适配器和网状网络的概念，构建Wifi热点，通过您家中现有的电线进行通信。
  • 威尔第 : 平台和灌溉硬件，帮助农民根据单个植物的需求浇水/施肥，旨在减少水资源浪费，提高作物产量。
  • Geecko : 打造“编程游戏”是为了融入招聘过程。

  • VIP化 : “验证即服务”，旨在为日益增多的二手转售平台办理产品认证。
  • Siro : 一款工具，旨在通过自动识别客户对话中的“可指导时刻”，帮助销售经理更积极有效地指导销售代表。
  • Stratodyne : 从软式飞艇中构建高分辨率航拍图像；例如，可以帮助农民识别从上面更容易发现的问题。

  • Neuronix AI: 旨在降低基于计算机视觉的人工智能的成本和功耗。
