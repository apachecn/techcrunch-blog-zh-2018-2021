思科再次出击，拿下威胁评估工具 Kenna Security，成为本周第三笔收购

思科本周一直忙于收购，今天该公司宣布收购威胁评估平台肯纳安全，这是该公司本周收购的第三家公司。两家公司没有透露收购价格。

有了 Kenna，思科就有了一家初创公司，它使用机器学习来整理每天进入安全系统的大量威胁数据，并优先考虑最有可能造成最大损害的威胁。这可能是一个非常有用的工具，当威胁比比皆是，并不总是很容易知道把你有限的安全资源放在哪里。思科计划采用该技术，并整合到其 SecureX 平台中。

思科高级副总裁兼安全业务集团总经理吉·里顿豪斯在宣布与 Kenna 交易的博客文章中写道，他的公司将获得一款产品，该产品将思科现有的威胁管理能力与 Kenna 基于风险的漏洞管理技能结合在一起。

“这就是为什么我们很高兴地宣布我们有意收购 Kenna Security，Inc .,这是一家公认的基于风险的漏洞优先排序领域的领导者，拥有超过 1400 万项受保护的资产和超过 127 亿个受管理的漏洞。里顿豪斯在博客中写道:“利用数据科学和现实世界的威胁情报，它有能力从多供应商环境中引入数据，并提供全面的 it 漏洞风险视图。

长期以来，安全领域一直很复杂，但随着员工因为 COVID 而从家里搬到办公室，这一问题在去年变得更加突出。在一个威胁形势瞬息万变的世界里，拥有一个能够优先考虑其武库中的哪些内容的工具可能会非常有用。

Kenna Security 首席执行官卡里姆·图巴(Karim Toubba)给出了一个典型的被收购的高管理由:在思科的领导下，他拥有了一个比他的公司独自一人所能建立的更大的市场。

“现在是我们改变行业的机会:一旦收购完成，我们将向今天使用思科 SecureX 的 7000 多名客户提供 Kenna 开创性的基于风险的漏洞管理(RBVM)平台更近一步。他在公司博客中写道“这一单个行动将成倍增加 Kenna 的技术对世界保护网络、端点和基础设施的影响。

该公司成立于 2010 年，自称是 RBVM 领域的先驱。根据 PitchBook 的数据，它以 3.2 亿美元的估值筹集了超过 9800 万美元的资金。客户包括汇丰银行、加拿大皇家银行、美泰和 Quest Diagnostics。

对于这些客户来说，随着两家公司共同努力将 Kenna 技术集成到 SecureX 平台中，该产品在某个时候将不再是独立的。完成后，独立客户将不得不购买思科解决方案，以继续使用肯纳技术。

思科在收购方面度过了忙碌的一周。该公司周二宣布有意收购塞多纳系统公司，周三宣布有意收购 T2 社会实验室，今天宣布了这一消息。对于任何一家公司来说，这都是一周内的大量活动。该交易预计将在思科 2021 财年第四季度完成。Kenna 的 170 名员工将加入由里顿豪斯领导的安全业务集团。
