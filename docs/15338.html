Stripe 收购 Bouncer，将把其卡认证集成到雷达欺诈检测工具中

原文：https://web.archive.org/web/https://techcrunch.com/2021/05/14/stripe-acquires-bouncer-will-integrate-its-card-authentication-into-the-radar-fraud-detection-tool/

继今年早些时候融资 6 亿美元之后，支付巨头 Stripe 一直在进行收购，以继续扩大业务。在最新的发展中，该公司已经收购了位于奥克兰的初创公司 Bouncer ，该公司已经建立了一个自动运行卡认证和检测基于卡的在线交易欺诈的平台。它的技术是为移动交易量身定制的，包括一个流程，帮助用户在被错误标记时验证自己，并合法地回到应用程序(因此得名)。

交易条款尚未披露，但 Stripe 正在收购 Bouncer 的技术和团队，并将整合到 Stripe Radar 中。始于 2018 年，Radar 是 Stripe 基于人工智能的反欺诈技术工具集，大部分技术——专注于防止 Stripe 平台上的欺诈交易——迄今为止都是内部开发的。Stripe 表示，雷达每年已经阻止了“数亿美元的商业欺诈”。

“Bouncer 是现代互联网企业的一个很好的工具。它使他们能够快速识别被盗卡，同时也确保合法客户可以交易而不会被阻止，”Stripe Radar 的业务主管西蒙·阿斯科特在一份声明中说。“我们非常高兴地欢迎 Bouncer 团队，以及他们多年来为企业构建支付认证软件的经验，为 Radar 用户提供他们的技术。随着高级卡扫描功能的增加，Stripe Radar 将能够阻止更多欺诈，并进一步增加全球数百万依赖 Stripe 的企业的收入。”

这笔交易是在 Stripe 宣布收购 TaxJar 以将基于云的销售税计算工具引入其支付平台几周之后达成的。

就像 Stripe 本身一样，Bouncer 是在 Y Combinator 孵化的，就其而言，是其 2019 年夏季团队的一部分。除了 YC，它还从商业风险投资公司和先锋基金筹集了资金，但从未披露总共筹集了多少资金。

不要与波兰营销技术初创公司 Bouncer 混淆，Oakland Bouncer 是由 Will Megson(首席执行官)和 Sam King(首席科学家)共同创立的，他们在身份验证方面有着有趣的经历，从学术界到在快速发展的公司工作，这些公司都是验证技术的最大采用者。

两人都曾在按需运输服务 Lyft 工作多年，从事欺诈、身份和支付管理工作。在此之前，梅格森在 Groupon 金除了在加州大学戴维斯分校担任计算机科学副教授之外，还在 Twitter 从事账户安全方面的工作，创建了假账户团队。

Groupon 是 Bouncer 目前合作的客户之一，另外还有 OfferUp、ibotta 和 Dealerware。Bouncer 将在交易后保持目前的服务和客户。

Radar 是目前在许多层级销售，从免费到每筛选交易 6 便士不等，取决于它的使用方式(有一个更基本的机器学习层，和一个针对欺诈团队的增强层，价格也取决于客户是使用 Stripe 的标准定价费用还是其他费用)。Stripe 还提供按存储容量使用计费保护服务，价格为每笔交易的 0.4%，并为 Radar 客户提供分析工具，以了解正在发生的事情。

Stripe 表示，自推出以来，Radar 已经阻止了超过 10 亿美元的欺诈交易。

Bouncer 目前也有不同的定价，从免费到 0.15 美元/扫描的基本解决方案，或者定制价格的定制服务。

将 Bouncer 的卡扫描和风险技术集成到 Radar stack 中，不仅会让人们从 Stripe 购买这些服务的交易变得更加划算，还会使这些工具更加有效。

正如 Stripe 所描述的那样，当 Radar 标记一笔交易时，Bouncer 的卡筛选和验证技术将作为一种“动态干预”来确认客户在交易时是否有合法的卡。这样做有助于减少误报，误报在高风险交易中更为常见(如大额交易，或者某人连续进行了几笔交易，或者其他在系统中不寻常的支付活动)。

我们一直处于新身份认证技术的浪潮中，包括生物识别和其他创新，但 Bouncer 采取了一种在摄取点上不太高科技的方法——只需要手机的摄像头和客户正在使用的卡。当交易被标记并发送给 Bouncer 进行验证时，Bouncer 通过请求支付卡的图片来工作(该图片可以基于任何支付卡类型，并且可以是弱光图片)。

然后，它通过其 PCI 和 GDPR 兼容的系统运行，看看它是被盗还是真实的。如果是真实的，交易继续；被盗，交易取消。整个过程不到一秒钟(当然不包括你拍一张照片的时间)。

对于 Bouncer 来说，这个想法是 Stripe 的机器学习引擎将反过来帮助 Bouncer 变得更有效。

Bouncer 首席执行官威尔·梅格森(Will Megson)在一份声明中表示:“我很高兴我们能够在整个 Stripe 网络中扩展我们先进的卡验证技术，以帮助企业增加收入，同时进一步减少幕后的欺诈行为。”“雷达学习的相同信号将使 Bouncer 更有效，而 Bouncer 反过来将使雷达更有效。加入雷达团队让我们无比激动。”

多年来，Stripe 进行了多次收购，以引进关键技术，其中一次是收购 Lagos 的 PayStack(另一位 YC 校友)，帮助 Stripe 进入非洲和更多新兴市场，并为其提供服务。

其中至少有两个是为了帮助引进技术专家和技术来构建其合规性和认证工具。在 2016 年，Stripe 悄悄收购了茶壶，这是一家硅谷初创公司，一直致力于开发用于身份验证、信任、信用和金融交易所需的其他工具的 API。该公司的联合创始人在转行做其他事情之前在公司呆了几年。

2019 年，Stripe 收购了爱尔兰一家名为 Touchtech 的初创公司，以引进技术，为欧洲强有力的客户认证法规做准备。

对更好、更复杂的工具以确保在线交易合法的需求不会很快出现。恶意黑客攻击——以及由此导致的获取可用于消费者欺诈的个人数据的后果——仍然是一个持续的威胁。与此同时，电子商务继续成为越来越主流的活动，扩大了消费者群体，也增加了出错的几率。
