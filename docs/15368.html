硬件黑客将在线多人游戏带到最初的 Game Boy

挪一下，Xbox 和 PlayStation。在线多人游戏世界中出现了一个新的敌人！是…呃，游戏机。就像 1989 年那辆打不坏的灰色 4.19 兆赫坦克。

虽然 Game Boy 从一开始就有一些本地多人游戏，但使用它意味着通过一种叫做 link cable 的配件将你的 Game Boy 与另一个 Game Boy 物理连接起来。如果你想和几英尺之外的人玩任天堂……那么，你只需要等上几十年。

在令人印象深刻的技能展示中，硬件黑客stack mashing成功地对游戏男孩的 link cable 协议进行逆向工程，并有效地欺骗它在互联网上工作。Game Boy 通过连接到 Raspberry Pi 的连接线连接到一个定制的桌面客户端，然后 pings 一个在线游戏服务器，作为你和你的对手之间的桥梁。游戏机以为它在和其他游戏机通话，却不知道它实际上是在和远在地球另一边的服务器通话。

他们做的第一个游戏？俄罗斯方块！

让任何给定的游戏运行起来(想象一下在互联网上和某人交易你在 1998 年抓到的神奇宝贝！)将需要对该游戏独特的通信协议进行逆向工程，所以目前只有俄罗斯方块。幸运的是，stacksmashing 已经开放了到目前为止已经构建的各种组件的源代码，因此有了一些可以构建的基础。因为没有人一起玩，整个事情就没意思了，所以还有一个不和谐频道，只是为了找到其他掉进这个兔子洞的人。甚至还有一个定制的 PCB 正在工作中(15 美元，预计 6 月份可以预订)，它将处理链接电缆和 Raspberry Pi 之间的连接，消除了您撕碎链接电缆以暴露其电线并使其工作的需要。

Stacksmashing 最近也因破解并修改苹果的空中标签，以及将 Game Boy 变成一个(可笑的动力不足)比特币矿工而成为头条新闻。
