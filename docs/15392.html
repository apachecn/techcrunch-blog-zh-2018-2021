谷歌今天在 I/O 上宣布的一切

原文：https://web.archive.org/web/https://techcrunch.com/2021/05/18/everything-google-announced-at-i-o-today/

今年来自谷歌的 I/O 活动强调的是“我们正在打造一些很酷的东西”，而不是“这是你明天可以使用或购买的东西”但在该公司山景城园区内外举行的半现场活动也有一些有趣的惊喜。请继续阅读所有有趣的内容。

Android 12 获得了全新的外观和一些生活质量功能

【T2

我们知道 Android 12 已经发布了几个月，但是今天是我们第一次真正看到这个世界上最受欢迎的操作系统的下一个重大变化。一种新的外观，称为 Material You(是的)，专注于用户，应用程序和一天中的时间或天气等事情，以动态地改变 UI 的颜色和其他方面。一些安全功能，如新的摄像头和麦克风使用指示器，以及一些“私人计算核心”功能，使用手机上的人工智能进程来定制回复和通知。今天有一个为喜欢冒险的人准备的测试版！

哇，Android 现在支持 30 亿台设备

Subhed 说明了一切(但在这里阅读更多)。高于 2017 年的 20 亿。

智能画布将文档、工作效率和视频通话融合在一起

【T2

数百万人和企业使用谷歌的生产力和协作工具套件，但该公司认为，如果他们不那么孤立，会更好。现在有了 Smart Canvas 您可以在一起处理共享文档时进行视频通话，并从硬盘和其他地方获取信息和内容。看起来很复杂，但可能很方便。

人工智能对话变得更有对话性

如果你脱离了脚本，用一种对你来说似乎正常但对语言模型来说完全不可理解的方式来问一些问题，这就有点太容易难倒人工智能了。谷歌的 LaMDA 是一种新的自然语言处理技术，它使与人工智能模型的对话对不寻常或意想不到的查询更具弹性，使其更像真人，而不像搜索功能的语音界面。他们展示了与拟人化版本的冥王星和纸飞机的对话。是的，听起来确实很奇怪。

谷歌建造了一个未来派的 3D 视频电话亭

T2】

在主题演讲中最令人惊讶的事情之一必须是项目 Starline，这是一个高科技的 3D 视频呼叫设置，它使用谷歌以前的研究和 Lytro DNA 来显示系统两侧的人的逼真 3D 化身。它仍处于实验阶段，但看起来很有前景。

Wear OS 进行了改造，推出了许多关注健康的应用

图片来源:谷歌

很少有人想在智能手表上看电影，但很多人喜欢用它来跟踪自己的脚步、冥想和其他与健康相关的活动。 Wear OS 注入了大量 Fitbit 的 DNA，集成了健康跟踪功能和许多第三方应用，如 Calm 和 Flo。

三星和谷歌宣布统一智能手表平台

这两个移动巨头多年来一直是手机世界的好朋友，但在可穿戴设备方面，他们仍然是竞争对手。然而，面对苹果在智能手表领域的绝对主导地位，两家公司搁置了分歧并宣布他们将在一个“统一的平台”上工作，这样开发者就可以制作在 Tizen 和 Wear OS 上都能工作的应用程序。

他们也在一起做可折叠的东西

显然，谷歌和三星意识到，除非他们做一些非常酷的事情，否则没有人会购买可折叠设备，合作是最好的前进方式。因此，两家公司也将合作改进折叠屏与 Android 的交互方式。

安卓电视点击量达到 8000 万台，并增加了电话遥控功能

图片来源:谷歌

智能电视领域是一个竞争激烈的领域，经过几次启动后，谷歌真的通过 Android TV 实现了这一点，该公司宣布 Android TV 的月活跃设备数量已经达到 8000 万台——将 it、Roku 和亚马逊(后两家拥有约 5000 万月活跃账户)置于同一联盟。该公司还展示了一个强大的新的基于手机的远程应用程序，它将(除其他外)使输入密码比使用遥控器上的 d-pad 更好。开发人员会很高兴听到有一个新的谷歌电视模拟器，Firebase 测试实验室将支持 Android 电视。

你的安卓手机现在(也)是你的车钥匙

好吧，假设你有一个非常新的内置超宽带芯片的安卓设备。谷歌首先与宝马合作，其他汽车制造商很可能很快也会合作，使成为一种当你靠近汽车时解锁汽车的新方法，或者在不使用遥控器或蓝牙的情况下交换基本命令。你问为什么不是蓝牙？蓝牙已经过时了。UWB 是新的。

Vertex 在一个地方收集机器学习开发工具

谷歌及其兄弟公司都是人工智能研究的领导者，也是其他人做自己人工智能工作的流行平台。但它的机器学习开发工具有点分散——有用但不连贯。 Vertex 是一个新的企业人工智能开发平台，它将许多这些工具放在一个地方，并与可选的服务和标准紧密集成。

有新一代的谷歌定制人工智能芯片

T2】

谷歌做了很多机器学习的东西。就像，很多很多。因此，他们一直在努力制造更好、更有效的计算硬件，以处理这些人工智能系统产生的大量处理负载。 TPUv4 是最新的，速度是老款的两倍，很快就会被打包成 4096 强的豆荚。为什么是 4096 而不是 4000？与计算中其他数字存在的原因一样:2 的幂。

他们正在为一些新的照片功能提供动力，包括一个可怕的功能

不，谢谢你

Google Photos 是一项伟大的服务，该公司正试图利用大多数用户拥有的庞大照片库来寻找像“与家人在沙发上自拍”和“带着我的幸运帽旅行”这样的模式，以此作为重新进入档案的有趣方式。太好了！但他们也在拍两张相隔一秒钟的照片，并让一个人工智能对它们之间的东西产生幻觉，导致一种真正奇怪的运动形式，射向神秘的山谷，希望它永远不会出现。

忘记密码了？谷歌机器人来救援了

谷歌的“人工智能为你预约头发”服务 Duplex 并没有引起轰动，但该公司已经找到了一种新的应用方式。如果您忘记了密码，Duplex 会自动填写您的旧密码，选择一个新密码并让您在提交到网站之前复制它，这一切都是通过与网站的正常重置界面进行交互。目前，它只能通过 Chrome 在 Twitter 和少数其他网站上运行，但是，嘿，如果你经常遇到这种情况，也许会给你省些麻烦。

进入购物图

图片来源:谷歌输入输出 2021

我们读者中的老年人可能还记得谷歌命运多舛的购物界面 Froogle。嗯，它又回来了……算是吧。该计划是当你搜索某样东西时，在谷歌界面中包含大量的产品信息，从价格到星级、可用性和其他信息。它从零售网站获取这些信息，包括你的购物车里是否有东西。很难想象这一切对谷歌以外的任何人都有什么好处，但很自然，他们把它定位为全面的胜利。尤其是对于新合作伙伴 Shopify。(我，我用 DuckDuckGo。)

Flutter 跨平台开发包获得更新

很多开发者已经接受了 Google 的 Flutter 跨平台 UI 工具包。今天发布的最新版本增加了一些安全设置、性能改进和工作流程更新。还有更多的未来。

Firebase 也得到了更新

流行的开发者平台 Firebase 也获得了许多新的和更新的特性。远程配置得到了很好的更新，允许开发人员根据个人用户类型定制应用程序体验，应用程序检查提供了抵御外部威胁的基本安全级别。这里有很多东西可供开发者咀嚼。

Android Studio 的下一个版本是北极狐

图片来源:谷歌

谷歌 Android Studio 环境的下一个版本的测试版即将推出，它被称为北极狐。它有一个全新的 UI 构建工具包，名为 Jetpack Compose，内置了一系列可访问性测试，以帮助开发人员使他们的应用程序对残疾人更容易访问。现在，连接到设备上进行测试应该也更容易了。哦，还有一个苹果芯片版本的 Android Studio。
