Finary 希望为下一代创造财富管理仪表板

原文：https://web.archive.org/web/https://techcrunch.com/2021/05/18/finary-wants-to-create-the-wealth-management-dashboard-for-the-next-generation/

来认识一下 Finary ，这是一家新的法国创业公司，它希望改变你管理储蓄、投资、抵押贷款、房地产资产和加密货币的方式。该公司允许你汇总你在不同银行和金融机构的所有账户，这样你就可以随着时间的推移全面跟踪你的财富。

在加入 Y Combinator 之后，这家初创公司刚刚完成了由 Speedinvest 牵头的 270 万美元(€220 万美元)的种子轮投资，基马风险投资公司和拉斐尔·沃利姆(raphal Vullierme)等天使投资人也参与了投资。

如果你认识有很多钱的人，他们很可能至少有 40 或 50 岁——毕竟，你不可能一夜暴富。他们倾向于通过提供量身定制服务的财富管理服务来管理自己的投资组合。

“财富管理中的技术含量很低。联合创始人兼首席执行官穆尼尔·拉古内告诉我:“顾问们也会受到激励，向你推销一些特别的金融产品。”。在这种情况下，负责金融产品的公司正在为顾问——而不是客户——创造收入。

与此同时，新一代投资者开始积累大量财富。然而，他们没有合适的工具来合理分配。年轻人希望直接看到信息。他们想要一种实时或接近实时跟踪信息的方法。他们希望能够根据这些数据采取一些行动。

Finary 希望在这些原则的基础上建立这项服务。它从一个基于 API 的聚合器开始。当您创建一个金融账户时，您可以将它与您的所有其他账户连接起来，包括银行账户、经纪账户、抵押贷款和房地产、黄金、加密货币等。

这家初创公司利用各种开放的银行 API，尽可能做到面面俱到。例如，“你可以将一个 Robinhood 账户和一个 Crédit Mutuel de Bretagne 账户联系起来，”Laggoune 说。在幕后，Finary 使用 Plaid 和 Budget Insight，运行自己的比特币和以太坊节点来跟踪钱包地址，并通过公共数据和专有算法来估计你家的价值。

之后，你可以看到你有多少钱，它是如何在你的投资池之间分配的，你的黄金和加密货币资产的当前价值等等。

“我们的长期愿景是，我们希望为欧洲建立一个虚拟财富管理机构，”拉格霍恩说。

这就是为什么 Finary 最近推出了名为 Finary+的高级订阅。使用 premium 帐户，您可以看到您支付了多少费用，并跟踪您的表现-随着时间的推移，将会添加更多功能。

在推出其平台几个月后，Finary 已经跟踪了€数千名用户的 20 亿美元资产。随着今天的融资，这家初创公司将向法国、欧洲和美国的更多国家和更多金融机构推出其服务。该公司还在开发移动应用程序。

这是对财富管理的一种有趣的看法，因为 Finary 没有试图重新发明轮子。传统玩家希望你使用一家银行来满足你所有的财务需求。但是你最终要支付很多费用，而且你不得不使用旧的笨重的界面。

金融不是另一种财富管理服务。这是一项全方位的服务，让您可以使用多家银行和服务，同时掌控自己的资产。
