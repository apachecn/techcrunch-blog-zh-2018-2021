谷歌评论 rss

原文：https://web.archive.org/web/https://techcrunch.com/2021/05/19/undead-again-google-brings-back-rss/amp/

图片来源: DSGpro / Getty Images

Chrome，至少在其 Android 上的实验性金丝雀版本(仅针对美国用户)，将在未来几周内获得一个有趣的更新，恢复 RSS ，这是一种曾经流行的格式，用于从谷歌阅读器和类似服务中获得所有你喜欢的网站的更新。

在 Chrome 中，用户将很快看到支持 RSS 的网站的“关注”功能，浏览器的新标签页将获得本质上(非常)基本的 RSS 阅读器——我猜你几乎可以称之为“谷歌阅读器”

现在我们不是在这里谈论一个成熟的 RSS 阅读器。新的标签页将按时间顺序显示你所关注的网站的更新，但它看起来不像你可以轻松地在订阅源之间切换。不过，这是个开始。

图片来源:谷歌

“今天，人们有许多方法来与他们喜欢的网站保持联系，包括订阅邮件列表、通知和 RSS。谷歌 Chrome 产品经理 Janice Wong 在今天的更新中写道:“对于任何一个人来说，这都是一个很大的挑战，所以我们正在探索如何简化在 Chrome 中直接从你最喜欢的网站获取最新和最棒的体验，建立在开放的 RSS 网络标准之上。”。“我们的愿景是帮助人们在网络上与他们喜爱的出版商和创作者建立直接联系。”

一位谷歌发言人告诉我，该公司实现这一目标的方式是让谷歌更频繁地抓取 RSS 订阅源，“以确保 Chrome 能够在新标签页的下一部分向用户提供最新、最棒的内容。”

RSS 是 Web 2.0 时代的基础技术之一。即使在今天，它仍然是从你最喜欢的网站获得及时更新的最简单的方法(尽管有些网站可能不再提供提要)，没有任何推荐算法妨碍你。然而，尽管 RSS 总是非常有用，但用户体验并不总是理想的，尽管谷歌阅读器(Google Reader)和 Feedly 等服务做了很多工作，使订阅提要和获取更新变得足够简单。但是当谷歌在 2013 年 Google+的祭坛上提供谷歌阅读器时，那个时代结束了，尽管顽固的新闻迷们仍然保留着他们的 Feedly 账户和旧的网络新闻专线。

我想很多人会很高兴看到谷歌将它重新作为其浏览器的核心功能。如果你更喜欢一个开放的网络，RSS，尽管偶尔有些笨拙，也是一个不错的选择。

然而，目前这只是一个实验。谷歌表示，它希望从“出版商、博客作者、创作者和开放网络的公民”那里收集反馈，因为它的目标是建立“用户和网络出版商之间在 Chrome 上的更深参与”。希望不会一直这样下去。
