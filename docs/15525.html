微软Surface Duo获得双屏游戏支持

12月，微软写了一篇博客文章，强调“Surface Duo的未来一年”该消息强调了这款双屏设备即将在美国以外地区上市(今年包括加拿大、英国、法国和德国)，以及一些功能。几乎所有评测过这款设备的人(包括在内的美国人)都承认，这是一项正在进行中的工作。

在测试版中推出该功能后，微软今天发布了安卓版 Xbox云游戏的更新，这将释放该产品的一些娱乐潜力。这是该公司已经讨论过，甚至预览过的东西，但直到现在，它仍然是少数几个盲点之一。

这款应用的工作方式与你在Duo上的预期一样——在顶部屏幕上显示游戏，并在合成模式方向上将底部转换为Xbox控制器的虚拟版本。显然，在触摸屏面前你只能做到这一步，但游戏是一个真正的应用程序，有第二块屏幕真的很方便。

正如 Engadget指出的，现在有超过50款游戏支持Xbox触摸控制。Xbox Game Pass Ultimate订阅用户将可以获得这些内容。

作为一款被大肆宣传和热切期待的可折叠外形的替代产品，这对组合在推出时令人失望。该公司此后将产品价格从1500美元降至1000美元——这并不完全是销售良好的迹象——但像这样持续的软件支持或许表明，尽管早期出现了失误，该公司将继续支持该系列。
