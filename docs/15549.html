微软 Azure 推出 PyTorch 企业支持

原文：https://web.archive.org/web/https://techcrunch.com/2021/05/25/microsoft-azure-launches-pytorch-enterprise/

微软今天宣布了 PyTorch Enterprise，这是一个新的 Azure 服务，为开发者在 Azure 上使用 PyTorch 提供了额外的支持。这基本上是微软对 PyTorch 的商业支持。

PyTorch 是一个以 Python 为中心的开源机器学习框架，专注于计算机视觉和自然语言处理。它最初是由脸书开发的，至少在某种程度上可以与谷歌流行的 TensorFlow 框架相媲美。

微软公司通信副总裁 Frank X. Shaw 称新的 PyTorch 企业服务为开发人员提供了“在数据科学工作中使用 PyTorch 的组织更可靠的生产体验”。

Shaw 解释说，通过 PyTorch Enterprise，微软首要统一支持计划的成员将获得优先请求、实际支持以及热修复、错误和安全补丁的解决方案等好处。微软每年还会选择一个 PyTorch 版本进行长期支持。

Azure 已经使 PyTorch 在 T2 变得相对容易使用，微软也长期投资于这个库，例如，去年接管了 PyTorch for Windows 的开发。正如微软在今天的公告中指出的，PyTorch 的最新版本将与 Azure 机器学习集成，该公司承诺将从开发者那里获得的 PyTorch 代码反馈给公共 PyTorch 发行版。

在 Windows 10 和许多流行的 Linux 发行版上，PyTorch 版本 1.8.1 和更高版本将提供企业支持。

“微软的这一新的企业级产品填补了一个重要的空白。PyTorch 为我们的研究人员设计模型和进行实验提供了前所未有的灵活性，”Nuance 的高级首席研究科学家 Jeremy Jancsary 说。“然而，在生产中服务这些模型可能是一个挑战。微软的直接参与让我们可以放心地将新版本的 PyTorch 部署到 Azure。”

通过这一新产品，微软通过在开源项目的基础上提供额外的服务，为初创公司提供了开源货币化剧本的一页。由于 PyTorch 不是由一家初创公司开发的，只是让一家主要的云提供商在开源代码的基础上提供自己的商业版本，这似乎是一个没有争议的举动。
