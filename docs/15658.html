Belvo 是 LatAm 对 Plaid 的回应，它筹集了 4300 万美元来扩展其用于金融服务的 API

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/01/belvo-latams-answer-to-plaid-raises-43m-series-a-to-scale-its-api-for-financial-services/

贝尔沃 ，一家建立了开放金融 API 平台的拉丁美洲初创公司，今天宣布在 A 轮融资中筹集了 4300 万美元。

硅谷和拉丁美洲的风险投资公司和天使投资人参与了融资，包括 Future Positive、Kibo Ventures、FJ 实验室、Kaszek、MAYA Capital、VentureFriends、Rappi 联合创始人兼总裁 sebastán mejíA、Wise(前 Transferwise)首席技术官 Harsh Sinha 和 Nubank 首席执行官兼创始人 David Vélez。

根据 Crunchbase 的数据，Belvo 认为这轮融资是拉丁美洲金融科技公司有史以来最大的一轮 A 轮融资。2020 年 5 月，Belvo 筹集了 1000 万美元的种子轮 co——由硅谷的 Founders Fund 和阿根廷的 Kaszek 领投。

Belvo 旨在与拉丁美洲领先的金融科技公司合作，涵盖 neobanks、信贷提供商和拉丁美洲人日常使用的个人金融产品等垂直行业。

这家初创公司的目标是开发人员优先的 API 平台，可用于访问和解释最终用户的金融数据，以在拉丁美洲构建更好、更高效和更具包容性的金融产品。流行的 neobank 应用程序、信贷提供商和个人金融工具的开发人员使用 Belvo 的 API 将银行账户连接到他们的应用程序，以释放开放银行的力量。

正如 TechCrunch 高级编辑 Alex Wilhelm 去年在 这篇文章 中解释的那样，Belvo 可能被认为与美国的 Plaid 类似，但更适合拉丁美洲市场，因此它可以接受更多样化的数据，以更好地满足其服务的各个市场的需求。

因此，尽管 Belvo 的目标“类似于 Plaid 的总体目标”，但联合创始人兼联合首席执行官 Pablo Viguera 告诉 TechCrunch，Belvo 不仅仅是建立一个银行 API 业务，希望将应用程序连接到金融账户。相反，Belvo 希望构建一个financeAPI，它可以获取比此类系统通常收集的信息更多的信息。拉丁美洲银行严重不足，没有银行账户，因此来自更多来源的数据越多越好。

“从本质上来说，当你考虑开放银行或开放金融时，我们正在推动类似的结果(如格子)，”维格拉说。“我们正在努力实现金融数据访问的民主化，让最终用户能够传输这些数据，并与他们想要的任何人共享这些数据。”

该公司经营的前提是，仅仅因为该地区的大量人口缺乏银行服务，并不意味着他们在财务上不再活跃。贝尔沃的目标是将各种账户联系起来。例如，Viguera 告诉 TechCrunch，拉丁美洲的一些零工经济公司正在发行自己的卡，允许工人在当地的小商店兑现现金。随着时间的推移，所有这些交易都是可以使用 Belvo 连接起来的数据，比我们在国内习惯的网络要广得多。

根据 Viguera 的说法，该公司将银行和非银行连接在一起的工作是该公司允许“任何金融科技或任何开发者访问和解释用户财务数据”的目标的关键。

维格拉和联合首席执行官奥里奥尔·廷托雷于 2019 年 5 月创立了贝尔沃，它是 Y Combinator2020 年冬季批次的一部分。该公司表示，自去年推出其平台以来，它已经在墨西哥、巴西和哥伦比亚建立了超过 60 家公司的客户群，每月处理数百万次 API 调用。

这一点很重要，因为正如 Alex 去年指出的那样， 与 API 领域的其他参与者类似，Belvo 对其客户使用的每个 API 调用收费(从这个意义上说，它有一个与 Twilio 类似的模型)。

图片来源:联合创始人兼联合首席执行官 Oriol Tintore 和 Pablo Viguera / Belvo

此外，在过去的一年里，Belvo 表示，它将其 API 覆盖范围扩展到了 40 多家金融机构，使公司能够连接到拉丁美洲 90%以上的个人和企业银行账户，以及税务机关(如墨西哥的 SAT)和 gig 经济平台。

“本质上，我们获取非结构化金融数据，这些数据可能是个人在银行之外获得的，例如我们与优步和 Rappi 等 gig 经济平台的集成。我们可以从他们的优步应用程序(有点像银行应用程序)中获取司机的信息，并将其转化为有意义的银行信息，第三方可以利用这些信息进行评估，就像这些数据来自银行一样，”Viguera 解释道。

这家初创公司计划利用新资本扩大其产品供应，继续扩大其地理覆盖范围，并将其目前 70 人的员工人数增加一倍。具体来说，Belvo 计划年底前在墨西哥和巴西招聘 50 多名工程师。它目前在墨西哥城、圣保罗和巴塞罗那设有办事处。该公司还计划在墨西哥和巴西推出银行对银行支付服务。

贝尔沃目前在墨西哥、哥伦比亚和巴西开展业务。

Viguera 告诉 TechCrunch，但是它在拉丁美洲的其他市场看到了“大量的机会”,尤其是在智利、秘鲁和阿根廷。"在适当的时候，我们将寻求在那里扩张."

Future Positive 的创始合伙人弗雷德·布莱克福特认为，贝尔沃代表着“该地区金融业真正的转型机遇”

kas zek 的联合创始人兼管理合伙人 Nicolás Szekasy 指出，拉丁美洲对金融服务的需求正以指数速度增长。

“贝尔沃正在开发基础设施，使大型机构和新一代年轻玩家能够成功部署他们的解决方案，”他说。“Oriol、Pablo 和 Belvo 团队一直在领导一个复杂平台的开发，该平台解决了非常复杂的技术挑战，该公司的指数级增长反映了它如何提供完全符合市场要求的产品。”
