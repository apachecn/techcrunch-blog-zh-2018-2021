巴西的 idwall 为身份验证平台 TechCrunch 筹集了 3800 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/01/brazils-idwall-raises-38m-for-identity-validation-platform/

在线欺诈和身份盗窃是一个全球性问题，随着新冠肺炎疫情在线交易的增加，这个问题变得更加严重。特别是，据估计，巴西公司每年因欺诈损失超过 410 亿美元。

为了正面解决这个问题，林肯·安藤和拉斐尔·梅洛在 2016 年年中创办了 idwall。总部位于圣保罗的 idwall 最初是一个自动化背景调查解决方案，后来发展成为一套数据和身份验证及风险分析产品。对于消费者市场，其“MeuID”应用程序针对的是那些希望改变自己身份和分享数据方式的用户。

现在，巴西 regtech 在由耐力公司牵头的 C 轮融资中筹集了 3800 万美元。

GGV 资本、monashees、Canary、Qualcomm Ventures LLC、ONEVC、Peninsula 和 Norte 也参与了融资，使其总融资额达到近 5000 万美元。

该公司表示，在 2017 年至 2020 年间，其增长了 1，458%，平均每年增长 144%。它的 300 多个客户包括 10 家独角兽公司，巴西三大银行中的两家，以及 iFood、克拉罗、Cielo、Loggi、Ebanx、QuintoAndar 和 OLX 等公司。

金融科技构成了其客户群的重要组成部分，2020 年，该公司仅金融行业的客户收入就比 2019 年增长了 588%。

Idwall 通过人脸匹配、背景调查、风险分析、身份验证和自动光学字符识别(OCR)产品，使用机器学习和人工智能来自动化入职流程，以帮助公司避免欺诈。

该公司表示，其应用编程接口通过“快速地并根据合规规则”在公共和私人数据库中搜索来验证个人文件和信息 Idwall 通过首先验证 ID 的真实性来完成所有这些工作。然后，它会确保使用它的人实际上是 ID 的所有者。最后，它会进行全面的背景调查。它声称这一切不到三分钟就完成了。

“我们帮助他们以更安全、更好、更快的方式完成所有这些入职流程，”idwall 联合创始人兼首席执行官 Ando 说道。

多年来，idwall 已经为其客户生成了超过 6500 万份数据报告，该公司表示，这一数字在 2017 年至 2020 年间激增了 5000 倍。该公司声称，这些报告帮助其客户扩大了业务规模，注册了更多自己的客户，优化了合规和 KYC 流程，并减少了欺诈行为。

图片来源: idwall

总的来说，疫情对数字化的推动导致了数字银行账户、移动支付服务以及适应数字平台和/或扩大数字业务的公司数量的大规模增长，从而带来了 idwall 业务的繁荣。

“公司越数字化，客户的期望就越高，市场竞争也越激烈，”安藤说。“我们的使命是始终保持市场创新的领先地位，这就是为什么我们在增长和建立最佳团队开发产品方面投入如此之多。”

其中一部分包括利用新资本招募更多的开发者，强化现有产品并发布新产品。Idwall 计划在未来几个月内将其员工人数从目前的 200 人增加到 300 人左右。该公司还在研究将业务从巴西扩展到整个拉丁美洲的可能性。

安藤说:“巴西面临的许多身份验证和欺诈问题在其他拉美国家也同样存在。“此外，墨西哥和哥伦比亚等地也有高度创新的公司，在身份和技术方面不断超越极限。在巴西，我们还有很多事情要做，但我们看到了一个巨大的机会，可以让我们的使命更进一步。”

尽管如此，在其祖国巴西，近年来的监管变化也导致对 idwall 产品的需求增加。

此外，该公司表示，巴西的文档数据库高度孤岛化，每个州都有自己最常见的身份证件 RG(“Registro Geral”或“General Registry”)的模型。另外，每个州的每个公民都可以获得不同的 RG 文件。

“不可否认的是，数字化入职和自动化身份验证流程对于拉丁美洲市场的发展至关重要，”Ando 说。“在巴西，理解和验证身份和个人数据极其困难。”

此外，总的来说，该公司已经观察到巴西人对于在常规活动中出示身份证是多么的厌倦。Idwall 通过其前面提到的“MeuID”解决方案来帮助解决这一问题，该解决方案在一个钱包中存储了金融科技公司、初创公司、办公楼和其他企业入职流程所需的所有文件。

它的投资者自然是乐观的。

GGV 资本的管理合伙人童士豪说，他的公司的目标是投资那些利用技术为用户和社会创造积极影响的行业领导者。

他形容 idwall 是一家“独一无二”的创业公司。

“idwall 的创始人正在解决巴西的一个巨大但未得到充分服务的问题，并带领公司实现了惊人的增长，”Tung 告诉 TechCrunch。“他们具备成为 LatAm 领先的企业个人数据平台的要素。”

Canary 的管理合伙人马科斯托莱多(Marcos Toledo)指出，idwall 是他的公司的首批投资之一。

他告诉 TechCrunch:“林肯和拉斐尔在解决巴西一个非常相关的问题方面建立和扩大业务的能力引起了我们的注意。”“作为一家公司，他们的文化、技术水平和灵活性在巴西市场也非常出色。”
