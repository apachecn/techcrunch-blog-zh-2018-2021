脸书向所有人开放Instagram的Messenger API

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/02/facebook-opens-its-messenger-api-for-instagram-to-all-businesses/

脸书的年度开发者大会F8刷新(new York Refresh)将于今天晚些时候开幕，它比过去几年更加精简，而且是虚拟的，在此之前，脸书发布了一些消息:所有企业现在都可以使用Messenger API在Instagram上与用户互动。该功能首先面向全球所有开发人员开放，并分阶段面向企业:

在第一阶段，关注人数超过10，000且低于100，000的Instagram账户将连接到API。它计划在7月份(第二阶段)将该计划扩展到粉丝数量在1，000到100，000之间的账户，其余账户将在第三季度上线。

该功能于10月以封闭测试版的形式首次在精选企业中公布——总共有30个开发者和700个品牌。现在，任何使用Instagram与客户互动的品牌或组织都可以使用它。

这个工具的关键点是，这种集成代表着公司如何利用更广泛的脸书平台向前迈出了重要的一步。

在过去，想要与客户互动的品牌要么需要直接通过Instagram，要么通过脸书的统一业务收件箱进行互动，这限制了它们的使用方式，尤其是那些可能需要处理大量流量的公司，或者渴望能够将这些客户互动与更广泛的客户服务数据库联系起来的公司。

相比之下，Messenger API可以集成到公司或品牌可能用来管理沟通的任何第三方应用程序中，无论是Hootsuite或Sprinklr等社交媒体管理平台，还是可以引入其他类型的客户数据(例如保修信息或积分卡号)的CRM应用程序。

脸书指出，从封闭测试中获得的一个关键信息是，品牌和公司希望从一个地方获得更好的沟通管理方式；另一个原因是，他们中的许多人正在加大软件投资，以更好地管理他们的通信和工作流程。因此，将Messenger API扩展到Instagram是这方面早就需要的功能。

将Messenger API扩展到Instagram的举动在几个不同的方面都有意义。首先，一段时间以来，脸书一直在加大对Instagram商业潜力的利用，从广告开始，但扩展到品牌或企业与用户之间的对话等领域，最近又增加了购物功能。脸书还指出，如今90%的Instagram用户至少关注一家企业，因此创建一个更好的途径来管理这些对话是一个合乎逻辑的举措。

与此同时，脸书一直在努力寻找更好地连接其各种应用程序和平台的方法，包括脸书本身、Messenger、WhatsApp、Instagram和Oculus，这不仅是为了让用户能够在它们之间进行互动，也是为了帮助企业在一个更加统一的社交战略中利用它们。推出Messenger API——最初是为了帮助品牌与机器人互动并管理Messenger上的对话——以包括对Instagram的支持符合这两个更大的战略。

对于那些想知道为什么在F8更新之前宣布的人来说？也许这暗示了社交网络在今年的活动中更重要的优先事项:合作伙伴关系，使更多的业务发生在社交网络巨头的平台上。
