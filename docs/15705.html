Prosus 以 18 亿美元收购 Stack Overflow

传奇的程序员问答网站(可能也是互联网上复制粘贴最多的网站之一) Stack Overflow 被收购。华尔街日报报道Prosus(中国游戏巨头/微信开发商腾讯的主要股东)将以 18 亿美元收购 Stack Overflow。

更新: Prosus 在这里的一份新闻稿中确认了 18 亿美元的数字。

虽然可能不是每个人都认识的名字，Prosus——南非 Naspers 的国际资产控股公司——是一个巨人。2001 年，母公司 Naspers 以 3200 万美元收购了腾讯 46.5%的股份。今年早些时候，他们以将近 150 亿美元的价格出售了腾讯 2%的股份。

今天我们很高兴地宣布堆栈溢出加入了 Prosus。Prosus 是一家投资和控股公司，这意味着此次发布的最重要部分是 Stack Overflow 将继续独立运营，由完全相同的团队根据完全相同的计划和完全相同的业务实践运营。不要指望看到重大变化或尴尬的“协同效应”。栈溢出的业务将继续关注可及性和相关性，以及团队的栈溢出。整个公司保持不变:我们只是现在有了不同的所有者。
