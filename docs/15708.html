银行从高盛获得 6000 万美元债务以扩大小型摩托车车队

总部位于柏林的 Tier Mobility 已经筹集了 6000 万美元，帮助这家电动滑板车公司在 2021 年扩大其车队和电池充电站网络。

这些资金来自投资银行公司高盛，就在几周前， Tier 与 Lime 和 Dott 一起获得了伦敦电动滑板车试点许可。随着一个主要新城市的出现和进一步扩张计划的暗示，Tier 将需要大量的前期投资来支付从车队订单到当地仓库到新团队的一切费用。

11 月， Tier 还完成了一轮 2.5 亿美元的 C 轮融资，由软银愿景基金 2 领投。最新的基金是资产支持融资，这意味着高盛实质上是在向 Tier 提供一笔贷款，这笔贷款由该公司的一项资产担保，可能是其小型摩托车。Tier 没有回应有关贷款细节的请求。

Tier 首席财务官亚历克斯·盖尔(Alex Gayer)在一份声明中表示:“这种高度可扩展的资产支持债务工具的规模是微移动领域的第一个改变游戏规则的工具，加速了我们的扩张，巩固了我们在欧洲的市场领导地位。”"这项贷款利用了我们最近的股权融资，将提高我们的资本效率增长."

除了伦敦，在过去的一年里，Tier 还将令人垂涎的迪拜和巴黎加入了它的名单。它在欧洲和中东 12 个国家的 100 多个城市提供。有了这笔新资金，Tier 计划扩大其国际覆盖面，并投资于其多式联运车队，将自行车和轻便摩托车加入其中。

Tier Energy Network 是 Tier Mobility 的一项计划，旨在零售商店中放置充电站，以激励骑行者更换滑板车电池。形象积分:等级

高盛(Goldman-Sachs)支持的资金也将使 Tier 能够扩大其 Tier 能源网络，这是一个在其覆盖区域内的零售店中放置电池充电站的风险投资。该能源网络将为骑行者提供一个激励结构，让他们在骑行结束时花一分钟更换滑板车的电池并获得免费积分，同时商店可以享受额外的步行交通。

高盛(Goldman Sachs)董事总经理本佩恩(Ben Payne)在一份声明中表示:“即使在全球疫情的背景下，Tier 也建立了盈利单位经济和资产寿命的良好记录。”。“我们很高兴能够帮助这位欧洲领导者将可持续交通推广到世界各地的更多人。”
