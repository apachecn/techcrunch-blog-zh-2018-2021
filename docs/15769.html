谷歌的Gradient Ventures因Vault Platform的不当行为报告SaaS而获得820万美元的首轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/06/googles-gradient-ventures-leads-8-2m-series-a-for-vault-platforms-misconduct-reporting-saas/

修正工作场所不当行为报告是一项任务，该任务得到了总部位于伦敦的 Vault Platform 的支持，该平台得到了谷歌专注于人工智能的基金Gradient Ventures的支持，Gradient Ventures是今天宣布的820万美元A轮投资的主要投资者。

参与这轮融资的其他投资者包括Illuminate Financial，以及包括Kindred Capital和Angular Ventures在内的现有投资者。其420万美元的种子轮于2019年关闭。

Vault向企业规模或大型/纵向扩展公司销售一套SaaS工具，以支持他们主动管理内部道德和诚信问题。除了供员工报告问题的工具之外，该平台还融入了数据和分析，因此可以支持客户更广泛的审计和合规性要求。

在接受TechCrunch采访时，联合创始人兼首席执行官Neta Meidav表示，除了完全参与升级遗留报告工具的总体任务，如向员工提供热线，以尝试揭露与行为相关的工作场所风险(无论是欺凌和骚扰；种族主义和性别歧视；或者贿赂、腐败和欺诈)，正如你所料，Gradient Ventures对应用人工智能进一步增强Vault位于SaaS的报告工具的潜力很感兴趣。

其当前平台的一个功能是“GoTogether”，包括一个托管系统，允许用户向相关内部机构提交不当行为报告，但前提是他们不是第一个或唯一一个报告同一个人的人——这一想法有助于鼓励员工(或外部人士，如果允许公开报告的话)报告他们可能会因各种原因而犹豫的问题。

Vault现在希望扩展该功能的功能，以便可以通过使用自然语言处理来帮助发现所报告活动类型中的模式和潜在联系，从而主动发现可能不仅仅与特定个人相关，甚至可能影响整个团队或部门的问题行为。

“我们今天的算法匹配了被指控的犯罪者的身份。然而，人们可能报道的许多事件并不与某个特定的人相关，它们可能更具描述性，”Meidav解释道。“例如，如果您的部门在会计方面遇到一些违规行为，并且您怀疑存在某种腐败或欺诈活动。”

“如果你想一想近年来发生的最大的[工作场所不当行为]灾难和危机——大众汽车的柴油门事件，波音公司发生的事情——所有这些案件的共同点是存在某种严重的道德违规或失败，这是由组织内偏远地区的几个人观察到的。这些点没有联系，”她继续说道。“因此，我们目前正在建立和增加的能力——建立在我们已经拥有的GoTogether的基础上——是连接这些重复事件的能力，能够连接、理解和阅读人类的输入。当重复事件发生时，将这些点联系起来——提醒公司董事会，他们需要去调查某个“热口袋”。

“这将使公司免于巨大的风险和成本，并从本质上防止巨大的损失。不仅是经济上的，还有名誉上的，有时甚至是人命的损失……这就是我们要去的地方，也是我们的目标。”

有一个问题是，如果你不能为一个想法申请专利，那么defensible Vault的GoTogether功能有多容易被复制。因此，人工智能烘焙可能是一种增加复杂性的方式，以试图保持竞争优势。

“后端有一些非常复杂、独特的技术，因此我们将继续在这方面投资。当我们问及防御能力时，Meidav说:“Gradient的投资和我们从Google now获得的具体信息只会增加我们业务的这一方面。”。

Gradient Ventures创始人兼管理合伙人安娜·帕特森在一份声明中对这笔资金进行了评论，他补充说:“ Vault 以创新和及时的解决方案处理了一个重要的领域。 Vault 的应用为组织提供了一种数据驱动的方法来应对职业欺诈、贿赂或腐败事件、安全故障和不当行为等挑战。鉴于他们令人印象深刻的团队、技术和客户吸引力，他们准备改善现代工作场所。"

这家总部位于伦敦的初创公司成立于2018年，尽管它最热衷于谈论破坏传统热线系统，这种系统只能为不当行为报告提供线性和被动的渠道，但还有许多其他初创公司在同一领域发挥作用。这方面的例子包括总部位于洛杉矶的AllVoices、T2、YC支持的Whispli、T4的Hootsworth和点点T7等等。

随着对工作场所报告的监管要求不断提高，竞争似乎会继续加剧。

即将出台的欧盟举报人保护指令是Vault预计将增加对更智能的合规解决方案(又名“信任技术”)的需求的一项监管措施，因为它试图将其标记为“信任技术”，因为它将要求员工超过250人的公司在2021年12月底之前建立举报解决方案，鼓励欧洲企业寻找工具来帮助降低与不当行为相关的风险。

她还建议，平台解决方案可以帮助弥合可能需要参与解决投诉的不同内部团队之间的差距，并通过提供与原举报人匿名聊天的能力来帮助加快内部调查。

Meidav还指出了美国监管机构对工作场所不当行为报告的日益关注——注意到SEC最近向外部举报者发放了一些巨额奖励，例如今年早些时候向一名举报者支付了2800万美元(与松下航空电子顾问腐败案有关)。

她还认为，越来越多的公司上市(如通过特殊目的收购公司趋势，在“空白支票”IPO之前，监管审查将有所减少)普遍提高了报告要求——这意味着，更多的公司将需要建立一个由第三方运营的系统，允许匿名和非匿名报告。(我们只能推测，相比选择更传统、更严格的上市路线的公司，在特殊目的收购公司上市的公司是否更需要不当行为报告服务。)

“就在几年前，我不得不说服投资者，这个类别确实是一个类别——快进到2021年，祝贺你！我们这里有市场。这是一个不断增长的类别，在这个领域存在竞争，”Meidav说。

“Vault的真正与众不同之处在于，我们不仅仅专注于将旧的传统流程数字化。我们专注于利用技术真正让更多的不当行为在内部浮出水面，并让员工以前所未有的方式畅所欲言。GoTogether以及我们在公司运营方面所做的事情(如协作)都是独一无二的。”

她举了一个例子，说明一位石油和天然气行业的客户如何配置该平台，以利用Vault应用程序中的匿名聊天功能，从而为员工提供一条与公司领导层的安全直线。

“他们正在利用匿名聊天，这款应用可以让人们直接联系到领导层，”她说。“太不可思议了。这是利用这一工具的一种进步的、前瞻性的方式。”

Vault Platform的工具套件包括一个员工应用程序和一个用于合规、人力资源、风险和法律团队的解决方案中心。图片来源:跳马平台

Meidav说，Vault现阶段有大约30个客户，分布在美国和欧盟——它的核心关注区域。

虽然它的平台面向企业，但其早期的客户群包括相当数量的纵向扩展——名单上有一些熟悉的名字，如柠檬水、Airbnb、卡瓦克、G2和OVO能源。

梅达夫表示，随着一家初创公司迅速转向扩大员工规模，公司文化可能会承受巨大压力，因此向上扩展可能是这类产品的天然客户。

“他们是早期采用者，而且他们对这类(工作场所)丑闻等事件非常敏感，因为这可能会对他们产生巨大影响……还有一个事实是，当一家公司经历超增长(阶段)时——通常你会看到科技公司的超增长比任何其他类型的行业都多——超增长是在你作为管理层和领导层真的非常重要的时候捍卫你的文化，”她建议道。

“因为它变化非常、非常快，这些变化可能会导致各种各样的事情——领导在它上面真的很重要。因此，当一家公司经历高速增长时，这是他们整合Vault等工具的绝佳时机。事实上，每一家考虑在未来几个月或几年进行IPO的公司都会很好地部署Vault这样的工具。”

在这一系列交易结束后，扩大Vault自己的团队也是可能的，因为它正在为自己业务的下一阶段增长而努力。不过，据推测，它并不缺少不当行为报告解决方案。
