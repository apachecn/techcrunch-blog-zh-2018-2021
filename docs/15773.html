99 minutos，墨西哥的最后一英里交付初创公司，筹集了 4000 万美元的 B 系列 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/07/99-minutos-mexicos-last-mile-delivery-startup-raises-a-40m-series-b/amp/

图片来源: 99 分钟

2014 年，Alexis Patjane 和几个朋友在墨西哥城当地的一个水烟酒吧，酒吧的烟草用完了。他们认为也许他们可以在网上购买一些，并让它实时送到酒吧，但事实证明这项服务并不存在。

当时，Patjane 经营着一家快餐车制造公司，该公司生产了墨西哥 80%的快餐车，所以他有在墨西哥做生意的经验。

几个星期后，为了解决他在水烟吧遇到的即时送货问题，Patjane 推出了 99 minutos 网站，销售产品并在 99 分钟内送货，因此得名。

今天，99 minutos 宣布从 Prosus 和 Kaszek Ventures 获得 4000 万美元的 B 轮投资，计划用于发展其在拉丁美洲的业务。

该公司目前在墨西哥、智利、哥伦比亚和秘鲁的 40 个主要市场开展业务，并提供四种服务:99 分钟内送达、当天送达、次日送达和无二氧化碳送达。

以快递起家的电商，迅速成为其他电商的最后一公里配送服务。

“我们开始构建 API 连接和插件，任何电子商务都可以将我们的送货服务添加到他们的业务中，”Patjane 告诉 TechCrunch。

99 minutos 通过向客户收取统一的送货费，然后向司机提供统一的费率来赚钱，但今天，每条路线的流量都很大，这变得非常有利可图。

“我们每条路线运送大约 60-80 个包裹，”Patjane 说，从消费者的角度来看，送货应用程序的工作方式类似于 Waze。“你可以暂停送货，你可以改变地址。你可以说，“哦，我不在家，我在街角的星巴克，你能把它放在那里吗？”他补充道。

Patjane 说，最初，该公司只在墨西哥城内提供送货服务，但它很快发展到在城市之间提供服务，现在在墨西哥的 21 个城市之间运营。

“电子商务在拉丁美洲发展迅速，但仍处于早期阶段。该公司在一份声明中说:“拉丁美洲的电子商务渗透率为 6%，而中国正达到 30%，美国为 20%。”

“当我们听到大型电子商务公司说 99 minutos 是‘他们最可靠的合作伙伴’，他们是‘最有潜力的提供商’，这告诉我们，这个团队执行得非常好，正在破坏拉丁美洲的电子商务交付，”Prosus Ventures 的美洲投资主管 Banafsheh Fathieh 说。

部分资金也将用于加速城市间的运输。“我们将在同一天从一个城市运送到另一个城市，并将使用小型飞机连接各个城市，”帕特珍说。
