以下是苹果在 WWDC 2021 主题演讲上宣布的一切

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/07/heres-everything-apple-announced-at-the-wwdc-2021-keynote/

又到时间了！今天早上，苹果公司以每年相同的方式开始了一年一度的全球开发者大会:一场主题演讲挤满了各种设备的新产品。

没有时间收听直播博客或观看视频流？我们明白了。这就是为什么我们将所有重大新闻浓缩成以下要点。闲暇时略读！

iOS 15

Craig Federighi 从 iOS 最新重大更新的细节开始，指出这次发布侧重于四点:保持联系，寻找焦点，利用智能和探索世界。他指出，iOS 15 将于 7 月进入公共测试版，正式版将于“今年秋天”发布。

  • 空间音频:使用兼容的音频设置时，空间音频会让您的朋友的声音听起来更像是从他们的屏幕位置发出的。
  • 声音隔离:更好的(可切换的)声音隔离，适合在嘈杂的环境中进行 FaceTiming。
  • 肖像模式:相机应用程序的肖像模式即将进入 face time——换句话说，漂亮的背景模糊。
  • FaceTime 链接:你现在可以生成可共享的 FaceTime 链接，让其他人(包括桌面和安卓用户)通过浏览器！)加入 FaceTime 通话。
  • SharePlay: 共享音频/视频流让你可以听音乐、看视频(最初选择视频合作伙伴，包括 Disney+、Hulu、HBO Max 和抖音)，还可以和朋友同步控制应用。你可以将共享的视频推送到你的 AppleTV 上，让它在设备和家庭之间保持同步。

“聚焦”模式

实时文本

相机现在可以识别照片(手写或打印)中的文本，并使其可选择、可搜索、可复制/可粘贴等。它还可以识别文本以外的东西，包括动物品种、地标等。

苹果钱包

苹果正在将家庭、办公室和酒店的钥匙整合到苹果钱包中，允许你用手机打开这些地方的兼容门。举例来说，你可以登记入住指定的酒店，一旦你的房间准备好了，你的房间钥匙就会显示在你的手机上。

他们还与美国运输安全管理局合作(首先在选定的州),将你的州驾照的加密副本(！)放在钱包里，接受安检。

苹果地图

图片来源:苹果

苹果地图正在进行注重细节的革新，增加了 3D 立体地图、3D 渲染地标、人行横道、自行车道等。

“地图”也从谷歌地图中获得了一些线索，包括一种模式，让你使用相机扫描附近的建筑物，以更精确地确定手机的方向，并帮助你确定你应该在步行开始时走哪个方向。

AirPods 更新

  • 对话模式:当你在嘈杂的环境中进行面对面的对话时，AirPods Pro 现在可以让你提高周围人的音量(并控制环境噪音的音量)。
  • 查找我的: AirPods 现在可以与“查找我的”应用程序配合使用，即使它们在箱子里，如果你把它们落在后面，它也会提醒你。
  • 空间音频将出现在 tvOS 和 M1 驱动的 MAC 电脑上。

iPadOS

  • iPadOS 将获得主屏幕小部件。它们与你在 iOS 上看到的非常相似，只是增加了一个更大的小工具选项，因为屏幕更大。
  • “应用程序库”的概念也正在从 iOS 走向 iPadOS。
  • 一个新的多任务菜单将(有望)简化你在 iPadOS 上处理多任务的方式，让系统远离目前需要的容易忘记的手势。
  • Notes 应用程序现在允许你在共享笔记中@提及某人，以引起他们对某项更改的注意，并在你回到共享文档时突出显示更改。与此同时，“快速笔记”可以让你在任何地方通过从屏幕右下角向上滑动来创建笔记。
  • 翻译: iPadOS 将获得一个全系统的文本翻译选项，以及一个实时翻译面对面对话的应用程序。
  • Swift Playgrounds 现在可以让你在 iPad 上开发应用，然后将它们提交到应用商店。
  • 7 月公测，全面发布《今秋》

安全更新

  • 邮件应用程序现在会自动隐藏你的 IP 地址/位置，不让那些营销人员喜欢塞进电子邮件的隐形跟踪像素看到。
  • “设置”中的“应用隐私报告”选项现在将总结任何给定应用访问你的位置、照片等的频率。希望能标记出任何意想不到的行为。
  • Siri 的语音识别现在将在设备上进行，将录音从苹果的服务器上移除，并允许在没有互联网连接的情况下使用 Siri 命令。
  • 您可以将恢复联系人添加到您的 iCloud 帐户，允许您选择朋友，如果您被锁定，他们可以帮助您重新进入 iCloud 帐户。您还可以选择“遗留”联系人，如果您去世，他们可以请求访问您的 iCloud 数据。

健康更新

  • 你的 iPhone 现在可以分析你的步态、步伐计时等。分析你的“行走稳定性”,如果你跌倒的风险随着时间的推移而增加的话，就有希望发出信号。
  • 健康应用程序中的“趋势”摘要将突出显示您的步数、心率等模式。
  • 新的“健康共享”选项将让您安全地与您关心的人共享健康数据，如您的父母或孩子。

watchOS

  • watchOS 现在将能够帮助跟踪你的呼吸频率(你每分钟呼吸多少次)，并注意到你一天中呼吸的不同(比如你睡觉时)。他们还在活动跟踪引擎中添加了新的锻炼模式，包括太极和普拉提。
  • 一个新的“肖像”表盘将使用你的肖像模式照片来创建一个整洁，层次分明的时钟布局。

  • 文本输入正在 watchOS 上进行微妙的改革，将“涂鸦”手写识别功能整合到了文本视图中。您可以使用顶部来放置文本光标，然后在文本上方绘制所需的字母/符号。
  • 7 月公测，全面发布《今秋》

Siri 适用于第三方设备

苹果表示，它将向第三方制造商及其设备开放 Siri，允许 Siri 从今年晚些时候开始在 ecobee 恒温器等东西上运行。

马科斯

  • macOS 的下一个主要版本将被称为 macOS Monterey 。
  • 所有上述 SharePlay 的东西——允许你通过 FaceTime 与朋友同步观看视频和听音乐——都将出现在 macOS 上。
  • "通用控制将让你使用一个键盘/鼠标设置来控制多台 MAC 和 iPads，当一个新的兼容设备放在附近时，它会自动识别。把一个 iPad 放在你的 MacBook 旁边，砰——它会自动展开，把两个设备变成一个大画布。
  • Mac 正在获得 AirPlay，允许您将 AirPlay 内容发送到 Mac 的显示器上，就像发送 AppleTV 一样。
  • iOS 的“快捷方式”自动化系统即将进入 macOS ，允许你快速拖放脚本来做你经常做的事情。(如果你已经在使用 macOS 的 automator 系统，它现在仍然会得到支持。)
  • Safari 正在进行 UI 大修；标签被做得越来越小，他们引入了一个“标签组”系统来保持事物的有序。标签组将在 iCloud 设备之间自动同步。
  • 7 月公测，全面发布《今秋》

开发的东西

虽然它被称为全球开发者大会，但苹果倾向于将主题演讲主要集中在面向消费者的内容上，并将大多数技术内容留给本周的许多分组会议。不过，他们确实提到了一些开发人员的亮点，包括:

  • 与 SharePlay、Focus 等挂钩的新 API。
  • iOS 正在获得一个内置的 3D 对象扫描系统，开发人员可以将其绑定到他们的应用程序中，允许他们集成 3D 扫描风格的功能，而不必重新发明轮子。
  • 开发者现在可以为同一个应用建立多个应用商店产品页面——用不同的图标、截图等。—看哪一个最能引起共鸣。
  • test flight——在苹果手机/平板电脑/手表操作系统上私下分发预发布/测试版应用的服务——即将登陆 macOS。
