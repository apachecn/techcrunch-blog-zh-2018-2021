Terra 筹集 280 万美元为健身数据建立格子 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/09/terra-raises-2-8m-to-build-the-plaid-for-fitness-data/

今天早上 Terra ，一个最近从 Y Combinator 毕业的为健身和健康数据开发 API，宣布它已经完成了一轮 280 万美元的投资。投资包括来自通用催化剂、三星 Next 和 NEXT VENTURES 的资金。

该公司打算用新资本招聘员工，从今天的四人团队——联合创始人和另外两名工程师——增加到本月的七人。Terra 预计在本季度内最多可招聘 10 名员工。

考虑到 Terra 正在建造的范围，更多的人是有意义的。该公司的 API 允许开发人员挂钩软件和硬件生成的健身和健康数据。按照操作顺序，Terra 首席执行官兼联合创始人基里亚科斯·埃勒夫瑟瑞乌告诉 TechCrunch，他的公司首先瞄准了提供 API 的公司的集成点，然后是 SDK，后来是支持蓝牙的健康可穿戴设备。这有很多可能的联系，所以让更多的人来帮助公司编写集成点是有意义的。

当然，提供健身和健康数据之间的联系是很好的，但只有使用现在解锁的数据建立的技术才是酷的。据 Eleftheriou 称，Terra 的产品已经吸引了大约 100 名开发人员，这家初创公司应该开始形成一个什么是可能的想法。在接受 TechCrunch 采访时，Eleftheriou 提出了一个将音乐选择与心率联系起来的好主意。因此，当你的心率上升时——比如说，由于锻炼——你的音乐播放器可以从你的可穿戴设备中读取数据，并调整它的播放。

或者相反，我补充说。说你和我一样，是个神经质的人。我可能想让 Spotify 更深入地播放肖邦的作品，因为我在工作时间的心跳越快。

链接到健身健康数据源有多难？按照 Eleftheriou 的说法，健身技术世界比我们预期的更容易连接。首席执行官说，Polar 公司生产健身追踪器等可穿戴设备，已经有了 Terra 可以插入的 API。

TechCrunch 对第三方在允许 Terra 插入各种健康和健身技术硬件和软件方面的犹豫感到好奇。但 Eleftheriou 表示，这对他的初创公司来说并不是一个大问题，一些公司——比如那些销售可穿戴设备的公司——有一种天然的动力，让他们的硬件尽可能有用。

称 Terra 为健康数据的格子不仅仅是一个有用的比较来帮助我们理解它在构建什么；该公司正在追求一种消费定价模式，这意味着它的经济规模将类似于 Plaid，或者 Twilio 自己的。

但比 Terra 如何看待健康和健身数据融入世界更重要的是，看看开发者实际上用它的服务构建了什么。公司现在有了几百万美元的资金，现在必须证明它的原料药有实质性的市场需求。如果是这样，我们可能会在未来看到一个更加互联、集成的健康和健身数据环境。我就是为此而来的。
