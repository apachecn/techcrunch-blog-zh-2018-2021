网络安全培训平台 Immersive Labs 完成由 Insight Partners 领投的 7500 万美元 C 轮融资 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/13/cyber-security-training-platform-immersive-labs-closes-75m-series-c-led-by-insight-partners/

沉浸式实验室(Immersive Labs)是一个通过“游戏化”方式使用真实、最新的威胁情报向企业员工传授网络安全技能的平台，该平台已完成了由新投资者 Insight Partners 以及 Menlo Ventures、花旗风险投资公司和现有投资者高盛资产管理公司(Goldman Sachs Asset Management)牵头的 7500 万美元的 C 轮融资。

这笔投资将用于扩大 Immersive 在美国的产品规模，并利用后疫情时代许多人远程工作引发的对网络威胁的新一轮兴趣。Immersive Labs 成立于 2017 年，目前拥有 200 名员工，在英国布里斯托尔和美国波士顿设有联合运营总部。它计划在未来两年内将员工人数增加到 600 人以上，并在 APAC 和欧洲的新地区建立运营。Immersive 的“网络劳动力优化”平台声称提供董事会级别的指标和基准，以衡量组织内部的技能如何应对。

Immersive 现在已经筹集了总计 1.23 亿美元的风险资金，并将汇丰银行、沃达丰和英国国民医疗服务体系作为客户。该公司表示，其年增长率“超过 100%”。

Immersive Labs 的首席执行官兼创始人詹姆斯·哈德利(James Hadley)表示:“随着网络风险成为越来越多业务职能部门的问题，网络安全知识和技能不应再是隐藏在后台办公室的少数技术人员的专利。从软件开发团队到首席执行官，每个人现在都需要在解决一个普遍的公司问题中发挥自己的作用。这需要在更广泛的人群中发掘和证明技能。”

Insight Partners 的董事总经理 Ryan Hinkle 表示:“随着过去几年全球客户和收入的显著增长，Immersive Labs 在快速发展的网络技能领域建立了强大的地位。凭借有影响力的领导力、不断增长的市场中的创新产品和强大的用户参与度，该公司有能力继续引领网络就绪市场。”

哈德利在一次采访中对我说:“我们选择 Insight Partners 是因为他们在企业 B2B 领域拥有真正的实力，这是我们向首席信息官和首席执行官销售的领域……我们希望成为下一个成功的英国网络安全公司 Darktrace。”

这种比较可能没那么离奇。沉浸式实验室来自 CYLON 网络加速器，类似于 Darktrace，与 Darktrace 有相同的投资者，但事实上它的 C 轮投资吸引了 7500 万美元，而 Darktrace 直到 d 轮投资才达到这个水平。Darktrace 现在已经在伦敦 IPO，融资 17 亿美元。

前 GCHQ 安全研究员和培训师哈德利在亲自领导网络培训时提出了网络技能平台的想法。我问他，为什么他认为 Immersive 的平台能够产生“飞轮效应”。

“人们总是谈论所有的网络威胁变得越来越严重，但现在确实如此，而且是在公共领域。我们坚信网络安全不再是地下室极客的责任。其实是全业务的。现在海啸来了。网络犯罪将在今明两年销声匿迹，因为公司正在支付赎金。因此，我们正在运用分析方法来衡量危机中的决策。无论是首席信息官还是垂直行业，每家公司都对此产生了很好的共鸣，”他告诉我。

沉浸式实验室也是慈善机构 TechVets 的主要支持者，该机构免费为英国退伍军人提供网络新职业培训。
