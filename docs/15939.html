2019年发布时，Powerbeats Pro表现突出。两年多后，它们仍然是市场上更全面的无线耳塞之一。当然，有些事情我会改变。即使在2019年，那个收费案也大得离谱。2021年，原案更加荒谬。当然，消除噪音在中端市场已经变得近乎标准化。

经过数周的谣言和泄露(包括世界上最著名的运动员之一的耳朵非常公开的客串)，Beats在这个领域的最新表现终于得到了官方认可。认识一下Beats Studio Pro。该公司很快会告诉你，它们不是Powerbeats Pro的替代品。这些仍在继续(这并不是说他们不会得到自己的升级)。

Beats可能是苹果公司所有，但在大多数方面，该品牌的运作一如既往。毕竟，在苹果染指之前，这是一个非常成功的品牌。所以该公司选择不修复明显没有损坏的部分。虽然这两个阵营显然共享技术(例如Powerbeats上的H1芯片)，但它在自有品牌音频产品(AirPods等)和beats系列之间保持着一条线。Beats从未真正出现在苹果的活动中是有原因的，尽管在接下来的一周有一个重大的宣布。

图片来源:布莱恩·加热器

与AirPods相比，Beats的线条可能有点复杂。实际上，新的Studio Beats是该公司的一款全无线耳塞线，借用了其优质耳挂线的名称。但新的buds实际上比Powerbeats Pro更紧凑，无论是从外壳还是buds本身来看。同样值得注意的——坦率地说有点令人惊讶——是价格。

Studio Buds的售价为150美元，比两年前的Powerbeats Pro便宜不少，后者目前在网上的售价在160美元至200美元之间。请记住，这低于250美元的发布价格。这也比AirPods便宜50美元，比Galaxy Buds便宜20美元。对于你在这里得到的东西来说，这是一个不错的价格——尽管可能我的标准有所改变，刚刚结束了对280美元的索尼WF-1000XM4 的评论。

当然，那些索尼公司是独一无二的。让相关各方把它们和其他中音耳机坑一坑，要公平得多。按照这个标准，他们表现得相当好。这里最大的附加功能是主动降噪——请记住，当Powerbeats Pro宣布时，它还远远不是标准的。然而，这些天来，在这个价格范围内，它感觉像是一个明显的忽略(谷歌，我在看着你)。

图片来源:布莱恩·加热器

另一个有趣的顶级功能是iOS和Android的快速配对，这使得Studio Beats成为首批走这条线的产品之一。有趣的是，它来自苹果的产品，但同样，该公司似乎在这方面至少有一点自由。这是一件小事——毕竟，许多人只会使用一次iOS/Android一键配对，但让尽可能多的潜在客户能够接触到该产品是很有意义的。

我喜欢花蕾的新流线型设计。如上所述，新的外壳只是Powerbeats的一小部分。不过，Studio Buds的电池续航时间与声明的一样，耳机续航时间为8小时，如果算上外壳，总共续航时间为24小时。这是一种健康的生活方式，如今正迅速成为标准。底部有一个USB-C端口(远离苹果专用的Lightning)，充电五分钟可以播放一小时。

图片来源:布莱恩·加热器

外壳比AirPods Pro更宽，更厚，但仍然很容易放入口袋。它有一点廉价的塑料感，但哑光处理是一个很好的触摸。品牌是标准的Beats级别的loud，黑色背景上有一个大而粗的白色“b”。花蕾也有标志，根据位置不同，标志可以是“9”或“6”。盖子上有一个搭扣，花蕾上的磁铁很好地扣在了合适的位置上——不过，和Powerbeats一样，要把它们放到合适的位置上可能需要一点技巧。

花蕾也相当紧凑。耳钩不见了。老实说，这是一个大杂烩。我不认为我会喜欢Powerbeats Pros耳挂，但作为一个经历过许多不同耳塞设计的耳朵疼痛的人，我发现它们是最舒适的选择之一，可以将负荷转移到耳朵顶部。

Studio Buds相当舒适，我可以穿着它们工作(IPX4等级FTW)，尽管我有时在保持它们的位置上有些困难。这在Powerbeats上从来都不是问题。如果你真的不想让它们移动，我建议施加一点压力，把它们真正拧到位。

Beats带回来的设计选择中，我非常欣赏的一个就是实体按键。Powerbeats拥有它们，它们现在回到了录音室的尽头。它有一个很好的小点击，我更喜欢纯粹的触摸按钮。单击将播放/暂停，长时间单击将打开和关闭ANC。

图片来源:布莱恩·加热器

当然，非国大是一个很好的补充。它在环境噪音方面做得不错，但不能真正接触到你在高端系统上发现的东西。音质在过去几年里也有了很大进步。Beats改进了一对8.2毫米驱动器，以其价格提供了坚实的声音。这些人不是坐在一起欣赏古典奏鸣曲或实验爵士乐唱片的人；然而，它们是坚固的，听音乐或播客的耳机。

buds有很多值得喜欢的地方，毫无疑问，它们在2021年比Powerbeats Pro好得多，即使它们不像它们的前辈在发布时那样具有开创性。

新的Beats Studio Buds今天开始接受预订，并于6月24日开始发货。
