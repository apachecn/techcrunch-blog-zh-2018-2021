Commsor收购Meetsy，为所有人构建社区工具——TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/14/commsor-buys-meetsy-to-build-community-tools-for-all/amp/

图片来源: SunnyGraph / Getty Images

Mac Reddin,Commsor的联合创始人兼首席执行官在午餐俱乐部(Lunchclub)上会见了他公司1600万美元首轮融资的第一位专业投资者，这是一个1:1视频介绍市场。Commsor将自己视为社区管理者的操作系统，然后亲身体验了在一个迷人的在线环境中匹配相似行业的人的好处。

因此，几个月后，Commsor将一家专注于社区内私人网络的公司加入其技术堆栈是非常合适的。Commsor已经收购了Meetsy，这是一家帮助社区内的人们促进彼此之间创造性和自发性对话的初创公司。Meetsy是一家完全自主经营的企业，由布拉德利·特拉梅尔创建，并以未公开的价格出售。

Commsor在几个月前推出，旨在帮助社区经理证明他们没有浪费预算，他们的工作与真实、有价值的结果相关联。Commsor可以得出这样的见解，“这里有三名Google工程师正在使用您的[在此插入社区平台名称]。因此，也许是时候去接近谷歌，问问他们是否想要一份企业合同了。”

通过这次收购，科姆索的野心似乎扩大了一些。为了让一个社区在外表上对一个企业有价值，它必须是内在健康的。成员们必须相互联系，反复讨论，并有机会讨论将他们放在首位的共同兴趣。Meetsy在公司内部工作，为员工和校友项目牵线搭桥，并与希望其投资组合公司相互联系的风险投资家合作。

“如果Commsor帮助你证明社区的商业价值，Meetsy帮助你建立一个更好的社区，这两者结合起来会产生更大的影响。”

尽管美国不断增长的疫苗接种率可能会打开面对面互动的水龙头，但许多人认为互动将继续存在虚拟成分，特别是与分布式团队的互动。毫无疑问，随着远程工作越来越深入人心，超级连接器应用的浪潮越来越大。前面提到的Lunchclub 上一轮估值为1亿美元， Workvivo融资1600万美元，Gatheround上个月获得了350万美元的种子。

“我见过的每种虚拟活动工具都内置了某种网络，但这通常是我认为的‘愚蠢的网络’，”雷丁说。他指的是常见的“快速约会”策略，即随机安排几分钟，然后拖着脚步重复。

他说，Meetsy已经完善了一个强大的匹配系统，因此社区经理可以进来，根据人们自选的属性来定义他们。然后，它可以根据他们希望匹配如何发生的准则来填写匹配规则。例如，Meetsy可以确保金融科技风投只与金融科技初创公司会面，以激发未来的投资，或者员工只能在没有当前团队以外的员工的情况下会面，以促进交叉合作。

“这样就不只是把两块石头砸在一起，然后希望火花四溅，”雷丁说。

这两家公司将继续在现有品牌下存在，但Meetsy用户可能会在Commsor上获得折扣，或者从未来的捆绑功能中受益。Meetsy一直是免费使用的，但未来会尝试新的付费功能，如白标功能，或要求公司在达到一定规模的社区成员后付费的层级系统。

Meetsy的创始人Tramer表示，将工具与Commsor相结合将有助于社区管理者“大规模培养有意义的联系，并衡量这些联系的影响。”

“在五年内，我认为‘策划介绍’是与活动和论坛一起的社区参与的核心支柱，”他说。
