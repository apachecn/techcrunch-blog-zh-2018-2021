英国金融科技公司 10x 筹资 1.87 亿美元为老银行打造新服务 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/15/10x-a-uk-fintech-raises-187m-to-build-new-services-for-old-banks/

随着所谓的新银行(neobanks)以其更现代的方式提供银行和其他金融服务，继续在市场上获得更多关注，一家正在开发技术以帮助现有参与者更好地竞争的初创公司宣布了一轮大规模融资。

总部位于伦敦的金融科技公司 10x Future Technologies 已经筹集了 1.87 亿美元，该公司帮助大型老牌银行构建下一代服务和工具，以帮助他们的旧服务更有效地工作。我们从接近该公司的消息人士处了解到，10x 本轮的估值在 7 亿美元左右。

(筹集的金额和估值也与天空新闻的数字大致相符，天空新闻本月早些时候报道称，10x 正在筹集新的资金。)

10x 将利用这笔资金向北美等新地区扩张，并继续为其旗舰平台开发更多技术。 SuperCore ，该平台被称为 SuperCore，是一个从零开始构建的一体化系统，用于运行广泛的银行服务，如支付、核心银行业务、抵押贷款、分析、安全和营销，10x 的银行客户可以通过 API 将其集成到他们现有的技术中，或者 10x 可以用来从头开始为这些客户构建新的服务。

这一轮 C 轮融资充满了重量级人物，证明了 10x 在市场上五年来获得的信誉。

由贝莱德和加拿大养老金计划投资委员会(CPP Investments)共同牵头，还包括现有投资者摩根大通、Nationwide、平安和澳洲西太平洋银行。

后四位包括战略支持者:Antony Jenkins，10x 的创始人兼首席执行官，他自己曾在大银行工作(他的最后一个角色是巴克莱银行的首席执行官，尽管他在阴影下离开，但他的声望和记录可能是该公司影响力的原因)，告诉我们 10x 目前正在为西太平洋银行和全国范围内建立服务。

詹金斯补充说，10x 还有另外两家尚未披露的银行客户，这将很快导致更多的客户，因为该行业目前正处于“过渡时刻”。他说，10x 的一些项目已经上线，“平台上的交易量越来越大”。其他的还没有发射。

10x 瞄准的机会很大，但也难以捉摸。

Neobanks 和其他新一代金融科技提供商正在慢慢蚕食现有银行在个人和企业银行业务上的优势。他们通常不是通过成为完全成熟的银行，而是通过其他金融科技公司的 API 将传统和更现代的银行服务组合在一起；为客户提供个性化服务的机器学习算法；和现代化的界面，让整个体验比传统银行更人性化。

现有银行希望用自己的竞争产品与这些新贵竞争，但在许多情况下，它们做不到:它们的基础设施太旧，公司文化往往更古老。

这就是 10x 的用武之地，它提供工具和建议来帮助他们启动和运行新的服务。

Jenkins 指出，目前，10x 看到的许多业务都是银行利用全新的服务，而不是建立服务来取代他们已经提供的服务。一个恰当的类比是，这有点像在一栋非常古老的房子上进行现代化扩建，而不是从头开始对整个老房子进行改造和现代化。

但是，我们现在似乎已经进入了 10x 作为一家企业的第五个年头，开始看到银行愿意探索如何将其核心数据迁移到更现代化的系统，以使其在更广泛的新服务中更具可扩展性和可用性的初步迹象，10x 相信它也可以成为后端转型的合作伙伴。

“遗留系统是银行的问题所在，因为它们都是围绕产品而不是客户来设计的，”他说但我们认为，该行业现在已经准备好考虑迁移过程。”他补充说，该公司目前还没有任何这类项目，但预计将在未来 12 个月内完成。

即使其他金融科技初创公司，如 FintechOS ，也在提供旨在帮助现有银行更加现代化的服务，这种预期也为投资者带来了机会。

贝莱德创新资本(Innovation Capital)负责人威廉阿贝卡西斯(William Abecassis)在一份声明中表示:“10x 的战略和雄心给我们留下了深刻印象，它将在技术创新、消费者预期和监管改革的推动下，在金融服务领域发生的变革中发挥关键作用。”。“我们很高兴能在业务扩展到新市场时投资该业务。”

CPP Investments 的医学博士兼主题投资主管 Leon Pedersen 补充道:“10x 非常适合改变大银行的构建方式和为客户提供服务的方式。10x 为 CPP Investments 这样的长期投资者提供了一个有吸引力的机会，因为我们相信他们将从金融机构投资数字化计划和更新核心技术基础设施的结构性增长趋势中受益，使银行能够以比使用传统平台更快的速度推出新产品。”
