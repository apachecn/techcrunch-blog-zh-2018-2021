Android 宣布六项新功能，强调安全性和可访问性

Android 今天分享了关于将于今年夏天推出的六项功能的信息。其中一些只是生活质量的升级，比如让短信更容易找到它们，或者根据你输入的内容获得上下文相关的表情符号厨房建议。但是这次更新的其他方面强调了安全性、安全性和可访问性。

去年夏天，谷歌在安卓系统上增加了一个功能，基本上就是用你的手机作为地震仪来创建“世界上最大的地震探测网络”该系统是免费的，自从在加州测试后，它也在新西兰和希腊推出。现在，谷歌将在土耳其、菲律宾、哈萨克斯坦、吉尔吉斯斯坦、塔吉克斯坦、土库曼斯坦和乌兹别克斯坦推出这一功能。该公司表示，他们今年将继续扩展该功能，优先考虑地震风险最高的国家。

图片来源:谷歌

谷歌还在扩展去年发布的另一项功能，该功能使谷歌助手兼容安卓应用。在最初的更新中，支持的应用程序包括 Spotify、Snapchat、Twitter、沃尔玛、Discord、Etsy、MyFitnessPal、Mint、Nike Adapt、Nike Run Club、易贝、Kroger、Postmates 和 Wayfair。今天的更新提到了易贝、雅虎金融、Strava 和 Capital One。这些功能堪比苹果对 Siri 与 iOS 应用程序的支持，包括打开应用程序、执行任务和录制自定义命令的能力。

图片来源:谷歌

谈到可访问性，谷歌正在提升其凝视检测功能，目前处于测试阶段。凝视检测允许人们要求语音访问仅在他们看着屏幕时做出响应，允许人们在与朋友交谈和使用手机之间自然移动。现在，语音访问还将拥有增强的密码输入功能——当它检测到密码字段时，它将允许你通过说出“大写 P”或“美元符号”来输入字母、数字和符号，例如，让用户更容易、更快速地输入这些敏感信息。10 月，谷歌助手在凝视驱动的无障碍设备上可用，同月，谷歌研究人员推出了一个演示，使使用手语的 T2 人可以被识别为视频通话中的“主动说话者”。苹果还没有可广泛使用的类似凝视检测功能，尽管他们在 2017 年收购了眼球追踪公司 SensoMotoric Instruments(SMI)。因此，希望类似的可访问性功能将在苹果公司工作，特别是当谷歌继续建立他们的。

今天的 Android 更新也让 Android Auto 用户定制更多的体验。现在，你可以从手机上设置启动屏幕，手动设置黑暗模式，并通过 A-Z 滚动条和“返回顶部”按钮更容易地浏览媒体应用程序上的内容。像 WhatsApp 和 Messages 这样的消息应用程序现在可以在启动屏幕上兼容——小心驾驶，不要分心——电动汽车充电、停车和导航应用程序现在也可以使用。
