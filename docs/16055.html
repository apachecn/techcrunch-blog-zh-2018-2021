原文：https://web.archive.org/web/https://techcrunch.com/2021/06/16/denmarks-templafy-raises-60m-for-its-b2b-saas-platform-that-does-business-document-creation/

丹麦的 Templafy 为其 B2B SaaS 平台融资 6000 万美元，该平台提供商业文档制作

诞生于丹麦的 B2B SaaS 平台 Templafy 已经获得了由 Blue Cloud Ventures 领投的 6000 万美元的 D 轮融资。之前所有的投资人也都参与了，包括 Insight Partners、种子资本、黎明资本和 Damgaard 公司。Templafy 现在已经筹集了 1.25 亿美元的资金。

某种程度上，Templafy 与 PandaDoc 是竞争关系。然而，创始人 Christian Lund 告诉我:“我们构建的平台非常关注企业，所以它不知道用例。它实际上是帮助员工制作他们需要的几乎任何类型的业务文档或内容，允许他们从他们工作的任何应用程序开始。它可能是 Office 或 Google，但也可能是 Salesforce 或 Teams 或 Slack。其他人则非常垂直地关注特定的用例，例如围绕销售。我们横向关注并帮助大型企业的一系列使用案例。”

Blue Cloud Ventures 的管理合伙人米尔·阿里夫(Mir Arif)表示:“Templafy 正在解决一个非常普遍、但经常被组织忽视的问题:内容脱节。虽然这个术语可能是新的，但问题本身并不新鲜。当公司内容没有集成到员工工作的应用程序中时，组织会遇到内容脱节的情况，这可能会导致一些破坏性的问题，包括失去合规性、降低效率，并最终对业务绩效产生负面影响。为所有企业解决不相关内容的雄心，加上成熟的市场、强大的运营团队和强大的用户友好型平台，使 Templafy 成为一个模范合作伙伴。”

Templafy 的C 轮 2500 万美元是 14 个月前。
