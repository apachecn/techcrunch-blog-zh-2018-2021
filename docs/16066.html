对于当代艺术家来说，将作品以不可替代的象征物(NFT)的形式附在区块链上，似乎是一种安全且可验证的在线出售艺术品的方式。

在某些方面，的确如此。区块链固有地记录所有交易的时间戳数据，并在分布式账本中永久显示所有权。对区块链交易的内部观察将提供关于 NFT 何时交易、谁参与了交易以及花费了多少的所有所需信息。

但 NFT 所有权的现实比人们想象的要复杂得多。作为一种新的加密资产类别，NFTs 似乎几乎不受当前监管系统的约束。但是和艺术结合起来，就有重叠的地方要考虑。理解当代 NFT 生态系统的法律陷阱是释放其潜力的第一步。

区块链上存在版权吗？

人们对 NFT 作为版权替代品的潜力寄予厚望，许多人认为它们本身就是版权。从表面上看，很容易理解这种困惑。

NFT 的购买者只拥有区块链上的一个唯一散列，其中有一个交易记录和一个到艺术品文件的超链接。

事实是，NFT 只是代表资产的令牌，与资产本身完全分离。因为每一个 NFT 都代表着一种独特的资产，所以一个单独的 NFT 不可能在保持与原作相同价值的情况下被复制。许多人将这种排他性的所有权形式等同于作品本身的所有权，但必须强调二者之间的区别。

这种误解更进一步。NFT 奖的各种可能性与符合版权条件的作品惊人地一致。尽管每个司法管辖区对“作品”的定义各不相同，但没有一个偏离本质太远。例如，在加拿大，版权保护除了表演、录音及其他相关作品之外，还延伸到文学、艺术、戏剧或音乐作品。创作者不需要申请这些保护——国家在作品创作时就给予他们固有的保护。

自然，NFT 代表的原创作品也受到这种保护。当艺术作品在 NFT 市场上被创作和拍卖时，版权的功能几乎与真人场景完全一样，版权由艺术家保留。但是，由于缺乏符合国际法的版权交易基础设施，在当前平台上交换 NFT 版权是不可能的。

因此，除非艺术家和买家之间达成了外部协议，否则 NFT 作品的版权仍然属于原艺术家。NFT 的购买者只拥有区块链上的一个唯一散列，其中有一个交易记录和一个到艺术品文件的超链接。

没有法律参数，欺诈是不可避免的

当考虑到潜在的盗窃和欺诈时，NFT 版权跟踪的问题变得更加棘手。为了添加到区块链，NFT 必须由上传者在一个称为“铸造”的过程中“签名”类似于画家在画作上的签名，这一特征旨在将 NFT 与其创作者联系起来。当造币者谎报身份时，事情可能会出错，这在许多 NFT 平台上并不罕见。

这个问题源于 NFT 市场缺乏强有力的法律框架。在某些平台上，你可以制作一条推文、一件艺术品，甚至是一张 Nyan Cat 的 gif，而无需成为实际的创作者。因此，许多艺术家报告说，他们的艺术作品在未经他们同意的情况下被盗走，并以 NFT 的形式出售，这在传统艺术市场上显然侵犯了版权。

这个问题在 NFT 的推特交流中尤其普遍。一个名为@tokenizedtweets 的 Twitter 机器人在今年早些时候掀起了一场造币狂潮，震惊了整个 Twitter 和 NFT 社区。它在未经作者同意甚至通知的情况下从病毒推特上创建 NFT 的政策引起了几位演员、艺术家和其他创作者的强烈抗议，激起了像威廉·夏特纳这样的名人的回应，他们表示担心“这些@tokenizedtweets 窃取内容，我上传的图片和我的推特都在我的版权之下，未经许可被标记和出售。”

盗窃和欺诈是缺乏强大法律基础设施的平台的自然结果。@tokenizedtweets 的行为很好地证明了这个问题，该网站现已被禁止在 Twitter 上发布。

少了什么？国际合规

迄今为止，还没有任何 NFT 平台涉足 NFT 拍卖会所代表的艺术版权的国际合规领域。这样做将是 NFT 生态系统的巨大飞跃。除了通过更强有力的版权执法来最大限度地减少欺诈之外，国际合规将允许在区块链境内进行象征性的版权交换。

1886 年伯尔尼公约已经奠定了基础，这是一项国际协议，保证在 179 个签署国中的任何一个国家创作作品时，都有标准化的版权保护。例如，该条约在 2014 年受到了考验，当时汤姆·佩蒂起诉萨姆·史密斯侵犯了史密斯的热门歌曲《和我在一起》(Stay With Me)的版权，这首歌与佩蒂的《我不会退缩》(I will not Back Down)在旋律上几乎完全相同这起诉讼和包括佩蒂遗产版税在内的和解协议，展示了《伯尔尼公约》的持续功能。

1996 年的世界知识产权组织版权条约正式将伯尔尼原则带入了数字艺术领域，但是许多伯尔尼公约的签署者并没有签署。在没有新条约的情况下，私营部门可能不得不收拾世界各国政府留下的烂摊子。

尽管国际条约规定了统一性，但 NFT 世界仍未能遵守世界各地版权法的多样性。为了让这个行业远离投机，走向全球化，必须将国际版权合规纳入这个新兴的生态系统。
