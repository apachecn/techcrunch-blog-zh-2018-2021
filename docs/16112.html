谷歌宣布总部位于 EPYC 的 Tau 云虚拟机

谷歌今天早上宣布推出 Tau，这是一个基于 AMD 第三代 EPYC 处理器的新的虚拟机家族。据该公司称，新的 x86 兼容系统比标准虚拟机的性价比提高了 42%。值得注意的是，谷歌早在 2017 年就开始将 AMD EPYC 处理器用于云，而亚马逊云的产品可以追溯到 2018 年。

谷歌声称 Tau 家族“跨越”了现有的云虚拟机。这些系统有多种配置，从每个虚拟机 60 个虚拟 CPU 到每个虚拟 CPU 4GB 内存不等。网络带宽高达 32 Gbps，并且它们可以与各种不同的网络连接存储相结合。

“各行各业的客户都在处理要求更高、数据密集型的工作负载，并寻找提高性能和降低成本的战略方法，”谷歌云首席执行官托马斯·库里安在一份新闻稿中说。“我们与 AMD 等关键战略合作伙伴的合作使我们能够扩大产品范围，为客户提供计算密集型、业务关键型应用程序的最佳性价比——所有这些都基于业界最干净的云。”

图片来源:谷歌

谷歌已经与一些知名客户签订了早期试用协议，包括 Twitter、Snap 和 DoIT。

Twitter 平台负责人 Nick Tornow 在一篇博客文章中说“在我们致力于为全球公众对话服务时，价位合适的高性能是一个关键的考虑因素。“初步测试显示了两位数的性能提升潜力，这让我们非常兴奋。我们正在与谷歌云合作，更深入地评估我们通过使用新的 Tau VM 系列可以实现的特定计算工作负载的性价比优势。”

图片来源:谷歌

Tau 虚拟机将于今年第三季度到达 Google Cloud。该公司已经向客户开放该系统进行预注册。价格取决于配置。例如，一台 32vCPU、内存为 128GB 的虚拟机每小时的运行成本约为 1.35 美元。
