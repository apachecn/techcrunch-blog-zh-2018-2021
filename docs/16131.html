Nylas，集成电子邮件和其他生产力工具的 API 的制造商，筹资 1 . 2 亿美元，超过 8 万名开发者 TechCrunch

像 Stripe 和 Twilio 这样的公司已经将 API 作为集成复杂功能的有效方法，这些功能可能不是您自己技术堆栈的核心，但却是您更广泛业务的必要部分。如今，一家采用这种模式创建了一种有效的方法，将电子邮件、日历和其他工具集成到使用 API 的其他应用程序中的公司，宣布将获得一大笔资金来扩展其业务。

Nylas 将自己描述为一个通信 API 平台——通过几行代码集成生产力工具，实现更多自动化，特别是在商业应用中——已经筹集了 1.2 亿美元的资金，这笔钱将用于继续扩展其提供的 API 类型，特别是不仅关注生产力应用，还关注人工智能和相关工具，以将更多自动化引入工作流。

Nylas 没有透露其估值，但这是一个非常重要的一步，该公司在这个时候看到强大的牵引力。

这比 Nylas 迄今为止筹集的资金(自 2015 年成立以来已筹集了 5500 万美元)高出一倍多，当它上次筹集资金时——2018 年的1600 万美元 B 轮——它说它的用户中有“数千名开发者”。现在，这个数字已经膨胀到 80，000，Nylas 每天处理大约 12 亿个 API 请求，每天处理 20tb 的数据。它还表示，在过去 12 个月里，收入增长了两倍。

系列 C 带来了一些有趣的名字 Nylas 的帽子表。新投资者 Tiger Global Management 领投本轮投资，之前的支持者花旗风险投资(Citi Ventures)、Slack Fund、8VC 和 Round13 Capital 也参与了投资。本轮其他新的支持者包括猫头鹰摇滚资本，蓝猫头鹰的一个部门；Stripe 联合创始人帕特里克·科利森和约翰·科利森；Klarna 首席执行官塞巴斯蒂安·西米阿特科夫斯基；还有托尼·法德尔。

与所谓的 API 经济中的其他公司一样，Nylas 发现的差距和机会是，有许多生产力工具主要存在于他们自己的筒仓中——这意味着当一个人想在应用程序中使用它们时，他们必须打开一个单独的应用程序来这样做。与此同时，构建新的工具，或者搭建一座桥梁来集成现有的应用程序，可能会非常耗时和复杂。

Nylas 首先通过电子邮件发现了这个问题。在其他应用程序中进行整合，使电子邮件和其中存储的数据更容易使用——这可以与微软和谷歌等主要提供商的电子邮件以及其他使用 IMAP 协议构建的服务一起使用——吸引了大量追随者，导致该公司扩展到其他领域，今天包括日程安排和日历，一种内置于情绪分析或生产力或工作流自动化等工具中的神经 API 和安全集成，以简化 Google OAuth 安全审查过程(例如在面向开发人员的应用程序中使用)。

“向数字通信和连接的根本转变导致各行各业的公司越来越依赖于开发人员来解决关键的业务挑战，并构建独特而迷人的产品和体验。因此，API 已经成为现代软件开发和数字化转型的核心，”Nylas 的联合创始人兼首席执行官格莱布·波利亚科夫在一份声明中说。

“通过我们强大的 API 套件，我们为开发人员提供了更快满足客户和市场需求所需的工具和应用，通过强大的定制用户体验创造竞争优势，并通过更高效、更智能的自动化流程和开发周期创造运营投资回报。我们很高兴能够继续推进我们的使命，让世界变得更有生产力，并且很荣幸能够得到杰出投资者和企业家的支持。”

事实上，Nylas 的崛起和它所实现的功能是我们在整个企业中看到的更大转变的一部分:随着组织变得更加数字化，并使用更多基于云的应用来完成工作，开发人员已经成为帮助机器运行的关键机制。更强调将服务集成在一起的 API 是他们常用的工具包的一部分，这也是今天投资者支持 Nylas 的决定性原因之一。

Tiger 的合伙人约翰·柯蒂斯(John Curtius)在一份声明中说:“公司正在迅速采用 API 作为自动化生产力的方式，并找到新的和创新的方式来支持现代工作和协作。”“这一趋势对于创建无摩擦、有意义的数据驱动型通信至关重要，它将推动数字化转型。我们相信 Nylas 在引领 API 经济的未来方面具有独特的优势。”柯蒂斯将在这一轮加入董事会。
