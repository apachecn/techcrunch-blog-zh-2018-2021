Transform 以 2450 万美元的资金启动，用于从数据仓库中查询和构建指标的工具

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/17/transform-metrics/

最大的科技公司已经投入大量时间和金钱，为他们的数据科学团队和那些与他们一起工作的人构建工具和平台，以便从他们公司产生的大量数据中收集见解和指标:如果你知道如何阅读数据，那么公司表现如何，新功能如何工作，什么时候出现问题，或者什么时候可能会卖得很好(以及为什么)，这些都是你可以弄清楚的事情。

现在，三名在大型技术领域从事数据工作的校友创办了一家初创公司，旨在建立一个“指标商店”，以便企业世界的其他部分——其中大部分缺乏从头构建这种工具的资源——也可以轻松地使用指标来解决这种问题。

这家初创公司被称为“转变”( Transform )( T1 ),如今它正悄然出现，并获得了大量的早期支持——这不仅表明投资者对这些特定创始人的信心，也表明投资者认识到，市场上存在一个缺口，正如该公司所描述的那样，这是一个可以有效填补的“商业数据真实的单一来源”。

该公司宣布，它已经秘密完成了 2000 万美元的首轮融资，以及更早的 450 万美元的种子轮融资——都是由 Index Ventures 和 Redpoint Ventures 牵头的。该公司表示，种子基金也有数十名天使投资者，其中包括 Color Genomics 的 Elad Gil、Airbnb 的 Lenny Rachitsky 和 conception 的 Cristina Cordova。

Transform 取得的重大突破是，它建立了一个公司可以应用于其结构化数据的指标引擎——一个类似于大型科技公司为自己使用而建立的工具，但它还没有真正为非大型科技公司的其他人创建(至少到目前为止)。

Transform 可以处理来自仓库的大量数据或实时跟踪的数据，以生成关于公司产品不同行为的见解和分析。Handel 说，Transform 可以被仍然需要处理数据的非技术人员使用和查询。

打造这款产品的动力来自尼克·汉德尔(Nick Handel)、詹姆斯·梅菲尔德(James Mayfield)和保罗·杨(Paul Yang)——他们分别是 Transform 的首席执行官、和软件工程师——当时他们一起在 Airbnb 工作(之前梅菲尔德和杨也在工作)，担任包括产品管理和工程在内的多种职务。

在那里，他们可以直接看到数据有助于围绕产品做出决策、衡量某样东西的使用方式或规划未来功能的前景，以及利用数据工作的需求，并让每个人都在同一页面上这样做。

“科技公司越来越倾向于测试每一个功能，每一次迭代。因此，作为其中的一部分，我们(在 Airbnb)建立了这个工具，基本上允许你定义你想要跟踪的各种指标，以了解你的实验，”亨德尔在采访中回忆道。“但你也想了解许多其他事情，比如，有多少人在搜索特定地区的房源？有多少人会立即预订这些房源？他们是否在联系客服，他们是否有信任和安全问题？”Airbnb 开发的工具是 Minerva T1，专门针对 Airbnb 通常可能会对自己的数据提出的问题进行了优化。

“通过锁定指标的所有定义，您基本上可以拥有一个数据工程团队，一个集中的数据基础架构团队，完成这些指标的所有计算，然后将这些数据提供给数据科学家，让他们进行更深入、更有趣的工作，因为他们不会陷入一次又一次计算这些指标的困境，”他继续说道。这个平台是在 Airbnb 内部发展起来的。“我们确实受到了这个工具的一些早期工作的启发。”

问题是，并不是每个公司都是为了构建这样的工具而构建的，不管它们自己的商业利益是什么。

“有一些公司在度量领域做类似的事情，”梅菲尔德说，“像 LinkedIn、Airbnb 和优步这样的顶级公司。他们真的开始投资度量标准了。但是，只有那些公司可以投入 8 到 10 人的团队，工程师，设计师，他们可以在内部建造这些东西。我认为这可能是，你知道，想要创办这家公司的很大一部分动力是说，不是每个组织都能够投入 8 或 10 名工程师来构建这个度量工具。”

另一个问题是，对于一家科技公司(或者默认情况下，任何一家)来说，在产品设计和更广泛的商业战略领域，指标已经成为越来越重要的——也许是最重要的——决策杠杆。

我们已经远离了“快速行动，打破常规”相反，我们现在信奉——正如梅菲尔德所说——“如果你不能衡量它，你就不能移动它。”

Handel 说，转型围绕三个基本优先事项进行。

第一个问题与度量标准的集体所有权有关:通过建立一个衡量和识别这些标准的单一框架，他们的理论是，公司更容易在使用这些标准时达成一致。第二个是使用 Transform，通过将提取洞察力的最重复部分转换为可以使用和重用的自动化脚本，使数据团队能够花更多时间分析数据，而不仅仅是构建数据集，从而使数据团队的工作更加高效和轻松。第三，为客户提供 API，他们可以使用这些 API 将指标提取工具嵌入到其他应用程序中，无论是在商业智能领域还是其他领域。

它今天推出的三个产品，称为 Metrics Framework、Metrics Catalog 和 Metrics API，都遵循这些原则。

Transform 今天才真正公开发布，但 Handel 表示，它已经在一个小测试版中与少数客户(未透露姓名)合作，足以确信它所构建的内容能够按预期工作。这笔资金将用于继续开发该产品，并吸引更多人才，希望有更多企业使用该产品。

“希望”这个词可能不会像其投资者会使用的那样含糊不清，他们相信它满足了市场的强烈需求。

“转型填补了行业内的一个重要空白。正如我们早期投资于 Looker 的创新商业智能方法一样，Transform 通过提供一个强大而简化的指标真实来源而更进了一步，”Redpoint Ventures 医学博士 Tomasz Tunguz 在一份声明中说。

“我们看到全球各地的公司都在努力理解无穷无尽的数据源，或将它们转化为可操作的可信指标。Index Ventures 的合伙人 Shardul Shah 补充说:“我们投资 Transform 是因为他们开发了一种解决这一问题的优雅解决方案，将改变公司对数据的看法。
