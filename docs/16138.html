我们都听说过这样一句话，“每家公司都是金融科技公司。”

但如今，随着越来越多甚至不属于金融服务领域的公司寻求在其产品中加入金融科技成分，这种情况变得越来越真实。

一批准备从这一转变中受益的初创公司是那些提供银行即服务的公司。一家这样的初创公司 Unit 已经在 B 轮融资中筹集了 5100 万美元，以进一步实现其目标，即让公司和金融科技公司能够“在几分钟内”构建银行产品。

总部位于硅谷的 Accel 引领了 Unit 的这轮融资，使该公司自 2019 年成立以来的融资总额接近 7000 万美元。现有的支持者 Better Tomorrow Ventures、Aleph、fluorescent Ventures 和 TLV 合伙公司也参与了最新的融资 。

创始人 Itai Damti 和 Doron Somech 对成长型公司并不陌生。这两人之前共同创立了一家总部位于特拉维夫的 B2B 交易技术提供商 Leverate。该单位在特拉维夫和纽约市有两个总部。

Damti 和 Somech 于 2019 年底成立了 Unit，并在第一年悄悄搭建了这个平台，其使命是 让公司能够将金融服务嵌入到他们的产品中，加快他们的上市时间。Unit 于 2020 年末正式推出其平台，在过去的三个月中，其存款量增长了 300%以上，新终端用户增长了 600%(尽管基数较小)。

有了它的平台，各行各业的公司——比如自由职业者或创造者经济和个人财务管理——可以直接在他们的软件中构建财务产品。这使他们能够 建立和推出下一代银行账户、卡、支付和借贷产品。客户包括 Wethos、Lance、Benepass、Moves 和 Tribevest 等。

“我们的使命是扩大所有人的金融服务，我们通过赋予下一代金融科技建设者权力来实现这一目标，”Damti 说。他说，只有大约 20%的客户可能被认为是真正的金融科技公司。其余 80%的公司不希望将银行服务嵌入到他们的产品中。

Damti 声称，Unit 采用了曾经“18 个月的非常昂贵和复杂的过程”,包括寻找和管理银行关系，建立合规团队和建立技术堆栈,“让你获得有竞争力的银行产品，并将其转变为一个 API 和一个仪表板，帮助公司在五周内推出账户卡、支付和贷款。”

在融资的同时，Unit 今天还宣布了一项新的产品 Unit Go，据称它允许 公司在几分钟内创建实时银行账户并发行实体和虚拟卡。创始人 和开发者可以通过创建一个免费账户，在 Unit 的 live 环境中构建，并使用真实资金测试他们的产品来进行尝试。Unit Go 目前处于测试阶段， 将于 2021 年秋季上市。

该公司计划利用其新资本增加 26 名员工，并加快其单位 Go 的发行。它还希望将其平台扩展到其他金融产品、软件开发工具包(SDK)和集成领域。(已经是 与格子 融为一体了，比如)。

当然，在蓬勃发展的银行即服务(BaaS)领域，Unit 并不是唯一的初创公司。它与类似的rail bankTreasury Prime和Stripe竞争。Damti 认为，有几件事有助于在日益拥挤的空间区分单位。

根据 Damti 的说法，首先，Unit 有意地“将合规性放在我们工作的首要位置”作为证据，今年早些时候，it 任命阿曼达·斯沃兰德 为首席合规官。

其次，丹提沿着 向斜 的思路强调，单位不是媒人，也不是市场。

“我们是一家将银行与科技生态系统连接起来的公司，银行是我们的重要供应商和合作伙伴，但我们将它们视为 Unit 的内置元素，因为我们相信这个生态系统中最优秀的体验只能来自软件公司，”Damti 告诉 TechCrunch。

最后，他指出，Unit 在技术上是独特的，因为它实际上是在建立一个分类账，Damti 将其描述为“生态系统中最关键和最敏感的部分。”

他说，通过拥有账本和不授权，Unit“能够提供更好的体验。”

“就交易环境、我们提供的数据的整洁度以及我们的客户能够控制和调整的费用而言，拥有那份账本对体验来说是极其重要的，”达姆蒂说。

Accel partner 艾米特·库马尔 指出，近年来，形势已经从数百家金融科技初创公司“试图用稍好的产品击败现有公司”转变为数千家科技公司试图在各自的垂直领域推出金融科技业务。

他告诉 TechCrunch:“该部门对管理合规性的高度重视解决了通常与提供银行服务相关的风险，并允许客户以比以前更快的速度将这些产品推向市场。”。" Unit 正在构建支持下一代金融科技的平台."
