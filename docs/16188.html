移动商务初创公司 Via 获得了 1500 万美元的首轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/21/via-a-mobile-commerce-startup-that-claims-51-million-in-sales-in-its-first-six-months-just-closed-its-a-round/

移动商务正处于风口浪尖，对所谓的对话商务初创公司的投资不断增加凸显了这一机遇。

Via 是一家成立两年的旧金山湾区初创公司，它是这股浪潮中的一员，已经发现了一些趋势，这些趋势在这个月变得越来越清晰。首先，今年手机上的电子商务销售额将超过台式机(有人估计高达 70%)，人们倾向于立即阅读短信，消费者每天花费超过 30 分钟使用移动通讯应用。

Via 还坚持认为，与专注于帮助零售商和其他人在短信中传播营销信息的不断扩大的创业公司不同，玩家有空间更好地解决许多其他环节，这些环节增加了愉快的消费者体验，从提供优惠券代码到开始退货过程。

事实上，据联合创始人兼首席执行官 Tejas kond uru(Brigham Young 的毕业生，其父母从印度移民到美国，自己曾在科技初创公司工作过)说，他的公司现在有 50 人，他早期的一个见解是，尽管他们的许多客户现在使用移动浏览器访问他们的商店并从他们的商店购物，但许多零售商使用 Shopify 或 BigCommerce 等网站建设者“将所有东西都塞进移动设备，只留下足够的空间，比如一张照片和一个购买按钮。”Konduru 认为，必须有一种方法来利用所有这些客户在他们的网站上对品牌的购物体验，并以快速、移动本地的方式实现这些体验。

他说，威盛的解决方案是帮助这些企业在客户最常用的设备和应用程序上与客户互动。“当有人使用 Shopify 或 BigCommerce 或任何这些平台时，”Konduru 说，“我们也将它连接到 Via，它基本上接受整个购物体验，并允许(顾客)快速浏览菜单或目录，例如 Facebook Messenger。Via 还将创建一个类似原生 iOS Android 应用的网站，将其克隆到原生 iOS Android 应用中，然后出售推送通知应用内聊天层。“本质上，”他补充道，“任何时候有人在手机上购物，而他们没有使用浏览器，这就是 Via 正在处理的事情。”

“消息”似乎传到了正确的人那里。去年推出的 Via 表示，它现在有 54 名全职员工，拥有 190 个品牌的客户，并刚刚获得由 Footwork 牵头的 1500 万美元 A 轮融资，Footwork 是一家新的风险投资公司，由前 Stitch Fix 首席运营官·迈克·史密斯和前 Shasta Ventures 投资者 Nikhil Basu Trivedi 共同创立。

这轮融资的其他参与者包括彼得森风险投资公司(Peterson Ventures)，Konduru 曾在那里实习；著名的多摩创始人乔希·詹姆斯，孔杜鲁也曾在这里被关押过；以及一长串其他著名的个人投资者，包括 Qualtrics 的瑞安史密斯(Ryan Smith)和 Lattice 的联合创始人兼首席执行官杰克奥尔特曼(Jack Altman)。

至于该公司如何收费，它不像传统的 SaaS 公司那样要求月费或年费，而是按照每次互动收费，无论是短信、语音通话、视频还是 GIF。

根据 Konduru 的说法，这开始增加，他说 Via 的普通客户看到了 15 倍的投资回报，从 2020 年 5 月——该公司的服务上线——到 12 月，该公司创造了 5100 万美元的销售额。Konduru 拒绝透露 Via 从这些交易中获得了多少，但表示该公司今年的年度经常性收入有望达到 1，000 万美元。

至于品牌如何开始使用威盛，根据该公司的说法，这很简单。Konduru 说，只要一家公司正在使用一个商务平台——从 Shopfiy 到 WooCommerce 再到 sales force——只需要大约五分钟就可以制作一个移动应用程序，其中的菜单显示了品牌可以通过 via 的平台实现的交互类型。

Konduru 在决定创办 Via 之前涉足了投资银行业务，他说，他对这家初创公司的快速发展并不感到惊讶，但他说，他对该公司看到的广泛对话感到吃惊。虽然他想象 Via 将成为品牌的一个强大营销渠道，这些品牌利用该平台发布关于废弃购物车和即将交付的通知，但这比他想象的更像是一条双向道路。

“每个月，大约有 15，000 人通过 Via 启动退货流程，并将从 Via 支持的渠道收到通知。但是突然之间——比如说顾客买错了 t 恤尺寸——人们开始和这个品牌交流。你可以看到从粉丝赞赏到地址变化，再到关于不良折扣代码的信息，再到“我的订单在哪里”类型的交换。这是我没有预料到的，”Konduru 说，他说，在进行首轮融资之前，Via 获得了由 Peterson Ventures 牵头的 420 万美元种子资金。

“我以为人们会看着通知，然后把它扔进某个深渊。相反，人们开始与品牌互动。”
