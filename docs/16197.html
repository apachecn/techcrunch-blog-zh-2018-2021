Airbank 集中了您所有的企业银行账户和财务数据 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/22/airbank-centralizes-all-your-business-bank-accounts-and-financial-data/

认识一下 Airbank，这是一家利用开放银行监管和相关 API 来整合你所有银行账户的初创公司。该公司专注于创业公司和中小型公司，希望建立一个一体化的银行界面来访问财务数据、启动支付、管理现金流等。

Airbank 刚刚筹集了 300 万美元(€250 万美元)的种子资金，由 Pia d'Iribarne 和 Jean de la Rochebrochard 在 New Wave 牵头，Speedinvest 和 Tiny VC 也参与了。少数商业天使也加入了这一轮，如克里斯康德(Accel 常驻高管)、卢卡阿斯卡尼(Accel scout)和马克麦凯布(Sequoia scout)。

这家初创公司的价值主张非常简单，可以用一张截图轻松解释。通过 Airbank，您可以输入您使用的所有银行账户和相关账户的登录信息。之后，您可以查看您的 Airbank 帐户中的所有内容:

图片来源: Airbank

许多公司出于多种原因不得不处理多个银行账户——当您成立公司时，您可能已经开立了一个银行账户，另一个银行账户用于申请贷款，一个用于降低国外交易费用的明智商业账户，一个用于为每个人获取借记卡的 Revolut 商业账户，等等。

除了银行账户，你还有可能通过 Stripe、PayPal 或 Shopify 获得收入。许多主管浪费了大量时间连接到门户网站，将数据导出为 CSV 文件，将这些文件导入 Microsoft Excel 并整合所有信息。

Airbank 会自动刷新您多个账户的余额。您可以查看多种货币的总余额。它还可以帮助您核对交易和未清发票，因为您可以一次搜索多个帐户。

这只是一个起点；Airbank 希望成为您所有银行需求的唯一接口。你可以对交易进行分类，查看你在每个供应商身上花了多少钱，跟踪经常性付款，并将一切导出到 Google Sheets 或 Microsoft Excel。很快，你就可以使用 Airbank 进行现金流预测，并与你的 Xero 或 QuickBooks 数据自动对账。

开放式银行不仅限于账户聚合。使用适当的 API，您应该能够从第三方产品发起支付。Airbank 计划利用这一点，因为该公司正在致力于支付。由于您可以管理访问权限，Airbank 可以充当财务团队的支付门户。

“开放式银行业务实现了与银行的顺利集成，我们可以利用它为我们的用户提供更丰富的银行和支付体验。我们的愿景是建立一个一体化的金融中心，在一个地方连接您的所有金融账户。联合创始人兼首席执行官克里斯托弗·泽米娜在一份声明中说:“我们的集成将把账单支付、费用管理和外汇都纳入一个易于使用的单一产品中。”

其他创业公司一直在致力于现金流管理，如 Agicap ，以及 B2B 支付，如 Libeo 和 Upflow 。Airbank 从账户聚合开始，希望以整体方式解决 B2B 金融问题。

垂直 SaaS 产品最近一直在蓬勃发展。这个领域竞争激烈是有原因的:围绕 B2B 金融科技和专业化的软件即服务产品，还有大量的事情要做。
