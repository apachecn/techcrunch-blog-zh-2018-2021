Transmit Security 筹集了 5.43 亿美元的 A 轮融资来取消密码

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/22/transmit-security-raises-543m-series-a-to-kill-off-the-password/amp/

图片来源: 卡尔·伊斯托克 / Getty Images

Transmit Security 是一家总部位于波士顿的初创公司，其使命是让世界摆脱密码，该公司已经在 A 轮融资中筹集了 5 . 43 亿美元。

这轮融资据说是网络安全历史上最大的 A 轮投资，也是自举公司的最高估值之一，由 Insight Partners 和 General Atlantic 牵头，Cyberstarts、Geodesic、SYN Ventures、Vintage 和 Artisanal Ventures 也进行了投资。T11】

Transmit Security 表示，其投资前估值为 22 亿美元，将利用新资金扩大其影响力，并投资于关键的全球领域，以发展该组织。

然而，最终，这轮融资将帮助该公司加速其使命，帮助世界实现无通行证。据这家初创公司称，由于“本质上不安全的”基于密码的身份认证，组织每年都会损失数百万美元；不仅 80%以上的数据泄露是由弱密码造成的，而且重置单个密码的平均帮助台人工成本超过 70 美元。T15】

Transmit 表示，其基于生物特征的身份认证器是第一个本地无密码身份和风险管理解决方案，已经被包括 Lowe's、Santander 和 UBS 在内的许多知名品牌采用。该公司表示，该解决方案目前每秒处理超过 9000 个认证请求，可以减少 96%的帐户重置，并将客户认证从 1 分钟减少到 2 秒。

“通过消除密码，企业可以立即减少流失和弃车，并为个人数据提供卓越的安全性，”Transmit Security 首席执行官 Mickey Boodaei 说，他是 2014 年该公司的联合创始人。“我们的客户，无论是在零售、银行、金融、电信还是汽车行业，都明白提供优化的身份体验是一项价值数百万美元的挑战。有了 premier partners 的最新一轮融资，我们可以显著扩大我们的覆盖范围，帮助世界摆脱密码。”

Transmit Security 并不是唯一一家致力于消除密码的公司。微软已经宣布计划让 Windows 10 免密码，而苹果最近预览了 iCloud Keychain 中的 Passkeys，这是一种由 WebAuthn 以及 Face ID 和触控 ID 支持的免密码认证方法。
