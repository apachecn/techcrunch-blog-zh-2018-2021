Vantage 筹集 400 万美元帮助企业了解其 AWS 成本

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/22/vantage-raises-4m-to-help-businesses-understand-their-aws-costs/amp/

图片来源: sihuo0860371 / Getty Images

帮助企业分析和降低 AWS 成本的服务公司 Vantage 今天宣布，它已经筹集了 400 万美元的种子资金，由 Andreessen Horowitz 牵头。包括 Brianne Kimmel、Julia Lipton、Stephanie Friedman、Calvin French Owen、Ben and Moisey Uretsky、Mitch Wainer 和 Justin Gage 在内的多位天使投资人也参与了此轮投资。

Vantage 从开始，专注于让 AWS 控制台更容易使用——并帮助企业在这个过程中了解他们在云基础设施预算上的支出。但正如 Vantage 联合创始人兼首席执行官 Ben Schaechter 告诉我的那样，真正吸引用户的是成本透明的特性。

“我们宣传自己是一个替代 AWS 控制台，专注于开发者体验和成本透明，”他说。“有趣的是，即使在 1 月份正式 GA 发布之前的早期访问中，我也可以说，我们从客户那里获得的 95%以上的反馈完全围绕着我们在 Vantage 中拥有的成本功能。”

图片来源: Vantage

像任何好的创业公司一样，Vantage 团队看到了这一点，并决定加倍重视这些功能，并在营销中突出它们，尽管它也保留了现有的 AWS 控制台相关工具。Schaechter 认为，其他工具没有大获成功的原因是因为越来越多的 AWS 用户已经习惯于将基础设施作为代码来进行他们自己的自动配置。这样一来，他们在 AWS 控制台上花费的时间就少多了。

“但有一点是一致的——总体而言——人们一年有 12 次真的真的很难过，他们会收到一份令人震惊的 AWS 账单，并不得不弄清楚发生了什么。Vantage 今天所做的是在透明度方面提供大量价值，”他说。

在过去的几个月中，该团队为其成本透明工具添加了许多新功能，包括机器学习驱动的预测(在总体客户级别和服务级别)以及跨团队共享报告的能力。

图片来源: Vantage

虽然 Vantage 希望在未来增加对其他云的支持，可能会从 Azure 开始，然后是 GCP，但这实际上不是该团队目前关注的重点。相反，Schaechter 指出，该团队计划增加对从第三方云服务中引入数据的支持。

“公司的首选往往是 AWS、GCP 和 Azure，”他说。“然后，在那之后，就是 Datadog、Cloudflare、Sumo Logic，诸如此类的东西。目前，没有办法从基于云使用的角度来看 P&L 或 ROI。Vantage 可以成为一种工具，让您从本质上了解所有云成本。”

这可能也是投资者的愿景，尽管 Vantage 现在正在与 Apptio 的 Cloudability 和 VMware 的 CloudHealth 等企业工具竞争，Schaechter 似乎并不担心竞争。他认为，这些工具诞生于 AWS 只有少量服务和几种交互方式的时代。他认为，Vantage 作为一个现代化的自助服务平台，将比这些更古老的服务具有相当多的优势。

“只需点击几下鼠标，您就可以启动并运行。你不必和销售团队交谈。在这个阶段，我们一直在帮助大量的初创公司，直到企业，而在我看来，Cloudability 和 CloudHealth 是一种过时的企业产品。据我所知，目前没有一家初创公司选择使用这些技术，”他说。

到目前为止，这个团队主要由 Schaechter 和他的联合创始人兼首席技术官 Brooke McKim 组成，他们将公司发展到了这一步。现在，他们计划用新的资本来建立自己的团队(该公司目前正在积极招聘)，包括开发和上市方面。

该公司为每月 AWS 成本高达 2500 美元的企业提供免费的入门计划，为需要跟踪更大账户的企业提供每月 30 美元的付费计划。
