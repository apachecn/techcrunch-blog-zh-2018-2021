广告技术初创公司孙铁麟筹集种子资金，让房地产广告和电子商务广告一样出色

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/23/adtech-startup-tomi-raises-seed-funding-to-make-real-estate-ads-perform-as-well-as-ecommerce/amp/?guccounter=1

图片来源:康斯坦丁·巴扬丁，孙铁麟创始人

房地产、汽车和金融服务等行业的线下销售周期很长，数字广告在这些领域往往表现不佳。转换率很低，而且由于现实世界的资产是离线的，广告商的诱惑是购买线索和点击，这可能会抬高客户获取成本。基本上，人们在浏览，但最终还是会在网下购买。

一家新的创业公司，孙铁麟计划通过处理用户在公司网站上的行为(使用跟踪像素，结合广告 API 和 CRM)来解决这个问题，以帮助公司以电子商务的方式接触更多的客户。

它现在从包括 Begin Capital 和 Phystech Leadership Fund 在内的投资者那里筹集了 100 万美元的种子资金。

孙铁麟由康斯坦丁·巴扬丁(Konstantin Bayandin)创建，他是 Compass 数字营销和技术的前高级总监，也是“俄罗斯的亚马逊”Ozon 的首席营销官，与类似的广告科技公司竞争，如 Anytrack、Sociaro、Meetotis、Alytics 和 Postclick。

然而，Bayandin 表示，不同之处在于孙铁麟“专注于线下转化，并与多个广告渠道合作，如脸书、Instagram 和谷歌。”

Bayandin 说:“房地产公司很乐意利用在线广告来销售他们的库存，但事实证明这太贵太难了。人们喜欢浏览，但很少转换，而且大多数交易都是在线下进行的。所以房地产客户不知道如何为他们真正的买家优化。孙铁麟使用机器学习来分析真正的买家浏览网站的方式，并优化广告活动以提高转化率。”

这一切的背景是，随着苹果关闭 IDFA，谷歌计划从其 Chrome 浏览器中删除第三方 cookies，以及最新的 iOS 14.5 更新允许用户选择退出“个性化广告”，整个广告业务都在不断变化，因此将需要新的工具。Bayandin 说，孙铁麟是广告技术新浪潮的一部分。
