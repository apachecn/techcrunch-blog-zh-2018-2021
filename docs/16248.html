NFT 市场初创公司 Rarible 完成 1420 万美元的 A 轮融资

你可能已经看到了围绕 NFT 销量从市场最近的峰值大幅下降的头条新闻，一些报告显示最近几周跌幅高达 90%。虽然许多加密投机者在这种快速下降中处于失败的一端，但其他人看到了数字收藏品和数字商品接管互联网的未来。

NFT 市场 Rarible 周三宣布，他们已经完成了来自 Venrock Capital、CoinFund 和 01 Advisors 的 1420 万美元 A 轮投资。这家初创公司允许用户购买和拍卖数字艺术。该公司还拥有自己稀有的治理令牌，允许用户就平台的功能和费用提出建议并进行投票。

这家初创公司表示，自 18 个月前推出以来，该平台的销售额已达 1.5 亿美元。

最近几个月，随着投资者联合起来支持数字收藏品领域的早期参与者，NFT 初创公司筹集了大笔资金。OpenSea 是 NFT 最受欢迎的枢纽之一，在早春从 Andreessen Horowitz 和 Naval Ravikant 那里筹集了 2300 万美元的资金。今年 3 月，受邀才能参加的 NFT 市场 SuperRare 筹集了 900 万美元的首轮融资。上个月， Bitski 筹集了 1900 万美元，试图成为“NFTs 的商店”

除了资金宣布，Rarible 还分享了他们与 NBA 顶级投篮制造商 Dapper Labs 的正式合作，将他们的 NFT 市场带到 Dapper's Flow 区块链。Dapper Labs 已经开始在其专有的区块链上建立一个更广泛的 NFT 平台联盟，这比以太坊网络上的交易更加节能，成本更低，尽管它也更加集中。Rarible 表示，他们将在“接下来的几个月内”完成这一转变

这家初创公司有一个广泛的目标，那就是让日常网络用户更容易接触到 NFT 空间，他们在一篇博客文章中说，他们希望利用这笔资金来探索更多用户友好的发展，比如接受信用卡支付。“我们需要一个独立创作者可以毫不费力地创作出 NFT 并以 5 美元的价格卖给他的支持者社区的世界，以及一个流行偶像可以使用 NFTs 以可持续的方式与她的数百万观众建立密切联系的世界，”帖子写道。
