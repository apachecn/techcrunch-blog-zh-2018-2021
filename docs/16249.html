NFT 市场初创公司 Rarible 完成 1420 万美元的首轮融资 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/23/nft-marketplace-startup-rarible-closes-14-2-million-series-a/amp/

你可能已经看到了围绕 NFT 销售额从市场最近的峰值大幅下降的头条新闻，一些报道显示近几周降幅高达 90%。尽管许多密码投机者在快速下跌中处于亏损状态，但其他人看到了数字收藏品和数字商品接管互联网的未来。

NFT 市场周三宣布，他们已经完成了由 Venrock Capital、CoinFund 和 01 Advisors 投资 1420 万美元的 A 系列。这家初创公司允许用户购买和拍卖数字艺术。该公司还拥有自己的稀有治理令牌，允许用户就平台的特点和费用提出建议并投票。

这家初创公司表示，自 18 个月前推出以来，该平台的销售额已经达到 1.5 亿美元。

最近几个月，随着投资者联合起来支持数字收藏品领域的早期参与者，NFT 初创公司筹集了大量资金。T2 最受欢迎的 NFT 中心之一 OpenSea 在早春从 Andreessen Horowitz 和 Naval Ravikant 那里募集了 2300 万美元的资金。今年 3 月，只接受邀请的 NFT 市场超级稀有筹集了 900 万美元的 A 系列。上个月，Bitski 筹集了 1900 万美元，希望成为“NFT 的商店”。

除了资金宣布之外，力宝还表示，他们已经正式与美国职业篮球联赛头号种子制造商达帕实验室合作，将他们在 NFT 的市场带到了达帕的流动区块链。Dapper Labs 已经开始在其专有的区块链平台上建立更广泛的 NFT 平台联盟，这比以太网网络上的交易更节能、成本更低，尽管它也更加集中。力宝表示，他们将在“未来几个月”实现这一转变。

这家初创公司有着让日常网络用户更容易接近 NFT 空间的广泛目标，在一篇博客文章中表示，他们希望利用这笔资金探索更方便用户的发展，比如接受信用卡支付。“我们需要一个独立创作者可以毫不费力地创作出一部 NFT，并以 5 美元的价格卖给他的支持者群体的世界，同时一个流行偶像可以利用 NFTs 以可持续的方式与她的数百万观众建立密切的联系，”该帖子写道。
