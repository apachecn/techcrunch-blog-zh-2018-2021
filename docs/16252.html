PlanetScale 为其数据库服务 TechCrunch 融资 3000 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/23/planetscale-raises-30m-series-b-for-its-database-service/

最初在 YouTube 上开发的 MySQL 开源数据库集群系统 Vitess 背后的公司 PlanetScale 今天宣布，它已经筹集了 3000 万美元的 B 轮融资，由 Insight Partners 牵头，a16z 和 SignalFire 参与其中。据 Crunchbase 称，该公司现在已经筹集了总计 5500 万美元的资金。

今天的声明是在 PlanetScale 推出其新的托管数据库平台(也称为 PlanetScale)仅几周后发布的。该公司之前提供了托管版本的 Vitess，但在这项新服务中，它更进一步，提供了它所谓的“开发者第一数据库”，该数据库抽象出所有的基础设施，以确保开发者不必考虑云区、集群大小和其他细节。

事实上，PlanetScale 首席执行官兼联合创始人 Jiten Vaidya 对这一早期产品的局限性相当开放。“我们去年建立的几乎是托管的 Vitess，这与今天许多云提供商给你的数据库没有什么不同，”他说。“因此，没有这种易用性，没有这种优雅，没有这些开发人员今天想要和期望的最先进的体验，我们已经内置到我们的产品中。”

但是几个月前，该公司请来前 GitHub 工程副总裁 Sam Lambert 担任首席产品官。Vaidya 指出，Lambert 为 PlanetScale 带来了许多开发人员的共鸣，并帮助它推出了这款新产品。

“人们来找你，因为他们不是数据库专家，但他们有数据，他们有问题，”兰伯特说。“太多的公司，尤其是数据库领域的公司，不像我们这样考虑用户的日常生活。他们没有考虑用户实际上想要做什么的完整旅程，也就是为他们的客户提供价值。他们只是对自己存储和检索数据的能力印象深刻。就像是，是的，我们一直在这么做。我们从 60 年代就开始这样做了。我们现在能做点别的吗？”

该公司今天的用户包括 Slack、Figma、GitHub 和 Square 等，因此它显然为许多用户提供了价值。正如兰伯特指出的，PlanetScale 旨在为他们提供一种简单易用的产品。“说实话，仅仅因为它简单易用、美观——就像设计精美的工具一样——并不意味着它就低人一等。这并不意味着它遗漏了什么。这意味着其他人错过了诗意和可以添加到基础设施产品中的额外美丽元素，”他说。

PlanetScale 计划使用新的资金在全球范围内扩大其团队，并加快其平台的采用。Insight Partners 董事总经理 Nikhil Sachdev 将加入公司董事会，该公司董事总经理 Praveen Akkiraju 也将作为董事会观察员加入。

Sachdev 说:“PlanetScale 为无服务器时代的基于云的数据库的简单性、性能和可扩展性树立了新的标杆。“长期以来，数据库开发人员的体验一直很痛苦。PlanetScale 正在打破这一链条，以一种极其优雅、有品位和有用的方式解决与可扩展性和可靠性相关的长期问题。”
