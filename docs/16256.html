Vercel 为其前端开发平台 TechCrunch 募集 1.02 亿美元的 C 轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/23/vercel-raises-102m-series-c-for-its-next-js-based-front-end-development-platform/amp/?guccounter=1

图片来源:路易斯·阿尔瓦雷斯/盖蒂图片社

广受欢迎的开源 Next.js React 框架背后的公司 Vercel 今天宣布，它已经筹集了由基岩资本牵头的 1.02 亿美元的 C 轮融资。现有投资者 Accel、CRV、Geodesic Capital、Greenoaks Capital 和 GV 也参与了这一轮投资，另外还有新投资者 8VC、Flex Capital、GGV、拉塔科拉、Salesforce Ventures 和 Tiger Global。总的来说，该公司现在已经筹集了 1.63 亿美元，目前的估值为 11 亿美元。

正如 Vercel 指出的那样，该公司最近几个月增长强劲，自 2020 年 10 月以来，其网络上所有网站和应用的流量都翻了一番。同期，全球最大的 10，000 家网站中使用 Next.js 的网站数量也增长了 50%。

图片来源: Vercel

鉴于 Next.js 框架的开源性质，并非所有这些用户都是 Vercel 的客户，但其当前的付费客户包括 Carhartt、Github、IBM、麦当劳和优步等。

“对我们来说，一切都始于前端开发人员，”Vercel 首席执行官 Guillermo Rauch 告诉我。“我们的目标是为这些开发人员和他们的团队创造并赋予他们能力，为他们的客户创造令人愉快、身临其境的网络体验。”

通过 Vercel，Rauch 和他的团队采用了 Next.js 框架，然后构建了一个专门迎合该框架的无服务器平台，允许开发人员专注于构建他们的前端，而不必担心扩展性和性能。

劳赫认为，旧的解决方案是在与云平台和无服务器技术隔离的情况下构建的，让开发人员来部署和扩展他们的解决方案。虽然一些潜在用户也可能满足于使用一个无头内容管理系统，但劳奇认为，开发者越来越需要能够构建比许多企业目前使用的现成解决方案更深入的解决方案。

Rauch 还指出，开发人员非常喜欢 Vercel 在每次编辑代码时为网站前端生成预览 URL 的能力。“因此，我们不再把所有时间都花在代码审查上，而是把你的时间转移到审查或体验你的前端。这使得体验更具协作性，”他说。“所以现在，设计师、营销人员、IT、首席执行官……现在可以一起合作构建前端，并说，‘这种蓝色不是正确的蓝色。’"

“Vercel 正在引领市场转型，通过这一转型，我们看到 web 和云应用程序开发中的大部分增值都是在最接近用户的前端交付的，在这里可以创造和享受真正的体验，”belt 的创始人兼管理合伙人 Geoff Lewis 说。“我们非常热情地与吉列尔莫和他组建的无与伦比的团队密切合作，推动这场革命向前发展，我们非常高兴能够共同领导这一轮。”
