Android 应用通过亚马逊应用商店进入 Windows 11

生态系统是奇怪的伙伴。这是最近记忆中的一个陌生人——也是今天 Windows 11 活动中最意想不到的消息之一。微软今天宣布，它将在其操作系统的下一个主要版本中提供 Android 应用程序。

首席产品官 Panos Panay 称这一增加“只是又一个小惊喜”，并指出移动应用程序可以集成到开始菜单和任务栏中。他们也将平铺或“窗口”作为操作系统新的应用放置 UI 的一部分。

图片来源:微软

这些应用将通过亚马逊应用商店的方式在微软商店出售。该公司强调抖音运行在一个演示的操作系统。该应用程序以垂直、纵向的方向呈现，正如你对移动优先设计的期望。

目前有 185 万个 Android 应用程序，这是一种向微软应用程序商店提供大量新内容的方式，并确保最新的热门移动应用程序突然在平台上可用。尽管时间会证明(基于英特尔桥构建的)体验最终有多好。

Windows 11 在这个假期到来。
