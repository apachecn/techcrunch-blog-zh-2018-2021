微软宣布 Windows 11 将于假期上市 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/24/microsoft-announces-windows-11-generally-available-by-the-holidays/amp/

图片来源:微软

经过数周的泄露和炒作，微软今天正式宣布了其桌面操作系统的下一个版本 Windows 11 。虽然该公司可能曾经说过 Windows 10 是 Windows 的最后一个版本，放弃了主要的发布点，而是定期进行两年一次的升级，但它显然认为这次更新中的变化——特别是重新设计的用户界面——值得一个新的版本号。

微软计划在假期向公众发布 Windows 11 ，所以我们大概可以在 11 月下旬左右期待它的到来。在此之前，我们可能会看到一系列的公开测试，从下周开始。这将是对 Windows 10 用户的免费更新。

如果你一直关注 Windows 10X 的发展和最终消亡，微软的操作系统为双屏和(最终)单屏笔记本电脑提供了简化的用户界面，你在这里看到的很多东西都会感到熟悉，包括重新设计的开始菜单。事实上，如果有人给你看 Windows 11 的截图和 Windows 10X 的早期预览，你会很难区分它们。

图片来源:微软

正如微软首席产品官 Panos Panay 在今天的公告中指出的那样，设计背后的总体想法是让你感觉“一种难以置信的平静感”，但与此同时，Windows 团队也努力让它变得更快。例如，Windows Updates 应该快 40 %,但 Panay 也指出，启动你的机器甚至浏览都应该感觉快得多。

图片来源:微软

一个令人惊讶的功能是，你现在也可以在 Windows 上运行 Android 应用程序了。举例来说，这些应用看起来就像可以集成到任务栏中的本地应用。这些应用将会出现在重新设计的微软商店里，通过——你可以坐下来看看——亚马逊应用商店。没想到会这样，是吧？

图片来源:微软

除了新的用户界面，它充分利用了半透明和阴影以及触摸屏用户的新功能，新的核心用户界面功能之一是微软所谓的 Snap Layouts，当你悬停在最大化窗口的图标上时，会弹出一个小部件，允许你将窗口移动到任何角落，这在以前需要将窗口拖到屏幕的角落(当你使用多个屏幕时，这通常很难)。

图片来源:微软

另一个主要的新功能是 Windows 11 将从一开始就内置团队。众所周知，微软看好团队。它最近推出了 Teams 的消费者版本，所以现在将其引入 Windows 11 也是有意义的。值得注意的是，微软从未将 Skype 引入 Windows，因此这是一个相当大的变化，但它基本上使团队微软的 Facetime。

图片来源:微软

如果你看到了 Windows 11 的漏洞，你就会知道网络小工具是更明显的新功能之一。Panay 说:“Windows widgets 是一个新的、个性化的 feed，由人工智能驱动，为你提供精选的内容。”当然，小工具并不是一个新东西，在很多方面，它们弥补了开始菜单中移除动态磁贴的不足。他们也将为开发者提供一个新的画布来展现他们应用程序中的信息。

图片来源:微软

当然，如果没有微软谈论游戏，这就不是一个新的 Windows。该公司辩称，Windows 11 将“提供迄今为止最好的 PC 游戏体验”，但他们还能说什么呢？

图片来源:微软

由于自动 HDR，微软承诺更好的图形，这是 Xbox 上已经可用的功能。微软表示，数千款游戏将在 Windows 11 上通过自动 HDR 得到自动增强。此外，该公司认为，由于 Windows 11 中新的存储 API，游戏将能够快速加载游戏资产，而不会拖累 CPU(但这需要兼容的 PC 才能做到)。哦，微软的游戏通行证订阅也将内置到 Windows 11 中。

图片来源:微软

正如所料，还将有一个新的微软商店，Panay 说它是为了速度而从头开始建造的(因为速度是人们抱怨的？).Panay 认为，微软希望帮助开发者将更多的应用程序带入商店。微软明显试图与苹果区别开来的一个领域是，它将允许开发者在他们的应用程序中使用自己的商业引擎，而微软不会从中收取任何费用。

图片来源:微软

事实上，微软首席执行官塞特亚·纳德拉表示，他希望开发者有机会在 Windows 11 上构建自己的平台。“Windows 是世界创造的舞台，”他说。“有了这个新版本的 Windows，我们正在释放我们每个人与生俱来的创新和创造力。[……]今天，世界需要一个更加开放的平台——一个允许应用成为独立平台的平台。Windows 是一个平台，比 Windows 更大的东西可以在这里诞生——比如网络。这是我们对 Windows 11 的期望。”
