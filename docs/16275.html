微软团队将直接内置于 Windows 11 中

Windows 11 日快乐。微软向我们展示了它即将在假期推出的操作系统 T1。在过去的一年里，我们绝大多数的人际交流都是通过电脑和手机屏幕进行的，现在，该公司正把它的通讯软件放在前台和中心位置。

Windows 11 将内置微软团队，旨在与苹果的 FaceTime 等通信平台进行更直接的竞争。和 FaceTime 一样，这里的关键是跨设备集成，随着人们从桌面移动到移动，再从移动移动回来，这使得服务更加与硬件无关。

图片来源:微软

在一个视频聊天平台多到不知如何是好的时代，我们很难摆脱这样一种感觉，即我们可能会看到曾经强大的 Skype 的最后一颗钉子，该公司早在 2011 年就以 85 亿美元的价格收购了 Skype。正如它对 Skype 所做的那样，微软希望通过这个平台模糊消费者和专业人士之间的界限。

新操作系统的所有版本都将团队直接加入到开始菜单中。
