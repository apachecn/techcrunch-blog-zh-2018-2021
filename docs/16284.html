Visa 以 20 多亿美元收购开放银行平台 Tink

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/24/visa-to-acquire-open-banking-platform-tink-for-more-than-2-billion/

Visa 宣布计划以 18 亿美元收购€的 Tink ，按照今天的汇率折合 21.5 亿美元。Tink 是欧洲领先的金融科技初创公司，专注于开放银行应用编程接口(API)。

几个月前，Visa 放弃了对另一家受欢迎的开放式银行初创公司 Plaid 的收购。最初，Visa 计划斥资 53 亿美元收购这家美国初创公司。但是在遭遇监管障碍后，该公司不得不取消收购。

Tink 提供了一个单一的 API，因此客户可以从他们自己的应用和服务连接到银行账户。例如，您可以利用 Tink 的 API 来访问帐户报表、启动支付、获取银行信息并定期刷新这些数据。

虽然由于欧盟支付服务指令 PSD2，银行和金融机构现在都必须提供开放的银行接口，但没有单一的标准。Tink 与 3，400 家银行和金融机构合作。

应用程序开发人员可以使用相同的 API 调用与各种金融机构的银行账户进行交互。正如您可能已经猜到的，它极大地简化了开放银行功能的采用。

300 家银行和金融科技初创公司使用 Tink 的 API 访问第三方银行信息——客户包括 PayPal、法国巴黎银行、美国运通和 Lydia。总体而言，Tink 覆盖了欧洲 2.5 亿银行客户。

总部设在瑞典斯德哥尔摩的 Tink 在收购后将继续照常运营。Visa 计划保留品牌和管理团队。

根据 Crunchbase 的数据，Tink 已经从 Dawn Capital、Eurazeo、HMI Capital、Insight Partners、PayPal Ventures、Creades、Heartcore Capital 和其他公司筹集了超过 3 亿美元的资金。

“在过去的 10 年里，我们坚持不懈地努力将 Tink 打造成为欧洲领先的开放银行平台，我们为 Tink 整个团队共同创造的一切感到无比自豪，”Tink 联合创始人兼首席执行官丹尼尔·杰伦在一份声明中说。“我们创造了一些不可思议的东西，但同时我们只是触及了表面。”

“加入 Visa 后，我们将能够比以往走得更快，覆盖面更广。Visa 是 Tink 下一阶段旅程的完美合作伙伴，我们对这将为我们的员工、客户和金融服务的未来带来什么感到无比兴奋。”
