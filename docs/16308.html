在 xCloud 今天的 iOS 发布会上，击中 iPhone 控制器主干 one 得分 Xbox 游戏通行证合作伙伴关系

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/28/hit-backbone-iphone-controller-scores-xbox-game-pass-partnership-at-xclouds-ios-launch-today/

在 xCloud 今天的 iOS 发布会上，击中 iPhone 控制器主干 one 得分 Xbox 游戏通行证合作伙伴关系

骨干一号，杀手 iPhone 游戏垫 I 去年年底在这里简介 ，刚刚拿下一个游戏配件的所有捆绑之母。随着 Xbox Game Pass Ultimate 今天在 iOS 上与 xCloud 游戏流媒体服务一起推出，它将与 Xbox Game Pass Ultimate 捆绑销售。

作为交易的一部分，Backbone 将与 Xbox Game Pass 捆绑在一个新的零售包装中，该包装获得了“专为 Xbox 设计”的标记，使 Backbone + iPhone 组合成为我们见过的最接近便携式 Xbox 的东西。

在我的上一篇文章中，我提到了 Backbone 如何巧妙地利用 iOS 内置的附件 API 为其跨游戏仪表盘提供超级流畅的功能。它一直在更新仪表板，以更好地向您展示游戏，向您展示新游戏，并让您使用其剪辑功能与您的朋友分享杀手级游戏。这是我多年来在 iOS 上看到的最好的游戏应用之一，它有很大的潜力为世界上最大的游戏观众创造一个跨宇宙的地方。

T2】

Backbone One 上的 Capture 按钮可与 Xbox Cloud Gaming 配合使用，并允许您将其作为链接进行分享。还有一个新的 Xbox 游戏 Pass feed，以及在同一界面中在 Xbox 和 iOS 游戏之间移动的方式。Xbox Remote Play 也是一个很大的问题，这个功能仍然没有得到充分利用，实际上在当代游戏机上非常好。

该包甚至利用 AppClip 为人们展示 AR 版本的 Backbone 运行 xCloud，发生在商店的零售包装上。

骨干团队继续以其巧妙的整合和坚实的本能给人留下深刻印象。该团队的游戏控制器谱系显示(一些原始 Xbox 360 控制器团队成员曾在主干上工作)但软件方面是最令人印象深刻的。Backbone 将 Warzone 或《我的世界》、Xbox 和 Playstation Remote Play 以及现在的原生 xCloud 游戏等 AAA iOS 端口的游戏体验统一起来的方式，感觉像是移动游戏的未来之路，包括苹果在内的所有玩家都没有成功做到这一点。

T2】

尽管 xCloud 游戏是基于网络的，但在 Backbone 出色的仪表盘中，它们被视为原生应用。我个人在 Backbone 上玩过很多 Stadia 上的 Destiny 2，感觉棒极了，我迫不及待地想让它像今天的 Xbox cloud 游戏一样，通过剪辑和社交功能更直接地集成到 dash 中。

发现、下载和玩游戏的体验是 比安装了主干网的 更好，因为现在团队越来越多地进入监管。他们还有一个非常灵活的链接机制，如果你在 feed 上看到他们的分享片段，你可以直接从 App Store 下载应用。这就是互联网原生玩家希望找到和玩新游戏的方式——一个持续超链接的游戏和流世界，使人们不必停下来手动搜索任何东西就可以看到、跟随和玩游戏。

这也让我渴望有一天发现游戏不仅仅是“下载”它们，而是进入一个世界，在那里我可以看到朋友分享的剪辑，点击它，玩一个让我着迷的关卡或比赛，然后再去购买游戏。对于新玩家来说，这将是一笔巨大的收购。

微软捆绑意味着某人从其网站 或今天在微软商店以 99 美元 购买主干可以获得 3 个月的 Game Pass Ultimate，考虑到 GPU 仅 45 美元，这对于新客户来说绝对是最好的交易。作为一家年轻的游戏初创公司，这对 Backbone 肯定是一个巨大的推动。
