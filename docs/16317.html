“移动应用商店”Tapcart 筹集了 5000 万美元的 B 轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/28/tapcart-a-shopify-for-mobile-apps-raises-a-50-million-series-b/

Shopify 改变了电子商务的格局，让商家更容易快速、经济地建立自己的网站。一家名为 Tapcart 的初创公司正在为移动商务做同样的事情。

该公司自称为“移动应用程序的 Shopify”，今天为顶级品牌提供购物应用程序，包括时尚新星、Pier One Imports、The bainings、Patta、Culture Kings 等。在一年的 3 倍收入增长(部分受疫情推动)后，Tapcart 今天宣布完成由 Left Lane Capital 领投的 5000 万美元 B 轮融资。Shopify 显然注意到了 Tapcart 对其自身商户群体的吸引力，因此也参与了这一轮融资。

本轮投资的其他投资者包括 SignalFire、Greycroft、Act One Ventures 和 Amplify LA。

Tapcart 的联合创始人新浪·莫巴斯和埃里克·内奇已经在移动应用行业工作多年。Mobasser 的前公司 TestMax 提供了 iOS 上的首批备考课程之一，而 Netsch 最近在代理方面工作，为品牌创造移动和数字体验。两人一起意识到了帮助在线商家将他们的业务转移到移动设备上的潜力，就像他们能够使用 Shopify 上网一样容易。

Tapcart 的创始人 Sina Mobasser 和 Eric Netsch 在他们的圣莫尼卡总部。图片来源: Tapcart

“现在，你可以在几周内就在我们的平台上推出一个应用程序，而在过去，如果你想定制一个应用程序，则需要长达一年的时间，”Mobasser 解释道。"而且你可以以很低的月租费做到这一点."

Tapcart 的平台本身提供了一个简单的拖放构建器，允许任何人使用工具为他们现有的 Shopify 商店创建一个移动应用程序，以设计他们的布局，定制产品详情页面，集成结账选项，包括产品评论，甚至可以选择添加其他品牌内容，如博客，lookbooks，视频(包括直播视频)等。一切都直接从 Shopify 实时同步到应用程序，因此商家的库存、产品和收藏都是最新的。这是与一些竞争对手的一大区别，后者需要重复的数据集和数据转换。

与此同时，Tapcart 利用 Shopify 的所有 API 和 SDK 来创建一个与 Shopify 现有数据结构兼容的原生应用程序。

图片来源: Tapcart

与 Shopify 的紧密集成有助于 Tapcart，因为它不必专注于电子商务基础设施，因为围绕库存和收藏的事情的结构在各个品牌之间大约 90%是相同的。相反，Tapcart 专注于让品牌脱颖而出的 10%，包括品牌、内容和设计等。其 CMS 允许商家创建独家内容，改变颜色和字体，添加视频等，使应用程序的外观和感觉完全定制化。

除了业务的移动应用程序创建方面，Tapcart 还帮助商家实现营销自动化。通过 Tapcart 平台，商家可以使用推送通知与他们的客户进行实时沟通，这些通知可以提醒他们新的销售，鼓励他们返回废弃的购物车或任何其他促销活动。营销活动也可以自动化，这有助于商家提前安排即将到来的发布和产品投放。该公司声称，由于其互动性和品牌效应，这些推送通知的点击率比传统的电子邮件或短信高出 72%。

图片来源: Tapcart

该平台很快吸引了中小型企业和中端市场企业客户，这些客户已经达到了他们的业务阶段，他们有必要加倍努力保持和转化客户，并优化他们的移动工作流程。

“我们的最佳时机是当你的数据库中有几百个客户时，”Netsch 指出。“这是一个绝佳的时机，现在不要太关注你业务中的付费收购部分，而要更多地关注如何留住和吸引现有客户，(这样他们就会)购物更多，体验更好，”他说。

在过去的 12 个月中，超过 12 亿美元的商家销售额通过 Tapcart 的平台流动。2020 年，Tapcart 的经常性收入增长了 3 倍，因为移动应用在疫情期间增长更快，这使得消费者的移动屏幕时间比 2019 年同比增长了 20%。该公司表示，移动商务支出也同比增长 55%，在假日购物季全球超过 530 亿美元。在此期间，Tapcart 自己的商家看到移动应用程序订单的速度超过每秒一次，它相信即使疫情即将结束，这些趋势也将继续。

今天，Tapcart 通过收取固定的 SaaS(软件即服务)费用来创收，这使它与许多竞争对手不同，竞争对手只收取商家总销售额的一定百分比。

图片来源: Tapcart

有了额外的资金，Tapcart 计划专注于成为垂直整合的移动商务工具套件的目标，最近还包括对 iOS 应用程序 Clips 的支持。它还将很快发布其 insights 分析平台的升级版本，并将提供脚本，商家可以将这些脚本安装在他们的移动网站上，以比较哪些在网站上有效，哪些在应用程序中有效。

今年晚些时候，Tapcart 计划推出一款全面的营销自动化产品，让品牌能够进一步自动化和个性化他们的通知。它还计划投资市场扩张，使其产品更好地适应移动和全球商务。

这笔资金将使总部位于圣莫尼卡的 Tapcart 在未来 24 个月内再雇佣 200 名员工，高于目前的 70 人。这些将包括跨时区的新加入，甚至在澳大利亚和欧洲等市场，因为它正在走向全球扩张。

Shopify 的投资也将带来许多新的机会，包括产品、工程、商业战略和合作伙伴关系。这也将有助于让 Tapcart 出现在 Shopify 的 170 万全球商家面前。

“仍有相当多的商家需要更好的移动体验，但尚未真正加倍努力，获得类似于原生应用的东西，”Netsch 指出。“有许多不同的方式和方法，商家正在尝试移动增长，我们试图在一个平台上提供所有最好的部分。因此，Tapcart 在现有的目标市场上有大量的扩张空间，”他说。

Shopify 产品加速副总裁萨蒂什·坎瓦尔(Satish Kanwar)在一份声明中表示:“我们相信，客户在哪里，品牌就在哪里，如今这意味着在他们的手机上。”“Tapcart 帮助商家创造顾客喜爱的移动优先购物体验，强化了 Shopify 的使命，即让商务对每个人都更好。我们期待看到 Tapcart 在 Shopify 上取得更大的成功，如今我们的平台上有超过 170 万商家。”
