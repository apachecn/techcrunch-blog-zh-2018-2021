Windows 11 的首个预览版现已发布 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/28/the-first-preview-of-windows-11-is-now-available/

微软今天向该公司 Windows Insider 项目的开发人员发布了 Windows 11 的首个预览版。如果你已经加入了 Insider 计划，并且满足了微软对新操作系统的新的——有点复杂的——系统要求，你应该很快就会看到更新。

这第一个预览版包括微软承诺的 Windows 11 的大部分新功能，包括新的外观和感觉，主题，小工具，新的快照布局和更新的文件浏览器。但是也有一些特性没有在第一版中出现。例如，对 Android 应用程序和新的内置团队集成的支持将在稍后的版本中推出，但新的 Windows 商店的预览版今天已经推出。

否则，举例来说，你可以尝试新的开始菜单(别担心，如果你不喜欢居中的外观，你可以将开始按钮移到左下角——但你不能将整个任务栏移到屏幕的另一边)。虽然开始菜单是 Windows 体验的标志性部分，但大多数高级用户可能从来不使用它，而是使用键盘或任务栏来启动 99%的应用程序。不过，微软正在尝试做一些不同的事情，它的新“推荐”部分突出了新安装的应用程序和最近使用的文件。

图片来源:微软

另一个你可能马上就会发现的新功能是新的文件浏览器，它现在取消了带状风格的菜单，取而代之的是更扁平的外观(微软称之为“命令栏”)和更现代的新图标。它看起来不错，但我们必须试一试，看看它是否在这个过程中失去了很多功能。

与 WIndows 11 中的其他应用一样，文件资源管理器也将支持微软的新捕捉布局，这种布局采用了 WIndows 中现有的“捕捉”手势或键盘快捷键，允许你将窗口捕捉到屏幕的任何一侧，并将其带到最大化按钮。虽然整体功能在这里并不是新的，但我很确定许多 Windows 用户从来不知道它的存在，所以这个新功能将向更多用户介绍窗口捕捉。

图片来源:微软

新的窗口小部件现在也突出显示在任务栏中。现在，有日历，天气，当地交通，微软待办事项和股票小部件，以及显示 OneDrive 和体育和电子竞技新闻(如果你喜欢的话)的最近照片的小部件。还有一个个性化的新闻源。

这里值得一提的最后一个新特性是新的设置菜单。自从命运多舛的 Windows 8 以来，Windows 本质上有两个设置菜单(控制面板和设置)。看起来那些令人困惑的日子还没有结束，但新的设置菜单至少看起来比 Windows 10 中现有的要干净得多。

图片来源:微软

当然，Windows 11 还有很多其他变化。这绝对不仅仅是又一次两年一次的 Windows 10 更新，只是在用户界面上做了一些小的改动。现在我们只需要看看所有这些在现实世界中是如何工作的——尽管要记住这仍然是一个早期版本。预览版现在正在向内部人士推出，所以我们可能会很快听到更多关于它如何表现的消息。我们还将在未来几天测试它的性能。
