联网汽车数据初创公司 Wejo 与微软、Palantir 和 Sompo 合作

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/29/connected-vehicle-data-startup-wejo-partners-with-microsoft-palantir-sompo/amp/

图片来源: Getty Images

联网汽车数据初创公司 Wejo 宣布与微软、Palantir 和 Sompo Holdings 合作，以提高其收集、存储和分析全球数百万辆联网汽车数据的能力。

此前，通用汽车支持的初创公司 宣布，它将通过与一家特殊目的收购公司【Virtuoso Acquisition corp .】合并而上市，该公司预计将于今年晚些时候关闭。来自微软和 Sompo 的 2500 万美元的承诺，加上已经承诺的投资者 GM 和 Palantir，使 Wejo 的管道融资总额达到 1.25 亿美元。

Palantir 曾是 Wejo 的战略投资者。2019 年，该软件开发商与保险提供商 Sompo 成立了一家日本合资公司。现在，这家合资企业与 Wejo 的合作将使这家初创公司有机会在日本，甚至是大亚太地区收集联网汽车数据。据 Wejo 创始人兼首席执行长理查德巴洛(Richard Barlow)说，该公司在韩国已经有一些实时车辆，但其 95%的数据来自美国。据该公司称，Sompo 将使用 Palantir Foundry 数据和分析平台 分析 Wejo 的联网车辆数据。

“现在全球销售的绝大多数汽车都有这种联网能力，所以这是一个巨大的机会，”巴洛告诉 TechCruch。“我们的平台上有 1100 万辆新车，而供应基数约为 5000 万辆。我们在该平台上与超过 17 家 OEM 合作，我们每天处理 160 亿个数据点，峰值约为每秒 414，000 个，这解释了为什么我们也很高兴得到微软的支持并迁移到他们的 Azure 云平台。”

Barlow 说，通过与通用、戴姆勒和现代等汽车制造商的合作，Wejo 可以看到 7%的车辆在纽约行驶，6%在加州行驶，20%在底特律行驶。该公司可以将原始的匿名数据——在车主同意的情况下从车辆上收集——交给企业、开发商或政府，或者为他们进行数据分析，这也是与微软合作可以派上用场的地方。

“微软提出了一个非常令人信服的解决方案，关于我们如何利用他们的机器学习和人工智能能力，为原始设备制造商和希望使用联网车辆数据的关键行业提供更令人难以置信的产品，”巴洛说。“因此，微软的 Azure 做了这些繁重的工作，真的会加快我们的业务。”

据 Wejo 称，最初的应用可能包括交通解决方案，以及远程诊断、集成支付、广告、零售和物流。两家公司还在讨论将 Wejo 用于微软地图解决方案的可能性。巴洛说，地图公司通常是 Wejo 数据的典型买家，并希望看到更多的保险提供商。

巴洛说:“我们已经看到了 1100 万次两辆车相撞的情况，我们正在实时从两辆车上获取数据。“所以我们开始先发制人，了解碰撞前后或车辆相互作用的特征或行为。”

Wejo 收集可以再现车祸的数据，从每个司机如何踩刹车，到哪个安全气囊展开，到撞击速度，以及哪个传感器被破坏。Barlow 说，然后它可以将这种数据分享给保险公司，以帮助加快索赔和恢复过程，并使维修更加准确。

所有这些证明人类在各种情况下的驾驶行为的数据都是在过去七年中收集的，这使得 Wejo 成为开发自动驾驶技术的公司的一个有吸引力的合作伙伴。

更正:本文的前一版本引用巴洛的话说，Wejo 每秒处理 40，000 个数据点，但准确的数字是 414，000。
