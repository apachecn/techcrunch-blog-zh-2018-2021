GitHub 预览新的人工智能工具，提供编码建议

GitHub 推出了一款新产品，它利用人工智能来帮助你更高效地编写代码。这款名为 GitHub Copilot 的新产品可以提示代码行，有时甚至可以提示整个功能。

GitHub 已经与 OpenAI 合作开发这个工具。它不会取代开发人员，它只是一个工具，应该可以提高生产率，让学习如何编码变得更容易。GitHub 把这个新工具框定为 AI pair 程序员。

GitHub Copilot 背后的模型已经接受了数十亿行代码的训练——其中许多代码都在 GitHub 上托管并公开提供。当你写代码的时候，GitHub Copilot 会在你打字的时候给出代码建议。您可以循环浏览建议，接受或拒绝它们。

为了弄清楚你正在编写什么，GitHub Copilot 试图解析一个注释的意思，你正在编写的函数的名字或者过去的几行。该公司在其网站的上展示了一些演示。

图片来源: GitHub

特别是，您可以在注释中用简单的英语描述一个函数，然后将它转换成实际的代码。如果您正在开始学习一门新的语言，或者您过去一直在使用无代码或低代码工具，这个特性可能会很有用。

如果你每天都在写代码，GitHub Copilot 可以用于一个新的框架或库。您不必从头到尾阅读文档，因为 GitHub Copilot 已经知道您正在使用的框架的具体功能和特性。它还可以取代许多堆栈溢出查询。

GitHub Copilot 直接与 Visual Studio 代码集成。你可以把它作为一个扩展来安装，或者通过 GitHub Codespaces 在云中使用它。随着时间的推移，服务应该会根据您与 GitHub Copilot 的交互方式进行改进。当你接受和拒绝建议时，这些建议会变得更好。

GitHub 目前是技术预览版，计划推出一款基于 GitHub Copilot 的商业产品。它目前与 Python、JavaScript、TypeScript、Ruby 和 Go 配合得最好。
