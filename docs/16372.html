脸书和马特波特合作开发人工智能的真实虚拟训练环境

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/30/facebook-and-matterport-collaborate-on-realistic-virtual-training-environments-for-ai/

为了训练机器人在房子里导航，你要么需要在很多真实的房子里给它很多真实的时间，要么在很多虚拟的房子里给它很多虚拟的时间。后者肯定是更好的选择，脸书和马特波特正在合作，为研究人员和他们贪婪的年轻人工智能提供数以千计的虚拟、互动的真实空间数字双胞胎。

在脸书方面，主要进步体现在两个方面:新的 Habitat 2.0 训练环境以及他们为实现这一环境而创建的数据集。你可能记得几年前的栖息地；在追求它所谓的“具体化的人工智能”，也就是说与现实世界互动的人工智能模型的过程中，脸书组装了许多逼真的虚拟环境供他们导航。

许多机器人和人工智能已经在理想化的、不现实的空间中学习了像运动和物体识别这样的事情，这些空间更像游戏而不是现实。真实世界的客厅和重建的客厅是完全不同的。通过学习在看起来像现实的东西中移动，人工智能的知识将更容易转移到现实世界的应用中，如家用机器人。

但最终这些环境只有多边形深度，只有最少的交互，没有真正的物理模拟——如果机器人撞到桌子，它不会摔倒，也不会把物品洒得到处都是。机器人可以去厨房，但它不能打开冰箱或从水槽里拿出东西。Habitat 2.0 和新的 ReplicaCAD 数据集改变了这种情况，增加了交互性和 3D 对象，而不是简单地解释 3D 表面。

在这些新的公寓规模的环境中，模拟机器人可以像以前一样滚动，但当它们到达一个物体时，它们实际上可以用它做一些事情。例如，如果机器人的任务是从餐桌上拿起一把叉子，并将其放入水槽中，几年前拿起和放下叉子只是假设，因为你无法实际有效地模拟它。在新的栖息地系统中，叉子是物理模拟的，它所在的桌子，它要去的水槽，等等。这使得它的计算量更大，但也更有用。

他们不是第一个到达这一阶段的人，但是整个领域正在快速前进，每次新系统出现时，它都会在某些方面超越其他系统，并指向下一个大瓶颈或机会。在这种情况下，Habitat 2.0 最接近的竞争对手可能是 AI2 的 ManipulaTHOR，它结合了房间尺度的环境和物理对象模拟。

Habitat 的优势在于速度:根据描述它的论文，模拟器的运行速度大约快 50-100 倍，这意味着机器人每秒钟可以进行更多的训练。(这种比较无论如何都不准确，而且这两个系统在其他方面也有所不同。)

用于它的数据集被称为 ReplicaCAD，它本质上是用定制的 3D 模型重新创建的原始房间级扫描。这是一个艰苦的手工过程，脸书承认，他们正在寻找扩大规模的方法，但它提供了一个非常有用的最终产品。

更多细节和更多类型的物理模拟正在路线图上——支持基本对象、运动和机器人存在，但在这一阶段，保真度必须为速度让路。

Matterport 还与脸书合作，采取了一些重大举措。在过去几年进行了巨大的平台扩展后，该公司已经收集了大量的 3D 扫描建筑。虽然它以前曾与研究人员合作过，但公司决定是时候向社区提供更大部分的宝藏了。

“我们已经研究了现存的每一种物理结构，或者接近它的结构。“公司首席执行官 RJ·皮特曼告诉我，住宅、高层建筑、医院、办公场所、游轮、喷气式飞机、塔可钟、麦当劳……所有包含在数字双胞胎中的信息对于研究都非常重要。“我们认为这肯定会对从计算机视觉到机器人到识别家用物品的所有领域产生影响。脸书不需要任何令人信服的…因为它就在球道的正中央。”

为此，它创建了一个数据集 HM3D，包含 1000 个精心 3D 捕捉的室内场景，从房地产浏览器可能识别的家庭扫描到商业和公共场所。这是此类收藏中最大的一个，被广泛出售。

图片来源: Matterport

这些环境由经过精确数字双胞胎训练的人工智能扫描和解释，在尺寸上精确到可以计算出窗户表面积或壁橱总体积的精确数字的程度。对于人工智能模型来说，这是一个有益的现实游戏场，虽然最终的数据集不是交互式的(还没有)，但它非常反映了真实世界的所有变化。(它不同于脸书互动数据集，但可以成为扩展的基础。)

“这是一个特别多样化的数据集，”皮特曼说。“我们希望确保我们有一个不同现实世界环境的丰富分组——如果你想从训练人工智能或机器人中获得最大收益，你就需要这种多样性的数据。”

所有的数据都是空间所有者自愿提供的，所以不要担心它被一些小字不道德地吸收了。Pittman 解释说，最终，该公司希望创建一个更大、参数化程度更高的数据集，可以通过 API 访问——基本上是作为服务的真实虚拟空间。

“也许你正在为美国某种风格的住宿和早餐制造一个待客机器人——如果能有一千个这样的机器人，那不是很好吗？”他沉思着。“我们希望看到我们能在第一个数据集上推进多远，获得这些知识，然后继续与研究社区和我们自己的开发人员合作，并从那里继续前进。这对我们来说是一个重要的起点。”

这两个数据集都是开放的，可供各地的研究人员使用。
