Noname Security 关闭 6000 万美元的 B 系列以消除 API 缺陷 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/30/noname-security-closes-60m-series-b-to-eliminate-api-flaws/

企业 API 安全初创公司 Noname Security 已经筹集了 6000 万美元的 B 轮融资，仅在六个月前完成了 2500 万美元的 a 轮融资

这轮融资由 Insight Partners 牵头，Next47、Forgepoint 和辛迪加集团(TSG)也参与其中，自 2020 年 12 月退出 stealth 以来，Noname 的总融资额达到 8500 万美元。

这家初创公司目前有 70 多名员工，在帕洛阿尔托和特拉维夫设有办公室，它表示，由于疫情助长了对 API 的日益依赖，它的融资速度很快。自然，API 的这种激增导致了 API 安全事件数量的增加。例如，今年早些时候， Experian API 暴露了几乎每个美国人的信用评分，几周后一个漏洞百出的 Peloton API 允许任何人直接从公司的服务器上获取用户的私人账户数据。脸书、 LinkedIn 、梯队和 Clubhouse 也成为了抓取攻击的受害者，这些攻击滥用访问 API 来获取他们平台上的用户数据。T15】

Noname 的首席执行官 Oz Golan 告诉 TechCrunch:“对 API 安全性的需求非常强烈，在疫情期间得到了极大的强调。”。"我们希望帮助组织安全地利用 API，我们希望消除所有的 API 漏洞。我们不希望再发生一起 Experian 事件。”

这家硅谷初创公司提供了一个整体安全平台，它使用人工智能和机器学习来使企业能够查看和保护由组织公开、由组织消费或在内部使用的托管和非托管 API，从而消除 API 安全盲点。根据 Noname 的说法，这些缺陷中的大多数通常会被忽视多年，这使得任何能够找到它们的人都可以不受约束地访问组织最敏感的操作。

“即使是经验丰富的安全专家也不知道他们的系统暴露了多少，”戈兰说。

自推出以来的 6 个月里，这家初创公司已经积累了 40 家技术、经销商和渠道合作伙伴，以及“数百家”正在生产或测试该平台的企业客户。

“由于我们已经看到了巨大的牵引力，我们希望加快速度——扩大我们的销售团队、营销团队、客户成功、研发。基本上是增长、增长、增长，”曾担任 NSO 集团工程总监的戈兰说。

在评论这轮融资时，Insight Partners 的负责人 Thomas Krane 说:“API 数量的激增和现代应用程序日益增加的复杂性导致了网络安全障碍的增加。最近在网络安全技能平台 Immersive Labs 中领导了 7500 万美元的 C 轮融资。Noname 在正确的时间带着完全实现的下一代技术进入市场，对全球客户产生了巨大影响。”

API 安全现在是投资者的热门话题。上个月，总部位于伦敦的 42Crunch 筹集了 1700 万美元的 A 轮融资，几周后，总部位于加州的 Salt Security 完成了 7000 万美元的 C 轮融资——使该公司去年筹集的资金总额达到 1 . 2 亿美元。
