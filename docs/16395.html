Slack 的新视频和语音工具正在改变工作的面貌

原文：https://web.archive.org/web/https://techcrunch.com/2021/06/30/slacks-new-video-and-voice-tools-are-nod-to-changing-face-of-work/amp/

图片来源: AlexSecret / Getty Images

Slack 去年年底开始谈论一套新的沟通工具来增强基于文本的渠道。今天，该公司发布了一款名为 Slack Huddles 的新音频工具，并提供了其他几款新工具的更多细节，包括可以留下视频信息和增强的员工名录，你可以从 Slack 内部访问这些信息。所有这些似乎都是考虑到工作性质的变化而设计的。

让我们从 Slack Huddles 开始，这款音频工具可以让你与 Slack 中的人进行实时对话，而不是将你所有的想法都打出来。对于觉得打字有挑战性的人来说，这将容易得多，但该公司也认为这将允许更多的自发讨论，至少在某种程度上模拟在办公室里。

“Huddles 是一种轻量级的，音频优先的交流方式。Slack 的首席产品官 Tamar Yehoshua 在昨天的新闻发布会上解释说:“它重现了在预定会议之外发生的自发和偶然的互动。

随着公司继续引进更灵活的工作模式，他们将不得不调整他们的工作方式。Slack 首席执行官斯图尔特·巴特菲尔德说，秘密会议是思考这个问题的一种方式。

“有些事情可以同步，但只需要三分钟。巴特菲尔德说:“下周二 11:30 到 12 点(安排会议)和整个半小时(因为这是我们安排的)，现在是两到三分钟，如果谈话在私下进行中失败了，你就让它开着，也许有人稍后会加入并说些什么，这是你在电话中不会做的。”

认识到不是每个人都能听到，新工具包括实时转录。

自去年以来，该公司也一直在谈论提供某种视频信息功能。这个想法几乎就像一个视频语音邮件或 Instagram 故事，你可以拍摄一个短视频并发布在 Slack 上。“我们一直在思考这个问题，我们相信，通过给人们一种表达和异步共享和消费信息的方式，我们可以让人们更加灵活地工作，并减少视频会议的需求，”耶霍舒亚说。

这项新功能将使 Slack 用户能够在 Slack 中播放视频、语音和屏幕记录。她解释说，人们可以录制短片并上传到频道或 DM 中，“使其他人能够按照自己的时间表观看和回应”。虽然这个功能还没有准备好发布，但 Yehoshua 报告说它正在试点，并将在未来几个月的某个时间提供给付费团队。

最后一篇文章是基于去年 Slack 收购 Rimeto 的文章，目的是提升他们的公司目录。事实上，Rimeto 产品已经被重新定位为 Slack Atlas，这是一个公司目录，用户可以直接在 Slack 中访问，而不必转到另一个程序来查找这些信息。这是 Slack 的另一种方式，可以让用户在 Slack 中找到他们需要的信息，同时避免上下文切换。据该公司称，这种服务目前只提供给有限的 Business+和 Enterprise Grid 客户，但今年晚些时候将会更广泛地提供。

Slack 在去年首次宣布了这些工具，最初说它们是实验性的，但很快就将其转移到了产品路线图上。巴特菲尔德在三月份与前 TechCrunch 记者 Josh Constine 一起出现在一个俱乐部采访中，Josh Constine 现在是 SignalFire 的投资者，表面上是谈论工作的未来，但他也第一次谈到了这些工具的更多细节。

很难不将这一讨论纳入工作的未来，事实上 Slack 作为 Salesforce 的一部分，去年以 270 亿美元收购了这家通讯工具。工作在变化，无论未来如何，Slack 都希望成为这个解决方案的一部分。
