显然，Twitter 现在正在制作 NFT

不，今天不是愚人节。但是 Twitter 的整个账户已经被 NFTs 接管。正如它的新标题所写的:“我已经停止保湿，因为关于 NFTs 的推特让我保持年轻。”

如果这是真的，推特鸟在一天结束时看起来会像一只小鸭子。今天早上，Twitter 发布了“140 个免费的 NFT 给 140 个你，闺蜜们”，这符合 Twitter 最初的 140 个字符的限制。

回想起来，在 Twitter 联合创始人兼首席执行官杰克·多西今年早些时候通过一个名为贵重物品中心的网站向出售他作为 NFT 的第一条推文后，我们应该已经预料到了这一点，该网站与 Twitter 无关，但给人们提供了制作和出售推文 NFT 的选项。就连一直在投资 NFT 空间的马克·库班也以 0.56 ETH 的价格(当时为 953 美元)出售了一条关于贵重物品的推文。多尔西的 NFT 以 1630 ETH 的价格售出，当时价值约 290 万美元。他将收益捐给了 GiveDirectly，以帮助非洲应对新冠肺炎病毒，但 NFT 仍然经常因其负面的环境影响而受到批评——其他人担心 NFT 市场可能只是昙花一现。但是随着像 Twitter 这样的主要平台制作 NFT，这种时尚可能还没有结束——或者，Twitter 可能正在试水，以衡量其用户群对即兴的免费 NFT 下降的反应，然后在该领域扩展其产品。

图片来源:推特

今天推特 NFT 下降的问题是没有人可以购买它们——推特用户可以回复推特，有机会获得 7 个 NFT 中的一个，这些 NFT 是在 NFT 市场 Rarible 上制作的，共 20 个版本，总共 140 个 NFT。

据 Twitter 称，该平台上关于 NFTs 的推文已经超过 2900 万条。Twitter 上 NFTs 的下一步是什么？我们不确定。但随着 Twitter 快速开发新功能，如 Spaces、Super Follows、Twitter Blue 等，这可能不会是我们最后一次听到 Twitter 涉足加密收藏品。
