最新的亲特朗普社交网络 Gettr 已经一团糟 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/06/gettr-trump-social-network-hack-defaced/

嗯，真快。就在前特朗普发言人杰森·米勒的推特克隆发布几天后，这个新的社交网络已经被问题困扰。

首先，黑客迅速利用 Gettr 的 API 获取了超过 85，000 名用户的电子邮件地址。网络安全公司 Hudson Rock 的联合创始人阿龙·加尔(Alon Gal)披露了用户名、姓名和生日等数据。

Gal 告诉 TechCrunch:“当威胁行为者由于疏忽的 API 实施而能够提取敏感信息时，其后果相当于数据泄露，应该由公司进行相应的处理，并由监管机构进行审查。”

上周，TechCrunch 的扎克·惠特克预测【Gettr 将很快看到它的数据通过它的 API 被抓取。

拼凑的数据只是 Gettr 头疼的问题之一。该应用程序实际上于上个月在 App Store 和 Google Play 上线，但在政治发布后于 7 月 4 日退出测试。虽然该应用旨在吸引众所周知的反华特朗普圈子，但Gettr 显然获得了中国亿万富翁郭文贵的早期资金，郭文贵是特朗普前顾问史蒂夫·班农的盟友。今年早些时候,《华盛顿邮报》报道称，郭是一个大规模网上虚假信息网络的核心人物，该网络散布反疫苗言论和 QAnon 阴谋。

7 月 2 日，该应用的团队以下载量激增为由为注册延迟道歉，但一点点启动停机时间可能是最不严重的问题。上周末，包括马乔里·泰勒-格林、史蒂夫·班农和米勒自己的账户在内的许多 Gettr 官方账户遭到泄露，引发了更多关于该应用粗制滥造的安全做法的问题。

撇开这件事不谈，虚假账户压倒了任何在 Gettr 上寻找真实用户的努力。这也适用于应用程序自己的推荐:在 TechCrunch 的测试中，Steam 的一个虚假品牌账户是应用程序自己的推荐之一。

另一个危险信号:这款应用的设计明显与 Twitter 相同，似乎使用了该公司的 API 来复制一些用户的粉丝数量和个人资料。Gettr 鼓励新用户在注册过程中使用他们的 Twitter 句柄，称这将允许在某些情况下复制推文(我们注册了，但这对我们不起作用)。TechCrunch 就 Gettr 惊人的相似性和其 API 的使用联系了 Twitter，但该公司拒绝置评。

在手机上，Gettr 基本上是 Twitter 的一个精确克隆——尽管它的边缘非常粗糙。Gettr 的一些内容有些生硬和奇怪，包括吹嘘它是一个“无偏见”的社交网络，“尽最大努力向用户提供最好的软件质量，允许任何人自由表达他们的观点。”

该公司将自己定位为任何认为主流社交网络敌视极右思想的人的替代选择。Gettr 的网站用熟悉的特朗普式信息吸引新用户:“不要被取消。灵活运用你的第一修正案。庆祝自由。”

“羟基氯喹管用！”米勒分享(Gettr'd？)周末，引用前总统的话。“没有人会撤下这个帖子或暂停这个帐户！#GETTR。”到目前为止，Gettr 上的内容审核要么很松散，要么根本不存在。但是正如我们在 Parler 和其他暴力阴谋的避风港所看到的，这种方法只能持续这么久。

尽管通过米勒和前特朗普竞选团队工作人员蒂姆·墨陶(Tim Murtaugh)与特朗普有着广泛的联系，但这位前总统还没有出现在该应用程序上。特朗普圈子里的一些人已经在 Gettr 上建立了个人资料，包括史蒂夫·班农(8.47 万粉丝)和迈克·蓬佩奥(130 万粉丝)，但搜索特朗普只会出现非官方账户。彭博报道川普没有加入该应用的计划。(鉴于 Gettr 的占优势的刺猬索尼克色情，我们不能完全责怪他。)

很难说这款应用的技术问题或特朗普的缺席是否会降低对 Gettr 的兴趣。根据来自 Sensor Tower 的估计，自 6 月份以来，Gettr 已经在全球范围内获得了大约 130 万次安装，巴西紧随美国成为该应用的第二大市场。

2021 年年中，在线亲特朗普生态系统仍然分散。随着 Trump 被禁，QAnon 周围混乱的阴谋网络在脸书和 Twitter 上不再受欢迎，Gettr 将自己定位为主流社交媒体许多被抛弃者的避难所。但是考虑到 Gettr 早期越来越多的问题，这个粗略的 Twitter 克隆体的阳光时刻可能已经结束了。
