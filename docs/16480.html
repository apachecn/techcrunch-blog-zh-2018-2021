Cloverly 获得 210 万美元的种子资金，继续开发 API 来测量和抵消碳的使用

总部位于亚特兰大的初创公司 Cloverly 开发了一种 API，可以帮助企业测量并抵消他们的碳排放。今天，该公司宣布了一轮 210 万美元的种子融资。

TechSquare Ventures 领投，软银机会基金(SoftBank Opportunity Fund)和全景风险投资公司(Panoramic Ventures)以及昼夜风险投资公司(circadian ventures)、克诺尔风险投资公司(Knoll Ventures)和 ventures . SaaS 公司(SaaS Japan)也参与了该轮投资

在此期间，该公司宣布创始人安东尼·奥尼(Anthony Oni)已经退出公司的日常运营，但仍将作为顾问留在董事会。该公司已聘请前易贝高管杰森·鲁布托姆接替他担任首席执行官。

“我们是一家可持续发展服务公司，帮助其他公司测量和减少碳足迹。我们的 API 测量企业内部各种活动或流程的碳排放，并允许企业或其客户抵消这些排放。然后提供全面的报道，”鲁布托姆解释道。

为该公司运营的 Rudy Krehbiel 表示，该 API 旨在灵活满足每个公司访问服务的需求，但一旦开发人员创建了一个应用程序，它就会自动测量排放量并购买补偿。“解决方案本身是自动化的。大部分工作都是预先进行的，一旦我们整合起来，它就变成了一个完全产品化和可操作化的持续测量和补偿解决方案，”他说。

当客户使用该工具构建解决方案时，他们可以通过从公共市场购买碳补偿来抵消他们的碳使用量，这可以根据给定公司的使用量自动进行。Cloverly 监测抵消市场，以确保来源是可信的，并增加新的，因为他们的发展。

该公司正在与 600 多个品牌合作，到目前为止已经抵消了超过 5500 万磅的碳。这个 API 最初是奥尼在南方公司工作时构思的，并在 2019 年地球日作为一家初创公司推出。

Oni 是一名黑人，随着他将接力棒交给 Rubottom，他正在脱离日常运营，但他从多元化的角度认识到了这笔资金的重要性。

“作为一家气候科技公司的黑科技创始人，拥有 TechSquare Ventures 和软银的机会基金作为投资者令人难以置信。这将需要不同的人和团队来寻找解决方案，以创造一个更可持续的未来，”他说。
