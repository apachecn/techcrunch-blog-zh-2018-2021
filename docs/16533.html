Circle 是一个很好的例子，说明 SPACs 为什么有用

随着比特币基地今年早些时候的直接上市，其他加密公司可能会寻求尽快上市。总部位于波士顿的技术公司 Circle 似乎就是这种情况，该公司提供 API 交付的金融服务和稳定的货币。

该交易所探索初创公司、市场和资金。

每天早上在 Extra Crunch 阅读或者每周六获取交易所简讯。T15】

Circle 不会直接上市，也不会寻求传统的 IPO。相反，该公司正在与特殊目的收购公司的一家空白支票公司 Concord Acquisition Corp .合并。该交易对 crypto shop 的企业价值估值为 45 亿美元，股权价值约为 54 亿美元。

这次发行标志着加密市场的一个有趣时刻。与比特币基地不同，Circle 运营一个交易平台，并以公开市场投资者普遍理解的方式收取费用，Circle 的产品更具异国情调。

Circle 的特殊目的收购公司演讲详细介绍了一家公司，其核心业务涉及一种稳定货币(stable coin)——一种与外部货币挂钩的加密资产，在这里是美元——以及一套为其他公司提供加密金融服务的 API。它还拥有股票众筹平台 SeedInvest，尽管 Circle 的大部分预期收入似乎来自其他业务。

关于交易本身的更多信息，TechCrunch 的 Romain Dillet 有一篇关于交易的文章。在这里，我们将深入研究该公司的投资者演示，谈论其商业模式，并重复其历史和预期的结果和估值倍数。

近几个季度，稳定的硬币变得流行起来。因为它们与外部货币挂钩，所以在加密世界里，它们是一种有趣的现金形式。如果你想拥有网上购买力，但又不想把所有的价值都储存在波动性更大、税收更高的加密货币中，而你可能不得不出售这些加密货币来购买其他任何东西，那么稳定的硬币可以作为一种更稳定的流动货币。比如说，他们可以将美元的稳定性和加密世界有趣的金融网络结合起来。
