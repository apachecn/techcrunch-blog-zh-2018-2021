Didomi 筹集 4000 万美元帮助您管理客户同意 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/09/didomi-raises-40-million-to-help-you-manage-customer-consent/

法国初创公司 Didomi 已经筹集了由大象和 Breega 领投的 4000 万美元 B 轮融资。该公司为网络出版商和应用开发商管理同意流程。Didomi 在欧洲已经做得很好，每月有数十亿次同意互动——它计划在今天的融资轮中扩展到美国。

“贾瓦德、拉斐尔和我共同创立了 Didomi，让隐私对每个人来说都更容易，也是公司的一个显而易见的选择。联合创始人兼首席执行官 Romain Gauthier 在一份声明中说:“这次筹款是我们实现这一使命的旅程中的一个重要里程碑。

“我们期待通过愉快的同意和偏好管理体验，帮助品牌和出版商使客户旅程更加透明和可信，”他补充说。

近年来，许多监管机构实施了新的注重隐私的框架。你可能会想到欧洲的 GDPR。

如果你生活在受这些变化影响的国家，你现在很清楚，每当你第一次访问一个新网站或打开一个应用程序时，你都会看到一个同意弹出窗口或横幅。

我不会说这些弹出窗口是“令人愉快的”，因为最好的同意弹出窗口是不存在的，因为你访问的网站不会收集和共享个人数据。但这并不总是可能的，你可能需要收集数据有不同的原因——包括在当前的网站 techcrunch dot com 上。

在这种情况下，像 Didomi 这样的产品真的很有帮助。认真对待这些同意流程是非常重要的，因为您不想搞砸实现并被罚款。Didomi 是一个以开发者为中心的同意平台，可以在许多不同的设备上运行。您可以为桌面网站、移动网站、移动应用程序或连接的电视配置您的同意流程。

有一个统一的解决方案也意味着你不必一遍又一遍地请求许可。Didomi 可以跨设备存储和同步偏好。一切都是可审计的，以防监管者想知道你是如何收集同意的。

随着今天的融资，该公司希望通过开放 API 和开源 SDK 使其产品对开发者更加友好。这并不意味着 Didomi 适合所有人，因为该公司特别关注高端客户。客户包括乐天、Orange、Giphy 和慧俪轻体国际。

该公司还将为不同的市场雇佣更多的本地营销和销售团队人员。Didomi 计划在德国、西班牙和美国开设办事处。

与此同时，形势也在迅速变化。网络浏览器正逐渐屏蔽第三方追踪器，苹果现在甚至会问你一个应用程序是否能在操作系统层面追踪你。看看 Didomi 如何随着用户的期望而发展将会很有趣。
