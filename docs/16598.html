MSCHF 丢弃你最喜欢的失败的启动硬件的微小的行动数字

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/12/mschf-drops-vinyl-action-figures-of-your-favorite-failed-startup-hardware/

硬件硬。你可以浏览这个网站的档案，找到几十个大胆的尝试，让新的消费电子产品发挥作用——其中一些 非常接近家 。但是，像所有创业公司一样，大多数硬件公司都遇到了将原子转化为值得购买的东西的核心难题。

为了纪念硬件的硬度，创意工厂/艺术公司 MSCHF 正在发布一套五个 死亡的创业玩具，作为乙烯基公仔 ，你可以以每个 39.99 美元或一套 159.99 美元的价格购买。它将这些称为“标志性的失败创业公司”，销售网站提供了每项努力的兴衰简史。它们包括从未真正存在过的产品，如 Theranos minilab，Jibo 等不合时宜的早期产品，以及 Juicero 等过度工程的练习。

鉴于我职业生涯的大部分时间都在研究并试图理解将消费类硬件推向市场的艰难而复杂的过程，我喜欢这些东西。这里可能有恶意的镜头，但我选择不这样看。欺诈就是欺诈，Theranos 和最酷的 Cooler 等崩溃背后的人已经或将看到法律体系的商业终结。

但是宏伟的愿景和硬件梦想并不总是那么明显地落入“失败”的陷阱有时硬件可以，但供应链不行。有时候，愿景是合理的，但产品还为时过早。产品失败的原因有很多——但是(因为它们实际上是真实的),你经常不得不放弃它，因为一群人和梦想家想要一个东西在宇宙中存在，并把它拖到那个地步。离开悬崖。

图片来源: MSCHF

这些数字本身做得真的很好，清晰的印记和精确的细节，可读的文字和印刷精美的标志。他们中的一些也是清晰的，并配有配饰。最酷的冷藏箱配有臭名昭著的搅拌机，而 Juicero 有一个可拆卸的(当然是专有的)“新鲜蔬菜”袋。总体而言，这些产品的质量相当高；我会将它们与我多年来购买的一些更好的新奇玩具进行排序——它没有打电话，很像冷却器的功能集。

包装也令人印象深刻；每个人都有一个定制的盒子，所有这些都放在一个更大的机架盒中。每一封信的背面都有“死亡原因”，告诉你每一次冒险失败的原因。MSCHF 不遗余力地使这成为一个非常优质的“玩具”下降，这是唯一合适的，因为它是一个实物产品的纪念碑。

与 MSCHF 的许多工作一样，有一个“等等，这合法吗？”此外，因为与这些产品相关的 IP 可能会落入许多漏洞，但其中一些漏洞可能仍然有法律实体。但是，正是这种危险因素使得它的许多项目至今仍能引起共鸣。所以我觉得他们不担心。

毕竟这些产品上没有一个有 的兽首 。总之，身体上。(图片来源: MSCHF)
