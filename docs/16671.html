微软推出 Windows 365

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/14/microsoft-launches-windows-365/amp/

图片来源:微软

微软今天发布了 Windows 365 ，这项服务让企业可以选择轻松地让他们的员工从云端访问 Windows 10 桌面(一旦 Windows 11 全面上市，Windows 11 也会随之推出)。想想游戏流，但你的桌面。它将于 2021 年 8 月 2 日面向商业用户(且仅限商业用户)推出。

通过一个有点难以理解的新闻稿宣布，Windows 365 被期待已久，实际上只是现有远程桌面服务的一个演变。

但是，嘿，你可能会说，微软不是已经提供了 Azure 虚拟桌面让企业可以选择让他们的员工在云端访问 Windows PC 吗？是的，但区别似乎是 Windows 365 更容易使用，并且不涉及在 Azure 云中设置完整的 Azure 虚拟桌面环境的复杂性。

但微软难道不能让 Azure 虚拟桌面更容易使用，而不是推出另一种虚拟桌面服务吗？是的，但 Azure 虚拟桌面在很大程度上是一种企业服务，默认情况下，这意味着它必须与公司现有基础设施的其余复杂性很好地配合。疫情将它应用于小公司，因为他们几乎没有选择，但从很多方面来说，今天的发布是微软承认对他们来说太难管理了。另一方面，Windows 365 有点像一块新的石板。它也可以通过基本的订阅服务获得。

该公司表示:“微软还将继续在 Azure 虚拟桌面方面进行创新，为那些拥有丰富虚拟化经验、希望获得更多定制和灵活性选项的组织提供服务。”。至少我们现在知道公司为什么要把 Windows 虚拟桌面改名为 Azure 虚拟桌面了。那会变得很混乱。

图片来源:微软

这也给了微软谈论“一种新的混合个人计算类别”的机会，其首席执行官塞特亚·纳德拉称之为“云个人电脑”还不清楚这到底是什么，但这是一个新的类别。

纳德拉在今天的新闻稿中解释道:“就像 SaaS 将应用程序带到云中一样，我们现在将操作系统带到云中，为组织提供更大的灵活性和安全的方式，使他们的工作人员无论身在何处都能够更高效地工作和保持联系。”。

但是这不就是瘦客户机吗？也许吧？但我们在这里谈论的不是硬件。它实际上只是云中的一个虚拟化操作系统，您可以从任何地方访问它，这一类别已经存在很长时间了。

“混合工作从根本上改变了技术在当今组织中的角色，”微软 365 公司副总裁 Jared Spataro 说。“随着工作人员比以往任何时候都更加分散，组织需要一种新的方式来提供出色的工作体验，同时增强通用性、简洁性和安全性。云电脑是一个令人兴奋的混合个人计算新类别，它可以将任何设备转变为个性化、高效和安全的数字工作空间。今天发布的 Windows 365 只是一个开始，随着我们模糊设备和云之间的界限，这一切都将成为可能。”
