Nooks 将在虚拟总部 TechCrunch 的世界中播种自己的种子

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/14/nooks-set-to-seed-its-own-in-the-world-of-virtual-hqs/

在测试运行一年后， Nooks，一个面向分布式团队的虚拟工作空间，已经吸引了成千上万的用户和数百万美元的风险投资。这家由斯坦福学生领导的新贵已经筹集了由 Tola Capital 牵头的 500 万美元种子轮，参与投资者包括 Floodgate 和 Julia 和 Kevin Hartz(分别是 Eventbrite 的首席执行官和董事长)以及 Awesome People Ventures 的创始人 Julia Lipton。

这笔融资标志着又一批投资者押注虚拟总部世界中的一家公司，这一群体包括数十家初创公司，他们认为分散的员工已经准备好从 Zoom 毕业，进入以生产力和游戏化为理念的“元游戏”。这是凯文·哈兹对虚拟总部的第二次投资，第一次是在集会。截至今天，红杉资本、安德森·霍洛维茨、门罗、Battery Ventures、Index Ventures、Y Combinator、Homebrew 和 Floodgate 都在不同的虚拟总部创业公司中持有股份。

换句话说:即使投资者投入大量资金，Nooks 也有自己的工作要做。

Nooks 于 2020 年 5 月由斯坦福大学的学生李仁港、罗翰·苏里、尼基尔·切拉和伦斯勒理工学院的安德鲁·屈共同推出。就像几乎所有其他意外投入远程工作的人一样，这三个人在学校和课堂上经历了极度疲劳。他们很快意识到，有必要为高绩效团队和志同道合的群体营造一个共享工作的空间。

两位联合创始人首先在斯坦福进行了 Nooks 试点，将其提供给助教，作为夏季虚拟课程的吸引层。Lee 说，Nooks 最初的用例看起来像是办公时间和家庭作业聚会。自从最初在学校试点以来，Nooks 越来越专注于帮助分布式团队工作，但这种风气一直保持不变。

“应该有一个持久的空间，而不仅仅是在一个像会议这样短暂的空间里，你可以去某个地方创造更多自发的联系，”李说。

角钩

当用户进入 Nooks 时，他们会看到一个松弛的界面。然而，员工被邀请进入“空间”，而不是左边的频道面板每个空间都有不同的用途，从前台模型到海滩聚会或设计聚会。Nooks 有一个专门打击代码中出现的 bug 的空间。在首次进入该平台时，UX 的 Nooks 与其他竞争对手有所不同。虽然 Branch 和 Gather 这样的公司看起来像是具有生产力元素的视频游戏，但 Nooks 完全跳过了化身的感觉，看起来更接近于 Teamflow 或 Tandem。该公司使用一个视频 API 让每个人占用小空间，并增加了谷歌文档、YouTube、Asana 或 GitHub 等平台的集成。

联合创始人苏里说，公司决定追求更简单的美学来促进对话，而不是更多的点击。

“实际上，我们并不认为，为了与某人交谈，你需要成为一个视频游戏玩家，带着你的虚拟形象走到他们面前，”他说。“应该就像在一个房间里看到他们，然后进入那个房间一样简单。”

图片来源:角落

当然，该公司正在努力平衡这种简单性与迷人的环境，包括空间和背景音乐的定制。有一个“耳语功能”，允许同行在演示过程中相互交谈，虚拟销售大厅，Nooks 创建了顶级卖家排行榜和共同工作空间，以促进思想的交叉传播。

简单可能会以自发性为代价。虽然其他虚拟总部平台使用空间音频来创造一种“碰撞”的感觉——当你靠近其他同事时，声音会变大，当你离开时，声音会变小——但 Lee 说，Nooks 总是通过让它“只需点击一下就可以与任何人交谈”，来促进即兴协作和随意对话

虽然无摩擦通信是一个重要的功能，但它不能成为 Nooks 的唯一钩子。像 Slack、Hangouts 甚至 Twitter DMs 这样的平台只需要一次点击(最多两次！)供用户与某人通信。更不用说 Slack 正在发布一系列围绕自发性和现场沟通的沟通工具。

尽管如此，Nooks 目前每周仍有数千名活跃用户，他们来自斯坦福、Embroker 和 Workato 等团队和组织。该公司表示，使用 Nooks 的团队平均每天在平台上花费 6 个小时。

混合工作的虚拟成长的烦恼

随着疫情在世界部分地区的消退，像 Nooks 这样的初创公司需要想办法适应混合团队在长期从事远程工作后的回归。这些创业公司面临的新挑战是如何定位自己，以融入新的工作文化。

邻近偏见可能会使这一点难以实现。

邻近偏见是指亲自去上班的员工比虚拟上班的员工更受重视。这是让混合工作如此难以大规模实现的现实之一:当一群员工仅仅因为可以去办公室而被定位为更重要或更受重视时，公平就会受到损害。

虚拟工作空间初创公司，尤其是那些希望将工作文化带到网上的公司，可能会意外地将在家工作的人与面对面工作的人过度分割开来。分裂将极大地影响历史上被忽视的个人，包括少数民族和妇女。值得注意的是，在现有的虚拟总部中，大多数都是由男性建造、运营和资助的。

当被问及他们如何对抗邻近偏见时，Lee 说“与远程员工更频繁、流畅和随意的交谈有助于他们与团队其他成员建立更强的联系。”果然，有一种观点认为，许多虚拟总部初创公司通过将办公室里的每个人带到同一个数字世界来消除邻近偏见。

最终，公平竞争需要积极的意图。一家初创公司如何确保虚拟总部的员工能够在会议室 A 自发地、面对面地站起来？一个平台如何让任何员工，无论身在何处，都有机会发表意见、提出异议或分享会后笑话？除了鼓掌或竖起大拇指之外，虚拟形象或浮动视频球能开始给出微妙的身体暗示吗？

我敢打赌，这些功能是虚拟总部创业公司长期的登月计划和生存黑客。
