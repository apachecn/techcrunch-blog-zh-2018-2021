Butlr Technologies 开发匿名人员传感器，投入 790 万美元种子资金 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/15/butlr-technologies-developing-anonymous-people-sensors-inks-7-9m-seed-round/

一轮新的 790 万美元的种子期增强了 Butlr Technologies 的能力，使其能够将其实时人体感应技术应用于商业房地产和零售用途之外，以监测正在就地老化的活跃老年人的跌倒和其他运动。

Hyperplane 领投，Founder Collective、Union Labs、500 Startups、SOSV、E14 Fund、structural Ventures、Scott Belsky、Chad Laurans 和 Sunny Vu 参与。

一年前，总部位于加州伯林盖姆的 proptech 公司以可转换票据的形式筹集了 120 万美元，这笔资金包含在 790 万美元中。它正在开发一个平台和热传感器，可以匿名检测某人的体温，以确定入住率、人数和活动。

联合创始人洪灏·邓和贾妮·曾在 2019 年将 Butlr 从麻省理工学院媒体实验室分离出来，他们将自己的技术称为“空间的 Alexa”他们的无线传感器每秒捕捉五帧体温数据——姿势、睡眠质量和温度——然后将这些数据转化为洞察力，客户可以利用这些洞察力做出任何决策，从建造建筑物到规划人们如何在医生空间共存。

邓告诉 TechCrunch，该公司是“唯一一家将准确性与无线结合起来的公司。”传感器的无线方面可以放在你的手掌中，这意味着它们可以放在任何地方，需要一次性设置，电池可以使用两年。Butlr 也是为了保护隐私，所以重播时不会显示人脸，而是显示人的“热量”在移动。

“这些传感器可以在任何环境下使用，”邓说。“它应该是无形的，人们不必担心。我们可以将尊严带回科技世界，同时也理解用户的需求。这是关于人们以前不能做的新事情，如何以一种易于部署的方式实时使用空间。"

基本套件起价 880 美元，可以从该公司的网站上购买，包括空间设计软件，人们可以使用该软件轻松确定在哪里放置传感器以实现最大覆盖范围。

Butlr 技术公司的运动和探测模拟。图片来源: Butlr 科技公司

这笔资金被指定用于持续的市场扩张和技术开发，包括更先进的 API，以满足对空间智能的爆炸性需求——该公司正在发运数十万个传感器，去年收入增长了近 10 倍。邓预计，随着各公司在疫情事件后制定复工政策，这种情况将会继续。

虽然 Butlr 最初是从房地产和零售商开始的，但邓看到了其他用例的潜力，如汽车交通流量和辅助生活设施，其中可以在事故(如摔倒)发生前监测早期诊断和认知功能。

“辅助生活市场是巨大的，”邓补充说。“监控带来的最重要的事情之一是了解一个人在哪里。我们可以提前预测它们的脆弱程度。”

例如，通过能够实时传输热信号，Butlr 能够更好地检测刚性运动或看起来像摔倒的行为。这与现有技术不同，邓说，现有技术在传输图像时会“错过几分钟”。

现任者还依赖可穿戴设备或健康腕带，它们必须充电，在高级护理或家庭健康环境中使用可能很麻烦。此外，通过传感器而不是摄像头，它还保留了尊严，曾说。

创始人集体合伙人 Eric Paley 告诉 TechCrunch，他喜欢 Butlr 正在做的事情，因为它“令人难以置信的简单”，他称之为“室内传感的圣杯”。

他看到其他创业公司也在追求同样的市场，但由于普通用户难以设置或需要电工而失败。佩利说，将传感器固定在一个地方并在那里放置数年“是巨大的”。

“他们在早期阶段所能做到的令人印象深刻，”他补充道。“Butlr 的技术有如此多的用途，许多行业都在追求使用它。我们一直在讨论什么是最好的机会，通过 API 方法，我们可以为很多客户提供服务。”
