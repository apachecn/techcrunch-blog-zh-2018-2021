味觉智能初创公司 Halla 完成了价值 450 万美元的 A1 系列，以预测购物者将购买哪些杂货商品 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/16/taste-intelligence-startup-halla-closes-4-5m-series-a1-to-predict-what-grocery-items-shoppers-will-buy/

Halla 想要回答人们如何决定吃什么的问题，现在已经从食品零售风险投资公司获得了 450 万美元的新鲜 A1 轮投资。

总部位于纽约的 Halla 于 2016 年由加布里埃尔·尼波特、亨利·迈克尔森和斯潘塞·普莱斯创立，旨在开发“味觉智能”，利用人类行为引导购物者找到他们想要的食物，同时在网上购物时发现新的食物。这一切都导致商店的购物篮订单增加。首席执行官普莱斯告诉 TechCrunch，SOSV 和 E&A 风险投资公司也加入了这轮融资，这使得 Halla 的总资本增加到 850 万美元。

该公司的 API 技术是一个即插即用的平台，利用超过 1000 亿个购物者和产品数据点，并将其输入三个引擎:搜索，考虑购物者的偏好；推荐，在某人购物时揭示相关的补充产品；和标识替换选项的替换。

Halla 的替代产品于今年早些时候发布，作为对缺货商品更好建议的回答，甚至像沃尔玛这样的零售商也在创造技术来解决。普莱斯引用了麦肯锡的一份报告，该报告发现 20%的杂货店购物者在糟糕的替代品产生负面结果后寻找竞争对手。

哈拉替代品。图片来源: Halla

这些数据点都没有与任何购物者的私人数据相关联，只是与购物本身相关的属性。相反，API 寻找上下文来返回相关的建议和替代。例如，Halla 的平台会考虑某人向购物车添加商品的方式，并建议下一个商品:如果你添加了火鸡和面包，平台可能会建议奶酪和调味品。

普莱斯说:“在杂货店购物和购买食品时，这也与个性化有关。”。“当你想要某个特定品牌的有机鸡蛋，而它又缺货时，这通常取决于你的私人购物者的判断。我们希望引导他们找到正确的替代品，这样你仍然可以烹饪你想要的食物，而不是“足够接近”"

Halla 的技术现已应用于 1100 多家电子商务商店。新的资金为 Halla 公司内部正在发生的火灾价格提供了一些燃料，包括将其支持的跨账户商店数量增加一倍的计划。他还预计将员工人数增加一倍至 30 人，以支持增长和客户群，并承认“我们可以处理更多的入境利益”Halla 一直忙于快速跟踪试点项目的大客户，同时希望在未来 18 个月内通过增加产品线进行国际扩张。

该公司还看到了“经常性收入的近乎无限的增长”，因为它吸引了六位数和七位数的合同，使该公司更接近现金流的积极性。普莱斯说，所有这些增长都是为了让 Halla 在需要的时候获得 B 轮融资。

与此同时，作为投资的一部分，食品零售风险投资公司的詹姆斯·麦肯将加入汉拿的董事会。

只投资食品和零售技术的 McCann 告诉 TechCrunch，杂货店需要一种激励购物者的方法，Halla 正在这样做，而且比他见过的其他智能版本更好。

“他们的技术远远领先于其他所有人，”他补充道。“他们有很棒的团队和很棒的产品。在建议和人们购买的东西方面，他们看到了巨大的提升，他们的衡量标准非常棒。”
