在过去十年的大部分时间里，金融科技行业都取得了巨大的成功(和丰厚的利润)，在疫情期间更是如此。但令人惊讶的是，业内许多人认为故事才刚刚开始，该行业将取得更大成就，预计金融科技的下一个十年将与过去 10 年截然不同。

早在疫情之前，监管银行的方式就在变化。像开放银行和修订的支付服务指令(PSD2)这样的倡议被提出来作为促进银行业竞争的一种方式——允许较小的挑战者公司进入一个长期由企业巨头主导的市场。

然而，现在这些举措已经到位，我们看到它们的效果远远超出了为挑战者银行打开缺口。由于开放银行要求银行通过 API 提供有价值的数据，这正在导致中小企业融资方式的革命——在这种方式下，数据而不是硬资本是推动金融科技成功的最重要因素。

开放银行和数据自由

为了理解席卷金融科技并重新配置该行业与小企业合作方式的变化，理解开放银行很重要。这是一个在过去十年里真正在政府和超国家银行监管机构中扎根的概念，我们现在开始看到它对整个银行业的影响。

允许第三方访问银行的数据将有助于评估中小企业的真实财务状况，其中许多是第一次这样做。

在最基本的层面上，开放银行指的是使用 API 向第三方开放消费者金融数据的过程。这使得这些第三方能够设计、构建和分发他们自己的金融产品。这些产品的效用(以及最终的盈利能力)并不依赖于它们持有的巨额资本，而是它们收获和包含的数据赋予了它们价值。

开放银行模式带来了许多挑战。一是银行业需要开发更加严格的系统，不断寻求消费者同意以这种方式共享数据。尽管金融科技的早期发展告诉我们，当涉及到放弃他们的数据时，消费者是非常放松的——一些研究表明，几乎 60%的美国人选择金融科技而不是隐私——通过开放银行框架共享的类型和数量比我们迄今为止看到的产品要广泛得多。

尽管存在这些担忧，世界各地都在推进开放银行业。在欧洲，PSD2(支付服务指令)要求大型银行与第三方共享金融信息，在亚洲，中国的支付宝和微信以及印度的 Tez 和 PayTM 等服务已经在改变金融服务市场。通过这些服务获得的额外功能已经导致呼吁美国银行系统以同样的程度接受开放银行业务。

为中小企业服务

如果美国银行业能够相信开放银行的效用，或者如果它被迫通过立法这样做，几个群体可能受益:

  • 消费者将获得基于比目前更详细的数据分析的新型银行和投资产品。
  • 设计和制造这些产品的金融科技公司也将看到其产品的使用增加，其利润率也随之增加。
  • 可以说，即使是银行也将受益，因为即使在最开放的模式下，银行仍然充当着看门人的角色，决定哪些第三方可以访问消费者数据，以及他们需要做些什么来访问这些数据。

然而，到目前为止，开放银行业的最大受益者将是中小企业。这并不一定是因为开放银行框架提供了对中小型企业有用的特定新功能。相反，它反映了一个事实，即传统银行对中小企业的服务历来很差。

中小企业在很多方面都没有得到足够的服务。传统银行查看中小企业总体财务状况的能力极其有限，因为中小企业持有多家机构和多种工具的资本，这使得获得融资非常困难。

此外，中小型企业通常必须处理过时且耗时的手动界面，才能将数据上传到银行。而且(也许是最糟糕的)大多数银行使用的 B2B 支付系统向使用它们的企业提供的反馈非常有限——信息的缺乏会让企业付出沉重的代价。

新功能

鉴于这些缺陷，金融科技初创公司热衷于向小企业贷款，中小企业积极寻找新颖的银行产品和服务就不足为奇了。当然，在这一领域已经有了一些成功的故事，今天中小企业(尤其是欧洲的中小企业)可以利用的银行系统比 10 年前的服务要先进得多。

然而，开放银行有望加速这一转变，并极大地改善普通中小企业可获得的金融服务。它将以几种方式做到这一点。允许第三方访问银行的数据将有助于评估中小企业的真实财务状况，其中许多是第一次这样做。

通过 API，金融科技公司将能够访问不同类型账户、保险、信用卡账户和租赁的信息，并将来自多个国家的数据整合成一幅整体画面。

这反过来将对中小企业的信用评估方式产生重大影响。目前，许多中小企业面临资金缺口，这主要是因为银行一直不愿脱离评估信贷风险的“资产负债表”模型。通过对中小企业当前业务活动进行实时分析，银行将能够更准确地评估这一风险，并向更多企业提供贷款。

事实上，这种情况已经在开放银行业发达的国家发生了——在英国，除了账户交易数据之外，劳埃德的商业工具箱还提供对公司和董事的无限制信用检查。

开放银行业务将也允许同行比较分析远远领先于我们迄今所看到的。API 可用于向中小企业提供关于其在市场部门中表现如何的实时反馈。同样，这种能力在英国已经可用，Barclays 的 SmartBusiness Dashboard 作为可定制业务仪表板的一部分提供了营销有效性工具。

这些功能对中小企业非常有用，它们可能会推动任何提供这些功能的金融科技产品的流行。对于中小企业来说，这一价值将主要在于基于智能数据分析的见解、建议和自动提示，它们可以建立在帐户聚合的基础上。

然后，从这些相同的监测工具中产生的额外见解可以使银行和替代贷款人更加积极主动地贷款——及时地向以前难以获得资金的中小企业提供预先批准的信贷额度。

底线

对金融科技行业至关重要的是，几乎可以肯定的是，中小企业将愿意为有助于它们发展的基于数据分析的增值服务付费。这就是为什么这个领域的一些初创公司已经吸引了大量资金，也是为什么开放银行是科技和经济之间关系的的核心。

因此，如果金融科技今年表现不错，这很可能只是故事的开始。在开放银行计划的支持下，该部门现在正处于银行业革命的前沿，这场革命最终将为中小企业提供应有的服务水平，并释放它们在整个经济中的真正潜力。
