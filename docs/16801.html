网络风险初创公司 Safe Security 从英国电信获得 3300 万美元

硅谷网络风险管理初创公司 Safe Security 获得了英国电信公司 T2 BT T3 3300 万美元的投资。

Safe Security 成立于 2012 年，原名 Lucideus，使用其企业安全评估框架(Safe)平台帮助组织测量和减轻企业范围的网络风险。包括脸书、软银和小米在内的许多公司都在使用这项服务，它可以帮助企业了解遭受重大网络攻击的可能性，计算客户风险的财务成本，并就应对这些风险的措施提供可行的见解。

包括思科前董事长兼首席执行官约翰·钱伯斯在内的 Safe Security 现有投资者参与了这轮融资，使 Safe Security 筹集的资金总额达到 4920 万美元。

英国电信表示，这是该公司自 2006 年以来在网络安全领域的第一笔重大第三方投资，反映了该公司在该领域快速增长的计划。英国电信首席执行官菲利普·詹森(Philip Jansen)表示:“网络安全现在是企业和政府的首要任务，他们需要能够相信自己受到保护，免受越来越多的攻击。T11】

“在一个高度分散的安全市场，英国电信已经是全球领先的提供商之一，这项投资清楚地表明了它进一步发展的雄心。”

这家初创公司的联合创始人兼首席执行官 Saket Modi 表示，他很“高兴”与英国电信合作。

“通过将英国电信的全球影响力和能力与外管局提供网络风险态势实时可见性的能力相结合，我们将从根本上改变全球安全的衡量和管理方式，”他说。

作为投资的一部分，到今年年底，SAFE Security 的工程团队将增加一倍，英国电信将把 Safe 平台与其托管安全服务相结合，并获得向英国企业和公共部门机构使用和销售 Safe 的独家权利。根据该公司的一份声明，英国电信还将与 Safe Security 合作开发未来的产品。

Safe Security 的竞争对手包括 UpGuard 、 Exabeam 和 VisibleRisk。
