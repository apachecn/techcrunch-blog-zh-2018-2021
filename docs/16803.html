Dwolla 筹集了 2100 万美元，为金融科技和品牌带来更多可定制的支付和转账选择 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/21/dwolla-raises-21m-to-bring-more-customizable-payment-and-money-transfer-options-to-fintechs-and-brands/

估值为 950 亿美元的 Stripe 一直在以一种整体平台的方式应对支付领域，引入了数十种相邻的服务，以通过 API 的方式吸引更广泛、更深入的客户群。但在所谓的“嵌入式金融”领域，规模较小的参与者仍有很大的空间，可以采用更复杂的方法来构建复杂的金融流程，第三方可以整合这些流程来开展自己的业务。如今，其中一家公司宣布提供一些资金来支持自己的使命。

Dwolla 提供了一个 API，允许公司建立和促进快速支付，特别是专注于 ACH(自动清算中心，或银行或其他金融机构之间的支付或转账)，已经完成了 2100 万美元的融资，这笔钱将用于继续构建其服务的功能，特别是如何集成和提供更多的卡支付响应能力；雇佣更多的人才；并开始将其铁路推向美国以外的更多市场，最有可能的是首先考虑加拿大、英国和澳大利亚。

Foundry Group 在这一轮中领先，Park West Asset Management LLC、Union Square Ventures、Detroit Venture Partners、Firebrand Ventures 和 Next Level Ventures 也参与其中。Traeger 的首席执行官杰里米·安德鲁斯(Jeremy Andrus)也作为个人投资者参与了这轮投资。该公司的其他投资者包括 Andreessen Horowitz、High Alpha、Thrive Capital 和 Ludlow Ventures，首席执行官布雷迪·哈里斯(Brady Harris)在接受采访时表示，Dwolla 目前不会披露其估值，但称其“在交易和支付方面总体上具有竞争力。”

德沃拉的总部设在爱荷华州的得梅因，多年来一直不为人知。自 2009 年以来，在本轮融资之前，它仅筹集了略高于 5000 万美元的资金，这对于如今的金融科技公司来说是一个相对适中的数额。这 2100 万美元是迄今为止最大的一轮。

但它也一直在悄悄地看到大量的增长。2019 年，Dwolla 通过其平台处理了 110 亿美元的总支付量。2020 年，这一数字增至 200 亿美元。哈里斯说，今年预计将达到 300 亿美元。客户包括大型机构和金融科技公司，它们希望将更快、更高效的基于 ACH 的支付整合到自己的服务中，而无需从头开始构建它们的繁重工作；客户还包括希望将这些支付整合到它们的堆栈中的企业，这些企业对它们如何贴上白色标签并进行定制有特殊要求。

该公司的平台上总共有大约 300 万最终用户，这些用户是通过使用其服务的大约 500 名客户获得的。这些客户包括房地产公司、教育机构、零售商和品牌，如山羊、伊博塔和拉力赛。其中一些客户比你想象的要大。Harris 向我提到，在白标服务中使用 Dwolla API 的客户之一是一家金融科技公司，其交易总额约为 9 万亿美元。(德沃拉属于 NDA，所以不能透露名字。哈里斯说，这 300 万的数字目前每个季度增长 150 万，所以现在它的交易量非常大。

当然，300 亿美元只是支付大饼的一小部分，2020 年交易估计价值为 5.4 万亿美元，预计到 2026 年将增长到 11.29 万亿美元。

正如 Harris 所描述的那样，虽然今天市场上有很多选择可供希望将支付，特别是基于银行转账的支付纳入其堆栈的公司选择，但 Dwolla 的独特方法是，它使这一特定服务更加高效，并易于为那些希望在流程中添加更多功能的公司定制。(这可能包括更多的计时，采用混合方式，包括卡支付或其他支付方式或其他方式。)

“ACH 产品是消费者可以在 Stripe 这样的支付公司下架的东西，但这是为了创造更多的定制，”哈里斯说。“我们有很多人正在与另一家提供商进行 mid 集成，但是 it 部门无法完成他们想要的工作，所以他们来找我们。我们喜欢认为自己是有计划性的、灵活的。”

这种对空间的专注和掌控不仅帮助 Dwolla 在客户中崛起，也帮助投资者崛起。

" Dwolla 继续推动现代商业支付的创新。在转向 B2B 领域的过程中， Dwolla 被定位为提供急需的商业支付解决方案，Foundry Group 合伙人 Chris Moody 在一份声明中表示。"现在， Dwolla 正利用这种动力和创新彻底改变当今领先品牌的资金运作方式。加倍投资是显而易见的，因为我们继续看到该公司在 B2B 支付现代化方面的价值，以及金融技术对公司在其巅峰时期运作的重要性。”
