Loop 在由 CRV 牵头的 6500 万美元的 B 轮融资中获得回报

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/21/loop-returns-locks-in-65-million-series-b-led-by-crv/

软件公司 Loop Returns 宣布完成 6500 万美元的 B 轮融资，该公司希望处理零售/电子商务退货的高成本和低效率流程。这轮融资由 CRV 牵头，参与方包括 Shopify 和 Renegade Partners，以及现有投资者 FirstMark Capital、Ridge Ventures、Peterson Ventures 和 Lerer Hippeau。

这笔交易对该公司的估值为 3.4 亿美元。

乔纳森·马坡是 Loop Returns 的联合创始人，此前他曾在一家机构工作，并为一家大型 Shopify 品牌提供咨询，帮助他们退货和换货。他与老朋友科比特·摩根合伙创办了 Loop Returns。

该软件与 Shopify 平台配合使用，以降低零售中常见问题(即退货)的成本和难度。事实上，

according to Shopify

，退货占电商销售额的 20 到 30%。对于大品牌和小品牌来说，这种趋势不仅要花钱，而且可能会失去一个客户和未来的收入。

Loop 通过引导用户回答一系列旨在保持业务的问题来接近回报。它从围绕商品尺寸的问题开始，然后转移到交换的概念，然后在退货的基础上向顾客提供品牌信用。

如果顾客想要的只是退货，Loop 会处理这个过程中一些比较棘手的部分，比如运输标签和品牌退款。

在疫情之前，该公司围绕国际扩张、平台扩张和产品扩张制定了宏大的计划。最终，黑天鹅时刻让该公司专注于其核心产品和照顾其客户，并获得了回报，尤其是在电子商务加速发展的情况下。

该公司的团队从 2019 年的 20 名员工发展到 100 名，团队中 41%的人是女性，16%的人是 BIPOC。

就吸引力而言，该公司的客户从 200 个增加到 700 个，在同一时间内，处理的退货从 2600 万美元增加到 1 亿多美元。

马坡告诉 TechCrunch，初创公司面临的最大挑战是扩大团队规模。

马坡说:“这是要把优秀的人才聚集起来，让他们朝着一个共同的目标前进，尤其是在这个遥远的第一世界。”。
