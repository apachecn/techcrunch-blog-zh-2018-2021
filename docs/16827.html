周四，互联网的一大部分都离线了。互联网上一些最受欢迎的网站、应用和服务都关闭了，包括 UPS 和联邦快递(后来又恢复了在线)、Airbnb、Fidelity 和其他公司都报告说 Steam、LastPass 和 PlayStation Network 都经历了宕机。

世界各地的许多其他网站也受到影响，包括欧洲的媒体。

原因似乎是互联网安全巨头 Akamai 的中断，该公司为公司提供网络和内容交付服务。东部时间上午 11 点左右，Akamai 报告了其 Edge DNS 的一个问题，该服务旨在保持网站、应用和服务平稳安全地运行。

DNS 服务对互联网的运作至关重要，但众所周知，它存在漏洞，而且很容易被恶意行为者操纵。像 Akamai 这样的公司已经建立了自己的 DNS 服务，旨在为他们的客户解决这些问题。但当事情出错或出现中断时，它会对所有依赖它的客户网站和服务产生连锁反应。

Akamai 表示，它正在“积极调查这个问题”，但当联系到一位发言人时，他不会说它的中断是否是目前离线的其他网站和服务中断的原因。思科 2020 年收购的互联网监控公司 ThousandEyes 的发言人将此次宕机归咎于 Akamai。

Akamai 没有说是什么原因导致了这个问题，但它已经在恢复中。

“我们已经针对此问题实施了一个修复程序，根据目前的观察，该服务正在恢复正常运行。我们将继续监测，以确保影响已经完全缓解，”Akamai 告诉 TechCrunch。

在随后的推文中，该公司称这“不是网络攻击的结果”

这不是我们第一次看到这么大的停电。去年，Cloudflare 也为世界各地的公司提供网络服务，在一个错误导致包括 Shopify、Discord 和政治在内的主要网站停止加载后，出现了类似的中断。11 月，亚马逊的云服务也绊住了，导致其在事件中无法更新自己的状态页面。在线工作空间初创公司“概念”今年也在遭遇了令人瞩目的宕机，迫使该公司转向 Twitter 寻求帮助。
