Magic 因其“即插即用”的无密码技术获得 2700 万美元的首轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/22/magic-27m-series-a-for-its-plug-and-play-passwordless-tech/

Magic 是一家总部位于旧金山的初创公司，开发“即插即用”无密码认证技术，已经在首轮融资中筹集了 2700 万美元。

这一轮由 Northzone 牵头，参与方包括 Tiger Global、Volt Capital、Digital Currency Group 和 CoinFund，一年多前 Magic 从 stealth 推出，更名为 Formatic。

和许多其他公司一样，该公司的使命是终结传统的基于密码的身份认证。Magic 的旗舰 SDK 于 2020 年 4 月推出，使开发人员只需几行代码即可实现各种无密码认证方法，并与许多现代框架和基础设施集成。

SDK 不仅使公司和开发人员更容易在他们的应用程序中实现无密码验证方法，而且还有助于减轻许多人不得不处理数据泄露所带来的昂贵后果。

“这就是密码如此危险的原因，”Magic 联合创始人兼首席执行官肖恩·李告诉 TechCrunch。“现在这就像一个叠木塔——入侵你系统的黑客可以下载整个加密密码数据库，然后轻松破解它们。这是一个巨大的失败中心。”

该公司最近开发了 SDK，增加了对 WebAuthn 的支持，这意味着它可以支持 Yubico 等基于硬件的认证密钥，以及移动设备上基于生物特征的 Face ID 和指纹登录。T11】

“现在它不是主流，但是我们正在为开发者把它变得超级简单，”李说。“通过这种方式，我们可以帮助推广新技术，这对用户的安全和隐私非常有利。”T13】

这一赌注似乎正在奏效:据 Magic 称，Magic 的开发者注册量已录得 13%的月环比增长，受保护的身份数量正以每周 6%的速度增长。它还获得了许多大牌客户，从加密新闻出版商 Decrypt 到筹款平台 Fairmint。

Northzone 的合伙人 Wendy Xiao Schadeck 表示:“我们非常高兴能够支持 Sean 和 Magic 团队，因为他们自下而上地重新定义了互联网的身份验证，解决了开发者、用户和公司的核心痛点。

“对我们来说很明显，他们绝对受到客户的喜爱，因为这个团队如此痴迷于为跨越几个社区的开发者之旅的每一个部分提供服务。可能更令人兴奋的是，他们将能够做些什么来赋予用户权力，并分散网络的身份层。”

该公司现在计划继续扩大其平台和团队，以满足 Magic 所描述的“飙升”的需求。这家初创公司目前有 30 名全职远程工作的员工，预计所有核心职能部门的员工人数将至少增加一倍，包括产品、工程、设计、营销、财务、人事和运营。

它还计划进一步开发 SDK 李说，他希望能够接入更多种类的技术，从低代码应用程序到工作流程自动化。T3】

“眼界比这大得多。我们希望成为互联网的通行证，”李补充道。
