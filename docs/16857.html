英国的 Mindtech 从 In-Q-Tel 等公司筹集了 325 万美元，用于训练闭路电视摄像头拍摄人造人

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/22/uks-mindtech-raises-3-25m-from-in-q-tel-among-others-to-train-cctv-cameras-on-synthetic-humans/

想象一个没有人的隐私被侵犯，没有人的面孔被扫描进庞大的数据库，没有隐私法被违反的世界。这是一个正在快速接近的世界。公司可以简单地放弃对真实世界闭路电视镜头的需求，转而使用人造人，将潜在的场景表演一百万遍吗？这是一家新的英国创业公司吸引了一批有影响力的投资者的资金的诱人前景。

总部位于英国的 Mindtech Global 已经开发了一个它称之为端到端的合成数据创建平台。说白了，它的系统可以想象视觉场景，比如某人在商店里的行为，或者过马路。这些数据随后被用于为大型零售商、仓库运营商、医疗保健、运输系统和机器人等客户训练基于人工智能的计算机视觉系统。它实际上是在一个合成的世界里训练一个“合成的”闭路电视摄像机。

它现在已经完成了一轮 325 万美元的早期融资，由英国地区支持者NPIF-麦西亚股权融资牵头， Deeptech Labs 和 In-Q-Tel 。

最后一个投资者意义重大。In-Q-Tel 投资于支持美国情报能力的初创公司，总部位于弗吉尼亚州阿灵顿。

Mindtech 的变色龙平台旨在帮助计算机理解和预测人类的互动。众所周知，目前训练人工智能视觉系统的方法需要公司提供数据，如闭路电视镜头。这一过程充满了隐私问题，而且既费钱又费时。Mindtech 表示，Chameleon 解决了这个问题，因为它的客户可以快速“使用照片般逼真的智能 3D 模型构建无限的场景和场景”。

一个额外的好处是，这些合成人可以用来训练人工智能视觉系统，以消除人类在多样性和偏见方面的缺点。

迈达科技首席执行官 Steve Harris。图片来源: Mindtech

Mindtech 首席执行官 Steve Harris 说:“机器学习团队可以花多达 80%的时间来寻找、清理和组织训练数据。我们的变色龙平台解决了人工智能训练挑战，让行业专注于更高价值的任务，如人工智能网络创新。这一轮将使我们能够加速增长，实现新一代人工智能解决方案，更好地理解人类相互之间以及与周围世界的互动方式。”

那么你能用它做什么呢？考虑以下情况:一个孩子在购物中心从父母手里溜走了。在 Mindtech 场景中运行的合成闭路电视经过数千次训练，知道如何实时发现它并提醒员工。另一个:送货机器人遇到在街上玩耍的孩子，并想出如何避开他们。最后:站台上的一名乘客行为异常，离铁轨太近——闭路电视经过训练，可以自动发现他们并发出帮助。

In-Q-Tel 董事总经理(伦敦)Nat Puffer 评论道:“Mindtech 的 Chameleon 平台的成熟及其对全球客户的商业吸引力给我们留下了深刻的印象。我们对该平台在不同市场的众多应用以及它在开发更智能、更直观的人工智能系统方面消除重大障碍的能力感到兴奋。”

Deeptech Labs 首席执行官 Miles Kirby 表示:“作为 Deeptech 成功的催化剂，我们的投资和加速器计划支持拥有新颖解决方案和建立改变世界的公司的雄心勃勃的团队。Mindtech 经验丰富的团队正致力于打破人工智能系统的训练方式，我们很高兴支持他们的旅程。”

当然也有潜在的更黑暗的应用，比如发现超市里的小偷小摸，或者以某种反乌托邦的方式“优化”压力重重的仓库工人。然而，理论上，Mindtech 的客户可以利用这个平台摆脱中层管理人员的偏见，更好地为客户服务。
