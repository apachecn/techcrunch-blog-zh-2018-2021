牛姆酒通过 2 亿美元的 Riverwood Capital 领投，超越了 1B 的估值

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/26/nium-crosses-1b-valuation-with-200m-riverwood-capital-led-round/amp/

图片来源:Tero Vesalainen/Getty Images

B2B 支付平台 Nium 周一宣布，它在 D 轮融资中筹集了超过 2 亿美元的资金，估值超过 10 亿美元。

该公司现在总部位于新加坡，但正在转移到旧金山湾区，它吹捧这项投资使其成为“东南亚第一家 B2B 支付独角兽”

Riverwood Capital 领投了这一轮投资，淡马锡、Visa、Vertex Ventures、Atinum Capital、Beacon Venture Capital 和 Rocket Capital Investment 也参与了投资，还有一批天使投资人，如 DoorDash 的 Gokul Rajaram、FIS 的 Vicky Bindra 和 Tribe Capital 的 Arjun Sethi。包括新的资金在内，Nium 迄今为止已经筹集了 3 亿美元，联合创始人兼首席执行官 Prajit Nanu 告诉 TechCrunch。

一些专家表示，B2B 支付领域已经炙手可热，但仍未充分渗透。Nanu 说，为了说明有多火，Nium 正在寻求 1.5 亿美元的 D 轮融资，从热切的投资者那里获得了 3 亿美元的承诺，最终确定了 2 亿美元。

“这是我们第四次或第五次筹资，但我们以前从未有过这种兴趣——我们甚至在五天内就拿到了投资意向书，”他补充道。“我相信这种兴趣是因为我们成功地创建了一个受到严格监管的全球平台，这使我们能够接触到许多网络。这是一个支付可见的环境，我们的核心是推动无摩擦商务，让任何人都能使用我们的平台。”

Nium 的新一轮融资为许多公司都在追逐每年价值 120 万亿美元的全球 B2B 支付市场火上浇油:上周，Paystand 在 C 轮融资中为筹集了 5000 万美元资金，以实现 B2B 支付的无现金化，而 Dwolla 为其 API 筹集了 2100 万美元，该 API 允许公司建立和促进快速支付。今年 3 月，Higo 筹集了 330 万美元在拉丁美洲做同样的事情，同时 Balance 开发了一个 B2B 支付平台，允许商家提供多种支付方式。2 月融资 550 万美元。

Nium 的方法是通过单一 API 提供对全球支付基础设施的访问，包括卡发行、应收账款和应付账款以及银行即服务。Nanu 说，该公司的网络使客户能够向 100 多个国家汇款，以 60 多种货币支付，接受 7 种货币的资金，并在 40 多个国家发行信用卡。该公司还在 11 个司法管辖区拥有资金转账、信用卡发行和银行执照。

Riverwood 的联合创始合伙人兼管理合伙人弗朗西斯科·阿尔瓦雷斯-德马尔德(Francisco Alvarez-Demalde)在一封电子邮件中表示，软件和监管许可证的结合，以及代表新银行和企业运营金融科技基础设施平台，是一种经历高速增长的全球趋势。

Riverwood 跟随 Nium 多年，正是它的未来愿景让该公司有兴趣成为这一轮的一部分。Alvarez-Demalde 表示，“Nium 拥有巨大的市场机遇、才华横溢的创始人和团队，我们相信，基于潜在的长期技术趋势，如增强的实时支付能力和跨境商务的激增，该公司已为全球增长做好准备。

他补充说:“作为一个 API 中的中央支付基础设施，Nium 是一种催化剂，可以通过当地市场许可证、合作伙伴和银行关系网络解锁跨境支付、本地账户和卡发行，以促进资金在世界各地的流动。”“各种类型的企业都在将金融服务嵌入到他们的消费体验中，而 Nium 是这一趋势的关键全球推动者。”

Nanu 表示，新的资金使该公司能够转移到美国，这占 Nium 收入的 3%。他希望在未来 18 个月内将这一比例提高至 20%，并在拉丁美洲扩张。这项投资还为该公司在 M&A 的进一步活动提供了 12 至 18 个月的时间。6 月份，Nium 收购了虚拟卡发行公司 Ixaris ，7 月份收购了 Wirecard Forex India 以将其引入印度市场。他还计划扩大该公司的支付网络基础设施，投资于产品开发，并增加 Nium 的 700 名员工。

Nium 已经拥有数百家企业客户，并计划在明年增加数千家。该公司每年处理 80 亿美元的支付，自 2015 年以来已发行超过 3000 万张虚拟卡。与此同时，收入同比增长超过 280%。

Nanu 说，所有这些增长使公司走上了首次公开募股的轨道。他已经与将帮助该公司在 2022 年第一季度正式启动这一旅程的人进行了交谈。

“与其他为新产品筹集资金的公司不同，我们的目标是扩大我们现有的业务范围，”纳努说。“美国是一个新市场，但我们有一个很好的品牌，并将利用新一轮为客户提供更好的体验。”
