Solarisbank 以 16.5 亿美元的估值筹集 2.24 亿美元收购 Contis，在欧洲拓展基于 API 的嵌入式银行技术

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/26/solarisbank-raises-224m-at-a-1-65b-valuation-to-acquire-contis-and-expand-its-api-based-embedded-banking-tech-in-europe/

嵌入式金融——金融服务的一些更复杂但也更商品化的方面被构建并包装在 API 中，供其他任何人在自己的产品中为最终用户实现的过程——已成为当今金融科技的巨大基石。现在，该领域中较早、规模较大的公司之一宣布将进行一轮大规模的增长融资，以建立自己的业务。

柏林初创公司 Solarisbank 通过大约 180 个 API 提供一系列金融服务，其他人使用这些 API 来构建面向最终用户的产品(包括基本的银行和信用卡服务、贷款、支付和了解你的客户服务)，该公司在 D 轮融资中为€筹集了 1.9 亿英镑(2.24 亿美元)，对该公司的估值为€14 亿英镑(16.5 亿美元)，并宣布收购该领域的竞争对手之一康蒂斯。

瑞士公司 decisional Capital Management 也支持保险科技初创公司 Wefox，该公司领投了这轮融资，Pathway Capital Management、CNP 和 Ilavska Vuillermoz Capital 也参与了融资，此外还有之前的支持者 Yabeo Capital、BBVA、Vulcan Capital 和 HV Capital。

这轮融资距离上一轮 6750 万美元的融资约一年，但作为时代的标志，或许更值得注意的是，该公司的估值自那以来已增长近五倍(2020 年 6 月为 3.6 亿美元)。

最近这一轮是要用来扩张的。首席执行官 Roland Folz 在接受采访时表示，收购 Contis 突显了金融科技领域更广泛的整合趋势，将有助于它更好地覆盖整个欧洲，并开始向亚洲迈出第一步。他说，目前没有将美国加入这一名单的计划。

他补充说，合并后的实体将实现“三位数欧元”的收入，也就是数亿欧元；该公司 2020 年的净收入为 3500 万€，如果愿意的话，明年就可以上市。

与此同时，它希望在我们正在金融服务领域看到的巨大转变上加倍努力，在这个领域，消费者和企业正在选择更新、更现代的银行体验，因为他们正在从更慢、更不灵活、有时更昂贵的现有机构中迁移出来。

“欧洲是我们工作的理想场所，”福尔兹说。“我们认为，欧洲大约有 8 亿个银行账户，其中约 4 亿个将改变‘所有权’，传统银行将被非传统银行取代。…如果我们着眼于 5 到 10 年的前景，我们希望确保这些客户中的很大一部分将出现在我们的平台上。”

SolarisBank 的客户包括 Trade Republic、American Express、BP、Samsung 和 Vivid ，为公司提供基本的银行业务、了解客户的支票、贷款、数字钱包和其他与金融相关的服务，反过来，这些公司可以将精力集中在建立更加用户友好的客户体验或其他服务上。福尔兹补充说，它每年以 40%-60%的速度增长，总共有大约 50 个“合作伙伴”(它这样称呼它的客户)，覆盖大约 200 万个客户账户。

与此同时，Contis 本身就是一个庞大的业务，拥有约 200 名客户，覆盖 200 多万用户，每年为€带来 99 亿英镑的交易。

Solarisbank 五年前成立于柏林的创业孵化器和投资者 finleap ，finleap 的创始人 Ramin Niroumand 实际上也是 Solarisbank 的“创始人”。(目前，他还是该公司顾问委员会的主席。)

嵌入式金融目前风靡一时，如今许多初创公司都在向第三方提供金融科技即服务或银行即服务工具。在同一细分市场的其他著名公司包括 Railsbank，它也在本月早些时候宣布融资； Rapyd ，今年早些时候以 25 亿美元的估值进行了一轮大融资；部门，另一家银行即服务创业公司获得资金并开始成长； FintechOS ，真的如其名(目前也在融资)；还有初创公司 10x ，讽刺的是它的目标是在职者。

Solarisbank 认为，其填补市场空白的特殊方法赋予了它更多的灵活性和里程:与竞争对手不同，扩张后的 Solarisbank 将在欧洲获得银行牌照，在立陶宛和英国获得电子货币牌照，其技术堆栈基于 AWS，这使它有机会建立更多的服务，扩大规模并在此过程中保持更高的利润率——这是规模经济中的一个关键细节。

它还认为，其多样化的客户基础——不仅包括 neobanks 等明显的金融科技公司，还包括三星等其他各种正在建立金融服务的公司(就其而言，是一种数字钱包)——使其更有持久力，可以满足该基础上任何特定时间增长最快的细分市场。正如 Niroumand 指出的那样，大约 70%的收入来自大约 30%的客户。“我们服务的客户相当多样化，”他说。

该公司目前活跃在德国、法国、意大利和西班牙，但表示它可以通过护照覆盖整个欧洲经济区。

“通过合并后的实体，我们看到了其他任何人都无法与之相比的数字，”Folz 补充道。

市场机遇，加上 Solarisbank 的方法及其当前的客户群，吸引了投资者。

“我们正在经历银行业的范式转变，客户希望金融服务能够适应他们的特定需求，”果断资本管理公司的合伙人托马斯·施利特-亨里克森(Thomas Schlytter-Henrichsen)在一份声明中表示。“技术是实现这一转变的关键，Solarisbank 强大的银行即服务平台使其在这个新的银行时代中处于完美的位置。我们都被这个团队所鼓舞，并为能一起完成这个使命而激动不已。”
