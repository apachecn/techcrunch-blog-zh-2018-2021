基于分散矩阵协议的消息应用 Element 融资 3000 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/27/element-a-messaging-app-built-on-the-decentralized-matrix-protocol-raises-30m/

在去年 Element 收购Gitter为自己的分散化、基于矩阵的消息应用带来更多用户和功能后，这家初创公司宣布为其增长投资一些资金。

Element 在 B 轮融资中获得了 3000 万美元。部分资金将用于扩展其技术，以继续满足已经在使用该技术的大型组织的使用案例。其中包括法国政府覆盖约 550 万公务员；Dataport(正在将其纳入德国的教育和公共管理系统)；以及为德国军队提供通信系统的 BWI，该公司称，总共约有 10 个政府项目。

Element 的首席执行官兼开源非营利组织 Matrix protocol 的技术联合创始人马修·霍奇森(Matthew Hodgson)也表示，部分资金将用于继续投资该公司的点对点架构，以消除对服务器的任何需求。第三个投资领域将是建设公司的分散语音和视频会议服务。

“在安全方面，这将是一次变革，”他在接受采访时说。

Element 现在可以作为内部部署系统或基于云的平台使用。这家初创公司表示，大多数公共部门的客户选择前者，私营部门的客户选择后者。在过去的 12 个月里，云收入增长了 300%，尽管该公司没有披露实际收入是多少，但这是 Element 正在增长的一个迹象(这是现在接受资金的另一个原因，尽管正如联合创始人 Amandine Le Pape 所说，它并不需要它)。

这轮投资包括来自协议实验室(libp2p、IPFS 和 Filecoin 背后的开源研发实验室)和meta planet(Skype 联合创始人贾恩·塔林设立的基金)。WordPress 母公司automatic和idea——都是过去的投资者——也参与其中。没有披露估值。迄今为止，Element 已经筹集了 4800 万美元。

Matrix 是 Element 所基于的非营利性协议，旨在通过创建一个地方来改变互联网的工作方式，在这个地方，多个孤立的通信环境可以集成到一个有凝聚力的单一平台中，而这些通信又可以由组织它的实体“拥有”。这里的想法是，通过将所有这些放在一起，从安全性和实际使用的角度来看，更容易管理这些对话。

Matrix 自身一直在增长，在过去 12 个月中使用量增长了 190%,现在部署了 75，000 个，拥有超过 3，500 万“可寻址”用户。这些部署可以部署在像树莓派这样的小东西上，也可以部署在政府运行的大型服务器上。但需要明确的是，“活跃”用户比可寻址用户要少得多。虽然 Matrix，更不用说 Element，无法“看到”其他人如何使用 Matrix(这是政府喜欢的安全优势之一)，但 Hodgson 指出，在其自己的服务器上有 120 万活跃用户。

element——最初被称为 Riot——被设定为一种 Slack/Discord 竞争对手，是 Matrix 之上的一个原生消息平台，旨在抓住已经在使用该协议做其他事情的受众，在只有少数平台控制着世界上大部分消息数据的时候，为他们提供一个干净安全的替代方案。

“15 年来，我们一直在开发信息技术，”Le Pape 说。“你让几个玩家(主导)并挟持每个人的数据，这感觉太疯狂了。这是打开任何人的交流。这就是我们试图解决的问题，数据主权。”

在这方面，元素(和矩阵)处于更大的技术趋势的核心，该领域一些最聪明的人正在考虑其工作方式的默认设置，并决定随着数据变得更有价值，通信渠道变得越来越重要，一种替代的，不那么集中的方法必须成为更大等式的一部分。

“互联网通信协议已经成为人类的基础。今天，大多数信息传递发生在易变的专有中央围墙花园，这些花园将我们作为他们短期商业前景的人质，”Protocol Labs 首席执行官胡安·贝内(Juan Benet)在一份声明中说。“Matrix 是希望的灯塔:一个安全、分散通信的开放网络，建立在健全的互联网基础设施原则之上。无论是产品还是公司，元素都能让组织在所有对话中拥有控制权、所有权和出色的用户体验。”

虽然市场上有许多其他加密消息平台正在被大量采用，包括 Telegram、Signal，甚至在某种程度上包括脸书的 WhatsApp，但 Element(和 Matrix)的早期采用者一直是大型组织。然而，随着引擎基础元素 Matrix 也与 Twitter 等公司就其 Blue Sky 分散平台努力开展早期工作，这可能会在早期采用者之外给 Element 带来另一个用户增长。

但是就像乌云有一线希望一样，阳光灿烂的前景也会投下阴影……就像 Telegram 和 Rocket.chat 等其他分散式通信平台一样，端到端加密也有两面性:它可以帮助防止通信被黑客攻击，但也可能被用来在策划恶意活动时逃避检测。Matrix(默认为 Element)一直在探索进入加密系统的后门的替代方案，这是政府正在考虑解决这个问题的一条途径，理由是授权一个只会将坏人转移到另一个。

最终，这里的投资，以及 Element 的使用，是对坚持去中心化方向的投票，尽管有这些疑虑和现存的问题。

“当交流集中时，它就成了一个非常容易被滥用的目标；不管是通过宣传、监视、审查还是更糟的方式，”Metaplanet 的塔林说。“消费者需要从监控资本主义中解救出来，组织需要一种安全中立的沟通方式。Matrix 是最先进的平台，可以提供缺失的通信层。”
