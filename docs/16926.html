微软收益超过预期，Azure 增长 51%；股票下跌

今天钟声过后，微软报告了其 2021 财年第四季度的收益，这一时期相当于今年的第二个日历季度。微软在此期间的收入为 462 亿美元，净收入为 165 亿美元，每股收益为 2.17 美元。与去年同期相比，该公司的收入增长了 21%，而同期净收入增长了更为可观的 47%。

该公司的结果超出了预期，雅虎财经报道收入为 441 亿美元，每股收益为 1.90 美元。消息传出后，这家软件巨头的股价下跌，可能是因为该公司的业绩遗漏了所谓的耳语数字；微软在最近几个交易日的交易价格达到或接近历史高点，这使得当前 3%的盘后跌幅成为现实。在今天的常规交易中，科技股普遍走弱，在此期间，微软的市值下降了不到 1%。

微软是如此之大的一家公司，以至于它的顶级结果几乎不清楚，所以让我们再深入一点。

首先，微软的云计算平台 Azure 公布，与去年同期相比，该季度的收入增长了 51%，如果剔除汇率波动，这一数字将降至 45%。根据初步分析，51%的数字是该公司自 2020 财年第三季度(即去年第一个日历季度)以来最好的 Azure 增长结果。

从这个角度来看，很难指责 Azure 在过去三个月的增长。

通过该公司的其他结果，我们可以对其三个主要部门的收入增长结果进行排序如下:

  • 智能云:30%的增长，这个数字部分是由 Azure 的增长推动的。
  • 生产力和业务流程:增长 21%，领衔的是 LinkedIn(增长 46%)和 Dynamics 365 CRM 产品(增长 49%)。
  • 更多个人计算:9%的增长，由搜索增长(53%，不包括流量获取成本)引领。

更大的雷德蒙收入审查中的弱点不难发现。办公室消费收入增长了 18%,这个数字看起来有些保守；Windows OEM 营收下滑 3%；表面收入下降了 20%。

但这些暗淡的光线不足以破坏该公司的总体增长前景和巨大的盈利能力。塞特亚·纳德拉的公司利润有多高？微软最近一个季度在股票回购和股息上花费了 104 亿美元。坦白地说，这是一笔有点令人困惑的金额。在这一点上，我们有点搞不清楚为什么微软要回购股票。它的市值略高于 2 万亿美元，这意味着该公司充其量只能随着时间的推移以巨大的代价逐渐减少其股票数量。它的现金肯定有更好的用途吧？

无论如何，该公司的业绩表明，最近一系列大型科技公司公布的令人印象深刻的巨额利润并没有结束。这可能有助于更广泛地增强投资者对科技公司的信心。你知道，这对创业公司来说不是一件坏事。
