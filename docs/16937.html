Shopify 允许商家直接通过他们的店面销售 NFT

Shopify 使符合条件的卖家可以通过其平台销售 NFT(不可替代的令牌)，这为电子商务商家开辟了一个全新的世界。

周一，NBA 芝加哥公牛队通过在 Shopify 上推出在线商店，推出了有史以来第一个 NFTs 包括 NBA 总冠军戒指的数字艺术品。公牛队的球迷现在可以使用信用卡或借记卡直接在该队的网上商店购买数字艺术品，而不必去 NFT 的市场。Shopify 的商业服务副总裁 Kaz Nejatian 表示，在推出 NFTs 的第一天，NBA 球队在短短 90 秒内就销售一空。

“以前你可以用信用卡购买 NFT，但老实说，现在 Shopify 之外的 NFT 购物体验对任何人来说都不是很棒，”他告诉 TechCrunch，“这就是我们决定做这项工作的原因。商家和买家不应该为了购买他们关心的东西而去上加密课程。”

这也是为了让消费者有更多的选择来购买 NFTs 特别是那些不精通加密货币的人。

该公司表示，通过让商家直接通过其 Shopify 店面销售 NFT 成为可能，它为想要销售 NFT 的商家创造了机会。Nejatian 说，他们最终将能够根据自己的产品和客户群选择在哪个区块链上销售，因为 Shopify 支持多个区块链。

“相比之下，如果商家想在 NFT 市场销售，他们需要根据该市场支持的区块链进行选择，”他补充道。

例如，芝加哥公牛队选择了区块链作为他们的 NFT。但总的来说，Shopify 商家今天可以从 Flow 和以太坊中进行选择，但根据 Nejatian 的说法，很快“将在 Shopify 上有更多的其他区块链选择”。

这一举动也是由商家的需求所驱动的，他们要求有能力销售 NFTs，并希望给创作者和艺术家另一个专业成长的论坛。

“许多创作者已经看到了向粉丝出售 NFT 的价值，但我们正在为他们和他们的买家消除一些摩擦，让他们更好地将他们的作品和他们与观众的联系货币化，”他补充道。“我们正在开辟一个世界，让他们的粉丝感觉到与他们的品牌有着有意义的联系，让 NFT 越来越成为我们在线购买和销售的一部分。”
