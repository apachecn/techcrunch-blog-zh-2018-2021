课堂，一个只缩放的虚拟教室，在软银检查后接近独角兽地位

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/28/class-a-zoom-only-virtual-classroom-nears-unicorn-status-after-softbank-check/

与 Zoom 独家集成的虚拟教室 Class 今天宣布，它在软银愿景基金 II(SoftBank Vision Fund II)牵头的一轮融资中筹集了 1.05 亿美元。成立仅 10 个月的初创公司目前已经从筹集了总计 1.46 亿美元的已知风险资金，超过了创始人迈克尔·蔡森的前上市公司 Blackboard 的融资额。

尽管还处于起步阶段，Class 正在迅速接近独角兽地位，证实其目前的融资后估值为 8.04 亿美元。此类投资者包括 GSV 风险投资公司和 Emergence Capital，他们领导了这家初创公司的 pre-seed 轮投资，以及美国顶级教育科技基金，包括 Reach Capital、Owl Ventures、Insight Partners 和 Learn Capital。

其他投资者包括 Maven Ventures 的 Jim Scheinman，他是 Zoom 的早期投资者，也是 Zoom 的命名者；比尔·泰是 Zoom 的第一个忠实支持者；史蒂夫·凯斯，美国在线的联合创始人和革命的首席执行官。

Class(以前称为 Zoom Class)使用管理和指导工具来增强视频会议通话体验。自推出以来，Class 与视频会议巨头进行了独家整合，后者在疫情的最初几个月里变得家喻户晓，并继续成为同步通信的中流砥柱。这是过去一年推出的变焦替代产品和增强产品浪潮的一部分，迄今已有 250 多名客户。

今天宣布软银的认可意味着 Class 做出了两个声明:第一，它正在认真对待全球扩张，第二，我认为这是一个信号，表明它不仅仅是 Zoom 的收购目标。

教育技术全球化

软银喜欢支持它认为某个领域的“赢家”，并向其投入数百万美元，以帮助其在国际市场上站稳脚跟。本月早些时候，这家日本企业集团向 Clearco(前 Clearbanc)投入了数百万美元，以帮助这家另类融资初创公司在欧洲、加拿大和美国以外的新地区发展。在这一点上，我想软银正在寻找那些有主见的初创公司，这些公司自然会走向国际，然后为它们提供大量资金。

阶级没有什么不同。Chasen 解释了自从 Class 宣布其种子期以来，国际上对该产品的需求一直很高。来自欧洲、中东和日本的学校在班级推出普遍可用性之前就已经联系了。现在，随着 Class 在 Mac、Windows、iOS、Android 和 Chromebook 上的全面推出，Chasen 正专注于将等待名单上的人转化为客户。

Class '的国际扩张将使其在英国和爱尔兰、EMEA、拉丁美洲和 APAC 等目标地区建立本地团队。这家初创公司预计将在现有 200 人的团队中增加 100 名来自世界各地的新成员。

Chasen 估计，65%的资金将用于推动 Class 的国际化，其余资金将用于产品开发。对班级的一个批评是，该平台为二年级学生提供了与高年级学生相同的体验。Chasen 同意，这家初创公司需要在未来的版本中为其产品增加更多的特异性——可能是 K-12 的游戏化和高等教育的考试监督。

“V1 为你提供了我们认为最起码的在线教学所需的东西，”他说，并提到了测试和成绩跟踪等功能。“目前，我们需要一款能在所有市场都运行良好的产品，未来我们将针对这些市场进行改进。”

到目前为止，用户是付费的。Class 表示，其收入在 2021 年环比增长了近 4 倍。

有 Zoom 福利的朋友

虽然围绕大融资和高估值有一种麻木效应，但 Class 最近的融资可能会打消人们关于它是否为最终被 Zoom 收购做好准备的疑问。

当 TechCrunch 第一次与 Chasen 交谈时，他说 Zoom 更关注规模，而不是 Class 想要提供的那种深度专业化。

尽管如此，该公司仍与 Zoom 的早期投资者合作，并在多个市场担任 Zoom 的经销商，这表明未来的整合不会太离谱。然而，今天过后，显然 Class 把自己视为一项独立的业务。创业公司不会只从精明的投资者那里筹集九位数的资金，除非他们有比整合更大的雄心。

展望未来，Class 可能会利用这数百万人中的一部分来建立自己的品牌，作为希望拥有课堂友好型缩放环境的学校或机构的首选。 Per Class' careers page ，市场营销是其目前最积极的招聘重点。该公司在营销团队中有六个公开的角色，包括国际营销经理和内容营销经理。

Class 最接近的竞争对手是 Engageli，它最近一次融资是在 2021 年 5 月，融资额为 3300 万美元。engeli 的联合创始人兼首席运营官杰米·法雷尔(Jamie Farrell)于 2021 年 2 月离开，去了另一家教育科技初创公司，该公司似乎并没有通过在线招聘板太积极地招聘员工。虽然细节是轶事，但 Engageli 可能会在带宽和营销方面面临更激烈的竞争，因为 Class 有了新的资本——以及不断增长的全球员工团队。
