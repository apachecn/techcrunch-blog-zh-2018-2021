搜索 API 初创公司 Algolia 以 22.5 亿美元的估值融资 1.5 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/28/search-api-startup-algolia-raises-150-million-at-2-25-billion-valuation/

Algolia 已经筹集了 1.5 亿美元的 D 轮融资，融资后估值为 22.5 亿美元。与 2019 年 10 月的 C 轮融资相比，该公司的估值翻了两番多。意味着 Algolia 现在是估值在 10 亿美元以上的独角兽。

该公司以其搜索即服务产品而闻名。它允许你使用开发者友好的 API 在你的应用或网站中集成实时搜索。使用 Algolia 支持的搜索功能就像在 Mac 上使用 Spotlight 一样。每次击键都会加载结果，并在几毫秒内显示出来。

该公司现在拥有超过 10，000 名客户，其中包括一些知名企业，如 Slack、Stripe、Medium、Zendesk 和 Lacoste。目前，该公司每年处理超过 1.5 万亿次搜索查询——如果你想看到所有的零，那就是 15 亿次。

Lone Pine Capital 主导了今天的融资。fidelity Management & Research Company LLC、STEADFAST Capital Ventures、Glynn Capital 和 Twilio 也参加了这一轮投资。但这还不是全部，一些现有的投资者也把更多的钱放在桌子上，如 Accel，Salesforce Ventures，DAG，Owl Rock 和世界创新实验室。

虽然该公司没有直接分享收入数字，但 Algolia 表示，其年度经常性收入同比增长了 180%。

“未来是 API 优先 Twilio、Stripe、Algolia 和其他 API 经济体的增长强调了这一现实。我们成功的很大一部分已经并将继续是我们通过 PLG 战略对开发者的不懈关注——使他们能够在自己的网站和应用中构建搜索，从而创造最相关和最动态的数字体验。”Algolia 首席执行官伯纳黛特尼克松在一份声明中说。“我们很高兴能够继续解决客户的问题，因为我们继续通过 Algolia 推荐和预测扩展搜索范围。"

除了搜索 API，Algolia 还扩展了其他实时 API。例如，你可以通过 Algolia 推荐在你的电子商务网站上提供实时产品推荐。这是公司产品多样化战略的一部分。

特别是，该公司现在正试图分析访问者的意图，以预测他们是否有可能购买某些东西。然后，公司可以利用这些信息来动态更新内容、发送推送通知、显示特价商品等。

这家公司最初成立于法国，在过去几年里发展迅速。Algolia 现在是一家以企业为中心的大公司，业务稳健。去年，其联合创始人兼首席执行官尼古拉斯·德塞格内决定转型为非运营角色。

该公司在过去 18 个月里招募了不少高管——米歇尔·亚当斯(Dropbox 的首席收入官)、卡尔顿·巴布(Alfresco 的首席财务官)、皮尤什·帕特尔(Capgemini 的首席业务发展官)、吉姆·沙汀(Alteryx 的首席客户官)、杰森·麦克莱兰(Salesforce 和 Adobe 的首席营销官)和巴拉特·古鲁普拉卡什(Twilio 的首席产品官)。

正如你所看到的，这是一个很长的人才名单，这意味着 Algolia 专注于建立一个长期的公司，而不是建立酷技术和优化收购。我不会对未来的首次公开募股感到惊讶。
