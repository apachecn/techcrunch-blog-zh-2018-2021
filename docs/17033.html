埃隆·马斯克称苹果的应用商店费用是“事实上的全球互联网税”

原文：https://web.archive.org/web/https://techcrunch.com/2021/07/30/elon-musk-calls-app-store-fees-a-de-facto-global-tax-on-the-internet/amp/

图片来源: 曹一川/NurPhoto / Getty Images

埃隆·马斯克在 App Store 垄断案中支持 Epic Games，特斯拉首席执行官周五早上发布了一条推文，称苹果的商店费用是“事实上的互联网全球税”，还补充说“Epic 是对的”

Epic Games 与苹果的法律战肯定会持续数年，这家堡垒之夜制造商也毫不掩饰其赢得舆论战的目标。马斯克的信任票可能会对尚未就该主题形成明确意见的消费者产生影响。

苹果公开表示，不满意的开发者可以将他们的产品带到 Android 或 iOS 上的移动网络，但 Epic Games 和其他公司表示，苹果对应用程序的束缚无异于垄断。

不太清楚的是，为什么马斯克现在开始讨论这个问题。虽然马斯克很少会放弃对一个不涉及他的有争议的问题提供外部意见，但马斯克目前的公司似乎都没有受到 App Store 收费的深刻影响，尽管幕后肯定有行动发生。
