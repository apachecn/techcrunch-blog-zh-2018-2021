Element Ventures 投资 1.3 亿美元，押注金融科技企业趋势

随着开放银行业务、PSD2 监管、insurtech 和整个金融科技热潮的兴起，科技投资者已经意识到，专用基金在这场持续的运动中有越来越多的位置。当你看到银行即服务产品、支付平台、保险技术、资产管理和基础设施提供商的崛起时，你会意识到正在发生一场相当巨大的革命。

根据 Dealroom 的数据，欧洲金融科技公司在 2021 年筹集了 123 亿美元，但市场仍然对 B2B 金融科技初创公司敞开大门。

因此，专注于 B2B 金融科技的 Element Ventures 宣布成立 1.3 亿美元的基金，对这一新的金融科技企业趋势进行双倍下注，也就不足为奇了。

Element 由金融服务资深人士斯蒂芬·吉普森(Stephen Gibson)和迈克尔·麦克法根(Michael McFadgen)创建，斯潘塞·莱克(汇丰银行前全球银行和市场副主席)也加入了该公司，它得到了以金融为导向的有限合伙人以及该行业约 30 名创始人和高管的支持。

Element 表示，它将专注于它所谓的“高信念投资策略”，这意味着每年只投资大约几家公司(该基金总共投资 15 家)，但它表示，将为其投资组合提供“高水平的支持”。

到目前为止，它已经支持了英国和欧洲的 B2B 金融科技公司，包括 Hepster(总共筹集了 1000 万美元)，德国的嵌入式保险平台，我最近在报道过；Billhop(共融资 670 万美元)，瑞典的 B2B 支付网络；Coincover(总共募集了 1160 万美元)，这是一家英国以外的加密货币恢复服务公司；以及瑞典的订阅管理平台 Minna(总共募集了 2500 万美元)。

Element Ventures 的合伙人麦克法根(McFadgen)在电话中对我说:“斯蒂芬和我一起投资 B2B 金融科技已经有很长时间了。2018 年，我们有机会启动 element，斯潘塞于 2019 年加入。因此，作为一家独立的风险投资公司，Element 实际上是我们长期参与的一项战略的延续。”

吉布森补充道:“我们对欧洲的运动和这些金融科技和保险科技公司在欧洲取得的突破非常信服。保险业一直是创新的沙漠，这种情况正在改变。你可以看到，我们试图围绕那些有突破性时刻的公司建立一个网络，不仅提供资本，还提供我们认为是故事一部分的所有其他东西。从 A 到 C 和 D 建立公司是我们试图卷起袖子帮助这些公司的领域。”

Element 说，它还将投资美国和亚洲。
