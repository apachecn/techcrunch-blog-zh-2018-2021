苹果的触控 ID 键盘终于可以独立使用了

在与触控 ID 推出神奇键盘三个半月之后，苹果公司终于从 iMac 套装中脱颖而出。该配件现在可以通过苹果商店和该公司的网站单独购买。

有两个版本:标准版和带数字键盘的加长版(几乎是该公司所有魔法键盘都提供的)，价格分别为 149 美元和 179 美元。还有一个 99 美元的版本，保持了新的圆形、紧凑的设计，但放弃了触控 ID，取而代之的是锁定系统的钥匙。但是这有什么意思呢？

图片来源:苹果

所有的型号都有专用于聚光灯、听写、请勿打扰和表情符号的按键(我最终禁用了我的表情符号，因为我无法避免意外按压🙁🙁).

所有这些都有一个重要的警告:触控 ID 只在运行 M1 芯片的 MAC 上工作，这使得目前市场上相当大一部分 MAC 失去了资格。如果你确实拥有一个这样的新系统，这个功能可以用于安全登录、购买等等。这种限制似乎是触控 ID 使用苹果新芯片上的安全飞地的结果。

图片来源:苹果

键盘包括一根编织的 USB-C 到 Lightning 电缆，尽管触控 ID 也可以通过蓝牙无线连接键盘。新推出的还有重新设计的魔法鼠标和触控板，售价分别为 79 美元和 129 美元。
