随着金融科技即服务的持续繁荣，Rapyd 以 87.5 亿美元的估值筹集了 3 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/03/rapyd-raises-300m-on-8-75b-valuation-as-fintech-as-a-service-continues-to-boom/

Neobanks、其他金融初创公司和“金融无处不在”的基本概念目前正获得巨大收益，今天，一家建设支持此类服务的基础设施的关键公司宣布了一轮重大的融资增长，以加倍抓住这一机会。

Rapyd 提供一系列金融服务，如支付、移动钱包、转账、信用卡发行、欺诈保护等，所有这些都通过一个 API 供第三方快速集成到他们自己的服务中。Rapyd 已经筹集了 3 亿美元，TechCrunch 从可靠来源了解到，该公司的 E 轮估值为 87.5 亿美元。

该公司去年一直在快速增长，这在很大程度上是受新冠肺炎疫情之后全球转向在线开展生活和业务的推动。Rapyd 的总支付量目标是今年超过 200 亿美元，比 2020 年的 50 亿美元增长四倍。该公司约有 12，000 家中小型企业使用其服务，另有 650 家大型企业客户。

Target Global 是欧洲风险投资公司，最近在金融科技和商业上下了一些大赌注，在这一轮中处于领先地位，新的支持者 Fidelity Ventures、Altimeter Capital、Whale Rock Capital、BlackRock 和 Dragoneer，以及以前的支持者 General Catalyst、Latitude、Durable Capital Partners、Tal Capital、Avid Ventures 和 Spark Capital 也参与了进来。这家初创公司过去的战略投资者包括支付巨头 Stripe 。

Rapyd 的首席执行官兼联合创始人 Arik Shtilman 在接受采访时表示，该计划将把部分投资用于收购，部分用于研发。

今年，Rapyd 已经开始认真进军 M&A，在 7 月份以 1 亿美元收购了支付和信用卡发行公司 Valitor，向欧洲进一步扩张，并成立了一家名为 Rapyd Ventures 的投资公司。收购可能会继续深入其当前市场。

在 R&D 方面，该公司已经有大约 900 种不同的服务，其 API 覆盖了 100 个国家。我曾经把 Rapyd 的方法比作类似于服务的“瑞士军刀”，Shtilman 说这些大致分为几个不同的类别。

“不管你是一家银行还是一家夫妻店，归根结底，地球上的金融服务有五个方面:收款、分散资金、存储资金、发卡和外汇。从这些你可以建立无限的能力，”他说。他补充说，现在的一个优先事项是专注于扩展与身份管理和欺诈相关的技术，以补充已经做的事情。

“了解你的客户[KYC]和合规工具将帮助我们更快地吸引更多的客户，”Shtilman 说。

最近一轮的时机对 Rapyd 来说是件大事。首先，就在 7 个月前，Rapyd 宣布了另一轮 3 亿美元的 D 轮融资(我发现那轮融资实际上是在 11 月结束的)。值得注意的是，最后一轮的估值为 25 亿美元，尽管该公司没有披露其融资情况，但 Shtilman 告诉我，自那以来，收入增长了 3.5 倍。一位非常接近该公司的消息人士告诉我，估值简单来说就是这两个数字的倍数:87.5 亿美元。

其次，在更广阔的市场背景下，以及 Rapyd 在其中的位置，这很重要。

我们目前正看到大量公司挖掘所谓的嵌入式金融(由一方构建和运营并通过 API 集成到另一方的服务中的金融服务)的潜力，以开发新产品，从世界各地的新银行到建立结账服务的电子商务公司，或者与商业业务无关的公司，他们现在推出产品以提高客户参与度或首次进入该领域。

这对 Rapyd 来说意味着很多竞争，其他一些大玩家包括 Fast 、 Checkout 、 Mambu 和 Railsbank ，它们也都筹集了大量资金。

换句话说，这一轮不仅是 Rapyd 自身成长的标志，也是向市场发出的信号，表明它如何定位自己，并在这个正在形成的有趣和竞争激烈的领域取得进展。

Rapyd 现在总部位于硅谷，但其 R&D 和首席执行官位于特拉维夫。rapyd 是该领域的早期参与者之一，Shtilman 喜欢回忆当他第一次创办该公司时，他遇到了金融服务界其他人的许多质疑，不仅因为这个想法听起来太难执行，还因为他们认为金融基础设施基本上是大多数金融服务公司皇冠上的宝石。

“当我们在 2016 年开始时，所有人都认为我们疯了，因为这个概念太大太广了，”他说。“然后，像雨后的蘑菇一样，所有人都看到了。每个人现在都明白，金融科技的未来是金融科技基础设施。比如云计算。”

事实上，这需要一点时间，但最近大多数人都承认，这些服务的基础非常难以构建，但本质上对每个人来说都是一样的，所以它们可以构建一次，然后打包并转化为你可以通过云服务利用的东西，从而转化为你可以花费资源、时间和策略进行个性化的商品。

同时，皇冠上的宝石实际上是你的顾客。因此，构建交易服务，要么补充你已经做的业务，要么以一种有趣和有用的方式增强它，这就是你最终如何加深与他们的接触和他们的承诺。

事实上，这可能适用于今天的任何在线业务，这意味着机会也是巨大的，这是投资者如此热衷于这个市场的一个原因。(这不仅仅适用于 Rapyd，因为它足够大，不会成为一个赢家通吃的市场。)

“Rapyd 建立了一个无边界的嵌入式金融科技基础设施，这对所有在全球运营的数字企业都至关重要。他们的平台包括支付、合规、外汇、欺诈管理、托管、虚拟账户和信用卡发行等。但现在，随着世界看到全球电子商务、演出经济、金融科技解决方案和技术平台的增长，Rapyd 必须采取下一步行动，”Target Global 普通合伙人迈克·洛巴诺夫在一份声明中说。“目前，我们前所未有地需要一个单一的合作伙伴来充当众多本地支付服务和商户之间的桥梁，为他们提供实现繁荣所需的灵活、快速集成和可扩展的解决方案。在 2018 年带领了 Rapyd 的 A 系列赛后，我们有信心 Rapyd 可以成为这样的合作伙伴，现在正在这一轮重新下注。”
