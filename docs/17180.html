批量支付初创公司 Comma 筹集了 600 万美元种子资金，由 Octopus 和 Connect 领投

总部位于英国的开放式银行批量支付初创公司 Comma 在由八达通风险投资公司(Octopus Ventures)和 Connect Ventures 牵头的种子轮融资中筹集了 434 万英镑(600 万美元)。投资者 Village Global、Wagestream 的创始人彼得·布里菲特(Peter Briffett)和波特曼·威尔斯(Portman Wills)也加入了他们的行列。

该公司表示，它使小型和微型企业能够使用现有的高街小型企业银行账户批量支付账单、工资和税款，节省了他们大量的管理时间和资金。这是因为 BACS 很难获得并且成本很高，虚拟账户需要 KYC 来建立并且增加了记账的复杂性。

Comma 表示，在疫情期间，外包金融业务大幅增长。几家高街银行的开放银行批量支付 API 的可用性使得该产品成为可能，这使得这家初创公司获得了更多的业务。

Comma 应用程序连接到会计系统(Xero、QuickBooks、Sage ),允许企业或其会计输入和管理供应商银行详细信息，而无需访问银行。他们可以一次支付 15-50 名收款人，系统会根据账单将支付情况发送回会计系统，以标记已支付的金额。

创始人汤姆·贝肯汉姆说:“我是一家公司的首席运营官，这家公司的账单遍布各大洲，给数百名员工发工资。这是一个非常手工的过程。我突然想到，大型企业拥有企业银行和支付管理系统。小企业没有，而且在很大程度上被忽视了。我注意到，为小型企业解决问题的传统方法具有很高的设置成本，从而淘汰了大部分市场。”

他说，他看到了一个利用新的开放式银行技术进入这个长尾企业并全面解决支付问题的机会:“到目前为止，我们的支付金额刚刚超过 100 万英镑。到本季度末，我们将达到每周 100 万部。”

在英国，这家初创公司在一定程度上与 Credec、Telleroo 和 ADP 等 BACS 机构存在竞争。在国际上，美国的 Melio 是最接近的比较。法国的 Libeo 也提供类似的服务。
