创建非洲宪章的初创公司 Raise 获得了 500 家初创公司的支持 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/06/raise-a-startup-building-africas-carta-gets-backing-from-500-startups/

随着非洲的初创公司继续以惊人的速度增长和融资，他们的 资本表也将扩大。大多数非洲初创公司的大部分风险投资资金来自外国投资者，这使得非洲初创公司必须在国外组建公司，尤其是在美国

成立公司的过程相当复杂，尽管大多数创始人仍然掌握了其中的诀窍，但他们仍有可能搞砸自己的资产负债表。例如，一些尼日利亚初创公司发行了奈拉优先股，然后在美国注册成立时取消了美元计价证券的发行

建立非洲 Carta 的初创公司 Raise 正在应对这些挑战，并获得了 500 家初创公司的支持，以扩大其技术规模。

2019 年，Marvin Coleby、Tina Nyamache 和 Eugene Mutai 开始创建一个区块链解决方案，让人们更容易买卖非洲上市前公司的股票。经过几次迭代后，他们发现大多数公司仍然在股权和流动性的概念上挣扎。 他们花钱为特拉华州、加拿大和欧洲的控股公司管理公司结构 但 在非洲各地维持基于纸面的子公司。

根据 Coleby 的说法，整个非洲的大部分股权仍然使用纸质证书、手动流程和分散的政府数据库进行存储、跟踪和更新。这增加了管理子公司和发放员工股票期权的交易成本。这也增加了进入和退出私营和上市公司的成本。

因此，他们开始帮助初创公司、投资者、员工和律师事务所管理交易、资本表和公司合规 。

在该平台上，Raise 客户还可以自动执行尽职调查、设定估值、跟踪员工股票归属，以及为尼日利亚和肯尼亚的许可证和政府文件制作常规文档。

当 2019 年 Raise 推出时，它处于私下测试阶段， 得到了 币安实验室的支持，后者是其 pre-seed 轮 的唯一投资者。自 2020 年进行公开测试以来，Raise 已经接纳了一些客户，如非洲最大的律师事务所Anjarwalla&Khan na；初创公司 Bamboo、Workpay 和 Mono 和蛹资本这样的风险投资公司。

科尔比在一次电话会议上告诉 TechCrunch，但融资试图解决的长期问题是流动性。

“我们所做的一切都是为了找到一种方法，让创始人、客户、员工和投资者更容易从投资公司中获得流动性，”他说。“公司在筹集资金，人们在投资，员工在获得股票期权。 然而 ，偶尔只有一两个出口。这是因为我们建立了硅谷模式，在那里我们必须增长，扩大规模，直到我们获得一些大的出口。从我们的角度来看，流动性不必如此。随着时间的推移，员工和投资者可以获得很小很小的流动性。”

以这种标准来衡量，非洲的私有和公有公司资本市场的流动性很差。购买或出售股票需要几个月或几年的时间，根据 Raise 的说法，非洲超过 1 万亿美元的股票是“非流动性的，基于纸张的，并以通货膨胀的货币定价。”

尼日利亚的股票交易平台，如 Chaka 、Bamboo 和 Trove，帮助尼日利亚人在当地和国际创造资产流动性。 然而 ，Raise 旨在搭建他们背后的平台，以精简更多的资产类别和投资机会。

虽然这仍在进行中，但 Raise 为非洲公司组织了所有权数据，并使它们可以访问。这类似于价值 30 亿美元的 Carta 公司为美国公司提供的 cap table 软件。

随着时间的推移，入职工资表和股权数据也将为 Carta 成为总部位于区块链的数字资产平台提供使用案例 。该计划将变得更像非洲私营公司的纳斯达克，因为我希望为它们出售指数、ETF、期货和资产。科尔比说，通过这种方式，融资将成为处理非洲数千亿美元贸易和证券交易量的股权引擎。

科尔比表示，上线的公司数量环比增长 60%。该平台管理着大约 200 个 cap 表，资产价值超过 4 亿美元。根据科尔比的说法，下一个增长阶段将是 A 轮和成长期公司登上平台。

这家公司活跃在尼日利亚和肯尼亚。Coleby 表示，一轮种子期正在进行中，以继续深入这些市场，并在非洲风险投资领域进行融资和流动性操作实验。

接下来，Raise 正在建立一个市场，继续在一个平台上连接和教育投资者、员工和创始人与他们的律师事务所，使用可信和经过验证的数据进行交易，并向员工发放股票期权。
