第一家进入 YC 的赞比亚创业公司正在开发非洲第一个发卡 API

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/06/the-first-zambian-startup-to-get-into-yc-is-developing-africas-first-card-issuing-api/

在过去的十年里，来自少数几个国家的 40 多家非洲创业公司已经走过了 YC。赞比亚今天加入了这个名单，它的新成员 Union54 是一个有价值的第一个成员。

Union54 (54 是对非洲国家数量的认可)是一家金融科技公司，由 Perseus Mlambo 和 Alessandra Martini 创办。这家初创公司声称是非洲第一个发卡 API，今年刚刚推出。 但要描绘这幅画面，Union54 并不是凭空而来的；这是这对夫妇早期创业公司沙祖的一个项目。

沙祖 于 2015 年作为挑战者银行在赞比亚推出 。与非洲大陆的任何金融科技一样，沙祖必须创建自己的借记卡，让用户可以连接到钱包。大多数情况下，沙祖要等几个月才能让国内的合作银行发行这些信用卡。姆兰博告诉我，他们一度不得不等了 18 个月。

与此同时，创始人开始与该地区的银行合作，开始自己发行信用卡。但是银行在他们的方法上是迟钝的。“我们 刚刚 意识到，无论是处理器还是银行都不一定能够很好地回答我们的问题，或者能够给我们提供我们正在寻找的产品，”姆兰博在接受 TechCrunch 采访时说。

这家初创公司决定瞄准靶心，与万事达会面。我的意思是，当你能从一开始就抓住发行这些卡的人时，为什么还要等银行呢，对吗？ 最终 ，该公司获得了万事达卡原理会员资格，第一家来自非洲的 fintech，它自称 。

作为主要成员，沙祖被授权作为“开证行”换句话说，他们可以提供借记卡并作为“收单方”，这意味着他们可以提供交易处理服务。

一路走来，创始人们意识到 到真的推进了非洲金融科技，当务之急是让任何非洲国家的金融科技更容易发行虚拟或实体借记卡。所以这个团队从沙祖分离出 Union54。该平台现在有几个API，使任何金融科技公司都能轻松发行可编程借记卡 。

“我们现在已经利用我们的会员资格来帮助其他公司，任何想要发行自己的卡 的非洲金融科技公司。他们可以 直接 来找我们，插入我们的 API，然后快速，而不需要花很长时间谈判，”姆兰博在谈到为其他非洲金融科技公司提供服务时说。

这位首席执行官补充道，该公司的目标客户是那些不想花费数十万美元的安装费来获得虚拟或实体卡的金融科技公司。Union54 声称通过一个 API 在几周内发行卡，该 API 提供 BIN 赞助、程序管理和结算等功能。

能够做到这一点让 Union54 成为非洲第一个发卡 API。 Fintechs 很少关注这个机会；大多数 都聚焦于 从支付网关到钱包 的其他环节。这是一个有趣的注意点，因为不知何故，这些领域的所有大玩家最终都试图为他们的客户创建虚拟和物理卡，并面临这样做的复杂性。这是 Union54 想要填补的空白，虽然它目前处于测试阶段，但该公司拥有一个令人印象深刻的匿名客户，他们已经在等待名单上注册并正在使用该平台。

“这些公司的迷人之处在于，它们不是 B 或 C 玩家。他们是非洲金融科技 前 5%的人。对我来说，我总是告诉人们，我们现在是非洲金融科技的黄金一代。 所以现在是一个发卡产品能够与所有这些人都被认为是他们领域的领导者 。这意味着我们 真的 有人们每天都想用的东西，”这位首席执行官补充道。

在该公司的网站上，其 API 有八个用例:基于分类账、收单机构/网关、立即购买、稍后支付、信用合作社、送货公司、数字银行、信用卡管理和公司卡。

使用 Union54 的金融技术人员还可以设计卡并设置他们希望卡被支付的货币，并设置一个广泛的目录，包括谁将使用它们，它们将 用于什么 ，它们将 用于什么时候 以及它们将如何 被使用

Union54 对每一个 API 调用都按照现收现付的方式收费。如果一家金融科技公司想要创建一张实体卡，他们会被收取 7-9 美元之间的固定费用以及交易时未披露的固定费用。

姆兰博表示，进入 YC 2021 的夏季批次让该公司签下了第一批客户，因为他们大多数来自 YC 的网络 。他称 YC 是一个“从第一天起就值得”的项目

“对于 Union54 成为 Y Combinator 接纳的第一家赞比亚 fintech，我真的 感到兴奋和自豪。第二个在南部非洲。正如你所知，当全球投资者关注非洲时，他们通常是从西非的角度出发，我们进入 Y Combinator 验证了我们更广泛假设的一小部分:从赞比亚等友好管辖区为非洲服务是可能的。”
