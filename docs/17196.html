zeroheight 在部落资本的带领下筹集了 1000 万美元，用于扩大 UX 团队的设计项目

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/06/zeroheight-raises-10m-round-led-by-tribe-capital-to-scale-designops-for-ux-teams/

zeroheight 在部落资本的带领下筹集了 1000 万美元，用于扩大 UX 团队的设计项目

网站和应用程序的高质量 UX 不再是一个好东西，如果一个公司想要成功，它是一个必须拥有的东西。但是扩大 UX 团队的影响并不简单，近年来，团队已经转向所谓的 DesignOps 平台来帮助他们。

现在，一家新的创业公司希望成为 UX 团队的关键设计平台，并筹集了资金来帮助它扩大规模。

Zeroheight 现在已经筹集了 1000 万美元的首轮融资，由 Tribe Capital 牵头，参与方包括 Adobe、Y Combinator、FundersClub 和 Expa，以及天使投资人，包括汤姆·普雷斯顿-沃纳(GitHub 的联合创始人)、布拉德利·霍洛维茨(谷歌产品副总裁)、艾琳·欧(为谷歌建立和运营 UX 设计)和尼克·考德威尔(Twitter 工程副总裁)。

总部位于伦敦的 zeroheight 现在将扩展到旧金山/海湾地区，并全面发展团队。到目前为止，它的重点是 UX 文档，但现在它也将探索其他领域，如缩小设计和开发之间的差距。

联合创始人杰罗姆·德·拉法格说:“zeroheight 为 UX 做的事情就像 GitHub 等 DevOps 平台为构建和发布代码做的事情一样，提供了一个记录和管理 UX 组件的中心位置，再加上设计 API，允许团队完全跳过设计移交阶段，加快 UX 交付流程。”

他说，公司解决了 UX 团队的规模问题:“问题已经出现，因为 UX 团队在过去几年里急剧增长，因为 UX 现在对大多数公司来说非常重要，只需竞争。正因为如此，你现在需要集中化，你需要可重复使用的组件，以便团队能够高效工作，并且在运送过程中不会降低质量。”

Zeroheight 拥有 1300 多家客户，其中包括几家财富 500 强公司，如 Adobe 和联合航空公司。
