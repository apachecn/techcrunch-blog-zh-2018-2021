立法者询问亚马逊计划如何处理掌纹生物识别数据

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/13/amazon-biometric-data-senate-letter/

立法者询问亚马逊计划如何处理掌纹生物识别数据

一群参议员周五给亚马逊新任首席执行官安迪·贾西(Andy Jassy)写了一封信，敦促该公司提供更多信息，说明它如何扫描和存储顾客的掌纹，供其一些零售店使用。

该公司通过一个名为 Amazon One 的项目推出了掌纹扫描仪，鼓励人们在实体店进行非接触式支付，而无需使用信用卡。亚马逊去年年底推出了亚马逊 One 扫描仪，现在可以在美国各地的亚马逊 Go 便利店和杂货店、亚马逊书店和亚马逊四星级商店找到这种扫描仪。这种扫描仪还安装在华盛顿州的八个全食超市。

在这封新的信中，参议员艾米·克洛布查尔(D-MN)、比尔·卡西迪(R-LA)和乔恩·奥索夫(D-GA)向杰西询问了亚马逊计划如何扩展其生物识别支付系统的细节，以及收集的数据是否有助于该公司定向广告。

“亚马逊通过 Amazon One 扩大生物特征数据收集的做法，引发了对亚马逊这些数据的计划及其对用户隐私的尊重的严重质疑，包括亚马逊可能如何将这些数据用于广告和跟踪目的，”参议员在信中写道，信的内容如下。

立法者还要求提供信息，说明迄今为止有多少人注册了亚马逊一号，亚马逊将如何保护敏感数据，以及该公司是否曾将掌纹与它在其他地方收集的面部识别数据进行配对。

“与苹果的 Face ID 和触控 ID 或三星 Pass 等将生物识别信息存储在用户设备上的生物识别系统相比，亚马逊 One 据说将生物识别信息上传到了云端，这带来了独特的安全风险，”参议员们写道。“…对于不可变的客户数据，如掌纹，数据安全性尤为重要。”

该公司颇具争议地为在该程序中注册掌纹的新用户提供了 10 美元的积分，这引起了隐私倡导者的强烈抗议，他们认为这是强迫人们交出敏感个人数据的廉价策略。

有足够的理由持怀疑态度。亚马逊因其另一个大型生物识别数据项目而面临激烈批评，该项目是被称为 Rekognition 的人工智能面部识别软件，该公司向美国执法机构提供了该软件，最终在去年因暂停该软件的警务应用而收回。
