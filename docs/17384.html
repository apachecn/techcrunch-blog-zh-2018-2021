非洲金融科技 Pngme 为其金融数据基础设施平台融资 1500 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/17/african-fintech-pngme-raises-15m-for-its-financial-data-infrastructure-platform/

随着主要参与者继续为规模 筹集更多资金，通过 API 分解金融数据并利用增值产品在非洲推动数据驱动的洞察力变得越来越令人兴奋。

总部位于旧金山和非洲的 fintech Pngme 在获得 300 万美元种子资金不到一年后，又为其金融数据基础设施项目 追加了 1500 万美元。该公司还将自己描述为一个机器学习即服务平台。

Octopus Ventures 领投 A 轮，跟投方包括 Lateral Capital、EchoVC、Raptor Group 和两家小鱼创投。其他投资者如 Unshackled Ventures、Future Africa、位于拉各斯的 Aruwa Capital 和 The51 也参与了进来。 Pngme 也收到天使投资人的支票；其中包括 RallyCap 的海登·西蒙斯、格子公司的丹·卡恩、加拿大皇家银行资本市场公司的前首席运营官理查德·塔尔博特和 Intersect VC 的凯尔·埃利科特。

Pngme 的平台面向撒哈拉以南非洲的金融科技公司和其他金融机构。 当创始人布兰登·普莱福德(Brendan Playford)和凯特·朗格(Cate Rung)最后一次与 TechCrunch 交谈时，Pngme 正在尼日利亚、肯尼亚和加纳 退出隐身模式。

目前，Pngme 在这三个市场为客户提供三种核心产品。 除了 到其已经为人所知的 API 和 mobile SDK，Pngme 又增加了一个客户管理平台。该公司表示，结合这三种产品将推动其客户采纳和使用个性化的用户体验和金融产品。

在与 TechCrunch 的一次对话中，Playford 提到了支付宝和微信在过去几年中所做的建立个性化用户金融科技体验的工作 。

当用户注册时，两个平台 在他们下定决心提供合适的产品之前，为 提供每一项金融服务的正确推荐 。

“这是一种高度数据驱动的用户体验。每一家金融科技公司或银行都希望提供同样的数据驱动型用户体验。 从即时贷款或储蓄，或透支，或无论它可能是什么，它都是 只是 就像围绕产品的用户体验，”普莱福德说，他指的是中国超级应用巨头 。“如果你抓住所有金融机构的业务问题的核心，他们正在做两件事。一是他们正在寻求降低客户获取成本。然后，他们会考虑增加客户的终身价值。”

Playford 表示 Pngme 希望效仿这种做法。但是为什么它突然变得对公司很重要了呢？

大多数非洲金融机构和金融科技公司都竞相提供完全定制的用户体验和根据客户需求量身定制的金融产品 。为了推动这些产品和用户体验，需要数据基础设施。机器学习模型 假设被训练获得保留 并最大化客户的终身价值。

这些过程可能既昂贵又耗时，让他们面临着在建设基础设施还是服务客户之间做出选择的艰难任务。

Pngme 允许金融机构和 fintechs 以 的规模采集 汇总 的金融数据。该公司表示，其移动 SDK 和数据处理管道收集替代金融数据，并将其与其他数据源相结合，以创建个人金融行为的整体画面 。

“我们解决的难题是构建基础架构的成本非常高。而数据科学、数据工程人才， 只是 在全球范围内是 真的 难求。

png me A 轮的结束使其总投资达到 1850 万美元，成为非洲大陆此类金融科技领域获得资金最多的公司。其他著名的初创公司包括总部位于南非的 Stitch ，以及尼日利亚的 Okra 、 Mono 和 OnePipe。

虽然每个平台都演变成提供更复杂的数据产品，但 Playford 表示，自 2 月份以来，Pngme 一直在考虑的一件重要事情是将自己与其他平台明显区分开来。

“我们所做的是，我们已经有点儿 真的 区分我们自己不是 只是 收集我们可以看到的数据，但我们也可以连接到 Mono 数据、Okra 数据，我们可以连接到银行的数据 。我们 本质上是 合并所有数据，然后为客户端将机器学习模型放在最上面。 可以预测的信贷模型、细分模型和 真正的 将自己定位为银行和金融科技公司的数据处理基础设施 。”

Playford 对他认为 Pngme 与众不同的解释与其他创始人对自己平台的看法产生了共鸣。但是时间会告诉我们这些产品的差异还会持续多久。

Pngme 的提议吸引了尼日利亚和南非一些未透露姓名的一级银行；fintechs Kuda、Umba、Renmoney、CredPal 和像 TransUnion Africa 这样的信用机构。

它表示，Pngme 将利用对的投资，获得更多的 客户。该公司计划实现这一目标的方法之一是扩大其高管团队。Pngme 聘请 Lorraine Kageni Maina 担任首席战略官，Nick Masson 担任首席技术官。

除了招聘关键高管，Pngme 还在全球范围内扩展其数据科学、工程和销售团队 。 首席运营官·荣格表示，Pngme 的基础设施已经处理了来自撒哈拉以南非洲数百家金融机构的数十亿个数据点。下一个计划是在下一年加倍投资其 Insights Library 产品，并将第三方数据连接扩展到其他市场 。

对于本轮的主要投资者 Octopus Ventures 来说，Pngme 表明需要可操作的数据来推动非洲数字金融科技服务的爆炸 。

关于风投公司投资的原因，Tosin Agbabiaka 说:“技术解决方案的优雅，加上优秀的团队和对大型机构的强大市场吸引力，强调了我们的信念，即 Pngme 将推动非洲下一代金融服务，帮助数百万人获得银行和贷款 。”
