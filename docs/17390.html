Brex 以 5000 万美元的价格收购了 Weav，这是一个面向商业平台的通用 API

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/17/brex-buys-weav-a-universal-api-for-commerce-platforms-for-50m/

去年夏天，fin techBrex首次与商业平台通用 API 开发商 Weav 合作。

今年 3 月，Brex 利用这家初创公司的技术，为 Shopify 卖家推出了即时支付服务。

结果令人印象深刻，到 4 月份，Brex 联合创始人 Henrique Dubugras 和 Pedro Franceschi 作为战略天使投资人参与了 Weav 的 430 万美元种子轮 。

在过去的几个月里，这两个人认为 Weav 的技术和团队太好了，不值得分享。TechCrunch 独家获悉，今天，金融科技公司宣布以 5000 万美元收购一岁的 Weav，这是其首次重大收购。

有趣的是，这笔交易是在两家公司的创始人都没有见过面的情况下达成的——这在新冠肺炎疫情奖之前可能更不寻常，但如今可能更常见了。(虽然他们后来见过面。)Brex 此前曾进行过“收购”,但此前从未收购过一家公司的团队和技术。

Dubugras 说，Brex 在 Weav 公司成立的“很早”就开始作为合伙人与它合作。

他告诉 TechCrunch:“我们对(首席执行官)Nadav(Lidor)和他的团队印象深刻，他们的建设速度如此之快，技术如此之好，我们希望扩大到更具战略意义的合作关系。”“然后，我们开始谈论收购。”

TechCrunch 采访了 Dubugras 和 Weav 首席执行官兼联合创始人 Lidor，以了解交易的细节，以及为什么这对两家公司都很重要。

首先，作为收购的一部分，Brex 将建立一个“创新中心”,并在 Weav 位于以色列和纽约的九人团队之外，在以色列雇佣员工，从而扩大其全球影响力。首席执行官 Lidor 将领导 Brex 的新以色列办事处。

Dubugras 表示，除了扩大其全球影响力，Brex 收购的技术将有助于加速其平台的金融科技连接。目前，Brex 在一个单一的仪表板中为其客户(主要是初创公司)提供信用卡、商业现金账户、支出管理和账单支付软件。其目标是继续扩大其产品和服务组合，成为“一个完全集成和全面的企业金融平台。”

“Weav 的技术帮助 Brex 为我们的客户提供更好的服务，”他说。

Weav 是去年由工程师 Ambika Acharya、Avikam Agur 和 Lidor 在参与 W20 YC 批次后创建的，是旨在推动金融科技和金融机构发展的金融科技基础设施公司之一。具体来说，Weav 的嵌入式技术旨在让组织能够访问“实时的、用户许可的”商业数据，他们可以使用这些数据为小企业创建新的金融产品。

其产品将允许客户通过一个 API 连接到多个平台，该 API 是专门为企业用来销售产品和接受支付的商务平台开发的。Weav 的经营前提是，允许公司建立和嵌入新的金融产品，为电子商务商人、创造者和其他企业家创造新的机会。

自去年成立以来，Weav 的 API 调用量每月增长 300%。

云计算和 SaaS 技术的日益普及导致数据存储在各种不同的系统中。Weav 的 API 旨在建立能够自动共享和分析的数字连接，从而(如上所述)允许商务平台 实时访问其客户的标准化交易数据。这对 Brex 很重要，因为它相信通过使用 Weav 的技术，使用其平台的企业将能够“更快、更准确地”获得金融服务和新产品

“我们想要建立一个一体化的金融平台，”Dubugras 告诉 TechCrunch。“这已经是我们合作的方向，但这次收购有助于我们更好地整合我们的所有金融产品，我们可以做得更多，比我们最初的计划快得多。”

例如，他补充道，Brex 与 Shopify 等平台整合。通过收购 Weav，它打算为其客户开发更多贷款、可视化和洞察产品。

“Weav 团队将基本上管理任何第三方集成，”Dubugras 说，“这样无论你的数据在哪里，Brex 都可以成为你的财务操作系统。你可以在一个地方拥有一切。”

Lidor 承认，Weav 成立后没想到会这么快退出。但他说，这些公司发现自己在同一页上。

“我们的目标一直是将企业、创造者和其他企业家与金融科技联系起来，以扩大金融服务，这与 Brex 的使命相一致，”Lidor 补充道。“在与恩里克和佩德罗合作后，我们意识到他们是再好不过的搭档了。Brex 团队给我们留下了深刻的印象，我们从他们那里学到了很多东西，并与他们一起建设。”

今年早些时候，该公司在 430 万美元的种子融资中没有披露其估值。杜布格拉斯说，5000 万美元的价格代表了“所有参与者的健康倍数”。

向以色列的扩张也令 Brex 团队兴奋不已，该团队去年远离新冠肺炎疫情，在美国、加拿大和巴西开展业务。

总部位于旧金山的 Brex 成立于 2017 年，今年早些时候在由 Tiger Global 领投的 4.25 亿美元 D 轮融资后，估值达到 74 亿美元。根据 Crunchbase 数据，该公司已经通过债务和股权融资筹集了 12 亿美元。

今年早些时候，该公司宣布推出一项新服务，名为 Brex Premium，每月收费 49 美元。

“我们现在的付费用户数量绝对超出了我们的预期，”杜布格拉斯说。

2 月，Brex 向最新的 fintech 申请银行执照 。

该公司销售一种专为初创企业定制的信用卡，目前由移民银行担任发行人， 该公司已向联邦存款保险公司(FDIC)和犹他州金融机构部(UDFI)提交了设立 Brex 银行的申请。

本月早些时候，该公司表示将自愿撤回银行执照和联邦存款保险申请。

该公司表示:“这将允许我们修改和加强我们的申请，并在晚些时候重新提交。”"我们感谢犹他州 DFI 和联邦存款保险公司的支持和周到的指导."
