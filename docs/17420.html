API 平台 Postman 在 2.25 亿美元的融资中价值 56 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/18/api-platform-postman-valued-at-5-6-billion-in-225-million-fundraise/

总部位于旧金山的 Postman 为开发者运营一个合作平台，帮助他们构建、设计、测试和迭代他们的 API，该公司周三表示，它在新一轮融资中筹集了 2.25 亿美元，估值为 56 亿美元，高于一年前的 20 亿美元。

这家初创公司的新一轮融资——D 轮——由总部位于纽约的现有投资者 Insight Partners 领投，现有投资者 Nexus Venture Partners 和 CRV 也参与了投资。包括 Coatue、Battery Ventures 和 BOND 在内的新投资者也参与了新一轮融资，使各轮融资总额超过 4.3 亿美元。

API 为开发人员提供了一种将其应用程序连接到其他内部和外部应用程序的方法。但直到过去十年，还没有多少公司试图精简这个领域。(开发人员依赖于——并且许多人继续这样做——开源 CLI 工具，如 curl 和 HTTPie。也就是说，邮差现在有很多竞争对手，包括 Stoplight、a16z 和老虎环球支持的孔。)

雅虎的前实习生阿比纳夫·阿斯塔那直接面对了这种挫折，为自己和朋友们开发了一个 Chrome 扩展。

他不知道有多少开发商和公司也需要它。

这家成立六年的初创公司的产品始于印度，如今被超过 1700 万开发者和超过 50 万个组织使用，包括微软、Salesforce、Stripe、Shopify、思科和 PayPal。

这个名单很长:Postman 的联合创始人兼首席执行官阿斯塔那告诉 TechCrunch，98%的财富 500 强公司都是 Postman 的客户。

“我们正在解决技术领域的一个根本问题。两年前，他告诉我:“大公司往往速度较慢，因为他们有许多其他事情要做。

Postman API 平台的产品。图片来源:邮递员

“当今世界每个行业的每个公司都在使用 API，都需要一个 API 平台。这种趋势只会随着向云和数字体验的转移而增长，”他在最近接受 TechCrunch 采访时表示。

如今，这家初创公司引领着市场。这可以解释投资者的兴奋。据一位知情投资者称，Postman 拒绝分享其收入，但其新一轮融资的金额是其 100 多倍。

Postman 的平台对开发者来说至关重要，但直到最近，这家初创公司才扩展到为开发者和公司创建一个公共市场，以找到现成的 API 来使用。

“邮差公共 API 网络连接了全球数百万开发者，为他们提供了一个致力于发现、探索和分享 API 的空间。这最终是由我们创建的公共工作空间推动的，它允许用户跨不同的组织进行连接，”Asthana 说。

“随着 API 的出现，我们相信这将迎来下一代无代码和‘公民开发者’我们鼓励一个充满创新的世界，让不同背景和不同技术水平的人都能参与进来。他说:“我们越来越多地看到销售、营销和财务人员越来越适应 API，并成为这项技术的冠军。

这家初创公司拥有超过 425 名员工，计划部署新的资金，在销售、营销、产品和工程部门雇用更多员工。

Postman 还将“大量”投资拓宽其产品路线图。“我们正在将 Postman 平台扩展到技术用户需要的领域，同时支持业务用户的需求。在高层次上，我们正在投资支持各种 API 的工作流——无论是私有 API、合作伙伴 API 还是公共 API，”他说。

路线图上即将出现的一些项目包括 Postman API 的新版本、对 gRPC、ProtoBuf 等协议的支持以及 GraphQL 更广泛的功能。“我们还非常重视与软件开发生命周期中的其他供应商的集成，如 AWS、GitHub 和 GitLab 等 Git 托管提供商。我们还发布了我们的 Flow Runner 工具，这是一个无代码 API 组合工具，使任何人都可以构建 API 驱动的程序。”

这家初创公司还计划通过 API 扫盲项目投资支持学生，并为开源项目做出贡献。

Insight Partners 董事总经理杰夫·霍林(Jeff Horing)在一份声明中表示:“API 已经迅速成为全球每个国家、每个行业的开发人员使用的软件的基本构建模块，Postman 已经牢牢确立了自己作为开发人员首选平台的地位。”。

“Postman 有机会成为企业如何构建、交付产品以及在整个生态系统中无缝实现合作伙伴关系的关键支柱。他们持续、快速的扩张和强大的管理团队为 Postman 指出了一个几乎无限可能的未来。”
