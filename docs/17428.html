创业公司/风险投资

我们最新的技术专家一直在努力工作，这意味着我们已经可以突出她的一些报道了。这是韩国二手市场 Danggeun Market 上的 Park，其最近一轮 1.62 亿美元的融资使该公司的估值达到 27 亿美元。

在我们进入融资阶段之前，梅菲尔德的投资者 Navin Chaddha 想知道风险投资发生了什么。他的文章是对信息中某一特定部分的回应。我们以前探索过这个概念，但是 Chaddha 的笔记非常值得一读。

现在，今天剩下的重要创业新闻:

  • API 是大生意: 事实上，提供 API 的初创公司并不是唯一一家通过构建开发人员友好的服务模式来筹集资金的公司。Postman 刚刚以 56 亿美元的估值融资 2.25 亿美元，帮助其他开发者构建 API。这样其他开发者就可以更容易地插入科技产品。邮差与红绿灯和孔等竞争。
  • 虫蛋白 cometh: 食品科技工整。谁不喜欢初创公司正在研究各种替代蛋白质，对吗？养牛对环境是垃圾，我们都知道。但是昆虫呢？Beta Hatch 刚刚为其专注于昆虫的蛋白质研究筹集了 1000 万美元。或许未来是脆弱的。
  • KaiPod 押注微型学校:2020 年和 2021 年 edtech 创业活动热潮中的一块，就涉及到了 Pod，或者说“微型学校”根据我们自己的 Natasha Mascarenhas 的说法，总部位于波士顿的 KaiPod 认为这种模式将会持续下去，并开始关注家庭学校市场。
  • 中等收入 fintech？金融科技领域的一个巧妙元素是利用金融科技来降低金融服务的成本，并为服务不足的社区带来更多的资金管理工具。一个是不这样做。相反，该公司正在为中产阶级打造金融科技。据玛丽·安·阿泽维多报道，这家初创公司希望打造一个“一体化”解决方案。
  • 为潮湿的未来做计划: 这就是 FloodMapp 正在做的事情，据丹尼·克赖顿说。这家总部位于澳大利亚的初创公司“旨在淘汰水文和预测分析的旧方法，并采用更现代的方法来帮助应急管理人员和公民知道洪水何时到来。”这种观点认为，一旦你知道将要发生什么，你就可以做好准备。
  • 更多风险投资无代码: 希望成为无代码应用开发主流的最新服务是 Stacker。你可能听说过他们。Ron Miller 报道称，该公司刚刚从 a16z 获得了 2000 万美元的首轮融资。与其他一些服务一样，Stacker 允许用户将电子表格转化为应用程序。(值得注意的是，一些初创公司采取了相反的方式。)
  • 今天在伟大的创业公司名称中， RaRa Delivery 刚刚筹集了 325 万美元 。这家印度尼西亚初创公司希望将当天送达服务带到国内市场。红杉资本印度的 Surge program 和 East Ventures 领投了这轮融资。

如何建立健康科技创业顾问委员会

大多数初创公司都可以使用顾问委员会，但在医疗科技领域，这是一个核心要求。

寻求在这一领域创新的创始人对导师有着独特的需求，这些导师在监管、融资和管理 R&D 等领域都有经验。

PatientPartner 的联合创始人兼首席运营官帕特里克·弗兰克(Patrick Frank)根据自己的经验，分享了一些非常具体的想法，涉及招聘谁、在哪里找到他们以及如何将他们纳入你的薪酬表。

弗兰克说:“你想利用这些人，这样你就能关注公司的全貌，以确保它是市场和投资者都想要的东西。”。
