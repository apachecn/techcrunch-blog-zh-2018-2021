Stacker 筹集 2000 万美元 A 轮融资，帮助业务部门在不编码的情况下构建软件 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/18/stacker-raises-20m-series-a-for-no-code-software-tool-for-business-units/

无代码平台已经发展成为一个热门市场，总部位于伦敦的无代码平台 Stacker 正试图将这一概念提升到一个新的水平。您不仅可以从电子表格创建 web 应用程序，还可以从各种来源提取数据来自动创建复杂的业务应用程序(尽管可能需要一些调整)。

今天，该公司宣布了由 Andreessen Horowitz 牵头的 2000 万美元 A 轮投资，现有投资者 Initialized Capital、Y Combinator 和 Pentech 也参与了投资。根据 Crunchbase 的数据，今天的投资总额达到了 2300 万美元。

Stacker 的首席执行官兼联合创始人迈克尔·斯凯利(Michael Skelly)表示，这个想法是将关键的商业数据转化为有用的应用程序，以帮助人们更有效地完成工作。Skelly 解释说:“(我们使)商界人士能够创建应用程序来帮助他们的工作生活——因此，像客户门户网站、内部工具和获取他们已经在使用的数据的东西，通常会运行一个流程，并将其转化为应用程序。”。

“我们真的认为，为了对业务真正有用，你需要了解企业关心的数据。因此，我们让人们带来他们的电子表格、SQL 数据库、Salesforce 数据，带来他们用于经营业务的所有数据，并自动将其转化为应用程序，”他说。

Skelly 说，一旦该公司将这些数据纳入并创建了一个应用程序，用户就可以开始调整事物的外观，但 Stacker 让他们在一开始就创造一些有用的东西方面有了很大的领先优势。

主要投资者安德森·霍洛维茨基金的合伙人李昕晢喜欢这家初创公司的无代码方式。“我们已经关注无代码领域有一段时间了，Stacker 因其周到的产品方法而与众不同，允许企业经营者立即生成一个完美适合现有业务流程的功能应用程序，”她在宣布融资轮的博客文章中说。

该公司目前有 19 名员工，计划在今年年底前将新资本投入到工作中，员工人数将达到 30-40 人。Skelly 认为建立一个多元化的公司是一个关键目标，并积极主动地寻找实现这一目标的方法。事实上，他已经确定了实现多元化的三种方式。

“首先，要确保我们有多元化的人才渠道。我真的认为，与你交谈的人的比例可能是你雇佣的人的最大指标。其次，我们试图找到一些方法，我们可以雇用那些可能在职业生涯中走得更远，但(希望)有所发展的人，”他说。

第三，我认为这是一个没有被充分谈论的问题，有很多人想进入编程角色，但他们的代表性不足，所以我们团队中的一些成员正在从各种非技术角色转换到开发人员，我认为这真的是一个很好的途径，可以增加(多元化候选人)的整体储备，”他说。

该公司是远程优先的，Skelly 在伦敦，他的联合创始人 Sam Davyson 在日内瓦，他们打算保持这种方式。他们在 2017 年创立了这家公司，最初创造了一种不同的产品，这种产品要复杂得多，需要大量的手握，然后最终得出结论，让它变得简单是一条出路。他们在 2019 年底发布了当前产品的第一个版本。

该公司有一个远大的愿景，那就是成为业务部门的软件开发工具。“我们真的认为，在未来，就像现在每个人都有电子邮件、聊天工具、电子表格和视频会议工具一样，他们也将有一个软件工具，他们可以在上面编写和运行他们经营业务的定制软件，”他说。
