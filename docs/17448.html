a16z 的 Arianna Simpson 说，Yield Guild Games 是该公司在加密+游戏上的最新赌注

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/19/arianna-simpson-of-a16z-on-yield-guild-games-the-firms-newest-bet-on-crypto-gaming/

作为 Andreessen Horowitz 的四名普通合伙人之一，Arianna Simpson 目前正在投资这家风险公司的第三个加密基金，一个价值 22 亿美元的工具，她高度关注如何将这笔资本以及更多的资本返还给公司的有限合伙人。

为此，她一直在跟踪更多将加密与游戏结合起来的创业公司。例如，上个月，她的团队共同领导了对虚拟人类工作室的投资，这是一家数字赛马服务 Zed Run 背后的初创公司，用户可以购买、出售和繁殖虚拟马，这些虚拟马的价值取决于它们与其他虚拟马的表现。(每个本质上都是不可替换的令牌，或 NFT，意味着它是唯一的。)

Simpson 对 NFT 的“玩到赚”模式很感兴趣，在这种模式下，游戏玩家可以赚取加密货币，然后他们可以根据自己的选择兑换成当地货币。事实上，a16z 今天宣布，它刚刚领导了对 Yield Guild Games (YGG)代币的 460 万美元投资，这是一家总部位于菲律宾的分散式游戏公司，邀请玩家通过玩“Axie Infinity”这样的游戏来分享公司的收入，这是一款总部位于区块链的游戏，玩家在游戏中繁殖和战斗名为 Axies 的数字生物，以赚取名为“小爱药水”的代币 YGG 借钱给玩家购买轴和其他数字资产来启动游戏，这样他们就可以开始赚钱了。

表面上的希望是，他们赚的钱比他们向 YGG 支付的资产使用费要多。但是为了了解更多，我们昨天与 Simpson 讨论了加密和游戏的交集，Simpson 在加入 a16z 之前首先支持了一些与该公司相同的初创公司，包括区块链基础设施公司 Dapper Labs 和全球支付平台 Celo 。她还分享了 a16z 最密切跟踪哪些平台来识别崭露头角的加密初创公司。

下面是我们的聊天记录，根据长度进行了编辑。

TC: Zed Run 真的很有意思。你是如何第一次接触到数字赛马业务的？

AS:我认为是加密 Twitter，老实说，这是我们发现很多游戏投资的地方。那里的社区真的很不可思议，通常是最先出现令人兴奋的新项目的地方之一。

Zed 确实标志着一种新的更复杂的加密游戏的出现。如果你看看[收藏游戏] CryptoKitties，它是第一个真正吸引了加密领域以外的人的注意力的 NFT 游戏之一。Zed 绝对是一个衍生扩展，因为你有一个正在玩的数字动物，但游戏要复杂得多，令人难以置信的是社区是多么兴奋。人们正在围绕如何玩这个游戏，如何阅读[种族]课程，如何在游戏中做各种不同的事情，以及世界各地成千上万的人[正在玩]整理各种非常复杂的指南。

也许这些已经存在了，但是在垂直领域是否有无限的机会，比如说，一个数字赛车的等价物或者 UFC 类型的等价物，人们可以购买和押注数字战斗机，并希望它们升值？

AS:在密码游戏的世界里，正在发生和将要发生的事情有着极其广泛的可能性。我认为这场运动的核心是将这些游戏资产的更多价值和所有权还给玩家。这是历史上的一个问题。你可能会花费数年时间来建立你的皮肤或游戏内资产，然后一个游戏会改变规则，从你那里拿走[你的一些奖金]或做任何可能让玩家感到非常失望和有点被欺骗的事情。(基于区块链的游戏)的想法是让它们更加开放，让玩家自己拥有游戏空间的所有权。

TC:这让我们想到了你最新的投资，屈服公会游戏，或者 YGG。为什么这家公司引起了公司的注意？

在疫情期间，许多人失业，无法养活自己和家人。这一次与一款名为“Axie Infinity”的游戏的兴起不谋而合，这是最早开创“玩到赚”模式的游戏之一，这正成为加密游戏中一个非常重要的主题。

为了玩“Axie Infinity”，你需要有三个轴，一般来说，这意味着你需要预先购买它们。显然，如果你失业了，你就没钱了(所以购买这些电子宠物)会成为一个非常具有挑战性的命题。因此，菲律宾的[YGG 创始人] Gabby Dizon，这位玩“Axie Infinity”的玩家开始借出他的 Axie，这样其他人就可以玩这个游戏并赚取代币，然后这些代币可以兑换成当地货币。所以基本上 YGG 是作为他们在这里所做的产品化出现的，所以 YGG 要么购买或培育有收益的游戏内资产，然后将它们借给我们的“学者”，他们是这些游戏内资产的接受者，然后 YGG 从玩家随着时间的推移产生的游戏内收入中抽取一小部分。

TC:一个“学者”必须是一个老练的玩家吗？

AS:有基本管理学者团队的管理者；他们才是真正决定把谁带进公会的人。

TC:所以这些轴可以兑换成现金，但是在哪里，谁在购买它们？

AS:他们可以在交易所买卖，如果他们需要在“Axie”繁殖，需要一些[Axie]，其他玩家也在购买；其他人出于投资目的购买它们。此外，他们不一定出售非功能性游戏，但他们可能会出售他们在游戏中获得的代币。

TC:现在有 5000 名这样的学者在玩这个游戏。他们大多在东南亚吗？

AS:大多数玩家和学者都在东南亚，但我们也看到了非常强劲的国际增长，尤其是“Axie Infinity”和 YGG。在这一点上，国际化无疑是 YGG 团队的核心关注点。

你提到了加密推特。Discord 和 Reddit 呢？你还在哪里寻找正在涌现并抓住人们想象力的新密码项目？

AS:以上都有。尤其是 Discord 被加密社区非常活跃地使用，有趣的是它真的可以让你了解一个社区有多活跃，人们有多投入，他们交谈的频率有多高，以及他们在谈论什么。它让你看到了整个社区，这是在寻求投资或评估项目健康状况时需要考虑的一件非常重要的事情。
