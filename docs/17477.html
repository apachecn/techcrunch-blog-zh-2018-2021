Tiger Global 以 5000 万美元支持短舱的电子商务基础设施

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/19/tiger-global-backs-nacelle-with-50m-for-its-e-commerce-infrastructure/

在全球疫情期间，消费者转向网上购物——并保持这一习惯——继续推动开发者工具制造商的收入，这些工具有助于电子商务网站提供更好的购物体验。

总部位于洛杉矶的短舱公司是继续吸引投资者注意力的电子商务基础设施公司之一，而且速度也很快。它完成了老虎环球公司 5000 万美元的 B 轮融资。这是在由英诺维亚牵头的1800 万美元 A 轮后的六个月，并在 2020 年进行 480 万美元的种子轮。

首席执行官布莱恩·安德森告诉 TechCrunch，该公司正在从事“无头”商务，这意味着它将网站的前端，也就是店面，与所有数据所在的后端分离，以创造更好的购物体验。通过这样做，商店的后端，实际上是所有神奇事情发生的地方，可以在不改变前端的情况下进行更新和维护。

他说:“网上购物并不新鲜，但顾客与它的关系一直在变化。”“网上购物的技术还没有达到标准——当你点击一些东西时，所有东西都必须重新加载，这与 Instagram 等应用程序相比。”

越来越多的人在他们的移动设备上购物，由于为每个品牌下载一个应用程序而产生摩擦。安德森补充说，这是“从网上购物中吸取乐趣”，因为没有人希望手机上有那么多应用程序。

Hybris 董事会成员、前执行副总裁史蒂文·克莱默(Steven Kramer)通过电子邮件表示，在过去的二十年里，电子商务行业经历了几次创新浪潮。现在，成熟的消费者行为和期望正在加速当前阶段。

“零售商和品牌正在努力采用最新的技术来满足当今对灵活性、速度和用户体验的要求，”Kramer 补充道。“短舱为组织提供了一种经得起未来考验的方式来加速创新，利用现有投资，并以较高的投资回报率实现这一目标。”

数据已经显示 COVID 时代的趋势将电子商务加速了大约五年，Gartner 预测到 2023 年 50%的新商务能力将被整合为以 API 为中心的 SaaS 服务。

这些趋势带来了竞争对手，也吸引了投资者的注意——例如， Shopistry 、 Swell 、 Fabric 、 Commerce Layer 和 Vue Storefront 仅仅是今年筹集资金的公司中的几个。

安德森指出，这个市场仍然很热，不容忽视，尤其是随着在线零售份额的增长。他解释说，他的一些竞争对手迫使客户从他们当前的技术体系迁移到他们各自的平台上，以便他们的用户可以获得良好的客户体验。相比之下，短舱使客户能够保留他们的技术堆栈，并按照他们认为合适的方式将组件组装在一起。

“这对任何垂直行业来说都是痛苦的，尤其是对电子商务来说，”他说。“这是你获得收入的直接途径。”

安德森说，与此同时，机舱本身的收入在过去一年增长了 690%，客户正在签署多年合同。

安德森的职业是工程师，随着无头商务的普及，他想全力开发新产品。这些措施包括在传统静态店面的技术堆栈之上提供一个动态功能层，甚至在今年晚些时候引入一些直播功能。

因此，短舱将把新一轮投资纳入其走向市场战略，并扩大其客户成功、合作伙伴关系和产品开发。他说，短舱已经是 Shopify Plus 商家“事实上的标准”。

“我们想把一切都放在一个为电子商务量身定制的 API 中，让前端开发人员轻松完成他们的工作，”安德森补充道。"我们还为商家提供初学者工具包，作为启动和运行的起点."
