Twitter 对其直接消息系统进行了一系列改进

你有没有试过通过 Twitter DM 和几个朋友分享一条搞笑的推文，结果却意外地开始了群聊？你并不孤单。今天，Twitter 宣布将在未来几周内对其直接消息系统推出一些生活质量改进，包括在单独对话中一次向多人发送推文的能力。研究员简·满春·王注意到推特上个月在 T2 开发了这个功能。

这一更新的一个潜在缺点是，它可能会邀请更多的垃圾邮件-你不能同时向超过 20 个人发送消息，但这仍然是很多人。现在收到这些消息的用户可能没有意识到他们是群发垃圾邮件的一部分，因为单个的 DMs 看起来像私人的 1:1 消息。

Twitter 表示，Android 用户将需要比 iOS 和网络推特用户等待更长的时间才能获得这项功能——目前还不清楚这将需要多长时间，因为在过去，iOS DM 更新需要几年才能到达 Android。但作为安慰奖，在 Android 和 iOS 上，如果你在 DM 对话中向上滚动，你可以通过按下向下箭头按钮快速滚动来返回最新消息。

Twitter 的另外两项 DM 改进目前只在 iOS 上推出——不再用日期和时间给单个消息打上时间戳，而是将消息按天分组。单个的 DMs 仍然会有一个时间戳，但是 Twitter 说这个改变将会减少时间戳的混乱。”

最后，在 DMs 中，iOS 用户可以通过双击和长按一条消息来访问“添加反应”菜单。长按朋友的消息还可以让你选择只删除你帐户上的消息，报告消息或复制文本。

Twitter 今天还宣布，它正在测试一项功能，将用户的时事通讯放在他们的个人资料上(Twitter 今年早些时候收购了时事通讯平台)。但是上周，它公布了更多引人注目的 UI 更新，专家认为这些更新使得平台更难访问。在更新后的两天内，Twitter 对其按钮进行了对比改变，并且发现了 Windows 上其专有字体啁啾声的问题。
