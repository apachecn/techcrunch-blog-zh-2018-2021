区块链初创公司 XREX 获得 1700 万美元以加快跨境贸易 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/22/blockchain-startup-xrex-gets-17m-to-make-cross-border-trade-faster/

XREX 的联合创始人温斯顿萧和黄伟晋。图片来源: XREX

一个世界贸易的很大一部分是用美元完成的，这给美元短缺的国家的商业带来了问题。区块链初创公司 XREX 的推出，旨在帮助新兴市场的跨境企业通过支付托管服务和 crypto-fiat 交换平台等产品更快地进行交易。

这家总部位于台北的公司今天宣布，它已经从 CDIB 资本集团领导的首轮融资中筹集了 1700 万美元。超额认购的一轮还包括 SBI 投资(SBI 控股公司的子公司)、全球创始人资本、ThreeD Capital、E.Sun 风险投资、Systex Corporation、MetaPlanet Holdings、AppWorks、BlackMarble、New Economy Ventures 和 Seraph Group 的参与。XREX 上一次融资是 2019 年的 700 万美元种子轮。

新一轮融资的一部分将用于申请新加坡、香港和南非的金融牌照，并与银行和金融机构合作，如支付网关。

“我们特别希望建立一个监管友好的上限表，”XREX 联合创始人兼首席执行官黄伟晋告诉 TechCrunch。“对于我们这样的初创公司来说，从银行和上市公司融资真的很难，但正如你所见，这一轮我们故意这么做，我们成功了。”

黄在 2013 年将他之前的初创公司——反恶意软件开发商 Armorize Technologies 卖给了 Proofpoint。Armorize 分析源代码以寻找漏洞，它的许多客户都是班加罗尔和钦奈的开发人员，所以黄花了很多时间在那里旅行。

“我们遇到了各种各样的跨境资金转移问题。这似乎是不可阻挡的，”黄说。“我们先后在美国和台湾长大，没有接触过这些问题。所以这埋下了一颗种子，然后当中本聪(Satoshi Nakamoto)发表了比特币白皮书，当然这对所有网络安全专家来说都是一件大事。”

他开始思考区块链如何支持像印度这样的新兴市场的金融包容性。这个想法在黄与 XREX 联合创始人、台湾首批比特币交易所之一 BTCEx-TW 的创始人温斯顿萧(Winston Hsiao)合作时实现了。萧在印度长大，创办了 Verico International，向其他国家出口台湾制造的半导体和电子产品，因此他也熟悉跨境贸易问题。

XREX 加密服务为商家，尤其是那些美元流动性低的国家的商家，提供了以数字法定货币进行交易的工具。“他们必须快速获得美元，并能够足够快地支付美元，以确保他们想要进口的重要商品，这就是我们想要解决的问题，”黄说。

为了使用该平台，商家和他们的客户注册了 XREX 的钱包，其中包括一项名为 Bitcheck 的商业托管服务。黄说，这类似于商业银行的备用信用证，因为买家可以用它来保证他们能够付款。比切克使用 USDT 和 USDC 等数字货币，这些稳定的货币与美元挂钩。

商家向供应商支付 stablecoin，XREX 保管资金，直到供应商提供发货证明，然后将货款转给他们。XREX 的 crypto-fiat exchange 允许用户将 USDT 和 USDC 兑换成美元，他们还可以通过该平台提取和存入美元。

XREX 的部分资金将用于扩展其法定货币平台，尽管黄表示，它不打算增加太多的加密货币，“因为我们不是为加密交易商而建，我们是为企业而建，品牌对他们来说非常重要。品牌和合规性，所以无论美国货币监理署说什么是好的稳定货币，他们都会使用。”

XREX 的一些合作伙伴包括合规和反洗钱提供商，如 CipherTrace、Sum&Substance 和 TRISA。XREX 的部分资金将用于扩展其安全和合规功能，包括公共档案，这对客户是强制性的，并使用信誉指数来增加透明度。

在一份关于融资的声明中，CDIB 资本创新基金负责人 Ryan Kuo 说，“CDIB 是 XREX 的早期投资者。在目睹了该公司的快速收入增长和他们对合规性的承诺后，我们决定将我们的投资增加一倍，并领导这一轮战略。”
