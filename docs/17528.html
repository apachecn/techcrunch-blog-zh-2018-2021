Pry Financials 筹集了 420 万美元，使创业会计更容易理解

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/23/pry-financials-raises-4-2m-to-make-startup-accounting-more-approachable/

窥探财务希望让整个团队都能接触到初创公司的财务，而不仅仅是负责会计电子表格的人。这位 Y Combinator 校友今天宣布，它已经从全球创始人资本、先锋基金、NOMO 风投、Liquid2 和 Hyphen Capital 筹集了 420 万美元。T3】

今年 3 月推出的 Pry 现在拥有 200 多名客户，并声称自 YC 的演示日以来，其月环比增长了 35%。它是由亚历克斯·赛勒、蒂芙尼·王、海登·詹森和苏德伦创立的。

在创办 Pry 之前，苏是 InDinero 的联合创始人，InDinero 是另一位校友，最初是一家“小型企业造币厂”，后来转型为一家提供全面服务的会计公司。当他还是加州大学伯克利分校的学生时，InDinero 就成立了，苏最终负责其财务规划。

窥探财务团队。图片来源:窥探财务状况

他告诉 TechCrunch，大多数创业公司都买不起像 Workday Adaptive Planning 这样的会计软件。相反，他们有时与外包的首席财务官服务合作，但大多数情况下一切都依赖电子表格:三方预测、预测跑道、招聘和承包商预算以及投资者更新。T11】

“我是首席技术官，多年来，我还负责财务职能，所以这是一种双重的 CTO/CFO 角色。这是从 2010 年到 2020 年，随着技术的发展，工程和产品团队每六个月左右就会得到各种新工具，而财务团队只是停留在 Excel 中，”他说。T13】

当苏还在 InDinero 时，Pry 作为一个副业项目开始，每月仅 50 美元，并用易于理解的会计、财务规划和场景建模仪表板取代这些电子表格。仪表板连接到 QuickBooks、Xero 或银行账户，因此数字会不断更新。

Pry 的客户通常在筹集种子资金后开始使用它，因为“对于大多数首次创业者来说，这是你收到的最大一笔钱，所以你需要花更多的时间来管理它，并每月审查它。而且你每个月都要花很多时间在工资单上，”苏说。与此同时，第二次创业的人注册了 Pry，因为他们厌倦了 Excel 电子表格。T15】

“审查一份电子表格是非常困难的，”苏说。“如果你看到一个错误的数字，你会得到这个奇怪的公式，如果你不是自己做的话。然后，你基本上必须给撰写邮件的财务分析师写一封长长的电子邮件，并希望他们在下班前回复你。”对于需要每月向贷款人或投资人更新信息的创始人来说，这意味着大量的工作。

Pry 通过将三方报告(资产负债表、损益表和现金流的组合)转化为财务报告仪表板，然后添加招聘计划、财务建模和情景规划等功能，使流程更加高效。T3】

场景规划功能充当了一个沙箱，为创业团队及其投资者提供了一种预测不同情况将如何影响财务的方法:例如，如果他们筹集到一定数量的资金或调整产品定价，他们会有多少跑道。

使用 Pry Financials 创建筹款仪表盘。图片来源:窥探财务状况

“我们正在改进，并试图以合作的方式对公司做出决策。我们的类比是 Git branching，你有自己的主要计划，想尝试新的收入模式或收购业务，但不想打乱当前的战略，”苏说。“你能做的是创建一个全新的分支机构，比如说，采用新的定价策略。您可以进行所有想要的更改，然后切换回您的旧分支，而不必担心覆盖它或与之冲突。”

这些投机性分支机构还会不断更新公司最新的银行账户和工资信息，因此，如果创始人希望以后重新考虑潜在的场景，他们不需要从头开始重新创建这些分支机构。T9】

Pry 计划建立更复杂预测工具，并将统计数据和基准等行业标准整合到模板中，以帮助创始人了解他们应该设定什么目标。T11】

因为 Pry 比一套 Excel 电子表格更容易管理，苏说它帮助初创公司发现重要的事情。例如，一位创始人通过处理税务问题找到了节省 15000 美元的方法。Pry 还能帮助初创公司的每个人了解公司的财务状况，即使他们之前没有接触过会计电子表格。该平台将很快添加角色和权限，因此创始人可以授予或限制不同人的访问权限，如特定部门的领导。T13】

苏说，Pry 不会与许多创业公司依赖的会计服务竞争，除非他们可以聘请财务主管，但创业公司可以更容易地与他们合作，因为他们可以共享他们的仪表板。T15】

“通常在早期，你可以外包给一家首席财务官公司。这是商业中的规范，对大多数公司来说都很有效。你可以找一个兼职首席财务官，非常努力地工作一个月，完成你的筹资结构，”苏说，并补充说，“我们很适合那个生态系统。”
