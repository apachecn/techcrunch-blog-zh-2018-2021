ICON 获得 2.07 亿美元的 B 轮融资，建造更多 3D 打印房屋，收入同比增长 400% TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/24/icon-lands-207m-series-b-to-construct-more-3d-printed-homes-after-400-yoy-revenue-growth/

利用 3D 打印机器人为无家可归者建造独户住宅。与美国国家航空航天局一起开发建筑系统，在月球上建造基础设施和栖息地，最终在火星上建造。交付据信是北美最大的 3D 打印建筑——德克萨斯州军事部门的兵营。

这些只是位于德克萨斯州奥斯汀的建筑科技初创公司 ICON 一直在做的一些事情。

今天，该公司的成就清单上又增加了一笔 2.07 亿美元的 B 轮融资。

自从 ICON 在 2018 年 10 月获得 900 万美元的种子期以来，我一直在报道它，所以看到该公司在不到三年后达到这一里程碑有点酷。

Norwest Venture Partners 领投了这家初创公司的 B 轮融资，参与方还包括 8VC、Bjarke Ingels Group (BIG)、BOND、花旗 Crosstimbers、Ensemble、Fifth Wall、LENx、Moderne Ventures 和 Oakhouse Partners。此次融资使 ICON 的总股本增至 2.66 亿美元。该公司拒绝透露其估值。

ICON 成立于 2017 年底，并在 2018 年 3 月的 SXSW 期间推出，其中 在美国首次允许 3D 打印房屋。350 平方英尺的房子需要大约 48 小时(以 25%的速度)才能打印出来。ICON 特意选择混凝土作为材料，因为正如联合创始人兼首席执行官杰森·巴拉德所说，“这是地球上最有弹性的材料之一。”

自那以后，这家初创公司表示，它已经在美国和墨西哥交付了二十多栋 3D 打印房屋和建筑。其中一半以上是为无家可归者或长期贫困者提供的。例如，i n 2020 年，ICON 与非营利合作伙伴 New Story 在墨西哥交付了 3D 打印的房屋。它还在德克萨斯州奥斯汀完成了一系列为长期无家可归者服务的家园，其中有非营利性的移动面包鱼。

这家初创公司在 2021 年初打入了主流住房市场，它说它为得克萨斯州奥斯汀的开发商 3Strands 在美国出售了第一套 3D 打印房屋 。这四所房子中有两所已经签约。剩下的两套房子将于 8 月 31 日上市。

最近，ICON 透露了其“下一代”Vulcan 建筑系统，并首次亮相其新的 探索系列 房屋。该系列的第一个房屋“零号房屋”是专门为 3D 打印优化和设计的。

在某些情况下，ICON 表示，其专有的 Vulcan 技术比传统的建筑方法更快地生产出“弹性、节能”的房屋，浪费更少，设计更自由。据 Ballard 称，该公司的新 Vulcan 建筑系统可以 3D 打印高达 3000 平方英尺的房屋和结构，比之前的 Vulcan 3D 打印机大 1.5 倍，快 2 倍。

从公司成立之初，Ballard 就坚持认为 ICON 的动力来自全球住房危机和缺乏解决方案。他认为，使用 3D 打印机、机器人和先进材料是解决经济适用房短缺的一种方式，这个问题在全国和奥斯汀越来越严重。

Ballard 说，ICON 的未来计划清单包括提供社会、救灾和更主流的住房，此外还与 NASA 一起开发建筑系统，在月球上，最终在火星上创建基础设施和栖息地。

ICON 还与美国宇航局有两个正在进行的项目。最近，美国宇航局刚刚宣布了火星沙丘阿尔法，ICON 和 BIG-ICON 目前已经完成了墙壁系统的打印，现在正在屋顶上。此外，美国国家航空航天局正在招募宇航员执行 nextfFall 任务，在 ICON 打印的第一个模拟火星栖息地生活。

奥林巴斯工程代表了 ICON 公司为未来探索月球开发太空建筑系统的努力，以及“想象人类在另一个世界的家园”

“我们的目标是在未来十年内让 ICON tech 登陆月球，”巴拉德说。

当被问及此事时，巴拉德表示，自去年 8 月该公司获得 3500 万美元的首轮融资以来，最重要的事情是“对 3D 打印房屋和建筑的需求急剧增长。”

Ballard 告诉 TechCrunch:“这个指标对我们来说意义重大。“人们不得不 想要 这些房子”

他补充说，为了解决住房短缺问题，世界需要增加供应，降低成本，提高速度，增加弹性，提高可持续性……所有这些都不能损害质量和美观。

“也许有一些方法可以做到这些事情，但只有建筑规模的 3D 打印才有潜力做到所有这些事情，”他说。

Ballard 称，ICON 的财务增长令人印象深刻，自成立以来几乎每年都有 400%的收入增长。在过去的一年里，它的团队增加了两倍，现在有 100 多名员工。它预计在下一年内规模会翻倍。

巴拉德说，B 轮基金将用于更多 3D 打印房屋的建设，“快速扩展和研发”，进一步推动太空技术进步，创造“对住房问题的持久社会影响”。

“我们已经建立了早期阶段的制造，并正在升级和加速这些努力，以满足对更多 3D 打印房屋的需求，即使我们结束了这一轮，”巴拉德说。“在接下来的五年里，我们相信我们每年将交付数千套住房，并且每年将交付数万套住房。”

作为融资的一部分加入 ICON 董事会的 Norwest Venture Partners 管理合伙人杰夫·克罗(Jeff Crowe)表示，他的公司相信 ICON 的 3D 打印建筑技术将“极大地影响美国和全球的住房短缺。”

Crowe 说，首先将先进的机器人技术、材料科学和软件结合起来开发一种强大的 3D 打印建筑技术是“非常困难的”。

他在电子邮件中写道:“更难的是开发一种技术，能够在不同的地理位置建造成千上万漂亮、负担得起、舒适、节能的房屋，并且具有可靠性和可预测性——而不仅仅是一两个受控环境中的示范单元。”。“ICON 做到了所有这一切，并且……具备了取得突破性、跨时代成功的所有要素。”
