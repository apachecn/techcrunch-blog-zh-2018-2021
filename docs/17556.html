Ramp 和 Brex 用 M&A 策略画出不同的市场计划

最近的交易告诉我们支出管理的战线在哪里

今天早些时候，spend 管理初创公司 Ramp 表示，已经筹集了 3 亿美元的 C 轮融资，估值为 39 亿美元。它还表示，它正在收购“谈判即服务”平台 Buyer，它认为这将帮助客户节省购买 SaaS 产品的费用。

在竞争对手 Brex 分享其自身收购的消息后一周，该轮交易宣布——T4 以 5000 万美元收购以色列金融科技初创公司 Weav。这笔交易是在 Brex 的创始人投资 Weav 后达成的，Weav 提供“商业平台的通用 API”。

从高层次来看，公司卡和消费管理领域最近的所有交易都表明，如今仅仅帮助公司追踪员工的消费是不够的。随着市场的成熟和功能集的开始融合，参与者正在寻求从竞争中脱颖而出。

但这里有趣的一点是，这些交易可以告诉我们，两家公司认为他们可以在哪里提供并从市场中获取最大价值。

这些差异是两家公司之间的另一层分歧:虽然 Brex 已经建立了其服务的付费软件层，但 Ramp 没有。

花更少的钱赚更多的钱

让我们从 Ramp 开始。该公司于 2019 年推出，在支出管理类别中是一个相对较新的公司。但从各方面来看，它产生了一些令人印象深刻的增长数字。正如我们的同事玛丽·安·阿泽维多写的那样:

自 2021 年初以来，该公司表示，其平台上的持卡人数量增加了 5 倍，目前有超过 2，000 家企业将 Ramp 作为其“主要支出管理解决方案”自 4 月份宣布上一次提价以来，其公司卡的交易量已经增长了两倍。令人印象深刻的是，据 Ramp 首席执行官兼联合创始人 Eric Glyman 称，Ramp 的交易量同比增长了 1000%。

Ramp 的重点一直是帮助客户省钱:它宣称对所有通过其信用卡进行的购买提供 1.5%的现金返还奖励，并表示其仪表板可以帮助企业识别欺诈订阅和许可证冗余。Ramp 还提醒客户，当他们可以节省年费和月费时，它说这导致许多客户放弃了 Concur 或 Expensify 等成熟的 T&E 平台。

总的来说，该公司声称，在转向其平台后，客户平均每年节省 3.3%的费用——而所有这些都是在将买家纳入考虑范围之前。
