Taktile 使得在金融行业利用机器学习变得更加容易

认识一下 Taktile，一家为金融服务公司开发机器学习平台的新公司。这并不是第一家想在金融产品中利用机器学习的公司。但 Taktile 希望通过让入门和切换到人工智能驱动的模型变得更容易，来使自己与竞争对手区分开来。

几年前，当你可以在每一份推介材料中读到“机器学习”和“人工智能”时，一些初创公司选择特别关注金融行业。这是有意义的，因为银行和保险公司收集了大量的数据，并了解他们客户的大量信息。他们可以使用这些数据来训练新的模型，并推出机器学习应用程序。

新的金融科技公司组建了自己的内部数据科学团队，并开始为自己的产品进行机器学习。像联合信贷和十月这样的公司使用预测风险工具来做出更好的贷款决策。他们开发了自己的模型，当他们根据过去的数据运行模型时，可以看到他们的模型运行良好。

但金融业的传统参与者呢？一些初创公司已经开发了可以集成到现有银行基础设施中的产品。你可以使用人工智能来识别欺诈交易，预测信誉，检测保险索赔中的欺诈行为等。

其中一些已经蓬勃发展，如特别关注保险的转变技术。但是很多创业公司建立了概念证明，并就此止步。未来没有有意义的长期商业合同。

Taktile 希望通过建立一个易于采用的机器学习产品来克服这一障碍。它已经筹集了 470 万美元的种子资金，由 Index Ventures 牵头，Y Combinator、firstminute Capital、Plug and Play Ventures 和几个商业天使也参与了进来。

该产品适用于现成的模型和客户定制的模型。客户可以根据自己的需求定制这些型号。模型由 Taktile 的引擎部署和维护。它可以在客户的云环境中运行，也可以作为 SaaS 应用程序运行。

之后，您可以使用 API 调用来利用 Taktile 的见解。它的工作方式非常类似于在你的产品中集成任何第三方服务。该公司试图提供尽可能多的透明度，对每个自动决策和详细日志进行解释。至于数据源，Taktile 支持数据仓库和数据湖以及 ERP 和 CRM 系统。

对于这家初创公司来说，现在仍处于早期阶段，看看塔克瓦的愿景能否实现将会很有意思。但该公司已经成功说服了一些有经验的支持者。所以让我们继续关注他们。
