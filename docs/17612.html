苹果降低了参与苹果新闻的新闻出版商的应用内购买佣金

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/26/apple-lowers-commissions-on-in-app-purchases-for-news-publishers-who-participate-in-apple-news/

苹果今天推出一项新计划，将允许订阅新闻机构参与苹果新闻应用并满足特定要求，将其在 App Store 应用内进行的合格应用内购买的佣金率降至 15%。通常情况下，苹果基于订阅的应用模式包括在应用商店的第一年收取 30%的标准佣金，然后在第二年降至 15%。但是今天宣布的新的苹果新闻合作伙伴计划，从第一天开始，参与者将获得 15%的佣金。

这种情况有几个注意事项，它们对苹果有利。为了获得资格，新闻出版商必须在苹果新闻上保持存在，并且他们必须以苹果新闻格式(ANF)提供他们的内容。后者是JavaScript Object Notation(JSON)格式，用于为苹果新闻撰写文章，这些文章针对 Mac、iPhone 和其他苹果移动设备进行了优化。通常，这涉及到一些设置，将来自出版商网站或 CMS(内容管理系统)的新闻文章翻译成支持的 JSON 格式。对于 WordPress 和其他流行的 CMS，也有插件可以使这个过程更容易。

与此同时，对于总部不在现有的四个苹果新闻市场之一——美国、英国、澳大利亚或加拿大——的出版商来说，他们可以通过向苹果提供 RSS 源来履行该计划的义务。

在 App Store 上，有资格获得 15%佣金的合作伙伴应用程序必须用于提供“原创、专业创作的”新闻内容，并且他们必须使用苹果的应用内购买系统提供自动续订订阅。

图片来源:苹果

虽然在建立出版商与苹果新闻的联系方面需要做一些初步工作，但值得注意的是，大多数主要出版商已经加入了苹果的平台。这意味着除了已经在做的工作之外，他们不必做任何额外的工作，就可以过渡到降低他们应用程序的佣金。然而，该计划也是推动新闻机构继续参与苹果新闻生态系统的一种方式，因为在更广泛的业务中这样做更有财务意义。

这可能会成为发行商争夺的一个领域，他们可能更希望降低的应用商店佣金没有附带条件。

一些出版商已经担心，他们将自己与苹果新闻生态系统捆绑在一起，会放弃对业务的太多控制。例如，去年,《纽约时报》宣布将退出与苹果新闻的合作关系，称苹果不允许它与读者建立自己想要的直接关系，而宁愿将读者导向自己的应用和网站。

然而，苹果会辩称，它并没有阻碍出版商的业务——它允许出版商为其内容付费，并从其销售的广告中获得 100%的广告收入。(如果他们不能全部卖出，或者更希望苹果代表他们卖出，那么他们将与苹果平分佣金，保留 70%的收入。)此外，对于该公司的苹果新闻+订阅服务——订阅收入分成要高得多—可以说是“找到了钱”也就是说，苹果公司将这项服务推销给了出版商自己无论如何也无法吸引的顾客。

新的苹果新闻合作伙伴计划的推出正值监管机构对苹果如何管理其应用商店业务进行审查，最近还提出了旨在解决美国和韩国等主要应用商店市场涉嫌反竞争问题的立法。

感受到市场的这种变化，苹果已经在努力通过调整其应用商店佣金来保护自己免受反垄断投诉和诉讼——就像现在正在进行的与 Epic Games 的诉讼一样。去年，推出了 App Store 小企业计划，该计划也将应用内购买的佣金从 30%降至 15%——但只针对收入不超过 100 万美元的开发者。

这个项目可能帮助了 T4 的小出版商 T5，但是很明显一些大出版商仍然不满意。在 11 月宣布降低小企业佣金后，出版商贸易组织Digital Content Next(DCN)——美联社、纽约时报、NPR、ESPN、Vox、华盛顿邮报、梅雷迪斯、彭博、NBCU、金融时报等的代表——就在下个月加入了倡导团体和游说组织应用公平联盟 (CAF)。

这些出版商曾写信给苹果公司首席执行官蒂姆·库克要求降低佣金，除了分成规模之外，他们还对收入分成有其他抱怨。他们也不想被要求使用苹果的应用内购买服务，称这种“苹果税”迫使他们为消费者提高价格。

这些出版商对苹果新闻合作伙伴计划的推出会有什么反应还有待观察。

虽然这给了他们降低应用商店费用的方法，但这并没有解决他们对苹果平台和规则的更广泛的抱怨。如果有什么不同的话，它将较低的费用与一个将他们进一步锁定在苹果生态系统中的项目联系起来。

苹果公司，作为一种善意的姿态，今天也表示它将再次承诺支持三家领先的媒体非营利组织:常识媒体、新闻素养项目和 geo servatorio Permanente Giovani-Editori。这些非营利组织提供无党派、独立的媒体素养项目，苹果认为这是其更大使命的关键，即让人们成为聪明、积极的新闻读者。苹果还表示，它稍后将宣布其他组织的进一步媒体素养项目。然而，该公司不会从财务角度透露其承诺的规模，也不会讨论过去向此类组织提供了多少资金。

苹果服务高级副总裁 Eddy Cue 在一份声明中表示:“从第一天起，为苹果新闻客户提供来自我们出版合作伙伴的可信信息就一直是我们的首要任务。”“十多年来，苹果为我们的客户提供了多种方式来访问和享受我们产品和服务中的新闻内容。我们在 App Store 中提供了来自全球几十个国家的数百款新闻应用，并创建了苹果新闻格式，为出版商提供了一个展示其内容的工具，并为数百万苹果新闻用户提供了良好的体验，”他补充道。

有关该计划的更多详情和申请表将在新闻合作伙伴计划网站上提供。
