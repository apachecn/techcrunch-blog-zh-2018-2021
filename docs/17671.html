North Base Media 在数字媒体初创公司 Vietcetera 中领投了 270 万美元的首轮前融资

总部位于胡志明市的数字媒体网络 Viet cetera(T1)最初是为千禧一代和 Z 世代创建的，在获得 270 万美元的首轮融资前资金后，它将扩大其目标人群。由专注于媒体的风险投资公司 North Base Media 牵头，今年进行了两轮融资。

其他投资者包括 Go-Ventures，Gojek 的企业风险部门；东方创投；峰会媒体；Genesia VenturesHustle 基金；以及由软银集团和 Naver Corporation 拥有的 Z Holdings 的企业风险部门 Z Venture Capital。

Vietcetera 于 2016 年由 Hao Tran 和 Guy Truong 创立，现在每月拥有 2000 万用户。其广告客户包括友邦人寿保险、谷歌、脸书、雀巢、万事达卡、Vingroup 和 Tiki。Tran 告诉 TechCrunch，Vietcetera 也将通过“优先考虑原创内容许可和开发”来赚钱

该网络最初是为千禧一代和 Z 世代观众创建的，他们想要“超越娱乐、轰动新闻和娱乐的内容。”为了服务更多的读者群体，Vietcetera 将在 2022 年推出新的垂直品牌，专注于女性内容、房地产和个人金融。

North Base Media 于 2013 年由《华尔街日报》和《华盛顿邮报》前主编马库斯·布劳奇利(Marcus Brauchli)和媒体发展贷款基金(Media Development Loan Fund)首席执行官萨阿·武契尼(SAA vu ini)创立，旨在支持移动互联网渗透率不断增长的市场中的数字媒体初创公司。它的其他投资组合公司包括 News Lens、Atlas Obscura、Rappler 和 Majarra。

Vietcetera 的新资金将用于内容，包括新节目和播客，移动应用程序开发，特许经营和内容许可，以及潜在的收购。
