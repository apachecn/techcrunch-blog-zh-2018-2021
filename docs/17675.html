羊驼融资 5000 万美元，迅速扩大其 API 交付的股票交易业务

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/30/alpaca-raises-50m-to-rapidly-scale-its-equities-trading-api-business/amp/

图片来源: Getty Images

羊驼今天早上表示，它已经完成了 5000 万美元的 B 轮投资。TechCrunch 此前报道了该公司的2019 年末 600 万美元的种子轮和2020 年末 1000 万美元的 A 轮。

羊驼通过 API 提供股票交易软件。该公司最初允许公司接入其技术，为投资集团的交易能力提供动力。最近，羊驼已经开始允许其他金融科技公司通过其服务向其消费者用户群提供股票交易，这符合更大的嵌入式金融趋势。

Tribe Capital 领投了该公司的 B 轮融资，现有投资者 Spark Capital、Portage Ventures 和 Social Leverage 也参与了融资。包括 Horizons Ventures 在内的新投资者也投入了资金。

羊驼是一个有趣的创业公司。在 2020 年储蓄和交易热潮期间，我们不仅将公司的交易量增长作为 API 交付软件初创公司增长的代表，还将其作为更广泛地买卖美国股票兴趣的窗口。

通过现在向金融科技公司的消费者终端用户提供交易服务 B2B2C 模式，羊驼已经扩大了它的市场范围。据这家初创公司称，它支持的经纪账户数量今年增长了约 1，500%，超过 10 万个。这家初创公司的首席执行官 Yoshi Yokokawa 告诉 TechCrunch，预计到 2021 年底，其股票交易技术将获得 100 个合作伙伴。在嵌入式金融产品发布之前，这一数字在 2020 年底为零。

对羊驼来说，与更多金融科技公司合作开辟了新的收入来源。该公司表示，将继续为订单流收入(PFOF)产生支付，但通过支持国际客户，它也可以从外汇费用等方面获得收入。

值得注意的是，羊驼打算通过与嵌入其金融科技 API 的合作伙伴分享 PFOF 收入，使其服务成为一个低成本中心。Yokokawa 拒绝与客户分享 PFOF 的分成，但我们猜测，大约 15%至 25%是合理的，这将激励潜在合作伙伴选择羊驼毛而不是竞争对手的技术，同时保持足够的羊驼毛收入，以继续建立风险投资规模的业务。

这家初创公司有着宏伟的计划:它将进军加密货币市场，今天早上宣布，并与 Plaid 合作，使投资者的资金转移更容易。在美国很受欢迎的消费者交易平台 Robinhood 最近的结果有助于强调加密交易对平台来说是多么有利可图。

为什么要筹集 5000 万美元？TechCrunch 很好奇为什么该公司会一次性向其账面投入如此多的资金，而不是筹集一轮更适度的资金，比如说 2500 万美元，这对于 B 轮融资来说仍然是一个健康的数字，并且在规模上与之前的 a 轮融资更接近。

横川说羊驼有很多东西要造。建造这一切需要很多人。当新冠肺炎袭击时，羊驼只有 10 名员工，这意味着该公司有大量的招聘工作要做。我们认为，它所需要的开发人员不会便宜。

尽管如此，大规模融资意味着投资者和观察团队(也就是我们)的巨大期望。我们将在几个月后再次与该公司核实，看看它是否有望实现其 2021 年的合作伙伴目标。
