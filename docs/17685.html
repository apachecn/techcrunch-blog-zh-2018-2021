从白手起家到几十亿

许多人试图重新发明日历，但都失败了。托普·阿沃托纳建造日历不是为了重新发明轮子，而是为了给混乱的人类交流和时间管理增加一层简单性。男孩成功了！这家曾经白手起家的公司现在价值超过 30 亿美元，为个人和企业提供服务。听听这位创始人兼首席执行官是如何起步的，为什么他最终决定接受机构投资，以及公司在成长过程中发生了怎样的变化。

不可阻挡的力量和不可移动的物体

Slack 和 Salesforce 是科技界两大巨头。通信工具(诞生于科技史上一个奇怪的支点)在几乎每个行业的组织中都很常见。这是一股不可阻挡的力量。销售 CRM 这个庞然大物被世界各地大大小小的销售团队所使用。不可移动的物体。2020 年 12 月，两家公司宣布了 277 亿美元的合并。听听 Slack 联合创始人兼首席执行官斯图尔特·巴特菲尔德和 Salesforce 总裁首席运营官·布雷特·泰勒讲述合并后实体的未来，为什么这笔交易有意义，写下这么多 0 是什么感觉。

利用云技术推动小型企业经济发展

小企业是创造就业、经济增长和创新的关键引擎，也是我们从全球疫情中复苏的驱动力。15 年前，一家名为 Xero 的新西兰初创公司成立，其宗旨是让小企业及其顾问的生活更美好。Xero 通过将会计实践转移到云并提供一套开放的 API 实现了这一目标，这使得 1000 多家应用程序合作伙伴能够构建连接到 Xero 平台的经济实惠的技术解决方案。 Xero 首席执行官 Steve Vamos 将讨论 Xero 如何通过使用云和其平台将实时数据与定制的业务解决方案连接起来，从而变革小企业的经营方式，帮助小企业主取得更大的成功。史蒂夫将谈到一系列关键举措，这些举措将改变希望在 Xero 平台上创新和合作的初创公司和企业家的游戏规则，他将解释 Xero 的愿景如何超越技术，激发全球社区的支持和目的，以帮助各地的小企业。由 Xero 提供。

推动下一步发展:企业软件市场洞察

在数字化转型和最近向远程工作转变的推动下，企业软件行业日益壮大，交易和估值的竞争空前激烈。虽然投资者对企业软件的胃口可能很大，但这并不意味着所有的科技企业都值得投资。在这个小组中，请听听 Vista 旗舰投资战略的联席主管 Michael Fosnaugh 和 Monti Saroya，以及一批 Vista 首席执行官关于一流软件公司的标志和推动行业发展的趋势。由 Vista Equity Partners 提供。

利用 Prometheus、Grafana 和 Hardware Sentry 实现可持续 IT

实施可持续发展计划以实现数据中心的零碳排放是一项重大挑战。加入 Bertrand Martin，Sentry 软件的联合创始人兼首席执行官，他为 Prometheus 介绍硬件 Sentry 出口商。利用这一独特的纯软件解决方案，测量 250 多个平台的功耗和温度。在 Grafana 中报告 CO₂排放、用电量以及应用和服务成本。通过智能优化环境温度，减少数据中心的碳足迹。由哨兵软件公司提供。
