Ideanomics 以 4.5 亿美元全股票交易收购电动汽车制造商威盛汽车

总部位于纽约的金融科技和电动汽车公司 Ideanomics 的收购清单上又增加了一项:商用电动汽车制造商 Via Motors(全股票交易，价值 4.5 亿美元)。

Ideanomics 今年一直在积极收购移动业务，寻求为车队运营商和过渡到电动汽车的交通部门提供垂直整合的产品。收购 Via Motors 的消息推动 Ideanomics 的股价自开市以来上涨 6%，至 2.43 美元。

仅在今年，Ideanomics 就完成了对美国 Hybrid 的收购，这是一家电动动力总成部件和燃料电池发动机制造商；电动拖拉机制造商 Solectrac 制造了唯一一台美国制造的电动拖拉机；总部位于犹他州的无线充电公司 Wave 和 Timios Holdings Corp .提供产权和托管服务。

通过马达对的收购是 Ideanomics 历史上最大的一笔。威盛设计和制造用于中短途运输的电动货车和卡车，在三种车型中使用模块化的“滑板”式架构。

“此次收购标志着 Ideanomics 的一个转型里程碑，”首席执行官阿尔弗雷德·普尔周一在投资者电话会议上就该交易表示。他指出，此次收购还提供了“全面的 OEM 制造能力”，这意味着该公司现在可以制造其融资并保持充电的电动汽车。

这笔交易包括 Via 股东高达 1.8 亿美元的潜在收益，取决于 2026 年的车辆交付情况。股东还将拥有合并后公司约 25%的股份。另外，Ideanomics 表示，将通过 operations 预付 5000 万美元的融资票据。

Ideanomics 目前为从电动汽车采购到建立充电管理基础设施的所有事情提供便利。通过其金融科技部门，Ideanomics 还提供融资以及充电即服务和车辆即服务服务，该公司表示，这将使车队公司将其投资模式从资本支出驱动型转变为运营支出驱动型。

“我们认为，从资本支出到运营支出的转变将对车队运营商产生深远影响，通过消除明显的进入壁垒，加快零排放车队的采用，但必须投资于新产品和基础设施，”普尔在最近的第二季度财报电话会议上表示。

虽然该公司对总部位于犹他州的 Via 到 2026 年的财务预测保持沉默，但普尔补充说，这些数字将包括在收购交易结束前提交给监管机构的 Ideanomics 的委托书中。
