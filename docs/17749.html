VanMoof 筹集 1.28 亿美元成为世界领先的电动自行车品牌 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/08/31/vanmoof-raises-128-million-to-become-the-worlds-leading-e-bike-brand/amp/

总部位于阿姆斯特丹的初创公司 VanMoof 已经筹集了 1.28 亿美元的 C 轮融资。该公司设计并销售电动自行车，在一些市场上很受欢迎。它现在希望以更快的速度迭代，成为世界领先的电动自行车品牌。

总部位于亚洲的私募股权公司高瓴投资(Hillhouse Investment)领投本轮投资，Booking.com 前首席执行官吉利恩·坦斯也参与了投资。一些现有投资者也拿出了更多资金，如 Norwest Venture Partners、Felix Capital、Balderton Capital 和 TriplePoint Capital。

与公司的 b 轮融资相比，今天的 C 轮融资代表了一个巨大的飞跃。去年，VanMoof 筹集了4000 万美元的 b 轮融资。总体而言，如果你把所有这些都加起来，这家初创公司总共筹集了 1.82 亿美元。

如果你不熟悉 VanMoof 的电动自行车，TechCrunch 评论了最新的 S3 和 X3 车型。理论上，他们是一样的。VanMoof X3 的特点是车架和车轮较小。

VanMoof 与普通电动自行车制造商的不同之处在于，该公司试图控制从供应链到客户体验的一切。VanMoof 电动自行车是主要为城市骑行设计的高级电动自行车。最新款目前售价 2298 美元，€售价 2198 美元。

它们配有一个电动马达和一个电子换挡系统。它有四个档位，你不用自己换档。你所要做的就是跳上自行车，开始踩踏板。

通过其标志性的三角形未来主义外观框架，S3 和 X3 也配备了液压制动器，集成灯和一些智能功能。有一个集成的运动探测器，结合了警报器，GPS 芯片和蜂窝连接。

如果你宣布你的自行车被盗，全球定位系统和手机芯片就会启动，你可以在 VanMoof 应用程序中跟踪你的自行车。该公司的自行车现在也与苹果的 Find My app 兼容。

该公司不再完全依赖现成的零件，而是与一小群供应商合作制造定制组件。通过这种方式，它可以削减尽可能多的中间环节，从而降低成本。也是很好的竞争优势。

发展像 VanMoof 这样的公司是一项资本密集型业务。该公司在全球 50 个不同的城市开设了零售店和服务中心。虽然该公司在欧洲起步，但美国现在是 VanMoof 增长最快的市场。

随着今天的融资轮，这家初创公司计划在当前战略上加倍努力。你可以期待更新的自行车有更好的设计和更多的定制零件。世界各地将会有更多的商店和服务中心。你也可以期待更多的网上销售。

“这将帮助我们在未来五年内让 1000 万人骑上我们的自行车，”联合创始人兼首席执行官 Taco Carlier 在一份声明中说。到目前为止，有 15 万人在使用 VanMoof 自行车。

今天的投资不应该让人感到意外。冠状病毒疫情加快了改造欧洲城市的计划——优先考虑自行车而不是汽车。去年，TechCrunch 的娜塔莎·洛马斯(Natasha Lomas)和我写了一篇关于巴黎、巴塞罗那、伦敦和米兰这四大城市关键政策发展的综合综述。VanMoof 现在正受益于这些政策转变。
