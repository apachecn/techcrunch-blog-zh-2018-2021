你应该发行信用卡或借记卡的所有理由

你不应该这么做的一个原因是

在过去的两三年里，我们看到消费和 B2B 金融科技初创公司以及我们传统上可能不认为是金融服务业参与者的公司推出了大量新的借记卡和信用卡产品。

在消费者方面，这意味着像 Venmo 或 PayPal 这样的公司提供借记卡作为用户消费账户资金的新方式。在 B2B 领域，像 Brex 和 Ramp 这样的初创公司发行公司卡带来了新的费用和支出管理选择。此外，品牌信用卡和借记卡在品牌和运动队中也有所增长。

但如果你的公司不知何故还没有找到发行借记卡或信用卡的方法，我们有个好消息:这样做比以往任何时候都容易，而且真的有钱可赚。要知道，如果你这样做了，你已经有很多竞争对手，而实际的客户使用情况可能取决于你的服务有多粘，以及你为最活跃的用户提供的回报有多有价值。

为了了解更多关于推出卡片产品的信息，TechCrunch 采访了来自 Marqeta 、 Expensify 、 Synctera 和 Cardless 的高管，讨论推出卡片产品的利弊。所以事不宜迟，以下是你应该考虑这么做的所有理由，以及你可能不想这么做的一个重要原因。

因为这(相对)容易

我们看到这么多新的金融科技公司和非金融科技公司急于向客户提供借记卡和信用卡，最大的原因可能是，对他们来说，这样做比以往任何时候都更容易。像 Marqeta 这样的企业的推出和成功使得 API 开发者的发卡变得更加友好，这在过去五年中大大降低了准入门槛。

Marqeta 的 SVP 和北美通用汽车公司的 Salman Syed 表示:“之所以会出现这种情况，是因为‘金融科技 1.0 基础设施’已经取得了成功。”。“当你有像(我们的)这样的公司时，推出贺卡产品就变得容易多了。”

Expensify 首席运营官 Anu Muralidharan 指出，至少在过去五六年中，信用卡发行和支付处理有了很好的选择，但他表示，金融科技基础设施的其他部分的技术资源激增，使得这些年来为信用卡发行开绿灯的过程变得更加容易。
