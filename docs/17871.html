微软推出个性化新闻服务，微软启动

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/07/microsoft-launches-a-personalized-news-service-microsoft-start/

微软今日推出自己的个性化新闻阅读体验，名为 Microsoft Start，既可作为网站也可作为移动应用，此外还可与其他微软产品集成，包括 Windows 10 和 11 及其微软 Edge 网络浏览器。该公司表示，feed 将结合新闻出版商的内容，但会根据用户的个人兴趣量身定制——这是一个定制系统，可以帮助微软更好地与苹果或谷歌等竞争对手以及 Flipboard 或 SmartNews 等第三方应用提供的新闻阅读体验竞争。

微软表示，该产品建立在该公司在线和移动消费者服务(如 MSN 和微软新闻)的基础上。但是，它不会取代 MSN。尽管推出了新的内部竞争对手，这项服务仍将继续提供。

要使用微软 Start，消费者可以访问 MicrosoftStart.com 的独立网站，它可以在谷歌 Chrome 和微软 Edge(但不是 Safari)上运行，或者他们可以下载 iOS 或 Android 的微软 Start 移动应用程序。

该服务还将增强 Windows 10 任务栏上的新闻和兴趣体验以及 Windows 11 上的小工具体验。在 Microsoft Edge 中，它也可以在新的选项卡页面中使用。

图片来源:微软

乍看之下，微软 Start 网站很像任何其他在线门户网站，提供来自各种出版商的新闻，以及天气、股票、体育比分和交通等小工具。当你点击阅读一篇文章时，你会被带到微软域名上的联合版本，包括顶部的微软开始顶部导航栏和标题下方的表情符号反应按钮。

用户还可以在浏览主页时用表情符号对故事做出反应。

这套表情符号和脸书今天推出的表情符号很相似，只是微软用一张思考脸取代了脸书备受争议的笑脸表情符号。(值得注意的是，脸书的笑脸已经越来越多地受到批评，因为它被用来公开嘲笑帖子和嘲弄人——甚至在描述悲惨事件的故事中，比如 COVID 死亡事件。)

微软的表情符号也做了另一项改变:当你用表情符号对一个故事做出反应后，你只能看到你的表情符号，而不是前三名和总反应数。

图片来源:微软

但是，尽管在线门户网站往往是新闻内容的静态聚合器，微软 Start 的 feed 将通过几种不同的方式来适应用户的兴趣。

用户可以点击“个性化”按钮进入一个页面，在那里他们可以手动添加和删除多个高级类别的兴趣，如新闻、娱乐、体育、科技、货币、金融、旅游、健康、购物等。或者他们可以搜索更具体或更适合的类别和兴趣。(而不是“养育”，例如，“养育青少年。”)这让人想起了最近 Flipboard 对其主页所做的更新，也就是让用户做出类似选择的 For You feed 。

当用户开始浏览他们的 Microsoft Start feed 时，他们还可以单击一个按钮来上下移动文章，以更好地根据他们的偏好调整 feed。微软表示，随着时间的推移，用户参与内容越多，提要就变得越精炼。该公司指出，这种定制将利用人工智能和机器学习，以及人类的节制。

像其他在线门户一样，这个 feed 是由广告支持的。当你向下滚动时，你会注意到每隔几行就有一个广告单元，其中的网址标有一个绿色的“广告”徽章。最初，这些广告大多是产品广告，与新闻内容截然不同。由于微软并没有关闭 MSN，而是将这项新闻服务整合到许多其他产品中，因此它正在通过此次发布扩大其可提供的广告空间。

根据 iOS 应用程序的隐私标签，用于跟踪其他公司网站和应用程序用户的数据包括用户 ID。相比之下，谷歌新闻不包括跟踪部分。微软 Start 和谷歌新闻都收集大量“与你相关的数据”，如位置、标识符、搜索历史、使用数据、联系信息等。然而，网站本身只链接到微软的一般隐私政策。

记者联系到微软进行评论，微软告诉我们，微软 Start 不会“打包或转售个性化数据给广告商、数据经纪人或其他第三方。”此外，用户可以随时删除他们的个人资料，如果没有登录，可以通过删除他们的网站 cookie 来删除，如果使用他们的 Microsoft 帐户登录，可以通过 Microsoft Privacy Dashboard、来删除。*

网站、应用程序和集成从今天开始推出。(如果你还找不到新的应用程序——它取代了微软新闻——你可以尝试从你的移动设备扫描二维码。我们目前发现该应用程序已经在 iOS 上推出，但链接指向了 Android 上的微软新闻。您的里程可能会有所不同。)

 

* 2011 年 9 月 9 日更新，提供了有关 Microsoft Start 隐私控制和数据使用政策的更多详细信息。
