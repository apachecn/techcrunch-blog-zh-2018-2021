虚拟会议平台元音筹集 1350 万美元，旨在治疗会议疲劳

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/07/virtual-meeting-platform-vowel-raises-13-5m-aims-to-cure-meeting-fatigue/amp/

图片来源:元音

会议是工作日不可避免的一部分，但随着过去 18 个月工作场所变得更加分散，元音首席执行官安迪·贝尔曼说，我们正稳步走向“会议死亡”

他的虚拟会议平台是最新一个获得风险投资基金——1350 万美元——的平台，目标是让会议在会前、会中、会后变得更有用。

元音是推出一个会议操作系统与工具，如实时转录；综合议程、说明和行动项目；会议分析；和可搜索的点播会议录音。该公司有一个免费增值的商业模式，并将在今年秋天推出一个每个用户每月 16 美元的商业计划。额外的功能将包括先进的集成，安全和管理控制。

A 轮融资由游说资本(Lobby Capital)的大卫·霍尼克(David Hornik)牵头，现有投资者 Amity Ventures 和 Box Group 以及一批个人投资者加入，包括 Calendly 首席执行官托普·阿沃托纳(Tope Awotona)、Intercom 联合创始人德斯·特雷纳(Des Traynor)、Slack 副总裁伊桑·艾斯曼(Ethan Eismann)、Yammer 前高管薇薇安娜·法加(Viviana Faga)、InVision 前总裁大卫·弗拉加(David Fraga)和 Okta 联合创始人弗雷德里克·克雷斯特(Frederic Kerrest)。

在开始元音之前，伯曼是婴儿监视器公司 Nanit 的创始人之一。该公司的团队分散在世界各地，因此沟通很困难。他说，2018 年，该公司开始寻找一种适用于同步和异步会议的工具，但仍然有很多时区需要管理。

受 Nanit 自己的婴儿监视器的启发，元音的想法诞生了，该公司开始关注分布式工作将会盛行的假设。

“人们最初认为我们疯了，但后来疫情来袭，每个人都在学习如何远程工作，”伯曼告诉 TechCrunch。“当我们现在回到混合工作时，我们认为这是一个机会。”

2017 年，《哈佛商业评论》报道称高管每周花 23 个小时开会。伯曼现在估计，普通员工每周有一半时间花在开会上。

元音是出带来 Slack，Figma 和 GitHub 组件的会议，通过录制音频和视频，可以随时暂停。用户可以添加笔记，并查看这些笔记在实时转录中的位置，使迟到或未能参加会议的人能够轻松赶上会议。会议结束后，它们可以被分享，元音有一个搜索功能，这样用户可以返回并查看特定的人或主题是在哪里讨论的。

新的资金将使该公司在产品、设计和工程方面壮大其团队。元音公司计划在接下来的一年里雇佣 30 名新员工。该公司最近结束了测试，并积累了一份 10，000 人的等待名单。伯曼说，公开发布将在秋季进行。

工作场所生产力和办公室通信工具并不是新概念，但正如伯曼解释的那样，在过去 18 个月里，当家变成办公室时，它们变得越来越重要。

竞争对手采取了不同的方法来解决这些问题:专注于视频会议或音频或带有插件的会议管理。伯曼说，许多人尚未成功的一个领域是将会议整合到典型的工作流程中。他补充说，这就是元音带“meeting OS”的原因。

“我们的目标是使会议更具包容性和价值，包括准备、会议和后续工作，”伯曼说。“我们看到未来将是知识管理，所以我们所做的事情与众不同，是确保你能够快速赶上并保持知识库。加纳的一份报告称，到 2025 年，75%的工作场所会议将被记录下来，这是我们正在彻底改造的一个趋势。

游说资本(Lobby Capital)的创始合伙人大卫·霍尼克(David Hornik)表示，他是从现有投资者友好投资公司(Amity Ventures)那里认识元音的。担任 GitLab 董事会成员的 Hornik 表示，在疫情之前，GitLab 是科技领域最大的分布式公司之一，亲眼目睹了让分布式团队发挥作用的挑战。

当 Hornik 听说元音时，他说他“很快”抓住了这个机会。他的公司通常投资有能力改变商业空间的平台业务。他补充说，许多是纯软件，如 Splunk 或 GitLab，而其他一些类似于 Bill.com，改变了小企业管理财务运营的方式。

他说，所有这些结合成一个公司，就像元音一样，特别是考虑到该公司对会议操作系统的愿景，以改变几十年来没有前进的会议空间。

“这对我来说很快就很明显，因为我每天都在开会——八倍变焦的一天是正常的——我只是希望我能记住所有的事情，”Hornik 说。“在与使用该产品的早期客户交谈时，当我问他们如果这种情况消失了，他们会怎么做时，他们说的第一句话是‘哭’，因为没有其他选择，他们会回到 Zoom 或其他工具，但这将是一个很大的挫折。”
