India 收购印度 Automate.io 以加速产品扩张

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/08/notion-acquires-indias-automate-io-in-push-to-accelerate-product-expansion/

organization 周三表示，它已经收购了印度初创公司 Automate.io ，该公司与 200 多种服务建立了连接和集成，因为这家工作场所生产力初创公司希望加快其产品扩张，以便对数千万越来越多地转向数字协作工具的个人和企业变得更具吸引力。

这家总部位于旧金山的初创公司最近一次在私人市场的估值为 20 亿美元(T3)，它表示，收购总部位于海德拉巴的 Automate 将有助于 idea 了解并利用这家印度初创公司开发的 200 个集成，让用户和企业能够将他们的工作流程带入 idea。

收购，首先是对概念来说，是一个“我们拼图的战略一块，”概念的首席运营官阿克谢·科塔里在接受 TechCrunch 采访时说。“这是一笔相当大的收购，”他说，但他没有透露交易的条款。

他说，收购 Automate——该公司只筹集过一次资金，主要来自朋友和家人，与 India 一样是盈利的——也使 India 能够在印度建立一个工程中心，这是它在美国以外的第一个工程中心。(他希望在印度设立更多办事处，这家初创公司的产品在印度已经很受欢迎，尤其是在初创公司圈子里。)

Automate 拥有约 40 名员工，已经与多个行业的公司开展了广泛的集成，包括电子商务、支付、营销、社交和生产力。它为其中一家公司做了大量的整合工作。

看看 Automate.io 已经开发集成的一些流行服务。图片来源: Automate.io

例如，用户/管理员可以使用 Automate 将协作平台连接到 Slack、Gmail、Mailchimp、Salesforce、Google Workspace、Office 365、Dropbox、WordPress、RSS Feed、PayPal 和许多其他服务。“每个产品都有一个 API。Automate 所做的是与每个 API 一起工作，以一种用户自己不必担心完成工作的方式来构建管道，”他说。

几个月前，当它向公众发布 API 时，idea 发现了 Automate。“我没听说过他们。但因为他们是我们的首批合作伙伴之一，所以我调查了他们，”科塔里说，几个月前，他联系了 Automate.io 的首席执行官阿肖克·古迪班德拉。

“我们的使命一直是通过自动化重复任务来支持业务，并提高效率。我们希望用户花更少的时间处理具有挑战性的集成，花更多的时间构建和创建他们需要的软件，”Gudibandla 在一份声明中说。

“有了 idea，我们将能够为全球更广泛的用户提供同样的集成和自动化体验。我们很高兴能与 ideal 联手，这是一个理想的合作伙伴，分享我们的价值观和承诺，以扩展协作、无缝软件体验的力量。”

Kothari 说，Automate.io 目前提供一些付费计划，并将继续向客户提供这些计划。

顺便说一下，作为一个长期的用户，我问 Kothari 公司是否正在增加对苹果铅笔和聚光灯搜索的支持。两者都在公司的待办事项中。他让我看了一眼这家初创公司正在做的一些事情——名单很长(这家初创公司跟踪每一个功能请求，甚至是在 Twitter 上提出的请求)iPhone 上的 Spotlight search 集成应该很快就会推出。

Kothari 说，有了 Automate.io，他希望 idea 能够更快地推出其中的一些功能。“我们面临的挑战是，这个概念目前被个人、在公司工作的个人以及那些公司所使用。每个人都有独特的要求。我们试图做的事情之一是保持水平，”他说。
