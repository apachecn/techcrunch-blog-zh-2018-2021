Tape It 为 iPhone 推出人工智能音乐录制应用 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/08/tape-it-launches-an-a-i-powered-music-recording-app-for-iphone/

今年早些时候，苹果正式停止了 Music Memos，这是一款 iPhone 应用程序，允许音乐家快速录制音频并开发新歌创意。现在，一家名为 Tape It 的新创公司正在用一款应用填补这一空白，该应用通过提供各种功能来改善音频录制，包括更高质量的声音、自动乐器检测、对标记、音符和图像的支持等等。

将它录制下来的想法来自两位朋友和音乐家，托马斯·瓦尔特和简·纳什。

在 Spotify 于 2017 年收购了他联合创立的音频检测初创公司 Sonalytic 之后，瓦尔特曾在 Spotify 工作了三年半。与此同时，纳什是一名受过传统训练的歌剧演唱家，他还会弹贝斯，还是一名工程师。

设计师兼音乐家克里斯蒂安·克鲁修斯(Christian Crusius)也加入了他们的行列，他之前是设计咨询公司峡湾(Fjord)的员工，该公司已被埃森哲(Accenture)收购。

沃尔特说，这些在一个乐队一起演奏了多年的创始人受到了制作磁带的启发，因为这是他们自己想要的东西。在结束了在 Spotify 的工作后，他在新成立的 Soundtrap 部门(在线音乐初创公司 Spotify 也于 2017 年收购)工作，他知道他想从事一个更专注于音乐制作方面的项目。虽然声音陷阱对一些人有用，但这并不是沃尔特和他的朋友们所需要的。相反，他们想要一个简单的工具，让他们可以用手机录制音乐——这是音乐家们今天经常使用苹果的语音备忘录应用程序，简称为音乐备忘录——直到它消亡。

“不管你是业余爱好者还是职业巡回赛爱好者，你都会用手机记录下你的想法，因为这是你随身携带的东西，”沃尔特解释道。“相机也是一样——最好的相机就是你随身携带的那台。而最好的录音工具就是你随身携带的那个。”

也就是说，当你想录音时，最简单的方法不是拿出你的笔记本电脑，连接一堆电缆，然后加载你的工作室软件——而是点击你 iPhone 上的录音按钮。

磁带 It 应用程序可以让你做到这一点，但增加了其他功能，使其更有竞争力的内置竞争，语音备忘录。

当你用磁带录制时，该应用程序利用人工智能自动检测乐器，然后用视觉指示注释录音，通过寻找彩色图标使这些录音更容易找到。音乐家也可以在录制时将自己的标记添加到文件中，然后添加注释和照片来提醒自己其他细节。瓦尔特说，这在以后回顾录音时会很有用。

“如果我有一个很好的吉他声音，我可以拍下我的放大器上的设置，我有它们。这是音乐家一直在做的事情，”他指出。“这是重现那种声音最简单的方法。”

磁带的另一个新颖但简单的变化是它将较长的录音分成多行，类似于一段文字。该团队称之为“时间段”，并认为这将使听更长的会话比默认设置更容易——默认设置通常是单个水平滚动的记录。

除了可选的标记和照片之外，该应用程序还被设计为更容易回到录音的正确部分，这要归功于它的智能波形。您还可以将录音标记为收藏，这样您就可以快速调出您的最佳创意和声音列表。该应用程序还提供了完整的媒体中心集成，所以你可以在任何时候有时间播放你的音乐。

然而，突出的功能是磁带它的“立体声高清”质量的支持。在这里，该应用程序利用 iPhone XS、XR 和其他更新型号的设备上的两个麦克风，然后使用人工智能技术和其他降噪技术来改善声音，这些技术是内部开发的。这项功能是其每年 20 美元的高级订阅的一部分。

随着时间的推移，Tape It 打算扩大对人工智能和其他 IP 的使用，以进一步提高音质。它还计划引入协作功能，并支持将录音导入和导出到专业工作室软件中。这最终可能会让 Tape It 进入 SoundCloud 最初追逐的市场，之后它将重点转向更面向消费者的服务。

但首先，在添加更多共享功能之前，它希望解决单用户工作流问题。

“我们认为确保它有用非常重要，即使只是对您而言。你可以合作的东西——如果你自己不喜欢用，你就不会用它，”沃尔特说。

Tape It 的三人团队位于斯德哥尔摩和柏林，目前正在启动。

这款应用程序本身可以在 iOS 上免费下载，之后将支持 Mac 和 Windows 上的桌面用户。没有计划推出安卓版本。
