Box 赢得与维权投资者 Starboard Value 的代理董事会之战

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/09/box-wins-proxy-board-battle-with-activist-investor-starboard-value/

Box 和其大股东 T2 Starboard Value 对董事会控制权的争夺于今天结束，公司董事会轻松击败了 Starboard。在争夺公司控制权的过程中，双方几个月的周旋达到了高潮。

Box 在发表了一份有点普通的声明，对结果表示感谢:

Box 感谢我们的股东在整个过程中给予我们的支持和观点。董事会和管理团队将继续专注于继续改造 Box，并执行 Box 的战略，以实现盈利增长，并为所有 Box 股东带来重大价值。

另一方面，如你所料，Starboard 对结果感到不满，并在今天早些时候发布的致股东的信中毫不掩饰。

“我们当然对这次选举的结果感到失望，它严重扭曲了与优先股融资相关的投票权，并利用股东资本在记录日期之前从可能支持变革的股东手中积极回购股票。在这个节骨眼上，Box 的未来掌握在董事会手中，还有大量的工作要做。已经做出了许多承诺，我们希望 Box 最终能够兑现其承诺，推动改善结果、问责制、治理和薪酬实践，”董事总经理 Peter A. Feld 在信中写道。

这一切都始于 Starboard Value 投资 Box，持有 7.5%的股份，最终将增长至该公司的 8.8%。凭借这些股份，该公司成为最大股东之一，但在今年 3 月之前，它一直保持相对低调。这是公众开始传言Starboard 对公司的方向不满，这场冲突可能最终导致创始人兼首席执行官 Aaron Levie 的下台或 Box 的出售。

当 Box 宣布将从 KKR 获得 5 亿美元投资时，情况发生了有趣的转变，Starboard 对此表示强烈反对，并在 5 月初发表的一封信中明确表示希望发生重大变化。正如我们当时写的:

虽然他们用最礼貌的语言表达了这封信，但很明显右舷被盒子激怒了。Starboard 在信中写道:“虽然我们感谢过去两年与 Box 的管理团队和董事会(简称“董事会”)进行的对话，但我们对持续糟糕的业绩、可疑的资本分配决策和低于标准的股东回报越来越失望。”

不到一周之后，右舷开始争夺船上的座位，控制权之争开始了。投票前，Box 发布了两份不错的收益报告，巩固了其地位；该公司采取了不寻常的举动，提前公布结果，以便在投票前向选民提供信息。

该公司还做出了不同寻常的举动，向美国证券交易委员会提交了一份文件，对 Starboard 的候选人名单进行了反击。最终，Box 赢得了这场战斗。Deep Analysis 的创始人兼首席分析师艾伦·佩尔茨-夏普(Alan Pelz-Sharpe)多年来一直关注 Box 运营的内容管理领域，他认为这是 Levie 和 Box 的胜利。

“对我来说，博克斯获胜并不奇怪。在我看来，Starboard 误读并低估了 Aaron Levie 产生的忠诚度。事实是，对于大多数 Box 员工和投资者来说，该公司是一个成功的故事，他们也知道客户群非常活跃，未来有很大的增长空间，”他说。

Pelz-Sharpe 补充说:“对 Box 来说，这一信任票将意味着他们可以(如果他们愿意)进行一些收购，并在 R&D 进行更多投资，而不会一直有一个激进的投资者在他们身后。”

很难知道接下来会发生什么，但 Starboard 目前仍保持着它的股份，它在这些数字上仍有一些影响力。正如最近的收益结果所显示的那样，在整个所有权期间，Box 表现得更好，该公司表示这仍然是最终目标。

“正如我们一再声明的那样，我们的唯一目标是帮助 Box 表现更好，并在运营绩效、财务业绩、治理和薪酬方面采用一流的做法，以便为所有股东的利益创造长期价值。Starboard 的费尔德写道:“我们将继续关注 Box 的进展，我们希望看到该公司接受由我们的参与推动的变革，并创造长期价值。”。
