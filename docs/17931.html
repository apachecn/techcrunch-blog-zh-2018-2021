Epic Games 将在 10 月关闭 Houseparty，包括视频聊天“堡垒之夜模式”功能

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/09/epic-games-to-shut-down-houseparty-in-october-including-the-video-chat-fortnite-mode-feature/

据报道，2019 年，堡垒之夜制造商 Epic Games 以 3500 万美元收购的社交视频聊天应用【Houseparty 即将关闭。公司称 Houseparty 将于 10 月停止，届时该应用将停止对其现有用户运行；然而，今天它将从应用商店下架。与此相关的是，Epic Games 的“堡垒之夜模式”功能，该功能利用 Houseparty 为堡垒之夜游戏玩家带来视频聊天，也将停止使用。

Houseparty 成立于 2015 年，为用户提供了一种与朋友进行群组视频聊天甚至玩游戏的方式，如 Uno、trivia、Heads Up 等。去年，Epic Games 将 Houseparty 与堡垒之夜整合，最初是为了让游戏玩家在玩游戏时可以看到朋友的实时反馈，后来将对直播游戏的支持直接添加到 Houseparty 中。当时，这些整合似乎是解释 Epic Games 当初为何收购这家社交初创公司的最终目标。

现在，就在宣布收购的两年多后，在应用程序中加入对直播的支持不到半年后，Houseparty 正在关闭。

该公司没有提供任何坚实的见解，乍一看，这似乎是承认未能利用其收购。但现实是，Epic Games 可能有比视频聊天更大的东西。也就是说，Epic Games 今天只能说，Houseparty 团队无法再给予该应用所需的关注——这一声明表明，管理层决定将团队的重点转移到其他事情上。

虽然没有 Houseparty 团队成员因为这一举动而被解雇，但我们被告知，他们将加入其他团队，在那里他们将致力于新的方式，以允许跨 Epic Games 系列产品的“社交互动”。该公司的声明暗示，这些社交功能将在“元宇宙规模”下设计和建造

“元宇宙”是一个越来越常用的时髦词，指的是一个共享的虚拟环境，就像堡垒之夜、Roblox 等大型在线游戏平台提供的那些虚拟环境一样。脸书也宣称元宇宙是社交网络的下一步大棋，首席执行官马克·扎克伯格将其描述为“你置身其中而不仅仅是看一眼的具体化互联网”

在某种程度上，堡垒之夜已经开始拥抱元宇宙，提供非游戏体验，比如你以虚拟形象参加的在线 演唱会，以及其他现场活动。在关闭之前，Houseparty 还尝试了现场活动，用户可以和朋友一起观看和参与。

Epic Games 发言人告诉 TechCrunch，Houseparty 团队已经(并将继续)致力于许多其他专注于社交的项目。但我们被告知，Epic Games 正在进行的一些“多个更大的项目”仍未公开。

在社交产品方面，Houseparty 的技术现在支撑着所有的堡垒之夜语音聊天，他们开发的功能可以通过 Epic Games Services 免费提供给开发者。他们还致力于建立新的社交体验，从堡垒之夜全球活动的社交 RSVP 功能，如最近的爱莉安娜·格兰德演唱会，到即将到来的合作任务和其他游戏机制的“行动:天火”活动。堡垒之夜的用户生成内容平台 Create Mode 也在增加更多的社交功能和新体验。

虽然关闭一个仅在去年因疫情而使用量激增的应用程序似乎有点奇怪，但 COVID 的增长似乎没有持久的力量。

在关闭最严重的时候，Houseparty 曾报告称，在一个月的时间里，它获得了 5000 万新注册用户,因为用户在世界关闭的时候，通过视频应用程序与家人和朋友联系。但是随着疫情的发展，其他的视频聊天体验越来越普及。Zoom 已经成为远程工作的必备工具，也成为了下班后和朋友一起出去玩的工具。脸书也开始吃 Houseparty 的午餐，去年首次推出视频聊天“房间”，提供类似的群组视频体验。无聊的用户转向基于音频的社交网络，如 Clubhouse 或 Twitter Spaces。

图片来源: Apptopia

根据 Apptopia 的数据，自从疫情事件以来，家庭聚会一直在持续下降。迄今为止，其应用程序在 iOS 和 Android 上的下载量共计 1.11 亿次，其中大多数(6300 万次)在 iOS 上。美国是 Houseparty 最大的市场，占下载量的 43.4%，其次是英国(9.8%)，然后是德国(5.6%)。

与此同时，Epic Games 表示，该应用服务于全球“数千万”用户。它坚持认为，关闭不是轻易决定的，关闭“堡垒之夜模式”的决定也不是因为缺乏采纳。

Houseparty 将在 10 月份最终关闭之前，通过应用内通知提醒用户关闭。届时，堡垒之夜模式也将不再可用。
