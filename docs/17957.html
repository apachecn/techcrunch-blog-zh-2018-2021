Twitter 引入了一个新标签，允许“好机器人”识别自己 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/09/twitter-introduces-a-new-label-that-allows-the-good-bots-to-identify-themselves/

Twitter 今天推出了一项新功能，允许帐户通过在个人资料中添加标签来自我识别为机器人。这项功能旨在帮助人们更好地区分自动账户——如转发新闻、公共服务公告或其他更新的机器人——和人工操作的账户。然而，它不是为了帮助用户识别“坏机器人”，即那些伪装成人类，经常传播错误信息或垃圾邮件的机器人。

该公司多年来一直在考虑给机器人贴标签。

2018 年， Twitter 首席执行官杰克·多西在参议院情报委员会听证会上被问及他是否认为用户在 Twitter 平台上与机器人或人类交谈时有“知情权”。他同意 Twitter 应该给推文添加更多的上下文，并正在考虑尽可能地识别机器人。然而，多尔西也指出，与那些利用 Twitter API 的机器人相比，识别使用脚本来表现人类外表的机器人将更加困难。

去年，公司最终巩固了这些计划，称它将在晚些时候推出新功能，让用户能够区分人工账户和自动账户。当 Twitter 在 5 月推出账户验证系统时，它提醒用户，除了渴望已久的蓝色徽章，它将很快提供其他方式来识别不同类型的账户——如机器人标签。

图片来源:推特

今天，Twitter 表示，其识别“好机器人”的新“自动账户”标签将对 500 多个开发者账户开放。该小组将测试该功能，并在它更广泛地向所有 Twitter 开发者开放之前提供反馈。因为暂时还是测试，所以标签就不需要了。

然而，当 Twitter 去年更新其开发者政策时，它确实要求开发者在他们的账户资料或简历中注明该账户是否是一个机器人，该账户是什么以及其背后的人是谁。这些帐户标签将允许开发者更容易地遵守该政策，而不是必须在他们的简历中手写这些信息。

Twitter 告诉 TechCrunch，根据在这次实验中了解到的情况，一旦它变得广泛可用，它可能会决定在未来要求所有运行自动账户的开发人员采用这个标签。

图片来源:推特

需要明确的是，Twitter 对那些运行良好的机器人没有任何问题，因为它知道自动化如何让账户向人们更新有用的、相关的或者有时只是有趣的信息。该公司甚至在宣布今天的开发者新闻时庆祝了几个它最喜欢的机器人，包括公共服务账号@ earthquakesSF；一个提供新冠肺炎更新的 bot，@ VAX _ progress；一个提供国会最近 100 项法案持续明细的机器人，@最近 100 项法案；一个关注可访问性的 bot，@ AltTxtReminder；其他人只是以他们自己的方式增加价值，像 @met_drawings ，它从大都会博物馆的图纸&印刷部门分享公共领域的作品，或者傻瓜 @EmojiMashupBot ，等等。

所有这些都将是初始测试组的一部分。

Twitter 也不太关心消费者如何使用自动化来更新自己的账户，也许是通过使用 IFTTT 等第三方工具来发布链接或其他内容。

Twitter 的政策建议 Twitter 用户，“你最终要对你的账户或与你的账户相关的应用程序所采取的行动负责”。在授权第三方应用程序访问或使用您的帐户之前，请确保您已经彻底调查了该应用程序并了解它将会做什么它还补充说，采用自动化的 Twitter 用户仍然需要遵守 Twitter 的准则。

该公司最近在推出新功能方面进展迅速。就在本周，它推出了社区、表情反应测试、对全幅照片和视频的支持以及一种方式来“软屏蔽”粉丝，等等。

Twitter 没有说明在自动账户标签被更广泛地推广之前，测试将持续多久。
