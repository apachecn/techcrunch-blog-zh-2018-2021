Epic Games 对上周与苹果反垄断斗争的裁决提出上诉

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/13/epic-games-appeals-last-weeks-ruling-in-antitrust-battle-with-apple/

堡垒之夜制造商 Epic Games 正在对上周与苹果的法庭大战中的裁决提出上诉，一名联邦法官表示，苹果将不再被允许阻止开发者添加替代支付机制的链接，但没有将苹果定性为垄断者。后者可能会让 Epic Games 提出为其 iOS 用户群提供服务的替代方式，可能包括通过第三方应用商店，甚至是内置于苹果移动操作系统中的侧装功能，类似于谷歌 Android 操作系统中的功能。

苹果公司立即宣布这场官司取得了胜利，因为法官同意了苹果公司的立场，即该公司“没有违反反垄断法”，并且认为苹果公司在应用和游戏生态系统中的成功“不违法”与此同时，Epic Games 创始人兼首席执行官蒂姆·斯维尼表示，这一裁决对开发商和消费者来说都不是一个胜利。在推特上，他暗示公司可能会对的决定提起上诉，他说“我们会继续战斗下去。”

在周日公布的一份法庭文件中(见下文)，Epic Games 正式表示将对美国地区法官伊冯娜·冈萨雷斯·罗杰斯的最终判决以及“导致或产生该判决的所有命令”提起上诉

作为法官裁决的一部分，Epic Games 被要求向苹果支付其在堡垒之夜 iOS 上引入替代支付系统时获得的 1200 万美元的 30%，这违反了其与苹果的法律合同。

上诉法院将重新审视冈萨雷斯·罗杰斯法官如何定义 Epic Games 辩称苹果是垄断者的市场。与双方的愿望相反，冈萨雷斯·罗杰斯明确地将其定义为“数字移动游戏交易”市场。虽然上诉可能会也可能不会让法院改变对 Epic Games 有利的观点，但一项新的裁决可能有助于澄清禁令中使用的模糊语言，该语言描述了苹果现在必须如何适应那些想让客户使用其他支付机制的开发者。

到目前为止，开发者社区的预期是，苹果将简单地将“阅读器应用”类别的例外扩展到所有非阅读器应用(提供购买内容访问的应用)。苹果最近与日本监管机构达成和解，同意允许阅读器应用程序将用户指向他们自己的网站，用户可以在那里注册并管理他们的账户，这可能包括付费订阅的客户——例如网飞或 Spotify 订阅。苹果表示，这一变化将是全球性的。

在与记者的简报中，苹果公司表示，与 Epic Games 裁决一起发布的禁令的细节仍需制定。鉴于这一决定是最近做出的，该公司尚未与开发者沟通这一变化将如何直接影响他们，也没有用新的语言更新其应用商店指南。

联系到 Epic Games 发表评论，Epic Games 表示，目前没有关于其上诉决定的任何进一步声明。
