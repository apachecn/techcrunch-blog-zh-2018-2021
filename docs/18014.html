贸易促进管理初创公司 Cresicor 筹集了 560 万美元来跟踪客户支出

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/13/trade-promotion-management-startup-cresicor-raises-5-6m-to-keep-tabs-on-customer-spend/

消费品贸易管理平台初创公司 Cresicor 筹集了 560 万美元的种子资金，以进一步开发其工具，获得更准确的数据和分析。

该公司位于远程，专注于中小型 CPG 公司，为他们提供一种自动化的方式来管理他们的贸易推广，该流程的联合创始人兼首席执行官亚历山大·沃特利(Alexander Whatley)表示，主要是使用电子表格手动完成的。

在贸易促销中会发生这样的事情:当一家公司想对他们销售较慢的商品打折时，该公司必须花钱这样做——在商店里设置展示或在某个货架上摆放该商品。如果行得通，更多的人会以更低的价格购买该商品。Whatley 告诉 TechCrunch，从本质上说，贸易促进是花钱在未来获得更多钱的过程。

Whatley 解释说，搞清楚所有的促销活动是一个复杂的过程。公司从几个不同的地方接收促销数据，从零售商那里接收收入数据，从会计源数据显示发货数量，然后可能直接从零售商那里接收数据。所有这些都必须与促销活动相匹配。

“没有 API 将这些数据带回品牌，所以我们的软件有助于自动化和跟踪这些手动过程，以便公司可以进行分析，看看促销活动进行得如何，”他补充说。它还有助于财务团队了解费用，包括哪些费用是有效的，哪些是无效的

Whatley 说，某些公司在贸易促销上的支出可能是仅次于制造业的第二大成本，公司通常会将 20%至 30%的收入再投资于贸易促销。这是一个巨大的市场，代表着尚未开发的增长，特别是美国的 CPG 销售额将在 2020 年超过 7200 亿美元。

“你可以看到整个行业是多么混乱，这就是为什么我们有一个光明的未来和巨大的 TAM，”他补充说。“有了这笔新资金，我们可以瞄准 P&L 的其他领域，比如供应链和薪资。我们还为他们的战略以及他们应该在哪里消费提供分析——哪家商店，哪种供应。通过正确分配资源，公司通常会看到销售额增加 10%。”

沃特利在 2017 年与他的哥哥丹尼尔、斯图尔特·肯尼迪和尼基·麦克尼尔一起创办了这家公司，当时他还是哈佛大学的本科生。自 2 月份筹集资金以来，该公司的收入增长了 2.5 倍，员工人数在过去 12 个月中增长了 4 倍，达到 20 人。

Costanoa Ventures 领投，Torch Capital 和一批天使投资人也参与了投资，包括五星电器首席技术官马特多卡和胡的厨房首席执行官马克拉马丹。

Costanoa 的合伙人约翰·考吉尔(John Cowgill)表示，尽管 Cresicor 筹集了一轮种子资金，但该公司在发布一款产品之前就已经在收购品牌和资本，并在没有任何外部资本的情况下几乎发展成为一家 A 轮公司，称其“让我震惊”

Cowgill 补充说，Cresicor 是 Costanoa 会感到兴奋的公司的“完美例子”——一家使用数据或机器学习来增加痛点的垂直软件公司。

“CPG 行业正处于快速变化之中，我们看到所有这些新兴的、数字化的本土和使命驱动的品牌正在迅速从现有品牌那里蚕食份额，”他补充道。“为了让下一代品牌参与竞争，它们必须在贸易推广管理方面取得胜利。Cresicor 超越贸易的机会意义重大。这只是建立一个公司的起点，这个公司是伟大品牌的核心推动者。”

亚历山大·沃特利说，新的资金将主要用于在工程和客户成功领域雇用更多的人才，以便公司能够达到下一个基准。他还打算将这笔资金用于收购新品牌和软件开发。Cresicor 拥有一份客户名单，其中包括 Perfect 零食、Oatly 和 Hint Water。

Whatley 说，零售业价值 5.5 万亿美元，其中五分之一是 CPG。因此，他着眼于 CPG 的其他垂直领域，如电子产品和宠物食品，然后扩展到其他领域。

“我们也将与企业公司合作——我们看到了与 P&G 和通用磨坊等公司合作的机会，我们也希望围绕贸易促进建立一个生态系统，并进入其他利润和损失领域，”沃特利说。
