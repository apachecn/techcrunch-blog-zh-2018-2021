原文：https://web.archive.org/web/https://techcrunch.com/2021/09/15/microsoft-now-lets-you-sign-in-without-your-password/

Microsoft 现在允许您无需密码即可登录

微软通过向所有消费者微软账户推出无密码登录选项，进一步推动用户远离密码。

与业内其他许多公司一样，这家科技巨头已经向传统的基于密码的身份认证发起了一场战争。这是因为密码是网络攻击的主要目标，因为弱密码或重复使用的密码可以通过自动攻击被猜到或强行破解。

为此，随着微软准备在几周后推出 Windows 11，微软将向所有微软账户推出其无密码登录选项，此前该选项仅面向商业客户。这意味着用户将能够登录 Outlook 和 OneDrive 等服务，而无需使用密码。相反，用户可以使用微软认证应用程序、Windows Hello、安全密钥和短信或电子邮件代码。

然而，一些微软应用程序仍然需要密码，包括 Office 2010 或更早版本、远程桌面和 Xbox 360。同样，那些使用现在不受支持的 Windows 版本的用户也不能放弃他们的密码，因为该功能将只在 Windows 10 和 Windows 11 上受支持。

微软表示，无密码登录将在未来几周内推广到消费者账户，所以你可能还不能摆脱你的密码。它补充说，它也在努力消除 Azure AD 帐户的密码，管理员可以选择特定用户是否需要、允许或不存在密码。
