Zonos banks 投资 6900 万美元开发用于跨境商务民主化的 API TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/15/zonos-banks-69m-to-develop-apis-for-democratizing-cross-border-commerce/

由 Silversmith Capital Partners 牵头的跨境商务公司 Zonos 在首轮融资中筹集了 6900 万美元，以继续开发其自动分类商品并计算国际交易总成本的 API。

总部位于犹他州圣乔治的 Zonos 将此轮投资归类为少数股权投资，其中包括个人投资者 Podium 首席执行官 Eric Rea 和 Pluralsight 联合创始人兼首席执行官 Aaron Skonnard。Zonos 创始人兼首席执行官 Clint Reid 告诉 TechCrunch，A 轮融资是 Zonos 自 2009 年成立以来筹集的第一笔外部资本。

正如 Reid 解释的那样，“总到岸成本”指的是来自另一个国家的人在从美国购买商品时可能支付的关税、税款、进口和运费。然而，企业通常很难计算出这些费用的确切成本。

2018 年，全球跨境电子商务预计超过 4000 亿美元，但增速是国内电子商务的两倍。这就是 Zonos 的用武之地:该公司的 API、应用程序和插件通过提供消费者在国际采购中为一件商品支付的准确最终价格，简化了跨境销售。企业可以选择与哪一家或多家运输公司合作，甚至允许客户在购买时进行选择。

“企业不可能知道一个国家的所有法律，”里德补充道。“我们的使命是在全球贸易中建立信任。如果你是透明的，你会带来信任。这在传统上被认为是一个运输问题，但实际上是一个技术问题。”

作为投资的一部分，Silversmith Capital Partners 的管理合伙人托德·麦克莱恩(Todd MacLean)加入了 Zonos 董事会。吸引麦克莱恩加入该公司的原因之一是，里德正在硅谷之外建立一家公司，并在远离任何港口的地方扰乱全球贸易。

他说，在调查国际商务时，他发现人们在已经购买了商品后，最终会被收取额外的费用，导致糟糕的客户体验，尤其是当商家试图建立品牌忠诚度时。

即使有人因为费用太高而选择不购买，麦克莱恩认为购买体验也会有所不同，因为价格和运输信息是预先提供的。

“我们的勤奋表示，Zonos 是唯一一家获取现有数据并加以理解的公司，”麦克莱恩说。“客户喜欢它，我们得到了最令人印象深刻的客户推荐，因为这种需求已经存在，他们看到了更多的收入，他们的客户也更加忠诚，因为它确实有效。”

里德补充说，事实上，公司销售额每年增长 25%到 30%是很常见的。他接着说，由于与运输相关的费用，这并不总是意味着公司收入的增加。可能会有小幅下降，但对客户的终身价值更长。

在这个时候寻求风险投资对里德来说很重要，他认为随着各国增加新的税法并停止使用其他贸易法规，全球贸易变得更加复杂。然而，这不仅仅是获得资金，而是找到合适的合作伙伴，认识到这个问题不会在未来五年内解决，而是需要长期解决，Reid 说他在 Silversmith 身上看到了这一点。

新的投资为 Zonos 在产品开发和上市方面的增长提供了动力，同时也将其全球团队扩展到了欧洲和亚太地区。18 个月前，该公司有 30 名员工，现在有 100 多名。它还在全球拥有 1500 多家客户，每天为他们提供数百万份到岸成本报价。

“目前，我们是跨境电子商务中 API 的领导者，但我们也需要成为技术领导者，不管是哪个行业，”Reid 补充道。“我们不能仅仅接受我们已经足够好，我们需要在这方面做得更好。我们正在考虑扩展到更多的市场，因为它不仅仅是为美国公司服务，而是需要与我们的客户在一起。”
