Avalanche 通过私下出售 AVAX 令牌筹集了 2.3 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/16/avalanche-raises-230-million-from-private-sale-of-avax-tokens/

Avalanche 是一家相对较新的区块链，专注于速度和低交易成本，已完成向一些知名加密基金私下出售 2.3 亿美元的 AVAX 令牌。Polychain 和三箭资本领投。

雪崩基金会早在 2021 年 6 月就完成了私人出售，并于今天披露。参与此次私募的其他公司包括 R/Crypto Fund、蜻蜓、CMS Holdings、Collab+Currency 和吕娜资本。

私人出售的收益将用于支持雪崩生态系统，例如，当你与以太坊区块链相比时，这是相对新生的。除其他事项外，该基金会计划通过赠款、代币购买和其他形式的投资支持 DeFi(分散融资)项目以及企业申请。

像索拉纳和其他新的区块链一样，Avalanche 想要解决老区块链面临的可扩展性问题。例如，如果你最近试图在区块链以太坊买一幅 NFT，你可能要支付 50 或 100 美元的交易费或油费。

雪崩基金会将其区块链定位为以太坊的可靠替代品。您可以运行 Dapps(分散式应用程序),只需很少的成本，而且完成时间要快得多。Avalanche 支持智能合同，这是支持 DeFi 项目的一个关键特性。

当你与以太坊和其他区块链竞争时，拥有更好的性能只是问题的一部分。Avalanche 还需要吸引开发人员，并建立一个强大的开发人员社区，以便成为其他加密项目的基础设施。

这就是为什么 Avalanche 希望尽可能容易地将您的以太坊 Dapp 移植到 Avalanche。Avalanche 的智能合同链执行以太坊虚拟机合同，这意味着如果你已经活跃在以太坊区块链上，你可以重用你的部分代码库。

类似地，查询以太坊网络的应用程序可以通过更改 API 端点和添加对新网络的支持来支持 Avalanche。Avalanche 团队也在研究一座桥梁，将以太坊的资产转移到 Avalanche 区块链。相当于 13 亿美元的加密资产已经通过这座桥转移。

这些是技术激励。至于财政激励，私人销售和赠款可能有助于刺激开发商的兴趣。雪崩基金会表示，目前有 225 个项目支持该平台，包括已经在其他区块链上运行的流行加密项目，如 Tether、SushiSwap、Chainlink、Circle 和 The Graph。与 MLB 和德甲有合作关系的 NFT 游戏 Topps 也在使用 Avalanche。

根据 CoinMarketCap 的数据，AVAX 及其底层令牌 AVAX 目前是总市值排名第 14 位的加密货币。Avalanche 目前的市值为 130 亿美元，领先于 Algorand 或 Polygon，但落后于 Polkadot 和 Solana。本周早些时候，索拉纳还遭遇了大停电，引发了对索拉纳扩展能力的质疑。看看这些区块链中的一个是否能在用途和价值上赶上甚至超过以太坊将会很有趣。
