CodeSignal 为其技术招聘平台 TechCrunch 获得 5000 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/16/codesignal-secures-50m-for-its-tech-hiring-platform/

在筹集了 2500 万美元的 B 轮融资后不到一年的时间里，技术评估公司 CodeSignal 宣布了 5000 万美元的 C 轮融资，为其平台提供新功能，帮助公司做出数据驱动的招聘决策，以发现和测试工程人才。

类似于吸引 B 轮大投资者——Menlo Ventures——它与 Index Ventures 合作领导 C 轮投资。门罗再次参加，并加入了头条新闻和一个资本。这一轮使 CodeSignal 的总融资额达到 8750 万美元。

联合创始人兼首席执行官 Tigran Sloyan 从他的联合创始人兼朋友 Aram Shatakhtsyan 在试图找一份工程工作时的经历中获得了创办该公司的想法。两人都来自亚美尼亚，上大学的道路各不相同，Shatakhtsyan 留在亚美尼亚，Sloyan 来到美国麻省理工学院学习。然后，他继续在谷歌工作。

Sloyan 告诉 TechCrunch:“当公司在招聘我和我的同学时，Aram 试图让他的简历被选中，但由于他在哪里上的大学而没有得到关注，尽管他是我所知道的最伟大的程序员。”“招聘人才是公司面临的第一大问题，但这里有最好的工程师，没有人愿意让他进来。”

他们和索菲亚·拜克(Sophia Baik)在 2015 年创立了 CodeSignal，作为一个自动驾驶面试平台，不管一个人的背景如何，都可以直接衡量技能。与需要参加驾照考试的人一样，Sloyan 将该公司的技术评估技术称为“开发人员的飞行模拟器”，它可以对候选人的技能进行模拟评估，并返回分数和突出的优势。

公司雇佣工程师的需求导致 CodeSignal 的收入同比增长了 3.5 倍，并收集了一份客户名单，其中包括 Brex、Databricks、脸书、Instacart、Robinhood、Upwork 和 Zoom。

Sloyan 表示，该公司尚未动用 B 轮融资中获得的资金，但希望抓住机会与 Index Ventures 的合伙人 Nina Achadjian 合作，他们在谷歌共事时就认识多年。Sloyan 说，一起工作并让 Achadjian 加入公司董事会是“我不能错过的事情”。

当 Achadjian 转向风险投资时，她帮助 Sloyan 联系导师和天使投资人，同时关注公司。Achadjian 解释说，雇用工程师对技术公司来说是“至关重要的”，但对她来说更明显的是，工程职能已经成为所有公司的必要条件。

在对该领域进行尽职调查时，她看到了利用 CodeSignal 的传统工程文化，但也看到了像银行和保险公司这样的非传统公司。

“他们的吸引力是不可否认的，我们投资组合中的许多公司都在使用 CodeSignal，”她补充道。“很难看到一家公司在目前的阶段加速增长。”

美国劳工部统计估计，全球人才劳动力短缺已经达到 4000 万人，到 2030 年，这一数字将超过 8500 万。Achadjian 说，在此期间，工程职位预计也会增加，鉴于所有这些职位和申请人，审查候选人将比以往任何时候都更重要，候选人无论在哪里都能申请也是如此。

新的资金使该公司能够推出其集成开发环境，供候选人在熟悉的机器上与相关的评估体验(如代码、文件和终端)进行交互，以便他们可以展示他们的技能，同时还能够预览他们的应用。同时，雇主能够根据空缺职位给每个候选人分配相同的编码任务。

此外，Sloyan 计划在未来几个月内将公司员工人数增加两倍，并扩展到技能评估的其他用例中。
