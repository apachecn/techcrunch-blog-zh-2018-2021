Tile 获得 4000 万美元，通过新产品挑战 Apple AirTag TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/16/tile-secures-another-40-million-to-take-on-apple-airtag-with-new-products/

蓝牙失物招领信标制造商 Tile T1 以及最近坚定批评苹果的 T2 T3 今天宣布，它已经从 Capital IP 获得了 4000 万美元的非稀释性债务融资。这笔资金将用于投资 Tile 的 finding technologies，先于该公司推出一系列新产品和新功能的计划，该公司认为这将有助于它更好地与苹果的 AirTags 竞争，并进一步扩大其市场。

该公司长期以来一直是失物招领领域的领导者，为消费者提供可以附着在物品上的小型设备，如手袋、行李、自行车、钱包、钥匙等，然后可以使用 iOS 或 Android 智能手机应用程序跟踪这些物品。当物品丢失时，Tile 应用程序利用蓝牙找到物品，并让它们发出声音。如果物品在更远的地方，Tile 会接入其更广泛的寻找网络，该网络由每个在手机和其他接入点上安装了该应用的人组成。通过这个网络，Tile 能够通过自己的 Tile 应用程序自动和匿名地将丢失物品的位置传达给失主。

图片来源:平铺

Tile 还建立了合作伙伴关系，专注于将其搜索网络集成到 40 多种不同的第三方设备中，包括音频、旅游、可穿戴设备和 PC 类别的设备。知名品牌合作伙伴包括惠普、戴尔、Fitbit、Skullcandy、Away、Xfinity、Plantronics、Sennheiser、Bose、Intel 等。Tile 表示，这些嵌入了其服务的设备的激活量同比增长了 200%。

迄今为止，Tile 已经销售了超过 4000 万台设备，拥有超过 42.5 万名付费用户——这是它首次披露这一指标。不过，它没有透露免费和付费用户的总数。Tile 表示，在 2021 年上半年，收入增长了 50%以上，但没有提供具体数字。

虽然 Tile 承认新冠肺炎疫情对国际扩张产生了一些影响，因为一些市场的反弹速度较慢，但它在美国以外仍有强劲的表现，并认为这是一个持续的重点。

然而，疫情并不是 Tile 唯一的减速带。

当苹果宣布其计划与 AirTags 的推出竞争时，公开称其为不公平竞争。与 Tile 设备不同，苹果的产品可以接入 iPhone 的 U1 芯片，通过使用新款 iPhone 上可用的超宽带技术，实现更准确的搜索。与此同时，Tile 公司计划开发自己的超宽带设备，但没有提供同样的接入。换句话说，苹果很早就给了自己的失物招领系统一个功能，这个功能可以让它在竞争中脱颖而出。(苹果后来宣布，它正在向第三方开发者提供超宽带 API，但这种访问从 AirTag 问世的第一天起就不可用。)

图片来源:瓷砖内部概念艺术

Tile 一直对苹果的反竞争行为直言不讳，与其他苹果批评者一起在多个国会听证会上作证，如 Spotify 和 Match。由于监管压力的增加，苹果后来向第三方设备开放了它的 Find My network ，以安抚 Tile 和它的 AirTags 将处于劣势的其他竞争对手。

但 Tile 不想让其客户使用苹果的第一方应用程序——它打算使用自己的应用程序，以便基于其专有的功能和服务进行竞争。除其他外，这包括瓷砖的订阅。基本计划是每年 29.99 美元，提供免费更换电池、智能提醒和位置历史记录等功能。每年 99.99 美元的计划还增加了各种保险——它每年为找不到的项目支付高达 1000 美元。(AirTag 不这么做。)

尽管有许多不同之处，Tile 仍面临来自超宽带 AirTags 的激烈竞争，后者的优势是可以进入苹果自己的潜在数亿 iPhone 用户的搜索网络。

然而，Tile 首席执行官 CJ Prober——于 2018 年加入该公司——声称 AirTag 没有影响公司的收入或设备销售。

“但这并不能抹杀他们给我们制造困难的事实，”他谈到苹果时说。“我们的业务在不断增长。我们赢得了消费者的心……而他们在进行不公平的竞争。”

“当你拥有这个平台时，你不应该能够确定你想要进入的类别，让该类别中的现有者处于劣势，然后让自己处于优势——就像他们在我们的案例中所做的那样，”他补充道。

Tile 正准备宣布即将到来的产品更新，这可能会使它更好地接受 AirTag。据推测，这将包括预先宣布的超宽带版本的瓷砖，但该公司表示，完整的细节将于下周共享。Tile 还可能以其他方式扩大其产品线，这将使其在外观和感觉、尺寸和形状以及功能上更具竞争力。

Tile 上一轮融资是 2019 年4500 万美元的成长股权。现在转移到债务上了。它说，除了新的债务融资，Tile 还通过此次融资对现有债务进行了再融资。

“我的哲学是，债务和股权相结合总是好的。因此，资产负债表上一定数量的债务是好的。普罗伯特说:“这也不会导致我们的股东权益被稀释。”。“我们认为这是我们正确的资本组合选择。”

该公司选择与 Capital IP 合作，过去三年来一直与该集团保持合作关系，Tile 曾考虑让该集团作为投资者。Prober 指出，该组织对 Tile 一直保持兴趣，并对其发展轨迹感到兴奋。

“我们很高兴与 Tile 团队合作，因为他们继续通过基于硬件和软件的创新来定义和引领发现类别，”Capital IP 的管理合伙人里亚德·沙迦汗在一份声明中说。“令人印象深刻的收入增长和快速攀升的用户趋势强调了 Tile 以平台无关的方式提供的价值主张，并且是我们决定投资的关键驱动因素。瓷砖团队有一个雄心勃勃的路线图，我们期待着支持他们进入新的市场和应用，以进一步巩固他们的市场领导地位，”他补充说。
