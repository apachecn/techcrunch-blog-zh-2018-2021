麻省理工学院的研究发现，当自动驾驶仪启动时，特斯拉司机变得注意力不集中

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/20/mit-study-finds-tesla-drivers-become-inattentive-when-autopilot-is-activated/

到本周末，潜在的数千名特斯拉车主将在公共道路上测试该汽车制造商最新版本的“全自动驾驶”测试软件，版本 10.0.1，即使监管机构和联邦官员在几次高调的撞车事故后调查该系统的安全性。

麻省理工学院的一项新的研究证实了 FSD 系统的想法，尽管它的名字实际上不是一个自主系统，而是一个高级驾驶辅助系统(ADAS)，实际上可能并不那么安全。研究人员研究了 290 个人类启动的自动驾驶仪脱离时期的扫视数据，发现司机在使用部分自动驾驶系统时可能会变得注意力不集中。

“视觉行为模式在[自动驾驶仪]脱离之前和之后都会发生变化，”该研究写道。“与过渡到手动驾驶后相比，在脱离之前，驾驶员较少关注道路，更多地关注与驾驶无关的区域。在脱离手动驾驶之前，越野视线的比例更高，但这并没有被更长的前方视线所补偿。”

特斯拉首席执行官埃隆·马斯克(Elon Musk)表示，并不是所有购买了 FSD 软件的人都能够获得测试版，该版本承诺提供更多自动驾驶功能。首先，特斯拉将使用遥测数据来捕捉七天内的个人驾驶指标，以确保司机仍然保持足够的注意力。这些数据还可以用来实现一个新的安全评级页面，该页面可以跟踪车主的车辆，并与他们的保险相关联。

麻省理工学院的研究提供了证据，表明司机可能没有按照建议使用特斯拉的自动驾驶仪(AP)。由于 AP 包括交通感知巡航控制和自动转向等安全功能，司机变得不那么专注，他们的手更多地离开方向盘。研究人员发现，这种类型的行为可能是误解了 AP 功能的作用及其局限性的结果，当它表现良好时，这种误解会得到加强。那些任务自动完成的司机在试图保持视觉和身体的警觉性后，可能会自然变得无聊，研究人员表示，这只会造成进一步的疏忽。

这份名为“围绕特斯拉自动驾驶脱离的自然扫视行为模型”的报告，已经在大波士顿地区跟踪了特斯拉 Model S 和 X 车主一年或更长时间的日常生活。这些车辆配备了实时智能驾驶环境记录数据采集系统 1，该系统从 CAN 总线、一个 GPS 和三个 720p 摄像机连续收集数据。这些传感器提供的信息包括车辆运动学、驾驶员与车辆控制器的互动、里程、位置以及驾驶员的姿势、面部和车辆前方的视野。麻省理工学院收集了近 50 万英里的数据。

这项研究的目的不是羞辱特斯拉，而是倡导驾驶员注意力管理系统，该系统可以实时向驾驶员提供反馈，或调整自动化功能以适应驾驶员的注意力水平。目前，Autopilot 使用手握方向盘的传感系统来监控驾驶员的参与度，但它不能通过眼睛或头部跟踪来监控驾驶员的注意力。

该研究背后的研究人员开发了一个扫视行为模型，“基于自然数据，可以帮助理解自动化下驾驶员注意力转移的特征，并支持开发解决方案，以确保驾驶员充分参与驾驶任务。”这不仅可以帮助驾驶员监控系统处理“非典型”眼神，还可以用作研究自动化对驾驶员行为的安全影响的基准。

像 Seeing Machines 和 Smart Eye 这样的公司已经与通用汽车、梅赛德斯-奔驰和据报道福特这样的汽车制造商合作，将基于摄像头的驾驶员监控系统带到装有 ADAS 的汽车上，同时也解决由醉酒或酒后驾驶引起的问题。技术是存在的。问题是，特斯拉会用吗？
