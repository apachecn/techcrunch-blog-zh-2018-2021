TrueLayer 以 1.3 亿美元的 1B+估值获得 1.3 亿美元，因为开放式银行业务已成为银行卡网络的一个可行选择

开放银行业务是一项颠覆性技术，旨在通过让银行以 API 的方式直接向开发者(和新服务)开放系统，绕过卡网络和其他传统金融轨道的主导地位。这项技术在金融服务领域不断发展壮大。作为这种牵引力的标志，一家在开放式银行应用程序中发挥核心作用的初创公司宣布了一轮具有里程碑意义的大规模融资。

TrueLayer 为开发者提供技术，以实现一系列基于开放式银行的服务，该公司在一轮融资中筹集了 1.3 亿美元，对这家总部位于伦敦的初创公司的估值超过 10 亿美元。

Tiger Global Management 在这一轮中领先，值得注意的是，支付巨头 Stripe 也参与了进来。

在金融科技领域，开放式银行是一个相对较新的领域——英国在 2018 年率先采用，欧洲随后加入，看起来我们现在看到更多的运动，美国可能很快也会加入这个行列 TrueLayer 被认为是该领域的先驱。

当今世界上的绝大多数交易仍然是使用 card rails 或更陈旧的银行基础设施进行的，但开放银行的机会是建立一个全新的基础设施，它可以更有效地工作，并可能对使用它的人收取更少(或免费)的费用，这是 API 的长期承诺:所有这些都是通过几行代码实现的。

“我们的愿景是金融应该开放，我们正在积极努力消除中介机构之间存在的摩擦，”首席执行官弗朗西斯科·西蒙尼斯基(Francesco Simoneschi)在接受采访时表示，他与卢卡·马丁内蒂(Luca Martinetti，现任首席技术官)共同创立了该公司。“我们想要一个为所有人服务的金融体系，但迄今为止情况并非如此。五年前，当开放银行在英国和其他地方成为法律时，出现了追求最令人印象深刻的寡头垄断的机会:信用卡网络和围绕它们的一切。现在，我们可以很容易地说，开放银行正在成为一个可行的替代方案。”

看起来金融和商业世界正在慢慢跟上，所以资金是紧随着公司的强劲增长而来的。

TrueLayer 目前提供的服务包括支付、支出、用户账户信息和用户验证；而最终用户的范围从新银行、加密初创公司和财富管理应用到电子商务公司、市场和游戏平台。

这家初创公司表示，它现在有“数百万”消费者通过 TrueLayer 的技术进行开放式银行交易，大约 1 万名开发者正在基于开放式银行标准构建服务。TrueLayer 今年到目前为止已经将其客户群翻了一番，获得了一些像 Cazoo 这样的关键客户，以实现基于开放银行的汽车支付；它已经处理了“数十亿”美元的支付，支付量增长了 400%，支付量增长了 800%。

该计划是利用这笔资金投资进一步发展这项业务——特别是将其支付网络扩展到更多地区(更多银行融入该网络)，以及吸引更多客户使用开放式银行服务进行更定期、更经常的交易。

Tiger Global 的合伙人亚历克斯·库克(Alex Cook)在一份声明中表示:“随着在线商务在全球的增长，向替代支付方式的转变正在加速，我们相信 TrueLayer 将在使这些支付方式更加普及方面发挥核心作用。”“我们很高兴与 Francesco、Luca 和 TrueLayer 团队合作，因为他们帮助客户提高转化率并继续扩大网络。”

值得注意的是，Stripe 目前不是 TrueLayer 的战略投资者，只是一个财务投资者。也就是说，它还没有将开放银行业务整合到自己的支付基础设施中。

但你可以想象它会如何对它感兴趣，将其作为客户更大选择的一部分，并可能建立自己的独立金融轨道，与卡网络提供的轨道进行真正的竞争(这是 Stripe 的一个紧密部分，其最早的 web 设计是基于物理卡，甚至其名称也是对它们背面条纹的引用)。

如今市场上还有其他开放银行连接的提供商——美国以外的格子是一个著名的名字——但 Simoneschi 认为 Stripe 和 TrueLayer 与公司是同一页上的。

“我们有一个深刻的共同信念，即进步来自开发者的眼睛，所以这是关于交付他们需要使用的工具，”他说。“我们处在一个非常互补的空间。”
