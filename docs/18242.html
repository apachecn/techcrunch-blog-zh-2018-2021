后疫情时代的转变意味着 Patch 将在英国的小城镇和郊区开展合作

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/21/post-pandemic-shifts-means-patch-will-take-co-working-to-uk-small-towns-and-suburbs/

公平地说，疫情对职场产生了巨大的影响，但它发生在其他因素已经在起作用的时候。电子商务加速了主街购物的衰落。向远程工作的转变飞速发展。人们不再想在早上 8 点到下午 6 点之间通勤。但是我们也发现在家工作并不像人们所说的那样好。此外，他们不明白通勤到大城市的意义，只是不得不在像 WeWork 这样的地方合作，而他们本可以很容易地去当地的一些地方。问题是，很少有本地的共同工作空间，尤其是在郊区或小城镇。

相反，如果你能把工作带到离家更近的地方(而不是在家里工作),那么，理论上来说，你会得到一种更平衡的生活方式，但也会得到许多人，尤其是家庭仍然渴望的工作和家庭的分离。

现在，一家新的英国创业公司提出了“分散工作空间”的想法，并计划在英国推广

Patch 将以其针对传统通勤者的“在家附近工作”主张，将当地空荡荡的高街商店变成“协作文化空间”。据估计，英国有 600 万从事知识工作的通勤者，Patch 将通过这类成员的每月订阅来运行。

它现在已经从一些英国领先的天使投资人那里筹集了 110 万美元的种子资金，包括罗宾·克莱因(LocalGlobe 的联合创始人)、马特·克利福德(Entrepreneur First 的联合创始人)，以及查理·桑赫斯特、西蒙·默多克(第一集)、温迪·贝克尔(Great Portland Estates 的前首席执行官杰克·威尔斯和内德)、卡米拉·多兰(可持续投资者 Eka Ventures 的创始合伙人)、佐伊·杰维尔(美国投资公司红杉的人才总监)和威尔·尼尔(Grabyo 的创始人和早期投资者)。

Patch 表示，其“在家附近工作”的想法是为了适应后 COVID“混合工作”运动，并计划创建公共场所，“重点是创业、技术和文化规划。”

每个地方都将提供一系列私人办公室、合作工作室、“可获得的低成本选择”和免费奖学金名额。

Patch 的第一个站点将于 11 月初在埃塞克斯郡的切姆斯福德开放，这家初创公司表示，计划在 2022 年再开设几个站点。据说已经收到了切斯特、圣奥尔本斯、威科姆、什鲁斯伯里、约维尔、伯里和泰晤士河畔金斯敦的人们的请求。

Patch 的创始人弗雷迪·福德(Freddie Fforde)说:“我们工作的地方和我们生活的地方传统上被视为截然不同的环境。这导致了工作日期间许多商业街的空心化，以及同样多余的办公区。我们认为技术从根本上改变了这一点，让人们可以在家附近工作，并创造一个专业、公民和文化交流的新混合环境。”

Fforde 曾是 Entrepreneur First 的创始人和员工，在伦敦和旧金山的早期科技公司担任过各种职务。产品负责人将是帕洛玛·斯特雷利兹，前 Assemble 联合创始人，Assemble 是一家赢得 2015 年特纳奖的设计工作室。

“企业家第一”和“代码第一女孩”的马特·克利福德(Matt Clifford)评论说:“技术一直在改变我们组织和合作的方式。Patch 将根据人才的身份为他们打开机会之门，不受他们居住地的限制。我们希望成为一个高技能工作随处可得的国家，而补丁是这个难题的关键部分。”

针对城镇和较小的城市，在居民区，而不是主要的城市中心，Patch 说，它将寻找城镇中心未被充分利用的标志性建筑。例如，在切姆斯福德，他们的第一个空间将是一个维多利亚时代的啤酒厂。

灰色庭院

切姆斯福德议员西蒙·戈德曼是负责经济发展和小企业的副内阁成员，也是申办委员会的代表，他说:“在格雷的院子里引入一个新的共同工作空间对这座城市来说是一个非常积极的计划。为居民提供本地工作选择将有助于他们减少通勤，这将有望实现更好的工作/生活平衡。在离家更近的地方工作对个人和家庭都有好处，对环境和当地经济也有好处。”

Patch 表示，它还将运营一种“回馈”模式，将 20%的高峰活动时段捐赠给地方和国家社区服务提供商，“以支持公共利益”。早期的国家合作伙伴包括技术技能提供商 Code First Girls 和 Coder Dojo，这是 Raspberry Pi 基金会的一项举措。
