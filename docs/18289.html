Nth Cycle 希望为回收商和矿山运营商革新金属加工技术 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/22/nth-cycle-wants-to-revolutionize-metals-processing-for-recyclers-and-mine-operators/

废物有很多价值；问问金属加工公司 Nth Cycle 就知道了，该公司开发了一种技术，可以帮助采矿和回收公司从他们的运营中回收每一点重要的矿物。

该公司将其技术称为“电提取”，Nth Cycle 的创始人兼首席执行官梅根·奥康纳在最近接受 TechCrunch 采访时将其比作滤水器。“我们找到了一种通电的方法，”她解释道。“我们推动电流通过过滤器，我们推动的电流就是我们如何选择性地去除感兴趣的金属。”

目前，所有迹象表明，金属回收和再循环将是经济电气化的关键，特别是因为如此多的零排放技术依赖于关键矿物:锂、钴、镍和锰，这些矿物在自然环境中以不同但最终有限的数量存在。

为了解决这个问题，Nth Cycle 开发了一个电提取系统，占地面积不到 1000 平方英尺，每天可以处理 5 吨材料，这是与大型传统回收公司的一个关键区别，传统回收公司使用的是需要更多资金和更大足迹的湿法冶金和火法冶金技术。该系统可以接受“黑色物质”，即电池回收过程中产生的粉末状废物，并将废物转化为镍、钴和锰等有价值的矿物质。

在采矿方面，Nth Cycle 可以在采矿过程的前端或后端工作，或者帮助公司从低品位(即金属含量相对较低)的地点提取价值。它可能有助于矿山经营者潜在地打开大量未开发的矿物。

“我们正在寻找北美大量未开发的资源，因为我们拥有能源转换所需的所有材料，只是我们还没有可持续开采它们的方法。”

Nth Cycle 的模块化装置还可以处理尾矿或采矿作业产生的废物，并可以收集原始采矿过程中未提取的高达 30%的金属。

一旦商业化，Nth Cycle 计划拥有和运营这些设备，其客户拥有任何回收的产品。

奥康纳说，她将电池回收公司 Redwood Materials 和 Li-Cycle 视为合作伙伴，而不是竞争对手，因为 Nth Cycle 的业务不是收集电池，也不是处理电池。

“我们真的只是在化学回收方面进行创新，”她说。“我们真的在努力创造一种更有效的方法，将这些金属从黑色物质中分离出来。我们不想像其他人一样去收集电池。我们不在物流方面。”

该公司的方法在资本支出方面也要轻得多。水力和火法冶金设施的建造成本可能高达数百万美元，这对于许多废物收集者来说根本不可行。奥康纳说，废物收集者和采矿公司将废物运送给回收者也不总是可行的，特别是如果是低品位的材料。

“对于绝大多数分散的废物来说，把它们运到那些集中的(回收)设施是不可行的，”她说。

Nth Cycle 是在奥康纳在杜克大学的环境工程论文答辩后一天成立的。“实际上，我们坐在波士顿的一家酒吧里，喝着啤酒，第二天我就在网上注册成为了一家有限责任公司，”她说。"从那以后，我们一直致力于商业化."

获得博士学位后，她成为橡树岭国家实验室创新十字路口的企业家研究员，这是一个为期两年的项目，面向致力于新技术的能源和先进制造业企业家。在那里，她获得了 50 万美元的资金，用于扩大技术规模，并准备向私人投资者演示。

展望未来，该公司将宣布几项全商业规模设备的交易，并计划在未来几个月内完成其 A 轮交易。到目前为止，该公司已经从气候技术风险投资公司清洁能源风险投资公司(Clean Energy Ventures)筹集了 125 万美元的非稀释性赠款和 320 万美元的种子资金。
