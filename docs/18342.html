2021 年的自举还有很长的路要走

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/24/bootstrapping-in-2021-goes-a-long-way/

自 2020 年下半年以来，科技创业市场一直享受着风险投资 资本融资的热潮，令人瞠目结舌。随着越来越多的公司加入投资初创公司的竞争，世界各地支付了创纪录的金额，后期资本像水一样流动。

不过，尽管风险投资游戏在最近几个季度似乎有所升温，但噪音却来自另一个极端:自举。

该交易所探索初创公司、市场和资金。

每天早上在 Extra Crunch 阅读或者每周六获取交易所简讯。T15】

就在几周前，Mailchimp 宣布以 120 亿美元退出 Intuit，这帮助建立了一个自举的高水位标志，证明了在没有巨额风险投资的情况下，在传统创业中心以外的地方建立大型、有价值的技术公司是可能的。据交易所最近报道，总部位于亚特兰大的 Mailchimp 只是该市众多创业公司中的一家。

今天的初创公司比以往有更多的融资选择，提供了许多扩大规模的途径。这些包括像 Pipe 这样的服务，它允许创业公司在竞争激烈的市场中出售未来的收入，以及 TechCrunch 广泛报道的无数风险债务参与者。

对于忙于用收入为增长融资的初创公司来说，还有更多好消息。今年早些时候，另一家总部位于亚特兰大的公司 Calendly，筹集了第一笔重大外部资本——估值刚刚超过 30 亿美元，表明早期启动并不意味着股权资本来源永远关闭。最近几个月，许多其他公司也纷纷效仿。

我们很好奇，不断变化的资本格局，以及由于产品导向型增长等商业概念而变得更加可行的创业市场，是否会改变创业创始人的人口结构，或许会使他们的队伍更加多样化。大家说说吧。

今日自举

回到网络繁荣时期，初创公司在物理硬件上花费早期资本是正常的。服务器不是你可以打电话到亚马逊按分钟租用的东西，这意味着你需要自己购买并支付托管费。这需要资金。

此外，正如已经令人作呕地讨论过的那样，几十年前建立数字公司要困难得多。公司没有无限的 API 来解决复杂的技术问题，而且接受过构建新工具培训的人和潜在客户都更少了。

在今天的创业市场，许多技术风险已经解决，通常同时降低了创业公司的建设成本。如今，创办一家公司比以往任何时候都更容易，这一老生常谈的说法可能会与或许成本也更低的概念相吻合。这对自举来说应该是个好兆头。
