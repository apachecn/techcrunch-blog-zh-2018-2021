Highnote 带着 5400 万美元秘密现身，并计划在发卡即服务领域挑战 Marqeta

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/28/highnote-emerges-from-stealth-with-54m-and-a-plan-to-take-on-marqeta-in-the-world-of-card-issuing-as-a-service/

金融科技初创公司通过为企业和消费者打造更灵活、更便宜和更友好的工具，向金融服务领域扔出了一个曲线球，这些企业和消费者反过来会离开旧的现任者，将他们的客户带到新的提供商。在最新的发展中，一家名为 Highnote 的初创公司正雄心勃勃地在发卡领域掀起波澜，让任何规模的公司都可以轻松地向客户提供虚拟支付卡。该公司由 PayPal 校友创建，今天退出秘密模式，并宣布获得 5400 万美元的资金，以迈出第一步。

这笔资金包括种子轮和 A 轮融资，Oak HC/FT 既牵头 A 轮融资，也与 Costanoa Ventures 共同牵头种子融资。XYZ、SVB 资本和 WestCap 也参加了，还有一些知名人士，包括比尔·雷迪(PayPal 的前首席运营官，现任谷歌商务和支付总裁)和雷诺·拉普朗什(Upgrade 的首席执行官)。

零售商和其他公司的自有品牌支付卡在这一点上远不是一个新颖的想法:它们和它们的忠诚度计划已经存在了几十年。甚至还有其他非常成功的创业公司，像 Marqeta ，它们都是建立在将机会民主化的前提下。

Highnote 的不同之处在于，该公司认为，目前的许多解决方案仍然需要太长的时间来提供，最终不如产品那样有创意:市场上有一个新的机会来虚拟化流程，就像苹果卡一样，使其成为任何规模的公司都可以引入的东西(不仅仅是苹果)，并为当今向消费者提供信贷的意义带来更大的灵活性。

PayPal 子公司 Braintree 的前总经理约翰·麦克韦恩(John Macllwaine)说:“今天有一个不可思议的机会来设计一个专门构建的平台，以满足嵌入式金融的需求，并让企业更快地移动，”他与 Braintree 的前建筑总监 Kin Kee 共同创立了 Highnote。“Marqeta 在阐明机会方面做得很好，但这些平台是在 10 年前构建的，是为特定的使用情形而设计的。他们在他们在的地方，”他耸耸肩总结道。_(ツ)_/

当前流行的嵌入式金融——各种金融服务，它们本身构建复杂，但易于通过 API 从第三方集成——在这里非常重要。传统的卡服务是基于非常有限的一套想法。

如果你是零售商，该卡旨在鼓励消费者更多地与你(而不是其他任何人)一起购物；非零售商(比如航空公司)将它们与可以用品牌兑换的忠诚度积分联系在一起，或者与参与品牌忠诚度计划的企业联系在一起。在每种情况下，品牌往往是大的，卡的基础是围绕支付限额，未付余额的利息等。要得到一张，需要在收银台填写申请表，整个过程需要几个星期才能完成(此时你会收到一张邮寄的实体卡片)。

人们对此的高度评价是，依靠嵌入式金融的概念，扩大可嵌入“支付卡”的东西的范围，并使任何类型的企业都更容易获得这种卡。

最初，该公司向其客户提供各种“卡”——虚拟的、物理的和令牌化的——并向他们提供方法，让他们的客户更智能地管理这些卡和他们的财务生活，包括让他们开设银行账户，以及管理和监控他们的卡如何被使用。它也给了公司更多的控制权来监控他们网络的使用和花费。

随着时间的推移，您可以想象在销售点推出许多不同的服务，这些服务也利用了信贷机会，包括潜在的立即购买、稍后支付选项或其他灵活支付的方式。是的，已经有 BNPL 公司拓展成卡了；是的，专注于支付的公司也在开发信贷产品。但 Highnote 提供的是不同的东西。可以说，品牌已经逐渐脱离了在线交易——复杂的过程基本上外包给了其他人，要么通过亚马逊这样的市场销售，要么嵌入由另一家公司构建和支持的复杂过程。Highnote 给这些品牌的承诺是再次拥有它和与之相关的数据。

这是你会从支付专家作为创始人那里期待的那种市场机会——讽刺的是，他们可能亲眼目睹了事情是如何演变的——这也是投资者热衷于在早期支持他们的原因。

Oak HC/FT 合伙人马特·斯特雷斯菲尔德(Matt Streisfeld)在一份声明中表示:“我们支持了金融科技领域的许多领先公司，但 John，Kin 和他们的团队真的很特别。”“我们了解企业目前在嵌入支付功能以提高客户参与度、忠诚度和收入方面面临的诸多挑战，也了解需要立即获得工资和资金的消费者和中小型企业。我们看到一个巨大的机会出现在任何能够破解密码的人面前。我们正在以 Highnote 建设，因此，我们很高兴为他们提供所需的支持，以继续他们令人难以置信的速度。”
