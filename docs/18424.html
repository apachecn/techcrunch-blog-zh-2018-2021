Swan 筹集了 1870 万美元来提供嵌入式银行服务

原文：https://web.archive.org/web/https://techcrunch.com/2021/09/28/swan-raises-18-7-million-to-provide-embedded-banking-services/

法国初创公司 Swan 获得了由 Accel 领投的 1870 万美元 A 轮融资(€1600 万)。该公司为其他产品和服务提供银行即服务平台。

嵌入式金融是过去几年金融科技领域的一个重要趋势。在这一领域经营的公司向其他公司提供金融服务。这些客户充当金融服务的分销渠道，而嵌入式金融公司则充当所有金融业务的骨干。

这样，一些不想经营金融服务的公司仍然可以提供金融服务。其他公司可以专注于基于 API 的产品，为他们的客户提供一些新功能。

Swan 与 SaaS 公司、市场甚至其他金融科技初创公司合作，因此他们可以提供白标银行功能。例如，您可以利用 Swan 的 API 通过 API 调用来生成账户和卡。

现有投资者 Creandum 和 Bpifrance 也参与了今天的融资轮。Swan 的一些客户包括 Agicap、Alma、Expensya、Fintecture、Mon Bel Appart、Pennylane 和 Ride。

在创建帐户时，每个帐户都有自己的 IBAN。最终客户可以使用 Swan 设计的网络银行界面访问这些账户。Swan 客户可以选择在 Swan 数据的基础上设计自己的界面。

说到卡，Swan 允许你生成虚拟卡和实体卡。他们与 Apple Pay 和 Google Pay 合作。Swan 客户可以限制一些功能，比如 ATM 取款，也可以让客户直接控制自己的卡。

这家初创公司拥有电子货币许可证，可以在整个欧洲经济区提供服务。它还处理法规要求，如身份验证和欺诈检测。单独解决这些棘手问题是一个重要的卖点，因为 Swan 客户不必自己处理法规要求。

随着今天的融资，该公司计划在 2022 年初扩展到德国，建立一个新的办公室。Swan 将与柏林的银行即服务公司 Solarisbank 竞争。但 Swan 还计划让开发者开始以自助服务模式直接使用该公司的 API。

“我们在这一轮融资中的目标是提供具有当地特色的银行功能，专门面向众多欧盟当地市场:荷兰的 Ideal、德国以 de 开头的 Ibans、法国的资本存款账户，等等！”联合创始人兼首席执行官尼古拉斯·贝纳迪在一份声明中说。

你可能已经注意到，金融科技已经成为科技创业公司的一个繁忙的垂直行业。凭借其 B2B2C 战略，天鹅不必创建一个金融超级 app，就可以成为拥有数百万用户的主要金融接触点。很多人可能已经在使用 Swan 服务，而没有意识到这一点。
