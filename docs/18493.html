Alloy 以 13.5 亿美元的估值融资 1 亿美元，帮助银行和金融科技公司利用其基于 API 的平台 TechCrunch 打击欺诈

为银行和金融科技公司构建身份操作系统的 Alloy 周四宣布，它以 13.5 亿美元的估值筹集了 1 亿美元。

光速创投(Lightspeed Venture Partners)领投了 C 轮融资，一年前，总部位于纽约的 Alloy 在 B 轮融资中筹集了 4000 万美元。现有的支持者 Canapi Ventures、Bessemer Venture Partners、Avid Ventures 和 Felicis Ventures 都在最新一轮投资中投入了资金，自 2015 年成立以来，Alloy 的总融资额已超过 1.5 亿美元。

Alloy 的成立主要是为了修复一个“破碎的”入职流程，该流程在历史上一直涉及人们在线申请银行账户时的人工审查。简而言之，这家初创公司的最初使命是利用其单一 API 服务和 SaaS 平台，帮助银行和金融科技公司做出更好的身份和风险决策。

在过去的一年里，Alloy 已经对该平台进行了改进，不仅实现了 入职身份决策的自动化，还实现了交易监控的自动化，不久还将实现信用核保的自动化。Alloy 首席财务官基兰·赫布巴尔表示，在过去的 12 个月里，该公司的年度经常性收入增长了两倍，客户数量增加了一倍。今天，该公司拥有 200 多家客户，包括 Ally Bank、HMBradley、Brex、Marqeta、Gemini、Ramp 和 Evolve Bank&Trust——比去年同期的 90 家有所增加。

Alloy 将其用户与来自 120 家身份提供商的数据联系起来，然后使用这些数据来帮助金融机构在初始客户入职和进行持续交易时避免欺诈。它旨在为银行和金融科技公司回答的问题包括:“这是一个真实的人吗？他们会不会诈骗我们？”

它为他们提供了一种创建即时决策系统的方法，从法规遵从性和风险的角度根据他们的需求进行定制。

首席执行官兼联合创始人汤米·尼古拉斯说:“对于金融科技公司和银行来说，部署对他们来说既安全的产品真的很难，这意味着他们不会承担一系列欺诈或合规损失，而且对他们的用户来说也是无缝的。”

从历史上看，金融机构降低风险的努力并没有带来理想的用户体验。

Nicholas 告诉 TechCrunch:“我们的整个任务就是让这种情况消失，让风险成为一种你可以安装并放在后台的东西，以解决那些可能从你这里拿钱的用户的所有问题。”。“我们非常擅长帮助他们自动化和优化这些流程。我们基本上是要求人们将他们运行的最重要的流程放到一个中央系统中。”

尼古拉斯表示，随着越来越多的公司成为金融科技公司，或者更确切地说，将支付等东西嵌入他们的平台(电子商务公司是一个主要例子)，Alloy 已经看到越来越多的公司可以利用它所做的事情。

此外，如上所述，该公司现在已经扩展到交易监控领域，计划在今年晚些时候推出一款专注于信用承销的产品。

图片来源:合金

这家初创公司计划将其新资本主要用于建立“不断发展”的客户身份 档案，以防止欺诈并最大限度地降低风险。沿着这条线，Alloy 计划纳入“更丰富的数据和风险信号”，其使命是让银行和金融科技公司对其客户有一个 360 度的视角。简单地说，它也想改善开发者的体验。

尼古拉斯说:“我们希望打造一款金融科技产品就像打造一款电子商务产品一样简单。身份及其关联风险不是商家应该搞清楚的东西，它 应该只是他们安装的东西。随着 Alloy 的发展……我们不仅可以让风险更容易理解，还可以通过让金融科技产品更容易制造来推动行业创新。”

光速创投(Lightspeed Venture Partners)合伙人贾斯汀·奥弗多夫(Justin Overdorff)在 2019 年的 A 轮融资中首次作为天使投资了 Alloy。

“当那一轮发生时，我在 Stripe 对这个问题有着敏锐的第一手知识和经验，经历了多年拼凑数据来源的过程，”他回忆道。

他说，这个过程“非常非常具有挑战性”

“你想对尽可能多的最佳客户说好，”奥多夫说，“但你不想误报或拒绝好客户，因为你拒绝的每一个好客户都是损失的收入。”

那次经历让他对 Alloy 正在制造的东西产生了兴趣。

他告诉 TechCrunch:“对我来说，令人兴奋的是收入在快速增长，公司也在快速增长。”。“我认为总体情况是，机会非常大。大大小小的金融服务公司数量激增，甚至在某种程度上，一些软件公司也在其产品中嵌入了金融服务。这些公司中的每一家都需要接纳客户，并首先通过 KYC(了解您的客户)、反洗钱(反洗钱)的接纳流程。”
