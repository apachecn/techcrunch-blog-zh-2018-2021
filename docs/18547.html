来自奥地利维也纳的一家默默无闻的自举创业公司——凭借为一些世界上最受欢迎的应用程序提供用户体验的技术获得了开发者的青睐——正在加倍努力，并宣布了第一笔外部投资，形式是一轮大规模增长融资。

PSP dfkit——它提供 API 和一个 SDK，开发者用它们来增强文档处理功能，如电子签名、文档查看和编辑、协作和等——已经为€筹集了 1 亿英镑(1.16 亿美元)。资金来自单一投资者 Insight Partners。

PSPDFkit 已经盈利了，而且已经有一段时间了，所以这次投资是为了加快它的增长速度。它计划利用这笔投资来开发更多的开发工具，进行战略性收购(联合创始人兼首席执行官乔纳森·雷恩对具体内容三缄其口，只是说将扩展它提供的有用工具套件)；并且，第一次在销售和市场营销领域做出一些协同努力。

事实上，到目前为止，PSPDFkit 的许多增长都是靠口口相传，这是一种策略，到目前为止它已经走得很远了。它的客户包括 Dropbox、DocuSign、SAP、IBM、大众汽车、Fabasoft、Wolters Kluwer Deutschland 和欧洲专利局，以及在 NDA 领导下与之合作的许多其他公司。

在许多情况下，并不是每个公司都乐意承认他们的用户体验和技术有多少是由第三方构建的，PSPDFkit 在哪里以及如何使用也是如此，但事实仍然是它非常庞大:总之，PSPDFkit 的技术，通过其 API 和 SDK，现在在 150 个国家接近 10 亿用户。

不出所料，这种吸引力也意味着 PSPDFkit 多年来一直有很多收购兴趣。大型技术公司正在开发生产力工具，已经在与开发者合作，并且已经在移动应用和云服务方面非常活跃，这些公司都已经敲开了 PSPDFkit 的大门。尽管这家初创公司没有披露其估值，但你可以猜测，鉴于其规模和盈利状况，在最近一轮融资中，它肯定会成为一笔更昂贵的收购。(如果一切按计划进行，情况会变得更糟。)

PSPDFkit 的故事是一个有趣的故事，它反映了多年来移动开发本身是如何成长的。

最初，该公司在 2011 年左右作为一个框架和工具集由奥地利工程师 Peter Steinberger 创建，他已经参与了 iOS 开发者社区，并看到了市场上许多应用程序对电子签名、文档编辑和文档查看等文档处理功能的需求。应用程序让我们将许多东西变成虚拟体验，而纸张正在成为最先消失的东西之一。

这种需求被证明是一个经典的用例，用于构建该功能，并使其成为许多其他人可以通过 SDK 和 API 访问的东西:文档操作工具——即使是预览基于云的文件夹中包含的文件这样的基本工具——也很难从头构建，不一定是公司核心业务的一部分，但仍然是它们如何工作的核心。

Steinberger 的框架因此成为了如何使用 SDK 和 API 将不同的服务和功能集成到其他应用程序中的早期例子之一——一个基本原则现在已经应用到其他一系列功能中:嵌入式金融服务，例如，那些用于构建新银行、新经纪人、保险技术的服务；嵌入式支付，如 Stripe 和其他公司的支付 APIs 嵌入式通信，例如 Twilio、Sinch 等公司的消息、语音或电子邮件 API 诸如此类。

(PSPDFkit 这个名字是指彼得·斯坦伯格的首字母缩写；因为 PDF 曾经是，并且在很大程度上仍然是公司最初的关注点；和“kit”来指代 SDK。)

随着移动应用程序的创建和使用开始真正起飞，Steinberger 的框架也开始起飞，不久 Martin Schürrer 也加入了他的行列。与此同时，Rhyne 在美国担任开发人员的代理律师，在一次开发人员活动中遇到他们后，他成为了他们的律师。

在这种关系中，三人很快意识到，不仅有一个适当的、不断增长的业务需要管理，而且斯坦伯格和施瑞对此也没什么兴趣。到 2014 年，Rhyne 放弃了律师行业，成为第三位联合创始人，与 Steinberger 和 Schürrer 一起在维也纳工作，Rhyne 的工作地点在美国北卡罗来纳州。

在这一轮中，斯坦伯格和舒勒将不再担任全职职位，但仍“大力投资”业务，而雷恩将继续担任首席执行官。

PSPDFkit 的发展很快，与应用程序本身的迅速崛起保持一致。

从 iOS 开始，今天 PSPDFkit 提供了可以用于在 Android 和 web 上构建应用程序的工具，使用 Flutter 和 React Native。策略是致力于构建一个更大的平台来处理多个相关的功能，开发人员可能希望使用与文档相关的功能，或者更广泛的生产力领域。

Rhyne 对 PSPDFkit 及其客户今天所做的事情的基本描述是“淘汰纸张”但随着时间的推移，就像 Stripe 如何从其核心功能(为支付提供 API)发展到与交易相关的更广泛的服务套件一样，PSPDFkit 看到了做更多事情的机会，尤其是因为世界的期望也发生了变化。

例如，Rhyne 回忆起 PSPDFkit 如何在 2015 年推出了一个实时协作平台，“但我们就像，‘伙计，它没有获得牵引力，人们并不真的想使用它。’然后新冠肺炎发生了，他继续说:“现在每个顾客都说，‘这看起来很棒。“是的，我们想利用这一点……”我认为我们看到了人们与文档互动方式的变化。"

这对创业公司和投资者来说都意味着大量的机会。

Insight Partners 的医学博士 Ryan Hinkle 在一份声明中说:“软件开发人员和工程师仅仅因为他们的工作性质就站在工作的最前沿。“他们工作和协作的方式也应该是最前沿的。凭借 PSPDFKit 的软件开发工具包和托管解决方案，该公司正在为企业和开发人员带来革命性的文档处理，他们的任务是让公司保持在创新的前沿。Insight 很高兴能在公司的成长过程中发挥作用。”辛克尔在这一轮加入了董事会。

然而，尽管如此，对 PSPDFkit 本身的期望变化不大:该公司是作为一个“分布式”运作，因为它一直是早在新冠肺炎疫情，主要 R&D 基地在维也纳，但雇员在世界各地。现在，该公司正以同样的模式招聘更多员工。
