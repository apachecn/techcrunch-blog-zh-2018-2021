第一个胜利:让早期客户和你一起冒险

新公司面临着无数的挑战，包括调整产品和构建合适的收入模式。你可能会期待筹集一些资金，并试图处理经营企业的行政方面，但在这之前，你的公司的目的是销售你的解决方案并创造收入。

但要做到这一点需要客户，那么你如何让别人和你一起冒险呢？

为了回答这个问题，我们上周在 TechCrunch Disrupt 2021 上召集了一个由三位技术行业资深人士组成的小组。该小组包括凯特泰勒，在概念的客户体验负责人；Belvo 联合首席执行官兼联合创始人 Pablo Viguera 以及 Egnyte 首席执行官兼联合创始人 Vineet Jain。

除了他们在帮助建立成功的创业公司方面的经验，这些小组成员的公司都采取了非常不同的创收方法。内容存储和管理平台 Egnyte 更像是一种直接的企业销售，而协作和内容组织工具 conception 则使用免费增值模式，然后追加销售带有附加功能的付费订阅。就其本身而言，Belvo 是一个提供给开发者组织的金融服务 API，当该 API 被添加到产品中并被人们使用时，它就会被货币化。

维格拉说，你不应该一开始就尝试用这种产品做太多事情。把东西拿出来，让顾客告诉你他们的想法。“在早期，你正在发布、正在构建的 v1 很难与两年后或十年后的版本相提并论。因此，集中精力获得大量反馈，不仅是在早期，而是随着时间的推移。他说:“真正建立反馈回路，认真寻求反馈，即使这是一件复杂的事情。

与此同时，泰勒说，你不能让顾客的反馈在太多的方向上推你和拉你。你必须找到一批忠诚的用户，让他们帮助你引导产品。“[你必须]管理干扰，因为你可以用很多不同的方式来使用产品，你会听到很多不同的反馈。所以，要找到那些积极参与的声音，以及你要带进这个圈子的人，他们会帮助你了解在哪里成长，”她说。
