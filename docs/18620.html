Sure 以 5.5 亿美元的估值筹集了 1 亿美元，帮助公司利用其“灵活的”API 推出保险产品

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/05/insurance-infra-startup-sure-raises-100m-series-c-at-a-550m-valuation/amp/

图片来源:确定

保险基础设施初创公司 Sure 今天宣布，以 5.5 亿美元的估值完成了 1 亿美元的 C 轮融资。

总部位于纽约的 Declaration Partners 和欧洲成长型投资者 Kinnevik 共同主导了此次融资，参与融资的还有 WndrCo 和现有支持者 W. R. Berkley 和 Menlo Ventures。自 2015 年成立以来，这轮融资使这家总部位于圣莫尼卡的初创公司的总融资额达到 1.231 亿美元。

Sure 在 2016 年初推出了其首款(企业 SaaS)产品，如今，传统金融服务和金融科技公司都是它的客户。客户包括 Farmers Insurance、Chubb、Intuit、Betterment、Revolut、Carvana、 几家汽车制造商和一家领先的全球信用卡网络 。所有这些都提供建立在 Sure 基础设施上的保险项目。

虽然 Sure 不会披露硬收入数据，但它声称自 2019 年以来一直盈利。此外，首席执行官兼联合创始人韦恩·斯拉文表示，其年度经常性收入(ARR)的增长“在过去几年中每年都超过 3 倍。”

简而言之，Sure 表示，它已经创建了技术基础设施，使整个保险流程现代化，并“允许它嵌入消费者体验中”换句话说，使用 Sure 的基础设施，公司可以“在几毫秒内”直接向消费者出售保险，该公司声称，通过完全数字化的体验，不涉及纸张或人类。

据 Slavin 称，该公司处于嵌入式保险的“前沿”。

他说，嵌入式保险是“整合到该品牌在所有行业的现有数字产品中的保险”。

Slavin 补充道:“从在线购买新的电动汽车或在 Carvana 上购买二手车所包含的汽车保险，到当你开始你公司的新银行账户时购买商业保险。

Slavin 说，使用其 API 的好处是，与其他方法相比，公司可以更快、更简单、更便宜地将新的保险产品推向市场。

“对我们的客户来说，这三大支柱确实与众不同:久经考验的技术基础设施、最快的上市速度以及提供完全嵌入式客户体验的能力，”Slavin 告诉 TechCrunch。“就像任何正在被颠覆的传统行业一样，许多玩家试图通过用一个花哨的新名字来重塑他们的 1.0 体验，从而驾驭‘嵌入式’的浪潮。真正的嵌入式保险体验是独一无二的，由 Sure 首创，因为它们是从头开始构建的，旨在实现嵌入式渠道中的端到端交易。”

图片来源: Sure /首席执行官兼联合创始人韦恩·斯拉文

该公司计划利用新一轮增长带来的资本加速其全球扩张，加快新产品的推出，并继续“简化嵌入式保险客户体验”

据 Slavin 称，国际扩张将以新客户和办事处的形式出现在欧洲、拉丁美洲和亚洲，以及拥有成功国内项目的现有客户，这些客户计划将其保险解决方案扩展到国际客户。

确定还打算继续招聘。Sure 目前的员工人数为 80 人，比去年增加了 50%以上。该公司计划在 2022 年将其工程人员增加两倍。

Declaration Partners 的合伙人布莱恩·斯特恩(Brian Stern)表示，他的公司之所以被 Sure 吸引，是因为它认为斯拉文是“一位有远见的领导者”，以及“ 这项变革性技术的巨大全球市场。”

他说:“Sure 专注于消费者体验、便利性，以及将这种能力嵌入消费者与品牌关系的力量。”。

Kinnevik 的 Ola Nordbye 说，他公司的几个投资组合公司已经在使用 Sure。

他在电子邮件中写道:“我们喜欢它如何实现更好的客户体验，同时为已经获得客户基础的品牌打开了一个巨大的未开发利润池。”。“谈到这一点，最难把握的金融科技领域之一是保险业。Sure 最先进的技术平台给我们留下了深刻的印象，它的先进程度远远超出了现阶段一家公司的预期，创始人对自己想要创造的东西有着清晰的愿景，结合深厚的行业知识，以及增长和盈利能力等重要的财务 KPI。”

Nordbye 补充说，Sure 的独特之处在于它如何在等式的两端(保险公司和品牌)实现新的收入流，而且他们是作为一种软件游戏而不是作为风险载体来实现这一点的。

“事实上，他们能够以令人难以置信的实施速度提供整体产品，这也是独一无二的，”他补充道。
