Streamlit 达到开源数据应用构建工具 TechCrunch 的 1.0 里程碑

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/05/streamlit-reaches-1-0-milestone-for-open-source-data-app-building-tool/amp/

图片来源: zf L / Getty Images

在过去的几年里，由机器学习老手创立的初创公司 Streamlit 一直致力于一个开源项目，以帮助数据科学家构建 web 应用程序来探索机器学习模型中的数据。今天，该公司宣布该项目已经达到 1.0 版本。

“两年前，我们推出了(Streamlit 的初始版本)。所以我们在上面贴上邮票，说这已经准备好了。首席执行官兼联合创始人 Adrien Treuille 告诉 TechCrunch 说:“我们非常非常自豪，事实上，随着我们的影响力不断增长，我们正在为社区建设一个伟大的路线图。

他说，当他们开始时，他们根据自己的经验意识到需要一种工具，让数据科学家获得他们正在处理的数据，并与组织的其他人分享，随着时间的推移，成千上万的人下载了该产品，他们发现这个想法是有效的。

“开源产品是免费的，所有的源代码都可以看到。它得到了一个庞大的社区的支持，这个社区的人们正在寻找漏洞，提出建议，甚至为代码做出贡献，”他说。

该公司的联合创始人兼首席运营官阿曼达·凯利(Amanda Kelly)表示，他们已经从早期的少数客户用户群发展成为一个庞大的用户社区，部分原因是 Python 的流行。

“我认为你知道核心的事情是，如果你理解 Python，现在有数千万人理解，即使只是一点点 Python，你真的可以开始使用 Streamlit，所以任何人都希望从某种类型的数据、某种类型的 API 开始，它们都在一起，你可以使用 Streamlit 来真正快速地得到答案，”她说。

该公司报告称，超过一半的财富 50 强企业正在使用开源产品，来自脸书的数据科学家 Tyler Richards写了一本关于如何使用 Streamlit 的书。

该公司在 4 月份获得了 3500 万美元的 B 轮融资，除了开源项目之外，该公司还一直在致力于商业云服务，预计将在今年年底前完成。该商业产品目前在 700 多家公司处于私人测试阶段，这意味着当它最终可用时，他们应该立即投入运行。

根据 Crunchbase 的数据，该公司于 2018 年推出，已融资 6200 万美元。
