技术面试平台 Byteboard 从谷歌的 120 区分离出来，接受新的资助 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/05/technical-interview-platform-byteboard-spins-out-of-googles-area-120-takes-on-new-funding/amp/

图片来源: Byteboard

科技博客 TechCrunch 获悉，谷歌证实，一项旨在用基于网络的替代方式取代公司招聘过程中现场技术面试的服务将从谷歌剥离出来。该产品最初是作为公司内部研发实验室“120 区”的一部分被孵化出来的，由首席执行官萨贡·考尔领导。通过此举，Byteboard 将成为第一个退出谷歌的 Area 120 项目，成为自己的独立公司。但谷歌指出，这将是一个例外，而不是规则。

谷歌告诉我们，分拆不会对任何现有的 Area 120 团队或该集团的未来战略产生影响。相反，该公司表示，其 R&D 部门将继续专注于资助最有可能促进谷歌自身机遇的项目。

除了分拆之外，Byteboard 还从 Cowboy Ventures 和其他公司获得了 1000 万至 1500 万美元的新投资。但我们知道这不是“收购”，因为谷歌将保留新公司的股权，新公司将继续由联合创始人考尔和尼克·哈德森-赫尔利领导。

“我们很高兴看到 Byteboard 在过去三年中在谷歌的 120 区内取得的进步，”谷歌发言人告诉 TechCrunch。“Byteboard 的解决方案为高增长企业提供了评估和聘用顶级技术人才所需的工具，提高了效率和公平性。我们期待着与萨贡、尼克和 Byteboard 团队的其他成员继续合作，”他们说。

Byteboard 于 2019 年推出，其想法是创建一个工具，让技术面试体验不那么乏味，更有效。该团队当时指出，目前面试软件工程师的过程并不能真正衡量一个人在日常工程工作中的表现。相反，面试往往有利于那些有时间和资源准备的人，因为他们更多地测试记忆，而不是实际应用人们的技能。

图片来源: Byteboard

Byteboard 通过向求职者提供真实世界的编码环境来扭转这一局面，他们可以从支持的语言中进行选择，如 Java、Python、Ruby、C++、C#、JavaScript (node.js)、Go 和 PHP。

基于网络的面试以 HTML、CSS 和 JavaScript 进行，而移动面试以 Swift (iOS)和 Kotlin (Android)进行，数据工程面试以 Python 和 Java 进行。

它的端到端服务包括面试平台、涵盖 20 多种基本软件工程技能的校准问题库以及事后评估。一组经验丰富的工程师审查和评价面试。这种评估是匿名进行的，目的是消除过程中的偏见。

该业务在 2019 年首次亮相后起飞。在 COVID 锁定早期的短暂低迷之后，随着公司恢复招聘，Byteboard 在 2020 年第三季度再次回升。据 Byteboard 的网站称，使用其服务的用户包括 Lyft、Hulu、Figma、perfect Foods、PlayVS、Betterment、Robinhood、GoodRx、ETHOS、Ezoic 和 Glowforge。

除了通过网络而不是亲自进行面试的好处(COVID 时代的另一个问题)、数据和评估的质量以及候选人压力较小的环境，Byteboard 还可以为公司节省资金，因为他们不再需要支付工程师来安排面试和衡量结果。

然而，作为谷歌旗下的产品，Byteboard 的潜力可能有限。

由于工程师是 Byteboard 人工评估过程的一部分，这将使谷歌成为其他公司招聘团队的一部分——在某些情况下，包括它直接竞争的公司。这显然引发了道德问题，因为公司通常不会让竞争对手参与到他们的招聘过程中。这在很大程度上是 Byteboard 作为自己的业务向前发展的原因，而不是谷歌拥有的项目。

一位知情人士告诉 TechCrunch，Byteboard 的六名团队成员中，有几个人加入了这家独立公司，有几个人选择留在谷歌，他们将在那里开始新的项目。谷歌没有证实交易这方面的细节，但指出 Byteboard 将继续扩展其核心产品和运营重点，在未来几个月内将分享更多内容。
