淡马锡牵头向 Orca Security 投资 5.5 亿美元，旨在获得进一步的国际立足点

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/05/temasek-leads-550m-series-c-extension-into-orca-security-which-aims-for-further-international-foothold/

Orca Security 是一家以色列安全公司，提供无代理平台来保护基于云的资产，该公司在 7 个月前筹集的 C 轮融资中获得了 5.5 亿美元的延期。

联合创始人兼首席执行官 Avi Shua 告诉 TechCrunch，第一轮2.1 亿美元的融资使公司的估值超过 10 亿美元，最新一轮融资将估值提高了 50%，达到 18 亿美元。

淡马锡领投了这轮融资，战略投资者 SAIC 和 Splunk Ventures 也参与了融资。首轮 C 轮由 CapitalG 领投，包括红点风险投资、GGV、亚由美资本、Lone Pine Capital、Stripes、Adams Street Partners、Willoughby Capital 和 Harmony Partners。

此次延期符合 Orca 安全部门在过去一年中积累的快速调查结果。该公司去年 12 月筹集了 5500 万美元的 B 轮融资，此前在 5 月份进行了 2050 万美元的 A 轮融资。

Shua 说，自今年早些时候的 C 系列以来，该公司一直在忙于建立一个新的平台，该平台将在几分钟内而不是几个月内将安全环境迁移到云。

“这就像是云的核磁共振成像，”他补充道。“一旦连接到云环境，您就可以毫无摩擦地全面了解风险。”

随着公司在过去两年转向数字化，组织被迫在数字化领域提供特性和功能，迫不及待。这导致云计算和安全解决方案的采用增加。Shua 说，对于 Orca Security 来说，这意味着“蓬勃发展”的增长。该公司拥有 200 多名员工，收入增长了 800%。

在完成 C 轮融资后，Shua 收到了更多希望与该公司合作的投资者的兴趣，其中一些名字对他来说非常突出，可以帮助该公司加速发展。

“淡马锡是世界知名的投资者，有 Splunk 和 SAIC 这样的战略合作伙伴，我们可以走得更远，”他补充道。“我们并不急需现金，但我们确实希望为我们正在经历的增长做好定位。”

他打算将这笔新资金投入三个领域:继续提供更多功能的工程设计、扩大其全球影响力和走向市场。

为了支持全球增长和走向市场，Orca Security 周二还宣布聘请 Meghan Marks 担任首席营销官。此前，Marks 是 Palo Alto Network Prisma 云业务部门的 CMO。

Orca Security 目前在超过 15 个国家运营，最近推出了德语、法语、中文和日语版本的网站。它将扩大在英国的业务，在伦敦开设一个办事处和 R&D 中心，还将扩展到 EMEA 和 APAC 地区。它计划在年底前为新的伦敦办事处配备 24 名员工。

Shua 认为云将继续快速发展，他预计云安全将在未来五年成为下一个万亿美元的市场。

“Orca Security 的定位是成为市场的领导者，我们专注于别人没有的技术，”他补充说。“我们生活在一个脆弱的世界，网络通常没有负面影响。如果你失败了，你只是在接下来的几分钟内再次尝试，这使得它更难控制。这就是网络市场增长的原因。我们部署的东西可以用来保护环境。”
