原文：https://web.archive.org/web/https://techcrunch.com/2021/10/06/databricks-acquires-8080-labs-to-extend-its-low-code-no-code-capabilities/

Databricks 收购 8080 实验室以扩展其低代码/无代码能力

Databricks 今天宣布，它已经收购了德国初创公司 8080 Labs，该公司是基于 Python 的熊猫数据分析和处理工具的流行 GUIbamboo lib的幕后公司。Bamboolib 允许数据科学家快速轻松地探索他们的数据并转换数据，而无需编写任何代码。8080 实验室提供了 bamboolib 的免费社区版和带有额外企业功能的付费专业版。Databricks 计划将这些 UI 驱动的功能集成到其 Lakehouse 平台中，并扩展其自身的低代码/无代码能力。

这标志着 Databricks 自 8 月完成 16 亿美元H 轮融资以来的首次收购。当时，Databricks 首席执行官阿里·古迪(Ali Ghodsi)告诉我们，他希望筹集这一轮资金，以便能够在这一领域与资金雄厚的竞争对手抗衡。在竞争激烈的领域中，快速增长的公司往往会遇到这种情况，这通常意味着进行收购，购买工程人才和软件能力。

“加上去年对 Redash 的收购，我们正在将用户群的焦点扩大到更广泛的偏好低代码/无代码解决方案的受众。Ghodsi 在今天的公告中指出:“为数据块带来简单的功能是关键的一步，可以让组织内的更多人轻松分析和探索大型数据集。

对于 Databricks 来说，这次收购也标志着重新重视将其平台带给公民数据科学家，否则他们可能不具备使用其工具的专业知识。该公司在今天的公告中指出:“随着数据和人工智能成为各种规模企业的战略优先事项，组织内的每个人都有权提出问题并根据数据采取行动，这一点至关重要。”。Databricks 认为，结合其自动化机器学习工具，其服务现在将允许更广泛的用户从他们的数据中提取价值，也使专业开发人员的生活更容易，他们可以使用其 UI 工具构建数据转换，然后在需要构建更定制的实现时导出 Python 代码。

8080 实验室的联合创始人 Tobias Krabel 说:“我们成立了 8080 实验室，以简化复杂的数据任务，并使任何技能组合的数据团队都可以利用数据科学和机器学习的力量。“凭借开源根源和重塑 lakehouse 类别的数据前景的不可思议的愿景，我们看到了 Databricks 的无限机遇，并且非常兴奋能够加入这个团队踏上这一旅程。”

这两家公司没有透露收购的细节，8080 实验室在这次出售之前没有筹集任何外部资金 更正 : 8080 实验室已经从四个商业天使那里筹集了天使资金，但从未透露细节。
