在代理权争夺战胜利后，是时候让 Box 采取一些大胆的行动了

作为一家上市公司，Box 经历了一段坎坷的历程。该公司成立于 2005 年，当时是一家简单的消费者文件共享服务公司，但几年后转向专注于企业，并为仍在发展的云和移动世界构建现代内容管理系统。

Box 曾经是硅谷初创公司的宠儿，在 2015 年申请 S-1 上市后，由于人们质疑其烧钱速度，Box 在遭遇重挫。这导致的首次公开募股被推迟了几个月，也许这是它在未来几年中寻找市场牵引力的一个迹象。最近， Box 与它的一个大股东 Starboard Value 进行了一场代理权争夺战，该公司最终赢得了。

对于云内容管理公司来说，现在是一个关键时刻。Box 从 KKR 获得了 5 亿美元的投资，这意味着它有资金收购一些小公司，填充产品路线图，或许还能推动一些无机增长，正如它在 2 月份收购电子签名公司 SignRequest 时所做的那样。

最近几个季度，Box 的收入也有所增长。继 2010 财年第三季度实现 13.6%的增长后，2011 财年第四季度开始出现下滑趋势，增长率降至 8.3%。这时候，右舷决定选择其代理战斗。但 Box 从低谷中走了出来，开始了缓慢的上升轨迹，其管理层预计这一过程将在未来几个季度继续。

这张图表没有详细说明巨大的增长，但它仍然显示了增长。联合创始人兼首席执行官 Aaron Levie 以前也遇到过麻烦，当涉及到他的批评者时，他采取了一种主要是和解的语气，承认还有很多需要学习的地方，他一直致力于改善公司。

我们采访了列维，了解了他所经历的考验和磨难，以及他希望将公司带向何方。

向前穿过一切

对于处于代理权争夺战中的公司领导人来说，过去的一年并不容易，但这位首席执行官表示，他并没有失去对工作的乐观和热情。当你考虑到他 2005 年在南加州大学的一间宿舍里创办了这家公司时，这是令人印象深刻的——他整个成年生活都在从事 Box 方面的工作。

“如果说有一条线索贯穿了我们经历过的任何困境，那实际上就是我对未来的极度兴奋和乐观。我对技术总体上和它对世界的影响以及它能为我们做什么持乐观态度。我对 Box 尤其如此，尤其是围绕我们正在构建的东西的机会，”Levie 告诉 TechCrunch。
