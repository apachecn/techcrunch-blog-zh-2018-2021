在 Greycroft 的支持下，Klasha 获得 240 万美元用于改善非洲的跨境贸易

原文：https://web.archive.org/web/http://techcrunch.com/2021/10/07/backed-by-greycroft-klasha-gets-2-4m-to-improve-cross-border-commerce-in-africa/

分析师称，非洲的电子商务价值预计明年将达到 290 亿美元，然而非洲人仍然发现很难在网上进行产品的国际支付。

卡、M-Pesa、银行转账和移动支付是非洲人最常用的支付方式。 但是，尽管各种支付网关提供了比十年前更好的体验，但仍有更多的工作要做。

总部位于拉各斯和旧金山的初创公司 Klasha 看到了跨境商务的利基，并提供了多种集成和 API 来促进该领域的交易。如今，它已经筹集了 240 万美元的种子资金。

杰西卡·阿努娜于 2018 年创立克拉莎。 当时，公司的重点是让非洲消费者更容易 从全球时装零售商 直接购买 产品。

她告诉我，但是克拉莎现在不仅仅是那样。它有几个特点和一个新的商业模式 围绕着帮助非洲人付款和得到他们想要的商品(只要它们不会腐烂)，不管他们在哪里 。

与 Checkout 非常相似的 Klasha Checkout 允许国际商人以当地货币从非洲收款。与只支持国际支付方式的 Checkout 或 Fast 不同，Klasha 使用非洲人习惯的方式——银行账户、信用卡、USSD、M-Pesa 和移动支付。

该公司表示，结账解决方案可以集成到任何电子商务平台中。它拥有知名电子商务网站 WooCommerce、OpenCart 和 BigCommerce 的插件，并将与 BigCommerce 签署官方合作伙伴关系，将其触角延伸到全球更多的商家。

有了 KlashaWire，消费者可以通过 多种 支付方式用非洲货币支付:奈拉、塞地、先令，Klasha 然后在两个工作日内用美元或欧元等主要货币汇给商家。

其支付链接功能支持没有店面的商家通过电子邮件或社交媒体与客户分享链接来接受支付 。

接下来是它的移动应用程序，让尼日利亚、加纳或肯尼亚的用户可以发送和接收资金。 他们还可以创建虚拟卡，用各自的货币为其注资。

除了 对其的付款发挥，Klasha 也是进入物流。 Anuna 声称，由于 Klasha 结合了支付和物流两个领域，它正在向非洲的消费者和希望无缝扩展到非洲的商家开放全球电子商务经济。

“我们的许多商家对我们说，如果我们要能够接受来自非洲的付款，我们需要一种方法来在短时间内无缝地运送到非洲，并为我们的消费者提供最佳的端到端物流体验 。这就是我们所做的，”首席执行官说。

据她所说，物流服务通过与第三方物流供应商的合作，帮助非洲客户在五到九天内从欧洲或美国获得产品。从更轻松的角度来看，与长达两周 的行业数据相比，Klasha 的时间框架相当值得称赞。

自从 Klasha 在经过几次迭代后于 5 月重新启动以来，它已经处理了超过 20，000 笔交易，并声称每月增长 366%。Klasha 通过销售佣金和订阅获得收入，零售商付费使用该平台进行分析，以衡量他们的产品在不同市场的表现。

“对许多零售商来说，这是他们第一次向非洲销售商品。 所以我们为他们提供完整的端到端电子商务套件，而不是不得不使用与技术无关的脱节服务，”她说。

Greycroft 的合伙人 Alison Lange Engel 说，Klasha 的技术允许无缝跨境交易，因为非洲正在迅速发展，需要在线商务的支付和物流解决方案。

这是这家风险投资公司继押注独角兽公司 Flutterwave 和 AZA Group 之后在非洲进行的第三笔重大投资。AZA Group 是一家由伊丽莎白·罗西耶洛(Elizabeth Rossiello)创立并领导的金融科技公司。

和罗塞罗一样，29 岁的阿努纳在创办由 Techstars 支持的 Klasha 之前就有行业经验。这位首席执行官曾在伦敦的亚马逊(Amazon)和 Shopify 工作，随后在中国创办了一家物流公司。在为大型批发商和供应商向英国和美国出口快速消费品的同时，她 认为中国(非洲最大的贸易伙伴)的商人需要更有效的方式从尼日利亚商人 那里接收付款。

的经历让阿努娜看到了跨境贸易和支付的世界，这也证明了她作为一名独立女性创始人 筹集风险资本的重要性。

黑人女性创始人获得全球风险投资的 0.6%。没有数据来描述非洲女性创业者的情况，但如果你能描绘一幅图画，它不会是一幅美丽的图画。

但阿努纳认为，投资者开始支持更多的女性创始人，特别是在金融科技领域，最近的初创企业和风险投资活动令人瞠目。

除了 Greycroft，其他投资者还包括 Seedcamp，Practical VC，Plug and Play，First Fund，2.12 Angels，MiLA Capital，Berrywood Capital，AVG Basecamp Fund，Expert Dojo。怀斯的乔·克罗斯和古姆特里的迈克尔·潘宁顿等天使投资人也进行了投资。

Klasha 希望利用这笔资金帮助 ASOS、Zara 和 H & M 等零售商接收非洲消费者的付款。Klasha 目前在尼日利亚、加纳和肯尼亚拥有 10，000 人的客户群，计划在今年年底前将业务扩展到另外三个非洲国家。
