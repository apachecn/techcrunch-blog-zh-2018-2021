Natasha 和 Mary Ann 和 Alex 本周都在船上，刻度盘上有 Grace 的声音，这意味着我们度过了一段平淡美好的时间为你记录公平。当然，Equity 是 TechCrunch 关注风险投资的播客，我们在这里挖掘最关键的融资回合，并谈论影响初创公司的关键新闻。

在我们进入本周的话题之前，你可以在 Twitter 上关注这个节目，我们经常在 Twitter 上举办即兴的空间，有时会变成剧集。来挂吧！

这是本周的概要:

  • Chalo 筹集了 4000 万美元用于改善印度的公交系统 : 这家初创公司赢得了本周名称(和初创公司名称)。Chalo 希望解决印度公交系统的低效问题，因此我们探讨了为什么这有意义以及未来可能面临的挑战。
  • 大师作品公司为部分艺术所有权筹集了 1.1 亿美元 : 如果你必须这么做，就称之为 A 轮融资，但大师作品公司刚刚筹集的巨额资金有助于强调全球向另类投资和部分所有权的转变。我们还要多久才能得到区块链的杰作？这将是真正的 IRL-NFT 交叉我们有点期待。
  • CostCertified 想拯救你的下家雷诺项目 : CostCertified 刚刚参加了 Y Combinator 的夏季队列，筹集了 845 万美元的种子资金。这家加拿大公司的最终目标是建立“建筑亚马逊”CostCertified 允许承包商向业主发送可购买的交互式评估，以便他们可以在项目期间选择，并立即看到对价格的影响。
  • 关于社区的一切 : 社区已经被淡化了，这是毫无疑问的。但是，仍然有关于它为什么有效的争论——而且我们经常争论。
  • 谷歌投资非洲 : 美国科技巨头谷歌正在非洲投入资金，但形式是基础设施投资和早期投资。坦率地说，考虑到广告巨头的商业模式，这两种说法都很有道理。
  • Edtech 走 B2B : Udemy 要上市了！我们已经挖掘了这些数字，但谢天谢地，有娜塔莎在节目中，我们可以更深入地了解教育技术的收入下一步可能来自哪里。
