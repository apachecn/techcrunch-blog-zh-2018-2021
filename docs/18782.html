Tiger Global 在 A 轮融资中以 1500 万美元支持非洲金融科技 Mono TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/11/tiger-global-backs-african-fintech-mono-in-15m-series-a-round/amp/

图片来源:单声道

今天，非洲初创公司 Mono 向 TechCrunch 证实，该公司帮助消费者将银行账户与金融应用程序联系起来，已经筹集了 1500 万美元的 A 轮融资。

本轮由 Tiger Global 牵头，这是继 Flutterwave 和 FairMoney 之后，风投公司今年在非洲的第三笔重大投资，新投资者 Target Global(新银行 Kuda 的投资者)、General Catalyst 和 SBI 投资 也参与了该轮投资。

现有投资者加入了新的投资者，包括 Entree Capital、Lateral Capital、Golden Palm Investments、Acuity VC 和 Ingressive Capital，使 Mono 自去年推出以来的总融资额略高于 1750 万美元。

超过一半的人口没有银行账户或者银行服务不足。 像 Mono 这样的非洲开放金融参与者——反映了 Plaid 在美国市场的成功——蓬勃发展的理念是，通过开放 API 进入金融生态系统将改善金融信息的获取，并降低银行不足者的进入成本 。

随着 Okra、Plaid、Stitch 和 OnePipe 等公司也在单一 API 中简化各种财务数据，向公司和第三方开发者提供客户财务信息的战斗正在升温 因此 需要推出更多创新产品。

据首席执行官 Abdulhamid Hassan 说，Mono 的两个产品给了它优势。首先是， DirectPay，这是一款 产品，帮助尼日利亚企业在他们的网络或移动应用程序内收取客户的银行转账付款，而无需使用他们的借记卡。T 想想 Flutterwave 和 Paystack 对卡做了什么；Mono 希望对银行账户也这样做。

接下来， 报表页面允许企业在不需要开发者的情况下访问客户的财务账户；哈桑称之为非洲的首例。

尽管有这些为尼日利亚观众量身定制的附加功能，Mono 仍在追逐加纳和肯尼亚等其他市场，首席执行官在 5 月份首次提到了这一点。

目前，Mono 仅在加纳推出，与 Oze 和 Tranzo 等几家银行和金融科技公司进行试点。 另一方面，肯尼亚仍然在其雷达范围内，因为 三个国家之一(包括埃及和南非)的 Mono 计划扩展到明年。

Hassan 补充道，Mono 可能会在今年通过与 Absa 银行的合作完成向南非的转移。

去年，当非洲的开放式金融初创公司开始获得一些关注时，人们一致认为，每家公司都可以在他们最初推出的大型个体市场中开展重要业务。 但现在看来已经不是那样了， 随着 Mono、Stitch、Pngme、Okra 现在也在其他非洲市场经营。

与普拉哈尔·辛格(Prakhar Singh)共同创办 Mono 的哈桑(Hassan)在电话中说:“我觉得，就我们能瞄准的企业数量而言，像尼日利亚这样的国家已经受到了限制。

到明年年底，Mono 将出现在五个非洲国家，这一举动略微表明该公司可能会把自己扩张得太薄。

当被问及投资者的压力或竞争是否促使 Mono 快速扩张时， Hassan 认为该公司做出这样的决定完全是因为它的客户。

“我们在 Mono 所做的一切都是 主要是 客户驱动的。想要在这些国家推出贷款或金融科技应用的客户表示，除非 Mono 在那里，否则他们无法这样做，”他说。“ 根据我们在尼日利亚建立的关系和我们生产的产品种类，人们依赖我们，并说当我们去一个国家时，他们也会想在那里推出 。”

该公司还有一个纵向扩张的机会，因为它希望为 FMCGs、律师事务所提供企业解决方案，这些非科技和传统行业有能力控制更多的交易量 。

谈到数字，Mono 声称已经处理了来自 270 多家企业、开发商和金融科技公司的超过 2 亿笔金融数据交易。

该公司还表示，在过去的两个月里，它已经连接了超过 15 万个银行账户，并且在这方面同比增长了 45 倍。据首席执行官称，Mono 目前约有 30 名员工，人数较上年翻了一番。

全球开放金融市场的活动一直受到 Visa 收购 Plaid 失败的刺激。投资者也没有放慢支持其他地区初创公司的步伐，他们认为，如果这家美国公司决定扩张，这些地区可能是其主要收购目标。

Mono 就是非洲的一个这样的候选公司，主要投资者 Tiger Global 知道这一点。根据 Hassan 的说法，风投公司寻求领导新一轮融资，尽管 Mono 今年没有计划筹集任何额外的资本，但 Tiger Global 的资本将帮助它更快地增长。

我们觉得Tiger 的团队觉得我们在做一些令人惊叹的事情，但认为我们筹集的资金不会帮助我们建立一个大公司。他们说，‘我们想帮你建立一个大公司，’”哈桑说。

Tiger Global 在给 TechCrunch 的一封电子邮件中证实，它领导了这笔投资。

作为非洲发展最快的初创公司之一，Mono 以非洲大陆罕见的方式完成了早期阶段的融资。去年 九月，该公司刚刚完成了一笔 50 万美元的前期种子投资，投资方是一些当地投资者。然后参加了Y Combinator 最近的冬季项目，在毕业时筹集了 200 万美元的种子资金，然后筹集了 A 轮融资。

然而，并不是每个人都这样衡量增长。如今，初创公司烧钱快速扩大规模，但收入很少或没有收入来展示 。在非洲，经常会有关于 这种模式可能会导致像 Mono 这样的高歌猛进的初创公司长期无利可图的附带评论 。

当被问及这是否是这家估值超过 1 亿美元的公司的一个担忧时，哈桑说:“当你建立一家初创公司时，你首先要考虑的不是盈利能力，但我可以说，自从我们上次在 5 月份融资以来，我们的收入已经增长了 10 倍。”

Mono 目前的收入相当于其过去三个月的收入总和，这位首席执行官继续说道，然后提到该公司如何通过精干的团队追求增长、规模和盈利能力，以“建立非洲最好的开放式银行平台”
