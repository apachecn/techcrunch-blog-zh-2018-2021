比特币基地正在推出自己的 NFT 平台来应对 OpenSea——TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/12/coinbase-is-launching-its-own-nft-platform-to-take-on-opensea/amp/

图片来源:史蒂夫·詹宁斯/盖蒂图片公司

比特币基地正准备推出一个 NFT 市场，它将与该领域现有的主要参与者竞争，如 OpenSea，推出一个桌面产品的用户等候名单，它希望在今年年底推出。

比特币基地在即将推出的平台上分享的细节有限，但在一篇博客文章中提到，他们希望将社交功能更深入地整合到平台中。"比特币基地 NFT 将使铸造、购买、展示和发现非专利技术比以往任何时候都更容易。我们通过构建直观的界面，将复杂性置于幕后，使 NFTs 更容易访问。我们正在增加社交功能，为对话&发现开辟新的途径。”

尽管比特币距离其在比特币基地直接上市时的历史高点越来越近，但比特币基地的股票仍未能跟上该领域一些最受欢迎的硬币的反弹增长，自 4 月份直接上市以来，其股价已下跌约 27%。该交易所同时受到监管机构的严密监视，美国证券交易委员会(SEC)最近威胁要起诉该公司的贷款收益率产品，后来他们取消了该产品的推出。随着 Robinhood 等交易所对加密技术的支持越来越强，比特币基地也在应对美国国内购买和销售加密技术的拥挤竞争局面。

图片鸣谢:假设 NFT 继续吸引那些寻找高价数字收藏品和艺术品的投资者的兴趣，CoinbaseCoinbase NFT 可能会为这家现已上市的公司开辟一个主要的收入来源。在今年早些时候和 8 月再次飙升后，NFT 市场显示出比许多人预期的更强的弹性，但仍然比已经波动的加密货币价格波动更大。8 月份，OpenSea 的交易额达到创纪录的 34 亿美元。

该公司推出了对以太坊区块链的支持，这并不是一个非常令人惊讶的地方开始在 NFT 推出，但比特币基地显然不会在推出时与任何第二层扩展网络集成，这意味着比特币基地 NFT 的用户仍然可能支付高额的燃气费，并在推出时处理 mainnet 以太坊的许多可扩展性问题。比特币基地在一篇博客文章中表示，它最终将在比特币基地 NFT 上提供“多链”支持，但将支持 ERC-721 和 ERC-1155 以太坊标准。竞争对手 OpenSea 最近利用 Polygon 网络推出了对大幅低价交易的支持。

OpenSea 是该领域的主导力量，但远非比特币基地必须与之竞争的唯一玩家。币安和 FTX 等竞争对手也宣布最近进入 NFT 市场生态系统。虽然许多加密投资者认为 NFTs 只是触及了其潜力的表面，但比特币基地的推出比一些即将成为竞争对手的公司晚得多。然而，鉴于比特币基地在消费密码领域的知名度，这可能会是一个大的启动。
