公司创建者 VC Antler 表示，它可以支持初创公司进行 C 轮融资，但它的条款是什么？TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/13/company-builder-vc-antler-says-it-can-back-startups-to-series-c-but-what-are-its-terms/

2017 年，鹿角在新加坡主要以“公司建设者”的身份出现，并以全球创业发电机计划的运营者和从非常早期的种子前阶段进行投资而闻名。它倾向于将团队聚集在一起，混合在一个技术堆栈中，并以惊人的速度产生创业公司。但是事情在这个基础上进展很快，鹿角公司现在的目标是从早期阶段到后期阶段全程投资。

去年，在获得施罗德和 Ferd 的投资后，T2 筹集了 7500 万美元。该公司现在表示，自去年以来，它又筹集了 2.25 亿美元。

一位发言人澄清说:“以前的累计总数是 7500 万，现在是 3 亿，也是全部。因此，自上次更新以来，又有 2.25 亿美元被筹集到。”

他补充说，这一数字此前从未披露过:“鹿角的基金不像许多基金那样是‘连续的’。他们不是在募集一只主要的“一只基金接一只基金”，也不是在做一只基金加一只“副业”之类的事情。他们在任何时候都有多只基金在运作，”他补充道。

但是，细节决定成败。

因为鹿角公司在所有不同的公司所在地都有资金来源，这些资金分布相当广泛，而且，如果你想稍微怀疑一下的话，很少。(它的业务遍及欧洲、亚太(APAC)、东南亚国家联盟(东盟)、东南亚、非洲(肯尼亚)，并在伦敦、纽约、新加坡和悉尼等地设有办事处。毕竟，一些风险投资者将 2.25 亿美元分散在像英国这样的一个国家，而不是整个地球。

因此，尽管今天发布消息称，它已经筹集了“迄今为止 3 亿美元”，但这只是在一定程度上是准确的，特别是当你考虑到它是在 2017 年成立的，所以五年内 3 亿美元在风险投资方面并不太重要。

该基金的投资者包括施罗德、kstfonden 和凤凰集团。

鹿角公司仍然专注于种子前期，但考虑到额外的 2.25 亿美元，它表示，随着投资公司的增长和规模扩大，它现在可以为它们提供后续资本，最多可达 c 轮。

在一份声明中，鹿角公司的首席执行官兼创始人马格努斯·格里梅兰(Magnus Grimeland)表示:“我们继续从最早的阶段为我们的创始人提供支持，并且很高兴能够在他们成长的过程中继续投资他们。我们期待在未来支持更多重新定义行业的企业家。”

该公司还宣布招聘新合伙人 Naman Budhdeo、Erik Jonsson、Jiho Kang 和 Subir Lohani，分别领导其新的加拿大、越南、韩国和印度尼西亚团队。软银的泰迪·希姆勒也作为后期投资团队的合伙人加入。

自 2018 年以来，鹿角投资了全球 30 多个不同行业的 350 多家公司。报告称，在这些公司中，40%的公司至少有一名女性联合创始人，代表着 70 个国家。

但是回到那些条款。鹿角表示，它可以一直支持一家初创公司进入 c 轮融资。当然，这将取决于每笔交易的条款，以及鹿角是否能够在初创公司的整个生命周期中一直提供资金。这就是其交易条款可能会受到审查的原因。

鹿角的成长并不是没有代价的。TechCrunch 从几个渠道获悉，安特尔对初创公司的条款并不“对创始人友好”。我把这个问题问了首席执行官格里梅兰，想听听他的反应。他承认，“早期”可能是这样。

他说:“我认为，总的来说，我们的创始人条款得到了非常好的反馈。在我们得到你提到的反馈的地方，我们实际上做了改变。我们希望确保我们尽可能地对创始人友好，同时提供强有力的保护。我认为我们在所有市场上都非常有竞争力。”

Grimeland 声称，由于它在如此多的司法管辖区开展业务，不同地区的法律框架可能会有很大不同:他告诉我，他们很早就意识到的一件事是，在美国可能很常见的创始人条款在应用到一个完全不同的国家时可能会“感觉荒谬”:“在这种情况下，我们实际上与当地的天使社区合作，更新我们的条款。他们非常有竞争力，符合其他早期风投和天使投资人希望在合同中看到的内容。”

无论情况如何，但由于增长速度如此之快，在全球范围内传播，鹿角将自己定位为 Y Combinator 和 Entrepreneur First 等公司的潜在未来挑战者，这将成为未来几年值得关注的有趣故事。
