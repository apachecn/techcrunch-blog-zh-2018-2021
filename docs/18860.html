这年头你们管 AI 叫什么？穿西装的 ML。

ML，即机器学习，在今天是一个很大的市场。这要归功于像囤积者这样的现代企业积累数据，以及数据科学作为一种工作类别的成熟。前者的证据可以在 Databricks 最近几个季度发布的增长中找到，而后者的证据则是大型科技公司愿意为专注于 ML 的角色投入多少现金。

因此，加权&偏向的市场这些天很活跃。这使得这家初创公司刚刚在一轮超大型 c 轮融资中筹集了超过 1.35 亿美元的资金并不令人惊讶。该公司在一份新闻稿中表示，该公司现在的市值约为 10 亿美元。Felicis、Insight Partners、Bond 和 Coatue 促成了这笔交易。

根据 Carta data的数据，自 2020 年初以来，以数据和分析为重点的 C 轮融资的中值为 4375 万美元，最终估值中值(融资后)约为 4.16 亿美元。鉴于历史数据，这使得本轮融资实际上是我们对该公司预期融资额的两倍。

在产品方面，Weights & Biases 在“MLOps”领域，即机器学习操作市场中发挥作用。MLOps 自然类似于 DevOps，尽管是一个较新的类别。

图像信用:权重&偏差。有问题的产品。

根据 Weights & Biases 联合创始人 Lukas Biewald 的说法，软件世界有一套为开发人员编写和部署代码的工具。这可能包括 git 风格的服务( GitLab 、 GitHub 等)。)、监控( Atlassian 、 Datadog 等。)，诸如此类。

他解释说，他的公司的目标是为 ML 世界建立一个类似的服务栈。今天，他解释说，许多 ML 团队正在使用特别的工具或者简单地没有软件的帮助。

对这样一个堆栈的需求可能是巨大的。根据 Biewald 的说法，开发和 ML 世界之间的一个区别是，当代码失败时会崩溃，而 ML 工作可以以更微妙的方式“表现糟糕”。

自然地，输入权重和偏差。这家初创公司的产品生命始于实验跟踪，Biewald 将其比作 DevOps 堆栈中的代码版本控制。他解释说，Git 虽然对人类编写的代码版本控制很好，但在处理不同版本的计算机生成代码方面有些差，比如来自机器学习工作的代码。这正是 Weights & Biases 想要解决的问题。

这一努力无疑吸引了投资者的关注。Felicis 的投资者 Aydin Senkut 告诉 TechCrunch，他关注 Weights & Biases 已经有一段时间了，但其他投资者最终领导了前两轮投资。这一次，Senkut 通过抢先收购该公司进入了资本表。根据 Biewald 的说法，如果 Felicis 没有带头发起攻击，Weights & Biases 可能会发起类似的一轮融资，尽管时间会晚一些。

在与该公司交谈之前，TechCrunch 深入研究了这家初创公司的定价方案。与 Weights & Biases 在构建其服务时似乎有意提供的生产率相比，其价目表看起来并不昂贵。注意，这本身并不是一种恭维；抑价是短期内将价值从公司和投资者转移给客户的一种方式。

Biewald 表示，Weights & Biases 正在对其服务进行定价，以便任何人都可以轻松访问。Senkut 补充说，在 Felicis 对投资进行尽职调查期间进行的客户检查中，客户表示，这家初创公司的服务定价低了三倍。

这位投资者补充说，他对这一前景感到兴奋，因为 Shopify 等其他公司也曾追求类似的长期贪婪，而不是短期收入。

坦率地说，我对 Weights & Biases 似乎想要建立的东西很感兴趣。让我们看看一张早来的九位数支票能让公司走多远。下一次我们谈话时，是时候更精确地关注增长指标，并检查其免费服务的利润影响(当然，除非它将该特定项目列入其销售和营销费用项目)。
