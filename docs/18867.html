核心可能是元宇宙的拉斯维加斯

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/14/core-manticore-metaverse-game-creation-hands-on/amp/

图片来源:蝎狮奥运会

Core 自称是“无尽的街机”，感觉就像 90 年代的赛博朋克热梦想成真。一半是可玩的游戏库，一半是无代码的游戏创建者，所有的霓虹灯，这个新平台是一个令人惊讶的很好实现的元宇宙事物的愿景，每个人最近似乎都在谈论它。

宣称自己是你的“多元宇宙的门户”，核心准备好测试古老的命题 如果你建造它，他们会来。像 Roblox 和脸书这样的大公司可能有巨大的成熟平台，但 Core 已经为创作者和玩家奠定了一些非常引人注目的基础。

登录后，玩家被传送到 Core 的中央枢纽，这是一个主题公园、高科技购物中心和赌场之间的恰当交叉，娱乐和购物在每个方向都有几个不受重力影响的大步。巨大的霓虹灯招牌吸引着玩家，吸引他们进入无数用户创造的虚拟世界。交换衣服和游戏装备，或者邀请一个朋友和你一起玩，只需要点击几下鼠标，四处逛逛，人们看着就足够有趣了。

如果 Core 看起来很像堡垒之夜，那不是巧合。核心，由蝎狮游戏公司制造，运行在堡垒之夜制造商 Epic 的虚幻引擎上。这些联系甚至更深:Epic 去年对该公司进行了一轮 1500 万美元的投资，该平台只能通过 Epic Games Store for PC 获得。今年 3 月，蝎狮从主要投资者那里筹集了 1 亿多美元，并将其 creator 平台投入使用。

图片来源:蝎狮奥运会

Core 可能还不是一个家喻户晓的名字，但它已经成为任何元宇宙有志者必须克服的挑战之一。在我玩 Core 的时候，从一个地方到另一个地方的体验经常是无缝的，以至于我不小心到了错误的地方。把这归咎于用户的失误，但在穿过各种门户网站后，立即被传送到一个死亡之秀，一个杂草丛生的反乌托邦荒原，一个等距的海盗游戏，这是我十多年来玩这些游戏中最无缝的在线多人游戏体验之一。

核心看起来很棒。这是对 Roblox 的一次打击，rob lox 是构建元宇宙愿景最成功的公司之一。很像堡垒之夜，核心的图形是卡通，但不是太卡通。Roblox 的 13 岁以下人群正在老龄化——这是该公司正在围绕积极规划的一个因素——那些不那么年轻的玩家可能很快就会寻找一个更成熟的新虚拟家园。

任何有抱负的 edgelord 都可以认真对待 Core 的各种定制服装和头像。或者你可以是一只小猫。

图片来源:蝎狮运动会

元宇宙居民 Deadmau5

Core 的大部分内容是 UGC，也就是用户生成的内容，这是一个定义时代的在线现象的新名字(如果这个缩写让人想起综合格斗，不要责怪自己)。但蝎狮也有足够的空间与音乐人和品牌合作，打造精致的主题游戏体验。

本周，DJ 和 EDM festival 常年 Deadmau5 推出了他自己的，一个庞大的，丰富多彩的系列体验，被描述为“元宇宙永久居留”Core 主要是用户自制游戏的家园，但它也是娱乐甚至教育的天然选择——该团队注意到一些用户开始举办游戏开发课程。

不同于最近在其他虚拟世界中的展示，如 Roblox 中的 Lil Nas X 或堡垒之夜中的爱莉安娜·格兰德，Deadmau5 主题的内容将在首次亮相后保持直播，供任何人探索。蝎狮的团队把这比作像 Penn 和 Teller 这样的表演者如何在拉斯维加斯为正在进行的演出露营，这个比喻非常恰当。但与拉斯维加斯不同的是，表演者可以同时出现在两个地方:Deadmau5 还宣布他将参加本周在以太坊虚拟平台分散之地举办的音乐节。

我和 Deadmau5，né Joel Zimmerman 一起看了这部剧，提前预览了一下。他戴着他标志性的巨型动物头盔之一(我想是一只猫？)和 cyborg angel wings，而我选择了低调的黑色连帽衫，元宇宙的小黑裙。

“我认为吸引我的是它的模块化，以及它如何为创作者提供更多工具，”齐默尔曼告诉我，一边在 Core 里疯狂地跳来跳去，一边斜靠在印有 Deadmau5 鼠标的游戏椅上。

图片来源:蝎狮奥运会

就像我们对虚拟音乐会的期望一样，互动表演充满了融化的迷幻视觉效果、迷你游戏和带有转盘耳朵的危险的 Chomp 式鼠标。齐默尔曼和 Core 的联合创始人弗雷德里克·德斯坎普斯(Frederic Descamps)和乔丹·梅纳德(Jordan Maynard)也和我一起看了至少 10 遍，但每个人似乎都很开心。

在某个时刻，我要么掉进了岩浆，要么被一个巨大的金属拳头砸在了传送带上，而一个以死亡之城为主题的恶棍就在附近出现了。梅纳德说:“我认为这是唯一一场你可以在其中死去的互动音乐会。”这场演出在视觉上非常有趣，富有创造性的互动，最终很像堡垒之夜的音乐会，为这种东西设立了一个高标准。

这个精心制作的虚拟体验名为 Oberhasli T1，也展示了一些由没有游戏开发经验的粉丝创造的独特世界，从一个阴森的丛林废墟到一个充满漂浮太空碎片的怪异世界。核心 Deadmau5 表演将于太平洋时间周五下午 3 点开始。它将在周末重播，并在随后按需提供，供任何想被打成电火花煎饼的人使用。

图片来源:蝎狮奥运会

创作者的核心

后来，在不和谐的气氛中，核心之旅变成了每个人穿过一堵可破坏的墙后面的秘密大门，在游戏类型中疯狂地跳来跳去，每个人都为一些不需要任何代码或游戏开发经验的东西而精心制作。即使在糟糕的 Wi-Fi 连接下，从一个游戏世界转移到另一个世界也需要几秒钟，包括我穿过看起来像魔兽世界黑暗门户的东西，最后驾驶一艘等距海盗船。

哇点头可能不是巧合。德斯坎普斯开始怀念《T4》WoW machinima 的全盛时期，这是一部通过捕捉游戏构建的叙事电影，就像只有一个严肃的老玩家才能做到的那样。德斯坎普斯和梅纳德之前也曾合作过《裂缝》,这是另一部 10 年后仍拥有忠实粉丝的奇幻 MMO 作品。(梅纳德是 7 号员工。)如今，每个人都对元宇宙赞不绝口，但令人惊讶的是，该领域很少有公司能将其根源追溯到多年来将人们聚集在一起的无缝虚拟游戏世界。

图片来源:蝎狮运动会

为了强调在 Core 中制作东西是多么容易，Maynard 快速构建了一个第一人称射击游戏供我们玩，这是一个拖放过程，花了大约两分钟的时间浏览 Core 的巨大原始游戏资产库，这些资产是使用其系统创建的。抓住一些 3D 物体，从模板选择中选择一种游戏模式(皇家战役、赛车或地下城爬虫？)而且你已经在 Core 的模块化沙盒中打造了一款外观精美的可玩游戏。将你的游戏设定在寒冷的雪景或贫瘠的沙漠也像拖放一样简单，给环境一种广阔的感觉。

抛开游戏性不谈，开箱即用的核心游戏看起来比你在 Roblox 中遇到的 UGC 要好几光年，尽管该平台的用户似乎从来不介意。视觉风格和游戏类型的广度也让任何在其他平台上跳出 samey UGC 的人难以置信。

创造内容的核心用户有相当多的货币化选择，蝎狮称之为“额外津贴”这包括提供游戏中的装饰品，但也对优质游戏收费，出售类似堡垒之夜的战斗通行证或实施订阅模式。收入分成是 50/50，与 Roblox 转给创作者的 25%相比，这看起来很慷慨。和其他模块化游戏制作平台一样，在核心平台上，每个人都是创作者——不需要开发经验。

Core 目前只在 PC 上使用，但蝎狮计划从明年开始将它引入其他平台，包括 iOS。游戏创作可能会局限于 PC，但这个想法是任何人都可以在任何地方玩核心游戏，这一平台不可知的愿景无疑推动了早期的堡垒之夜和最近的 Roblox。

“[游戏开发]有点像烘焙:一个非常精确的公式，技术性的，可能需要数周的迭代，”德斯坎普斯说。但在核心中，技术方面的东西不再碍事，通常会拖延的过程可以在几分钟内完成，剩下的时间留给实验和玩耍。

"如果你把传送门枪放进马里奥赛车会怎么样？"梅纳德问，我相当肯定我们当时就能发现。
