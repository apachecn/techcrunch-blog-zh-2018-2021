本周，创业领域的关键投资故事是 GitLab 的大规模 IPO。

如果你落后了， GitLab 申请上市，我们注意到，按照目前的市场价格，DevOps 巨头可能价值约 100 亿美元。结果是保守的。GitLab 最终将其 IPO 价格区间提高到远远高于其最初的估计，然后定价为 77 美元(T21)。周五下午晚些时候，当我给你写信时，它的价值超过每股 108 美元。

我与 GitLab 首席执行官 Sid Sijbrandij 通了电话，讨论了这笔交易。我已经和 Sijbrandij 聊了一段时间，从这个特别的故事开始。因此，在 IPO 当天与他交谈很有意思，因为他受到美国证券交易委员会正常规则的约束。以下是我学到的:

  • git lab 为什么现在上市？Sijbrandij 说，它达到了所有指标，包括收入规模、收入可预测性和合规性。从联合创始人 Dmitriy Zaporozhets 为该公司编写第一个代码的那个月算起，IPO 日期已经过去了 10 年。所以这是一个很好的时间循环。人类确实喜欢整数。
  • git lab 强大的净留存指标对收入的可预测性有帮助吗？是的，但是 Sijbrandij 不想明确地说出来。
  • 开源现在是一个优势，而不是一个障碍:这一点与我们前几个月提到的关于创业公司的一些事情相呼应，但仍然值得一提。对于希望与开发者建立长期关系的公司来说，拥有开源代码现在是一个的福音，我敢说，这通常是产品导向型增长的关键。这与十年前的世界截然相反，也许这就是为什么微软不久前改变了对开放代码概念的态度。
  • 我们会看到 GitLab 进入 MLOps 和 DevOps 吗？也许？Sijbrandij 在这件事上并不是非黑即白的，但是随着 MLOps 世界加速发展，如果我们看到 GitLab 及时进入所谓的创业领域，我不会感到震惊。它现在当然有资金做它想做的任何事情。

云闪对世界

让我们回到 9 月下旬的新闻，即 Cloudflare 正在进入“存储即服务”市场。消息是，Cloudflare 打算通过其全球数据中心集合提供云存储。这一产品新闻与 Cloudflare 最著名的功能相去甚远，即让网站更快更安全地出现。

这家现已上市的公司为什么会涉足像存储这样的商品化领域？当时，Ron Miller 写道，Cloudflare 正在将它为自己构建的一些东西转向其他人。通过取消一些费用，Cloudflare 的存储服务——R2——将比亚马逊提供的服务更便宜，例如，通过其 AWS 的基础设施服务集合。

我有一个想法。也就是说，如果我们看到大型但不庞大的全球技术公司提供特定风格的数字服务也开始提供看起来最初是小众的基础设施工具，与亚马逊和微软目前通过 AWS 和 Azure 提供的工具进行适度竞争，我不会完全惊讶。

这可能很愚蠢，但是我们可以通过类比来解释一下。我的论点类似于英特尔如何用其特殊的 CPU 方法长期统治世界，结果不仅让被迫进入加密货币盐矿的 GPU 失去了未来，也失去了对创业公司的控制，比如说人工智能调整的硅。在我们的类比中，AWS 是英特尔，AI 芯片是像 Cloudflare 的 R2 一样的东西。

AWS 和 Azure 来回降价的日子已经过去了。下一步是什么？
