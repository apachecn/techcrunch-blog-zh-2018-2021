交叉思维引入新资本，提供不需要可识别信息的推荐 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/18/crossing-minds-brings-in-new-capital-to-provide-recommendations-that-dont-require-identifiable-information/

我们大多数的媒体和购物世界都涉及到我们正在看的东西下面的小盒子，这些小盒子提供了其他可以看的节目或可以购买的物品，这些东西要么是相似的，要么是其他人也买了的。

消费者还期望网站更加个性化，这意味着网站需要了解你的一些情况，以便向你展示你想看的东西。

首席执行官 Alexandre Robicquet 与 Emile Contal 和 Google X 创始人兼斯坦福大学教授巴斯蒂安·特龙组成的全明星创始团队一起，启动了 2018 年在 TechCrunch 的 Disrupt 战地发布的 Crossing Minds 和，作为一款以消费者为中心的应用程序，名为 Hai，为用户提供书籍、音乐、节目、视频游戏和餐厅等娱乐推荐。

两年后，该公司拥有数千名活跃用户和一些大型组织作为客户，但推荐存在一些挑战，这导致该公司暂停并转向 B2B。

“个性化无处不在，甚至搜索的下一次进化也会被过滤，”Robicquet 告诉 TechCrunch。“这不是为一个企业或 API 建立推荐引擎的问题，而是做一些可以扩展并服务于许多企业的事情。”

Robicquet 表示，高达 60%的消费者通常是网站的新用户，因此“冷启动”可能是个问题。他补充说，再加上其他障碍，如第三方 cookie 贬值，更严格的隐私法以及实时链接当前和过去的网络会话的困难，几乎没有办法知道是谁在访问该网站。

Crossing Minds 正在应对这一挑战，并开发了一种使用现场操作提供建议的方法，以便客户可以更容易地发现和参与企业，但不需要个人身份信息。

该公司与客户的数据库相连，能够在两周左右的时间内根据关键绩效指标构建出一个模型。然后，它的 SaaS 模式按推荐收费。

周一，Crossing Minds 宣布在由 Radical Ventures 牵头的首轮融资中获得 1000 万美元。现有投资者 Index Ventures、Partech 和 Lerer Hippeau 也参加了会议。Robicquet 希望将新资本注入其工程团队、产品开发和客户群，并充实其领导团队。

Crossing Minds 正在与几家公司合作，包括企鹅兰登书屋、达能和 Inkbox。他估计，该公司已经帮助其客户将销售额平均提高了 93%，点击率提高了 120%。

激进风险投资公司的联合创始人孙铁麟·普塔宁说，像 Inkbox 这样的公司能够“在与 Crossing Minds 合作时跨越卢比孔河”，他们看到的转化、重复销售和流失的指标是驱动经济公式的因素，他说。

例如，Inkbox 在 2 月份开始与 Crossing Minds 合作，并已经看到了一些指标，包括电子邮件点击率增加了 250%，购物车增加了 348%，全新用户的现场转化率增加了 68.6%。

“亚历山大，埃米尔和塞巴斯蒂安有一些独特的东西，”普塔宁说。“数学和高科技的血统和程度的基础在其他地方不容易找到。他们深刻理解产品库存以及如何使用它。”
