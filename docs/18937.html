原文：https://web.archive.org/web/https://techcrunch.com/2021/10/18/magsafe-is-back/amp/

苹果当然喜欢带我们乘坐连接的过山车。新款 MacBook Pro 电脑非常耗电，以至于 MacBook 系列依赖了一段时间的 USB-C 连接器不再管用了。你仍然可以使用 Thunderbolt 端口给你的电脑充电，但如果你打算将新的 M1 Pro 和 M1 Max 处理器的寿命缩短到一微米以内，现在是时候了。

实际上，这是一个聪明的设计:这意味着你可以将 MagSafe 3 充电器留在你的主工作站上，在那里你可以进行繁重的计算机工作。你可以在旅途中随身携带一个更小、功率更低的充电器，当你把笔记本电脑用作美化的打字机或网络浏览设备时。这是两个世界的最佳选择——尽管很明显，USB-C 电缆没有 MagSafe 快速断开的优势。这有点愚蠢，因为如果有人被你的线索绊倒，这更有可能发生在你外出时，而不是你在家或在工作时安全地坐在办公桌前。

全新的 MagSafe 3 连接器，以及 MacBook Pro 上的新端口系列。图片来源:苹果

新的 MagSafe 端口具有熟悉的磁性快速释放功能，这是苹果在 2006 年开创的——如果有人笨拙地绊倒你的电源线，可以防止你的笔记本电脑被摔到地上。当时，苹果公司宣称它是所有电脑故障的解决方案。

新的 MagSafe 3 连接器当然有一定的吸引力。图片来源:苹果

十年后，苹果改变了主意，全力以赴开发 USB-C，这让人有些挠头；电脑坠落的问题真的不存在吗？看到连接器回到该公司的笔记本电脑是受欢迎的，但进一步让我想知道，到底是什么，信息在这里。

我怀疑苹果只是需要比 USB-C 能够可靠提供的更多的功率(更不用说，人们正在把功率低得可怜的手机充电器插到他们的笔记本电脑上)，如果他们打算设计一个全新的连接器，为什么不回到过去最受欢迎的时代呢。

MagSafe 3 可用于苹果今天发布的全新 MacBook Pro 电脑系列。
