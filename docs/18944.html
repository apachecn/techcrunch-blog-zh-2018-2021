SaaS 的未来将是开源的 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/18/saas-future-will-be-open-sourced/

在过去十年左右的时间里，开源发生了巨大的变化。曾经主要是关于大教堂、集市和微软的邪恶的哲学讨论——这反过来导致人们构建编译器和 Linux 内核之类的东西——成为事实上的标准，甚至是传统企业如何在公开场合构建现代软件。随之而来的是新的商业模式，从为开源用户销售专业服务，到添加带有额外“企业”功能的专有付费扩展，以及越来越多地推出完全托管的开源产品 SaaS 版本。

不出所料，在我们的 TC 会议:10 月 27 日的 SaaS 2021 活动上，我们将讨论这个问题:如何在开源的基础上构建成功的 SaaS 产品。为此，木偶奇遇记首席技术官艾比·卡恩斯、首席执行官、总裁兼联合创始人孔奥古斯托·“阿希”·马利蒂和红点创投董事总经理杰森·华纳将加入我们。

Kearns 目前是 IT 自动化服务 Puppet 的 CTO，该服务提供其核心工具的开源版本，以及扩展其服务的开源模块的大型生态系统。但在加入该公司之前，她还是 Cloud Foundry Foundation 的执行董事，在这个变革的时代，她帮助管理了一个最大、最活跃的以企业为中心的开源项目。Cloud Foundry 最初是在 VMware 内部开发的，它本身形成了 Pivotal 和 SUSE 等公司的许多类似 SaaS 的产品的基础。

孔，之前被称为 Mashape ，是开源商业化的一个典型代表。它的 API 和微服务管理工具的许多核心服务现在可以在开源许可下获得。早在 2015 年，该公司首次开源了 Kong API 管理服务，并一直将这一重点放在开源上。例如，该公司最近推出了新的开源项目，如其库马控制平面(随后将其捐赠给云计算原生计算基金会)。与此同时，该公司现在已经筹集了超过 1.71 亿美元的资金，最近的估值为 14 亿美元。它的客户从德国电信到棒约翰和西门子。出生于意大利的 Marietti 早在 2009 年就创建了这家公司，该团队实际上在当时的 TechCrunch50 大会上展示了第一个原型。

Redpoint 的 Jason Warner 在 Heroku 和 than 担任 GitHub 的 CTO 期间成名，但即使在他今年夏天跳槽到风投界之前，他已经是一名非常活跃的天使投资人，他的名字不断出现在近年来一些最有趣的初创公司的新闻稿中。在过去一年多的时间里，他投资了一些公司，比如 Magic 、 Census 、 Jam 和 Sym 。其中一些人自己构建了大量开源软件，一些人只是消费它，但无论如何，华纳将能够给我们提供关于开源行业状况的风险投资视角，以及是什么让他想要投资(或不投资)以开源为中心的初创公司。

除了我们对开源的讨论，会议还将包括谷歌的 Javier Soltero，以及投资者 Kobie Fuller 和 Casey Aylward 等人。我们希望你能加入我们。这将是一个发人深省的阵容。
