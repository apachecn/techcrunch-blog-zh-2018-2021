墨西哥金融科技的成功:科技如何推动人们转向银行业

More posts by this contributor

本月早些时候，墨西哥第二大银行花旗墨西哥银行宣布正在开发新的 API，以允许金融科技公司实时访问其服务和 ATM 位置等信息。此举正值墨西哥金融科技生态系统蓬勃发展，传统银行不得不接受它们，否则就有落后的风险。

墨西哥一直是——并且仍然是——拉丁美洲金融科技的领头羊。该国是该地区第一个实施专门针对该行业的法律的国家，墨西哥金融科技公司占整个拉丁美洲该行业风险投资总额的近 20% 。更不用说，墨西哥人口对金融科技服务既有人才又有需求。墨西哥多次在最具创新性的金融科技中心名单中名列前茅，neobank 和数字钱包交易持续增长——在 COVID 锁定期间，估计增长了 80% 。

考虑到墨西哥有近 4000 万人(几乎占总人口的三分之一)没有银行账户，金融科技一直是信贷、保险和汇款新途径的必要载体。由于许多墨西哥人不太信任银行，金融科技期权的透明度在人们采用新技术的过程中发挥了重要作用。

没有迹象表明墨西哥的金融科技正在放缓。全球远程革命已经进行了一年多，现在是反思中国金融科技之旅、影响最大的本地参与者以及该行业未来发展的好时机。

墨西哥长期以来一直在培育其金融科技生态系统

墨西哥金融科技的成功并非巧合——这要归功于一系列因素，这些因素使该国成为创造和发展金融科技解决方案的沃土。这种自然演变使人们逐渐熟悉 fintech 作为一种安全和简化的解决方案。

举例来说，近年来，墨西哥的智能手机使用率和互联网普及率都有了显著提高，形成了一个熟悉科技的群体，他们已经习惯于通过设备管理资金。再加上人们对传统银行的厌恶，这种技术素养使金融科技成为一种有吸引力的替代选择。

该国的金融科技法( Ley Fintech )于 2018 年生效，为众筹、加密货币和 API 设定了参数，并详细说明了金融科技产品的监管沙箱。这部法律在促进健康的金融科技领域方面创造了奇迹，由于监管框架的安全网，用户得到了保护，投资者也更加活跃。目前，已有 52 家金融科技公司获准依法运营，还有 38 家正在等待批准。该法律预计将在今年得到进一步发展。
