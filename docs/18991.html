SEC 表格 D 文件显示 Sisu Game Ventures 基金新成立 5000 万美元，目标是游戏初创公司

根据 SEC 的表格 D 文件，Sisu Game Ventures 是一家投资游戏初创公司的种子前和种子基金，它已经筹集了 5000 万美元的新基金。不过，该公司拒绝置评。

然而，Sisu 的保罗·布拉吉尔(Paul Bragiel)确实评论说，该公司在夏季进行了几笔投资，包括 Zynga 创始人的一家新企业。Sisu 团队成员是 Remedy、Unity、Disney、Niantic、Funcom 和 EA 等公司的企业家、经营者和投资者。

该基金迄今已进行了 14 项投资。一半投资于现有的投资组合公司，如 Resolution Games 和 Lightheart Entertainment，其余投资于新加入的公司，包括 Theorycraft Games、Playco 和 Core Loop。

Bragiel 表示，Sisu fund III 的投资者群体几乎完全来自游戏行业，包括 Supercell 这样的全球工作室，索尼和育碧这样的平台持有者和发行商，以及工作室的个人创始人。

“许多来自 Sisu 家族的创始人已经开始了多项成功的投资，这些是我们喜欢支持的创始人。我们的目标是，如果我们投资一个创始人一次，我们希望支持他们创办的所有未来的公司，”他说。

Sisu 表示，它在产品准备好和指标到位之前就进行投资，它还以“工作室建造者”的形式与个人创始人合作。

最近的例子包括 Lightheart Entertainment 的第一款游戏《自动开火先生》，以及 Mainframe Industries 尚未宣布的云原生 MMO。

到目前为止，该基金已经有八次退出，包括 Zynga 以 7 亿美元收购小巨人游戏公司，以及 King 收购 Hatrabbit，Next Games 收购赫尔辛基 Gameworks 和 Flaregames 收购 Kopla Games。
