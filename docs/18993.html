Skyflow 的数据隐私 API 业务融资 4500 万美元 B 轮 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/19/skyflows-data-privacy-api-business-raises-45m-series-b/?guccounter=1

企业数据隐私产品市场正在升温，今天早上 Skyflow 宣布了 4500 万美元的 B 轮融资，就在一天前 TripleBlind 宣布了 2400 万美元的 B 轮融资。

自 2020 年初种子融资以来，TechCrunch 一直在覆盖的 Skyflow 提供一种 API 交付的服务，为客户存储个人身份信息(PII)和其他敏感数据。

当我们上次讨论 Skyflow 时，该公司刚刚在 2020 年末完成了 1750 万美元的首轮融资。此后，它扩大了产品组合，加入了一项专注于金融科技数据的服务，帮助金融科技公司存储信用卡和其他敏感信息。该公司还为医疗保健公司提供各种服务。

Insight Partners 领导了新的融资活动，其中包括来自与桑坦德银行相关的风险部门的资金。该公司在一份新闻稿中表示，该初创公司现已筹集到 7000 万美元。

像许多 API 交付的公司一样，Skyflow 希望将一个困难的问题抽象成一个开发者挂钩。或者，更简单地说，Skyflow 希望为其客户处理存储 PII 和其他形式的潜在放射性数据的所有困难的隐私、加密和数据问题。(如果你好奇的话，Skyflow 使用多态加密构建的零信任保险库。)

在接受 TechCrunch 采访时，Skyflow 联合创始人兼首席执行官 Anshu Sharma 强调，他的公司正在努力确保能够处理数据驻留要求和其他相关监管事宜，以便世界各地的公司可以使用其服务，而不会引起任何特定政府的愤怒。

到目前为止，这家初创公司的模式取得了一些成功，声称在过去的九个月中实现了“8 倍的增长”。不清楚这种增长是从什么基础上开始的，以及这种增长将公司的收入增长率扩大到什么水平。B 轮大概是我们对创业公司要求更严格的指标的点；这将是 Skyflow 的最后一轮，我们将在没有更难的数字的情况下讨论它。

这家初创公司目前约有 65 名员工，预计到 2022 年底将增加一倍。

Skyflow 关闭 b 轮时不需要筹集新的资金，毕竟距离上一轮融资还不到一年。该公司的账面上仍有之前投资的资本——每 Sharma 约 1100 万美元。这一数据很重要，因为它暗示该公司在银行账户缩水的情况下进行了融资，这可能会让它在融资中发挥杠杆作用。

该公司拒绝公开透露估值，但考虑到 4500 万美元的融资，我们预计该公司现在已经轻松进入九位数的区间，尽管尚不清楚它会在这个区间内停留多久。PitchBook 数据显示在这轮融资之前，Skyflow 的价值不到 1 亿美元，所以如果我们是猜测型的，3 亿至 5 亿美元的估值对其 B 轮融资来说是正确的。

数据隐私问题不小。TripleBlind 和 Skyflow 并不是唯一致力于这一问题的公司，尽管它们都有面向医疗保健的产品出售。当然，通过新的融资，Skyflow 有足够的现金来证明它已经为目标市场制造了合适的产品，因此不会发现自己最终被归入某个更大的技术商店。

从我们正在讨论的初创公司的角度来看，公司只是在为自己积累更多的数据，久而久之。如果不是这样，我们就不会看到 Databricks 像现在这样迅速地扩大其收入和估值。数据越多，风险越大。风险越大，对注重隐私的软件的需求越大。从长期来看，专注于解决数据存储和安全问题的初创公司正在走下坡路。
