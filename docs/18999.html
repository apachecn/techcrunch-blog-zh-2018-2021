我们能从中国手机游戏经济中学到什么

预算高达数百万，视频游戏不是小生意。快速增长的手机游戏行业正在形成一个更加有利可图的市场。

这在中国庞大的手机游戏市场尤其如此，但该国严格的监管要求让寻求进入的国际游戏开发商踌躇不前。

但移动广告平台 Mintegral 美国区总经理杰夫苏认为，中国出版商如何应对监管环境，还有很多值得学习的地方。

Sue 在 TechCrunch+客座博文中深入探讨了中国移动游戏行业，概述了全球发行商如何通过寻求当地合作伙伴关系、关注中国新兴小城市以及研究新兴社交媒体趋势，在中国成功进行广告宣传并占领市场份额。

需要资金的高科技初创企业应该考虑联邦政府的资助

到目前为止，2021 年似乎是筹集创业资金的黄金时代，但深度科技创业公司仍面临挑战，因为很少有风投有耐心或风险偏好进行实验——或等待投资回报。

然而，根据 CoCoPie 联合创始人兼首席技术官沈锡鹏的说法，深度科技创业公司有一个资本来源，也可以带来相当大的声誉:联邦拨款。

“从成千上万的美国申请人中脱颖而出，表明你的创新具有强大的技术和商业价值，以及对美国经济产生广泛影响的潜力，”他写道。

在一篇详细的帖子中，沈概述了他们接受小企业创新研究/小企业技术转移(SBIR/STTR)计划的过程，并指出他的公司正在战术性地使用资金“将我们的技术转化为最小可行的产品”

极度活跃的拉丁美洲开放银行市场:该地区是如何被资本化的

图片来源:布莱斯·德宾

拉丁美洲正非常迅速地成为金融科技初创公司的温床，风投们，不管是否精明，都在争相投资。

推动这一增长的因素有很多，但技术的一个重要焦点似乎是创新的核心:围绕开放式银行构建的 API。

Prometeo 的联合创始人兼首席业务发展官希梅纳·埃勒曼表示，在传统的限制性银行环境中，像 Nubank 和 Rebaja Tus Cuentas 这样的初创公司正在彻底改变银行和金融服务，他们的努力可以通过使用开放的银行 API 来实现。

“一个开放的 API 生态系统，其中第三方服务提供商从金融机构访问消费者数据，这意味着金融服务将得到优化，以降低交易成本和时间，并改善用户体验，”她在 TechCrunch+客座博文中写道。

Rad Power Bikes 创始人 Mike Radenbaugh 为电动自行车革命推波助澜

图片来源:布莱斯·德宾

电动自行车在新冠肺炎疫情期间大受欢迎，该行业没有一家美国公司比 Rad Power Bikes 今年筹集的 1.5 亿美元资金更多。

这家直接面向消费者的公司口袋里装满了现金，正在寻求积极扩大其国际业务和规模，同时专注于迄今为止帮助其发展的一件事:“用户友好”，创始人迈克·拉登博说。

在与丽贝卡·贝兰的自由采访中，拉登博谈到了电动自行车革命，供应链的多样化，以及电动自行车销售的繁荣是否会持续。

中国不断变化的监管环境并没有阻止风险资本的狂热

当中国实施任何新的监管规定时，整个创业生态系统变得极度紧张——毕竟，他们的生计依赖于此。

但风险投资在中国的势头似乎没有放缓。亚历克斯·威廉和安娜·海姆今天早上发现，2021 年第三季度是迄今为止中国创业公司最好的一个季度。

“中国政府未来的监管活动会影响未来的创业投资吗？我们不知道。但投资者正在像他们一样下注，这一点值得注意。”
