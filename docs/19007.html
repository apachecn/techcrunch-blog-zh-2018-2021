由 QED 和 Kaszek 牵头，巴西 Hash 为其白标支付软件融资 4000 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/20/brazils-hash-raises-40m-in-round-led-by-qed-and-kaszek-for-its-white-label-payments-software/

支付基础设施 fin techHash今天宣布，在 QED 投资者和卡泽克共同牵头的 C 轮融资中筹集了 4000 万美元。

仅在六个月前，这家巴西初创公司在由弗吉尼亚州亚历山大市的 QED 牵头的 B 轮融资中筹集了 1500 万美元，自 2017 年成立以来，融资总额接近 5900 万美元。Endeavor Scale-Up Ventures 也进行了投资。

Hash 为希望提供银行服务的非金融 B2B 公司提供“端到端”支付基础设施— 从销售点(POS)软件和移动应用程序到仪表盘和支付。其 白标软件构建定制支付服务，专注于拥有大型客户链和合作伙伴的 B2B 公司。

Hash 的创始人兼首席执行官 joo Miranda 工程师表示，自该公司上次融资以来，交易数量增长了“6 倍”,客户数量翻了一番，达到 16 家。在过去的一年里， Hash 的 POS 交易量增长了 10 倍，Miranda 预计到年底总支付额将超过 2.75 亿美元。

该公司自称的使命是“分散”金融服务，重点是中小企业。

“我们为 B2B 公司提供金融基础设施，将支付解决方案嵌入他们自己的定制生态系统，”他说。“通过这样做，这些公司为其生态系统内的中小型企业提供了更好、更具竞争力的解决方案。”

Hash 的 16 个直接客户使用其基础设施为自己的客户提供金融服务，这些客户是中小企业。目前，超过 16，000 家中小企业正在使用其软件。

Miranda 表示，Hash 旨在提供一个“完整的”支付服务基础设施，帮助企业获得新的收入来源。

他告诉 TechCrunch:“我们明白，市场中的每个领域、每个生态系统都有自己的特殊性和必要性。“这就是为什么我们使用专有技术，能够适应不同的垂直市场。”

Hash 首席执行官兼创始人若昂·米兰达/保罗·维塔莱版权所有

这家初创公司计划利用新资本在它已经涉足的市场上“进一步”发展，无论是地理上还是细分市场。如今，Hash 只在巴西运营。米兰达说，它的目标是扩展到“整个国家领土，这是巨大的，具有巨大的增长潜力”。

未来，它希望在其支付量的基础上实现更多的金融服务，如信贷、贷款和发卡基础设施。和大多数已经融资的初创公司一样，Hash 也计划招聘更多员工。与一年前的 74 人相比，该公司的员工人数已增至近 170 人。它的目标是到年底拥有 190 名员工。

kas zek 的合伙人 Santiago Fossatti 认为，Hash“能够解决行业中的几个问题”,同时仍然为客户的业务“增加真正的价值”。

“Hash 将在支付行业扮演重要角色，并成为拉丁美洲的参考，”他补充道。

QED Investors 合伙人 Mike Packer 认为这是一个“巨大的”市场机会，他认为 Hash 与该领域其他公司的主要区别在于它“能够将最好的技术与如何将支付整合到合作伙伴的商业模式中的独特视角相结合。”
