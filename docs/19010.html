Deci 斥资 2100 万美元，用于开发基于现有数据和计算能力的更好的人工智能模型

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/20/deci-snaps-up-21m-for-tech-to-build-better-ai-models-based-on-available-data-and-compute-power/

构建可用的模型来运行人工智能算法，不仅需要足够的数据来训练系统，还需要合适的硬件来运行它们。但是，由于理论和实践往往不是一回事，数据科学家希望做的事情和他们实际做的事情之间往往存在差距。如今，一家名为 Deci 的初创公司建立了一个深度学习平台，以帮助弥合这一差距——通过建立可以与可用数据和硬件一起工作的模型——在发现财富 500 强科技公司运营基于视频和其他基于计算机视觉的服务的大众市场人工智能产品对其产品有强大吸引力后，该公司宣布提供一些资金。

这家总部位于特拉维夫的初创公司已经获得了 2100 万美元的 A 轮融资，这笔资金将用于继续扩大其产品和客户群。Insight Partners 在这一轮中领先，之前的支持者有 Square Peg、Emerge 和 Jibe Ventures，还有一些新的支持者:Samsung Next、Vintage Investment Partners 和 Fort Ross Ventures。一年前，Square Peg 和 Emerge 领导了 Deci 公司在 T2 的 910 万美元种子融资。它还与其他非战略或金融投资者(但很可能是潜在投资者)密切合作？).英特尔与它合作开发了 MLPerf ，其中 Deci 的技术在英特尔 CPU 上运行时加快了 ResNet-50 神经网络的推理速度。

到目前为止，Deci 一直专注于基于计算机视觉的产品模型，其平台——建立在自己专有的 AutoNAC(自动神经架构构建)技术基础上——能够快速构建并持续更新服务模型，否则可能需要更长时间和大量的反复试验才能设计出来。

例如，一个重要的客户是世界上最大和最知名的视频会议平台之一(不幸的是，名称未披露)，该平台正在使用 Deci 建立人工智能建模，以便用户可以在视频通话中模糊他们的背景。在这里，执行模糊所需的所有计算都发生在“边缘”，在用户自己的基于 CPU 的设备上(也就是说，通常不会针对人工智能工作负载进行优化)。

与 Ran El-Yaniv 和 Jonathan Elial(人工智能专家三人组)共同创建 Deci 的首席执行官 Yonatan Geifman 表示，该计划现在开始从计算机视觉应用扩展到另一个挑战，建立更好的自然语言模型，你可能需要用语音界面运行任何类型的服务，例如，从电话或智能扬声器上的个人助理到基于音频的搜索或任何类型的客户服务界面。

尽管 Deci 通过帮助公司解决在不一定针对人工智能进行优化的设备环境中运行人工智能服务的挑战，赢得了很多业务，但它也发现组织对使用 Deci 为自己的内部计算建立更好的模型很感兴趣，即使理论上他们手头有 GPU 和计算能力来运行任何东西。这利用了长期存在于企业 IT 中的一种有趣的权力平衡，并在今天的人工智能中得到了很好的发挥，企业将试图利用他们手中的资产做更多的事情，而与此同时，他们经常被推动在更新、更昂贵和更强大的设备上投资更多。

Geifman 在一次采访中说，“一直都在进行更大模型的竞赛”，他引用 Nvidia 和微软本月早些时候宣布的新语言模型作为这种演变的一个例子。“所以硬件是不够的。从某种意义上说，也许投资新硬件的竞赛和动力是由硬件制造商自己推动的，但型号正在变得更大。算法和硬件供应之间存在差距。因此，我们需要根据现有的硬件进行一些整合。Deci 正在弥合甚至弥合这一差距。”

随着足够的训练数据成为人工智能的另一个长期问题，Deci 也在努力推动等式的数据侧。Geifman 说，Deci 本质上是建立合成数据集，以便在需要更多数据来建立模型时补充数据。在所有情况下，产品在组织的开发人员环境中工作，数据留在原处，而不去 Deci 或构建模型过程中的任何其他地方。

除此之外，Deci 还利用 AutoNAC 生产更多产品。其中最新的是 DeciNets，Deci 将其描述为“一系列计算机视觉模型”，它们基本上跳过了从头开始构建模型的一些工作，因此运行时使用的计算能力更少。

Insight Partners 董事总经理 Lonne Jaffe 在一份声明中说:“Deci 处于人工智能和深度学习加速的前沿，拥有高度差异化的技术，让客户可以优化速度惊人的深度学习模型，以适应任何硬件平台的推理。”。“我们很高兴成为 Deci 扩展之旅的一部分，并期待支持公司的快速发展。”Jaffe 在这一轮加入了董事会。
